📐 Responsive Spacing

Padding / margin breakpoint variants — Issue #10065

How It Works

Base (no prefix) .ease-sm-* ≥640px .ease-md-* ≥768px .ease-lg-* ≥1024px .ease-xl-* ≥1280px

Resize your browser to see the padding/margin change at each breakpoint. The examples below use padding-4 → md:padding-8 → lg:padding-12 and margin-4 → md:margin-8 → lg:mx-auto.

Responsive Padding

Classes: ease-padding-4 ease-md-padding-8 ease-lg-padding-12

padding: 4 → md:8 → lg:12

Directional: ease-pt-4 ease-md-pr-8 ease-lg-pb-12 ease-xl-pl-4

pt:4 → md:pr:8 → lg:pb:12 → xl:pl:4

Axis: ease-px-4 ease-md-px-8 ease-lg-px-12

px:4 → md:px:8 → lg:px:12

Responsive Margin

ease-margin-4 ease-md-margin-8 ease-lg-mx-auto (outer bg shows margin effect)

margin:4 → md:8 → lg:mx-auto

Bottom margin only: ease-mb-4 ease-md-mb-8 ease-lg-mb-12

mb:4 → md:mb:8 → lg:mb:12

All Breakpoint Variants (Generated Pattern)

Each of the 4 breakpoints generates the following classes (shown for md here): 

/* Padding generics */
.ease-md-padding-{1,2,3,4,6,8,10,12}
/* Padding directional */
.ease-md-pt-4  .ease-md-pr-4
.ease-md-pb-4  .ease-md-pl-4
/* Padding axis */
.ease-md-px-4  .ease-md-py-4
.ease-md-px-8  .ease-md-py-8
/* Margin generics */
.ease-md-margin-{1,2,3,4,6,8}
.ease-md-margin-auto
/* Margin axis + directional */
.ease-md-mx-auto  .ease-md-my-{4,8}
.ease-md-mt-4  .ease-md-mr-4
.ease-md-mb-4  .ease-md-ml-4

Replace md → sm / lg / xl for other breakpoints.

HTML Structure

<!-- Responsive padding -->
<div class="ease-padding-4 ease-md-padding-8 ease-lg-padding-12">
  Padding scales up at each breakpoint.
</div>

<!-- Responsive margin + centering -->
<div class="ease-margin-4 ease-md-mx-auto ease-md-margin-8">
  Centered card on tablet+.
</div>

<!-- Responsive axis padding -->
<div class="ease-padding-4 ease-md-px-8 ease-lg-px-12">
  Horizontal padding scales, vertical stays.
</div>
