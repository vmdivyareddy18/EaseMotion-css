Issue #9857 — Logical (Flow-Relative) Property Utility Classes

Proposed additions to core/utilities.css: margin-inline, margin-block, padding-inline, padding-block, and inset logical utilities that respect writing-mode and direction.

How Logical Properties Work

Logical properties respect the document's writing direction. In LTR, inline-start = left; in RTL, it flips to right.

LTR (left-to-right)

mi-s (start margin)

RTL (right-to-left)

mi-s flips!

Margin Inline

.ease-mi-s (start) / .ease-mi-e (end) + scale: {1,2,3,4,6,8,10,12}

ease-margin-inline-auto centers block elements in both LTR and RTL.

Margin Block

.ease-mbs (block-start) / .ease-mbe (block-end) + scale: {1,2,3,4,6,8,10,12}

block-start = top, block-end = bottom (same in both LTR and RTL).
