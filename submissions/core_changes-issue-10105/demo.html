✂️ Text Truncation

Single & multi-line truncation utilities — Issue #10105

Single Line

ease-truncate

This is a very long title that should never wrap to a second line no matter how much text there is in this element

Two Line Clamp

ease-truncate-2

This card heading is too long and should clamp at exactly two lines before showing the ellipsis truncation

Three Line Clamp

ease-truncate-3

This description text can span up to three lines before being truncated with an ellipsis character at the very end of the visible text. Any content beyond the third line is hidden.

Reset

ease-truncate-none

This text has truncation reset and will show the full content regardless of length. Useful when toggling between truncated and expanded states with JavaScript.

HTML Structure

<!-- Single line -->
<p class="ease-truncate">Long text...</p>

<!-- Two line clamp -->
<h3 class="ease-truncate-2">Long heading...</h3>

<!-- Three line clamp -->
<p class="ease-truncate-3">Long description...</p>

<!-- Reset -->
<p class="ease-truncate-none">Full content</p>
