Issue #9880 — Font Variation Utility Classes

Proposed additions to core/utilities.css: font-size-adjust and font-stretch utilities.

Font Size Adjust

.ease-font-size-adjust-{none, from-font, 05, 06, 07}

Normalizes the visual size of fallback fonts when the primary font fails to load, preventing layout shifts.
