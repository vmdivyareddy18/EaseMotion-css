Issue #9876 — Object Position & Object View Box Utility Classes

Proposed additions to core/utilities.css: utilities for controlling how replaced elements fit within their box.

Object Position

.ease-object-{center, top, bottom, left, right, top-left, top-right, bottom-left, bottom-right}

Controls alignment of replaced elements (<img>, <video>) within their box when using object-fit. Each cell uses object-fit: cover with the specified position.

Object View Box

.ease-object-view-box-none

Resets the object view box to none — useful for undoing custom inset() cropping applied via inline styles.
