Issue #9870 — Table Utility Classes

Proposed additions to core/utilities.css: table-layout, border-collapse, border-spacing, and caption-side utilities.

Table Layout

.ease-table-auto / .ease-table-fixed

ease-table-fixed forces equal-width columns regardless of content; ease-table-auto lets content determine widths.

ClassShort ContentLong Content That Would Normally Expand

ease-table-autoItem BThis column would be wider in auto mode
