Color Variants

Success

Error

Something went wrong. Please try again.

Warning

Info

New updates are available. Refresh to see them.

Size Variants

ease-toast-sm

Compact toast for short messages.

default

Standard toast for most use cases.

ease-toast-lg

Large toast for longer messages or when you need more visual weight.
