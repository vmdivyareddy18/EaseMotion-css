List Style Position

.ease-list-inside / .ease-list-outside

Inside places the marker within the list item content; outside places it to the left.

  • Inside — this text wraps under the bullet itself
  • Notice the alignment
ease-list-inside
  • Outside — bullet is in the margin
  • Default list behavior
ease-list-outside
