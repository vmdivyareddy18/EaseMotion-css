Bundle Import Reference

This page documents the difference between the two EaseMotion CSS entrypoints, addressing #27879 — where easemotion/all.css was undocumented as a partial bundle relative to easemotion.css.

Entrypoint Coverage Recommended use
easemotion.css Full component surface (current, always up to date) Default choice for new projects
easemotion/all.css Legacy / partial bundle — may lag behind newly added components Existing projects already depending on this entrypoint
Heads up: easemotion/all.css is not guaranteed to include every component shipped in easemotion.css. Newer components (e.g. announce-bar, avatar, breadcrumb, btn-magnetic, command-palette, compare-table, connection-status) may only be available through the main entrypoint until the legacy bundle is refreshed.
