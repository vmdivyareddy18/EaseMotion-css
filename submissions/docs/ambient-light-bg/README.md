# Ambient Light Background

## Overview

This submission proposes a subtle ambient light background for the EaseMotion CSS documentation in light mode.

Instead of using a flat white canvas, the demo introduces soft purple radial gradients that create a calmer, more polished reading experience while preserving readability.

## Features

- Soft off-white background
- Ambient purple radial gradients
- Lightweight CSS-only implementation
- Optional subtle animation
- Supports `prefers-reduced-motion`
- No JavaScript required

## Usage

Open `demo.html` in any modern browser.

The demo recreates a documentation page to demonstrate how the ambient background behaves with common documentation elements such as:

- Navigation bar
- Sidebar
- Headings
- Tables
- Code blocks
- Callouts
- Cards

## Why is this useful?

A softer ambient background makes long documentation sessions more comfortable by reducing the harshness of a pure white page.

The effect also gives the documentation a more modern, premium appearance while remaining subtle enough that it never distracts from the content.

Since the implementation is CSS-only, it is lightweight, easy to maintain, and can be integrated into the existing documentation stylesheet with minimal changes.

## Files

- `demo.html` – Demonstrates the proposed documentation layout with the ambient background.
- `style.css` – Contains the complete CSS implementation for the ambient background and supporting styles.
