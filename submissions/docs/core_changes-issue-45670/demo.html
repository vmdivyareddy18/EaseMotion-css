Issue #45670 — Reduced Motion Fallback Fix

This page demonstrates the fallback rule to fully disable CSS animations, transitions, and smooth scrolling for users who prefer reduced motion.

System Status: Normal Motion Enabled 🏃‍♂️ Reduced Motion Enabled 🛡️ (Animations Disabled)

Below is a box animated with CSS @keyframes (bouncing and spinning) and equipped with hover transitions (scaling and color shift). Hover over it or test it by toggling your system's "Reduce motion" preference.

Hover Me

To test this in Chrome/Edge DevTools: press Ctrl+Shift+P (or Cmd+Shift+P on Mac), type "prefers-reduced-motion", and select "Emulate CSS prefers-reduced-motion: reduce".
