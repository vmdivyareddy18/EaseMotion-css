# Toast / Notification Component — Issue #30377

CSS-only toast/notification component with slide-in animation, multiple positions, and semantic variants.

## Classes

| Class | Purpose |
|-------|---------|
| `.ease-toast-container` | Fixed positioned container |
| `.ease-toast-container-tl/tr/bl/br/tc/bc` | Position: top-left, top-right, bottom-left, bottom-right, top-center, bottom-center |
| `.ease-toast` | Individual toast item |
| `.ease-toast-body` | Text content wrapper |
| `.ease-toast-title` | Bold title line |
| `.ease-toast-desc` | Description line |
| `.ease-toast-icon` | Icon area |
| `.ease-toast-close` | Close button |
| `.ease-toast-success` | Green success variant |
| `.ease-toast-error` | Red error variant |
| `.ease-toast-warning` | Amber warning variant |
| `.ease-toast-info` | Blue info variant |
| `.is-dismissing` | Triggers fade-out exit animation |

## CSS Variables

- `--ease-toast-gap`, `--ease-toast-min-width`, `--ease-toast-max-width`
- `--ease-toast-bg`, `--ease-toast-text`, `--ease-toast-border`
- `--ease-toast-radius`, `--ease-toast-shadow`, `--ease-toast-duration`
