ease-zoom-in Fix

Preventing scrollbar flicker during zoom animation

With Fix (No Flicker)

Uses clip-path: inset(0) to contain the transform.

Hero with Fixed Zoom Animation

No scrollbar flicker thanks to clip-path containment

How to Apply the Fix

/* Contain the transform within bounds */
.hero-container {
  overflow: hidden;
}

.ease-zoom-in {
  clip-path: inset(0);
}
