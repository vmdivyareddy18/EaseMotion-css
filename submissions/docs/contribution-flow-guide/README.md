# Contribution Flow Guide

## Purpose

This submission provides a beginner-friendly guide explaining the expected contribution workflow.

## Problem

New contributors may not immediately understand that only the `submissions/` directory should be modified.

## Proposed Improvement

Provide a visual example of the contribution workflow and required folder structure.

## Folder Structure

```
submissions/
└── docs/
    └── contribution-flow-guide/
        ├── demo.html
        ├── style.css
        └── README.md
```

## How to View

Open `demo.html` in your browser.
