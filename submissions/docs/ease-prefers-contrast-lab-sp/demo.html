Copy-ready CSS snippet

@media (prefers-contrast: more) {
  .ease-hover-glow:hover {
    filter: none !important;
    outline: 2px solid #0f172a;
    outline-offset: 2px;
  }

  .ease-card-glow:hover {
    box-shadow: none;
    border: 2px solid #0f172a;
  }

  .ease-card {
    border-width: 2px;
    border-color: #0f172a;
    box-shadow: none;
  }

  :focus-visible {
    outline-width: 3px;
    outline-offset: 3px;
  }
}

Contrast guidance

What changes?

  • Remove soft glows and drop-shadows.
  • Increase border width to 2px+.
  • Thicken focus outlines for keyboard users.
  • Prefer solid colors over transparency.

Media query

  • prefers-contrast: more — user wants higher contrast.
  • prefers-contrast: less — user prefers softer UI.
  • Chrome 96+, Firefox 101+, Safari 15.4+.

Motion + contrast

  • Glow animations reduce perceived contrast.
  • Pair with prefers-reduced-motion when needed.
  • Test with OS high-contrast mode enabled.
