Usage guidance

When injection happens

  • ≤ 20 tabs → pure CSS, no JS rules needed.
  • > 20 tabs → tabs.js adds data-tabs-id and injects rules for tabs 21+.
  • Each extra tab adds 4 CSS rules (focus, checked, underline, panel).

Underline behavior

  • updateUnderlineWidth() sets pixel width and left offset.
  • Recalculates on tab change and window resize (150ms debounce).
  • Use horizontal scroll nav for high tab counts.

High-tab-count tips

  • Prefer shorter tab labels for 25+ items.
  • Include core/tabs.js after your tab markup.
  • Use keyboard: Tab to focus inputs, arrow keys between radios.
  • Consider grouping or a dropdown for 30+ tabs in production.
