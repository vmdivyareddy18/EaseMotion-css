# E-Commerce Toggle Showcase (SB)

## Overview

This project demonstrates an interactive e-commerce pricing toggle built using HTML and CSS. Users can switch between monthly and yearly billing plans with a smooth pricing update.

## Features

- Monthly and yearly billing toggle
- Three pricing plans
- Responsive layout
- Smooth hover animations
- Glassmorphism-inspired cards
- Keyboard-accessible toggle using `:focus-visible`
- Self-contained demo with no external dependencies

## Folder Structure

```
e-commerce-toggle-sb/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern web browser.
2. Use the toggle to switch between monthly and yearly pricing.
3. Hover over the pricing cards to view the animation.
