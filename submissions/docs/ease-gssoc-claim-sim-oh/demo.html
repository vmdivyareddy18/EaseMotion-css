⚠️ Common Mistakes

  • Claiming 3+ issues - Will be rejected, follow the 2-issue limit
  • Going inactive - 24-hour cooldown triggers automatic unassign
  • No labels - Issues without proper labels won't be accepted
  • Spam PRs - Low-quality submissions violate the spam policy
  • Missing README - All submissions need documentation
