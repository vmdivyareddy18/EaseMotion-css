Quick Reference Checklist

  • ✅ Only animate properties that trigger composite (opacity, transform)
  • ✅ Use `will-change` sparingly only for elements that will animate
  • ✅ Reduce/remove animations for users with `prefers-reduced-motion`
  • ✅ Use CSS animations over JS where possible (compositor thread)
  • ✅ Test your page on Lighthouse or WebPageTest
  • ✅ Keep animation durations short and purposeful (300-500ms)
