# Web Vitals & Animations Guide

A core web vitals impact of animations guide for EaseMotion CSS under submissions/docs/!

## What does this do?

This is a simple, self-contained documentation lab where users can:
- Compare 'heavy motion' vs 'optimized motion' UI patterns side by side
- See how different animation techniques affect LCP, CLS, and INP
- Learn which EaseMotion patterns are safer for production
- Apply recommended best practices for performant animations

## Features

- 📊 Side-by-side before/after panels (Heavy Motion vs Safe Motion)
- 🐢 LCP impact examples (lazy/heavy entrance timing and load behavior)
- 📐 CLS impact examples (show shift from animated position/size changes)
- 🖱️ INP impact examples (show click animation delay on interaction)
- 📝 Safe motion recommendations using transform and opacity
- 💡 Guidelines on limiting simultaneous animations
- ✨ "Copy-ready" safe class combinations snippets
- 📋 Educational performance checklist for production pages
- ♿ Accessible and clear UI!
