# Animation Performance Best Practices Guide

## Description

This example demonstrates simple CSS animation techniques that promote better rendering performance.

## Files

- demo.html
- style.css

## Best Practices

- Animate `transform` instead of `left`, `top`, `width`, or `height`.
- Use `opacity` for fade animations.
- Keep animation duration reasonable.
- Avoid unnecessary infinite animations.
- Test animations on different devices.

## How to Run

Open `demo.html` in your browser and observe the smooth animation created using `transform` and `opacity`.

## Benefits

- Better rendering performance
- Smoother animations
- Beginner-friendly reference
