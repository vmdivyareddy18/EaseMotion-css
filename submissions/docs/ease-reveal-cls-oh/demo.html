Fixed HTML (With Dimensions)

  <img src="image.jpg" alt="..." width="400" height="300">
  <div class="skeleton"></div>
</div>

<style>
.img-container {
  position: relative;
  aspect-ratio: 4 / 3; /* Reserve space */
}
.skeleton {
  position: absolute;
  inset: 0;
  background: linear-gradient(90deg, #f0f0f0 25%, #e0e0e0 50%, #f0f0f0 75%);
  animation: shimmer 1.5s infinite;
}
img[src] + .skeleton { display: none; } /* Hide when loaded */
</style>

Fix Patterns

1. Explicit Width/Height

Add width and height attributes to reserve space

<img src="..." width="400" height="300">

2. Aspect Ratio CSS

Use CSS aspect-ratio to reserve space

.img-container {
  aspect-ratio: 4 / 3;
}

3. Skeleton Placeholder

Show loading state while image loads

.skeleton {
  position: absolute;
  inset: 0;
  background: #e0e0e0;
  animation: shimmer 1.5s infinite;
}

4. Delayed Reveal

Wait for images to load before revealing

window.addEventListener('load', () => {
  document.querySelectorAll('.ease-reveal')
    .forEach(el => el.classList.add('revealed'));
});

Best Practices

  • Always specify width and height attributes on images
  • Use CSS aspect-ratio to reserve space
  • Add skeleton loading states for visual feedback
  • Delay reveal animations until images are loaded
  • Test with throttled network (Slow 3G) in DevTools
  • Monitor CLS in Google PageSpeed Insights
