Safari-safe CSS

.flip-container {
  perspective: 1000px;
}
.flip-inner {
  transform-style: preserve-3d;
  transition: transform 0.6s;
}
.flip-face {
  position: absolute;
  inset: 0;
  backface-visibility: hidden;
  -webkit-backface-visibility: hidden; /* Safari still requires this prefix */
}
.flip-back {
  transform: rotateY(180deg);
}

Required properties checklist

  • perspective set on the outer container (not the flipping element itself)
  • transform-style: preserve-3d set on the inner flipping element
  • backface-visibility: hidden AND -webkit-backface-visibility: hidden set on both faces — Safari still needs the prefixed version
  • Faces positioned with position: absolute; inset: 0; so they overlap correctly during the flip
