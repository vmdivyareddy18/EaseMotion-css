# Footer Contrast Fix

### 1. What does this do?
This submission fixes the low-visibility issue of the footer text when the website is viewed in Light Mode.

### 2. How is it used?
It is a self-contained CSS override. When the `style.css` in this folder is loaded, it automatically adjusts the text and background variables to ensure high contrast against light backgrounds. No changes to existing HTML markup are required.

### 3. Why is it useful?
It improves accessibility for users who prefer Light Mode, ensuring that the footer content meets WCAG contrast standards and remains readable.
