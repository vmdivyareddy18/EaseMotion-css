Package release check

npm metadata is behind the repository version

The repository declares EaseMotion CSS version 1.2.0, while the npm registry currently reports 1.0.0 as the latest published package.

Repository package.json 1.2.0
npm latest 1.0.0
Reproduction command npm view easemotion-css name version dist-tags versions --json

This audit supports issue #40040 and gives maintainers a compact reference for aligning the published package, README release notes, and repository metadata.
