CPU / paint cost guidelines
  • Each infinite loop triggers continuous compositor + paint work every frame.
  • Cap dashboard loops to 1–3 simultaneous infinite animations.
  • Use --ease-animation-iterations: 3 for attention without eternal cost.
  • Prefer ease-fade-in / ease-slide-up for one-shot entrances.
  • Test on throttled CPU in DevTools → Performance → CPU 4× slowdown.
  • Always verify prefers-reduced-motion behavior.
