# Basic Animation Examples

## Purpose

This submission provides beginner-friendly examples demonstrating some commonly used EaseMotion CSS animation classes.

## Included Examples

### Fade In

```html
<div class="ease-fade-in">
    Fade In
</div>
```

---

### Slide Up

```html
<div class="ease-slide-up">
    Slide Up
</div>
```

---

### Hover Grow

```html
<div class="ease-hover-grow">
    Hover Me
</div>
```

---

## Files

- demo.html
- style.css

## How to Run

Open `demo.html` in your browser.

## Benefits

- Easy for beginners to understand.
- Demonstrates common animation classes.
- Can serve as a quick reference without modifying the project's main documentation.
