EaseMotion CSS

❌ Broken Layout (100vh)

Uses height:100vh. On iOS Safari, the address bar expanding or collapsing can cause the layout to jump.

✅ Fixed Layout (100dvh)

Uses modern viewport units like 100dvh for a stable full-screen experience.

Recommended Fixes

Use Dynamic Viewport

.hero{
height:100dvh;
}

Fallback

.hero{
height:100svh;
min-height:-webkit-fill-available;
}

Safe Area Insets

padding-top:
env(safe-area-inset-top);

padding-bottom:
env(safe-area-inset-bottom);

100vh 100dvh
