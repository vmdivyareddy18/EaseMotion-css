# Mobile Responsive Demo

## Overview

This submission demonstrates a mobile-responsive version of the EaseMotion CSS demo page. It showcases how the layout adapts to smaller screen sizes while maintaining a clean and user-friendly interface.

The implementation focuses on improving the responsiveness of the navigation bar, hero section, button groups, and grid layouts for mobile devices.

---

## Features

- Responsive navigation with wrapped menu items
- Improved spacing for mobile viewports
- Mobile-friendly hero section
- Responsive button layout
- Single-column grid layout on smaller screens
- Better readability and usability across different screen sizes

---

## Usage

Open `demo.html` in any modern web browser.

To view the responsive behavior:

1. Open the page in Chrome, Edge, or another modern browser.
2. Open Developer Tools (`F12`).
3. Enable Device Toolbar (`Ctrl + Shift + M`).
4. Test different viewport sizes (e.g. **504 × 925**, **390 × 844**, or **375 × 667**).

The layout automatically adapts based on the screen width.
