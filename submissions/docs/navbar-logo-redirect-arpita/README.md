# Navbar Logo Redirect to Landing Page
## Overview
This demo showcases the expected navigation behavior where clicking the EaseMotion CSS logo redirects users to the Landing/Home page instead of the Documentation page.

---

## Features
- Pure HTML & CSS implementation
- Responsive navigation layout
- Logo redirects to the Landing page
- Documentation remains accessible through a dedicated navigation link
- CSS-only modal demonstration
- Modern and clean UI

---

---

## What does this do?
Demonstrates the correct navigation behavior by making the project logo return users to the Landing/Home page.

---

## How is it used?
```html
<header class="navbar">
    <a href="#home" class="logo">EaseMotion CSS</a>
</header>
```

---

## Why is it useful?
Redirecting the logo to the Landing page follows standard web navigation practices, improves usability, and provides users with an intuitive way to return to the main page while keeping documentation accessible through a separate navigation item.
