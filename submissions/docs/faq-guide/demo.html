EaseMotion CSS - Frequently Asked Questions

Quick answers for new contributors.

How do I install EaseMotion CSS?

Follow the installation instructions provided in the project documentation.

Can I use a CDN?

Yes. Use the official CDN documented by the project.

Where should I add my contribution?

Add new work only inside the submissions/ directory unless maintainers request otherwise.

How do I preview my example?

Open demo.html in your browser.

How do I create a Pull Request?

Create a feature branch, commit your changes, push it, and open a Pull Request on GitHub.
