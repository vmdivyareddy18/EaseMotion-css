Deep-linking guidance

Why hash modals?

  • Shareable URLs open the same dialog state.
  • Browser back button closes the modal naturally.
  • Works without a router or state library.
  • CSS :target provides a no-JS fallback.

Accessibility checklist

  • Set role="dialog" and aria-modal="true".
  • Label with aria-labelledby pointing to a heading.
  • Trap focus and restore it on close.
  • Lock body scroll while open.

modal.js responsibilities

  • Sync .is-active on hashchange.
  • Focus the .ease-modal container on open.
  • Close on backdrop click and Escape.
  • Wrap Tab inside the overlay.
