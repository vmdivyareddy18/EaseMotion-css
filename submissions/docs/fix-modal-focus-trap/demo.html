Modal Focus Trap Fix Showcase

This showcase demonstrates the focus trap leakage bug and how the fix forces focus back inside the active modal overlay.

Instruction:
  1. Open the demo modal.
  2. Click the "Background interactive input" (or focus it programmatically).
  3. Press Tab.

Status: Fix is active. Tab will trap you inside the modal even if you focus this background input.
