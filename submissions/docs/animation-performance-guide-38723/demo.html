Animation Performance Comparison

Open your browser's Developer Tools (Performance tab) and record while hovering over the boxes below to see the difference in Layout/Paint vs. GPU Compositing.

❌ The Bad Way

Animates margin-left and width. This triggers Layout and Paint on the CPU for every frame, causing "jank" on mobile devices.

Hover Me
.box-bad:hover {
  width: 150px;
  margin-left: 200px;
}

✅ The Good Way

Animates transform: translate3d() and scaleX(). This runs entirely on the GPU (Compositor Thread), guaranteeing 60FPS.

Hover Me
.box-good:hover {
  transform: translate3d(200px, 0, 0) scaleX(1.5);
}
