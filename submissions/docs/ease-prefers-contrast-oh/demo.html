Understanding prefers-contrast

The prefers-contrast media query detects if users prefer higher contrast. Use it to adjust borders, focus rings, and shadows for better readability.

CSS 
/* Default subtle border */
.card { border: 1px solid rgba(255,255,255,0.1); }

/* High contrast: stronger border */
@media (prefers-contrast: more) {
  .card { border: 2px solid currentColor; }
}

📋 Best Practices

  • Use prefers-contrast: more for higher contrast modes
  • Increase border width and opacity in high contrast
  • Make focus indicators more prominent
  • Reduce or remove glow shadows
  • Ensure text maintains minimum 4.5:1 contrast ratio
  • Test with actual accessibility tools
