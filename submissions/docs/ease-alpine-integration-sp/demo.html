Integration guidance


      

        
x-show + EaseMotion

        

          x-show toggles display but does not replay
          CSS keyframes. Call a retrigger helper in $nextTick after
          opening so ease-fade-in runs every time.
        

      


      

        
x-transition vs EaseMotion

        
    
          
  • x-transition — Alpine's built-in opacity/scale tweens.
    • 
          
  • EaseMotion classes — richer keyframe presets from the framework.
    • 
          
  • Pick one per element; avoid stacking conflicting transitions.
    • 
          
  • Use x-cloak to hide unstyled Alpine markup on load.
    • 
        

      


      

        
Production tips

        
    
          
  • Load Alpine with defer after your markup.
    • 
          
  • Close modals on @keydown.escape.window.
    • 
          
  • Pair role="dialog" + aria-modal for modals.
    • 
          
  • Hover classes work without any Alpine state at all.
