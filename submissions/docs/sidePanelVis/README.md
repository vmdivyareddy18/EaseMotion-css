# Side Panel Visibility Improvements (Light Mode)

## Issue
The documentation sidebar in light mode had low visual distinction from the page background, making navigation less prominent and reducing the visibility of the active navigation item.

## Changes Made
- Added a soft lavender background to the sidebar.
- Improved the active navigation item's background and text contrast.
- Preserved the existing layout and spacing.
- Kept the changes limited to light mode without affecting dark mode.

## Before
- Sidebar blended into the page.
- Active navigation item had lower visual prominence.

## After
- Sidebar is visually separated from the main content.
- Active navigation item is easier to identify.
- Overall navigation experience is improved while maintaining the project's design language.

- `docs/docs.css`
