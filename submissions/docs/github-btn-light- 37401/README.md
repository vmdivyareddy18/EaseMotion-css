# Improved GitHub Outline Button

## Description

This submission improves the visibility and usability of an outline-style GitHub button across both Light and Dark themes.

The component focuses on better contrast, clearer hover feedback, and improved accessibility while maintaining a clean, modern design.

---

## Features

- Better border visibility in Light Theme
- Improved contrast in Dark Theme
- White hover text for better readability
- Smooth hover animation
- Elevated shadow effect
- Rounded pill design
- Theme toggle included for demonstration

---

## Light Theme

- Strong purple outline
- White background
- Dark readable text

Hover

- Purple background
- White text
- Soft shadow
- Lift animation

---

## Dark Theme

- Transparent background
- Soft white border
- White text

Hover

- Purple background
- White text
- Enhanced glow

---

## Usage

```html
<a href="#" class="outline-btn">
    GitHub ↗
</a>
```
