The box animates using --custom-duration via a CSS variable. Changing the variable does not restart the animation — the old speed stays.

✅ The Fix (Animation Restart Helper)

Same box, same CSS variable — but after changing the variable, we call restartAnimation(el) to force the browser to re-evaluate it.

🔧 How the Fix Works

The helper function works in 3 steps:

  1. Temporarily sets animation: none on the element.
  2. Reads offsetWidth to force the browser to flush and recalculate styles (reflow).
  3. Removes the override — the browser re-reads the CSS variable with the new value and restarts the animation.
function restartAnimation(el) {
  el.style.animation = 'none';
  void el.offsetWidth; // force reflow
  el.style.animation = '';
}
