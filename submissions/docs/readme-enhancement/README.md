### ✅ Installation Guide

#### Using CDN

```html
<link rel="stylesheet" href="path/to/easemotion.css">
```

#### Local Installation

```bash
git clone https://github.com/JAY4IGNITE/EaseMotion-css.git
```

or download the CSS files and include them in your project.
