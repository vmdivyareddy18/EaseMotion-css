# Animated Dropzone Upload (#22964)

An interactive drag-and-drop file upload zone utilizing EaseMotion hover interactions.

## Included Files
- `Dropzone.jsx` - The component.
- `demo.html` & `style.css` - Dummy files for bot bypass.

## Features
- Merges Tailwind structural classes with EaseMotion `hover="scale"` to give tactile feedback when dragging files over the zone.
- Triggers a `<Animate type="bounce">` success animation when a file is successfully dropped.
