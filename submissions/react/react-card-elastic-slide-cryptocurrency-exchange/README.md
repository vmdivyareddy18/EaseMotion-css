# React Elastic Slide Card

## Description

Starter template for a cryptocurrency exchange card component built with React.

## Props

| Prop | Description |
|------|-------------|
| title | Coin name |
| subtitle | Coin symbol |
| price | Current price |
| change | 24-hour change |

## Usage

```jsx
<ElasticSlideCard
  title="Bitcoin"
  subtitle="BTC"
  price="$118,000"
  change="+2.1%"
/>
```

## Customize

- Add your own elastic slide animation.
- Improve responsiveness.
- Add accessibility enhancements.
- Style it to match a cryptocurrency dashboard.
