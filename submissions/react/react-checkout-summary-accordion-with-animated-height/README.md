# React Checkout Summary Accordion with Animated Height

A lightweight, reusable React checkout summary accordion featuring smooth animated height transitions. Ideal for e-commerce checkout pages where users can expand or collapse order details.

## Features

- Smooth expand/collapse animation
- Animated height transition
- Responsive layout
- Clean and reusable React component
- Pure CSS styling
- No external dependencies

## Installation

Copy the following files into your project:

- `CheckoutSummaryAccordion.jsx`
- `style.css`

Import both into your React application.

## Usage

```jsx
import CheckoutSummaryAccordion from "./CheckoutSummaryAccordion";
import "./style.css";

function App() {
  return <CheckoutSummaryAccordion />;
}

export default App;
```
