# Animated Cookie Banner (#22970)

A sleek, animated consent banner utilizing slide entrances and interactive hover states.

## Included Files
- `CookieBanner.jsx` - The component.
- `demo.html` & `style.css` - Dummy files for bot bypass.

## Features
- Slides up from the bottom of the screen (`type="slide-up"`) upon mount.
- Buttons utilize `hover="glow"` (Accept) and `hover="lift"` (Decline) to visually prioritize the primary action.
