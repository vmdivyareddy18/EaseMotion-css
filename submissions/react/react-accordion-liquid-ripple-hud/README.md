# React Liquid Ripple Accordion – Futuristic HUD

A reusable React accordion component featuring a Liquid Ripple animation designed for futuristic HUD interfaces. Built with EaseMotion CSS utility classes, responsive layouts and accessible accordion interactions.

---

## Features

- React functional component
- Liquid Ripple animation
- Futuristic HUD inspired design
- Responsive layout
- Accessible accordion interaction
- EaseMotion CSS utility classes
- Lightweight and reusable
- Optional external stylesheet

---

## Usage

```jsx
import LiquidRippleAccordion from "./LiquidRippleAccordion";

function App() {
  return <LiquidRippleAccordion />;
}
```

---

## Styling

Import the stylesheet:

```jsx
import "./style.css";
```

The component uses EaseMotion CSS utility classes including:

- `ease-fade-in`
- `ease-hover-lift`

along with a custom Liquid Ripple animation.

---

## Accessibility

- Uses semantic button elements
- Keyboard accessible
- Expand and collapse interaction
- Responsive layout

---

## Responsive

The accordion automatically adapts spacing and typography for smaller screens while maintaining readability.
