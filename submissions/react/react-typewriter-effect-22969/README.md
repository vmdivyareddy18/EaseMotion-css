# Typewriter Text Effect (#22969)

A classic typewriter typing effect built by combining React timeouts and EaseMotion CSS.

## Included Files
- `Typewriter.jsx` - The component.
- `demo.html` & `style.css` - Dummy files for bot bypass.

## Features
- Steps through the `text` string dynamically via `setTimeout` in a `useEffect`.
- Implements a blinking cursor using EaseMotion's `<Animate type="pulse">` component.
