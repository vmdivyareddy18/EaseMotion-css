# Video Playlist Sidebar with Active Track Glow

Modular copy-paste ready React component featuring built-in state management hooks and an active track neon pulsing effect.

## Component Props

| Prop | Type | Description |
| :--- | :--- | :--- |
| `playlist` | Array | Data array containing track info configurations. |
| `onTrackSelect` | Function | Triggers an event handler on element click events. |
