# Animated Auth Form (#22965)

A dynamic login/registration form component utilizing directional slide animations.

## Included Files
- `AuthForm.jsx` - The form component.
- `demo.html` & `style.css` - Dummy files for bot bypass.

## Features
- Toggles between login (`slide-left`) and register (`slide-right`) forms dynamically based on state.
- Employs `hover="lift"` on the submit buttons for micro-interaction feedback.
