# React Tooltip - Elastic Slide for SaaS Showcase Layouts

A reusable React Tooltip component with a smooth Elastic Slide animation designed for modern SaaS product showcase interfaces.

## Features

- Smooth elastic slide animation
- Clean SaaS dashboard style
- Lightweight React component
- Multiple tooltip positions
- Uses EaseMotion utility classes
- Easy customization


## Installation

Copy the component folder into your React project.
