AnimateGroup

A declarative stagger wrapper for EaseMotion CSS animations. The demo below shows five cards entering with a slide-up motion, a replay control, and a reverse-order toggle that mirrors the React API.

Equivalent React API

<AnimateGroup
  type="slide-up"
  stagger={100}
  delay={200}
  duration="medium"
  reverse={false}
  tag="div"
  className="my-grid"
>
  <Card />
  <Card />
  <Card />
</AnimateGroup>

Design system block

Each child inherits its existing props while AnimateGroup layers on the EaseMotion animation class and computed delay.

Reusable wrapper

Use any valid HTML tag as the group container and keep your original child markup intact.

Safe child merging

Reverse sequence
