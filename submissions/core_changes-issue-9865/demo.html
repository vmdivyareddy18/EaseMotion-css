Issue #9865 — Border Image Utility Classes

Proposed additions to core/utilities.css: border-image gradient shorthands, border-image-slice, and border-image-repeat utilities.

Border Image — Gradient Shorthand

.ease-border-gradient-{primary, success, danger}

Combines border-image-source, border-image-slice, and border-image-width in one declaration.
