# Flexbox Alignment & Layout Utility Mixins

A clean, reusable module for managing flexible grid alignment structures seamlessly within the design system.

## Mixins Included
1. `flex-layout` - Configures a container layout with full alignment configurations.
2. `flex-center` - Shortcut to center inner elements perfectly along both axes.
3. `flex-item` - Allows granular layout sizing configurations directly on child flex elements.
