# Hover Effect SCSS Each Loop

## Overview

This SCSS module generates multiple hover effect utility classes using the `@each` directive, reducing repetitive code and improving maintainability.

---

## Features

- Uses SCSS `@each`
- Generates multiple hover utility classes
- Easy to extend by adding new effects
- Cleaner and reusable SCSS

---

## Usage

```scss
.ease-hover-lift
.ease-hover-scale
.ease-hover-rotate
```

---

## Generated Classes

- `.ease-hover-lift`
- `.ease-hover-scale`
- `.ease-hover-rotate`

---

## Example

```html
<button class="ease-hover-lift">Hover Me</button>
<button class="ease-hover-scale">Hover Me</button>
<button class="ease-hover-rotate">Hover Me</button>
```
