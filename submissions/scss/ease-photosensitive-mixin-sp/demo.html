Copy-ready SCSS usage

@use 'mixin' as psm;

// One-line recipes
.badge-pulse {
  @include psm.photosensitive-pulse();
}

.notify-ping {
  @include psm.photosensitive-ping($duration: 2.4s, $iterations: 2);
}

// Or wrap your own keyframes
@include psm.psm-keyframes-pulse(my-safe-pulse, $min-opacity: 0.6);

.hero-cta {
  @include psm.photosensitive-safe-animation(
    my-safe-pulse,
    $duration: 2s,
    $iterations: 2
  );
}

// Configurable guardrails (optional overrides)
// $psm-opacity-floor: 0.6;
// $psm-max-iterations: 2;
// $psm-duration-floor: 1.2s;

Photosensitive guidance

Flash rate limits

  • WCAG 2.3.1: avoid more than 3 flashes per second.
  • Mixin enforces ≥ 1s cycle duration by default.
  • infinite is capped to 3 iterations max.

Opacity caps

  • Keyframes never drop below $psm-opacity-floor (0.55).
  • Scale pulses capped at 1.35× by default.
  • Prefer soft fades over hard step transitions.

Reduced motion

  • Mixin emits prefers-reduced-motion reset rules.
  • Animations become static; opacity returns to 1.
  • Toggle simulation above to preview the fallback.
