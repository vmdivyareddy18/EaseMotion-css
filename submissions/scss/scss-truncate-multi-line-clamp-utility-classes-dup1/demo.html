This block of text is allowed to be a little bit longer. It will take up exactly four lines before it gets truncated. This is incredibly useful for product description cards in an e-commerce grid, where you want to show a good amount of context but you still need all the cards in the row to be exactly the same height so the layout doesn't look messy and disorganized.
