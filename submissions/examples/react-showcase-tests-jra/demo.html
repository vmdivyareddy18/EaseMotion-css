05 — Accessibility

Reduced-Motion Support

All animations in this showcase respect the user's motion preference.

♿
prefers-reduced-motion: reduce
When a user has enabled the "reduce motion" system setting, all animation durations are collapsed to 0.01ms via a @media query — ensuring no layout flashes and full accessibility compliance.

EaseMotion CSS — Animation Info
