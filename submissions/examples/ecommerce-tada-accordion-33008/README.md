# E-commerce Tada-Click Accordion

This submission resolves issue #33008.

## Features
- **E-commerce Product Theme**: A realistic e-commerce product page mockup. The layout includes a split product view with an image placeholder on the left and product details (title, price, add to cart button, and the accordion) on the right.
- **Component Structure**: Integrated product specification accordion. Utilizes standard HTML markup combined with CSS transitions for smooth height expansions typical in modern retail sites.
- **Animation**: Implements the `ease-click-tada` animation. The accordion headers perform a playful "tada" bounce upon clicking, adding an engaging micro-interaction to the shopping experience.

## File Structure
- `demo.html`: The complete e-commerce product page layout.
- `style.css`: Retail-focused styling and the custom Tada keyframes for interaction.
