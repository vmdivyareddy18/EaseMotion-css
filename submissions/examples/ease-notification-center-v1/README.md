# Ease Notification Center

## Overview

Ease Notification Center is a reusable dashboard component that displays user notifications with badges, timestamps, icons, and hover animations using pure HTML and CSS.

---

## Features

- 🔔 Notification cards
- 🟢 Unread indicator
- ⏰ Timestamps
- ✨ Hover animation
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Notification panel
- Unread notification
- Icons
- Timestamp labels
- Hover interaction

---

<div class="notification-panel">
...
