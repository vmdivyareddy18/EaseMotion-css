# Asset Warranty Status Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive asset warranty dashboard that displays warranty status using modern cards with progress indicators. Suitable for IT asset management systems, inventory dashboards, enterprise hardware portals, and equipment lifecycle management applications.

## Features

- Warranty status cards
- Active, Expiring, and Expired indicators
- Animated progress bars
- Summary statistics
- Responsive layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="asset active">

    <div class="top">

        <h3>MacBook Pro 16"</h3>

        <span class="badge success">
            Active
        </span>

    </div>

</div>
```
