# HeadShake Entry Animation Example

## Feature Overview
This example demonstrates a new `headShake` entry animation. This animation provides a slight shaking motion on the horizontal axis combined with minor rotations, drawing attention to the element in an engaging way.

## Installation Instructions
1. Copy the CSS from `style.css` into your project's stylesheet.
2. Apply the `.ease-headShake` class to the element you want to animate.

## Usage Example
```html
<div class="ease-headShake">
    This element will shake its head!
</div>
```

## Customization Guidance
- **Duration**: Adjust the `animation-duration` in the CSS (default is `1s`).
- **Intensity**: Modify the `translateX` and `rotateY` values in the `@keyframes` definition to make the shake stronger or milder.
