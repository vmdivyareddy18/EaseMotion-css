# ease-card-product

A reusable e-commerce product card built with pure HTML and CSS.

## Features

- Product image
- Image hover zoom
- Product name
- Price display
- Star rating
- Quick View hover overlay
- Add to Cart button using the `ease-btn` pattern
- Lightweight and responsive

---

## Usage

```html
<article class="ease-card-product">
    ...
</article>
```
