# Potoo

A pure CSS and HTML potoo, the wide eyed nocturnal bird, frozen upright on a broken branch pretending to be a stump until a moth passes and it gapes.

## What it shows

- Cryptic bark plumage from a repeating linear gradient plus mottling, over a body that continues the trunk's colour so the bird disappears into it
- The huge eye layered from stacked radial gradients, with a lid that snaps open from a near closed slit
- An enormous gape: the lower mandible swings wide open on a clip-path just as the moth flies by
- A moon, twinkling stars and a wandering moth to break the stillness
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
