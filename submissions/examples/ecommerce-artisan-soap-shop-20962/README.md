# App Landing: Artisan Soap Shop (`ecommerce-artisan-soap-shop-20962`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a handcrafted, natural soap and skincare brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A natural artisan brand requires a UI that feels organic, earthy, and raw. This template achieves a crafted aesthetic using a natural palette (clay, sage, sand, cream), an elegant serif combined with a warm sans-serif (`Playfair Display` & `Karla`), and smooth, floating animations that mimic nature and slow processes.

The layout includes:
- A soft fading Promo Banner.
- A minimalist, sticky Navigation.
- A Hero section featuring abstract CSS soap bars floating gently, accompanied by swaying botanical leaves.
- A Collections grid showcasing color-coordinated soap bars with CSS-based textures (speckles, grain) and slow hover zooms.
- A Story/Process section with an abstract CSS illustration of a soap curing rack, animating bars sliding onto the shelves.
- A Gift Bundles section utilizing CSS to render a stack of soaps tied with twine.
- A refined, elegant Reviews grid with star ratings.
- A bold, contrasting Newsletter signup section.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use natural, organic `cubic-bezier(0.42, 0, 0.1, 1)` timings that convey a slow, mindful craft:
- `.ease-fade-up` (Configured for a slow, organic vertical entrance).
- `.ease-reveal` (A gentle blur-to-focus reveal).
- `.ease-slide-right` (Used for the curing rack animation, sliding items softly into place).
- `.ease-float-slow` (A continuous animation mimicking objects suspended in air or water, combined with CSS variables for unique rotation angles).
- `.ease-sway` (A continuous animation mimicking leaves blowing in a gentle breeze).
- `.ease-hover-zoom` (A slow scale and slight rotation on hover for product cards).
- `.ease-hover-lift-text` (A subtle vertical shift and opacity change for text links).
- `.ease-hover-float` (A soft, organic hover effect for buttons).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the natural layout architecture, the earthy palette, and the complex abstract CSS illustrations (soap bars, textures, curing racks, twine). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the slow entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
