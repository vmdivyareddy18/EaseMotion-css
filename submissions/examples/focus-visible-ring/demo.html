:focus-visible Ring

Use Tab to navigate focusable elements. :focus-visible shows a focus ring only when navigating via keyboard (not mouse click). Compare with the always-on :focus style below.

:focus-visible (keyboard only)

Learn More

Focus ring appears only when tabbing with the keyboard. Clicking with a mouse does not show the ring.

:focus (always visible)

Details

Focus ring appears on every focus event — both keyboard and mouse. This can be visually noisy for mouse users.

Accessibility Notes

:focus-visible provides the best of both worlds: visible focus indicators for keyboard users without the visual clutter of persistent focus rings for mouse users. It is the recommended approach for accessible custom focus styles.
