# Media — Local News Publication

## What does this do?
A complete, copy-paste ready full-page website for a Local News Publication named `Valley Herald`, featuring breaking news header banners, editorial featured story heroes (Main Street revitalization), news grids (school lab upgrades, river clean-ups), community events calendars with date badges, journalist profile cards, subscriber support call-to-actions, and footer.

## How is it used?
Open `demo.html` directly in any web browser to view the page. No build step is required. The page transition, reveal animations, and hover components use the project's standard CSS classes.

## Why is it useful?
This showcases how EaseMotion CSS can be applied to design community journalism, local news, and publishing layouts with classic editorial typography, event list alignments, and support panels.

## EaseMotion CSS Classes Showcased
- **Entrance & Hover Animations**:
  - `.ease-slide-in-left` / `.ease-slide-in-right`: Used for the hero content entrances.
  - `.ease-hover-grow`: Applied to header logo elements and button states.
  - `.ease-hover-lift`: Used for event items and journalist profiles.
  - `.ease-hover-underline`: Applied to nav menu links.
  - `.ease-card-lift`: Used for news article cards.
- **Scroll Reveal Animations**:
  - `.ease-reveal`: Applied to sections to hide elements offscreen.
  - `.ease-reveal-up` / `.ease-reveal-scale`: Sliding directional reveals.
  - `.ease-reveal-delay-1` / `.ease-reveal-delay-2` / `.ease-reveal-delay-3`: Staggered timing delays for grid columns.
