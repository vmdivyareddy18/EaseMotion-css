# Radio Card Select

A selectable radio card with a scaling marker and lifted selected state.

## Files

- `demo.html` - radio input wrapped in a visible card label.
- `style.css` - selected-state border, marker animation, focus-within styling, and reduced-motion support.

## Highlights

- Uses a native radio input for selection.
- Shows selected state through border, lift, and marker scale.
- Keeps keyboard focus visible.
