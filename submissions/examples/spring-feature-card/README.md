# Ease Spring Feature Card

A spring animated feature card component inspired by Creative Agency design patterns.

## Features

- Smooth spring entrance animation
- Modern glassmorphism card style
- Hover interaction
- Pure CSS implementation
- Responsive design
- No JavaScript required


## Usage

Add the card structure:

```html
<article class="feature-card">
  <h2>Creative Solutions</h2>
  <p>Feature description</p>
</article>
