# Ease Ivy Crawl Animation

A horizontal crawling CSS animation mimicking a sprawling ivy vine moving along a background surface, utilizing staggered node expansion delays paired with an overarching smooth displacement function.

## How to View
1. Launch `demo.html` in a contemporary browser.
2. Hit the "Crawl Again" action layout component to replay the sequence manually.

## Technical Details
- **Architecture:** CSS3 linear keyframe offset tracking combined with back-easing transform dynamics.
- **Easing Curve:** Primary travel profile runs via `cubic-bezier(0.22, 1, 0.36, 1)`.
