Safari Hover Animation Lag Fix

GPU acceleration via will-change + translateZ(0) eliminates Safari stuttering

ease-pulse — Before vs After Fix

❌ Before (lags on Safari)

Pulse
/* No GPU hint */
.ease-pulse {
  animation: pulse 2s infinite;
}

✅ After (smooth on Safari)

Pulse
/* GPU accelerated */
.ease-pulse-fixed {
  animation: pulse 2s infinite;
  will-change: opacity;
  transform: translateZ(0);
  backface-visibility: hidden;
}

ease-hover-grow — Before vs After Fix

❌ Before (lags on Safari)

Hover
/* No GPU hint */
.ease-hover-grow {
  transition: transform 0.3s ease;
}
.ease-hover-grow:hover {
  transform: scale(1.06);
}

✅ After (smooth on Safari)

Hover
/* GPU accelerated */
.ease-hover-grow-fixed {
  transition: transform 0.3s ease;
  will-change: transform;
  transform: translateZ(0);
  backface-visibility: hidden;
}
.ease-hover-grow-fixed:hover {
  transform: scale(1.06) translateZ(0);
}

Universal Fix — .ease-safari-fix utility

Add .ease-safari-fix to any element experiencing lag on Safari. This forces GPU layer promotion without changing the animation itself.

<div class="ease-pulse ease-safari-fix">...</div>
