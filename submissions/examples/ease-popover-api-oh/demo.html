Popover API + EaseMotion

Native popovers with smooth entrance animations

About the Popover API

The Popover API provides native support for tooltips and menus without JavaScript libraries. Combined with EaseMotion animations, you get smooth, accessible popovers with minimal code.

1. Tooltip with Fade-In

Hover or click to see the tooltip with ease-fade-in animation

This is a tooltip with fade-in animation!

Press Escape or click outside to close

HTML 
<button popovertarget="tooltip1">Hover for Tooltip</button>
<div id="tooltip1" popover class="popover tooltip-popover ease-fade-in">
  <p>Tooltip content here...</p>
</div>

2. Menu with Zoom-In

Click to open menu with ease-zoom-in animation

HTML 
<button popovertarget="menu1">Open Menu ▼</button>
<div id="menu1" popover class="popover menu-popover ease-zoom-in">
  <button class="menu-item">Profile</button>
  <button class="menu-item">Settings</button>
  <button class="menu-item danger">Sign Out</button>
</div>

3. Confirmation Dialog

Modal-style dialog with slide-up animation

Confirm Deletion

Are you sure you want to delete this item? This action cannot be undone.

HTML 
<button popovertarget="dialog1">Delete Item</button>
<div id="dialog1" popover class="popover dialog-popover ease-slide-up">
  <h3>Confirm Deletion</h3>
  <p>Are you sure?</p>
  <button popovertarget="dialog1">Cancel</button>
  <button>Delete</button>
</div>

4. Notification Toast

Toast notification with bounce animation

✓ Item saved successfully!
HTML 
<button popovertarget="toast1">Show Success Toast</button>
<div id="toast1" popover class="popover toast-popover ease-bounce-in">
  <span>✓ Success message</span>
</div>

Key Features

  • ✓ Native Popover API - no JavaScript libraries needed
  • ✓ Accessibility built-in - keyboard navigation support
  • ✓ Light dismiss - click outside to close
  • ✓ Escape key support - press to close
  • ✓ Smooth EaseMotion entrance animations
  • ✓ Multiple popover types: tooltips, menus, dialogs, toasts
