# Smart Mobility Analytics Dashboard

A modern mobility intelligence dashboard built using pure HTML and CSS.

## Features

- Fleet Performance Analytics
- Route Optimization Monitoring
- Vehicle Utilization Metrics
- Emissions Tracking Dashboard
- Mobility KPI Overview
- Fleet Status Monitoring
- Driver Performance Insights
- Trip Timeline Feed
- Glassmorphism UI
- Emerald-Blue Dark Theme
- Fully Responsive Design

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
