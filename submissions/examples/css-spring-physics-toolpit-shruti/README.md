# Spring Physics Tooltip – Accessible Web

A responsive tooltip featuring a spring physics animation built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- No JavaScript
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- Spring Physics animation

## Files

- demo.html
- style.css

## Accessibility

- Keyboard focus support
- Uses `aria-describedby`
- Uses `role="tooltip"`
