# Bounce Maintenance Page

A responsive **Bounce Maintenance Page** inspired by food delivery applications. It features a CSS-only bouncing illustration and a clean maintenance message using pure HTML and CSS.

## Features

- Pure HTML & CSS
- CSS bounce animation
- Responsive layout
- Accessible structure
- No JavaScript required

## Folder Structure

```
bounce-maintenance-page-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
