# Ease Ripple Pulse Animation

## What does this do?

Ease Ripple Pulse is a reusable CSS animation that combines a gentle scaling pulse with an expanding ripple effect to attract attention in a smooth, modern way.

---

## Features

- Pure HTML & CSS
- Lightweight
- Infinite pulse animation
- Expanding ripple effect
- Suitable for buttons, cards, badges, and icons
- No JavaScript required

---

## Usage

```html
<button class="ease-ripple-pulse">
    Explore Now
</button>
```

---

## Animation

The animation consists of:

- Subtle scale pulse
- Expanding circular ripple
- Fade-out ripple
- Smooth ease-in-out timing

---

## Why is it useful?

This animation provides a modern attention effect that can highlight important interactive elements without being distracting. It fits the EaseMotion CSS philosophy by offering reusable, animation-first utilities that enhance user interfaces while remaining lightweight and easy to integrate.
