# Weather Snow Icon

1. What does this do? Displays a cloud icon with animated snowflake dots that fall with a gentle sway, sized for a weather app icon.

2. How is it used? Wrap the `.weather-icon` and `.snow-icon` classes around the cloud and snowflake dot elements.

3. Why is it useful? It provides a lightweight, pure-CSS snow animation at icon scale — ideal for weather widgets, forecast cards, and app launcher icons without JavaScript or external assets.
