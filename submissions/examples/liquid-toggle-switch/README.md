# Liquid Toggle Switch

## What does it do?
A CSS-only toggle switch where the knob behaves like a squishy blob of liquid — stretches horizontally and compresses vertically while traveling across the track.

## Features
- Squishy blob morphing animation via keyframes
- `width` and `border-radius` transitions for liquid effect
- Pure CSS, no JavaScript
- Accessible (uses hidden checkbox input)

## Usage
```html
<label class="liquid-toggle">
  <input type="checkbox" class="liquid-toggle-input">
  <span class="liquid-toggle-track">
    <span class="liquid-toggle-knob"></span>
  </span>
</label>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
