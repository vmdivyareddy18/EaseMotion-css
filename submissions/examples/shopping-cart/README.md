# Shopping Cart Drawer Component

An interactive side sheet utility component handling modal viewport overlays, custom numerical input increments, and high-performance flex row item exit transformations.

## Visual Elements
- **Blur Shrouded Context Overlays:** Opacity mapping that decouples surrounding viewport elements safely.
- **Sliding Lateral Panels:** Matrix transformations sliding right-to-left into native alignment coordinates.
- **Collapsible Exit Rows:** Multi-property animation tracking offsets scaling down spacing smoothly upon item deletion.

## Usage Layout Structure
```html
<div class="ease-cart-overlay">
  <div class="ease-cart-drawer">
    <div class="ease-cart-items-feed">
      <div class="ease-cart-item-row"> ... </div>
    </div>
  </div>
</div>
```
