# Elephant Stomping

1. What does this do? A CSS-only elephant that stomps the ground with body shake and trunk swing.

2. How is it used? Place .elephant inside .scene. The body is a large gray ellipse; the trunk is a lighter shape that sways.

3. Why is it useful? Shows heavy-animal motion with ground impact via sudden translateY and squash/stretch.
