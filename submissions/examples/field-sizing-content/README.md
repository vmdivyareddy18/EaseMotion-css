# CSS field-sizing: content Auto-Growing Form

## What does this do?

Demonstrates the `field-sizing: content` CSS property, which makes form fields (textareas, inputs, selects) automatically grow and shrink to fit their content. This eliminates the need for JavaScript-based auto-resize solutions.

## Key Features

- `field-sizing: content` on textareas for auto-growing height
- `field-sizing: content` on `<input>` elements for auto-expanding width
- `min-height` / `max-height` constraints to prevent extreme sizes
- `min-width` / `max-width` for width-bounded expansion
- Form demo with multiple field types showcasing the behavior
