# API Request Status Stack

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern API monitoring dashboard that visualizes request states, response metrics, endpoint health, and service status using pure HTML and CSS.

## Features

- API request monitoring
- Success, running, warning, and failed states
- Response time indicators
- Animated activity pulse
- Service health section
- Status badges
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="request-card success">
  ...
</div>
```

## Main Classes

```css
.request-stack
.request-card
.badge
.pulse
.health-panel
.health-row
```
