# 🍽️ Restaurant & 💊 Pharmacy Dashboard

## Overview
The **Restaurant and Pharmacy Dashboard** is a responsive web application built with **EaseMotion CSS**.  
It provides a unified interface for managing restaurant orders and pharmacy inventory, combining elegant UI transitions with real-time analytics.

---

## 🎯 Objectives
- Build a dual-purpose dashboard for restaurant and pharmacy operations.
- Integrate order tracking, inventory management, and analytics visualization.
- Use EaseMotion CSS for smooth animations and transitions.
- Ensure accessibility and responsiveness across all devices.

---

## 🧩 Tech Stack
| Component | Description |
|------------|-------------|
| **HTML5** | Semantic structure for dashboard sections |
| **EaseMotion CSS** | Animation and transition utilities |
| **JavaScript (ES6)** | Dynamic interactivity and data handling |
| **Chart.js** | Visual analytics for sales and stock trends |
| **Font Awesome** | Icons for navigation and actions |
