# App Landing: Dog Walking Marketplace (`app-landing-dog-walking-21003`)

This proposal provides a complete, copy-paste ready B2C landing page designed for a pet services/dog walking marketplace app, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A pet services marketplace must project friendliness, trust, and approachability. This template implements a highly playful, warm aesthetic using colors like Coral Orange (`#FF7B54`), Teal (`#2D9596`), and bright Yellow (`#FFD24C`), alongside the heavily rounded, friendly `Nunito` font.

The layout includes:
- A Hero section featuring a detailed CSS mobile app mockup (showing a GPS tracked walk in progress with floating notification alerts).
- A two-sided marketplace split layout (For Pet Parents vs For Walkers).
- A 4-column services grid (Walking, Sitting, Drop-ins, Boarding).
- A Trust & Safety section showcasing a CSS-rendered "Report Card" mockup.
- Reviews from both sides of the marketplace.
- Bold App Store download Call-To-Action (CTA) sections.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities, specifically configured with a "bouncy" bezier curve to enhance the playful pet aesthetic:
- `.ease-fade-up` / `.ease-slide-up` (Configured with a playful, overshooting curve `cubic-bezier(0.34, 1.56, 0.64, 1)`)
- `.ease-zoom-in` (Applied to background organic blobs and service cards to pop them into view)
- `.ease-slide-left` / `.ease-slide-right` (Used for floating app UI alerts)
- `.ease-hover-lift` (Standard bouncy interaction for all buttons and service cards)
- `.ease-hover-scale` (Applied to the logo with an added subtle rotation for a playful touch)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, the phone/report card mockups, the organic background blobs, and the highly rounded UI geometry. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.
