# Swing Speed Dial

A Social Media-inspired Swing Speed Dial built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Swing animation
- Floating social action buttons
- Responsive layout
- Accessible HTML
- No JavaScript required
