# True Crime Podcast Template
A suspenseful, professional template designed for true crime podcasts, investigative journalism series, and mystery storytelling brands.

- **Design**: Uses a "Noir-Investigative" palette with deep noir-blacks, professional evidence-greys, and high-impact crimson-alert accents to build tension and authority.
- **Use Case**: Investigative podcasts, mystery storytelling series, and archival crime documentary brands.
