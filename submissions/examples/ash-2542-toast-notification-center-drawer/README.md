# Toast Notification Center Drawer

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern notification center drawer featuring stacked toast alerts with different status states.

## Features

- Notification center drawer
- Success, info, warning, and error toasts
- Slide-in drawer animation
- Staggered toast appearance
- Responsive design
- Glassmorphism styling
- Dashboard-friendly layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="toast success">

    <div class="icon">✓</div>

    <div>
        <h3>Payment Successful</h3>
        <p>Your subscription has been renewed.</p>
    </div>

</div>
```

## Main Classes

```css
.notification-drawer
.drawer-header
.toast
.success
.info
.warning
.error
.icon
```
