# Animated Gradient Button

A CSS-only animated gradient button that shifts colors on hover, slightly elevates, and shows a soft glowing box-shadow.

## Files
- `demo.html` - Contains the demo structure.
- `style.css` - Contains the core utility classes.

## Usage
Add the class to any link or button element:
```html
<a class="em-gradient-btn" href="#">Explore</a>
