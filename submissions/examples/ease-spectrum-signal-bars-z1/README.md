# Ease Spectrum Signal Bars

## Overview

A futuristic signal visualization featuring animated spectrum bars inside a frosted glass panel using only HTML and CSS.

## Features

- Animated spectrum bars
- Gradient glow effect
- Glassmorphism panel
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-spectrum-signal-bars-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Adds a reusable animated spectrum component suitable for music visualizers, monitoring dashboards, AI interfaces, and analytics applications.
