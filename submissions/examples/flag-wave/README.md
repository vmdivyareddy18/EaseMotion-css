# Flag Wave Distortion

A smooth, continuous fabric-waving animation simulating a flag waving in the wind. This effect is achieved purely in CSS using combinations of `skewY` and `scaleX` transform oscillations.

## EaseMotion CSS Classes Showcased

This feature submits a self-contained prototype demonstrating how standard skew and scale distortions can be chained to create a fabric-waving illusion.

### Classes:
- `.ease-flag-wave`: Applies continuous horizontal and vertical fabric waving over a configurable looping duration (`4s` by default).

### Usage in HTML:
```html
<div class="ease-flag-wave">
  <!-- Insert flag graphic or text banner -->
</div>
```
