# Linear Gradient Mask Reveal

Using CSS mask-image with linear gradients to create smooth fade transitions on images and content. Includes both static and animated mask effects.

## Files

- `demo.html` - Image fade mask and text reveal mask
- `style.css` - WebKit-prefixed mask-image with gradient animations

## Usage

Open `demo.html` to see gradient mask effects in action.
