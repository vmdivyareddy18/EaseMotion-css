# Stellar Flash Burst Animation Utility

## Overview
This utility creates a bright stellar burst effect where glowing light rays flash outward from a central energy core, simulating a star-like explosion.

## Usage

```html
<div class="ease-stellar-flash">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first interaction
- Great for hero sections, notifications, loaders, and futuristic interfaces
