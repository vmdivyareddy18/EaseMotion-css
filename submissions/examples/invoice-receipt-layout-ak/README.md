# Invoice Receipt Layout

## What does this do?
A printable invoice/receipt layout with dynamic line items, automatic tax calculation, and a print button.

## How is it used?
Open demo.html in a browser. Add line items with name, quantity, and price. View subtotal, tax, and total. Use Print to print the invoice.

## Why is this useful?
Demonstrates a clean printable document layout with dynamic item management and responsive print styling.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
