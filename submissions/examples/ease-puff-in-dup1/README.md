# ease-puff-in

A fast entrance animation that makes an element appear as if it is puffing into existence.

## Features

- Starts from scale(0)
- Expands beyond final size
- Settles smoothly to normal size
- Fades in simultaneously
- Ideal for tooltips, popovers, badges, and notifications

## Usage

```html
<div class="ease-puff-in">
  Content
</div>
