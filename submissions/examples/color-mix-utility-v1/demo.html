EaseMotion CSS color-mix() Utility

Blend colours in CSS.
No preprocessor needed.

The native color-mix() function mixes two colours at any ratio, right in your stylesheet — using EaseMotion design tokens as the source colours. Hover each segment below to see the token name.

--ease-color-primary
75% primary
50% / 50%
25% primary
--ease-color-secondary

Hover to reveal token names · Colors computed by color-mix() in oklch

Syntax

color-mix(in oklch, var(--ease-color-primary) 75%, var(--ease-color-secondary) ) /* ↑ color space · ↑ colour A + ratio · ↑ colour B (remainder) */

💡 Why oklch? The oklch colour space produces perceptually uniform blends — midpoints look visually balanced rather than muddy. You can swap it for srgb, hsl, or lab without changing the ratios; only the blending path changes.
