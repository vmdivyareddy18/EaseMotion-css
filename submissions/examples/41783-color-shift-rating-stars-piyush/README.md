# Color Shift Rating Stars

A responsive and accessible CSS-only animated rating stars component for the EaseMotion CSS library.

## Features

- ⭐ Pure HTML & CSS (No JavaScript)
- 🌈 Smooth color-shift animation
- ♿ Keyboard accessible
- 📱 Responsive design
- 🎨 Uses EaseMotion-style variables and animation naming

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in any modern browser.

## Preview

Hover over or select the stars to see the animated color shift effect.
