# Holographic Notification Bell

Resolves #42423.

### What does this do?
Adds a Holographic Notification Bell component to EaseMotion CSS. It creates a sci-fi projector effect using a clipped linear-gradient projection cone (`ease-pulse`). The bell icon floats gently using a slowed down `ease-swing`, and features a glowing neon aesthetic via CSS `filter: drop-shadow()`.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. It requires a dark background to make the holographic glow prominent.

### Why is it useful?
Provides an incredibly striking notification element for dark-mode apps, health & fitness dashboards, or any interface leaning towards a cyber or futuristic design language.
