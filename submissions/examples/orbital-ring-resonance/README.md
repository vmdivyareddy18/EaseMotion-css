# Orbital Ring Resonance Animation Utility

## Overview
This utility creates glowing orbital rings that rotate around a central energy core, producing a futuristic resonance effect.

## Usage

```html
<div class="ease-orbital-ring">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first interaction
- Great for futuristic dashboards, loaders, cards, and sci-fi interfaces
