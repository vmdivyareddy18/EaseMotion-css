# Multi-Layer Box Shadow Elevation Scale

A card layout system demonstrating realistic multi-layered ambient light box shadows.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting low, medium, and high shadow cards, layout dividers, and depth details.
2. **`style.css`**: Layered ambient occlusion offsets, blur settings, card formats, translations, and hover rules.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Compare the depth profiles of the three cards.
3. Verify that the **Elevation Level 3** card displays the deepest and softest lighting blur.
