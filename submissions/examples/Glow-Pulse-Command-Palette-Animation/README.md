# Glow Pulse Command Palette Animation

A responsive Glow Pulse animation designed for Command Palette interfaces using only HTML and CSS.

## Features

- Pure CSS
- Responsive layout
- Glow Pulse animation
- Command Palette interface
- CSS custom properties
- `prefers-reduced-motion` support
- No JavaScript required

## CSS Variables

```css
--duration
--accent
--surface
--background
--text
--muted
--radius
```
