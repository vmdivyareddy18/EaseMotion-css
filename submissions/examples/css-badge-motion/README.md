# 🏷️ CSS Badge Motion

A collection of beautiful, pure CSS animated badges and tags for modern web interfaces.

## 🚀 Features

### Badge Styles (9 types)
1. **Solid Badges** - 6 color variations with hover lift
2. **Outline Badges** - Border-only with fill on hover
3. **Gradient Badges** - 6 gradient variations
4. **Pill Badges** - Rounded pill shape
5. **Neon Badges** - 6 neon glow colors
6. **Status Badges** - Online/Offline/Busy/Away with indicators
7. **Notification Badges** - Icon badges with count
8. **Animated Badges** - Pulse, bounce, shake, glow, shine, rotate
9. **Icon Badges** - Badges with emoji icons

### Animation Features
- Pulse effect
- Bounce animation
- Shake animation
- Glow effect
- Shine sweep
- Rotate animation
- Status indicator pulse
- Notification badge bounce
