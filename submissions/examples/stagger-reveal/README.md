# CSS Stagger Animation System for List/Grid Reveal

A pure CSS animation system for staggered list/grid reveals on scroll using CSS custom properties.

## Features
- Staggered entrance animations with fade and slide
- CSS custom property delays: `--delay: calc(n * 80ms)`
- Base animation: 0.5s ease-forwards with delay
- Accessibility: respects `prefers-reduced-motion`
- No JavaScript required
