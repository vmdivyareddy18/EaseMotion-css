Why EaseMotion?

EaseMotion uses human-readable animation names that improve code readability and maintainability.

Using Tailwind with EaseMotion

Tailwind can handle layout, spacing, and typography while EaseMotion provides animation and motion effects. 

<div class="flex items-center justify-center ease-bounce">
    Hello World
</div>
