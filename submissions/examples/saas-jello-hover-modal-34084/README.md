# SaaS Showcase Modal with Jello Hover Animation

This submission resolves issue #34084.

## Features
- **SaaS Theme**: Clean, professional design with high contrast, system fonts (Inter), and subtle drop shadows suited for B2B SaaS dashboards.
- **Component**: A responsive, centered modal dialog.
- **Animation**: Implements a custom `ease-hover-jello` animation. When the user hovers over the primary "Upgrade Plan" button (and the trigger button), it performs a playful, engaging jello wobble effect to encourage clicks.

## File Structure
- `demo.html`: The interactive layout containing the modal and the JavaScript to toggle it.
- `style.css`: The completely original SaaS-themed styles and the custom Jello keyframes.
