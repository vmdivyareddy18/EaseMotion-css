# CSS Counter Badges

## What does this do?
Auto-numbered badges and lists using CSS counter-increment and counter() for numbered steps, ordered lists, and section headings.

## How is it used?
```html
<ol class="cnb-steps"><li class="cnb-step"><div class="cnb-circle"></div><div class="cnb-body"><strong>Title</strong><p>Description</p></div></li></ol>
```

## Why is it useful?
Automatic numbering without hardcoding or JavaScript. Connected steps with visual connectors. Multiple styles: circle badges, list counters, section headings.
