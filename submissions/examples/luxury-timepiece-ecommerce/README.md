# Luxury Timepiece E-Commerce

## Overview

A full-page luxury watch e-commerce experience inspired by timeless editorial design and Swiss horology craftsmanship.

## Features

- Product hero section with watch showcase
- Collections grid
- Heritage storytelling section
- Materials and craftsmanship highlights
- Limited edition showcase
- Customer reviews
- Boutique visit CTA
- Responsive footer

## Tech Stack

- HTML5
- CSS3
- EaseMotion CSS

## Usage

Open `demo.html` directly in a browser. No build step or dependencies are required.

## Browser Support

- Chrome
- Firefox
- Edge

## EaseMotion Classes Showcased

- `ease-fade-in`
- `ease-fade-in-up`
- `ease-slide-down`
- `ease-slide-left`
- `ease-slide-right`
- `ease-hover-lift`
- `ease-hover-grow`
