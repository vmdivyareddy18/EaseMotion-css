# Advanced: Business Landing — Veterinary Clinic

A responsive veterinary clinic landing page built using **HTML**, **CSS**, and **EaseMotion CSS**. This example showcases a modern, pet-friendly healthcare website with smooth animations powered entirely by existing EaseMotion CSS classes.

## Features

* Responsive navigation bar
* Warm hero section with appointment CTA
* Veterinary services grid
* Meet the veterinarians section
* Responsive layout for desktop, tablet, and mobile
* Modern healthcare-inspired design
* Smooth entrance animations
* Interactive hover effects

## EaseMotion CSS Classes Showcased

### Entrance Animations

* `ease-fade-down`
* `ease-fade-up`
* `ease-fade-left`
* `ease-fade-right`
* `ease-scale-in`

### Hover Animations

* `ease-hover-lift`
* `ease-hover-grow`
