# Footer Animation

## Overview

Demonstrates fade-in animation for footer section in EaseMotion CSS.

## Features

- **Fade-in Animation**: Smooth entrance effect using `ease-fade-in`
- **Customizable Delay**: Use `ease-delay-*` classes to control timing
- **Responsive Design**: Works on all screen sizes

## Usage

Add animation classes to your footer:

```html
<footer class="demo-footer ease-fade-in ease-delay-500">
  <div class="footer-content">
    <!-- Footer content -->
  </div>
  <div class="footer-bottom">
    <p>© 2026 Your Company</p>
  </div>
</footer>
```

## Available Animation Classes

| Class | Effect |
|-------|--------|
| `ease-fade-in` | Opacity transition from 0 to 1 |
| `ease-slide-up` | Slide up from bottom |
| `ease-slide-down` | Slide down from top |
| `ease-delay-100` | 100ms delay |
| `ease-delay-300` | 300ms delay |
| `ease-delay-500` | 500ms delay |
