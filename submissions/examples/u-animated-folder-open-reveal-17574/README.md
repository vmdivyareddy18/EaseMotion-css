# Animated Folder Open Reveal

## Overview

Animated Folder Open Reveal is a lightweight CSS animation that simulates opening a folder and revealing a document.

## Features

- CSS-only animation
- Folder reveal effect
- Document emergence animation
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="folder-reveal">
  <span class="folder">📂</span>
  <span class="document">📄</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
