# Animated Card Hover UI Example

## 1. What does this do?

This submission illustrates card components with animated hover effects, smooth elevation transitions, and interactive visual depth. Hovering triggers subtle movement, shadow expansion, and content fade-ins to create a modern card UX.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes several card styles with hover-triggered reveals and animated accent highlights. Copy the `.card-hover` markup and CSS into your project to use polished card hover interactions.

---

## 3. Why is it useful?

Animated hover cards improve engagement by providing responsive visual feedback when users explore content. The combination of shadow motion and reveal animations adds sophistication while keeping the interface lightweight and CSS-first.
