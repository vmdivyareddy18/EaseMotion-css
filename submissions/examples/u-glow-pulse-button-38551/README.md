# Glow Pulse Card

A responsive **Glow Pulse Card** created with pure CSS for **E-Commerce Checkout Layouts**. The component highlights the checkout summary using a subtle glowing pulse animation while remaining lightweight, responsive, and accessible.

---

## Features

- Pure CSS implementation
- Glow Pulse animation
- Responsive checkout card
- Keyboard accessible
- CSS Custom Properties
- Zero JavaScript dependency
