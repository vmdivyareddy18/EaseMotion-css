# CSS Accordion Component

## What does this do?
Pure CSS accordion using details/summary elements and checkbox hack with nesting support.

## How is it used?
```html
<details class="ac-details"><summary class="ac-summary">Title</summary><div class="ac-body"><p>Content</p></div></details>
```

## Why is it useful?
No JavaScript expand/collapse. Two variants: details/summary (native) and checkbox hack (multi-open). Nested support. Animated transitions. Ideal for FAQs and docs.
