▶ What is EaseMotion?

EaseMotion is a lightweight CSS framework for building smooth animations and interactive UI components without JavaScript overhead. It focuses on performance and accessibility.

▶ Browser Support

EaseMotion CSS works in all modern browsers including Chrome, Firefox, Safari, and Edge. Some newer CSS features like scroll-driven animations require Chromium 115+.

▶ Getting Started

Clone the repository and link style.css in your project. Customize CSS variables for theming. No build tools or npm required - just vanilla CSS.

▶ License

EaseMotion CSS is open source under the MIT license. You can use it in personal and commercial projects freely.
