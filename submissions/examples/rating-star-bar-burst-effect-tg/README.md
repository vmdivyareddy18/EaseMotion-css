# Rating Star Bar with Interactive Burst Effect

### 1. What does this do?
Provides an interactive five-star rating component with hover previews, click-to-select functionality, and a burst animation for visual feedback.

### 2. How is it used?
Apply the `.rating` container with `.star` buttons in your HTML and include the accompanying `style.css` and JavaScript from the demo.

```html
<div class="rating">
  <button class="star">★</button>
  <button class="star">★</button>
  <button class="star">★</button>
  <button class="star">★</button>
  <button class="star">★</button>
</div>
```

### 3. Why is it useful?
It provides an engaging, responsive, and accessible rating interaction that aligns with EaseMotion CSS's goal of creating simple, reusable UI effects with smooth animations.
