# Liquid Ripple Fitness Tracker Accordion

A responsive Fitness Tracker accordion featuring a smooth Liquid Ripple animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Native `<details>` and `<summary>`
- Keyboard accessible
- Responsive design
- Liquid Ripple opening animation
- Fitness Tracker inspired layout
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--primary
--secondary
--surface
--background
--text
--radius
```
