# Creative Zoom-Out Popover (#46634)

An elegant, standard-compliant portfolio layout transition that dynamically zooms out, blurs, and sets perspective depth back onto primary page contents when a detailed details popover modal is requested.

### Usage
Integrate the wrapper arrangement into your view setups:

```html
<div class="portfolio-perspective-wrapper">
    <main class="portfolio-content">
        </main>
    <div id="info-popover" class="popover-overlay">
        </div>
</div>
