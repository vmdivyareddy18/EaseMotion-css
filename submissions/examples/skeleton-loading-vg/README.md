# CSS-only Skeleton Loading Components

A responsive collection of modern skeleton loading components built using pure HTML and CSS.

The showcase demonstrates reusable loading placeholders for common web interface layouts without using JavaScript.

## Components Included

- Profile Card Skeleton
- Blog Card Skeleton
- Dashboard Widget Skeleton
- Product Card Skeleton
- Table Row Skeleton

## Features

- Pure HTML and CSS
- No JavaScript dependencies
- Smooth shimmer animation
- Fully responsive
- Reusable skeleton utility classes
- Accessible loading states
- Reduced-motion support
- Forced-colors support
- Mobile and desktop layouts
- Easy animation customization using CSS variables
