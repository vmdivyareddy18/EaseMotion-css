# Image Reveal with Clip-Path (Issue #18270)

An image (represented by a gradient) that reveals itself with a circle wipe animation using CSS `clip-path`. Triggers automatically on page load.

## Usage

Open `demo.html` in a browser. The image reveals from center outward.
