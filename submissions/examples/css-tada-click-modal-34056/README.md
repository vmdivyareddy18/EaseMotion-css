# CSS Tada-Click Modal for SaaS Showcase Layouts

## Overview

This project demonstrates a responsive SaaS showcase modal featuring a smooth **Tada-Click** entrance animation built entirely with HTML and CSS. The component presents a modern pricing plan with highlighted features, a clean interface, and an engaging call-to-action without requiring JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- Smooth Tada-Click entrance animation
- Responsive SaaS showcase modal
- Modern pricing card layout
- Feature checklist
- CSS custom properties for easy customization
- Hover and active button effects
- Keyboard-accessible focus styles
- No JavaScript or external libraries required

## Customization

The component uses CSS custom properties.

```css
:root{
    --primary:#4f46e5;
    --primary-dark:#4338ca;
    --accent:#10b981;
    --bg:#eef3fb;
    --radius:22px;
    --duration:.8s;
}
```

You can customize:

- Theme colors
- Background color
- Animation duration
- Border radius
- Typography
- Shadow intensity
- Button appearance
- Feature icon colors
- Modal width
