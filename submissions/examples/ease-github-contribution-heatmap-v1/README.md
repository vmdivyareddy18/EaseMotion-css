# Ease GitHub Contribution Heatmap

## Overview

Ease GitHub Contribution Heatmap is a reusable HTML & CSS component inspired by GitHub's contribution graph. It features animated activity cells, hover interactions, and a contribution legend without requiring JavaScript.

---

## Features

- 🟩 GitHub-style contribution grid
- ✨ Hover glow effect
- 📊 Activity levels
- 🎯 Staggered reveal animation
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Weekly contribution columns
- Activity intensity colors
- Hover interactions
- Contribution legend

---

## Usage

```html
<div class="heatmap">
...
</div>
```

---

## Customization

- Grid size
- Activity colors
- Cell spacing
- Animation timing
- Border radius

---

## Browser Support

Compatible with modern browsers supporting CSS Grid, Flexbox, transitions, and animations.

---

## Why This Fits EaseMotion CSS

Perfect for developer portfolios, Git dashboards, coding profiles, contribution analytics, and educational demos. It provides a lightweight, reusable contribution heatmap component built entirely with HTML and CSS.
