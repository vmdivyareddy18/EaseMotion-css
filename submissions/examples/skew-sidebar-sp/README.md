# Skew Sidebar

A modern corporate-style skew sidebar built with pure HTML and CSS for EaseMotion CSS.

## Features

- 📐 Smooth skew-to-straight hover animation
- 🎨 Pure CSS (No JavaScript)
- 📱 Responsive layout
- ♿ Keyboard accessible navigation
- ⚡ Lightweight and reusable
- 💼 Corporate-inspired design

## File Structure

```
skew-sidebar-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Hover over the sidebar to expand it and reveal a smooth skew transition with interactive navigation links.

## Why is it useful?

The Skew Sidebar provides a modern navigation component suitable for dashboards, admin panels, business portals, and corporate web applications while maintaining a clean and responsive user experience.
