# CSS Slide-Up Carousel for Cyberpunk Neon Layouts

A pure HTML and CSS Slide-Up Carousel inspired by Cyberpunk Neon UI aesthetics.

## Features

- Pure HTML & CSS
- Smooth slide-up animation
- Neon glow effects
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- No JavaScript required

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Customization

You can modify:

- Neon colors
- Animation duration
- Border radius
- Glow intensity
- Overlay background

using CSS variables.
