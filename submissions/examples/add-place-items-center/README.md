# Place Items Center Utility

Centers content both horizontally and vertically using CSS Grid.

## CSS

```css
.place-items-center {
  display: grid;
  place-items: center;
}
```

## Usage

```html
<div class="place-items-center">
  Centered Content
</div>
```

## What it does

The `.place-items-center` utility combines:

```css
display: grid;
place-items: center;
```

This makes it easy to perfectly center text, icons, buttons, loaders, and other content inside a container.

## Use Cases

- Centering buttons
- Empty states
- Loading spinners
- Icons
- Hero content
- Cards and panels
