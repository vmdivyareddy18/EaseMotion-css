# Animated Credit Score Meter Sweep

## Overview

Animated Credit Score Meter Sweep is a lightweight CSS animation that simulates a credit score gauge with a sweeping needle and animated score display.

## Features

- CSS-only animation
- Gauge meter sweep effect
- Credit score visualization
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="credit-score-meter">
  <span class="needle">▲</span>
  <span class="score">780</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
