# CSS Spring Physics Accordion - Minimalist Tech Layouts

## Overview

A pure CSS Spring Physics Accordion designed for minimalist technology interfaces.

This component provides smooth spring-inspired expand and collapse animations using only HTML and CSS without JavaScript.

## Features

- Pure CSS implementation
- Smooth spring motion animation
- Responsive layout
- Keyboard accessible using native details and summary elements
- Customizable CSS variables
- Lightweight and reusable
