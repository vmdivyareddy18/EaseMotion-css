# Vertical-Align Utilities

1. What does this do?
Provides utility classes to control the vertical alignment of inline, inline-block, or table-cell boxes relative to their parent element.

2. How is it used?
Apply classes like `.align-middle`, `.align-top`, or `.align-baseline` directly to inline-block indicators, badges, or table-cell elements:
```html
<p>
  Text label
  <span class="badge align-middle">Centered Badge</span>
</p>
```

3. Why is it useful?
It allows aligning icons, subscripts, superscripts, and buttons with their neighboring text line, ensuring clean visual alignments across inline components and grid/table cells.
