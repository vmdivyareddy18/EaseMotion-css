✅ Test 1: Utilities override Components

When an element has both .card (components layer) and .bg-override (utilities layer), the utility's background should win — even though both are single-class selectors at the same specificity, because utilities layer comes after components.

element only: .card Component style (default)

✓ PASS: .bg-override (utilities layer) correctly overrides .card (components layer)

✅ Test 2: Components override Base

Base layer sets color: #1a1a2e, but component layer sets color: #e0e0e0. The component value wins due to layer order.

✅ Test 3: Unlayered User Styles override all

Styles with no @layer (unlayered) always win over any layered style, regardless of layer order. .unlayered-text sets color: #f59e0b and should beat all layers.
