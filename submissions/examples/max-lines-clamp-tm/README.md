# Max Lines Clamp

line-clamp — Clamps text to a maximum number of visible lines

## Features
- Uses `--ease-color-surface`
- Uses `--ease-radius-md`
- Uses `--ease-color-muted`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Max Lines Clamp</span>
</div>
```

## Why is it useful?

This submission demonstrates the max lines clamp CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
