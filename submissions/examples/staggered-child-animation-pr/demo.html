Staggered Child Animation

A pure CSS utility that animates sibling children one after another using animation-delay — no JavaScript required.

Cards — 100 ms stagger

Each card fades up 100 ms after the previous one.

class="card-grid stagger-fade-up stagger-100"
⚡
Zero JavaScript

The entire stagger is driven by CSS animation-delay on nth-child selectors — no script needed.

Pure CSS
🎯
nth-child Targeting

Delay is applied with :nth-child(n) selectors directly on the children of the utility container.

Selector API
🎨
Custom Easing

Uses a smooth spring-like cubic-bezier curve for a polished, physical feel on every entrance.

Easing
♿
Accessible

Fully respects prefers-reduced-motion — all animations are disabled for users who prefer it.

A11y
📐
Responsive Grid

Cards use auto-fit minmax grid so the layout adapts from 1 to 3 columns without a media query breakpoint.

Responsive
🔁
Composable

Mix any animation class (stagger-fade-up, stagger-fade-in, stagger-slide-right) with any delay class freely.
