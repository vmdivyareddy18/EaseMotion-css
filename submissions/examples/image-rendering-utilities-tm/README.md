# image-rendering CSS utilities

Utilities for controlling how images are rendered — crisp edges vs smooth interpolation across browsers.

## Properties Covered

`image-rendering`, `-webkit-optimize-contrast`, `interpolate-police`

## Usage

Import the stylesheet and apply utility classes to your elements:

```html
<link rel="stylesheet" href="easemotion.css" />
<div class="...">Content</div>
```

## Responsive

All utilities include responsive variants with `sm:` and `md:` prefixes for mobile and tablet breakpoints.

## Dark Mode

Dark mode variants are automatically applied via `@media (prefers-color-scheme: dark)`.

## Accessibility

Respects `prefers-reduced-motion` — all transitions are disabled for users who prefer reduced motion.
