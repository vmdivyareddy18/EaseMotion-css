# Digital Twin Operations Center Component

## What does this do?

Provides a centralized interface for monitoring digital twins, connected assets, simulations, maintenance activities, and operational performance metrics.

## How is it used?

```html
<div class="asset-card healthy">
    <h3>Manufacturing Line A</h3>
</div>
```

## Why is it useful?

Organizations using digital twins need visibility into asset health, simulation accuracy, operational efficiency, and maintenance workflows. This component provides a reusable operations dashboard aligned with Industry 4.0 and IoT management practices.
