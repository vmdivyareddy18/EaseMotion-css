# Animated Empty State / Zero Data UI

A clean, modern, and fully responsive **Empty State Component** that turns a "zero data" scenario into an inviting starting point. This example demonstrates how to combine **EaseMotion CSS** for continuous floating effects, staggered text entrances, and a bouncy call-to-action button to create an engaging user experience.

## 🚀 Features

- **Floating Illustration**: The main folder icon and surrounding sparkles use `ease-float` with staggered delays to create a lively, continuous floating effect.
- **Pulsing Background**: A subtle background circle uses `ease-pulse` to draw attention to the illustration area.
- **Staggered Text Entrances**: The title, subtitle, and secondary link fade in sequentially using `ease-fade-in-up` and `ease-delay-*`.
- **Bouncy Call-to-Action**: The primary button uses `ease-bounce-in` to playfully pop into view, encouraging user interaction.
- **Interactive Hover States**: Uses `ease-hover-grow` on the button for satisfying tactile feedback.
- **Clean Card Layout**: Centered card design with soft shadows and rounded corners for a modern aesthetic.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing and full-width button on small devices.
