# Copy to Clipboard – Button with checkmark feedback

> A production-ready copy to clipboard component that provides instant visual feedback with a checkmark animation when content is successfully copied. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

**Live Preview:** Open `demo.html` in your browser.

---

## ✨ Features

- ✅ **One-click copy** – Copy text with a single click
- ✅ **Checkmark feedback** – Visual confirmation when copied
- ✅ **Multiple targets** – Inputs, textareas, code blocks
- ✅ **Fallback support** – Works on older browsers
- ✅ **Copy counter** – Track number of copies
- ✅ **4 variants** – Primary, Success, Outline, Minimal
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
