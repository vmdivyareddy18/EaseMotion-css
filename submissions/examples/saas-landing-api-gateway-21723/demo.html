Declarative configuration

Configure your API gateway exactly like your code. Use our YAML spec or native Terraform provider to version control your infrastructure.

  • ✓ GitOps ready
  • ✓ Terraform & Pulumi providers
  • ✓ CI/CD GitHub Actions
edgegate.yml
main.tf
index.js (SDK)
name: production-api
version: 1.0

routes:
  - path: /api/v1/*
    methods: [GET, POST, PUT, DELETE]
    backend:
      url: https://internal.api.company.com
    
    policies:
      - name: jwt-auth
        config:
          jwks_uri: https://auth.company.com/.well-known/jwks.json
      
      - name: rate-limit
        config:
          limit: 100
          window: 1m
          by: jwt.claims.sub
          
      - name: cache
        config:
          ttl: 300s
          methods: [GET]
