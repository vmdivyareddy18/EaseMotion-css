# Animated 404 Error Card Showcase

Resolves #43651.

### What does this do?
Provides a premium 404 error state card featuring a floating "404" text effect, a subtle rotating radial gradient background, and a responsive layout.

### How is it used?
Use the `.ease-error-card` class as a container for your empty state or 404 page content. The `.ease-error-illustration` applies the floating animation and gradient text effect.

### Why is it useful?
Empty states and 404 pages are often neglected. This component provides a polished, out-of-the-box solution that turns a negative user experience (a broken link) into a visually pleasing one.
