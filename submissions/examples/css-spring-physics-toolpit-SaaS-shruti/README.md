# Spring Physics Tooltip – SaaS Showcase

A responsive, accessible tooltip featuring a spring-like entrance animation using only CSS.

## Features

- Pure CSS
- Spring Physics animation
- Responsive design
- Keyboard accessible
- CSS custom properties for easy customization
- No JavaScript required

## Files

```
demo.html
style.css
README.md
```

## Custom Properties

```css
--ease-duration
--ease-easing
--ease-distance
--ease-scale
--ease-primary
--ease-tooltip
```

## Accessibility

- Keyboard focus supported
- Uses `role="tooltip"`
- Linked using `aria-describedby`

## Preview

Hover or focus on the button to display the animated tooltip.
