My animation isn't working

Answer each question to find the likely cause.

1. Is the class spelled correctly?

Check both the HTML class="..." attribute and the CSS selector — typos are the #1 cause.

2. Is the CSS file actually loaded?

Open DevTools → Network tab and confirm the stylesheet returns 200, not 404.

3. Is another style overriding it?

In DevTools → Elements, check if your animation/transition rule has a strikethrough.

4. Is the element display:none initially?

Animations don't run on elements with display:none — they need to already be in the render tree.

🔧

Found it: typo

Fix the class name so the HTML and CSS match exactly (case-sensitive, no extra spaces).

🔧

Found it: stylesheet not loading

Double-check the <link> path, your build output folder, and any caching issues.

🔧

Found it: specificity conflict

A more specific selector or !important elsewhere is winning. Increase specificity or reorder your stylesheets.

🔧

Found it: hidden on load

Toggle visibility:hidden + opacity:0 instead of display:none, or add the class after the element is already visible.

🔍

Dig deeper

Common remaining causes: a typo in the animation-name/keyframes name, animation-duration:0s, or the element being animated lacking a property that can actually transition (e.g. animating height:auto).
