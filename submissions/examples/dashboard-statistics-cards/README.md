# Dashboard Statistics Cards

A modern responsive dashboard statistics card component built using pure HTML and CSS.

## Features

- Multiple statistics cards
- Animated hover effects
- Trend indicators
- Responsive grid layout
- Icon support
- Modern dashboard styling
- Pure HTML and CSS

## Folder Structure

```text
dashboard-statistics-cards/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any browser.
2. Ensure `style.css` is in the same directory.
3. Customize icons, metrics, colors, and trends.

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
- CSS Transitions
- Responsive Design

## Components Included

- Revenue Card
- Users Card
- Orders Card
- Bounce Rate Card

## Animations

- Card hover lift effect
- Icon scale animation
