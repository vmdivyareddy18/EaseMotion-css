# ⌨️ Animated Keyboard Shortcut Showcase

A modern collection of animated keyboard shortcut cards built using pure HTML and CSS. This component presents common keyboard shortcuts in a clean, responsive card layout suitable for documentation websites, developer portfolios, blogs, learning platforms, and UI showcases.

---

## ✨ Features

- Modern glassmorphism-inspired design
- Responsive CSS Grid layout
- Animated keyboard keycaps
- Smooth hover interactions
- Floating card animation
- Reusable shortcut card component
- Pure HTML & CSS (No JavaScript)
- Mobile-friendly layout

---

## 📂 Included Categories

- 💻 VS Code Shortcuts
- ✏️ Editing Shortcuts
- 🌐 Browser Shortcuts
- 🖥️ Windows Shortcuts
- 🍎 macOS Shortcuts

## 🚀 Usage

Include the stylesheet in your HTML.

```html
<link rel="stylesheet" href="style.css">
```

Then create a shortcut card like this:

```html
<div class="card">

    <div class="shortcut-icon">📋</div>

    <h3>Copy</h3>

    <div class="keys">
        <kbd>Ctrl</kbd>
        <span>+</span>
        <kbd>C</kbd>
    </div>

    <p>
        Copies the selected content.
    </p>

</div>
```
