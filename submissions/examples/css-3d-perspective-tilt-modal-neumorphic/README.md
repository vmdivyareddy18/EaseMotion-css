# CSS 3D Perspective Tilt Modal (Neumorphic Soft UI)

A **pure CSS 3D animated modal component** designed for **Neumorphic Soft UI interfaces** in the EaseMotion CSS library.

---

## ✨ Features

- Pure CSS 3D perspective animation
- Neumorphic soft shadow design
- Hover-based 3D tilt interaction
- Smooth modal open/close transitions
- Fully responsive layout
- Keyboard accessible (ESC support)

---

## 🎯 Purpose

This component enhances **soft UI / neumorphism design systems** by combining:

- depth-based shadows
- 3D perspective motion
- interactive hover feedback

All implemented using only CSS.

---

## 🧪 Usage

```html
<button class="open-btn" onclick="openModal()">Open Modal</button>

<div class="modal active">
  <div class="modal-content">
    Neumorphic content here
  </div>
</div>
