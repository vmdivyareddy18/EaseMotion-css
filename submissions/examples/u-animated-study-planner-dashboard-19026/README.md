# Animated Study Planner Dashboard

A modern study planner dashboard featuring subject progress tracking, exam countdowns, achievement badges, and responsive layouts.

## Features

- Subject progress cards
- Animated progress bars
- Exam countdown widgets
- Study statistics
- Achievement badges
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any browser.

## Customization

- Subjects
- Progress values
- Exam dates
- Achievement badges
- Colors and animations

## Use Cases

- Student dashboards
- Learning management systems
- Study planners
- Educational platforms
- Productivity trackers
