# Ease Package Dependency Tree

## Overview

Ease Package Dependency Tree is a reusable visualization component that displays hierarchical package relationships inspired by npm dependency graphs. Built entirely with HTML and CSS.

---

## Features

- 🌳 Hierarchical dependency tree
- 🔗 Animated connector lines
- ✨ Pulsing package nodes
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Root application node
- Multiple dependency levels
- Animated tree connectors
- Interactive package nodes

---

## Usage

```html
<ul class="tree">
...
</ul>
```

---

## Customization

- Node colors
- Tree depth
- Connector colors
- Animation speed
- Typography

---

## Browser Support

Compatible with all modern browsers supporting CSS animations and pseudo-elements.

---

## Why This Fits EaseMotion CSS

Perfect for developer documentation, package visualization, dependency explorers, technical blogs, and educational content. Provides a lightweight, reusable tree visualization using only HTML and CSS.
