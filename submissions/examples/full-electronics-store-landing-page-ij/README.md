# Full Electronics Store Landing Page

1. What does this do? A modern electronics e-commerce landing page with product categories, deals section, featured products grid, service features, and newsletter CTA using only EaseMotion CSS framework classes.

2. How is it used? Open `demo.html` — every layout grid, product card, hover effect, scroll reveal, and responsive behavior is powered by EaseMotion classes (`ease-grid-cols-3`, `ease-card-hover`, `ease-hover-glow`, `ease-reveal-up`, `ease-delay-*`, etc.).

3. Why is it useful? Shows EaseMotion handling a product-heavy tech e-commerce page with deal badges, price tags, category icons, and service feature cards — all framework-native with no custom keyframes.
