# 🚀 Space Jumper

Jump between asteroids in zero gravity! Collect stars, avoid meteorites, and survive the cosmos in this addictive space jumping game.

## 🎮 How to Play

- **Click/Tap** to jump from one asteroid to another
- **Press Space** to jump (keyboard)
- **Land on asteroids** to survive and score points
- **Collect ⭐ stars** for bonus points
- **Avoid 💥 meteorites** or it's game over!
- **Different gravity** on each asteroid changes your jump

## ✨ Features

- **🌍 Variable Gravity** - Each asteroid has different gravity (0.5x to 2.0x)
- **🪐 Procedural Asteroids** - Randomly generated with unique sizes and colors
- **⭐ Star Collectibles** - Bonus points for extra challenge
- **💥 Meteorites** - Dodge falling space rocks
- **🎵 Sound Effects** - Jump, land, collect, and game over sounds
- **🌠 Particle Effects** - Beautiful visual feedback for all actions
- **📊 Real-time Stats** - Score, asteroid count, current gravity
- **📱 Mobile Friendly** - Touch support with responsive design

## 🎯 Scoring

| Action | Points |
|--------|--------|
| Land on asteroid | + (size/10) |
| Collect star | +50 |
| Survive longer | +1 per frame |

## 🏆 Gravity System

| Gravity | Color | Effect |
|---------|-------|--------|
| 0.5x - 0.8x | 🟢 Teal | Floaty jumps |
| 0.8x - 1.2x | 🟡 Gold | Normal jumps |
| 1.2x - 1.6x | 🔴 Red | Heavy jumps |
| 1.6x+ | 🟠 Orange | Super heavy |

## 🛠️ Technical Details

### Built With
- **HTML5 Canvas** - Game rendering
- **CSS3** - Styling and animations
- **Vanilla JavaScript** - Game logic and physics

### Physics System
- **Multi-body gravity** - Influenced by nearest asteroid
- **Variable jump force** - Based on asteroid gravity
- **Momentum preservation** - Realistic space movement
- **Collision detection** - Circle-based asteroid collisions

### Game Loop
- **60fps rendering** - Smooth animations
- **RequestAnimationFrame** - Optimized performance
- **State management** - Start, playing, game over

## 💡 Tips & Strategies

1. **Plan your jumps** - Look ahead to the next asteroid
2. **Watch gravity** - High gravity asteroids need stronger jumps
3. **Collect stars** - They appear on random asteroids
4. **Dodge meteorites** - They come from all directions
5. **Build momentum** - Use velocity to reach distant asteroids
6. **Don't panic** - The game gets easier with practice!
