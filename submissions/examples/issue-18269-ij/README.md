# Masonry Grid with Staggered Entry (Issue #18269)

A CSS columns-based masonry grid with items of varying heights. Each item fades and slides up with a staggered `animation-delay` based on its order.

## Usage

Open `demo.html` in a browser. Grid items animate in sequentially.

## Files

- `demo.html` — HTML5 page with masonry grid
- `style.css` — Dark theme styles with `@keyframes fadeSlideUp`
