# Procurement Bid Comparison Table

A feature-rich Procurement Bid Comparison Table built entirely with HTML and CSS.

## Overview

This example demonstrates a modern enterprise procurement evaluation dashboard designed for sourcing teams to compare supplier bids across financial, technical, operational, and sustainability criteria. The interface combines KPI cards, an advanced comparison matrix, weighted evaluation metrics, procurement recommendations, and executive summaries into a responsive purchasing workspace.

Unlike a traditional table, this example presents a complete procurement decision-support interface suitable for ERP systems, supplier portals, and enterprise sourcing applications.

---

## Features

- Enterprise procurement dashboard
- Advanced supplier comparison matrix
- Sticky comparison table header
- Procurement KPI cards
- Vendor recommendation badges
- Financial risk indicators

- Technical evaluation comparison
- Delivery timeline analysis
- Warranty comparison
- Weighted evaluation progress bars
- Animated procurement score ring
- Executive recommendation panel
- Responsive enterprise layout
