# Grayscale Filter Utility Showcase

## What does this do?
Demonstrates the implementation of CSS grayscale filter utility classes (`.ease-filter-grayscale` and `.ease-filter-grayscale-0`) with interactive hover states on cards.

## How is it used?
Apply the utilities to images, avatars, or other elements to control grayscale level dynamically.

```html
<div class="card">
  <img src="avatar.jpg" class="ease-filter-grayscale hover:ease-filter-grayscale-0 transition-all duration-300" alt="Avatar">
</div>
```

## Why is it useful?
Provides native utility support to apply standard grayscale filters and transition states in EaseMotion CSS, allowing developers to create highly interactive, premium UI effects (such as gallery hover reveals or inactive profile badges) without writing custom CSS rules.
