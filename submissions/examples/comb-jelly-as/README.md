# Comb Jelly

A pure CSS and HTML ctenophore drifting in the black deep, rainbow light chasing down the eight comb rows that beat it through the water.

## What it shows

- Comb rows as rainbow repeating linear gradients scrolled with background-position, so colour visibly races down each row like real ciliary diffraction
- Staggered delays across the rows so the shimmer travels around the body instead of firing at once
- A translucent gelatinous bell from layered soft gradients with a rim highlight, pulsing as it swims
- Trailing tentacles and a second jelly scaled down and blurred for depth in the dark
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
