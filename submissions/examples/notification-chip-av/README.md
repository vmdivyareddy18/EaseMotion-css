# Animated Notification Chip UI Example

Notification chips with bounce and color transitions for updates!

## What does this do?

Animated notification chips that show up with a bounce animation when triggered, with success/info/warning color variants!

## How is it used?

Add the chip HTML structure and use the JavaScript from demo.html to trigger them!

## Why is it useful?

Perfect for notifying users of events, fits EaseMotion's animation-first philosophy!

## Features

- ✨ Bounce entrance animation
- 🟢 Success variant
- 🔵 Info variant
- 🟡 Warning variant
- ❌ Close button
- ⏱️ Auto-removal after 5 seconds
- ♿ Accessible and clear UI
- 📱 Responsive design
