# Rotating Bars Spinner (Issue #18261)

A classic loading spinner with 8 bars radiating from the center. Each bar fades in and out with staggered animation delays. Pure CSS.

## Usage

Open `demo.html` in a browser. The spinner animates continuously.

## Files

- `demo.html` — HTML5 page with rotating bars spinner
- `style.css` — Dark theme styles with `@keyframes spinFade`
