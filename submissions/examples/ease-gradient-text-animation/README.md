# CSS Gradient Text Animation (`ease-gradient-text-animation`)

A pure CSS animated text gradient component using `background-clip: text` and animated `background-position`.

## 🚀 Features & EaseMotion Showcase

- **Pure CSS**: Zero JavaScript required. Smooth hardware-accelerated animations using `background-position`.
- **Cross-Browser**: Properly utilizes both standard and `-webkit-` prefixes for Safari and Chrome compatibility.
- **Continuous Flow**: Features a seamless looping linear-gradient that translates cleanly across the text.
- **Multiple Palettes**: Includes 4 stunning starting palettes (Rainbow, Sunset, Cyberpunk, Ocean).

## 🛠️ Usage

This demo is self-contained. Open `demo.html` in your browser. All required CSS is inside `style.css`.

```html
<h1 class="ease-text-gradient ease-text-gradient--cyber">
  Animated Text
</h1>
