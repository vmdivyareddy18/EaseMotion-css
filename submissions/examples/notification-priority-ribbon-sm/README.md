# Notification Priority Ribbon

1. What does this do? Adds a responsive notification stack with animated priority ribbons, slide-in entry motion, hover lift feedback, and keyboard-visible focus states.
2. How is it used? Apply `.notification-list` to the wrapper and `.notice-card` with a priority class such as `.critical`, `.warning`, `.success`, or `.info` to each notification.
3. Why is it useful? It makes urgent messages easier to scan while keeping the interaction lightweight, accessible, and CSS-only.
