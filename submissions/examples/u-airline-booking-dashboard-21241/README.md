# Pharmacy Management Dashboard

A responsive **Pharmacy Management Dashboard** built with **HTML5** and **CSS3** that demonstrates medicine inventory tracking, prescription management, stock monitoring, and pharmacy workflow animations using EaseMotion CSS.

---

## Features

- Responsive dashboard layout
- Pharmacy overview cards
- Medicine stock progress
- Prescription queue
- Expiry alerts
- Recent pharmacy activity
- Notification panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Medicines in Stock
- Prescriptions Today
- Pending Orders
- Daily Revenue

### Pharmacy Widgets

- Medicine Stock Progress
- Prescription Queue
- Expiry Alerts

### Activity

- Recent Pharmacy Activity Feed
- Notifications Panel

### Quick Actions

- Add Medicine
- Process Prescription
- Generate Invoice

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations.
