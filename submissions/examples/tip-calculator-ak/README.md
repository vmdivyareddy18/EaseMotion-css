# Tip Calculator

## What does this do?
A tip calculator with preset percentages, custom tip entry, and split bill functionality for up to 20 people.

## How is it used?
Open demo.html in a browser. Enter the bill amount, select a tip percentage (or enter a custom one), and adjust the split count. The tip, total, and per-person amounts update instantly.

## Why is this useful?
This provides a practical dining or service cost calculator for everyday use. It demonstrates preset button groups, split controls, and dynamic result aggregation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
