# Magnetic Hover Lift

A reusable CSS hover effect that creates a smooth magnetic lifting animation with scaling and glowing shadows.

## Features

- Pure CSS
- Responsive
- Lightweight
- Modern hover interaction
- Reusable for cards, buttons, dashboards, pricing sections, and feature panels

## Files

```
u-magnetic-hover-lift-20853/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<div class="magnetic-card">
    <h2>Interactive Card</h2>
    <p>Hover to experience the effect.</p>
</div>
```

## Customization

Modify the accent color:

```css
.magnetic-card{
    --accent:#8b5cf6;
}
```

Available demo variants:

- Purple
- Blue
- Orange
- Green

## Browser Support

Works in all modern browsers supporting:

- CSS Transforms
- CSS Transitions
- CSS Variables
- Box Shadow
