# Ease Banner Dismissible

This demo implements a **top-of-page announcement banner** with animated entrance, dismiss functionality, and persistence using localStorage.

## 🧠 How It Works
- Slides down from the top on page load.
- Includes a dismiss button that collapses the banner smoothly.
- Exit animation combines height and opacity transitions.
- Uses `localStorage` to persist dismissal across sessions.
- Supports color variants: info, warning, and success.

## 🧩 Files
- `demo.html` — main demo with banner logic.
- `style.css` — styling, transitions, and color variants.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Observe the banner sliding down from the top.
3. Click the ✖ button to dismiss it.
4. Refresh the page — the banner remains hidden due to persistence.

## 🧰 Technologies
- HTML5
- CSS3 (transitions, transforms)
- JavaScript (localStorage)
