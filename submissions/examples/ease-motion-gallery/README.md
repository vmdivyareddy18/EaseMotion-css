# 🎨 EaseMotion CSS Gallery

A beautiful collection of CSS animations and easing effects showcasing modern motion design techniques.

## ✨ Features

### Animation Cards
- 💫 **Pulse Effect** - Smooth pulsing animation
- 🔄 **3D Rotate** - Continuous rotation
- ⚡ **Bounce** - Elastic bounce motion
- 🎈 **Floating** - Gentle floating effect
- 🔔 **Shake** - Attention-grabbing shake
- 🎯 **Scale** - Growing & shrinking
- ➡️ **Slide** - Smooth sliding motion
- ✨ **Glow** - Radiant glowing effect

### Animated Buttons
- Ripple Effect
- Slide Fill
- Glow Button
- Bounce Click

### Easing Functions Demo
- linear
- ease
- ease-in-out
- cubic-bezier

## 🚀 How to Use

1. Clone or download the repository
2. Open `demo.html` in any modern browser
3. Explore the animations and effects
