# Animated Modal Popup Component

## Description

A CSS-only animated modal popup with smooth fade effects, scale animations, overlay background, and interactive buttons.

---

## Features

- Smooth fade-in overlay animation
- Popup scale entrance effect
- Animated modal icon
- Hover interactions on buttons
- Responsive modal layout
- Lightweight CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.modal-overlay` – Background overlay layer
- `.modal-card` – Main popup container
- `.modal-icon` – Animated icon element
- `.modal-buttons` – Action button wrapper
- `.btn` – Modal buttons

---

## Accessibility

Supports `prefers-reduced-motion` to disable animations for users who prefer reduced motion.

---

## Use Cases

- Alert messages
- Confirmation dialogs
- Login popups
- Promotional messages
- Information cards
