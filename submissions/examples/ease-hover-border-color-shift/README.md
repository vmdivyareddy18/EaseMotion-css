# Border Color Shift on Hover

Border color shifts on hover — pure CSS.

## Features
- Smooth border-color transition on hover
- Pure CSS, no JavaScript

## Usage
```css
.border-card { border: 3px solid #2a2a2a; transition: border-color 0.3s; }
.border-card:hover { border-color: #6366f1; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
