# EaseMotion VS Code Snippets

This example documents common VS Code snippets that developers can use while working with EaseMotion CSS.

## Included Snippets

- em-fade-in
- em-hover-card
- em-stagger-list
- em-slide-up
- em-scale-hover
- em-button
- em-card
- em-navbar

## Example

```html
<div class="ease-fade-in">
    Content
</div>
```

## Benefits

- Faster development
- Easy autocomplete
- Consistent class usage
- Better developer experience
