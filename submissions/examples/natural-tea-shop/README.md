# 🍃 Natural Tea Shop

A modern and responsive **Natural Tea Shop** landing page built using **EaseMotion CSS**. The page showcases premium organic teas with a clean and elegant design, featuring smooth animations and responsive layouts.

## ✨ Features

* Hero section with tea lifestyle imagery
* Featured tea collection
* Organic tea categories
* Monthly subscription CTA
* Responsive navigation
* Modern product cards
* Responsive footer
* Smooth EaseMotion animations
