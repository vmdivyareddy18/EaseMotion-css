CSS Architecture Fix: Z-Index Scale

This demo showcases how to properly structure z-indexes using a centralized CSS variable scale, fixing catastrophic layering conflicts across 2,000+ components.

Incorrect Architecture

Dropdown (999)
Modal (99)

The dropdown is incorrectly rendering on top of the modal overlay.

Correct Architecture

Dropdown (var(--z-dropdown))
Modal (var(--z-modal))

The modal correctly overlays everything else.
