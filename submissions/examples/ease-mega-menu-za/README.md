# CSS Mega Menu Component

## What does this do?
Full-width CSS dropdown mega menu with hover activation, multi-column categories, and smooth transitions. Perfect for content-rich navigation.

## How is it used?
```html
<nav class="mgm-nav">
  <ul class="mgm-menu">
    <li class="mgm-has-mega"><a href="#">Products</a>
      <div class="mgm-mega">
        <div class="mgm-col"><h3>Category</h3><a href="#">Link</a></div>
      </div>
    </li>
  </ul>
</nav>
```

## Why is it useful?
Provides an organized multi-column dropdown navigation for sites with many categories. Pure CSS hover activation, responsive collapse on mobile, and customizable styling via variables.
