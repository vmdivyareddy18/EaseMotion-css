# ease-hover-text-up

A lightweight hover animation that shifts text upward by **3px**, creating a subtle interactive effect perfect for navigation links and buttons.

## Features

- Pure CSS
- No JavaScript
- Smooth 0.25s transition
- Ideal for navigation menus
- Easy to customize with CSS variables

## Usage

```html
<a class="ease-hover-text-up" href="#">
    Home
</a>
```

## CSS Variables

```css
:root{
    --ease-hover-text-up-distance: -3px;
    --ease-hover-text-up-duration: 0.25s;
}
```
