# Activity Feed / Notification Timeline

A vertical timeline showing user activity items. Each event has an indigo dot connected by a vertical line to the next item. Displays notification text with bold mentions and a relative timestamp.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the activity feed.

## Accessibility notes

The timeline is rendered with semantic HTML. The connecting lines are purely decorative CSS pseudo-elements.
