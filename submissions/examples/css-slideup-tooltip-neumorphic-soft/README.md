# CSS Slide-Up Tooltip – Neumorphic Soft Layout

## Overview

A responsive Neumorphic Soft UI tooltip component built entirely with HTML and CSS. The component features a smooth slide-up animation, soft-shadow cards, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Slide-Up tooltip animation
- Neumorphic Soft UI design
- Responsive soft-shadow cards
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--background`
- `--surface`
- `--primary`
- `--accent`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--slide-distance`
- `--shadow-dark`
- `--shadow-light`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any **Learn More** button to display the animated Slide-Up Tooltip. Colors, spacing, shadows, animation timing, and slide distance can all be customized using the CSS variables defined in `style.css`.
