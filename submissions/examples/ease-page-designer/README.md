# Full UI/UX Designer Portfolio (`ease-page-designer`)

A modern, clean, and image-forward portfolio template for Product/UX/UI Designers, showcasing how EaseMotion classes can elevate a professional profile.

## 🚀 Features & EaseMotion Showcase

This landing page integrates multiple core `.ease-*` classes to create a polished, premium feel typical of top-tier design portfolios:

- **Elegant Typography Reveal**: Uses staggered `.ease-slide-up` classes to reveal the hero text and subtitle sequentially, creating a thoughtful, narrative introduction.
- **Hover Interactions**: Incorporates `.ease-hover-lift` on project cards and tool icons to provide tactile feedback and depth, which is critical for UI design portfolios.
- **Continuous Animation**: Introduces an `.ease-float` infinite animation on abstract UI elements in the hero section to keep the page feeling "alive" and interactive even when the user isn't scrolling.
- **Staggered Grids**: Employs an `IntersectionObserver` combined with `animation-delay` attributes on `.ease-slide-up` and `.ease-slide-right` to create satisfying waterfall effects for the process steps and case studies.

## 🛠️ Usage

This demo is entirely self-contained. You can open `demo.html` directly in your browser. All required EaseMotion CSS classes have been defined at the top of the `style.css` file for immediate testing. 

If using this as a template for a real project, simply migrate the HTML structure into your application and ensure your global CSS bundle includes the `ease-*` utility classes.
