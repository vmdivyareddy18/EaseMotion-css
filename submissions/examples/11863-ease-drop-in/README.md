# ease-drop-in

1. What does this do? It animates an element dropping from above, easing into place with a bounce and a subtle squash on landing.
2. How is it used? Apply the demo class to the element you want to drop in, for example `<div class="drop-card is-animating">...</div>`.
3. Why is it useful? It gives common placement moments, like cards, pins, and alerts, a readable physics feel that fits EaseMotion CSS's animation-first style.
