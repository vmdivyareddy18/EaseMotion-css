# Checkbox Variants

## What does this do?

Provides a colorful collection of custom checkbox styles with modern hover effects and multiple visual themes.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<label class="check purple">

<input type="checkbox">

<span></span>

Purple

</label>
```

---

## Why is it useful?

Checkboxes are one of the most frequently used form controls in web applications. This showcase demonstrates multiple reusable checkbox styles—including colorful and glass-inspired variants—using only HTML and CSS while aligning with EaseMotion CSS's lightweight and animation-first design philosophy.
