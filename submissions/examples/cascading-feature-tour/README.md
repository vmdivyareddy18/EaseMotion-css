# Cascading Feature Tour

A CSS-only animated feature tour component inspired by Health & Fitness design patterns.

## Features

- Cascading entrance animation
- Modern card-based layout
- Smooth hover interaction
- Responsive design
- Pure CSS implementation
- No JavaScript required

## Usage

```html
<section class="tour">

  <article class="feature">
    <h2>Feature Title</h2>
    <p>Description</p>
  </article>

</section>
