# Mac-Style Dock Navigation

## What does this do?
CSS dock navigation with hover scale-up effect, tooltips, active indicators, divider, and multiple style themes.

## How is it used?
```html
<div class="dkn-dock"><div class="dkn-item dkn-active"><div class="dkn-icon dkn-finder"></div></div></div>
```

## Why is it useful?
Desktop-style dock navigation without JavaScript. Hover scale with sibling cascade. Tooltip labels on hover. Dark, glass, and minimal themes. Bounce animation variant.
