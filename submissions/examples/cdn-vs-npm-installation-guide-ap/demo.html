Detailed Setup Workflows

      

        

          
CDN Setup

          

            Copy the jsDelivr <link> tag and paste it inside the HTML <head> element.
            This fetches the pre-compiled, minified stylesheet directly from the CDN server edges.
          

        


        

          
NPM Setup

          

            Run npm install easemotion-css to add the package as a dependency.
            Import the style module in your index file (e.g., import 'easemotion-css') to compile it with your bundler.
