# Gradient BG Shimmer

An animated gradient background component with a flowing shimmer effect. Colors shift smoothly across the viewport in a continuous loop.

## Features

- Smooth gradient color cycling via `background-position` animation
- Multiple gradient presets (Ocean, Sunset, Forest, Neon, Lavender)
- Play / Pause toggle for the animation
- Customizable colors, angle, and duration via CSS custom properties

## Custom Properties

| Property          | Default    | Description                  |
| ----------------- | ---------- | ---------------------------- |
| `--grad-duration` | `10s`      | Animation cycle duration     |
| `--grad-color-1`  | `#0c3b5a`  | First gradient color         |
| `--grad-color-2`  | `#1a6b8a`  | Second gradient color        |
| `--grad-color-3`  | `#3b9ec4`  | Third gradient color         |
| `--grad-angle`    | `135deg`   | Gradient direction angle     |
