# Ease Floating Action Menu

A clean Floating Action Button (FAB) component that expands upward on hover to reveal multiple secondary action buttons.

## Features

- Floating Action Button (FAB)
- Smooth CSS transform animations
- Opacity transition
- Hover expansion
- Responsive layout
- Pure HTML & CSS
- Beginner-friendly component

## Technologies

- HTML5
- CSS3

## Usage

Open `demo.html` in any modern browser.
