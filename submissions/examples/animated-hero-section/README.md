# Animated Hero Section

A responsive animated hero section component built using pure HTML and CSS.

## Features

- Animated headline effects
- Floating background elements
- CTA button hover animations
- Responsive layouts
- Smooth entrance transitions
- Modern landing page aesthetics
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated hero section component.

## Files

- `demo.html` → Hero section structure
- `style.css` → Hero styles and animations
- `README.md` → Documentation
