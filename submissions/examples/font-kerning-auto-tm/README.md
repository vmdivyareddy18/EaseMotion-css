# Font Kerning Auto

font-kerning — Enables or disables Kerning pair letter spacing

## Features
- Uses `--ease-color-text`
- Uses `--ease-text-base`
- Uses `--ease-color-bg`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Font Kerning Auto</span>
</div>
```

## Why is it useful?

This submission demonstrates the font kerning auto CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
