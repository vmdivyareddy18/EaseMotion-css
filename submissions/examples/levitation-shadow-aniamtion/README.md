# Levitation Shadow Animation

A lightweight levitation animation utility built using pure CSS.

## Features

- Smooth floating effect
- Dynamic shadow movement
- Realistic levitation feel
- Infinite ease-in-out animation
- No JavaScript required

## Usage

```html
<div class="float-card">
  Floating Card
</div>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable motion effect
