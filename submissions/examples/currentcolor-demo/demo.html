currentColor Inheritance

SVG icons, borders, box-shadows, and pseudo-elements that all inherit the parent text color.

Theme:

SVG Icons

stroke="currentColor" makes the icon match the card's text color.

Borders & Outline

border: 2px solid currentColor

The border automatically changes when the theme color updates.

Box Shadow

box-shadow: 0 4px 20px currentColor

No hardcoded shadow color — it inherits from the text.
