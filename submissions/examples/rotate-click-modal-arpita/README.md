# Rotate Click Modal

## What does this do?
This component provides a pure CSS modal with a smooth rotate-and-scale opening animation for modern SaaS interfaces.

---

## How is it used?
Include the HTML structure and apply the provided CSS classes.

```html
<input type="checkbox" id="modal-toggle" class="modal-toggle">

<label for="modal-toggle" class="open-btn">
    Open Modal
</label>

<div class="overlay">
    <div class="modal">
        <!-- Modal Content -->
    </div>
</div>
```

---

## Why is it useful?
This component demonstrates how modern modal interactions can be achieved using only HTML and CSS. It improves user experience with smooth animations, responsive design, customizable CSS variables, and zero JavaScript, aligning with EaseMotion CSS's philosophy of lightweight, reusable, and visually engaging UI components.
