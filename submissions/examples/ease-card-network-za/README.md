# Card Network Grid

## What does this do?
CSS connected card layouts with visual link indicators, hub-and-spoke diagrams, tree views, and relationship cards.

## How is it used?
```html
<div class="cn-grid"><div class="cn-card cn-linked">Node</div><div class="cn-card cn-child">Child</div></div>
```

## Why is it useful?
Creates visual relationship diagrams without JavaScript or SVG. Linked cards with CSS pseudo-element connectors. Hub-and-spoke, tree, and relationship variants. Ideal for org charts, dependency graphs, and network visualizations.
