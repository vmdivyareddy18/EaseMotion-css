# Sepia Filter Showcase - EaseMotion CSS

A beautiful, interactive component showcase presenting the sepia filter transition classes.

## Features Included

- `.ease-filter-sepia`: Applies `filter: sepia(100%) !important` to an element.
- `.ease-filter-sepia-0`: Applies `filter: sepia(0%) !important` to an element.
- Serifs journal cards layout showing how warm tones interact with layouts on hover.

## How to Run

1. Open `demo.html` directly in any modern web browser.
2. Hover over the cards to observe the interactive sepia transitions.

## Why is it useful?

Sepia filter transitions provide warm, vintage, and classic visual designs. Developers can use them for photograph grids, interactive memory walls, travel logs, and hover effects that shift color temperatures.
