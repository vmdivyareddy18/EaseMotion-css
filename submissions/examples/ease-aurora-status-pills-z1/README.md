# Ease Aurora Status Pills

## Overview

A premium collection of animated status pills featuring aurora-inspired highlights, glowing indicators, and glassmorphism styling built entirely with HTML and CSS.

## Features

- Aurora sweep animation
- Animated status indicators
- Glassmorphism container
- Responsive layout
- Lightweight CSS
- No JavaScript

## Folder Structure

```text
ease-aurora-status-pills-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Designed to adapt smoothly across desktop, tablet, and mobile screens.

Showcases reusable animated UI states with subtle motion and clean styling suitable for dashboards, developer tools, and modern interfaces.
