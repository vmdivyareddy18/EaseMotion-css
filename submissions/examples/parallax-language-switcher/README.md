# Parallax Language Switcher — Corporate Design Patterns

A pure CSS language selection interface implementing layered spatial depth transitions using perspective transforms and controlled parallax motion.

## Specifications

- **Spatial Depth Rendering**: Utilizes CSS `perspective` and `translateZ` transformations to create multi-layer language card separation along the Z-axis.
- **Declarative Interaction**: Implements hover-driven parallax transitions entirely through CSS selectors without JavaScript dependencies.
- **Motion Architecture**: Integrates EaseMotion CSS variables for consistent animation timing, easing curves, and framework-aligned transitions.
- **Accessible Controls**: Uses semantic button elements with ARIA labels and visible focus states for keyboard navigation support.
- **Responsive Composition**: Maintains adaptive positioning and scaling behavior across different viewport sizes.
