# Wind Direction Arrow

1. What does this do? A compass-style arrow that smoothly rotates to a target wind direction angle (set via JS) with a subtle idle sway animation at rest.

2. How is it used? Include `.wind-arrow` on the arrow element and set `--target-deg` via JavaScript. Toggle `.swaying` and `.rotating` classes for idle sway and rotation respectively.

3. Why is it useful? It provides a realistic wind direction indicator for weather dashboards and widgets with smooth transitions and natural-feeling idle motion, all within the EaseMotion CSS animation philosophy.
