# Double Border Expand

## What does this do?
Triggers an expanding secondary offset border frame around the element on hover.

## How is it used?
Add the class to your button:

```html
<button class="double-border-btn">Interactive Button</button>
```

## Why is it useful?
It provides a retro/tech styled, crisp interactive highlight that visually locks the active target cleanly.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
