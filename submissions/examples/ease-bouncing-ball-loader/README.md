# Bouncing Ball Loader – Ball bounces across the screen

> A production-ready bouncing ball loader component that features a realistic ball bouncing across the screen with physics simulation, customizable speed, bounce, gravity, and multi-ball support. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Realistic physics** – Gravity, bounce, friction
- ✅ **Smooth animation** – 60fps with trails
- ✅ **Customizable** – Speed, bounce, gravity, color
- ✅ **Multi-ball** – Toggle between single and multiple balls
- ✅ **Pause/Resume** – Control the animation
- ✅ **Reset** – Reset the ball(s)
- ✅ **Bounce counter** – Tracks total bounces
- ✅ **Keyboard shortcuts** – Space (pause), R (reset), M (multi-ball)
- ✅ **4 color variants** – Aurora, Sunset, Gold, Dream
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
