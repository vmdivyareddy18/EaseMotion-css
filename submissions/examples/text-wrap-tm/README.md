# Utility: text-wrap balance and pretty

Demonstrates the `.ease-text-balance` and `.ease-text-pretty` utilities using modern CSS text-wrap properties for improved typography.

## Features

- Pure HTML & CSS
- `text-wrap: balance` for headings — prevents orphan lines
- `text-wrap: pretty` for body text — better line breaking
- Side-by-side comparison (with/without utilities)
- Dark mode compatible
- Reduced motion support
- No JavaScript required

## Usage

```html
<h2 class="ease-text-balance">Balanced Headline Text</h2>
<p class="ease-text-pretty">Body text with improved line breaks</p>
```

## Why is it useful?

Modern CSS `text-wrap: balance` and `text-wrap: pretty` are widely supported (Chrome 117+, Safari 18+, Firefox 121+). They eliminate manual line-break hacks and improve readability automatically — a quality-of-life upgrade for any text-heavy UI.
