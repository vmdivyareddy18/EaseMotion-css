# 13 — named-grid-areas

## Overview
Showcases CSS `grid-template-areas` for named grid layouts. The demo includes an ASCII art map showing the intended layout (nav, main, sidebar, ads, footer) and a live grid container with responsive reordering at three breakpoints via area name reassignment.

## Files
- `demo.html` — page with ASCII layout map and a responsive named-area grid
- `style.css` — dark-themed styles with three `grid-template-areas` breakpoints

## Usage
Open `demo.html` and resize the browser window. The grid items automatically reorder as area names are reassigned to different positions at 700px and 480px breakpoints.
