# Cascading Navigation Bar Component

A lightweight, accessible navigation bar component featuring smooth cascading motion effects and Neumorphic aesthetics. Built natively for the **EaseMotion CSS** library.

## Features
- **100% Pure CSS:** Interactivity is managed entirely without JavaScript via an unblocking checkbox structural hack.
- **Staggered Cascade:** Target links gracefully drop down in sequence using CSS Custom Properties and real-time calculation matrices (`calc()`).
- **Neumorphic Aesthetic:** Precise flat and inset outer box-shadow variables designed to match modern soft UI interfaces.
