# Zoom Hover Card

Cards that scale up on hover with a zoom effect on the image area.

## Features

- Card scale transform on hover with bounce cubic-bezier
- Image area scales independently for parallax-like effect
- Shadow intensifies on hover
- Clean card layout with icon, title, and description
- Pure CSS hover transitions

## Usage

Include `style.css` and structure cards with `.zoom-card`, `.zoom-img`, and `.zoom-body`. The hover affects both the card scale and the image scale independently.
