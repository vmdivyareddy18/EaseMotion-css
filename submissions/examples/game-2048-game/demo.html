🎮 2048 – classic game
• Use arrow keys (↑ ↓ ← →) to slide tiles.
• Tiles with the same number merge into one.
• Each move adds a new tile (2 or 4) on an empty cell.
• Goal: create a tile with the value 2048.
• Game ends when no moves left — press New Game to restart.
• Built with vanilla HTML + CSS + JS. No dependencies.
