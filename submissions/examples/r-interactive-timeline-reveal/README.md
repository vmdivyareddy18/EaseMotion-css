# Interactive Timeline Reveal Animation

A responsive timeline component demonstrating reveal and hover animations using EaseMotion CSS concepts.

## Features

- Alternating timeline layout
- Smooth reveal animation
- Hover elevation effect
- Responsive design
- Lightweight and customizable

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser to view the timeline animation example.

## Customization

You can:

- Add more timeline items
- Change colors and spacing
- Adjust animation duration
- Integrate EaseMotion utility classes
