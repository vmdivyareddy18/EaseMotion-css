# Alert Banner with Dismiss Animation

Four alert banners for info, success, warning, and error states. Each alert has a dismiss button that collapses the banner with a height and opacity transition. A "Reset All" button restores all dismissed alerts.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click ✕ on any alert to dismiss it. Click "Reset All" to restore them.

## Accessibility notes

Each alert uses semantic color coding with icon indicators. The dismiss button is a `<button>` element. Reduced motion disables the dismiss transition.
