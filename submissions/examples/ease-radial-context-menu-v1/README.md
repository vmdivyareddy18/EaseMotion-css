# Ease Radial Context Menu

## Overview

Ease Radial Context Menu is a floating circular action menu inspired by modern design systems. Menu items expand outward from a central action button with smooth radial animations using only HTML and CSS.

---

## Features

- ⭕ Circular expanding menu
- 🚀 Radial animation
- ✨ Staggered reveal
- 🎯 Hover interactions
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---
