# Kakapo

A pure CSS and HTML kakapo, the flightless night parrot of New Zealand, booming from a mossy log.

## What it shows

- Mottled green plumage built from stacked hard stop radial gradients
- An owl like facial disc textured with a faint repeating conic gradient
- Chest inflation on the whole bird, with sound rings expanding outward on staggered delays
- Ferns swaying behind the log
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
