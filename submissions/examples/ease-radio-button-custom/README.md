# Ease Radio Button Custom

A modern **Custom Radio Button** component built using **pure HTML and CSS**. It features animated selection effects, responsive design, and customizable styling using CSS custom properties.

## Features

- Pure HTML & CSS
- Custom styled radio buttons
- CSS-only selection animation using `@keyframes`
- Hover interactions
- CSS Custom Properties for easy customization
- Responsive design
- Keyboard accessible
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
ease-radio-button-custom/
├── demo.html
├── style.css
└── README.md
```

## Customization

Modify the CSS variables in the `:root` section to customize:

- Primary color
- Background color
- Border radius
- Shadow
- Transition speed

## Browser Support

Compatible with all modern browsers.
