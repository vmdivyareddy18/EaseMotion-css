# App Landing: Custom Stationery E-commerce (`ecommerce-custom-stationery-20985`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a high-end, personalised stationery and invitation brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A luxury stationery brand requires a highly refined, minimalist, and elegant aesthetic. This template achieves a sophisticated feel using a strict neutral color palette (monochrome with a single warm gold accent), thin borders, and premium typography (`Bodoni Moda` serif paired with `Inter` sans-serif).

The layout includes:
- A stark, high-contrast Announcement Bar.
- A sticky Navigation with clean, muted hover states.
- A split Hero section featuring an abstract CSS representation of stacked stationery (envelope, letterpress card, wax seal).
- A "Curated Collections" 3-column grid with custom CSS mockups of different paper goods.
- A Personalisation split-layout highlighting print techniques with CSS-simulated foil shimmer and letterpress effects.
- A dark-mode Process Timeline.
- A large, typography-focused Review section.
- A clean, grid-based Bespoke Wedding Inquiry Form.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use sharp, crisp, and elegant `cubic-bezier(0.16, 1, 0.3, 1)` timings (fast acceleration, very slow, smooth deceleration):
- `.ease-fade-up` / `.ease-slide-up` (Configured for crisp, sophisticated entrances).
- `.ease-reveal-right` / `.ease-reveal-left` (Used for the hero envelope and personalisation sections to smoothly unfold into view).
- `.ease-zoom-in` (Applied to the wax seal and review text for a subtle, elegant scale effect).
- `.ease-shimmer` / `.ease-press` (Custom continuous animations applied specifically to the abstract text in the personalisation section to simulate metallic foil and deep letterpress printing).
- `.ease-hover-scale` (Custom interaction class applied to the logo).
- `.ease-hover-lift` (Applied to cards and buttons to gently and sharply elevate on hover).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the elegant minimalist layout, the abstract CSS paper goods simulations, and the premium typography styling. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the crisp, refined entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
