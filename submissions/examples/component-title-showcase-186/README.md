# Component Title Showcase 186

A clean, premium, CSS-only title showcase component created for EaseMotion issue `#25715`.

## Files

- `demo.html` - browser-ready component preview
- `style.css` - responsive styling, glass effects, gradients, and CSS animation

## Features

- Pure HTML and CSS implementation
- Responsive title showcase layout
- Glassmorphism panel with animated decorative rings
- No JavaScript or external dependencies

## Usage

Open `demo.html` directly in a browser to preview the component.
