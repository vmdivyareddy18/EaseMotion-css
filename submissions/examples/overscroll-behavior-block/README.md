# overscroll-behavior-block

## Purpose
Demonstrates `overscroll-behavior: contain`, `none`, and `auto` for controlling scroll chaining in nested scroll containers.

## Features
- Nested scroll containers with different overscroll-behavior values
- Visual comparison of scroll chaining propagation
- Prevents background page scroll when inner containers reach boundaries
- Interactive explanation of each mode

## Usage
Open `demo.html`. Scroll each nested container to its boundary and observe whether the parent scrolls (chaining) or stops (contain/none).
