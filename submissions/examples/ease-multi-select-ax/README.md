# ease-multi-select – Multi-Select with Tags

A multi‑select component where selected items are shown as removable tags. Includes a search/filter input, a dropdown list of options, and a smooth tag collapse animation on removal.

## Features
- **Search/Filter**: Type to filter the dropdown options.
- **Tag Display**: Selected items appear as coloured chips with a remove button.
- **Tag Removal Animation**: Tags shrink and fade out when removed.
- **Dropdown**: Click to open, filter, and select items.
- **Minimal JavaScript**: Uses vanilla JS for interaction; no framework needed.

- **Layout:** `ease-container`, `ease-flex`, `ease-flex-wrap`, `ease-items-center`, `ease-justify-center`, `ease-min-h-screen`, `ease-py-16`, `ease-mx-auto`, `ease-gap-2`, `ease-mb-2`, `ease-mb-4`, `ease-mb-8`, `ease-mt-8`
- **Background:** `ease-bg-gray-50`
- **Typography:** `ease-text-3xl`, `ease-font-bold`, `ease-text-gray-500`, `ease-text-sm`, `ease-text-gray-400`
- **Components:** `ease-input`, `ease-card`, `ease-badge`
- **Animation:** `ease-fade-in`, `ease-delay-200`, `ease-delay-500`
- **Utilities:** `ease-hidden`, `ease-list-none`, `ease-w-full`, `ease-shadow-lg`

## How to Use
1. Copy the HTML, CSS, and JS from `demo.html` into your project.
2. Adjust the option list (`<ul id="optionsList">`) to your own data.
3. Ensure the path to `easemotion.css` is correct.
