GitHub Actions Auto‑Label Workflow

A workflow that automatically adds labels to new issues based on keywords in the title.

Keyword Rules

  • animation → animation
  • component → component
  • bug or fix → bug
  • docs → documentation

How to use

# Copy the workflow file to .github/workflows/
.github/workflows/auto-label.yml

Once placed in the correct directory, the workflow runs on every new issue and adds matching labels.

Built for EaseMotion CSS — reduce maintainer labeling burden.
