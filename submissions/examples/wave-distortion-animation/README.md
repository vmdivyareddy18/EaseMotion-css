# Wave Distortion Animation

A lightweight wave distortion animation utility built using pure CSS.

## Features

- Dynamic skew and scale distortion
- Wave-like oscillation effect
- CSS variable intensity control
- Infinite smooth animation
- Pure CSS implementation

## Usage

```html
<div class="distort-box">
  DISTORT
</div>
```

## Customization

```css
:root {
  --ease-distort-intensity: 18deg;
}
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable distortion effect
- Pure CSS visual animation
