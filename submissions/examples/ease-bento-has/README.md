# Staggered Bento Grid (`ease-bento-has`)

### 1. What does this do?
A bento grid layout displaying dashboard items with entering stagger animations and hover-scale offsets.

### 2. How is it used?
Configure a CSS grid and wrap card nodes with `.bento-card`:
```html
<div class="bento-grid">
  <div class="bento-card col-span-2">
    <h3>Title</h3>
    <p>Body</p>
  </div>
</div>
```

### 3. Why is it useful?
Provides an structured bento dashboard design with smooth page entrance delays that capture attention.
