# Team Members Grid Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive team members section.

---

## Components

- Responsive Grid
- Six Team Member Cards
- Avatar
- Name
- Position
- Bio
- Social Icons

---

## Folder Structure

```
submissions/examples/team-grid-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Resize the browser window.
3. Observe the responsive layout and EaseMotion animations.

---

## Features

- Responsive team section
- Modern card layout
- Social media icons
- Uses existing EaseMotion CSS animation classes
