# Shake Input Error

A form input component with a shake animation on validation errors, including animated error messages.

## Features

- Shake keyframe animation on input with .error class
- Red border and glow on error state
- Error message appears with fade-in animation
- Focus state with highlight border
- Clean dark theme form design

## Usage

Add the `.error` class to `.field-input` to trigger the shake animation. The `.field-error` element appears below the input with a fade-in transition.
