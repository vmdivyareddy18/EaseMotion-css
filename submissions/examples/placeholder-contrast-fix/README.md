# Placeholder Contrast Fix

A simple accessibility improvement for `.ease-input` placeholders.

## Problem
The placeholder text used reduced opacity, making it difficult to read on colored backgrounds and potentially failing accessibility contrast requirements.

## Fix
- Removed reduced opacity from placeholder text.
- Used a readable placeholder color.
- Improved visibility and accessibility.

## Files
- demo.html
- style.css

## Result
Placeholder text remains clear and readable across different background colors while maintaining a clean appearance.
