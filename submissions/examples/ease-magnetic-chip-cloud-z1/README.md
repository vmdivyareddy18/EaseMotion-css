# Ease Magnetic Chip Cloud

## Overview

A premium floating tag cloud featuring glassmorphism chips, magnetic hover interactions, and soft floating motion built entirely with HTML and CSS.

## Features

- Floating chip animation
- Magnetic hover effect
- Glassmorphism styling
- Responsive flex layout
- Lightweight component
- Pure HTML & CSS

## Folder Structure

```text
ease-magnetic-chip-cloud-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in a modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Works seamlessly across desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Demonstrates reusable floating micro-interactions, elegant hover animations, and a clean CSS architecture suitable for developer portfolios, dashboards, and tag interfaces.
