# Weight Loss Clinic Template
A supportive, encouraging, and science-backed template for medical weight loss clinics, nutrition-focused medical practices, and wellness-transformation centers.

- **Design**: Uses a "Revitalization-Energy" palette with revitalize-teal primaries, energy-orange growth accents, and light, airy layouts to convey positivity, medical rigor, and personal transformation.
- **Use Case**: Medical weight management clinics, nutrition therapy practices, and wellness centers.
