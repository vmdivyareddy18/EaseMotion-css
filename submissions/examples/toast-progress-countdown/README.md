# Toast Progress Countdown

A toast notification with a slim progress bar that counts down before dismissal.

## Files

- `demo.html` - toast status markup.
- `style.css` - toast entrance, countdown bar, and reduced-motion fallback.

## Highlights

- Uses `role="status"` for the toast message.
- Animates the progress bar with `scaleX`.
- Stops motion for reduced-motion users.
