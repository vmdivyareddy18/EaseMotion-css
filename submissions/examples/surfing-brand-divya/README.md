# Surfing Brand Demo

A responsive **Surfing Brand** landing page built using EaseMotion CSS.

## Features

- Hero section with ocean-inspired design
- Surfboards, Wetsuits & Accessories showcase
- Team Riders section
- Customer Reviews
- Newsletter for Surf Tips
- Responsive layout
- Hover effects
- EaseMotion CSS animations

## Files

```
demo.html
style.css
README.md
```

## EaseMotion CSS Classes Used

- ease-fade-up
- ease-slide-left
- ease-slide-right
- ease-zoom-in

- CSS3
- EaseMotion CSS
