# CSS-only Expanding Dock Navigation

A modern floating dock navigation built entirely with **HTML and CSS**.

## Features

- Pure HTML & CSS
- Glassmorphism design
- Smooth hover animations
- Responsive layout
- Icon scaling effect
- Labels appear on hover
- No JavaScript required

## Folder Structure

```
expanding-dock-navigation/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern browser.
2. Hover over the dock items.
3. Customize the icons, colors, or spacing as needed.

## Customization

You can easily modify:

- Background gradient
- Dock size
- Hover scale
- Animation duration
- Border radius
- Icons and labels

## Browser Support

Works in all modern browsers that support:

- CSS Flexbox
- CSS Transitions
- `backdrop-filter` (glass effect)
