# Ripple Wave Toggle Switch

A modern **Ripple Wave Toggle Switch** component built using pure HTML and CSS.

The component features a smooth sliding toggle combined with a ripple wave animation inspired by modern food delivery applications.

## Features

- Pure HTML & CSS
- Responsive
- Accessible
- No JavaScript
- Ripple Wave animation
- Modern food delivery inspired UI
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<label class="toggle">
  <input type="checkbox" />

  <span class="slider">
    <span class="thumb"></span>
  </span>
</label>
```

Include the provided `style.css` in your project.

## Accessibility

- Uses semantic checkbox input
- Keyboard accessible
- Focus-visible styles
- Responsive design

## Why EaseMotion CSS?

This component demonstrates how subtle motion can improve user interaction using a lightweight ripple animation while remaining reusable, responsive, and entirely CSS-based.
