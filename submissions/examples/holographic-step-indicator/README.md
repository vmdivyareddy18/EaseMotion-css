# Holographic Step Indicator

A pure CSS animated holographic step indicator component for EaseMotion CSS.

## Features

- Futuristic holographic glow effect
- Animated active steps
- Smooth transitions
- Responsive layout
- No JavaScript required


## Usage

Add the HTML structure and include `style.css`.

Example:

```html
<div class="step active">
    <span>1</span>
    <p>Account</p>
</div>
