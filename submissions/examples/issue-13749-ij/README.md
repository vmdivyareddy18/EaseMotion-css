# ease-burn-ember

**Closes #13749**

An ember glow effect with irregular box-shadow flicker timing and rising ember particles via pseudo-elements.

## Acceptance Criteria

- [ ] `box-shadow` flickers with warm orange/red tones
- [ ] Rising ember particles animate upward via `::before`/`::after`
- [ ] Toggle button ignites/extinguishes the effect
- [ ] Warm colour palette (orange, red, amber)

## Files

- `style.css` – Flicker and ember keyframes, box-shadow animation
- `demo.html` – Ember container with toggle button
- `README.md` – Issue documentation
