# Issue #14829: Advanced: Full Pest Control Landing Page

This folder contains a standalone demonstration of a Pest Control Landing Page as requested in Issue #14829.

## Files
- `demo.html`: The HTML structure for the landing page.
- `style.css`: The CSS styling emphasizing a clean, hygienic, and highly visible design.

## How it works
The demo provides a complete layout for a pest control service featuring:
- A striking color palette of high-visibility green against dark charcoal.
- A hero section emphasizing urgent service and reliability.
- A grid showing common pests handled (Termites, Rodents, Insects).
- A trust-building badge section (Licensed, Insured, Guaranteed).
- Dynamic CSS hover effects that maintain a clean, professional aesthetic.
