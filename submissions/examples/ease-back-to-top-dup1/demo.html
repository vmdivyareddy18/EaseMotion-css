EaseMotion CSS — Submission

Ease Back to Top

Scroll down to see three animated back-to-top button variants appear: a classic circle, a scroll-progress ring, and an expanding pill. All pure CSS + minimal JS for the scroll trigger.

Section 01

Smooth Scroll Behaviour

The button fades and springs into view after the user scrolls past 200 px, keeping the viewport clean at the top of the page.

Section 02

Zero Dependencies

No libraries, no frameworks. The scroll listener is fewer than 30 lines of vanilla JavaScript — the visual layer is entirely CSS.

Section 03

Three Visual Variants

Choose the circle button on the left, the progress ring in the centre, or the expanding pill on the right. Hover each one to see micro-interactions.

Section 04

Accessibility First

Every button carries aria-label, works on keyboard focus, and respects prefers-reduced-motion.
