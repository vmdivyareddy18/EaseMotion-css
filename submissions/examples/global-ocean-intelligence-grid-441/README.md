# Global Ocean Intelligence & Maritime Security Grid

A premium CSS-only dashboard concept for EaseMotion issue `#24401`.

## Files

- `demo.html` - standalone browser preview
- `style.css` - responsive dashboard styling, radar animation, cards, and hover states

## Features

- Naval fleet monitoring interface concept
- AUV coordination and submarine communications modules
- Offshore infrastructure and marine ecosystem cards
- Maritime security and shipping corridor visualization
- Pure HTML and CSS with no JavaScript

## Usage

Open `demo.html` directly in a browser to view the dashboard component.
