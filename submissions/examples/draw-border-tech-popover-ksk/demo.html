fn ease.animate()
ease.animate() — reference
function ease.animate(
  target: string | Element,
  keyframe: EaseKeyframe,
  options?: AnimateOptions
): Promise<void>

Applies an EaseMotion keyframe animation to the target element(s). Returns a promise that resolves when the animation completes.

paramtypedescription
target string CSS selector or Element reference
keyframe EaseKeyframe Named keyframe from EaseMotion library
options object? duration, delay, easing, fill
async v1.0+ stable
fn ease.sequence()
ease.sequence() — reference
function ease.sequence(
  steps: AnimateStep[],
  options?: SequenceOptions
): Promise<void>

Runs a series of animation steps in order. Each step waits for the previous to finish before starting. Supports stagger delay between items.

paramtypedescription
steps array Array of {target, keyframe, options}
options object? stagger, direction, repeat
async v1.2+ beta
fn ease.register()
ease.register() — reference
function ease.register(
  name: string,
  keyframes: Keyframe[],
  defaults?: AnimateOptions
): void

Registers a custom named keyframe animation into the EaseMotion library. Once registered, use the name as any built-in keyframe string.

paramtypedescription
name string Unique keyframe identifier
keyframes array Web Animations API keyframe array
defaults object? Default timing options for this keyframe
sync v1.0+ stable
