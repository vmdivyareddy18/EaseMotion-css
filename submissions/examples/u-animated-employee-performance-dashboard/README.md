# Animated Employee Performance Dashboard

A modern **Employee Performance Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example provides an interactive interface for monitoring employee performance, attendance, productivity, KPI progress, achievements, and activity using smooth CSS-only animations.

## Features

* 👨‍💼 Employee overview cards
* 📅 Attendance statistics
* ⚡ Productivity tracking
* ⭐ Monthly performance rating
* 📊 KPI progress bars
* 🏅 Skill badges
* 👥 Employee profile cards
* 🟢 Animated online/offline status indicators
* 📈 Performance timeline
* ✨ Smooth hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

All animations follow the EaseMotion CSS naming convention.

* `ease-fadeUp`
* `ease-progressGrow`
* `ease-pulse`
* `ease-rotateIn`
* `ease-slideIn`
* `ease-hoverLift`
* `ease-cardReveal`

## Customization

All colors, spacing, and theme settings are controlled through CSS variables.

Example:

```css
:root{
    --primary:#4f46e5;
    --success:#10b981;
    --warning:#f59e0b;
    --surface:#ffffff;
    --bg:#f5f7fb;
}
```
