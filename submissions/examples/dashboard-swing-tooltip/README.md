# Dashboard Swing-Hover Tooltip (Pure CSS)

An ultra-performant, responsive dashboard utility tooltip built with 100% hardware-accelerated motion strings. The drop-down transition features pendulum momentum swing physics.

## Features
- **Pure CSS Execution Engine:** Eliminates the need for JavaScript `Layout/Paint` step triggers by driving physics strictly on the GPU composite layer.
- **Top-Center Hanging Pivot:** Utilizes explicit `transform-origin` configurations to realistically match real-world gravity behaviors.
- **Native Parameter Tuning:** Offers easy adjustments directly through clean CSS customization variables (`:root`).
- **Full A11y Support:** Completely accessible out-of-the-box via explicit focus sequencing mapping (`:focus-visible` / `aria-describedby`).
