# ease-grid-layout-toggle

A layout toggle that switches between a multi-column grid view and a single-column list view with animated transitions. Items smoothly reposition with scale and opacity changes.

## Features

- Toggle between grid (3-column) and list (single-column) layouts
- Smooth transition on `grid-template-columns`
- Items animate with scale and opacity during layout switch
- Toggle button with grid/list icon indicators
- Content cards with color-coded image blocks and labels

## CSS Custom Property Control

- `--grid-toggle-duration`: Duration of the layout transition
- `--grid-item-bg`: Background color of each item card
- `--grid-item-shadow`: Box shadow applied to each item
- `--grid-radius`: Border radius for item cards
- `--grid-gap`: Gap between grid items
