# Ease Repeat Last Action

A CSS animation component that replays a smooth pulse animation whenever the user hovers over the element. It demonstrates a simple repeat-last-action interaction using CSS keyframes.

## Files

- `demo.html` – Demo page showcasing the animation.
- `style.css` – Contains the animation and styling.

## Usage

```html
<button class="ease-repeat-last-action">
    Hover Me
</button>
```

Hover over the button to replay the animation.
