Tabs Dark Mode Fix

Issue #9136 — tabs.css nav border uses --ease-color-neutral-200 with no dark override

Enable dark mode emulation in Chrome DevTools → More Tools → Rendering → set Emulate CSS media feature prefers-color-scheme to dark. The broken tabs show a bright light-grey separator. The fixed tabs use a dark border that matches the dark background.
