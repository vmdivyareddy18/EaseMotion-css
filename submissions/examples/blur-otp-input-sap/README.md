# Blur OTP Input Component

### What does this do?
Adds a pure-CSS multi-field secure verification input card styled with frosted-glass music player layouts that transitions an individual input block dynamically from heavily blurred states to clear, vibrant neon focus states upon interaction.

### How is it used?
```html
<div class="otp-container-sap">
  <div class="otp-fields-sap">
    <input type="text" maxlength="1" class="otp-field-sap" required />
    <input type="text" maxlength="1" class="otp-field-sap" required />
  </div>
</div>
```

### Why is it useful?
It replaces standard flat utility input elements with an ultra-smooth micro-interaction sequence using high-performance CSS backdrops and focus pseudoclasses without running any custom JavaScript framework routines.
