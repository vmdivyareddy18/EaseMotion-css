# CSS Timeline Component

## What does this do?
A vertical CSS timeline with alternating left/right cards, compact version, and icon-based step variants.

## How is it used?
```html
<div class="tl-timeline"><div class="tl-item tl-left"><div class="tl-dot tl-dot-blue"></div><div class="tl-card"><span class="tl-date">Date</span><h3>Title</h3><p>Content</p></div></div></div>
```

## Why is it useful?
Showcases chronological events cleanly without JavaScript. Alternating layouts maximize space. CSS-only responsive breakpoints. Ideal for roadmaps, changelogs, and history views.
