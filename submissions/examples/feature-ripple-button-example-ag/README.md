> Ripple Button (Warning State)
Overview

This submission showcases a warning-themed ripple button built using semantic HTML and pure CSS. The component is fully self-contained, requires no JavaScript or external libraries, and emphasizes accessibility, smooth interactions, and reusable styling.

> Technologies Used

HTML5 (Semantic elements)
CSS3
CSS Transitions
CSS Keyframe Animations
CSS Pseudo-elements (::before)
Media Query (prefers-reduced-motion) for accessibility

> Features 

Warning-themed orange gradient button
Smooth ripple animation on hover
Slight lift effect with shadow on hover
Press animation on click
Keyboard-accessible focus indicator
Responsive and lightweight design
Supports reduced-motion accessibility preferences
No JavaScript or external dependencies required

> Button Appearance

Rounded rectangular button
Orange-to-red warning gradient background
White bold text
Soft elevation shadow on hover
Expanding white ripple originating from the center
Smooth, modern micro-interactions
Clean and minimal UI suitable for reusable components

> Usage 

<button class="ripple-button-example-ag">
    Hover Me
    <span class="ripple-circle-ag"></span>
</button>

> Why is it useful?

This component enhances user interaction through subtle visual feedback while maintaining accessibility and performance. It demonstrates how modern CSS can create engaging UI effects without relying on JavaScript, making it suitable for buttons used in alerts, warnings, confirmations, and other interactive interfaces.
