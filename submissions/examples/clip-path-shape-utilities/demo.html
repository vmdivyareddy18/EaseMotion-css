Clip Path Shape Utilities

CSS clip-path utilities to create complex geometric shapes and clip outer margins. Perfect for decorative headers, custom avatars, and polygonal buttons.

What is clip-path?
The clip-path CSS property creates a clipping region that defines what part of an element should be visible. Anything outside the path is hidden from rendering, allowing you to crop rectangular blocks into custom circles, ellipses, or complex polygons.

Hover over the shapes below to trigger micro-scaling animations.

Geometric Clip Shapes

Circle Shape .cp-circle

Crops the element into a perfect circle with a radius of 50% from the center point.

Ellipse Shape .cp-ellipse

Crops the element into an oval with a 50% horizontal and 35% vertical radius.

Triangle Shape .cp-triangle

Crops the element using a 3-point polygon to form a symmetric upright triangle.

Diamond Shape .cp-diamond

Crops the element using a 4-point polygon to form a centered vertical diamond.

Hexagon Shape .cp-hexagon

Crops the element using a 6-point polygon to form a flat-topped hexagon shape.

Star Shape .cp-star

Crops the element using a 10-point polygon to form a classic 5-pointed star.
