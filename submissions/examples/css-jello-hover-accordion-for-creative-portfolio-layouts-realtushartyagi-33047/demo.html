Jello-Hover Accordion (Creative Portfolio)

Hover over the header above. The "Jello" animation applies a mathematically sequenced skewX and skewY oscillation to the icon, creating a gelatinous, highly organic wobble effect perfectly suited for creative portfolios.

The accordion opens via a pure CSS Grid grid-template-rows: 0fr -> 1fr transition, eliminating the need for heavy JavaScript height calculations and ensuring buttery-smooth 60fps expansions.

For users with system-level motion reduction enabled (prefers-reduced-motion), the Jello skew effect is cleanly bypassed, falling back to a standard subtle rotation.
