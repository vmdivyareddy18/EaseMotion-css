# Animated Image Gallery Component

## Description

A modern CSS-only image gallery with smooth zoom animations, elegant overlay captions, and responsive grid layouts.

---

## Features

- CSS-only implementation
- Smooth image zoom effect
- Animated overlay captions
- Responsive gallery grid
- Rounded image cards
- Hover shadow interactions
- Lightweight and customizable
- Reduced motion accessibility support

---

## Classes

- `.gallery` – Responsive image grid
- `.gallery-card` – Individual image container
- `.overlay` – Caption overlay
- `.gallery-section` – Main gallery wrapper

---

## Accessibility

Supports `prefers-reduced-motion` to reduce animations for users who prefer less motion.

---

## Use Cases

- Photography portfolios
- Product showcases
- Creative galleries
- Business websites
