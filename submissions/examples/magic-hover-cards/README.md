# ✨ Magic Hover Cards

A collection of stunning, pure CSS hover effects designed to elevate UI interactions. Perfect for portfolios, product showcases, or just adding some flair to your website.

## 🚀 Features

This project includes 4 unique, GPU-accelerated hover animations:

1. **💎 Shine Sweep** - A smooth light reflection that sweeps across the card.
2. **🌈 Glowing Border** - An animated, blurred gradient border that activates on hover.
3. **💡 Neon Glow** - A classic, vibrant neon box-shadow and icon glow.
4. **🎲 3D Tilt** - A realistic 3D perspective tilt with depth (using `translateZ`).
