# Animated Countdown Timer

A vibrant, modern, and fully responsive **Event Countdown Timer** perfect for product launches, event registrations, and marketing campaigns. This example demonstrates how to combine **EaseMotion CSS** for continuous pulsing animations, staggered entrance effects, and smooth hover transitions to create an engaging, urgency-building component.

## 🚀 Features

- **Continuous Number Pulsing**: Uses `ease-pulse` with staggered delays on each time unit to create a lively, attention-grabbing effect.
- **Staggered Entrance Animations**: Each time unit fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a premium, cascading load effect.
- **Glassmorphism Design**: Features frosted glass cards with backdrop blur and subtle borders for a modern aesthetic.
- **Interactive Hover States**: Time boxes lift on hover with enhanced shadows and accent-colored borders using custom CSS transitions.
- **Floating Decorative Elements**: Background decorations use `ease-float` and `ease-spin-slow` for continuous ambient motion.
- **Call-to-Action Button**: Uses `ease-hover-grow` for satisfying tactile feedback on the "Notify Me" button.
- **Fully Responsive**: The grid adapts gracefully from desktop to mobile, scaling down time boxes and adjusting spacing.
