# Animated Globe Rotation Orbit

## Overview

A reusable globe icon animation featuring a gentle rotation effect suitable for networking platforms, internet services, worldwide connectivity dashboards, and global communication interfaces.

## Features

- CSS-only animation
- Globe orbit rotation effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="globe-button">
  <span class="globe-icon">🌐</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
