# Volunteer Finder App Template
A mission-driven, inspiring, and community-focused landing page template for volunteer matching apps, social impact platforms, and nonprofit engagement tools.

- **Design**: Uses a "Community-Impact" palette with impact-emerald growth headers, heart-rose action accents, and clean, uplifting layouts designed to foster altruism and community connection.
- **Use Case**: Volunteer coordination apps, nonprofit resource platforms, and community service dashboards.
