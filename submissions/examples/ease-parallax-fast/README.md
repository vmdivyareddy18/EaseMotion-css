# Ease Parallax Fast

This demo implements a **fast parallax animation** where elements move at 150% of scroll speed, creating a reverse parallax effect.

## 🧠 How It Works
- JavaScript maps scroll position to vertical translation.
- The element moves faster than the scroll speed (`1.5x` multiplier).
- Ideal for foreground elements to create depth contrast.

## 🧩 Files
- `demo.html` — main demo structure.
- `style.css` — styling and layout.
- `script.js` — parallax logic.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Scroll down — the element moves faster than the page.
3. Adjust `speed` in `script.js` for different parallax intensities.

## 🧰 Technologies
- HTML5
- CSS3 (transforms)
- JavaScript (scroll event)
