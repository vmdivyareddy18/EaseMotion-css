# Animated Event Schedule Timeline

A responsive event schedule timeline showcasing sessions, workshops, networking breaks, and keynote talks.

## Features

- Vertical event timeline
- Time indicators
- Event cards with descriptions
- Hover animations
- Responsive design
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more events
- Change timeline colors
- Modify spacing and layout
- Adjust animation timing
