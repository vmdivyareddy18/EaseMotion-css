# Blob Morphing Animation

## What does this do?
Organic CSS shape-shifting blob animations using border-radius keyframe manipulation with floating motion, gradient fills, and cluster compositions.

## How is it used?
```html
<div class="bl-blob bl-5 bl-grad-1"></div>
```

## Why is it useful?
Creates fluid organic animations for modern UI backgrounds. Multiple blob variants, floating combinations, cluster layouts, noise texture overlays, and interactive hover. All CSS, no JavaScript.
