# ease-alarm-bell-shake

Animated alarm bell shake effect for notifications and alarm triggers.

## Features

- Rapid bell shake animation
- Triggered by class addition
- Uses rotation-based wiggle motion
- Pure HTML and CSS
- Suitable for alarms and notifications

## Usage

```html
<div class="alarm-bell triggered">🔔</div>
```

Remove or add the `triggered` class to control the animation.

## Animation Details

- Bell rotates left and right rapidly.
- Uses a shake/wiggle pattern.
- Transform origin placed at the top for realistic bell movement.
