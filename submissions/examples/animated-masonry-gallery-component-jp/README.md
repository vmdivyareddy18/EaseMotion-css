# Animated Masonry Gallery Component

## Overview

A CSS-only masonry image gallery featuring responsive columns, smooth zoom animations, overlay captions, and elegant hover transitions.

## Features

- Responsive masonry layout
- Smooth image zoom effect
- Animated overlay captions
- Fade-in entrance animation
- Mobile-friendly responsive design
- Lightweight CSS-only implementation

## Files

- demo.html – Displays the animated masonry gallery
- style.css – Contains gallery styles and animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in a modern browser to view the masonry gallery.

Hover over images to see the zoom and overlay animation effects.
