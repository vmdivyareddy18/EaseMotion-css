# Ease Quantum Loader Grid

## Overview

A futuristic loading animation featuring synchronized glass tiles that pulse in a quantum-inspired wave pattern using only HTML and CSS.

## Features

- Quantum wave animation
- Glassmorphism tiles
- Animated center highlight
- Responsive CSS Grid
- Lightweight component
- Pure HTML & CSS

## Folder Structure

```text

├── demo.html

```

## Installation

Open `demo.html` in any modern browser.

- Firefox
- Edge

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Showcases reusable loading animations with modern glass styling and staggered motion, making it ideal for dashboards, AI interfaces, and application loading states.
