# Minimalist Tech Tada-Click Modal

A modern **Pure CSS Tada-Click Modal** designed for **Minimalist Tech** interfaces. The component features a smooth Tada-Click animation, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Tada-Click animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Minimalist Tech inspired design
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:0.8s;
    --easing:cubic-bezier(.68,-0.55,.27,1.55);
    --scale:1.08;

    --bg:#0f172a;
    --surface:#1e293b;
    --primary:#38bdf8;
}
```

These variables allow you to customize:

- Animation duration
- Animation easing
- Animation scale
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
