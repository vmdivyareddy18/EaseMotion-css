# CSS Pricing Cards

## What does this do?
CSS pricing comparison cards with three-tier layout, featured badge, check/cross feature lists.

## How is it used?
```html
<div class="pc-card pc-feat"><div class="pc-badge">Popular</div><div class="pc-hdr"><h3>Plan</h3><div class="pc-pr"><span class="pc-amt">$29</span></div></div><ul><li class="pc-ok">Feature</li></ul></div>
```

## Why is it useful?
Showcases pricing tiers without JS. Featured card with accent border. Check/cross indicators. Hover elevation.
