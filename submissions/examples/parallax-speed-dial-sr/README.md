# Parallax Speed Dial

A modern **Parallax Speed Dial** component inspired by **Banking Dashboard** design patterns for the **EaseMotion CSS** library.

This component provides a beautiful floating action menu with smooth CSS-only animations, glassmorphism styling, staggered motion, and responsive behavior. It is built using HTML and CSS only—no JavaScript required.

---

## ✨ Features

- 💳 Banking-inspired modern UI
- 🚀 Pure HTML & CSS (No JavaScript)
- 🎨 Glassmorphism design
- 🌊 Parallax animated background
- 📱 Fully responsive
- ♿ Accessible with `aria-label`
- ⚡ Smooth staggered Speed Dial animation
- 🎯 Uses EaseMotion animation utility classes
- 🖱️ Hover effects and ripple interaction
- 🎭 Keyboard-friendly toggle

## 🚀 Demo

The component displays a modern banking dashboard card with a floating **Speed Dial**.

Available quick actions include:

- 💸 Transfer Money
- 💳 Payments
- 👛 Wallet
- 🏦 Loans
- 📞 Customer Support

The Speed Dial expands using a CSS checkbox toggle and staggered animations.
