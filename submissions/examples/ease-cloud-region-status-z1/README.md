# Ease Cloud Region Status

## Overview
A CSS-only component for monitoring cloud region health.

## Features
- Animated status indicators
- Responsive layout
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for cloud dashboards, DevOps platforms, and infrastructure monitoring tools.
