# CSS object-fit Gallery

1. **What does this do?** Side-by-side comparison of all five `object-fit` values — cover, contain, fill, scale-down, and none — applied to identical images in same-sized containers.

2. **How is it used?** Set `object-fit` on `<img>` or `<video>` elements that have explicit `width` and `height` (or are inside a sized container).

3. **Why is it useful?** Images in responsive layouts frequently stretch or crop unexpectedly. This demo makes each value's behavior visible, helping developers choose the right fit mode.
