# EaseMotion - Feature: Add Interactive Animated Button Variants

This feature expands the core library of EaseMotion CSS with 5 unique, motion-first interactive button modules designed to capture tactile feedback, increase visual click metrics, and deliver smooth transitions under hardware acceleration profiles.

## Animated Variants Included

* **Ripple Button:** Captures click positions smoothly expanding an absolute coordinate indicator layer outward.
* **Shimmer Hover Button:** Leverages skew transforms mapping a repeating reflective gradient overlay tracking left-to-right across focus states.
* **Magnetic Hover Button:** Integrates wrapper coordinates tracking interactive pointer positioning calculations to gently gravitate button bodies natively toward mouse vectors.
* **Neon Glow Button:** Combines standard borders with stacked emission box-shadow filtering to simulate realistic high-contrast lighting elements.
* **Liquid Fill Button:** Blends stacked border-radius transformations rotating on focus layers to rise upwards mimicking authentic fluid physics models.
