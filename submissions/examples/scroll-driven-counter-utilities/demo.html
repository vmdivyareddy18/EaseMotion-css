CSS Scroll-Driven Counters

Combine CSS Counters with Scroll-Driven Animations to create section numbering, progress indicators, and document milestones that update naturally as the user scrolls — no JavaScript required.

🔢 CSS Counters 📜 Scroll Animations 🪄 No JavaScript

💡 How it works: We initialize a counter on a container (e.g., the body). Then, on specific elements, we increment the counter and display its value using content: counter(...). By combining this with the modern animation-timeline: view(), we can animate the counter element as it enters the viewport during scrolling.

1. Section Numbering — .ease-counter

The primary utility from the issue snippet. It automatically numbers sections with a leading zero. As you scroll down this page, notice how each numbered section animates into view.

Introduction

CSS Counters are a powerful way to automatically number elements in a document. They let you adjust the appearance of content based on its location in the document.

Installation

You can use counters to number headings, lists, tables, or any other elements. They are initialized with counter-reset, incremented with counter-increment, and displayed using the counter() function.

Configuration

With Scroll-Driven Animations, we tie an animation to the scroll position of a scroll container rather than the clock. The view() timeline allows us to animate elements as they cross the viewport.

Deployment

Combining these two powerful CSS features allows for rich, interactive, and semantic document structures without relying on complex JavaScript observers.

🔧 CSS Reference

/* 1. Scope the counter (from the issue snippet) */
.ease-counter-scope {
  counter-reset: section;
}

/* 2. The standard counter utility (matches snippet exactly) */
.ease-counter::before {
  counter-increment: section;
  content: counter(section, decimal-leading-zero);
  font-size: 2rem;
  color: #6c63ff;
  margin-right: 1rem;
}

/* 3. Adding Scroll-Driven Animation */
.ease-counter {
  animation: slideInFade linear both;
  animation-timeline: view();
  animation-range: entry 10% cover 30%;
}

@keyframes slideInFade {
  from {
    opacity: 0;
    transform: translateY(30px) scale(0.95);
  }
  to {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
}
