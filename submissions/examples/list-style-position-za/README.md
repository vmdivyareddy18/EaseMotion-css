# List Style Position Inside vs Outside

A CSS demo comparing list-style-position: inside and outside, showing how bullet markers align differently with multi-line list items.

## Files

- `demo.html` - Two lists comparing inside vs outside positioning
- `style.css` - list-style-position property, serif typography

## Usage

Open `demo.html` to see the alignment difference between inside and outside list markers.
