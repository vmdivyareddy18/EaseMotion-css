# Ripple Button

## What does this do?

Creates a premium call-to-action button featuring an expanding ripple animation, gradient background, and smooth hover interactions.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<button class="ripple-button">
    <span>Get Started</span>
</button>
```

---

## Why is it useful?

Ripple animations provide immediate visual feedback and improve perceived interactivity. This standalone component demonstrates how a ripple effect can be achieved using only HTML and CSS while remaining lightweight and framework-independent.
