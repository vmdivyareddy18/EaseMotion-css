# Traffic Light

A traffic light component with red/yellow/green glow/pulse animations, pedestrian walk/don't-walk symbols, and crosswalk stripe animation.

**Directory:** `ease-traffic-light-ij`

## Features
- Red/Yellow/Green lights with glow pulse effect
- Pedestrian walk/don't-walk symbol toggle
- Crosswalk stripe bounce animation
- Auto-cycle mode with countdown timer
- Manual state selection buttons

## CSS Custom Properties
| Property | Default | Description |
|---|---|---|
| `--trl-duration` | `0.4s` | Transition duration |
| `--trl-red-color` | `#ef4444` | Red light color |
| `--trl-yellow-color` | `#eab308` | Yellow light color |
| `--trl-green-color` | `#22c55e` | Green light color |
| `--trl-box-bg` | `#1e293b` | Traffic light box background |
| `--trl-radius` | `12px` | Box border radius |

## Usage
Open `demo.html` in a browser. Click "Auto Cycle" to start the automatic light sequence. Click individual color buttons for manual control.
