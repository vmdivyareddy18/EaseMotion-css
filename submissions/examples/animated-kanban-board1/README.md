# Animated Kanban Board

A clean, modern, and fully responsive **Kanban Board Component** perfect for project management and task tracking applications. This example demonstrates how to combine **EaseMotion CSS** for staggered card entrances, smooth hover effects, and interactive column animations to create an engaging task management interface.

## 🚀 Features

- **Staggered Card Entrances**: Each task card fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a cascading load effect.
- **Interactive Hover States**: Task cards lift on hover with enhanced shadows and accent borders using `ease-hover-grow`.
- **Priority Indicators**: Color-coded badges (High/Medium/Low) for quick visual identification of task importance.
- **Task Metadata**: Each card displays assignee, due date, and task ID for comprehensive information.
- **Completed Task Styling**: Done tasks feature reduced opacity and strikethrough text for clear visual distinction.
- **Horizontal Scrolling**: Board scrolls horizontally on smaller screens with custom scrollbar styling.
- **Column Task Counts**: Each column header displays the number of tasks it contains.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted column widths.
