# OceanVoyager – Full Cruise Line Landing Page

## 1. What does this do?
This is a complete cruise line landing page for OceanVoyager, featuring a hero section with ship/ocean imagery, a popular itineraries grid with destinations and pricing, ship amenities showcase, cabin types and pricing, guest reviews/testimonials, special offers section, and a booking CTA form. All motion and animation effects are powered exclusively by EaseMotion CSS framework classes.

## 2. How is it used?
Open `demo.html` in any modern browser. The page is fully responsive across mobile, tablet, and desktop. The booking form and interactive elements are static UI demonstrations. Users can browse itineraries, cabin types, amenities, and special offers as a showcase template.

## 3. Why is it useful?
This is a production-ready template for cruise line, travel, or hospitality websites. It demonstrates how to build a visually rich, multi-section landing page using only EaseMotion animation classes, ensuring consistent motion design across the entire page while keeping custom CSS limited to layout, colors, and typography.
