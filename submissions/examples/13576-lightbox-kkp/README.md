# Animated Lightbox

CSS-only lightbox/modal viewer demo for EaseMotion. The gallery uses anchor targets to open a large image preview with fade and scale motion.

## Files

- `demo.html` - gallery and target-based lightbox markup
- `style.css` - thumbnail layout, overlay, and modal animation

## Usage

Open `demo.html`, then click any thumbnail to view the animated lightbox.
