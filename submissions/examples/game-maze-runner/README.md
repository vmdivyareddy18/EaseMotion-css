# 🧩 Maze Runner

A classic Maze Runner game built with HTML, CSS, and JavaScript. Navigate through randomly generated mazes from start to finish while tracking your time and moves!

## 🎮 How to Play

1. Click **"Start"** or press **Space** to generate a maze
2. **Navigate** from the green start (▶) to the red finish (🏁)
3. Use **Arrow Keys** or **click** on cells to move
4. **Track** your time and moves
5. **Beat** your best time!

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `↑` `↓` `←` `→` | Move player |
| `Space` | Start new maze |
| `S` | Show solution path |
| `R` | Reset game |

## ✨ Features

- 🧩 **Random Maze Generation** – Unique maze each time (DFS algorithm)
- 📐 **Multiple Sizes** – Choose from 5×5 to 31×31
- ⏱️ **Timer** – Tracks your completion time
- 👣 **Move Counter** – Track your steps
- 🏆 **Best Time** – Saved in browser local storage
- ✨ **Solve Feature** – Show the solution path
- 🖱️ **Click or Arrow Keys** – Two ways to move
- 📱 **Responsive Design** – Works on desktop and mobile
