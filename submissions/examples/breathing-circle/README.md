# Breathing Circle — `ease-breathe`

A meditative breathing guide animation that slowly expands and contracts, popular in wellness and focus apps.

## Usage

```html
<div class="ease-breathe circle">Breathe</div>
```

```css
@import 'style.css';
```

## Acceptance Criteria

- ✅ scale(1) → scale(1.3) → scale(1)
- ✅ Slow timing: 4s per cycle
- ✅ ease-in-out on both phases
- ✅ Infinite loop
- ✅ Works with text content inside

## Preview

Open `demo.html` in a browser to see the animation in action.
