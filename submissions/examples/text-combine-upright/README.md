# text-combine-upright

## Overview

Demonstrates the `text-combine-upright` CSS property for combining multiple characters into the width of a single character in vertical writing mode. Shows `none`, `all`, `digits 2`, `digits 3`, and `digits 4` variants using Japanese era years and other numeric contexts.

## Usage

Open `demo.html` in a browser that supports `text-combine-upright` (Chrome, Edge, Safari). Five sections compare how digits of varying lengths appear in vertical text with and without combination. The Japanese year section shows practical use for era notation.

## Features

- Five combination modes: none, all, digits 2, digits 3, digits 4
- Japanese era year examples (Reiwa, Heisei, Showa, Taisho, Meiji)
- Temperature, version numbers, and page ranges
- Vertical writing mode with `writing-mode: vertical-rl`
- `prefers-reduced-motion` support
- Dark theme with pink accent and blue section headers
