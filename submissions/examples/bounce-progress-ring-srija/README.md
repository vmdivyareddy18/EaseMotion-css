# Bounce Progress Ring

A bounce animated progress ring component for EaseMotion CSS inspired by food delivery applications.

## Features

- Pure CSS implementation
- No JavaScript required
- Bounce animation
- Circular progress indicator
- Responsive design
- Accessible markup using ARIA

## Usage

Open `demo.html` in a browser to view the component.

## Animation

The progress ring gently bounces to draw attention while displaying the current progress percentage.
