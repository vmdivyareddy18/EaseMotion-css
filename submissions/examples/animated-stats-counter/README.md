# Animated Stats Counter Dashboard Component

An interactive dashboard component for **EaseMotion CSS** featuring high-performance stat metric cards. The numerical counters dynamically tick upwards from zero the exact moment they enter the user's viewport.

## 🚀 Features
- **Scroll-Triggered Interactivity:** Utilizes an efficient `Intersection Observer` engine to run numerical counting only when visible on screen.
- **Micro-interactions:** Includes smooth 3D-esque hover states and entry fade-in/up transitions.
- **Dynamic Trend Scaling:** Contextual badge states (`--up` and `--down`) matching professional enterprise SaaS analytical dashboards.
