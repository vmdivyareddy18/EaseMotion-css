# Spring Barcode

A responsive **Spring Barcode** component built with pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Nature-inspired design
- Responsive layout
- Accessible structure
- Animated spring barcode effect
- Uses EaseMotion CSS variables and `ease-*` animation class

## Folder Structure

```text
spring-barcode-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Technologies

- HTML5
- CSS3

## Animation

- `ease-fade-in`
- `spring` keyframe animation
- Responsive and lightweight
