# E-commerce: Christmas Decorations (`e-commerce-christmas-decorations-20991`)

This proposal provides a complete, copy-paste ready landing page template designed for a seasonal e-commerce brand (Christmas/Holiday Decor), built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A seasonal e-commerce site requires a UI that evokes warmth, elegance, and festivity. This template achieves a "Classic Christmas" aesthetic using a rich palette of Evergreen (`#1A362D`), Crimson (`#9B2226`), and Metallic Gold (`#D4AF37`) against Warm Cream (`#FDFBF7`) backgrounds. It pairs elegant typography (`Cinzel` for headings, `Lora` for body) to feel premium and timeless.

The layout includes:
- A festive promo top bar and elegant sticky navigation.
- A Hero section featuring abstract CSS snowfall and pulsing warm glow effects.
- A "Decor by Theme" grid featuring pure CSS abstract illustrations (bauble, star, tree) with elegant hover lifts.
- A highlighted "Advent Calendar" product section featuring an interactive, pure CSS 12-day grid mockup.
- A "Curated Gift Sets" masonry-style grid featuring abstract CSS gift boxes.
- A high-contrast personalization banner.
- An elegant, minimalist review carousel layout.
- A crimson Newsletter signup section with a golden glow effect.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use gentle, magical `cubic-bezier(0.16, 1, 0.3, 1)` and `cubic-bezier(0.25, 0.46, 0.45, 0.94)` timings tailored for a premium brand:
- `.ease-slide-up-stagger` (Configured for a slow, elegant vertical entrance).
- `.ease-drift-slow` / `.ease-drift-fast` (Custom continuous CSS animations simulating gentle snowfall in the hero background).
- `.ease-flicker` (A slow, pulsing opacity animation applied to background gradients to simulate warm firelight/glow).
- `.ease-hover-gold` / `.ease-hover-gold-fill` (Elegant text and background color transitions).
- `.ease-hover-scale-glow` (Applied to primary buttons to add a rich crimson shadow on hover).
- `.ease-scale-x` (Used for elegant horizontal line drawing effects on section headers).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the rich holiday color palette to the elegant serif typography and abstract CSS illustrations (baubles, trees, gift boxes). The standard `ease-*` interaction classes are attached directly to HTML elements to provide gentle, magical animations. No JavaScript is required for this template.
