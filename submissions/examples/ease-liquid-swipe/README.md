# Liquid Swipe – Liquid-like transition between elements

> A production-ready liquid swipe component that creates smooth, organic liquid-like transitions between slides with customizable colors, speed, wave intensity, and auto-play. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Liquid-like transitions** – Smooth, organic wave effects
- ✅ **Customizable colors** – Primary and background colors
- ✅ **Adjustable speed** – Control transition speed
- ✅ **Wave intensity** – Control wave amplitude
- ✅ **Auto-play** – Automatic slide rotation
- ✅ **Keyboard navigation** – Arrow keys to navigate
- ✅ **Toggle wave** – Enable/disable wave effect
- ✅ **4 color variants** – Ocean, Sunset, Forest, Gold
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
