# Playlist Item Active

1. What does this do? A playlist where the currently playing item gets a smooth background highlight transition and an animated equalizer bars icon (building on the volume bars animation pattern).

2. How is it used? Add `.playlist-item` elements with a `.pl-eq` container of `.eq-bar-inner` bars. Toggle `.active` class to trigger the background transition and the equalizer bounce animation.

3. Why is it useful? It provides a clear visual indicator of the currently playing track with a subtle animated equalizer icon, making the playback state immediately obvious in playlist UIs.
