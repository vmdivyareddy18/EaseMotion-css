# Floating Profile Cards

## Overview

A set of floating team profile cards designed for dark dashboards, featuring gradient borders, glowing avatars, and subtle hover motion.

## Features

- Developer, Designer, Product Manager, and Freelancer cards
- Circular avatar placeholders
- Floating hover animation and soft glow
- Gradient accent borders for each variant
- Pure HTML and CSS with responsive layout

## Usage Example

```html
<link rel="stylesheet" href="style.css">
<section class="card-grid">
  <article class="profile-card profile-dev">
    <div class="card-ring"></div>
    <div class="avatar"></div>
    <div class="profile-body">
      <p class="profile-role">Developer</p>
      <h2>Alex Reed</h2>
      <p class="profile-text">Building performant web experiences with animation and UI polish.</p>
      <div class="social-row">
        <span></span><span></span><span></span>
      </div>
    </div>
  </article>
</section>
```

## Why it fits EaseMotion CSS

This component is fully self-contained, responsive, and motion-forward while staying pure CSS. It delivers modern visual polish, flexible card variants, and hover interaction without any external dependencies, matching EaseMotion CSS submission goals.
