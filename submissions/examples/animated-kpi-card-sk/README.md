# animated-kpi-card-sk

A KPI dashboard metric card component with CSS-only icon (emoji circle), large numerical value, label, and trend arrow (▲/▼) with green/red coloring.

## Interactions
- **Hover**: Card lifts upward with a smooth transform, number value pulses briefly with a scale animation, and trend arrow bounces.

## Features
- Pure CSS, no JavaScript
- Dark theme
- Responsive grid layout via `auto-fit`
- Respects `prefers-reduced-motion`
