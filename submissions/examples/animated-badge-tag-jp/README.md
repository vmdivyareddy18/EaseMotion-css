# Animated Badge and Tag Component

## Description

A CSS-only collection of animated badges and tags with hover effects, pulse animations, gradient styles, and multiple color variants.

---

## Features

- Smooth hover lift animations
- Animated pulse notification badge
- Gradient and colored badge variants
- Modern pill-shaped tags
- Responsive flexible layout
- Lightweight CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.badge` – Base badge component
- `.tag` – Outlined tag style
- `.success` – Green status badge
- `.warning` – Yellow warning badge
- `.danger` – Red alert badge
- `.gradient` – Colorful gradient badge
- `.pulse` – Animated notification badge

---

## Accessibility

Supports `prefers-reduced-motion` to disable animations for users who prefer less motion.

---

## Use Cases

- Notification indicators
- User status labels
- Categories and topics
- Skills and technologies
- Product labels
