# 3D Flip Card on Hover

A card that flips 180 degrees on hover using CSS 3D transforms and backface-visibility. Front shows an icon and title; back shows description and action.

## Files

- `demo.html` - Two flip cards in a grid
- `style.css` - Perspective, preserve-3d, backface-visibility, rotateY

## Usage

Open `demo.html` and hover over the cards to see them flip.
