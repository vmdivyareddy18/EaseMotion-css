How to Use

Add the markup structure below to your project, link `style.css`, and initialize the tab coordinates. The CSS filter uses blur and contrast inside the background layer to melt the shapes together while keeping buttons readable on top.

HTML Markup Structure

<div class="ease-liquid-tabs">
  <!-- Background Layer -->
  <div class="ease-liquid-bg-layer" id="bg-layer">
    <div class="ease-active-bubble" id="active-bubble"></div>
  </div>
  
  <!-- Foreground Layer -->
  <div class="ease-liquid-fg-layer">
    <button class="ease-liquid-tab ease-active" data-target="dashboard">Dashboard</button>
    <button class="ease-liquid-tab" data-target="analytics">Analytics</button>
    <button class="ease-liquid-tab" data-target="settings">Settings</button>
  </div>
</div>
