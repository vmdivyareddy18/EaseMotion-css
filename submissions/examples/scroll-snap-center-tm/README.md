# Scroll Snap Align Center Utility

### What does this do?
Adds a utility mechanism to align scroll items to the center of a scroll container, ideal for building carousels.

### How is it used?
Apply the alignment class to child elements within a scroll-snapped flex or grid track wrapper.

### Why is it useful?
It provides smooth, focused item presentation for galleries and media carousels, aligning closely with EaseMotion's smooth user experience philosophy.
