# scroll-start-target

## Purpose
Demonstrates the `scroll-start-target` CSS property for controlling the initial scroll position of a scroll container, with auto-scrolling to a target element on load.

## Features
- Smooth initial scroll behavior to a designated target element
- CSS `scroll-start-target` property on child elements
- `scroll-behavior: smooth` for graceful entry
- Visual highlight on the targeted section

## Usage
Open `demo.html` in a Chromium-based browser. The page auto-scrolls to "Section 3" on load using the `scroll-start-target` property applied to that element.
