# Incident Escalation Path Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive incident escalation workflow component that visualizes each escalation stage using timeline-style cards. Suitable for incident management dashboards, DevOps platforms, ITSM tools, and operations centers.

## Features

- Escalation path cards
- Timeline-style workflow
- Priority-based color indicators
- Escalation time badges
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="step">

    <div class="number">1</div>

    <div class="content">
        ...
    </div>

    <span class="badge">
        15 min
    </span>

</div>
```

## Main Classes

```css
.card
.timeline
.step
.number
.content
.badge
.critical
.major
.urgent
.resolved
```
