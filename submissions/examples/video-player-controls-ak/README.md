# Custom Video Player Controls

## What does this do?
A custom HTML5 video player with styled play/pause, progress bar with seek, and volume slider controls.

## How is it used?
Open demo.html in a browser. Use the custom controls to play/pause, seek through the video, and adjust volume.

## Why is this useful?
Demonstrates the HTML5 Video API with custom CSS-styled controls, including progress bar syncing and volume control.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
