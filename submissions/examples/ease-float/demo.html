Composability — Mix & Match

PreviewClassesBehaviour
🚀 .ease-float .ease-float-fast .ease-float-high Fast high float (1s, 20px)
🍃 .ease-float .ease-float-slow .ease-float-subtle Slow subtle float (4s, 4px)
📦 .ease-float .ease-float-hover .ease-float-high High float on hover only
⭐ .ease-float .ease-float-slow .ease-delay-500 Starts after 500ms delay
🎯 .ease-float .ease-float-once .ease-float-high One-shot high float
Infinite by default: Unlike other ease-* utilities that play once, .ease-float loops infinitely — floating is a continuous ambient animation. Use .ease-float-once for a single play.

display: inline-block is set automatically so transform works on inline elements like <span> and <a>.

prefers-reduced-motion: All float animations are disabled for users who have enabled "Reduce Motion" in OS accessibility settings.
