Wobble Code Block Variant

A startup-inspired code block UI with wobble accent animation, no JavaScript, keyboard focus visibility, and reduced-motion support.

TypeScript

type MotionPreset = {
  name: "wobble-code-block";
  duration: 420;
  easing: "cubic-bezier(.2,.8,.2,1)";
};

export const preset: MotionPreset = {
  name: "wobble-code-block",
  duration: 420,
  easing: "cubic-bezier(.2,.8,.2,1)"
};

CSS

.wobble-code-block {
  position: relative;
  border-radius: 16px;
  animation: wobble-accent 7s linear infinite;
}

@media (prefers-reduced-motion: reduce) {
  .wobble-code-block {
    animation: none;
  }
}

Usage Pattern

Apply wobble-on-hover to any code wrapper and keep decorative motion on pseudo-elements so content remains crisp and readable.

Responsive

Stacks cards on narrow screens and keeps touch targets above 44px.

Accessible

Focus-visible rings and skip link included for keyboard navigation.

No JS

Pure CSS visual behavior. Buttons are non-functional demo controls.

EaseMotion ready

Demo includes ease utility classes such as ease-fade-in and ease-slide-up.

Readability first

Code text remains stable while accent layers perform subtle motion around it.

Gradient frame

Conic and linear gradients create contrast-safe visual structure.

Wobble rhythm

Animation cadence is intentionally slow to avoid attention fatigue.

Reduced motion

All non-essential transitions and keyframes are neutralized when requested.

Composability

Utility classes allow this component to blend into docs, blogs, and dashboards.
