# CSS Elastic Slide Animation (Sci-Fi Console)

## Overview
A pure CSS implementation of an elastic slide animation effect designed specifically for Sci-Fi Console layouts.
It uses a `cubic-bezier` transition to create a bouncy, elastic motion when the user hovers over the element.

## Usage
Simply load `style.css` and view `demo.html` for the markup structure.
