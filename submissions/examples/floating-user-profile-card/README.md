# Animation: Floating User Profile Card

A premium profile card featuring glassmorphism styling, animated gradient borders, follower statistics, online status indicators, and interactive action buttons.

## Features

- Pure HTML & CSS
- Glassmorphism card design
- Animated gradient border
- Online status indicator
- User statistics section
- Interactive action buttons
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="profile-card">

    <div class="avatar">
        A
        <span class="status"></span>
    </div>

</div>
```

## Why is it useful?

Profile cards are commonly used in social platforms, team pages, portfolios, developer communities, and SaaS products. This component provides a modern way to showcase user information while maintaining a polished and interactive appearance.
