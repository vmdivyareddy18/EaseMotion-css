# Ease Loading Ring Spinner — Rotating Segments

> **PR:** #36780  
> **Component:** `<loading-ring-spinner>`

A spinning ring composed of colored segments for loading states.

## CSS Custom Properties

| Property          | Default   | Description          |
|-------------------|-----------|----------------------|
| `--ring-color`    | `#8b5cf6` | Ring segment color   |
| `--ring-size`     | `48px`    | Overall ring size    |
| `--segment-count` | `8`       | Number of segments   |
| `--spin-duration` | `1s`      | Full rotation time   |

## Usage

```html
<loading-ring-spinner></loading-ring-spinner>
```

## Features

- Configurable segment count
- Smooth continuous rotation
- Dark theme by default
