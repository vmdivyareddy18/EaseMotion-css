# Ease Chat Bubble Pop Component

A beautiful, micro-interactive CSS-animated Chat Bubble component designed for smooth UI transitions. It implements a spring-loaded scaling entrance pop animation using hardware-accelerated CSS properties.

## Features
- **Smooth `@keyframes` Animation:** Uses a custom `cubic-bezier` timing curve to create an organic, bouncy entry scaling effect.
- **Interactive Trigger:** Type and submit a message through the chat bar to dynamically trigger new popping chat bubbles. Hovering over bubbles gives a gentle hover feedback loop.
- **Customizable:** Fully driven by CSS Global Custom Properties variables (`--pop-duration`, `--bubble-sent-bg`, etc.).
- **Responsive Web Design:** Completely accessible and scales beautifully across mobile and desktop Viewports.
