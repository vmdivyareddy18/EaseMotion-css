# Morphing Nav – Navigation changes shape on scroll

> A production-ready morphing navigation component that elegantly transforms shape on scroll, from a rounded pill to a sleek compact header with smooth animations. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Morphing shape** – Smoothly transforms from rounded to compact
- ✅ **Scroll detection** – Triggers morph based on scroll position
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Mobile menu** – Hamburger with slide-in animation
- ✅ **Accessibility** – Keyboard navigable with ARIA labels
- ✅ **Customizable** – Colors, sizes, morph offset
- ✅ **Performance** – Optimized for 60fps scrolling
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
