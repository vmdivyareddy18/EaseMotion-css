# CSS Job Listing Card Placeholder

This submission details the `.ease-card-job-listing` component, optimized for job board listing grids, hiring panels, and career pages.

---

## Acceptance Criteria Handled

- **Company Logo:** A placeholder logo containing a stylized gradient background with centered text.
- **Featured Pulse Badge:** A green category badge with a small circular icon pulsing continuously via CSS keyframes.
- **Job Details:** Role title, company name, location tags, and experience attributes.
- **Salary Indicator:** Displayed clearly alongside an apply call-to-action button.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Verify the featured badge contains a green pulse indicator at the top-right of the card.
3. Hover over the card. Verify:
   - The card raises vertically by `4px` with a purple glow outline.
4. Verify location tag layout and apply CTA button align correctly.
