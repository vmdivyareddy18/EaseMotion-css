# Cyberpunk Button

## What does this do?

Creates a futuristic cyberpunk-inspired call-to-action button featuring glowing neon borders, animated scan effects, and interactive hover animations.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<button class="cyber-button">
    <span>Launch Mission</span>
</button>
```

---

## Why is it useful?

Cyberpunk-themed buttons are popular in gaming interfaces, developer portfolios, landing pages, dashboards, and futuristic UI concepts. This standalone component demonstrates advanced hover effects and modern visual styling while remaining lightweight and framework-independent
