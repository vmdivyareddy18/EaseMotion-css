# Unicode Bidi Isolate

unicode-bidi: isolate — Isolates RTL text fragments in LTR context

## Features
- Uses `--ease-color-surface`
- Uses `--ease-color-primary`
- Uses `--ease-radius-md`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Unicode Bidi Isolate</span>
</div>
```

## Why is it useful?

This submission demonstrates the unicode bidi isolate CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
