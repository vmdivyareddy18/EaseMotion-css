# Smooth Scrollbar Styling & Hover Mixins

An overflow text component demonstrating scrollbars styled with webkit pseudo elements.

---

## What is Included

1. **`demo.html`**: A scrollable panel container holding overflow paragraphs, descriptions, and headers.
2. **`style.css`**: Webkit scrollbar width presets, track roundings, thumb clips, colors, and hover transitions.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the custom scrollbar thumb on the right of the scroll card.
3. Scroll the content card to verify scrollability.
4. Hover over the scrollbar thumb — confirm that it changes color to bright cyan.
