# Staggered Entrance Pricing Grids

## Description
A beautifully designed pricing card system featuring staggered layout entrance animations and interactive scale/depth hover animations.

## Usage
Simply structure your pricing cards under a `.pricing-grid` container:
```html
<div class="pricing-grid">
  <div class="pricing-card">
    <div class="pricing-header">
      <h3>Starter</h3>
    </div>
    ...
  </div>
</div>
```

## Why is it useful?
Provides high-fidelity, polished onboarding page entry visual states. Built utilizing CSS animation delays for seamless, lightweight, and hardware-accelerated animations. Supports reduced motion preference compliance.
