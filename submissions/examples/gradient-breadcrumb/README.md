# Gradient Breadcrumb

A neon-inspired animated breadcrumb navigation component built using pure HTML and CSS.

## Features

- Neon gradient text animation
- Smooth entrance animation
- Responsive design
- Pure CSS implementation
- No JavaScript dependency
- Lightweight and reusable


## Usage

Add the breadcrumb structure:

```html
<nav class="breadcrumb">

  <a href="#">Home</a>

  <span>/</span>

  <a class="active" href="#">
    Current Page
  </a>

</nav>
