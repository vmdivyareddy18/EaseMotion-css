# Progressive Disclosure in Homepage Button Section

An implementation of a progressive disclosure mechanic for lists of buttons. It keeps layouts clean and distraction-free by showing an initial subset of choices and revealing more upon interaction.

## Features
- Initially reveals 3 items.
- Displays hidden items when clicking "Load More".
- Switches to "View Less" to collapse items.
