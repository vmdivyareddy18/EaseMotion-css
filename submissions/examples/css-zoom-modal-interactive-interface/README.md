# CSS Zoom-In Modal – Interactive Interface Layout

## Overview

A responsive Interactive Interface modal component built entirely with HTML and CSS. The component features a smooth zoom-in animation, modern dashboard cards, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-In modal animation
- Interactive Interface dashboard
- Responsive card layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--background`
- `--surface`
- `--text`
- `--muted`
- `--duration`
- `--easing`
- `--zoom-scale`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Open Workspace** to display the animated Zoom-In modal. Colors, animation timing, spacing, shadows, and scaling can all be customized using the CSS variables defined in `style.css`.
