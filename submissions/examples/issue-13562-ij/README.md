# ease-status-badge

Animated status indicator badge. Online (gentle ping), Away (slow pulse), Busy (fast pulse, red), Offline (static gray). Each status has a distinct animation personality.

## Usage

Include `style.css` and use `.ease-status-badge` with `.dot` and a status class (`.status-online`, `.status-away`, `.status-busy`, `.status-offline`).

## Demo

Open `demo.html` in a browser.
