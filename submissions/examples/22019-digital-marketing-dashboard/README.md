# Animated Digital Marketing Dashboard

> **Issue:** [#22019](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/22019)

## Overview

A self-contained animated digital marketing analytics dashboard for campaign monitoring. Demonstrates EaseMotion CSS entrance animations across stat cards, campaign performance bars, funnel steps, activity feed, and notification panels.

## EaseMotion CSS Classes Showcased

| Class | Applied To |
|---|---|
| `ease-fade-in-left` | Sidebar navigation, notification cards |
| `ease-fade-in-down` | Top navigation bar |
| `ease-zoom-in` | Stat overview cards (staggered), quick action buttons |
| `ease-fade-in-up` | Widget panels (staggered delays) |
| `ease-fade-in-right` | Funnel steps, marketing activity items |
| `ease-fade-in` | Progress/campaign performance bars |
| `ease-pulse` | Live indicator dot |

## Sections

- **Overview Cards:** Campaigns, Leads, Conversions, ROI
- **Campaign Performance Widget:** 4 campaign bars with gradient fills
- **Traffic Sources Widget:** Organic, Paid Social, Email, Direct, Referral
- **Conversion Funnel Widget:** Impressions → Clicks → Leads → Conversions
- **Marketing Activity:** Timestamped activity feed with category icons
- **Notifications Panel:** 3 alert notifications with severity accents
- **Quick Actions:** Launch Campaign, Export Analytics, Generate Report

## How to Test

1. Open `demo.html` directly in a browser (no build step required).
2. Observe staggered entrance animations on load for all cards and widgets.
3. Hover over stat cards — verify the lift and glow shadow effect.
4. Check the sidebar, topbar, and layout at ≤ 900px viewport (responsive).
5. Verify the live pulsing indicator and animated progress bars render correctly.
