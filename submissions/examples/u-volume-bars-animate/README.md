# ease-volume-bars-animate

Animated volume/equalizer bars for media and podcast players.

## Features

- Equalizer style animation
- Independent staggered bars
- Playing state activation
- Responsive design
- Pure HTML and CSS

## Acceptance Criteria Covered

### Volume Bar Animation

- Multiple animated bars
- Independent staggered delays
- scaleY oscillation effect
