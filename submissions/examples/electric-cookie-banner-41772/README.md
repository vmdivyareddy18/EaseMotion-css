# Electric Cookie Consent Banner (`ease-electric-cookie-banner`)

A high-performance, glassmorphic space-inspired legal telemetry 
notification container powered by pure CSS layout operations.

---

## 📋 Overview

The `electric-cookie-banner` variant represents an enterprise-grade 
UI solution for cookie management inside science, telemetry, and galactic dashboards.

The visual layout implements highly polished neon lighting features 
alongside responsive glass surfaces without imposing scripting resources. 

Dismiss loops are bound directly to standard HTML inputs to assure 
structural actions complete flawlessly without requiring JS triggers.

---

## ✨ Features

* **Glassmorphic Spatial Chassis**
  Applies advanced `backdrop-filter` rules integrated with cosmic 
  canvas variables to float on the main dashboard layers cleanly.

* **Moving Laser Energy Pulse Lines**
  Integrates a looping horizontal energy scanning bar styled exclusively 
  with CSS keyframes to draw immediate user focus.

* **No-Asset CSS Cookie Design**
  Uses pure HTML and CSS radial mask structures to form a fully stylized 
  cookie icon with no external assets needed.

* **Zero-Script Trigger Channels**
  Controls closing states directly via checkbox hack workflows to 
  reduce script execution requirements.
