# Kids Clothing Brand Demo

A responsive Kids Clothing Brand landing page built using **EaseMotion CSS**.

## Features

- Hero section with call-to-action
- Collections by age group (0–2, 3–5, 6–9, 10–12)
- Kids Size Guide
- Safety Certifications
- Parent Reviews
- Gift Finder CTA
- Newsletter Signup
- Responsive Design
- Footer
- Hover Effects
- Smooth Entrance Animations

## Files

```
demo.html
style.css
README.md
```

## EaseMotion CSS Classes Used

- ease-fade-up
- ease-slide-left
- ease-slide-right
- ease-zoom-in

## Technologies Used

- HTML5
- CSS3
- EaseMotion CSS

The page is fully responsive for:
