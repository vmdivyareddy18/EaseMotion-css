# Neumorphic Music Player UI

A sleek, modern music player interface utilizing the "Neumorphism" (soft UI) design trend. It features an animated spinning vinyl record and a smooth CSS progress bar, built entirely with HTML and CSS.

## Features
- Pure CSS implementation
- Neumorphic design using complex `box-shadow` techniques (both outset and inset)
- Animated spinning vinyl record using CSS keyframes and radial gradients
- Interactive neumorphic buttons with inset shadow `:active` states
- Animated progress bar

## Usage
Simply include the HTML structure and the accompanying `style.css`. The neumorphic effect relies on the background color (`#e0e5ec`) matching the container's background to create the illusion of extruded shapes. Be sure to maintain this color relationship if you adapt the design into an existing project.
