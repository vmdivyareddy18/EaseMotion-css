# 🛒 FreshGo Grocery Delivery Demo

A responsive landing page for an on-demand grocery delivery app built using **EaseMotion CSS** utility classes. This demo highlights modern UI components, smooth entrance animations, and interactive hover effects without using custom keyframes.

---

## 📁 Folder Structure

```
submissions/
└── examples/
    └── grocery-delivery-app/
        ├── demo.html
        ├── style.css
        └── README.md
```

---

## ✨ Features

- 🚚 Hero section with grocery delivery mockup
- ⚡ 30-minute delivery feature
- 🥦 Fresh produce showcase
- 📍 Live order tracking feature
- 🏪 Retail partner section
- 💰 Free delivery pricing information
- ⭐ Customer testimonials
- 👨‍💼 Driver/Shopper signup CTA
- 📱 Google Play & App Store buttons
- 📱 Fully responsive design

---

## 🎬 EaseMotion Classes Used

- `ease-fade-up`
- `ease-fade-in`
- `ease-slide-up`
- `ease-zoom-in`
- `ease-hover-grow`
- `ease-hover-glow`
- `ease-hover-lift`

---

## ▶️ How to Run

1. Open the project folder.
2. Open `demo.html` in your browser.
3. No installation or build tools are required.
