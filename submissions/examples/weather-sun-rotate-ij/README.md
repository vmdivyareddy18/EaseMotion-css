# Weather Sun Rotate

1. What does this do? A sun icon slowly rotates continuously (rotate 360deg) while its rays subtly pulse in opacity at staggered intervals.

2. How is it used? Apply `.sun-icon` to the container, with `.sun-body` for the circle and `.ray` elements for the rays.

3. Why is it useful? It delivers a live weather widget icon animation with two simultaneous motion layers — rotation and pulse — using pure CSS, suitable for forecast cards and app launchers.
