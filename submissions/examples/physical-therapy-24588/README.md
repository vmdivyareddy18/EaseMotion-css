# Physical Therapy Practice Template
A dynamic, results-driven, and empowering template for physical therapy clinics, rehabilitation centers, and sports performance practices.

- **Design**: Uses a "Motion-Vitality" palette with motion-azure blues, vitality-green growth accents, and high-contrast, professional layouts to signify movement, recovery, and clinical excellence.
- **Use Case**: Physical therapy clinics, rehabilitation centers, and sports-medicine practices.
