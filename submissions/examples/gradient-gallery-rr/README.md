# Gradient Collection Gallery Component

## What does this do?

Provides a curated collection of modern CSS gradients displayed in an interactive gallery layout.

## How is it used?

```html
<div class="gradient-card sunset">
    <h3>Sunset Glow</h3>
</div>
```

## Why is it useful?

Gradients are widely used in modern interfaces for cards, hero sections, buttons, and backgrounds. This component showcases reusable gradient patterns with hover animations using pure HTML and CSS.
