# Morphing Shapes Animation

A pure CSS animation that smoothly transitions between three geometric shapes: **Circle → Square → Triangle**.

## Demo

Open `index.html` in any modern browser.

## Features

- ✅ Pure CSS — zero JavaScript
- ✅ 60fps hardware-accelerated animations
- ✅ Multiple size, speed, and color variants
- ✅ Respects `prefers-reduced-motion` for accessibility
- ✅ Perfect for loading states, decorative elements, and playful UIs
