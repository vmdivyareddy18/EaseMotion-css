# Animated Expanding Search Bar

A modern expanding search bar built with pure CSS and EaseMotion-inspired animations.

## Features

- Smooth width expansion
- Focus animations
- Responsive design
- Modern UI styling
- CSS-only implementation

## Folder Structure

```text
r-expanding-search-bar/
├── README.md
├── demo.html
└── style.css
```

## Usage

Open `demo.html` in a browser.

## EaseMotion Concepts Demonstrated

- Smooth transitions
- Focus interactions
- Micro-animations
- Reusable UI components
