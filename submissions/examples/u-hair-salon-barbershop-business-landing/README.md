# Advanced: Business Landing — Hair Salon & Barbershop

A responsive business landing page for a modern hair salon and barbershop built using **HTML**, **CSS**, and **EaseMotion CSS**. This example demonstrates how to create a stylish grooming website using only existing EaseMotion CSS animation and utility classes.

## Features

* Responsive navigation bar
* Stylish hero section with booking CTA
* Services and pricing overview
* Professional stylist profiles
* Modern responsive layout
* Elegant salon-inspired design
* Smooth entrance animations
* Interactive hover effects

## EaseMotion CSS Classes Showcased

### Entrance Animations

* `ease-fade-down`
* `ease-fade-up`
* `ease-fade-left`
* `ease-fade-right`
* `ease-scale-in`

### Hover Animations

* `ease-hover-grow`
* `ease-hover-lift`
