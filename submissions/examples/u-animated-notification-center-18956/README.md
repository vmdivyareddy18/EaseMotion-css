# Animated Notification Center UI Component

A modern notification center component showcasing smooth card animations, unread indicators, hover interactions, and responsive design.

## Features

- Slide-in notification animations
- Pulsing unread indicators
- Hover movement effects
- Responsive layout
- Modern dashboard-inspired UI

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser to view the component.

## Customization

You can modify:

- Colors
- Animation timing
- Card spacing
- Notification content
- Hover effects
