# Animated Search Bar Component

A modern **Animated Search Bar Component** built with **HTML** and **CSS** for EaseMotion CSS.

---

## ✨ Features

- Modern rounded search bar
- Smooth hover animations
- Expand on focus
- Animated search icon
- Animated clear button
- Outline variant
- Filled variant
- Glassmorphism variant
- Rounded pill variant
- Responsive design
- Dark mode support
- Reduced motion support
