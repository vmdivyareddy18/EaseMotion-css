# Perspective Date Picker

A modern date picker component with a subtle 3D perspective animation built using pure HTML and CSS.

## Features

- Pure HTML and CSS (No JavaScript)
- 3D perspective hover effect
- Smooth transitions
- Responsive layout
- Modern card-based design
- Lightweight and reusable

## Files

```text
perspective-date-picker-ak/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<div class="date-picker-card">
    <h2>Select Date</h2>
    <input type="date">
</div>
```

The appearance and animation can be customized by modifying the CSS properties in `style.css`.

## Preview

Displays a modern date picker inside a clean card with a smooth 3D perspective animation and interactive hover effects.

## Why is it useful?

This lightweight, dependency-free component provides an engaging date selection experience while following EaseMotion CSS's philosophy of simple, reusable, animation-first UI components.
