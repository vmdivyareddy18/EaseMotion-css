# Rotate-In Entrance

**What does this do?**
Provides a dynamic entrance animation where an element fades in, scales up, and rotates into its final resting position.

**How is it used?**
Apply the class to any element you want to animate when the page loads (or when dynamically inserted):
```html
<div class="rotate-in-entrance">
    I am spinning into view!
</div>
```

**Why is it useful?**
It adds energetic, playful motion to hero images, icons, or modal popups, aligning perfectly with EaseMotion's animation-first design language. The cubic-bezier curve gives it a satisfying bounce at the end of the rotation.
