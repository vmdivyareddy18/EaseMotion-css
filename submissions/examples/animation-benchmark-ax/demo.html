Animation Performance Benchmark

Use the included Puppeteer script to measure frame rate and jank for EaseMotion CSS animations.

How to run

# Install dependencies
npm install puppeteer

# Run the benchmark
node benchmark.js

The script loads each animation class for 3 seconds, measures average FPS and jank, and prints a report.

Sample Output

Animation          Avg FPS   Jank %   Jank Frames
-------------------------------------------------
ease-fade-in       58.4      2.1%     3
ease-slide-up      59.1      1.8%     2
ease-bounce        57.2      4.5%     7
ease-pulse         56.8      5.2%     8
