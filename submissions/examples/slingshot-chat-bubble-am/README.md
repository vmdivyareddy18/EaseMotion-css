# Slingshot Chat Bubble — Banking Design Pattern

A sleek, premium, responsive **Slingshot Chat Bubble** component inspired by private banking interface patterns. Built using pure HTML and CSS for the EaseMotion CSS library.

## Features

- **Slingshot Physics**: Elegant incoming animations that mimic elastic pullbacks (`cubic-bezier(0.34, 1.56, 0.64, 1)`) for bubbles, ensuring an organic springiness.
- **Pure CSS Interactive States**: Simulates multi-step interactive chat dialog choices using hidden checkbox selectors without a single line of JavaScript.
- **Staggered Delays**: Staggered transition delays (`delay-1`, `delay-2`, `delay-3`) mimic realistic typing speeds for back-and-forth advisor conversations.
- **Luxury Banking Aesthetic**: A high-end dark vault theme utilizing gold-accented borders, rich typography (Inter), active pulse states, and clean gradients.
- **Responsive & Accessible**: Fully optimized for mobile and desktop viewports, includes full keyboard-focused support, and honors user-preferred reduced motion media requests.
