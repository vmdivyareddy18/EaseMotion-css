Understanding Transform Origin

The transform-origin property changes the pivot point around which a transformation is applied. See how different origins affect the rotation path.

right bottom

Used in this demo

  • Pivot at lower-right corner
  • Element swings from upper-left
  • Creates a natural entrance arc
center

Default behavior

  • Pivot at center of element
  • Element rotates in place
  • No positional shift — spins on spot
top left

Opposite pivot

  • Pivot at upper-left corner
  • Element swings from lower-right
  • Opposite arc direction
ⓘ
Key insight: With transform-origin: right bottom, the initial rotate(45deg) causes the element's visible mass to shift toward the upper-left. As it animates to rotate(0deg), the element settles into position, creating a smooth, lifelike entrance.

How It Works

@keyframes
/* Start state — element is rotated and hidden */
@keyframes rotateFromUpperLeft {
  0% {
    opacity: 0;
    transform: rotate(45deg);
  }
  100% {
    opacity: 1;
    transform: rotate(0deg);
  }
}

/* The transform-origin is set on the element itself */
.element {
  transform-origin: right bottom;
  animation: rotateFromUpperLeft 0.8s
    cubic-bezier(0.22, 1, 0.36, 1) forwards;
}
