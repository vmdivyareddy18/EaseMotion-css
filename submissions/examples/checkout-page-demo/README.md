# Checkout Page Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive checkout page.

---

## Components

- Billing Form
- Shipping Address
- Payment Method
- Order Summary
- Place Order Button

---

## Folder Structure

```
submissions/examples/checkout-page-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to verify responsiveness.
3. Observe the EaseMotion CSS animations applied to the form and order summary.

---

## Features

- Responsive checkout layout
- Billing information form
- Payment method selector
- Order summary
- Place Order button
- Uses existing EaseMotion CSS animation classes
