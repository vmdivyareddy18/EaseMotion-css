# Ease Fractal Tree Grow

## Overview

Ease Fractal Tree Grow is a nature-inspired CSS animation where a tree grows progressively from trunk to branches and finally glowing leaves.

---

## Features

- 🌳 Animated trunk growth
- 🌿 Sequential branch animation
- ✨ Leaf appearance effect
- 💡 Soft glowing leaves
- ⚡ Pure HTML & CSS
- 📱 Responsive demo

---

## Demo

The demo includes:

- Tree trunk growth
- Four animated branches
- Sequential leaf reveal
- Natural growing animation

---

## Usage

```html
<div class="tree">
...
</div>
```

---

## Customization

You can customize:

- Tree height
- Branch angles
- Leaf colors
- Glow intensity
- Animation delays

---

## Browser Support

Works in modern browsers supporting CSS animations and transforms.
