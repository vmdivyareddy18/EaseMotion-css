# Animated Notification Toast / Snackbar System

A clean, modern, and fully responsive **Notification Toast System** perfect for providing user feedback with success messages, error alerts, warnings, and info notifications. This example demonstrates how to combine **EaseMotion CSS** for smooth slide-in animations, auto-dismiss progress bars, and interactive close buttons to create a polished, non-intrusive notification experience.

## 🚀 Features

- **Smooth Slide-In Animations**: Toasts slide in from the right using `ease-slide-in-right` for a smooth entrance.
- **Four Toast Types**: Success (green), Error (red), Warning (amber), and Info (blue) with distinct colors and icons.
- **Auto-Dismiss with Progress Bar**: Toasts automatically dismiss after a configurable duration, with a visual progress bar showing time remaining.
- **Configurable Position**: Toasts can appear in four positions: top-right, top-left, bottom-right, or bottom-left.
- **Configurable Duration**: Choose from 3s, 5s, 8s, or disable auto-dismiss entirely.
- **Stackable Toasts**: Multiple toasts can be displayed simultaneously and stack vertically.
- **Interactive Close Button**: Each toast has a close button with `ease-hover-grow` for tactile feedback.
- **Smooth Dismissal**: Toasts slide out smoothly when closed or auto-dismissed.
- **Fully Responsive**: Adapts gracefully to mobile screens with full-width toasts on small devices.
