# Ease Lava Lamp Background

## Overview

Ease Lava Lamp Background creates a smooth animated background using colorful morphing blobs inspired by classic lava lamps. The animation is built entirely with HTML and CSS.

---

## Features

- 🌈 Animated morphing blobs
- 🎨 Multiple gradient-inspired colors
- 🔄 Infinite smooth animation
- ✨ Soft blur effect
- 📱 Responsive hero section
- ⚡ Pure HTML & CSS

---

## Demo Includes

- Five animated blobs
- Hero heading
- Description
- Primary & secondary buttons

---

## Usage

```html
<div class="lava-bg">
    <span class="blob"></span>
</div>
```

---

## Customization

You can customize:

- Blob colors
- Blob sizes
- Animation duration
- Blur amount
- Background color
- Number of blobs

---

## Browser Support

Compatible with modern browsers supporting CSS animations, transforms and filters.

---
