# Ease Aurora Background

## Overview

Ease Aurora Background creates a premium animated background inspired by the Northern Lights using layered gradients and smooth motion.

---

## Features

- 🌌 Animated aurora light effect
- 🎨 Multi-layer gradient animation
- ✨ Soft blur transitions
- 💡 Modern hero background
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Green Aurora
- Blue Aurora
- Purple Glow Layer
- Hero Content

---

## Usage

```html
<div class="aurora"></div>
```

---

## Customization

You can customize:

- Aurora colors
- Animation speed
- Blur intensity
- Layer size
- Background color

---

## Browser Support

Compatible with modern browsers supporting CSS gradients, filters, and transforms.
