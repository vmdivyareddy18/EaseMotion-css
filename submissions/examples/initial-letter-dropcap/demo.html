Browser Support & Fallback

initial-letter is supported in Safari and Chrome 110+. For non-supporting browsers, the .drop span is styled as a floated, large-font drop cap with ::first-letter as a secondary fallback. The @supports at-rule scopes the modern property so the float fallback is the default and initial-letter overrides it when supported.

Resize the viewport to see how drop caps reflow. All styles are self-contained in style.css.
