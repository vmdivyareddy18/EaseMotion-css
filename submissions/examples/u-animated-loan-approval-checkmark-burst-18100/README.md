# Animated Loan Approval Checkmark Burst

## Overview

Animated Loan Approval Checkmark Burst is a lightweight CSS animation that simulates successful loan approval using a pulsing checkmark badge.

## Features

- CSS-only animation
- Approval confirmation effect
- Success feedback visualization
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="loan-approval-burst">
  <span class="approval-icon">✅</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
