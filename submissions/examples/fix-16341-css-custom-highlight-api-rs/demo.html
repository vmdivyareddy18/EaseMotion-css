CSS Custom Highlight API Utilities

Search Results Highlight

The CSS Custom Highlight API provides a mechanism for styling arbitrary text ranges on a document by using JavaScript to create the ranges, and CSS to style them. This is incredibly useful for implementing custom search features, syntax highlighting, or collaborative editing cursors without modifying the DOM structure with extra span tags.

Multiple Highlight Ranges (Syntax/Grammar)

You can register multiple custom highlights. For instance, we can highlight grammatical errors in red, and warnings in yellow, entirely independent of the DOM elements.
