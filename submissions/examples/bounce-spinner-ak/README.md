# Bounce Spinner

A glassmorphism-inspired bounce spinner built using HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML and CSS (No JavaScript)
- Glassmorphism-inspired design
- Responsive layout
- Accessible loading indicator using `role="status"` and `aria-label`
- Uses EaseMotion animation class (`ease-fade-in`)
- Smooth bouncing animation

## Files

```
bounce-spinner-ak/
├── demo.html
```

## Usage

```html
<div class="bounce-spinner">
    <span></span>
    <span></span>
    <span></span>
</div>
```
The spinner can be customized by modifying the CSS properties in `style.css`.

## Preview

The component displays three animated bouncing dots inside a frosted glass card with a gradient background, making it suitable for loading states in modern web interfaces.


## Why is it useful?

This lightweight, dependency-free animation follows EaseMotion's philosophy of creating simple, reusable, and visually engaging motion effects for modern web interfaces.
