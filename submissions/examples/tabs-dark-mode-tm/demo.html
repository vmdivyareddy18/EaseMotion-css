Interactive Tabs

The .ease-tabs component uses radio inputs for state and CSS selectors for the active tab indicator. Dark mode overrides the tab label color, active indicator, and tab panel background.

Overview

EaseMotion CSS is a lightweight, accessible animation and UI component library for modern web projects. It provides a collection of composable utility classes and components.

Features

  • Zero-dependency, pure CSS components
  • Accessible by default (keyboard + screen reader)
  • CSS variable-based theming and dark mode
  • Reduced-motion safe animations

Pricing

Free and open source under the MIT license. Use it in personal and commercial projects with no attribution required.
