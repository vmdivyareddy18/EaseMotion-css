# Ease Orbital Notification Radar

## Overview

A futuristic notification radar featuring rotating scan beams, glowing ping indicators, and concentric orbit rings built entirely with HTML and CSS.

## Features

- Rotating radar sweep
- Animated notification pings
- Concentric orbit rings
- Glassmorphism container
- Responsive layout
- Pure HTML & CSS

## Folder Structure

```text
ease-orbital-notification-radar-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile screens.

## Why it fits EaseMotion CSS

Demonstrates reusable radar-style animations, orbital motion, and modern CSS effects suitable for dashboards, monitoring systems, developer tools, and futuristic interfaces.
