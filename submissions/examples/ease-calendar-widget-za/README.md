# CSS Calendar Widget

## What does this do?
CSS calendar widget with monthly grid, today highlight, event dots, date range picker, and upcoming events list.

## How is it used?
```html
<div class="cw-calendar"><div class="cw-weekdays"><span>Sun</span>...<span>Sat</span></div><div class="cw-grid"><span class="cw-today">16</span><span class="cw-event">18</span></div></div>
```

## Why is it useful?
Provides a calendar date UI without JavaScript. Monthly grid with weekday headers. Today highlight. Event date indicators. Date range picker for check-in/check-out. Events list with date badges. Ideal for booking widgets, scheduling dashboards, and event calendars.
