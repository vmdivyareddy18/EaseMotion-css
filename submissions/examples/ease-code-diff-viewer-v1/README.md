# Ease Code Diff Viewer

## Overview

Ease Code Diff Viewer is a reusable component inspired by GitHub's code review interface. It displays added and removed code with smooth animations and responsive styling using only HTML and CSS.

---

## Features

- 📝 Side-by-side code comparison
- ✅ Added line highlighting
- ❌ Removed line highlighting
- 🎯 Hover interaction
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

## Usage

```html
<div class="diff">
...
</div>
```

## Customization

Works in all modern browsers supporting CSS Grid, Flexbox, and CSS animations.
