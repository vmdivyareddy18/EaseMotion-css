# Cat Walking

1. What does this do? A CSS-only cat silhouette that walks in place with a subtle body bounce and tail wag animation.

2. How is it used? Include .cat inside a .scene container. The cat is built with pseudo-elements (body ellipse + tail triangle) and animated with keyframes.

3. Why is it useful? Demonstrates how pure CSS shapes and keyframe animations can create expressive animal motion without images.
