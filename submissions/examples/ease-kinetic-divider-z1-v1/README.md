# Ease Kinetic Divider

A CSS-only animated divider with a glowing kinetic center point for separating sections.

## Features

- Pure HTML & CSS
- Animated center glow
- Responsive
- Lightweight
- Reusable divider

## Structure

```
ease-kinetic-divider-z1/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any browser.

## Why EaseMotion CSS

Provides a reusable animated divider that enhances layouts while keeping the code minimal and animation-focused.
