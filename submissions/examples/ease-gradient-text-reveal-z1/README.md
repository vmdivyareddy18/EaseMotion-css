# Ease Gradient Text Reveal

## Overview

A lightweight text reveal effect with an animated gradient created using HTML and CSS.

## Features

- Pure HTML & CSS
- Animated gradient text
- Responsive
- Lightweight
- Easy to customize

## Folder Structure

```
ease-gradient-text-reveal-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why This Example

This example provides a reusable animated heading effect suitable for hero sections, banners, and landing pages.
