# Pulse File Uploader

A dark mode inspired animated file uploader component built with pure HTML and CSS.

## Features

- Pure CSS implementation
- Pulse animation interaction
- Dark mode UI style
- Responsive design
- No JavaScript required
- Accessible file input structure


## Usage

```html
<label class="pulse-uploader">

  <input type="file" hidden>

  <div class="upload-icon">
    ↑
  </div>

  <h2>Upload File</h2>

  <p>Drag & drop your files</p>

</label>
