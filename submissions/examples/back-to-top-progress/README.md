# Back to Top with Progress Ring

A floating back-to-top button that appears after scrolling 200px. An SVG progress ring fills clockwise to indicate the user's scroll progress through the page. Clicking the button smoothly scrolls to the top.

## EaseMotion CSS classes used

None required — built with plain CSS and SVG.

## How to run

Open `demo.html` in a browser and scroll down. The button will appear in the bottom-right corner with a progress ring.

## Accessibility notes

The button has an `aria-label="Back to top"`. Reduced motion disables the opacity and ring transitions.
