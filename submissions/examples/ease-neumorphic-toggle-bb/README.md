# Neumorphic Soft Toggle Switch Animation

## Description
A soft UI interactive toggle switch constructed strictly utilizing light-source drop shadow transitions for neumorphic aesthetics.

## Usage
Add the toggle template to your layout:
```html
<div class="toggle-wrapper">
  <input type="checkbox" id="toggle" class="neumorphic-toggle-input">
  <label for="toggle" class="neumorphic-toggle-label">
    <span class="neumorphic-toggle-switch"></span>
  </label>
</div>
```

## Why is it useful?
Delivers a highly modern, tactile neumorphic toggling interaction using clean CSS box-shadows. Safely adapts to low-motion accessibility environments.
