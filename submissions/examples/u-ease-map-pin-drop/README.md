# ease-map-pin-drop

Animated map marker placement effect with bounce landing feedback.

## Features

- Drops from above using translateY(-100px)
- Bounce landing animation
- Squash and stretch effect
- Infinite looping demo
- Pure HTML and CSS
- Suitable for maps and navigation interfaces

## Usage

```html
<div class="map-pin">📍</div>
```

## Animation Details

- Marker starts above the map.
- Falls downward using translateY.
- Squashes slightly on impact.
- Performs a small bounce before settling.
