# Magic Lamp

### What does this do?

It shows a golden genie lamp with glowing magic smoke curling up out of its spout. The wisps grow and drift as they rise, and sparkles twinkle in the air around them. Under reduced motion the smoke settles and the sparkles hold still.

### How is it used?

```html
<div class="lamp">
  <span class="spout"></span>
  <span class="body"></span>
  <span class="lid"></span>
  <span class="handle"></span>
</div>
<span class="wisp w1"></span>
```

The lamp body is a rounded gold gradient with a curved spout, a lidded top, a looped handle, and a foot. Each `wisp` is a blurred blue glow that runs the `rise` animation, floating up and to the side while growing and fading, so the smoke looks like it is streaming from the spout. Sparkles twinkle on their own timing.

### Why is it useful?

Fantasy, wish, and reward themes use a magic lamp. This builds a genie lamp with rising magic smoke using pure CSS shapes and animation, no images, with a reduced motion fallback.
