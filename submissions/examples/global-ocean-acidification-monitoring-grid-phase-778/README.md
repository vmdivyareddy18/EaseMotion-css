# Global Ocean Acidification Monitoring Grid (Phase 778)

## Overview

A responsive single-page HTML/CSS dashboard that visualizes key ocean acidification metrics through a modern glassmorphism interface. The layout presents environmental statistics such as ocean pH, carbon absorption, coral reef health, and monitoring station coverage.

## Features

- Responsive grid layout
- Modern glassmorphism cards
- Smooth hover animations
- Gradient ocean-inspired background
- Fully responsive design
- Pure HTML and CSS
- Easy to customize

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- CSS Animations

## Folder Structure

```
global-ocean-acidification-monitoring-grid-phase-778/

```

## How to Run

1. Download or clone the repository.
2. Open `demo.html` in any modern browser.
3. Resize the browser window to see the responsive layout.
