# Shake Avatar

A responsive animated avatar component for **EaseMotion CSS**.

## Preview

The avatar performs a subtle shake animation when hovered or focused.

## Features

- Pure CSS
- Responsive
- Accessible
- Keyboard friendly
- Works without JavaScript
- Respects prefers-reduced-motion
- Uses EaseMotion CSS variables

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<button class="shake-avatar ease-duration-normal ease-ease-standard">
  <img src="avatar.jpg" alt="Profile picture" />
</button>
```

## Accessibility

- Semantic button element
- Keyboard focus support
- Visible focus indicator
- Alt text for avatar images
- Reduced motion support
