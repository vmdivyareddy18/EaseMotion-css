# CSS Scale-Hover Carousel for Accessible Web Layouts

A pure HTML and CSS carousel featuring smooth scale-hover animations designed for accessible web interfaces.

## Features

- Pure HTML & CSS
- Smooth scale-hover interaction
- Responsive layout
- Keyboard accessible
- CSS Custom Properties support
- No JavaScript required

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` directly in any modern browser.

## Customization

Modify CSS variables to control:

- Animation duration
- Hover scale
- Transition easing
- Colors
- Border radius
- Spacing
