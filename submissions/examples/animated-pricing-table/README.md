# Animated Pricing Table

An interactive pricing comparison component built with pure CSS. Features a 3-tier plan layout (Starter, Professional, Enterprise) with animated hover effects, a monthly/annual billing toggle with price flip animation, and a magnetic spotlight effect that follows the cursor.

### GSSoC 2026 Contribution

**Features:**

- Three pricing tiers with feature comparison (checkmark/cross icons)
- Monthly/Annual billing toggle with smooth price flip animation
- "Most Popular" badge on the featured plan with pulse effect
- Cursor-following spotlight gradient on each card
- Staggered card entry animation on page load
- Responsive single-column layout on mobile
