White-Space Utilities

Control how spaces, tabs, and line breaks are handled inside text blocks

This is a long line of code. Multiple spaces and tab indentations. Another line.

normal: Collapses multiple spaces and tabs. Breaks lines automatically to fit container width. Ignores raw line breaks.
Wrapping & Collapsing
whitespace-normal

Sequences of white space are collapsed. Newline characters in source are treated as single spaces. Text wraps automatically when it reaches container limits.

whitespace-nowrap

Collapses white space like normal, but suppresses all automatic line wrapping. Text remains on a single line, causing horizontal overflow if wider than the parent container.

Formatting Preservation
whitespace-pre

Preserves all source spaces, tabs, and line breaks exactly as written. Does not wrap text automatically. Typical for displaying raw code snippets.

whitespace-pre-line

Collapses source spaces and tabs, but preserves raw source line breaks (newlines). Automatically wraps text to fit container boundaries.

whitespace-pre-wrap

Preserves source spaces, tabs, and line breaks exactly. Text will wrap automatically when reaching container limits.
