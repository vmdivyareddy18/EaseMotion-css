# Typewriter Text Animation

**What does this do?**
Reveals text character-by-character with a blinking cursor using pure CSS keyframes and `steps()`. No JavaScript required.

**How is it used?**
```html

<h1 class="ease-typewriter">Hello, EaseMotion!</h1>

<h1 class="ease-typewriter" style="--ease-typewriter-chars: 30">
  Animation-first CSS for humans.
</h1>

<h1 class="ease-typewriter ease-typewriter-no-cursor">Build faster.</h1>
```

**Why is it useful?**
Typewriter effects are widely used in hero sections, terminal UIs, and portfolios. The `--ease-typewriter-chars` property must match your text's character count exactly — mismatches cause choppy animations. This fits EaseMotion CSS's animation-first, zero-JS philosophy perfectly.
