# Animated FAQ Accordion

A modern FAQ accordion component featuring expandable questions, animated content reveals, and responsive layouts.

## Features

- Accordion-style FAQ cards
- Smooth answer reveal animation
- Rotating plus icon
- Hover interactions
- Responsive layout
- Staggered reveal animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

You can customize:

- Questions and answers
- Colors
- Animation timing
- Border radius
- Typography

## Use Cases

- Documentation websites
- SaaS landing pages
- Help centers
- Product websites
- Community portals
