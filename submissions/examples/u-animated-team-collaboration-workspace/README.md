# Animated Team Collaboration Workspace

A modern **Team Collaboration Workspace** built using **pure HTML and CSS** for the EaseMotion CSS project. This example demonstrates a collaborative workspace dashboard featuring team members, task management, project activity, and productivity metrics with smooth CSS-only animations.

## Features

* 👥 Workspace overview cards
* 📊 Productivity progress indicator
* 📋 Task management board
* 🟢 Animated online/offline member status
* 🧑 Team member cards with hover interactions
* 📁 Activity feed with recent updates
* ✨ CSS-only hover and entrance animations
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

All animations follow the EaseMotion CSS naming convention.

* `ease-progressGrow`
* `ease-pulse`
* `ease-rotateIn`
* `ease-slideIn`
* `ease-hoverLift`
* `ease-cardReveal`

## Customization

All colors and spacing are controlled using CSS variables.

Example:

```css
:root{
    --primary:#4f46e5;
    --secondary:#10b981;
    --warning:#f59e0b;
    --surface:#ffffff;
    --bg:#f5f7fb;
}
```
