# Glowworm Cave

A pure CSS and HTML glowworm cave: hundreds of bioluminescent points glowing on the ceiling and dangling silk lines, mirrored in the still water below.

## What it shows

- Glowworm larvae as tiny dots with layered box-shadow halos, twinkling out of phase across the whole ceiling
- Dangling silk snares as thin gradient threads, each ending in a live glowing bead
- A cratered cave ceiling from stacked dark radial gradients
- The still pool reflecting the lights as a blurred, rippling field of points
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
