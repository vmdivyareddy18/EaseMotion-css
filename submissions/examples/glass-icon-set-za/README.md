# Glassmorphism Icon Set

A set of glassmorphism-styled icons using backdrop-filter blur, semi-transparent backgrounds, and hover elevation with colored border accents.

## Files

- `demo.html` - Six glass icons in a grid
- `style.css` - Backdrop blur, glass border, hover animations

## Usage

Open `demo.html` and hover over icons to see the glass effect with border color accents.
