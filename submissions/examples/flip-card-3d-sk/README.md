# Flip Card 3D SK

Hover-activated 3D flip card for front/back content reveals.

## Usage

```html
<div class="flip-card">
  <div class="flip-card__inner">
    <div class="flip-card__face flip-card__face--front">Front</div>
    <div class="flip-card__face flip-card__face--back">Back</div>
  </div>
</div>
```

## Why EaseMotion CSS

Uses `perspective`, `transform-style: preserve-3d`, and smooth easing for tactile motion.
