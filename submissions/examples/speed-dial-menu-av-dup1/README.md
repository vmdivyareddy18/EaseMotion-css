# Speed Dial Menu

## What does this do?

Creates a colorful floating speed dial menu that expands into multiple quick-action buttons using pure HTML and CSS.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="speed-dial">
    <button class="fab">+</button>
</div>
```

---

## Why is it useful?

Speed dial menus provide quick access to common actions while minimizing interface clutter. They are widely used in mobile applications, dashboards, productivity tools, and admin panels. This standalone component demonstrates an animated floating action menu using only HTML and CSS.
