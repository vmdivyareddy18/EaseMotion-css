User Select Utilities

CSS user-select utilities to specify how users can highlight and select text. Improve web-app interactivity, block button highlights, and simplify copy-pasting code.

What is user-select?
The user-select property regulates whether users are allowed to highlight text. By controlling text selection, you can prevent accidental highlighting of UI menus, enable quick copy-pasting for code, or keep text standard.
