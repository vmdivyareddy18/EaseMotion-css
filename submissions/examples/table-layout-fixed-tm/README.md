# Table Layout Fixed

table-layout: fixed — Equal-width table columns for predictable layouts

## Features
- Uses `--ease-color-neutral-200`
- Uses `--ease-color-primary`
- Uses `--ease-radius-md`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Table Layout Fixed</span>
</div>
```

## Why is it useful?

This submission demonstrates the table layout fixed CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
