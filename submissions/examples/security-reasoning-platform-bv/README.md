# Enterprise Security Reasoning Platform

An interactive threat decision tree and propagation reasoning dashboard showcasing CSS stepped progress indicators, dial gauges, and hover animations.

---

## Features
- **Stepped Decision Progress:** Step-by-step model evaluations styled with interactive border state changes.
- **Model Confidence Gauge:** A circular progress dial displaying model assessment risk confidence.
- **Threat Mitigation Cards:** Layout panels for rapid security mitigations and quarantine options.
