# Ease XP Level Progress System

## Overview
This feature simulates a gamified XP system with animated progress tracking and level updates.

## ✨ Features
- Animated progress bar fill
- XP gain interaction
- Level increment system
- Smooth transitions and glow effects

## 🎯 Utility Classes
- ease-fade-in
- ease-slide-up
- ease-hover-grow

## 💡 Usage

```html
<div class="progress-bar">
  <div class="progress-fill"></div>
</div>
