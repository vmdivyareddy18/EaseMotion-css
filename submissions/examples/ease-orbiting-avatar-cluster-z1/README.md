# Ease Orbiting Avatar Cluster

## Overview

A futuristic avatar cluster featuring rotating orbital rings and floating profile nodes built entirely with HTML and CSS.

## Features

- Orbiting avatar layout
- Dual rotating rings
- Glassmorphism styling
- Responsive design
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-orbiting-avatar-cluster-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Provides a reusable orbit-based avatar component suitable for team dashboards, collaboration tools, AI workspaces, and modern profile interfaces.
