# Table Row Hover Highlight

## What does this do?
Adds a smooth highlight transition to table rows on hover, improving readability and scanability of tabular data.

## How is it used?
Apply the `.table-hover` class to a `<table>` element. Rows will smoothly highlight on hover with a subtle background color transition.

## Why is this useful?
Enhances data table usability by providing visual tracking as users scan rows, reducing the chance of misreading across columns.
