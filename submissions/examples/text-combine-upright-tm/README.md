# Text Combine Upright

text-combine-upright — Combines characters into single vertical glyph

## Features
- Uses `--ease-text-sm`
- Uses `--ease-color-text`
- Uses `--ease-color-bg`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Text Combine Upright</span>
</div>
```

## Why is it useful?

This submission demonstrates the text combine upright CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
