# Neon Glow Accordion — E-Commerce Checkout

A pure CSS accordion with neon glow animation for E-Commerce Checkout.

## Features
- neon glow animation
- Smooth expand/collapse with icon rotation
- E-Commerce Checkout aesthetic styling
- Keyboard accessible (native <details> element)

## Usage
```html
<details class="accordion-item">
  <summary class="accordion-header">Title</summary>
  <div class="accordion-body">Content...</div>
</details>
```

## Browser Support
- Chrome 12+, Firefox 49+, Safari 6+

## Tech Stack
- HTML + CSS only, no JavaScript
