What is the :has() selector?

The :has() pseudo-class lets you select a parent element based on its children. For example, .card:has(img) matches any .card that contains an image. It's a relational selector — often called the "parent selector" CSS developers have requested for decades.

How is this accordion CSS-only?

It uses native <details> / <summary> HTML elements for the toggle behavior — no JavaScript needed. :has(details[open]) on the parent .accordion__item lets us style the container when its child is open, enabling the border highlight and dimming of siblings.

Height animation technique

The smooth height transition uses the CSS Grid trick: setting grid-template-rows from 0fr to 1fr. This avoids the need to know the content's height in advance and works in pure CSS, unlike max-height hacks that produce uneven easing.

Browser support for :has()

Supported in Chrome 105+, Safari 15.4+, Edge 105+, and Firefox 121+. Coverage is now above 90% of global users. The accordion still works in older browsers — only the parent-styling via :has() degrades (the dimming effect disappears; open/close still works).
