# API Token Permission Scope Chips

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive API permission component that displays token scopes using colorful permission chips for developer dashboards, API management portals, and authentication interfaces.

## Features

- Permission scope chips
- Color-coded access levels
- Responsive flex layout
- Active token status
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="chip-group">

    <span class="chip read">Read</span>

    <span class="chip write">Write</span>

    <span class="chip admin">Admin</span>

</div>
```
