# CSS Magnetic Dock Navigation

A modern **CSS-only Magnetic Dock Navigation** inspired by the macOS Dock.

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Smooth hover animations
- Magnetic scaling effect
- Glassmorphism design
- Responsive layout
- Easy to customize

---

## Preview

Hover over the dock icons to see:

- Enlarged hovered icon
- Neighbor icons slightly scale
- Floating hover animation
- Frosted glass background

---

## Folder Structure

```
magnetic-dock-navigation/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Download the files.
2. Keep all three files in the same folder.
3. Open `demo.html` in your browser.

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Glassmorphism
- CSS Transitions
- `:has()` selector
- Transform animations

---

## Customization

You can easily change:

- Background gradient
- Icon size
- Hover scale
- Border radius
- Glass opacity
- Transition speed
- Emoji icons with SVG or Font Awesome

---

## Browser Support

The magnetic neighbor effect uses the CSS `:has()` selector.

Works in all modern browsers that support `:has()`.
