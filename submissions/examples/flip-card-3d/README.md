# 3D Perspective Flip Card Component

Pure CSS 3D flip card using perspective, transform-style preserve-3d, and backface-visibility.

## Features
- 3x3 grid of flip cards
- 3D perspective (1800px)
- Smooth 0.7s cubic-bezier transition
- backface-visibility for hidden back face
- Accessibility: aria-pressed, keyboard support
- Hover and focus triggers
