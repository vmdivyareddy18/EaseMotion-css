# Bulk Import Review Queue

## What does this do?

Shows ready, duplicate, invalid, and imported states for a bulk import review queue.

## How is it used?

```html
<li class="import-row has-warning">
  <span>leads.csv</span>
  <strong>Duplicate email</strong>
</li>
```

## Why is it useful?

Import flows need row-level validation feedback before users commit incoming data.
