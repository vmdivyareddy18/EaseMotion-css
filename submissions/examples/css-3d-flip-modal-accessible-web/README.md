# CSS 3D Flip Modal – Accessible Web Layout

## Overview

A responsive Accessible Web modal built with pure HTML and CSS. The component features a smooth 3D flip opening animation, accessibility-focused styling, responsive design, and keyboard accessibility without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D flip animation
- Accessible Web interface
- Glassmorphism modal
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Reduced motion support

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--success`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--blur`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Open Settings** to display the animated 3D flip modal and click **×** to close it.

Customize colors, animation timing, flip angle, scaling, blur, and spacing using the CSS custom properties in `style.css`.
