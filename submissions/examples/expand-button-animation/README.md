# Expand Button Utility

A lightweight CSS utility that creates an expanding effect for buttons during user interactions. The animation smoothly enlarges the button on hover and focus, providing clear visual feedback while maintaining performance.

## Features

* Pure CSS implementation
* Smooth expand animation using keyframes
* Supports hover, focus, and active states
* Lightweight and reusable
* Cross-browser compatible (Chrome, Firefox, Safari, Edge)
* No JavaScript required

## Usage

```html
<button class="ease-expand-btn">
  Hover Me
</button>
```

## Behavior

* Smoothly expands when hovered
* Supports keyboard accessibility through focus state
* Provides responsive interaction feedback
* Uses CSS easing for natural motion

## Why is it Useful?

It improves user interaction feedback by making buttons feel more responsive and engaging. The subtle expansion effect helps draw attention to clickable elements without being distracting.
