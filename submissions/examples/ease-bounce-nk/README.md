# Ease-Bounce Tutorial

## What does this do?
This is a beginner-friendly tutorial showing how to make your first contribution to EaseMotion CSS - from finding an issue to submitting a PR!

## How is it used?
Add the class to any HTML element:

    <div class="Ease-Bounce">I am bouncing!</div>

## Why is it useful?
This tutorial helps first time contributors understand the full GSSoC contribution workflow - finding issues, forking, cloning and submitting PR.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.

## Contribution Notes
