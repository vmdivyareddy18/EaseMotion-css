# Ease Kanban Board

## Overview

Ease Kanban Board is a reusable project management component inspired by Trello and Jira. It features animated task cards, responsive columns, and smooth hover interactions using pure HTML and CSS.

---

## Features

- 📋 Three Kanban columns
- 🎴 Animated task cards
- ✨ Hover lift effect
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- To Do column
- In Progress column
- Done column
- Animated cards
- Status indicators

---

## Usage

```html
<div class="board">
...
</div>
```

## Customization

- Card styles

## Browser Support

Compatible with all modern browsers supporting Flexbox, CSS animations, and transitions.
