How to Use

Apply the desired class to any block-level element. Each class works independently:

<div class="border-gradient-rotate">Content</div>
<div class="border-pulse">Content</div>
<div class="border-shimmer">Content</div>
<div class="border-dash-running">Content</div>

Accessibility: All animations respect prefers-reduced-motion: reduce and fall back to static border styles.
