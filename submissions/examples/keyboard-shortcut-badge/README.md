# Keyboard Shortcut Hint Badge

A display component for keyboard shortcut hints. Each shortcut shows a label followed by one or more `<kbd>` elements styled as pill-shaped badges with a subtle inset shadow. Modifier keys like Ctrl are separated by a plus sign.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to see a list of keyboard shortcut hints.

## Accessibility notes

Uses semantic `<kbd>` elements for keyboard input representation. The hints are purely visual documentation.
