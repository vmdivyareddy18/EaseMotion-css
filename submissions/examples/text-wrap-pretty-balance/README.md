# CSS text-wrap: pretty vs balance

1. **What does this do?** Compares `text-wrap: wrap` (default), `text-wrap: balance`, and `text-wrap: pretty` side-by-side so designers can see exactly how each value affects multi-line text rendering.

2. **How is it used?** Apply `text-wrap: balance` to headings and `text-wrap: pretty` to paragraph text via CSS. Resize the browser to observe how each value re-flows differently.

3. **Why is it useful?** Most developers default to `wrap` without realizing `balance` and `pretty` exist. This demo makes the trade-offs visible, helping teams adopt better typography practices.
