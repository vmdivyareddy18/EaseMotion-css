# Elastic Testimonial Card (`elastic-testimonial-card-ksk`)

1. What does this do?  
Displays neumorphic soft-UI testimonial cards that pop in with a springy elastic entry animation on page load and play a squish-stretch bounce keyframe when hovered.

2. How is it used?  
Wrap testimonial content inside a `.elastic-card` container placed inside a `.testimonial-grid`. Avatar SVGs sit inside `.author-avatar`, which is also neumorphically shadowed.

3. Why is it useful?  
Delivers a physically intuitive, tactile testimonial section that feels alive without any JavaScript — leveraging CSS animation staggering and elastic `cubic-bezier` curves.
