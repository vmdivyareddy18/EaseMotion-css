Features & Accessibility Details

  • Faceted Pendulum Swing: Hovering swatches triggers an elastic pendulum keyframe animation (swatch-swing-ag) that simulates gravitational damping and pivot tension.
  • Pure CSS Display Variables: Color selections bind values globally at the console panel level, allowing child preview elements and HEX displays to update coordinates dynamically in CSS.
  • Futuristic HUD Glow: Implements active box-shadow layers and light lens reflections to build rich neon lighting effects.
  • Keyboard Accessibility: Custom label buttons are fully focusable with keyboard focus indicator shadows. Selecting items using Space/Enter works natively.
