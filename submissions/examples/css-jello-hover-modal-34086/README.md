# CSS Jello-Hover Modal for Minimalist Tech Layouts

## Overview

This project presents a modern **Minimalist Tech Modal** featuring a smooth **Jello-Hover** animation built entirely with HTML and CSS. The component highlights a technology product with feature descriptions, platform statistics, and call-to-action buttons while maintaining a clean, lightweight, and responsive interface without requiring JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- CSS-only Jello-Hover animation
- Minimalist technology-inspired interface
- Responsive modal layout
- Feature highlight section
- Product statistics cards
- Dual call-to-action buttons
- Keyboard-accessible controls
- CSS custom properties for easy customization
- No JavaScript or external libraries required

## Customization

The component uses CSS custom properties for easy styling.

```css
:root{
    --bg:#f4f7fb;
    --card:#ffffff;
    --accent:#2563eb;
    --text:#1f2937;
    --muted:#6b7280;
    --radius:22px;
}
```

You can customize:

- Background colors
- Accent colors
- Typography
- Border radius
- Button styles
- Card shadows
- Feature card appearance
- Animation duration
- Modal spacing
