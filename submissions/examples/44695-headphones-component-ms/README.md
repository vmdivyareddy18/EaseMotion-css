# Headphones Product Card

A sleek, modern product card component for headphones or audio gear.

## Features
- **Hover Interactions**: Smooth 3D-like hover effects on the entire card and the headphone illustration.
- **Radial Glow**: A subtle, elegant radial background glow appears on hover.
- **Action Button**: A vibrant gradient "Add to Cart" button with satisfying press animations and box-shadow transitions.
- **Glassmorphism**: Built using dark-themed glassmorphism principles (backdrop filters and subtle borders).

## Files
- `demo.html`: The HTML structure of the product card.
- `style.css`: The styling and animations for the card, glows, and interactions.

## Usage
Add `.ease-headphone-card` to your project and include the provided styles. This component is completely responsive and relies purely on CSS for all visual effects.
