# Animated Trophy Victory Bounce

## Overview

A reusable trophy icon animation featuring a celebratory bounce effect suitable for achievements, milestones, rankings, rewards, leaderboards, and success states.

## Features

- CSS-only animation
- Trophy bounce effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="trophy-button">
  <span class="trophy-icon">🏆</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
