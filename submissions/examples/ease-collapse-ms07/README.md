# Ease Collapse

A pure CSS exit animation that smoothly collapses an element instead of removing it abruptly.

## Features

- Pure CSS
- Suitable for accordions
- Smooth exit animation
- Composable with delay classes
- Prevents abrupt layout jumps

## Usage

```html
<div class="ease-collapse">
    Content
</div>
```

## Effect

The element gradually:

- Shrinks vertically
- Fades out
- Collapses its height smoothly

## Why it fits EaseMotion CSS

This animation provides a natural exit transition for accordions, dismissible cards, and list items while keeping interactions fluid and human-friendly.
