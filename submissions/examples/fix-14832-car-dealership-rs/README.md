# Issue #14832: Advanced: Full Car Dealership Landing Page

This folder contains a standalone demonstration of a Car Dealership Landing Page as requested in Issue #14832.

## Files
- `demo.html`: The HTML structure for the landing page.
- `style.css`: The CSS styling emphasizing a sleek, premium, and high-performance aesthetic.

## How it works
The demo provides a complete layout for a luxury/performance car dealership featuring:
- A sophisticated color palette of Obsidian Black and Racing Red.
- A dynamic, full-screen hero section with glassmorphism search elements.
- A featured vehicles grid displaying car specs and pricing with hover reveals.
- A sleek "Why Choose Us" section.
- Modern CSS properties including flexbox, grid, backdrop-filter, and smooth transitions.
