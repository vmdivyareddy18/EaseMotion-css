# Text Decoration Utilities: Underline Offset, Thickness, And Style Variants

Demonstrates CSS text-decoration utilities including underline-offset, text-decoration-thickness, text-decoration-style, text-decoration-color, and text-underline-offset with EaseMotion color tokens.

## Features

- Demonstrates **text-underline-offset** CSS property with multiple practical variants
- Uses EaseMotion design tokens (`--ease-color-*`, `--ease-space-*`, `--ease-radius-*`)
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`
- Multiple size/style variants for real-world usage

## Usage

Apply the utility class to any element:

```html
<div class="underline-offset-xs">Content</div>
```

## Why is it useful?

The **text-underline-offset** CSS property is part of the modern CSS specification and enables
fine-grained control over text-decoration-utilities. It integrates seamlessly with EaseMotion's design token
system, making it easy to maintain consistent spacing, colors, and typography across your project.
