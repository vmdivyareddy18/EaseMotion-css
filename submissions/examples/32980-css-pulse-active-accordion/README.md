# Pulse Active Accordion

A modern Pulse Active Accordion built using only HTML and CSS. It features a clean minimalist design with a smooth pulse animation when expanded.

## Features

- Pure HTML & CSS
- No JavaScript
- Pulse animation on open
- Responsive design
- Accessible using `<details>` and `<summary>`
- Modern minimalist UI

Open `demo.html` in any modern web browser.
