# CSS 3D Flip Modal – Cyberpunk Neon Layout

## Overview

A responsive Cyberpunk Neon modal built entirely with HTML and CSS. The component features a smooth 3D flip opening animation, vibrant neon styling, glassmorphism effects, responsive layout, and keyboard accessibility without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D flip animation
- Cyberpunk neon interface
- Glassmorphism design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--blur`
- `--shadow`

## Files

- `demo.html` – Demonstration page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Launch Console** to display the animated Cyberpunk Neon 3D Flip Modal. Customize colors, animation timing, flip angle, scaling, blur, and shadows through the CSS custom properties in `style.css`.
