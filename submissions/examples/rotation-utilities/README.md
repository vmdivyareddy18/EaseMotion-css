Markdown# Rotation Utilities

A lightweight collection of CSS utility classes that apply common rotation transformations to elements. These utilities make it easy to rotate elements without writing custom CSS.

## Features

* Pure CSS implementation
* Five predefined rotation utilities
* Lightweight and reusable
* Human-readable class names
* Cross-browser compatible
* No JavaScript required

## Available Classes

| Class | Rotation |
|---------|---------|
| `ease-rotate-0` | 0° |
| `ease-rotate-45` | 45° |
| `ease-rotate-90` | 90° |
| `ease-rotate-180` | 180° |
| `ease-rotate-270` | 270° |

## Usage

```html
<div class="ease-rotate-45">Rotated Element</div>

## Why is it Useful?

These utility classes provide a quick and readable way to rotate elements for icons, badges, decorative elements, UI indicators, and creative layouts without requiring custom styles.
