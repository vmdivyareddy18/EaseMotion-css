# Accordion Icon Rotate

A native details accordion with a rotating icon and a small content reveal animation.

## Files

- `demo.html` - native `details` and `summary` accordion markup.
- `style.css` - icon rotation, content reveal, focus-visible state, and reduced-motion support.

## Highlights

- Uses semantic accordion behavior from the browser.
- Rotates the summary icon when the section is open.
- Keeps keyboard focus styling visible.
