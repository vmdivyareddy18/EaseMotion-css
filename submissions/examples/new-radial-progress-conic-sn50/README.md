# Conic Radial Progress Ring

A pure CSS circular progress indicator using conic gradients for dashboard metrics and completion states.

## Features

- Uses `conic-gradient()` to render progress without JavaScript.
- Customizable `--value` and `--accent` variables per metric.
- Includes a clean center label and card layout.
- Works for analytics, goal tracking, health scores, and profile completion.
- Responsive grid stacks cleanly on smaller screens.

## Usage

Open `demo.html` to view three radial progress examples. Set `--value` from `0` to `100` and customize `--accent` to change the progress color.
