# Animated Timeline Component

## Description

A modern vertical timeline component with smooth scroll reveal animations, glowing timeline markers, and interactive hover effects.

---

## Features

- Smooth fade-in entrance animation
- Glowing timeline markers
- Hover lift effect on cards
- Responsive mobile layout
- Lightweight JavaScript IntersectionObserver
- Reduced motion accessibility support

---

## Classes

- `.timeline` - Timeline container
- `.timeline-item` - Individual milestone
- `.dot` - Animated timeline marker
- `.card` - Content card
- `.date` - Timeline date label

---

## Accessibility

Supports `prefers-reduced-motion` to disable animations for users who prefer less motion.

---

## Browser Support

Works in all modern browsers supporting CSS animations and IntersectionObserver.
