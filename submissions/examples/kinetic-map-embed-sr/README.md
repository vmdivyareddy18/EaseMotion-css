# Kinetic Map Embed

## What does this do?

This component creates a simple animated map card using only HTML and CSS. It includes a moving background grid, a pulsing location marker, a glowing border, and a smooth floating effect.

## How is it used?

Open the `demo.html` file in your browser and make sure `style.css` is in the same folder.

Example:

```html
<div class="ease-map-card ease-float">
    ...
</div>
```

## Why is it useful?

Map sections are used on many websites like contact pages, portfolios, business websites, and travel sites. Instead of using a plain map placeholder, this component adds a clean animated look while keeping everything lightweight and responsive. Since it is built with only HTML and CSS, it is easy to reuse and customize.
