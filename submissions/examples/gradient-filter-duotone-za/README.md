# Duotone Gradient Filter Effect

A duotone image effect using CSS grayscale filters, gradient overlays, and mix-blend-mode. Three color variations demonstrate different duotone palettes.

## Files

- `demo.html` - Three duotone card variations
- `style.css` - Grayscale filter, mix-blend-mode overlay, hover opacity

## Usage

Open `demo.html` and hover over the duotone images to intensify the color overlay.
