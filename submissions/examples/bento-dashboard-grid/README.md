# CSS Bento Dashboard Grid

A modern **CSS-only Bento Dashboard Grid** demonstrating responsive grid layouts with elegant hover animations.

## Features

- Responsive CSS Grid layout
- Hover lift animation
- Smooth scaling effect
- Gradient cards
- Pure HTML & CSS
- No JavaScript required

## Folder Structure

```text
bento-dashboard-grid/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in your browser.
2. Hover over the cards to see the animation.
3. Customize colors, spacing, and card sizes as needed.

## Technologies Used

- HTML5
- CSS3 Grid
- CSS Transitions
- Responsive Design

## Preview

The demo showcases a dashboard-style Bento grid with cards that gently lift and scale on hover, making it suitable for admin panels, landing pages, and portfolio dashboards.
