# CSS 3D Flip Modal – Modern SaaS Layout

## Overview

A responsive Modern SaaS modal built entirely with HTML and CSS. The component features a smooth 3D flip opening animation, a modern SaaS landing section, pricing cards, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip animation
- Modern SaaS interface
- Responsive pricing cards
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--bg`
- `--surface`
- `--text`
- `--muted`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **View Platform** to display the animated 3D Flip Modal showcasing Modern SaaS pricing plans. Colors, animation timing, spacing, shadows, and flip effects can be customized using the CSS variables defined in `style.css`.
