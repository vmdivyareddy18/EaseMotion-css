New Component

ease-checkbox-animated

An accessible, customizable custom checkbox. Animates checked fills via cubic-bezier spring bounces and draws vector paths dynamically with pure CSS.

Installation & Usage

Wrap the hidden input checkbox and the custom box elements inside a parent label container to trigger selections globally.

<!-- Checkbox component -->
<label class="ease-checkbox">
  <input type="checkbox" class="ease-checkbox-input">
  <div class="ease-checkbox-box">
    <svg class="ease-checkbox-checkmark" viewBox="0 0 24 24">
      <path class="ease-checkbox-path" d="M20 6L9 17L4 12"></path>
    </svg>
  </div>
  <span class="ease-checkbox-label">Label text</span>
</label>

Modifier Classes

Class Suffix Target Element Description
.ease-checkbox Parent Label Base container. Handles inline layout alignments and typography.
.ease-checkbox-input Input Element Accessible hidden checkbox toggle layer. Handles states.
.ease-checkbox-box Box Core Custom square element. Bounces scale and fills color on active check.
.ease-checkbox-path Check SVG Path Checkmark outline path. Transitions stroke-dashoffset to draw check.
.ease-checkbox--sm Parent Label Shrinks dimensions to 18px size and limits padding heights.
.ease-checkbox--lg Parent Label Enlarges dimensions to 30px size and inflates labels.
.ease-checkbox--success Parent Label Overrides accent toggled backgrounds and borders to green.
.ease-checkbox--danger Parent Label Overrides accent toggled backgrounds and borders to red.
.ease-checkbox--warning Parent Label Overrides accent toggled backgrounds and borders to amber.

Customization Variables

CSS Variable Name Default Value Description
