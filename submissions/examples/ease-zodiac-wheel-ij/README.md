# Ease Zodiac Wheel

A zodiac wheel with rotating segments, featuring astrological symbols in a circular layout.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.zodiac-wheel` element.

## Custom CSS Variables

| Variable            | Default    | Description                       |
|---------------------|------------|-----------------------------------|
| `--wheel-bg`        | `#1a1a2e`  | Background color of the wheel     |
| `--sign-color`      | `#4ecdc4`  | Color of the zodiac signs         |
| `--rotate-duration` | `12s`      | Duration of one full rotation     |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
