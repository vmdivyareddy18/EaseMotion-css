# Copy Clipboard

## What does this do?
Copy Clipboard component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"cc-primary\">Copy Clipboard Primary</div>
```

## Why is it useful?
Provides a reusable copy clipboard UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
