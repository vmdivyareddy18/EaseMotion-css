ease-focus-ring

Use Tab to navigate between elements and see the focus ring

Tab through the elements above to see each focus ring variant.
.ease-focus-ring uses box-shadow instead of outline, so it respects border-radius and works on any element.

Usage

      
<!-- Primary Ring --!>
<button class="ease-focus-ring">Primary Ring</button>
<!-- Success Ring --!>
<button class="ease-focus-ring ease-focus-ring-success">Success Ring</button>
<!-- Danger Ring --!>
<button class="ease-focus-ring ease-focus-ring-danger">Danger Ring</button>
<!-- Info Ring --!>
<button class="ease-focus-ring ease-focus-ring-info">Info Ring</button>
<!-- Thick Ring --!>
<button class="ease-focus-ring ease-focus-ring-thick">Thick Ring</button>
<!-- Works on inputs too --!>
<input class="ease-focus-ring" type="text" placeholder="Focus me">
