# Cursor Glow Gauge

## Description

The Cursor Glow Gauge is a reusable UI component inspired by modern neon design patterns. It features a glowing cursor effect with a clean gauge layout, making it suitable for dashboards, analytics pages, and interactive interfaces. The component is built using only HTML and CSS and is designed to be lightweight, responsive, and easy to customize.

## Features

- Neon-inspired cursor glow effect
- Pure HTML and CSS
- Responsive design
- Accessible and easy to use
- Reusable for different UI layouts

## File Structure

```
cursor-glow-gauge-sharanya/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern web browser.
2. The component styles are defined in `style.css`.
3. Customize colors, size, and glow intensity as needed.

## Why EaseMotion CSS?

This component follows the EaseMotion CSS philosophy by providing a reusable, animation-focused UI element with clean HTML structure and lightweight CSS. It can be easily integrated into modern web projects without requiring JavaScript.
