# Animated Tabs (`ease-tabs-cv`)

A lightweight animated tabs component built with **HTML5** and **Pure CSS**.

## Features

- Animated underline
- Hover transitions
- Pill tab style
- Multiple visual themes
- Responsive layout
- No JavaScript
- No external libraries

## Folder Structure

```
ease-tabs-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Notes

This example focuses on animated tab navigation styles. Since the project requires **Pure CSS with no JavaScript**, the demo showcases static active states and hover animations rather than interactive tab switching.
