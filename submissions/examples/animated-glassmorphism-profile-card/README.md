# Animated Glassmorphism Profile Card

A stunning, modern, and fully responsive **Glassmorphism User Profile Card** perfect for social platforms, portfolios, and team pages. This example demonstrates how to combine **EaseMotion CSS** for bouncy avatar entrances, staggered text reveals, and smooth hover interactions to create a premium, frosted-glass UI component.

## 🚀 Features

- **Glassmorphism Design**: Uses `backdrop-filter: blur()` and semi-transparent backgrounds to create a beautiful frosted glass effect over a vibrant gradient.
- **Bouncy Avatar Entrance**: The profile picture uses `ease-bounce-in` to playfully pop into view, overlapping the cover photo.
- **Staggered Content Reveal**: The name, handle, bio, and stats fade in sequentially using `ease-fade-in-up` and `ease-delay-*` for a premium cascading load effect.
- **Interactive Follow Button**: Toggles between "Follow" and "Following" states with a smooth color and shadow transition, utilizing `ease-hover-grow`.
- **Animated Social Icons**: Social links fade in with staggered delays and scale up on hover for tactile feedback.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing and spacing.
