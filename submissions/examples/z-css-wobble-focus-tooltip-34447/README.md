# CSS Wobble Focus Tooltip

Pure CSS animated tooltip utilizing a Wobble-Focus interaction designed for Modern SaaS layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Wobble animation
- Modern SaaS styling
- CSS custom properties
- Glassmorphism cards
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--tooltip
--duration
--ease
```
