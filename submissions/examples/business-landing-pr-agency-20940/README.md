# Business Landing: PR Agency (`business-landing-pr-agency-20940`)

This proposal provides a complete, copy-paste ready landing page template designed for a Public Relations Firm, Communications Agency, or Media Strategy Consultancy, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A high-end PR agency website must convey authority, sophistication, and editorial polish. This template achieves an "Editorial Authority" aesthetic using a stark Black and White color palette (`#111827` / `#FFFFFF`) punctuated by a single, aggressive Editorial Red accent (`#D32F2F`). It pairs a bold, authoritative serif font (`Cinzel`) for display text with a clean, modern geometric sans-serif (`Montserrat`) for body copy.

The layout includes:
- A stark, confident navigation bar.
- A Hero section featuring a massive, subtle typography background graphic and an infinite-scrolling marquee of media coverage logos.
- An "Expertise" grid with bold, oversized numbering and sharp hover elevations.
- A "Recent Placements" section showcasing custom CSS-constructed newspaper/article layout mockups with dynamic scale-and-shadow hover states.
- A "Leadership Team" section featuring grayscale CSS-abstracted portraits that reveal color and social links upon hover.
- A "Client Success Stories" list utilizing bold typography and hover-triggered arrow animations.
- A high-contrast retainer inquiry form.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use extremely sharp, decisive `cubic-bezier(0.86, 0, 0.07, 1)` timings tailored for an authoritative, high-stakes brand:
- `.ease-slide-up-stagger` (Configured for a fast, sharp vertical entrance with zero bounce).
- `.ease-marquee-fast` (Drives the infinite scrolling logo bar in the hero section).
- `.ease-hover-lift` (Provides a sharp, immediate elevation effect on expertise cards).
- `.ease-hover-scale-shadow` (Applied to the article mockups to create a commanding "pop" effect when hovered).
- *Custom Transitions*: The template relies heavily on group hover effects (`group-hover-translate-y-0`, `group-hover-filter-none`, `group-hover-transform-translate-x-10`) combined with sharp transition timings to reveal data and animate arrows precisely.

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the bold typography and stark color palette to the pure CSS illustrations (the article mockups and team portraits). The standard `ease-*` interaction classes are attached directly to HTML elements to provide sharp, decisive animations. No JavaScript is required for this template.
