# Animated Floating Action Button Menu

A modern Floating Action Button (FAB) menu with smooth expand and collapse animations.

## Features

- Expandable action menu
- Smooth animations
- Hover interactions
- Mobile-friendly design
- Pure CSS implementation

## Folder Structure

```text
r-floating-action-menu/
├── README.md
├── demo.html
└── style.css
```

## Usage

Open `demo.html` in any modern browser.

- Floating action buttons
