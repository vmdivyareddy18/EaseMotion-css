Hardware Acceleration Test

Scroll up and down to test the backdrop-filter on the sticky header. The glass effect should remain perfectly smooth without flickering, flashing, or repainting artifacts on Safari (macOS/iOS) and Firefox.

Layer Promotion Verified

By utilizing backface-visibility: hidden alongside transform: translateZ(0) and a defined perspective, we force the browser engine to composite the header entirely on its own GPU layer, isolating it from the main thread layout computations.

High-Diffusion Rendering

Keep scrolling so this text passes beneath the header. The 12px blur radius should diffuse the text edges smoothly. If the stabilization patches were missing, Safari would drop frames or temporarily remove the blur during the scroll phase.

Performance Metrics

If you open the Safari Web Inspector and record a rendering timeline, you should observe a steady 60fps (or 120fps on ProMotion displays) with zero main-thread layout thrashing while scrolling.
