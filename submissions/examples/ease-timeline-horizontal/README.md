# Horizontal Timeline

An animated horizontal timeline with staggered entry — pure CSS.

## Features
- Horizontal connector line via `::before`
- Staggered fade-up animation per item
- Dot highlight on hover
- Pure CSS, no JavaScript

## Usage
```html
<div class="timeline">
  <div class="tl-item"><div class="tl-dot"></div><div class="tl-content"><h3>Step</h3><p>Desc</p></div></div>
  <!-- more items with nth-child delays -->
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
