Technical Specifications & Usage

💡 Container Queries

Unlike normal media queries which monitor the whole browser window, this component uses CSS Container Queries. Styling scales and snaps based directly on the width of the simulated parent frame.

.viewport-container {
  container-type: inline-size;
}

@container (max-width: 768px) {
  .preview-grid {
    grid-template-columns: 1fr;
  }
}

🎨 CSS Variable Breakpoints

Easily customizable CSS breakpoint variables mapped for clean and maintainable UI design system configuration.

:root {
  --bp-mobile: 375px;
  --bp-tablet: 768px;
  --bp-laptop: 1024px;
  --bp-desktop: 1240px;
}
