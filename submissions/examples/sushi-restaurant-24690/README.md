# Sushi Restaurant Template
A clean, minimalist template designed for sushi restaurants, omakase bars, and contemporary Japanese cuisine.

- **Design**: Uses a "Zen Minimalism" palette with deep ink-black backgrounds, vibrant wasabi-green accents, and crisp paper-white typography to emphasize clarity and artisan focus.
- **Use Case**: Sushi bars, omakase-style dining, and modern Japanese restaurants.
