Copy-ready snippets

Sticky promo

<div class="ease-announce-bar is-fixed is-warning" role="banner">
  <div class="ease-announce-bar-content">
    <strong>Sale:</strong> 30% off — <a href="#">Shop now</a>
  </div>
</div>

Dismissible slide-down

<input type="checkbox" id="bar-dismiss" class="ease-announce-bar-dismiss" />
<div class="ease-announce-bar is-info">
  <div class="ease-announce-bar-content">Your message here</div>
  <label for="bar-dismiss" class="ease-announce-bar-close" aria-label="Dismiss">×</label>
</div>

Rotating messages

<div class="ease-announce-bar is-success" role="status" aria-live="polite">
  <div class="ease-announce-bar-content">
    <span id="msg">Message 1</span>
  </div>
</div>
<!-- Cycle messages with JS: messages[i++] -->

Sale countdown

<div class="ease-announce-bar is-danger" role="timer">
  <div class="ease-announce-bar-content">
    <strong>Sale ends in</strong> <span id="countdown">00:00:00</span>
  </div>
</div>

Cookie consent

<div class="ease-announce-bar ab-cookie-bar" role="dialog" aria-label="Cookie consent">
  <span>We use cookies. <a href="#">Learn more</a></span>
  <button type="button">Accept</button>
  <button type="button">Decline</button>
</div>

Pattern guidance

Component classes

  • .ease-announce-bar — base bar with slide-down entrance
  • .ease-announce-bar-content — message area
  • .ease-announce-bar-close — dismiss label/button
  • .is-fixed — sticky top positioning

Variant modifiers

  • .is-info — primary purple (default)
  • .is-success — green confirmations
  • .is-warning — amber promos
  • .is-danger — red urgency / countdown

Accessibility tips

  • Use role="banner" for site-wide promos.
  • Add aria-label="Dismiss" on close controls.
  • Rotating text needs aria-live="polite".
  • Limit simultaneous bars — one announce strip per viewport.
