# Orbit Rotation Animation

A lightweight orbit rotation animation utility built using pure CSS.

## Features

- Continuous orbital motion
- Smooth infinite rotation
- Ambient decorative animation
- Adjustable rotation speed
- No JavaScript required

## Usage

```html
<div class="orbit-container">
  <div class="center-dot"></div>
  <div class="orbit-item"></div>
</div>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable ambient motion effect
