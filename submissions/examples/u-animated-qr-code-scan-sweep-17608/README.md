# Animated QR Code Scan Sweep

## Overview

Animated QR Code Scan Sweep is a lightweight CSS animation that simulates QR code scanning using a moving scan line.

## Features

- CSS-only animation
- Moving scan line
- QR scanner effect
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="qr-scan">
  <div class="qr-code"></div>
  <div class="scan-line"></div>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
