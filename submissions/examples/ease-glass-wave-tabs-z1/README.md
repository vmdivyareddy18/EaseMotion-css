# Ease Glass Wave Tabs

## Overview

A premium tab navigation featuring animated wave fills, glassmorphism styling, and smooth hover transitions using only HTML and CSS.

## Features

- Animated wave fill
- Glassmorphism tabs
- Floating hover effect
- Gradient highlights
- Responsive layout
- Zero JavaScript
