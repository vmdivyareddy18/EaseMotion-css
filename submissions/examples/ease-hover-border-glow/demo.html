ease-hover-border-glow

Hover any element below — its border lights up with a soft, colorful glow that spreads and blurs outward, while the rest of the element stays untouched.

Cards — Custom Glow Colors

💜

Violet Glow

Default --ease-border-glow-color, set to a soft violet.

💚

Teal Glow

Override the color variable per-instance, nothing else changes.

💗

Rose Glow

Same class, same transition — only the hue is different.

Buttons

Inputs

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Add the class to a button, card, or input */
<button class="ease-hover-border-glow">Hover me</button>
<input class="ease-hover-border-glow" type="text" />

/* 3. Customise the glow color per-instance */
.my-card {
  --ease-border-glow-color: #ff4d8d;     /* rose */
  --ease-border-glow-duration: 0.45s;   /* slower fade-in */
}
