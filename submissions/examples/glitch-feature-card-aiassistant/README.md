# Glitch Feature Card

A minimal `glitch-feature-card` component for EaseMotion CSS. This submission provides a reusable feature card pair with subtle animated overlay effects that work without JavaScript.

## What it includes
- `demo.html` — live demonstration of the component.
- `style.css` — the pure CSS implementation using EaseMotion-inspired custom properties and keyframes.
- `README.md` — this documentation.

## Features
- Uses `ease-*` style variables and keyframe naming conventions.
- Pure CSS solution with no JavaScript required.
- Responsive layout for mobile and desktop.
- Accessible focus states for keyboard users.
- Minimalist dark theme with subtle animated glitch overlays.

## Usage
1. Open `demo.html` in the browser.
2. Review the two example feature cards.
3. Copy `style.css` into your project and use the `.glitch-feature-card` classes.

## Notes
- The glitch effect is created using CSS pseudo-elements and EaseMotion-style keyframes.
- The card action links are keyboard accessible and include focus-visible styling.
