# Screen Shake

## What does this do?
A screen/element shake effect simulating impact, explosion, or error feedback with rapid oscillation on the X axis.

## How is it used?
Apply the `.screen-shake` class to trigger a shaking animation. The `.intense` modifier creates a stronger shake for dramatic effects.

## Why is this useful?
Provides impactful feedback for errors, explosions, collisions, or warning states in games, forms, and interactive applications.
