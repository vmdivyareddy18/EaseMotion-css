# Minimalist Tech Slide-Right Accordion

A modern **Pure CSS Slide-Right Accordion** designed for **Minimalist Tech** layouts. The component features a clean interface, smooth slide-right interaction, responsive design, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Smooth Slide-Right interaction
- Minimalist tech-inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and reusable

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.4s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --primary:#2563eb;

    --radius:14px;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Border radius
- Shadows
- Background colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
