# CSS Elastic Slide Tooltip - Glassmorphism UI Layouts

A pure CSS animated tooltip featuring an elastic slide transition with a modern glassmorphism visual style.

## Features

- Pure CSS implementation
- No JavaScript required
- Glassmorphism UI appearance
- Smooth elastic slide animation
- Responsive design
- Keyboard accessible
- Custom CSS animation variables

## Usage

Add the tooltip structure:

```html
<div class="tooltip-wrapper">

  <button class="tooltip-button">
    Hover Me
  </button>

  <div class="tooltip">
    Tooltip content
  </div>

</div>
