# Counter Reset and Counter Increment Demo

### What does this do?

Demonstrates how CSS `counter-reset` and `counter-increment` can be used to automatically number elements without manual numbering.

### How is it used?

Apply counters to lists, cards, sections, or steps.

Example:

```html
<ol class="steps">
    <li>Install EaseMotion CSS</li>
    <li>Import the stylesheet</li>
</ol>

### Why is it useful?

CSS counters help create maintainable, automatically numbered UI elements while keeping HTML clean and easy to update.
