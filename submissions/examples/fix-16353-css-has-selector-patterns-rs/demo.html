CSS `:has()` Utilities

The "Parent Selector" is finally here. Style parent elements based on their descendants or state, without writing a single line of JavaScript.

1. Form Validation State Hoisting

The entire `div.ease-form-group` highlights red because the `input` inside it has `:invalid` state.

2. Conditional Component Styling

The first card gets a glowing amber border because it `:has(.ease-badge-featured)`. The second card loses its padding because it `:has(img)`.

Special Item

This card dynamically styled its own border based on the badge.

Image Card

Because an image is present, the parent card removed its padding automatically.

3. Global Sibling Dimming on Hover

Hover over any item. The parent `.ease-has-list` detects the hover, and tells all items that are *not* hovered to dim themselves.
