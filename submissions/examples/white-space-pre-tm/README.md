# White Space Pre

white-space: pre — Preserves whitespace and line breaks in text

## Features
- Uses `--ease-font-mono`
- Uses `--ease-color-surface`
- Uses `--ease-radius-md`
- Uses `--ease-color-muted`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">White Space Pre</span>
</div>
```

## Why is it useful?

This submission demonstrates the white space pre CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
