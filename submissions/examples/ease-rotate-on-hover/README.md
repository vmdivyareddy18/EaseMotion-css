# Rotate on Hover – Icon spins on interaction

> A production-ready rotation component that spins icons smoothly on hover with customizable angles, direction control, and multiple animation styles. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Smooth rotation** – Icons spin on hover
- ✅ **Multiple angles** – 90°, 180°, 360°, custom
- ✅ **Direction control** – Clockwise / Counter-clockwise
- ✅ **Animation styles** – Bounce, elastic, slow, fast
- ✅ **Toggle rotation** – Enable/disable effect
- ✅ **Reset all** – Reset all icons to default
- ✅ **Keyboard shortcuts** – Ctrl+R (toggle), Ctrl+D (direction), Escape (reset)
- ✅ **6 variants** – Different rotation styles
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
