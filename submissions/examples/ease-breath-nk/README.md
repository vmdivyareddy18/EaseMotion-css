# ease-breath Animation

## Description

A very subtle breathing animation that slowly expands and contracts an element.

The animation follows an inhale–exhale cycle:

```css
scale(1) → scale(1.06) → scale(1)
```

Duration:

```css
4s ease-in-out infinite
```

## Usage

```html
<div class="ease-breath"></div>
```

## Use Cases

- Meditation apps
- Wellness dashboards
- Ambient UI indicators
- Focus timers
- Calming interface elements

## Accessibility

Includes support for:

```css
prefers-reduced-motion
```

to disable animation when users request reduced motion.

## Why EaseMotion CSS?

The animation demonstrates a lightweight motion pattern that creates a calm and natural feeling without distracting users.
