# Easings Comparison

A side-by-side visual comparison of CSS easing functions — ease, ease-in, ease-out, ease-in-out, linear, and a spring simulation. Each ball animates across its track with the corresponding easing, making the differences in acceleration and deceleration immediately apparent.

## Features

- Six parallel tracks, each labeled with the easing function name.
- Spring easing simulated via a secondary `@keyframes` for squish-and-stretch effect.
- Reset button to restart all animations simultaneously.
- Responsive layout that adapts to mobile screens.

## Topics Covered

- `animation-timing-function` property
- `cubic-bezier()` vs named easings
- Combining multiple `@keyframes` on one element
- `prefers-reduced-motion` fallback
