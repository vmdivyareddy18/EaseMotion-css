Background Clip

Determines how far the background extends within an element's box model.

CSS Property: background-clip

Variants Demo

background-clip: border-box Background Clip variant
background-clip: padding-box Background Clip variant
background-clip: content-box Background Clip variant
background-clip: text Background Clip variant
border-box
Background to border edge
padding-box
Background to padding edge
content-box
Background to content only
text (webkit)
Gradient text clip

Usage

<div class="background-clip">Content here</div>
