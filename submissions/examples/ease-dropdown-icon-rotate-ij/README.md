# Icon Rotate Dropdown

A dropdown component with a chevron icon that rotates smoothly on toggle.

## Features

- Chevron icon rotates 180° on open/close
- Clean, minimal design
- Supports multiple independent dropdown instances

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--dd-rotate-duration` | `0.3s` | Rotation animation duration |
| `--dd-icon-color` | `inherit` | Chevron icon color |
| `--dd-bg` | `#ffffff` | Background color |
| `--dd-text-color` | `#1f2937` | Text color |
| `--dd-border` | `#d1d5db` | Border color |
| `--dd-radius` | `8px` | Border radius |
