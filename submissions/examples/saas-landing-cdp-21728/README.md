# Business Landing — Customer Data Platform SaaS (`saas-landing-cdp-21728`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for MarTech and Customer Data Platforms (CDP), using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A CDP SaaS landing page needs to feel data-driven, unified, and powerful (similar to Segment, mParticle, or RudderStack). This template uses a clean, modern aesthetic with a trustworthy emerald green color palette.

The layout includes:
- A hero section featuring an animated CSS-built unified customer profile mockup
- A logo banner for enterprise trust
- A 3-column platform feature grid (identity resolution, segmentation, activation)
- A split layout showcasing integration ecosystems
- Customer testimonials from marketing/data teams
- A 2-tier pricing layout based on MTUs (Monthly Tracked Users)
- A high-impact dark CTA section

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth, authoritative text and card entrances)
- `.ease-slide-right` / `.ease-slide-left` / `.ease-slide-up` (For layout structural animations)
- `.ease-zoom-in` / `.ease-bounce-in` (For drawing attention to primary CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For buttons, feature cards, and mockups)
- `.ease-hover-glow` (For subtle link interactions)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
