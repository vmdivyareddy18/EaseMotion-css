# Animated Bento Grid / Feature Showcase

A sleek, modern, and fully responsive **Bento Grid Component** perfect for showcasing product features on SaaS landing pages. This example demonstrates how to combine **EaseMotion CSS** for staggered card entrances, hover lift effects, and continuous floating/pulsing elements inside the cards to create a stunning, dynamic feature section.

## 🚀 Features

- **Asymmetrical Grid Layout**: Uses CSS Grid to create a visually interesting, magazine-style layout with large, wide, and tall cards.
- **Staggered Card Entrances**: Each card fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a premium cascading load effect.
- **Hover Lift & Glow**: Cards smoothly change border color and reveal a subtle radial glow on hover using `ease-hover-grow` and CSS transitions.
- **Floating Elements**: Icons and avatars inside the cards use `ease-float` to create a lively, continuous movement.
- **Pulsing Accents**: Security shields and map dots use `ease-pulse` to draw attention to key features.
- **Animated Bar Chart**: The analytics card features bars that grow from zero using custom CSS keyframes.
- **Dark Mode Aesthetic**: Sleek dark theme with subtle gradients and glowing borders.
- **Fully Responsive**: Gracefully collapses from a 4-column asymmetrical grid to a 2-column layout, and finally to a single column on mobile.
