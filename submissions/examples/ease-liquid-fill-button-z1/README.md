# Ease Liquid Fill Button

## Overview

This example demonstrates a reusable CSS-only liquid fill button with a rotating wave animation that rises on hover.

## Features

- Pure HTML and CSS
- Liquid hover animation
- Multiple color themes
- Responsive layout
- Reusable utility
- No JavaScript

## Folder Structure

```text
ease-liquid-fill-button-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

1. Open the project folder.
2. Launch `demo.html`.
3. Customize button labels and colors.

## Browser Support

Compatible with modern browsers supporting CSS transforms and animations.

## Why This Example

This utility provides an engaging hover interaction for call-to-action buttons while remaining lightweight and easy to reuse across different interfaces.
