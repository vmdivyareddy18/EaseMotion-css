# Interactive Button Hover Animations

## Overview

This contribution contains a collection of lightweight CSS-only button hover animations.

The examples demonstrate:

- Icon movement without changing layout
- Border drawing animation
- Smooth gradient transitions
- Subtle neon glow effects


## Usage

Copy the required HTML structure and CSS styles into your project.

These animations work without JavaScript or external dependencies.


## Design Philosophy

These animations follow EaseMotion CSS principles:

- Lightweight CSS transitions
- Smooth user interactions
- Minimal visual distraction
- Easy customization
- Accessibility support through reduced-motion handling


## Included Examples

### Icon Glide Button

Moves the icon slightly on hover while keeping text unchanged.


### Border Draw Button

Creates a smooth border reveal effect.


### Animated Gradient Button

Transitions between modern gradient colors.


### Neon Glow Button

Adds a subtle glow effect for emphasis.
