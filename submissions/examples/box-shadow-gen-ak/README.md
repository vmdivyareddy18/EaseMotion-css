# Box Shadow Generator

## What does this do?
An interactive box-shadow generator with sliders for offset, blur, spread, color, opacity, and inset toggle, plus copyable CSS output.

## How is it used?
Open demo.html in a browser. Adjust any slider or toggle to see the shadow update in real-time on the preview box. Click "Copy CSS" to copy the generated box-shadow declaration.

## Why is this useful?
This provides a visual tool for quickly prototyping CSS box-shadow values without manual trial and error. It demonstrates real-time CSS property manipulation and color conversion.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to generate shadows.
