# Project Management Dashboard

A modern and responsive Project Management Dashboard built using HTML, CSS, and JavaScript. This example demonstrates how multiple EaseMotion CSS-style animations and UI patterns can be combined to create a realistic productivity application interface.

## Features

### Dashboard Layout

* Responsive sidebar navigation
* Top navigation bar
* Statistics overview cards
* Project summary widgets
* Team member section
* Activity timeline
* Notification center

### Project Management Components

* Project status cards
* Progress tracking
* Task counters
* Priority indicators
* Deadline tracking
* Team assignments

### Interactive Elements

* Hover card animations
* Animated progress bars
* Status indicators
* Notification alerts
* Interactive buttons
* Responsive navigation

### Accessibility

* Semantic HTML structure
* Keyboard-friendly controls
* Responsive design
* Clear visual hierarchy
