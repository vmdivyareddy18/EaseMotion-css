# ease-loader-spinner

A classic circular loading spinner using the CSS border trick.

## Usage

```html
<div class="ease-loader-spinner"></div>
```

## CSS Variables

| Variable | Default | Description |
|---|---|---|
| `--ease-loader-size` | `48px` | Size of the spinner |
| `--ease-loader-color` | `#6366f1` | Color of the spinner |
| `--ease-loader-speed` | `0.8s` | Animation duration |
