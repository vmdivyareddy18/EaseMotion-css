# Ease Cell Collapse Expand

A CSS-only component that smoothly expands and collapses table cell content using checkboxes, CSS variables, and `@keyframes`.

## Features

- Pure HTML & CSS
- No JavaScript
- Smooth expand/collapse animation
- Uses CSS variables
- Responsive layout
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## How to Use

1. Open `demo.html`.
2. Click any table section.
3. The content expands and collapses smoothly.

## Technologies

- HTML5
- CSS3
- CSS Variables
- @keyframes
