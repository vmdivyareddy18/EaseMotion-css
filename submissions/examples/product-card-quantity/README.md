# Product Card with Quantity Selector

A product card with a gradient image placeholder, category label, title, price, a quantity selector with +/- buttons, and an Add to Cart button. The quantity is clamped between 1 and 99.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Use the + and − buttons to adjust the quantity.

## Accessibility notes

Quantity buttons are `<button>` elements. The Add to Cart button uses a high-contrast indigo fill.
