# Cookie Consent Banner

## What does this do?
CSS-styled cookie consent banners with bottom, top, modal, and floating card layouts with preference toggles.

## How is it used?
```html
<div class="ck-banner ck-bottom"><div class="ck-content"><div class="ck-text"><strong>Title</strong><p>Description</p></div><div class="ck-actions"><button class="ck-btn ck-accept">Accept</button></div></div></div>
```

## Why is it useful?
Demonstrates multiple cookie consent UI patterns. Glassmorphism styling. Preference checkboxes, accept/decline actions, and modal variant. Customizable via CSS variables.
