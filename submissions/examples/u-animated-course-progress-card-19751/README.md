# Animated Course Progress Card

A modern educational progress tracking component built using HTML and CSS.

## Features

- Animated course progress bar
- Completion percentage display
- Lessons and modules statistics
- Study streak tracker
- Achievement section
- Responsive design

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- Learning platforms
- LMS dashboards
- Online courses
- Student portals
- Educational applications
