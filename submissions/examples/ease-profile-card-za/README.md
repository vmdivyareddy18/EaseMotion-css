# CSS Profile Card Component

## What does this do?
A complete user profile card with cover photo, avatar with online status indicator, user details, stats counters, skills tags, social media links, and action buttons. Includes a secondary card variant.

## How is it used?
```html
<div class="pr-card">
  <div class="pr-card-body">
    <div class="pr-avatar-section">
      <div class="pr-avatar">
        <div class="pr-avatar-img">JD</div>
      </div>
    </div>
    <h1 class="pr-name">John Doe</h1>
    <p class="pr-title">Developer</p>
    <div class="pr-actions">
      <button class="pr-btn pr-btn-primary">Follow</button>
    </div>
  </div>
</div>
```

## Why is it useful?
Creates a professional profile card for user directories, team pages, and social platforms. Features a gradient cover photo with edit overlay, avatar with online status indicator, user details with name title and bio, location with email link, statistics counters with dividers, skill tags with accent colors, social profile links with hover effects, and action buttons for follow message and more. Includes a second card variant with different styling. Fully responsive with CSS custom property theming.
