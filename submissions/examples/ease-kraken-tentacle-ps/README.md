# Ease Kraken Tentacle Animation

1. What does this do? Creates a dramatic grabbing animation where an element rises from below, grips tightly, and then pulls violently down off-screen.
2. How is it used? Apply `.ease-kraken-tentacle-ps` for an immediate pull animation, or `.ease-kraken-tentacle-hover-ps` to trigger the sequence on hover. 
3. Why is it useful? It is an extremely cinematic exit animation, perfect for dismissing notifications, deleting items from a list, or thematic storytelling in a UI where things need to "disappear into the depths".
