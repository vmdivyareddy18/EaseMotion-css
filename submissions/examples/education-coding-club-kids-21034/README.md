# Education: Coding Club for Kids (`education-coding-club-kids-21034`)

This proposal provides a complete, copy-paste ready landing page template designed for a youth coding education program or after-school club, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A Coding Club for Kids brand requires a UI that feels highly energetic, colorful, fun, and tech-forward. This template achieves this aesthetic using a vibrant, high-contrast color palette with Bright Purple (`#5D5FEF`), Bright Pink (`#EF5DA8`), Bright Yellow (`#FFD166`), and Green (`#06D6A0`). It utilizes a chunky, playful display font (`Fredoka One`) paired with a highly readable body font (`Nunito`) and a monospace font (`Fira Code`) for coding elements. 

The layout includes:
- A playful Navigation with a CSS curly-brace logo.
- A Hero section featuring a CSS-animated "laptop screen" mockup simulating a coding environment with a typewriter effect, flanked by floating game sprites.
- A "Why Coding" features grid with spinning emoji icons.
- A Programs by Age grid utilizing colorful headers and abstract CSS icons (blocks, python snake, web layout).
- A dark-mode Student Project Gallery grid with hover-scaling cards.
- A Parent Testimonials section.
- A highly visible Pricing tier grid highlighting the "Best Value" option.
- A bold, high-contrast CTA section.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use extremely bouncy, jelly-like `cubic-bezier(0.68, -0.55, 0.265, 1.55)` and `cubic-bezier(0.175, 0.885, 0.32, 1.275)` timings tailored for a fun, child-friendly feel:
- `.ease-bounce-in` / `.ease-pop-in` (For extremely playful, springy component reveals).
- `.ease-float-fast` (Applied to background game sprites for a fast, dynamic hovering effect).
- `.ease-wiggle` (Applied to the sticker badges to draw attention playfully).
- `.ease-hover-jelly` (A very squishy, jelly-like hover interaction for primary buttons).
- `.ease-hover-spin` (Applied to feature icons to make them spin on hover).
- `.ease-typewriter` (A custom CSS animation simulating code being typed on the laptop screen).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the vibrant color palette to the chunky button styles (with hard shadows) and the abstract CSS illustrations (laptop, coding blocks, game sprites). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the precise, energetic animations. No JavaScript is required for this template.
