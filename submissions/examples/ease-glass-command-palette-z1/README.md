# Ease Glass Command Palette

## Overview

A premium command palette interface inspired by modern developer tools, featuring glassmorphism styling and animated command rows built entirely with HTML and CSS.

## Features

- Glass command palette
- Animated command items
- Gradient action icons
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

