# Ease Gradient Signal Loader

A lightweight loading animation built with pure HTML and CSS, featuring animated gradient signal bars.

## Features

- Pure HTML & CSS
- Gradient signal animation
- Responsive design
- Zero dependencies
- Easy to customize

## Structure

```
ease-gradient-signal-loader/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in a browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why EaseMotion CSS

A reusable loading animation that showcases smooth keyframe motion in a minimal, animation-focused example.
