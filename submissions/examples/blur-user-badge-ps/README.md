# Blur User Badge (#42055)

### 1. What does this do?
This adds a beautiful, modern user badge variant that features a soft-entry cinematic blur effect when loaded, complete with responsive micro-interactions on wrapper elements.

### 2. How is it used?
Developers structure the element inside their container components using the base badge and theme classes:

```html
<div class="profile-card">
    <span class="user-badge badge-admin">Admin</span>
</div>
