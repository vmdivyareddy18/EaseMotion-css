# Serval

A pure CSS and HTML serval stalking through savanna grass, huge ears swivelling for sound.

## What it shows

- Spotted coat from nine stacked hard stop radial gradients sharing the body silhouette
- Ear angles parked in custom properties so one swivel keyframe can rebuild each rotation
- A banded tail from a repeating linear gradient, flicking on its own cycle
- Far legs dimmed with filter: brightness to sit behind the body
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
