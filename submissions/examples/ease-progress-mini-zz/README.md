# Mini Progress

## What does this do?
Mini Progress component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"pm-primary\">Mini Progress Primary</div>
```

## Why is it useful?
Provides a reusable mini progress UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
