# Creative Sticker E‑Commerce Website

This demo site showcases a fun, colorful, creative sticker shop aesthetic using **EaseMotion CSS classes** only.

## 🧩 Sections
- Hero with colorful sticker imagery
- Products by theme/style
- Artist spotlight section
- Bundle packs
- Custom order CTA
- Reviews
- Newsletter signup
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — entrance fade animations
- `ease-slide` — sliding entrance for headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs, artist card, and buttons
- `ease-hover-opacity-reduce` — hover dimming for product cards, bundles, and reviews

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll through sections to see entrance animations.
3. Hover over buttons, cards, and testimonials to see interactive effects.
4. Replace placeholder images and text with real sticker e‑commerce content.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion animation classes
