# Wheel Prize — Spinning Prize Wheel

CSS prize wheel built with `conic-gradient` for the colored segments. The `spin-wheel` keyframe creates a realistic deceleration with an overshoot bounce.

## Files

- `demo.html` — wheel, marker, spin button
- `style.css` — conic-gradient segments, spin keyframe, inner hub

## Usage

Open `demo.html` and click "Spin Again" to re-trigger the wheel animation.
