# Text Decoration Utilities

A lightweight collection of CSS utility classes for applying common text decoration styles. These utilities provide a simple and readable way to control text decoration directly through class names.

## Features

* Pure CSS implementation
* Human-readable utility classes
* Lightweight and reusable
* Cross-browser compatible
* No JavaScript required

## Available Classes

| Class | Effect |
| ------- | ------- |
| `ease-underline` | Underlines text |
| `ease-no-underline` | Removes text decoration |
| `ease-line-through` | Adds a line through text |

## Usage

```html
<p class="ease-underline">Underlined text</p>

<p class="ease-no-underline">Normal text</p>

<p class="ease-line-through">Strikethrough text</p>
