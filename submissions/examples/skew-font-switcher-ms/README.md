# Animated Font Switcher (Skew Style)

Resolves #42432.

### What does this do?
Adds a Skew Font Switcher component to EaseMotion CSS within a Neumorphic design environment. It utilizes hidden radio buttons combined with the modern CSS `:has()` pseudo-class to change the typography of the content area. The buttons feature the `ease-skew-right` animation on hover for a dynamic interaction.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Click the typography labels to instantly switch the font family of the target content without JavaScript.

### Why is it useful?
Provides an accessible, interactive reader settings panel ideal for blogs, documentation sites, or any reading app, leveraging modern CSS to handle state management cleanly.
