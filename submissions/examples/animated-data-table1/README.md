# Animated Data Table / Data Grid

A clean, modern, and fully responsive **Data Table Component** perfect for admin panels, dashboards, and SaaS applications. This example demonstrates how to combine **EaseMotion CSS** for staggered row entrances, smooth hover transitions, and bouncy checkbox selections to create a polished, interactive data display.

## 🚀 Features

- **Staggered Row Entrances**: Each table row fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a smooth cascading load effect.
- **Smooth Row Hover**: Rows smoothly change background color on hover for clear visual tracking.
- **Animated Sort Icons**: Clickable column headers rotate their sort icons (↕ to ↑/↓) when sorting is applied.
- **Bouncy Checkbox Selection**: Checkboxes use custom styling and provide clear visual feedback when selected.
- **Status Badges**: Color-coded pills for Active (green), Pending (amber), and Inactive (gray) states.
- **Interactive Action Buttons**: Edit and delete buttons use `ease-hover-grow` with distinct hover colors.
- **Pagination Controls**: Clean pagination footer with active state highlighting.
- **Fully Responsive**: On mobile devices, the table transforms into a stacked card layout for better readability.
