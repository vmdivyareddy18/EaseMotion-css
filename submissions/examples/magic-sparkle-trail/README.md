# Magic Sparkle Trail Animation

## What does this do?
A glowing orb that moves along a path leaving behind sparkling particles in a trail effect.

## How is it used?
Add the `.trail-container` with `.orb` and `.spark` elements.

## Why is it useful?
Magic trail effects add elegance to spell animations, enchantments, and fantasy UIs.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
