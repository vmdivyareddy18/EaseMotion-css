# NFT Card Gallery

An NFT gallery card with 3D tilt hover animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--card-bg` | `#2d2d44` | Card background |
| `--accent-color` | `#f72585` | ETH price accent |
| `--text-color` | `#ccc` | Text color |
| `--tilt-duration` | `0.35s` | Hover tilt speed |
