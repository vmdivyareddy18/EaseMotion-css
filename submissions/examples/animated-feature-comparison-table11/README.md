# Animated Feature Comparison Table

A clean, modern, and fully responsive **Feature Comparison Table** perfect for SaaS pricing pages, product feature matrices, and decision-making tools. This example demonstrates how to combine **EaseMotion CSS** for staggered row entrances, smooth hover transitions, bouncy checkmarks for included features, and interactive plan highlighting to create a polished, engaging comparison experience.

## 🚀 Features

- **Staggered Row Entrances**: Each feature row fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a cascading load effect.
- **Highlighted "Most Popular" Plan**: The Professional plan is visually distinguished with a scale transform, enhanced shadow, and a bouncy "Most Popular" badge using `ease-bounce-in`.
- **Bouncy Checkmarks**: Included features use `ease-bounce-in` to playfully pop into view with a subtle bouncing animation.
- **Interactive Hover States**: Feature cells smoothly change background color on hover for clear visual feedback.
- **Plan Button Variations**: Primary and secondary button styles for different call-to-action priorities.
- **Three-Tier Plan Structure**: Starter, Professional, and Enterprise plans with distinct color schemes.
- **Section Organization**: Features are grouped into logical sections (Core, Advanced, Security) for better readability.
- **Fully Responsive**: On mobile devices, the table transforms into a stacked card layout for better mobile UX.
