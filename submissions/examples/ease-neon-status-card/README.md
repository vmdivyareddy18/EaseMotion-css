# Ease Neon Status Card

A premium CSS-only status card with a glowing neon ring and animated dashboard style.

## Features

- Pure HTML & CSS
- Animated rotating ring
- Glassmorphism card
- Responsive layout
- Neon glow effects
- Zero dependencies

## Folder Structure

```
ease-neon-status-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Edge
- Firefox
- Safari

## Why EaseMotion CSS

Demonstrates smooth animation, reusable UI, and modern CSS with a lightweight implementation.
