# Issue #14824: Advanced: Full Construction Company Landing Page

This folder contains a standalone demonstration of a Construction Company Landing Page as requested in Issue #14824.

## Files
- `demo.html`: The HTML structure for the landing page.
- `style.css`: The CSS styling using bold, industrial aesthetics.

## How it works
The demo provides a complete layout for a construction firm featuring:
- A high-contrast color palette utilizing industrial yellow/orange and dark charcoal.
- Bold, heavy typography to communicate strength and reliability.
- An angled design motif representing structural building.
- Services grid and a statistics/trust section.
- Dynamic hover effects for interactive elements.
