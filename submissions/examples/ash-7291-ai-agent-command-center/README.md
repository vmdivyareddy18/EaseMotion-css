## Overview

A responsive dashboard for monitoring autonomous AI agents, task execution, workflow automation, and operational insights. Suitable for AI agent platforms, orchestration systems, automation dashboards, and enterprise AI operations.

## Features

- AI agent KPI cards
- Performance visualization
- Live activity feed
- Responsive dashboard layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required
