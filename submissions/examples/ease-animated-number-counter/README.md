# Animated Number Counter – Counts up from 0 to target

> A production-ready animated number counter that smoothly counts up from 0 to a target value with easing animations. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Smooth counting animation** – Counts up with easing
- ✅ **Multiple counters** – Run multiple counters simultaneously
- ✅ **Progress tracking** – Visual progress bar
- ✅ **Pause/Resume** – Control animations
- ✅ **Reset functionality** – Reset to 0
- ✅ **Custom targets** – Set any number
- ✅ **Easing options** – Linear, easeOut, easeInOut
- ✅ **Auto-start on scroll** – Intersection Observer support
- ✅ **Status indicators** – Running, Paused, Complete
- ✅ **Keyboard shortcuts** – Ctrl+Enter (start), Ctrl+Escape (reset)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
