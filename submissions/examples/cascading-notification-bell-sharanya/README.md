# Cascading Notification Bell

## Description

The Cascading Notification Bell is a reusable notification component inspired by modern creative agency interfaces. It features a notification bell with cascading notification cards that appear in sequence using smooth CSS animations. The component is built using only HTML and CSS, making it lightweight, responsive, and easy to customize.

## Features

- Cascading notification animation
- Modern creative agency inspired design
- Pure HTML and CSS
- Responsive layout
- Accessible structure
- Easy to customize and reuse

## Usage

1. Open `demo.html` in any modern web browser.
2. The component styles are defined in `style.css`.
3. Customize the notification text, colors, spacing, and animation timing as needed.

## Why EaseMotion CSS?

This component follows the EaseMotion CSS philosophy by providing a reusable, animation-focused notification pattern with clean HTML structure and lightweight CSS. It can be integrated into dashboards, creative agency websites, and modern web interfaces without requiring JavaScript.
