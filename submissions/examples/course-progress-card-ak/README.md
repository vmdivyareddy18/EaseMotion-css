# Course Progress Card

## What does this do?
A course progress dashboard showing multiple courses with colored progress bars, lesson counts, instructor names, and a Continue/Review button that simulates lesson completion.

## How is it used?
Open demo.html in a browser. Each card shows course progress. Click "Continue" to simulate completing a lesson and watch the progress bar update.

## Why is this useful?
Demonstrates a learning management system (LMS) course card pattern with visual progress tracking and interactive state updates.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
