# Neumorphic Range Slider Input

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A soft-shadow neumorphic range slider component designed for modern dashboards, settings pages, and control panels.

## Features

- Neumorphic design
- Soft shadow effects
- Custom slider thumb
- Value display badge
- Responsive layout
- Multiple slider examples
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<input
  type="range"
  class="neumorphic-slider"
>
```

## Main Classes

```css
.slider-card
.neumorphic-slider
.value-badge
.slider-scale
.volume-value
```
