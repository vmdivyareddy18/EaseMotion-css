# Autonomous Global Investigation Network

A responsive **Autonomous Global Investigation Network** dashboard built entirely with **HTML** and **CSS**.

## Features

- Executive investigation command dashboard
- Autonomous case intelligence monitoring
- Investigation intelligence matrix
- AI-powered evidence analysis
- Investigation workflow timeline
- Enterprise analytics cards
- Circular case success indicator
- Responsive CSS Grid layout
- Modern command center UI
- No JavaScript or external libraries
