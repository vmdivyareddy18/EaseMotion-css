# 🐍 Snake Game

A classic Snake game built with HTML, CSS, and JavaScript. Control the snake to eat food, grow longer, and avoid collisions with walls or your own tail.

## 🎮 How to Play

- **Arrow keys (⬆ ⬇ ⬅ ➡)** – Change the snake's direction
- **Spacebar** – Pause / resume the game
- **R key** – Restart the game at any time
- **Restart button** – Click to reset your game

## 🧠 Game Rules

- Eat the red food (🍎) to grow and increase your score.
- The game ends if you hit the wall or your own tail.
- Win by filling the entire board with your snake!
