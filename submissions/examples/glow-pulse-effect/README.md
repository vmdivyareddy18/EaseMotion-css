# Glow Pulse Animation Effect

A smooth, attention-grabbing animation utility that combines scale pulsing with a glowing effect. Perfect for CTAs, badges, notifications, and interactive elements.

## Features
- ✨ Smooth scale pulse animation (1.0 → 1.15 → 1.0)
- 💡 Glowing drop-shadow effect with customizable intensity
- ⏱️ Configurable animation duration via CSS variables
- 🚀 Zero JavaScript required

## Usage
```html
<button class="btn glow-pulse">Get Started</button>
