🧭 CSS Logical Properties

margin-inline, padding-block, inset-inline-start, border-inline-end — all flip automatically with the writing direction.

margin-inline — Horizontal Spacing

start middle end

margin-inline: auto on .inline-middle pushes it to the center; margin-inline-start/end on the siblings flips with direction.

padding-block — Vertical Padding

padding-block: 1rem 2rem

The first value maps to padding-block-start (top), the second to padding-block-end (bottom).

inset-inline-start — Positioning

This box uses inset-inline-start: 20px instead of left. In RTL it shifts to the right side.

border-inline-end — Directional Borders

This border-inline-end appears on the right in LTR, left in RTL.
