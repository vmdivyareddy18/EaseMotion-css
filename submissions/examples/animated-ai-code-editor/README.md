# Animated AI Code Editor Component

A modern **VS Code-inspired AI Code Editor** built entirely with **HTML & CSS**. This component features a clean glassmorphism interface, animated code editor, AI assistant panel, syntax-highlighted sample code, and responsive design without using JavaScript.

---

## ✨ Features

- 🖥️ VS Code inspired interface
- 🤖 Built-in AI Assistant panel
- 🎨 Modern glassmorphism design
- 📁 File explorer sidebar
- 📑 Editor tabs
- 💻 Syntax-highlighted sample code
- ⌨️ Animated blinking cursor
- ▶️ Animated Run button
- 💬 AI suggestions panel
- ✨ Smooth hover effects
- 🌌 Animated glowing background
- 📱 Fully responsive layout
- 🚀 Pure HTML & CSS (No JavaScript)
