# CSS Tada Click Tooltip

Pure CSS animated tooltip utilizing a Tada-Click interaction designed for Cyberpunk Neon layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Tada animation
- Cyberpunk neon styling
- CSS custom properties
- Modern interface aesthetics
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--tooltip
--duration
--ease
```
