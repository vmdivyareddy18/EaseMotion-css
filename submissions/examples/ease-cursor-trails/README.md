# Cursor Trails – Glowing trail follows cursor

> A production-ready cursor trails component that creates a mesmerizing glowing particle trail that follows your cursor with customizable length, size, speed, and colors. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Glowing particle trail** – Mesmerizing trail follows cursor
- ✅ **Customizable** – Length, size, speed, color
- ✅ **Trail toggle** – Enable/disable trail effect
- ✅ **Particle system** – Glowing particles with wobble
- ✅ **Keyboard shortcuts** – Space (toggle), C (clear), T (trail toggle)
- ✅ **4 color variants** – Ocean, Sunset, Gold, Dream
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
