# Zoom Image Gallery

A responsive **Glassmorphism Zoom Image Gallery** built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Glassmorphism cards
- Smooth zoom animation
- Responsive grid layout
- Accessible images
- CSS custom properties
- Supports prefers-reduced-motion
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Customization

Edit the CSS variables inside `:root` to change colors, border radius, blur, and animation timing.
