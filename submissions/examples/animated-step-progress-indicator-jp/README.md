# Animated Step Progress Indicator Component

## Description

A CSS-only multi-step progress indicator with animated active states, completed steps, and smooth transitions.

---

## Features

- Animated active step pulse effect
- Completed and pending states
- Smooth hover animations
- Responsive vertical layout for mobile devices
- CSS-only implementation
- Lightweight and customizable
- Accessibility support using reduced motion

---

## Classes

- `.step-progress` – Main progress wrapper
- `.step` – Individual progress item
- `.circle` – Step number or completed icon
- `.line` – Connector between steps
- `.active` – Current active step
- `.completed` – Finished step state

---

## Accessibility

Supports `prefers-reduced-motion` to disable unnecessary animations for users who prefer reduced motion.

---

## Use Cases

- Checkout process
- Registration forms
- User onboarding
- Task trackers
- Multi-step workflows
