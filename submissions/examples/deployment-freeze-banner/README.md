# Deployment Freeze Banner

An intermediate EaseMotion CSS example for release governance screens. The banner uses animated stripes, a pulsing status dot, and a filling timeline to communicate freeze-window state.

## Files

- `demo.html` - Semantic freeze-window notice markup.
- `style.css` - Responsive banner layout, moving stripe overlay, pulse motion, and timeline fill.

## Usage

Open `demo.html` in a browser to preview the animated deployment freeze banner.
