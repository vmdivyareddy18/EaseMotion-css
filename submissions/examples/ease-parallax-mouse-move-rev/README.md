# Parallax Mouse Move

1. What does this do? Creates a multi-layer parallax effect where layers move at different speeds based on mouse position to create depth.

2. How is it used? Apply `.parallax-layer` class to elements with `data-speed` attribute for movement intensity, and use JavaScript to set `--mouse-x` and `--mouse-y` custom properties.

3. Why is it useful? It provides an engaging, performant way to add depth and interactivity to web interfaces using CSS transforms driven by mouse position.
