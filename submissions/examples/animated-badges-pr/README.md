# Animated Badge and Tag Components

### What does this do?
Provides a modular suite of micro-UI elements, including live status badges featuring infinite radial scale pulse loops and tactile category tags with glowing hover behaviors.

### How is it used?
Incorporate `.ease-badge-pulse` paired with nested `.status-dot` containers for active states, or drop `.ease-tag-interactive` modifiers directly onto text blocks.

### Why is it useful?
It updates common interface indicator components with fluid, hardware-accelerated animations while maintaining zero bundle overhead.
