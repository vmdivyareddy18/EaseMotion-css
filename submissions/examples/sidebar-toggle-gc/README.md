# Animated Sidebar Toggle UI Example

## 1. What does this do?

This submission provides a collapsible sidebar menu with sliding animation and active item highlight. The sidebar smoothly opens and closes while keeping the main content visible.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes a sidebar with menu items and a toggle button. Copy the `.sidebar-toggle` markup and CSS into your project to implement an animated sidebar.

---

## 3. Why is it useful?

Animated sidebar toggles save space while offering easy navigation access. The sliding motion and active highlight create a polished experience for responsive layouts.
