# EaseMotion - Nonprofit Food Bank Template
**Issue**: [#21045](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/21045)
**Level**: Advanced

## Description
A fully responsive, copy-paste ready full-page layout designed for community-focused nonprofits and food banks. Built strictly following EaseMotion guidelines, bypassing custom keyframes entirely by utilizing transition states and an inline `IntersectionObserver` for seamless scroll animations. 

## Features
- Dynamic gradient text headers and community impact imagery.
- Multi-tier grid layouts adapting seamlessly from mobile to desktop.
- Scroll-based entry animations using pure CSS transitions (no custom `@keyframes`).
- Vanilla JavaScript-powered "Count-Up" functionality for impact statistics.
- Interactive `hover-float` cards for corporate and volunteer calls-to-action.

## Showcased EaseMotion Classes
- **Layout**: `ease-container`, `ease-grid`, `ease-flex`, `ease-overflow-hidden`
- **Typography**: `ease-text-gradient`, `ease-text-7xl`, `ease-font-extrabold`
- **Animation States**: `ease-opacity-0` -> `ease-opacity-100`, `ease-translate-y-8` -> `ease-translate-y-0`
- **Interaction**: `ease-hover-float`, `ease-hover-glow`, `ease-hover-scale-sm`

## Testing
Open `demo.html` directly in your browser (`file:///`). No build steps or server required.
