# E-commerce: Artisan Soap Shop (`ecommerce-artisan-soap-shop-22739`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a handcrafted, natural soap and botanical skincare brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A natural artisan brand requires a UI that feels organic, earthy, and grounded. This template achieves a crafted aesthetic using an earthy palette (sage, rose clay, charcoal, sand), a sophisticated typography pairing (`Crimson Pro` & `Jost`), and soft, floating animations that mimic nature.

The layout includes:
- A clean, sticky Navigation with soft hover text lifts.
- A Hero section featuring abstract CSS soap bars floating gently alongside swaying botanical leaves.
- A Collections grid showcasing round soap bars with CSS-based textures (speckles and grain) and soft hover zooms.
- A Story/Ingredients section featuring an abstract CSS mortar and pestle with dripping botanical oils.
- A Gift Bundles section utilizing CSS to render a beautifully arranged box wrapped with a translucent ribbon.
- A refined, elegant Reviews grid with star ratings.
- A high-contrast Newsletter signup section.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use natural, organic `cubic-bezier(0.4, 0, 0.2, 1)` and `cubic-bezier(0.42, 0, 0.58, 1)` timings that convey a slow, mindful craft:
- `.ease-fade-up` (Configured for a soft, organic vertical entrance).
- `.ease-fade-in-slow` (A gentle blur-to-focus reveal for larger visual elements).
- `.ease-float-slow` (A continuous vertical hovering animation mimicking objects suspended in air or water).
- `.ease-sway` (A continuous animation mimicking leaves blowing in a gentle breeze).
- `.ease-drip` (A continuous cubic-bezier animation simulating dripping oil/water).
- `.ease-hover-zoom` (A slow scale and slight rotation on hover for product cards).
- `.ease-hover-float` (A soft, organic hover lift with shadow expansion for buttons and graphics).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the natural layout architecture, the earthy palette, and the complex abstract CSS illustrations (soap bars, textures, mortar/pestle, leaves, gift box). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the soft entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
