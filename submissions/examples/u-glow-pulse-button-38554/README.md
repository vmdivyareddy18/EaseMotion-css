# Glow Pulse Button

A modern **Glow Pulse Button** built with pure CSS for **Visual Editor Layouts**. The component provides a smooth glowing pulse animation to emphasize important actions such as publishing, exporting, or saving creative work.

---

## Features

- Pure CSS implementation
- Smooth Glow Pulse animation
- Fully responsive layout
- Keyboard accessible
- CSS Custom Properties for customization
- Zero JavaScript dependency
