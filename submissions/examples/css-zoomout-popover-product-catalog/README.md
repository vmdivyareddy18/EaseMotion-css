Zoom-Out Popover — Product Catalog

A production-quality Zoom-Out Popover component built using only HTML5 and CSS3. This example demonstrates a premium e-commerce product card with an interactive popover that reveals detailed product information using a smooth zoom-out animation.

---

Features

- Pure HTML5 & CSS3
- No JavaScript
- Modern premium e-commerce UI
- Glassmorphism design
- Smooth Zoom-Out Popover animation
- Hover and keyboard focus support
- Product image placeholder
- Product title
- Category badge
- Rating
- Price
- Discount badge
- Stock availability
- Wishlist button
- Add to Cart button
- Large product preview
- Product specifications
- Features list
- Available colors
- Available sizes
- Shipping information
- Buy Now button
- Soft shadows
- Modern gradients
- Rounded corners
- Responsive layout
- CSS Custom Properties
- Accessible markup
- Visible keyboard focus
- Screen reader friendly
- Supports "prefers-reduced-motion"
- Modular CSS architecture
