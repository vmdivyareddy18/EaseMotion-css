# Toggle Switch Component

Pure CSS toggle switches using checkbox inputs. No JavaScript required.

## Demo

Open `demo.html` in any modern browser.

## Features

- ✅ **Pure CSS** — uses checkbox input, zero JavaScript
- ✅ **3 sizes** — small, default, large
- ✅ **4 colors** — default (indigo), success, warning, danger
- ✅ **Icon support** — on/off icons inside slider
- ✅ **Rounded variant** — squared pill shape
- ✅ **Disabled state** — non-interactive with reduced opacity
- ✅ **Toggle group** — stacked switch list
- ✅ **Accessible** — keyboard focus, reduced-motion support

## Usage

### Basic Toggle

```html
&lt;label class="toggle"&gt;
  &lt;input type="checkbox" class="toggle__input"&gt;
  &lt;span class="toggle__slider"&gt;&lt;/span&gt;

&lt;/label&gt;
