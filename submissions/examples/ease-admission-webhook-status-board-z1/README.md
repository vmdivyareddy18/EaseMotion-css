# Ease Admission Webhook Status Board

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes Admission Webhook health and availability.

## Features

- Webhook health overview
- Configuration update monitoring
- Latency warning detection
- Admission failure alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable Kubernetes security dashboard for monitoring Admission Webhooks, validating deployment policies, and improving cluster reliability.
