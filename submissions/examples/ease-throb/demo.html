ease-throb

A slow, deep heartbeat-like throb for elements that need maximum emphasis — scale and glow expand together over a full 2s cycle.

Default (Red)

Teal

ease-throb vs. ease-pulse

ease-pulse · 0.8s · quick & light
🔔
ease-throb · 2s · deep & heavy
🚨

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Add the class to any important CTA */
<button class="ease-throb">Buy Now</button>

/* 3. Customise color, scale, or speed */
.my-cta {
  --ease-throb-color: #6c63ff;     /* glow + tint color */
  --ease-throb-scale: 1.18;        /* deeper throb */
  --ease-throb-duration: 2.5s;     /* even slower */
}
