# DNA Helix Animation

A stunning, production-ready DNA helix animation that simulates two strands rotating with connecting base pairs. Built with vanilla HTML, CSS, and JavaScript — no dependencies required.

---

## 📸 Demo

![DNA Helix Animation](https://via.placeholder.com/500x400/4facfe/ffffff?text=🧬)

**Live Preview:** Open `demo.html` in your browser.

---

## ✨ Features

- ✅ **Two strands rotating** with smooth 3D-like animation
- ✅ **Connecting bars** between base pairs
- ✅ **Real-time depth sorting** for 3D perspective
- ✅ **Interactive controls** – Play/Pause, Reset, Speed, Theme
- ✅ **Color themes** – Ocean, Fire, Neon, Purple
- ✅ **Keyboard shortcuts** – Space (pause), R (reset), S (speed), C (theme)
- ✅ **Particle background** for atmospheric depth
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Accessible** – Keyboard controls and ARIA support
- ✅ **Customizable** – Easy to tweak colors, speed, and base pairs
