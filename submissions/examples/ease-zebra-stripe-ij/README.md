# Ease Zebra Stripe

A CSS animation of alternating table rows highlighting like zebra stripes with a pulsing effect.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.zebra-table` element.

## Custom CSS Variables

| Variable           | Default | Description                     |
|--------------------|---------|---------------------------------|
| `--stripe-color`   | `#eee`  | Highlight color for even rows   |
| `--row-bg`         | `#2d2d44`| Background color for odd rows  |
| `--stripe-duration`| `0.3s`  | Transition duration per stripe  |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
