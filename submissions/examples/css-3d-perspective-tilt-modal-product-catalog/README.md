# CSS 3D Perspective Tilt Modal (Product Catalog UI)

A **pure CSS 3D animated modal component** designed for **Product Catalog interfaces** in EaseMotion CSS.

---

## ✨ Features

- Product preview modal UI
- Pure CSS 3D perspective animation
- Hover-based tilt interaction
- Smooth open/close transitions
- Fully responsive design
- Keyboard accessible (ESC support)

---

## 🎯 Purpose

Enhances product browsing experience with:
- interactive product previews
- modern motion effects
- clean catalog UI structure

---

## 🧪 Usage

```html
<button class="open-btn" onclick="openModal()">View Product</button>

<div class="modal active">
  <div class="modal-content">
    Product details here
  </div>
</div>
