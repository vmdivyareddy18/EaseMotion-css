# Confetti Celebration – Burst on success (form submit)

> A production-ready confetti celebration component that bursts into colorful particles on success events like form submissions. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **🎊 Burst on success** – Trigger confetti on form submit
- ✅ **💥 Multiple shapes** – Circle, square, rectangle, triangle, star
- ✅ **🎨 Rich colors** – Vibrant, colorful particles
- ✅ **📊 3 intensity levels** – Low, Medium, High
- ✅ **🔄 Animated particles** – Physics-based motion
- ✅ **⌨️ Keyboard shortcuts** – Space (burst), R (reset), 1-3 (intensity)
- ✅ **🎮 Programmatic API** – Trigger from anywhere
- ✅ **📱 Responsive** – Adapts to any screen size
- ✅ **🌙 Dark mode** – Auto-adjusts to system preference
- ✅ **⚡ Zero dependencies** – No libraries or frameworks
