# Marketing Landing Page Pulse-Active Modal

A modern **Pure CSS Pulse-Active Modal** designed for **Marketing Landing Page** interfaces. The component features a smooth pulse animation, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Pulse-Active animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Marketing landing page inspired design
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:1s;
    --easing:ease-in-out;
    --scale:1.04;

    --primary:#ff6b35;
    --secondary:#ff3d81;
}
```

You can customize:

- Animation duration
- Animation easing
- Pulse scale
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in your browser.

No JavaScript or build tools are required.
