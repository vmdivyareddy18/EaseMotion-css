One @keyframes, Infinite Variants

Each box uses the same .motion-atom class and the same @keyframes parametric-motion. Only the inline custom properties differ — var() inside @keyframes resolves per-element.

The single shared keyframe — all variants above come from this alone

@keyframes parametric-motion {
  from {
    transform: translate(0, 0) rotate(0deg) scale(1);
  }
  to {
    transform:
      translate(var(--travel-x, 0px), var(--travel-y, -60px))
      rotate(var(--rotation, 0deg))
      scale(var(--end-scale, 1));
  }
}
