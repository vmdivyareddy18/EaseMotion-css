# Toolbar Icon Pop

A compact toolbar where icons lift on interaction and the active tool pops into place.

## Files

- `demo.html` - toolbar markup with three tool buttons.
- `style.css` - icon hover state, active pop animation, focus-visible ring, and reduced-motion support.

## Highlights

- Uses `role="toolbar"` for the button group.
- Keeps button dimensions stable during transforms.
- Mirrors hover styling on keyboard focus.
