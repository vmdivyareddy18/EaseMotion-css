# Dark Mode Animation Demo

## Purpose

This submission demonstrates how EaseMotion CSS animation classes can be used in a dark-themed interface.

---

## Features

- Dark mode layout
- Responsive cards
- Animation examples
- Beginner-friendly structure

---

## Included Animation Classes

- ease-fade-in
- ease-slide-up
- ease-hover-grow

---

## Folder Structure

```
dark-mode-animation-demo/
│
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`
2. Hover over the cards.
3. Resize the browser window to test responsiveness.

---

## Benefits

- Demonstrates dark mode compatibility.
- Provides reusable UI examples.
- Helps contributors understand animation usage in modern interfaces.
