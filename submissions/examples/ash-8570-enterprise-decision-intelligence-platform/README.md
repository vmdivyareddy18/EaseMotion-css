# Enterprise Decision Intelligence Platform

A responsive **Enterprise Decision Intelligence Platform** dashboard built entirely with **HTML** and **CSS**.

## Features

- Executive decision intelligence dashboard
- AI-powered business insights
- Decision intelligence matrix
- Predictive analytics overview
- Enterprise workflow timeline
- Executive metric cards
- Circular decision accuracy indicator
- Responsive CSS Grid layout
- Modern enterprise command center
- No JavaScript or external libraries
