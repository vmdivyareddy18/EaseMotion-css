# Upload Dropzone Pulse

## What does this do?
A file upload dropzone with a pulsing border, focus feedback, and animated selected-file chips.

## How is it used?
```html
<label class="dropzone">
  <input type="file">
  <span class="upload-icon"></span>
  <strong>Drop files here</strong>
</label>
```

## Why is it useful?
It gives EaseMotion CSS a practical form component for upload workflows with clear motion and accessible focus treatment.
