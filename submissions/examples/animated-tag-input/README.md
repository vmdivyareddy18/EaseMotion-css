# Animated Tag/Chip Input Component

A clean, modern, and fully responsive **Tag Input Component** perfect for email recipients, skill selection, and category management. This example demonstrates how to combine **EaseMotion CSS** for smooth tag entrances/exits, interactive hover states, and continuous pulse effects to create an engaging, interactive tagging experience.

## 🚀 Features

- **Bouncy Tag Entrances**: Tags use `ease-bounce-in` with staggered delays for a playful, cascading load effect.
- **Smooth Tag Removal**: Tags fade out and scale down when removed with a custom CSS animation.
- **Color-Coded Tags**: Multiple color variations (default, blue, green, purple, orange) for categorization.
- **Pill-Shaped Tags**: Optional rounded pill shape with icons for visual variety.
- **Interactive Hover States**: Tags change color on hover using `ease-hover-grow` for tactile feedback.
- **Pulsing Add Button**: The add button uses `ease-pulse` to draw attention.
- **Focus States**: Container highlights when input is focused.
- **Suggestion Panel**: Clickable suggestion tags that can be added to the input.
- **Keyboard Support**: Press Enter to add tags.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted layout.
