# Environment Variable Scope Badges

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive environment variable management component that displays deployment scopes using color-coded badges for developer dashboards, DevOps tools, and cloud platforms.

## Features

- Environment variable list
- Scope badges
- Production, staging, development, and shared scopes
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="variable">

    <div class="info">
        <h3>DATABASE_URL</h3>
    </div>

    <span class="badge production">
        Production
    </span>

</div>
```

## Main Classes

```css
.card
.variable
.info
.badge
.production
.staging
.development
.shared
```
