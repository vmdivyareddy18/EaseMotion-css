# Marketing Landing Page Staggered Entrance Popover

A pure CSS animated popover component designed for modern marketing landing pages.

## Features

- Pure CSS animation
- Staggered entrance effect
- Responsive layout
- Keyboard accessible
- No JavaScript required
- Customizable animation variables


## Usage

Include the HTML structure from `demo.html`
and import `style.css`.


Example:

```html
<button class="cta">
 Explore Features

 <div class="popover">
   <div class="item">
     Campaign Tools
   </div>
 </div>

</button>
