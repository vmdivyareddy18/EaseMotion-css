# Audio Waveform Visualizer

## What does this do?
An animated audio waveform visualizer with genre-specific animation patterns (Pop, Rock, Classical, Electronic) and play/pause controls.

## How is it used?
Open demo.html in a browser. Click "Play" to start the animated waveform bars. Select different genres to change the animation pattern, speed, and color scheme.

## Why is this useful?
This provides a visually engaging audio visualization component for music players or ambient displays. It demonstrates requestAnimationFrame-based animation with genre-driven parameter variation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see the visualizer.
