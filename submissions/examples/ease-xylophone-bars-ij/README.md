# Ease Xylophone Bars

A CSS animation of a xylophone being played sequentially from left to right, with a hammer that strikes each bar and a glow effect on the active bar.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.xylophone` element.

## Custom CSS Variables

| Variable          | Default  | Description                         |
|-------------------|----------|-------------------------------------|
| `--bar-width`     | `30px`   | Width of each xylophone bar         |
| `--bar-color`     | `#ff6b6b`| Base color of the bars              |
| `--hammer-color`  | `#ddd`   | Color of the hammer head and handle |
| `--hit-duration`  | `0.4s`   | Duration each bar is struck         |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
