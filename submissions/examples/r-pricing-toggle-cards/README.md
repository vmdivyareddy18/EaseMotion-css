# Animated Pricing Toggle Cards

A responsive pricing section featuring a billing toggle UI, animated pricing cards, and highlighted plans.

## Features

- Monthly/Yearly billing toggle interface
- Three pricing tiers
- Featured plan highlight
- Hover animations
- Responsive layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more pricing plans
- Change pricing values
- Customize colors and typography
- Modify animation timing
