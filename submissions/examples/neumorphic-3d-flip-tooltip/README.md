# Neumorphic 3D Flip Tooltip

A responsive tooltip component built entirely with CSS featuring a smooth **3D Flip** animation and a soft neumorphic design.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard Accessible
- 3D Flip Animation
- CSS Custom Properties
- Soft Neumorphic UI

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## CSS Variables

```css
--duration
--ease
--flip-angle
--bg
--tooltip-bg
--text
--radius
```

These variables allow customization of animation speed, colors, rotation angle, and border radius.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## EaseMotion CSS

This example demonstrates a lightweight, reusable tooltip using a CSS-only 3D flip animation with a modern neumorphic aesthetic. It follows EaseMotion CSS principles by being animation-focused, customizable, accessible, and JavaScript-free.
