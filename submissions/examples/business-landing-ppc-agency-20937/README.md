# Business Landing: PPC / Google Ads Agency (`business-landing-ppc-agency-20937`)

This proposal provides a complete, copy-paste ready landing page template designed for a Performance Marketing or PPC Advertising Agency, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A performance marketing agency website must project an image of data-driven precision, technical capability, and high conversion. This template achieves a "Tech-Forward Performance" aesthetic using a dark mode palette (`Zinc 950/900`) contrasted with a highly visible "Cyber Green" accent (`#00ff9d`). It pairs a highly legible interface sans-serif (`Inter`) with a monospace font (`JetBrains Mono`) for data points and technical details.

The layout includes:
- A dark, tech-oriented navigation bar.
- A Hero section featuring glowing orb backgrounds, a CSS grid pattern, and a custom CSS-built data visualization graphic (bar chart) demonstrating ROAS growth.
- A high-impact metrics banner.
- A "Services" grid focusing on specific ad networks (Google, Meta, LinkedIn).
- A "Case Studies" section highlighting specific ROI metrics.
- A "Methodology" process timeline utilizing an alternating vertical layout.
- A transparent pricing grid.
- A high-conversion "Free Audit" form section.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use snappy, mechanical `cubic-bezier(0.175, 0.885, 0.32, 1.275)` timings tailored for a data-driven brand:
- `.ease-grow-up` (Drives the animated bar chart in the hero section, simulating data loading).
- `.ease-float` (Creates a continuous, subtle levitation effect on the data cards in the hero).
- `.ease-slide-up-stagger` (Configured for a snappy, slightly overshooting vertical entrance).
- `.ease-hover-scale-subtle` (Provides a constrained, precise interaction on the case study cards).
- *Custom Animations*: Includes a slow pulse (`ease-pulse-slow`) for background orbs and a pinging status indicator (`animate-ping`) to simulate a live, active system.

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the dark mode palette and typography to the pure CSS data visualizations (the animated bar chart). The standard `ease-*` interaction classes are attached directly to HTML elements to provide snappy, responsive animations. No JavaScript is required for this template.
