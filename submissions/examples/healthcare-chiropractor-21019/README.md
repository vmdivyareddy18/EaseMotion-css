# Healthcare: Chiropractor (`healthcare-chiropractor-21019`)

This proposal provides a complete, copy-paste ready B2C healthcare landing page designed for a holistic chiropractic practice, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A chiropractic and wellness site should evoke feelings of natural healing, balance, and trust. This template achieves a "holistic and trustworthy" aesthetic using an earthy color palette (sage greens, warm beige, stone), a dual typography system featuring a classic serif font for headings (Lora) and a clean sans-serif for body copy (Lato), and slow, deliberate entrance animations.

The layout includes:
- A hero section featuring soft CSS blob background shapes and an abstract arched image frame with a floating feature card.
- A 3-column "Conditions Treated" grid.
- A split-layout "Treatment Approach" section with a step-by-step methodology.
- Professional team profile cards.
- Trust-building patient testimonial cards set against the primary brand color.
- A robust, dual-pane appointment booking form integrating insurance and contact information.
- A comprehensive clinic footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (Configured with slightly longer durations (0.8s) for slower, more deliberate and calming entrances of text and cards)
- `.ease-zoom-in` (Applied to the hero image frame and methodology visual to draw focus softly)
- `.ease-hover-scale` (Applied with a gentle cubic-bezier to team portraits for a subtle, high-end interaction feel)
- `.ease-hover-lift` (Standard interaction for booking buttons)
- `.ease-slide-left` / `.ease-slide-right` (Used in split-layout sections, such as the booking form, to bring content together seamlessly)
- `.ease-focus-ring` (Applied to form inputs to provide a soft, earthy glow outline upon focus, avoiding harsh border lines)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, organic shapes, and typography mapping, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/healthcare-chiropractor-21019/` directory to avoid modifying core files directly and to ensure zero deletions.*
