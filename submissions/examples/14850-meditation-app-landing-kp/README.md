# Meditation App Landing Page

## What does this do?

It provides a responsive mobile app landing page for a meditation product with a phone preview, wellness sections, and progress stats.

## How is it used?

```html
<section class="hero">
  <h1>
    Build calm habits with guided breathing, sleep stories, and focus sessions.
  </h1>
  <a class="button" href="#sessions">Start breathing</a>
</section>
```

## Why is it useful?

It demonstrates a wellness app page with a CSS-only breathing animation and self-contained responsive layout.
