# Neon Cyber Operations Interface

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A futuristic cyber operations dashboard featuring neon-inspired styling, security metrics, activity monitoring, and incident tracking. Ideal for cybersecurity dashboards, SOC interfaces, network monitoring tools, and enterprise security platforms.

## Features

- Neon cyber dashboard
- Security KPI cards
- Live activity chart
- Recent events panel
- Responsive layout
- Neon glow effects
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="dashboard">

    <aside class="sidebar">
        ...
    </aside>

    <main class="content">
        ...
    </main>

</div>
```
