# CSS 3D Perspective Tilt Modal (E-Commerce Checkout UI)

A **pure CSS 3D animated modal component** designed for **E-Commerce checkout interfaces** in EaseMotion CSS.

---

## ✨ Features

- Checkout-style UI modal
- Pure CSS 3D perspective animation
- Hover-based tilt interaction
- Smooth open/close transitions
- Fully responsive design
- Keyboard accessible (ESC support)

---

## 🎯 Purpose

Enhances e-commerce flows like:
- order summary
- checkout confirmation
- product preview modal

with modern motion and depth effects.

---

## 🧪 Usage

```html
<button class="open-btn" onclick="openModal()">Open Checkout</button>

<div class="modal active">
  <div class="modal-content">
    Checkout content
  </div>
</div>
