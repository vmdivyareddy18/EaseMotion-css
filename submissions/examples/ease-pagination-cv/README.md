# Animated Pagination (`ease-pagination-cv`)

A lightweight animated pagination component built using **HTML5** and **Pure CSS**.

## Features

- Sliding active page indicator
- Hover transitions
- Rounded style
- Square style
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-pagination-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Variants

- Rounded pagination
- Square pagination

## Browser Support

Works in all modern browsers.
