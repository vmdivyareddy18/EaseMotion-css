# Ease Spectrum Equalizer

## Overview

A modern animated spectrum equalizer featuring glowing gradient bars that rise and fall in sequence using only HTML and CSS.

## Features

- Animated equalizer bars
- Staggered wave motion
- Gradient glow effect
- Glassmorphism container
- Responsive design
- Pure HTML & CSS

## Folder Structure

```text
ease-spectrum-equalizer-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Scales cleanly across desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Demonstrates reusable loading and audio-inspired animations with concise CSS, making it ideal for dashboards, music interfaces, AI assistants, and visual status components.
