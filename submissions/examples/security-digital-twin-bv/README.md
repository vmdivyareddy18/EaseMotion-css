# Security Digital Twin Platform

Virtual security ecosystem modeling platform designed to simulate threat propagation paths, evaluate impact, and perform sandbox what-if scenario testing.

---

## Features
- **Virtual Replica Grids:** Visualizes active and quarantined nodes on a topology layout.
- **Attack Scenario Controllers:** Interactive toggle buttons to trigger ransomware, insider, or DDoS simulations.
- **Blast Radius Indicators:** Telemetry grids displaying forecasted recovery status.
