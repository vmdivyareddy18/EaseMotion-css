# Milestone Progress Bar

A CSS-only milestone progress bar designed for modern SaaS dashboards.

## What it is

This component displays a horizontal project progress tracker with five milestone nodes. Completed milestones show filled circles, the current milestone pulses, and future milestones remain dimmed. Labels sit beneath each milestone, and the progress fill animates from left to right.

## Features

- CSS-only implementation with no JavaScript
- Horizontal progress track with milestone nodes
- Animated progress fill and pulse effect
- Clean SaaS dashboard styling
- Responsive layout for mobile and desktop
- Clear state styling for completed, current, and future milestones

## Usage

1. Copy `demo.html` and `style.css` into your submission folder.
2. Open `demo.html` in a browser.
3. Customize milestone labels or progress width as needed.

```html
<link rel="stylesheet" href="style.css" />
<div class="progress-page">
  <section class="progress-card ease-fade-in">
    <!-- milestone content here -->
  </section>
</div>
```
