# Elastic Sidebar

1. **What does this do?**  
   A collapsible sidebar that expands with a satisfying elastic bounce effect upon hover.

2. **How is it used?**  
   ```html
   <div class="elastic-sidebar">
       <div class="sidebar-item">Item</div>
   </div>
   ```

3. **Why is it useful?**  
   It provides a space-saving navigation component with a highly polished, bouncy micro-interaction out of the box.
