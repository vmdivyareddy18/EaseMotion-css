# Cyberpunk HUD Card

1. **What does this do?**  
   Adds a premium cyberpunk HUD styled card with glowing corners, a pulsing indicator, a scanning line animation, and a responsive tilt hover effect.

2. **How is it used?**

   ```html
   <div class="hud-card">
     <div class="hud-scanline"></div>
     <!-- Card content -->
   </div>
   ```

3. **Why is it useful?**  
   Demonstrates a highly stylized tech/futuristic card layout combining performance-optimized transform animations with accessible prefers-reduced-motion fallbacks.
