Animation Play State Utilities

Three composable classes that pause and resume any looping animation — no JavaScript required for hover control.

The three utility classes

Add any of these to an element that already has a looping animation. They only set animation-play-state — nothing else changes. 

/* Freeze any running animation */
.animate-pause  { animation-play-state: paused;  }

/* Explicitly resume (overrides animate-pause) */
.animate-running { animation-play-state: running; }

/* Pure CSS hover-to-pause — no JavaScript */
.hover-pause       { animation-play-state: running; }
@media (hover: hover) and (pointer: fine) {
  .hover-pause:hover { animation-play-state: paused;  }
}

1. .animate-pause — freeze on load

Adding animate-pause to an already-animating element freezes it at its current frame. Useful for animations that should only play on user interaction, or for pausing before a scroll trigger activates them.

Without utility
🟣 Bouncing
anim-bounce ▶ running
With animate-pause
⏸ Paused
anim-bounce animate-pause ⏸ paused 
/* ✅ Freeze the bounce — element stays at first keyframe */
<div class="anim-bounce animate-pause">⏸ Paused</div>

2. .hover-pause — pause on hover, no JS

The most useful of the three. Move your cursor over any element below and the animation freezes instantly — purely with CSS. This also benefits users who find spinning / looping elements distracting but haven't enabled the OS "reduce motion" setting.

3. JS toggle — classList.toggle('animate-pause')

For richer controls (play/pause buttons, keyboard shortcuts), toggling the animate-pause class from JavaScript is all you need — no inline styles, no framework state.

🔵 Pulse
Rotate — toggle button
⚙️ Rotate 
/* HTML */
<div class="anim-bounce" id="my-anim">🟣 Bounce</div>
<button onclick="toggle()">Pause / Resume</button>

/* JavaScript — that's all it takes */
function toggle() {
  document.getElementById('my-anim')
          .classList.toggle('animate-pause');
}

4. Pause / resume all — one querySelectorAll call

Because the utility is just a class, you can pause every animation on the page at once — useful for "reduce motion" toggles or focus modes.

All demo animations:
🟣 Bounce
🔵 Pulse
🎈 Float
⚙️ Rotate 
/* Pause every animation with .global-anim in one line */
document.querySelectorAll('.global-anim')
        .forEach(el => el.classList.add('animate-pause'));

/* Resume them all */
document.querySelectorAll('.global-anim')
        .forEach(el => el.classList.remove('animate-pause'));

Class reference

Class CSS property Needs JS? Best for
animate-pause animation-play-state: paused No (static) / Yes (toggle) Freeze on load, JS-controlled pause
animate-running animation-play-state: running No Override a paused parent state
hover-pause paused on :hover No Pause on hover — pure CSS

All three classes work with any existing animation — from EaseMotion CSS built-ins to your own @keyframes. They only set animation-play-state and nothing else.
