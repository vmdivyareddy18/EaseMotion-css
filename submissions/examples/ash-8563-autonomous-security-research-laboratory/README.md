# Autonomous Security Research Laboratory

A responsive **Autonomous Security Research Laboratory** dashboard built entirely with **HTML** and **CSS**.

## Features

- Executive security research dashboard
- Autonomous AI research monitoring
- Threat analysis overview
- Security research pipeline table
- Circular security score indicator
- Experiment timeline
- Analytics metric cards
- Responsive CSS Grid layout
- Modern enterprise UI
- No JavaScript or external libraries
