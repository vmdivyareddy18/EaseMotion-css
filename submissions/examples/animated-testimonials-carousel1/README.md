# Animated Testimonials Carousel

A clean, modern, and fully responsive **Testimonials Carousel Component** with smooth slide transitions, star ratings, and auto-play functionality. This example demonstrates how to combine **EaseMotion CSS** for smooth slide animations, staggered content entrances, and interactive navigation to create an engaging social proof component.

## 🚀 Features

- **Smooth Slide Transitions**: Uses `ease-slide-in-right` effect (via CSS transforms) for smooth carousel transitions between testimonials.
- **Staggered Content Animations**: Star ratings use `ease-bounce-in` with delays for a playful entrance effect.
- **Interactive Navigation**: Clickable arrows and dots with `ease-hover-grow` for tactile feedback.
- **Auto-play Functionality**: Automatically cycles through testimonials every 5 seconds with toggle control.
- **Keyboard Navigation**: Arrow keys (← →) navigate between testimonials for accessibility.
- **Star Ratings**: Visual 5-star rating display with bouncy entrance animation.
- **Quote Icon Decoration**: Large decorative quote mark in the background for visual interest.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing and navigation positioning.
