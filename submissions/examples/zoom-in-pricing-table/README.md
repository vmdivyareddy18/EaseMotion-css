# CSS Zoom-In Pricing Table

A responsive pricing table built using pure HTML and CSS.

## Features

- Responsive layout
- Zoom-in hover effect
- Pure HTML & CSS
- Accessible
- prefers-reduced-motion support
- Modern UI
- CSS Variables
- No JavaScript
