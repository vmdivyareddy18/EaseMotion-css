# Ease Rain Effect

A pure CSS rain animation component featuring falling raindrops, atmospheric effects, and customizable rain intensity.

## Features

- Pure HTML & CSS
- CSS Keyframe Animations
- Falling Rain Drops
- Multiple Rain Layers
- Responsive Design
- Smooth Infinite Animation
- Easy Customization

## Files

- demo.html
- style.css
- README.md

## Usage

Add the rain container:

```html
<div class="rain-container">
  <div class="rain-layer"></div>
</div>
```
