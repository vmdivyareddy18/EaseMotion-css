# Enterprise Capability Mapping Dashboard Component

## What does this do?

Provides a centralized interface for visualizing business capabilities, supporting applications, ownership structures, maturity scores, governance metrics, and strategic alignment.

## How is it used?

```html
<div class="capability-card healthy">
    <h3>Customer Management</h3>
</div>
```

## Why is it useful?

Enterprise capability mapping helps organizations understand how business functions are supported by technology and teams. This component provides a reusable dashboard for capability governance, modernization planning, and strategic transformation initiatives.
