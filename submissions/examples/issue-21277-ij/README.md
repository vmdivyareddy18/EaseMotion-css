# Pendulum Wave — Multiple Pendulums Wave

CSS pendulum array with staggered delays to create a wave effect. Each arm rotates from its top origin with a phase offset.

## Files

- `demo.html` — five pendulum elements
- `style.css` — arm/bob styling, staggered wave keyframes

## Usage

Open `demo.html` to see the pendulums sway in a wave pattern.
