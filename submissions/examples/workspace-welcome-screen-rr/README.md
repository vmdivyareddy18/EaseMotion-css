# Workspace Welcome Screen Component

## What does this do?

Provides a modern workspace welcome screen featuring a greeting section, quick actions, workspace statistics, and animated UI interactions.

## How is it used?

```html
<section class="hero">
    <h1>Welcome Back</h1>
</section>
```

## Why is it useful?

Welcome screens help onboard users, surface important actions, and improve the first experience within productivity applications. This component demonstrates modern SaaS welcome-state patterns using pure HTML and CSS.
