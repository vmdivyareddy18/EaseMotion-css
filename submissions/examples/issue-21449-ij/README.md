## Northern Lights Animation

A mesmerizing aurora borealis display with three overlapping ribbons of green, teal, blue, and purple that wave and pulse independently. A starfield twinkles in the background.

**Features:**
- Three aurora bands with independent wave animations
- Screen blend mode for realistic light layering
- Twinkling starfield via radial gradients
- Respects `prefers-reduced-motion`
