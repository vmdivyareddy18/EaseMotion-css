Tada-Click Accordion (Responsive Dashboard Layout)

When you click the header above, observe the "Tada" animation on the chevron icon. It briefly scales down, then scales up while oscillating left and right before settling into its final 180-degree rotation.

This effect is powered entirely by CSS @keyframes, allowing it to remain lightweight and highly performant within complex responsive dashboard layouts where JavaScript threads might be busy parsing heavy chart data.

The accordion respects prefers-reduced-motion natively. If enabled, the Tada animation is bypassed, and the chevron simply rotates instantly without the oscillating scale effect.
