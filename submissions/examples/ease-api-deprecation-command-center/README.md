# API Deprecation Command Center Showcase

A premium operational dashboard built with pure HTML and CSS to visualize endpoint deprecation, migration progress, consumer impact, and rollout risks. Ideal for developer portals and API lifecycle management.

## EaseMotion classes used
- **Layout:** ease-container, ease-grid, ease-lg-col-span-2, ease-flex, ease-gap, ease-space-y, ease-items-center, ease-justify-between
- **Background:** ease-bg-slate-900, ease-bg-slate-800/50, etc.
- **Typography:** ease-text-4xl, ease-font-bold, ease-text-lg, ease-text-slate-400, ease-text-sm, ease-font-semibold
- **Spacing:** ease-py-12, ease-px-4, ease-mb-12, ease-mb-6, ease-mt-6, ease-p-5, etc.
- **Components:** ease-card
- **Animation:** ease-fade-in, ease-delay-200, ease-delay-300, ease-delay-500
- **Utilities:** ease-backdrop-blur (if available), ease-border, ease-rounded-*

## Sections
- Deprecation overview header
- Statistics cards (endpoints, migration, consumers, blockers)
- Endpoint status cards with migration progress bars
- Affected consumer list
- Sunset timeline
- Readiness matrix
- Risk & blocker register

## How to use
1. Open demo.html in any modern browser – no build step required.
2. Customize the data to fit your own API lifecycle.
3. Ensure the path to easemotion.css is correct.
