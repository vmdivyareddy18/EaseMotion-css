# CSS Tooltip Component

A pure CSS tooltip component built for EaseMotion CSS.

## Features

- Pure CSS implementation
- Four tooltip positions
  - Top
  - Bottom
  - Left
  - Right
- Smooth fade and slide animation
- Lightweight and reusable
- No JavaScript required

## Files

- demo.html
- style.css

## Usage

Apply the `ease-tooltip` class and provide tooltip text using the `data-tooltip` attribute.

Example:

```html
<button class="ease-tooltip" data-tooltip="Tooltip on top">
  Hover Me
</button>
