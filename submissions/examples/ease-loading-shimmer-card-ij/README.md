# Ease Loading Shimmer Card — Pulse Placeholder

> **PR:** #36781  
> **Component:** `<loading-shimmer-card>`

A card skeleton placeholder with a shimmering pulse overlay.

## CSS Custom Properties

| Property            | Default   | Description          |
|---------------------|-----------|----------------------|
| `--card-bg`         | `#1e293b` | Card background      |
| `--shimmer-color`   | `#334155` | Shimmer pulse color  |
| `--shimmer-duration`| `1.8s`    | Shimmer cycle time   |
| `--border-radius`   | `0.75rem` | Card border radius   |

## Usage

```html
<loading-shimmer-card></loading-shimmer-card>
```

## Features

- Animated shimmer sweep
- Configurable pulse color and speed
- Dark theme by default
