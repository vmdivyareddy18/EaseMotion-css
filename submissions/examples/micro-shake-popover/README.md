# CSS Micro Shake Popover

A responsive glassmorphism popover with a subtle micro-shake animation built using pure HTML and CSS.

## Features

- Pure HTML5 & CSS3
- Glassmorphism design
- Micro shake animation
- Keyboard accessible
- Responsive layout
- CSS custom properties
- No JavaScript required

## Folder Structure

```text
micro-shake-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.
