# Corporate Glow Pulse Card

## 1. What does this do?
It presents a polished, pure CSS glow-pulse card for modern corporate UI, with a soft ambient glow that strengthens on hover and keyboard focus.

## 2. How is it used?
```html
<article class="em-card" tabindex="0">
  <p class="em-card__eyebrow">Strategy pulse</p>
  <h1 class="em-card__title">Scale your next quarter with calm confidence.</h1>
  <p class="em-card__copy">Coordinate launches and customer momentum from one polished command center.</p>
</article>
```

## 3. Why is it useful?
It stays lightweight, responsive, semantic, and accessible while showing how EaseMotion CSS submissions can deliver refined motion without JavaScript.

### Customization
- `--em-duration`
- `--em-delay`
- `--em-easing`
- `--em-scale`
- `--em-glow-color`
