# Ease Gradient Beam Button

A modern button with an animated gradient light beam built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Animated light beam
- Responsive
- Lightweight
- Reusable button

## Structure

```
ease-gradient-beam-button-z1/
```

## Usage

Open `demo.html` in any browser.

A reusable micro-interaction demonstrating smooth motion with minimal code.
