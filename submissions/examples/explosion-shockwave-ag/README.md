# Explosion Shockwave

## What does this do?
An expanding shockwave ring effect simulating an explosion or impact with concentric rings radiating outward.

## How is it used?
Apply the `.shockwave` class to a container. Multiple rings expand outward sequentially to create a shockwave visual.

## Why is this useful?
Creates dramatic visual feedback for game events, button actions, or transitional effects in themed and gamified interfaces.
