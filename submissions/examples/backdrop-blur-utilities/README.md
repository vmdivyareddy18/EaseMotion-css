# Backdrop Blur Utilities

A lightweight collection of CSS utility classes for applying backdrop blur effects. These utilities are useful for glassmorphism, frosted-glass cards, overlays, and modern UI patterns.

## Features

* Pure CSS implementation
* Small, medium, and large blur levels
* Ideal for glassmorphism interfaces
* Lightweight and reusable
* Cross-browser compatible
* No JavaScript required

## Available Classes

| Class | Effect |
| ------- | ------- |
| `ease-backdrop-blur-sm` | Applies a 4px backdrop blur |
| `ease-backdrop-blur` | Applies an 8px backdrop blur |
| `ease-backdrop-blur-lg` | Applies a 16px backdrop blur |

## Usage

```html
<div class="ease-backdrop-blur-sm">Small Blur</div>

<div class="ease-backdrop-blur">Medium Blur</div>

<div class="ease-backdrop-blur-lg">Large Blur</div>
```

## Why is it Useful?

Backdrop blur is a key building block for glassmorphism and frosted UI designs. These utilities provide a simple, readable, and reusable way to add modern visual depth while staying aligned with EaseMotion CSS's human-readable philosophy.
