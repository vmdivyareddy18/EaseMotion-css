# Component: Mask Fade Edges

This submission implements a CSS mask-image fade edge effect for issue **#13619**.

## Description

A `.ease-mask-fade-edges` class that applies a `mask-image` with a `linear-gradient` to fade the edges of horizontally scrollable content. The fade direction is customizable via `--ease-mask-fade-direction`. Includes a scrollable card row with direction toggle buttons.

## Acceptance Criteria

- `.ease-mask-fade-edges` class with `mask-image: linear-gradient`
- Horizontal scroll container with hidden scrollbar
- `--ease-mask-fade-direction` controls fade orientation
- Interactive direction buttons to change fade orientation
- Works with horizontally scrollable content
