# Animated Job Application Tracker

A responsive dashboard component for tracking internship and job applications with animated progress indicators and status tracking.

## Features

- Application tracking cards
- Progress bar animations
- Status badges
- Statistics overview
- Hover interactions
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

- Company names
- Job titles
- Status labels
- Progress percentages
- Colors
- Animation timing

## Use Cases

- Internship tracking
- Job search dashboards
- Career management apps
- Student placement portals
- Recruitment systems
