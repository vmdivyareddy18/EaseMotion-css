# Automated Broken Link Checker

## Overview

This example demonstrates a documentation page for an **Automated Broken Link Checker** using GitHub Actions. The workflow periodically scans project documentation for broken hyperlinks and automatically reports any issues by creating a GitHub Issue.

The page is designed as a responsive documentation example for the EaseMotion CSS repository.

---

## Features

* Weekly scheduled GitHub Action
* Automatic link validation
* Broken link detection
* Automatic GitHub Issue creation
* Example GitHub Actions workflow
* Example link-check report
* Responsive documentation layout
* Pure HTML and CSS implementation

No build tools or installation are required.

---

## Demonstrated Workflow

The example illustrates the following automation process:

* GitHub Actions runs every week.
* Repository documentation is scanned for broken links.
* Any invalid URLs are detected automatically.
* A GitHub Issue is created when broken links are found.
* Maintainers receive a report for further action.
