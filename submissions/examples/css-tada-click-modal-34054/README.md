# CSS Tada-Click Modal for Product Catalog Layouts

## Overview

This project demonstrates a responsive product catalog modal with a smooth **Tada-Click** entrance animation built entirely using HTML and CSS. The component presents product information in a clean, modern card with engaging animations and requires no JavaScript.

---

## Features

- Pure HTML5 and CSS3
- Smooth Tada-Click entrance animation
- Modern product catalog modal
- Responsive design
- Accessible HTML structure
- CSS custom properties for easy customization
- Hover and active button effects
- Lightweight with no external libraries

## Preview

The modal contains:

- Product image/icon
- Product title
- Product description
- Customer rating
- Product specifications
- Price section
- "Add to Cart" button

### Animation

- The background overlay fades in smoothly.
- The modal appears with a **Tada-Click** animation using scale and rotation effects.
- The action button includes hover and active animations for improved interactivity.

## How to Use

1. Download or clone the project.

```
demo.html

```

3. Open `demo.html` in any modern web browser.

No JavaScript or build tools are required.

## Customization

The component uses CSS custom properties.

```css
:root{
    --primary:#2563eb;
    --primary-hover:#1d4ed8;
    --bg:#f4f6fb;
    --radius:22px;
    --duration:.8s;
}
```

You can customize:

- Primary colors
- Background color
- Border radius
- Animation duration
- Font styles
- Modal width
- Button appearance
- Shadow intensity
- Product image styling
