# Dog Grooming Salon Business Landing

A responsive business landing page showcasing a modern dog grooming salon built using EaseMotion CSS.

## Features

* Hero section
* Services grid
* Breed-specific services
* Before & After gallery
* Pricing by dog size
* Appointment booking form
* Responsive footer
* Mobile-friendly layout

## EaseMotion CSS Classes Used

* `ease-slide-up`
* `ease-slide-in-right`
* `ease-fade`
* `ease-hover-scale`
* `ease-hover-shadow`
* `ease-hover-lift`
