# ease-order-confirmation-card

## What this is
A CSS-only order confirmation card with a success check animation.

## Why
It gives checkout and order-complete screens a clearer success state while keeping the component lightweight and reusable.

## How to test
1. Open `demo.html` in a browser.
2. Confirm the success check animation plays on load.
3. Check that the card content is readable and visually balanced.
4. Resize the browser to verify responsive behavior.
5. Update the CSS custom properties and confirm the component still looks correct.

## Note
This is a visual demo component and uses only HTML and CSS.
