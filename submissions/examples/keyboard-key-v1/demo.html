Keyboard Key Styling Component

A CSS-only mechanical keycap component featuring a fluid pressed downward motion layout.

Light Theme (Inline Flow)

Press Ctrl Shift P to open the command palette.

Dark Theme Context

Use W A S D or ↑ arrow keys to steer navigation characters.

System Shortcuts

Copy Ctrl C , Paste Ctrl V , Cut Ctrl X , Undo Ctrl Z , Redo Ctrl Y , Select All Ctrl A , Save Ctrl S .

Browser & Dev Tools

Find Ctrl F , New Tab Ctrl T , Close Tab Ctrl W , Reopen Tab Ctrl Shift T , Dev Tools Ctrl Shift I .
