# Animated Progress Bar

## Description

A modern animated progress bar component with smooth loading effects, percentage labels, and multiple design variants.

## Features

- Smooth loading animation
- Percentage indicators
- Basic progress bar
- Gradient progress bar
- Rounded style
- Animated striped effect
- Compact variant
- Responsive layout
- Hover brightness effect

## Variants

### Basic
Simple solid color progress indicator.

### Gradient
Colorful gradient filled progress bar.

### Rounded
Extra rounded progress bar style.

### Striped
Animated moving stripe effect.

### Compact
Minimal progress bar with smaller height.

## Usage

Include `demo.html` and `style.css` to preview the component.

## Accessibility

Designed with clear labels and readable progress percentages.
