# Ease API Response Status

## Overview
A CSS-only dashboard component for displaying API response statuses.

## Features
- Animated status badges
- Responsive layout
- Hover interactions
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in your browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component designed for developer dashboards, API monitoring tools, and admin interfaces.
