# Conic Progress Ring

A CSS-only circular progress indicator built using `conic-gradient()`.

## Features

- Pure CSS implementation
- No JavaScript required
- Smooth entrance animation
- Hover scale effect
- Multiple progress variants (25%, 50%, 75%, 100%)
- Responsive layout

## Preview

Displays animated circular progress rings commonly used in:

- Dashboards
- Analytics Panels
- Portfolio Websites
- Loading Indicators
- Performance Metrics

## File Structure
```

conic-progress-ring/
├── demo.html
├── style.css
└── README.md

```

## Usage

Open `demo.html` directly in a browser.

The component uses CSS `conic-gradient()` to create visually appealing progress rings.

## Technologies Used

- HTML5
- CSS3
- Conic Gradient
- CSS Animations
- Flexbox

## Contribution Type

Submission-only example for EaseMotion CSS.

No framework files are modified.
