# Bounce Badge

A reusable animated badge component for EaseMotion CSS.

## Features

- Pure CSS animation
- No JavaScript required
- Responsive layout
- Keyboard accessible
- Uses EaseMotion naming (`ease-*`)
- Beginner-friendly example

## Files

```
demo.html
style.css
README.md
```

## Animation

The badge performs a smooth bounce animation when hovered or focused.

## Accessibility

- Supports keyboard focus using `:focus-visible`
- High contrast text
- Responsive on mobile devices

## Preview

Hover over any badge to see the bounce effect.
