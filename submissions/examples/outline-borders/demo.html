Creative Outline Borders

CSS outline & outline-offset let you create multi-layered, decorative borders without extra elements.

double
outline-style: double
groove
outline-style: groove
offset -
outline-offset: -4px
offset +
outline-offset: 8px
ridge
outline-style: ridge
gradient
outline + gradient bg

Key properties

outline Shorthand for outline-width, outline-style, outline-color
outline-offset Space between border and outline (accepts negative values!)
outline-style Supports all border styles + auto for default focus
