# CSS Outline Borders

1. **What does this do?** Shows creative uses of `outline`, `outline-offset`, and `outline-style` to create multi-layered decorative borders with a single element.

2. **How is it used?** Apply `outline: 3px solid color` and adjust `outline-offset` for positive (outside) or negative (inside) positioning relative to the border box.

3. **Why is it useful?** Most developers only use outlines for focus indicators. Combined with `outline-offset` and non-standard styles like `groove`/`ridge`, outlines create sophisticated border effects without extra HTML.
