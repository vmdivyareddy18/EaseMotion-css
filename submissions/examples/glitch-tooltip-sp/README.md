# Glitch Tooltip

A gaming-inspired tooltip with a neon glitch animation built entirely using HTML and CSS.

## Features

- 🎮 Gaming-inspired design
- ⚡ Animated glitch effect
- 💻 Pure HTML & CSS
- 📱 Responsive layout
- ♿ Accessible hover interaction
- 🚫 No JavaScript required

## File Structure

```
glitch-tooltip-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

Hover over the button to reveal the animated glitch tooltip.
