CSS Containment for Performance

Scoped Rendering with contain

Shows how CSS containment scopes paint and layout cycles — improving rendering performance in complex UIs.

Containment Variants

contain: layout
Scopes layout recalculations to this element. Other elements do not reflow when this changes.
LAYOUT
contain: paint
Prevents overflow from being painted outside this element bounds.
PAINT
contain: strict
Combines layout, paint, style, and size — the most aggressive containment.
STRICT
contain: content
Combines layout and paint only — style isolation without size constraints.
