# CSS Fade-Out Accordion for Accessible Web Layouts

## Overview

A lightweight accordion component built entirely with HTML and CSS, designed for accessible web interfaces. The component uses semantic HTML elements and smooth fade transitions to present information clearly while remaining responsive and keyboard friendly.

---

## Features

- Pure HTML5 & CSS3
- CSS-only fade transition
- Accessible `<details>` and `<summary>` elements
- Keyboard friendly
- Responsive layout
- CSS custom properties
- Modern minimal design
- No JavaScript required
