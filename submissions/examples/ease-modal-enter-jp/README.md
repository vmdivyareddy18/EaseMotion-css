# Ease Modal Enter

A premium modal popup animation built using pure HTML and CSS featuring backdrop fade, content scale-up, and smooth entrance transitions.

## Features

- Backdrop Fade Animation
- Modal Scale-Up Entrance
- Smooth Opacity Transition
- Glassmorphism Design
- Responsive Layout
- Pure HTML & CSS
- No JavaScript Required
