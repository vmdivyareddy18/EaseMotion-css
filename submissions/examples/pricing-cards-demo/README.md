# Pricing Cards Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be applied to a modern responsive pricing section.

---

## Included Components

- Basic Plan
- Pro Plan
- Enterprise Plan
- "Most Popular" badge
- Animated buttons
- Responsive layout

---

## Folder Structure

```
submissions/examples/pricing-cards-demo/
├── demo.html

```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to test responsiveness.
3. Observe how the existing EaseMotion animation classes enhance the pricing cards.
