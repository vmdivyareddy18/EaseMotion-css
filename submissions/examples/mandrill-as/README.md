# Mandrill

A pure CSS and HTML mandrill face, the most colourful of primates, staring out of the rainforest.

## What it shows

- The bristling mane and yellow chin tuft drawn from repeating conic gradients, each masked to a ring or fan
- Ridged blue muzzle flanks from a repeating linear gradient of two blues
- A red nose stripe with paired nostril holes punched by radial gradients
- An occasional blink driven by scaleY on the eyes
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
