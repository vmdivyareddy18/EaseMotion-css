# CSS Elastic Slide Tooltip - Organic Craft Layouts

A pure CSS animated tooltip component inspired by organic craft and handmade design interfaces.

## Features

- Pure CSS animation
- Elastic slide transition effect
- Natural craft themed UI
- Responsive layout
- Keyboard accessible
- Custom CSS animation variables


## Usage

Open `demo.html` directly in a browser.

Hover or focus on craft buttons to display the animated tooltip.


## Customization

Update animation values:

```css
:root {
    --tooltip-time: 0.45s;
    --tooltip-scale: 1;
    --tooltip-easing: cubic-bezier(.68,-0.55,.27,1.55);
}
