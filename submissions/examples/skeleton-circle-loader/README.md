# Circular Skeleton Shimmer + Profile Card Skeleton

This project demonstrates a **loading UI skeleton system** using shimmer animations. It includes circular loaders and a profile card skeleton with avatar and text placeholders.

---

## ✨ Features

- Circular skeleton loaders (sm, md, lg)
- Profile card skeleton UI
- Circular avatar shimmer
- Text line skeleton shimmer
- Smooth infinite shimmer animation
- Fully centered responsive layout
