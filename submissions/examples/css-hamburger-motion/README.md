# ☰ CSS Hamburger Motion

A collection of beautiful, pure CSS animated hamburger menu icons for modern responsive websites.

## 🚀 Features

### Hamburger Styles (10 types)
1. **Classic X** - Traditional X transformation
2. **Arrow** - Transforms into left arrow
3. **Squeeze** - Middle bar squeezes out
4. **Spin** - Whole icon spins 180°
5. **Elastic** - Elastic bounce effect
6. **Collapse** - Bars collapse with scale
7. **Rotate Arrow** - 90° rotation to arrow
8. **Slider** - Middle bar slides out
9. **Cross Fade** - Color change with fade
10. **Neon** - Cyberpunk neon glow

### Animation Features
- Smooth bar transformations
- Rotation and scaling
- Elastic easing effects
- Slide and fade animations
- Neon glow pulsing
- Color transitions
- Hover state effects

## 📂 Files
- `demo.html` - Main HTML structure
