# CSS Zoom-Out Modal for Neumorphic Soft Layouts

A lightweight, responsive **CSS-only Zoom-Out Modal** designed for **Neumorphic Soft** interfaces. The component uses a smooth zoom-out transition, requires **no JavaScript**, and is fully customizable using CSS custom properties.

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Smooth Zoom-Out modal animation
- Soft Neumorphic UI design
- Fully responsive
- Keyboard accessible
- Customizable using CSS variables
- Lightweight and reusable

## 🚀 Usage

1. Open `demo.html` in any modern browser.
2. Click the **Open Modal** button.
3. The modal appears with a smooth Zoom-Out transition.
4. Click **Close** to dismiss the modal.

## 🎨 Customization

Edit the CSS variables in `style.css`:

```css
:root {
  --duration: 0.5s;
  --ease: cubic-bezier(0.22, 1, 0.36, 1);

  --primary: #5b5ce6;
  --primary-hover: #4546d4;

  --radius: 22px;
}
```

You can customize:

- Animation duration
- Animation easing
- Primary colors
- Border radius
- Shadows
- Background colors
