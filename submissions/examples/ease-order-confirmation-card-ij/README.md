# Order Confirmation Card

A success confirmation card with animated checkmark.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--card-bg` | `#2d2d44` | Card background |
| `--success-color` | `#4ecdc4` | Checkmark/accent color |
| `--text-color` | `#ccc` | Detail text color |
| `--check-duration` | `0.5s` | Checkmark draw speed |
