# Destination Wedding Planner Template
A romantic, high-end template designed for destination wedding planners, bridal travel coordinators, and international event organizers.

- **Design**: Uses an "Ethereal Romance" palette with blush pinks, soft golds, and ocean-inspired teals to evoke luxury, dream-like locations, and celebratory elegance.
- **Use Case**: Destination wedding planners, luxury honeymoon coordinators, and boutique travel-event agencies.
