# Button Animation Showcase

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be applied to common button styles.

---

## Included Buttons

- Primary Button
- Secondary Button
- Outline Button
- Rounded Button
- Icon Button

---

## Folder Structure

```
submissions/examples/button-animation-showcase/
```

---

## Files

- demo.html
- style.css

---

## How to Run

Open `demo.html` in your browser.

Hover over or interact with the buttons to observe the applied animation classes.

---

## Benefits

- Demonstrates real-world usage of EaseMotion CSS.
- Beginner-friendly example.
- Responsive layout.
- Useful as a component reference.
