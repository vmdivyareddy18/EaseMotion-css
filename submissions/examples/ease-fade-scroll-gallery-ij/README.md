# Fade Scroll Gallery

A gallery grid with staggered fade-in and slide-up entry animations, simulating a scroll-triggered reveal.

## Features

- Staggered fade-in animation for gallery items
- Slide-up and scale entry effect
- Hover scale and shadow effect on images
- Responsive CSS Grid layout
- Gradient color backgrounds with emoji icons

## Usage

Include `style.css` and structure gallery items with `.gallery-item` inside `.gallery`. Use CSS custom property `--i` on each item for staggered animation delays.
