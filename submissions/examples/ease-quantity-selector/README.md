# Quantity Selector

A product quantity +/- selector with button animations — pure CSS.

## Features
- Hover scale + background transition on buttons
- Active press (scale down) effect
- Clean bordered layout
- Pure CSS, no JavaScript

## Usage
```html
<div class="qty-selector">
  <button class="qty-btn">−</button>
  <span class="qty-value">1</span>
  <button class="qty-btn">+</button>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
