# 🧩 Tetris

A classic Tetris puzzle game built with HTML, CSS, and JavaScript. Stack falling blocks, complete horizontal lines, and score points as the game speeds up over time.

## 🎮 How to Play

1. Click **"Start"** or press **Space** to begin
2. **Move** the falling piece left/right using arrow keys
3. **Rotate** the piece with the Up arrow
4. **Drop** the piece faster with Down arrow
5. **Hard Drop** instantly with Space
6. **Complete lines** to score points
7. Game speeds up as you level up!

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `←` | Move Left |
| `→` | Move Right |
| `↑` | Rotate Piece |
| `↓` | Soft Drop |
| `Space` | Hard Drop / Start |
| `R` | Reset Game |

## ✨ Features

- 🧩 **7 Classic Tetrominoes** – I, O, T, S, Z, L, J shapes
- 📈 **Progressive Difficulty** – Speed increases with level
- 🎯 **Scoring System** – Points based on lines cleared (1, 2, 3, 4)
- 🏆 **High Score** – Saved in browser local storage
- 👻 **Ghost Piece** – Shows where the piece will land
- 🎨 **Visual Feedback** – Glowing blocks and smooth animations
- 📱 **Responsive Design** – Works on desktop and mobile
- ⌨️ **Keyboard Controls** – Full keyboard support

## 🎯 Scoring

| Lines Cleared | Points |
|---------------|--------|
| 1 Line | 100 × Level |
| 2 Lines | 300 × Level |
| 3 Lines | 500 × Level |
| 4 Lines (Tetris) | 800 × Level |

## 📊 Level Progression

- Level increases every **10 lines cleared**
- Speed increases with each level
- Maximum speed at level 10+
