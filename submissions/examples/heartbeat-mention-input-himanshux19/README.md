# Heartbeat Mention Input

A responsive mention input component with a subtle heartbeat animation built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Heartbeat focus animation
- Responsive layout
- Accessible form controls
- Modern e-commerce inspired design
- No JavaScript required

## Files

```text
demo.html
style.css
README.md
```

## Usage

```html
<div class="mention-box">
  <label for="mention">Mention</label>

  <textarea
    id="mention"
    placeholder="@username Write your message..."
    rows="5"
  ></textarea>
</div>
```

## Accessibility

- Semantic label
- Keyboard accessible
- Responsive
- Visible focus state

## Why EaseMotion CSS?

This component demonstrates how motion can subtly enhance user interaction using a lightweight heartbeat animation while remaining simple, reusable, and entirely CSS-based.
