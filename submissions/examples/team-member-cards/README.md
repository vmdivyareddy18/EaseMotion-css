# Animation: Team Member Cards

A modern team showcase component featuring profile avatars, role badges, and social links that appear on hover.

## Features

- Pure HTML & CSS
- Responsive team grid
- Hover elevation effect
- Social links reveal
- Role badges
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="member-card">

    <div class="avatar">A</div>

    <h3>Aryan Kumar</h3>

    <span class="role">
        Frontend Developer
    </span>

</div>
```

## Why is it useful?

Team showcases are common on company websites, startup landing pages, portfolios, and agency sites. This component presents team members in a clean, modern layout while adding subtle motion and interaction.
