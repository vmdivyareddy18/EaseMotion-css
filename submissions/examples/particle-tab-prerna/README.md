# Minimalist Particle Tabs

1. What does this do? It is a zero-JavaScript, pure CSS tab component that triggers hardware-accelerated minimalist particle animations (such as radial bursts, rising sparks, or horizontal pulses) when a tab is selected.
2. How is it used? Place hidden `.tab-input` radio inputs alongside a `.tabs-nav` navigation bar, and apply interactive label wrappers like `.tab-label-burst` (containing `.particles-burst`), `.tab-label-rise` (containing `.particles-rise`), or `.tab-label-pulse` (containing `.particles-pulse`).
3. Why is it useful? It aligns with the EaseMotion CSS philosophy of building rich, premium micro-interactions and transitions with absolute zero JavaScript overhead, utilizing responsive and accessible CSS rules that adapt to dark modes and respect user motion preferences.
