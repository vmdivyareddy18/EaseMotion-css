# Animated Light Bulb Idea Glow

## Overview

A reusable light bulb icon animation featuring a glowing effect suitable for tips, ideas, hints, AI assistants, educational platforms, and productivity tools.

## Features

- CSS-only animation
- Glow and scale effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="idea-button">
  <span class="idea-icon">💡</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
