# Orphans Widows

orphans / widows — Controls page breaks inside and after paragraphs

## Features
- Uses `--ease-color-surface`
- Uses `--ease-text-base`
- Uses `--ease-color-muted`
- Uses `--ease-color-text`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Orphans Widows</span>
</div>
```

## Why is it useful?

This submission demonstrates the orphans widows CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
