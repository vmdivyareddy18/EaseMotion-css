# VORTEX — Elite Sports Nutrition E-Commerce

A full-page, fully responsive e-commerce landing page designed for a high-performance fitness nutrition and supplement brand. Built exclusively utilizing the `EaseMotion` CSS utility class architecture to deliver an aggressive, high-energy dark-mode aesthetic.

## ⚡ Theme & Aesthetic
**High-Performance Athletic:**
* **Colors:** Deep Carbon Black (`#090909`), Dark Surface Card (`#141414`), Neon Cyan (`#00E5FF`), and Pure White (`#FFFFFF`).
* **Typography:** `Oswald` (a bold, condensed sans-serif for aggressive, high-impact headings) paired with `Inter` (a clean sans-serif for UI elements and readable body copy).
* **Vibe:** Intense, scientific, results-driven, and premium.

## 🚀 How to Run
This template requires **zero build steps**. 
1. Clone or download the repository.
2. Ensure `demo.html` and `style.css` are in the same directory.
3. Open `demo.html` directly in your favorite web browser.

## 🛠️ EaseMotion Classes Showcased
This project relies entirely on the core EaseMotion CSS utility framework to build complex grids and smooth micro-interactions.

### Layout & Structure
* `ease-container`
* `ease-flex`, `ease-grid`
* `ease-grid-cols-2`, `ease-grid-cols-3`, `ease-grid-cols-4`
* `ease-gap-4`, `ease-gap-6`, `ease-gap-8`, `ease-gap-12`
* `ease-items-center`, `ease-justify-between`, `ease-justify-center`
* `ease-relative`

### Theming Utilities
* `ease-bg-main`, `ease-bg-surface`, `ease-bg-primary`, `ease-bg-light`
* `ease-text-main`, `ease-text-primary`, `ease-text-white`, `ease-text-muted`
* `ease-border`, `ease-border-primary`, `ease-border-y`
* `ease-font-bold`, `ease-font-display`

### Animations & Scroll Reveals
* `ease-reveal`, `ease-reveal-up`, `ease-reveal-scale` (Intersection Observer triggered for smooth scroll-in)
* `ease-fade-in`, `ease-slide-up`, `ease-bounce-in` (Initial page load transitions)
* `ease-delay-100` through `ease-delay-500` (Staggered element loading)

### Micro-Interactions (Hover)
* `ease-hover-lift` (Y-axis translation + glowing shadow expansion for buttons and cards)
* `ease-hover-scale` (Subtle zoom for goal cards and bundle images)
* `ease-hover-fade` (Subtle opacity shift on logos)
* `ease-hover-underline` (Animated neon bottom border on navigation links)

## 📱 Responsiveness
The layout adapts aggressively for different viewports. Desktop 4-column goal grids and 3-column pricing tables smoothly degrade to 2-column on tablets and a stacked, single-column design on mobile devices. Typography utilizes `clamp()` to scale heading sizes perfectly across viewports.
