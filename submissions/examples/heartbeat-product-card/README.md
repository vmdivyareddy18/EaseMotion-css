# Heartbeat Product Card

A pure HTML and CSS animated product card component built for EaseMotion CSS.

## Features

- Heartbeat pulse animation
- Modern e-commerce card design
- Hover interaction
- Responsive layout
- No JavaScript required

## Usage

Add the HTML:

```html
<section class="product-card">
  <h2>Product Name</h2>
  <button class="buy-btn">
    Add to Cart
  </button>
</section>
