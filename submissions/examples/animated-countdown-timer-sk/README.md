# Animated Countdown Timer

A CSS-only countdown timer with flip-card digit transitions and a glowing accent on the active time unit.

## Created By

[@saurabhhhcodes](https://github.com/saurabhhhcodes)

## Features

- **Days / Hours / Minutes / Seconds** boxes
- **Flip animation** using CSS pseudo-elements and `rotateX`
- **Glowing accent ring** on the currently active unit
- Fully responsive

## Usage

1. Link `style.css` in your HTML.
2. Use the component markup with data attributes for displayed values.
