# Animated Course Progress Tracker

A responsive course progress tracking interface demonstrating learning modules, completion percentages, and animated progress indicators.

## Features

- Multiple course cards
- Progress bars with completion states
- Status badges
- Hover animations
- Responsive layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more courses
- Change progress percentages
- Update colors and typography
- Modify animation timing
