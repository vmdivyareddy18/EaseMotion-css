# Weather Rain Icon

1. What does this do? Displays a static cloud icon with small animated raindrop lines falling continuously beneath it, at weather app icon scale.

2. How is it used? Apply `.weather-icon` and `.rain-icon` to a container with `.rain-cloud` and `.rain-drop` child elements.

3. Why is it useful? It offers a lightweight, pure-CSS rain animation ideal for weather forecast cards and app launcher icons, with no JavaScript or external assets needed.
