# Issue #22741 - Advanced: E-commerce - Specialty Coffee Brand

## What this submission adds

This submission adds a complete premium one-page DTC specialty coffee landing page for **Veloura Roast** using only:

- `demo.html`
- `style.css`
- existing EaseMotion CSS classes from the repository

The page is designed to open directly in a browser with no build step, no JavaScript, and no external assets.

## Required sections included

- Hero with inline SVG coffee illustration
- Coffee origins grid with flavor profiles
- Subscription box CTA
- Brewing guides
- Roastery story
- Product showcase / coffee collection
- Reviews / testimonials
- Footer

## EaseMotion classes used

The page uses existing EaseMotion motion and interaction classes, including:

- `ease-fade-in`
- `ease-slide-up`
- `ease-delay-100`
- `ease-delay-150`
- `ease-delay-200`
- `ease-delay-300`
- `ease-delay-400`
- `ease-delay-500`
- `ease-hover-lift`
- `ease-hover-glow`
- `ease-hover-underline`
- `ease-input`
- `ease-select`

## How to open `demo.html`

Open the file directly in any modern browser:

1. Navigate to `submissions/examples/22741-specialty-coffee-brand/`
2. Double-click `demo.html`

It links to the repository's local `easemotion.min.css`, so no server is required.

## Responsiveness

- Mobile-first structure with stacked layouts on small screens
- Expands into multi-column editorial layouts on tablet and desktop
- Hero, cards, forms, and footer all adapt without JavaScript

## Browser compatibility

Designed for standards-based modern browsers that support:

- CSS Grid
- Flexbox
- Inline SVG
- Gradients and backdrop blur

Expected to work well in current versions of:

- Chrome
- Edge
- Firefox
- Safari
