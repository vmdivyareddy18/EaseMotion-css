# Queue Health Compass

An intermediate EaseMotion CSS example for operations queue dashboards. A compass needle rotates toward the highest load while backlog metrics animate underneath.

## Files

- `demo.html` - Semantic queue health compass markup.
- `style.css` - Responsive panel layout, animated compass needle, and staggered metric rows.

## Usage

Open `demo.html` in a browser to preview the animated queue health compass.
