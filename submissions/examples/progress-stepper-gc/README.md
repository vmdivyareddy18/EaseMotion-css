# Progress Stepper UI Example

## 1. What does this do?

This submission adds a multi-step progress stepper component with animated step transitions and an active state motion effect. It visually communicates progress through sequential interface steps.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes a stepper bar with active step transitions and a progress indicator. Copy the `.progress-stepper` markup and CSS into your project to display multi-step progress.

---

## 3. Why is it useful?

Progress steppers guide users through workflows and indicate completion states. Animated transitions improve clarity and make step changes feel smoother and more intuitive.
