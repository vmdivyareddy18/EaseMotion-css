# Hover Wobble Animation

## What does this do?
This adds a smooth, playful "wobble" animation that gently shakes and tilts an element from left to right when a user hovers over it.

## How is it used?
Apply the `.hover-wobble` class to any element (such as a button, card, or icon) that you want to animate on hover.

Example:
```html
<button class="hover-wobble">
  Hover me!
</button>
```

## Why is it useful?
- Improves visual feedback by providing a clear, satisfying response when a user interacts with an element.
- Enhances user interaction by making the interface feel more alive and engaging.
- It is highly suitable for buttons, cards, and CTAs without being overly distracting.
