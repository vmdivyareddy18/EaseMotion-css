# Navigation Components

A set of responsive, animated navigation components designed to act as the primary structural shell for web applications. Includes a sticky top Navbar and a collapsible side Sidebar.

## Files
- `demo.html` - Interactive demonstration of a full dashboard shell utilizing both components.
- `style.css` - The CSS implementation utilizing EaseMotion tokens.

## Details
This addresses the "Navigation components (navbar, sidebar)" requirement currently listed as "Planned" in the project Roadmap. 
- **Navbar**: Features a pure-CSS mobile hamburger menu toggle using the checkbox hack.
- **Sidebar**: Features smooth hover-indent animations and highlighted active states.
Both components require zero JavaScript to function properly.
