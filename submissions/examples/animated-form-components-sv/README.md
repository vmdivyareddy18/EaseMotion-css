# Animated Form Components

## What does this do?

This example provides a collection of modern animated form elements built entirely with HTML and CSS:

- Floating label text inputs
- Animated custom checkboxes
- CSS-only toggle switches
- Interactive submit button

All interactions are lightweight and accessible, with support for reduced-motion preferences.

---

## How is it used?

### Floating Input

```html
<div class="input-group">
  <input type="text" required>
  <label>Username</label>
</div>
```

### Custom Checkbox

```html
<label class="checkbox-wrapper">
  <input type="checkbox">
  <span class="custom-checkbox"></span>
  Remember Me
</label>
```

### Toggle Switch

```html
<label class="toggle-wrapper">
  <input type="checkbox" class="toggle-input">
  <span class="toggle-slider"></span>
</label>
```

---

## Why is it useful?

Forms are one of the most commonly used UI patterns across websites and applications.

These components provide:

- Better user feedback
- Modern interactive states
- Improved visual polish
- Consistent styling across projects
- No JavaScript dependencies

The component aligns with EaseMotion CSS's animation-first philosophy while remaining lightweight and reusable.
