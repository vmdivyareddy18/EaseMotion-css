# Animated Smart Home Control Dashboard

A responsive smart home dashboard component built with HTML and CSS animations.

## Features

- Animated smart home dashboard
- Connected room cards
- Energy usage progress bar
- Security status panel
- Responsive layout
- Smooth hover and reveal animations

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

- CSS Animations
