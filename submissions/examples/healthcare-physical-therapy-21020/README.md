# Healthcare: Physical Therapy Practice (`healthcare-physical-therapy-21020`)

This proposal provides a complete, copy-paste ready B2C healthcare landing page designed for a physical therapy or rehabilitation clinic, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A physical therapy site needs to convey clinical expertise and trust while remaining warm, encouraging, and focused on recovery. This template achieves a "clinical yet warm" aesthetic using a crisp sans-serif font (Inter), a clean white/gray background scheme, and a color palette featuring a calming medical blue and a healing green.

The layout includes:
- A hero section featuring abstract CSS "recovery waves", a welcoming headline, and a floating stat card.
- A 3-column service grid categorized by body part/condition.
- A vertical "Treatment Approach" timeline documenting the patient journey.
- Professional therapist profile cards with hover-scaling portraits.
- A dedicated insurance acceptance banner.
- A comprehensive appointment request form with integrated contact details.
- A structured, professional clinic footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (For standard, clean entrances of text blocks, service cards, and timeline steps)
- `.ease-zoom-in` (Applied to the insurance banner and hero visual to draw focus to critical trust elements)
- `.ease-hover-scale` (Applied to therapist portraits to make the team feel interactive and approachable)
- `.ease-hover-lift` (Standard interaction for primary booking buttons)
- `.ease-slide-left` / `.ease-slide-right` (Used in split-layout sections, such as the booking form, to bring content together)
- `.ease-focus-line` (Applied to form inputs to provide a clean, clinical bottom-border highlight upon focus)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, timeline structure, and clean clinical aesthetics, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/healthcare-physical-therapy-21020/` directory to avoid modifying core files directly and to ensure zero deletions.*
