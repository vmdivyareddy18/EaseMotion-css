# SaaS Pricing Intelligence Platform

An ultra-premium subscription management and pricing optimization dashboard featuring multi-tier plans, ROI calculators, revenue bar charts, and competitor analytics. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Branding Header**: Logo layout and billing model status badge indicators.
- **Annual Billing Toggle**: Switch toggles Monthly/Annual pricing with glowing discount indicators.
- **SaaS Pricing Cards**: Four layouts (Sandbox, Pro, Premium, Enterprise) with checklist bullet features.
- **ROI Calculator Slider**: Real-time event calculator forecasting values.
- **Revenue Breakdown Chart**: Bar visualization detailing plan MRRs.
- **Value Segmentation Table**: Telemetry tracking segments, user sizes, and churn risk parameters.
- **Billing Ledger**: Detailed transaction history tables showing invoicing, plans, amounts, and statuses.
- **Competitor Pricing Matrix**: Detailed comparison tables detailing competitive advantages.
- **A/B Testing Insights**: Comparison graphs demonstrating revenue growth from price modifications.
- **Price Optimization Suggestions**: AI feedback lists giving recommended upgrade configurations.

## 🎨 Design Theme

- **Palette**: Midnight Slate (`#0f172a`), Dark Surface (`#1e293b`), Indigo Purple (`#6366f1`), Growth Emerald (`#22c55e`), Churn Rose (`#f43f5e`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Transitions**: Spotlight hover highlights, vertical card lifts, and soft transition fills.
