@property Gradient Button

True CSS gradient interpolation via registered typed custom properties

conic spin — --grad-angle
hue shift — --grad-color-start
border ring — --border-hue
@property registers typed custom properties ( <angle>, <color>) that the browser can interpolate inside @keyframes. Without it, gradient stops cannot be animated natively. Requires Chrome 85+, Firefox 128+, Safari 16.4+.
