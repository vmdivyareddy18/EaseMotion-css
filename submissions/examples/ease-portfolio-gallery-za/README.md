# CSS Portfolio Gallery Component

## What does this do?
A responsive portfolio gallery with filterable categories, overlay project details with hover reveal, statistics counter section, call-to-action, and a comprehensive project showcase grid with 8 portfolio items.

## How is it used?
```html
<section class="pf-gallery">
  <div class="pf-item pf-web">
    <div class="pf-item-img" style="background:gradient">
      <div class="pf-overlay">
        <h3>Project Title</h3>
        <p>Description text goes here</p>
        <button class="pf-btn">View Case Study</button>
      </div>
    </div>
    <div class="pf-item-info">
      <h3>Project Title</h3>
      <span>Category</span>
    </div>
  </div>
</section>
```

## Why is it useful?
Creates a professional portfolio showcase for designers and agencies. Features filterable category system with six categories, hover-reveal overlay with project details, skill tags, and case study buttons. Includes statistics display section and call-to-action. Fully responsive with smooth hover animations and CSS custom property theming.
