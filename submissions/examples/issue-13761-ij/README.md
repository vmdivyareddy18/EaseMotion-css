# ease-wifi-pulse

**Closes #13761**

A WiFi icon with concentric arc pseudo-elements that pulse outward with staggered opacity.

## Acceptance Criteria

- [ ] Three concentric arcs pulse outward from the WiFi icon
- [ ] Arcs have staggered animation delays (0s, 0.4s, 0.8s)
- [ ] Toggle button activates/deactivates the signal animation
- [ ] Dark theme with blue primary color for arcs

## Files

- `style.css` – WiFi arc pulse keyframes, staggered delays
- `demo.html` – WiFi icon with toggle for pulsing arcs
- `README.md` – Issue documentation
