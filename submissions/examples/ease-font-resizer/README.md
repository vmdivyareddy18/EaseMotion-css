# Font Resizer – Buttons to increase/decrease text size

> A production-ready font resizer component that allows users to increase or decrease text size with simple buttons, making content more accessible and customizable. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Increase/Decrease** – Adjust font size with buttons
- ✅ **Real-time preview** – See changes instantly
- ✅ **Keyboard shortcuts** – Ctrl++ (increase), Ctrl+- (decrease), Ctrl+0 (reset), Ctrl+T (toggle)
- ✅ **Toggle resizer** – Enable/disable functionality
- ✅ **Reset to default** – One-click reset
- ✅ **Size tracking** – Shows current size and steps
- ✅ **Multiple instances** – Independent resets for different sections
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
