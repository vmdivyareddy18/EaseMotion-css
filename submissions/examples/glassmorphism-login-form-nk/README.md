# Glassmorphism Login Form

## What does this do?

Creates a reusable glassmorphism-style login form with frosted glass effects, animated focus states, and accessibility support.

## How is it used?

```html
<div class="login-card">
  ...
</div>
```

## Why is it useful?

It demonstrates modern authentication UI design patterns using glassmorphism, smooth interactions, responsive layouts, and reduced-motion accessibility support while remaining fully self-contained.
