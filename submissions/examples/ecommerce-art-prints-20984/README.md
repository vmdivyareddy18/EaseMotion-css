# App Landing: Art Prints & Posters E-commerce (`ecommerce-art-prints-20984`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a modern, curated art print gallery, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A modern art retailer requires a minimalist, stark, gallery-like aesthetic to allow the artwork to command attention. This template achieves a clean feel using a strict black/white/gray color palette with a single "Gallery Blue" accent, ample whitespace, and bold, structural typography (`Space Grotesk` paired with `Inter`).

The layout includes:
- A high-contrast Announcement Bar.
- A sticky Navigation with a frosted glass effect and subtle opacity hover states.
- A Hero section featuring a split layout with a CSS-built gallery wall simulation.
- A "Curated Collections" 4-column grid with abstract CSS art mockups.
- An Augmented Reality Visualizer CTA featuring a custom CSS mobile phone projecting a pulsing AR frame.
- An Artist Spotlight section with an overlapping grid layout.
- A gallery-grade Framing Options grid.
- A clean Collector Reviews section.
- A dark-mode Custom Print CTA.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use snappy, highly polished `cubic-bezier(0.215, 0.61, 0.355, 1)` timings (fast out, smooth deceleration) to feel modern and responsive:
- `.ease-fade-up` / `.ease-slide-up` (Configured for sharp, structural entrances).
- `.ease-reveal-bottom` / `.ease-reveal-left` (Used for the hero frames and AR visualizer phone to slide into view smoothly).
- `.ease-sway` / `.ease-float` (Custom continuous animations applied to abstract elements in the hero art to give life to the mockups).
- `.ease-pulse` (Applied to the AR bounding box in the visualizer section).
- `.ease-hover-fade` (Custom interaction class that drops opacity to 50% for a subtle UI feel).
- `.ease-hover-zoom` (Applied to collection cards to scale the artwork up within its container on hover).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the minimalist gallery layout, the abstract CSS artwork/frame simulations, and the typography styling. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the snappy, modern entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
