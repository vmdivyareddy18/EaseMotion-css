# Animated Notification Pulse Badge

A reusable notification badge animation built with pure CSS.

## Features

- Pulse notification effect
- Lightweight CSS-only implementation
- Works with icons, avatars, and buttons
- Smooth infinite animation
- Responsive and reusable

## Folder Structure

```text
r-notification-pulse-badge/
├── README.md
├── demo.html
└── style.css
```

## Usage

Open `demo.html` in any modern browser.
