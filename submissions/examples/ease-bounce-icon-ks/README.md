# Bounce Icon Wave

## What does this do?
Triggers a repeating vertical bounce animation on the inner SVG icon when the button container is hovered.

## How is it used?
Add the class to your button containing an SVG icon:

```html
<button class="bounce-icon-btn">
  <span>Upload Files</span>
  <svg>...</svg>
</button>
```

## Why is it useful?
It adds playful, micro-interaction feedback to functional buttons (upload, download, arrow links) that draws visual interest and encourages clicks.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
