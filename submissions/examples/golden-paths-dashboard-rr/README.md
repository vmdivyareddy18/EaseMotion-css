# Platform Engineering Golden Paths Dashboard Component

## What does this do?

Provides a centralized interface for tracking Golden Path adoption, workflow completion, onboarding success, compliance coverage, and developer engagement.

## How is it used?

```html
<div class="path-card healthy">
    <h3>Service Creation Path</h3>
</div>
```

## Why is it useful?

Golden Paths help organizations standardize engineering workflows and accelerate developer productivity. This component enables platform teams to measure adoption, identify friction points, and improve developer onboarding experiences.
