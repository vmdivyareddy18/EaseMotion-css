# Ease Voltage Flow

## Overview

Ease Voltage Flow creates an animated electric border effect using CSS conic gradients and rotation. The animation gives the appearance of energy continuously flowing around an element.

---

## Features

- ⚡ Animated electric border
- 🎨 Three color variants
- 🚀 Hover elevation
- ✨ Smooth infinite animation
- 💯 Pure HTML & CSS

---

## Included Variants

- Default Cyan
- Blue
- Purple

---

## Usage

```html
<div class="voltage-card">
    Content
</div>
```

---

## Customization

You can modify:

- Border colors
- Rotation speed
- Card size
- Border radius
- Glow intensity

---

## Browser Support

Works in all modern browsers supporting CSS conic-gradient and transforms.
