# RTL-Friendly Animation Demo

## Description

This example demonstrates how directional animations can be adapted for Right-to-Left (RTL) layouts using CSS.

## Files

- demo.html
- style.css

## Features

- Left-to-Right animation example.
- Right-to-Left animation example.
- Uses the HTML `dir` attribute.
- No JavaScript required.

## Usage

Open `demo.html` in a browser.

Observe:

- The LTR example moves from left to right.
- The RTL example moves from right to left.

## Benefits

- Better multilingual support.
- Improved internationalization.
- Easy to understand and reuse.
