# Animated Pagination Component

## Overview

A modern pagination component with smooth hover animations, active page indicators, and stylish navigation controls.

## Features

- Smooth hover lift animation
- Active page highlighting
- Circular modern button design
- Lightweight and reusable component
- Pure HTML and CSS implementation

## Files

- demo.html – Pagination component preview
- style.css – Styling and animation effects
- README.md – Documentation and usage guide

## Usage

Open demo.html in any modern browser to see the animated pagination component.

Hover over the page buttons to experience the interactive animations.
