# Empty Cells Show

empty-cells — Shows or hides borders on empty table cells

## Features
- Uses `--ease-color-surface`
- Uses `--ease-color-neutral-200`
- Uses `--ease-radius-sm`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Empty Cells Show</span>
</div>
```

## Why is it useful?

This submission demonstrates the empty cells show CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
