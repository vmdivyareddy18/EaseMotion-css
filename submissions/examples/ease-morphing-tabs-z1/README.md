# Ease Morphing Tabs

## Overview

A lightweight tab navigation with a smooth morphing active state built using HTML and CSS.

## Features

- Pure HTML & CSS
- Morphing active tab
- Responsive
- Easy to customize
- Zero dependencies

## Folder Structure

```
ease-morphing-tabs-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why This Example

This example demonstrates a reusable tab navigation component with clean transitions suitable for dashboards, admin panels, and documentation websites.
