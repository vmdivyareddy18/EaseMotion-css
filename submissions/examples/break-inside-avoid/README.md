## break-inside-avoid

Explores CSS break properties — `break-inside`, `break-before`, and `break-after` — for controlling column and page fragmentation.

### Key Features

- `break-inside: avoid` prevents cards from splitting across columns
- `break-before: page` for print pagination and chapter starts
- Column break management with `break-before: column`

### Break Properties

The modern `break-*` properties unify `page-break-*`, `column-break-*`, and `region-break-*` under a single syntax. Use `break-inside: avoid` to keep elements intact, `break-before` / `break-after` to force or avoid breaks at specific points.
