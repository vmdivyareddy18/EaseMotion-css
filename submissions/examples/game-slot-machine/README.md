# 🎰 Slot Machine

A classic Slot Machine game built with HTML, CSS, and JavaScript. Spin the reels, match symbols, and win big with multiple paylines and bonus features!

## 🎮 How to Play

1. **Set your bet** using the +/- buttons or arrow keys
2. Click **"Spin"** or press **Space** to spin the reels
3. **Match symbols** across active paylines to win
4. **Watch your balance** grow with each win!
5. **Reset** to start over

## 🖱️ Controls

| Action | Input |
|--------|-------|
| Spin | Click Spin or press `Space` |
| Increase Bet | Click + or press `↑` |
| Decrease Bet | Click - or press `↓` |
| Reset Game | Click Reset or press `R` |

## ✨ Features

- 🎰 **8 Symbols** – Cherry, Lemon, Orange, Watermelon, Star, Bell, Diamond, Jackpot
- 🎯 **3 Paylines** – Top, Middle, Bottom (with optional V/Λ paylines)
- 💰 **Betting System** – Adjustable bet from 1-50
- 🔊 **Sound Effects** – Spin, win, and lose sounds via Web Audio
- 🎨 **Visual Feedback** – Win animations and particle effects
- 📊 **Statistics** – Balance, bet, win, and spin counter
- 📱 **Responsive Design** – Works on desktop and mobile
