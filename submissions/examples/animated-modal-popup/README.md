# Animated Modal Popup

A responsive animated modal popup component built using pure HTML and CSS.

## Features

- Fade and scale animations
- Overlay blur effects
- Animated close button interactions
- Responsive modal layout
- Smooth transitions and motion effects
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated modal popup component.

## Files

- `demo.html` → Modal popup structure
- `style.css` → Modal styles and animations
- `README.md` → Documentation
