Scroll Progress Ring

Scroll down the page — the ring in the corner fills as you go, and the percentage label updates with it.

This is the circular counterpart to a linear top-of-page loading/progress bar: a common reading-progress indicator for long articles, docs pages, or blog posts.

Keep scrolling to watch the ring fill all the way around.

The ring only needs the scroll percentage — everything else (the circle math, the stroke animation) is handled in CSS via a single custom property.
