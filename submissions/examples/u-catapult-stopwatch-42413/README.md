# Catapult Stopwatch

A Space-inspired Catapult Stopwatch built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Catapult entrance animation
- Stopwatch interface
- Responsive layout
- Accessible HTML
- No JavaScript required
