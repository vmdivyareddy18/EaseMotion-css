# University Student Portal Dashboard

A responsive **University Student Portal Dashboard** built with **HTML5** and **CSS3** that demonstrates academic progress tracking, attendance monitoring, assignment management, and campus workflow animations using EaseMotion CSS.

---

## Features

- Responsive dashboard layout
- Student overview cards
- Attendance progress tracker
- Upcoming classes panel
- Assignment timeline
- Recent academic activity
- Notifications panel
- Quick action buttons
- Smooth hover effects
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Enrolled Courses
- Attendance
- Assignments Due
- Current GPA

### Student Widgets

- Attendance Progress
- Upcoming Classes
- Assignment Timeline

### Activity

- Recent Academic Activity
- Notifications Panel

### Quick Actions

- View Results
- Submit Assignment
- Download Transcript

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
