# ease-static-noise

A TV static noise overlay animation inspired by vintage monitors and glitch effects.

## Features

- Rapid flickering opacity changes
- Static noise overlay effect
- Small scale and translate jitter
- Screen-like appearance
- Pure HTML and CSS
- Responsive layout

## Files

- demo.html
- style.css
- README.md

## Animation Details

The effect combines:

- Irregular opacity flickering
- Subtle position jitter
- Scale distortion
- Noise texture overlay

Suitable for monitor, terminal, cyberpunk, and glitch-themed interfaces.
