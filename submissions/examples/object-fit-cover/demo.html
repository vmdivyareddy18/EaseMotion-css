Object-Fit & Object-Position

The same image, five different object-fit values

cover

Fills the box while preserving aspect ratio. Content is cropped to fit — no distortion.

Best for hero banners and thumbnails where full coverage is desired.

contain

Fits entirely within the box. Letterboxed if aspect ratios differ.

Ideal when the full image must be visible (product shots, logos).

fill

Stretches to fill the box exactly, ignoring aspect ratio. May distort.

Use sparingly — only when exact box coverage is required.

scale-down

Acts like contain or none, whichever results in a smaller image.

Prevents upscaling — works well for responsive icons.

none

Ignores the box size — uses the image's intrinsic dimensions. May overflow.

Only viable when source and container have matching sizes.
