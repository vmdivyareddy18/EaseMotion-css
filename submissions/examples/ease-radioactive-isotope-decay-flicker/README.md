# ease-radioactive-isotope-decay-flicker

**What does this do?**
Adds an ultra-unique ease-radioactive-isotope-decay-flicker CSS animation.

**How is it used?**
Include style.css and apply the .ease-radioactive-isotope-decay-flicker class to an element.

**Why is it useful?**
Provides a highly specific, visually distinct effect that cannot be mistaken for generic templates.
