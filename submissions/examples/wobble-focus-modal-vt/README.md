# CSS Wobble-Focus Modal for Minimalist Tech Layouts

1. **What does this do?**  
   It renders a pure CSS animated modal component utilizing a smooth Wobble-Focus interaction transition, styled for modern minimalist tech interfaces.

2. **How is it used?**  
   Open `demo.html` in any modern web browser. Click the **Launch Modal** button to display the modal with a Wobble-Focus entrance animation. Close it using the **Close** button or the **×** icon. No JavaScript is required.

3. **Why is it useful?**  
   It provides a lightweight, zero-JavaScript modal solution featuring responsive design, keyboard accessibility, customizable CSS variables for animation timing, easing, and scale, making it easy to integrate into modern web applications.
