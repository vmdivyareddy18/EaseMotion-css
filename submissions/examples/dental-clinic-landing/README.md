# Premium Dental Clinic Landing Page

A complete, high-fidelity, and fully responsive landing page for a friendly dental clinic ("BrightSmile Dental") built under issue #14813 using the **EaseMotion CSS** design system.

## EaseMotion Features Showcased

- **Entrance Animations:** Leverages `.ease-reveal` and direction helpers (`.ease-reveal-up`, `.ease-reveal-right`) with scroll-reveal triggers.
- **Hover Micro-interactions:** Showcases hover scale card effects, avatar transformations, and interactive form feedback.
- **Component Systems:** Utilizes standardized `.ease-btn`, `.ease-btn-primary`, `.ease-btn-outline`, and layout classes (`.ease-flex`, `.ease-grid`, `.ease-card`).

## Required Sections Included

1. **Friendly Hero:** Warm welcome theme featuring trust badges and a direct appointment booking CTA.
2. **Services Grid:** List of premium treatments (e.g. routine cleaning, professional whitening, precision implants) with hover highlights.
3. **Before/After Smile Gallery:** An interactive comparison slider demonstrating smile restorations.
4. **Doctor Profiles:** Professional cards introducing certified specialist dentists and orthodontic surgeons.
5. **Insurance/Pricing Section:** A clean table and checklist detailing accepted insurance coverages and transparent costs.
6. **Patient Testimonials:** Authentic feedback and success stories from real patients.
7. **Appointment Request Form:** Form container built with active input states and request status alerts.

Open `demo.html` in any browser to explore the design.
