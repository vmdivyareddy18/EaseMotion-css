# Deployment Readiness Ladder

An intermediate EaseMotion CSS example for release dashboards. The demo shows four deployment stages climbing into place while a gradient rail draws between them.

## Files

- `demo.html` - Semantic release readiness ladder markup.
- `style.css` - Responsive layout, staged entrance animations, and status color tokens.

## Usage

Open `demo.html` in a browser to preview the animated ladder component.
