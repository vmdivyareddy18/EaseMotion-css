# CSS Rating System

## What does this do?
Pure CSS rating components with star ratings (static and interactive via checkbox hack), emoji reactions, numeric scores, review cards, and distribution bars.

## How is it used?
```html
<div class="rt-stars"><span class="rt-star rt-filled">★</span><span class="rt-star">★</span></div>
<div class="rt-interactive"><input type="checkbox" class="rt-check" id="s1"><label for="s1" class="rt-check-label">★</label></div>
```

## Why is it useful?
Provides comprehensive rating UI patterns without JavaScript. Interactive star rating via checkbox hack. Emoji reaction selector. Review cards with avatar. Distribution bars for aggregate ratings. Customizable colors and sizes.
