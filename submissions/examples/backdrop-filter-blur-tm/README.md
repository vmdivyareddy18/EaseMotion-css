# Backdrop Filter: Blur And Blur Variants

Demonstrates backdrop-filter blur variants using CSS backdrop-filter property with blur, saturate, brightness, and contrast effects, all powered by EaseMotion design tokens.

## Features

- Demonstrates **backdrop-filter** CSS property with multiple practical variants
- Uses EaseMotion design tokens (`--ease-color-*`, `--ease-space-*`, `--ease-radius-*`)
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`
- Multiple size/style variants for real-world usage

## Usage

Apply the utility class to any element:

```html
<div class="blur-sm">Content</div>
```

## Why is it useful?

The **backdrop-filter** CSS property is part of the modern CSS specification and enables
fine-grained control over backdrop-filter. It integrates seamlessly with EaseMotion's design token
system, making it easy to maintain consistent spacing, colors, and typography across your project.
