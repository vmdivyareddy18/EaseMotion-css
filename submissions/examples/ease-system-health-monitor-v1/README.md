# Ease System Health Monitor

## Overview

Ease System Health Monitor is a reusable dashboard component that displays animated system metrics such as CPU, Memory, Network, and Disk usage using pure HTML and CSS.

---

## Features

- 💻 System health dashboard
- 🟢 Animated status indicators
- 📊 Smooth progress bars
- ✨ Hover interactions
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- CPU monitor
- Memory monitor
- Network monitor
- Disk monitor
- Animated progress bars
- Live status indicators

---

## Usage

```html
<div class="dashboard">
...
</div>
```

## Customization

- Card colors
- Progress values

Compatible with modern browsers supporting CSS animations, transitions, and Grid.
