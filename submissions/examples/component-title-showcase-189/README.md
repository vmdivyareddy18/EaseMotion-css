# Component Title Showcase 189

A clean, premium, CSS-only title showcase component created for EaseMotion issue `#25723`.

## Files

- `demo.html` - component markup and preview page
- `style.css` - responsive styling, glass effects, gradients, and CSS animation

## Features

- Pure HTML and CSS implementation
- Responsive hero-style title layout
- Glassmorphism card with animated decorative orbs
- No JavaScript or external dependencies

## Usage

Open `demo.html` in a browser to preview the component.
