# Smart Dock Menu

A floating dock navigation inspired by desktop operating systems using pure HTML and CSS.

---

## Features

- Floating dock layout
- Magnify hover effect
- Active icon animation
- Glassmorphism style
- Responsive design
- Pure HTML & CSS

---
