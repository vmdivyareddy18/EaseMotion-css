# Deployment Stage Checklist

## What does this do?

Shows deployment stages with passed, running, warning, and blocked states.

## How is it used?

```html
<ol class="stage-list">
  <li class="running"><span>Preview deploy</span><strong>Running</strong></li>
</ol>
```

## Why is it useful?

It gives EaseMotion CSS a readable release workflow pattern for product and developer dashboards.
