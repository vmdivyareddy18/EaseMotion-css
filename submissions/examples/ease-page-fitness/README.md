# Full Fitness Gym Landing Page (`ease-page-fitness`)

A high-energy, aggressive, and highly contrasting fitness landing page template built using core EaseMotion CSS classes to deliver impact.

## 🚀 Features & EaseMotion Showcase

This landing page integrates multiple core `.ease-*` classes alongside an `IntersectionObserver` script to create an energetic, bold entrance:

- **Aggressive Typography & Contrast**: Pairs a heavily styled UI with rapid `.ease-slide-up` and `.ease-fade-in` entrance animations. The timing functions are deliberately quick to match the high-energy aesthetic of a fitness brand.
- **Directional Entrances**: Section headers utilize opposing `.ease-slide-right` and `.ease-slide-left` classes to create dynamic tension as the user scrolls down the page.
- **Punchy Interactions**: Interactive elements utilize a snappy `.ease-hover-scale` with a custom bouncy `cubic-bezier` timing curve to make buttons and cards feel responsive and active.

## 🛠️ Usage

This demo is entirely self-contained. You can open `demo.html` directly in your browser. All required EaseMotion CSS classes have been defined at the top of the `style.css` file for immediate testing. 

If using this as a template for a real project, simply migrate the HTML structure into your application and ensure your global CSS bundle includes the `ease-*` utility classes.
