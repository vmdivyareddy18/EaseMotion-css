# CSS Stat Counters

## What does this do?
Animated number counting cards with scroll-triggered CSS animations for stats, metrics, and KPIs.

## How is it used?
```html
<div class="sc-crd"><span class="sc-num" data-target="1250">0</span><span class="sc-lbl">Label</span></div>
```

## Why is it useful?
Animated metrics display. Tabular numbers, fade-up animation, grid/row/inline layouts. Pure CSS counters.
