# Product Cards Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be applied to responsive product cards.

---

## Included Components

- Product Image
- Product Name
- Product Price
- Product Rating
- Buy Now Button

---

## Folder Structure

```
submissions/examples/product-cards-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to verify responsiveness.
3. Observe the animation effects applied to the product cards.
