# Photography Presets Shop

A premium, copy-paste ready dark-mode landing page for a Lightroom photography presets e-commerce store built with EaseMotion CSS.

## Description

This project showcases a complete e-commerce landing page tailored for photography presets. It features a moody, premium dark aesthetic highlighted by golden accents. It leverages the EaseMotion library for smooth entrance animations and hover states, and includes a pure CSS interactive before/after image slider using the `resize` property.

## Features Included

- **Hero Section**: Impactful typography, clear CTA, and a static visual representation of before/after edits.
- **Style Categories**: Animated pills representing different preset moods (Moody, Bright, Film, VSCO).
- **Preset Collections**: Grid of products with hover lifts and detailed pricing.
- **Interactive Before/After Slider**: A pure CSS implementation of an image comparison slider demonstrating raw vs edited.
- **Reviews & Testimonials**: Trust-building social proof cards.
- **Bundle Deals**: A prominent, highlighted bundle box with massive value proposition.
- **Responsive Design**: Adapts beautifully across mobile, tablet, and desktop views.
