# CSS Fade-In Modal for Product Catalog Layouts

A responsive product catalog layout with a modern CSS fade-in modal for displaying product details. This component is ideal for eCommerce stores, online marketplaces, product landing pages, and digital catalogs.

---

## Features

- ✨ Smooth fade-in and slide-up modal animation
- 🛍️ Responsive product grid
- 🔍 Quick View modal for product details
- 🎨 Clean and modern card design
- 📱 Mobile-friendly layout
- ❌ Close button support
- 🖱️ Click outside to dismiss
- ⌨️ ESC key closes the modal
- 🚀 Lightweight HTML, CSS, and Vanilla JavaScript
- 🌐 No external dependencies

## Getting Started

1. Download the project files.
2. Place `demo.html` and `style.css` in the same directory.
3. Open `demo.html` in your browser.
4. Click any **Quick View** button to display the animated product modal.

---

## Animation

### Overlay

- Uses a semi-transparent dark background.
- Smoothly fades in with an opacity transition.

### Modal

Hidden State:

- `opacity: 0`
- `transform: translateY(30px) scale(0.95)`

Visible State:

- `opacity: 1`
- `transform: translateY(0) scale(1)`

The combination creates a polished fade-and-slide entrance suitable for modern eCommerce interfaces.
