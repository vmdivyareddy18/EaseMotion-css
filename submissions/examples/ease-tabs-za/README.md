# CSS Tab Panel

## What does this do?
Pure CSS tabbed interface using radio button hack with horizontal, pill, and vertical layout variants.

## How is it used?
```html
<div class="tb-tabs"><input type="radio" name="tabs" id="tab1" class="tb-inp" checked><div class="tb-labels"><label for="tab1" class="tb-label">Tab</label></div><div class="tb-panels"><div class="tb-panel">Content</div></div></div>
```

## Why is it useful?
No JavaScript tab switching. Three layout variants: underlined horizontal, pill-style, and sidebar vertical. Accessible via label elements. Clean CSS-only panel transitions.
