HTML5

HTML provides the semantic structure for web documents. Elements like <header>, <nav>, <main>, and <footer> describe content meaning. Modern HTML supports multimedia, forms, canvas, and accessibility features.

CSS3

CSS controls the visual presentation of HTML elements. It handles layout (flexbox, grid), colors, typography, animations, and responsive design. Modern CSS includes custom properties, container queries, and cascade layers.

JavaScript

JavaScript adds interactivity and dynamic behavior to web pages. It runs in the browser and can manipulate the DOM, handle events, fetch data from APIs, and manage application state. ES6+ brings classes, modules, and async/await.

REST API

APIs enable communication between frontend and backend systems. RESTful endpoints use HTTP methods (GET, POST, PUT, DELETE) to perform CRUD operations. JSON is the standard data interchange format.
