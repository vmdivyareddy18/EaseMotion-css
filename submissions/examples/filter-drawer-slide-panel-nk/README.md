# Filter Drawer Slide Panel

A responsive filter drawer example built with pure HTML and CSS.

## Features

- Slide-in filter drawer
- Desktop side panel layout
- Mobile bottom-sheet behavior
- Product grid showcase
- Filter chips
- Price range indicator
- Hover animations
- Responsive design
- prefers-reduced-motion support

## Usage

```html
<aside class="filter-drawer is-open">
    ...
</aside>
```

## Why it fits EaseMotion CSS

This example demonstrates practical UI motion patterns commonly used in dashboards, e-commerce applications, analytics platforms, and job boards. It uses human-readable class names, smooth transitions, responsive layouts, and accessibility-friendly reduced-motion handling while remaining completely framework-free.
