# Glassmorphic Statistics Card

A modern, highly visual dashboard statistics card engineered using CSS backdrop-filters to achieve a premium frosted glass design.

## Features
- **True Glassmorphism**: Utilizes combined translucent layers, `backdrop-filter`, and glowing borders.
- **Micro-interactions**: Clean and snappy 3D lift hover effects.
- **Fully Responsive**: Built with CSS Grid to adapt flawlessly across mobile, tablet, and desktop viewports.
- **Semantic Structure**: Easily customizable for different types of dashboard metric layouts.

## Structure
- `demo.html` - Interactive layout displaying various trend conditions (Positive, Negative, Neutral).
- `style.css` - Component structural and aesthetic styling variables.
