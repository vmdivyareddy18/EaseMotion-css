# CSS Hero Banners

## What does this do?
Hero sections with gradient backgrounds, overlays, CTAs, split layout, and minimal variants.

## How is it used?
```html
<div class="hb-hero hb-grad"><div class="hb-content"><h2>Title</h2><p>Text</p><button class="hb-btn">CTA</button></div></div>
```

## Why is it useful?
Hero section patterns without JS. Gradient/pattern backgrounds. Split and minimal variants. Gradient text.
