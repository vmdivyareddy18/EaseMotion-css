# Glassmorphic Music Player Layout

## Description
A visually striking glassmorphic lofi music player layout utilizing backdrop blurs, active keyframe rotating vinyl record styling, animated sound waveform bars, and range sliders.

## Installation
Save the folder files and open `demo.html` locally.

## Usage
Include files normally in HTML wrapper layouts.

## Accessibility
- Audio controls labeled clearly via `aria-label`.
- Fully interactive range progress inputs.
