# Digital Clock

## What does this do?
Digital Clock component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"dc-primary\">Digital Clock Primary</div>
```

## Why is it useful?
Provides a reusable digital clock UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
