# Weather Cloud Drift

1. What does this do? A cloud-shaped element drifts slowly from left to right in a seamless looping translateX animation, at a slow dreamy speed.

2. How is it used? Apply `.cloud-drift` to the cloud element inside a `.drift-container` with overflow hidden.

3. Why is it useful? It provides a calm, atmospheric background motion for weather widgets, dashboards, or landing pages using pure CSS with no JavaScript.
