# CSS Positioning Visualizer

A lightweight CSS-only utility that visually demonstrates how different CSS `position` values affect element placement.

## Features

- Static Position
- Relative Position
- Absolute Position
- Fixed Position
- Sticky Position
- Responsive Grid Layout
- Pure HTML & CSS
- No JavaScript Required

## Folder Structure

```
css-positioning-visualizer-ajit/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in your browser.
2. Compare the behavior of all five CSS positioning modes.
3. Resize the browser to test responsiveness.

## Technologies Used

- HTML5
- CSS3
