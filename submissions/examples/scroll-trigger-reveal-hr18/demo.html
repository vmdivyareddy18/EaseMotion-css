Scroll-Trigger Reveal

A standalone, dependency-free utility that gates any entrance animation behind an IntersectionObserver instead of firing it on mount. Scroll down — everything below fires only once it enters the viewport.

Each section below uses a data-ease-scroll attribute like "ease-slide-up-hr18 threshold-0.3 once delay-100" — no per-section JavaScript, just the attribute.

Feature grid

Each card slides up with a small staggered delay.

Composable

One attribute, any animation class — no bespoke JS per section.

Configurable

Threshold, once/repeat, and delay are all readable modifiers.

Dependency-free

Just IntersectionObserver — no scroll-event polling, no library.
