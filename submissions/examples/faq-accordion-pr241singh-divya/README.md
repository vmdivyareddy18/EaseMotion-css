# CSS FAQ Accordion

A responsive FAQ Accordion component built using only HTML and CSS for the EaseMotion CSS project.

## Features

- Pure HTML & CSS
- Semantic `<details>` and `<summary>` elements
- Smooth expand/collapse animation
- Arrow indicator
- Hover effects
- Responsive design
- Keyboard accessible
- Dark mode support
- Reduced motion support
- Multiple visual variants

## Variants

- Basic
- Bordered
- Glass
- Gradient
- Filled
- Minimal

## Browser Support

Works in all modern browsers that support the HTML5 `<details>` element.
