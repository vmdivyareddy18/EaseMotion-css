# Avatar Status Swing

A profile avatar card where the online status badge swings gently on hover or keyboard focus.

## Files

- `demo.html` - focusable avatar card markup.
- `style.css` - card lift, status badge swing, focus-visible state, and reduced-motion support.

## Highlights

- Keeps the status badge anchored to the avatar.
- Mirrors hover behavior on keyboard focus.
- Disables badge motion for reduced-motion users.
