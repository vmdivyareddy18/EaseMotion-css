# Interactive Pricing Cards

## Description

A modern pricing card component with smooth hover animations, featured plan badges, responsive layouts, and attractive UI effects.

---

## Features

- Responsive pricing card layout
- Hover lift and scale animation
- Featured "Most Popular" badge
- Smooth transitions
- Modern shadows and rounded corners
- Mobile friendly design

---

## Files

- `demo.html` – Pricing card structure
- `style.css` – Styling and animations
- `README.md` – Documentation

---

## Usage

Open `demo.html` in your browser to preview the pricing cards.
