# Floating Orbit Modal – Cyberpunk Neon

A futuristic **Floating Orbit Modal** created entirely with HTML and CSS for the EaseMotion CSS collection.

## Features

- Pure HTML & CSS
- Floating Orbit animation
- Cyberpunk Neon UI
- Responsive design
- Keyboard accessible
- Zero JavaScript
- CSS Custom Properties
- Smooth scale entrance animation
- Neon glow effects
- Lightweight and reusable
