# Clear Both

clear — Stops floating elements from flowing around an element

## Features
- Uses `--ease-color-surface`
- Uses `--ease-radius-md`
- Uses `--ease-shadow-sm`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Clear Both</span>
</div>
```

## Why is it useful?

This submission demonstrates the clear both CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
