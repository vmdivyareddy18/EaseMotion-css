EaseMotion CSS — Card Enhancement

Missing Card Accent Variants

EaseMotion CSS cards feature primary, success, danger, and warning accents. This demo adds the missing info (cyan/blue) and secondary (slate/gray) accent classes.

Primary

Standard System

This card utilizes the default primary accent line on the left side of the border.

Info

System Analytics

Utilizes the newly introduced ease-card-accent-info class with custom cyan/blue indicators.

Secondary

Archive Storage

Utilizes the newly introduced ease-card-accent-secondary class with custom slate/gray indicators.
