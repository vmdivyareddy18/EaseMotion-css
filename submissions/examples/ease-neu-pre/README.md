## ease-neu

**What does this do?**
Neumorphism (soft UI) card and button styles using dual inner/outer
shadows to simulate raised and pressed elements on a flat background.

**How is it used?**
<div class="ease-neu-card">Card content</div>
<div class="ease-neu-card pressed">Pressed card</div>

<button class="ease-neu-btn">Default</button>
<button class="ease-neu-btn green">Confirm</button>

<input class="ease-neu-input" placeholder="Input..." />

**Why is it useful?**
Provides a complete neumorphic UI system with high-contrast mode
override for accessibility — works best on light gray backgrounds,
consistent with EaseMotion CSS's component philosophy.
