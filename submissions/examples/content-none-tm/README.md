# Content None

content — Sets generated content in ::before and ::after pseudo-elements

## Features
- Uses `--ease-color-surface`
- Uses `--ease-color-primary`
- Uses `--ease-color-muted`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Content None</span>
</div>
```

## Why is it useful?

This submission demonstrates the content none CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
