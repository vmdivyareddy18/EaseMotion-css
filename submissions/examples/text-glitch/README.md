# Text Glitch Effect

## Description
A pure CSS glitch text animation effect. It uses pseudo-elements and CSS clip-paths to create a cyberpunk-style RGB split and skew glitch effect on text. This requires zero JavaScript.

## Files
- `demo.html`: Contains the HTML structure for the glitch text.
- `style.css`: Contains the CSS animations and styling for the glitch effect.

## How to use
1. Open `demo.html` in your browser to see the effect.
2. Copy the HTML and CSS into your project.
3. **Important:** Ensure the `data-text` attribute in your HTML matches the inner text of the element exactly (e.g., `<h1 class="glitch-text" data-text="YOUR TEXT">YOUR TEXT</h1>`).
4. Adjust the colors in `style.css` (`#00ffff` and `#ff00ff`) to fit your design theme.
