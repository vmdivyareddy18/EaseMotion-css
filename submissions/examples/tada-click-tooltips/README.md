# Pure CSS Tada-Click Tooltip

A fast, semantic, high-contrast tooltip module tailored for minimalist technical application layouts. It applies standard interactive states alongside a dramatic CSS-driven celebratory "Tada" layout burst when actively pressed.

## Features
- **Zero JavaScript Runtime:** Relies natively on matching `:active` pseudo-states synchronized with CSS `@keyframes`.
- **Minimal Tech Theme:** Monospaced structural parameters styled with high-contrast surfaces cleanly aligned with terminal systems.
- **Micro-Interaction Burst:** Vibrates and bounces dynamically on user click inputs to create satisfying action confirmation.
- **Exposed Configuration Variables:** Tailor scales and visual timing metrics using CSS variables.

## Configurable Properties

| Property | Purpose | Default Vector |
| :--- | :--- | :--- |
| `--tech-accent` | Core focal point accent outline indicator | `#00ffa3` |
| `--tooltip-bg` | Core surface container fill parameter | `#f1f5f9` |
| `--tooltip-fade-duration` | Opacity processing transition metric | `0.2s` |
| `--tooltip-tada-duration` | Execution period of the click-shake sequence | `0.5s` |
