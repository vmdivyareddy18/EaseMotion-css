# Glitch Text

1. **What does this do?**  
   It creates a chaotic RGB-split glitch animation on text, simulating digital distortion.

2. **How is it used?**  
   ```html
   <div class="glitch-text" data-text="Your Text">Your Text</div>
   ```

3. **Why is it useful?**  
   Perfect for 404 pages, cyberpunk themes, or emphasizing errors in a visually striking way.
