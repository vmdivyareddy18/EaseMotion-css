# Animated Recruitment Hiring Dashboard

A modern **Recruitment Hiring Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example showcases recruitment metrics, hiring pipeline, recruiter management, interview tracking, and hiring analytics with smooth CSS-only animations.

## Features

* 👥 Total Applicants overview
* 📅 Interview schedule tracking
* 📨 Offers Sent statistics
* 🎉 Hires Completed overview
* 📋 Hiring pipeline progress
* 👤 Recruiter profile cards
* 🟢 Online/Offline recruiter status
* 📊 CSS-only weekly hiring chart
* 📈 Recruitment progress indicators
* 🔔 Recruitment activity feed
* ✨ Smooth hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries
