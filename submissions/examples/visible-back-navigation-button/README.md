# Visible Back Navigation Button

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A reusable back navigation component featuring prominent visibility, hover interactions, glassmorphism styling, sticky positioning, and a floating action variant.

## Features

- Visible back button
- Animated arrow movement
- Hover slide interaction
- Glassmorphism styling
- Sticky header variant
- Floating back button variant
- Responsive design
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<a href="#" class="back-button">
  <span class="arrow">←</span>
  <span>Back</span>
</a>
```

## Main Classes

```css
.back-button
.arrow
.floating-back-btn
.page-header
```

## Why It Fits EaseMotion CSS

- Reusable navigation utility
- Human-readable class names
- Animation-first interactions
- Lightweight implementation
- Modern UI design pattern
