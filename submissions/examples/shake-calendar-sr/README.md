# Shake Calendar

## Overview

Shake Calendar is a modern animated calendar component built using only HTML and CSS. It features a clean glassmorphism design, hover shake animation, floating effects, and a responsive layout. No JavaScript is required.

## Features

- Pure HTML and CSS
- Responsive design
- Shake hover animation
- Floating card animation
- Animated highlight for the current day
- Accessible keyboard focus styles
- Easy to customize using CSS variables

## Folder Structure

```
shake-calendar-sr/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern web browser.

## Customization

You can change colors, spacing, animations, and fonts by editing the CSS variables defined at the top of `style.css`.

## Browser Support

Works in all modern browsers that support CSS Grid, Flexbox, and CSS animations.
