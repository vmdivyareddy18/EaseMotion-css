# Carousel Go-Round — Merry-Go-Round

CSS merry-go-round with rotating arms and counter-rotating ride pods. Five colored rides orbit the center axle with an upright counter-spin.

## Files

- `demo.html` — axle, arms, ride pods
- `style.css` — carousel spin, counter-spin keyframes

## Usage

Open `demo.html` to see the carousel rotate.
