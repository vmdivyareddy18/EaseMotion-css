# Ease Question Answer Card

A modern **Question & Answer Card** component built using **pure HTML and CSS**. It uses semantic HTML elements to create an expandable Q&A interface with smooth animations and responsive design.

## Features

- Pure HTML & CSS
- Semantic `<details>` and `<summary>` elements
- CSS-only expand animation using `@keyframes`
- Responsive layout
- CSS Custom Properties for easy customization
- Hover effects
- Keyboard accessible
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
ease-question-answer-card/
├── demo.html
├── style.css
└── README.md
```

## Customization

Modify the CSS variables in the `:root` section to customize:

- Primary color
- Background color
- Border radius
- Shadow
- Transition speed

## Browser Support

Compatible with all modern browsers that support the HTML5 `<details>` element.
