# CSS Sticky Notes Component

## What does this do?
Visual corkboard-style sticky note cards with pin effect, color variants, and tilt rotations for playful note-taking UIs.

## How is it used?
```html
<div class="stn-note stn-yellow stn-tilt-left">
  <div class="stn-pin"></div>
  <h3>Title</h3>
  <p>Content here</p>
</div>
```

## Why is it useful?
Creates playful note-taking interfaces for dashboards, kanban boards, and brainstorming tools. Multiple colors, tilt options, and hover effects. Pure CSS, no JavaScript.
