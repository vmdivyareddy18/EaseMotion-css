# Deep Space Operations Command Center

A deep space operations command center dashboard built with EaseMotion CSS tokens. Tracks active missions, crew status, ship systems, communications, navigation data, resource levels, alerts, and fleet overview in a deep-space dark theme.

## Features

- **Stat cards** — active missions, crew onboard, fleet range, system integrity
- **Ship systems** — 7 systems with nominal/caution/critical/standby status
- **Command crew** — 6 officers with rank, name, and role
- **Communications log** — timestamped messages from base, scout, med, Mars relay
- **Navigation** — heading, velocity, distance, ETA, nearest celestial, gravity well
- **Resource levels** — fuel, oxygen, water, food, power with color-coded bars
- **Alert log** — alpha/beta/gamma severity events with timestamps
- **Fleet overview** — active ships, in dock, on EVA, total crew, planets surveyed, days since incident
- Responsive 12-column grid with deep-space dark (default) and light mode support
- Reduced-motion compliant
