# Feedback Sentiment Breakdown Bars

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive sentiment analysis component displaying customer feedback distribution using animated progress bars.

## Features

- Positive, neutral and negative sentiment bars
- Animated progress indicators
- Responsive layout
- Modern glassmorphism design
- Clean dashboard UI
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="sentiment">

    <div class="label">
        <span>😊 Positive</span>
    </div>

    <div class="bar">

        <div class="fill positive"></div>

    </div>

</div>
```

## Main Classes

```css
.card
.sentiment
.label
.bar
.fill
.positive
.neutral
.negative
```
