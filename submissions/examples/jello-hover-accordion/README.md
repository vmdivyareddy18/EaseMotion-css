# Jello Hover Accordion

A modern SaaS-style accordion component built using **pure HTML and CSS**. This component features a smooth jello hover animation, responsive design, and semantic HTML for improved accessibility.

## Features

- Pure HTML & CSS
- No JavaScript required
- Jello hover animation
- Smooth expand/collapse interaction
- Semantic `<details>` and `<summary>` elements
- CSS Custom Properties (variables)
- Responsive design
- Keyboard accessible
- Focus-visible styles
- Dark mode support
- Reduced motion support

## Folder Structure

```
jello-hover-accordion/
├── demo.html
├── style.css
└── README.md
```

## Browser Support

Compatible with all modern browsers that support the HTML5 `<details>` element.
