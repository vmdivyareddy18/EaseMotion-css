Jello Hover Accordion

Modern SaaS UI built with Pure HTML & CSS

What is EaseMotion CSS?
EaseMotion CSS is a modern collection of reusable UI components, animations, and utilities created entirely with HTML and CSS.
Does this require JavaScript?
No. This accordion is powered completely by semantic HTML and CSS.
Is it responsive?
Yes. It automatically adapts to desktop, tablet and mobile devices.
Can I customize the colors?
Yes. Simply edit the CSS variables defined at the top of the stylesheet.
Does it support dark mode?
Yes. Dark mode is supported using the prefers-color-scheme media query.
