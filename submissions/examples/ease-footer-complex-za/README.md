# CSS Complex Footer Component

## What does this do?
A comprehensive footer section with five-column navigation grid, brand description with social media links, newsletter subscription form, contact information bar, legal links, copyright notice, SVG wave divider, and a floating back-to-top button.

## How is it used?
```html
<footer class="ft-footer">
  <div class="ft-inner">
    <div class="ft-grid">
      <div class="ft-col">
        <div class="ft-logo">Brand</div>
        <ul class="ft-links">
          <li><a href="#">Link</a></li>
        </ul>
      </div>
    </div>
    <div class="ft-bottom">
      <p class="ft-copy">&copy; 2026</p>
    </div>
  </div>
</footer>
```

## Why is it useful?
Creates a professional website footer for modern web applications. Features a five-column responsive grid with brand column including logo description and social media links plus four link column groups for Product Resources Company and Support. Includes a newsletter subscription section with email input and CTA button contact information row with icons and legal links. SVG wave divider at top provides visual separation. A fixed back-to-top button appears for easy navigation. Fully responsive with smooth hover effects and CSS custom property theming.
