# Album Art Spin

1. What does this do? A vinyl record-style album art that rotates continuously (rotate 360deg) during playback and pauses smoothly when stopped, with a tonearm that moves into position.

2. How is it used? Add a `.vinyl-record` element and toggle the `.playing` class to start/stop the rotation animation. The CSS transition on the element ensures a smooth stop when paused.

3. Why is it useful? It creates a nostalgic vinyl record visual metaphor for music playback, giving users a clear and delightful audio-visual feedback for the now-playing state.
