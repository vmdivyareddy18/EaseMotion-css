# Ease Floating Action Menu

A lightweight, pure CSS Floating Action Button (FAB) that expands upward to reveal a menu of secondary actions on hover. 

## Features
- **Pure CSS transitions**: Leverages `transform`, `opacity`, and `visibility` for performance-friendly rendering.
- **Rotation effect**: Rotates the primary `+` icon on hover to turn it into a close `x` icon.
- **Responsive & accessible**: Adapts to any viewport size, utilizing basic semantic elements and ARIA labels.

## File Structure
- `demo.html` - Minimal demonstration of the component's interactive hover state.
- `style.css` - Component-specific layout, absolute positioning, hover effects, and base demo styles.

## How to Use
Copy the markup from `demo.html` and the CSS under the component section of `style.css` into your project.
