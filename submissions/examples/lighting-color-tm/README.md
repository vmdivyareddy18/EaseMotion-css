# Lighting Color

lighting-color — Sets the color of the light source in SVG lighting filters

## Features
- Uses `--ease-color-primary`
- Uses `--ease-shadow-lg`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Lighting Color</span>
</div>
```

## Why is it useful?

This submission demonstrates the lighting color CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
