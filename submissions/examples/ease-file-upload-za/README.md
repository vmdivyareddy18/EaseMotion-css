# CSS File Upload

## What does this do?
File upload components with drag-drop zone, progress bars, file list, format badges, and multi-upload.

## How is it used?
```html
<div class="fu-drop"><div class="fu-drop-inner"><button class="fu-btn">Browse</button></div></div>
```

## Why is it useful?
Upload UI without JS. Dashed drop zone, animated progress bars, success/error states, format badges.
