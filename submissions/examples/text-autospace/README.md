# CSS text-autospace

## Overview
This example demonstrates the CSS 	ext-autospace property for controlling automatic spacing between different writing systems (e.g., CJK and Latin text). It improves readability in multilingual typography.

## Features
- **	ext-autospace: normal** — inserts spacing between mixed script boundaries
- **	ext-autospace: no-autospace** — removes all automatic spacing
- **	ext-autospace: ideograph-alpha** — spacing only between ideographs and alphabetic characters
- **	ext-autospace: ideograph-numeric** — spacing only between ideographs and numerals

## How to Use
1. Apply 	ext-autospace: normal to multilingual paragraphs
2. Use with lang attributes for correct script detection
3. Combine with 	ext-spacing-trim for comprehensive typographic control
