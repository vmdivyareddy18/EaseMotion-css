CSS text-autospace

Automatic spacing between CJK and Latin characters for better multilingual typography.

What is text-autospace?

The text-autospace property automatically inserts spacing at boundaries between different writing systems — for example between Chinese/Japanese (CJK) ideographs and Latin letters. This eliminates the need to manually wrap every foreign word in <span> elements with margin.
