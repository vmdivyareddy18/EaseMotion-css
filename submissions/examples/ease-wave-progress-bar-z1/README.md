# Ease Wave Progress Bar

## Overview

A lightweight animated progress bar with a flowing gradient effect built using HTML and CSS.

## Features

- Pure HTML & CSS
- Animated gradient fill
- Responsive
- Lightweight
- Easy to customize

## Folder Structure

```
ease-wave-progress-bar-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why This Example

This example demonstrates a reusable animated progress indicator suitable for dashboards, upload screens, and analytics interfaces.
