# NPS / CSAT Feedback Widget with Motion

A beautiful, interactive NPS/CSAT feedback widget built for EaseMotion CSS!

## What does this do?

Builds a concise, self-contained example where users can select a score from 0-10 on an NPS-style feedback widget, see a reaction to their score on the button, submit feedback and view a success animation, and transition to a thank you state after submission!

## How is it used?

Include the HTML structure from demo.html and style.css, and use the JavaScript for interactivity!

## Features

- 🔢 0-10 NPS score selector UI
- ⏱️ Staggered hover animation on score buttons
- 🎨 Glass-effect / aesthetic styling
- ✅ Submit feedback and view a success animation
- 🎉 Thank-you state with entrance animation after submit
- 🎯 (optional) CSAT-style label variants (e.g. "Not likely"/"Very likely")
- 📋 Copy-ready HTML snippet for the feedback widget pattern
- ♿ Accessible labels, keyboard-friendly score selection, readable contrast!

## Why is it useful?

EaseMotion CSS includes animation utilities built to attach to UI, but there is currently only one NPS submission in this repository! Beginners building SaaS/feedback flows with a practical example showing how to combine NPS/CSAT score selection, staggered hover motions, and subtle thank you states with EaseMotion animations!
