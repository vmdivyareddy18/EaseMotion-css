# Dot Pattern Grid

An animated dot matrix pattern grid with staggered color-shifting pulses and a centered overlay text.

## Features

- 7×7 grid of dots with staggered animation delays
- Multi-color pulse animation (red → pink → purple → red)
- Scale and opacity changes per dot
- Overlay text with backdrop blur for readability
- Pure CSS grid layout and keyframe animations

## Usage

Include `style.css` and create a `.dot-grid` container with `.dot` elements. Use the CSS custom property `--i` on each dot to stagger animation delays.
