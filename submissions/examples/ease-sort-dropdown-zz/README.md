# Sort Dropdown

## What does this do?
Sort Dropdown component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"sd2-primary\">Sort Dropdown Primary</div>
```

## Why is it useful?
Provides a reusable sort dropdown UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
