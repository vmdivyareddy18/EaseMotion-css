# Authentication Tabs Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive Login and Signup interface.

---

## Components

- Login Form
- Signup Form
- Tab Switching
- Responsive Layout

---

## Folder Structure

```
submissions/examples/auth-tabs-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Click **Login** or **Sign Up** tabs.
3. Observe the tab switching and EaseMotion CSS animations.

---

## Features

- Responsive authentication page
- Login & Signup tabs
- Minimal JavaScript
- Uses existing EaseMotion CSS animation classes
