# Neumorphic Soft Fade-Out Modal

A modern **Pure CSS Fade-Out Modal** designed for **Neumorphic Soft** layouts. The component features a smooth fade-out closing animation, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Smooth Fade-Out modal animation
- Soft Neumorphic UI design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.45s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --bg:#e9eef5;
    --surface:#e9eef5;

    --accent:#5b7cfa;
    --radius:22px;
}
```

You can customize:

- Animation duration
- Animation easing
- Background color
- Accent color
- Border radius
- Shadow intensity

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
