# 🎠 Ease Carousel Slider

A responsive Carousel/Slider example built using EaseMotion CSS.

## Features

- Responsive design
- Previous & Next controls
- Indicator dots
- Optional autoplay
- Smooth hover interactions
- Mobile friendly

## Files

```
ease-carousel-slider/
├── demo.html
├── style.css
├── script.js
└── README.md
```

## EaseMotion Classes Used

- ease-fade-up
- ease-fade-left
- ease-fade-right
- ease-duration-700
- ease-hover-scale
- ease-hover-lift

Made with ❤️ using EaseMotion CSS.
