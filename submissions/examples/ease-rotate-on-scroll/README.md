# Ease Rotate on Scroll

This demo implements a **scroll-linked rotation animation** where an element continuously rotates based on scroll progress.

## 🧠 How It Works
- JavaScript maps `scrollY` to rotation degrees.
- The rotation value is stored in a CSS variable `--ease-scroll-rotate`.
- The element rotates smoothly as the user scrolls.
- Ideal for spinning icons or decorative elements.

## 🧩 Files
- `demo.html` — main demo structure.
- `style.css` — styling and rotation logic using CSS variable.
- `script.js` — scroll event listener mapping scroll position to rotation.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Scroll down to see the icon rotate continuously.
3. Adjust rotation mapping or speed in `script.js` as needed.

## 🧰 Technologies
- HTML5
- CSS3 (variables, transforms)
- JavaScript (scroll event)
