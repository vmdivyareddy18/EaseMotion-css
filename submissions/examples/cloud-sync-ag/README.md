# Cloud Sync

## What does this do?
A cloud sync icon with rotating/pulsing animation to indicate active synchronization of data.

## How is it used?
Apply the `.cloud-sync` class to a container. The cloud icon has a pulsing glow and the arrows rotate to show sync activity.

## Why is this useful?
Provides a recognizable sync state indicator for cloud-based applications without JavaScript, aligning with CSS-first UI patterns.
