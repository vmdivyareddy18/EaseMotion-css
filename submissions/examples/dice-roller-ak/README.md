# Dice Roller

## What does this do?
A dice roller simulation with animated dice faces, configurable number of dice (1-5), total sum display, and roll history.

## How is it used?
Open demo.html in a browser. Select the number of dice and click "Roll" to roll them. The total is displayed and history is tracked.

## Why is this useful?
This provides a dice rolling component for board games, RPGs, or decision-making tools. It demonstrates animation keyframes, random number generation, and history tracking.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to roll dice.
