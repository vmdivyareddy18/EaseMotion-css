## grid-column-row-span

Covers `grid-column` and `grid-row` with `span` for explicit placement across multiple tracks in CSS Grid.

### Key Features

- Column spanning with `grid-column: span <n>`
- Row spanning with `grid-row: span <n>`
- Named line placement using custom line names in `grid-template-columns` and `grid-template-rows`
- Named grid areas with `grid-template-areas` and spanning behavior

### Usage Notes

The `span` keyword counts tracks from the auto-placement position. Combine with named lines for maintainable layouts. Grid areas (`grid-template-areas`) provide a visual map of the layout that is easy to read and modify.
