# Tada Map Embed

Resolves #42425.

### What does this do?
Adds a Tada Map Embed component to EaseMotion CSS tailored for Creative Agencies. It features a stylized map container with a prominent location marker. The marker utilizes an infinitely looping `ease-tada` animation with a custom 3-second duration to playfully draw the user's eye to the agency's location.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Adjust the `top` and `left` percentages on `.map-marker` to position it correctly over your specific map background.

### Why is it useful?
Provides an aesthetic and engaging alternative to standard, boring iframe map embeds. The "tada" animation gives the component a friendly, creative personality suitable for design studios or boutique agencies.
