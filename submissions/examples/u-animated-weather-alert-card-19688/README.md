# Animated Weather Alert Card

A lightweight weather notification component built with HTML and CSS.

## Features

- Animated weather alerts
- Storm warning card
- Heat alert card
- Flood watch card
- Wind advisory card
- Hover effects
- Responsive design

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- Weather dashboards
- Notification systems
- Emergency alert platforms
- Monitoring applications
