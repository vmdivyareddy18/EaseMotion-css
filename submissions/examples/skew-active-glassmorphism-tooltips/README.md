# Pure CSS Skew-Active Glassmorphism Tooltip

A declarative, high-performance, zero-JavaScript tooltip interface component crafted precisely to complement frosted glass plates, blurry backdrops, and modern dark translucent layouts.

## Features
- **Zero Runtime Script Overhead:** Drives the complete structural reveal and skew transformation sequence using native browser CSS interpretation engines.
- **Dynamic Skew-Active Motion:** Leverages multi-axis `skew()` parameters combined with localized `scale()` adjustments on element reveal events.
- **Vibrant Frosted Aesthetics:** Modeled with premium translucent backing panels, distinct fine border strokes, and vibrant underlying radial lighting elements.
- **Keyboard Access Arrays:** Fully interactive via traditional keyboard focus paths while remaining accessible to system screen readers.

## Parameter Customization Options

| CSS Variable Token | Layout Configuration Target Utility | Default Baseline Matrix Value |
| :--- | :--- | :--- |
| `--tooltip-skew-angle` | Peak horizontal distortion skew angle threshold | `6deg` |
| `--tooltip-fade-duration` | Base time parameters for visibility arrival transitions | `0.25s` |
| `--glass-blur` | Standard backplane blur setting for frosted glass items | `16px` |
