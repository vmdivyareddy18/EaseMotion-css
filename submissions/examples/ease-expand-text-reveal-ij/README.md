# Expand Text Reveal

Interactive read-more / show-less card with animated height transition.

## CSS Custom Properties

| Variable | Default | Description |
|---|---|---|
| `--text-bg` | `#1a1a2e` | Card background |
| `--text-color` | `#e0e0e0` | Body text colour |
| `--expand-color` | `#e94560` | Toggle button accent |
| `--expand-duration` | `0.4s` | Height reveal transition |

## Interaction

- Click **Read more** to expand the text container; the gradient fade disappears
- Click **Show less** to collapse back to the truncated view

## Preview

A dark card with a truncated multi-line paragraph and a fade-out gradient at the bottom. A button toggles full visibility.
