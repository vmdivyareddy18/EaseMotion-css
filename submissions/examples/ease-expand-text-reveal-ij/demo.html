What is EaseMotion?

EaseMotion is a lightweight CSS animation library built for modern web interfaces. It provides a collection of drop-in components that bring interfaces to life with smooth, performant animations.

Each component is designed with custom properties in mind, making it easy to theme and integrate into existing projects. From error pages to interactive trees, EaseMotion covers a wide range of UI patterns.

The library emphasises accessibility and respects user preferences for reduced motion. All animations are GPU-accelerated and tested across modern browsers.

Start using EaseMotion today and give your UI the motion it deserves. Simply import the component's CSS and customise the variables to match your brand.
