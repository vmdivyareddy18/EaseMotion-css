# Error Page 404

Interactive 404 error page with floating geometric elements.

## CSS Custom Properties

| Variable | Default | Description |
|---|---|---|
| `--page-bg` | `#16213e` | Card background |
| `--text-color` | `#e0e0e0` | Body text colour |
| `--float-color` | `#0f3460` | Floating element tint |
| `--float-duration` | `6s` | Float animation cycle |

## Interaction

- **Go Home** — triggers a burst-out effect on the floating shapes
- **Toggle Float** — pauses / resumes the floating animation
