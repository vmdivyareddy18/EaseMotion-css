# Scale Chat Bubble

Premium CSS only chat UI for EaseMotion Project.

## Features
- Pure HTML/CSS
- Glassmorphism
- Animated gradient background
- Phone mockup
- Two-sided chat bubbles with tails
- Scale + fade entrance animation
- Floating blurred blobs
- Online status indicator
- Timestamps
- Responsive & accessible
- Hover effects

## Installation
Open `demo.html`.

## Folder Structure
```
scale-chat-bubble-sammuuu11151171/
├── demo.html
├── style.css
└── README.md
```

## Customization
Edit colors, animation timing, gradients and message content in `style.css` and `demo.html`.
