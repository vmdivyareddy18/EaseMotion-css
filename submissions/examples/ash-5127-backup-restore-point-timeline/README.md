# Backup Restore Point Timeline

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive backup restore timeline that displays available restore points in chronological order. Ideal for admin panels, cloud backup systems, disaster recovery dashboards, and infrastructure management interfaces.

## Features

- Vertical restore point timeline
- Latest and archived backup indicators
- Restore point summary
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="event completed">

    <div class="dot"></div>

    <div class="content">
        <h3>Today • 09:30 AM</h3>
    </div>

</div>
```
