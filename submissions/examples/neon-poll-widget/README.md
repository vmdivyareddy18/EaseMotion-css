# Neon Poll Widget

A glowing animated poll component built with pure HTML and CSS.

## Features

- Neon glow effect
- Animated poll bars
- Responsive layout
- No JavaScript required
- Accessible structure
- Lightweight reusable component


## Usage

Add the widget markup:

```html
<section class="poll-widget">
  <h1>Your Question</h1>

  <div class="poll-option">
    <span>Option</span>

    <div class="bar">
      <div class="fill"></div>
    </div>

  </div>

</section>
