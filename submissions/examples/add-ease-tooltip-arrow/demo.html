Animated Directional Tooltip

A CSS-only interactive tooltip featuring a responsive scale entrance and a directional pointing arrow that wiggles towards the trigger element.

Standard Position Variants

Hover or focus the triggers below to see the default entrances. The directional arrow wiggles dynamically on hover.

Viewport-Aware Auto-Positioning

Powered by CSS Anchor Positioning & Popover API. Hover elements placed near screen or box edges. The tooltips auto-flip to stay visible in the viewport and reposition the wiggling arrow dynamically.

⚠️
Browser Requirement: Viewport edge-awareness utilizes modern CSS Anchor Positioning (Baseline 2026). Ensure you are using a modern browser (e.g. Chrome 125+, Edge 125+, Safari 18+ or Firefox 128+) to see native auto-positioning.

<!-- Traditional CSS usage -->
<div class="tooltip-container">
  <button>Hover me</button>
  <div class="tooltip tooltip-top">Tooltip text</div>
</div>

<!-- Viewport Edge-aware usage -->
<button anchor-name="--trigger" popovertarget="tip">Trigger</button>
<div id="tip" popover class="tooltip-anchor-tip" style="position-anchor: --trigger;">
  Tooltip Text
  <span class="anchor-arrow arrow-bottom"></span>
