# Full Job Board / Recruitment Landing Page

1. What does this do? A complete job board landing page featuring job listings with tags, category navigation, company showcase, and applicant CTA built entirely with EaseMotion CSS classes.

2. How is it used? Open `demo.html` in any browser — the page uses `ease-navbar-glass-sticky`, `ease-card`, `ease-badge`, `ease-reveal-up`, `ease-hover-lift`, and other EaseMotion classes for every visual element without custom keyframes.

3. Why is it useful? Shows how EaseMotion utility and component classes compose to build a recruitment platform UI with professional animations, responsive grid layouts, and interactive job cards — all framework-native.
