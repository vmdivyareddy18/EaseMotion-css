# Smart Home Control Panel Dashboard

A responsive Smart Home Dashboard that allows users to monitor and control various connected devices through an interactive interface.

## Features

* Responsive dashboard layout
* Sidebar navigation menu
* Device management cards
* Smart Lights control
* Air Conditioner control
* Ceiling Fan speed selector
* Security Camera status panel
* Smart Door Lock controls
* Energy Monitoring widget
* Animated status indicators
* Interactive toggle buttons
* Activity feed section
* Notification alerts
* Mobile-friendly design
* Pure HTML, CSS, and JavaScript
