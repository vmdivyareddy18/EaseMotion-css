# Ease Chimera Animation

1. What does this do? Creates a highly chaotic, glitching, morphing animation that rapidly shifts scales, translations, skews, and colors.
2. How is it used? Apply `.ease-chimera-ps` to any element you want to look unstable, possessed, or "glitchy" (like a hybrid monster/chimera).
3. Why is it useful? It provides a unique, aggressive visual effect perfect for error states, Halloween themes, gaming UIs, or high-energy marketing elements without relying on complex SVG filters or JS canvas manipulation.
