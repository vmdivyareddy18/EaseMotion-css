# Active Tab Indicator – Sliding underline between tabs

> A production-ready tab navigation component with a smooth sliding underline indicator that animates between active tabs. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Sliding underline indicator** – Smooth sliding animation
- ✅ **Keyboard navigation** – Arrow keys to navigate tabs
- ✅ **Customizable position** – Bottom or top indicator
- ✅ **Animated transitions** – Elastic easing on switch
- ✅ **Accessible** – ARIA roles and keyboard support
- ✅ **4 color variants** – Primary, Success, Danger, Dark
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
