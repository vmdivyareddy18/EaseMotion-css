# Jello Toggle Switch

A playful toggle switch component with a jello animation effect built using pure HTML and CSS.

## Features

- Jello bounce animation
- Neumorphism inspired design
- Pure CSS implementation
- No JavaScript required
- Responsive layout
- Accessible toggle input


## Usage

Add the toggle structure:

```html
<label class="toggle">

  <input type="checkbox">

  <span class="slider">
    <span class="circle"></span>
  </span>

</label>
