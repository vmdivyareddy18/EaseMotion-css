# Baobab

A pure CSS and HTML baobab standing against a savanna sunset, birds crossing the sky.

## What it shows

- The bottle trunk cut from a single multi point clip-path polygon, with a second clipped layer for bark striation
- Branches placed by a custom property angle, each growing two twigs from ::before and ::after
- Bird silhouettes made of two pseudo element wings, flapping on opposite keyframes while they drift across
- Grass tufts cut from a spiky clip-path
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
