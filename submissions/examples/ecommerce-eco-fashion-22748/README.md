# App Landing: Sustainable Eco-Fashion E-commerce (`ecommerce-eco-fashion-22748`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a premium, ethical, and sustainable fashion brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A sustainable fashion brand needs to project an earthy, organic, and highly transparent aesthetic. This template achieves a relaxed, editorial feel using warm off-whites (`#F7F5F0`), sand tones, and moss greens. The typography pairs a soft, elegant serif (`Lora`) for headings with a clean, readable sans-serif (`Work Sans`) for details.

The layout includes:
- A top shipping/returns Announcement Bar.
- An absolute-positioned transparent Navigation.
- A Hero section featuring an abstract CSS linen-texture background and an overlay gradient to ensure text readability.
- A 3-column "Curated for Longevity" collections grid with simulated garment shapes.
- A Split-layout highlighting the brand's materials and water-saving statistics.
- A "Radical Transparency" Carbon Footprint section featuring a CSS-rendered vertical bar chart.
- A Press Mentions grid.
- A Customer Reviews section.
- A Newsletter signup block.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities, configured with a slow, organic, flowing `cubic-bezier(0.25, 1, 0.5, 1)` to mimic natural, unhurried movement:
- `.ease-fade-up` / `.ease-slide-up` (Configured for slow, graceful text and card entrances, reinforcing the "slow fashion" ethos).
- `.ease-reveal-right` (Uses a `clip-path` inset animation to elegantly unveil the material texture block).
- `.ease-zoom-in` (Applied to the hero background for a subtle, cinematic breathing effect).
- `.ease-hover-opacity` (Standard interaction for text links and navigation).
- `.ease-hover-lift` (Applied to collection cards for a gentle, floating interaction).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, the organic easing curves, the abstract CSS garment simulations, and the carbon footprint bar chart. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the premium entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
