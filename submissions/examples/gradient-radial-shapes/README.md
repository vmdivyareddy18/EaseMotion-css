# Gradient Radial Shapes

Explores the `radial-gradient()` CSS function with all shape and size keywords: `circle`, `ellipse`, `closest-side`, `farthest-corner`, `closest-corner`, and `farthest-side`. A comparison section shows identical color stops with each keyword for direct visual comparison.

## Usage

Open `demo.html` in a browser. The six-card grid shows each shape keyword applied to a radial gradient with the same color stops. The comparison section uses a custom `at 30% 40%` position to highlight how each size keyword affects the gradient extent.

## Notes

- `circle` forces a circular gradient; `ellipse` (default) stretches to the element's aspect ratio.
- Size keywords control the gradient's ending shape relative to the element's dimensions.
- Multiple color stops (up to 5) demonstrate smooth transitions between vibrant hues.
