# Capacity Risk Radar

An intermediate EaseMotion CSS example for infrastructure dashboards. A radar sweep rotates across capacity pings while a compact legend explains regional risk levels.

## Files

- `demo.html` - Semantic capacity radar and legend markup.
- `style.css` - Responsive panel layout, rotating radar sweep, pulsing pings, and risk legend styling.

## Usage

Open `demo.html` in a browser to preview the animated capacity risk radar.
