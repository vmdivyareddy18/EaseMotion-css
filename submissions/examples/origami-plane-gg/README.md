# Origami Paper Plane Submit Button

## What it does
This is an interactive "Submit" button micro-interaction. Upon clicking the button, it morphs into an origami-style paper plane and flies off the screen in a smooth, curved trajectory using CSS animations.

## How to use it
1. Clone or download the repository.
2. Navigate to the `submissions/examples/origami-plane-gg/` directory.
3. Open the `demo.html` file directly in any modern web browser.
4. Click the "Submit" button to trigger the animation.

## Why it fits EaseMotion CSS
This component perfectly follows the animation-first philosophy by using CSS `clip-path` for morphing and `@keyframes` for the flight path. It keeps the interaction smooth, lightweight, and visually engaging without requiring heavy JavaScript, fitting seamlessly into modern UI design patterns.
