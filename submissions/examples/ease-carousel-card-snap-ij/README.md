# Ease Carousel Card Snap Component

A high-performance CSS carousel featuring robust structural snap alignment configurations. The slider works completely fluidly through hardware-backed browser engines using minimal script resources.

## Features
- **Native CSS Scroll Snapping:** Employs `scroll-snap-type: x mandatory` and `scroll-snap-align: center` properties for crisp position anchoring.
- **Dynamic Variable Configurations:** All layout parameters including maximum wrapper boundaries, sliding margins, gaps, and custom thematic styling gradients are managed securely via central CSS properties.
- **Two-way Interactive Triggers:** Supports pure multi-directional dragging gestures or trackpads along with click actions linking down to individual tracking page bullets.
- **No Heavy Layout Overlap:** Leverages smart responsive calc parameters ensuring a portion of adjacent hidden cards remains visible to optimize visual affordance.
