# KPI Dashboard Cards

## What does this do?
CSS dashboard metric cards with animated values, trend indicators, progress bars, and mini sparkline charts.

## How is it used?
```html
<div class="kp-card kp-revenue"><div class="kp-icon">$</div><div class="kp-body"><span class="kp-label">Revenue</span><span class="kp-value">$125K</span></div></div>
```

## Why is it useful?
Displays key metrics at a glance. Color-coded categories, trend arrows, progress bars, and sparklines. No JavaScript needed. Ideal for admin dashboards and analytics panels.
