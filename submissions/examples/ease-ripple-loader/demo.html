ease-ripple-loader

Expanding rings
that fade out.

A pure CSS loader — no JavaScript, no dependencies. Drop in style.css, add the markup, and customise with modifiers.

Colour variants

--blue (default)
--teal
--rose
--amber

Size variants

--sm (40 px)
--md (80 px)
--lg (140 px)

Speed variants

--slow (3 s)
--normal (1.8 s)
--fast (1 s)

On dark background — white rings

Usage

<!-- 1. link the stylesheet -->
<link rel="stylesheet" href="style.css" />

<!-- 2. drop in the markup -->
<div class="ease-ripple ease-ripple--blue ease-ripple--md">
  <div class="ease-ripple__ring"></div>
  <div class="ease-ripple__ring"></div>
  <div class="ease-ripple__ring"></div>
  <div class="ease-ripple__dot"></div>   <!-- optional centre dot -->
</div>

<!-- 3. or override via CSS custom properties -->
<style>
  .my-loader {
    --ripple-color: #a78bfa;
    --ripple-size: 100px;
    --ripple-duration: 2.4s;
  }
</style>
