# CSS 3D Perspective Tilt Modal for Marketing Landing Page Layouts

## Overview

This project showcases a modern marketing landing page modal featuring a smooth **3D Perspective Tilt** entrance animation built entirely with HTML and CSS. The component is designed to highlight promotional content, feature highlights, pricing information, and call-to-action buttons in an engaging, responsive interface without relying on JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- CSS-only 3D Perspective Tilt entrance animation
- Glassmorphism-inspired modal design
- Responsive marketing landing page layout
- Interactive feature cards
- Primary and secondary call-to-action buttons
- Keyboard-accessible focus styles
- CSS custom properties for easy customization
- No JavaScript or external libraries required

### Animation

- The modal enters using a smooth 3D perspective tilt effect.
- Rotation along the X and Y axes creates depth during the entrance.
- The card gently settles into place with a subtle scaling transition.
- Hover interactions apply a slight 3D tilt for added interactivity.
- Buttons and feature cards include lightweight hover animations.

## Customization

The component uses CSS custom properties for quick styling.

```css
:root{
    --primary:#7c3aed;
    --secondary:#38bdf8;
    --bg1:#0f172a;
    --bg2:#1e293b;
    --glass:rgba(255,255,255,.12);
    --text:#ffffff;
    --radius:24px;
}
```

You can customize:

- Background gradients
- Accent colors
- Glassmorphism appearance
- Border radius
- Typography
- Feature card styling
- Animation duration
- Button styles
- Shadows and spacing
