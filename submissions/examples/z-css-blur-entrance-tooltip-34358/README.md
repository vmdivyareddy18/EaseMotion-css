# CSS Blur Entrance Tooltip

Pure CSS animated tooltip utilizing a Blur Entrance interaction designed for Responsive Dashboard layouts.

## Features

- Pure CSS implementation
- Blur Entrance animation
- Dashboard aesthetics
- Responsive design
- Keyboard accessible
- CSS custom properties
- Glassmorphism styling
- Smooth 60fps animations
- Zero JavaScript dependency

## Variables

```css
--primary
--secondary
--tooltip-bg
--duration
--ease
```
