# Hover Underline Link

Animated underline that grows from left to right on hover. Color via `--ul-color`, duration via `--ul-dur`. Uses `::after` pseudo-element with CSS transform scaleX transition.

## Features

- Animated underline on hover via ::after pseudo-element
- Custom color via `--ul-color` CSS variable
- Custom duration via `--ul-dur` CSS variable
- Bounce cubic-bezier transition
- Smooth color change on text

## Usage

Add `.hu-link` class to any `<a>` tag. Set `--ul-color` for underline color and `--ul-dur` for animation duration.
