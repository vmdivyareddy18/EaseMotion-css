# Product Return Reason Selector

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive product return reason selector that allows customers to choose a return reason using interactive CSS-only option cards.

## Features

- CSS-only radio selection
- Return reason cards
- Priority badges
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<label class="option">

    <input type="radio" name="reason">

    <span class="indicator"></span>

    <div class="content">
        ...
    </div>

</label>
```

## Main Classes

```css
.card
.options
.option
.indicator

.tag

```
