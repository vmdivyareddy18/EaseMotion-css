# 📄 CSS Pagination Motion

A collection of beautiful, pure CSS animated pagination styles for modern websites.

## 🚀 Features

### Pagination Styles (8 types)
1. **Classic Numbered** - Traditional numbered pages with border
2. **Rounded** - Pill-shaped pages with green theme
3. **Gradient** - Vibrant gradient backgrounds
4. **Neon** - Cyberpunk neon glow effect
5. **Minimal Dots** - Dot-based pagination
6. **Underline** - Animated underline on hover
7. **Pill** - Rounded pill style with purple theme
8. **3D** - 3D depth effect with shadow

### Animation Features
- Smooth hover transitions
- Scale and lift effects
- Underline animations
- Neon glow pulsing
- 3D press effects
- Background fills

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All pagination animations
- `readme.md` - Documentation

## 🛠️ How to Use

### Basic Pagination
```html
<nav class="pagination classic">
    <a href="#" class="page-link">«</a>
    <a href="#" class="page-link">1</a>
    <a href="#" class="page-link active">2</a>
    <a href="#" class="page-link">3</a>
    <a href="#" class="page-link">»</a>
</nav>
