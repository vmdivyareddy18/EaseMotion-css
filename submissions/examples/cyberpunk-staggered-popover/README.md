# Cyberpunk Staggered Entrance Popover

A pure CSS animated popover featuring a staggered entrance animation with a cyberpunk neon interface style.

## Features

- Pure CSS animation
- Staggered entrance effect
- Cyberpunk neon design
- Responsive layout
- Keyboard accessible
- No JavaScript required
- CSS custom properties


## Usage

Open demo.html directly in the browser.


## Customization

Change animation timing:

```css
:root{
 --duration:.5s;
}
