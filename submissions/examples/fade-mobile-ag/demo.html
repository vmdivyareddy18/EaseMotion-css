Fading Element

Expected Behavior

The box should smoothly transition from 0% opacity to 100% opacity over 1.5 seconds.

Actual Behavior on Mobile

On Chrome (Android) and Safari (iOS), the element occasionally appears instantly without any fade transition, or the animation gets skipped.
