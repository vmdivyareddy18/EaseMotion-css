# Glow Drag Drop Zone

A responsive Cyberpunk-inspired Glow Drag Drop Zone built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Cyberpunk glowing hover effect
- Animated glowing border
- Floating upload icon
- Responsive layout
- Accessible HTML
- No JavaScript required

## Files

```
demo.html
style.css
README.md
```

## Animation

- Animated glowing border sweep
- Floating upload icon
- Glow shadow on hover
- Smooth transitions
