# Ease Aurora Timeline Beam

## Overview

A premium animated vertical timeline featuring glowing aurora beams, floating milestone nodes, and elegant glassmorphism styling using pure HTML and CSS.

## Features

- Animated aurora timeline
- Glowing milestone nodes
- Glassmorphism cards
- Smooth hover interaction
- Responsive layout
- Pure HTML & CSS
