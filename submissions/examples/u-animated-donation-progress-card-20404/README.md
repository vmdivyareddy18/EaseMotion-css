# Animated Donation Progress Card

A modern donation and fundraising progress component built using HTML and CSS.

## Features

- Animated fundraising progress bar
- Raised amount and target statistics
- Donor count section
- Campaign milestone card
- Responsive layout
- Interactive action buttons

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- Crowdfunding platforms
- Charity campaigns
- NGO websites
- Fundraising dashboards
- Donation management systems
