# 🔄 CSS Loading Spinners

A collection of beautiful, pure CSS loading animations for modern web applications.

## 🚀 Features

6 different loading spinners:
1. **Classic Spinner** - Traditional rotating circle
2. **Bouncing Dots** - Three dots with staggered animation
3. **Pulse Ring** - Expanding ring effect
4. **Wave Bars** - Vertical bars with wave motion
5. **3D Flip** - Square with 3D flip animation
6. **Gradient Spin** - Conic gradient rotation

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All spinner animations
- `readme.md` - Documentation

## 🛠️ How to Use
1. Open `demo.html` in your browser
2. Copy the CSS classes you need
3. Add the HTML structure to your project
4. Customize colors and sizes as needed
