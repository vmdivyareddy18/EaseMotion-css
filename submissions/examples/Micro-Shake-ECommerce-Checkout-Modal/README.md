# Micro Shake E-Commerce Checkout Modal

A responsive checkout modal featuring a subtle Micro Shake interaction built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive layout
- Keyboard accessible
- E-commerce checkout interface
- Order summary section
- Secure checkout badge
- Micro Shake animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--shake
--primary
--success
--bg
--surface
--text
--muted
--radius
```
