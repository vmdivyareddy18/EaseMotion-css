# Bar Chart Race

A horizontal bar chart race built with CSS transitions and vanilla JavaScript. Bars animate their width from 0% to their target value with staggered delays, creating a racing effect.

## Features

- Horizontal bars with animated width transitions
- Staggered entrance delays for each bar
- Randomize button to generate new values and re-trigger the race
- Color-coded categories
- Value labels on each bar

## CSS Variables

| Variable                | Default  | Description            |
| ----------------------- | -------- | ---------------------- |
| `--bar-race-duration`   | `1.2s`   | Transition duration    |
| `--bar-race-bar-color`  | `#4f46e5`| Default bar color      |
| `--bar-race-bg`         | `#ffffff`| Background color       |
| `--bar-race-label-color`| `#1e293b`| Label text color       |

## Usage

Open `demo.html` in a browser. Click **Randomize & Race** to generate new values and watch the bars animate.
