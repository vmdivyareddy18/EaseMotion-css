# Animated Gradient Background Showcase

## What does this do?
A showcase of animated gradient backgrounds — slow-shifting linear, breathing radial, and spinning conic gradients — each demoed in its own card.

## How is it used?
```html
<div class="gradient-preview slow-shift"></div>
<div class="gradient-preview radial-breathe"></div>
<div class="gradient-preview conic-spin"></div>
```

## Why is it useful?
Animated gradients are popular for hero sections, loading states, and decorative backgrounds. This showcase demonstrates how EaseMotion can bring static backgrounds to life using keyframe animations on background-position, background-size, and gradient angles — fitting the framework's animation-first philosophy.
