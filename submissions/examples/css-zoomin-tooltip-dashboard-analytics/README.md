# CSS Zoom-In Tooltip – Dashboard Analytics Layout

## Overview

A responsive Dashboard Analytics tooltip component built entirely with HTML and CSS. The component features a smooth zoom-in animation, modern analytics widgets, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-In tooltip animation
- Dashboard Analytics interface
- Responsive analytics widgets
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--background`
- `--surface`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--zoom-scale`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any **Details** button to display the animated Zoom-In Tooltip. Colors, spacing, animation timing, zoom scale, and shadows can all be customized using the CSS variables defined in `style.css`.
