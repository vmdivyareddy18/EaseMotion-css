# Real Estate — Commercial Real Estate

## What does this do?
A complete, copy-paste ready full-page website for a Commercial Property Broker named `Nexus Commercial`, featuring industry sector grids (office, retail, industrial), featured active listings, custom market report download inputs, advisor portfolios, client company logos, and inquiry submission forms.

## How is it used?
Open `demo.html` directly in any web browser to view the page. No build step is required. The page transition, reveal animations, and hover components use the project's standard CSS classes.

## Why is it useful?
This showcases how EaseMotion CSS can be applied to build professional, corporate brokerage pages with structured selection grids, custom select dropdown integrations, and alignment layouts.

## EaseMotion CSS Classes Showcased
- **Entrance & Hover Animations**:
  - `.ease-slide-in-left` / `.ease-slide-in-right`: Used for the hero content entrances.
  - `.ease-hover-grow`: Applied to header logo elements and button states.
  - `.ease-hover-lift`: Used for interactive active listings cards.
  - `.ease-hover-underline`: Applied to nav menu links.
  - `.ease-card-lift`: Used for the industry sectors showcase and advisor profiles.
- **Scroll Reveal Animations**:
  - `.ease-reveal`: Applied to sections to hide elements offscreen.
  - `.ease-reveal-up` / `.ease-reveal-scale`: Sliding directional reveals.
  - `.ease-reveal-delay-1` / `.ease-reveal-delay-2` / `.ease-reveal-delay-3`: Staggered timing delays for grid columns.
