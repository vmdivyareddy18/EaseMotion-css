# Ease Zeppelin Float

A CSS animation of a blimp/zeppelin drifting across the sky with floating clouds and a spinning propeller.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.sky` element.

## Custom CSS Variables

| Variable           | Default   | Description                   |
|--------------------|-----------|-------------------------------|
| `--blimp-color`    | `#ddd`    | Color of the zeppelin body    |
| `--gondola-color`  | `#888`    | Color of the gondola and fins |
| `--float-duration` | `6s`      | Duration of one full crossing |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
