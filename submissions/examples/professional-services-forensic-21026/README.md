# Professional Services: Forensic Accountant (`professional-services-forensic-21026`)

This proposal provides a complete, copy-paste ready B2B professional services landing page designed for a forensic accounting, fraud investigation, or litigation support firm, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A forensic accounting site must convey extreme technical competence, authority, and uncompromising objectivity. This template achieves a highly technical, data-driven aesthetic using a dark theme (Deep Charcoal and stark Black), sharp alert-red accents, and a dual typography system pairing a clean sans-serif (IBM Plex Sans) with a monospace font for data points and UI elements (IBM Plex Mono).

The layout includes:
- A hero section featuring a CSS-animated "scanner line" over a data grid background, alongside a simulated anomaly detection visualizer.
- A highly structured 3-column practice areas grid.
- Detailed case study cards mimicking formal investigative reports.
- A split layout showcasing rigorous professional credentials (CPA, CFF, CFE) alongside a client testimonial.
- A confidential consultation form featuring "glow" focus states.
- A technical, organized footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (For standard, clean entrances of text blocks and service cards)
- `.ease-zoom-in` (Applied to the abstract data visualizer in the hero section to simulate a UI element loading)
- `.ease-expand` (Applied to the red divider lines to provide sharp, decisive section breaks)
- `.ease-slide-left` / `.ease-slide-right` (Used in split-layout sections to bring content together from opposite sides of the screen)
- `.ease-hover-lift` (Standard interaction for primary buttons to indicate interactivity)
- `.ease-focus-glow` (Applied to form inputs to create a stark, red "alert" glow on focus, leaning into the technical aesthetic)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, dark-theme styling, and monospace typography mapping, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.
