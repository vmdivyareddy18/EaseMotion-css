# Animated Smart City Monitoring Dashboard

A modern dashboard for monitoring smart city infrastructure, utilities, transportation systems, and environmental metrics.

## Features

- City metric cards
- Animated counters
- Service status indicators
- Infrastructure progress bars
- Responsive layout
- Smooth animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any browser.

- Smart city platforms
- IoT dashboards
- Government portals
- Urban analytics systems
- Infrastructure monitoring
