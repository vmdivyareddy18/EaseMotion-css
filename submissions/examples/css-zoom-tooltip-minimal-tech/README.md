# CSS Zoom-In Tooltip – Minimal Tech Layout

## Overview

A responsive Minimal Tech tooltip component built entirely with HTML and CSS. The component features a smooth zoom-in animation, interactive developer workspace controls, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-In tooltip animation
- Minimal Tech interface
- Interactive developer toolbar
- Responsive layout
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--background`
- `--surface`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--zoom-scale`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any toolbar button to display the animated Zoom-In Tooltip. Colors, animation timing, spacing, shadows, and scaling can all be customized using the CSS variables defined in `style.css`.
