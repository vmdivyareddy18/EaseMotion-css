# Diagonal Skew Slide Button

## What does this do?

Hovering shifts a solid background across an action button diagonally (using a slanted skew transition), revealing contrast text.

## How is it used?

Apply the `skew-slide-uj` class to any button or link styled like a button:

```html
<button class="skew-slide-uj">Explore Features</button>
```

You can customize the background fill color and border colors using standard CSS or properties:

```css
.skew-slide-uj {
  border-color: #6c63ff;
  color: #6c63ff;
}
```

## Why is this useful?

It offers a high-fidelity visual animation on hover, transitioning from an outline button to a filled solid button with a dynamic, slanted slide-in effect. This fits EaseMotion's philosophy of highly visual, zero-dependency, and responsive interaction utilities.
