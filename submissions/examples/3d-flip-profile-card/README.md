# 3D Flip Profile Card

## Description

A modern profile card that flips in 3D to reveal additional content.

## Features

- Smooth 3D rotation
- Pure CSS implementation
- Interactive hover effect
- Lightweight and reusable

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and hover over the card.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
