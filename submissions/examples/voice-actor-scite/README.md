# Voice Actor — Full Page Site

A complete, copy-paste ready full-page website for a professional
voice actor built using EaseMotion CSS classes. Opens directly in
the browser with no build step.

## Structure

```
voice-actor-site/
├── demo.html    — full single-page site
├── style.css    — layout, theming, responsive styles
└── README.md
```

## Sections

| Section | Description |
|---|---|
| Navigation | Fixed dark nav with mobile hamburger |
| Hero | Headline, animated mic visual, soundwave, trust stats |
| Genres | 6 genre cards (Commercial, Animation, Audiobook, eLearning, Games, Corporate) |
| Demo Reels | 4 player mockups with waveform, progress bar, play/pause |
| Client List | 10 brand logos |
| Testimonials | 3 client testimonials |
| Pricing | 3-tier pricing cards with featured highlight |
| Booking Form | Full project inquiry form |
| Footer | 4-column dark footer |

## EaseMotion CSS Classes Used

| Class | Usage |
|---|---|
| `ease-fade-in` | Nav, eyebrows, descriptions, trust stats |
| `ease-slide-up` | Hero title, section headings, cards |
| `ease-zoom-in` | Mic badge, featured pricing card |
| `ease-delay-100` through `ease-delay-500` | Staggered entrance timing |

## Design

- **Display font:** Bebas Neue (high-impact editorial)
- **Body font:** Inter
- **Palette:** Near-black, gold accent, cream for light sections
- **Aesthetic:** Premium, bold, professional, broadcast-ready

## Features

- Pure HTML & CSS — no framework, no build step
- Animated soundwave bars in hero
- Pulsing LIVE badge
- Play/pause toggle on demo reel cards
- Scroll-triggered entrance animations via IntersectionObserver
- Hover lift on all cards
- Fully responsive (mobile, tablet, desktop)
- Respects `prefers-reduced-motion`

## Use Cases

- Voice actor portfolio sites
- Audio production studios
- Podcast host pages
- Narrator / announcer websites
