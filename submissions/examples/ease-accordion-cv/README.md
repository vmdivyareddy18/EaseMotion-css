# Animated Accordion (`ease-accordion-cv`)

A responsive accordion component built entirely with **HTML5** and **Pure CSS**.

## Features

- Smooth expand/collapse animation
- Animated plus/minus icon
- Hover transitions
- Filled and outlined styles
- Responsive layout
- Uses native `<details>` and `<summary>`
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-accordion-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in a browser.

## Browser Support

Compatible with all modern browsers supporting the `<details>` element.
