# Animated Search Bar (`ease-search-cv`)

A lightweight animated search bar built using **HTML5** and **Pure CSS**.

## Features

- Expanding search field
- Animated search icon
- Focus glow animation
- Rounded variant
- Minimal variant
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```
ease-search-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Variants

- Rounded
- Minimal

## Browser Support

Works in all modern browsers.
