# Gradient Stat Cards

## What does this do?

This example creates modern dashboard statistic cards with colorful gradient accent bars, hover elevation effects, and subtle glow interactions.

Included examples:

- Revenue card
- Active users card
- Projects completed card
- Growth rate card

---

## How is it used?

### Basic Structure

```html
<div class="stat-card revenue">
  <div class="accent"></div>

  <h3>$24,580</h3>
  <p>Monthly Revenue</p>
</div>
```

### Available Variants

```html
revenue
users
projects
growth
```

---

## Why is it useful?

Statistic cards are one of the most common dashboard UI patterns.

They are widely used in:

- Analytics dashboards
- SaaS products
- Admin panels
- Business reporting tools
- Portfolio metrics

The component provides a reusable, animation-focused card design while remaining lightweight, customizable, and framework-independent.
