# Rotate Text Exit

A lightweight CSS utility that creates a rotating exit effect for text elements. The animation combines rotation, scaling, and fading to produce a smooth and visually engaging exit transition.

## Features

* Pure CSS implementation
* Smooth rotate exit animation
* Fade-out and scale-down effect
* Lightweight and reusable
* Cross-browser compatible
* No JavaScript required

## Usage

```html
<h1 class="ease-rotate-text-exit">
  Goodbye, EaseMotion!
</h1>
```

## Behavior

* Text rotates during exit
* Gradually fades out
* Slightly scales down for a smoother transition
* Uses easing for natural motion

## Why is it Useful?

It provides a visually appealing way to remove text from the screen, making UI transitions feel smoother and more polished while maintaining simplicity and performance.
