# FAB Keyboard Focus Accessibility Fix

### Overview
This submission provides a fix for the missing keyboard focus state on the Floating Action Button (`.ease-fab`), making it accessible for keyboard users.

### How to use
Open `demo.html` in your browser and use the `Tab` key to focus the button. You will see a clear focus ring.

### Fits EaseMotion
Improves core component accessibility without altering existing layout or hover animations.
