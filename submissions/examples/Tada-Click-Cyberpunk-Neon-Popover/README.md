# Tada-Click Cyberpunk Neon Popover

A responsive **Cyberpunk Neon Popover** built entirely with HTML and CSS featuring a Tada-Click interaction animation.

## Features

- Pure HTML & CSS
- Responsive layout
- Keyboard accessible
- Cyberpunk neon styling
- Tada click animation
- CSS custom properties
- Supports `prefers-reduced-motion`
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Customization

Update the variables inside `:root` to customize colors, glow effects, animation timing, spacing, and border radius.
