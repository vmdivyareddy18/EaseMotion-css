# Dynamic Color Palette Shading & Tinting Mixins

A color display component demonstrating dynamic tint (lightened) and shade (darkened) blends.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting color boxes, text keys, labels, and description text.
2. **`style.css`**: Color-mix definitions, parent palettes, hover transitions, text variables, and card backgrounds.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Verify the Purple Color Scale loads correctly.
3. Confirm that the boxes transition smoothly from light lavender (Tint 2) to deep dark indigo (Shade 2).
