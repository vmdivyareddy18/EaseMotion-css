ease‑command‑bar

A fully animated CMD+K command palette: backdrop fade, panel scale-in from center, results that stagger in as you type, and complete keyboard navigation — all driven by CSS animations with minimal JS for state.

or press ⌘K / Ctrl+K

What's Inside

Each animation layer is a distinct CSS keyframe + transition.

🎭

Backdrop Fade

Frosted glass overlay fades in with opacity + backdrop-filter animation.

⚡

Scale-In Panel

Command panel scales in from 95% with a spring-like cubic-bezier easing.

🎯

Staggered Results

Each result item animates in with a --delay CSS custom property for the stagger effect.

⌨️

Keyboard Nav

↑↓ arrow keys to navigate, Enter to select, Escape to close — fully accessible.

CSS Animation Blueprint

The core animations powering every part of the command bar.

/* Backdrop: fade in with frosted glass */
.cmd-backdrop.is-open {
  animation: backdropIn 0.2s ease forwards;
}
@keyframes backdropIn {
  from { opacity: 0; }
  to   { opacity: 1; }
}

/* Panel: scale from center with spring */
.cmd-panel.is-open {
  animation: panelIn 0.28s cubic-bezier(0.34, 1.56, 0.64, 1) forwards;
}
@keyframes panelIn {
  from { opacity: 0; transform: scale(0.93) translateY(-8px); }
  to   { opacity: 1; transform: scale(1) translateY(0); }
}

/* Results: stagger in with --delay custom property */
.cmd-result-item {
  animation: resultIn 0.22s ease var(--delay, 0ms) both;
}
@keyframes resultIn {
  from { opacity: 0; transform: translateY(6px); }
  to   { opacity: 1; transform: translateY(0); }
}
