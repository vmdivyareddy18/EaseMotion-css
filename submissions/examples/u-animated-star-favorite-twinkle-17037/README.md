# Animated Star Favorite Twinkle

## Overview

A reusable star icon animation featuring a subtle twinkle effect suitable for favorites, ratings, featured content, achievement systems, and bookmarking interfaces.

## Features

- CSS-only animation
- Twinkle effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="favorite-button">
  <span class="star-icon">⭐</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
