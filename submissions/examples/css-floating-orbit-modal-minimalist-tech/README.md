# CSS Floating Orbit Modal for Minimalist Tech Layouts

A lightweight **pure CSS Floating Orbit Modal** designed for Minimalist Tech interfaces. The component features a smooth floating orbit entrance animation, responsive layout, accessibility support, and customizable CSS variables—all without JavaScript.

---

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Smooth Floating Orbit animation
- Responsive design
- Keyboard-friendly structure
- CSS Custom Properties for easy customization
- Lightweight and reusable
- Modern minimalist appearance

## 🚀 Usage

1. Open `demo.html` directly in any modern browser.
2. Click **Launch Modal**.
3. Close the modal using the **×** button or by clicking outside the modal.
