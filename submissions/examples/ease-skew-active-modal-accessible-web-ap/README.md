# Skew-Active Modal — Accessible Web

1. **What does this do?** Adds a smooth Skew-Active animated modal dialog styled for Accessible Web interfaces.
2. **How is it used?** Apply the class `.modal-overlay` with a child `.modal` element, then toggle `.active` to show/hide. See demo.
3. **Why is it useful?** This pattern gives developers a polished, accessible modal component with zero JavaScript dependencies for the animation, fitting the EaseMotion CSS philosophy of human-readable, performant motion.
