How to Use & Customization

Apply the utility class smooth-hover-ag alongside your button, card, link, or custom container class to soften the transitions. It transitions specific properties (colors, borders, shadows, transforms, opacity) using standard EaseMotion tokens. 

<!-- Compose it on any element -->
<button class="btn-ghost smooth-hover-ag">Ghost Button</button>

<!-- Customize speed and easing directly inline -->
<div class="card smooth-hover-ag" style="--smooth-speed: 500ms; --smooth-ease: ease-out;">
  Card Content
</div>

CSS Properties API

Variable Description Default Fallback Value
--smooth-speed Controls the transition duration var(--ease-speed-medium, 300ms)
--smooth-ease Controls the timing transition function var(--ease-ease, cubic-bezier(0.4, 0, 0.2, 1))
