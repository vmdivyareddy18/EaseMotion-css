# Animated Search Icon Pulse

## Overview

A reusable search icon animation featuring a subtle pulse effect that helps draw attention to search functionality.

## Features

- CSS-only animation
- Lightweight micro-interaction
- Search-focused visual cue
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="search-button">
  <span class="search-icon">🔍</span>
</button>
```

## Accessibility

Supports:

```css
@media(prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
