# Animated Personal Finance Budget Tracker

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern finance dashboard showcasing budget tracking, savings goals, spending categories, transaction history, and animated progress indicators using pure HTML and CSS.

## Features

- Budget overview dashboard
- Income, expense, and savings cards
- Animated progress bars
- Savings goal tracker
- Category spending breakdown
- Transaction history panel
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="summary-card income">
  <h3>Income</h3>
  <h2>$5,250</h2>
</div>
```

## Main Classes

```css
.summary-card
.panel
.progress
.fill
.goal-circle
.transaction
```
