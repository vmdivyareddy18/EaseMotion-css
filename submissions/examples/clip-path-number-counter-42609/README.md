# Clip Path Number Counter

## Overview
A responsive, CSS-only travel-inspired number counter that reveals statistics through modern clip-path motion, paired with a polished dashboard-style card layout.

## Installation
1. Open [demo.html](demo.html) directly in a browser.
2. Keep [style.css](style.css) in the same folder so the demo stays self-contained.
3. If you embed the component elsewhere, preserve the local stylesheet links and the repository stylesheet reference.

## Usage
```html
<link rel="stylesheet" href="../../../easemotion.css" />
<link rel="stylesheet" href="./style.css" />
```

Then include the semantic structure from [demo.html](demo.html). The counter uses CSS-only hover and clip-path reveal effects, so no JavaScript is required.

## Accessibility
- Uses semantic structure with a main landmark, heading, and meaningful content sections.
- The cards remain readable and navigable on smaller screens.
- Motion is reduced automatically when the user prefers less animation.

## Why it fits EaseMotion CSS
This submission reflects the EaseMotion philosophy by combining utility-driven layout, smooth motion, and a reusable component pattern in a lightweight, accessible format.
