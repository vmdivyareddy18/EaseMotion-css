# Ease Zigzag Border

A CSS animation of an animated zigzag border pattern that moves continuously along the edges of a box.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.zigzag-box` element.

## Custom CSS Variables

| Variable          | Default   | Description                  |
|-------------------|-----------|------------------------------|
| `--border-color`  | `#4ecdc4` | Color of the zigzag pattern  |
| `--zig-duration`  | `2s`      | Duration of one zigzag cycle |
| `--border-size`   | `4px`     | Thickness of the border      |

## Interactive Controls

Open `demo.html` to use the color pickers and range sliders for live customization.
