# Equalizer Preset Selector

1. What does this do? Audio equalizer preset selection buttons where the active preset gets an animated glow pulse and gradient background highlight.

2. How is it used? Add `.preset-btn` elements in a group. Toggle `.active` class on the selected button to trigger the glow animation and highlight transition.

3. Why is it useful? It provides a polished audio settings UI pattern — preset buttons animate smoothly on selection with a pulsing glow, making the active state clearly visible and engaging.
