# Data Freshness Indicator States

**What does this do?** Shows live, recent, stale, refreshing, and failed data-source states beside dashboard labels.

**How is it used?** Combine the base `badge` class with `live`, `recent`, `stale`, `refreshing`, or `failed`.

**Why is it useful?** It makes data age and sync health scannable through text, borders, symbols, color, and motion with a complete reduced-motion fallback.
