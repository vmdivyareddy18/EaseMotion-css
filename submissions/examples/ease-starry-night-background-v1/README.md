# Ease Starry Night Background

## Overview

Ease Starry Night Background creates a peaceful animated night scene featuring twinkling stars, a glowing moon, and a moving shooting star using only HTML and CSS.

---

## Features

- 🌙 Glowing moon
- ⭐ Twinkling stars
- ☄️ Shooting star animation
- 🌌 Animated night sky
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Moon glow
- 20 animated stars
- Shooting star
- Hero content
- Call-to-action button

---

## Usage

```html
<div class="sky">
...
</div>
```

---

## Customization

- Moon size
- Star count
- Shooting star speed
- Sky colors
- Glow intensity

---

## Browser Support

Compatible with all modern browsers supporting CSS animations and transforms.

---
