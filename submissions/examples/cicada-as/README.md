# Cicada

A pure CSS and HTML cicada clinging to sun baked bark, wings shivering as it belts out its buzzing song.

## What it shows

- Lacy veined wings from two crossed repeating linear gradients over a translucent membrane, quivering in mirrored pairs
- A ridged abdomen and segmented thorax from repeating linear gradients and stacked radial gradients
- Big red compound eyes and angled legs gripping the bark
- Expanding sound rings pulsing outward in time with the wing buzz
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
