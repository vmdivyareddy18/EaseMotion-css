# @starting-style Animations

Explores the `@starting-style` CSS rule for entry animations on dynamically added elements, dialogs, and popovers — enabling smooth first-render transitions.

## Features

- Basic entry animation for dynamically added DOM elements
- Animated `<dialog>` open with `::backdrop` fade
- Popover entry animation using the `popover` attribute
- Side-by-side comparison with and without `@starting-style`

## Usage

Open `demo.html` in a browser that supports `@starting-style` (Chrome 117+ or Edge 117+). Click "Add Card" to see entry animations, open the dialog and popover for overlay animations.
