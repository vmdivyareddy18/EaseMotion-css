# Moray Eel

A pure CSS and HTML moray eel peering from its reef hole, gaping its jaws to breathe and lunging as a fish drifts past.

## What it shows

- Needle teeth on both jaws from a clip-path sawtooth, each row hinged to its own jaw so they part with the gape
- The gape, the lunge and the teeth all driven from one timeline so the whole strike reads as a single motion
- Mottled hide from stacked hard stop radial gradients over a graded body
- A dark reef hole with an inset shadow that the eel genuinely emerges from, plus swaying coral
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
