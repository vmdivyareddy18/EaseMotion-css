# Ease Neural Signal Grid

## Overview

A futuristic grid of animated signal nodes inspired by AI neural networks. Each tile features glowing pulses and glassmorphism styling using only HTML and CSS.

## Features

- Animated signal pulses
- AI-inspired visual design
- Glassmorphism cards
- Responsive CSS Grid
- Smooth hover interactions
- Pure HTML & CSS
