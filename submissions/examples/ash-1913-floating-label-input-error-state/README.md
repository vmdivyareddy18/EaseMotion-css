# Floating Label Input With Error State

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern floating label form input component with validation feedback and animated label transitions.

## Features

- Floating labels
- Error state styling
- Success state styling
- Validation messages
- Smooth focus animations
- Responsive design
- Glassmorphism card layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="input-group error">
    <input type="email" placeholder=" ">
    <label>Email Address</label>
</div>
```

## Main Classes

```css
.input-group
.error
.success
.error-message
.success-message
.save-btn
```
