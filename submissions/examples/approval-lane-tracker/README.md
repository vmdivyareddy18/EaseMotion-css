# Approval Lane Tracker

Approval Lane Tracker is a responsive governance dashboard for release approvals. It shows which review lanes are cleared, which are still active, and how far each approval path has progressed.

## Features

- Six approval lanes with status labels
- Progress bars for cleared and active reviews
- Summary panel for overall approval completion
- Mobile-friendly stacked layout

## Files

- `demo.html` contains the approval lane markup.
- `style.css` contains the layout, lane states, and progress styling.
