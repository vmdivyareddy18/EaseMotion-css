# Issue #22771 - Advanced: E-commerce - Eco Cleaning Products

## What this submission adds

This submission adds a premium, responsive, direct-open landing page for `Verdant Home`, a sustainable home cleaning products e-commerce brand. It is built using only HTML, CSS, inline SVG artwork, and existing EaseMotion CSS classes.

## Required sections included

- Hero with custom inline SVG product advertisement scene
- Products grid for all-purpose cleaner, laundry detergent, dish soap, and surface wipes
- Ingredients transparency section
- Environmental impact stats section
- Subscription plan section
- Customer reviews section
- Newsletter signup section
- Footer

## Responsive behavior

- Desktop: split hero layout, four-column product grid, four impact cards, and multi-column footer
- Tablet: stacked feature areas where needed, two-column product and impact layouts
- Mobile: single-column content flow, stacked cards, wrapped hero badges, and full-width actions

## Browser compatibility

- Modern Chromium browsers
- Firefox
- Safari
- Edge

The page is designed to open directly from `demo.html` without any build step or server runtime.

## How to open `demo.html`

Open `submissions/examples/22771-eco-cleaning-products/demo.html` directly in a browser.

## Notes

- No JavaScript is used
- No frameworks are used
- No external images, icon libraries, or external SVG files are used
- The hero art and product visuals are created with inline SVG and CSS only
- The page links to the repository root `easemotion.css` using a relative path for direct-open compatibility
