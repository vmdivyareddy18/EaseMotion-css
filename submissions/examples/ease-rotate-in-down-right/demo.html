EaseMotion CSS

ease-rotate-in-down-right

An origin-based rotation entrance — the element rotates into view from the upper-left, pivoting around its bottom-right corner while fading in.

How the Pivot Works

transform-origin: right bottom

Entrance in Action

🔄

rotate(45deg)

Starts rotated a full 45° before settling to rotate(0).

📍

Fixed Pivot Point

transform-origin: right bottom anchors the rotation corner.

✨

Opacity 0 → 1

Fade and rotation run on the same timeline, together.

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Add the class to any element */
<div class="ease-rotate-in-down-right">Hero content</div>

/* 3. Optional stagger delays */
<div class="ease-rotate-in-down-right delay-1">First </div>
<div class="ease-rotate-in-down-right delay-2">Second</div>

/* 4. Customise angle, origin, or speed */
.my-card {
  --rotate-angle: 60deg;
  --rotate-duration: 1.1s;
}
