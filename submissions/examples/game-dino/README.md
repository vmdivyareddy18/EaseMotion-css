# Dino Game · bro 🦕

A classic Chrome-dino-style endless runner built with HTML, CSS, and vanilla JavaScript.  
Jump over cacti, survive as long as possible, and beat your high score.

## 🎮 How to Play
- **Press SPACE** or **click/tap** the game screen to start.
- **Press SPACE** or **click/tap** to make the dino jump.
- Avoid the green cacti — every collision ends the game.
- Your score increases the longer you survive.

## 📁 Files
- `demo.html` – main page with embedded JavaScript logic.
- `style.css` – all visual styling and responsive layout.
- `README.md` – this file.

## 🚀 Run It
1. Download all three files into the same folder.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, etc.).
3. Press SPACE or click the canvas to jump right in.

## ✨ Features
- Clean retro aesthetic with animated clouds and grass.
- Responsive canvas that adapts to smaller screens.
- Score tracking and game-over overlay with final score.
- Touch-friendly (tap to jump on mobile devices).

## 🛠️ Tech Stack
- **HTML5** – structure & canvas.
- **CSS3** – styling, shadows, and responsive layout.
- **Vanilla JavaScript** – game loop, physics, collision detection, and rendering.
