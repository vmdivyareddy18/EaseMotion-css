# Component: Animation Fill-Mode Documentation

This submission documents animation-fill-mode behavior for EaseMotion classes for issue **#13633**.

## Description

A documentation page explaining how `animation-fill-mode` values (none, forwards, backwards, both) affect EaseMotion animation classes. Includes a table mapping each animation class to its recommended fill-mode, behavior description, and use case.

## Acceptance Criteria

- Table with class name, fill-mode value, behavior, and use case
- Covers all major EaseMotion entrance and emphasis animations
- Explains why specific fill-mode values are chosen
- Demo element with fill-mode: both
