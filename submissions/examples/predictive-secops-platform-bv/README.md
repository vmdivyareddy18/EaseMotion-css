# Predictive Security Operations Platform

Centralized security operations intelligence dashboard incorporating SVG telemetry charts, forecast confidence counters, and warning status badges.

---

## Features
- **Threat Probability Chart:** Renders multi-hour vector forecasts using raw SVG lines.
- **Predicted Incidents Feed:** Visualizes upcoming incidents with neon indicators.
- **Forecast Integrity Counters:** Displays predictive confidence and metrics.
