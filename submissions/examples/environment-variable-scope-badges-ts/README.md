# Environment Variable Scope Badges

## What does this do?

Shows environment variable rows with development, preview, production, secret, and inherited scope states.

## How is it used?

```html
<div class="env-row is-production">
  <span>STRIPE_KEY</span>
  <b>Production</b>
</div>
```

## Why is it useful?

Developer settings screens need compact visual language for environment scope and secret state.
