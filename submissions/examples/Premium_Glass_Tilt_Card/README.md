# Premium Glass Tilt Card

**What does this do?**
Creates a premium glassmorphism card that lifts and tilts in 3D space when hovered. The effect combines perspective transforms, enhanced shadows, and a glowing gradient border to create a modern depth illusion.

**How is it used?**

```html
<div class="card-tilt">
  <div class="badge">POPULAR</div>

  <h2>Premium Plan</h2>

  <p>
    Unlock advanced analytics, unlimited projects,
    priority support, and premium features.
  </p>

  <div class="price">
    $19<span>/month</span>
  </div>

  <button>Get Started</button>
</div>
```

**Why is it useful?**
Interactive cards are a core element of modern web interfaces. While simple hover scaling is common, a subtle tilt effect creates a stronger sense of depth and responsiveness without overwhelming the user. The glassmorphism styling and animated glow make cards feel more premium, helping important content such as pricing plans, featured products, portfolio projects, and dashboard widgets stand out. The effect remains lightweight, visually appealing, and highly reusable across different layouts.
