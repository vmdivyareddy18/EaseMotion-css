# Glassmorphism Staggered Entrance Accordion

A modern **Pure CSS Staggered Entrance Accordion** designed for **Glassmorphism UI** layouts. The component features frosted glass styling, staggered entrance animations, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Staggered Entrance animation
- Glassmorphism UI design
- Frosted glass effect using backdrop blur
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.6s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --bg1:#4f46e5;
    --bg2:#06b6d4;

    --glass-bg:rgba(255,255,255,.15);
    --glass-border:rgba(255,255,255,.28);

    --radius:20px;
}
```

You can customize:

- Animation duration
- Animation easing
- Background gradient
- Glass transparency
- Border opacity
- Border radius
