# Animated Scroll Indicator (`ease-scroll-indicator-cv`)

A lightweight animated scroll indicator component built using **HTML5** and **Pure CSS**.

## Features

- Animated mouse scroll indicator
- Animated chevron arrows
- CSS progress bar animation
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-scroll-indicator-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Browser Support

Works in all modern browsers.
