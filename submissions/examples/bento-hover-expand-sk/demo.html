How it works

:has() as a Relational Selector

The rule .bento:has(.bento__cell:hover) .bento__cell:not(:hover) selects all non-hovered cells inside a grid that contains a hovered cell. This is the "parent selector" developers have wanted for decades — now in CSS.

Supported in Chrome 105+, Safari 15.4+, Edge 105+, Firefox 121+.
