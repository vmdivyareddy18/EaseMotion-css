# Ease Cyber Orbit Dashboard

## Overview

A futuristic dashboard widget featuring rotating orbital rings, glowing status nodes, and a glassmorphism control hub built entirely with HTML and CSS.

## Features

- Dual orbital animations
- Animated status nodes
- Glassmorphism center panel
- Responsive layout
- Lightweight implementation
- Pure HTML & CSS

## Folder Structure

```text

```

Open `demo.html` in any modern browser.
