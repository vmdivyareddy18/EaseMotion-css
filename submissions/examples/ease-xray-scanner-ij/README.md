# Ease X-Ray Scanner

A CSS animation of a scan line sweeping across a dark panel with a glow effect, simulating an X-ray or security scanner display.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.scanner-panel` element.

## Custom CSS Variables

| Variable          | Default    | Description                      |
|-------------------|------------|----------------------------------|
| `--scan-color`    | `#4ecdc4`  | Color of the scan line and glow  |
| `--scan-duration` | `2s`       | Duration of one full scan cycle  |
| `--bg-color`      | `#1a1a2e`  | Background color of the page     |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
