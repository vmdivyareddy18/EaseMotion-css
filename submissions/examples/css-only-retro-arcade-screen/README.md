# Pure CSS Animated Retro Arcade Start Screen

A highly aesthetic, premium component that perfectly recreates the nostalgic "Insert Coin" / "Press Start" interface of classic 80s arcade machines, built entirely without JavaScript. 

## Features
- **CRT Scanlines:** Pure CSS `repeating-linear-gradient` overlays to simulate CRT monitor scanning.
- **Neon Text Glows:** Advanced `text-shadow` layers creating a realistic 8-bit neon glow effect.
- **Blinking Animations:** Keyframe-driven pulsing opacity for the iconic "Press Start" text.
- **Vignette & Scan Distortion:** Uses `radial-gradient` and subtle keyframes to simulate screen flicker.

## Files
- `demo.html` - Full-screen interactive showcase of the arcade screen.
- `style.css` - The CSS implementation detailing the keyframes, variables, and typography.

## Details
This component serves as a showcase for high-end CSS rendering. It demonstrates how developers can use EaseMotion-css principles to build fully immersive, themed UI elements relying strictly on hardware-accelerated animations (`opacity`, `text-shadow`, `background-position`) without needing external JS libraries or Canvas elements.
