ease-hover-float

Hover any element below — it lifts off the surface while its shadow grows and softens beneath it, simulating a real physical float.

Hover the Cards

🪶

Light Lift

A subtle 6px rise paired with a soft, wide shadow underneath.

🎈

Physical Feel

A bouncy cubic-bezier overshoot makes the float feel weightless.

🌤️

Shadow Spread

Shadow blur and spread both increase, not just the offset.

🔥

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Add the class to any element */
<div class="ease-hover-float">Hover me</div>

/* 3. Customise the float per-instance */
.my-card {
  --float-distance: -10px;     /* rise further */
  --float-duration: 0.45s;     /* slower float */
  --shadow-lift: 0 30px 50px rgba(0,0,0,0.5);
}
