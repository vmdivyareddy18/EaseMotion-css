Introduction

Welcome to the target pseudo-class demo. Click the navigation links above to jump between sections. Each targeted section will highlight with a distinct border glow and fade-in animation.

The :target pseudo-class applies styles only to the element whose id matches the current URL fragment. This enables pure-CSS section highlighting without JavaScript.

Features

  • Zero JavaScript required
  • Custom highlight per section
  • Smooth scroll behavior
  • Graceful fallback for older browsers

Each section in this demo has a unique accent color that appears when it becomes the target. The highlight effect includes a colored border, subtle background change, and a fade-in animation.
