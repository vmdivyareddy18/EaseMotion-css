# Animated NFC Tap Ripple

## Overview

Animated NFC Tap Ripple is a lightweight CSS animation that simulates a contactless payment interaction using expanding ripple waves.

## Features

- CSS-only animation
- NFC ripple effect
- Contactless payment UI
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="nfc-tap-ripple">
  <span class="device">📱</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
