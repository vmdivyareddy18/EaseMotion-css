# Animated Course Progress Tracker

A responsive learning dashboard showcasing course progress, achievement badges, streak tracking, and animated progress indicators.

## Features

- Animated progress bars
- Course cards
- Achievement badges
- Learning streak section
- Hover interactions
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

You can customize:

- Course names
- Progress percentages
- Badge icons
- Colors
- Animation durations
- Layout spacing

## Use Cases

- LMS platforms
- Coding bootcamps
- Online courses
- Student dashboards
- Certification tracking
