# CSS Ripple Click Effect

## What does this do?
Material-inspired expanding circular ripple animation on click for buttons, cards, and interactive elements.

## How is it used?
```html
<button class="rp-btn rp-ripple">Click Me</button>
<div class="rp-card rp-ripple">Card content</div>
```

## Why is it useful?
Provides tactile feedback on interactions without JavaScript. Works on any element. Multiple color themes, shapes, and hover variants. Accessible with reduced-motion support.
