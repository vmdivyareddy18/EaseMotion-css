# Success Check Draw

A confirmation state that scales in a success circle and draws a check mark across it.

## Files

- `demo.html` - status card markup for a completed action.
- `style.css` - circle pop, check draw, and reduced-motion fallback.

## Highlights

- Uses a pseudo-element to draw the check mark.
- Keeps the confirmation message visible beside the animation.
- Provides a static success state for reduced-motion users.
