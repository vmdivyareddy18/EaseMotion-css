# 🦘 Doodle Jump

A classic Doodle Jump arcade game built with HTML, CSS, and JavaScript. Control a cute character as it bounces endlessly upward on moving platforms. How high can you go?

## 🎮 How to Play

1. **Click "Start Game"** or press **Space** to begin
2. **Jump** automatically on platforms
3. **Move left/right** using **Arrow keys** or **A/D** to land on platforms
4. **Keep climbing** as high as possible!
5. **Fall off** and the game ends

## ⌨️ Controls

| Action | Key |
|--------|-----|
| Move Left | `←` or `A` |
| Move Right | `→` or `D` |
| Start Game | `Space` |
| Reset Game | `R` |
| Touch | Swipe left/right on canvas |

## ✨ Features

- 🦘 **Cute Doodle Character** with animated legs
- 📈 **Endless Climbing** with procedural platform generation
- 🏆 **Score & High Score** tracking with local storage
- 🎨 **Beautiful Visuals** with gradient backgrounds and stars
- 🖐️ **Touch Support** for mobile play
- ⌨️ **Keyboard Controls** with multiple key options
- 📱 **Responsive Design** for all devices
- 🎯 **Dynamic Platforms** with moving and static types

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3 (Gradients, responsive design)
- Vanilla JavaScript
