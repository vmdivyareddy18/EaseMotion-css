# Skeleton Avatar with Shimmer Effect

A loading placeholder avatar component with an animated shimmer effect.

## Features

- Circular avatar placeholders
- Smooth shimmer loading animation
- Multiple avatar sizes
- Profile card example
- Responsive design
- Reduced motion support

## Sizes

- Small
- Medium
- Large
