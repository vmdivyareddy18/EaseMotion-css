# 🎯 Shooting Gallery

A fast-paced shooting gallery game where targets appear randomly and you must click them before they disappear. Test your aim and reflexes in this 30-second challenge!

## 🎮 How to Play

1. Click **"Start Game"** or press **Space** to begin
2. **Targets** appear randomly around the screen
3. **Click** or **tap** targets to shoot them
4. Each target has a different point value
5. **Score as many points** as possible in 30 seconds!
6. Targets disappear after a short time — be quick!

## 🎯 Target Types & Points

| Target | Points |
|--------|--------|
| 🎯 Bullseye | +10 |
| 🎪 Clown | +8 |
| 🦆 Duck | +7 |
| 🎈 Balloon | +6 |
| ⭐ Star | +5 |

## ⌨️ Controls

| Action | Key / Input |
|--------|-------------|
| Start Game | `Space` or Click Start |
| Shoot Target | Click or Tap on target |
| Reset Game | `R` |

## ✨ Features

- 🎯 **5 Target Types** – Each with unique emoji and point values
- ⏱️ **30-Second Timer** – Race against the clock
- 📊 **Real-time Stats** – Score, hits, and time remaining
- 🏆 **Best Score** – Saved in browser local storage
- 🎨 **Visual Feedback** – Hit animations, miss indicators
- 🔊 **Sound Effects** – Satisfying hit sounds via Web Audio
- 📱 **Responsive Design** – Works on desktop and mobile
- 🖐️ **Touch Support** – Play on mobile devices
- ✨ **Smooth Animations** – Targets appear with pop effect

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations, gradients, responsive design)
- Vanilla JavaScript (Web Audio API for sounds)
