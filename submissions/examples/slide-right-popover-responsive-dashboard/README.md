# Slide Right Popover - Responsive Dashboard Layouts

A pure CSS animated slide-right popover component designed for modern responsive dashboard interfaces.

## Features

- Smooth slide-right entrance animation
- Pure CSS (No JavaScript)
- Responsive mobile support
- Keyboard focus support
- Customizable animation variables
- Lightweight and reusable


## Usage

Add the popover inside any interactive element:

```html
<button class="dashboard-btn">
  View Details

  <div class="popover">
    Dashboard information
  </div>

</button>
