What is the difference between row-gap and column-gap? +
row-gap sets the spacing between rows (vertically), whereas column-gap sets the spacing between columns (horizontally). gap is a shorthand that sets both.
Which layout displays support row-gap? +
Modern CSS flexbox and grid layouts fully support the row-gap property. It does not apply to block layouts unless they are converted to flex or grid containers.
Why use separate row-gap utilities? +
Sometimes you want tight columns but wide rows (or vice versa). By using row-gap independently of column-gap, you gain absolute control over vertical pacing.
