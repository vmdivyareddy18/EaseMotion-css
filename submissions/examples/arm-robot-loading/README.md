# Robotic Arm Loading Component (CSS-Only)

An advanced loading indicator that coordinates multiple synchronized CSS timelines to simulate a functional keyframe-driven robotic manufacturing loop.

## Features
* **Iterative Multi-Axis Kinematics:** Leverages layered structural hierarchies to cascade translation offsets cleanly down base segments without rendering interference.
* **Reflow Isolation Compliance:** Isolates structural loops inside `transform: rotate() translate3d()` properties, keeping frame paint rates at a fluid 60fps.
* **Simplified Variable Injectors:** Exposes a concise configuration map (`--ease-robot-*`) to quickly adjust variables like colors, speeds, sizing, and acceleration.

## Custom Property Profiles

| CSS Custom Variable | Baseline Token Value | Target Context |
| :--- | :--- | :--- |
| `--ease-robot-speed` | `2.8s` | Full sequence timeline duration index. |
| `--ease-robot-accent` | `#38bdf8` | Visual highlights used on claw elements. |
| `--ease-robot-package-color` | `#f97316` | Material fill token given to the payload block. |
