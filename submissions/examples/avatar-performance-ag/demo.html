1. Hover Lift Effect

Uses transform: translateY() instead of margin/padding for smooth, layout-free lifting.

2. Active Selection Ring

Uses a pseudo-element with opacity and transform: scale() instead of animating borders to avoid layout shifts.

3. Notification Pulse

Continuous status indicator animation utilizing transform: scale() and opacity on a pseudo-element.

4. Avatar Group Expansion

Uses transform: translateX() on hover to reveal adjacent avatars rather than transitioning margins.
