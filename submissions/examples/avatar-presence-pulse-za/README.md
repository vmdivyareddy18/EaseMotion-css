# Avatar with Presence Pulse Indicator

Avatars with pulsing presence indicator rings showing online, idle, and offline states using CSS box-shadow animations.

## Files

- `demo.html` - Three avatars with different presence states
- `style.css` - Pulsing box-shadow keyframes for each state

## Usage

Open `demo.html` to see avatars with animated presence indicators.
