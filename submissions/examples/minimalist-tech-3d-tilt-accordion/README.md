# Minimalist Tech 3D Perspective Tilt Accordion

A modern **Pure CSS 3D Perspective Tilt Accordion** designed for **Minimalist Tech** layouts. The component features subtle 3D tilt interactions, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- 3D Perspective Tilt hover animation
- Clean Minimalist Tech design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.4s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --bg:#f8fafc;
    --surface:#ffffff;

    --accent:#2563eb;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent color
- Background color
- Border radius
- Border color

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
