# Ripple Wave Time Picker

Issue: #42312

## Overview

The Ripple Wave Time Picker is a lightweight animated time input component created for the EaseMotion CSS library. When the user hovers over or focuses on the time input, a ripple wave animation expands smoothly, providing visual feedback without using JavaScript.

## Features

- Pure HTML & CSS
- Responsive layout
- Accessible native time input
- Ripple wave animation
- No JavaScript required
- Uses EaseMotion-style CSS variables and keyframes

## Usage

```html
<div class="ease-ripple-time">
    <input type="time">
    <span class="ripple"></span>
</div>
```

Include:

```html
<link rel="stylesheet" href="style.css">
```
