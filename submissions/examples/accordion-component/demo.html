Accessible Accordion Component

A fully accessible accordion component using semantic HTML and EaseMotion CSS animations. Built with proper ARIA attributes and keyboard navigation support.

Features

  • ✅ Semantic HTML using <details> and <summary>
  • ✅ Keyboard navigation support (Enter/Space to toggle)
  • ✅ Smooth expand/collapse animations
  • ✅ Screen reader compatible
  • ✅ Focus management

What is EaseMotion CSS?

EaseMotion CSS is an animation-first CSS framework with reusable UI components, modern effects, and zero dependencies. It's lightweight, beginner-friendly, and open-source.

How do I get started?

Simply include the EaseMotion CSS file in your project and start using the predefined classes. Check the documentation for a complete list of available animations and components.

Is it free to use?

Yes! EaseMotion CSS is released under the MIT License. You can use it in personal and commercial projects for free.

Does it support all browsers?

EaseMotion CSS supports all modern browsers including Chrome, Firefox, Safari, and Edge. Some animations may require fallback styles for older browsers.

Implementation

HTML

<div class="accordion">
  <details class="accordion-item">
    <summary class="accordion-header">
      <span class="accordion-icon">
        <!-- Chevron icon -->
      </span>
      <span class="accordion-title">Question Here</span>
    </summary>
    <div class="accordion-content">
      <p>Answer content here.</p>
    </div>
  </details>
</div>

CSS

.accordion-item {
  background: var(--ease-color-surface);
  border-radius: var(--ease-radius-md);
  margin-bottom: var(--ease-space-2);
  overflow: hidden;
}

.accordion-header {
  display: flex;
  align-items: center;
  gap: var(--ease-space-3);
  padding: var(--ease-space-4);
  cursor: pointer;
  user-select: none;
}

.accordion-header:focus {
  outline: 2px solid var(--ease-color-primary);
  outline-offset: -2px;
}

.accordion-icon {
  transition: transform var(--ease-speed-normal) var(--ease-ease);
}

details[open] .accordion-icon {
  transform: rotate(180deg);
}

.accordion-content {
  padding: 0 var(--ease-space-4) var(--ease-space-4);
}
