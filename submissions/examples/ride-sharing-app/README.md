# 🚖 RideNow - Ride Sharing App Demo

A responsive landing page for a **Ride Sharing / Transportation App** built using **EaseMotion CSS** utility classes. This example demonstrates how to create a modern transportation landing page with smooth entrance animations and interactive hover effects.

---

## 📁 Folder Structure

```
submissions/
└── examples/
    └── ride-sharing-app/
        ├── demo.html
        ├── style.css
        └── README.md
```

---

## ✨ Features

- 🚗 Hero section with ride booking mockup
- 📍 Real-time tracking feature
- ⭐ Driver rating system
- 💳 Secure payment options
- 👨‍✈️ Driver signup CTA
- 🗺️ City coverage section
- 💰 Ride pricing model
- 💬 Driver & rider testimonials
- 📱 Google Play & App Store download buttons
- 📱 Fully responsive layout

---

## 🎬 EaseMotion Classes Used

- `ease-fade-up`
- `ease-fade-in`
- `ease-slide-up`
- `ease-zoom-in`
- `ease-hover-grow`
- `ease-hover-glow`
- `ease-hover-lift`

---

## ▶️ How to Run

1. Open the project folder.
2. Open `demo.html` in your preferred browser.
3. No installation or build process is required.
