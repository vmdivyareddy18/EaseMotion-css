# ApexCFO — Fractional Financial Leadership E-Commerce

A full-page, fully responsive landing page designed for a Fractional CFO and financial advisory firm. Built exclusively utilizing the `EaseMotion` CSS utility class architecture to deliver a trustworthy, modern corporate aesthetic.

## 📈 Theme & Aesthetic
**Strategic Corporate Leadership:**
* **Colors:** Deep Trust Navy (`#0F2C59`), Light Slate (`#F8F9FA`), Crisp White (`#FFFFFF`), and Professional Gray (`#5C6B7A`).
* **Typography:** `Plus Jakarta Sans` (sharp, modern headings conveying authority) paired with `Inter` (highly legible, trustworthy body text).
* **Vibe:** Professional, data-driven, strategic, and secure.

## 🚀 How to Run
This template requires **zero build steps**. 
1. Clone or download the repository.
2. Ensure `demo.html` and `style.css` are in the same directory.
3. Open `demo.html` directly in your favorite web browser.

## 🛠️ EaseMotion Classes Showcased
This project relies on the core EaseMotion CSS utility framework to build complex layouts and micro-interactions without writing custom CSS logic in the HTML.

### Layout & Structure
* `ease-container`
* `ease-flex`, `ease-grid`
* `ease-grid-cols-2`, `ease-grid-cols-3`, `ease-grid-cols-4`
* `ease-gap-4`, `ease-gap-6`, `ease-gap-8`, `ease-gap-12`
* `ease-items-center`, `ease-justify-between`, `ease-justify-center`

### Theming Utilities
* `ease-bg-main`, `ease-bg-surface`, `ease-bg-primary`, `ease-bg-light`
* `ease-text-main`, `ease-text-primary`, `ease-text-white`, `ease-text-muted`
* `ease-border`, `ease-border-y`
* `ease-font-bold`, `ease-font-display`

### Animations & Scroll Reveals
* `ease-reveal`, `ease-reveal-up`, `ease-reveal-scale` (Intersection Observer triggered for smooth scroll-in)
* `ease-fade-in`, `ease-slide-up`, `ease-bounce-in` (Initial page load transitions)
* `ease-delay-100` through `ease-delay-500` (Staggered element loading)

### Micro-Interactions (Hover)
* `ease-hover-lift` (Y-axis translation + shadow expansion for buttons and cards)
* `ease-hover-scale` (Scale scaling for pricing tiers to emphasize value)
* `ease-hover-fade` (Subtle opacity shift on logos)
* `ease-hover-underline` (Animated bottom border on navigation links)

## 📱 Responsiveness
The layout seamlessly adapts across devices. Desktop 3-column service grids and pricing tables degrade to a centralized, single-column design on mobile devices. Font sizes utilize `clamp()` to scale perfectly across viewports without breaking the financial dashboard-style layout.
