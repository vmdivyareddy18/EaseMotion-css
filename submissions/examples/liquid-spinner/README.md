# Liquid Spinner

A CSS-only animated liquid spinner component inspired by Admin Panel loading patterns.

## Features

- Smooth liquid wave animation
- Multiple layered motion effects
- Pure CSS implementation
- Responsive design
- Accessible loading indicator
- No JavaScript required

## Usage

```html
<div class="spinner" role="status">
  <span></span>
  <span></span>
  <span></span>
</div>
