# Animated Skill Badge Component

## What does this do?
This component provides reusable animated skill badges for displaying technologies, skills, or categories with smooth CSS-based motion effects.

## How is it used?

Add the badge class to any skill or tag element:

```html
<span class="skill-badge">Python</span>
<span class="skill-badge">Machine Learning</span>
<span class="skill-badge">React</span>
