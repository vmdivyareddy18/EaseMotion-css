# Business Landing — URL Shortener SaaS (`saas-landing-url-shortener-21711`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for Link Management, URL Shortening, and Analytics platforms (similar to Bitly or Dub.co), using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A link shortener SaaS needs a clean, utility-focused, and developer-friendly aesthetic. This template uses a crisp UI with monospace accents for data/code and bold typography.

The layout includes:
- A hero section featuring an inline link shortener input and an animated CSS-built link management dashboard mockup.
- A global stats banner with count-up style numbers.
- A 6-card platform feature grid.
- A dark-mode developer API section featuring a simulated CSS code editor with cURL requests and JSON responses.
- A volume-based tiered pricing layout.
- A comprehensive footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth text entrances, stats banners, and simulating data loading in the mockups)
- `.ease-slide-up` (For structural layout animations in pricing, features, and specifically the cascading link items in the dashboard mockup)
- `.ease-zoom-in` (For drawing attention to the primary mockups and the API code window)
- `.ease-bounce-in` (For the primary link shortener input group and CTA buttons)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For interactive buttons, feature cards, and the dashboard mockups)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes. The dashboard and code windows are built entirely with HTML/CSS.
