# Scale Chip (`ease-scale-chip-ksk`)

1. What does this do?  
Displays interactive category/selection pills (chips) that elevate and scale in 3D perspective space on hover, and pop-out with a springy checkmark animation when selected.

2. How is it used?  
Associate a hidden checkbox input `.chip-input` with a label `.scale-chip` containing the text and the checkmark SVG `.chip-check-icon`.

3. Why is it useful?  
It provides a responsive, highly tactile filtering element built purely with CSS 3D transforms, avoiding JavaScript selection state listeners.
