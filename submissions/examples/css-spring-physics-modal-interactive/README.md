# CSS Spring Physics Modal (Interactive UI Theme)

A pure CSS modal design variant calibrated using high-rebound spring acceleration variables, styled specifically to match rich interactive console designs.

## Features
- **Scriptless Execution Layers**: Completely managed using structural CSS label references and active pseudoselectors.
- **Parametric Elastic Layout**: Controls velocity states natively via token structures located under the component stylesheet core file.
- **Interface Acceleration Curve**: Programmed with custom parameters that amplify spatial overshoot before deceleration.

## Parameter Map
Adjust the timing dynamics by editing these variables in `style.css`:

| Custom Token Variable | Default Value | Role |
|---|---|---|
| `--spring-duration` | `0.72s` | Elastic stabilization timeline window. |
| `--spring-easing` | `cubic-bezier(0.34, 1.76, 0.42, 0.94)` | Elastic physics configuration sequence. |
| `--modal-scale-start` | `0.65` | Initial scaling entry displacement point. |
