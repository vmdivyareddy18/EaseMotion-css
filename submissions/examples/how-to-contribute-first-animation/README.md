# How to Contribute Your First Animation to EaseMotion CSS

A beginner-friendly walkthrough to make your first open source contribution!

**Time needed:** ~15 minutes

## What You'll Learn
- How to find an issue and fork the repo
- How to clone and create a branch
- How to build a simple bounce animation
- How to submit a Pull Request

## Animation: ease-bounce

A simple bouncing animation that moves an element up and down smoothly.

### Usage
```html
<div class="ease-bounce">Hello! I bounce!</div>
```

### CSS Class
```css
.ease-bounce {
  animation: ease-bounce 0.8s ease-in-out infinite;
}
```

## Steps to Contribute

1. Fork the repo on GitHub
2. Clone your fork locally
3. Create a new branch
4. Copy the TEMPLATE folder
5. Build your animation
6. Commit and push
7. Open a Pull Request

## Common Mistakes

- Working directly on main branch
- Wrong folder structure
- Missing required files (demo.html, style.css, README.md)
