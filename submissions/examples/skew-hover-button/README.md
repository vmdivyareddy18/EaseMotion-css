# Skew Hover Button Utility

## Description

A reusable CSS utility that applies a subtle skew animation to buttons on hover. The button slightly tilts, scales up, and displays a stronger shadow to create a modern interactive effect.

## Features

- Smooth skew hover animation
- Slight scale effect
- Enhanced shadow on hover
- Lightweight and reusable
- Pure HTML & CSS

## Usage

```html
<button class="skew-hover-button">
    Hover Me
</button>
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
