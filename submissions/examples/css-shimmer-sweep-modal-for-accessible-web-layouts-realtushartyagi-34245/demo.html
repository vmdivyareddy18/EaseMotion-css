Why Accessibility Matters

This modal demonstrates a Shimmer Sweep effect combined with strict accessibility standards.

  • High contrast text and borders.
  • Large minimum touch targets (48x48px for the close button).
  • Clear, high-contrast :focus-visible rings for keyboard users.
  • Disables the shimmer animation entirely if prefers-reduced-motion is active.
  • Utilizes the CSS :target pseudo-class for state management without JavaScript.
