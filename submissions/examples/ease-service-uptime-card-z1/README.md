# Ease Service Uptime Card

## Overview
A CSS-only component for displaying service uptime statistics.

## Features
- Animated uptime circle
- Responsive layout
- Hover interactions
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first widget ideal for monitoring dashboards, cloud platforms, and admin panels.
