# Gravitational Pendulum

A pendulum animation with realistic swing motion.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--pendulum-color` | `#ff6b6b` | Color of the pendulum arm and bob |
| `--pivot-color` | `#ffd93d` | Color of the pivot point |
| `--swing-angle` | `30deg` | Maximum swing angle |
| `--swing-duration` | `2s` | Duration of one full swing cycle |
