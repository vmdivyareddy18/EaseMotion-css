# Animated Project Timeline Dashboard

A modern **Project Timeline Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. It showcases project progress, milestones, activity tracking, and animated visual components without using JavaScript.

## Features

* 📊 Project overview cards
* 📅 Animated vertical project timeline
* ✅ Completed, active, and upcoming milestones
* 📈 Animated progress bars
* ⭕ Circular project progress indicator
* 📉 CSS-only weekly progress chart
* 🔔 Recent activity feed
* ✨ Hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

## Included Animations

All animations follow the EaseMotion CSS naming convention.

* `ease-fadeUp`
* `ease-progressGrow`
* `ease-badgePulse`
* `ease-rotateIn`
* `ease-slideRight`
* `ease-pulse`
* `ease-barGrow`

## Customization

You can easily customize the design by editing the CSS variables in `:root`.

Example:

```css
:root{
    --primary:#4f46e5;
    --secondary:#10b981;
    --warning:#f59e0b;
    --surface:#ffffff;
    --bg:#f5f7fb;
}
```
