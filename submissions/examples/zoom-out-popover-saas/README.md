# Zoom-Out Popover for Modern SaaS Layouts


## What does this add?

A pure CSS animated zoom-out popover component designed for modern SaaS interfaces.

The component creates a smooth scale-up reveal animation without JavaScript.


## How to use it?


Add the wrapper structure:

```html
<div class="popover-wrapper">

<button class="trigger">
Open Features
</button>

<div class="popover">
Your content here
</div>

</div>
