# Ease Session Activity Log

## Overview
A CSS-only component for displaying recent user session activity.

## Features
- Animated status indicators
- Responsive layout
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component suitable for authentication dashboards, admin panels, analytics systems, and SaaS applications.
