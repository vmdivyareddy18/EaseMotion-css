# Glassmorphism Dashboard Card

A dashboard metric card with glassmorphism and an animated gradient border — pure CSS.

## Features
- Glassmorphism effect with backdrop-filter blur
- Rotating conic-gradient border animation
- Reusable card component for dashboards

## Usage
```html
<div class="card-glow">
  <div class="glass-card">
    <div class="card-header">
      <span class="label">Total Revenue</span>
      <span class="badge">+12.5%</span>
    </div>
    <div class="value">$24,560</div>
  </div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
