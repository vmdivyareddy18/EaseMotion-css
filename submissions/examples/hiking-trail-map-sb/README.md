# 🥾 Hiking Trail Map Component

An interactive, CSS-powered hiking trail map with animated waypoints, elevation markers, and a moving hiker.

## Features
- Animated trail drawing effect
- Pulsing waypoints with staggered delays
- Moving hiker along the trail
- Elevation markers
- Start/End markers
- Responsive design
- Uses EaseMotion CSS variables

## Usage
```html
<div class="ease-trail-map">
    <svg class="ease-trail-svg" viewBox="0 0 600 400">
        <!-- Trail path, waypoints, and hiker -->
    </svg>
</div>
