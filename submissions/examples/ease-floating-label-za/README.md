# Floating Label Input

## What does this do?
CSS-only floating input labels using :focus-within and :placeholder-shown pseudo-classes with color themes and sizes.

## How is it used?
```html
<div class="fl2-field"><input type="text" class="fl2-inp" placeholder=" " id="name"><label class="fl2-lbl" for="name">Label</label></div>
```

## Why is it useful?
Clean form UX without JavaScript. Labels float up on focus or when filled. Multiple themes, sizes, and icon support. Accessible and lightweight.
