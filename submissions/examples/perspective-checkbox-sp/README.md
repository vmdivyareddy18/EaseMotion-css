# Perspective Checkbox

A dark-mode inspired checkbox featuring a subtle 3D perspective flip animation built with pure HTML and CSS.

## Features

- 🌙 Dark mode UI
- 🎬 3D perspective animation
- ✅ Animated checkmark
- 💻 Pure HTML & CSS
- 📱 Responsive layout
- ♿ Accessible label structure
- 🚫 No JavaScript required

## File Structure

```
perspective-checkbox-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Hover over the checkbox to experience the perspective effect and click to toggle the animated checkmark.

## Use Cases

- Settings Panels
- Preferences
- Dashboard Controls
- Admin Interfaces
- Dark Theme Applications
