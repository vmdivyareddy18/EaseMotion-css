Build Pipeline Fix: Sourcemap Generation

This submission provides the architectural fix for the core Node.js build script to generate proper .map files alongside the minified CSS.

Current Pipeline (Regex)

build-minified-css.mjs

Strips comments via regex. Does NOT generate sourcemaps. Developers see "easemotion.min.css:1" in DevTools, making debugging mathematically impossible.

Proposed Pipeline (AST/Clean-CSS)

build-minified-css.mjs

Generates perfect easemotion.min.css.map files. Developers can instantly trace production styles back to their original component source files.
