# Project Milestone Health Timeline

## What does this do?

Shows project milestones with completed, on-track, at-risk, and delayed states.

## How is it used?

```html
<article class="mile risk">
  <span>Jul 03</span><h2>Beta</h2><p>At risk</p>
</article>
```

## Why is it useful?

It provides a planning dashboard pattern for milestone health and project timing.
