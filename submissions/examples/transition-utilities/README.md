# Transition Utilities

1. **What does this do?**  
   Adds readable transition utility ideas for common non-animation property changes such as transform, color, shadow, opacity, and spacing.

2. **How is it used?**  
   ```html
   <button class="transition-smooth hover-lift">Hover me</button>
   <button class="transition-colors color-swap">Change color</button>
   ```

3. **Why is it useful?**  
   It fits EaseMotion CSS by turning everyday interaction polish into plain-language classes that developers can understand without memorizing timing shorthand.
