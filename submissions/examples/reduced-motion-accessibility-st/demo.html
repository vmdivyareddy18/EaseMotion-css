Reduced Motion Accessibility Support

This demo implements the prefers-reduced-motion: reduce media query to globally disable animations and transitions for users with motion sensitivities.

Interactive Demo

The box below spins infinitely and scales on hover.

Hover Me

How to test this:

Windows: Settings > Accessibility > Visual effects > Turn off "Animation effects"

macOS: System Settings > Accessibility > Display > Turn on "Reduce motion"

Chrome DevTools: Open the Rendering tab (Ctrl+Shift+P > type "Rendering") and set "Emulate CSS media feature prefers-reduced-motion" to "reduce".

Once enabled, the spinning will stop instantly and the hover scale transition will snap instantly without animating.
