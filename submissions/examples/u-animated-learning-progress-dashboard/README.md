# Animated Learning Progress Dashboard

A modern **Learning Progress Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example provides a responsive interface for tracking course progress, completed lessons, learning streaks, achievements, and study analytics with smooth CSS-only animations.

## Features

* 📚 Total Courses overview
* ✅ Completed Lessons tracking
* ⏱️ Learning Hours statistics
* 🔥 Current Learning Streak
* 📖 Course progress indicators
* 🏅 Achievement badges
* 📊 CSS-only weekly learning chart
* 📈 Learning progress bars
* 📋 Learning activity feed
* ✨ Smooth hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries
