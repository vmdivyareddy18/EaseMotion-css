# Ease Comet Trail Buttons

## Overview

A premium button collection featuring an animated comet trail that sweeps across each button, creating a futuristic interaction using only HTML and CSS.

## Features

- Animated comet trail
- Glassmorphism buttons
- Hover lift effect
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

