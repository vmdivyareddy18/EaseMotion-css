# ease-open-close-toggle

## What this is
A CSS-only open/close toggle with a rotating handle animation.

## Why
It gives a simple open/close control a more expressive motion style while staying lightweight and easy to theme.

## How to test
1. Open `demo.html` in a browser.
2. Click the toggle to switch between open and closed.
3. Confirm the handle rotates smoothly when the state changes.
4. Resize the window to check the responsive layout.
5. Verify the component still looks correct with custom property changes.

## Note
This is a visual demo component. The toggle uses only HTML and CSS, and the animation loops are not needed.
