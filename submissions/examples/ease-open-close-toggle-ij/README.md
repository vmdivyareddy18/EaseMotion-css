# Open/Close Toggle

A circular toggle button with rotating handle animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--toggle-bg` | `#2d2d44` | Border color (closed) |
| `--handle-color` | `#888` | Handle line color |
| `--open-color` | `#4ecdc4` | Active/border color |
| `--rotate-duration` | `0.3s` | Rotation animation speed |
