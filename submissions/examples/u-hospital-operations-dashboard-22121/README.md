# Hospital Operations Dashboard

A responsive **Hospital Operations Dashboard** built using **HTML5** and **CSS3** that demonstrates hospital management, patient monitoring, doctor availability, appointment scheduling, and emergency operations through a modern interactive dashboard.

---

## Features

- Responsive dashboard layout
- Hospital overview cards
- Bed occupancy tracker
- Department status panel
- Appointment progress monitoring
- Recent admissions feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
