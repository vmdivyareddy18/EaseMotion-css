# Testimonial Cards Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build responsive testimonial cards.

---

## Components

- Responsive testimonial section
- Customer avatar
- Customer name
- Designation
- Five-star rating
- Review text

---

## Folder Structure

```
submissions/examples/testimonial-cards-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to test responsiveness.
3. Observe the applied EaseMotion CSS animations.

---

## Features

- Responsive card layout
- Three testimonial cards
- Modern UI
- Uses existing EaseMotion CSS animation classes
