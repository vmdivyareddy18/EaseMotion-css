# Staggered List Animation Component

A cascading list item entrance component utilizing index-based animation delays.

---

## What is Included

1. **`demo.html`**: A notifications feed card holding reload actions, list elements, and custom CSS stagger indices.
2. **`style.css`**: Transition delay formulas, slide-up keyframes, and user layout styles.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the notification feed items fading and sliding up in a cascading sequence.
3. Click the **Reload** button to re-trigger the staggered animation sequence.
