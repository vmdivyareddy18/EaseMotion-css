# Clip-Path Shapes

Demonstrates CSS `clip-path` capabilities including circle, ellipse, polygon, and inset shapes with hover transitions, text clipping via `background-clip: text`, and image clipping.

## Features

- Four basic clip-path shapes with live hover transitions
- Text clipping using gradient backgrounds clipped to text glyphs
- Image gallery with distinct clip-path geometries
- Accessible reduced-motion support

## Usage

Open `demo.html` in any modern browser. Hover over shape cards and images to see smooth clip-path transitions. No build tools or external dependencies required.
