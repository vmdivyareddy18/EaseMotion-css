# Animated Notification Panel

## Overview

A modern notification panel with smooth sliding animations, colorful notification cards, icons, and interactive hover effects.

## Features

- Sliding entrance animation
- Smooth hover interactions
- Multiple notification types
- Responsive card layout
- Modern dashboard-style design
- Pure HTML and CSS implementation

## Files

- demo.html - Displays the notification panel
- style.css - Contains styling and animation effects
- README.md - Component documentation

## Usage

Open demo.html in a browser to view the animated notification panel.

Hover over each notification card to experience the interaction effects.
