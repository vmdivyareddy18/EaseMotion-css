# Font Language Override

font-language-override — Uses language-specific glyph variants

## Features
- Uses `--ease-color-text`
- Uses `--ease-text-base`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Font Language Override</span>
</div>
```

## Why is it useful?

This submission demonstrates the font language override CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
