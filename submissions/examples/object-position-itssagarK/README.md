# Object Position Utilities

1. What does this do? Adds object-position utility classes (`.object-center`, `.object-top`, `.object-bottom`, `.object-left`, `.object-right`, `.object-left-top`, etc.) to control placement of replaced element contents (images or videos) within their containers.
2. How is it used? Apply `.object-top` or `.object-right-bottom` alongside `.object-cover` to align the crop region.
3. Why is it useful? It gives precise layout control over image focal points and cropping without changing the image source.
