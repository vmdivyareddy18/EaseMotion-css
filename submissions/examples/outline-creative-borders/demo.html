Creative Borders with outline

Using outline + outline-offset for effects that border cannot achieve.

Offset Outline

Outline sits outside the element box, creating a gap between edge and border.

outline-offset: 6px

Double Ring

A border inside + an outline outside produces two concentric rings.

border + outline

Dashed Outline

Outline supports all line styles — dash, dot, double — just like border.

outline-style: dashed

Animated Width

The outline-width animates outward in a pulsing ring.

animation on outline-width

Inset Offset

Negative outline-offset pulls the outline inward, inside the element.

outline-offset: -8px

Outline vs Border

Outline does not affect layout — it overlays. Border takes space inside the box.

box-sizing difference
