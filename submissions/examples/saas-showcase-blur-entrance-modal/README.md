# SaaS Showcase Blur-Entrance Modal

A modern **Pure CSS Blur-Entrance Modal** designed for **SaaS Showcase** interfaces. The component features a smooth blur entrance animation, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Blur-Entrance animation
- Glassmorphism inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:0.8s;
    --easing:ease;
    --blur:12px;

    --primary:#2563eb;
    --secondary:#60a5fa;
}
```

These variables allow you to customize:

- Animation duration
- Animation easing
- Blur intensity
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
