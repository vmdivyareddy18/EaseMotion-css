# Sticky Notes Board

## What does this do?
A digital sticky notes board with add/delete notes, editable text, random color assignment, auto-save to localStorage, and a note count.

## How is it used?
Open demo.html in a browser. Click "Add Note" to create a new colored sticky note. Type directly into the note body to auto-save. Click the &times; button to delete a note.

## Why is this useful?
Demonstrates a flexible note-taking UI with grid layout, editable content, persistent storage, and color-coded visual distinction.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript (localStorage)

## Preview
