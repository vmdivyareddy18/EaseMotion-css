# Animated Wildlife Conservation Dashboard

A modern dashboard for tracking wildlife populations, habitat health, conservation goals, and environmental monitoring activities.

## Features

- Species population cards
- Habitat monitoring indicators
- Conservation goal progress bars
- Research activity feed
- Responsive layout
- Smooth animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Wildlife conservation portals
- Environmental dashboards
- Biodiversity monitoring systems
- Research organizations
- Sustainability projects
