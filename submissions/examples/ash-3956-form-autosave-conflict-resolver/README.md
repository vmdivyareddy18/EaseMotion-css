# Form Autosave Conflict Resolver

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive conflict resolution component that helps users resolve autosave conflicts between local drafts and server versions in forms and editors.

## Features

- Local vs server comparison cards
- Conflict resolution actions
- Responsive layout
- Glassmorphism design
- Hover animations
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="conflict">

    <div class="version local">
        ...
    </div>

    <div class="divider">
        VS
    </div>

    <div class="version remote">
        ...
    </div>

</div>
```
