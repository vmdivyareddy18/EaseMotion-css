# ease-hover-icon-right

A subtle hover animation where an icon inside a link or button nudges to the right, creating a directional call-to-action effect.

---

## ✨ Features

- Pure CSS implementation
- Smooth 0.25s animation
- Works with links and buttons
- Great for "Read more →" UI patterns
- Lightweight and reusable

---

## 🚀 Usage

```html
<a href="#" class="ease-hover-icon-right">
    Read More <span class="icon">→</span>
</a>
```

---

## 🎛 CSS Variables

```css
:root {
    --ease-hover-icon-right-distance: 4px;
    --ease-hover-icon-right-duration: 0.25s;
}
```
