# Chat Emoji Reaction

Emoji reaction picker with pop-in animation. Click an emoji to send it with a bouncing pop-in effect over a chat message bubble.

## Features

- Emoji reaction buttons with pop-in animation
- Animated emoji flying from picker to message
- Customizable emoji size, pop duration, and reaction background
- Smooth CSS transitions and @keyframes pop effect
- Chat-style message UI

## Usage

Set CSS vars on `.er-picker`:
- `--emoji-size`: emoji font size (default `1.8rem`)
- `--pop-duration`: pop animation duration (default `0.4s`)
- `--reaction-bg`: reaction badge background (default `#1e293b`)
