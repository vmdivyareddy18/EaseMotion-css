# Ease Quicksand Loader

Dots sinking like quicksand, creating a playful loading indicator.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.quicksand` element.

## Custom CSS Variables

| Variable          | Default   | Description                     |
|-------------------|-----------|---------------------------------|
| `--sand-color`    | `#d4a574` | Color of the sinking dots       |
| `--dot-size`      | `12px`    | Diameter of each dot            |
| `--sink-duration` | `1.2s`    | Duration of one sink cycle      |

## Interactive Controls

Open `demo.html` to use the color picker and range sliders for live customization.
