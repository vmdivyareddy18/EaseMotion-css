# Ease Scroll Snap Carousel

## Overview

This example demonstrates a reusable CSS-only horizontal carousel using CSS Scroll Snap for smooth navigation across cards.

## Features

- Pure HTML and CSS
- CSS Scroll Snap
- Touch-friendly scrolling
- Responsive layout
- Lightweight utility
- No JavaScript required

## Folder Structure

```text
ease-scroll-snap-carousel-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

1. Open the project folder.
2. Launch `demo.html`.
3. Add or remove cards as needed.

## Browser Support

Works in modern browsers supporting CSS Scroll Snap.

## Why This Example

Scroll Snap provides an elegant, accessible way to build horizontal content sliders without introducing JavaScript dependencies.
