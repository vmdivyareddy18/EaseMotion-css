Caret Color Utilities

CSS caret-color utilities to customize the color of the text insertion cursor. Improve focus tracking, error reporting, and visual branding in form fields.

What does caret-color do?
The caret-color property controls the color of the text insertion cursor (caret) inside editable elements like <input> and <textarea>. Adjusting this color creates high-contrast cursors matching your form states.

Instructions: Click into each field below and type text to view the insertion cursor color.

Input Caret Demos

Default Cursor .cc-auto

Uses the browser's default caret color (typically white or light gray in dark themes).

Primary Brand Focus .cc-primary

Applies a blue insertion caret to match secondary links and primary action states.

Success / Validated Input .cc-success

Applies a green cursor. Excellent for validated fields or checkout input success.

Textarea Caret Demos

Danger / Validation Error .cc-danger

Applies a red caret, immediately drawing attention to fields with validation errors or issues.

Warning / Alert State .cc-warning

Applies a yellow caret cursor, indicating a pending validation check or warnings.
