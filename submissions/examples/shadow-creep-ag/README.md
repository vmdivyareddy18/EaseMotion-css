# Shadow Creep

## What does this do?
A dark shadow creeping/spreading effect where a shadow gradually expands from below an element, creating a sense of looming presence.

## How is it used?
Apply the `.shadow-creep` class to an element. The box-shadow gradually expands and darkens to create a creeping shadow effect.

## Why is this useful?
Adds atmospheric depth to UI elements for storytelling, game interfaces, or dramatic hover effects and loading states.
