# Loading Skeleton List UI Example

## 1. What does this do?

This submission adds skeleton-loading cards with staggered shimmer animation to simulate content loading states. It uses pure CSS to create placeholder layouts and animated shimmer effects.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes a list of loading cards and shimmering placeholders. Copy the `.skeleton-list` markup and CSS into your project for skeleton screen loading patterns.

---

## 3. Why is it useful?

Skeleton loaders improve perceived performance by showing an animated placeholder while real content loads. They reduce layout shift and make loading states feel smoother and more engaging.
