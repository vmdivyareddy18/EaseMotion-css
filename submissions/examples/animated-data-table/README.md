# Animated Data Table

A clean, modern, and fully responsive **Data Table Component** with sortable columns, search functionality, and smooth animations. This example demonstrates how to combine **EaseMotion CSS** for staggered row entrances, animated sort indicators, and interactive hover effects to create a polished data management interface.

## 🚀 Features

- **Staggered Row Entrances**: Each table row fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a cascading load effect.
- **Animated Sort Indicators**: Sortable column headers feature animated arrows that rotate and highlight when active.
- **Interactive Row Hover**: Rows smoothly change background color on hover for better readability.
- **Search Box Animation**: Search input with smooth focus states and icon positioning.
- **Action Buttons**: Edit and delete buttons with lift effects on hover.
- **Status & Role Badges**: Color-coded badges for user status and roles with distinct visual styles.
- **Responsive Design**: Horizontal scrolling on mobile with adjusted header layout.
- **Mini Pagination**: Compact pagination controls in the footer.
