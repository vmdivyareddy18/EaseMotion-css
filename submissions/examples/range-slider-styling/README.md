# Range Slider Styling

Custom range input styling with gradient track fill, multiple color themes, and orientation support.

## Features

- Custom track and thumb styling via `-webkit-appearance` / `-moz-appearance`
- Gradient track fill that follows the slider value
- Four color themes: blue, green, purple, orange
- Horizontal and vertical orientation support
- Live value display with tabular-nums
- Focus-visible ring for keyboard accessibility
- `prefers-reduced-motion` support

## Files

- `demo.html` — Interactive demo with all themes and orientations
- `style.css` — Self-contained CSS (192 lines)
