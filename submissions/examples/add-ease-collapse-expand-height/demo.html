EaseMotion CSS

Smooth Height Auto Animation Utility

Interactive FAQ Accordion

This utility uses the CSS Grid grid-template-rows transition technique. By transitioning the grid rows from 0fr to 1fr, the browser handles the layout engine calculation dynamically, allowing content to expand smoothly to its exact auto height without layout thrashing.
Traditional transitions using max-height require setting a large arbitrary value (e.g. max-height: 1000px) to avoid clipping. This results in animation lag, delay, or mismatched timing when opening/closing elements with small content heights. The CSS Grid technique transitions directly to the content's exact size.
No size calculations in JavaScript are needed! The browser takes care of calculations natively in CSS. JS is only used here to toggle the class names (which can also be done using pure CSS techniques like checkbox hacks or target pseudo-classes).
