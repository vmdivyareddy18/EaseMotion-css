# Rotate Click Tooltip

Responsive tooltip component designed for Product Catalog interfaces.

---

## Features

- Pure CSS implementation
- Rotate-Click animation
- Product-focused styling
- Responsive layout
- Keyboard accessibility
- Focus support
- CSS variables
- Zero JavaScript dependency
