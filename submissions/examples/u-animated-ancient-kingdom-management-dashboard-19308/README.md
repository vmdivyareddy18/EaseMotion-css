# Animated Ancient Kingdom Management Dashboard

A medieval-themed dashboard for tracking kingdom resources, military strength, construction projects, and expansion progress.

## Features

- Kingdom resource cards
- Treasury and population monitoring
- Construction progress tracking
- Military status indicators
- Activity feed
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Strategy game dashboards
- Historical simulation interfaces
- Resource management systems
- Educational history projects
- Medieval-themed applications
