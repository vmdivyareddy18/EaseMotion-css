# Animated Input Field UI Example

## 1. What does this do?

This submission demonstrates animated input fields with floating labels, focus border highlights, and optional icon variants. It uses CSS `:placeholder-shown`, `:focus`, and `@keyframes` to create a polished, modern form input experience.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes `style.css` for visual styling and uses pure HTML/CSS for the floating label and focus states. Copy the `.animated-input` markup and CSS into your project to reuse the input wrapper and variants.

---

## 3. Why is it useful?

Animated input fields improve form usability by keeping labels visible and clearly showing focus state. This pattern reduces placeholder ambiguity, provides better context, and feels more interactive while remaining accessible and lightweight.
