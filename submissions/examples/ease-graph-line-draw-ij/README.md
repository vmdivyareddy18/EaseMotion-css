# Graph Line Draw (#36693)

Graph line draw with animated stroke in a dark-themed card layout.

## CSS Custom Properties

| Property          | Default     | Description                     |
|-------------------|-------------|---------------------------------|
| `--graph-bg`      | `#0f0f1f`   | Background color of the graph   |
| `--line-color`    | `#4ecdc4`   | Color of the drawn line         |
| `--line-width`    | `3px`       | Stroke width of the line        |
| `--draw-duration` | `2s`        | Duration of the draw animation  |

## Interactive Controls

- Color picker for graph background
- Color picker for line color
- Range slider for line width
- Range slider for draw speed
- Replay button to restart animation
