# Animated Calendar Reminder Bounce

## Overview

A reusable calendar icon animation featuring a gentle bounce effect suitable for reminders, events, bookings, schedules, productivity dashboards, and task management systems.

## Features

- CSS-only animation
- Calendar reminder bounce effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="calendar-button">
  <span class="calendar-icon">📅</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
