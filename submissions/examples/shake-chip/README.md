# Shake Chip

A responsive **Shake Chip** component built with pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Cyberpunk-inspired design
- Responsive layout
- Accessible markup
- Hover shake animation
- Uses EaseMotion-style variables and `ease-*` animation class

## Folder Structure

```
shake-chip-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3

## Animation

- `ease-shake`
- Pure CSS keyframes
- Smooth hover interaction
