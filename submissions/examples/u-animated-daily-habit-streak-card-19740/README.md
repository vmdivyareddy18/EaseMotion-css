# Animated Daily Habit Streak Card

A modern habit-tracking component built using HTML and CSS.

## Features

- Animated streak counter
- Weekly progress indicator
- Habit completion list
- Motivation message
- Achievement badge
- Responsive design

Open `demo.html` in any modern browser.
