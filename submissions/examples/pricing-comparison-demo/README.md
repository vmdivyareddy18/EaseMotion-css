# Pricing Comparison Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive pricing comparison table.

---

## Components

- Pricing Table
- Feature Comparison
- Three Pricing Plans
- Action Buttons

---

## Folder Structure

```
submissions/examples/pricing-comparison-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Resize the browser window.
3. Observe the EaseMotion CSS animations applied to the pricing comparison table.

---

## Features

- Responsive comparison table
- Three pricing plans
- Feature comparison
- Call-to-action buttons
- Uses existing EaseMotion CSS animation classes
