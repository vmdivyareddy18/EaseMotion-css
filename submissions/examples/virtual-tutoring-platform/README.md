# 🎓 Virtual Tutoring Platform Demo

A modern, responsive landing page for a **Virtual Tutoring Platform** built using **EaseMotion CSS** animation utility classes. This example showcases how EaseMotion can be used to create an engaging educational website with smooth entrance animations and interactive hover effects.

---

## 📁 Folder Structure

```
submissions/
└── examples/
    └── virtual-tutoring-platform/
        ├── demo.html
        ├── style.css
        └── README.md
```

---

## ✨ Features

- 📚 Hero section with tutor session mockup
- 👨‍🏫 One-to-one tutoring features
- 🗂 Subject categories
- ⭐ Tutor profile preview cards
- 💰 Pricing plans
- 💬 Student testimonials
- 🚀 Get Started call-to-action
- 📱 Fully responsive design

---

## 🎬 EaseMotion Classes Used

- `ease-fade-up`
- `ease-fade-in`
- `ease-slide-up`
- `ease-zoom-in`
- `ease-hover-grow`
- `ease-hover-glow`
- `ease-hover-lift`
- `ease-hover-shadow`

---

## ▶️ How to Run

1. Open the project folder.
2. Open `demo.html` in your browser.
3. No installation or build tools are required.
