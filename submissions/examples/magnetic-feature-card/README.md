# Magnetic Feature Card

## What does this do?
A layered feature card with hover lift, animated preview tiles, and a magnetic call-to-action.

## How is it used?
```html
<article class="feature-card">
  <span class="badge">Interactive</span>
  <h2>Magnetic Feature Card</h2>
  <a class="magnetic-link" href="#">Explore pattern</a>
</article>
```

## Why is it useful?
It gives EaseMotion CSS a polished marketing or dashboard card pattern where motion reinforces the main action without JavaScript.
