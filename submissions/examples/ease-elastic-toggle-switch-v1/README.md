# Ease Elastic Toggle Switch

## Overview

Ease Elastic Toggle Switch is a reusable micro-interaction component featuring an elastic thumb movement, glowing track, and smooth spring animation using only HTML and CSS.

---

## Features

- 🎛 Elastic toggle motion
- ✨ Glow effect
- 🌈 Three color variants
- ⚡ Pure HTML & CSS
- 📱 Responsive demo
- 💎 Modern UI interaction

```html
<label class="toggle">
```
