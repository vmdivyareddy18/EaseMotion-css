# Overflow Wrap Utilities

1. What does this do? Adds overflow-wrap utility classes (`.wrap-normal`, `.wrap-break`, `.wrap-anywhere`) to control wrapping behavior of long, unbreakable words and URLs.
2. How is it used? Apply `.wrap-break` or `.wrap-anywhere` to containers handling dynamic text that might overflow.
3. Why is it useful? It prevents layout breakage and clipping caused by extremely long words or links in responsive layouts.
