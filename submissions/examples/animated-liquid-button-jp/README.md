# Animated Liquid Button Component

A futuristic CSS-only liquid button featuring flowing wave animations, glowing gradients, and smooth hover interactions.

---

## Features

- Animated liquid wave effect
- Smooth floating hover animation
- Multiple colorful themes
- Glowing button shadows
- Responsive design
- Pure HTML and CSS implementation
- No JavaScript required

---

## Included Variants

- 🔵 Blue Liquid Button
- 🌸 Pink Liquid Button
- 🟢 Green Liquid Button

---

## Files Included

- `demo.html` — Demonstrates liquid button styles.
- `style.css` — Contains animations, gradients, and responsive styles.
