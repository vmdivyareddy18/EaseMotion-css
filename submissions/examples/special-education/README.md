# Inclusive Education Website

This demo site showcases an inclusive, supportive, and empowering education layout using **EaseMotion CSS classes** only.

## 🧩 Sections
- Hero with inclusive imagery
- Resources by learning difference (dyslexia, ADHD, autism)
- Programs for teachers and parents
- Specialist profiles
- Family testimonials
- Free resources download
- Newsletter signup
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — entrance fade animations
- `ease-slide` — sliding entrance for headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs and specialist cards
- `ease-hover-opacity-reduce` — hover dimming for resource cards and testimonials

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll through sections to see entrance animations.
3. Hover over buttons, cards, and testimonials to see interactive effects.
4. Replace placeholder images and text with real inclusive education content.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion animation classes
