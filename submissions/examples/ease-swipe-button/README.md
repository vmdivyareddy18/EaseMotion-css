# iOS-style Swipe Button

A production-ready "Swipe to Confirm" component built with vanilla HTML, CSS, and JavaScript. Perfect for confirmation dialogs, form submissions, payments, or any action requiring intentional user confirmation.

**Live Preview:** Open `index.html` in your browser.

---

## ✨ Features

- ✅ **Drag-to-confirm** – iOS-style swipe interaction
- ✅ **Smooth progress** – Visual feedback while dragging
- ✅ **Success state** – Confirmed state with animation
- ✅ **Touch & mouse** – Works on all devices
- ✅ **Keyboard accessible** – Press Enter/Space to confirm
- ✅ **Fully responsive** – Adapts to any screen size
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Customizable** – CSS variables for easy theming
- ✅ **RTL-ready** – Supports right-to-left layouts
- ✅ **Auto-reset** – Resets after confirmation (configurable)
