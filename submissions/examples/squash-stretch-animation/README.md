# Squash Stretch Animation Utility

A lightweight squash-and-stretch animation utility built using pure CSS.

## Features

- Pure CSS squash animation
- Elastic stretch interaction
- Lightweight and reusable
- Classic animation principle
- No JavaScript required

## Usage

```html
<button class="squash-btn">
  Click Me
</button>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Smooth interactive feedback
