# Elastic Bounce Accessible Web Modal

A responsive accessibility-focused modal featuring a smooth Elastic Bounce entrance animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- High-contrast interface
- Large typography and controls
- Visible focus indicators
- Elastic Bounce animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--scale
--primary
--secondary
--bg
--surface
--text
--radius
```
