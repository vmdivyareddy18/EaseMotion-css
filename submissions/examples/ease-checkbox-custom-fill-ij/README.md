# Checkbox Custom Fill

Custom styled checkboxes with an animated bottom-to-top fill effect.

## Features

- Fill color slides up via `clip-path` on `:checked`
- Checkmark fades in after fill completes
- Hover state highlights the row
- `:focus-visible` outline for keyboard accessibility

## Custom Properties

| Property              | Default  | Description              |
|-----------------------|----------|--------------------------|
| `--cb-duration`       | `0.35s`  | Fill animation duration  |
| `--cb-checked-color`  | `#22c55e`| Fill color when checked  |
| `--cb-unchecked-bg`   | `#fff`   | Background when unchecked|
| `--cb-border-color`   | `#cbd5e1`| Default border color     |
| `--cb-checkmark-color`| `#fff`   | Checkmark color          |
