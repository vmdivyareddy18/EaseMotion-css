# Shatter Effect – Element shatters into pieces

> A production-ready shatter effect component that breaks an element into flying pieces with customizable count, force, duration, colors, and gravity physics. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Element shattering** – Breaks into flying pieces
- ✅ **Physics simulation** – Gravity, velocity, friction
- ✅ **Customizable** – Pieces, force, duration
- ✅ **Color control** – Primary color for shards
- ✅ **Gravity toggle** – Enable/disable gravity
- ✅ **Particle effects** – Glowing shards with rotation
- ✅ **Keyboard shortcuts** – Space (shatter/reset), R (reset), G (gravity)
- ✅ **4 color variants** – Ocean, Sunset, Gold, Dream
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
