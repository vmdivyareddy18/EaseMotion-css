# Page Loader Progress

A top-bar style loading progress indicator.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--loader-bg` | `#2d2d44` | Track background |
| `--fill-color` | `#4ecdc4` | Progress fill color |
| `--track-height` | `8px` | Bar height |
| `--fill-duration` | `2s` | Fill animation speed |
