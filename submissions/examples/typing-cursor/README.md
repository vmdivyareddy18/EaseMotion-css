# Typing Cursor Effect

A lightweight CSS-only typing cursor animation for EaseMotion.

## Features

- Pure CSS
- Blinking caret effect
- Works with headings
- Works with inline text
- Customizable through CSS variables
- No JavaScript required

## Usage

```html
<span class="em-cursor"></span>
```

### Small Variant

```html
<span class="em-cursor em-cursor-sm"></span>
```

## Customization

```css
:root {
  --em-cursor-color: currentColor;
  --em-cursor-width: 2px;
  --em-cursor-height: 1em;
  --em-cursor-duration: 0.8s;
}
```
