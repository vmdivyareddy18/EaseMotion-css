# Spotlight Carousel

A pure CSS spotlight carousel component for EaseMotion CSS.

## Features

- Spotlight focus animation
- Smooth scaling effect
- CSS-only animation
- Responsive design
- No JavaScript required


## Usage

Include the HTML structure and import `style.css`.

Example:

```html
<section class="spotlight-carousel">
    <div class="carousel-track">
        <article class="card active">
            Content
        </article>
    </div>
</section>
