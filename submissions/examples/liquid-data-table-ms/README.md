# Liquid Data Table

Resolves #42427.

### What does this do?
Adds a Liquid Data Table component to EaseMotion CSS suited for News Portals or live data tracking. It uses the `ease-blob` animation hidden within the candidate cell (`overflow: hidden`). When a user hovers over a row, the blob slides in from the left, acting as an organic, liquid-like highlight for that specific candidate.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Hover over the table rows to see the liquid highlight effect slide in behind the candidate name.

### Why is it useful?
Standard table hover states can be boring. This liquid highlight provides a dynamic, fluid interaction that makes exploring dense data (like election results or sports stats) much more engaging for the user.
