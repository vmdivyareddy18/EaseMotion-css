# Error Boundary Fallback

Interactive error boundary fallback card with an animated broken-circle illustration.

## CSS Custom Properties

| Variable | Default | Description |
|---|---|---|
| `--error-bg` | `#1a1a2e` | Card background |
| `--error-color` | `#e94560` | Primary error accent |
| `--icon-color` | `#ff6b6b` | Broken-circle icon tint |
| `--shake-duration` | `0.5s` | Shake/slide animation duration |

## Interaction

- **Try Again** — triggers a shake animation on the illustration
- **Dismiss** — toggles visibility of the error detail trace
