# Shimmer Sweep Popover

A Marketing Landing Page inspired popover with a smooth shimmer sweep animation built entirely with HTML and CSS.

## Features

- ✨ Shimmer sweep animation
- 🎯 Marketing landing page design
- 📱 Responsive layout
- ♿ Keyboard accessible (`:focus-within`)
- 🎨 Customizable using CSS custom properties
- 🚫 No JavaScript required

## CSS Variables

```css
--accent
--secondary
--duration
--easing
--scale
```

## File Structure

```
shimmer-sweep-popover-marketing-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser and hover over the CTA button to reveal the shimmer popover.
