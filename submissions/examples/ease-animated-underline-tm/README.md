# ease-animated-underline

Animated underline wipe effects for EaseMotion CSS.

## Variants

- `.underline-center` - expands from center outward
- `.underline-left` - slides in from left
- `.underline-right` - slides in from right
- `.underline-double` - two parallel lines
- `.underline-up` - grows upward
- `.underline-gradient` - gradient sweep

All use a `::after` pseudo-element with width animation.
