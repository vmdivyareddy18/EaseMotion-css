# Design System Landing Page

## What does this do?

It provides a responsive landing page for a design system and component library with token previews, component cards, and library metrics.

## How is it used?

```html
<section class="hero">
  <h1>
    Ship consistent interfaces with tokens, patterns, and accessible components.
  </h1>
  <a class="button" href="#components">Explore components</a>
</section>
```

## Why is it useful?

It demonstrates a complete component-library marketing page with accessible structure and CSS-only motion.
