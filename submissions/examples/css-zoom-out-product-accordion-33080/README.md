# CSS Zoom-Out Accordion for Product Catalog Layouts

## Overview

A modern Product Catalog accordion built entirely with HTML and CSS. The component features a subtle zoom-out animation when expanding sections, making product information easy to browse while maintaining a clean and responsive layout.

---

## Features

- Pure HTML5 & CSS3
- Responsive product catalog layout
- CSS-only zoom-out animation
- Native `<details>` and `<summary>` accordion
- Keyboard accessible
- Smooth expand/collapse transitions
- Rotating toggle icon
- CSS custom properties
- No JavaScript required
