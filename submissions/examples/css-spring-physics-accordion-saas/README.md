# CSS Spring Physics Accordion for SaaS Showcase Layouts

A **pure CSS Spring Physics Accordion** designed for modern **SaaS showcase interfaces**. It delivers smooth spring-inspired expand/collapse animations using only HTML and CSS—no JavaScript required.

## Features

- 🚀 Pure HTML & CSS
- 🎯 Spring-inspired accordion animation
- 💻 SaaS-style modern interface
- 📱 Fully responsive layout
- ♿ Keyboard accessible using native `<details>` and `<summary>`
- 🎨 Customizable with CSS Custom Properties
- ⚡ Lightweight and JavaScript-free

## Folder Structure

```text
css-spring-physics-accordion-saas/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern browser.
2. Click any accordion section to expand or collapse it.
3. Customize colors, spacing, animation timing, and other properties through the CSS variables in `style.css`.

## Customization

Modify the variables in the `:root` selector:

```css
:root {
  --primary: #4f46e5;
  --secondary: #7c3aed;
  --radius: 18px;
  --duration: .45s;
  --easing: cubic-bezier(.22,1,.36,1);
}
```
