# Ease Hover Scale Icon

This demo implements a **hover-triggered icon scale animation** where the child icon scales up when the parent element is hovered.

## 🧠 How It Works
- The `.ease-icon` scales to `1.2` when its parent `.button` is hovered.
- The transition duration is `0.25s` for smooth animation.
- Ideal for icon buttons or interactive UI elements.

## 🧩 Files
- `demo.html` — main demo with button and icon.
- `style.css` — styling and hover animation logic.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Hover over the button to see the icon scale up.
3. Adjust scale or transition duration in `style.css` as needed.

## 🧰 Technologies
- HTML5
- CSS3 (transitions, hover effects)
