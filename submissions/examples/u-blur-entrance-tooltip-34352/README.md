# Blur Entrance Tooltip

Modern tooltip component built for SaaS showcase interfaces.

Inspired by contemporary SaaS landing pages and product presentation sections.

---

## Features

- Pure CSS implementation
- Blur Entrance animation
- Glassmorphism styling
- Responsive layout
- Keyboard accessible
- Focus-visible support
- Customizable CSS variables
- No JavaScript required

---

## CSS Variables

```css
--duration
--ease
--scale
--blur
--radius
--primary
--secondary
--glass
```
