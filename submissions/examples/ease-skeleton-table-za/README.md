# CSS Skeleton Table

## What does this do?
Loading skeleton patterns for data tables, list items, and card grids with wave and pulse animations.

## How is it used?
```html
<div class="sk-table"><div class="sk-thead">...</div><div class="sk-tbody"><div class="sk-tr"><div class="sk-td"><div class="sk-cell sk-wave"></div></div></div></div></div>
```

## Why is it useful?
Provides loading states for table-heavy UIs without JavaScript. Wave and pulse animation variants. Table, list, and card grid patterns. Customizable via CSS variables.
