# Animated Project Roadmap Timeline

A responsive roadmap timeline showcasing project milestones, phases, and progress states.

## Features

- Vertical roadmap layout
- Completed, active, and upcoming milestones
- Timeline indicators
- Hover animations
- Responsive design
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more milestones
- Change status colors
- Modify timeline spacing
- Adjust animation effects
