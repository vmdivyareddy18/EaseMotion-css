# Micro Shake Cyberpunk Neon Modal

A futuristic cyberpunk-inspired modal built entirely with HTML and CSS featuring a subtle Micro Shake interaction and glowing neon interface.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Cyberpunk neon styling
- Micro Shake animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--shake
--bg
--card
--cyan
--pink
--radius
```
