# Animated Toast Notification

A standalone animated toast notification component for EaseMotion CSS.

## Features

- Success toast
- Error toast
- Warning toast
- Info toast
- Auto dismiss using `data-ease-duration`
- Manual close button
- Toast stacking
- Five position variants
    - top-right
    - top-left
    - top-center
    - bottom-right
    - bottom-left
- Entrance animation
- Exit fade animation
- Supports `prefers-reduced-motion`
