# ease-card-event

A reusable event listing card built with pure HTML and CSS. It features a prominent date badge, event details, and a registration call-to-action.

---

## ✨ Features

- Prominent date badge (day + month)
- Event title
- Location
- Time
- Register / RSVP button
- Hover lift animation
- Pure HTML & CSS
- Responsive layout

---

## 🚀 Usage

```html
<article class="ease-card-event">
    ...
</article>
```

---

## ✅ Acceptance Criteria

- Date badge with day and month
- Event title
- Event location
- Event time
- Register button
- Hover lift animation
- CSS-only implementation
