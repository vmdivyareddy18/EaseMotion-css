How it works: overflow: hidden + white-space: nowrap clips text. steps(30, end) creates 30 discrete width jumps — each jump reveals roughly one character. The border-right acts as the blinking cursor using a separate ease-kf-cursor-blink animation.

Character count tip: The default steps(30) works for text up to ~30 characters. For longer text, the effect still works but each "step" reveals more than one character.

prefers-reduced-motion: Full text shown instantly — width: 100%, no animation, no cursor.
