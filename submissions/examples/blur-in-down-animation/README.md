# Blur In Down Animation

A lightweight blur-in descending animation utility built using pure CSS.

## Features

- Smooth downward entrance motion
- Blur-to-clear transition
- Opacity fade-in effect
- Lightweight CSS animation
- Pure CSS implementation

## Usage

```html
<div class="blur-box">
  BLUR IN
</div>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable entrance animation
- Pure CSS motion effect
