# Animation Card Gallery

## Purpose

This demo showcases several animation styles together in a single responsive gallery.

---

## Included Cards

- Fade
- Slide
- Grow
- Rotate
- Bounce
- Scale

---

## Folder Structure

```
animation-card-gallery/
│
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Watch the animations play.
3. Resize the browser to see the responsive grid.

---

## Benefits

- Compare multiple animation styles in one place.
- Easy for beginners to explore.
- Responsive layout for desktop and mobile.
