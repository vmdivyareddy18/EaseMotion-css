# Approval Request Decision Bar

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern approval workflow component featuring request information, priority indicators, approval actions, and status tracking.

## Features

- Approval request banner
- Approve and reject actions
- Priority badge
- Animated notification pulse
- Request details section
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="approval-bar">
  Approval Request
</div>
```

## Main Classes

```css
.approval-bar
.approval-info
.priority
.approval-actions
.approve-btn
.reject-btn
```
