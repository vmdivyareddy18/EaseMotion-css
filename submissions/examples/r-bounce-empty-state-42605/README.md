# Bounce Empty State

A responsive **Bounce Empty State** component built with pure CSS for the **EaseMotion CSS** library. Inspired by modern **Startup Design**, it provides a friendly empty-state interface with a smooth bounce entrance animation and floating illustration.

---

## ✨ Features

- Pure CSS implementation
- Bounce entrance animation
- Floating illustration
- Startup-inspired design
- Responsive layout
- Keyboard accessible
- No JavaScript required
- Easy customization using CSS variables
