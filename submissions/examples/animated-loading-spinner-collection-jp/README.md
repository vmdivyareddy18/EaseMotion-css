# Animated Loading Spinner Collection

## Overview

A collection of CSS-only loading animations featuring rotating circles, bouncing dots, pulse rings, and colorful gradient spinners.

## Features

- Rotating circle loader
- Bouncing dot animation
- Pulse ring effect
- Gradient spinner animation
- Smooth infinite CSS animations
- Lightweight and reusable implementation

## Files

- demo.html – Demonstrates all loading spinner animations
- style.css – Contains loader styles and keyframe animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in a modern browser to view different animated loading spinner styles.
