# Business Landing — Moving Company

## What does this do?
A complete, copy-paste ready full-page website for a Moving & Logistics company styled with a trustworthy corporate aesthetic, featuring entrance animations, interactive calculators, checklist grids, and testimonial carousels.

## How is it used?
Open `demo.html` directly in any web browser to view the page. You do not need any build step or local server. The CSS transitions and animations are triggered on scroll using the project's classes.

## Why is it useful?
This showcases how EaseMotion CSS can be applied to real-world business landing pages to provide premium, interactive motion designs without requiring heavy third-party animation libraries or custom keyframes.

## EaseMotion CSS Classes Showcased
- **Entrance & Hover Animations**:
  - `.ease-slide-in-left` / `.ease-slide-in-right`: Used for hero grid entrance slide effects.
  - `.ease-hover-grow`: Used for main logos, navigational elements, and review selectors.
  - `.ease-hover-lift-shadow`: Used for primary action cards and pricing buttons.
  - `.ease-hover-underline`: Used for header navigation menu items.
  - `.ease-card-lift`: Used for interactive service info boxes.
  - `.ease-hover-glow`: Used for call-to-action booking button emphasis.
- **Scroll Reveal Animations**:
  - `.ease-reveal`: Applied to sections to hide elements offscreen.
  - `.ease-reveal-up` / `.ease-reveal-left` / `.ease-reveal-right` / `.ease-reveal-scale`: Directional variations that slide or scale items when intersecting the viewport.
  - `.ease-reveal-delay-1` / `.ease-reveal-delay-2` / `.ease-reveal-delay-3`: Staggered timing delays for grid columns.
- **Looping & Status Animations**:
  - `.ease-ping`: Used for active map pins to indicate service terminal locations.
