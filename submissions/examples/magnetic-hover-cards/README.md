# Magnetic Hover Cards

## Overview
This feature introduces a cursor-reactive "magnetic" hover effect where elements respond dynamically to mouse movement.

## ✨ Features
- Cursor-based motion interaction
- Smooth transform animations
- Subtle magnetic attraction effect
- Lightweight JavaScript integration

## 🎯 Utility Classes

- `ease-fade-in`
- `ease-hover-grow`

## 💡 Usage

```html
<div class="card magnetic">
  Content
</div>
