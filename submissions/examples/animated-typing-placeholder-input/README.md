# Animated Typing Placeholder Input

## Description

An animated input field that simulates placeholder text being typed in real time.

## Features

- Typing animation effect
- Blinking cursor
- Multiple rotating suggestions
- Lightweight implementation

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and watch the placeholder suggestions animate automatically.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
