# Wedding Accessories Shop Demo

A responsive **Wedding Accessories Shop** landing page built using EaseMotion CSS.

## Features

- Hero section with romantic wedding theme
- Shop by category
- Real Wedding Gallery
- Customization CTA
- Customer Reviews
- Wedding Checklist Download CTA
- Responsive layout
- Hover effects
- EaseMotion CSS animations

README.md
