# Pulse Image Cropper

A responsive **Pulse Image Cropper** component created with pure HTML and CSS for the EaseMotion CSS library. It features a neumorphism-inspired design with a continuously pulsing crop frame that highlights the image selection area.

## Features

- Pure HTML & CSS
- No JavaScript required
- Neumorphism-inspired UI
- Smooth pulse animation
- Responsive layout
- Accessible and lightweight
- Easy to customize

## Folder Structure

```
pulse-image-cropper-srujana-manda/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern browser.
2. Keep `style.css` in the same folder.
3. Replace the image URL with your own image if desired.

## Customization

You can customize:

- Crop frame size
- Pulse animation speed
- Colors
- Border radius
- Shadows
- Image dimensions
