Stylelint Property Order

This submission demonstrates configuring Stylelint with the stylelint-order plugin to enforce a consistent CSS property ordering convention across the codebase. 

{
  "plugins": ["stylelint-order"],
  "rules": {
    "order/properties-order": [
      "position",
      "display",
      "width",
      "height",
      "margin",
      "padding",
      "border",
      "background",
      "color",
      "font",
      "animation"
    ]
  }
}

Benefits

  • Improves readability of stylesheets.
  • Produces cleaner code reviews and diffs.
  • Creates consistency across contributors.
  • Reduces formatting discussions in pull requests.
