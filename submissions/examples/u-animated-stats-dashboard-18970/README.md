# Animated Statistics Dashboard Cards

A modern dashboard component featuring animated statistics cards, trend indicators, progress bars, and responsive layouts.

## Features

- Animated card reveal effects
- Progress bar animations
- Trend indicators
- Responsive dashboard grid
- Hover interactions
- Analytics-inspired UI

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in your browser.

## Customization

You can customize:

- Statistics values
- Icons
- Progress percentages
- Trend indicators
- Colors
- Animation timing

## Use Cases

- Admin dashboards
- Analytics platforms
- SaaS applications
- Business reporting tools
- Project monitoring systems
