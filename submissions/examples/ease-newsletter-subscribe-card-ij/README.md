# Newsletter Subscribe Card

An email subscription card with slide-in confirmation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--card-bg` | `#2d2d44` | Card background |
| `--input-bg` | `#1a1a2e` | Input field background |
| `--accent-color` | `#4ecdc4` | Button/accent color |
| `--slide-duration` | `0.35s` | Transition speed |
