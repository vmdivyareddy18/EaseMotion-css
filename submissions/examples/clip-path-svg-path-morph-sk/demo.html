Same-Command Constraint

For clip-path: path() animation to interpolate smoothly, every keyframe path must contain the same number of commands in the same order. Morphing between M C C C C Z and M L L Z will snap rather than interpolate. Export paths from a vector tool with the same anchor count to guarantee clean transitions.
