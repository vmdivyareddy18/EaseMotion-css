# Interactive Bento Dashboard Grid

## Description
A premium responsive Bento Grid layout tailored for modern web dashboards. Features customized spotlight cursor tracking glows, micro-interaction states, and full responsive alignment layouts.

## Installation
Add the HTML file and include `style.css` in your project folder.

## Usage
Semantic layout example:
```html
<section class="bento-grid">
  <div class="bento-card card-large" tabindex="0">
    <div class="card-glow"></div>
    <div class="card-content">
      <h2>Title</h2>
      <div class="metric">Details</div>
    </div>
  </div>
</section>
```

## Accessibility
- Tab focusable cards via `tabindex="0"`.
- Keyboard indicator states using high contrast outline rings.
