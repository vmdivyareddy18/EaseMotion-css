Smooth-Animated Accordion / Disclosure

CSS-only expand/collapse animation using native <details>/<summary> — no JavaScript, fully keyboard accessible.

What is EaseMotion CSS? ▾
EaseMotion CSS is an animation-first CSS framework with human-readable utility classes, letting you build expressive, motion-rich interfaces without a build step or custom keyframes.
Does this accordion require JavaScript? ▾
No. The expand/collapse motion is driven entirely by CSS, using a grid-template-rows: 0fr → 1fr transition on the content wrapper. Native <details> handles the open/closed state and keyboard interaction.
Is it accessible? ▾
Yes — native <details>/<summary> gives you keyboard navigation, focus handling, and screen reader support for free. The component also respects prefers-reduced-motion.
