# Portfolio Projects Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive portfolio project showcase.

---

## Components

- Responsive Grid
- Six Project Cards
- Project Image
- Category
- Description
- View Project Button

---

## Folder Structure

```
submissions/examples/portfolio-projects-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Resize the browser window to verify responsiveness.
3. Observe the animation effects applied to each project card.

---

## Features

- Responsive portfolio layout
- Six project showcase cards
- Modern card design
- Uses existing EaseMotion CSS animation classes
