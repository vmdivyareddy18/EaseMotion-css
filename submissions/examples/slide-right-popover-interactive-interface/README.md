# CSS Slide-Right Popover

A pure CSS animated popover component with a smooth slide-right entrance effect.

## Features

- Pure CSS animation
- No JavaScript required
- Responsive design
- Keyboard accessible using focus states
- Custom animation controls using CSS variables

## Usage

```html
<div class="container">

<button class="popover-btn">
  Hover Me
</button>

<div class="popover">
  Content here
</div>

</div>
