# Ease Ripple

1. What does this do? Adds a pure CSS Material Design-style ripple expansion animation to elements upon clicking.
2. How is it used? Apply the `.ease-ripple` class to any clickable element (like buttons, links, or cards), and customize it using `--ease-ripple-color` and `--ease-ripple-duration` if desired.
3. Why is it useful? It provides immediate, high-fidelity tactile feedback on user interaction without any JavaScript dependency, while natively respecting reduced motion preferences.
