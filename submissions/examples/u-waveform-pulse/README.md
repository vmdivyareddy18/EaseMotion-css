# ease-waveform-pulse

Animated audio waveform component inspired by media player visualizers.

## Features

- Pulsing waveform bars
- Organic waveform appearance
- Staggered animation delays
- Responsive design
- Pure HTML and CSS

## Acceptance Criteria Covered

### Waveform Animation

- Multiple animated waveform bars
- Similar to volume/equalizer bars
- Organic pulse behavior

### Random Looking Heights

Bars use varying heights to simulate a realistic audio waveform:

- Short bars
- Medium bars
- Tall peaks

### Use Cases

- Music players
- Podcast apps
- Audio dashboards
- Streaming platforms
- Voice recording interfaces
