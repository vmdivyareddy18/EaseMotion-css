# Animated Breadcrumb (`ease-breadcrumb-cv`)

A lightweight animated breadcrumb component built with pure HTML and CSS.

## Features

- Smooth slide-in animation
- Animated separators
- Gradient hover underline
- Three color themes
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```
ease-breadcrumb-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

Customize the theme by changing the breadcrumb class:

- `blue`
- `purple`
- `green`

## Browser Support

Works in all modern browsers.
