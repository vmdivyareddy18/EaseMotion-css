# Animated Card Carousel Component

## Overview

A CSS-based card carousel component featuring smooth scaling effects, active card highlights, and interactive hover animations.

## Features

- Responsive card layout
- Active card highlighting
- Smooth hover scaling animation
- Glowing shadow effects
- Modern card design
- Lightweight CSS-only implementation

## Files

- demo.html – Demonstrates the animated card carousel
- style.css – Contains carousel styles and animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in a modern browser to view the animated card carousel.

Hover over the cards to experience smooth scaling and glow effects.
