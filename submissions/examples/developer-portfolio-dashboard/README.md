# Animation: Developer Portfolio Dashboard

A modern developer dashboard inspired by GitHub, Vercel, and Linear. It combines profile information, statistics, and activity feeds with subtle motion and interactive cards.

## Features

- Pure HTML & CSS
- Responsive dashboard layout
- Animated activity feed
- Interactive statistic cards
- Developer-focused design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="stat-card">
    <h3>Repositories</h3>
    <span>42</span>
</div>
```

Cards respond to user interaction while maintaining a clean and professional dashboard appearance.

## Why is it useful?

Developer dashboards are commonly used for portfolios, analytics panels, SaaS products, and admin interfaces. This example demonstrates how motion can improve hierarchy and user engagement without overwhelming the layout.
