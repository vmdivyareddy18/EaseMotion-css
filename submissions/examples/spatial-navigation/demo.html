CSS Spatial Navigation Utilities

Directional keyboard navigation between elements using arrow keys, TV remotes, and game controllers — entirely CSS-driven with spatial-navigation-contain and nav-up/down/left/right.

⌨️ Arrow Keys 📺 TV / Remote 🎮 Game Controller ♿ Accessibility

🎯 How to try: Click any button to focus it, then use ↑ ↓ ← → arrow keys to navigate spatially between items.

Navigation Grid

A 2×2 grid with spatial-navigation-contain: contain keeps arrow key navigation inside the grid. Each button has nav-up/down/left/right: auto so the browser computes the best target based on geometry.

A vertical nav list — focus any item and press ↑ / ↓ to move between items. Each item has explicit nav-up and nav-down targets defined by ID for precise control.

TV Remote Interface

A simulated smart TV app layout with a sidebar and a content grid. Uses ease-nav-zone on the sidebar and ease-nav-grid on the main content, each with spatial-navigation-contain: contain to keep focus within its zone. The large focus ring makes navigation visible at TV viewing distance.

🔧 CSS Reference

/* 1. Create a spatial navigation container */
.ease-nav-grid {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 1rem;
  spatial-navigation-contain: contain; /* keep focus inside */
}

/* 2. Enable auto spatial navigation on focusable items */
.ease-nav-grid button {
  nav-up: auto;    /* browser picks best target upward */
  nav-down: auto;
  nav-left: auto;
  nav-right: auto;
}

/* 3. Explicit targets for precise control */
#nav-dashboard { nav-down: #nav-analytics; }
#nav-analytics  { nav-up: #nav-dashboard; nav-down: #nav-messages; }
#nav-messages   { nav-up: #nav-analytics; nav-down: #nav-settings; }

/* 4. Zone containers (vertical list, etc.) */
.ease-nav-zone {
  spatial-navigation-contain: contain;
}
.ease-nav-zone :is(a, button, [tabindex]) {
  nav-up: auto;
  nav-down: auto;
  nav-left: auto;
  nav-right: auto;
}
