# Hexagonal Grid Pattern (#36726)

A honeycomb-style hexagonal grid where cells morph (scale, rotate, and round corners) on hover.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--hex-color` | `#8b5cf6` | Fill color of hexagon cells |
| `--grid-bg` | `#0f0f13` | Background color of the grid area |
| `--hex-size` | `50px` | Width of each hexagon cell |
| `--morph-duration` | `0.5s` | Duration of the morph transition |

## Interactive Demo

Open `demo.html` and hover over any hex cell. Adjust the color, size, and morph speed using the controls.
