Drag-and-Drop Visual Feedback Collection

Five CSS-only visual states for drag interactions. This collection only owns the look — drop-zone highlight, drag ghost, invalid/success flashes, and a grip-handle affordance. The actual drag mechanics (native HTML5 Drag and Drop, or any JS sortable library) just need to toggle the state classes shown below.

ease-drag-handle + ease-drop-zone

Grip affordance on each item, dashed zone that highlights on dragover. Try actually dragging an item with your mouse — this uses native HTML5 Drag and Drop.

ease-drop-invalid / ease-drop-success

Shake + red flash for a rejected drop, scale-pulse + green flash for an accepted one. Click the buttons below to preview each animation.

  • ⠿ Drop target preview

In real use, your drag logic adds .is-invalid or .is-success on drop, then removes it after the animation ends (~450ms) so it can replay on the next interaction.
