# Autonomous Space Exploration Operations Center

Phase 430 submission for the EaseMotion-css repository issue: an enterprise-grade static mission control dashboard for autonomous space exploration operations.

## Included Files

- `demo.html` - semantic HTML5 dashboard with all requested mission sections
- `style.css` - responsive CSS3 styling, animations, custom properties, dark and light mode support
- `README.md` - setup notes and feature summary

## Feature Coverage

- 12 mission KPI cards
- 8 spacecraft monitoring cards
- 6 scientific research cards
- 4 executive leadership cards
- 16 dashboard sections covering mission overview, Mars, lunar, probes, orbital stations, instruments, navigation, communications, crew, launch, planning, science data, partners, safety, executive, and future exploration
- Real-time style telemetry displays
- Trajectory visualization
- Deep Space Network communication status
- Scientific observation scheduler

## Technical Notes

- Pure HTML5 and CSS3
- No JavaScript
- CSS Grid and Flexbox layouts
- Mobile-first responsive design
- Dark mode and light mode via `prefers-color-scheme`
- Accessible skip link, semantic landmarks, descriptive labels, visible focus states, and reduced-motion support
- Animated starfield, orbital motion, telemetry pulse, and hover interactions

## Usage

Open `demo.html` in any modern browser. The page is self-contained and links only to the local `style.css` file.
