# Writing Mode Tb

writing-mode — Switches between horizontal and vertical text flow

## Features
- Uses `--ease-color-text`
- Uses `--ease-text-sm`
- Uses `--ease-color-surface`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Writing Mode Tb</span>
</div>
```

## Why is it useful?

This submission demonstrates the writing mode tb CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
