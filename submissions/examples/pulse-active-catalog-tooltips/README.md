# Pure CSS Pulse-Active Product Catalog Tooltip

An elegant, highly performant, zero-JavaScript tooltip component styled explicitly to complement high-end product display grids and retail catalog matrices.

## Features
- **Zero Running Overhead:** Drives continuous animation loops strictly via native hardware-accelerated CSS properties.
- **Ambient Breathing Pulse:** Gently loops between minor scale matrices and accent shadow values to anchor user vision on core info tags.
- **Clean Luxury Aesthetic:** Complemented by neutral product surfaces, sleek minimal borders, and crisp, modern typography.
- **Accessible Interactions:** Seamlessly supports keyboard tab focus arrays with custom tracking highlights.

## Variable Override Configurations

| Property Variable | Targeting Focus Utility | Default Core Setup |
| :--- | :--- | :--- |
| `--catalog-accent` | Core focal highlight and pulse drop glow color | `#0284c7` |
| `--tooltip-pulse-scale` | Peak scaling factor during the breathing pulse cycle | `1.04` |
| `--tooltip-pulse-speed` | Complete duration metric for a singular pulse loop | `2s` |
| `--tooltip-fade-speed` | Base element entry visibility fade delay | `0.2s` |
