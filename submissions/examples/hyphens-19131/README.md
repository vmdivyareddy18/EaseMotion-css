# Hyphens Text Layout Utilities
Utility classes to control word breaking via hyphenation.

- **hyphens-auto**: Lets the browser automatically determine break points based on language settings.
- **hyphens-none**: Disables hyphenation entirely.
- **hyphens-manual**: Hyphenates only at points explicitly marked by the user (e.g., &shy;).
- Requires a `lang` attribute on the HTML element for optimal performance.
