# Cyberpunk Glass Card

A modern, glowing cyberpunk-themed glassmorphism card with a rotating neon border effect on hover. Perfect for futuristic UI layouts or premium feature showcases.

## Features
- **Glassmorphism**: Uses `backdrop-filter` for a sleek frosted glass look.
- **Neon Glow Hover**: A hidden conic gradient rotation creates a smooth glowing border sweep when hovered.
- **3D Pop**: Scales and translates up slightly for a satisfying physical interaction.

## How to use

```html
<div class="ease-cyberpunk-glass-card">
    <h2>Cyberpunk Glow</h2>
    <p>Hover me for an awesome neon 3D glass effect!</p>
</div>
```
