# 🦍 Donkey Kong

A classic Donkey Kong arcade game built with HTML, CSS, and JavaScript. Climb ladders, jump over barrels, and rescue the princess from the top of the construction site!

## 🎮 How to Play

1. Click **"Start Game"** or press **Space** to begin
2. Use **Arrow Keys** to control Jumpman (Mario)
   - **←** / **→** – Move left/right
   - **↑** – Climb up ladders
   - **↓** – Climb down ladders
   - **Space** – Jump
3. **Avoid rolling barrels** and fireballs
4. **Climb** to the top of the structure
5. **Rescue the princess** to advance to the next level!

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `←` / `→` | Move Left/Right |
| `↑` | Climb Up Ladder |
| `↓` | Climb Down Ladder |
| `Space` | Jump |
| `Space` (Menu) | Start Game |
| `R` | Reset Game |

## ✨ Features

- 🦍 **Classic Donkey Kong Gameplay** – Climb, jump, and rescue
- 🏗️ **Multiple Levels** – Different layouts with increasing difficulty
- 🛢️ **Rolling Barrels** – Avoid barrels thrown by Donkey Kong
- 🔥 **Fireballs** – Bouncing fireballs in higher levels
- 🎯 **Scoring System** – Points for completing levels
- 🏆 **High Score** – Saved in browser local storage
- ❤️ **Lives System** – 3 lives to rescue the princess
- 🎨 **Retro Visuals** – Classic arcade aesthetic
- 📱 **Responsive Design** – Works on desktop and mobile

## 🎯 Scoring

| Action | Points |
|--------|--------|
| Complete Level | 100 × Level |

## 📊 Level Progression

- **Level 1** – Classic layout with 4 ladders
- **Level 2** – More complex platform layout
- **Level 3+** – Increasing difficulty with more platforms and ladders

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3 (Gradients, responsive design)
- Vanilla JavaScript
