# CSS Draw-Border Line Modal – Dashboard Analytics

A responsive Dashboard Analytics modal featuring a sequential Draw-Border Line animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Draw-Border Line animation
- Dashboard Analytics design
- Statistic cards
- Animated progress bar
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript animations

## Files

```
demo.html
style.css
README.md
```

## Customization

Edit the CSS variables in `:root`:

```css
--primary
--secondary
--bg
--radius
--duration
```
