# color-mix() Basics

Demonstrates the CSS `color-mix()` function, which blends two colors in a specified color space. Features an interactive color mixer and prebuilt examples across sRGB, HSL, OKLCh, linear sRGB, and LAB spaces.

## Usage

Open `demo.html` in any modern browser that supports `color-mix()`. Use the interactive color pickers and slider to see real-time mixing across five color spaces. Each example card shows a fixed mix with its source code.

## Browser Support

`color-mix()` is supported in Chrome 111+, Firefox 113+, Safari 16.2+, and Edge 111+. The demo degrades gracefully — unsupported browsers show fallback background colors. The dark theme builds on `#0a0f1e` with `#e2e8f0` text for comfortable viewing.
