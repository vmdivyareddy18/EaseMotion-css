# Animated Stock Market Portfolio Dashboard

A responsive investment portfolio dashboard built using HTML and CSS animations.

## Features

- Animated portfolio value
- Investment statistics cards
- Portfolio goal progress bar
- Daily performance status
- Responsive dashboard layout
- Smooth hover animations

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations
