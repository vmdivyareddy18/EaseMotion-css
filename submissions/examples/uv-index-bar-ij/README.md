# UV Index Bar

1. What does this do? A horizontal color-coded bar that fills with an animated width transition based on UV index value, changing from green (low) to purple (extreme).

2. How is it used? Apply `.uv-bar-fill` to the fill element and set `--uv-pct` (width percentage) and `--uv-color` via JavaScript. The `.uv-bar-track` wraps the bar.

3. Why is it useful? It gives weather dashboards and widgets a smooth, animated UV index visualisation that communicates risk level instantly through both width and colour, aligning with EaseMotion CSS's utility animation approach.
