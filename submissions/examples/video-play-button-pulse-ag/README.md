# Video Play Button Pulse

## What does this do?
Adds a pulsing animation to video play buttons, inviting users to click and start playback.

## How is it used?
Apply the `.play-pulse-btn` class to a button element wrapped around a play icon. The pulse ring animation draws the eye to the control.

## Why is this useful?
Provides an intuitive visual cue that a video element is interactive and ready to play, improving discoverability of media controls without JavaScript.
