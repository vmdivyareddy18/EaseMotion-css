:focus-visible Fix Demo

Fixes the bug where :focus shows the outline ring on mouse clicks as well as keyboard navigation. Now the ring only appears on keyboard (Tab) navigation, while mouse clicks stay clean.

❌ Before (buggy)

Uses *:focus — ring shows on mouse click too.

A link 
*:focus {
  outline: 2px solid var(--ease-color-primary);
  outline-offset: 2px;
}

✅ After (fixed)

Uses :focus-visible — ring only on keyboard Tab.

A link 
*:focus {
  outline: none;
}

*:focus-visible {
  outline: 2px solid var(--ease-color-primary);
  outline-offset: 2px;
  border-radius: 2px;
}
