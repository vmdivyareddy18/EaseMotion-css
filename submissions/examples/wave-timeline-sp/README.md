# Wave Timeline

A Health & Fitness inspired timeline featuring animated wave markers built using pure HTML and CSS.

## Features

- 🌊 Wave pulse animation
- 💪 Health & Fitness inspired layout
- 📱 Responsive design
- ♿ Accessible semantic HTML
- 🚫 No JavaScript required

## File Structure

```
wave-timeline-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

The animated timeline displays fitness milestones with a continuous wave effect.

## Use Cases

- Workout plans
- Fitness trackers
- Project timelines
- Progress tracking
- Event schedules
