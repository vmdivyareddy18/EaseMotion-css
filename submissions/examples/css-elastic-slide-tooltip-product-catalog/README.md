# CSS Elastic Slide Tooltip - Product Catalog Layout

A pure CSS animated tooltip designed for modern product catalog interfaces.

## Features

- Pure CSS implementation
- Elastic slide entrance animation
- Product UI friendly design
- Keyboard accessible
- Responsive layout
- Custom CSS animation variables

## Usage

```html
<div class="tooltip-wrapper">

<button class="product-btn">
View Product
</button>

<div class="tooltip">
Product information
</div>

</div>
