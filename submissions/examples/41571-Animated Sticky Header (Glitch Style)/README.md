# Animated Sticky Header (Glitch Style)

A neon-inspired, pure CSS glitch sticky header component using EaseMotion CSS variables.

## Features
- Pure CSS glitch effect
- Sticky positioning
- Neon glow effects using EaseMotion color variables
- Responsive design
- No JavaScript required

## Files
- `demo.html` - Live demo of the component
- `style.css` - Component specific styles and keyframes
