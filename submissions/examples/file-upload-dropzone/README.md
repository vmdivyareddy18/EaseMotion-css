# File Upload Drop Zone

A drag-and-drop file upload area with visual feedback. Dragging a file highlights the zone with an indigo border and background. Users can also click "browse files" to open a file picker. Selected files are displayed as preview chips below the drop zone.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Drag files onto the drop zone or click "browse files" to select files.

## Accessibility notes

The drop zone can be activated via the file input trigger. Multiple file selection is supported. Reduced motion disables the border/background transitions.
