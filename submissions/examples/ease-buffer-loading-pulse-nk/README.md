# ease-buffer-loading-pulse-nk

### What does this do?

Provides a subtle animated loading indicator for media buffering states.

### How is it used?

Include the loading container inside a video or audio player overlay.

```html
<div class="buffer-loading-pulse">
    <span></span>
    <span></span>
    <span></span>
</div>
Why is it useful?

Media applications frequently need lightweight buffering indicators. This component provides a clean, non-intrusive pulse animation suitable for video and audio loading states.
