# Upload Drop Bounce

A drag-and-drop upload surface with a subtle lift and a bouncing upload icon.

## Files

- `demo.html` - file input wrapped in a visible drop-zone label.
- `style.css` - hover, focus-within, icon bounce, and reduced-motion fallback.

## Highlights

- Uses native file input behavior.
- Provides the same visual treatment for hover and keyboard focus.
- Disables movement for reduced-motion users.
