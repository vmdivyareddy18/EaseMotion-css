# Floating Particles – Interactive particle system following cursor

> A production-ready interactive particle system where particles float, connect, and respond to cursor movement with mesmerizing animations and full customization. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Interactive particles** – Follow cursor movement
- ✅ **Particle connections** – Lines between nearby particles
- ✅ **Glow effects** – Soft glow around each particle
- ✅ **Trail effects** – Particles leave subtle trails
- ✅ **Customizable** – Count, speed, size, color
- ✅ **Explode effect** – Particles burst outward
- ✅ **Toggle particles** – Enable/disable system
- ✅ **Randomize** – Random settings with one click
- ✅ **Keyboard shortcuts** – Space (toggle), R (reset), E (explode)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
