# CSS Noise Texture Generator

## What does this do?
A CSS noise texture generator that creates grainy overlay effects using repeating-conic-gradient patterns with adjustable intensity, color, and opacity.

## How is it used?
Open demo.html in a browser. Adjust the intensity, color, and opacity sliders to create custom noise textures. Click "Copy CSS" to copy the generated CSS code for use in your projects.

## Why is this useful?
This provides a lightweight, CSS-only way to add film grain or noise textures to UI elements without external images or JavaScript canvas. It demonstrates procedural CSS gradient generation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to generate textures.
