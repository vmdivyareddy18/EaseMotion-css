# Ease Interactive Timeline

A beautiful, premium, and fully responsive interactive timeline component designed for modern web applications. 

## Features
- **Modern Aesthetics**: Utilizes a sleek dark mode with glassmorphism effects and curated color palettes.
- **Dynamic Interactions**: Features micro-animations and satisfying hover effects that scale up content and add glowing accents.
- **Responsive Design**: Automatically adjusts to a single-column layout on smaller screens.
- **Scroll Animations**: Timeline items gracefully fade and slide in as they scroll into view.

## Usage
Simply include the `style.css` and the HTML structure from `demo.html` into your project. 

### Files
- `demo.html`: The markup and a small JavaScript snippet for scroll-triggered animations.
- `style.css`: All styling, variables, and keyframes.

## Why it fits EaseMotion CSS
This component aligns perfectly with EaseMotion's philosophy by utilizing modern CSS techniques (glassmorphism, variables, gradients) to provide a premium, highly-interactive visual experience without relying on complex external libraries. It focuses on fluid motion, such as scroll-triggered fade-ins and dynamic hover states, to bring the timeline to life organically.

## Customization
You can easily customize the theme by modifying the CSS variables found at the top of `style.css`:
```css
:root {
    --bg-color: #0f172a;
    --accent-color: #3b82f6;
    /* ... */
}
```
