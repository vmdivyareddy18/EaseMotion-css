# Banking Loan Dashboard

A responsive **Banking Loan Dashboard** built using **HTML5** and **CSS3** that demonstrates loan management, approval tracking, EMI monitoring, repayment status, and banking workflow animations.

---

## Features

- Responsive dashboard layout
- Banking overview cards
- Loan approval progress
- Repayment status panel
- Credit score overview
- Recent loan activity feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Active Loans
- Pending Approvals
- EMI Collected
- Loan Revenue

### Banking Widgets

- Loan Approval Progress
- Repayment Status
- Credit Score Distribution

### Activity

- Recent Loan Activity
- Notifications Panel

### Quick Actions

- Approve Loan

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
