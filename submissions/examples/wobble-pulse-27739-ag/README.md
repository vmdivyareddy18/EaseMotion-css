# Keyframe Wobble & Pulse Animation Mixins

An interactive button component demonstrating micro-interactions powered by CSS keyframes.

---

## What is Included

1. **`demo.html`**: A showcase dashboard containing interactive triggers, position containers, and headers.
2. **`style.css`**: Button styles, rotation keyframes, scaling animation triggers, and media rules.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Hover over the **Wobble Button** — verify that it oscillates slightly back and forth.
3. Hover over the **Pulse Button** — verify that it pulses smoothly and expands its glow shadow.
