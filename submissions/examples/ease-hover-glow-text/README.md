# Glow Text on Hover

Text glows with animated `text-shadow` on hover — pure CSS.

## Features
- Multiple text-shadow layers for neon glow
- Smooth opacity transition on hover
- Pure CSS, no JavaScript

## Usage
```css
.glow-text { transition: text-shadow 0.3s; }
.glow-text:hover { text-shadow: 0 0 10px #a78bfa, 0 0 20px #a78bfa, 0 0 40px #a78bfa; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
