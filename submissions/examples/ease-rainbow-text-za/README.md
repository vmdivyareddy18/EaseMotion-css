# CSS Rainbow / Animated Text

## What does this do?
CSS animated text effects including gradient shift, shimmer, neon glow, color cycle, and gradient mask text.

## How is it used?
```html
<p class="rw-grad-shift">Animated gradient text</p>
<p class="rw-neon">Neon glow text</p>
```

## Why is it useful?
Adds visual flair to headings and highlights without JavaScript or SVG. Six effects: gradient shift, shimmer, neon, cycle, gradient labels, hover underline. All CSS.
