# details-summary-styling

## Purpose
Demonstrates `details` and `summary` element styling with custom open/close indicators, animated transitions, and accordion-style FAQ.

## Features
- Custom `::marker` replacement with CSS indicators
- Smooth animated open/close transitions
- Accordion-style FAQ layout
- Rotating chevron indicator on open state

## Usage
Open `demo.html`. Click each summary to expand/collapse the details sections with animated transitions and custom icons.
