What is EaseMotion?

EaseMotion is a CSS utility library for smooth animations and transitions, designed to work with modern CSS features like scroll-driven animations, view transitions, and container queries.

How do I install it?

You can install via npm: npm install easemotion, or use a CDN link in your HTML. No build tools required — just include the CSS file.

Does it support prefers-reduced-motion?

Yes. All animations respect the user's prefers-reduced-motion setting and gracefully fall back to instant transitions or no animation.

Can I use it with frameworks?

Absolutely. EaseMotion is framework-agnostic. It works with React, Vue, Svelte, Angular, or vanilla HTML/CSS projects without any configuration.

Is it production-ready?

EaseMotion follows semantic versioning and is tested across modern browsers. Check the documentation for browser support details.
