Contrast Filter Utilities

Enhance readability, accessibility, and visual drama using simple, elegant CSS contrast filters. Standardized utility classes for any design context.

Low Contrast

Reduces contrast to 50%, perfect for disabled states, sub-elements, or muted backgrounds.

.ease-filter-contrast-50

Normal Contrast

Default contrast value (100%). Explicitly restores normal contrast levels on demand.

.ease-filter-contrast

High Contrast

Boosts contrast to 150%. Excellent for highlight effects, intense hover styles, or dark theme accessibility.

.ease-filter-contrast-150
