# Swing Hover Accessible Web Modal

A responsive, accessibility-focused modal built entirely with HTML and CSS using a smooth Swing-Hover animation.

## Features

- Pure CSS
- Responsive layout
- Keyboard accessible
- High-contrast interface
- Focus-visible support
- Swing-Hover animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--rotate
--primary
--primary-dark
--background
--surface
--radius
```
