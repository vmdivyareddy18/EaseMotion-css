# Ease Constellation Command Wheel

## Overview

A futuristic radial command menu featuring a glowing central hub surrounded by orbit-inspired action buttons using pure HTML and CSS.

## Features

- Radial command layout
- Glassmorphism buttons
- Animated center hub
- Responsive design
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-constellation-command-wheel-z1/
├── demo.html

└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

Introduces a reusable radial command interface suitable for AI dashboards, admin panels, developer tools, productivity software, and futuristic web applications.
