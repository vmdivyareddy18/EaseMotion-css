# Floating Action Cards Component

A premium collection of floating action cards designed for EaseMotion CSS (Issue #7616). 

## Overview

This component implements highly interactive, modern card designs that respond smoothly to user interactions. Utilizing advanced CSS transitions, transforms, and pseudo-elements, the cards create a stunning "lifting" effect on hover while seamlessly revealing contextual action buttons.

This package includes three unique stylistic variants to match different modern UI aesthetics.

## Features

- **3 Premium Variants:**
  - **Gradient:** Rich, vibrant colors with deep gradient shadow drop.
  - **Glassmorphism:** Frosted glass effect with backdrop blur and delicate luminous borders.
  - **Neon Glow:** High contrast cyber-aesthetic with intense glow and sharp actions.
- **Smooth Hover Lift Effect:** Uses CSS transforms (`translateY`, `scale`) for a natural hover lift.
- **Action Button Emphasis:** Action buttons smoothly slide up from the bottom while the card content subtly shifts upward.
- **Enhanced Shadow Depth:** Shadows transition deeply on hover, maximizing the 3D space illusion.
- **Pure HTML & CSS:** Absolutely zero JavaScript required. 
- **Fully Responsive:** Grid-based layout that adapts naturally to varying viewport sizes.

## File Structure

- `demo.html`: The HTML layout featuring the three card variants, embedded SVG icons, and decorative backgrounds.
- `style.css`: The styling rules defining the core layout, the three aesthetic variants, and complex hover state animations.
- `README.md`: Component documentation.

## Usage

Simply open `demo.html` in any modern web browser to view the component.

Hover over any of the three cards to experience the floating interaction and see the hidden action buttons animate into view.
