# Animated Event Management Dashboard

A modern **Event Management Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example showcases event analytics, attendee management, schedules, registrations, and activity tracking with smooth CSS-only animations.

## Features

* 🎉 Total Events overview
* 👥 Registered attendees statistics
* 🎤 Session tracking
* 🎟️ Tickets sold indicator
* 📅 Event schedule section
* 🧑‍💼 Attendee cards
* 🟢 Online/Offline indicators
* 📊 CSS-only registration analytics chart
* 📈 Progress indicators
* 🔔 Activity feed
* ✨ Smooth hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

## Customization

Modify the dashboard theme easily using CSS variables.

```css
:root{
    --primary:#2563eb;
    --success:#10b981;
    --warning:#f59e0b;
    --danger:#ef4444;

    --surface:#ffffff;
    --bg:#f5f7fb;
}
```
