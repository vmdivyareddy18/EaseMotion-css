# Search Result Highlight

A search result row that lifts and reveals a subtle highlight sweep on hover or focus.

## Files

- `demo.html` - linked search result markup.
- `style.css` - highlight sweep, card lift, focus-visible state, and reduced-motion support.

## Highlights

- Uses a pseudo-element for the highlight sweep.
- Mirrors hover interaction on keyboard focus.
- Keeps the result text readable during animation.
