# Meter Gauge SK

Animated progress meter driven by a CSS custom property.

## Usage

```html
<div class="meter-gauge" style="--value: 72;">
  <div class="meter-gauge__track">
    <div class="meter-gauge__fill"></div>
  </div>
</div>
```

Set `--value` from 0 to 100.

## Why EaseMotion CSS

Declarative progress animation with one variable and no JavaScript.
