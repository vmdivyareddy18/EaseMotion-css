# Digital Products Shop (EaseMotion CSS)

A production-ready landing page for a modern digital products shop, built entirely using EaseMotion CSS classes. This example showcases how to create a premium, responsive, and animated UI without writing custom CSS or external frameworks.

## 📁 Folder Structure

```
submissions/examples/digital-products-shop/
├── demo.html
└── README.md
```

## ✨ Features

- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop viewports using EaseMotion's responsive grid and utility classes.
- **Premium Aesthetics**: Leverages glassmorphism (`ease-card-glass`), neumorphism (`ease-card-neumorphic`), soft shadows (`ease-shadow-lg`), and curated typography.
- **Animations & Micro-interactions**: Includes entrance animations (`ease-slide-up`, `ease-zoom-in`, `ease-fade-in`), scroll reveal animations (`ease-reveal`, `ease-reveal-up`), and hover effects (`ease-hover-lift-shadow`, `ease-card-hover`).
- **Accessible & Semantic HTML**: Uses proper heading hierarchy, ARIA labels, and semantic HTML5 tags.
- **No External Dependencies**: 100% built with EaseMotion built-in utility and component classes (plus Unsplash for placeholder imagery).

## 🚀 EaseMotion Classes Used

Here are some of the key EaseMotion classes highlighted in this project:

- **Layout**: `.ease-container`, `.ease-grid`, `.ease-flex`, `.ease-flex-col`, `.ease-grid-cols-4`
- **Spacing**: `.ease-gap-6`, `.ease-py-8`, `.ease-padding-12`, `.ease-mx-auto`
- **Typography**: `.ease-text-4xl`, `.ease-text-primary`, `.ease-font-bold`, `.ease-text-muted`, `.ease-gradient-text-animated`
- **Components**: `.ease-btn`, `.ease-btn-primary`, `.ease-card`, `.ease-card-glass`, `.ease-card-neumorphic`, `.ease-badge-info`, `.ease-card-stat`
- **Animations**: `.ease-slide-up`, `.ease-zoom-in`, `.ease-bounce`, `.ease-reveal-up`, `.ease-delay-200`, `.ease-hover-lift`
