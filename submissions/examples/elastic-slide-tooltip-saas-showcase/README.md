# Elastic Slide Tooltip - SaaS Showcase Layout

## Description

A pure CSS animated tooltip with a smooth elastic slide interaction designed for SaaS showcase interfaces.

## Features

- CSS-only animation
- Elastic slide transition
- Responsive layout
- Keyboard accessible
- No JavaScript dependency
- Custom animation variables

## Usage

```html
<div class="tooltip-wrapper" tabindex="0">

  <button class="saas-button">
    View Product
  </button>

  <div class="tooltip">
    Tooltip content
  </div>

</div>
