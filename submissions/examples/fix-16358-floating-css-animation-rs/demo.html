Floating / Levitating Animations

Add smooth, organic floating animations to objects like hero images, 3D assets, or decorative icons.

Standard Float

`ease-animate-float`

Fast Float

`ease-animate-float-fast`

🚀

Dynamic Shadow

`ease-animate-float-shadow`

Implementation Detail

Notice the "Dynamic Shadow" variant. As the rocket object moves *up*, the underlying drop-shadow pseudo-element shrinks in scale and reduces its opacity, perfectly mimicking real-world lighting and distance.
