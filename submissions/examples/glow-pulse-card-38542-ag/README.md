# Glow Pulse Card for Corporate Layouts

A corporate metric dashboard card component demonstrating subtle glow pulse animations.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting financial metric panels, status badges, and description elements.
2. **`style.css`**: Glow pulse loop keyframes, box-shadow spreads, active borders, and accessibility blocks.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the **Monthly Revenue** KPI card.
3. Verify that the card border and shadow glow pulse slowly and continuously between low and high cyan/blue light ranges.
