## column-count-multi

Demonstrates the `column-count` property for creating multi-column layouts with balanced text flow.

### Key Features

- Three-column layout with column-gap and subtle column-rule styling
- Four-column layout with dashed rule for editorial aesthetics
- Two-column layout with a heavy double rule for visual separation

### Column Properties

`column-count` sets the number of columns. `column-gap` controls spacing between columns. `column-rule` adds a decorative divider. `column-fill: balance` (default) distributes content evenly across columns.
