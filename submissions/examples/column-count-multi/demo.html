Four Columns with Rule

The column-rule property adds a decorative line between columns. You can control its width, style, and color independently.

Column balance is controlled by column-fill. The default value "balance" tries to make columns equal height, while "auto" fills them sequentially.

Multi-column layout is great for long-form reading. The shorter line lengths improve readability compared to full-width text.

Browser support is excellent. All modern browsers support column-count, column-width, column-gap, column-rule, and column-span.

Two Columns with Heavy Rule

A two-column layout with a thick, styled column rule. The column rule sits in the middle of the column gap and adds visual separation between content blocks.

Use column-gap to control the spacing between columns and column-rule-width for the divider thickness. Together they create a polished editorial look.

Multi-column properties: column-count, column-gap, column-rule, column-fill, column-span.
