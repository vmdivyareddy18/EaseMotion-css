# Apex Martial Arts — Advanced Martial Arts School Business Landing

A premium, interactive, disciplined martial arts dojo relocation business landing page designed using **EaseMotion CSS** framework classes.

## Showcase Features

*   **Responsive Glassmorphism Navigation**: Implements `ease-navbar-glass` along with `ease-navbar-glass-sticky` for a modern, responsive sticky navbar.
*   **Uniform Special Promo Banner**: Features a custom-styled, elegant seasonal promo uniform registration banner with a pulsing indicator (`ease-ping`).
*   **Active Dojo Mat Utilization Progress**: Features a custom live mat utilization meter using `.ease-progress-striped`.
*   **Detailed Programs Grid**: Grid card system detailing Little Ninjas (kids), Adult Fitness, and Elite Competition Team classes using `.ease-card-hover` and `.ease-card-glow` interactions.
*   **Instructor Credentials**: Profile cards for Dojo head Senseis and masters showcasing ranks, skills, and background credentials.
*   **Belt Progression Timeline**: Vertical dotted timeline showing the path from white belt to black belt mastery.
*   **Dojo Schedule & Pricing**: Timetable calendar summary and class tuition plans (Basic Care, Unlimited Pro, Elite Competition) side-by-side.
*   **Interactive Booking/Scheduling Form**: Secure contact trial class scheduler using standard framework form components (`.ease-field`, `.ease-input`, `.ease-select`, `.ease-textarea`, `.ease-btn-block`).
*   **Premium Composed Footer**: Structured newsletter sign-up and social media navigation built utilizing the `.ease-footer` suite.

## EaseMotion CSS Classes Showcased

### Layout & Spacing
*   `.ease-container` — Constrains layout width to the design grid.
*   `.ease-center` — Centers contents using standard flex variables.
*   `.ease-flex` — Applies standard display flex.
*   `.ease-flex-col` — Vertical flexbox container direction.
*   `.ease-flex-wrap` — Wraps overflow elements safely.
*   `.ease-justify-center` / `.ease-justify-between` — Horizontally aligns items.
*   `.ease-items-center` / `.ease-items-start` — Vertically aligns items.
*   `.ease-gap-2` / `.ease-gap-3` / `.ease-gap-4` / `.ease-gap-8` / `.ease-gap-10` — Flexbox/Grid row and column spacing.
*   `.ease-grid` — Enables standard CSS Grid layout.
*   `.ease-grid-cols-2` / `.ease-grid-cols-12` — Set specific column layouts.
*   `.ease-col-span-5` / `.ease-col-span-7` — Responsive column span configurations.
*   `.ease-stack` / `.ease-stack-sm` — Flexbox stack helpers.
