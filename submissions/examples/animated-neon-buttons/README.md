# Animated Neon Border Button Collection

An interactive set of premium cyberpunk neon-styled buttons utilizing multi-layered CSS custom variables, ripple triggers, and flickering physics variables.

## Contribution Metadata

1. **What is this component doing?**
   It generates dynamic, customizable neon interactive buttons with structural themes (Plasma, Acid, Infrared, Arctic) across structural components (CTA, outlined dashboards, round icon targets). It embeds breathing neon pulse frames, hover tube micro-flickers, and coordinate-mapped click ripples.

2. **Why should this be added to EaseMotion CSS?**
   EaseMotion CSS currently features grid structures and glassmorphism windows but possesses zero standalone button macro collections. This adds high-density dark UI utilities utilizing specialized `box-shadow` layering mechanics that do not override layout constraints.

3. **How to test this module?**
   Simply open the `demo.html` directly inside any modern internet browser framework. Hover over different variations to inspect rapid flicker states, and execute clicks to confirm vector center points for responsive calculations.
