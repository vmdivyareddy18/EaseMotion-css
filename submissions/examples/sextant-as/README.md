# Sextant

A pure CSS and HTML marine sextant sighting a star above the horizon, its index arm swinging along the graduated arc.

## What it shows

- The graduated limb built as a full brass ring cut down to a 60 degree arc: a radial mask makes the ring, a clip-path wedge keeps only the sixth
- Degree ticks from a repeating conic gradient masked to the same band
- An index arm pivoting about the exact centre of the arc, so its tip tracks the graduations
- A tiny star that twinkles above a glowing horizon line
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

The index arm and star stop under `prefers-reduced-motion: reduce`.
