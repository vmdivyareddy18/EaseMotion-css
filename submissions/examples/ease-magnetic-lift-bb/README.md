# Magnetic Drag and Lift Card Animation

## Description
An interactive UI container card featuring physics-simulated drag resistance and elastic auto-centering bounce transitions.

## Usage
Simply drop the card structure into your layout:
```html
<div class="magnetic-container">
  <div class="magnetic-lift-card" id="card">
    <div class="magnetic-lift-card-content">
      <h4>Magnetic Terminal</h4>
      <p>Click, hold, and drag.</p>
    </div>
  </div>
</div>
```

## Why is it useful?
Delivers a highly tactile, premium feel to grid dashboards or interactive layout components using lightweight CSS translates. Safe transitions for systems utilizing reduced motion preferences.
