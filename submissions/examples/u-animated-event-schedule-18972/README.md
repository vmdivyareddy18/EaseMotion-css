# Animated Event Schedule Timeline

A responsive event schedule component featuring animated timeline cards, time indicators, category badges, and staggered reveal effects.

## Features

- Timeline layout
- Animated event reveal
- Category badges
- Hover interactions
- Responsive design
- Conference and workshop friendly UI

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

You can customize:

- Event categories
- Timeline colors
- Animation timing
- Event details
- Layout spacing

## Use Cases

- Conferences
- Hackathons
- Workshops
- College events
- Meeting schedules
- Webinar agendas
