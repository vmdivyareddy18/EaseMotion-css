# Ease Holographic File Stack

## Overview

A futuristic stack of animated glassmorphism file cards inspired by developer workflows. Built entirely with semantic HTML and modern CSS.

## Features

- Layered file stack
- Holographic gradient overlay
- Floating animation
- Hover lift interaction
- Responsive layout
- Pure HTML & CSS

## Folder Structure

```text
ease-holographic-file-stack-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Adapts smoothly across desktop and mobile devices.

## Why it fits EaseMotion CSS

Demonstrates reusable layered card animations with lightweight CSS, making it suitable for dashboards, documentation sites, developer portfolios, and productivity interfaces.
