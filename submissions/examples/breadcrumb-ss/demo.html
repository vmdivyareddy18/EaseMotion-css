♿ Accessibility

  • ✅ <nav aria-label="Breadcrumb"> — semantic navigation landmark
  • ✅ <ol> structure — ordered list for hierarchy
  • ✅ focus-visible outline — keyboard navigation visible
  • ✅ prefers-reduced-motion — disables staggered entrance
  • ✅ Current page marked with <span> (not link) — no self-referencing
  • ✅ Separators are decorative — hidden from screen readers via CSS
