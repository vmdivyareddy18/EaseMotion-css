# Fullscreen Overlay – Menu expands to fullscreen

> A production-ready fullscreen overlay menu component that expands with smooth animations, elegant typography, and full keyboard accessibility. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Fullscreen expansion** – Smooth, immersive menu
- ✅ **Staggered animations** – Elegant content reveal
- ✅ **Background blur** – Modern glass effect
- ✅ **Keyboard accessible** – Escape to close
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
