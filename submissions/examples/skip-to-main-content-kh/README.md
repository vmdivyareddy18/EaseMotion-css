# Skip To Main Content Link

## What does this do?
Adds an accessible skip navigation link for keyboard users.

## Why is it useful?

It helps keyboard users bypass repeated navigation links and jump directly to the main content area, improving accessibility and supporting WCAG best practices.

## How is it used?
```html
<a href="#main" class="skip-link">Skip to main content</a>

<main id="main">
    ...
</main>
