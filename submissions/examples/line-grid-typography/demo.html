📐 Baseline Rhythm 🔲 Line Grid ✍️ Vertical Spacing

💡 What is a baseline grid? It's an invisible vertical rhythm where every line of text sits on a consistent unit. Like notebook paper for your layout. Without it, mixing headings and paragraphs creates uneven gaps. With it, the eye flows naturally down the page.

1. Basic Line Grid — .ease-line-grid

The primary utility from the issue snippet. Sets a consistent line-height and uses row-gap: calc(1lh) to space children by exactly one line-height unit. All child margins are reset to zero so spacing is controlled entirely by the grid.

The Art of Typographic Rhythm

Good typography is invisible. When readers notice your type, something has gone wrong. A consistent vertical rhythm is the foundation of great editorial design.

What is Vertical Rhythm?

Vertical rhythm refers to the consistent spacing between lines of text and blocks of content on a page. When every element aligns to a baseline grid, the eye can flow naturally from one line to the next.

"Typography is the craft of endowing human language with a durable visual form." — Robert Bringhurst

How Line Grid Helps

  • Eliminates manual margin calculations
  • Keeps headings and body text in sync
  • Works with any font size or line height
  • Scales naturally across viewport sizes

With .ease-line-grid, all of this is automatic. Set it once on your article container and every child aligns perfectly.

🔧 CSS Reference

/* 1. Base Line Grid (from the issue snippet exactly) */
.ease-line-grid {
  line-height: 1.75;
  display: grid;
  row-gap: calc(1lh); /* 1lh = 1 line-height unit */
}

.ease-line-grid > * {
  margin: 0; /* Reset all child margins — grid handles spacing */
}

/* 2. Compact (tighter rhythm for sidebars / data) */
.ease-line-grid-sm {
  line-height: 1.5;
  display: grid;
  row-gap: calc(1lh);
}

.ease-line-grid-sm > * {
  margin: 0;
}

/* 3. Relaxed (generous rhythm for editorial / articles) */
.ease-line-grid-lg {
  line-height: 2;
  display: grid;
  row-gap: calc(1lh);
}

.ease-line-grid-lg > * {
  margin: 0;
}

/* WHAT IS 1lh?
   The 'lh' unit equals the computed line-height of the element.
   calc(1lh) therefore always equals exactly one "line" of space,
   keeping every child spaced by a single baseline unit.
   Browser support: Chrome 109+, Firefox 120+, Safari 17.2+
   Fallback: Use em-based row-gap for older browsers. */
