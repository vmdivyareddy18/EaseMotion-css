# Analog Clock

Analog clock with animated hour, minute, and second hands using CSS transforms. Roman numerals surround the face and a digital time display sits below.

## Features

- Smooth hand rotation via CSS transform-origin
- Continuous second hand movement (no tick skipping)
- Roman numeral tick marks (I–XII)
- Digital time display below the clock face
- Customizable size, colors, and duration via CSS custom properties

## Usage

The clock updates continuously via JavaScript. All hands rotate from the center bottom. Set `--ancl-size` to adjust the clock diameter.
