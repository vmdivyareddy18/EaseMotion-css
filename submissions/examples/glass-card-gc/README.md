# Glass Card CTA UI Example

## 1. What does this do?

This submission showcases a glassmorphism card with hover lift, blur background, and a pulsating call-to-action accent. The design blends translucent surfaces with motion to create a premium CTA card.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example contains glass-style cards and CTA buttons with hover interaction. Copy the `.glass-card` markup and CSS into your project to integrate the glass card CTA pattern.

---

## 3. Why is it useful?

Glass cards provide a polished visual layer for promotional content and CTAs. Animated hover effects and accent pulses draw attention while maintaining a modern, lightweight UI aesthetic.
