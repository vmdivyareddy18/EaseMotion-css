# Ease Gradient Border Glow Animation

## Overview

This animation creates a glowing animated gradient border around an element using only CSS.

The border continuously rotates through multiple colors, producing a modern premium effect while keeping the content readable.

---

## Features

- Pure CSS implementation
- Animated glowing gradient border
- Lightweight
- Reusable
- Responsive
- Accessible with reduced motion support

---

## Example

```html
<div class="ease-gradient-border-glow">
    Premium Card
</div>
```

---

## Use Cases

- Cards
- Login Forms
- Pricing Cards
- Buttons
- Hero Sections
- Product Highlights

---

## Browser Support

Works in all modern browsers supporting:

- CSS Animations
- CSS Gradients
- Pseudo Elements

---

## Accessibility

Supports

`prefers-reduced-motion`

to disable the animation for users who prefer reduced motion.
