# Animated Compass Direction Spin

## Overview

A reusable compass icon animation featuring a gentle directional spin effect suitable for maps, navigation systems, route planners, travel applications, and exploration interfaces.

## Features

- CSS-only animation
- Compass direction effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="compass-button">
  <span class="compass-icon">🧭</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
