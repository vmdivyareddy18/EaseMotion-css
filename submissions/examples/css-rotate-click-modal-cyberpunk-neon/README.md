# CSS Rotate-Click Modal – Cyberpunk Neon Layout

## Overview

A responsive Cyberpunk Neon modal built with pure HTML and CSS. The component features a smooth rotate-click opening animation, glassmorphism styling, responsive design, and keyboard accessibility with no JavaScript required.

## Features

- Pure HTML & CSS
- Rotate-click opening animation
- Cyberpunk neon interface
- Glassmorphism modal
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Reduced motion support

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--duration`
- `--easing`
- `--rotate`
- `--scale`
- `--radius`
- `--blur`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Launch Console** to open the animated modal and click **×** to close it.

Adjust the CSS custom properties in `style.css` to customize the colors, animation, rotation angle, scaling, blur, and overall appearance.
