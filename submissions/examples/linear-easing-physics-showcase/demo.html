How it works: A hidden checkbox is toggled by the label button. The :checked ~ .comparison-grid .demo-card sibling selector applies transform: translateY(-140px), which each card transitions using its own timing function. The linear() spring physically overshoots and oscillates — something cubic-bezier cannot replicate. An @supports guard provides a cubic-bezier fallback for browsers that do not yet support linear().
