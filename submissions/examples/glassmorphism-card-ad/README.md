# Glassmorphism Card

1. **What does this do?**  
   It provides a stylish glassmorphism card component with a frosted glass background blur and a smooth hover lift effect.

2. **How is it used?**  
   ```html
   <div class="glass-card">
       <h2>Card Title</h2>
       <p>Card content goes here.</p>
   </div>
   ```

3. **Why is it useful?**  
   It brings a modern, premium aesthetic to UI designs, fitting perfectly into the sleek, animated style of EaseMotion CSS.
