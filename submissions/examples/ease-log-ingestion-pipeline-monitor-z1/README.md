# Ease Log Ingestion Pipeline Monitor

## Overview

A responsive HTML and CSS component for monitoring log ingestion pipelines, processing delays, event failures, and overall pipeline health.

## Features

- Log ingestion metrics
- Processing latency monitoring
- Failed event tracking
- Health status indicators
- Responsive design
- Pure HTML & CSS
- CSS variables and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

A reusable component designed for observability dashboards, centralized logging systems, SIEM platforms, and cloud-native monitoring tools.
