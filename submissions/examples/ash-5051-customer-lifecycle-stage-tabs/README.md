# Customer Lifecycle Stage Tabs

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive customer lifecycle interface that organizes customer journey stages using modern tab navigation and summary cards. Suitable for CRM platforms, marketing dashboards, sales pipelines, and customer success tools.

## Features

- Lifecycle stage tabs
- Active stage indicator
- Customer metrics cards
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="tabs">

    <button class="tab active">
        Qualified
    </button>

</div>
```
