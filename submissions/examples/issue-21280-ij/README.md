# Ball Roll — Ball Rolling with Rotation

CSS ball rolling horizontally along a track. The `roll-ball` keyframe combines `left` translation with `rotate` to create a convincing roll.

## Files

- `demo.html` — rolling stage and ball
- `style.css` — horizontal translation, rotation, floor line

## Usage

Open `demo.html` to see the ball roll back and forth.
