# Improve light mode UI - enhance visual design and polish (#39432)

1. What does this do? Improves visual hierarchy, contrast (meeting WCAG AA), button variants, depth, and loading spinners in light mode.
2. How is it used? Apply `./style.css` which overrides core light theme tokens, adds shadows and borders to normal/ghost buttons, and fixes loading indicator colors.
3. Why is it useful? It ensures the framework looks premium, accessible, and holds to standard WCAG AA contrast ratios in light mode.
