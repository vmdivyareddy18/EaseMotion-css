# Marquee Logo Cloud Component

## What does this do?

Creates an infinitely scrolling logo cloud used for showcasing customers, partners, sponsors, or technologies.

## Usage

```html
<div class="marquee-wrapper">
    <div class="marquee-track">
        <div class="logo">Brand</div>
    </div>
</div>
```

## Why does it fit EaseMotion CSS?

The component demonstrates smooth motion-driven UI design using pure CSS. It is lightweight, reusable, responsive, and includes accessibility support through reduced-motion preferences.
