# CSS Slide-Right Accordion for SaaS Showcase Layouts

## Overview

A responsive accordion designed for SaaS landing pages and dashboards. Built entirely with HTML and CSS using semantic elements and a smooth slide-right animation.

---

## Features

- Pure HTML5 & CSS3
- CSS-only slide-right animation
- Responsive design
- Semantic `<details>` and `<summary>`
- Keyboard accessible
- CSS variables
- No JavaScript required
