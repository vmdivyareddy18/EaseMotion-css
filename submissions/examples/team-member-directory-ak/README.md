# Team Member Directory

## What does this do?
A filterable grid of team member cards with search-by-name and role-based filtering.

## How is it used?
Open demo.html in a browser. Type a name in the search box or select a role from the dropdown to filter the team grid.

## Why is this useful?
Demonstrates a common directory UI pattern with live client-side filtering and responsive card layout.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
