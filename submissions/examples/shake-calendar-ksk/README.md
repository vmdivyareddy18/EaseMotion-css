# Animated Calendar - Shake Style (`shake-calendar-ksk`)

1. What does this do?  
Displays a console calendar where hovering day cells triggers a playful jiggle/vibration animation and the selected date pulses continuously mimicking audio waveforms.

2. How is it used?  
Nest a 7-column calendar grid inside a `.calendar-console` container using `.calendar-day` items.

3. Why is it useful?  
It showcases bouncy, organic micro-interactions and periodic alerts suitable for music player screens or alarm systems.
