# Animated Cloud Upload Float

## Overview

A reusable cloud upload animation featuring a gentle floating effect suitable for file uploads, backups, cloud storage platforms, synchronization systems, and SaaS dashboards.

## Features

- CSS-only animation
- Cloud upload float effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="upload-button">
  <span class="upload-icon">☁️⬆️</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
