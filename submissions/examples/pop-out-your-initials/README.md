# Pop Out Animation

## What does this do?
An intermediate-level exit animation that makes an element briefly pop larger for visual emphasis before quickly shrinking to zero and fading out completely.

## How is it used?
Apply the animation to any class that handles closing or dismissing components like cards, notifications, or modals.

## Why is it useful?
It creates an organic, snappy micro-interaction that provides immediate tactile feedback when an object exits the screen.
