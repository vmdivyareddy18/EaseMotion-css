Manage and configure your active dashboard integrations.

Hover over the collapsed panels to see the Scale-Hover effect. The entire card container slightly scales up (transform: scale(1.02)) while elevating its box-shadow, providing a highly tactile, physical response common in modern app interfaces.

State management is handled purely natively using the hidden CSS radio button hack. When a panel is active, the CSS :has() selector detects this and seamlessly updates the parent card's border color to match the primary dashboard accent blue.

Height transitions are achieved using CSS Grid (grid-template-rows: 0fr to 1fr). This ensures fluid, jank-free animations regardless of how much content is nested inside, making it robust for dynamic dashboard data injection.
