# Typewriter Text Effect

An animated typewriter effect that types out phrases character by character, then deletes and types the next one. The cursor blinks with a CSS animation. The text rotates through a set of phrases in an infinite loop with a pause between each.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Watch the text type itself out on a dark background.

## Accessibility notes

Reduced motion disables the cursor blink animation. The typewriter effect itself is driven by JavaScript.
