# Scroll Triggered Animation Demo

## Description

This demo shows how the Intersection Observer API can trigger CSS animations when elements enter the viewport.

## Files

- demo.html
- style.css

## Features

- Scroll-triggered animations
- Smooth fade and slide effect
- Uses the Intersection Observer API
- Lightweight implementation

## How to Run

Open `demo.html` in a browser and scroll down the page. Each card animates into view when it becomes visible.

## Benefits

- Better performance
- Modern animation approach
- Improved user experience
