# Radial Particle Burst Animation Utility

## Overview
This utility creates a particle explosion effect where glowing particles burst outward radially from a central core.

## Usage

```html
<div class="ease-radial-particle">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first interaction
- Great for celebrations, loaders, hero sections, and game-inspired UI effects
