# Dismissible Alert

A simple dismissible alert component with smooth slide-out animation.

## Features

- Success, error, warning, and info alert types
- Color-coded alert styles
- Close button for each alert
- Smooth slide-out transition
- CSS custom property used for slide distance

## Files

- `index.html` - Alert structure
- `style.css` - Alert styling and animation
- `script.js` - Dismiss functionality
