CSS-Only Accordion

Expandable FAQ sections using native <details> + <summary>. No JavaScript.

What is EaseMotion CSS?
EaseMotion CSS is a human-readable, animation-first CSS framework. It provides a rich set of utility classes for entrance animations, hover effects, transitions, and UI components — all with zero dependencies and no build step required.
Do I need JavaScript to use this accordion?
No! This accordion is powered entirely by the native HTML <details> and <summary> elements, which provide toggle behaviour built into every modern browser. CSS handles all the visual styling and animation.
Can I customise the colours and animation speed?
Absolutely. The accordion uses EaseMotion's design tokens like --ease-color-primary and standard transition durations. Override those CSS variables in your own stylesheet to instantly retheme the accordion to match your brand colours and preferred animation speed.
Is this component accessible?
Yes. Because we use the semantic <details> element, screen readers natively understand the open/closed state. The toggle is keyboard-accessible via the Enter and Space keys with no additional ARIA attributes required.
