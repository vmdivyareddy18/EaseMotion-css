# Sidebar Navigation Hover Contrast Fix

## What does this do?

This update improves the visibility of sidebar navigation items when hovered in **Light Mode**. It addresses the low-contrast hover styling that made the topic text difficult to read.

## How is it used?

1. Open the EaseMotion CSS documentation website in **Light Mode**.
2. Hover over any sidebar navigation item.
3. The updated hover state provides a darker text color and a more distinguishable background, making the navigation easier to read.

## Why is it useful?

* Improves text readability and accessibility.
* Provides better visual feedback during navigation.
* Enhances the overall user experience by making sidebar topics clearly visible on hover in Light Mode.
