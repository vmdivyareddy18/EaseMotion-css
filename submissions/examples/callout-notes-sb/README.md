# Callout Note Blocks

Documentation-ready callout blocks for tips, notes, warnings, danger states, and success messages.

## Features

- Includes primary, note, warning, danger, and success variants.
- Uses a shared `.ease-callout` base class with variant accent variables.
- Pairs an icon slot with a structured title/body layout.
- Responsive spacing keeps long documentation notes readable.
- No JavaScript required.

## Usage

Open `demo.html` to view every variant. Use `.ease-callout` with one of the variant classes, then add `.ease-callout-icon`, `.ease-callout-body`, and `.ease-callout-title` inside it.
