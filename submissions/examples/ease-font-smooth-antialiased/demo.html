Premium Typography Smoothing

Using -webkit-font-smoothing: antialiased to prevent dark backgrounds from artificially fattening light text on macOS and iOS devices.

Default Browser Rendering

The Default Rendering

By default, browsers use subpixel antialiasing. On dark backgrounds, especially on Mac/iOS, this causes the white text to bleed into the dark background, making the font look artificially bold and slightly muddy.

Look closely at the edges of these letters. They might appear slightly heavier than intended.

Antialiased (Premium)

The Premium Rendering

By applying grayscale antialiasing, we stop the subpixel color bleeding. The text appears noticeably thinner, crisper, and exactly as the type designer originally intended it to look.

This is the secret sauce used by Apple, Vercel, and Stripe to make their dark modes look incredibly sleek.

Note: This effect is most visible on macOS and iOS devices with Retina displays. Windows handles font rendering differently natively.
