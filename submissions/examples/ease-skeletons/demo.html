CSS Token Reference

TokenLight defaultDark defaultPurpose
--ease-skeleton-base #e2e8f0 #313244 Background base color
--ease-skeleton-shine #f8fafc #45475a Shimmer highlight color
--ease-skeleton-radius 6px 6px Border radius
--ease-skeleton-duration 1.6s 1.6s Shimmer sweep duration
How the shimmer works: A linear-gradient(90deg) with a bright midpoint is set as background-image. The gradient is wider than the element (background-size: 200%) and background-position is animated from -200% to 200% — creating the left-to-right sweep effect.

prefers-reduced-motion: Animation is disabled and a solid base color is shown — static placeholder, no motion.
