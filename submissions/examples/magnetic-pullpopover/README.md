# CSS Magnetic Pull Popover

A responsive and accessible popover component that uses a smooth magnetic pull animation with pure HTML and CSS.

No JavaScript or external dependencies are required.

## Features

- Pure HTML and CSS implementation
- Smooth magnetic pull hover interaction
- Native HTML Popover API
- Keyboard-accessible controls
- Visible focus states
- Responsive layout
- Reduced-motion support
- Forced-colors accessibility support
- Customizable animation properties
- Zero JavaScript dependencies
