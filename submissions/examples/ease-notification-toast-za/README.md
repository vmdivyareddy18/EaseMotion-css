# CSS Notification Toast

## What does this do?
CSS notification toast component with info/success/warning/error variants, slide-in animation, dismiss button, and action buttons.

## How is it used?
```html
<div class="nt-toast nt-info"><div class="nt-icon">ℹ</div><div class="nt-body"><strong>Title</strong><p>Message</p></div><button class="nt-dismiss">&times;</button></div>
```

## Why is it useful?
Provides toast notification UI patterns without JavaScript. Four severity types with color-coded left borders. Slide-in animation. Stackable layout. Dismiss button. Optional action buttons. Position variants. Customizable via CSS variables.
