# Animated Refresh Spinner Pulse

## Overview

A reusable refresh indicator animation that combines smooth rotation with a subtle pulse effect.

## Features

- CSS-only implementation
- Continuous spin animation
- Gentle pulse effect
- Lightweight micro-interaction
- Responsive demo
- Accessibility support

## Usage

```html
<button class="refresh-button">
  <span class="refresh-icon">⟳</span>
  Refresh
</button>
```

## Accessibility

Supports:

```css
@media(prefers-reduced-motion: reduce)
```

to disable animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
