Image Rendering Utilities

Pixel scaling controls proposed for core/utilities.css.

These utility classes control how the browser scales images up or down. This is particularly useful for crisp pixel art, QR codes, or low-resolution textures.

.rendering-auto (Default)

The browser uses its default algorithm (usually bilinear interpolation), resulting in a blurred image when scaled up.

.rendering-pixelated

The browser preserves sharp edges (nearest-neighbor), resulting in a crisp, blocky image when scaled up.
