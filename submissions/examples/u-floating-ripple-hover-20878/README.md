# Floating Ripple Hover

A reusable CSS hover animation that combines a gentle floating motion with an expanding ripple effect. The ripple originates from the center of the element and fades outward, creating a smooth, modern micro-interaction without JavaScript.

## Features

* 🌊 Expanding ripple hover animation
* ⬆️ Smooth floating hover effect
* 🎨 Pure CSS implementation
* ⚡ Lightweight and reusable
* 📱 Responsive design
* 🔧 Easy to customize

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Apply the class to any interactive element:

```html
<button class="floating-ripple-hover">
    Hover Me
</button>
```

The effect also works with cards and icons:

```html
<div class="card floating-ripple-hover">
    Interactive Card
</div>
```

## Customization

You can customize:

* Floating distance
* Ripple size
* Ripple duration
* Background colors
* Border radius
* Shadow effects

Example:

```css
.floating-ripple-hover:hover {
    transform: translateY(-12px);
}
```
