# Progress Step Glow

A three-step progress row where the active step glows while completed steps remain steady.

## Files

- `demo.html` - ordered progress step markup.
- `style.css` - completed step styling, active glow, and reduced-motion fallback.

## Highlights

- Uses semantic ordered list markup.
- Keeps each step in a stable grid cell.
- Stops active glow motion for reduced-motion users.
