# FAQ Section for CONTRIBUTING.md (Issue #1737)

This submission provides the proposed FAQ section for the `CONTRIBUTING.md` file as requested in [Issue #1737](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/1737).

## What does this do?
It includes `FAQ.md`, which contains answers to the most common questions external contributors have (e.g., why their PRs get automatically closed, how to properly submit bug fixes, and what files are required). 

## How is it used?
Since external contributors are restricted from modifying the `CONTRIBUTING.md` file directly (doing so causes the automated PR validator to immediately close the PR), this FAQ content is submitted here. The repository maintainer can manually review `FAQ.md` and append its contents to the core `CONTRIBUTING.md`.

## Why is it useful?
Adding a FAQ section drastically reduces confusion among new contributors, lowers the number of closed/rejected PRs, and makes the strict repository rules easier to understand at a glance.
