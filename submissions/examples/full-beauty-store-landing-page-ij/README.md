# Full Beauty/Skincare Store Landing Page

1. **What does this do?** An elegant beauty and skincare e-commerce landing page featuring product collections, bestsellers with ratings, ingredient spotlights, and testimonials in a soft rose/pink theme built entirely with EaseMotion CSS framework classes.

2. **How is it used?** Open `demo.html` — the page uses `ease-navbar-glass` for the header, `ease-card-glass`/`ease-card-hover` for product and collection cards, `ease-reveal-up`/`ease-reveal-scale` with stagger delays for scroll animations, `ease-hover-glow`/`ease-hover-lift` for product card interactions, `ease-slide-up`/`ease-fade-in` for entrance animations, `ease-grid-cols-2`/`ease-grid-cols-3`/`ease-grid-cols-4` for responsive grids, `ease-badge` for bestseller tags, `ease-btn-pill`/`ease-btn-primary` for CTAs, `ease-chip` for ingredient tags, and `ease-footer` for the footer.

3. **Why is it useful?** Demonstrates EaseMotion building a beauty brand experience with glassmorphism cards, zoom-in product reveals, and shimmer hover effects — all framework-native, proving EaseMotion handles soft, aesthetic UIs as well as technical ones.
