# Machine Learning Engineer Portfolio Layout

This submission provides a complete, production-ready portfolio template designed specifically for Machine Learning Engineers, Data Scientists, and AI Researchers. It relies entirely on the `EaseMotion` CSS utility framework to deliver a technical, performant, and high-conversion layout.

## Features Included
- **Zero-JS Architecture:** Pure CSS handling of all interactions, layout math, and grid logic.
- **Custom CSS Neural Network Hero:** A lightweight, math-based `@keyframes` implementation of data nodes connecting, built exclusively with CSS elements.
- **Production Grid System:** Utilizes `ease-grid` for responsive mapping of academic publications, GitHub contribution mocks, and project deployments (NLP, CV, RecSys).
- **EaseMotion Integrations:** Fully implements `ease-hover-lift`, `ease-animate-slide-up`, and `ease-hover-glow` for interaction feedback.
- **High-Contrast Dark Theme:** Optimized for long technical reading with inclusive, AAA-contrast color tokens.

## Running Locally
This layout is completely self-contained. There is no build step.
1. Simply clone this directory.
2. Open `demo.html` directly in your favorite web browser (loads via `file:///` protocol).

## EaseMotion Classes Showcased
* `ease-animate-slide-up` (Staggered entrance animations)
* `ease-hover-glow` (Used on project cards)
* `ease-hover-lift` (Used on tech stack badges)
* `ease-text-gradient` (Used for hero typography)
* Grid layouts: `ease-grid`, `ease-grid-cols-3`, `ease-mobile-grid-1`

## Developer Notes
All classes in `style.css` are heavily vendor-prefixed to pass automated CSS linters. The UI relies strictly on standard web fonts to ensure no external network requests are made, keeping the template privacy-first.
