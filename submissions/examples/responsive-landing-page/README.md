# Responsive Landing Page Demo

## Purpose

This example demonstrates how multiple EaseMotion CSS animation classes can be combined to build a modern responsive landing page.

---

## Sections Included

- Hero
- Features
- Statistics
- Call-to-Action
- Footer

---

## Folder Structure

```
submissions/examples/responsive-landing-page/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to test responsiveness.
3. Observe how the EaseMotion animation classes enhance each section.
