# Breadcrumb Collapse

## What does this do?
Creates a collapsing breadcrumb navigation where long paths gracefully truncate with an animated ellipsis expander.

## How is it used?
Apply the `.collapsible-breadcrumb` class to a nav element with `.crumb` items. Long breadcrumbs show an animated ellipsis to indicate hidden sections.

## Why is this useful?
Handles deep navigation paths gracefully without overflow, keeping the UI clean while maintaining context of the current page location.
