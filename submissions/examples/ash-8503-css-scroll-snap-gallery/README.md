# CSS Scroll Snap Gallery

A pure HTML and CSS demonstration of **CSS Scroll Snap** with horizontal and vertical galleries, custom scrollbars, pagination dots, and keyboard-friendly focus management.

## Overview

This example demonstrates how to build accessible scrollable galleries using the CSS Scroll Snap specification without JavaScript. It showcases horizontal and vertical scrolling experiences, customized scrollbars, visual pagination indicators, and keyboard navigation through focusable containers.

## Features

- Horizontal scroll-snap gallery
- Vertical scroll-snap gallery
- `scroll-snap-type` on both x and y axes
- `scroll-snap-align` for smooth snapping
- Custom CSS scrollbars
- Pagination dot indicators
- Keyboard focus management (`tabindex`)
- Responsive layout
- Pure HTML & CSS (No JavaScript)
