# ease-hover-skew-left-isk

## What does this do?
Slightly skews the element to the left on hover for a dynamic, energetic lean effect.

## How is it used?
Add the class to any hoverable HTML element like a button or card:

```html
<button class="ease-hover-skew-left-isk">Hover Me</button>
```

You can customize the skew angle by overriding the `--ease-skew-angle` CSS variable:

```css
.custom-element {
  --ease-skew-angle: -8deg;
}
```

## Why is it useful?
Skew-based hover interactions add physical energy and dynamism to otherwise static UI elements. It is an excellent choice for buttons, navigation links, and call-to-action cards, providing interactive depth without disrupting the overall layout.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
