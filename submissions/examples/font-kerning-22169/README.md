# Font-Kerning Utilities
This submission introduces utilities to control the `font-kerning` property, which allows developers to toggle the use of kerning information stored in fonts.

- `.kerning-auto`: Browser decides whether to apply kerning (default).
- `.kerning-normal`: Forces the browser to use kerning info.
- `.kerning-none`: Disables kerning entirely.

Proper kerning is essential for professional typography, especially with character combinations like 'AV' or 'WA'.
