# Interactive HTML Code Generator for Components

An interactive development utility designed to streamline developer workflows within the EaseMotion CSS ecosystem. This generator lets users visually toggle component classes—spanning element size metrics, entrance behaviors, structural depth shadows, and interactive states—and view live updates on a unified preview canvas while instantly generating the exact composable utility HTML markup markup strings.

## Features
- **Real-Time Reactive Previews:** Instant visual rendering sync across component card states as options get modified.
- **Granular Utility Configuration Controls:** Supports deep selection vectors including button/card wrappers, sizing thresholds, transform modifiers, and delay multipliers.
- **One-Click Clipboard Export:** Built-in copy utilities using native browser Clipboard APIs for swift component exporting.
- **Production-Ready Theme footprint:** Deep workspace layout design curated to feel right at home inside elite documentation hubs.
