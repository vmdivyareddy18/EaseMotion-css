# ease-slide-fade-right

A smooth entrance animation where an element fades into view while sliding from the left to its original position.

## Features
- Fade-in effect
- Left-to-right slide motion
- Lightweight and reusable
- Pure CSS animation

## Files
- demo.html
- style.css
- README.md

## Usage

```html
<div class="ease-slide-fade-right">
  Welcome to EaseMotion CSS
</div>
```

## Animation Details
- Starts at `opacity: 0`
- Moves from `translateX(-30px)`
- Ends at `opacity: 1`
- Ends at `translateX(0)`
- Duration: `0.8s ease`
