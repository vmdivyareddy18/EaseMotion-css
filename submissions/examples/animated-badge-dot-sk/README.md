# Animated Badge Dot

Status badge dot indicators (online/offline/away/busy) with pulse animation. Available in three sizes (small, medium, large) and positioned on avatar/icon mockups.

Created by @saurabhhhcodes

## Features
- Four status states with distinct colors
- Pulse animation for live statuses
- Three sizes: small (8px), medium (12px), large (16px)
- Avatar mockup with absolute-positioned badge
- BEM naming convention

## Status Colors
- Online — Green with pulse
- Offline — Gray, static
- Away — Amber with pulse
- Busy — Red with pulse
