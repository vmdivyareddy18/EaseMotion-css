🎨 Theme Skin System

Buttons — themed via CSS custom properties

Cards — surface / border / shadow inherit theme

📦

Card Surface

Background, border, and shadow adapt to the active data-theme. Hover lifts with a theme-appropriate shadow.

🎯

Theme Variables

Uses --surface, --border, --text, --shadow — switch any theme and all cards re-theme instantly.

🌀

No Flicker

Because CSS custom properties cascade via data-theme attribute, switching is instant with zero layout shift.

Alerts — semantic color tokens

✅ Success — --success token colors the left border.
⚠️ Warning — --warning token adapts per theme.
❌ Error — --error token for destructive alerts.
ℹ️ Info — --info token for informational messages.

Usage

Set data-theme on any container element. All children inherit the theme via CSS custom properties.

document.documentElement.setAttribute('data-theme', 'ocean')
