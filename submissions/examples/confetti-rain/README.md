# Confetti Rain Effect

**What is it?**
A confetti rain celebration scene with 25 multicolored confetti pieces continuously falling against a gradient purple background.

**How it works**
Each `.confetti` div uses CSS custom properties for color (`--c`), width (`--w`), height (`--h`), duration (`--d`), and delay (`--del`). The `confetti-fall` keyframe drops them with rotation, scaling, and fade-out.

**Why this approach**
Pure CSS with custom properties enables varied confetti behavior without JavaScript. The staggered delays and different sizes create an organic, festive rain effect.
