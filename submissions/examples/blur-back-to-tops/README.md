# Blur Back to Top

A CSS-only **Blur Back to Top** component inspired by Space design patterns.

## Features

- Pure CSS
- Floating blur animation
- Glassmorphism effect
- Responsive layout
- Accessible button
- No JavaScript required

## Usage

```html
<a href="#" class="back-top" aria-label="Back to top">
  ↑
</a>
```

Include `style.css` in your project.

- `demo.html` – Live demonstration
