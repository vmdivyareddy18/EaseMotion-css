# Checkbox & Radio Group

Custom styled checkboxes and radio buttons with animated selection indicators.

## Features

- Custom checkbox checkmark scales in on `:checked`
- Custom radio dot scales in on `:checked`
- Accessible — native inputs are visually hidden but remain focusable
- `:focus-visible` outline for keyboard users

## Custom Properties

| Property              | Default  | Description              |
|-----------------------|----------|--------------------------|
| `--cr-duration`       | `0.25s`  | Animation duration       |
| `--cr-checked-bg`     | `#6366f1`| Background when checked  |
| `--cr-unchecked-bg`   | `#fff`   | Background when unchecked|
| `--cr-border-color`   | `#cbd5e1`| Default border color     |
| `--cr-checkmark-color`| `#fff`   | Checkmark / dot color    |
| `--cr-radius`         | `6px`    | Checkbox border radius   |
