# ease-sidebar-collapse

An animated collapsible sidebar navigation. Smoothly collapses to icon-only width and expands back with text label fade-out.

## Usage

Include the `style.css` in your project and use the HTML structure with `.sidebar` and `.sidebar-nav`. Toggle is controlled by adding/removing the `.collapsed` class.

## Demo

Open `demo.html` in a browser to see the component in action.
