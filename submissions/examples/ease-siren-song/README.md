# ease-siren-song

An enchanting siren-inspired animation set for EaseMotion CSS.

## Usage

```html
<!-- Hypnotic sway -->
<div class="ease-siren-song">🧜‍♀️</div>

<!-- Glowing text -->
<div class="ease-siren-glow">Come closer...</div>

<!-- Sound ripple -->
<div class="ease-siren-ripple">🎵</div>

<!-- Color enchant -->
<div class="ease-siren-color">✨</div>

<!-- Combined enchant -->
<div class="ease-siren-enchant">Enchanted</div>
```

## Speed Variants

```html
<div class="ease-siren-song ease-siren-slow">...</div>
<div class="ease-siren-song ease-siren-fast">...</div>
```

## Classes

| Class | Description |
|---|---|
| `ease-siren-song` | Hypnotic sway animation |
| `ease-siren-glow` | Enchanting color glow pulse |
| `ease-siren-ripple` | Sound wave ripple effect |
| `ease-siren-color` | Hue-rotate color enchant |
| `ease-siren-enchant` | Combined sway + glow |
| `ease-siren-slow` | Slow speed variant |
| `ease-siren-fast` | Fast speed variant |

## Features
- Hypnotic swaying motion
- Enchanting glow text effect
- Sound wave ripple
- Color shift enchantment
- Combined enchant variant
- Speed variants
- Respects prefers-reduced-motion
- Uses EaseMotion CSS design tokens
