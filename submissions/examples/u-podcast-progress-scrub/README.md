# ease-podcast-progress-scrub

Podcast and audio player progress bar with enhanced hover scrubbing experience.

## Features

- Enlarged progress bar on hover
- Scalable scrub thumb
- Hover preview tooltip
- Responsive layout
- Pure HTML and CSS

## Acceptance Criteria Covered

### Progress Bar Hover

The progress bar increases in height when hovered for easier scrubbing.

### Thumb Hover Animation

The scrub thumb scales up smoothly on hover.

### Hover Preview

Displays a timestamp preview tooltip above the scrub point.

## Use Cases

- Podcast players
- Audio streaming apps
- Music players
- Learning platforms
- Video players
