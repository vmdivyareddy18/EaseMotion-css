# Animated Notification Badge (`ease-notification-cv`)

A lightweight notification badge component built using **HTML5** and **Pure CSS**.

## Features

- 💓 Pulse animation
- 🏀 Bounce animation
- ✨ Glow effect
- 🔢 Number badge variants
- 📱 Responsive
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-notification-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
