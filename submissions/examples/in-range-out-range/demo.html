:in-range / :out-of-range

Number inputs with defined min/max attributes. Values within range get a green indicator; values outside range get a red warning. Try entering numbers below or above the allowed range.

Age Input

Enter a value between 18 and 120

Rating Input

Rate from 1 to 10

Temperature Input

Range: -20°C to 50°C

Discount Percentage

Enter a percentage between 0 and 100

Validation Notes

:in-range applies when the value is within the min/max bounds (green border). :out-of-range applies when the value is outside (red border + shake). Browser-native validation bubbles also appear on form submission attempts.
