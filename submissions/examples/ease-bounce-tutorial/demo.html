ease-bounce

An infinite vertical bounce animation — zero config, one class.

Step 1 — See it live

Three flavours of bounce

All three balls below use the same ease-bounce class — only speed and height differ via a custom property override.

Step 2 — Install

Add the CDN link

Paste this inside your <head>. No build step, no npm.

<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/SAPTARSHI-coder/EaseMotion-css@main/easemotion.min.css" />
Step 3 — Use it

Apply the class

Add ease-bounce to any element. Works on divs, images, icons — anything.

<!-- basic -->
<div class="ease-bounce">🎾</div>

<!-- slower -->
<div class="ease-bounce" style="--bounce-dur: 1.6s;">🐢</div>

<!-- higher -->
<div class="ease-bounce" style="--bounce-height: 80px;">🚀</div>

Step 5 — When to use it

Good use cases

🔔
Notifications
Draw attention to a badge or alert icon without being jarring.
⬇️
Scroll cues
A bouncing chevron at the bottom of a hero section guides users to scroll.
🎮
Games / fun UIs
Characters, mascots, or playful loading states that need energy.
🛒
Cart / CTA buttons
Brief bounce on add-to-cart confirms the action with delight.
