# Page Pagination Nav

A page-turn pagination navigation with animated transitions.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--nav-bg` | `#2d2d44` | Navigation bar background |
| `--page-color` | `#888` | Inactive button color |
| `--active-color` | `#4ecdc4` | Active button color |
| `--turn-duration` | `0.4s` | Page turn animation speed |
