# Elastic Bounce Cyberpunk Neon Modal

A responsive Cyberpunk Neon modal featuring a smooth Elastic Bounce entrance animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Cyberpunk neon interface
- Elastic Bounce animation
- Neon glow effects
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--scale
--bg
--surface
--primary
--secondary
--text
--radius
```
