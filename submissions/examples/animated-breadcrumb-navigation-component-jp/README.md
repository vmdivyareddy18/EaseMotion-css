# Animated Breadcrumb Navigation Component

## Overview

A modern CSS-only breadcrumb navigation component with smooth hover effects, animated separators, and active page highlighting.

## Features

- Smooth hover animations
- Animated breadcrumb separators
- Active page highlighting
- Responsive and lightweight design
- CSS-only implementation
- Easy integration into websites and dashboards

## Files

- demo.html – Demonstrates the animated breadcrumb component
- style.css – Contains styles, transitions, and animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in any modern browser to see the animated breadcrumb navigation.

Hover over breadcrumb links to experience smooth interaction effects.
