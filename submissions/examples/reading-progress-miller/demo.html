Reading Progress Bar

Scroll down to see the progress bar fill. Use the buttons below to switch variants.

How it works

A single div fixed at the top of the page. Its width is updated via one line of JavaScript on scroll, reflecting how far through the page the user has scrolled.

The gradient uses EaseMotion's primary and secondary color tokens. A subtle glow shadow gives it depth against any background.

Variants

Four variants available: default (purple-pink-amber), cool (cyan-indigo-violet), warm (amber-red-pink), and flat (no glow). A thick variant doubles the bar height for higher visibility.

Accessibility

Respects prefers-reduced-motion — the smooth transition is disabled for users who prefer reduced motion, while the bar still tracks scroll position accurately.
