# animated-comment-thread-sk

A nested comment thread with 2-3 levels of replies. Each comment has a gradient initials avatar, username, timestamp, text content, and action row (Like/Reply).

## Interactions
- **Hover**: Comment row highlights with a background change, and the avatar pulses subtly.

## Structure
- Depth-1 and depth-2 comments are indented with a left border connector for visual hierarchy.

## Features
- Pure CSS, no JavaScript
- Dark theme
- Respects `prefers-reduced-motion`
