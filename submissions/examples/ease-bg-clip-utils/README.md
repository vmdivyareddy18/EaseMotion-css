# ease-bg-clip utilities

Controls `background-clip` to define how far the background extends.

## Usage

```html
<div class="ease-bg-clip-text ease-bg-primary">Text with clipped gradient</div>
```

## Classes

| Class                  | CSS Property                               |
| ---------------------- | ------------------------------------------ |
| `ease-bg-clip-border`  | `background-clip: border-box`              |
| `ease-bg-clip-padding` | `background-clip: padding-box`             |
| `ease-bg-clip-content` | `background-clip: content-box`             |
| `ease-bg-clip-text`    | `background-clip: text` + transparent text |
