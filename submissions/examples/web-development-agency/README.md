# Web Development Agency Landing Page

## What does this do?

This is a complete, ready-to-copy B2B web development agency landing page built entirely with EaseMotion CSS utility classes, components, and layout systems.

## How is it used?

Link `easemotion.css` (or `easemotion.min.css`) in your project's `<head>` and use the pre-configured layout and animation classes:

```html
<!-- Link to EaseMotion CSS bundle -->
<link rel="stylesheet" href="easemotion.css" />
<!-- Page-scoped custom design classes -->
<link rel="stylesheet" href="style.css" />
<!-- Reveal logic for scroll entrance animations -->
<script src="core/reveal.js" defer></script>
```

## Why is it useful?

This landing page provides a real-world B2B agency template that developers can copy and customize. It demonstrates how to combine EaseMotion's variables, components (navbar, cards, badges), utility classes (flexbox grid, padding, margin), scroll-entrance animations, and interactive hover transitions to create a premium, responsive user interface without writing complex custom animations or relying on bloated JavaScript frameworks.

## Showcase of EaseMotion CSS Classes

This template showcases the following EaseMotion features:

- **Components**:
  - Sticky glassmorphic navbar: `.ease-navbar-glass` with `.ease-navbar-glass-sticky` and `.ease-navbar-glass-blur`
  - Cards: `.ease-card`, `.ease-card-hover`, `.ease-card-glass`
  - Badges: `.ease-badge`, `.ease-badge-primary`
  - Buttons: `.ease-btn`, `.ease-btn-primary`, `.ease-btn-outline`
- **Animations & Interactive states**:
  - Combined Pulse + Glow Hover: `.ease-hover-pulse-glow`
  - Grow on hover: `.ease-hover-grow`
  - Lift + Shadow on hover: `.ease-hover-lift` / `.ease-hover-lift-shadow`
  - Shimmer effect: `.ease-shimmer-text`, `.ease-shimmer-sweep`
  - Floating element keyframes: `.ease-float`
  - Text underline hover: `.ease-hover-underline`
  - Scroll entrance reveal classes: `.ease-reveal`, `.ease-reveal-up`, `.ease-reveal-slide-left`
  - Stagger delay helpers: `.ease-reveal-delay-1`, `.ease-reveal-delay-2`, `.ease-reveal-delay-3`, `.ease-reveal-delay-4`
- **Utilities**:
  - Flexbox and Alignment: `.ease-flex`, `.ease-items-center`, `.ease-justify-center`, `.ease-center`
  - Grid layouts: `.ease-grid`, `.ease-grid-cols-3`, `.ease-gap-6`
  - Margins & Paddings: `.ease-mb-4`, `.ease-gap-2`
