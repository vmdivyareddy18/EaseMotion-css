# Animated Tabs Navigation

A responsive animated tabs navigation component built using pure HTML and CSS.

## Features

- Animated active tab indicator
- Smooth content transitions
- Hover interactions
- Responsive layouts
- Modern tab switching effects
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated tabs navigation component.

## Files

- `demo.html` → Tabs navigation structure
- `style.css` → Tabs styles and animations
- `README.md` → Documentation
