# Talent Intelligence Command Center

A modern, responsive Talent Intelligence Command Center dashboard built using only HTML and CSS.

## Overview

This example showcases a recruitment analytics dashboard designed for HR teams, recruiters, and talent acquisition professionals. It presents hiring metrics, recruitment pipelines, candidate information, department statistics, and skill demand in a clean enterprise-style interface.

## Features

- Responsive dashboard layout
- Professional sidebar navigation
- KPI statistic cards
- Recruitment pipeline overview
- Hiring analytics bar chart
- Skill demand progress indicators
- Recent candidates table
- Department hiring donut visualization
- Modern gradients and glass-inspired cards
- Pure HTML and CSS (No JavaScript)
