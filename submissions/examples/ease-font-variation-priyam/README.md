# Font Size Adjust and Font Stretch Utilities

## What does this do?

This submission demonstrates raw CSS utility classes for `font-size-adjust` and `font-stretch` typography control.

## How is it used?

Add one of the utility classes to the text element that needs font size adjustment or font stretch control.

```html
<p class="font-size-adjust-from-font">Font size adjust from font</p>
<p class="font-size-adjust-05">Font size adjust 0.5</p>
<p class="font-stretch-condensed">Condensed text</p>
<p class="font-stretch-expanded">Expanded text</p>
<p class="font-stretch-pct-75">75% stretched text</p>
```

## Why is it useful?

These utilities help improve typography stability and flexibility by reducing fallback font layout shifts and supporting condensed or expanded font faces, which fits EaseMotion CSS's goal of providing practical, reusable visual utilities.
