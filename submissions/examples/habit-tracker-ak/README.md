# Daily Habit Tracker

## What does this do?
A daily habit tracker that lets users add habits, check them off for each day of the week, and track weekly progress percentage with localStorage persistence.

## How is it used?
Open demo.html in a browser. Type a habit name and click Add. Check the boxes for each day you complete the habit. The progress bar shows your weekly completion rate. Delete habits with the X button.

## Why is this useful?
This provides a personal productivity tool for building and tracking daily routines. It demonstrates localStorage persistence, checkbox state management, and progress calculation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to track habits.
