## Tornado Spin Animation

A menacing tornado funnel sways and twists across the frame while debris particles orbit at different altitudes, creating a dynamic storm effect.

**Features:**
- Funnel-shaped tornado with blur and gradient
- Wobbling sway animation on the funnel
- Multiple debris particles in orbital paths
- Respects `prefers-reduced-motion`
