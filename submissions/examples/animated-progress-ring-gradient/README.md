# Animated Progress Ring (Gradient Style)

A circular animated progress ring built using HTML, CSS, SVG and JavaScript.

## Features

- Gradient progress stroke
- Smooth animation from 0 to target
- Stroke-dasharray animation
- Percentage counter
- Responsive layout
- Glow effect
- Hover scaling animation

## Technologies

- HTML5
- CSS3
- SVG
- JavaScript

## Usage

Open `demo.html` in any browser.
