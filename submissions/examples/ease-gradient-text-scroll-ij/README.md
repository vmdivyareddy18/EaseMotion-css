# Gradient Text Scroll

Animated gradient text with a shifting color background that scrolls horizontally using CSS background-position animation.

## Features

- Gradient text filled with moving background
- Multiple gradient lines with staggered animation delays
- Large heading with wide color spectrum
- Dark theme background
- Pure CSS using background-clip and background-position

## Usage

Apply `background: linear-gradient(...)` with `background-clip: text` and `-webkit-text-fill-color: transparent`. Animate `background-position` with `@keyframes` to shift the gradient across the text.
