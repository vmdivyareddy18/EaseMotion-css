# Invoice Payment Status Strip

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive invoice payment status component that displays invoice progress using compact status strips. Suitable for finance dashboards, billing portals, accounting software, ERP systems, and payment management applications.

## Features

- Invoice status strips
- Paid, Processing, Pending, and Overdue states
- Color-coded status badges
- Summary statistics
- Responsive layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="invoice paid">

    <span class="dot"></span>

    <div class="details">

        <h3>INV-1042</h3>

    </div>

    <span class="badge success">
        Paid
    </span>

</div>
```
