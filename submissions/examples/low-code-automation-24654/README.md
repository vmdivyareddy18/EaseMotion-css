# Low-Code Automation Template
A modular, visual-first template for low-code/no-code platforms, workflow automation services, and integration tools.

- **Design**: Uses a "Builder-Canvas" palette with deep logic-blues, canvas-greys, and accent-teal nodes to represent the drag-and-drop nature of low-code environments.
- **Use Case**: Workflow automation SaaS, visual API integration platforms, and business logic orchestration tools.
