# Neumorphic Soft 3D Perspective Tilt Accordion

A modern **Pure CSS 3D Perspective Tilt Accordion** designed for **Neumorphic Soft UI** layouts. The component features soft neumorphic styling, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- 3D Perspective Tilt hover animation
- Soft Neumorphic design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Smooth shadow transitions
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.45s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --bg:#e9eef5;
    --surface:#e9eef5;

    --accent:#5b7cfa;
    --radius:20px;
}
```

You can customize:

- Animation duration
- Animation easing
- Background color
- Accent color
- Border radius
- Shadow intensity

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
