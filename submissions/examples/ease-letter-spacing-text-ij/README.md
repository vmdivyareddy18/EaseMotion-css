# Letter Spacing Text

Text with animated letter-spacing on hover, creating a breathing expand/collapse effect using CSS transitions.

## Features

- Letter-spacing expands smoothly on hover
- Color transitions from white to accent color
- Subtitle fades out when heading is hovered
- Multiple demo lines with individual hover effects
- Bounce cubic-bezier timing for natural feel

## Usage

Apply `transition: letter-spacing 0.6s cubic-bezier(...)` and change `letter-spacing` on `:hover`. Works on any text element for a drifting-apart animation.
