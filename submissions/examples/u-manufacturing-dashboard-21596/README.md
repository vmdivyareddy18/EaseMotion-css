# Agriculture Farm Dashboard

A responsive **Agriculture Farm Dashboard** built using **HTML5** and **CSS3** that demonstrates crop monitoring, irrigation management, weather tracking, and harvest progress through an interactive farming dashboard.

---

## Features

- Responsive dashboard layout
- Farm overview cards
- Crop growth progress tracker
- Water usage monitoring
- Weather overview panel
- Recent farm activity feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Active Fields
- Crop Health
- Irrigation Status
- Harvest Progress

### Farm Widgets

- Crop Growth
- Water Usage
- Weather Overview

### Activity

- Recent Farm Activity
- Notifications Panel

### Quick Actions

- Add Crop
- Schedule Irrigation
- Generate Report

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
