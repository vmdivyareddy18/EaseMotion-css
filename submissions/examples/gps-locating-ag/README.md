# GPS Locating

## What does this do?
A GPS location finding pulse animation with concentric expanding rings to indicate active location detection.

## How is it used?
Use the `.gps-locating` class on a container. Multiple expanding rings pulse outward from the center pin to indicate searching for location.

## Why is this useful?
Provides clear spatial feedback for location-aware features, making the device's positioning activity visible to the user.
