# Bounce Tabs

Tab switching component with a bouncing indicator and panel animation using the CSS radio hack.

## Features

- Sliding tab indicator with bounce cubic-bezier transition
- Tab panel content animates with bounce entrance
- Active tab label color highlights
- Pure CSS using hidden radio inputs for tab state
- Dark theme design

## Usage

Include `style.css` and structure tabs with hidden radio inputs, `.tab-label` elements, and `.tab-panel` content sections. The checked radio determines the active tab.
