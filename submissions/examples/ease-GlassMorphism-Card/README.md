# Glass Card

**What does this do?**
Creates a modern glassmorphism card using a semi-transparent background, backdrop blur, frosted border, and soft shadowing. The effect mimics frosted glass, allowing background colors and gradients to subtly show through the component.

**How is it used?**

```html
<div class="glass-card">
  <h2>Premium Workspace</h2>

  <p>
    Build beautiful interfaces with a clean,
    modern glassmorphism aesthetic.
  </p>

  <button>Get Started</button>
</div>
```

**Why is it useful?**
Glassmorphism has become one of the most popular UI design trends across dashboards, landing pages, mobile applications, and SaaS products. Developers frequently recreate this pattern from scratch, making it a strong candidate for a reusable EaseMotion CSS component.

The effect improves visual hierarchy while maintaining a lightweight and modern appearance. Because it relies only on CSS properties such as `backdrop-filter`, transparency, and subtle borders, it remains easy to use and highly customizable.
