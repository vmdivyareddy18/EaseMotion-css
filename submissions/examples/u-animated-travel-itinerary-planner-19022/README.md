# Animated Travel Itinerary Planner

A modern travel itinerary planner featuring animated trip cards, travel information panels, and responsive layouts.

## Features

- Animated itinerary cards
- Day indicators
- Hotel and flight information
- Budget overview
- Hover interactions
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any browser.

## Customization

- Destinations
- Travel dates
- Hotel details
- Flight information
- Budget values
- Colors and animations

## Use Cases

- Travel planning apps
- Tourism websites
- Trip management dashboards
- Vacation planners
- Travel agencies
