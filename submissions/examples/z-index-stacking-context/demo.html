Stacking context via opacity

Child (z: 99)
opacity: 1
Child (z: 99)

Opacity < 1 creates a new stacking context — the child's z-index is scoped to its parent.

Stacking context via transform

No transform (z: 10)
transform: scale(1) (z: 5)
transform: rotate(2deg) (z: 1)

Elements with transform also create a stacking context, restricting child z-index values.
