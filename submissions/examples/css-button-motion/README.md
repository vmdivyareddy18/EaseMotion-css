# 🔘 CSS Button Motion

A comprehensive collection of beautiful, pure CSS animated buttons for modern web applications.

## 🚀 Features

### Gradient Buttons (4 styles)
- Purple gradient
- Pink gradient
- Blue gradient
- Green gradient

### Neon Buttons (4 colors)
- Neon Blue
- Neon Pink
- Neon Green
- Neon Purple

### 3D Buttons (4 colors)
- 3D Red with depth effect
- 3D Blue with depth effect
- 3D Green with depth effect
- 3D Orange with depth effect

### Outline Buttons (4 styles)
- Solid outline with fill
- Rounded outline with fill
- Square outline with fill
- Dashed outline with fill

### Special Effect Buttons (6 effects)
- Ripple Effect - Material design ripple
- Shine Effect - Light sweep animation
- Pulse Effect - Continuous pulse glow
- Bounce Effect - Bouncy hover
- Slide Fill - Background slide animation
- Rotate Icon - Rotating icon on hover

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All button animations
- `readme.md` - Documentation
