# Bounce Background Hover

**What does this do?**  
Adds a hover effect where the element background bounces in as a soft, elastic color layer behind the content.

**How is it used?**  
```html
<button class="bounce-bg-button">
  Save changes
</button>

<article class="bounce-bg-tile">
  <h2>Live crop insight</h2>
  <p>Hover to reveal the bouncing background.</p>
</article>
```

**Why is it useful?**  
It gives buttons, cards, and feature tiles an energetic hover state without JavaScript. The effect uses a pseudo-element so content stays readable, and the colors can be changed through CSS variables before the maintainer standardizes it for EaseMotion CSS.
