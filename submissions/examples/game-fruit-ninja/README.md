# 🍉 Fruit Ninja

A classic Fruit Ninja arcade game built with HTML, CSS, and JavaScript. Swipe to slice flying fruits, avoid bombs, and build massive combos!

## 🎮 How to Play

1. Click **"Start Game"** to begin
2. **Swipe** across the screen to slice fruits
3. **Avoid bombs** — they cost you a life!
4. **Slice multiple fruits** in one swipe for combos
5. **Score points** for each fruit sliced
6. **Lose all 3 lives** and the game ends!

## 🖱️ Controls

| Action | Input |
|--------|-------|
| Slice Fruits | Swipe on canvas (mouse/touch) |
| Start Game | Click Start or press `Space` |
| Reset Game | Press `R` |

## ✨ Features

- 🍉 **10 Fruit Types** – Each with unique emoji and colors
- 💥 **Combo System** – Slice multiple fruits for bonus points
- 💣 **Bomb Avoidance** – Bombs cost lives
- 🎯 **Scoring System** – Points based on fruit value and combo
- 🔥 **Visual Feedback** – Slice trails, particles, and animations
- 🎵 **Sound Effects** – Slice and bomb sounds via Web Audio
- 📱 **Responsive Design** – Works on desktop and mobile
- 🖱️ **Touch Support** – Swipe on mobile devices

## 🎯 Scoring

| Fruit | Points |
|-------|--------|
| 🍎🍊🍋🍓🍒 | 1 |
| 🍉🍇🍑🥝 | 2 |
| Combo Bonus | +1 per 5 combo |

## 📊 Stats

- **Score** – Your current points
- **Lives** – 3 lives (lose one per bomb)
- **Combo** – Consecutive slices without missing
- **Best Score** – Saved in browser local storage

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3 (Gradients, responsive design)
- Vanilla JavaScript (Web Audio API)
