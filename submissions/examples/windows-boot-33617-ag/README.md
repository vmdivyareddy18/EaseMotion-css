# Pure CSS Windows Startup Animation

A complete Windows startup boot screen and user sign-in transition component built using only HTML/CSS.

---

## What is Included

1. **`demo.html`**: A monitor-style simulated screen displaying the blue Windows tiles, orbiting spin dot patterns, user PIN dialogs, and status controls.
2. **`style.css`**: Windows grid styles, coordinate transformations, orbiting timing calculations, user login forms.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the Windows grid logo with spinning dots loading orbitally below it.
3. Verify that after 4.8 seconds, the boot loader fades into the Windows PIN Login screen.
