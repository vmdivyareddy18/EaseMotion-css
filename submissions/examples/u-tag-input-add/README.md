# ease-tag-input-add

Animated tag input component where new tags appear with a scale and bounce effect.

## Features

- Animated tag chips
- Scale-in effect
- Bounce animation
- Expanding input field
- Responsive layout
- Pure HTML and CSS

## Acceptance Criteria Covered

### Tag Animation

New tag chips animate using:

- scale(0)
- scale(1.15)
- scale(1)

with a subtle bounce effect.

### Input Field

The input expands smoothly when focused.

### Common Use Cases

- Blog editors
- Search filters
- Category selection
- Skill input forms
- Dashboard tagging systems
