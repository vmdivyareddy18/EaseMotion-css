# Fade-Out Accordion

## What does this do?

This submission provides a responsive dashboard-style accordion with a smooth fade-out and scale animation using only HTML and CSS.

---

## How is it used?

```html
<details class="accordion">
    <summary>Revenue Overview</summary>

    <div class="content">
        <p>Your content goes here.</p>
    </div>
</details>
```

---

## Why is it useful?

- Pure CSS implementation
- No JavaScript required
- Responsive layout
- Keyboard accessible through native `<details>` and `<summary>`
- Smooth fade and scale transition
- Suitable for dashboards, analytics panels, FAQs, and documentation sections
