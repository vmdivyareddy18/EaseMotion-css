# CSS Scale Hover Tooltip

Pure CSS animated tooltip utilizing a Scale Hover interaction designed for Minimalist Tech layouts.

## Features

- Pure CSS implementation
- Scale Hover animation
- Minimalist interface aesthetics
- Responsive design
- Keyboard accessible
- CSS custom properties
- Lightweight styling
- Smooth 60fps animations
- Zero JavaScript dependency

## Variables

```css
--primary
--accent
--tooltip-bg
--duration
--ease
```
