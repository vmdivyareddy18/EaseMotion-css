# Animated Details Summary Disclosure Example

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern disclosure component using native HTML `details` and `summary` elements with smooth reveal animations.

## Features

- Native disclosure pattern
- CSS-only animations
- Smooth content reveal
- Expand and collapse interaction
- Keyboard accessible
- Responsive layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<details class="disclosure-card">

    <summary>
        Section Title
    </summary>

    <div class="content">
        Content goes here
    </div>

</details>
```

## Main Classes

```css
.disclosure-card
.content
```
