# Ease Gradient Mesh Overlay

## Overview

This example demonstrates a reusable CSS-only animated gradient mesh background created with layered radial gradients.

## Features

- Pure HTML and CSS
- Animated gradient mesh
- Responsive layout
- No images required
- Lightweight utility
- Easy to customize

## Folder Structure

```text
ease-gradient-mesh-overlay-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

1. Open the example folder.
2. Launch `demo.html`.
3. Adjust gradient colors or animation speed.

## Browser Support

Supported in modern browsers with CSS gradient and animation support.

## Why This Example

Gradient mesh overlays create visually rich sections while keeping projects lightweight and dependency-free.
