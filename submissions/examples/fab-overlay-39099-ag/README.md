# Morphing Floating Action Button (FAB) Overlay

A floating button layout demonstrating geometric morph transitions from circular keys to full panels.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting circular action buttons, quick menu lists, links, and switches.
2. **`style.css`**: Bottom floating coordinates, border-radius morph transitions, rotate X angles, and list borders.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Click the floating **+** button in the bottom-right corner.
3. Verify that the button morphs smoothly into a rectangular menu panel displaying quick setting options.
4. Verify that the **+** symbol rotates 45 degrees to represent a close **X** button.
5. Click it again to confirm it returns to its circular FAB shape.
