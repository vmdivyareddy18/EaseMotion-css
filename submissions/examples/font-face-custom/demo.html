font-display: swap

Text renders in a fallback face instantly; the custom font loads asynchronously and swaps in once ready. Prioritises content visibility.

font-display: optional

If the custom font fails to load within ~100 ms, the browser sticks with the fallback permanently. Great for performance-sensitive pages.

font-display: block

The browser hides text for up to 3 seconds while the font loads, then swaps. Ensures no flash of unstyled text (FOUT) but may delay content.

This demo uses Satoshi by Fontshare. Font files live alongside the CSS in a production setup.
