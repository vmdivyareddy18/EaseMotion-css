# Pie Chart Explode – Segment pops out on hover

> A production-ready pie chart component where segments smoothly pop out on hover and click, with animated transitions and interactive controls. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Segment explode on hover** – Smooth pop-out animation
- ✅ **Click to explode** – Click a segment to keep it exploded
- ✅ **4 color palettes** – Default, Pastel, Vibrant, Monochrome
- ✅ **Interactive legend** – Hover and click from legend
- ✅ **Tooltip** – Shows label and value on hover
- ✅ **Animate transitions** – Smooth data changes
- ✅ **Toggle labels** – Show/hide percentage labels
- ✅ **Add/Remove segments** – Dynamic data management
- ✅ **Explode/Collapse all** – Bulk operations
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
