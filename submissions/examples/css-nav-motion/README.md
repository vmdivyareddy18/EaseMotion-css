# 🧭 CSS Nav Motion

A collection of beautiful, pure CSS animated navigation menus for modern websites.

## 🚀 Features

### Navigation Styles (6 types)
1. **Underline Slide** - Smooth underline animation from center
2. **Pill Hover** - Rounded pill shape with hover effect
3. **Neon Glow** - Cyberpunk neon text glow
4. **Gradient Fill** - Gradient background slide-in
5. **Icon Menu** - Icon + label with bounce effect
6. **Hamburger Menu** - Animated hamburger to X toggle
7.6. **Hamburger Menu** - Animated hamburger to X toggle


## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All nav animations
- `readme.md` - Documentation

## 🛠️ How to Use

### Simple Nav
```html
<nav class="nav underline-slide">
    <a href="#" class="active">Home</a>
    <a href="#">About</a>

</nav>
