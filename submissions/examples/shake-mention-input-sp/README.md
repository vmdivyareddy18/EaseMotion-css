# Shake Mention Input

A retro-inspired mention input featuring a shake animation built with pure HTML and CSS.

## Features

- 🎯 Mention input with `@` prefix
- 🎬 Retro shake animation
- 💻 Pure HTML & CSS
- 📱 Responsive layout
- ♿ Accessible using `:focus-within`
- 🚫 No JavaScript required

## File Structure

```
shake-mention-input-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Hover over or focus the mention input to trigger the retro shake animation.

## Use Cases

- Chat applications
- Team collaboration tools
- Social media platforms
- Comment systems
- Community dashboards
