# Animated Back-to-Top Button

A floating Back-to-Top button that appears after scrolling and smoothly returns the user to the top of the page.

## Features

- Floating button
- Smooth scrolling
- Fade-in/Fade-out animation
- Hover scaling effect
- Responsive
- Accessible (`aria-label`)
- Lightweight implementation

## Technologies

- HTML5
- CSS3
- JavaScript

## Usage

Open `demo.html` in any modern browser and scroll down to see the button appear.
