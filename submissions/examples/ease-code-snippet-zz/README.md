# Code Snippet

## What does this do?
Code Snippet component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"cs-primary\">Code Snippet Primary</div>
```

## Why is it useful?
Provides a reusable code snippet UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
