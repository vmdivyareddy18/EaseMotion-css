# Dark Mode Design System Showcase

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern Dark Mode Design System showcasing reusable UI components, color palettes, typography, design tokens, and interface patterns for contemporary web applications.

## Features

- Hero section
- Color palette showcase
- Typography preview
- Component chips
- Design tokens panel
- Responsive grid
- Hover animations
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="container">

    <header class="hero">
        ...
    </header>

    <section class="grid">
        ...
    </section>

</div>
```
