# Climate Monitoring Dashboard

## Overview
Design an environmental analytics dashboard.

This project implements the Phase 112 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Climate metrics**: Built with glassmorphism and CSS grid/flexbox.
- **Monitoring widgets**: Built with glassmorphism and CSS grid/flexbox.
- **Trend cards**: Built with glassmorphism and CSS grid/flexbox.
- **Environmental statistics**: Built with glassmorphism and CSS grid/flexbox.
- **Responsive layout**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
