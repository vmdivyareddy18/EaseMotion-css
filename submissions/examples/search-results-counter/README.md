# Live Search Results Counter

1. What does this do?

Displays a live counter that updates automatically to show how many animations match the user's current search query.

2. How is it used?

Type into the search field and the counter updates in real time:

```html
<input type="text" id="searchInput" placeholder="Search animations..." />

<p id="resultsCount">Showing 5 animations</p>
```

3. Why is it useful?

This feature provides instant feedback during searching and filtering, helping users quickly understand how many animations match their query. It improves discoverability, enhances the user experience, and makes navigating large animation collections more intuitive.
