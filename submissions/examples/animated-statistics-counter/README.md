# Animated Statistics Counter Section

A clean, modern, and fully responsive **Statistics Counter Section** perfect for landing pages. This example demonstrates how to combine **EaseMotion CSS** staggered entrance animations with custom CSS keyframes to create smooth, animated number counters that grab attention and build trust.

## 🚀 Features

- **Staggered Card Entrance**: Uses `ease-zoom-in` combined with `ease-delay-*` classes so each stat card pops in sequentially on page load.
- **Animated Number Counters**: Custom CSS `@keyframes` animate the numbers fading in and sliding up, synchronized with the card entrance delays.
- **Continuous Icon Pulsing**: Uses `ease-pulse` with staggered delays on the icons to draw attention to each metric.
- **Interactive Hover Effects**: Cards lift on hover with an enhanced shadow and a gradient top border that scales in from the left.
- **Gradient Typography**: The heading uses a gradient text effect for a modern, eye-catching look.
- **Fully Responsive**: The grid adapts from 4 columns to 2 columns to a single column based on screen size.
