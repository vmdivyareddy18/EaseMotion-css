# Ease Avatar Border Spin Component

A highly performant, responsive CSS-animated component where an avatar's custom conic-gradient border rotates smoothly upon interaction.

## Features
- **Conic-Gradient Mechanics:** Utilizes native CSS `conic-gradient` elements to deliver an uninterrupted spectrum of border colors.
- **Hardware-Accelerated:** Runs animations smoothly using the `transform: rotate()` rule.
- **Keyboard Accessible:** Includes focus triggers (`:focus`) alongside `:hover` triggers to maintain accessible standards.
- **Highly Custom Options:** Control sizing, border thickness, colors, and spinning rotation cycles directly through custom properties.
