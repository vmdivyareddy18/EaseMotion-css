# Cursor Follow Hover

## Overview

A hover effect where an element subtly shifts toward the cursor position using CSS variables driven by minimal JavaScript.

## Features

- Cursor tracking
- CSS variable driven transforms
- Smooth animation
- Spring reset on mouse leave
- Pure CSS animation with minimal JS

## Usage

```html
<div class="follow-card">
  Hover Me
</div>
```

## HTML Structure

```html
<div class="wrapper">
  <div class="follow-card"></div>
</div>
```

## CSS Classes

### `.wrapper`

Centers the demo.

### `.follow-card`

Interactive element that follows the cursor.

## Why It Fits EaseMotion CSS

- Lightweight
- Animation focused
- Easy integration
- No dependencies
- Demonstrates CSS variable driven motion
