## Snowfall Animation

A serene winter snowfall scene with ten unique snowflakes drifting down at varying speeds and sizes. Each flake sways side-to-side and rotates as it falls.

**Features:**
- Snowflakes with six-point crystal shape via pseudo-elements
- Randomized size, speed, opacity per flake
- Gentle horizontal sway during descent
- Respects `prefers-reduced-motion`
