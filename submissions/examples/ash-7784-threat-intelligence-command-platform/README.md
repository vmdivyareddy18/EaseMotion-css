# Threat Intelligence Command Platform

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive Threat Intelligence Command Platform dashboard for monitoring cyber threats, indicators of compromise (IOCs), attack campaigns, and security operations. Suitable for SOC teams, SIEM platforms, threat intelligence portals, and cybersecurity command centers.

## Features

- Threat intelligence KPI cards
- Threat activity visualization
- Live threat feed
- Responsive dashboard layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="dashboard">

    <section class="stats">
        ...
    </section>

    <section class="content">
        ...
    </section>

</div>
```
