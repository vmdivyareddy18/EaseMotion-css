# Animation Iteration Count

Demonstrates how `animation-iteration-count` controls how many times an animation plays. Includes infinite, whole numbers, decimal values, and combinations with `animation-direction`.

## Features

- Five iteration-count demos: infinite, 3, 2, 1, and 2.5 (decimal stops mid-cycle).
- Direction combinations: normal, reverse, alternate, alternate-reverse.
- Restart button to replay all animations simultaneously.
- Colour-coded: iteration demos in indigo, direction demos in amber/red.

## Topics Covered

- `animation-iteration-count` property values
- `animation-direction` (normal, reverse, alternate, alternate-reverse)
- Decimal iteration counts stopping mid-animation
- `animation-fill-mode: forwards` to hold final state
