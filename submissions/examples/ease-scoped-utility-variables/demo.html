Locally Scoped Animation Variables

Using locally scoped CSS Custom Properties (--ease-local-duration) to prevent global cascading pollution when combining multiple animation utilities.

The Problem with Global Cascades

If a parent container sets a global animation duration, it can accidentally overwrite all nested children. Locally scoped variables prevent this.
