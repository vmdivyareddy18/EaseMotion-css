# Ease Magnetic Card Stack

A premium animated card stack component built using **HTML** and **CSS**. The design features layered glassmorphism cards with a magnetic hover effect, depth animation, floating background elements, and smooth transitions. It is lightweight, reusable, and ideal for modern web interfaces.

---

## ✨ Features

- Premium glassmorphism design
- Layered card stack effect
- Magnetic hover animation
- Floating blurred background
- Gradient badges
- Shine sweep hover effect
- Responsive layout
- Pure HTML & CSS
- No JavaScript required
- Easy to customize

## 🚀 Installation

1. Clone the repository.
2. Navigate to the project folder.
3. Open `demo.html` in your browser.
4. Ensure `style.css` is located in the same directory.
