# Frosted Glass Carousel

A modern landing-page carousel component built with pure CSS, featuring a frosted glass effect, smooth transitions, and EaseMotion CSS animation utilities.

Designed for SaaS websites, product showcases, portfolios, and modern hero sections.

## Features

- ✨ Pure CSS implementation — no JavaScript required
- 🧊 Frosted glass / glassmorphism UI
- 🔄 Automatic slide transitions using CSS animations
- 🌈 Animated gradient background
- 🎨 Modern landing-page inspired design
- ⚡ Uses EaseMotion CSS animation utilities
- 📱 Fully responsive layout
- ♿ Accessibility-friendly motion handling
- 🚀 Zero dependencies
