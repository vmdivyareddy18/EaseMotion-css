# Course Cards Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build responsive course cards for educational platforms.

---

## Included Components

- Course Image
- Course Title
- Instructor Name
- Duration
- Rating
- Enroll Button

---

## Folder Structure

```
submissions/examples/course-cards-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to test responsiveness.
3. Observe the applied EaseMotion CSS animations.
