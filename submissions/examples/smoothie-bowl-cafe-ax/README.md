# Bliss Bowls – Smoothie Bowl Cafe Landing Page

A complete, ready‑to‑copy landing page for a health food smoothie bowl cafe built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – colorful smoothie bowl imagery with an order CTA
- **Menu Categories** – smoothie bowls, cold‑pressed juices, wholesome snacks
- **Ingredients and Sourcing Story** – fresh organic ingredients and cafe philosophy
- **Nutritional Information** – calories, protein, fibre, and added sugar stats
- **Online Order CTA** – rose‑coloured call‑to‑action section
- **Reviews** – customer testimonials
- **Location / Hours** – address and opening hours
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-rose-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-rose-600`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-full`, `ease-shadow-lg`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
