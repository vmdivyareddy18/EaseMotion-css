# Ease Portal Hover Reveal

## Overview

Ease Portal Hover Reveal creates a futuristic interaction where content appears to emerge through a glowing portal using CSS transforms, blur, opacity, and scaling.

---

## Features

- 🌀 Portal reveal animation
- ✨ Glowing circular portal
- 🌟 Blur-to-sharp transition
- 🚀 Hover depth effect
- 🎨 Three visual variants
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

## Usage

```html
<div class="portal-card">
...
</div>
```

---

## Customization

You can customize:

- Portal color
- Ring size
- Reveal speed
- Blur amount
- Card dimensions

---

## Browser Support

Compatible with all modern browsers supporting CSS transforms, transitions, filters, and box shadows.
