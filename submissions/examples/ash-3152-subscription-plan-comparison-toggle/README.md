# Subscription Plan Comparison Toggle

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive pricing comparison component featuring a billing toggle and subscription plan cards for SaaS and subscription-based applications.

## Features

- Monthly/Yearly billing toggle
- Responsive pricing cards
- Featured plan highlight
- Hover animations
- CTA buttons
- Modern glassmorphism UI
- Mobile-friendly layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="billing-toggle">

    <button class="toggle active">
        Monthly
    </button>

    <button class="toggle">
        Yearly
    </button>

</div>
```

## Main Classes

```css
.billing-toggle
.toggle
.plans
.plan
.featured
.badge
```
