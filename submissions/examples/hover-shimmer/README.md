# Hover Shimmer

**What does this do?**  
Adds a sweeping light-shimmer effect that animates across a card on hover, creating a polished, premium feel.

**How is it used?**  
```html
<div class="hover-shimmer-card">Your content</div>
```

**Why is it useful?**  
Cards are the most common UI element in EaseMotion CSS. A shimmer hover adds motion feedback without relying on heavy JavaScript libraries, and it fits the animation-first philosophy of EaseMotion CSS.

---
## Usage

1. Include the CSS file in your project:
```html
<link rel="stylesheet" href="style.css">

2.Apply the class to your element:
<div class="hover-shimmer">Hover me</div>
