# Neobrutalism Hover Card

A high-contrast content card featuring thick borders and a harsh shadow, designed for trendy brutalist interfaces.

## Files
- `demo.html`: The HTML structure of the card.
- `style.css`: The CSS rules defining the card's stark aesthetic and dynamic hover state.
- `README.md`: Documentation.

## Features
- **Harsh Shadows & Borders:** Uses bold `3px solid black` borders and solid black shadows.
- **Dynamic 3D Hover:** On `:hover`, the card pops upwards and rotates slightly, while the solid black shadow dynamically extends downwards, emphasizing a playful paper-cutout feel.
- **High Contrast:** Bright, retro colors combined with stark black lines.

## Usage
Add the `.neo-brutalism-card` class to any container element.
