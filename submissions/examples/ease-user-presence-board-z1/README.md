# Ease User Presence Board

## Overview
A CSS-only team presence component for collaboration and dashboard interfaces.

## Features
- Online, away and offline indicators
- Responsive design
- Hover animation
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any modern browser.

## Browser Support
Chrome, Firefox, Edge and Safari.

## Why EaseMotion CSS?
A lightweight, reusable, animation-first component for chat apps, dashboards, SaaS platforms and admin panels.
