CSS Motion Path Orbital Elements

offset-path and offset-distance animate elements along geometric paths — no trigonometry, no JavaScript, no SVG animateMotion.

Single circular orbit — offset-path: circle()

offset-path: circle(130px at 50% 50%) defines the orbit. offset-rotate: 0deg keeps the planet facing up as it travels.

Multi-body solar system — varied radius & duration

Each planet has its own offset-path radius and animation-duration. Inner planets orbit faster — no physics engine required.

Elliptical orbit — offset-path: ellipse() + offset-rotate: auto

offset-path: ellipse(160px 100px at 50% 50%) for an elliptical track. offset-rotate: auto makes the comet shape align to the path tangent as it travels.
