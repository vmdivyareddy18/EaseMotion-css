# CSS Spring Physics Popover

A pure CSS Spring Physics Popover with a smooth spring-like hover animation designed for accessible web interfaces.

## Features

- Pure HTML & CSS
- Spring physics transition
- Keyboard accessible
- Responsive design
- CSS Custom Properties
- No JavaScript

## Folder Structure

```
spring-physics-popover/
│── demo.html
│── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Customization

```css
:root{
    --duration:.45s;
    --scale:1.05;
    --primary:#2563eb;
}
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
