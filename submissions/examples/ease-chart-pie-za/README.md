# CSS Pie / Donut Chart

## What does this do?
Pure CSS pie and donut charts using conic gradients with animated segments, center labels, and interactive legend cards.

## How is it used?
```html
<div class="pc-pie" style="background:conic-gradient(#667eea 0% 40%, #48bb78 40% 65%)"></div>
<div class="pc-pie pc-donut" style="background:conic-gradient(#667eea 0% 60%, #48bb78 60% 100%)"><div class="pc-center"><span>60%</span></div></div>
```

## Why is it useful?
Creates data visualization charts without JavaScript or SVG libraries. Pie and donut variants with center labels. Segmented donut with gaps. Color-coded legend items. Small chart sizes for dashboards. Customizable via CSS variables.
