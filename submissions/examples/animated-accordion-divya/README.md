# Animated Accordion Component

A CSS-only accordion/collapse component with smooth expand and collapse animations.

## Features

- Pure HTML & CSS
- Smooth expand/collapse animation
- Rotating chevron icon
- Responsive design
- Hover effects
- Reduced motion support
