# Financial Dashboard Showcase Hub

## Overview

Financial Dashboard Showcase Hub is a comprehensive EaseMotion CSS showcase that demonstrates how multiple animation patterns can work together inside a modern financial analytics dashboard.

This example combines portfolio visualization, investment tracking, budgeting tools, spending analytics, approval states, invoice confirmations, notifications, and financial KPIs into a single cohesive interface.

The goal is to provide developers with a practical reference for building rich finance-related user experiences using lightweight CSS animations.

## Accessibility

Supports reduced-motion preferences:

```css
@media (prefers-reduced-motion: reduce)
```

This disables non-essential animations for users who prefer reduced motion.
