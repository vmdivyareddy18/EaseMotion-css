# Wave Mention Input

A reusable mention input component with a smooth wave animation built using pure HTML and CSS.

## Features

- Pure CSS animation
- No JavaScript dependency
- Responsive design
- Modern mention input UI
- Smooth wave motion effect
- Easy customization

## Usage

Add the HTML structure:

```html
<div class="wave-input">
  <span class="at-symbol">@</span>
  <input type="text" placeholder="Enter username">
</div>
