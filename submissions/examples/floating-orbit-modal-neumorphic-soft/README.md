# CSS Floating Orbit Modal – Neumorphic Soft

A modern **Floating Orbit Modal** inspired by soft neumorphic interfaces. The component uses only HTML and CSS, featuring a floating entrance animation, orbiting decorative element, responsive layout, and accessibility-friendly focus styles.

## Features

- Pure HTML & CSS
- Floating Orbit animation
- Soft Neumorphic UI
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- No JavaScript required
