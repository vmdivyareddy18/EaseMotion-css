# ease-water-intake-fill

Animated hydration progress indicator showing a glass filling with water.

## Features

- Water fill animation using height transitions
- Wave-like moving top edge
- Glass-shaped container
- Infinite animation loop
- Pure HTML and CSS
- Suitable for health and wellness applications

## Usage

```html
<div class="glass">
    <div class="water">
        <div class="wave"></div>
    </div>
</div>
```

## Animation Details

- Water level rises using animated height changes.
- Wave element creates a realistic liquid surface.
- Designed for hydration tracking and wellness dashboards.
