# Image Reveal Effects

## What does this do?
CSS clip-path and transform reveal animations for images with directional, shape, and swipe hover-triggered effects.

## How is it used?
```html
<div class="imr-frame"><div class="imr-img imr-clip-left" style="background:url(image.jpg)"></div></div>
```

## Why is it useful?
Engaging image entrance effects without JavaScript. 10 reveal patterns: directional clips, circle, diamond, skew, swipes. Hover-triggered with smooth CSS transitions.
