# Ease Tie-Breaker Game

A lightweight, animated tie-breaker widget that randomly picks a winner among tied options.

## What does this do?
Lets you add a list of tied names/options and randomly picks one winner with a short animated flicker effect.

## How is it used?
Open `demo.html` in your browser. Type a name into the input and click **Add** (or press Enter) to add options. Once you have at least two, click **Break the Tie 🎉** — the display flickers through the options before landing on a random winner.

## Why is it useful?
Teams often need a quick, fun way to break ties (e.g. choosing who goes first, deciding between two equal options). This component does that with a lightweight, dependency-free, animated interaction that fits EaseMotion CSS's animation-first, human-readable philosophy.
