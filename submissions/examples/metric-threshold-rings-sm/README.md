# Metric Threshold Rings

1. What does this do? Adds responsive dashboard metric cards with conic threshold rings, animated card entrance, status pulse indicators, and health-state colors.
2. How is it used? Apply `.metric-grid` to the wrapper and use `.metric-card` with state classes like `.healthy`, `.watch`, or `.urgent` for each metric.
3. Why is it useful? It makes dashboard health states easier to understand at a glance while keeping the effect CSS-only and motion-aware.
