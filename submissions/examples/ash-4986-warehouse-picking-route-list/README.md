# Warehouse Picking Route List

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive warehouse picking route component that guides warehouse staff through an optimized picking sequence using CSS-only route cards.

## Features

- Warehouse picking route list
- Completed and active route states
- Picking summary cards
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="stop active">

    <div class="marker">2</div>

    <div class="details">

        <h3>Aisle C • Shelf 04</h3>

    </div>

</div>
```

## Main Classes

```css
.card
.route
.stop
.completed
.active
.marker
.details
.time
.summary
.stat
```
