# Price Strikethrough

## What does this do?
Animated strikethrough for discounted prices, where the original price line animates in with a smooth transition effect.

## How is it used?
Apply the `.price-old` class to the original price element. The strikethrough line animates from left to right on page load.

## Why is this useful?
Adds visual emphasis to discount/sale pricing, making the savings more noticeable while keeping the original price readable.
