# Firefly Glow Animation

CSS-only firefly glow demo for EaseMotion. The example creates small light particles that blink, drift, and scale across a night panel without JavaScript.

## Files

- `demo.html` - demo markup
- `style.css` - firefly layout, glow, drift, and blink animation

## Usage

Open `demo.html` in a browser to view the animation.
