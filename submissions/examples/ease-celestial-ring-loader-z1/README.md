# Ease Celestial Ring Loader

## Overview

A futuristic loading animation featuring multiple rotating celestial rings orbiting around a glowing core using only HTML and CSS.

## Features

* Multi-ring orbital animation
* Glowing center core
* Smooth continuous rotation
* Responsive layout
* Lightweight CSS
* Pure HTML & CSS

## Folder Structure

```text
ease-celestial-ring-loader-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

* Chrome
* Firefox
* Edge
* Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Demonstrates reusable loading animations, orbital motion, and modern CSS techniques suitable for splash screens, AI interfaces, dashboards, and futuristic web applications.
