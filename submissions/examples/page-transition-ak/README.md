# Page Transition Showcase

## What does this do?
A page transition showcase demonstrating fade, slide, and zoom transition effects between 4 simulated pages with keyboard navigation support.

## How is it used?
Open demo.html in a browser. Use the Next/Prev buttons or arrow keys to navigate between pages. Select a transition type (Fade, Slide, or Zoom) to change the animation effect.

## Why is this useful?
This provides a reference implementation for CSS page transition patterns. It demonstrates CSS keyframe animations, state-based class toggling, and accessible keyboard navigation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see transitions.
