# Draw Border Modal
A pure HTML and CSS modal featuring a border drawing animation with a soft neumorphic design.

---

## Features
- Draw-border animation
- Pure HTML & CSS
- Responsive layout
- Neumorphic UI
- Hover effects
- No JavaScript required

---

## Usage
```html
<label for="modal-toggle" class="open-btn">
Open Modal
</label>
```

---

## Why this fits EaseMotion CSS
This component demonstrates an elegant animated modal with a border drawing effect, smooth transitions, and modern neumorphic styling. It provides a reusable UI pattern that aligns with EaseMotion CSS's animation-first philosophy.
