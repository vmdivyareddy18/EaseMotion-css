How it works

Each dropdown uses the popover attribute on a <div> and popovertarget on the trigger button. The arrow indicator points up from the dropdown to the button via ::before.

The "Account" dropdown demonstrates .dropdown--right alignment for edge-of-screen positioning.

Click the trigger again or press Esc to close. Light-dismiss (click outside) also works automatically.

Built with the CSS Popover API — no JavaScript required for open/close behavior.
