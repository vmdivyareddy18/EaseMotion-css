# Animated Dropdown Menu

A responsive animated dropdown menu component built using pure HTML and CSS.

## Features

- Fade and slide animations
- Hover interactions
- Smooth open and close transitions
- Responsive layouts
- Reusable dropdown menu structure
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated dropdown menu component.

## Files

- `demo.html` → Dropdown menu structure
- `style.css` → Dropdown styles and animations
- `README.md` → Documentation
