# Ease License Expiry Card

## Overview
A CSS-only component for tracking software license expiry.

## Features
- Expiry status badges
- Responsive layout
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for enterprise dashboards, subscription management, and admin panels.
