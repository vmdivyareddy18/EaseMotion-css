# Ripple Wave Testimonial Card

A CSS-only animated testimonial card component inspired by modern landing page designs.

## Features

- Ripple wave animation effect
- Glass-style testimonial card
- Pure CSS implementation
- Responsive layout
- No JavaScript required

## Usage

```html
<article class="testimonial-card">
  <p>Your testimonial text</p>
  <h3>Author Name</h3>
</article>
