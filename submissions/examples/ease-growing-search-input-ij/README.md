# Growing Search Input

A search input that expands on click with a smooth width transition.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--input-bg` | `#2d2d44` | Input background color |
| `--input-color` | `#eee` | Text color |
| `--input-focus-bg` | `#3d3d5a` | Background color on focus |
| `--expand-duration` | `0.3s` | Expansion animation duration |
