# Ease Fusion Data Cascade

## Overview

A futuristic data visualization featuring animated cascading streams flowing into a frosted information panel using only HTML and CSS.

## Features

- Cascading data animation
- Gradient stream effect
- Glassmorphism panel
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-fusion-data-cascade-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

Optimized for desktop, tablet, and mobile devices.
