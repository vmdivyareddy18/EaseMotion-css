# 🚀 Animation Performance Monitor UI

An advanced performance monitoring component to track animation smoothness using FPS (Frames Per Second).

---

## ✨ Features

- Real-time FPS counter  
- Performance tracking using requestAnimationFrame  
- Animated element for testing  
- Lightweight monitoring tool  

---

## 📦 Use Cases

- Debug animation performance  
- Optimize UI animations  
- Benchmark CSS animations  
- Developer tools integration  

---

## 🧩 How It Works

- Uses requestAnimationFrame loop  
- Counts frames per second  
- Displays FPS dynamically  

---

## 🛠️ Tech Used

- HTML  
- CSS  
- JavaScript  

---

## 💡 Why this fits EaseMotion CSS

This enhances animation quality by providing insights into performance, aligning with animation-first and optimized UI design.
