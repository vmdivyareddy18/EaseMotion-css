# Animated FAQ Accordion

A smooth, responsive FAQ accordion component built with vanilla JS and EaseMotionCSS-style transitions.

## Features
- Smooth expand/collapse animation using `max-height` transition
- Rotating `+` icon on open/close
- Only one FAQ open at a time (accordion behavior)
- Fully responsive, mobile-friendly layout
- Clean, modern dark UI

## Files
- `demo.html` — Markup and interaction logic
- `style.css` — Styling and animations

## Usage
1. Open `demo.html` in your browser.
2. Click any question to expand its answer.
3. Click another question to collapse the current one and open the new one.

## Customization
Adjust animation speed by changing the `transition` duration in `.faq-icon` and `.faq-answer` inside `style.css`.
