# CSS Scale-Hover Carousel – Marketing Landing Page

A responsive **Marketing Landing Page Carousel** built using pure HTML and CSS with smooth scale-hover animations.

## Features

- Pure HTML & CSS
- Marketing landing page card design
- Smooth scale-hover animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## CSS Variables

```css
:root{
  --primary:#ff6b35;
  --secondary:#ff9f43;
  --scale:1.08;
  --duration:.35s;
}
```

## Folder Structure

```
scale-hover-carousel-marketing-landing/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser. Customize colors, animation speed, scale, and spacing through the CSS variables.
