# Border Image Source

Explores `border-image-source` with CSS gradients (linear, radial, conic, multi-stop) as border fill. Compares border-image borders against regular solid borders.

## Gradient Borders

While `border-color` only accepts a single solid color, `border-image-source` accepts any CSS image — including gradients. Combined with `border-image-slice` and `border-image-repeat`, complex multi-color borders are possible. This demo shows linear, radial, and conic gradient borders at various widths.

## Usage

Open `demo.html` in a modern browser. Gradient source variants are presented in a grid layout with labeled code snippets. A side-by-side comparison highlights the difference between `border: 20px solid` and `border-image-source` with a gradient. Edit the gradient functions in `style.css` to create custom border appearances.
