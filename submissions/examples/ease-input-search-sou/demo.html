.ease-input-search

A search input variant with an inline SVG magnifier icon — added via background-image using the same approach as the existing .ease-select dropdown icon.

1. Basic Search Input

Add .ease-input-search to any .ease-input.

.ease-input .ease-input-search

2. Size Variants

Combine with .ease-input-search-sm or .ease-input-search-lg.

.ease-input-search-sm
.ease-input-search (default)
.ease-input-search-lg

3. Icon Position

Use .ease-input-search-right to place the icon on the right (useful for RTL layouts or alternative designs).

.ease-input-search (left, default)
.ease-input-search.ease-input-search-right

4. State Variants

Error, success, disabled, and loading states.

.is-error
.is-success
disabled
.is-loading (with spinner)

5. Real-World Form Example

Drop-in search bar — semantic, accessible, zero JS.
