# Neon Glow Text

1. **What does this do?**  
   It applies a multi-layered text shadow to create a realistic neon glow effect with a subtle flickering animation.

2. **How is it used?**  
   ```html
   <div class="neon-text">Your Text</div>
   ```

3. **Why is it useful?**  
   It brings a highly requested cinematic aesthetic to typography, matching the vibrant animation goals of EaseMotion CSS.
