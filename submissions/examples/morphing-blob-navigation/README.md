# Morphing Blob Navigation

A modern CSS-only navigation menu featuring a smooth morphing blob highlight on hover and the active navigation item.

## Features

- 🎨 Pure HTML & CSS
- ✨ Smooth blob animation
- 📱 Responsive layout
- 🚀 No JavaScript required
- 🎯 Easy to customize

## Folder Structure

```
morphing-blob-navigation/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern browser.
2. Hover over the navigation items to view the morphing blob animation.
3. Apply the `active` class to highlight the current page.

```html
<nav class="blob-nav">
  <a href="#" class="active">Home</a>
  <a href="#">About</a>
  <a href="#">Services</a>
</nav>
```

## Customization

You can easily customize:

- Background colors
- Navigation spacing
- Border radius
- Animation duration
- Font styles
- Hover colors
