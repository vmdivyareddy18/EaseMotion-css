# Animated Rating (`ease-rating-cv`)

A lightweight animated rating component built using **HTML5** and **Pure CSS**.

## Features

- ⭐ Star hover animation
- ✨ Glow effects
- ⭐ Static half-star example
- 🎨 Multiple color themes
- 📱 Responsive layout
- No JavaScript
- No dependencies

## Folder Structure

```
ease-rating-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Themes

- Gold
- Blue
- Half-star demonstration

## Browser Support

Works in all modern browsers.
