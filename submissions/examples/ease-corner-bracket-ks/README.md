# Corner Brackets Slide-in

## What does this do?
Reveals and slides in framing brackets (`[` and `]`) from both sides of the text when hovered.

## How is it used?
Add the class to your text link or inline element:

```html
<a href="#" class="corner-bracket-link">Explore Features</a>
```

## Why is it useful?
It offers a clean, minimalistic link highlight that frames the interactive target cleanly without adding vertical layout shifting.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
