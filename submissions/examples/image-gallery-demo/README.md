# Image Gallery Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be applied to a responsive image gallery.

---

## Components

- Responsive Grid
- Six Image Cards
- Image Titles
- Hover Interaction

---

## Folder Structure

```
submissions/examples/image-gallery-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to test responsiveness.
3. Hover over the cards to observe the interaction.

---

## Features

- Responsive gallery
- Six image cards
- Modern layout
- Uses existing EaseMotion CSS animation classes
