# Scroll Snap Utilities

Demonstrates CSS `scroll-snap-type` for building smooth, predictable scrolling carousels and galleries. Includes horizontal, vertical, and 2D grid snap demos.

## Features

- `scroll-snap-type: x mandatory` — horizontal carousel snapping
- `scroll-snap-type: y mandatory` — vertical section snapping
- `scroll-snap-type: both mandatory` — 2D grid snapping
- `scroll-snap-type: x proximity` — gentle snapping only near snap points
- Styled custom scrollbars matching the primary color
- Dark mode and reduced-motion safe

## Design Tokens Used

- `--ease-color-primary` — accent and scrollbar color
- `--ease-color-neutral-*` — surface and background variants
- `--ease-radius-*` — rounded containers
- `--ease-shadow-sm` / `--ease-shadow-md` — card elevation
- `--ease-font-sans` / `--ease-font-mono` — content and badge fonts
- `--ease-space-*` — spacing scale
