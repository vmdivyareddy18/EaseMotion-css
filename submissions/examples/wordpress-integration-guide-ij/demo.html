WordPress Integration Guide

Add EaseMotion animations to any WordPress site

1. CDN via Theme

Add the EaseMotion CDN link to your theme's header.php or via the Customizer "Additional CSS" field.

<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/easemotion-css/dist/easemotion.min.css">

2. Block Editor (Gutenberg)

In the WordPress Block Editor, add the "Custom HTML" block and wrap your content with EaseMotion classes.

<div class="ease-fade-in">
  <h2>Welcome to My Site</h2>
</div>

3. Classic Editor / Page Builders

Use the "Text" tab in the Classic Editor or your page builder's HTML widget to add EaseMotion classes directly to elements.

  • Elementor: Add class under Advanced → CSS Classes
  • WPBakery: Add class under Design Options → Extra Class
  • Divi: Add class under Advanced → CSS ID & Classes

4. Enqueue in functions.php

For theme developers, enqueue EaseMotion from your CDN in your theme's functions.php.

function enqueue_easemotion() {
  wp_enqueue_style(
    'easemotion',
    'https://cdn.jsdelivr.net/npm/easemotion-css/dist/easemotion.min.css',
    array(),
    '1.0.0'
  );
}
add_action('wp_enqueue_scripts', 'enqueue_easemotion');

5. Use in Templates

Apply EaseMotion classes directly in your PHP template files.

<div class="ease-slide-up ease-delay-200">
  <?php the_title( '<h1>', '</h1>' ); ?>
</div>
