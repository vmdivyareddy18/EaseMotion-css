# Component: WordPress Integration Guide

This submission implements a WordPress integration guide for EaseMotion for issue **#13641**.

## Description

A step-by-step guide showing WordPress users how to add EaseMotion animations to their sites. Covers CDN inclusion via theme files, the Block Editor (Gutenberg), Classic Editor, page builders (Elementor, WPBakery, Divi), enqueuing in functions.php, and direct use in PHP templates.

## Acceptance Criteria

- CDN link via theme header.php / Customizer
- Block Editor custom HTML block usage
- Page builder class instructions
- functions.php enqueue snippet
- PHP template usage example
