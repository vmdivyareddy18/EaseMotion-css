# CSS Spring Physics Accordion - Minimalist Tech Layouts

## Overview

A pure CSS Spring Physics Accordion designed for minimalist technology interfaces.

This component provides smooth expand and collapse animations inspired by natural spring motion while keeping the implementation lightweight and JavaScript-free.

## Features

- Pure HTML and CSS
- Spring-inspired expand/collapse animation
- Responsive design
- Keyboard accessible
- Native details and summary elements
- Customizable CSS variables
- No JavaScript dependency
