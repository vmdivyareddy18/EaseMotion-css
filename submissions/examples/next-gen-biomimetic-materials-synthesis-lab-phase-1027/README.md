## Features

- Fully static HTML/CSS lab showcase with no JavaScript.
- Premium dark biomimetic dashboard aesthetic with glassmorphism panels.
- Inline SVG hero visualization for the biomaterial sample, lattice ring, organic leaf influence, and nano-fiber mesh.
- CSS-only motion for pulsing molecule nodes, rotating lattice ring, scan line, floating telemetry cards, soft glow, and hover lift.
- Responsive layout tuned for desktop, tablet, and mobile direct-open viewing.

## UI sections

1. Header and hero panel for the Biomimetic Materials Synthesis Lab
2. Central biomaterial sample visualization
3. Molecular lattice and nano-fiber structure viewer
4. Live synthesis telemetry grid
5. Material property indicators
6. Bio-inspired design cards
7. Lab process timeline
8. Environmental stability and stress test cards
9. Alert and status timeline
10. Control panel buttons and lab controls
11. Footer reference to the local README
