# Shimmer Card Skeleton (Issue #18252)

A card-shaped skeleton loader with title and body placeholder lines. A shimmer gradient sweeps across the card to indicate loading state.

## Usage

Open `demo.html` in a browser. The shimmer animation runs continuously.

## Files

- `demo.html` — HTML5 page with the shimmer card
- `style.css` — Dark theme styles with `@keyframes shimmerCard`
- `README.md` — This file
