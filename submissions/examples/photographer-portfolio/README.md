# Premium Photographer Portfolio Page

A complete, high-fidelity, and fully responsive single-page photographer portfolio ("Aether Photography") built under issue #14802 using the **EaseMotion CSS** design system.

## EaseMotion Features Showcased

- **Entrance Animations:** Leverages `.ease-reveal` and direction helpers (`.ease-reveal-up`, `.ease-reveal-right`) with scroll-reveal triggers.
- **CSS-Only Lightbox:** Implements `.ease-lightbox` using the CSS `:target` pseudo-class for JavaScript-free, full-screen image viewing overlays.
- **Hover Micro-interactions:** Showcases hover scale card effects, image opacity transitions, and interactive form feedback.
- **Component Systems:** Utilizes standardized `.ease-btn`, `.ease-btn-primary`, `.ease-btn-outline`, and layout classes (`.ease-flex`, `.ease-grid`, `.ease-card`).

## Required Sections Included

1. **Full-Bleed Hero:** Dark, high-contrast visual display with a fade-in overlay text and direct scheduling CTA.
2. **Categories Filter Menu:** Clickable selectors to dynamically filter gallery categories (e.g. Wedding, Portrait, Landscape).
3. **Masonry Gallery & Lightbox:** Asymmetric masonry photogrid with built-in full-screen `:target` closeable slide lightbox elements.
4. **About the Photographer:** Sleek, visual profile detailing the photographer's creative vision and gear selection.
5. **Pricing Packages:** Flexible packages (Bronze, Silver, Gold session options) to drive bookings.
6. **Instagram Feed Grid Mockup:** A modern photography block showing client highlight stories.
7. **Booking Form CTA:** Responsive booking request module with focus validation and feedback screens.

Open `demo.html` in any browser to explore the design.
