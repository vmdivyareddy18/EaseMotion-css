# Buffer Loading Pulse

## What does this do?
Adds a pulsing buffering indicator animation for media loading states, simulating the spinning/beating pattern of content buffering.

## How is it used?
Apply the `.buffer-pulse` class to a container element. Three dots pulse sequentially to indicate ongoing loading activity.

## Why is this useful?
Provides a familiar buffering indicator pattern for video/audio players without JavaScript, aligning with EaseMotion's CSS-first approach to UI feedback.
