# Gauge Meter – Needle sweeps to value

> A production-ready gauge meter component that animates a needle sweeping to a target value with smooth easing, multiple themes, and real-time controls. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Smooth needle animation** – Sweeps with easing
- ✅ **Multiple themes** – Default, Dark, Success, Danger
- ✅ **Real-time controls** – Slider, buttons, keyboard
- ✅ **Speed control** – Slow, Normal, Fast
- ✅ **Status indicators** – Low, Medium, High
- ✅ **Tick marks** – With value labels
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Highly customizable** – Colors, range, labels, callbacks
