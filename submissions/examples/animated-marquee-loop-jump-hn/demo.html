Seamless Infinite Horizontal Scroll Marquee

Issue Fixed: Previously, horizontal marquees translated track width by exactly -50%. When looping, this caused a visual jump equal to half the gap size ($G/2$). By using calc(-50% - var(--marquee-gap) / 2), the translation accounts for the gap separating the duplicate items, resulting in a perfect, stutter-free loop.
