# ease-border-color Utilities

Border color utility classes for EaseMotion CSS.

## Usage

```html
<div class="ease-border ease-border-success">Success bordered box</div>
<div class="ease-border-2 ease-border-danger">Danger bordered box</div>
```

## Classes

| Class                   | CSS Property                              |
| ----------------------- | ----------------------------------------- |
| `ease-border-success`   | border-color: var(--ease-color-success)   |
| `ease-border-danger`    | border-color: var(--ease-color-danger)    |
| `ease-border-warning`   | border-color: var(--ease-color-warning)   |
| `ease-border-info`      | border-color: var(--ease-color-info)      |
| `ease-border-secondary` | border-color: var(--ease-color-secondary) |
