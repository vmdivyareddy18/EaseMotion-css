# Notification Preference Toggle Grid

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive notification preference settings component that organizes multiple notification options into a clean toggle grid. Suitable for account settings, user profiles, admin dashboards, SaaS applications, and mobile-friendly preference screens.

## Features

- Toggle switch grid
- Email, SMS, Push, Marketing, Security, and Reports preferences
- Responsive card layout
- Accessible form controls
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="item">

    <div>

        <h3>Email Alerts</h3>

    </div>

    <label class="switch">

        <input type="checkbox" checked>

        <span class="slider"></span>

    </label>

</div>
```

## Main Classes

```css
.grid
.item
.switch
.slider
.container
.card
```
