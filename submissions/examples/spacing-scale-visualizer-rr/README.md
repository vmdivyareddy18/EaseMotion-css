# Interactive Spacing Scale Visualizer Component

## What does this do?

Provides a visual representation of spacing tokens commonly used in design systems, helping developers compare spacing scales and understand layout consistency.

## How is it used?

```html
<div class="spacing-item">
    <span>16px</span>
    <div class="bar bar-16"></div>
</div>
```

## Why is it useful?

Spacing is one of the most important design tokens in modern UI systems. This component visually demonstrates spacing relationships using animated bars and responsive layouts built entirely with HTML and CSS.
