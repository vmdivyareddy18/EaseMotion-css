# Music Visualizer Bars

Audio visualizer bars that animate with varying heights simulating music frequencies. Bars rise and fall with staggered delays and gradient colors.

## Features

- 20 animated bars with staggered animation delays
- Gradient color from pink to purple
- Play/pause toggle for animation
- Frequency band presets (Pop, Rock, Electronic, Classical, Bass)
- Dark theme with neon visual style

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--mvb-duration` | `0.4s` | Animation duration |
| `--mvb-bar-color-1` | `#ec4899` | Start color (bottom) |
| `--mvb-bar-color-2` | `#8b5cf6` | End color (top) |
| `--mvb-bar-gap` | `4px` | Gap between bars |
| `--mvb-bar-radius` | `4px` | Bar border radius |

## Usage

Open `demo.html` in a browser. Click **Play** to start the animation, **Pause** to stop. Select a preset to change frequency patterns.
