EaseMotion CSS Submission #14015

ease-map-pin-drop

A map pin drops from above with a bounce landing and a slight squash on impact — the standard marker placement animation for map UIs.

Variants — Hover to Replay

ease-map-pin-drop Default
ease-map-pin-drop--slow 0.8s duration
ease-map-pin-drop--delay 0.4s delay

Multi-Pin Stagger — Click Map to Replay

Click to replay

How to Use

html
<!-- Default pin drop -->
<div class="ease-map-pin-drop">
  <svg ...>...</svg>
  <div class="pin-shadow"></div>
</div>

<!-- Variants -->
<div class="ease-map-pin-drop ease-map-pin-drop--slow">...</div>
<div class="ease-map-pin-drop ease-map-pin-drop--delay">...</div>

<!-- Staggered multi-pin -->

<div class="ease-map-pin-drop" style="animation-delay: 0.15s">...</div>
<div class="ease-map-pin-drop" style="animation-delay: 0.3s">...</div>
