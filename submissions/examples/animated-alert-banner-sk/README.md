# Animated Alert Banners

CSS-only alert banners with slide-in animation, dismissible close buttons, and color-coded themes.

## Features
- Four variants: info, success, warning, error
- Slide-in entrance animation
- Dismissible close button with hover effect
- Icon support per variant
- Color-coded backgrounds, borders, and icons
- Stackable with staggered animation delays
- Fully responsive

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`. Add the appropriate modifier class (`--info`, `--success`, `--warning`, `--error`) to the `.alert` element.

## Browser Compatibility
Modern browsers supporting CSS Animations, Flexbox, and Custom Properties.
