# Android Startup Animation

A pure HTML & CSS Android boot animation created for EaseMotion CSS.

## Features

- Pure CSS Android robot
- Eye blinking animation
- Arm swing animation
- Walking legs animation
- Four animated loading dots
- Animated progress bar
- Cycling boot status messages
- Responsive design
- No JavaScript

## Files

```
android-startup-animation/
├── index.html
├── style.css
└── README.md
```

## Usage

Open `index.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Keyframes
