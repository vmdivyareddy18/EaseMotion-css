# Notification Badge (Issue #18254)

A circular notification badge with a pop-in bounce animation. Click the button to increment the count and trigger the animation.

## Usage

Open `demo.html` in a browser. Click the **+1** button to change the count and see the badge bounce.

## Files

- `demo.html` — HTML5 page with badge and JS increment button
- `style.css` — Dark theme styles with `@keyframes badgePop`
- `README.md` — This file
