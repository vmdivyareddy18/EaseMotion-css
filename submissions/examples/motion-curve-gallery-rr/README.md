# CSS Motion Curve Gallery Component

## What does this do?

Provides a visual gallery for comparing CSS easing functions and motion curves through animated examples.

## How is it used?

```html
<div class="curve-card">
    <h3>Ease In</h3>
    <div class="track">
        <div class="dot ease-in"></div>
    </div>
</div>
```

## Why is it useful?

Understanding motion curves is important when creating smooth and engaging UI animations. This component visually demonstrates how different easing functions behave using pure HTML and CSS.
