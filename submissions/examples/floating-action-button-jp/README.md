# Floating Action Button (FAB)

## Description

A modern floating action button component with smooth hover scaling, animated ripple effects, and multiple size variants.

---

## Features

- Hover scale animation
- Continuous ripple effect
- Multiple sizes:
  - Small FAB
  - Default FAB
  - Large FAB
  - Extended FAB
- Smooth transitions
- Responsive layout
- Keyboard focus support
- Reduced motion accessibility

---

## Classes

- `.fab` — Base floating action button
- `.small` — Small size variant
- `.large` — Large size variant
- `.extended` — Extended button with text

---

## Accessibility

Supports keyboard navigation using `:focus-visible` and respects `prefers-reduced-motion`.
