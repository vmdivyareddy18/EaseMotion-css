# CSS Spring Physics Popover (Neumorphic Soft Layouts)

An accessible, responsive soft popover component configured around tactile neumorphic design patterns leveraging fluid cross-axis spring mechanics.

## Exposed Parameters
The following custom properties are exposed globally inside the stylesheet to tweak popover bounce factors:
- `--spring-duration`: Total acceleration vector time.
- `--spring-easing`: The custom `cubic-bezier` timing rule that models soft spring snap recoil.
- `--spring-pullback`: Initial scale and translate offsets applied before deployment.
