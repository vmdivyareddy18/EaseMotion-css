EaseMotion CSS — Media Utilities

Object Scale Down

Demonstrates the utility class ease-object-scale-down. Compare how layout elements containing images scale, crop, or preserve natural proportions.

Object-Fit: Cover

Crops the image to completely fill the bounds, causing circle distortion at the edges.

Object-Fit: Contain

Scales the image to fit within the box. Preserves aspect ratio, but leaves empty side gaps.

Object-Fit: Scale-Down

Dynamically chooses the smaller of none or contain, preventing pixelation on smaller assets.
