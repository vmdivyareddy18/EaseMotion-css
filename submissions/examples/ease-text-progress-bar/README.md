# Text Progress Bar – Progress bar that spells out percentage

> A production-ready text progress bar component that displays the percentage value as text inside the bar with smooth animations, full customization, and interactive controls. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Text percentage** – Percentage displayed inside the bar
- ✅ **Smooth animation** – Fluid progress updates
- ✅ **Play/Pause/Reset** – Full control
- ✅ **Customizable** – Color, speed, size
- ✅ **Manual control** – Slider for direct progress
- ✅ **Random progress** – Jump to random value
- ✅ **Complete animation** – Finish with style
- ✅ **Keyboard shortcuts** – Space (start/pause), R (reset), C (complete)
- ✅ **4 variants** – Blue, Success, Danger, Warning
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
