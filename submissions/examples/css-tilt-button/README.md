# Pure CSS Tilt Hover Button

An ultra-performant, zero-JavaScript interactive tilt button component engineered for creative portfolio layouts and smooth micro-interactions.

## Features
- **Zero JS Overhead:** Relies entirely on highly optimized CSS 3D contexts and hardware-accelerated transforms.
- **Customizable Properties:** Exposes intuitive design tokens (`--tilt-duration`, `--tilt-deg-positive`, `--btn-bg`, etc.) via standard CSS `:root` variables.
- **Accessible Design:** Features keyboard tab-focus accessibility indicators (`:focus-within`) and explicit `aria-label` scaling fallback support.
