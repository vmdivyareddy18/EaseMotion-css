# Blur Entrance Tooltip

A modern marketing landing page tooltip component built entirely with CSS.

## Features

- Pure CSS
- Blur Entrance Animation
- Glassmorphism Design
- Responsive Layout
- Keyboard Accessible
- Focus States
- CSS Variables
- Zero JavaScript

---

## Custom Properties

```css
--duration
--blur
--scale
--radius
--primary
--secondary
```

---

## Accessibility

Supports:

- Hover
- Keyboard navigation
- Focus-visible
- Screen reader compatible tooltip role

---

## Usage

Ideal for:

- SaaS landing pages
- Hero sections
- Pricing cards
- Product showcases
- Marketing websites
