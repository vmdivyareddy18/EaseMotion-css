# Ease Hover Opacity Reduce

This demo implements a **hover opacity reduction** animation where an element dims smoothly when hovered.

## 🧠 How It Works
- The element starts at full opacity (`1`).
- On hover, opacity transitions to `0.6`.
- The transition duration is `0.25s` for a smooth fade.

## 🧩 Files
- `demo.html` — main demo with a hoverable box.
- `style.css` — styling and animation logic.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Hover over the box to see the dimming effect.
3. Adjust opacity or transition duration in `style.css` as needed.

## 🧰 Technologies
- HTML5
- CSS3 (transitions)
