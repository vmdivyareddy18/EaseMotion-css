# CSS Slide Deck

## What does this do?
A lightweight presentation slide deck with CSS transition slides, arrow key navigation, dot indicators, and touch/swipe support.

## How is it used?
Open demo.html in a browser. Navigate slides using left/right arrow keys or clicking the dot indicators. Swipe left/right on touch devices.

## Why is this useful?
This provides a presentation-style component for showcasing content in a slideshow format without any framework dependencies. It demonstrates CSS transform transitions, keyboard event handling, and touch gesture support.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
