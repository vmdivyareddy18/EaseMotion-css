# CSS Morphing Button Component

## What does this do?
Interactive buttons with shape-shifting hover effects including circle-to-rectangle, expanding width, border-to-fill, icon reveal, skew, and shimmer animations.

## How is it used?
```html
<button class="mph-btn mph-circle-rect">Hover Me</button>
<button class="mph-btn mph-icon-expand">
  <span class="mph-icon">&#9654;</span>
  <span class="mph-text">Label</span>
</button>
```

## Why is it useful?
Adds playful, interactive button experiences that guide user attention through morphing shapes and sizes. Pure CSS transitions and animations, no JavaScript needed.
