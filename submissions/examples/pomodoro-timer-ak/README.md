# Pomodoro Timer

## What does this do?
A Pomodoro timer that alternates between 25-minute work sessions and 5-minute breaks with a visual progress ring and session counter.

## How is it used?
Open demo.html in a browser. Click Start to begin the countdown. The progress ring fills as time passes. When a session ends, a tone plays and it auto-switches between work and break modes. Use the mode tabs to manually switch, or Pause/Reset to control the timer.

## Why is it useful?
This provides a complete Pomodoro productivity tool as a single-file frontend component. It demonstrates timer logic, SVG ring animation, and mode toggling using vanilla JavaScript and CSS transitions.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript
