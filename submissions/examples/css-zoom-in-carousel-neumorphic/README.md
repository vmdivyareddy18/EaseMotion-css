# CSS Zoom-In Carousel – Neumorphic Soft

A pure CSS animated carousel with a smooth zoom-in interaction designed for Neumorphic Soft interfaces.

## ✨ Features

- Pure CSS
- Zoom-in animation
- Neumorphic soft shadows
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="carousel">
  <div class="card active">Card</div>
  <div class="card">Card</div>
</div>
```

## Custom Properties

```css
:root{
  --zoom-scale:1.08;
  --duration:.35s;
  --easing:cubic-bezier(.22,1,.36,1);
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component delivers a lightweight neumorphic carousel using only HTML and CSS. It follows the EaseMotion CSS philosophy by being reusable, responsive, accessible, and customizable through CSS variables.
