# Interactive Interface Floating Orbit Accordion

A modern **Pure CSS Floating Orbit Accordion** designed for **Interactive Interface** layouts. The component features subtle floating orbit animations, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Floating Orbit animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Interactive interface inspired design
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:4s;
    --easing:ease-in-out;
    --primary:#7c3aed;
    --secondary:#38bdf8;
}
```

You can customize:

- Animation duration
- Animation easing
- Theme colors
- Border styling

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
