Single-Expand Accordion

An animated, pure-CSS accordion component. Opening a new panel smoothly closes the previous one.

What is GSSoC 2026?

GirlScript Summer of Code is a 3-month-long open-source program conducted every summer by the GirlScript Foundation. It helps newcomers dive deep into the world of open-source development while working on real-world projects under experienced mentors.

How does the single-expand mechanism work?

By leveraging the native HTML name attribute on the <details> element, browsers automatically manage exclusive accordion groups. No JavaScript is required to close one item when another opens!

Can I customize the animation properties?

Absolutely! The entire component is driven by exposed CSS Custom Properties (variables). You can adjust transitions, durations, sizing metrics, and theme colors locally or globally with minimal effort.
