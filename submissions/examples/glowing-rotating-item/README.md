# Glowing Rotating Item

## What does this do?

Creates a glowing item animation with a pulsing box-shadow and slow rotation to highlight special or power-up items.

## How is it used?

```html
<div class="item-glow">
  ⭐
</div>
```

## Why is it useful?

This animation helps draw attention to important elements while remaining lightweight and reusable, aligning with EaseMotion CSS's philosophy of creating elegant and expressive motion effects with minimal code.
