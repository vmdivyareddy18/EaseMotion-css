# Cocktail Bar Template
A high-atmosphere, luxury-focused template designed for cocktail bars, speakeasies, and nightlife lounges.

- **Design**: Uses a "Midnight Luxury" aesthetic with gold accents and dark, textured backgrounds to convey an upscale, intimate night-out experience.
- **Use Case**: Cocktail bars, speakeasies, rooftop lounges, and premium nightlife establishments.
