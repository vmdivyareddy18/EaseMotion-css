How to Use

Basic HTML Structure

<div class="popover-trigger" data-popover="Your text here">
  <span class="trigger-label">Hover me</span>
</div>

CSS Custom Properties (Optional)

/* Customize animation timing */
--jello-duration: 0.4s;

/* Customize popover appearance */
--popover-bg: #f0f9ff;
--popover-text: #0c4a6e;
--popover-border: #0284c7;
--popover-shadow: 0 4px 12px rgba(0, 0, 0, 0.1);

/* Position variations */
.popover-top
.popover-bottom
.popover-left
.popover-right

Examples

/* Change animation speed */
<div class="popover-trigger" data-duration="0.3s">...</div>

/* Change scale factor */
<div class="popover-trigger" data-scale="1.3">...</div>

/* Position at top */
<div class="popover-trigger popover-top">...</div>

/* Apply custom theme */
<div class="popover-trigger" 
     style="--popover-bg: #1f2937; 
            --popover-text: #f3f4f6;">...</div>
