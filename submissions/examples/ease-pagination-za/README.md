# CSS Pagination

## What does this do?
CSS page navigation with multiple sizes (sm, default, lg), styles (default, pill, outline), and ellipsis truncation.

## How is it used?
```html
<nav class="pg-pages" aria-label="Pagination"><ul><li><a href="#">1</a></li><li><a href="#" class="pg-active">2</a></li></ul></nav>
```

## Why is it useful?
Clean page navigation without JavaScript. Three sizes, three visual styles. Ellipsis for large page sets. Active/disabled states. Accessible aria-label support.
