# Cosmic Spiral Collapse Animation Utility

## Overview
This utility creates a cosmic spiral collapse effect where glowing particles orbit inward toward the center and disappear.

## Usage

```html
<div class="ease-spiral-collapse">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first visual interaction
- Great for hero sections, cards, loaders, and futuristic UI
