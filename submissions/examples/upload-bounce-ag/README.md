# Upload Bounce

## What does this do?
An upward bouncing arrow animation for file upload indicators, showing active upload state with a playful bounce effect.

## How is it used?
Apply the `.upload-bounce` class to a container. The arrow icon bounces upward while the progress bar fills to indicate active upload.

## Why is this useful?
Provides clear visual feedback for upload operations, making the process feel more responsive and transparent to users.
