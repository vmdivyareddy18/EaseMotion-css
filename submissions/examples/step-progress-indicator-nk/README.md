# Step Progress Indicator

## What does this do?

Creates a reusable step progress indicator component for multi-step workflows such as onboarding, checkout, and registration flows.

## How is it used?

```html
<div class="stepper">
  <div class="step completed">
    <div class="circle">✓</div>
    <span>Account</span>
  </div>
</div>
```

## Why is it useful?

It provides clear visual feedback during multi-step processes while demonstrating EaseMotion CSS transition and layout capabilities.
