Accessibility Guidance (WCAG 2.1)

Success Criterion 1.3.1

Grid & Info Relationships

Hiding an empty cell visually does not remove it from the accessibility DOM tree. Screen readers will still read out hidden table cells during keyboard navigation. Ensure empty cells have placeholder labels (like "N/A" or "-") or aria-hidden="true" if they are purely decorative layout boxes.

Prerequisites

Border Collapse Rules

The empty-cells property only functions when border-collapse is set to separate. If the table uses border-collapse: collapse;, empty-cell borders are merged into adjacent cells, making empty-cell rules ineffective.
