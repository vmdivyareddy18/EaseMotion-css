# Plasma Video Player

A modern **Plasma Video Player** component built using **pure HTML and CSS** for the **EaseMotion CSS** library.

This component showcases a premium glassmorphism-inspired video player with a glowing plasma border, animated play button, floating background effects, and a responsive layout. It is designed as a reusable UI block that can be integrated into landing pages, dashboards, portfolios, and media applications.

---

## Features

- Modern glassmorphism design
- Animated plasma gradient border
- Pulsing play button
- Smooth CSS-only animations
- Video information section
- Animated progress bar
- Fully responsive
- Accessible HTML structure
- No JavaScript required
- Beginner-friendly and reusable
