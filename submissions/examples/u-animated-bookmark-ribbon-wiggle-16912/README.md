# Animated Bookmark Ribbon Wiggle

## Overview

A reusable bookmark ribbon animation featuring a playful wiggle effect designed for bookmark, save, favorite, and reading list interactions.

## Features

- CSS-only implementation
- Hover-triggered wiggle animation
- Lightweight micro-interaction
- Responsive demo layout
- Accessibility support
- Easy customization

## Usage

```html
<button class="bookmark-button">
  <span class="bookmark-ribbon">🔖</span>
</button>
```

## Accessibility

Supports:

```css
@media(prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
