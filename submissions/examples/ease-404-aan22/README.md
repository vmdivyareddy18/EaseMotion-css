# Ease Animated 404 Error Page

## What does this do?
A beautifully animated 404 error page featuring a glitching "404" text animation, a floating astronaut, animated drifting background particles, and a "Return to Earth" button with a hover effect.

## How is it used?
```html
<div class="ease-404">
  <div class="ease-404__particles">
    <div class="ease-404__particle"></div>
  </div>
  <div class="ease-404__content">
    <div class="ease-404__astronaut">🧑‍🚀</div>
    <div class="ease-404__number">
      <span class="ease-404__digit ease-glitch" data-text="4">4</span>
    </div>
    <h2 class="ease-404__title">Houston, we have a problem!</h2>
    <a href="/" class="ease-404__btn">🚀 Return to Earth</a>
  </div>
</div>
```

## Why is it useful?
404 pages are seen by users when they hit broken links — a missed opportunity for most sites. This fits EaseMotion's animation-first philosophy: a beautifully animated error page turns a frustrating experience into a memorable brand moment.
