# Expand Card

An expandable card component using the CSS checkbox hack for toggling between preview and detailed views.

## Features

- Smooth expand/collapse animation using max-height transitions
- Floating icon animation with CSS keyframes
- Glassmorphism card design
- Fade-in content on expand
- Pure CSS toggle without JavaScript

## Usage

Include `style.css` and use a checkbox input paired with card preview and details sections. The `.card-toggle` checkbox controls the expanded state, with `.card-details` transitioning via `max-height`.
