# Profile Card Component

## Description
A clean, responsive, and reusable `.ease-profile-card` component. It includes a user avatar, name/role display, and an action button row, perfect for dashboards and user directories.

## Features
- **Hover Lift Effect**: Subtle `translateY` animation on hover for better user engagement.
- **Responsive**: Adapts to parent containers with ease.
- **Customizable**: Easy to override styles for specific brand themes.

## Usage
Add the following structure to your HTML:
```html
<div class="ease-profile-card">
    <div class="avatar"></div>
    <h3>Name</h3>
    <p>Role</p>
    <div class="action-row">
        <button>Action 1</button>
        <button>Action 2</button>
    </div>
</div>
