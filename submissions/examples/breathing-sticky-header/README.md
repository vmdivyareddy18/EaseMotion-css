# Breathing Sticky Header

A responsive **Breathing Sticky Header** inspired by modern landing pages. The header stays fixed at the top while displaying a subtle breathing animation for an elegant appearance.

## Features

- Pure HTML & CSS
- Sticky header
- Smooth breathing animation
- Responsive layout
- Accessible navigation
- No JavaScript required

## Folder Structure

```
breathing-sticky-header-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
