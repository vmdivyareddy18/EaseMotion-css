# CSS Ribbon Badges

## What does this do?
Corner ribbon badges using absolute positioning with clip-path and pseudo-element fold effects for promotional and status indicators.

## How is it used?
```html
<div class="rbb-card"><div class="rbb-ribbon rbb-top-right rbb-primary"><span>Featured</span></div><div class="rbb-body"><h3>Title</h3><p>Content</p></div></div>
```

## Why is it useful?
Decorative ribbon badges without JavaScript. Top-right, top-left, bottom-right positions. Color variants. Fold style for depth. Multiple sizes. Ideal for pricing cards and featured items.
