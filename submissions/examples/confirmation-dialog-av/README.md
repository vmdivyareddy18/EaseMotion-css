# Confirmation Dialog

## What does this do?

Creates a modern confirmation dialog with glassmorphism styling, warning icon, project details, and responsive action buttons.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="dialog">

    ...

</div>
```

---

## Why is it useful?

Confirmation dialogs help prevent accidental destructive actions such as deleting projects, removing files, or resetting settings. This standalone component provides a polished, responsive dialog using only HTML and CSS while following EaseMotion CSS's lightweight and animation-first design philosophy.
