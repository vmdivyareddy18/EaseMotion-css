# CSS Positioning Visualizer (`css-positioning-visualizer`)

A lightweight visual demonstration of different CSS positioning modes built using **HTML5** and **Pure CSS**.

### Features

- Static Position
- Relative Position
- Absolute Position
- Fixed Position
- Sticky Position
- Responsive layout
- Pure CSS
- No JavaScript
- Beginner Friendly

### Folder Structure

```text
css-positioning-visualizer/
├── demo.html
├── style.css
└── README.md
```

### Usage

Open `demo.html` directly in any modern browser.

> Note: This example is created for learning CSS positioning concepts visually using only HTML and CSS.

### Browser Support

Works in all modern browsers.
