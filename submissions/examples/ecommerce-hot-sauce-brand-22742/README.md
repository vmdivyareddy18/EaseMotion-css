# E-commerce: Hot Sauce Brand (`ecommerce-hot-sauce-brand-22742`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a bold, personality-driven hot sauce brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A hot sauce brand needs a UI that feels bold, aggressive, and energetic. This template achieves a fiery aesthetic using a high-contrast palette (black, flame red, orange, yellow), heavy display typography (`Anton` & `Inter`), and explosive, popping animations that simulate heat and energy.

The layout includes:
- A stark, high-contrast Navigation.
- A Hero section featuring a CSS-rendered bottle surrounded by explosive sparks and a glowing, flickering background.
- A Product grid categorized by heat level, featuring CSS bottles and aggressive hover shakes.
- A Story section with abstract CSS peppers and a sharp clip-path reveal.
- A Recipes grid with CSS-illustrated food icons (wings, tacos, chili).
- An Awards marquee-style row.
- A dark, high-contrast Reviews grid.
- A bold, fiery Shop CTA.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use energetic, popping `cubic-bezier(0.175, 0.885, 0.32, 1.275)` and `cubic-bezier(0.34, 1.56, 0.64, 1)` timings:
- `.ease-explode-in` (Configured for an explosive scale-up and fade-in entrance).
- `.ease-slide-up` and `.ease-slide-up-stagger` (Sharp, fast vertical entrances).
- `.ease-reveal` (A sharp, horizontal clip-path wipe).
- `.ease-flicker` (A continuous animation simulating the erratic glow of a flame).
- `.ease-float-angry` (A faster, slightly erratic floating animation).
- `.ease-hover-pop` (An aggressive scale and shadow expansion on hover).
- `.ease-hover-shake` (A rapid, shaky animation applied on hover to simulate danger/heat).
- `.ease-hover-flame` (A bold underline expansion).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the bold layout architecture, the high-contrast palette, and the complex abstract CSS illustrations (bottles, peppers, sparks, food icons). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the explosive entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/ecommerce-hot-sauce-brand-22742/` directory to avoid modifying core files directly and to ensure zero deletions.*
