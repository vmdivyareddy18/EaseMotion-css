# Image Hue Shift Hover Effect

An image hover effect using CSS filter hue-rotate() to cycle colors through the spectrum when hovering over an image card.

## Files

- `demo.html` - Image card with hue shift on hover
- `style.css` - hue-rotate filter transition

## Usage

Open `demo.html` and hover over the gradient image to see the hue shift.
