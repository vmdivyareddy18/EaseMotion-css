# E-Commerce Analytics Dashboard Showcase

## What does this do?

Provides an animated e-commerce analytics dashboard with revenue tracking, sales monitoring, product performance insights, and customer activity updates.

## How is it used?

Open `demo.html` directly in a browser and explore the e-commerce analytics interface.

## Why is it useful?

It demonstrates practical e-commerce analytics UI patterns and combines multiple animation techniques into a realistic sales and customer-monitoring experience.
