# CSS Grid Auto Flow Dense Layout

A CSS demo of grid-auto-flow property showing dense packing, row fill, and column fill behaviors in a grid layout with varying item sizes.

## Files

- `demo.html` - Three grid panels with different auto-flow values
- `style.css` - grid-auto-flow property, grid placement, dark theme

## Usage

Open `demo.html` to compare how row, column, and dense auto-flow fill grid items.
