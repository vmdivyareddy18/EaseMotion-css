# Glowing Orb Particles (#36689)

Glowing orb particles with floating bubbles in a dark-themed card layout.

## CSS Custom Properties

| Property           | Default   | Description               |
|--------------------|-----------|---------------------------|
| `--orb-color1`     | `#ff6b6b` | First orb color           |
| `--orb-color2`     | `#4ecdc4` | Second orb color          |
| `--orb-color3`     | `#ffe66d` | Third orb color           |
| `--float-duration` | `5s`      | Duration of float cycle   |

## Interactive Controls

- Color pickers to change orb colors
- Range slider to adjust float speed
