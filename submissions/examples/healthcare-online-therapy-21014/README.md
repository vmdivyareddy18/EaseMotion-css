# Healthcare: Online Therapy Platform (`healthcare-online-therapy-21014`)

This proposal provides a complete, copy-paste ready B2C healthcare landing page designed for an online therapy or mental health counseling platform, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A mental health platform must convey profound safety, empathy, and trustworthiness. This template achieves an "empathetic and safe-space" aesthetic using a very soft, calming color palette (deep forest greens, soft mints, warm sand/oatmeal backgrounds), a warm typography system featuring Lora (serif) and Nunito (sans-serif), heavily rounded corners (`border-radius: 24px`), and gentle, slow-paced CSS entrance animations.

The layout includes:
- A hero section featuring abstract soft CSS blur shapes and a uniquely rounded image frame (to avoid sharp, clinical edges).
- A 3-step "How it Works" guide.
- A split-layout "Therapist Credentials" showcase emphasizing vetting and trust.
- A grid outlining different therapeutic approaches (CBT, DBT, EMDR).
- A highly prominent, dark-green privacy and HIPAA compliance banner.
- Anonymized patient testimonials to build social proof while respecting privacy.
- A clear pricing and insurance section.
- A comprehensive footer with emergency crisis contact information.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (Configured with a softer, slower cubic-bezier (`0.25, 1, 0.5, 1`) and longer durations to create a calming, unhurried entrance effect)
- `.ease-zoom-in` (Applied to the hero image frame to draw the eye softly)
- `.ease-slide-left` / `.ease-slide-right` (Used for the floating descriptive cards in the hero section)
- `.ease-hover-lift` (Standard interaction for primary buttons and approach cards to encourage engagement)
- `.ease-hover-scale` (Applied gently to the logo)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, organic shapes, typography, and the extremely rounded "safe" UI elements, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/healthcare-online-therapy-21014/` directory to avoid modifying core files directly and to ensure zero deletions.*
