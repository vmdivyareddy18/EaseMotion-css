# Brightness Filter Showcase - EaseMotion CSS

A beautiful, interactive component showcase presenting the brightness filter transition classes.

## Features Included

- `.ease-filter-brightness`: Applies `filter: brightness(100%) !important` to an element.
- `.ease-filter-brightness-50`: Applies `filter: brightness(50%) !important` to an element.
- `.ease-filter-brightness-150`: Applies `filter: brightness(150%) !important` to an element.

## How to Run

1. Open `demo.html` directly in any modern web browser.
2. Observe normal, dimmed, and bright showcase variations side-by-side.

## Why is it useful?

Brightness filter utilities allow developers to control visual illumination states of images, icons, and cards. This helps implement ambient glows, active highlight effects, dimmed inactive card overlays, and dynamic visual indicators.
