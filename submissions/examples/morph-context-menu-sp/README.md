# Morph Context Menu

A nature-inspired context menu with a smooth morphing animation built using pure HTML and CSS.

## Features

- 🌿 Nature-inspired design
- ✨ Morphing scale animation
- 💻 Pure HTML & CSS
- 📱 Responsive layout
- ♿ Keyboard accessible using `:focus-within`
- 🚫 No JavaScript required

## File Structure

```
morph-context-menu-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Hover over or focus on the button to reveal the morphing context menu.

## Use Cases

- Dashboard actions
- File explorers
- Design tools
- Productivity applications
- Admin panels
