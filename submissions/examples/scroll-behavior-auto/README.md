# scroll-behavior-auto

## Purpose
Compares `scroll-behavior: smooth` vs `scroll-behavior: auto` for anchor link scrolling and programmatic `scrollTo` calls.

## Features
- Side-by-side panels with smooth and auto scroll behavior
- Anchor link navigation with both modes
- JavaScript `scrollTo` buttons for comparison
- Clear visual feedback during scroll

## Usage
Open `demo.html` and click the anchor links or buttons. Watch the left panel scroll smoothly while the right panel jumps instantly.
