# Animated Social Share Buttons Bar

A floating, fully responsive **Social Share Bar** perfect for blogs, articles, and content pages. This example demonstrates how to combine **EaseMotion CSS** for staggered icon entrances, continuous floating effects, and smooth hover transitions to create an eye-catching sharing component that encourages social engagement.

## 🚀 Features

- **Staggered Icon Entrances**: Each social icon uses `ease-bounce-in` with `ease-delay-*` to playfully pop in sequentially when the bar appears.
- **Continuous Floating Effect**: The entire bar has a subtle vertical float animation to make it feel alive and draw attention.
- **Smooth Bar Entrance**: The share bar slides in from the left using `ease-slide-in-left` for a polished reveal.
- **Brand-Colored Hover States**: Each button transitions to its brand color on hover with a smooth background fill effect.
- **Interactive Icon Scaling**: Icons scale up slightly on hover using `ease-hover-grow` for satisfying tactile feedback.
- **Responsive Positioning**: On desktop, the bar floats vertically on the left side. On mobile, it repositions to the bottom center.
- **Accessible**: Includes `aria-label` attributes for screen readers.
