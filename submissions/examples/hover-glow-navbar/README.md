# Hover Glow Navbar

## What does this do?
A responsive navbar with hover glow, active underline animation, and keyboard-visible focus states.

## How is it used?
```html
<nav class="glow-navbar">
  <a class="active" href="#">Home</a>
  <a href="#">Work</a>
</nav>
```

## Why is it useful?
It gives EaseMotion CSS a clean navigation pattern where motion clarifies hover, active, and focus states.
