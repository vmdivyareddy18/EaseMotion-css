# Responsive Grid Columns

1. **What does this do?**  
   Adds responsive grid column utility ideas for quickly switching layouts from one column to two, three, or four columns.

2. **How is it used?**  
   ```html
   <div class="grid-cols-3 grid-collapse-mobile">
     <article>Card one</article>
     <article>Card two</article>
     <article>Card three</article>
   </div>
   ```

3. **Why is it useful?**  
   It fits EaseMotion CSS by giving developers readable layout classes that make responsive grids easy to describe and reuse.
