# Ease Build Queue Status

## Overview
A CSS-only component for visualizing CI/CD build queue statuses.

## Features
- Status badges
- Responsive layout
- Hover animations
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component ideal for DevOps dashboards, CI/CD pipelines, and developer monitoring tools.
