## Overview

A responsive dashboard for visualizing animation regression testing results, performance metrics, and comparison reports. Designed for animation testing pipelines, QA dashboards, design systems, CI/CD validation tools, and frontend quality assurance workflows.

## Features

- Regression testing KPI cards
- Animation regression timeline
- Latest findings panel
- Responsive dashboard layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="dashboard">

    <section class="stats">
        ...
    </section>

        ...
```
