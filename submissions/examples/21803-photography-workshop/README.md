# Photography Workshop Landing Page (#21803)

## Overview

This submission delivers a premium editorial-style business landing page for a luxury photography workshop. The page is fully self-contained, opens directly in the browser, and stays within the repository submission rules by using only local HTML and CSS with an inline SVG hero illustration.

## Features

- Hero section with a detailed inline SVG photography illustration:
  DSLR camera body, layered lens reflections, aperture blades, viewfinder, floating photo frames, film strip, bokeh circles, soft light rays, a photography grid, glow treatment, and floating glass information cards.
- Workshop overview with premium glassmorphism feature cards.
- Instructor profile with an inline SVG portrait card.
- Student gallery built without external images or assets.
- Schedule, pricing, enrollment form, testimonials, FAQ, and footer.
- Smooth hover states, floating motion, polished spacing, and editorial typography using CSS only.

## Responsive Behavior

- Desktop:
  Two-column hero, spacious editorial layout, floating overlay cards, and multi-column content grids.
- Tablet:
  Key sections collapse to simpler grids while preserving hierarchy and breathing room.
- Mobile:
  Single-column flow, full-width actions, stacked cards, and touch-friendly spacing for the form and FAQ.
