# Dependency Risk Ledger

Dependency Risk Ledger is a responsive supply-chain review panel for tracking package risk before release. It highlights high, medium, and low dependency risk items with short ownership notes.

## Features

- Supply-chain review summary
- Dependency rows with risk labels
- Clear high, medium, and low states
- Responsive layout for narrow screens

## Files

- `demo.html` contains the dependency risk content.
- `style.css` contains the ledger layout and status styles.
