ease-tada

Celebration animation utility — scale + rotation wiggle effect. Great for achievement badges, completion states, and welcome animations.

Use cases

Achievement Unlock

🏆

Achievement badge with infinite tada loop.

Button Celebration

🎉

Click to trigger the animation.

Fast Pulse

✨

Fast variant (.ease-tada-fast) at 0.6s.

Slow & Subtle

⭐

Slow variant (.ease-tada-slow) at 1.5s.

CSS source

@keyframes ease-kf-tada {
  0% { transform: scaleX(1) scaleY(1) rotate(0deg); }
  10%, 20% { transform: scaleX(0.9) scaleY(0.9) rotate(-3deg); }
  30%, 50%, 70%, 90% { transform: scaleX(1.1) scaleY(1.1) rotate(3deg); }
  40%, 60%, 80% { transform: scaleX(1.1) scaleY(1.1) rotate(-3deg); }
  100% { transform: scaleX(1) scaleY(1) rotate(0deg); }
}

.ease-tada {
  --ease-tada-duration: 1s;
  animation: ease-kf-tada var(--ease-tada-duration) var(--ease-ease) both;
  animation-iteration-count: var(--ease-animation-iterations, 1);
}

Usage

<!-- One-shot tada -->
<div class="ease-tada">🎉</div>

<!-- Infinite looping -->
<div class="ease-tada ease-tada-infinite">🏆</div>

<!-- Fast variant -->
<div class="ease-tada ease-tada-fast">✨</div>

<!-- Custom duration -->
<div class="ease-tada" style="--ease-tada-duration:2s;">⭐</div>
