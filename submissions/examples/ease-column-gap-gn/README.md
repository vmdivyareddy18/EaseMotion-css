# Ease Column Gap Utilities

1. **What does this do?** Adds `column-gap` utility classes using CSS design tokens for consistent spacing in grid and flex layouts.

2. **How is it used?**
```html
   <!-- Grid layout -->
   <div class="grid ease-col-gap-4">...</div>

   <!-- Flex layout -->
   <div class="flex ease-col-gap-6">...</div>

   <!-- Responsive -->
   <div class="ease-col-gap-2 ease-col-gap-md-6">...</div>
```

3. **Why is it useful?** Consistent column-gap utilities using CSS variables — human-readable class names, responsive variants, and token-based spacing fit EaseMotion CSS's utility-first philosophy perfectly.
