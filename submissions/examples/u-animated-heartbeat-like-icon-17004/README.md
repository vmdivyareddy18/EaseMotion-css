# Animated Heartbeat Like Icon

## Overview

A reusable heart icon animation featuring a subtle heartbeat effect suitable for likes, favorites, reactions, and engagement indicators.

## Features

- CSS-only animation
- Heartbeat scale effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="like-button">
  <span class="heart-icon">❤</span>
</button>
```

## Accessibility

Supports:

```css
@media(prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
