# prefers-contrast Media Query

Demonstrates the `prefers-contrast` media query for adapting designs to user contrast preferences. Shows high-contrast (more) and low-contrast (less) variants with buttons, form fields, links, and visual bars.

## Usage

Open `demo.html` and adjust your OS contrast setting to see cards and elements adapt. Use Windows High Contrast mode or macOS Accessibility settings. The comparison bars show visual intensity across contrast levels.

## Browser Support

`prefers-contrast` is supported in Chrome 96+, Firefox 101+, Safari 16+, and Edge 96+. The demo uses CSS variables for dynamic opacity and border thickness. Dark theme `#0a0f1e` background with `#e2e8f0` text ensures a consistent baseline.
