# Unit Converter

## What does this do?
A unit converter with category tabs for Length, Weight, Temperature, and Volume. Supports bidirectional conversion between units within each category.

## How is it used?
Open demo.html in a browser. Select a category tab, enter a value, and choose source/target units. The result updates in real-time.

## Why is this useful?
This provides a versatile conversion tool suitable for educational tools, engineering calculators, or recipe converters. It demonstrates category-based tab switching and multi-unit conversion logic.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the converter.
