# Animated Drip Feed Tooltip

**What does this do?**  
Tooltips that reveal text character by character (drip-feed animation) when hovering over a trigger element. Uses CSS `steps()` for the typing effect.

**How is it used?**  
Add `data-tooltip="your text"` to any element with class `.tooltip-trigger`. The tooltip appears above the trigger on hover with a smooth character-by-character reveal. Arrow indicator included.

**Why is it useful?**  
Adds a delightful typing feel to tooltips, drawing attention to each word. Pure CSS, zero JavaScript, works with buttons, icons, and links.
