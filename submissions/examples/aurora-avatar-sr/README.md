# Aurora Avatar

## What does this do?

Creates an animated aurora-style avatar using HTML and CSS.

## How is it used?

```html
<div class="aurora-avatar ease-float">
    <div class="aurora-ring ring-one"></div>
    <div class="aurora-ring ring-two"></div>

    <div class="avatar-shell">
        <img src="avatar.jpg" alt="Profile Avatar">
    </div>
</div>
```

## Why is it useful?

It provides a reusable, responsive, and lightweight animated avatar component for blogs, portfolios, and profile sections.
