# EaseMotion CSS — Pagination Component Showcase

A premium, interactive showcase displaying the **EaseMotion CSS** Pagination component.

## Showcase Features

*   **Responsive Pagination Scaling**: Previews `.ease-pagination-sm`, standard size, and `.ease-pagination-lg` buttons.
*   **Aria-Accessible Attributes**: Incorporates `aria-label` properties on button elements to align with visual navigation benchmarks.
*   **Pulsing Active State Indicators**: Custom styled active buttons utilizing repeating glows.
*   **Disabled Previous / Next Controls**: Illustrates disabled previous/next controls using `disabled` selectors.
*   **Interactive Dynamic Dataset**: Fully functional JavaScript data list that paginates rows immediately on click events.

---

## EaseMotion CSS Classes Showcased

### Layout & Spacing
*   `.ease-flex` — Standard display flex.
*   `.ease-justify-between` — Aligns list footer elements.
*   `.ease-items-center` — Vertically aligns content items.
*   `.ease-gap-2` — Spacing between buttons.
*   `.ease-stack-lg` — Vertical layout spacing.

### Components

#### Pagination
*   `.ease-pagination` — Core pagination layout block.
*   `.ease-page-btn` — Standard page button styling.
*   `.ease-page-btn-active` — Color accent styling indicating the current page.
*   `.ease-page-btn-ellipsis` — Non-interactive dots representing hidden intermediate pages.
*   `.ease-pagination-sm` / `.ease-pagination-lg` — Sizing modifier classes.

#### Cards
*   `.ease-card` — Standard card layout container.
*   `.ease-card-outlined` — Outlined border variant.
*   `.ease-card-header` / `.ease-card-body` — Cards content organization sections.
*   `.ease-card-flat` — Container background panel.

#### Badges & Typographies
*   `.ease-badge` — Status indicators.
*   `.ease-font-semibold` — Bold titles.

### Animations
*   `.ease-fade-in` — Entrance loading animation.
