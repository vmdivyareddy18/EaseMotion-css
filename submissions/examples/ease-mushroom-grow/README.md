# Ease Mushroom Grow Animation

An organic, playful CSS animation mimicking the pop-up sprout expansion dynamics of a forest mushroom using targeted scale axis metrics and specialized elastic easing handles.

## How to View
1. Run `demo.html` in any web browser ecosystem.
2. Hit the "Trigger Growth" element to watch the cycle restart.

## Technical Details
- **Stack:** HTML5 structural nodes / CSS3 Keyframes & Transform matrices
- **Easing Curve:** `cubic-bezier(0.34, 1.56, 0.64, 1)` to induce a cartoonish overshooting elasticity effect.
