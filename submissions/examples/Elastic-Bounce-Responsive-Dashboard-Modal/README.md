# Elastic Bounce Responsive Dashboard Modal

A responsive dashboard modal featuring a smooth Elastic Bounce entrance animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive dashboard layout
- Elastic Bounce animation
- Keyboard accessible
- KPI statistic cards
- Simple chart visualization
- Customizable CSS variables
- No JavaScript required

```css
--duration
--scale
--primary
--secondary
--bg
--surface

```
