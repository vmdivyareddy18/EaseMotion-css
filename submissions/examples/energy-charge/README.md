# Energy Charge Animation

## What does this do?

Creates a battery-style energy charging animation with a rising fill level and floating energy particles, all in pure CSS.

## How is it used?

Add the `.battery` container with `.battery-body`, `.charge-level`, and `.particles` sub-elements.

```html
<div class="battery">
  <div class="battery-head"></div>
  <div class="battery-body">
    <div class="charge-level"></div>
  </div>
</div>
```

## Why is it useful?

Energy charging animations are common in game UIs, power-up indicators, battery displays, and loading screens. This pure CSS approach provides an engaging visual without JavaScript.

## Tech Stack

- HTML
- CSS (no frameworks, no JavaScript)

## Preview

Open `demo.html` directly in your browser to see the effects.
