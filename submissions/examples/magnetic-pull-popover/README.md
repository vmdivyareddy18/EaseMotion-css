# CSS Magnetic Pull Popover

A pure CSS animated popover with a smooth magnetic pull interaction effect.

## Features

- CSS-only animation
- Magnetic movement effect
- Responsive layout
- Keyboard accessible
- Customizable CSS variables
- No JavaScript required

## Usage

```html
<div class="magnetic-popover">

<button class="magnetic-trigger">
Hover Me
</button>

<div class="magnetic-content">
Content here
</div>

</div>
