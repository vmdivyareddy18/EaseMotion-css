# CSS Lazy Load Blur Component

## What does this do?
Blur-up image loading effect with progressive reveal animations including standard blur, skeleton-to-blur, and scale reveal transitions.

## How is it used?
```html
<div class="llb-image llb-blur-up"></div>
<div class="llb-image llb-skeleton-blur"><div class="llb-shimmer"></div></div>
```

## Why is it useful?
Creates smooth progressive image loading experiences similar to Medium and Pinterest. GPU-accelerated blur and transform animations. Multiple variants for different loading patterns.
