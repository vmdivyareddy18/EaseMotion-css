Scroll-to-Bottom Button

A fixed-position button that smoothly animates into view when you're near the top of the page, and scrolls you to the bottom on click. Scroll down to see the button appear!

Scroll down

How it Works

The button uses CSS opacity and transform transitions. When the user scrolls past a threshold (20% of the page), the .scroll-to-bottom-btn--visible class is toggled via JavaScript.

Once visible, clicking the button smoothly scrolls the page to the very bottom using window.scrollTo({ behavior: 'smooth' }). When the user is near the bottom, the button hides itself again.

Animation Details

The enter transition slides the button up from 12px below its final position while scaling from 0.9 to 1.0. The exit reverses this — it fades and sinks back down. The hover state adds a gentle downward nudge and glow shadow, and the active (pressed) state scales the button down slightly for tactile feedback.

Use Cases

Chat interfaces, live activity feeds, log viewers, comment sections — any context where the latest content lives at the bottom of a long page.

Integration

Composable with existing ease-btn classes for additional styling. The visibility toggle can be handled by a tiny JavaScript snippet or a dedicated Intersection Observer.
