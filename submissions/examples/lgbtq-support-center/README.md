# Community Support Center

A responsive nonprofit landing page showcasing a welcoming community support organization built using EaseMotion CSS.

## Features

- Hero section
- Community services
- Coming Out resources
- Volunteer opportunities
- Donate call-to-action
- Events section
- Community stories
- Responsive footer

## EaseMotion Classes Used

- ease-slide-up
- ease-hover-scale
- ease-hover-lift
