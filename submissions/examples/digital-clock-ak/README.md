# Live Digital Clock

## What does this do?
A live digital clock that displays the current time in 12h or 24h format with the full date, updating every second.

## How is it used?
Open demo.html in a browser. The clock shows the current time and updates in real-time. Click "12h" or "24h" to switch between formats.

## Why is it useful?
This provides a clean real-time clock component suitable for dashboards, status bars, or full-screen time displays. It demonstrates real-time DOM updates and time formatting with vanilla JavaScript.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see the live clock.
