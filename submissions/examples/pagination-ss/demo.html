♿ Accessibility

  • ✅ <nav aria-label="Pagination"> — semantic navigation landmark
  • ✅ <ul> structure — list semantics for screen readers
  • ✅ aria-current="page" — active page marked for assistive tech
  • ✅ focus-visible outline — keyboard navigation visible
  • ✅ prefers-reduced-motion — disables staggered entrance
  • ✅ Disabled state — not-allowed cursor, reduced opacity
  • ✅ Jump input — aria-label for context, min/max validation
