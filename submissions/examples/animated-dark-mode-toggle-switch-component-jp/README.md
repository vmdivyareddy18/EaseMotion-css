# Animated Dark Mode Toggle Switch Component

## Overview

A modern CSS-only dark mode toggle switch with smooth sliding animations, animated sun and moon icons, glowing effects, and interactive hover states.

## Features

- Smooth toggle sliding animation
- Animated sun and moon icons
- Glowing border and hover effects
- Modern gradient switch design
- Lightweight CSS-only implementation
- Easy to customize and integrate

## Files

- demo.html – Demonstrates the animated dark mode toggle
- style.css – Contains toggle styles and animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in a modern browser and click the toggle switch to experience the animated transition between sun and moon modes.
