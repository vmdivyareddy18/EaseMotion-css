# Ease Notification Inbox

## Overview
A CSS-only notification inbox component for dashboards and SaaS applications.

## Features
- Animated unread indicator
- Responsive layout
- Hover animation
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
Provides a reusable, animation-first notification component that is lightweight and easy to customize.
