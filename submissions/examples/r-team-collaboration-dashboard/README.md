# Team Collaboration Dashboard

A responsive Team Collaboration Dashboard example built with EaseMotion CSS.

## Features

- Responsive dashboard layout
- Team member cards
- Project overview section
- Activity timeline
- Smooth animations
- Mobile-friendly design

## Folder Structure

```text
r-team-collaboration-dashboard/
├── README.md
├── demo.html
└── style.css
```

## How to Run

Open `demo.html` in any modern browser.

## EaseMotion Features Used

- Fade-in animations
- Hover effects
- Smooth transitions
- Responsive layouts
