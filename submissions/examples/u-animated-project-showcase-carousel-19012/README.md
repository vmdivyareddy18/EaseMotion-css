# Animated Project Showcase Carousel

A modern project showcase component featuring animated project cards, navigation controls, category badges, and responsive layouts.

## Features

- Project showcase cards
- Featured project highlight
- Navigation controls
- Progress indicators
- Hover interactions
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Project titles
- Descriptions
- Category badges
- Colors
- Animation timing
- Layout spacing

## Use Cases

- Developer portfolios
- Agency websites
- Startup landing pages
- Product showcases
- Open source projects
