# Animated Image Gallery Filter

A responsive **Animated Image Gallery Filter** built using **pure HTML and CSS**. This example demonstrates a modern gallery layout with animated filter buttons, fade-in effects, and image hover scaling.

## Features

- Pure HTML & CSS
- Responsive CSS Grid gallery
- Animated category filter buttons
- Fade-in animation using `@keyframes`
- Scale hover transition for gallery items
- CSS Custom Properties for easy customization
- Responsive design
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
animated-image-gallery-filter/
├── demo.html
├── style.css
└── README.md
```

## Customization

Edit the CSS variables in the `:root` section to customize:

- Primary color
- Background color
- Card shadow
- Border radius
- Transition speed

## Browser Support

Compatible with all modern browsers.
