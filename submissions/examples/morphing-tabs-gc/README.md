# Morphing Tab Menu UI Example

## 1. What does this do?

This submission creates a morphing tab menu with an animated underline that smoothly moves between active tabs and content panels that fade in on selection.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes a tab list and content sections. Copy the `.morphing-tabs` markup and CSS into your project to build animated tabbed navigation.

---

## 3. Why is it useful?

Animated tabs improve navigation clarity and make content switching feel fluid. The underline movement and fading transitions create a polished interaction for tabbed interfaces.
