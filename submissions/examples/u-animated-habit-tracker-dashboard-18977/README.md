# Animated Habit Tracker Dashboard

A responsive productivity dashboard featuring habit tracking, streak counters, progress indicators, weekly completion tracking, and achievement badges.

## Features

- Animated progress bars
- Habit cards
- Weekly tracker
- Achievement badges
- Streak counter
- Hover interactions
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in your browser.

## Customization

- Habit names
- Progress percentages
- Achievement badges
- Colors
- Animations
- Layout spacing

## Use Cases

- Productivity apps
- Habit tracking platforms
- Wellness dashboards
- Learning trackers
- Personal development tools
