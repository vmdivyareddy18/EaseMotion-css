# Ease Yellow Fade

A gentle yellow glow that fades in and out, like a warm pulse of light.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.glow` element.

## Custom CSS Variables

| Variable          | Default  | Description               |
|-------------------|----------|---------------------------|
| `--fade-from`     | `#ffe082`| Inner glow / peak color   |
| `--fade-to`       | `#ffd54f`| Outer edge / dim color    |
| `--fade-duration` | `2s`     | Duration of one fade cycle|

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
