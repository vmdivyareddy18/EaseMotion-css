How to Use

Basic HTML Structure

<div class="swing-popover" data-popover="Your text here">
  <span class="popover-label">Hover me</span>
</div>

CSS Custom Properties (Optional)

/* Customize animation timing */
--swing-duration: 0.5s;
--swing-angle: 15deg;

/* Customize popover appearance */
--popover-bg: #f3f4f6;
--popover-text: #1f2937;
--popover-accent: #6366f1;

/* Direction variations */
.swing-from-top
.swing-from-bottom
.swing-from-left
.swing-from-right

Practical Examples

/* Portfolio card with top swing */
<div class="portfolio-card swing-popover swing-from-top" 
     data-popover="View project">
  <img src="project.jpg" alt="Project">
  <h3>Project Name</h3>
</div>

/* Skill badge with center swing */
<div class="skill-badge swing-popover" 
     data-popover="Expert in UI/UX">
  Design
</div>

/* Custom timing and colors */
<div class="swing-popover" 
     data-popover="Custom swing"
     style="--swing-duration: 0.4s;
            --popover-bg: #1f2937;
            --popover-text: #f3f4f6;">
  Dark Mode
</div>
