# Ease Cyber Data Pods

## Overview

A premium dashboard component featuring futuristic data pods with animated energy lines and glassmorphism styling built using only HTML and CSS.

## Features

* Animated energy line
* Glassmorphism cards
* Responsive CSS Grid
* Hover lift interaction
* Lightweight implementation
* Pure HTML & CSS

## Folder Structure

```text
ease-cyber-data-pods-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

* Chrome
* Firefox
* Edge
* Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.
