# Pill Switch Snap

A CSS-only pill switch with a snappy thumb movement and a soft active-state glow.

## Files

- `demo.html` - accessible checkbox-based switch markup.
- `style.css` - track color transition, thumb movement, focus-visible state, and reduced-motion support.

## Highlights

- Uses a native checkbox for state.
- Keeps focus styling visible for keyboard users.
- Includes a reduced-motion fallback.
