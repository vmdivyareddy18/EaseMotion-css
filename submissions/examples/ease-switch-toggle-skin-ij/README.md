# Switch Toggle Skin

Toggle switch component with multiple skin styles and color options.

## Features

- Three skin variants: Pill (rounded), Square, and Material
- Five color presets with visual color picker
- Icons inside the handle that change on toggle state
- Smooth handle translation and track background transitions

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--sts-duration` | `0.3s` | Toggle animation speed |
| `--sts-track-bg` | `#334155` | Track off-state background |
| `--sts-handle-bg` | `#ffffff` | Handle background |
| `--sts-active-color` | `#22c55e` | Track on-state background |
| `--sts-radius` | `999px` | Default border radius |
