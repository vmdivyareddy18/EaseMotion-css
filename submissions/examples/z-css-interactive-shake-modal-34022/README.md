# CSS Micro Shake Modal

Pure CSS animated modal utilizing a Micro Shake interaction designed for Interactive Interface layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS custom properties
- Interactive UI styling
- Micro shake animation
- Glassmorphism design
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--duration
--ease
```
