# fieldset-legend-styling

## Purpose
Demonstrates `fieldset` and `legend` element styling for grouped form controls with custom legend positioning and decoration.

## Features
- Custom fieldset borders, backgrounds, and padding
- Styled legend with positioned decoration and badges
- Grouped form controls with visual hierarchy
- Disabled fieldset state styling

## Usage
Open `demo.html` to see styled fieldset groups with custom legends for different form sections.
