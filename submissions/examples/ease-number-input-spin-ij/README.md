# Number Input Spin

A number input with up/down buttons that changes value with a scale animation on change.

## Features

- Increment / decrement buttons
- Scale transform animation on value change
- Focus border highlight

## CSS Custom Properties

| Property                    | Default   | Description              |
| --------------------------- | --------- | ------------------------ |
| `--nis-transition-duration` | `0.2s`    | Duration of animations   |
| `--nis-accent-color`        | `#6c63ff` | Accent / button color    |
| `--nis-border-color`        | `#ccc`    | Border color             |

## Usage

Click the + and − buttons to adjust the numeric value.
