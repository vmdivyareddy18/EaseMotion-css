# Lighthouse Beam

A pure CSS and HTML lighthouse at night, its lamp sweeping a soft beam across the starry sky above a dark sea.

## What it shows

- A rotating light: a blurred clip-path wedge of beam pivots from the lamp room, sweeping back and forth from a small pivot element
- The tapered tower cut from a clip-path trapezoid, with candy stripe banding from a repeating linear gradient clipped to the same shape
- A glowing lamp room from a warm radial gradient behind a caged glass of vertical bars, pulsing its halo
- A rippling sea from a scrolling repeating linear gradient over the rocks
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
