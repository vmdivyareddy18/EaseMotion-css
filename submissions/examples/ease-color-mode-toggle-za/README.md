# Color Mode Toggle

## What does this do?
CSS dark/light mode toggle switches with sun/moon icons, slider, label, button group, and icon-only variants.

## How is it used?
```html
<label class="cm-switch cm-sunmoon"><input type="checkbox"><span class="cm-slider"><span class="cm-icon cm-sun">☀</span><span class="cm-icon cm-moon">☾</span></span></label>
```

## Why is it useful?
Provides multiple toggle UI patterns for theme switching without JavaScript. Animated transitions between modes. Customizable colors via CSS variables. Demo preview shows light/dark mode effect.
