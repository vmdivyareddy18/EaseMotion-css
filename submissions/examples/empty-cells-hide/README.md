# empty-cells-hide

## Overview

Demonstrates the `empty-cells` CSS property for controlling whether borders and backgrounds are shown in empty table cells (`<td>` with no content). Compares `empty-cells: show` (default) and `empty-cells: hide` on identical data, including a split comparison with colored backgrounds.

## Usage

Open `demo.html` in any browser. The first two tables show the same sales data with empty cells rendered differently. The third section provides a side-by-side split view with colored backgrounds to highlight how `empty-cells: hide` removes both border and background from empty cells.

## Features

- Side-by-side comparison of show vs hide on identical dataset (sales table with missing quarters)
- Colored background section highlighting the visibility difference
- Split grid comparison for direct visual contrast
- Responsive grid layout
- `prefers-reduced-motion` support
- Dark theme with yellow accent and blue headers
