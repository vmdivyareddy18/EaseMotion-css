## align-self-individual

Explores the `align-self` property for overriding alignment on individual flex and grid items within a container.

### Key Features

- Flexbox and Grid demos showing `stretch`, `center`, `start`, `end`, `baseline`, and `auto`
- Individual item alignment overriding the parent `align-items` value
- Mixed-height gallery demonstrating per-item alignment in a uniform container

### Property Values

`align-self` accepts: `auto`, `stretch`, `center`, `flex-start`, `flex-end`, `start`, `end`, `baseline`, `first baseline`, `last baseline`. The `auto` value inherits the container's `align-items` setting.
