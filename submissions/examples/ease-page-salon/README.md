# Full Barbershop Landing Page (`ease-page-salon`)

A complete, production-ready landing page template demonstrating the power and flexibility of EaseMotion CSS utilities when combined in a real-world scenario.

## 🚀 Features & EaseMotion Showcase

This landing page integrates multiple core `.ease-*` classes alongside an `IntersectionObserver` script to create a stunning, scroll-triggered experience:

- **Scroll Entrances**: Utilizes `.ease-slide-up`, `.ease-slide-left`, `.ease-slide-right`, and `.ease-fade-in` with CSS `animation-delay` staggered properties to create cascading entry animations as the user scrolls down the page.
- **Interactive Galleries**: The Instagram-style gallery applies `.ease-hover-zoom` to scale images up on hover smoothly.
- **Card Hover States**: The barber profiles and the map placeholder utilize `.ease-hover-lift` to add a physical depth-bounce on mouseover.
- **Premium Buttons**: All Call-to-Action buttons use `.ease-hover-scale` combined with a sleek gold/dark color palette for a highly clickable aesthetic.

## 🛠️ Usage

This demo is entirely self-contained. You can open `demo.html` directly in your browser. All required EaseMotion CSS classes have been defined at the top of the `style.css` file for immediate testing. 

If using this as a template for a real project, simply migrate the HTML structure into your application and ensure your global CSS bundle includes the `ease-*` utility classes.
