# Animated Upload Cloud Bounce

## Overview

Animated Upload Cloud Bounce is a lightweight CSS animation that simulates a cloud upload process using a bouncing cloud and rising upload arrow.

## Features

- CSS-only animation
- Cloud bounce effect
- Upload arrow motion
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<div class="upload-cloud">
  <span class="cloud">☁️</span>
  <span class="arrow">⬆️</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
