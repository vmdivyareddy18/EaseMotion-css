# Custom Animated Radio Buttons (`ease-radio-button-custom-ij`)

Animated radio buttons with a fill ripple effect and check mark indicator. Part of the EaseMotion CSS submissions.

## Features

- Smooth border-to-fill transition on selection
- Expanding ripple animation from the selected circle
- Scaling check mark appearing inside
- Keyboard accessible (hidden native input remains in DOM)
- ARIA radiogroup for screen reader support

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--rbc-duration` | `0.3s` | Transition duration for all animations |
| `--rbc-outer-color` | `#cbd5e1` | Outer ring and fill color |
| `--rbc-inner-color` | `#ffffff` | Inner circle background |
| `--rbc-check-color` | `#ffffff` | Check mark dot color |
| `--rbc-label-color` | `#1e293b` | Label text color |

## Usage

Open `demo.html` in any modern browser. Click any radio option to see the animated transition.
