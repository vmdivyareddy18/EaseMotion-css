# Justify-Items Utilities

1. What does this do?
Provides utility classes to set the default inline-axis alignment for all child elements inside a grid container.

2. How is it used?
Apply classes like `.justify-items-start`, `.justify-items-center`, `.justify-items-end`, or `.justify-items-stretch` directly to a grid container:
```html
<div class="grid-container justify-items-center">
  <div>Item 1 (Centered)</div>
  <div>Item 2 (Centered)</div>
  <div>Item 3 (Centered)</div>
</div>
```

3. Why is it useful?
It offers container-level alignment control over grid columns, allowing developers to establish a uniform, clean layout structure efficiently without needing to define layout classes on individual children.
