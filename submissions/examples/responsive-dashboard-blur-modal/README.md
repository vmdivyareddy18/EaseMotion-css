# Responsive Dashboard Blur Entrance Modal

A pure CSS responsive modal featuring a smooth blur entrance animation for dashboard-style interfaces.

## Features

- Pure CSS
- Blur Entrance Animation
- Responsive Layout
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript Required

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## CSS Variables

```css
--duration
--ease
--radius
--scale
--overlay
--primary
```

These variables allow customization of animation speed, easing, colors, scale, and border radius.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## EaseMotion CSS

This component follows the EaseMotion CSS philosophy by providing a reusable, animation-first modal built entirely with CSS, requiring zero JavaScript while remaining responsive and accessible.
