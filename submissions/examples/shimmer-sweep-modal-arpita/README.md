# Cyberpunk Shimmer Sweep Modal
A pure HTML and CSS modal featuring an animated shimmer sweep effect with glowing cyberpunk neon aesthetics for futuristic SaaS interfaces.

---

## Features
- Pure HTML & CSS (No JavaScript)
- Cyberpunk neon design
- Animated shimmer sweep effect
- Glassmorphism modal
- Responsive layout
- Hover animations
- Keyboard accessible using the `:target` selector

---

## How to Use
Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Example HTML:

```html
<a href="#modal" class="open-btn">Launch Modal</a>

<div id="modal" class="overlay">
    <div class="modal">
        <!-- Modal Content -->
    </div>
</div>
```

Open `demo.html` directly in any modern browser to see the component in action.

---

## Why is it Useful?
This component provides a visually engaging cyberpunk modal using only HTML and CSS. The animated shimmer sweep, glowing neon effects, and glassmorphism styling create a modern UI while remaining lightweight, responsive, and free of JavaScript dependencies. It aligns with EaseMotion CSS by demonstrating expressive animation techniques that are simple to understand and easy to customize.
