# 3D Tilt Screen Recorder

A responsive **3D Tilt Screen Recorder** inspired by minimalist interfaces. The component uses CSS-only 3D transforms to create an interactive recording card without JavaScript.

## Features

- Pure HTML & CSS
- 3D tilt effect
- Responsive design
- Accessible layout
- No JavaScript required

## Folder Structure

```
3d-tilt-screen-recorder-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
