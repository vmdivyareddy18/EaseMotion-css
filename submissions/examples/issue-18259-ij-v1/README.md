# Toggle Switch (Issue #18259)

A pill-shaped toggle switch with a sliding circular knob. Track color changes from gray to blue on toggle. Smooth CSS transitions on `transform` and `background-color`.

## Usage

Open `demo.html` in a browser. Click the toggle to switch it on/off.

## Files

- `demo.html` — HTML5 page with toggle switch
- `style.css` — Dark theme styles with CSS transitions
