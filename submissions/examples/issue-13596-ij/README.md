# ease-floating-action-menu

A floating action button (FAB) with expanding sub-actions. Implements a Material Design-style FAB pattern using CSS-only `:focus-within` / checkbox technique.

## Usage

Include the `style.css` in your project and use the HTML structure with a checkbox input and the `.fab-container` wrapper. Clicking the main FAB rotates the icon and fans out sub-actions with staggered delays.

## Demo

Open `demo.html` in a browser to see the component in action.
