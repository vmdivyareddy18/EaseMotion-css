What is EaseMotion? ▶

EaseMotion is a CSS library and design system for creating smooth, performant animations and layouts with minimal code. It provides utilities, patterns, and examples for modern CSS features.

How do I install it? ▶

You can install via npm: npm install easemotion, or include it directly from a CDN. Check the documentation for the latest version and setup instructions.

Is it compatible with all browsers? ▶

EaseMotion targets modern browsers that support CSS Grid, Custom Properties, and the latest CSS features. We recommend checking the browser support table in the docs for specific feature compatibility.

Can I use it with React / Vue / Svelte? ▶

Yes. EaseMotion is framework-agnostic — it's pure CSS. You can use it with any framework or vanilla HTML. The patterns and utilities integrate seamlessly into component-based architectures.

Does it support dark mode? ▶

Yes. EaseMotion ships with built-in dark mode support using CSS custom properties. You can toggle themes by switching a data-theme attribute on the root element.

How can I contribute? ▶

Contributions are welcome! Fork the repo, create a feature branch, and submit a pull request. See the contributing guide for coding standards, commit conventions, and review process.
