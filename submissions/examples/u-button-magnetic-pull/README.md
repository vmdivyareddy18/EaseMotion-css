# ease-button-magnetic-pull

Magnetic button animation that subtly follows cursor movement.

## Features

- Magnetic cursor attraction effect
- CSS variable driven translation
- Smooth transform transitions
- Interactive hover feedback
- Pure HTML, CSS and minimal JS

## Acceptance Criteria Covered

### CSS Variables

Uses:

- --x
- --y

updated through JavaScript.

### Magnetic Pull

Button movement is controlled through:

transform: translate(var(--x), var(--y))

### Use Cases

- Landing pages
- CTA buttons
- Portfolio websites
- Interactive dashboards
