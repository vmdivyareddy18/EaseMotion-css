# Device Health Signal Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive dashboard component that displays device health signals using color-coded status cards. Suitable for monitoring dashboards, DevOps tools, cloud infrastructure, IoT management systems, and network operations centers.

## Features

- Device health cards
- Healthy, Warning, and Critical states
- Health score display
- Status badges
- Responsive grid layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="card healthy">

    <div class="top">
        <h3>Edge Router</h3>
    </div>

    <div class="metric">
        99%
    </div>

</div>
```

## Main Classes

```css
.dashboard
.cards
.card
.top

```
