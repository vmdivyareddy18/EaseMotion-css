# App Landing: Herbal Remedies & Holistic Wellness (`ecommerce-herbal-remedies-20987`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a natural, holistic wellness and herbal remedy store, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A holistic wellness brand requires an organic, earthy, and grounding aesthetic. This template achieves a natural feel using a sophisticated color palette of sage greens, warm sand, terracotta, and deep forest green. The typography pairs an elegant serif (`Cormorant Garamond`) with a clean, modern sans-serif (`Jost`).

The layout includes:
- A prominent Announcement Bar for seasonal sales.
- A sticky Navigation with a frosted glass effect and earthy hover states.
- A Hero section featuring abstract CSS botanical shapes swaying gently.
- A "Shop by Benefit" 4-column grid with custom CSS tincture bottle mockups.
- A Sourcing Story split-layout highlighting organic/wildcrafted credentials with a breathing CSS landscape.
- A high-impact Product Spotlight section.
- A "Voices of Healing" customer review section highlighting the serif typography.
- A Wellness Quiz CTA with a slanted background layout.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use slow, organic, and deliberate `cubic-bezier(0.25, 1, 0.5, 1)` timings:
- `.ease-fade-up` / `.ease-slide-up` (Configured for slow, grounding entrances (1.2s - 1.4s) to feel natural and unhurried).
- `.ease-reveal-left` / `.ease-reveal-right` (Used for large background shapes and the sourcing image to smoothly unveil).
- `.ease-sway` (A custom continuous animation applied to the abstract hero shapes to mimic leaves blowing in a gentle breeze).
- `.ease-breathe` (A continuous scaling animation applied to the abstract sun to mimic a deep breath).
- `.ease-hover-earth` (Custom interaction class that transitions text to a warm terracotta color).
- `.ease-hover-lift` (Applied to cards and buttons to gently elevate on hover, providing tactile feedback without feeling digital or jumpy).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the earthy layout, the abstract CSS tincture/landscape simulations, and the typography styling. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the organic, grounding entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
