# Cursor Glow Progress Bar

A lightweight animated progress bar component featuring a moving cursor glow effect.

## Features

- Pure HTML and CSS
- No JavaScript required
- Smooth glow animation
- Responsive design
- Accessible progress indication
- Reusable for portfolio, dashboard and loading pages

## Usage

Add the component markup:

```html
<div class="progress-bar">
  <span class="progress-glow"></span>
</div>
