# Progress Bar (cv variant)

## What does this do?
A collection of animated progress bar components with 4 visual styles (Classic, Striped, Shimmer, Gradient Flow), 5 color variants, and 3 size options. Perfect for indicating loading progress, task completion, and process status.

## How is it used?

### Basic Usage:

```html
<div class="ease-progress-bar-cv ease-progress-bar-cv-classic">
    <div class="ease-progress-bar-cv-fill" style="width: 75%"></div>
</div>

<!-- Classic gradient -->
<div class="ease-progress-bar-cv ease-progress-bar-cv-classic">
    <div class="ease-progress-bar-cv-fill" style="width: 60%"></div>
</div>

<!-- Animated stripes -->
<div class="ease-progress-bar-cv ease-progress-bar-cv-striped">
    <div class="ease-progress-bar-cv-fill" style="width: 60%"></div>
</div>

<!-- Shimmer effect -->
<div class="ease-progress-bar-cv ease-progress-bar-cv-shimmer">
    <div class="ease-progress-bar-cv-fill" style="width: 60%"></div>
</div>

<!-- Gradient flow -->
<div class="ease-progress-bar-cv ease-progress-bar-cv-gradient">
    <div class="ease-progress-bar-cv-fill" style="width: 60%"></div>
</div>

<div class="ease-progress-bar-cv ease-progress-bar-cv-classic ease-progress-bar-cv-primary">
    <div class="ease-progress-bar-cv-fill" style="width: 75%"></div>
</div>

<div class="ease-progress-bar-cv ease-progress-bar-cv-classic ease-progress-bar-cv-success">
    <div class="ease-progress-bar-cv-fill" style="width: 75%"></div>
</div>

<!-- Available: primary, success, info, warning, danger -->

 <!-- Small (6px) -->
<div class="ease-progress-bar-cv ease-progress-bar-cv-classic ease-progress-bar-cv-sm">
    <div class="ease-progress-bar-cv-fill" style="width: 65%"></div>
</div>

<!-- Medium (10px - default) -->
<div class="ease-progress-bar-cv ease-progress-bar-cv-classic">
    <div class="ease-progress-bar-cv-fill" style="width: 75%"></div>
</div>

<!-- Large (16px) -->
<div class="ease-progress-bar-cv ease-progress-bar-cv-classic ease-progress-bar-cv-lg">
    <div class="ease-progress-bar-cv-fill" style="width: 85%"></div>
</div>
