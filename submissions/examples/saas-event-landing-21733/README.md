# SaaS Landing Page Showcase: EventHub

This is a complete, copy-paste ready HTML/CSS landing page demonstrating how to utilize EaseMotion CSS to rapidly build a dynamic, animated marketing site for a SaaS product (Event Management Platform).

## EaseMotion Classes Utilized

This showcase uses a wide variety of EaseMotion classes to build engaging micro-interactions and entrance flows:

### Entrances
- `ease-fade-in`
- `ease-slide-up`
- `ease-slide-down`
- `ease-slide-left`
- `ease-slide-right`
- `ease-zoom-in`

### Utilities
- `ease-duration-fast`
- `ease-duration-medium`
- `ease-delay-100` to `ease-delay-700`

### Interactions (Hovers)
- `ease-hover-scale`
- `ease-hover-lift`
- `ease-hover-glow`

### Continuous
- `ease-pulse`
- `ease-infinite`

## Usage
Simply open `demo.html` in your browser. No build steps are required. 
Layout is powered by Tailwind CSS (via CDN) and animations are powered entirely by EaseMotion classes.
