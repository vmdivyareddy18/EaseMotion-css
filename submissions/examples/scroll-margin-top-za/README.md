# Scroll Margin Top Anchor Offset

A CSS demo of scroll-margin-top property creating offset anchor positions so target sections are not hidden behind fixed headers when navigating with hash links.

## Files

- `demo.html` - Page with fixed nav and scroll-margin sections
- `style.css` - scroll-margin-top property, smooth scrolling, fixed header

## Usage

Open `demo.html` and click the nav links to see scroll-margin in action.
