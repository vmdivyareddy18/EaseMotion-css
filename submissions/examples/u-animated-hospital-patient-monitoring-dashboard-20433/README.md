# Animated Hospital Patient Monitoring Dashboard

A responsive healthcare dashboard built using HTML and CSS animations.

## Features

- Animated patient monitoring dashboard
- Vital signs metric cards
- Recovery progress indicator
- Health status notification
- Responsive dashboard layout
- Smooth hover and heartbeat animations

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations
