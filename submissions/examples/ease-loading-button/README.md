# Loading Button – Button turns into spinner on click, resets

> A production-ready loading button component that transforms into a spinner on click, then displays success or error states before resetting. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Loading state** – Button transforms into animated spinner
- ✅ **Success state** – Displays checkmark with green animation
- ✅ **Error state** – Displays X mark with shake animation
- ✅ **Auto-reset** – Automatically resets after success/error
- ✅ **Interactive controls** – Simulate success or error states
- ✅ **Keyboard accessible** – Press Enter/Space to trigger
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Customizable** – Easy to tweak colors, durations, and callbacks
