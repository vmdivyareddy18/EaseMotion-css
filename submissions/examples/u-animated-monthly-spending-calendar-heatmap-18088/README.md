# Animated Monthly Spending Calendar Heatmap

## Overview

Animated Monthly Spending Calendar Heatmap is a CSS-only visualization that highlights spending activity across a calendar-style grid using animated heatmap cells.

## Features

- CSS-only animation
- Sequential heatmap pulse effect
- Spending activity visualization
- Responsive layout
- Easy customization
- Accessibility support

## Usage

```html
<div class="spending-heatmap">
  <div class="day-cell active"></div>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
