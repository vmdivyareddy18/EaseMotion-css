# CSS 3D Perspective Tilt Modal (Interactive Interface)

A modern **pure CSS 3D animated modal component** designed for interactive UI systems in EaseMotion CSS.

---

## ✨ Features

- Pure CSS 3D perspective animation
- Interactive hover tilt (rotateX / rotateY)
- Smooth open/close transitions
- Fully responsive modal layout
- Keyboard accessible (ESC support)
- No external libraries or frameworks

---

## 🎯 Purpose

This component enhances **interactive interface designs** by adding depth-based motion using only CSS transforms, improving user engagement without JavaScript-heavy animations.

---

## 🧪 Usage

```html
<button class="open-btn" onclick="openModal()">Open Modal</button>

<div class="modal active">
  <div class="modal-content">
    Your content here
  </div>
</div>
