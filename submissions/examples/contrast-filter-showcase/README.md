# Contrast Filter Showcase - EaseMotion CSS

A beautiful, interactive component showcase presenting the contrast filter transition classes.

## Features Included

- `.ease-filter-contrast`: Applies `filter: contrast(100%) !important` to an element.
- `.ease-filter-contrast-50`: Applies `filter: contrast(50%) !important` to an element.
- `.ease-filter-contrast-150`: Applies `filter: contrast(150%) !important` to an element.

## How to Run

1. Open `demo.html` directly in any modern web browser.
2. Observe standard, low-contrast, and high-contrast styling variations side-by-side.

## Why is it useful?

Contrast filter modifications are standard requirements for creating accessible user interfaces (e.g. togglable high-contrast view styles), handling soft backdrop layers, or adjusting layout visibility parameters natively without manual CSS.
