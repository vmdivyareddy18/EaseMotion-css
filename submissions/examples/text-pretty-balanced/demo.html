Paragraph Behavior

pretty

CSS text-wrap: pretty is designed to produce the most visually appealing line breaks by considering the entire paragraph. It may add small performance cost but looks better for body text. Notice how orphans and widows are minimized in this block.

balance

CSS text-wrap: balance aims to make each line roughly the same length. This is ideal for headlines and short blocks. For longer paragraphs the effect is less dramatic but still noticeable on the final line.

wrap (default)

CSS text-wrap: wrap is the standard wrapping behavior. Lines break at natural overflow points without any special optimization. This is the fastest rendering mode and works well for most body text content on the web.

nowrap

CSS text-wrap: nowrap forces all content onto a single line. It will overflow its container if the text is longer than the available width. Use with overflow: hidden or text-overflow: ellipsis for truncation effects.
