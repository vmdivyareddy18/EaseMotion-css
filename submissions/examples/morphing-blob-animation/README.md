# Morphing Blob Animation

A multi-step morphing blob shape animation — pure CSS keyframes.

## Features
- Organic shape-shifting via border-radius keyframes
- Gradient color fill with smooth transitions
- Continuous looping animation

## Usage
```css
.blob { animation: morph-blob 6s ease-in-out infinite; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
