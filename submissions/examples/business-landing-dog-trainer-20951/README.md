# Business Landing: Dog Trainer (`business-landing-dog-trainer-20951`)

This proposal provides a complete, copy-paste ready landing page template designed for a Pet Services Business (Professional Dog Trainer), built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A dog training website requires a UI that is warm, trustworthy, and organic. This template achieves an "Earthy Trust" aesthetic using a calming palette of Terracotta (`#D97757`), Sage Green (`#8A9A86`), Warm Sand (`#E6DFD3`), and Cream. It uses a combination of classic serif typography (`Merriweather`) for authority and clean sans-serif (`Open Sans`) for readability.

The layout includes:
- A trust-building promo bar and sticky navigation.
- A Hero section featuring abstract organic floating shapes, a paw-print background pattern, and a custom CSS-animated Dog illustration (featuring bobbing head, moving ears, and a wagging tail).
- A "Services" grid with sliding top-accents and bounce-on-hover icons.
- An "About the Trainer" section featuring an abstract CSS portrait and a pulsing certification badge.
- A "Success Stories" layout utilizing a clear Before/After comparison structure for behavioral issues.
- A transparent Pricing tier layout with a highlighted "Most Popular" option.
- A warm, inviting free consultation booking form.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use gentle, organic `cubic-bezier(0.4, 0, 0.2, 1)` and smooth `cubic-bezier(0.25, 0.46, 0.45, 0.94)` timings tailored for a trustworthy brand:
- `.ease-slide-up-stagger` (Configured for a gentle, organic vertical entrance).
- `.ease-hover-lift-shadow` (Provides soft, elevated feedback on service and story cards).
- `.ease-float-slow` (Applied to organic background blobs and UI float-cards for a calming effect).
- *Custom Character Animations*: The template includes unique keyframes attached to standard EaseMotion triggers to animate the CSS dog mockup (`ease-dog-bob`, `ease-dog-wag`, `ease-dog-ear-left`).
- `.ease-pulse-subtle` (Used to draw attention to certification badges).
- `.ease-zoom-in` (Used for soft entrances of visual focal points like the consultation form).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the earthy color palette and typography to the pure CSS illustrations (the animated dog mockup and portrait). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the warm, organic animations. No JavaScript is required for this template.
