# ease-map-cluster-expand

Animated map clustering effect where a marker cluster expands into individual map pins.

## Features

- Cluster marker with count display
- Pins expand outward from center
- Staggered entrance timing
- Scale-in animation
- Pure HTML and CSS
- Inspired by real-world map clustering interfaces

## Usage

```html
<div class="cluster">
  12

  <div class="pin">📍</div>
</div>
```

## Animation Details

- Pins originate from the cluster center.
- Each pin scales outward to its own position.
- Staggered delays create a natural expansion effect.
- Commonly used in maps, logistics, travel, delivery, and location-based apps.
