# Media Preview Thumbnail

## Overview

A responsive media preview component featuring image/video thumbnails with a centered play button overlay, duration badges, and smooth hover interactions.

## Features

- Media thumbnail card layout
- Play button overlay with scale animation
- Duration badge
- Thumbnail zoom on hover
- Responsive and accessible design
- Suitable for galleries, feeds, and video showcases

## Tech Stack

- HTML5
- CSS3
- CSS Transforms
- CSS Transitions
- Flexbox and Grid

## Usage

1. Open `demo.html` directly in a browser.
2. Replace the sample thumbnails with your own media assets.
3. Customize durations, titles, and hover timings as needed.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
