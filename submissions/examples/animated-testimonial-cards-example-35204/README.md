# Animated Testimonial Cards Example

## 1. What does this do?
Creates a responsive section with three animated testimonial cards, demonstrating card styling, slide-up entrance motion, staggered delays, and hover-lift interaction.

## 2. How is it used?
Apply the following EaseMotion utility classes to each card element:
- `ease-card`
- `ease-slide-up`
- `ease-delay-100` / `ease-delay-200` / `ease-delay-300`
- `ease-hover-lift`

Example (one card):
```html
<article class="ease-card ease-slide-up ease-delay-100 ease-hover-lift" tabindex="0">
  <!-- content -->
</article>
```

## 3. Why is it useful?
It shows how EaseMotion CSS utilities can be composed to build polished, reusable testimonial UI patterns—without custom keyframes—while keeping the markup readable and the animation behavior consistent across contributors.
