# Animated User Avatar Group Component

## Overview

A modern CSS-only avatar group component featuring overlapping user avatars, hover scaling effects, glowing shadows, and smooth transitions.

## Features

- Overlapping avatar layout
- Smooth hover lift animation
- Scaling and glowing effects
- Responsive and lightweight design
- Easy customization for different avatar sizes
- Pure HTML and CSS implementation

## Files

- demo.html – Demonstrates the animated avatar group
- style.css – Contains avatar styles and animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in any modern browser to view the animated user avatar group.

Hover over avatars to experience the scaling and glowing effects.
