# CSS Scale Hover Tooltip

Pure CSS animated tooltip utilizing a Scale Hover interaction designed for Creative Portfolio layouts.

## Features

- Pure CSS implementation
- Scale Hover animation
- Creative portfolio aesthetics
- Responsive design
- Keyboard accessible
- CSS custom properties
- Glassmorphism styling
- Smooth 60fps animations
- Zero JavaScript dependency

## Variables

```css
--primary
--secondary
--tooltip-bg
--duration
--ease
```
