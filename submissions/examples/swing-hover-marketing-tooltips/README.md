# Pure CSS Swing-Hover Marketing Landing Page Tooltip

An elegant, highly performant, zero-JavaScript user interface overlay system engineered with a top-pivoted 3D rotational swing sequence to naturally boost interactive engagement.

## Features
- **Zero Runtime Dependencies:** Drives the full 3D swinging physics simulation using native hardware-accelerated CSS properties.
- **Top-Hinged Pendular Motion:** Sets a clear `transform-origin` configuration to achieve a realistic gravity swing effect upon reveal.
- **Modern Landing Aesthetics:** Styled with crisp color indicators, responsive grids, and subtle drop shadow depths.
- **Robust Focus Navigation:** Fully integrated with clear layout triggers and semantic ARIA helper rules.

## Configurable Layout Tokens

| Variable Parameter Name | Assignment Target Utility Scope | Default Baseline Vector |
| :--- | :--- | :--- |
| `--tooltip-swing-angle` | Core maximum peak angle bounds of the swing | `15deg` |
| `--tooltip-swing-duration` | Complete timeline duration of the pendular animation | `0.6s` |
| `--tooltip-bg` | Interior tooltip canvas surface filling color | `#0f172a` |
| `--tooltip-fade-duration` | Layout entry transformation visibility timing fallback | `0.2s` |
