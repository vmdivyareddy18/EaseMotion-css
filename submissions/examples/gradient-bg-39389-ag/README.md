# Animated Gradient Background Showcase

A visual demonstration of fluid shifting linear gradient backgrounds utilizing keyframe loops.

---

## What is Included

1. **`demo.html`**: A showcase dashboard container hosting description cards and layout margins.
2. **`style.css`**: Background gradient definitions, scaling parameters, shift keyframes, and motion overrides.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the background colors shifting slowly across slate, indigo, and deep violet.
3. Confirm that the transitions are smooth and seamless.
