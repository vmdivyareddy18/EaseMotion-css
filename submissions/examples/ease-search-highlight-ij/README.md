# Search Highlight

A text search highlighting component with animated pulse effect on matched terms.

## Features

- Highlighted search terms with animated pulse and glow effect
- Styled search input with focus border animation
- Dark theme design matching the EaseMotion aesthetic
- Pure CSS animation with staggered keyframes

## Usage

Include `style.css` and wrap search terms in `<span class="highlight">` elements. The highlight animation uses a pulsing glow effect with CSS `box-shadow` and `background` transitions.
