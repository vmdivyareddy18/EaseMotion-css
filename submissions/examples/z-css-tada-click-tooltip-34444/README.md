# CSS Tada Click Tooltip

Pure CSS animated tooltip utilizing a Tada-Click interaction designed for Accessible Web layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Focus states
- Tada animation
- CSS custom properties
- Accessible UI patterns
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--tooltip
--duration
--ease
```
