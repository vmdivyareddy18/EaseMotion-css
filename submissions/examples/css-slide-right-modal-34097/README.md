# CSS Slide-Right Modal for E-Commerce Checkout Layouts

## Overview

This project demonstrates a responsive e-commerce checkout modal with a smooth **Slide-Right** entrance animation built entirely using **HTML** and **CSS**. The component showcases how CSS-only animations can create engaging user interfaces without relying on JavaScript.

---

## Features

- 🚀 Pure HTML and CSS implementation
- 🎨 Smooth Slide-Right entrance animation
- 📱 Fully responsive layout
- ♿ Keyboard-accessible modal structure
- 🎯 Clean and modern e-commerce checkout design
- ⚙️ Easily customizable through CSS variables
- 💻 No JavaScript or external libraries required

## Customization

The following CSS variables can be modified in `style.css`:

```css
:root{
    --modal-width:420px;
    --radius:18px;
    --duration:.6s;
    --easing:cubic-bezier(0.22,1,0.36,1);
}
```

You can customize:

- Modal width
- Border radius
- Animation duration
- Animation easing
- Colors
- Typography
- Button styling
