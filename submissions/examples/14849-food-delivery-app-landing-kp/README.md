# Food Delivery App Landing Page

## What does this do?

It provides a responsive mobile app landing page for food delivery with order progress, restaurant feature cards, and delivery metrics.

## How is it used?

```html
<section class="hero">
  <h1>Order local favorites and follow every step from kitchen to doorstep.</h1>
  <a class="button" href="#restaurants">Find dinner</a>
</section>
```

## Why is it useful?

It demonstrates a conversion-focused app page with CSS-only delivery status motion and responsive layout.
