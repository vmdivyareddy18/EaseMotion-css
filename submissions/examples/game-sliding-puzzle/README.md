# 🧩 Sliding Puzzle

A classic 4×4 Sliding Puzzle game built with HTML, CSS, and JavaScript. Arrange the numbered tiles in order by sliding them into the empty space. Track your moves and time, and challenge yourself to beat your best!

## 🎮 How to Play

1. Click **"Shuffle"** to scramble the puzzle
2. **Click** a tile adjacent to the empty space to slide it
3. **Arrange** tiles in order from 1 to 15
4. **Track** your moves and time
5. **Beat** your best time!

## ⌨️ Controls

| Action | Input |
|--------|-------|
| Slide Tile | Click on adjacent tile |
| Move Up | `↑` Arrow Key |
| Move Down | `↓` Arrow Key |
| Move Left | `←` Arrow Key |
| Move Right | `→` Arrow Key |
| Shuffle Puzzle | Click "Shuffle" or press `R` |
| Reset Puzzle | Click "Reset" |
| Auto-Solve | Click "Solve" or press `S` |

## ✨ Features

- 🧩 **4×4 Sliding Puzzle** – 15 numbered tiles, 1 empty space
- 🎯 **Move Counter** – Tracks your moves
- ⏱️ **Timer** – Tracks your completion time
- 🏆 **Best Time** – Saved in browser local storage
- 🔀 **Shuffle** – Randomizes the puzzle with guaranteed solvability
- ✨ **Auto-Solve** – BFS algorithm to solve the puzzle automatically
- 🎨 **Beautiful Design** – Colorful tiles with gradients and gloss
- 📱 **Responsive Design** – Works on desktop and mobile
- 🖱️ **Touch Support** – Swipe to slide tiles on mobile
