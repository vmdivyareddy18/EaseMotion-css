# Glassmorphism Floating Profile Card Animation

A CSS-only animated glassmorphism profile card component created for EaseMotion CSS.

## Features

- Frosted glass UI effect
- Floating card animation
- Animated gradient border
- Avatar scale interaction
- Hover lift effect
- Smooth transitions
- No JavaScript required

## Usage

Add the HTML structure from `demo.html`
and include the styles from `style.css`.

Example:

```html
<div class="profile-card">
  <img src="avatar.png" />

  <h2>Alex Carter</h2>

  <p>Frontend Developer</p>

  <button>View Profile</button>
</div>
```

Animation Details

The component uses:

- float-card animation for floating movement
- gradient rotation for border motion
- hover transformations for interaction
