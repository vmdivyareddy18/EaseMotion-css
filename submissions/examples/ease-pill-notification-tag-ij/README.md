# Pill Notification Tags

Pill-shaped notification tags with animated entrance, dismiss, and dynamic add/remove.

## Features

- Scale + opacity entrance animation
- Shrink + fade dismiss animation
- Multiple color types (default, success, warning, danger, info)
- Dynamic add/remove via controls
- ARIA labels on close buttons

## CSS Custom Properties

| Property | Default | Description |
|----------|---------|-------------|
| `--pnt-duration` | `0.35s` | Transition duration |
| `--pnt-tag-bg` | `#e8e8ed` | Tag background |
| `--pnt-text-color` | `#1c1c1e` | Tag text color |
| `--pnt-accent` | `#007aff` | Accent color |
| `--pnt-radius` | `999px` | Border radius |
