# Rating Distribution Bar Chart

A horizontal bar chart showing the distribution of star ratings from 1 to 5. Each row shows the star label, a gradient amber bar proportional to the percentage, and the percentage value. The bars create a visual summary of customer feedback.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the static rating distribution chart.

## Accessibility notes

The percentage is displayed as text alongside each bar. The bar widths convey the distribution visually. No JavaScript is required.
