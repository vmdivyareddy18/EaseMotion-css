# Clip Path Sections

## What does this do?
CSS clip-path section dividers with five distinct shapes: diagonal, arrow, wave, curve overlap, and tilted zigzag patterns.

## How is it used?
```html
<section class="cps-sec cps-diagonal"><div class="cps-inner"><h2>Title</h2><p>Content</p></div></section>
```

## Why is it useful?
Creative page section transitions without images or JavaScript. Five divider styles. Overlapping negative margins create seamless flows. Colorful gradient backgrounds.
