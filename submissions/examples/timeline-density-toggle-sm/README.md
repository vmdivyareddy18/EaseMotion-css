# Timeline Density Toggle

1. What does this do? Adds a CSS-only timeline that switches between comfortable and compact density modes with animated spacing, a moving toggle indicator, and staggered timeline cards.
2. How is it used? Place the `.density-toolbar` controls before an `.activity-timeline` list and use `.timeline-item` with `.timeline-card` for each event.
3. Why is it useful? It lets dense activity feeds remain readable for scanning while using purposeful motion and no JavaScript.
