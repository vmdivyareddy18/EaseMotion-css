# Ease Conic Progress Ring

## Overview

This example demonstrates a reusable CSS-only circular progress ring built with `conic-gradient()`. It is suitable for analytics, dashboards, skills, and profile metrics.

## Features

- Pure HTML and CSS
- Conic gradient progress ring
- Responsive layout
- Smooth entrance animation
- Easily customizable values
- No JavaScript required

## Folder Structure

```text
ease-conic-progress-ring-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

1. Open the example folder.
2. Launch `demo.html`.
3. Update the `--value` custom property to display different percentages.

## Browser Support

Works in modern browsers supporting CSS `conic-gradient()` and custom properties.

## Why This Example

Circular progress indicators are widely used for analytics, dashboards, portfolios, and statistics. This implementation remains lightweight while requiring no JavaScript.
