# Bird Flying

1. What does this do? A CSS-only bird that flaps its wings (scaleY) while soaring up and down.

2. How is it used? Place .bird inside .scene. The body is an ellipse, wings are created with the ::after pseudo-element scaled on the Y axis.

3. Why is it useful? Shows how scaleY animation on a pseudo-element creates convincing wing-flapping motion.
