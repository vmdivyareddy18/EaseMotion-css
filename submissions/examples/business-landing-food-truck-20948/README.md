# Business Landing: Food Truck (`business-landing-food-truck-20948`)

This proposal provides a complete, copy-paste ready landing page template designed for a Street Food or Food Truck Business, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A food truck website requires a UI that is loud, energetic, and highly appetizing. This template achieves a "Bold Street" aesthetic using a high-contrast palette of True Dark (`#121212`), Mustard Yellow (`#FFB300`), and Spicy Red (`#FF3D00`). It uses chunky, comic-style display typography (`Bangers`) paired with a clean, modern sans-serif (`Outfit`).

The layout includes:
- An animated Promo bar for location updates.
- A Hero section featuring abstract CSS graphic bursts, floating labels, and a complex custom CSS-animated Giant Burger.
- An infinite scrolling CSS Marquee.
- A "Menu" section utilizing a dark-mode card layout with subtle hover reveals.
- A "Location/Schedule" section with a bouncing map pin and rotated card layouts for an organic, paste-up feel.
- An "Our Story" section paired with a CSS-constructed placeholder grid for an Instagram feed.
- A high-impact Catering Call-To-Action.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use snappy, elastic `cubic-bezier(0.175, 0.885, 0.32, 1.275)` and bouncy `cubic-bezier(0.68, -0.55, 0.265, 1.55)` timings tailored for a fun, energetic brand:
- `.ease-slide-up-stagger` (Configured for a fast, bouncy vertical entrance, scaling up slightly as it arrives).
- `.ease-hover-scale` (Provides an aggressive, tilted scale effect on buttons).
- `.ease-marquee` (Used for the infinite scrolling text banner).
- *Custom Character Animations*: The template includes unique keyframes attached to standard EaseMotion triggers to animate the CSS Burger mockup (`ease-burger-float`), making the individual ingredients hover and breathe.
- `.ease-spin-slow` (Applied to the background burst graphic in the hero).
- `.ease-hover-scale-img` (Creates an interactive press-down effect on the social feed placeholders).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the high-contrast color palette and typography to the pure CSS illustrations (the animated Giant Smash Burger and the nav icon). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the snappy, bouncy animations. No JavaScript is required for this template.
