# Ripple Hover Effect

A lightweight and reusable ripple hover animation utility built using pure CSS.  
This effect creates a smooth expanding ripple animation when hovering over elements like buttons.

---

## Features

- Pure CSS implementation
- Smooth ripple animation
- Lightweight and reusable
- Uses pseudo-elements and keyframes
- Easy to customize

---

## Usage

### HTML

```html
<button class="ease-hover-ripple">Hover Me</button>
```
