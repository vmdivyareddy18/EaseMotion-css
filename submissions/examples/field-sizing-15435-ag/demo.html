EaseMotion CSS — Form Elements

CSS field-sizing Auto-Growing Inputs

Say goodbye to scrollbars inside small textareas and complex JavaScript size calculators. The new field-sizing property allows form fields to adapt directly to their content.

Traditional Sizing

Default inputs have fixed widths/heights, forcing scrollbars or hidden content when values overflow.

Auto-Growing Sizing

With field-sizing: content active, the elements expand horizontally and vertically to fit what's typed.

CSS Implementation

To enable content-aware auto-sizing, apply field-sizing: content and set appropriate bounds using min-width/max-width or min-height/max-height:

.auto-grow-textarea {
  field-sizing: content;
  min-height: 48px;
  max-height: 300px;
  width: 100%;
}
Browser Support Notice: Natively supported in Chrome, Edge, and Opera. Fallback fixed styling automatically handles older engines.
