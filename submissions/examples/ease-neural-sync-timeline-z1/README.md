# Ease Neural Sync Timeline

## Overview

A futuristic animated timeline featuring glowing synchronization nodes connected with gradient links using only HTML and CSS.

## Features

- Animated timeline nodes
- Gradient connector lines
- Glassmorphism cards
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS
