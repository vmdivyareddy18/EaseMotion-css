# Glow Pulse Sleek Editorial Card

A responsive Sleek Editorial themed card featuring a subtle Glow Pulse animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Glow Pulse animation
- Editorial magazine layout
- Clean typography
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--accent
--accent-light
--background
--surface
--text
--muted
--radius
```
