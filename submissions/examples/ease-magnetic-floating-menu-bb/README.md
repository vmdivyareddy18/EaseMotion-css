# Magnetic Floating Action Menu

A premium floating action menu utilizing CSS transitions and transform properties to achieve a magnetic attraction feel upon hovering over menu items.

## What does this do?
This demo displays a floating sticky navigation dock. Menu icons react with a fluid, spring-like magnetic pull towards the cursor when hovered, expanding with a glowing backdrop ring.

## How is it used?
1. Include `style.css` in your HTML document.
2. Structure the navigation menu:
   ```html
   <nav class="magnetic-menu">
     <a href="#" class="menu-item">...</a>
   </nav>
   ```
3. Use pre-configured CSS classes to configure transitions and spring easing profiles.

## Why is it useful?
Adding magnetic attraction to interactive floating components improves hover feedback. It makes menus feel more tactile and responsive, which is popular in modern creative portfolios and mobile-first dashboards.
