# Nested Comment Thread

A comment thread with nested reply expansion.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--comment-bg` | `#2d2d44` | Comment background |
| `--border-color` | `#4a4a6a` | Border accent |
| `--reply-indent` | `24px` | Reply indentation |
| `--expand-duration` | `0.3s` | Reply form expand speed |
