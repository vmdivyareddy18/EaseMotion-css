# Radial FAB Navigation Menu

## Description
A premium floating action radial shortcut expansion component utilising pure CSS trigonometric transforms, springy animations, and proper screen reader mappings.

## Installation
Drop the CSS styles in your layout folder and map shortcuts.

## Usage
Anchor wrapper layout inside fixed body modules.

## Accessibility
- Implements custom `aria-expanded` status toggle.
- Tab-accessible keyboard shortcuts in active radial lists.
