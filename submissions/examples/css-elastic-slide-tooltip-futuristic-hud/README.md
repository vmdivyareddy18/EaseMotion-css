# CSS Elastic Slide Tooltip - Futuristic HUD Layouts

A pure CSS animated tooltip component inspired by futuristic HUD interfaces.

## Features

- Pure CSS animation
- Elastic slide transition
- Futuristic dashboard style
- Responsive design
- Keyboard accessible
- CSS variable customization


## Usage

Open `demo.html` directly in a browser.

Hover or focus on HUD buttons to display the tooltip animation.


## Customization

Change animation values:

```css
:root {
    --tooltip-time: 0.45s;
    --tooltip-scale: 1;
    --tooltip-easing: cubic-bezier(.68,-0.55,.27,1.55);
}
