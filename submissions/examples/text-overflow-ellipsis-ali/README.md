# Text Overflow Ellipsis Utility

## Overview
Adds a utility class to handle single-line text truncation.

## Class Name
`.ease-text-overflow-ellipsis`

## Why this is needed
In grid-based layouts (like the EaseMotion Card component), long titles can cause cards to stretch to different heights. This utility ensures all cards in a row maintain a consistent height by forcing titles to remain on a single line with an ellipsis (`...`).

## Usage
Add the class to any heading or paragraph inside a container with a fixed or constrained width.
