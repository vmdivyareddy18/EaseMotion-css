# Animated Dropdown Menu Component

## Overview

A modern CSS-only dropdown menu with smooth slide and fade animations, hover interactions, and a responsive navigation design.

## Features

- Smooth dropdown opening animation
- Fade and slide transition effects
- Hover interactions
- CSS-only implementation
- Lightweight and reusable design
- Navigation bar style layout

## Files

- demo.html – Demonstrates the animated dropdown menu
- style.css – Contains dropdown styling and animations
- README.md – Component documentation

## Usage

Open demo.html in a modern browser.

Hover over the "Services" menu item to see the animated dropdown effect.
