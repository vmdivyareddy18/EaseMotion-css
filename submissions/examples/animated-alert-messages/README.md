# Animated Alert Messages

A clean, modern, and fully responsive **Alert/Notification Messages System** perfect for user feedback in forms, actions, and system messages. This example demonstrates how to combine **EaseMotion CSS** for smooth directional entrances, bouncy icons, and interactive dismiss buttons to create a polished, accessible alert system.

## 🚀 Features

- **Smooth Slide-In Entrance**: Uses `ease-slide-in-right` to elegantly introduce each alert from the right side.
- **Bouncy Icon Animation**: The alert icons use `ease-bounce-in` with staggered delays to playfully pop into view.
- **Four Alert Types**: Success (green), Error (red), Warning (amber), and Info (blue) with distinct colors and icons.
- **Interactive Hover States**: Alerts shift slightly on hover, and dismiss buttons use `ease-hover-grow` for tactile feedback.
- **Staggered Load Sequence**: Multiple alerts cascade in sequentially using `ease-delay-*` for a premium feel.
- **Accessible**: Includes `aria-label` on dismiss buttons for screen readers.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted spacing and font sizes.
