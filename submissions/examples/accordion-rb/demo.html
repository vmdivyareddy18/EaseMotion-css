Default Accordion

What is EaseMotion CSS?
EaseMotion CSS is an animation-first CSS framework with zero dependencies. It gives you human-readable utility classes and curated UI components that just work by linking a single stylesheet.
Does it require JavaScript?
Nope! This accordion is built entirely with the native HTML <details> element and CSS grid transitions. No frameworks, no build steps, no runtime JS.
How do I customize the speed?
Override the CSS custom property --accordion-speed on the .accordion container. The default is 0.4s.

Glassmorphism Accordion

Glassmorphism Design
This variant uses backdrop-filter: blur() and translucent backgrounds to create a modern frosted-glass look. Perfect for hero sections and dashboards.
Accessibility Built-in
Because we use the native <details> element, keyboard navigation (Enter / Space to toggle) and screen-reader announcements work out of the box.
Composable with Cards
Drop this accordion inside an .ease-card (once standardized) or any flex/grid container. It adapts to the parent width automatically.
