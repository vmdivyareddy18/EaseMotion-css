# Voice Search Indicator

1. What does this do?

Creates a CSS-only voice search indicator with animated sound-wave rings radiating outward from a microphone icon, simulating an active listening or voice input state.

2. How is it used?

Apply the component class to a microphone element:

```html
<div class="voice-search-indicator">
  🎤
</div>
```

3. Why is it useful?

This component provides clear visual feedback when voice search, voice commands, or speech recognition features are active. It helps users instantly recognize that the application is listening for input, improving usability and engagement without requiring JavaScript.
