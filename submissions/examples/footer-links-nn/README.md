# Footer Links NN

## 1. What does this do?

This is a modern, responsive documentation page footer with a dark theme grey background (#1f1f1f), featuring smooth, interactive CSS hover animations, social integrations, and key documentation links.

## 2. How is it used?

Include the stylesheet and apply the namespaced `*-nn` classes to your markup:

```html
<footer class="footer-nn">
  <div class="footer-container-nn">
    <!-- Top Section -->
    <div class="footer-top-nn">
      <div class="footer-brand-nn">
        <div class="footer-brand-logo-nn">
          <span class="footer-brand-icon-nn">⚡</span>
          <span class="footer-brand-name-nn">EaseMotion CSS</span>
        </div>
        <p class="footer-tagline-nn">
          Supercharge your web projects with animations.
        </p>
        <div class="footer-socials-nn">
          <a href="#" class="footer-social-link-nn" aria-label="Twitter">...</a>
        </div>
      </div>

      <div class="footer-links-grid-nn">
        <div class="footer-links-col-nn">
          <h5 class="footer-links-title-nn">Resources</h5>
          <ul class="footer-nav-nn">
            <li>
              <a href="#" class="footer-link-nn">Documentation</a>
            </li>
          </ul>
        </div>
      </div>
    </div>

    <div class="footer-divider-nn"></div>

    <!-- Bottom Section -->
    <div class="footer-bottom-nn">
      <p class="footer-copyright-nn">&copy; 2026 EaseMotion CSS.</p>
    </div>
  </div>
</footer>
```

## 3. Why is it useful?

It offers a performant, dependency-free, and accessibility-minded footer solution tailored for dark-themed documentation portals, embedding micro-interactions and smooth transitions that reflect the core philosophy of EaseMotion CSS.
