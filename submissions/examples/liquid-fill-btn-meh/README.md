# Liquid Fill Button Hover Effect

A pure CSS button hover effect where a liquid/water-like fill animates inside the button using `clip-path` and `::before` pseudo-element. Creates smooth fluid motion without any JavaScript.

## What It Does

Creates a liquid fill animation on button hover:
- 🌊 **Fluid wave effect** using CSS `clip-path`
- 🎨 **Multiple color variants** (blue, purple, green, coral)
- ➡️ **Fill directions** (bottom-up, left-right, right-left, center-expand)
- 🌊 **Wave styles** (smooth, bubbly, elliptical)
- ⚡ **Zero JavaScript** — pure CSS implementation

## How to Use

```html
<!-- Basic Liquid Fill -->
<button class="btn-liquid">Hover Me</button>

<!-- Color Variants -->
<button class="btn-liquid btn-liquid-purple">Get Started</button>
<button class="btn-liquid btn-liquid-green">Download</button>
<button class="btn-liquid btn-liquid-coral">Subscribe</button>

<!-- Fill Directions -->
<button class="btn-liquid btn-liquid-left">Swipe Right</button>
<button class="btn-liquid btn-liquid-right">Swipe Left</button>
<button class="btn-liquid btn-liquid-center">Radial Fill</button>

<!-- Wave Styles -->
<button class="btn-liquid btn-liquid-smooth">Smooth</button>
<button class="btn-liquid btn-liquid-bubbly">Bubbly</button>
<button class="btn-liquid btn-liquid-elliptical">Elliptical</button>
