# Line Height Grow on Hover

Line height grows when hovering text — pure CSS.

## Features
- Smooth line-height transition on hover
- Pure CSS, no JavaScript

## Usage
```css
.lh-text { line-height: 1.2; transition: line-height 0.3s; }
.lh-text:hover { line-height: 2; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
