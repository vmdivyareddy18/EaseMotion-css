# 🌿 Spotlight Carousel - EaseMotion CSS

A nature-inspired **Spotlight Carousel** component built with pure HTML and CSS.  
This component adds a smooth spotlight animation effect to carousel cards, making the focused item stand out with scale, brightness, and glow animations.

## ✨ Features

- Pure CSS animation (No JavaScript required)
- Smooth spotlight focus effect
- Scale and brightness transitions
- Responsive design
- Accessible image alt text support
- Uses EaseMotion-style animation naming (`ease-*`)
- Lightweight and reusable component
