# Confetti Drag Drop Zone

Resolves #42601.

### What does this do?
Adds a Confetti Drag Drop Zone component to EaseMotion CSS. It uses `ease-bounce` for the icon, `ease-hover-scale` for the dropzone container, and staggering `ease-zoom-in` animations for a burst of confetti particles upon drop (simulated with a CSS-only checkbox hack).

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Click on the dropzone to trigger the confetti animation, simulating a successful file drop/upload.

### Why is it useful?
Provides an engaging, positive visual reinforcement for file uploads without requiring any JavaScript.
