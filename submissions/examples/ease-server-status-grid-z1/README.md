# Ease Server Status Grid

## Overview
A responsive server monitoring component built using only HTML and CSS.

## Features
- Responsive grid
- Animated status indicators
- Hover effects
- CSS variables
- Pure CSS
- No JavaScript

## Usage
Open `demo.html` in any browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?
This component is lightweight, reusable, animation-first, and ideal for SaaS dashboards, admin panels, and monitoring interfaces.
