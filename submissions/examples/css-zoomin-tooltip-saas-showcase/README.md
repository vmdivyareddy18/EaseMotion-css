# CSS Zoom-In Tooltip – SaaS Showcase Layout

## Overview

A responsive SaaS Showcase tooltip component built entirely with HTML and CSS. The component features a smooth zoom-in animation, modern SaaS feature cards, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-In tooltip animation
- SaaS Showcase interface
- Responsive feature cards
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--background`
- `--surface`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--zoom-scale`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any **Explore** button to display the animated Zoom-In Tooltip. Colors, spacing, animation timing, zoom scale, and shadows can all be customized using the CSS variables defined in `style.css`.
