# Animated Space Mission Control Dashboard

A futuristic dashboard for monitoring spacecraft telemetry, mission progress, communication systems, and operational status.

## Features

- Mission metric cards
- Telemetry indicators
- System status panel
- Mission timeline
- Progress tracking bars
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Space mission simulations
- Sci-fi dashboards
- Telemetry monitoring UIs
- Educational aerospace projects
- Futuristic control panels
