# CSS-Only Toggle Switch

**What does this do?**
This adds a beautifully animated, zero-JavaScript toggle switch (checkbox alternative) to the framework.

**How is it used?**

```html
<label class="ease-toggle">
  <input type="checkbox" class="ease-toggle-input">
  <span class="ease-toggle-slider"></span>
  Toggle Me
</label>
```

**Why is it useful?**
It provides a smooth, accessible form element with native CSS transitions. It relies purely on the `:checked` pseudo-class, which perfectly aligns with the framework's zero-dependency, animation-first philosophy.
