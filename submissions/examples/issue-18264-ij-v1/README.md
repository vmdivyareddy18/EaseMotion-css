# Flip Clock Digit (Issue #18264)

A flip clock digit card with a 3D fold animation. Uses CSS `perspective` and `rotateX` to create the flipping effect. Minimal JavaScript to trigger the flip.

## Usage

Open `demo.html` in a browser. Click **Flip** to animate the digit.

## Files

- `demo.html` — HTML5 page with flip clock and JS trigger
- `style.css` — Dark theme styles with flip keyframe animations
- `README.md` — This file
