# ✈️ Animated Airline Booking Dashboard

A modern and responsive Airline Booking Dashboard built using **HTML5** and **CSS3**. This example demonstrates a clean dashboard layout with airline management components and smooth UI animations.

## Features

- 📊 Overview Cards
  - Flights Today
  - Bookings
  - Passengers
  - Revenue

- 🛫 Flight Status Table

- 📈 Boarding Progress Section

- 🚪 Gate Information Grid

- 🔔 Flight Activity Feed

- 🔔 Notifications Panel

- ⚡ Quick Actions
  - Book Flight
  - Check In Passenger
  - Download Report

- 📱 Fully Responsive Layout

- ✨ Smooth Hover & Fade Animations

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
- Font Awesome Icons

## Project Structure

```
animated-airline-dashboard/
├── demo.html
├── style.css
└── README.md
```

## How to Run

1. Open the `animated-airline-dashboard` folder.
2. Ensure `demo.html` and `style.css` are in the same directory.
3. Open `demo.html` in your browser.

## Preview

This dashboard includes airline statistics, flight status monitoring, boarding progress, gate information, activity updates, notifications, and quick action buttons in a modern responsive interface.
