# Ease Gravity Card

A reusable card animation that simulates the effect of gravity.

Unlike traditional hover cards that move upward, the card briefly drops downward, bounces, and smoothly settles back into place.

## Features

- Pure CSS
- Gravity-inspired motion
- Smooth bounce effect
- Customizable animation duration
- Customizable drop distance
- Lightweight and reusable

## Installation

Include the CSS file in your project.

```html
<link rel="stylesheet" href="style.css">
```

## Usage

<div class="ease-gravity-card">
    <h2>Gravity Card</h2>
    <p>Hover me!</p>
</div>
