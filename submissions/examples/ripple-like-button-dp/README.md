# Ripple Like Button

A pure-CSS like/heart toggle button with an expanding ripple animation.

## What does this do?

Provides a reusable, accessible like button component that toggles between an empty and filled heart icon with a ripple halo effect.

## How is it used?

\`\`\`html
<button class="ripple-btn">
<input class="ripple-checkbox" type="checkbox" aria-label="Like">
<span class="ripple-icon"></span>
</button>
\`\`\`

Include \`style.css\` in your page, then use the button markup above. Works by default with no JavaScript.

## Why is it useful?

Like buttons are among the most commonly needed UI elements. This component delivers tactile, polished feedback using pure CSS and EaseMotion's existing animation keyframes, making it lightweight and immediately reusable across projects.
