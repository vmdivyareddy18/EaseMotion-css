# Ease Luminous Node Tree

## Overview

A futuristic node tree visualization featuring glowing hierarchical connections and animated glass panels built entirely with HTML and CSS.

## Features

- Hierarchical node layout
- Animated floating nodes
- Glassmorphism styling
- Responsive design
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-luminous-node-tree-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Introduces a reusable hierarchy visualization suitable for architecture diagrams, workflow trees, developer documentation, system maps, and analytics dashboards.
