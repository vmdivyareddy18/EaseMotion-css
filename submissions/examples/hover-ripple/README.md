# Hover Ripple Effect (`ease-hover-ripple-[your-github-username]`)

## Description
A smooth, expanding ripple effect that triggers on hover, combined with a subtle scale and shadow transition. It provides excellent tactile feedback for interactive elements like buttons and cards.

## Usage
Add the `ease-hover-ripple-[your-github-username]` class to any interactive element.

```html
<button class="ease-hover-ripple-[your-github-username]">Click Me</button>
