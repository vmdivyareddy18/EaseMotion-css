# CSS Staggered Entrance Accordion for Marketing Landing Page Layouts

## Overview

This project showcases a modern **Marketing Landing Page FAQ Accordion** featuring a smooth **Staggered Entrance** animation built entirely with HTML and CSS. Each accordion item enters sequentially with a subtle slide-and-fade effect, creating an engaging first impression while maintaining a clean, professional interface without requiring JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- CSS-only staggered entrance animation
- Marketing landing page inspired design
- Responsive accordion layout
- Sequential fade and slide animations
- Smooth accordion expand and collapse interaction
- Keyboard-accessible using native `<details>` and `<summary>`
- CSS custom properties for easy customization
- No JavaScript or external libraries required

- Accordion cards enter one after another with staggered timing.
- Each item slides upward while fading into view.
- Accordion content expands smoothly using native HTML elements.

## Customization

The component uses CSS custom properties for quick styling.

```css
:root{
    --primary:#2563eb;
    --bg:#f5f8fc;
    --card:#ffffff;
    --text:#1f2937;
    --muted:#6b7280;
    --border:#e5e7eb;
    --radius:18px;
}
```

You can customize:

- Primary color
- Background colors
- Typography
- Border radius
- Card spacing
- Entrance animation timing
- Hover effects
- Shadows
- Accordion transitions
