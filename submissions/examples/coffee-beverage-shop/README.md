# ☕ Coffee Beverage Shop

A responsive coffee and specialty beverages landing page built using EaseMotion CSS.

## Features

- Hero section
- Coffee product grid
- Featured roasters
- Brewing guide
- Subscription plans
- Customer reviews
- Newsletter signup
- Responsive footer

## Project Structure

```
coffee-beverage-shop/
├── demo.html
├── style.css
└── README.md
```

## EaseMotion Classes Used

- ease-fade-down
- ease-fade-up
- ease-fade-left
- ease-fade-right
- ease-duration-700
- ease-hover-scale
- ease-hover-lift
