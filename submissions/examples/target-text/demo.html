The Text Fragments API

Text fragments allow you to link directly to specific text on a page using the #:~:text= syntax in the URL. When a user follows such a link, the browser scrolls to and highlights the matching text.

With the ::target-text pseudo-element, you can customize the appearance of that highlight — controlling background color, text color, and even adding animations.
