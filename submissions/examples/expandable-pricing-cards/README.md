# Animation: Expandable Pricing Cards

A modern pricing section featuring expandable plan cards, hover elevation effects, and animated feature reveals.

## Features

- Pure HTML & CSS
- Pricing card layout
- Expandable feature lists
- Featured plan highlighting
- Hover elevation effect
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="pricing-card">

    <h2>Starter</h2>

    <div class="price">$0</div>

</div>
```

Hovering a pricing card expands its feature list and elevates the card for emphasis.

## Why is it useful?

Pricing sections are essential for SaaS products, developer tools, and startup websites. This component improves plan comparison while maintaining a clean and professional appearance.
