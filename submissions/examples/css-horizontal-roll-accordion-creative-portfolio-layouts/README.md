# CSS Horizontal Roll Accordion

A pure CSS animated accordion featuring a smooth horizontal roll interaction designed for creative portfolio layouts.

## ✨ Features

- Pure CSS implementation
- No JavaScript required
- Horizontal roll opening animation
- Responsive design
- Keyboard accessible using native `<details>` and `<summary>`
- Customizable animation values using CSS variables
- Lightweight and reusable


## 🚀 Usage

Add the accordion structure:

```html
<div class="accordion">

    <details>
        <summary>
            Portfolio Section
        </summary>

        <div class="content">
            Your content here
        </div>

    </details>

</div>
