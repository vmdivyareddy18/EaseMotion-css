# Ruby Position

ruby-position — Positions ruby annotations relative to base text

## Features
- Uses `--ease-color-text`
- Uses `--ease-color-primary`
- Uses `--ease-text-sm`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Ruby Position</span>
</div>
```

## Why is it useful?

This submission demonstrates the ruby position CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
