# Animated Gradient Text

A pure CSS animated gradient text effect with flowing color transitions. No external dependencies, no JavaScript — just clean HTML and CSS.

## What It Does

Creates smooth, flowing gradient animations on text elements. The gradient background moves continuously from left to right, creating an eye-catching shimmer effect perfect for hero sections, landing pages, and marketing headlines.

## How to Use

```html
<!-- Basic Usage -->
<h1 class="text-gradient">Your Heading Here</h1>

<!-- Fast Animation (1.5s) -->
<h1 class="text-gradient text-gradient-fast">Fast Animation</h1>

<!-- Slow Animation (5s) -->
<h1 class="text-gradient text-gradient-slow">Slow & Subtle</h1>

## Available Classes

Class	Description
text-gradient	Base gradient text with 3s animation
text-gradient-fast	Faster 1.5s animation
text-gradient-slow	Slower 5s animation for subtle effect
