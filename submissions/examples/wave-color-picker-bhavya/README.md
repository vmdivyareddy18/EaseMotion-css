# Wave Color Picker

A pure CSS product color picker with animated wave-ring selection feedback.

## How to use

Open `demo.html` directly in a browser and select a color option.

```html
<label class="color-option is-violet">
  <input type="radio" name="product-color" checked />
  <span class="swatch"></span>
  <span class="color-name">Violet</span>
</label>
```

## Why it is useful

Color selectors are common in e-commerce, product cards, and checkout flows. This component adds a polished EaseMotion-style wave animation while remaining responsive, accessible, and JavaScript-free.
