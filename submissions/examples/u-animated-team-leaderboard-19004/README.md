# Animated Team Leaderboard

A modern leaderboard component featuring animated ranking cards, podium highlights, score displays, and responsive layouts.

## Features

- Animated leaderboard cards
- Top 3 podium section
- Rank indicators
- Hover interactions
- Responsive design
- Smooth reveal animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

- Member names
- Scores
- Rank colors
- Animations
- Layout spacing

## Use Cases

- Coding competitions
- Hackathons
- Open source programs
- Gaming leaderboards
- Student rankings
- Productivity dashboards
