# Elastic Slide Tooltip - Neumorphic Soft Layout

## Description

A pure CSS animated tooltip featuring an elastic slide transition with a soft neumorphic design style.

## Features

- Pure CSS implementation
- Smooth elastic slide animation
- Neumorphic soft UI appearance
- Responsive design
- Keyboard accessible
- No JavaScript required
- Custom CSS animation variables

## Usage

```html
<div class="tooltip-wrapper" tabindex="0">

  <button class="soft-button">
    Hover Me
  </button>

  <div class="tooltip">
    Tooltip content
  </div>

</div>
