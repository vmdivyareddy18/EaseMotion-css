# CSS 3D Flip Modal – Minimalist Tech Layout

## Overview

A responsive Minimalist Tech modal built entirely with HTML and CSS. The component features a smooth 3D flip opening animation, clean modern styling, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip animation
- Minimalist Tech interface
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--bg`
- `--card`
- `--text`
- `--muted`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in a modern browser.

Click **Open Dashboard** to display the animated Minimalist Tech 3D Flip Modal. Customize colors, animation timing, flip angle, scaling, spacing, and shadows using the CSS custom properties defined in `style.css`.
