CSS Liquid Ripple Accordion

A responsive, keyboard-accessible accordion built with native HTML, smooth liquid ripple transitions, and customizable CSS variables.

Getting Started Learn how to use the component in your project. ▼

This accordion uses native HTML <details> and <summary> elements, making it keyboard accessible without JavaScript.

Accessibility Features Designed with usability and semantic HTML in mind. ▼

Native focus management, keyboard interaction, responsive layout, and semantic HTML provide a solid accessible foundation.

Customization Adjust animation timing, easing, colors, and ripple scale. ▼

All motion and appearance settings are controlled through CSS custom properties, making the component easy to theme without editing the core styles.
