EaseMotion CSS · #11892

ease-backflip-in

A full aerial backflip entrance — the element arcs through 3-D space, rotating on the X-axis with a natural lift and land.

🚀

Hero Elements

Dramatic entrances for CTAs, product highlights, and splash cards.

🌀

Full 360° Rotation

Starts back-face-up at –180°, arcs overhead, and lands front-facing at 0°.

✨

Aerial Arc

Y-translation lifts the element at the apex — a genuine flip, not a hinge.

🎯

Stagger Ready

Four built-in delay helpers let you cascade groups of elements with ease.

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Add the class — that's it */
<div class="ease-backflip-in">Hero content</div>

/* 3. Optional stagger delays */
<div class="ease-backflip-in delay-1">First </div>
<div class="ease-backflip-in delay-2">Second</div>
<div class="ease-backflip-in delay-3">Third </div>

/* 4. Custom duration via inline style */
<div class="ease-backflip-in"
     style="animation-duration: 1.2s">Slow flip</div>
