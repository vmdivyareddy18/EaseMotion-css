# Shimmer Sweep Popover

A Dashboard Analytics inspired popover with a smooth shimmer sweep animation built using pure HTML and CSS.

## Features

- ✨ Shimmer sweep animation
- 📊 Dashboard analytics design
- 🎨 Uses CSS custom properties for customization
- 📱 Responsive layout
- ♿ Keyboard accessible with `:focus-within`
- 🚫 No JavaScript required

## CSS Custom Properties

```css
--duration
--easing
--scale
--accent
--popover-bg
--popover-text
```

## File Structure

```
shimmer-sweep-popover-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html`.

Hover over or focus the **Analytics** button to display the animated shimmer popover.

## Use Cases

- Analytics Dashboards
- SaaS Products
- Admin Panels
- KPI Cards
- Business Reports
