# Scroll Spy – Active section highlight on scroll

> A production-ready scroll spy navigation component that automatically highlights the active section as you scroll through content, with smooth transitions and keyboard accessibility. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Real-time highlighting** – Active section updates on scroll
- ✅ **Smooth transitions** – Visual feedback feels natural
- ✅ **Click navigation** – Jump to sections with one click
- ✅ **Progress indicator** – Shows scroll progress
- ✅ **Keyboard shortcuts** – Ctrl+Shift+Up/Down to navigate
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
