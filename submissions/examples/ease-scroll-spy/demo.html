01

Introduction

Scroll Spy is a navigation component that automatically highlights the active section as you scroll through a page. It provides a visual indicator of your current position within the content.

The sidebar navigation updates in real-time, making it easy for users to navigate long-form content and understand their location within the document.

Navigation Scroll UX
02

Getting Started

To use Scroll Spy in your project, simply include the component and define your sections with unique IDs. The navigation will automatically detect and highlight the active section.

<nav class="scroll-spy-nav">
  <ul>
    <li><a href="#section1">Section 1</a></li>
    <li><a href="#section2">Section 2</a></li>
  </ul>
</nav>

The component uses Intersection Observer API for efficient scroll detection.

Installation

Install Scroll Spy using one of the following methods:

📦 NPM npm install scroll-spy
🧩 CDN <script src="cdn.scroll-spy.js">
📁 Manual Download and include in your project

Choose the method that works best for your workflow.

API Reference

Method Description Returns
scrollSpy.activate(index) Activate a specific section void
scrollSpy.getActive() Get the currently active section string
scrollSpy.destroy() Clean up the component void
scrollSpy.update() Manually update the spy void
