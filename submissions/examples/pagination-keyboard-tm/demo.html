Pagination Keyboard Navigation

Submission proposing keyboard navigation enhancements to components/pagination.css. Page buttons use <a> elements (not <span>) for native focusability, with clear :focus-visible ring and aria-current="page" on the active page. Linked to issue #11694.

Interactive demo (use Tab and arrow keys)

Tab to the pagination. Use ArrowLeft / ArrowRight to move between pages. Press Enter or Space to activate a page.

Accessibility checklist

  • <a> elements (not <span>) — native keyboard focus
  • aria-current="page" on the active page button
  • aria-label on prev/next buttons ("Go to previous page")
  • aria-label on each page button ("Go to page N")
  • :focus-visible ring for keyboard navigation
  • ArrowLeft / ArrowRight keyboard navigation via JS
