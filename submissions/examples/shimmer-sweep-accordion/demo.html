How does Auto-Height transition in pure CSS?

Historically, animating height from 0 to 'auto' in pure CSS was impossible. This accordion resolves the limitation by utilizing a CSS Grid Track Transition. By shifting the grid item row from 0fr to 1fr inside a grid layout, the browser computes the auto-height and animates the transitions flawlessly. 


.em-accordion-content-wrapper {
  display: grid;
  grid-template-rows: 0fr;
  transition: grid-template-rows 0.45s ease;
}

.em-accordion-input:checked + .em-accordion-item .em-accordion-content-wrapper {
  grid-template-rows: 1fr;
}


.em-accordion-content {
  overflow: hidden;
  min-height: 0;
}
