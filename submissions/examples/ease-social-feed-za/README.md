# CSS Social Media Feed

## What does this do?
CSS social media feed layout with post cards, user avatars, interaction buttons, image placeholders, and stories bar.

## How is it used?
```html
<div class="sf-feed"><div class="sf-post"><div class="sf-header"><div class="sf-avatar">JD</div><div class="sf-meta"><strong>Name</strong><span>Time</span></div></div><div class="sf-body"><p>Content</p></div><div class="sf-actions"><button class="sf-btn sf-like">♥</button></div></div></div>
```

## Why is it useful?
Creates social media feed UI without JavaScript. Post cards with avatar, metadata, content, image placeholder, and action buttons. Stories bar with gradient rings. Hover interactions on like/comment/share. Customizable via CSS variables.
