# Background Attachment Fixed Parallax

A CSS demo of background-attachment: fixed creating a parallax scrolling effect. Multiple sections with fixed gradient backgrounds create depth without JavaScript.

## Files

- `demo.html` - Multi-section page with fixed backgrounds
- `style.css` - background-attachment: fixed, gradient backgrounds, full-page sections

## Usage

Open `demo.html` and scroll to see the parallax effect.
