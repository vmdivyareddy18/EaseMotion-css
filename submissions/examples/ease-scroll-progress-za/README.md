# Scroll Progress Indicator

## What does this do?
A pure CSS scroll progress bar that tracks reading progress using CSS animation-timeline: scroll().

## How is it used?
```html
<div class="sp-bar"></div>
Add this div at the start of your body. The bar automatically tracks scroll progress.
```

## Why is it useful?
Provides clear reading progress feedback without JavaScript. GPU-accelerated, accessible, and follows EaseMotion lightweight CSS philosophy. Customizable via CSS variables for colors and sizing.
