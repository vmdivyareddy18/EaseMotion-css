# Animated Fitness Tracking Dashboard

A modern fitness analytics dashboard built using pure HTML and CSS.

## Features

- Workout overview
- Calories burned
- Daily steps
- Heart rate tracking
- Activity feed
- Progress indicators
- Responsive layout
- CSS Variables
- Smooth animations
- No JavaScript

## Folder Structure

u-animated-fitness-tracking-dashboard/
├── demo.html
├── style.css
└── README.md

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
- CSS Variables
- CSS Animations
