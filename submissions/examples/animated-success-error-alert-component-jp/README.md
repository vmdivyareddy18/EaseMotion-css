# Animated Success and Error Alert Component

## Overview

A CSS-only collection of animated alert messages with smooth slide, fade, hover, and glow effects for different notification states.

## Features

- Success, error, warning, and information alerts
- Smooth slide and fade entrance animations
- Hover scaling and glow effects
- Modern responsive design
- Lightweight CSS-only implementation

## Files

- demo.html – Demonstrates the animated alert collection
- style.css – Contains alert styles and animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in a modern browser to view the animated alerts.

Hover over alert cards to see the interactive effects.
