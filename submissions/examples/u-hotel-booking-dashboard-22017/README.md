# 🏨 Hotel Booking Dashboard

## Overview
The **Hotel Booking Dashboard** is a responsive web application built with **EaseMotion CSS** to manage hotel reservations, room availability, and customer analytics.  
It provides a seamless experience for hotel administrators and guests through elegant UI transitions and modular components.

---

## 🎯 Objectives
- Design a dashboard with real-time booking and room management.
- Integrate customer profiles, booking history, and payment tracking.
- Use EaseMotion CSS for smooth animations and transitions.
- Ensure accessibility and responsiveness across all devices.

---

## 🧩 Tech Stack
| Component | Description |
|------------|-------------|
| **HTML5** | Semantic structure for dashboard sections |
| **EaseMotion CSS** | Animation and transition utilities |
| **JavaScript (ES6)** | Dynamic interactivity and data handling |
| **Chart.js** | Visual analytics for occupancy and revenue |
| **Font Awesome** | Icons for navigation and actions |
