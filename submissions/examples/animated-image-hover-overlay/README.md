# Animated Image Hover Overlay Component

A modern CSS-only animated image card component with smooth hover interactions, gradient overlays, zoom animations, and reveal effects for content.

## Features

- Smooth image zoom animation
- Gradient dark overlay
- Animated text reveal
- Animated action buttons
- Fully responsive design
- Pure HTML and CSS
- Easy to customize
- Suitable for portfolios, galleries, and product showcases

## Files

- `demo.html` – Component structure and demo layout
- `style.css` – Styling, animations, transitions, and responsive design

## Hover Effects Included

- Image scale zoom
- Brightness adjustment
- Overlay fade-in
- Text slide-up animation
- Button pop animation

## Customization

You can easily modify:

- Card height
- Overlay colors
- Animation speed
- Fonts and spacing
- Button styles
- Grid layout

## Browser Support

Works with all modern browsers:

- Chrome
- Firefox
- Edge
- Safari
