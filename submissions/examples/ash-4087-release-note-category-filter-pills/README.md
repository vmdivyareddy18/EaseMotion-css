# Release Note Category Filter Pills

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive release notes interface featuring category filter pills for organizing product updates such as features, bug fixes, security improvements, and performance enhancements.

## Features

- Category filter pills
- Release note cards
- Color-coded categories
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="filters">

    <button class="pill active">All</button>

    <button class="pill feature">Features</button>

    <button class="pill fix">Bug Fixes</button>

</div>
```
