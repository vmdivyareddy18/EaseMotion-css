# Mask Image Utilities

1. What does this do? Adds CSS mask-image utility classes (`.mask-none`, `.mask-linear-to-bottom`, `.mask-linear-to-right`, `.mask-radial`, `.mask-vignette`, `.mask-size-cover`, etc.) to control elements visual transparency shape.
2. How is it used? Apply classes like `.mask-linear-to-bottom` to images or gradient blocks to smoothly fade them into the background.
3. Why is it useful? It allows developers to create beautiful fade effects, spotlight masks, vignette layers, and complex vector mask layouts natively with CSS.
