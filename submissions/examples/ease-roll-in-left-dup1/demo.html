ease-roll-in-left

Rolls in from the left with matching rotation — translateX(-100%) + rotate(-120deg) → 0 with ease-out timing. Rotation speed matches horizontal movement for a physically correct feel.

Default

⚽
.ease-roll-in-left (100%, -120deg, 0.6s, ease-out)

Variants

🎱
.ease-roll-in-left-sm (50%, -60deg)
🔴
.ease-roll-in-left-fast (0.4s)
🟡
.ease-roll-in-left-slow (1s)

Physical correctness — rotation matches translation speed

The rotation angle (120°) is proportional to the distance travelled (100%). Both use ease-out so they decelerate together — the ball naturally slows as it arrives, with no visual disconnect between spin and horizontal movement.
