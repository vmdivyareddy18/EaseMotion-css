# Premium Yoga Studio Landing Page

A complete, serene, and fully responsive wellness and yoga studio landing page ("Prana Yoga Lounge") built under issue #14807 using the **EaseMotion CSS** design system.

## EaseMotion Features Showcased

- **Entrance Animations:** Leverages `.ease-reveal` and direction helpers (`.ease-reveal-up`, `.ease-reveal-right`) with scroll-reveal triggers.
- **Calm breathing animation:** Showcases the `.ease-breathe` breathing core simulation inside the hero area.
- **Hover Micro-interactions:** Showcases hover scale card effects, avatar transformations, and interactive form feedback.
- **Component Systems:** Utilizes standardized `.ease-btn`, `.ease-btn-primary`, `.ease-btn-outline`, and layout classes (`.ease-flex`, `.ease-grid`, `.ease-card`).

## Required Sections Included

1. **Calm Hero:** Serene styling featuring the signature custom breathing bubble element animated using EaseMotion's `.ease-breathe` pattern.
2. **Class Types Grid:** Detailed menu of core offerings (e.g. Vinyasa Flow, Hatha Alignment, Yin Meditation) with hover details.
3. **Instructor Profiles:** Clean cards highlighting certified yoga masters and mindfulness coaches.
4. **Schedule Timetable:** Interactive weekly grid showing scheduled classes, hours, and instructors.
5. **Pricing / Memberships:** Tiered pricing packages (Flow, Balance, Infinite) to drive studio registrations.
6. **Student Testimonials:** Genuine stories of personal wellness achievements.
7. **Booking CTA Form:** Sleek class reservation form with interactive input validation alerts.

Open `demo.html` in any browser to explore the design.
