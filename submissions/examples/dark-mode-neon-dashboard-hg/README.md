# Dark Mode Neon Dashboard

A modern dark-themed dashboard showcasing how **EaseMotion CSS** animations can be combined with a responsive neon-inspired interface.

## ✨ Features

- Responsive dark mode dashboard
- Animated sidebar navigation
- Interactive statistic cards
- Glassmorphism-inspired UI
- Neon hover effects
- Smooth entrance animations using EaseMotion CSS
- Mobile-friendly layout

## 📂 Files

```
dark-mode-neon-dashboard-hg/
├── demo.html
├── style.css
└── README.md
```

## 🚀 How to Use

1. Open `demo.html` directly in your browser.
2. Ensure the internet connection is available so the EaseMotion CSS CDN loads.
3. Hover over cards and navigation items to see the animations.

Example:

```html
<link rel="stylesheet"
href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css">

<div class="card ease-slide-up">
    Animated Card
</div>
```

## 🎯 Why it Fits EaseMotion CSS

This example demonstrates how EaseMotion CSS animation classes can be used to build a polished dark-mode dashboard with minimal code. It highlights reusable animation utilities in a practical UI while keeping the implementation clean, lightweight, and beginner-friendly.
