# Heat Map Grid (#36724)

An animated grid where each cell pulses with dynamic intensity, blending from cold to hot colors.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--grid-bg` | `#0f0f13` | Background color of the grid container |
| `--cold-color` | `#1e3a5f` | Color for low-intensity cells |
| `--hot-color` | `#ef4444` | Color for high-intensity cells |
| `--animate-duration` | `2s` | Duration of the pulse animation cycle |

## Interactive Demo

Open `demo.html` to see the grid. Hover over any cell to trigger an intensity burst. Use the controls to customize colors and animation speed.
