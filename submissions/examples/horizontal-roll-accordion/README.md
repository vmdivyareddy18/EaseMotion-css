# Horizontal Roll Accordion

Ek modern, pure CSS accordion component jo e-commerce checkout flows ke liye design kiya gaya hai. Isme JavaScript ka koi use nahi hai, yeh sirf CSS states par kaam karta hai.

## Features
- **Pure CSS:** No JS required.
- **Responsive:** Flexbox based layout.
- **Customizable:** CSS variables ka use karke speed aur colors change kar sakte hain.

## CSS Variables
- `--roll-speed`: Animation ki speed control karne ke liye.
- `--accent-color`: Active step ka color.

## How to use
Simple `input:radio` buttons ka use karke accordion states manage ki gayi hain. `input:checked` selector expansion trigger karta hai.
