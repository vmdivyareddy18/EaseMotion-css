# Ease Origami Menu

## Overview

Ease Origami Menu is a navigation animation where menu items unfold like paper flaps using CSS 3D transforms. It provides a lightweight, interactive navigation effect built entirely with HTML and CSS.

---

## Features

- 📄 Origami-inspired folding animation
- ✨ Smooth hover interaction
- 🎯 CSS 3D transforms
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Five navigation links
- Folding hover effect
- Smooth transitions

---

## Usage

```html
<nav>
    <a href="#">Home</a>
</nav>
```

---

## Customization

- Link colors
- Fold speed
- Border radius
- Hover color
- Menu spacing

---

## Browser Support

Compatible with modern browsers supporting CSS transforms and transitions.

---

## Why This Fits EaseMotion CSS

Suitable for navigation bars, landing pages, dashboards, portfolios, and interactive websites. It adds a reusable navigation animation utility with no JavaScript dependency.
