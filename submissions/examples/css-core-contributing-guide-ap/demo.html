Core Integration Rules

      

        

          
Core vs Submissions

          

            Submissions folder stores custom templates, landing pages, and demo showcases.
            Core package (core/animations.css) accepts only highly reusable, pre-compiled utilities.
          

        


        

          
Naming Conventions

          

            All selectors must use the .ease- prefix to avoid collisions.
            Keyframes should match camelCase styles (e.g. @keyframes easeZoomIn).
          

        


        

          
Compositor Properties

          

            Animations must run only on the compositor thread. Animate using transform and opacity.
            Avoid animating margin, width, height, or top/left positions.
