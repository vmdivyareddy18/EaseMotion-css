# Scale Avatar

Resolves #42417.

### What does this do?
Adds a Scale Avatar component to EaseMotion CSS suited for Admin Panels. It features an avatar image that dynamically scales up on hover/focus without affecting layout flow. It utilizes pure CSS (`:focus-within`) to toggle a dropdown menu and flip a chevron icon.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Click or focus the user menu to expand the dropdown without JS.

### Why is it useful?
Provides a standard admin-panel user menu with a high-end feel. The scale animation focuses the user's attention, while the CSS-only dropdown menu keeps implementation lightweight.
