Animation Play State Utilities

These utility classes allow you to control CSS animations without writing custom CSS.

1. Pause on Hover (.hover-pause)

The element spins infinitely. Hovering over it pauses the animation.

Hover

2. Static Paused (.animate-pause)

The element has an animation applied, but is permanently paused by the utility class.

Paused

3. Run on Hover (.hover-running)

The element is paused by default. Hovering over it resumes the animation.

Hover
