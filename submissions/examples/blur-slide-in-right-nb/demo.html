Animation Breakdown

Three CSS properties work together to create this multi-dimensional entrance effect.

transform translateX(30px) → translateX(0)

The element begins 30 pixels to the right of its final position and slides horizontally into place. This creates a natural sense of direction and momentum.

filter blur(10px) → blur(0)

A 10-pixel Gaussian blur softens the element on entry, then resolves to a sharp, crisp state. This mimics the way objects come into focus as they settle into view.

opacity 0 → 1

The element starts fully transparent and fades in to full opacity. Combined with the slide and blur, this prevents any harsh popping into the layout.

timing ease-out

An ease-out timing function decelerates the animation as it approaches the final state, creating a natural settling motion that feels organic and refined.

@keyframes Definition

@keyframes blur-slide-in-right {
    0% {
        transform: translateX(30px);
        filter: blur(10px);
        opacity: 0;
    }
    100% {
        transform: translateX(0);
        filter: blur(0);
        opacity: 1;
    }
}
