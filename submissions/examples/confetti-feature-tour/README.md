# Confetti Feature Tour

A responsive **Confetti Feature Tour** component inspired by modern corporate onboarding experiences. It features a clean information card with a CSS-only falling confetti animation.

## Features

- Pure HTML & CSS
- CSS-only confetti animation
- Responsive layout
- Accessible design
- No JavaScript required

## Folder Structure

```
confetti-feature-tour-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
