# Animated Developer Portfolio Showcase

A modern developer portfolio component featuring animated project cards, skill badges, contribution statistics, and responsive layouts.

## Features

- Animated portfolio cards
- Skills section
- Statistics dashboard
- Project showcase
- Social links
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Use Cases

- Developer portfolios
- Student portfolios
- Open source contributor pages
- Personal websites
- Resume landing pages
