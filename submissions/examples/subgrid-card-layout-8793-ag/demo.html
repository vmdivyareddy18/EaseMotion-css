CSS Implementation

To align components across sibling items, let the card span rows on the parent grid, and apply grid-template-rows: subgrid directly to the card elements:

.card-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  /* Rows repeat for each card inner section (Header, Body, Footer) */
  grid-template-rows: auto auto auto;
}

.card {
  grid-row: span 3; /* Stretch across the 3 rows */
  display: grid;
  grid-template-rows: subgrid; /* Inherit parent grid row sizing */
}
