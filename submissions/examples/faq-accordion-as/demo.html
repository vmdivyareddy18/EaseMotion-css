How do I install EaseMotion CSS?
Link one CSS file in your page head and start applying the readable class names. There is no build step required.
Does it need JavaScript?
No. The animations and components work with pure CSS, so they run without any JavaScript.
Is it accessible?
Yes. This FAQ uses native details and summary elements, so keyboard and screen reader support come for free.
Can I use it with any framework?
Yes. Because it is plain CSS and markup, it works with React, Vue, Svelte, or plain HTML.
