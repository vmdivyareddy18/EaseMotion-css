# Smart Cyber-Physical Power Grid Command

## Overview
This submission creates a premium futuristic smart power-grid command dashboard using only HTML and CSS. The experience is designed as a direct-open, self-contained showcase with high-contrast visuals, cyber-physical energy infrastructure cues, readable telemetry cards, inline SVG grid graphics, and polished CSS-only motion.

## Features
- Premium dark command-center theme using the required high-contrast palette
- Hero panel with protected grid intelligence status badges
- Central inline SVG smart-grid visualization with substations, transmission lines, battery storage, renewable nodes, a control tower, and city distribution blocks
- Cyber-physical control network map with secure relay connections
- Live telemetry grid covering all requested dashboard metrics
- Grid stability indicators with animated progress bars
- Renewable source cards for solar, wind, and storage
- Substation and transformer asset status cards
- Demand and load-balancing chart panels with CSS-only shimmer and glow
- Cybersecurity alert timeline
- Responsive control panel button set
- Responsive layout for desktop, tablet, and mobile
- No JavaScript, no external assets, no external images, no frameworks

## EaseMotion Classes Used
This submission uses submission-local EaseMotion-style class naming to keep the implementation self-contained:

- `ease-fade-in`
- `ease-float-up`
- `panel`
- `panel-overlay`
- `status-pill`
- `metric-card`
- `control-btn`

These classes are defined inside `style.css` for this folder so `demo.html` opens directly without depending on any external stylesheet.
