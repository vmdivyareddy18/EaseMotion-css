# Pure CSS Linux Tux Startup Animation

A complete Linux Tux console boot and bash shell transition component built using only HTML/CSS.

---

## What is Included

1. **`demo.html`**: A monitor-style simulated screen displaying the Arch Linux version banner, Tux penguin mascot logo, system loading files, and active bash commands.
2. **`style.css`**: Screen borders, bouncers, terminal prompts, and blink actions.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe Tux pulsing and console system logs compiling vertically.
3. Verify that after 5.2 seconds, the terminal logs fade into an active bash session with a blinking green cursor.
