Platform Notes

macOS: antialiased uses Quartz rendering (no subpixel). subpixel-antialiased uses LCD subpixel rendering (legacy).

Windows: antialiased uses ClearType. subpixel-antialiased uses standard subpixel. auto respects system ClearType settings.

Linux: Behavior depends on fontconfig and FreeType configuration.
