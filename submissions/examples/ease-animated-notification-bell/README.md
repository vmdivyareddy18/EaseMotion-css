# 🔔 Animated Notification Bell

A fully interactive notification bell component with smooth animations, badge counter, and a dropdown notification center.

## ✨ Features

- **Smooth Shake Animation** – Bell shakes when new notifications arrive
- **Animated Badge** – Bouncy counter showing unread notifications
- **Dropdown Menu** – Slides down smoothly with blur backdrop
- **Individual Notifications** – Each has a read/unread state, dot indicator, and dismiss button
- **Add / Clear / Mark All** – Interactive controls for managing notifications
- **Responsive** – Works on all screen sizes
- **Glassmorphism Design** – Modern, dark theme with glass effects
