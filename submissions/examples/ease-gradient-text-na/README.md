# ⚡ Ease Animated Gradient Text

A pure-CSS typographic enhancement module that shifts multi-color vector loops directly across background clipping surfaces.

## ✨ Features
- **GPU Canvas Position Optimization:** Keeps your page render loops performant by altering background indexes instead of structural text geometries.
- **Seamless Loop Anchors:** Preserves design aesthetics by matching terminal color profiles with your primary gradient start colors.
- **Accessible Fail-safes:** Connects to `@media (prefers-reduced-motion)` flags to freeze active colors instantly for accessible displays.

## 🚀 Usage Guide

Apply the primary configuration token along with your preferred theme option variant to any typography block element:

```html
<h1 class="ease-gradient-text ease-gradient-text--aurora">
  Your Fluid Headline Text
</h1>
