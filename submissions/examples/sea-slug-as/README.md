# Sea Slug

A pure CSS and HTML nudibranch creeping across the sea floor, cerata rippling along its back.

## What it shows

- Six cerata that each park their own angle in a custom property, so one shared keyframe can rebuild the rotation instead of wiping it
- A gill plume drawn with a repeating conic gradient and masked into a fan
- Layered mantle: a yellow rim and pale spots stacked over the orange body in a single background stack
- Staggered delays so the ripple travels down the back
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
