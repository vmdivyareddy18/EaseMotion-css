# Ease Constellation Builder

## Overview

Ease Constellation Builder creates a beautiful animated night-sky effect where glowing stars are connected through animated constellation lines.

---

## Features

- ⭐ Animated glowing stars
- 🌌 SVG constellation paths
- ✨ Twinkling animation
- 💫 Flowing connection lines
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Five animated stars
- SVG connection paths
- Flow animation
- Twinkling effect

---

## Usage

```html
<div class="sky">
    ...
</div>
```

---

## Customization

You can customize:

- Number of stars
- Glow color
- Line color
- Animation duration
- Star positions

---

## Browser Support

Works in modern browsers supporting CSS animations and SVG.
