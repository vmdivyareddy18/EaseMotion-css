# Fitness Workout Dashboard

A responsive dashboard showcasing fitness tracking and workout management animations built with **EaseMotion CSS**.

## Features

- Responsive dashboard layout
- Animated overview cards
- Workout progress tracker
- Heart rate monitor
- Weekly activity chart
- Recent workout activity feed
- Notifications panel
- Quick action buttons
- Hover animations
- CSS keyframe animations
- Accessibility support
- Reduced motion support

## Dashboard Components

### Overview Cards

- Calories Burned
- Workout Sessions
- Active Minutes
- Daily Goal

### Widgets

- Workout Progress
- Heart Rate Monitor
- Weekly Activity Chart

### Activity

- Recent Workout Feed
- Notifications Panel

### Quick Actions

- Start Workout
- View Progress
- Add Exercise

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- CSS Animations

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations.
