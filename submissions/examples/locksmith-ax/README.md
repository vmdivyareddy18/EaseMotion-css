# KeyRescue – Emergency Locksmith Services Landing Page

A complete, ready‑to‑copy landing page for a 24/7 emergency locksmith business built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – 24/7 emergency CTA with phone number
- **Services** – lockout, rekeying, installation, safe services
- **Response Time Stats** – 20 min average, 15k+ jobs, 24/7, 98% reviews
- **Service Area Map** – placeholder map image
- **Certifications Badges** – licensed, bonded, insured, certified
- **Customer Reviews** – three testimonials
- **Call Now CTA** – red call‑to‑action section
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-red-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-red-700`, `ease-hover-opacity-100`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`, `ease-opacity-80`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
