3D Parallax Starfield & Hyperdrive

An animation-first, interactive, pure CSS cosmic simulation using radial stretching and 3D coordinate mapping.
How it works: The stars are generated on diverse radial angles and delays, translating outward using CSS keyframes. Clicking the Engage Hyperdrive switch in the dashboard triggers the checkbox hack. This speeds up the animation, stretches the stars along their motion vectors (using scaleY), intensifies the nebula glow, and fills the velocity meter—all done natively in pure CSS.
