# ⚡ Reaction Time Test

A simple and addictive reaction time testing game built with HTML, CSS, and JavaScript. Click when the screen changes color to measure how fast your reflexes are.

## 🎮 How to Play

1. **Click** the colored box or press **Space** to start
2. **Wait** for the color to change (random delay between 1-5 seconds)
3. **Click immediately** when the color changes!
4. **Your reaction time** will be displayed in milliseconds
5. **Challenge yourself** to get the fastest time possible!

## ⌨️ Controls

| Action | Key / Input |
|--------|-------------|
| Start / Click | Click the box or press `Space` |
| Reset Stats | Press `R` |

## ✨ Features

- **Random Delays** – Waits 1-5 seconds before changing color to prevent anticipation
- **Random Colors** – Each test uses a different vibrant color
- **Instant Feedback** – See your reaction time immediately
- **Best Time Tracking** – Your fastest reaction time is saved in your browser
- **Last Time Display** – See your most recent reaction time
- **Total Attempts** – Track how many tests you've completed
- **Performance Categories** – Get feedback on your speed (Incredible, Fast, Good, Slow, Too Slow)
- **Keyboard & Mouse Support** – Works with both inputs
- **Responsive Design** – Plays perfectly on desktop, tablet, and mobile
- **Visual Feedback** – Color changes, status messages, and animations

## 📊 Reaction Time Categories

| Time | Category |
|------|----------|
| < 150ms | ⚡ Incredible! (Pro gamer) |
| 150-200ms | 🔥 Fast! |
| 200-300ms | 👍 Good! |
| 300-400ms | 🐢 Slow... |
| > 400ms | 😴 Too slow! |

## 🛠️ Technologies Used

- HTML5
- CSS3 (Gradients, animations, responsive design)
- Vanilla JavaScript (no dependencies)
