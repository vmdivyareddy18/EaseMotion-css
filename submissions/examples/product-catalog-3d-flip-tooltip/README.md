# Product Catalog 3D Flip Tooltip

A responsive **Pure CSS 3D Flip Tooltip** designed for product catalog interfaces.

## Features

- Pure CSS
- Responsive
- Keyboard Accessible
- 3D Flip Animation
- CSS Custom Properties
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## CSS Variables

```css
--duration
--ease
--rotateX
--bg
--primary
--radius
```

These variables allow easy customization of animation speed, colors, and border radius.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## EaseMotion CSS

This component provides a lightweight and reusable tooltip with a smooth 3D flip animation suitable for modern product catalog layouts.
