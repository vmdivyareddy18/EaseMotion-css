# Animated Resizable Panel Divider Component

A layout infrastructure splitting component built for editor tracking windows, expandable sidebars, and customizable multi-panel tracking charts under **EaseMotion CSS**.

## ✨ Layout Interactions
* **Dynamic Hover/Active Rails**: Modifies alignment profiles smoothly (`ease-panel-divider-active`) using hardware layers to avoid canvas paint lag.
* **Boundary Flash Warnings (`barrierSnapFlash`)**: Signals minimum/maximum layout boundaries directly to the user by pulsing semantic warning channels.
* **Rotational Collapsing Vectors**: Employs isolated matrix overrides to cleanly handle expansion states asynchronously via toggled view states.
