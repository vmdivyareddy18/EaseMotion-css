# Height Utilities

A lightweight collection of CSS height utility classes that provide quick and readable control over element sizing.

## Features

* Pure CSS implementation
* Human-readable utility classes
* Lightweight and reusable
* Cross-browser compatible
* No JavaScript required

## Available Classes

| Class | Effect |
| ------- | ------- |
| `ease-h-full` | Sets height to 100% |
| `ease-h-screen` | Sets height to 100vh |
| `ease-h-auto` | Sets height to auto |
| `ease-min-h-screen` | Sets minimum height to 100vh |

## Usage

```html
<div class="ease-h-full"></div>

<section class="ease-h-screen"></section>

<div class="ease-h-auto"></div>

<main class="ease-min-h-screen"></main>
```

## Why is it Useful?

These utilities make common height-related layouts easier to build while keeping class names readable and intuitive. They align with EaseMotion CSS's philosophy of writing UI styles in plain English.
