# CSS Elastic Slide Tooltip - Holographic Light Layouts

A pure CSS animated tooltip featuring an elastic slide transition designed for holographic light interface designs.

## Features

- Pure CSS animation
- No JavaScript dependency
- Smooth elastic slide motion
- Responsive design
- Keyboard accessible
- Custom CSS variables for animation control

## Usage

```html
<div class="tooltip">
  <button class="tooltip-trigger">
    Holographic Status
  </button>

  <div class="tooltip-content">
    System Online
  </div>
</div>
