# Animated Neon Border Card Component

A futuristic CSS-only card component featuring animated neon gradient borders, glowing shadows, smooth hover scaling, and vibrant cyberpunk-inspired effects.

---

## Features

- Animated moving neon gradient border
- Smooth floating hover animation
- Glowing shadow effects
- Multiple neon color themes
- Interactive buttons
- Responsive design
- Lightweight HTML & CSS implementation
- No JavaScript required

---

## Included Examples

- 🌟 Premium Pricing Card
- 💻 Developer Toolkit Card
- 🎮 Gaming Pass Card

## Files Included

- `demo.html` — Demonstration of neon cards.
- `style.css` — Contains animations, gradients, and responsive styling.
