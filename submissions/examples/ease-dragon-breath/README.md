# 🐉 Ease Dragon Breath

A reusable **CSS-only glow animation** inspired by the visual effect of a dragon breathing fire. The component creates a smooth pulsing glow using gradients, shadows, and keyframe animations, making it suitable for modern user interfaces.

---

## ✨ Features

- 🔥 Pure HTML & CSS
- 🎨 Smooth glowing pulse animation
- 🚀 Reusable `.ease-dragon-breath` class
- 💡 Works with buttons, cards, badges, icons, and loaders
- 📱 Fully responsive
- ⚡ No JavaScript required
- ♻️ Easy to customize

## 💻 Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Apply the animation class to any element:

```html
<button class="ease-dragon-breath">
    Launch Project
</button>
```

You can also use it with:

- Cards
- Icons
- Badges
- Loaders
- Hero Buttons
- Feature Boxes

---

## 🎨 Customization

Change the primary colors inside `:root`.

```css
:root{
    --orange:#ff6b00;
    --yellow:#ffb703;
    --red:#ff3b30;
}
```

You can also customize:

- Animation duration
- Glow intensity
- Shadow colors
- Background gradients
- Border radius
