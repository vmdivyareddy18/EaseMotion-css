Wave Floating Action Button

A study of gentle concentric wave propagation in modern editorial and blog design.

Floating action buttons in editorial platforms should provide utility without breaking the immersive flow of reading. Rather than loud bounces, scale transitions, or glassmorphism overload, the Wave FAB communicates interactivity through a gentle concentric ripple effect.

This wave is created using pure CSS keyframes that animate three staggered absolute ring elements. Each ring starts at a scale of 1 and expands outwards to 2.4 while fading out. The resulting phase difference mimics an organic, expanding water ripple.

Scroll down this page. As you scroll away from the top, the fixed Wave FAB remains in the bottom right corner, ready to return you to the header with a smooth scroll. Hovering or focusing on the button intensifies the wave duration and switches its tone to highlight active state.

Try using your keyboard. Tab to focus the button to see the outline indicator and wave interaction.

Accessibility & Optimization Checklist

  • Semantic anchor structure utilizing native smooth scrolling.
  • Concentric rings utilize GPU-friendly transform and opacity properties.
  • No layout shifts, layout repaints, or rendering artifacts.
  • Prefers-reduced-motion media query immediately scales down and freezes the active wave.
