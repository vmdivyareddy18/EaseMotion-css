# Fluid Typography & Clamp Mixin System

A typography showcase page demonstrating smooth relative header and body text clamp scaling.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting adaptive headers, sizing paragraphs, and viewport descriptions.
2. **`style.css`**: CSS clamp properties, minimum bounds, viewport width variables, and layout sizes.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Resize the browser window width from 320px up to 1200px.
3. Verify that the **Fluid Header Title** scales smoothly without jumps or breaks.
4. Verify that the title size clamps at exactly `1.4rem` on small screens and `2.5rem` on large monitors.
