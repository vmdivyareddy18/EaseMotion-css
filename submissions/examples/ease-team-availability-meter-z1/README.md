# Ease Team Availability Meter

## Overview
A CSS-only widget that displays team availability using animated progress bars.

## Features
- Animated progress meters
- Responsive layout
- CSS variables
- Pure HTML & CSS
- Lightweight and reusable

## Usage
Open `demo.html` directly in a browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
Perfect for collaboration dashboards, HR systems, and SaaS applications with a simple animation-first design.
