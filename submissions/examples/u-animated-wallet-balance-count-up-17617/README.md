# Animated Wallet Balance Count-Up

## Overview

Animated Wallet Balance Count-Up is a lightweight CSS animation that simulates wallet balance updates using a pulsing wallet icon and highlighted balance display.

## Features

- CSS-only animation
- Wallet pulse effect
- Financial dashboard UI
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="wallet-balance">
  <span class="wallet">👛</span>
  <span class="amount">₹5,000</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
