# Floating Select UI Example

## 1. What does this do?

This submission demonstrates a custom select field with floating label styling, focus state animation, and optional validation variants. It uses CSS `:focus-within`, transitions, and `@keyframes` to make the select control feel polished and modern.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example uses `style.css` for the select field styling and floating label behavior. Copy the wrapper markup and CSS to use the `.floating-select` component in your project.

---

## 3. Why is it useful?

Custom select fields with floating labels preserve context and improve usability, especially in dense forms. This submission shows how to build a reusable, animated input pattern without JavaScript.
