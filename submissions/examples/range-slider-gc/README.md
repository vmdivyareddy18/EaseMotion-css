# Range Slider UI Example

## 1. What does this do?

This submission demonstrates a CSS-first range slider component with custom thumb styling, animated track fill, and hover/focus effects. It shows advanced form input styling using `::-webkit-slider-thumb`, `::-webkit-slider-runnable-track`, and modern CSS variables with a dynamic linear-gradient track.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example loads `style.css` for all visual styling and includes a small inline script to keep each slider’s fill percent and displayed value in sync. Copy the HTML markup and CSS into your project to reuse the `.range-slider` component and update colors using CSS custom properties.

---

## 3. Why is it useful?

Range sliders are essential for filtering, pricing selection, and value input forms. This submission provides a polished, accessible design pattern that preserves native `input[type="range"]` behavior while giving the track and thumb a custom, branded visual treatment.
