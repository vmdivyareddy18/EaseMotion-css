# Animated Habit Tracker Dashboard

A responsive habit tracking dashboard featuring progress indicators, streak counters, and achievement badges.

## Features

- Habit tracking cards
- Progress bars
- Streak indicators
- Achievement badges
- Responsive layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more habits
- Modify progress values
- Change colors and badges
- Adjust animation timing
