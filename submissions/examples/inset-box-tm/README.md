# Inset Box

inset — Shorthand for positioning elements with top/right/bottom/left

## Features
- Uses `--ease-color-primary`
- Uses `--ease-radius-lg`
- Uses `--ease-shadow-md`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Inset Box</span>
</div>
```

## Why is it useful?

This submission demonstrates the inset box CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
