Build Script Refactor Proposal

Proposing the migration from a legacy JS-based CSS minifier to LightningCSS (Rust-based) for exponential performance gains.

Expected Benchmark Results

By migrating the root build script, EaseMotion CSS compilation times will drop drastically, improving the CI/CD pipeline efficiency.

Current Pipeline (JS)
~1200ms
LightningCSS (Rust)
~15ms

Why LightningCSS?

  • Speed: Written in Rust, it parallelizes parsing and minification.
  • Prefixing: Replaces PostCSS Autoprefixer entirely by handling browser targets natively.
  • Dead Code: Automatically removes unused rules if configured.
