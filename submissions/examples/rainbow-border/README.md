# Rainbow Border Animation

A pure CSS animated rainbow border effect using `conic-gradient` and pseudo-elements. Perfect for call-to-action buttons, premium badges, and highlighted cards.

## Demo

Open `index.html` in any modern browser.

## Features

- ✅ Pure CSS — zero JavaScript
- ✅ Uses `conic-gradient` for smooth color transitions
- ✅ Hardware-accelerated with `transform: rotate()`
- ✅ Multiple component variants included
- ✅ Adjustable animation speed
- ✅ Static mode available
- ✅ Respects `prefers-reduced-motion`
