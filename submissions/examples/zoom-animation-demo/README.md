# Zoom Animation Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can create zoom-style interactions for cards and images.

---

## Included Components

- Product Card
- Feature Card
- Gallery Card

## Folder

```
submissions/examples/zoom-animation-demo/
```

## How to Run

1. Open `demo.html`.
2. Hover over the cards.
3. Observe the zoom interaction.
