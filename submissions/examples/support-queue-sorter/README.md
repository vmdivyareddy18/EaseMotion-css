# Support Queue Sorter

An intermediate EaseMotion CSS example for support operations screens. Priority ticket cards slide into ordered lanes while status chips and background pulses make urgency easy to scan.

## Files

- `demo.html` - Semantic support queue ticket markup.
- `style.css` - Responsive board layout, staggered sorting motion, and priority color tokens.

## Usage

Open `demo.html` in a browser to preview the animated support queue sorter.
