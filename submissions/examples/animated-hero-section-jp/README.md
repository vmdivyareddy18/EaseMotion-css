# Animated Hero Section Component

## Description

A reusable hero section with animated headings, floating background elements, smooth entrance effects, and interactive call-to-action buttons.

---

## Features

- Smooth fade-up entrance animation
- Floating animated background elements
- Interactive hover effects on buttons
- Gradient background design
- Fully responsive layout
- Lightweight CSS animations
- Reduced motion accessibility support

---

## Components

- `.hero` – Main hero section container
- `.hero-content` – Text and button area
- `.circle` – Floating background shapes
- `.btn` – Interactive call-to-action button
- `.primary` – Filled button variant
- `.secondary` – Outline button variant

---

## Accessibility

Supports `prefers-reduced-motion` to disable animations for users who prefer less motion.
