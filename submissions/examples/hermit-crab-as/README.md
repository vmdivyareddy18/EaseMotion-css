# Hermit Crab

A pure CSS and HTML hermit crab hauling its borrowed spiral shell across wet sand, one big claw waving, an empty shell waiting further down the beach.

## What it shows

- A spiral shell drawn from a repeating conic gradient crossed with a repeating radial gradient, giving whorls and growth ridges from two gradients alone
- Claws built as nested arms with two pincer halves that snap open and shut on mirrored keyframes
- Four jointed legs, each parking its own angle in a custom property so a shared step keyframe drives the scuttle, with a knee on ::after
- Stalked eyes swivelling independently over a lapping wave and a drag track in the sand
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
