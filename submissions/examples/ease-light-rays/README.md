# ease-light-rays

Animated volumetric light beams sweep gently across a dark background container.

## Usage

```html
<div class="ease-light-rays" id="rays">Content</div>
```

Add `.ray` children via script with staggered delays.

## Why it fits EaseMotion CSS

Rays are plain `div`s styled entirely by CSS — width, gradient, and animation timing are all CSS properties. Script only places them.
