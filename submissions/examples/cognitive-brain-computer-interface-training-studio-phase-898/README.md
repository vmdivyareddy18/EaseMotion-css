# Cognitive Brain-Computer Interface Training Studio

## Overview
This submission is a premium futuristic dashboard showcase for a cognitive brain-computer interface training studio. It presents a high-contrast neurotechnology UI with a hero command surface, a central neural visualization, telemetry cards, waveform monitoring, training modules, and coaching panels using only HTML and CSS.

## Features
- Fully static `demo.html` that opens directly in a browser
- High-contrast futuristic interface using the required dark palette
- Original inline SVG brain network with animated neural nodes and signal paths
- CSS-only EEG waveform field, rotating cognitive rings, scanline sweep, shimmer, and hover lift
- Responsive dashboard layout for desktop, tablet, and mobile
- Readability-focused card overlays, strong typography, and clear metric hierarchy

## No JavaScript / No External Assets Note
This submission uses HTML and CSS only. It does not use JavaScript, external images, icon libraries, frameworks, build tools, or network-hosted assets. Visuals are created with inline SVG and CSS, and the page is designed to open directly from `demo.html`.
