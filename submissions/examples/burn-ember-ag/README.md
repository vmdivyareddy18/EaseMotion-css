# Burn Ember

## What does this do?
A smoldering ember glow effect simulating burning material with pulsing orange-red light and floating ash particles.

## How is it used?
Apply the `.burn-ember` class to a container. An ember core pulses while simulated ash particles float upward.

## Why is this useful?
Creates atmospheric fire/burning effects for themed interfaces, game HUDs, fantasy elements, and immersive storytelling.
