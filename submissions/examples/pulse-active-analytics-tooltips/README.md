# Pure CSS Pulse-Active Dashboard Analytics Tooltip

A declarative, high-performance, zero-JavaScript interactive tooltip setup optimized explicitly for layout tracking nodes, data grids, and business telemetry cards.

## Features
- **Pure CSS Execution Vector:** Operates continuous scaling and ambient background glow loops using hardware-accelerated animations (`:hover`/`:focus` maps).
- **Concentric Halo Waves:** Blends a component-level transform breathing system with a radiating background border layer (`pulse-halo`).
- **Data-Centric Analytics Theme:** Structured with clean data summary sections, colored status parameters, and bold typography.
- **Accessible-Compliant Framework:** Fully navigable via keyboard actions with customizable highlight rings.

## Configuration Tokens

| Variable Token Name | Custom Assignment Utility Targets | Default Initial Baseline |
| :--- | :--- | :--- |
| `--dash-accent` | Core focal brand color and wave loop ring | `#6366f1` |
| `--tooltip-pulse-scale` | Peak sizing factor during the breathing pulse cycle | `1.04` |
| `--tooltip-pulse-duration` | Complete timeline duration of the breathing pulse loop | `2s` |
| `--tooltip-fade-duration` | Opacity entry transformation timing metrics | `0.2s` |
