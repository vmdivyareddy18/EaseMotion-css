# Notification Banners

## What does this do?
CSS dismissible notification banners with color variants, slide/fade animations, toast positioning, action buttons, auto-dismiss timer indicators, and stacked layout.

## How is it used?
```html
<div class="nb-banner nb-info"><span class="nb-icon">ℹ</span><div class="nb-body"><strong>Title</strong><p>Message</p></div><span class="nb-close">&times;</span></div>
```

## Why is it useful?
Provides rich notification UI patterns without JavaScript. Four severity types with color-coded left borders. Multiple entry animations. Toast positioning. Action button integration. Customizable via CSS variables.
