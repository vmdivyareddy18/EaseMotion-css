# Animated Kanban Task Board

A modern animated Kanban board component for organizing tasks into workflow stages.

## Features

- Animated column reveal effects
- Interactive task cards
- Priority badges
- Responsive design
- Modern dashboard UI
- Hover lift animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any browser.

## Customization

You can customize:

- Column colors
- Priority labels
- Card animations
- Layout spacing
- Responsive breakpoints

## Use Cases

- Project management dashboards
- Team task boards
- Agile workflows
- Productivity tools
- Admin panels
