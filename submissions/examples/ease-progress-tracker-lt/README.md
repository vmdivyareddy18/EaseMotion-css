# CSS-only Progress Tracker

A responsive CSS-only progress tracker component for multi-step workflows such as onboarding, checkout, and registration.

## Features

- Completed, active, and upcoming steps
- Smooth hover effects
- Pulse animation for the active step
- Responsive layout
- Pure HTML and CSS implementation
