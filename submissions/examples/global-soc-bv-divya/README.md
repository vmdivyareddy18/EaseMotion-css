# Global Security Operations Center

A modern enterprise Security Operations Center (SOC) dashboard built using pure HTML and CSS.

## Features

- Enterprise dashboard layout
- Security metrics cards
- Threat distribution progress bars
- Automation playbook toggles
- Incident monitoring table
- Animated terminal logs
- Responsive design
- Pure HTML & CSS
- No JavaScript required

## Folder Structure

```
global-soc-bv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern web browser.
