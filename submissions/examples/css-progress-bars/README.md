# 📊 CSS Progress Bars

A collection of beautiful, pure CSS animated progress bars for modern web applications.

## 🚀 Features

### Linear Progress Bars
- Smooth fill animation
- Customizable colors
- Shimmer effect overlay

### Striped Progress Bars
- Animated diagonal stripes
- Moving stripe animation
- Multiple color options

### Circular Progress Bars
- SVG-based circular indicators
- Smooth stroke animation
- Percentage display in center

### Gradient Progress Bars
- Moving gradient backgrounds
- Multi-color transitions
- Smooth animations

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All progress bar animations
- `readme.md` - Documentation

## 🛠️ How to Use

### Linear Progress Bar
```html
<div class="progress linear">
    <div class="bar" style="--width: 75%; --color: #4facfe;"></div>
</div>
