## Cloud Drift Animation

Four clouds drift across a bright blue sky at varying speeds, sizes, and opacities to create a calming parallax effect.

**Features:**
- Multiple cloud layers with parallax drift
- Pseudo-element cloud bumps for fluffy shape
- Different speeds and staggered start times
- Respects `prefers-reduced-motion`
