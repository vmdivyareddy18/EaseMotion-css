# CSS Star Rating

## What does this do?
Interactive star rating with radio and checkbox selection, display sizes, and score badges.

## How is it used?
```html
<div class="sr-stars"><input type="radio" name="r" id="s5" class="sr-inp"><label for="s5">&#9733;</label></div>
```

## Why is it useful?
Rating interaction without JS. Radio for single, checkbox for multi. Display sizes. Color-coded score badges.
