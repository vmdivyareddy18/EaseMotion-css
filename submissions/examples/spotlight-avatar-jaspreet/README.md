# Creative Spotlight Avatar Component

A professional, high-fidelity portfolio-focused avatar component with volumetric projection spotlight mechanics built entirely in pure HTML and CSS for the EaseMotion utility library.

## Features
- **Zero JS Volumetric Light Cone:** Implements custom CSS `conic-gradient` elements and layered high-radius gaussian blur properties to simulate authentic physical 3D spotlight displacement beams over user media.
- **Dynamic Orbital Ring Arrays:** Harnesses secondary modular ring layouts configured with overlapping radial scale transformations to project real-time micro-pulsations on user activation.
- **Optimized Portal Glare Effect:** Employs precise translation mechanics to trigger linear glare overlays dynamically across the avatar face upon component focus or hover state.
- **Hardware-Accelerated Layout Pipeline:** Uses isolated CSS layers coupled with defensive `will-change` configurations to protect rendering performance on mobile devices.
- **Semantic HTML & Screen Reader Friendly:** Engineered using explicit semantic section blocks, header hierarchies, self-explanatory descriptions, and targeted screen reader tag variables (`aria-label`, `aria-hidden`).
- **Vestibular Safe Override Architecture:** Integrates global systemic media constraints (`prefers-reduced-motion: reduce`) which instantly drops rendering layers and animation paths to safeguard sensitive users.
