# Global Security Operations Center

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive Global Security Operations Center (SOC) dashboard for monitoring cyber threats, security alerts, infrastructure health, and incident activity. Suitable for SOC teams, cybersecurity platforms, SIEM dashboards, and enterprise security monitoring.

## Features

- Security KPI cards
- Threat detection chart
- Security events feed
- Responsive dashboard layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="dashboard">

    <section class="stats">
        ...
    </section>

    <section class="content">
        ...
    </section>

</div>
```
