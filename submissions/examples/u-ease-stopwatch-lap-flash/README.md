# ease-stopwatch-lap-flash

Animated stopwatch lap entry feedback effect.

## Features

- New lap entry slides in from the top
- Brief highlight/flash effect
- Background color fades back to normal
- Pure HTML and CSS
- Suitable for stopwatch and timer interfaces

## Usage

```html
<div class="lap-item new-lap">
    <span>Lap 3</span>
    <span>00:55.27</span>
</div>
```

## Animation Details

- New lap entries animate into view.
- Background flashes briefly when recorded.
- Flash fades smoothly back to the default appearance.
