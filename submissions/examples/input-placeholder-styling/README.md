# input-placeholder-styling

## Purpose
Demonstrates `::placeholder` pseudo-element styling for custom placeholder color, font, alignment, and animated effects.

## Features
- Custom placeholder colors, fonts, and text alignment
- Animated placeholder effects (fade, slide, typewriter)
- Comparison of default vs styled placeholders
- Accessible and reduced-motion aware

## Usage
Open `demo.html` and interact with the inputs. Placeholder styles change on focus and blur with smooth transitions.
