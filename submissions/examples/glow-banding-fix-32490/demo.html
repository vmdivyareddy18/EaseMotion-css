Glow Colour-Banding Fix

On 16-bit colour-depth displays, glow animations that transition a wide alpha range in a single box-shadow layer produce visible banding — discrete colour steps instead of a smooth gradient. This fix uses multi-layer shadow stacking and an SVG noise dithering overlay to eliminate banding across all display depths.

✗ Without Fix 🌑

Banding on 16-bit

Single box-shadow with a wide alpha range (0.1 → 0.6). On low colour-depth screens the gradient quantises into discrete colour rings — visible as harsh concentric bands in the glow halo.

✓ With Fix 🔮

Smooth on All Displays

Five stacked shadows each covering a narrow alpha slice, plus an SVG Perlin noise overlay at 4% opacity. The human eye blends the layers; the noise breaks up quantisation edges. Looks smooth on 16-bit and flawless on 24-bit.

Technique How it works Why it eliminates banding
Multi-layer shadow stacking 5 concentric box-shadow layers, each ≈ 0.06–0.08 alpha wide Each layer's alpha range is too narrow to produce visible quantisation steps on its own; layers sum perceptually into a continuous gradient
SVG noise dithering (::before) 64×64 feTurbulence noise tile at 4% opacity with mix-blend-mode: overlay Randomly offsets pixel colours by ±1–2 levels across quantisation boundaries — the same principle image encoders use for low-bit-depth output
filter: brightness() animation Adds a secondary animated property alongside box-shadow Increases the number of intermediate compositor values the browser must blend, spreading colour transitions across more frames
ease-in-out timing Non-linear interpolation — spends more time near the endpoints Slows down the transition in the dark region where banding is most visible (low colour values with few available palette entries)
will-change: box-shadow, filter Signals both animated properties upfront Lets the browser allocate dedicated compositor resources; avoids mid-animation recalculation that can cause colour snapping

🎨 The CSS Fix

Apply .glow-smooth to any element. The noise layer is a self-contained ::before pseudo-element — no extra HTML needed.

/* ① Apply to the element */
.glow-smooth {
  position: relative;        /* contains the ::before noise layer */
  border-radius: inherit;   /* noise inherits card corner radius */
  will-change: box-shadow, filter;
  animation: glow-smooth 2.5s ease-in-out infinite;
}

/* ② SVG Perlin noise dithering layer (no extra HTML needed) */
.glow-smooth::before {
  content: "";
  position: absolute;
  inset: 0;
  border-radius: inherit;
  pointer-events: none;
  opacity: 0.04;            /* invisible on 24-bit, active on 16-bit */
  background-image: url("data:image/svg+xml,..."); /* feTurbulence noise */
  mix-blend-mode: overlay; /* scatters values across quant. steps */
}

/* ③ 5 shadow layers — narrow alpha slices prevent banding */
@keyframes glow-smooth {
  0%, 100% {
    box-shadow:
      0 0  6px rgba(99,102,241, 0.22),  /* layer 1: tight inner */
      0 0 14px rgba(99,102,241, 0.14),  /* layer 2: mid ring   */
      0 0 26px rgba(99,102,241, 0.08),  /* layer 3: outer halo */
      0 0 42px rgba(99,102,241, 0.04),  /* layer 4: ambient    */
      0 0 64px rgba(99,102,241, 0.02);  /* layer 5: atmosphere */
    filter: brightness(1);
  }
  50% {
    box-shadow:
      0 0  8px rgba(99,102,241, 0.55),
      0 0 18px rgba(99,102,241, 0.38),
      0 0 34px rgba(99,102,241, 0.20),
      0 0 56px rgba(99,102,241, 0.09),
      0 0 80px rgba(99,102,241, 0.03);
    filter: brightness(1.08);
  }
}

♿ Reduced motion — static glow preserved

Animation stops, but a static mid-intensity glow is kept so the element still looks intentionally designed.

@media (prefers-reduced-motion: reduce) {
  .glow-smooth {
    animation: none !important;
    box-shadow:
      0 0  8px rgba(99,102,241, 0.38),
      0 0 20px rgba(99,102,241, 0.18),
      0 0 38px rgba(99,102,241, 0.08);
  }
}
