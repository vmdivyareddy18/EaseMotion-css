# Fade-Out Popover for Neumorphic Soft Layouts


## What does this add?

A pure CSS Fade-Out Popover component designed for Neumorphic Soft UI interfaces.

It provides a smooth opacity-based disappearing animation without JavaScript.


## How does a developer use it?

```html
<div class="popover-wrapper">

<button class="neumo-btn">
Open Profile
</button>

<div class="popover">
Content
</div>

</div>
