# Profile Switcher Panel

A CSS-only profile switcher panel for modern SaaS and team dashboards.

## What it is

This component displays multiple user profiles in a vertical list. The active profile is highlighted, avatars scale on hover, and each row includes user name, role, and status indicator.

## Features

- Pure HTML and CSS implementation
- Vertical profile list with active highlight
- Smooth hover interactions and subtle transitions
- Avatar scale effect on hover
- Status badges for online, away, busy, and offline
- Modern SaaS dashboard styling
- Responsive layout for mobile screens

## Usage

1. Copy `demo.html` and `style.css` into your submission folder.
2. Open `demo.html` in a browser.
3. Update profile names, roles, and status classes as needed.

```html
<link rel="stylesheet" href="style.css" />
<div class="switcher-page">
  <section class="switcher-card ease-fade-in">
    <!-- profile switcher content -->
  </section>
</div>
```
