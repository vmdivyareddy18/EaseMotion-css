# Accessible Range Slider with Tooltip and Tick Marks

A modern, responsive, and accessible range slider component built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

This component features a floating tooltip, customizable tick marks, keyboard accessibility, dark/light theme support, and smooth animations without using any external libraries.

---

## ✨ Features

- 🎯 Accessible HTML range slider
- 💬 Floating value tooltip
- 📍 Tick marks with labels
- 🌗 Dark & Light theme toggle
- 📱 Fully responsive layout
- ⌨️ Keyboard accessible
- ♿ ARIA compliant
- 🎨 Glassmorphism UI
- ✨ Smooth transitions and animations
- ⚡ Pure HTML, CSS & JavaScript
- 🚫 No external frameworks

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Open `demo.html` in your browser.

No installation or build process is required.
