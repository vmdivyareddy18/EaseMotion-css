# Expanding Search Bar Reveal

## Description

A compact search icon that expands into a full search input when focused.

## Features

- Smooth width expansion
- Modern search interaction
- Lightweight implementation
- Mobile-friendly design

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and click the search icon.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
