# Smart Cyber-Physical Power Grid Command Phase #990

A futuristic, ultra-modern dashboard UI designed to simulate a smart energy grid control center, built entirely with pure HTML5 and CSS3 for the EaseMotion CSS framework showcase.

## Project Overview

This isolated demo represents a cyber-physical system (CPS) dashboard used to monitor high-voltage transmission, renewable energy integration (solar/wind), micro-grid stability, and active cyber defenses. It utilizes the EaseMotion CSS philosophy to generate complex, hardware-accelerated 60fps animations like flowing electricity along transmission lines, pulsing energy nodes, and an interactive grid topology map—all without a single line of JavaScript.

## Features

- **Pure HTML/CSS Engineering**: 100% vanilla HTML5 and CSS3. Zero JavaScript, zero frameworks (no Bootstrap/Tailwind), zero CDNs, and zero external image assets.
- **Cyber-Physical Topology Map**: A central CSS-animated map showing real-time energy flow (`@keyframes flow`) between nuclear core nodes, renewable arrays, and city sectors.
- **Dynamic Energy Monitoring**: Interactive substation cards, load analytics, and cyber-defense metric panels featuring expanding CSS progress bars.
- **AI Analytics Simulation**: A CSS-only area chart simulating load balancing predictions over a 24-hour period.
- **Smart City Aesthetics**: Deep navy backgrounds with electric cyan, voltage yellow, and cyber green glowing accents. Features floating energy particles in the background.
- **Responsive Bento Grid**: Beautifully adapts from a multi-column command center desktop layout to a stacked mobile viewport using CSS media queries.

## Animation Details (EaseMotion CSS Integration)

This project strictly adheres to the EaseMotion CSS framework's structural animation philosophy:
- `.ease-fade-in`: Smooth opacity transitions for HUD panel initialization.
- `.ease-slide-up` / `.ease-slide-down` / `.ease-slide-right`: Directional entrance animations for the bento grid.
- `.hover-lift` / `.hover-glow`: Interactive micro-animations translating the Y-axis and injecting neon border/box shadows.
- `.pulse-cyan` / `.pulse-yellow` / `.power-pulse`: Breathing/glowing animations mimicking electrical surges and active nodes.
- `@keyframes flow`: Custom linear translation animations representing electricity moving across transmission lines.
