# Animated Shield Security Pulse

## Overview

A reusable shield icon animation featuring a subtle pulse effect suitable for security indicators, authentication systems, cybersecurity dashboards, and protected interfaces.

## Features

- CSS-only animation
- Shield pulse effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="security-button">
  <span class="shield-icon">🛡️</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
