Transform Box Utilities

CSS transform-box utilities to control the layout box used to calculate transform origins and coordinate offsets. Crucial for centering SVG rotations and box-model transforms.

What is transform-box?
It defines the reference box that the transform-origin and transform properties are calculated against.
  • fill-box / view-box: Used primarily in SVGs. fill-box aligns coordinates relative to the shape's bounds, while view-box aligns them relative to the SVG viewport.
  • border-box / content-box: Used in HTML elements. Aligns origin coordinates relative to either the outer border edge or the inner content area.
