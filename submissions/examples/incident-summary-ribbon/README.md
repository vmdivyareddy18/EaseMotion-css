# Incident Summary Ribbon

A responsive incident summary demo with severity chips, expanding priority ribbons, hover lift cards, and keyboard focus states.

## Files

- `demo.html` contains the incident summary card markup.
- `style.css` contains the responsive card grid, severity colors, ribbon animation, and focus-visible styling.

## Usage

Open `demo.html` in a browser. Hover or focus each card to expand the ribbon and lift the summary.
