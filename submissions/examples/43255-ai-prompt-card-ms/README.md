# Animated AI Prompt Card Showcase

Resolves #43255.

### What does this do?
Provides a sleek, modern AI prompt input card featuring a continuous gradient border animation (`easeGradientPan`) and an interactive send button with hover physics.

### How is it used?
Wrap a textarea and button inside `.ease-ai-prompt-wrapper` > `.ease-ai-prompt-inner`. The wrapper handles the gradient border, while the inner container provides the white background.

### Why is it useful?
AI interfaces demand a futuristic and highly polished aesthetic. This prompt card immediately communicates "AI capability" to the user through its glowing, moving border and premium micro-interactions.
