### 1. What does this do?
This adds a sticky navigation bar that remains fixed at the top of the viewport when the user scrolls down the page.

### 2. How is it used?
Apply the `.sticky-navbar` class to the `<nav>` element.

```html
<nav class="sticky-navbar">
  <div class="logo">Brand</div>
  <ul>
      <li><a href="#home">Home</a></li>
      <li><a href="#about">About</a></li>
  </ul>
</nav>
```

### 3. Why is it useful?
It improves the user experience by ensuring that navigation links are always accessible, preventing users from having to scroll all the way back to the top of a long page to navigate to other sections.
