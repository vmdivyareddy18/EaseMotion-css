# Particle Barcode

A responsive **Particle Barcode** component inspired by real estate interfaces, built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Particle-style animated barcode
- CSS custom properties
- Accessible markup
- Supports `prefers-reduced-motion`
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` directly in any modern browser.

## Customization

Update the CSS variables in `:root` to adjust colors, animation speed, spacing, and border radius.
