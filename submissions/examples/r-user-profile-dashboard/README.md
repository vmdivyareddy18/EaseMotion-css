# Animated User Profile Dashboard

A responsive profile dashboard featuring profile information, statistics cards, achievement badges, and recent activity.

## Features

- Profile card
- Statistics widgets
- Achievement badges
- Recent activity section
- Hover animations
- Responsive layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more statistics
- Update profile information
- Add custom badges
- Modify colors and animations
