# Bookstore Library Shelf

## What does this do?
A library shelf display with searchable and genre-filterable book cards showing title, author, year, and genre with color-coded spines.

## How is it used?
Open demo.html in a browser. Search by title or author, filter by genre dropdown. Each book card shows a colored spine strip and metadata.

## Why is this useful?
Demonstrates a browsable library UI with dual search/filter capabilities and visual book spine design.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
