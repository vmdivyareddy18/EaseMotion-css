# Circular Countdown

An SVG-based circular countdown timer with stroke-dashoffset animation.

## Features

- Circle outline depletes as time counts down
- Displays remaining seconds
- Start/Pause and Reset controls
- Configurable starting seconds via number input

## Custom Properties

| Property                     | Default   | Description              |
|------------------------------|-----------|--------------------------|
| `--countdown-duration`       | `10s`     | Total countdown duration |
| `--countdown-size`           | `200px`   | Diameter of the circle   |
| `--countdown-stroke-width`   | `8`       | Stroke width of circles  |
| `--countdown-color`          | `#6366f1` | Progress stroke color    |
| `--countdown-track-color`    | `#e2e8f0` | Background track color   |
