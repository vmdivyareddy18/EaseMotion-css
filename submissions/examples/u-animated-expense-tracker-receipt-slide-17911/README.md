# Animated Expense Tracker Receipt Slide

## Overview

Animated Expense Tracker Receipt Slide is a lightweight CSS animation that simulates expense logging using a sliding receipt motion.

## Features

- CSS-only animation
- Receipt slide effect
- Expense tracking UI
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="expense-receipt-slide">
  <span class="receipt">🧾</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
