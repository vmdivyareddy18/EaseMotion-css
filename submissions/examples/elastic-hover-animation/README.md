# Elastic Hover Animation

**What does this do?**
Creates a highly responsive, spring-like "elastic" bounce when you hover over an element.

**How is it used?**
Apply the class to an element like a button or card:
```html
<button class="elastic-button">Hover Me</button>
```

**Why is it useful?**
It provides extremely satisfying, tactile feedback to the user. The custom `cubic-bezier` pushes the scale past its final resting point and snaps back, which fits perfectly with EaseMotion's animation-first philosophy.
