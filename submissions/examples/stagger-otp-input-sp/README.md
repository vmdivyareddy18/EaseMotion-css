# Stagger OTP Input

A modern dark-mode OTP input component featuring staggered entrance animations using pure HTML and CSS.

## Features

- 🌙 Dark mode design
- ✨ Sequential stagger animation
- 🎯 Individual hover & focus effects
- 📱 Responsive layout
- ♿ Keyboard accessible
- 🚫 Pure CSS (No JavaScript)

## File Structure

```
stagger-otp-input-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Each OTP field animates into view with a staggered effect, providing a clean and modern verification interface suitable for authentication flows.

## Why is it useful?

The Stagger OTP Input improves user experience by combining subtle sequential animations with responsive design and accessibility, making it suitable for login, banking, and verification interfaces.
