# Carbon Emissions Target Tracker

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive sustainability dashboard component that visualizes carbon emissions reduction progress toward annual environmental targets. Suitable for ESG dashboards, sustainability reports, environmental monitoring systems, and corporate analytics platforms.

## Features

- Circular emissions progress tracker
- Animated progress bar
- Sustainability metrics
- Summary cards
- Responsive layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="tracker">

    <div class="ring">

        <div class="inner">

            <strong>74%</strong>

        </div>

    </div>

</div>
```
