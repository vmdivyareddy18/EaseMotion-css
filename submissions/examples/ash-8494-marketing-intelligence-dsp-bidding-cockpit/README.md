# Marketing Intelligence & DSP Bidding Cockpit

A modern Marketing Intelligence & DSP Bidding dashboard built entirely with HTML and CSS.

## Overview

This example showcases an enterprise-grade digital advertising dashboard for monitoring campaign performance, real-time bidding, audience targeting, conversion analytics, and marketing intelligence. The interface is fully responsive and built using only HTML and CSS.

## Features

- Responsive enterprise dashboard
- DSP bidding performance visualization
- Campaign health metrics
- Marketing KPI cards
- Top campaigns analytics table
- Traffic source distribution chart
- Audience and conversion insights
- Modern analytics interface
- Pure HTML & CSS (No JavaScript)
