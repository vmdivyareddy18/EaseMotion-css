# ease-card-lift

A lightweight, reusable CSS utility that makes cards lift upward on hover with a subtle shadow increase — a common, polished interaction seen across modern SaaS dashboards.

## Use Cases
- Product cards
- Pricing cards
- Portfolio items
- Dashboard widgets

## Usage

Add the `ease-card-lift` class to any card-like element:

```html
<div class="ease-card-lift">
  Card Content
</div>
