# OTP Input – Auto-advancing OTP fields

> A production-ready OTP input component with auto-advancing fields, paste support, keyboard navigation, and full customization. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Auto-advancing fields** – Automatically moves to next input
- ✅ **Paste support** – Paste entire OTP to fill all fields
- ✅ **Keyboard navigation** – Arrow keys, Backspace, Home, End
- ✅ **Mask toggle** – Show/hide OTP digits
- ✅ **Configurable length** – 4-8 digits
- ✅ **Auto-submit** – Optional auto-verification
- ✅ **Resend timer** – Countdown with resend functionality
- ✅ **Keyboard shortcuts** – Enter (verify), Escape (clear)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
