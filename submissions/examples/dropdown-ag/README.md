# Dropdown Entry Animation Example

Demonstrates a smooth dropdown entry animation for interactive UI components like navigation menus and notification panels.

## Usage

Apply the `ease-dropdown-entry` class to the dropdown container. The animation will play when the element is displayed.

```html
<div class="dropdown-menu ease-dropdown-entry">
  <a href="#" class="dropdown-item">Item 1</a>
  <a href="#" class="dropdown-item">Item 2</a>
</div>
```

## Why is it useful?

It provides a natural, physics-based expansion effect (scaling from the top origin while fading in) that makes UI reveals feel responsive and premium, fitting perfectly with EaseMotion's philosophy of smooth, purposeful motion.
