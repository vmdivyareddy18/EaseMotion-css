# Smart City Monitoring Dashboard

A responsive **Smart City Monitoring Dashboard** built using **HTML5** and **CSS3** that demonstrates real-time city operations, traffic monitoring, public utilities, environmental metrics, and infrastructure management through a modern interactive dashboard.

---

## Features

- Responsive dashboard layout
- Smart city overview cards
- Traffic monitoring progress
- Utility status panel
- Environmental monitoring
- City alerts feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
