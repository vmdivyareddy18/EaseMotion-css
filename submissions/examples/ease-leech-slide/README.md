# Ease Leech Slide Animation

A specialized biological locomotion CSS simulation mimicking the specific contraction-expansion inching and sliding rhythm of an aquatic leech via multi-segmented structural nodes and alternating `scaleX` parameters.

## How to View
1. Render `demo.html` in a web browser tool.
2. Fire the "Slide/Inch" control element to replay the loop lifecycle manually.

## Technical Details
- **Architecture:** Segmented block scaling timelines synchronized within an absolute horizontal carriage container.
- **Easing Profile:** Built around a smooth `cubic-bezier(0.45, 0, 0.55, 1)` step profile.
