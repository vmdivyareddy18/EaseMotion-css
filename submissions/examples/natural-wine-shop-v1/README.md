# 🍷 Natural Wine Shop Landing Page

A modern, responsive landing page for a **Natural Wine Shop**, built using **EaseMotion CSS** utility classes. The design highlights premium wines, regional collections, artisan winemakers, and subscription services while maintaining a clean and elegant user experience.

## ✨ Features

- Responsive navigation bar
- Hero section with call-to-action
- Featured wines by region
- Stylish product cards
- Responsive grid layout
- Interactive buttons and hover effects
- Elegant typography
- Footer section
- Mobile-friendly design
