# CSS Pricing Toggle Switch

## What does this do?
CSS pricing toggle component with monthly/yearly switch, three tier pricing cards with animated price transitions, feature comparison table, and CTA buttons.

## How is it used?
```html
<div class="ps-toggle-wrap"><label class="ps-toggle"><input type="checkbox" class="ps-check"><span class="ps-slider"><span class="ps-label ps-mo">Monthly</span><span class="ps-label ps-yr">Yearly</span></span></label></div>
<div class="ps-card"><h3>Plan</h3><div class="ps-price"><span class="ps-amount" data-mo="29" data-yr="23">29</span></div></div>
```

## Why is it useful?
Provides a complete pricing section for SaaS products. CSS-only toggle between monthly/yearly billing. Animated price transitions via checkbox hack. Feature comparison table. Highlights popular plan with badge. Customizable via CSS variables.
