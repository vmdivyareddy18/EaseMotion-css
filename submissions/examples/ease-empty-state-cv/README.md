# Animated Empty State (`ease-empty-state-cv`)

A lightweight animated empty state component built using **HTML5** and **Pure CSS**.

## Features

- 🎈 Floating illustration animation
- ✨ Fade-in entrance
- 🚀 Animated call-to-action button
- 🌊 Decorative floating background
- 📱 Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-empty-state-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
