# Animated Digital Wallet Dashboard

A modern digital wallet dashboard component built with HTML and CSS.

## Features

- Animated wallet balance
- Financial statistics cards
- Savings goal progress bar
- Transaction status notification
- Responsive layout
- Smooth hover animations

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

- HTML5
- CSS3
- CSS Animations
