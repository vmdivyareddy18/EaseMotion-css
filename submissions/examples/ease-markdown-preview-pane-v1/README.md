# Ease Markdown Preview Pane

## Overview

Ease Markdown Preview Pane is a reusable split-view component inspired by Markdown editors. It displays editable Markdown content alongside a live preview layout using only HTML and CSS.

---

## Features

- 📝 Split-screen layout
- 📄 Markdown preview
- ✨ Smooth entrance animation
- 📱 Responsive design
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Markdown panel
- Preview panel
- Responsive layout
- Hover interaction

---

## Usage

```html
<div class="editor">
...
</div>
```

## Customization

- Panel colors

## Browser Support

Compatible with all modern browsers supporting CSS Grid and animations.
