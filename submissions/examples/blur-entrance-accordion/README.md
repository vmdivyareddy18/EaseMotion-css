# Blur Entrance Accordion

## Description

A modern pure CSS accordion featuring a smooth blur entrance animation, glassmorphism-inspired design, responsive layout, and keyboard accessibility.

---

## Features

- Pure HTML & CSS
- No JavaScript required
- Blur + Fade + Slide entrance animation
- Rotating arrow indicator
- Glassmorphism card design
- Fully responsive
- Keyboard accessible using `<details>` and `<summary>`
- CSS Custom Properties for easy customization
- Modern hover effects

---

## Files

- `demo.html` – Demo implementation
- `style.css` – Styling and animations

---

## Customization

Modify these CSS variables inside `:root`:

```css
--primary
--blur
--duration
--radius
--shadow
```

These allow developers to customize colors, animation speed, blur intensity, border radius, and shadows.

---

## Browser Support

Works in all modern browsers supporting the `<details>` element and CSS custom properties.
