# Leafy Seadragon

A pure CSS and HTML leafy seadragon drifting among the weed, its leaf shaped camouflage appendages waving so it reads as floating seagrass.

## What it shows

- Leaf appendages sprouting all round the body, each parking its own angle in a custom property so a shared wave keyframe drifts them out of phase
- Transparent fins whirring on very fast keyframes while the animal itself barely moves, the way a real seadragon swims
- Body rings and dappling from a repeating linear gradient plus hard stop radial gradients
- Weed that sways with a rotate and skew so the dragon blends into it
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
