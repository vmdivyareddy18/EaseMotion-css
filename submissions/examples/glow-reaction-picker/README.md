# Glow Reaction Picker

A reusable animated reaction picker component with glowing hover effects.

## Features

- Pure CSS animation
- Neon glow pulse effect
- Smooth hover interaction
- Responsive layout
- Accessible buttons
- No JavaScript required

## Usage

Add the component:

```html
<div class="ease-glow-reaction-picker">

  <button class="reaction">❤️</button>
  <button class="reaction">👍</button>
  <button class="reaction">😮</button>
  <button class="reaction">🎉</button>
  <button class="reaction">😂</button>

</div>
