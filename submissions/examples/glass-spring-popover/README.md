# Glass Spring Physics Popover

A pure CSS Spring Physics Popover with a modern Glassmorphism interface.

## Features

- Pure CSS animation
- Spring physics inspired transition
- Glassmorphism UI
- Responsive design
- Keyboard accessible
- CSS variables customization
- No JavaScript


## Usage

Open `demo.html` directly in your browser.


## Custom Properties

```css
:root{
 --duration:.45s;
 --scale:1.08;
}
