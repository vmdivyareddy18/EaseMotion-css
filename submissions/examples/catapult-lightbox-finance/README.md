# Catapult Lightbox — Finance Dashboard Patterns

A high-performance pure CSS lightbox modal layout implementing an explicit structural kinetic catapult spring entry mechanism optimized for confirmation grids.

## Specifications
- **Mechanical Physics**: Employs complex physical bounce timelines (`cubic-bezier(0.175, 0.885, 0.32, 1.275)`) combined with inverse multi-axis transforms to build rebound force context.
- **Declarative Operations**: Houses structural transition maps and glass backdrop values completely inside raw CSS with zero runtime dependency footprints.
- **Hardware Layering**: Maps positional shifts, rotational steps, and scaling matrices directly into GPU compositor pipelines to secure steady frames.
