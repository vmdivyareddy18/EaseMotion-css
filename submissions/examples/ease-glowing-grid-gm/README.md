# Vercel-Style Glowing Grid Portal Dashboard (`ease-glowing-grid-gm`)

1. What does this do?  
Displays a 60fps server metrics dashboard with cursor-tracking neon grid lines, card border shines, and lag-free 3D parallax tilt effects.

2. How is it used?  
Use the wrapper `.ease-grid-portal` and nested grid `.ease-glow-grid` containing `.ease-glow-card` and `.ease-glow-card-body` containers.

3. Why is it useful?  
It showcases high-performance GPU-accelerated cursor masking and 3D rotations on isolated child wrappers that prevent parent layout stutter.
