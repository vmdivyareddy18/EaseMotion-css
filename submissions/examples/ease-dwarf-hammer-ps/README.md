# Ease Dwarf Hammer Animation

1. What does this do? Creates a weighty, rotational animation that simulates pulling a heavy object back, pausing at the apex, and then swinging it down with crushing force.
2. How is it used? Apply `.ease-dwarf-hammer-ps` for an immediate swing, or `.ease-dwarf-hammer-hover-ps` to trigger the animation on hover. It uses `transform-origin: bottom right` by default to simulate the pivot of a swung tool.
3. Why is it useful? Excellent for thematic interactions, aggressive visual feedback on destructive actions (like a 'Delete' confirmation), or adding immense "weight" to an element's movement profile.
