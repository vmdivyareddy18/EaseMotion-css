# Alert Banner

## What does this do?
Alert Banner component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"ab-primary\">Alert Banner Primary</div>
```

## Why is it useful?
Provides a reusable alert banner UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
