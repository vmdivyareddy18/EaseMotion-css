# Background Gradient Utilities

A collection of reusable background gradient utility classes for EaseMotion CSS.

These utilities provide clean and modern gradient backgrounds using simple class-based styling while following the animation-first and developer-friendly philosophy of EaseMotion CSS.

---

## Features

- Reusable gradient utility classes
- Modern linear gradient backgrounds
- Lightweight pure CSS implementation
- Easy integration into existing UI components
- Human-readable class names

---

## Available Classes

### Primary Gradient

```css
.ease-bg-gradient-primary
```
