# Flash Reaction Picker

A responsive **Flash Reaction Picker** component built with pure CSS for the **EaseMotion CSS** library. Inspired by **Real Estate** design patterns, it provides an engaging way for users to express feedback through animated reaction buttons.

---

## ✨ Features

- Pure CSS implementation
- Flash hover animation
- Responsive layout
- Keyboard accessible
- Uses CSS custom properties
- No JavaScript required
- Lightweight and reusable
