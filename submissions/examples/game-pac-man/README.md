# 🟡 Pac-Man

A classic Pac-Man arcade game built with HTML, CSS, and JavaScript. Navigate the maze, eat dots, avoid ghosts, and use power pellets to turn the tables!

## 🎮 How to Play

1. Click **"Start Game"** or press **Space** to begin
2. Use **Arrow Keys** (↑ ↓ ← →) to move Pac-Man
3. **Eat all the dots** to clear the level
4. **Avoid ghosts** — they will chase you!
5. **Eat power pellets** to turn ghosts blue
6. **Eat blue ghosts** for bonus points!
7. **Multiple levels** with increasing difficulty

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `↑` | Move Up |
| `↓` | Move Down |
| `←` | Move Left |
| `→` | Move Right |
| `Space` | Start Game |
| `R` | Reset Game |

## ✨ Features

- 🟡 **Classic Pac-Man Gameplay** – Eat dots, avoid ghosts
- 👻 **4 Ghosts with AI** – Blinky, Pinky, Inky, and Clyde
- ⚡ **Power Pellets** – Turn ghosts blue and eat them!
- 🎯 **Scoring System** – Points for dots, power pellets, and ghosts
- 📈 **Multiple Levels** – Increasing difficulty with each level
- 🏆 **High Score** – Saved in browser local storage
- ❤️ **Lives System** – 3 lives to complete the maze
- 🎨 **Beautiful Visuals** – Glowing effects, animated ghosts
- 📱 **Responsive Design** – Works on desktop and mobile
