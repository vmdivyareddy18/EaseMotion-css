# Rotate Nav Indicator

A navigation bar with a rotating active indicator using CSS 3D transforms and the radio input hack.

## Features

- Active indicator slides and rotates between nav items
- Color changes per tab with rotateY transitions
- Pure CSS using hidden radio inputs
- Active item text highlights to white
- Smooth cubic-bezier animation

## Usage

Include `style.css` and structure the nav with hidden radio inputs, `.nav-item` labels, and a `.nav-bg` indicator element that translates and rotates between positions.
