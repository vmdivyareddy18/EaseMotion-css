How the fix works:
width: min(100% - 2×padding, max-width)

At 1200px viewport: min(1200 − 48, 1200) = 1152px → content is 1152px wide, leaving 24px breathing room on each side.

At 1400px viewport: min(1400 − 48, 1200) = 1200px → max-width wins, container is centred with auto margins.

At 375px mobile: min(375 − 48, 1200) = 327px → always 24px padding from screen edge.
