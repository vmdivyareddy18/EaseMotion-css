# Filter Chip

## What does this do?
Filter Chip component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"fc-primary\">Filter Chip Primary</div>
```

## Why is it useful?
Provides a reusable filter chip UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
