## ease-obsidian-editorial-testimonials

**What does this do?**
A premium dark editorial-style testimonial section inspired by
luxury magazines with decorative quotes, verified badges,
featured cards, and hover animations.

**How is it used?**
<section class="flow-editorial-testimonials">
  <div class="testimonials-grid">
    <div class="testimonial-card featured">
      <p class="testimonial-quote">Quote here</p>
      <div class="testimonial-author">...</div>
    </div>
  </div>
</section>

**Why is it useful?**
Provides a storytelling-focused testimonial layout for agency sites,
SaaS pages, and luxury brands — consistent with EaseMotion CSS's
animation-first, human-readable philosophy.
