# Emoji Picker Popup

**Issue:** [#36640](https://github.com/easemotion/easemotion-css/issues/36640)

An emoji picker popup with a staggered grid reveal animation. Click the trigger button to open the popup — each emoji scales and fades in sequentially with a playful bounce.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--popup-bg` | `#181825` | Background color of the popup panel |
| `--emoji-size` | `40px` | Width and height of each emoji cell |
| `--grid-gap` | `6px` | Gap between grid items |
| `--reveal-duration` | `350ms` | Duration of each individual emoji reveal animation |

## Features

- Staggered grid reveal with pop-in animation per emoji
- Search/filter emoji in real time
- Click outside or press X to close
- Adjustable emoji size, grid gap, and reveal duration via sliders
- Selected emoji updates in the trigger button
- Dark theme
