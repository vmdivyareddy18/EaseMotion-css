# Animation: ease-text-flash

### What does this do?
Triggers a high-visibility, one-shot text color flash profile that changes color dynamically and then gracefully eases back to its standard resting color.

### How is it used?
Apply `.ease-text-flash` to text elements, passing the target color variant token through the inline property `--ease-text-flash-color`.

### Why is it useful?
It offers a non-disruptive option for emphasizing data changes on live dashboards without risking layout shifts or forcing browser reflow cycles.
