# CSS Shimmer Skeleton Loading

A lightweight shimmer skeleton loading component built using only CSS.

## Features

- Pure CSS
- No JavaScript required
- Responsive
- Lightweight
- Easy to customize
- Reusable utility classes
- Smooth shimmer animation

## Files

- demo.html
- style.css

## Components

- Circle Loader
- Title Loader
- Text Loader
- Button Loader
- Card Skeleton

## Preview

Open `demo.html` in your browser to view the shimmer loading effect.

## Customization

You can modify:

- animation duration
- colors
- border radius
- widths
- heights

using CSS variables.
