# Responsive Data Table with Row Striping

A clean data table with alternating row colors, hover highlight, and a status badge column. On mobile screens (below 600px), the "Location" column hides to conserve space. The table is wrapped in a horizontal scroll container for overflow safety.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Resize the viewport to see the responsive column hiding behavior.

## Accessibility notes

Uses semantic `<table>`, `<thead>`, and `<tbody>` elements. Status badges have appropriate color contrast.
