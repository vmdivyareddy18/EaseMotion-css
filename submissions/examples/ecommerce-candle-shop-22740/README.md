# E-commerce: Candle Shop (`ecommerce-candle-shop-22740`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a handmade, artisanal candle and home fragrance brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A home fragrance brand requires a UI that feels cozy, warm, and relaxing. This template achieves a calming aesthetic using a warm, muted palette (warm gray, beige, soft orange), an elegant serif combined with a clean sans-serif (`Cormorant Garamond` & `Lato`), and soft, gentle animations that simulate glowing light and slow melting processes.

The layout includes:
- A soft, sticky Navigation.
- A Hero section featuring abstract CSS candles with flickering flames and a gently pulsating ambient background glow.
- A Curated Scents grid with CSS-rendered minimal candle jars and a subtle hover overlay reveal.
- A Process section featuring a CSS illustration of wax being poured continuously into a jar.
- A Gift Bundles section utilizing CSS transforms to create a box lid that lifts open on hover.
- A refined, elegant Reviews grid with star ratings.
- A clean Subscribe CTA block.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use gentle, drifting `cubic-bezier(0.25, 0.46, 0.45, 0.94)` and `cubic-bezier(0.33, 1, 0.68, 1)` timings that convey relaxation:
- `.ease-fade-up` (Configured for a slow vertical entrance with a subtle blur-to-focus filter).
- `.ease-fade-in` (A slow, soft opacity reveal).
- `.ease-pulse` (A continuous ambient animation for background glows).
- `.ease-flicker` (A continuous erratic animation simulating a candle flame).
- `.ease-hover-glow` (A text-shadow and color transition hover effect).
- `.ease-hover-lift` (A slow, gentle vertical shift on hover for buttons).
- `.ease-hover-open` (A custom 3D transform hover effect on the gift box lid).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the soft layout architecture, the warm palette, and the complex abstract CSS illustrations (candles, flames, pouring wax, gift box). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the slow entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/ecommerce-candle-shop-22740/` directory to avoid modifying core files directly and to ensure zero deletions.*
