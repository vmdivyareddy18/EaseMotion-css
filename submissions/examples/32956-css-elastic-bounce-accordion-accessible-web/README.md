# CSS Elastic Bounce Accordion (Accessible Web Layouts)

A pure CSS responsive accordion designed for clean public directories and documentation indexes. Features a spring-like elastic bounce expansion when content panels open.

## Features
- **Elastic Bounce Transition**: Employs a spring-like `cubic-bezier` curve that bounces slightly past its final height, giving a physical organic feel.
- **Bespoke Forest & Slate Theme**: Clean slate-gray borders, pure white cards, and vibrant forest green active indicators.
- **Pure CSS checked Logic**: Built using native checkbox inputs for zero-JS toggle support.
- **A11y Compliant Outlines**: High-visibility focus rings and screen-reader compliant structures.

## Files
- `demo.html`: Live interactive demonstration of the documentation catalog.
- `style.css`: Stylesheet containing the spring-loaded transitions and layout structures.
