Elastic Bounce Directory

A clean documentation directory triggering a spring-like elastic bounce expansion when content panels open.

It uses custom cubic-bezier math transitions to make content boxes slide open slightly beyond their target height limits, then spring back smoothly to settle into place, representing natural physical momentum.
To comply with WCAG 2.1 AA accessibility guidelines, we combine high-contrast slate-gray text with forest green highlight accents, ensuring high legibility and clean outlines.
Exposed properties at the top of the stylesheet allow you to edit bounce durations, change spring curves, or customize card border radius tokens cleanly.
