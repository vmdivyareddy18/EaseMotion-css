# Pulse Card Utility

**What does this do?**  
Adds a subtle pulse animation to cards on hover/focus/active, making them scale slightly and show a soft shadow glow.

---

**How is it used?**
```html
<div class="em-pulse-card">...</div>
<div class="card em-pulse-card">Card</div>
<button class="em-pulse-card">Button</button>

## Why is it useful?
It gives smooth interactive feedback and makes UI feel more responsive and modern. Works on all elements like cards, buttons, and containers.

## Behavior
- Slight scale-up (pulse effect)  
- Soft shadow glow  
- Smooth easing animation  
- Triggered on hover, focus, active  
- Pure CSS, no JS
