# CSS Fade-In Popover

A lightweight CSS Fade-In Popover component built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Smooth fade-in animation
- Responsive design
- Modern UI
- No JavaScript required
- Accessible with `prefers-reduced-motion`

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="popover-container">
    <button class="trigger">Hover Me</button>

    <div class="popover">
        <h3>Creative Portfolio</h3>
        <p>This popover smoothly fades into view using CSS animations.</p>
    </div>
</div>
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports users who prefer reduced motion using the `prefers-reduced-motion` media query.
