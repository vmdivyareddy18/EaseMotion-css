ease-clock-loader

Time in motion.
Minimal. Mechanical.

A pure-CSS analog clock loader. Hour and minute hands glide smoothly; the second hand ticks with steps(60) — just like the real thing. No JS, no dependencies.

Theme variants

default (light)
--slate
--cream
--steel
--midnight

Size variants

--xs (32 px)
--sm (48 px)
--md (64 px)
--lg (96 px)
--xl (140 px)

Turbo mode — accelerated processing state

--turbo
--paused

Add ease-clock--turbo during active processing, swap back to normal when waiting.

Usage

<!-- 1. link the stylesheet -->
<link rel="stylesheet" href="style.css" />

<!-- 2. add the markup -->
<div class="ease-clock ease-clock--md ease-clock--slate">
  <div class="ease-clock__face"></div>
  <div class="ease-clock__hand ease-clock__hand--hour"></div>
  <div class="ease-clock__hand ease-clock__hand--minute"></div>
  <div class="ease-clock__hand ease-clock__hand--second"></div>
  <div class="ease-clock__center"></div>
</div>

<!-- 3. or override tokens directly -->
<style>
  .my-clock {
    --clock-size:         80px;
    --clock-face:         #faf6ef;
    --clock-hand-second:  #e74c5e;
    --clock-dur-second:   10s;   /* speed up just the second hand */
  }
</style>
