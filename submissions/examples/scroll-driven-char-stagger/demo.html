Each character reveals as you scroll. No JavaScript. No Intersection Observer. Pure CSS using animation-timeline: scroll().

Performance

GPU-accelerated

Opacity, translate, and filter are all compositor-thread properties. Scroll-driven animations using these properties produce zero layout recalculations.

Requires animation-timeline: scroll() — Chrome 115+, Firefox 114+ (behind flag). Characters are fully visible as a static fallback in unsupported browsers via the @supports not block.

Composable

Apply .char to any inline element and set --i to control its scroll offset. Works on headings, labels, or standalone words.

Accessible

The prefers-reduced-motion media query disables all animation and forces full opacity. Screen readers see the plain text regardless of animation state.
