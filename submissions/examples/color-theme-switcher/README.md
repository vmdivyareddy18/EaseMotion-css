# Color Theme Switcher

A theme switcher that toggles between four color schemes: Light, Dark, Sunset, and Ocean. Each button applies a corresponding CSS class to the body element, transitioning the background and text colors with a smooth CSS transition.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click each theme button to see the page colors change.

## Accessibility notes

Theme buttons have sufficient color contrast in all variations. Reduced motion disables the color transition.
