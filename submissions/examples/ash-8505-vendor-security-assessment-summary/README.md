# Vendor Security Assessment Summary

A modern Vendor Security Assessment Summary dashboard built entirely with HTML and CSS.

## Overview

This example showcases an enterprise dashboard for assessing vendor security posture, monitoring third-party risks, tracking compliance frameworks, and managing remediation activities. The interface is fully responsive and built using only HTML and CSS.

## Features

- Responsive vendor security dashboard
- Third-party risk assessment overview
- Compliance monitoring
- Security control metrics
- Vendor risk summary table
- Risk distribution visualization
- Assessment activity analytics
- Modern enterprise UI
- Pure HTML & CSS (No JavaScript)
