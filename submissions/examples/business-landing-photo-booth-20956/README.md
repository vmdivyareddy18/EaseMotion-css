# Business Landing: Photo Booth Rental (`business-landing-photo-booth-20956`)

This proposal provides a complete, copy-paste ready landing page template designed for an Event Rental Business (specifically Photo Booths), built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

An event industry website requires a UI that is highly energetic, fun, and modern. This template achieves a "Brutalist Pop" aesthetic using a vibrant color palette of Electric Purple (`#7C3AED`), Hot Pink (`#EC4899`), and Pop Yellow (`#FDE047`) against high-contrast dark accents and thick borders. It uses playful, bold typography (`Poppins` and `Nunito`) and hard drop shadows (`box-shadow: 10px 10px 0px black`) to create a tactile, energetic feel.

The layout includes:
- A snappy, high-contrast navigation bar.
- A Hero section featuring abstract floating background shapes, a CSS-animated "Photo Booth" mockup (simulating a flashing screen and ring light), and playfully tilted floating photo strips.
- A seamless, infinite scrolling CSS marquee banner for event types.
- A "Choose Your Vibe" packages grid with dramatic hover-lift interactions.
- A masonry-style "Recent Smiles" abstract emoji gallery with scale/bounce hover effects.
- A clear, bold Pricing tier grid highlighting a "Most Popular" option.
- A high-impact booking form/CTA section with gradient buttons.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use highly bouncy, energetic `cubic-bezier(0.68, -0.55, 0.265, 1.55)` and springy `cubic-bezier(0.175, 0.885, 0.32, 1.275)` timings tailored for a fun brand:
- `.ease-pop-in` (Configured for a highly elastic, bouncy entrance).
- `.ease-hover-scale-bounce` / `.ease-hover-lift-shadow` (For extreme, tactile hover feedback that lifts elements and exposes hard shadows).
- `.ease-hover-bounce-icon` (Applied to emojis inside the gallery to make them pop out on hover).
- `.ease-scroll-infinite` (A custom continuous CSS animation for the event ticker tape).
- `.ease-flash` / `.ease-flash-bright` (Custom animations simulating the camera flash on the photo booth mockup).
- `.ease-float-tilt` (Applied to the photo strips to make them bob playfully in the air).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the bold brutalist color palette and typography to the pure CSS illustrations (photo booth mockup with ring light, photo strips, ticket banner). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the energetic, bouncy animations. No JavaScript is required for this template.
