# Full Furniture Store Landing Page

1. What does this do? A warm, inviting furniture store landing page with room category cards, product grid, inspiration gallery, service features, and design consultation CTA built entirely with EaseMotion CSS framework classes.

2. How is it used? Open `demo.html` — the page uses `ease-card-hover`, `ease-reveal-up`, `ease-hover-lift`, `ease-grid-cols-*`, and dozens of other EaseMotion classes for all layout, components, and animations.

3. Why is it useful? Proves EaseMotion handles content-rich home decor e-commerce pages with image galleries, category browsing, product cards, and trust signals — all using composable framework classes with zero custom CSS animations.
