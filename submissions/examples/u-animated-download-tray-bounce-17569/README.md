# Animated Download Tray Bounce

## Overview

Animated Download Tray Bounce is a lightweight CSS animation that simulates a completed file download using a falling download arrow and a bouncing tray.

## Features

- CSS-only animation
- Download tray effect
- Responsive design
- Easy customization
- Accessibility support
- Lightweight implementation

## Usage

```html
<div class="download-tray">
  <span class="download-arrow">⬇️</span>
  <span class="tray">📥</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
