# Ease Floating Dock Menu

A macOS-inspired floating dock menu built with pure HTML and CSS.

## Features

- Floating dock layout
- Hover lift animation
- Responsive
- Pure HTML & CSS
- Lightweight

## Structure

```
ease-floating-dock-menu-z1/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Why EaseMotion CSS

A reusable navigation micro-interaction showcasing smooth hover motion with minimal code.
