# Pulse Progress Ring UI Example

## 1. What does this do?

This submission adds a circular progress ring with animated pulse and percentage fill. The ring visually conveys completion progress with motion and gradient highlights.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example showcases multiple progress ring states and pulsing center effects. Copy the `.progress-ring` markup and CSS into your project to display animated circular progress.

---

## 3. Why is it useful?

Circular progress indicators are compact and visually striking. The pulse animation draws attention to progress updates and adds a polished loading or completion state.
