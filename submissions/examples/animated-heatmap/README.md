# Animated Heatmap Calendar Grid

An advanced, data-first component utility configured via CSS Grid tracking algorithms. It presents metrics and analytical contribution patterns dynamically using sequential entry blooms.

## Utility Structural Classes
- `.ease-heatmap-container` : Configures the core isolated card boundaries.
- `.ease-heatmap-grid` : Enforces strict 7-row columns tracking timelines natively.
- `.ease-heatmap-cell` : Core cell interaction point maps scale shifts and delays.

## Composable Value Scales
- `.ease-heat-empty`
- `.ease-heat-low`
- `.ease-heat-med`
- `.ease-heat-high`
- `.ease-heat-max`
