# Overflow Wrap Break

overflow-wrap: break-word — Wraps long words without overflow

## Features
- Uses `--ease-color-text`
- Uses `--ease-text-base`
- Uses `--ease-color-muted`
- Uses `--ease-color-bg`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Overflow Wrap Break</span>
</div>
```

## Why is it useful?

This submission demonstrates the overflow wrap break CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
