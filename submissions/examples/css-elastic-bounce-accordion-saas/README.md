# Elastic Bounce Accordion

A responsive accordion with a smooth elastic bounce opening animation built using pure HTML and CSS.

## Features

- Elastic bounce opening animation
- Pure HTML & CSS
- Responsive layout
- No JavaScript
- Beginner Friendly

## Folder Structure

```text
css-elastic-bounce-accordion-saas/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Browser Support

All modern browsers.
