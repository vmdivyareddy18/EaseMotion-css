# Sustainability & Carbon Footprint Dashboard Component

## What does this do?

Provides a centralized interface for monitoring carbon emissions, energy consumption, renewable energy adoption, ESG metrics, and sustainability initiatives.

## How is it used?

```html
<div class="category-card healthy">
    <h3>Data Centers</h3>
</div>
```

## Why is it useful?

Organizations increasingly need visibility into environmental impact and sustainability goals. This component provides a reusable ESG dashboard for tracking emissions, energy efficiency, renewable adoption, and sustainability performance.
