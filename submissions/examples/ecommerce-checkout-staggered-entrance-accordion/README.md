# E-Commerce Checkout Staggered Entrance Accordion

A modern **Pure CSS Staggered Entrance Accordion** designed for **E-Commerce Checkout** layouts. The component features sequential entrance animations, responsive design, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Staggered Entrance animation
- Modern checkout-inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.6s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --primary:#16a34a;
    --secondary:#22c55e;

    --radius:18px;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Border radius
- Shadows
- Background colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
