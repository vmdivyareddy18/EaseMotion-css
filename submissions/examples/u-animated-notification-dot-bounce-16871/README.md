# Animated Notification Dot Bounce

## Overview

A reusable notification indicator featuring a subtle bouncing animation designed for alerts, messages, inboxes, and dashboard notifications.

## Features

- CSS-only implementation
- Lightweight animation
- Notification attention indicator
- Responsive layout
- Accessibility support
- Easy customization

## Usage

```html
<button class="icon-button">
  🔔
  <span class="notification-dot"></span>
</button>
```

## Accessibility

Supports:

```css
@media(prefers-reduced-motion:reduce)
```

to disable animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
