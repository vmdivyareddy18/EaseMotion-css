# Toast Notification Example

Reusable Toast component inspired by EaseMotion CSS.

## Features

- Success toast
- Error toast
- Warning toast
- Information toast
- Responsive
- Pure HTML & CSS
- Reduced motion support

## Folder Structure

```
toast-sp/
├── demo.html
├── style.css
└── README.md
```

## Browser Support

- Chrome
- Firefox
- Safari
- Edge

## Accessibility

Supports:

- prefers-reduced-motion
- Semantic structure
- Responsive layout
