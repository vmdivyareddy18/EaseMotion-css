# Tooltip Pop (Issue #18255)

A tooltip that pops up on hover. The tooltip scales from 0 to 1 with a smooth CSS transition.

## Usage

Open `demo.html` in a browser. Hover over the trigger text to see the tooltip.

## Files

- `demo.html` — HTML5 page with tooltip trigger
- `style.css` — Dark theme styles with CSS transition
- `README.md` — This file
