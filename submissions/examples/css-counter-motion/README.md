# 🔢 CSS Counter Motion

A collection of beautiful, pure CSS animated counters and stats for modern websites.

## 🚀 Features

### Counter Styles (8 types)
1. **Classic Number** - Gradient text with fade-in animation
2. **Gradient Counter** - Icons with gradient backgrounds
3. **Circular Progress** - SVG-based circular indicators
4. **Flip Counter** - Clock-style flip digits
5. **Neon Counter** - Cyberpunk neon glow effect
6. **Bar Counter** - Animated progress bars with shimmer
7. **Percentage Counter** - Conic gradient circles
8. **Icon Counter** - Icons with rotating animation

### Animation Features
- Staggered fade-in animations
- Shimmer effects on bars
- Circular progress fill
- Flip digit rotation
- Neon flicker effect
- Icon rotation on hover
- Smooth transitions

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All counter animations
- `readme.md` - Documentation
