# Immigration Legal Services Website

This demo site showcases a trustworthy, professional, multilingual legal services layout using **EaseMotion CSS classes** only.

## 🧩 Sections
- Hero with consultation CTA
- Services (visa, green card, citizenship, deportation defense)
- Attorney profiles
- Case success stats (count-up style)
- Multilingual service badges
- Client testimonials
- Free consultation form
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — entrance fade animations
- `ease-slide` — sliding entrance for headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs, badges, and attorney cards
- `ease-hover-opacity-reduce` — hover dimming for service cards and testimonials

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll through sections to see entrance animations.
3. Hover over buttons, cards, and badges to see interactive effects.
4. Replace placeholder images and text with real immigration legal service content.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion animation classes
