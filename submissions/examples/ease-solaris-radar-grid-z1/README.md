# Ease Solaris Radar Grid

## Overview

A futuristic radar visualization featuring rotating scan beams, glowing targets, and concentric grid rings using pure HTML and CSS.

## Features

- Rotating radar beam
- Animated target pulses
- Glassmorphism styling
- Responsive design
- Lightweight implementation
- Pure HTML & CSS

## Folder Structure

```text
ease-solaris-radar-grid-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Adds a reusable radar-style animation suitable for monitoring dashboards, cybersecurity interfaces, AI platforms, and futuristic control panels.
