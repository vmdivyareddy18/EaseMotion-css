# Glowing Status Badge (`ease-badge-has`)

### 1. What does this do?
An animated pulse status indicator displaying different colors (Green, Amber, Red) representing dashboard monitoring states.

### 2. How is it used?
Deploy the inline `.pulse-badge` status node:
```html
<div class="badge-item">
  <span class="pulse-badge green"></span>
  <span>Systems Online</span>
</div>
```

### 3. Why is it useful?
Provides an non-blocking notification signal element that loops continuously without slowing down rendering processes.
