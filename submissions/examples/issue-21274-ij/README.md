# Coin Spin — Spinning Coin Wobble

CSS-only coin spin with a 3D wobble illusion. The `scaleX` axis compresses at 90° and 270° to simulate edge-on perspective as the coin rotates.

## Files

- `demo.html` — coin container with face character
- `style.css` — radial gradient coin, wobble keyframe

## Usage

Open `demo.html` to see the coin spin with wobble effect.
