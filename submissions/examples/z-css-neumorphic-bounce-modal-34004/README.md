# CSS Elastic Bounce Modal

Pure CSS animated modal utilizing a smooth Elastic Bounce transition designed for Neumorphic Soft layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS Custom Properties
- Elastic bounce animation
- Neumorphic styling
- Soft shadows
- Smooth 60fps animations

## Variables

```css
--duration
--ease
--bg
--card
--text
--primary
```
