# Neon Kinetic Map Embed Component

A complex, pure-CSS tactical map interface overlay component featuring hardware-accelerated kinetic animations, pulsing radar markers, and high-fidelity Neon UI designs. Engineered natively for the **EaseMotion CSS** UI toolkit.

## Layout Features
- **Neon Design Identity:** Leverages multi-layered `box-shadow` structures, dark canvas grids, and electric high-vibrancy accent glow metrics.
- **Pure CSS Mode Management:** Handles state switches natively between the vertical tracking matrix scanner and continuous radar pings via input radio bindings.
- **Kinetic Animations:** Uses non-blocking CSS `@keyframes` logic paths for fluid, hardware-accelerated loops.
