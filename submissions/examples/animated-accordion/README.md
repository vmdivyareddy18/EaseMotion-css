# Animated Accordion

A pure CSS animated accordion component built using the native HTML `<details>` and `<summary>` elements. It features a bouncy icon rotation and a smooth fade-and-slide animation for the content when opened.

## Features
- **Zero JavaScript**: Entirely pure CSS implementation.
- **Accessible**: Uses native `<details>` and `<summary>` tags, providing built-in keyboard navigation and accessibility features automatically.
- **Exclusive Toggling**: Leverages the `name` attribute on the `<details>` tag so that only one accordion panel can be open at a time (supported in modern browsers).
- **Smooth Animations**: Includes a custom `cubic-bezier` rotation on the toggle icon and a keyframe slide-in animation for the body content.

## Why it fits EaseMotion
This component embodies the "animation-first" philosophy by taking a traditionally rigid native HTML element and making it feel fluid, responsive, and interactive, without relying on external libraries or JavaScript.
