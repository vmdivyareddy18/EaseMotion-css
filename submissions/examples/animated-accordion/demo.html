Frequently Asked Questions

What is EaseMotion CSS? +

EaseMotion CSS is a zero-dependency, animation-first CSS framework that lets you build polished interfaces with readable class names. No build step required.

Does it require a build step? +

No! You can drop in the CDN link and start using the utility classes instantly. No Webpack, Vite, or PostCSS configuration required, making it incredibly fast to set up.

Can I customize the animations? +

Yes, all animations and speeds are controlled by CSS variables which you can easily override in your own stylesheet to match your project's unique design system.
