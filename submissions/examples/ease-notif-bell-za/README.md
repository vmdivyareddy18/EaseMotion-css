# CSS Notification Bell

## What does this do?
Notification bell icon with badge counts, dot indicator, and notification dropdown list.

## How is it used?
```html
<div class="nb-bell-wrap"><button class="nb-bell">&#128276;</button><span class="nb-badge nb-bdg-danger">3</span></div>
```

## Why is it useful?
Notification UI patterns without JS. Dot/count badges, dropdown list, unread indicators, bell icon states.
