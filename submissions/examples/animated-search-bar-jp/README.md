# Animated Search Bar Component

## Description

A modern animated search bar with smooth focus effects, icon animations, responsive layout, and interactive buttons.

---

## Features

- Smooth search bar focus animation
- Animated search icon
- Interactive search button
- Modern rounded design
- Responsive mobile layout
- Lightweight CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.search-section` – Main search area
- `.search-container` – Search bar wrapper
- `.search-input` – Input field
- `.search-icon` – Animated search icon
- `.search-btn` – Search button

---

## Accessibility

Supports `prefers-reduced-motion` to reduce animations for users who prefer less motion.

---

## Use Cases

- Website navigation
- Admin dashboards
- Blogs and articles
- E-commerce search systems
