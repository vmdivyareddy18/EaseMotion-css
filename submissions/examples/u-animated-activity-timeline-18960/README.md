# Animated Activity Timeline

A responsive animated timeline component for displaying project updates, achievements, events, and user activity.

## Features

- Animated timeline reveal
- Hover interactions
- Pulsing timeline indicators
- Multiple activity categories
- Responsive design
- Clean dashboard-style layout

## Files

- demo.html
- style.css

## Usage

Open demo.html in your browser.

## Customization

You can customize:

- Timeline colors
- Animation duration
- Timeline categories
- Card styling
- Hover effects

## Example Use Cases

- Project tracking
- User activity feeds
- Product roadmaps
- Company history pages
- Event timelines
