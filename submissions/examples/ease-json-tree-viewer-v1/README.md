# Ease JSON Tree Viewer

## Overview

Ease JSON Tree Viewer is a reusable HTML & CSS component that visually represents nested JSON structures in a clean tree layout inspired by browser developer tools.

---

## Features

- 🌳 Tree-style JSON structure
- 🔵 Colored keys
- 🟢 Colored values
- ✨ Hover interaction
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

- Nested JSON tree
- Object hierarchy
- Color-coded keys and values
- Interactive hover effect

---

## Usage

```html
<div class="viewer">
...
</div>
```

---

## Customization

- Key colors
- Value colors
- Tree spacing
- Typography
- Hover animation

---

## Browser Support

Works in all modern browsers supporting Flexbox and CSS pseudo-elements.
