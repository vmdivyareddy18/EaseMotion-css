# ease-tilt-right – Tilt Right

A subtle hover animation: the element rotates 5 degrees to the right with a smooth 0.3s transition.

- **Components:** ease-card
- **Hover Effects:** ease-hover-shadow-xl
- **Animation:** ease-fade-in, ease-delay-200, ease-delay-500, ease-transition
