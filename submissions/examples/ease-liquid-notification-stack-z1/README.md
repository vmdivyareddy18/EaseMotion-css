# Ease Liquid Notification Stack

## Overview

A modern notification stack featuring liquid-inspired motion, glowing status indicators, and glassmorphism styling using only HTML and CSS.

## Features

- Glassmorphism notification cards
- Floating primary notification
- Animated glowing status dots
- Smooth hover interactions
- Responsive layout
- Pure HTML & CSS

## Folder Structure

```text

```

Open `demo.html` in any modern browser.

Optimized for desktop, tablet, and mobile devices.
