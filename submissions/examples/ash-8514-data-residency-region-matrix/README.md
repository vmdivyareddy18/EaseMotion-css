# Data Residency Region Matrix

A high-fidelity **Data Residency Region Matrix** built entirely with HTML and CSS.

## Overview

This example demonstrates a modern enterprise governance dashboard for visualizing global data residency compliance across cloud regions. It combines an advanced compliance matrix, executive KPI cards, regulatory readiness indicators, animated analytics, regional distribution summaries, and operational insights into a responsive interface suitable for security and compliance platforms.

Designed as a realistic enterprise dashboard rather than a simple table, this example can be adapted for cloud governance, cybersecurity, privacy engineering, and regulatory compliance applications.

---

## Features

- Enterprise governance dashboard
- Advanced data residency comparison matrix
- Sticky table header
- Executive KPI cards
- Region compliance indicators
- Cross-border transfer policies
- Encryption readiness monitoring
- Risk classification badges
- Animated compliance score ring
- Regulatory timeline panel
- Policy readiness progress indicators
- Regional distribution analytics
- Responsive enterprise layout
