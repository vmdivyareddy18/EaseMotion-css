# Animated Status Indicator (`ease-status-cv`)

A lightweight animated status indicator component built using **HTML5** and **Pure CSS**.

## Features

- 🟢 Online status
- ⚪ Offline status
- 🔴 Busy status
- 🌙 Away status
- 💫 Pulse animation
- ✨ Glow/Ripple effect
- 📱 Responsive layout
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-status-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
