Scroll Snap Type Utilities

Configure scroll containers for responsive native snap behaviors on X, Y, or both axes.

Horizontal Mandatory Snapping

Using .snap-x .snap-mandatory to lock scrolling coordinates to the nearest child element on scroll release.

01

Mandatory Slide One

Scroll horizontally. Release your scroll gesture to watch the container snap immediately to this item.

.snap-x .snap-mandatory
02

Mandatory Slide Two

Mandatory strictness forces alignment to a snap point when scrolling stops.

.snap-x .snap-mandatory
03

Mandatory Slide Three

Ideal for full slide carousels where only one item should be focused at a time.

.snap-x .snap-mandatory

Horizontal Proximity Snapping

Using .snap-x .snap-proximity. Snaps only when the scroll gesture stops close to a snap coordinate.

01

Proximity Slide One

Scroll slowly. If you release close to this card, it snaps. If you release far, it rests normally.

.snap-x .snap-proximity
02

Proximity Slide Two

Proximity strictness allows more natural, freeform scrolling but locks when near boundaries.

.snap-x .snap-proximity
03

Proximity Slide Three

Great for general lists where users want custom scrolling control but clean alignment when near edges.

.snap-x .snap-proximity

Vertical Snapping

Using .snap-y .snap-mandatory to lock scrolling coordinates vertically.

01

Vertical Slide One

Scroll vertically inside this frame. Release scroll to watch the frame lock onto this card.

.snap-y .snap-mandatory
02

Vertical Slide Two

Excellent for vertical dashboard grids, story decks, or paginated columns.

.snap-y .snap-mandatory
03

Vertical Slide Three

Provides clean, native pagination without any JavaScript scrolling event overhead.

No Snapping

Using .snap-none to disable snapping entirely (default scroll physics).

01

Normal Card One

Scroll is completely fluid. Released scrolling positions rest naturally based on drag friction.

.snap-none
02

Normal Card Two

Standard scroll behavior. Ideal when snapping interferes with rapid data scanning.
