# Campaign Budget Pacing Meter

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive campaign budget pacing dashboard that visualizes advertising spend against planned budgets using a circular progress meter and supporting analytics. Suitable for marketing dashboards, advertising platforms, campaign management tools, and analytics systems.

## Features

- Circular budget pacing meter
- Animated progress bar
- Budget summary metrics
- Campaign performance cards
- Responsive layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="meter">

    <div class="ring">

        <div class="inner">

            <strong>68%</strong>

        </div>

    </div>

</div>
```
