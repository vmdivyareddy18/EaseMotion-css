# Modern Admin Dashboard Suite

A premium responsive admin dashboard example built using pure HTML and CSS.

## Features

- Responsive Sidebar Navigation
- Top Navigation Bar
- Analytics Widgets
- Revenue Chart
- Traffic Chart
- User Management Table
- Responsive Layout
- Accessibility Friendly Design

## Components

### Sidebar
Provides navigation links for dashboard sections.

### Top Navigation
Includes search bar, notifications, and profile access.

### Analytics Widgets
Displays:

- Revenue
- Users
- Orders
- Conversion Rate

### Charts

- Animated bar chart
- Circular traffic chart

### Table

Responsive user management table with status badges.

## Responsive Support

- Desktop
- Tablet
- Mobile
