print-color-adjust for Print

Optimize color rendering for print with ink-saving palettes and color scheme adjustments.

Screen Palette (vivid)

print-color-adjust: economy — browser may reduce color fidelity.

Print Palette (ink-saving)

print-color-adjust: exact — preserve these print-specific colors.

Color Adjustment Modes

economy Browser may swap, dither, or convert colors to save ink.
exact Browser must render colors as specified (where possible).

Print Preview

Use Ctrl+P to preview the print layout. The ink-saving palette uses fewer saturated colors for economical printing. Elements with print-color-adjust: exact preserve their specified colors.
