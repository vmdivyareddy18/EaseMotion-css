# Animated Archaeological Expedition Dashboard

A modern dashboard for tracking archaeological excavations, artifact discoveries, preservation activities, and expedition progress.

## Features

- Excavation statistics cards
- Artifact discovery monitoring
- Site mapping progress bars
- Preservation status indicators
- Research activity feed
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Archaeological research projects
- Historical preservation programs
- Museum field operations
- Scientific expeditions
- Educational history platforms
