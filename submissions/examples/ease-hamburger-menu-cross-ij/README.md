# Hamburger Menu Cross (`ease-hamburger-menu-cross-ij`)

An animated hamburger menu icon that morphs into a close (X) icon on click. Built with **HTML5**, **CSS3**, and **Vanilla JavaScript**.

## Features

- Smooth rotate & fade transitions
- Customizable CSS variables for color, size, and duration
- Accessible button with `aria-label`
- No external dependencies (except Inter font via Google Fonts)

## CSS Custom Properties

| Variable                    | Default    | Description            |
|-----------------------------|------------|------------------------|
| `--hmc-line-color`          | `#1e293b`  | Color of the lines     |
| `--hmc-line-thickness`      | `3px`      | Thickness of each line |
| `--hmc-line-length`         | `28px`     | Width of each line     |
| `--hmc-transition-duration` | `0.3s`     | Animation speed        |

Open `demo.html` directly in any modern browser. Click the button to toggle between hamburger and close icon.

## Browser Support

Works in all modern browsers.
