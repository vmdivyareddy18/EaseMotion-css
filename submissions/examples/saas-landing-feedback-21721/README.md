# Business Landing — Customer Feedback Platform (`saas-landing-feedback-21721`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template designed specifically for Survey, NPS, and Customer Feedback software (similar to Typeform), utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A modern survey tool needs to feel conversational, elegant, and frictionless. This template achieves a premium aesthetic using a warm color palette, large typography, and extremely smooth structural animations to mimic the feeling of a seamless, interactive form experience.

The layout includes:
- A hero section featuring an animated CSS-built interactive NPS survey mockup.
- A trusted-by integration logo strip.
- A 4-item feature grid with subtle rotation hover states.
- A dark-mode data analytics dashboard mockup highlighting sentiment analysis.
- A clear, 3-tier pricing layout.
- High-contrast customer testimonials and a strong footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-expand` (Used creatively on the survey progress bar to simulate completion loading)
- `.ease-slide-up` (Applied heavily with custom cubic-bezier timing in the CSS to create the signature "smooth slide in" effect associated with premium form builders)
- `.ease-pulse` (Used on the active '8' NPS score to draw attention to the interactive element)
- `.ease-fade-in` / `.ease-fade-up` (For smooth, staggered text and UI entrances across the page)
- `.ease-bounce-in` / `.ease-zoom-in` (Used on decorative floating badges and mockups to create depth)
- `.ease-hover-lift` / `.ease-hover-rotate` (For interactive NPS score buttons and feature icons)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, typography, and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The NPS survey mockup and the analytics dashboard are built entirely with structural HTML and CSS, requiring zero JavaScript for the visual rendering and entrance animations.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/saas-landing-feedback-21721/` directory to avoid modifying core files directly and to ensure zero deletions.*
