Glassmorphism UI

Transparent, frosted-glass UI components using CSS backdrop-filter — a modern design trend that creates depth and layering in web interfaces.

What is Glassmorphism?

Glassmorphism is a design trend where UI elements appear as translucent, frosted-glass surfaces floating over colorful backgrounds. The effect is created using the CSS backdrop-filter: blur() property combined with semi-transparent backgrounds and subtle borders.

Overlay Dialog

Glass modals create a sense of depth by placing a frosted overlay above the page content. This technique works especially well over colorful or image-heavy backgrounds.

Design Token Integration

This glassmorphism system is built on EaseMotion's design tokens: --ease-glass-bg and --ease-glass-border provide consistent translucent values, while --ease-glow-* tokens add colored glows to interactive elements.
