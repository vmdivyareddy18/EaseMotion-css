CSS-Only Morphing Submit Button

A submit button that morphs into a loading spinner seamlessly — pure CSS.

Demo

Click and hold (or :active) the button to see it morph

Use :active state — click and hold to see the spinner.

How It Works

On :active, the button shrinks to a circle, hides the text, and reveals a spinning border via ::after. Uses transition for the morph and animation for the spin.
