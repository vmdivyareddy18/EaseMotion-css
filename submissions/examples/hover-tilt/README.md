# Magnetic Micro-Tilt with Spotlight Overlays

This directory introduces a performance-focused, zero-dependency 3D interaction layer utilizing hardware-accelerated CSS perspective utilities.

### Classes
- `.ease-hover-tilt`: Attaches the structural 3D perspective variables and rotation matrices.
- `.ease-spotlight-purple`: Projects an ambient indigo directional spotlight flare on interaction.
- `.ease-spotlight-green`: Projects an ambient emerald directional spotlight flare on interaction.
- `.ease-spotlight-red`: Projects an ambient crimson directional spotlight flare on interaction.
