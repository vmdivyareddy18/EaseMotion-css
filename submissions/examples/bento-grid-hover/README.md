# Bento Grid Hover Expansion

A modern CSS-only Bento Grid layout with smooth hover interactions.

## Features

- Responsive Bento Grid layout
- Card lift animation on hover
- Subtle scaling effect
- Focus effect by dimming neighboring cards
- CSS-only implementation
- Mobile responsive

## Folder Structure
```

bento-grid-hover/
├── demo.html
├── style.css
└── README.md

```

## Usage

1. Open `demo.html` in a browser.
2. Hover over any card to see the expansion effect.
3. Customize colors, spacing, and grid sizes as needed.

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- CSS Transitions

## Preview

Hover over a card to:
- Lift the card
- Scale it slightly
- Highlight it with a shadow
- Dim surrounding cards
