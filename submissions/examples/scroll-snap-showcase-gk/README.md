# Scroll Snap Showcase

## Overview

A responsive Scroll Snap Showcase built using pure HTML and CSS for the EaseMotion CSS project.

This example demonstrates multiple practical uses of CSS Scroll Snap, including horizontal galleries, vertical sections, and product carousels.

---

## Features

- Horizontal Scroll Snap
- Vertical Scroll Snap
- Product Carousel
- Responsive Design
- Pure HTML & CSS
- Modern UI
- Smooth Scrolling
