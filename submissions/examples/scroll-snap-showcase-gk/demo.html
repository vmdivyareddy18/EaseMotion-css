How Scroll Snap Works

CSS Scroll Snap makes scrolling smoother by automatically aligning content while scrolling. 

.scroll-container{
    overflow-x:auto;
    scroll-snap-type:x mandatory;
}

.card{
    scroll-snap-align:center;
}
