# macOS Window

## What does this do?

Creates a macOS-inspired application window with classic traffic-light controls and a terminal-style content area.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="window">

    <div class="window-bar">

        <div class="window-controls">

            <span class="control close"></span>
            <span class="control minimize"></span>
            <span class="control maximize"></span>

        </div>

        <div class="window-title">
            README.md
        </div>

    </div>

    <div class="window-body">

        ...

    </div>

</div>
```

---

## Why is it useful?

macOS-style windows are commonly used in documentation sites, developer portfolios, product landing pages, and code showcases. This standalone example demonstrates a polished desktop-inspired UI while aligning with EaseMotion CSS's clean and modern design philosophy.
