Features & Accessibility Details

  • Elastic Squash-and-Stretch: Opening a popover triggers a physical squash-and-stretch wobble keyframe animation (wobble-focus-ag) simulating structural fluid motion.
  • Zero-JS Click-Outside Dismiss: Leverages transparent sibling label overlays (`.popover-overlay-ag`) linked to trigger checkboxes to uncheck state and close popovers on document clicks.
  • Modern Discrete Transitions: Uses new CSS transition attributes (`transition-behavior: allow-discrete` and `@starting-style`) to animate popover displays smoothly without JS toggles.
  • Keyboard Accessibility: Interactive labels are fully focusable with keyboard outline indicators. The toggle elements can be activated using standard Space/Enter keys.
  • Custom Parameters: Variables such as squash amplitude, wobble duration, and timing eases are fully exposed at the component level using CSS Custom Properties (e.g., --wobble-scale-x).
