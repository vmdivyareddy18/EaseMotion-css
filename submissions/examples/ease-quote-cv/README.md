# Animated Quote Block (cv variant)

## What does this do?
A collection of 5 beautifully styled animated quote blocks perfect for blog citations, testimonials, article pull quotes, and design system callouts. Features include hover effects, fade-in animations, and multiple color options.

## How is it used?

### Basic Usage:

```html
<!-- Classic quote -->
<blockquote class="ease-quote-cv ease-quote-cv-classic">
    <p>Design is not just what it looks like. Design is how it works.</p>
    <cite>— Steve Jobs</cite>
</blockquote>

<!-- Classic -->
<blockquote class="ease-quote-cv ease-quote-cv-classic">...</blockquote>

<!-- Glass (Glassmorphism) -->
<blockquote class="ease-quote-cv ease-quote-cv-glass">...</blockquote>

<!-- Gradient (with glow border) -->
<blockquote class="ease-quote-cv ease-quote-cv-gradient">...</blockquote>

<!-- Highlight (yellow highlighter) -->
<blockquote class="ease-quote-cv ease-quote-cv-highlight">...</blockquote>

<!-- Card (full card with shadow) -->
<blockquote class="ease-quote-cv ease-quote-cv-card">...</blockquote>

<blockquote class="ease-quote-cv ease-quote-cv-classic ease-quote-cv-info">...</blockquote>
<blockquote class="ease-quote-cv ease-quote-cv-classic ease-quote-cv-success">...</blockquote>
<blockquote class="ease-quote-cv ease-quote-cv-classic ease-quote-cv-pink">...</blockquote>
