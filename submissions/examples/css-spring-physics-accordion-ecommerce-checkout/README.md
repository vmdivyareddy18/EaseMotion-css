# CSS Spring Physics Accordion for E-Commerce Checkout Layouts

A **pure CSS Spring Physics Accordion** created for modern **E-Commerce Checkout** interfaces. The component provides smooth spring-inspired expand/collapse animations while remaining lightweight, responsive, accessible, and completely JavaScript-free.

## Features

- 🛒 Pure HTML & CSS
- 🌱 Smooth spring-inspired accordion animation
- 📦 Designed for E-Commerce checkout pages
- 📱 Fully responsive
- ♿ Keyboard accessible using native `<details>` and `<summary>`
- 🎨 Customizable with CSS Custom Properties
- ⚡ No JavaScript required

## Folder Structure

## Usage

1. Open `demo.html` in any modern browser.
2. Click an accordion section to expand or collapse it.
3. Modify the CSS variables in `style.css` to match your project's branding and animation preferences.

## Customization

Edit the variables in the `:root` selector:

```css
:root {
  --primary: #2563eb;
  --accent: #22c55e;
  --radius: 16px;
  --duration: .45s;
  --easing: cubic-bezier(.22,1,.36,1);
}
```

You can customize:

- Primary and accent colors
- Border radius
- Animation duration
- Easing function
- Shadows and spacing
