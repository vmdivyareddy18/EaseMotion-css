# Steps Timing Function

Demonstrates CSS `steps()` timing function used for sprite-sheet animation, typewriter effects, and discrete step progress. Each example shows a different application of step-based timing.

## Features

- Typewriter effect using `steps(18)` with a blinking cursor.
- Step count comparison: 4, 8, and 12 steps side by side.
- Jump keyword comparison: `jump-start`, `jump-end`, and `jump-both`.
- All animated elements share the same duration for fair comparison.

## Topics Covered

- `steps()` timing function with count and jump keywords
- `animation-timing-function` with step-based easing
- Typewriter reveal pattern
- Discrete vs continuous animation
