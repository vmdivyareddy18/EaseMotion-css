CSS Scrollbar Color Theme Utilities

Style native browser scrollbars using standardized CSS properties — scrollbar-color and scrollbar-width — with vendor-prefixed fallbacks for cross-browser consistency.

The base utility from the issue snippet. Uses a purple thumb on a light track. Applies both the modern scrollbar-color standard and ::-webkit-scrollbar fallbacks so it works in all browsers.

🔧 CSS Reference

/* 1. Base Theme (from the issue snippet exactly) */
.ease-scroll-theme {
  overflow: auto;
  scrollbar-width: thin;
  scrollbar-color: #6c63ff #ececec;
}

/* WebKit fallback (Chrome, Safari, Edge) */
.ease-scroll-theme::-webkit-scrollbar {
  width: 8px;
}

.ease-scroll-theme::-webkit-scrollbar-thumb {
  background: #6c63ff;
  border-radius: 999px;
}

/* 2. Dark Theme */
.ease-scroll-dark {
  overflow: auto;
  scrollbar-width: thin;
  scrollbar-color: #64748b #1e293b;
}

.ease-scroll-dark::-webkit-scrollbar { width: 8px; }
.ease-scroll-dark::-webkit-scrollbar-thumb {
  background: #64748b;
  border-radius: 999px;
}
.ease-scroll-dark::-webkit-scrollbar-track {
  background: #1e293b;
}

/* 3. Hidden Scrollbar */
.ease-scroll-hidden {
  overflow: auto;
  scrollbar-width: none; /* Firefox */
}

.ease-scroll-hidden::-webkit-scrollbar {
  display: none; /* Chrome, Safari, Edge */
}
