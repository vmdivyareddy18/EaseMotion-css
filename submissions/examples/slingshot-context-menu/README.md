# Slingshot Context Menu

A responsive **Slingshot Context Menu** component built with pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML and CSS
- No JavaScript
- Retro-inspired design
- Slingshot style opening animation
- Smooth hover interaction
- Responsive layout
- Accessible menu buttons
- Uses EaseMotion CSS variables and `ease-*` animation classes
