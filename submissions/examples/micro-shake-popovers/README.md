# CSS Micro Shake Popover

A responsive **Pure CSS Micro Shake Popover** designed for **Responsive Dashboard Layouts**. The component displays contextual information using a subtle micro-shake animation and a hover/focus popover.

## Features

- Pure HTML & CSS
- Responsive Layout
- Keyboard Accessible (`:focus-visible`)
- Micro Shake Animation
- Hover & Focus Popover
- CSS Custom Properties
- No JavaScript

## Technologies

- HTML5
- CSS3

## CSS Variables

```css
--shake-duration
--shake-distance
--primary
--popover-bg
--transition
--radius
--shadow
```
