# Mantis Shrimp

A pure CSS and HTML peacock mantis shrimp smashing a snail shell with its club, the fastest strike in the sea.

## What it shows

- The raptorial club on a nested arm that cocks back and snaps out on a fast keyframe, timed to a cavitation flash on the shell
- Independently swivelling stalked eyes, each iris a conic gradient spectrum for the animal's famous colour vision
- A rainbow banded tail fan and body plates from repeating linear gradients
- A snail shell drawn with a repeating conic gradient that rattles and cracks on the hit
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
