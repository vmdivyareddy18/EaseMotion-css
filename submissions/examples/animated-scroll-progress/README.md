# Animated Scroll Progress Indicator

A clean, modern, and fully responsive **Scroll Progress Indicator with Reading Time** perfect for blogs, articles, and documentation. This example demonstrates how to combine **EaseMotion CSS** for smooth entrance animations, continuous pulse effects, and interactive hover states to create an engaging reading experience.

## 🚀 Features

- **Smooth Progress Bar**: A gradient progress bar at the top that fills as users scroll, with a glowing effect.
- **Real-time Percentage**: Displays the current scroll percentage with smooth updates.
- **Reading Time Indicator**: Calculates and displays estimated reading time with a continuous pulse animation using `ease-pulse`.
- **Staggered Content Entrances**: Article content fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a cascading load effect.
- **Back to Top Button**: Smooth scroll-to-top button that appears after scrolling down, with hover lift effect using `ease-hover-grow`.
- **Interactive Tags & Share Buttons**: Hover effects on tags and social share buttons for tactile feedback.
- **Responsive Design**: Adapts gracefully to mobile screens with adjusted sizing and layout.
- **Accessible**: Includes proper `aria-label` attributes for screen readers.
