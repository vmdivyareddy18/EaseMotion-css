# Hover Jelly Animation

## What does this do?
This adds a physics-inspired "jelly" or "squish" animation. When a user hovers over an element, it rapidly stretches and compresses horizontally and vertically before settling back to its original shape.

## How is it used?
Apply the `.hover-jelly` class to any inline-block or block element (such as a button, card, or icon) that you want to animate on hover.

Example:
```html
<button class="hover-jelly">
  Hover me!
</button>
```

## Why is it useful?
- Makes UI elements feel more tactile and responsive by simulating organic, elastic physics.
- Excellent for playful brands and calls-to-action (CTAs) that need a highly noticeable, fun micro-interaction.
