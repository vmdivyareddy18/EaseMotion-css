# Animated Border Effects

## What does this do?
CSS animated border effects including rotating gradient borders, neon pulse borders, marching dashed ants, corner accents, and hover slide borders.

## How is it used?
```html
<div class="ab-card ab-rotate"><div class="ab-content">Content</div></div>
```

## Why is it useful?
Provides visually rich border animations without JavaScript. Five distinct border effect families. Customizable colors via CSS variables. Hover-triggered slide borders. GPU-accelerated animations.
