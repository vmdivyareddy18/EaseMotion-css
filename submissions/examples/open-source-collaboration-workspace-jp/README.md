# Open Source Collaboration Workspace

A modern Open Source Collaboration Workspace dashboard built using pure HTML and CSS.

## Features

- Community Overview Statistics
- Pull Request Analytics
- Contributor Leaderboard
- Repository Insights Panel
- Activity Timeline Feed
- Glassmorphism UI
- Cyan & Lime Gradient Theme
- Hover Animations
- Fully Responsive Layout
- Accessibility Friendly Structure
