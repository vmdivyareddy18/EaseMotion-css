## Overview

A pure CSS toggle switch component with smooth bounce animations, multiple sizes, color variants, and full accessibility support.

## Classes

| Class | Description |
|-------|-------------|
| `.ease-toggle` | Base toggle wrapper |
| `.ease-toggle-sm` | Small size variant |
| `.ease-toggle-lg` | Large size variant |
| `.ease-toggle-success` | Green active state |
| `.ease-toggle-danger` | Red active state |
| `.ease-toggle-warning` | Amber active state |

## Usage

```html
&lt;label class="ease-toggle"&gt;
  &lt;input type="checkbox"&gt;
  &lt;span class="ease-toggle-track"&gt;&lt;/span&gt;
  &lt;span class="ease-toggle-label"&gt;Label text&lt;/span&gt;
&lt;/label&gt;
