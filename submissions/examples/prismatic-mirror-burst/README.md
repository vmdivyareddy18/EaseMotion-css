# Prismatic Mirror Burst Animation Utility

## Overview
This utility creates a prismatic mirror burst effect where glowing mirrored shards burst outward from the center and fade away.

## Usage

```html
<div class="ease-prism-burst">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first interaction
- Great for premium cards, hero sections, and futuristic UI
