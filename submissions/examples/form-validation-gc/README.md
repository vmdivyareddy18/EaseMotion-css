# Animated Form Validation UI Example

## 1. What does this do?

This submission demonstrates a form field validation pattern with animated error and success messages. It includes input states, icon feedback, and animated message reveal transitions using CSS `@keyframes`.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example uses `style.css` for validation field styling and motion. Copy the `.validation-field` markup and CSS to integrate the animated validation patterns into your forms.

---

## 3. Why is it useful?

Animated validation feedback helps users quickly identify and correct input issues, improving form completion and reducing errors. This example shows how to provide clear, attractive state feedback with CSS-only animations.
