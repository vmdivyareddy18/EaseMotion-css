# Virtual Event Management Platform

## Overview
Create a conference and event operations dashboard.

This project implements the Phase 123 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Event analytics**: Built with glassmorphism and CSS grid/flexbox.
- **Attendee metrics**: Built with glassmorphism and CSS grid/flexbox.
- **Session overview**: Built with glassmorphism and CSS grid/flexbox.
- **Schedule widgets**: Built with glassmorphism and CSS grid/flexbox.
- **Responsive design**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
