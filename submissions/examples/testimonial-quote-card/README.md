# Testimonial Quote Card

A testimonial card component with a quotation mark icon, quote text, and author details including an initial avatar. Two variants are shown: light and dark theme. The avatar displays the author's first letter in a neutral colored circle.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the testimonial cards side by side.

## Accessibility notes

Uses the `<blockquote>` element for semantic quoting. The quote icon SVG is purely decorative and has no interactive role.
