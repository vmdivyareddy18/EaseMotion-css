# Border Image Width

Demonstrates `border-image-width` — the property that determines the width of the border image area, independent of `border-width`. Covers auto, number, pixel, percentage, and asymmetric values.

## Width Controls

`border-image-width` can be set as `auto` (inherits from `border-width`), a unitless number (multiplies `border-width`), a length (absolute), or a percentage (of border-box). When larger than `border-width`, the image extends into the padding area; when smaller, the border background shows through.

## Usage

Open `demo.html` in any modern browser. Each card shows a gradient border image with a specific `border-image-width` configuration. Side-by-side comparisons demonstrate how `border-image-width` and `border-width` interact. Edit values in `style.css` to explore different sizing behaviors.
