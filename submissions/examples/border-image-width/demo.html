Border Image Width

How border-image-width sizes the border image area

Automatic vs Explicit Width

Auto
border-image-width: auto

Defaults to the computed border-width value

Number (1.5)
border-image-width: 1.5

Multiplied by border-width (1.5x wider)

Pixel (40px)
border-image-width: 40px

Explicit pixel width, independent of border-width

Percentage (50%)
border-image-width: 50%

Relative to the element's border-box size

Width vs Border-Width

Thin border, thick image
border-width: 10px
border-image-width: 40px
Thick border, thin image
border-width: 40px
border-image-width: 10px
Zero border, image set
border-width: 0
border-image-width: 30px
Border set, zero image
border-width: 30px
border-image-width: 0

Asymmetric Widths

Two Values
border-image-width: 20px 40px

Top/bottom = 20px, left/right = 40px

Three Values
border-image-width: 10px 30px 50px

Top = 10px, left/right = 30px, bottom = 50px

Four Values
border-image-width: 10px 20px 30px 40px

Top, right, bottom, left — each unique

border-image-width can be auto, number, length, or percentage.
