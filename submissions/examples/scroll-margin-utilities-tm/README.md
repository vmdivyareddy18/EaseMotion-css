# Scroll Margin Utilities: Scroll Margin And Scroll Padding

Demonstrates CSS scroll-margin and scroll-padding utilities for precise scroll-into-view behavior, anchor navigation, and sticky section headers using EaseMotion spacing tokens.

## Features

- Demonstrates **scroll-margin** CSS property with multiple practical variants
- Uses EaseMotion design tokens (`--ease-color-*`, `--ease-space-*`, `--ease-radius-*`)
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`
- Multiple size/style variants for real-world usage

## Usage

Apply the utility class to any element:

```html
<div class="section-margin-top">Content</div>
```

## Why is it useful?

The **scroll-margin** CSS property is part of the modern CSS specification and enables
fine-grained control over scroll-margin-utilities. It integrates seamlessly with EaseMotion's design token
system, making it easy to maintain consistent spacing, colors, and typography across your project.
