# Animated Disaster Response Dashboard

A modern emergency management dashboard for tracking disaster response operations, evacuation progress, resource allocation, and emergency alerts.

## Features

- Emergency statistics cards
- Alert monitoring panel
- Response progress indicators
- Timeline tracking
- Responsive layout
- Smooth animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Emergency management systems
- Disaster response centers
- Crisis coordination platforms
- Public safety dashboards
- Humanitarian operations
