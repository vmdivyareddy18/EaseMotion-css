# CSS isolation Stacking Context Example

## Description
An interactive, self-contained educational demonstration of the CSS `isolation` layout property. Showcases how to encapsulate z-index stacking layers cleanly within component blocks to prevent layout layer pollution.

## Key Concepts Illustrated
- **Z-Index Leakage**: How elements with high relative z-index layers can unintentionally cover global page elements.
- **Stacking Isolation**: Using `isolation: isolate` to lock depth context variables into parent blocks natively.
