# Pure CSS Shake Radio Button Component

A lightweight, responsive, and accessible radio button component built entirely with CSS for EaseMotion CSS. The selected radio button features a subtle shake animation with a glowing accent, making it ideal for dark mode settings, preference selectors, and modern forms.

## Features

- **Pure CSS:** No JavaScript required.
- **EaseMotion Ready:** Uses EaseMotion CSS utility classes for animations and layout.
- **Shake Animation:** Selected radio button performs a subtle shake effect.
- **Dark Mode Design:** Modern interface with glowing selection states.
- **Responsive Layout:** Adapts smoothly to desktop and mobile devices.
- **Accessible:** Built using native radio inputs for keyboard and screen reader support.

## Configuration Parameters

The component can be customized using the following CSS custom properties:

| Variable | Description | Default Value |
| :--- | :--- | :--- |
| `--primary` | Primary accent color | `#38bdf8` |
| `--primary-glow` | Glow effect color | `rgba(56, 189, 248, 0.45)` |
| `--bg` | Background color | `#09090b` |
| `--surface` | Card background color | `#18181b` |
| `--border` | Border color | `#3f3f46` |
| `--text` | Primary text color | `#f8fafc` |
| `--muted` | Secondary text color | `#a1a1aa` |
| `--radius` | Card border radius | `18px` |
| `--transition` | Default transition timing | `0.3s ease` |
