What is EaseMotion?

EaseMotion is a lightweight CSS utility library providing drop-in animations and transitions for modern web interfaces. It prioritizes performance, accessibility, and ease of use.

v2.1.0

Key Features

Every animation duration, timing function, and color can be overridden through CSS custom properties. Components support keyboard navigation, ARIA attributes, and reduced motion preferences.

Getting Started

Install via npm with npm install easemotion or include directly from a CDN. Refer to the getting started guide for detailed installation instructions.
