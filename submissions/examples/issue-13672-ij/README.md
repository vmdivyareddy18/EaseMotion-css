# Ease Loader Bar Indeterminate

Closes #13672

## Description
An indeterminate progress bar following the Material Design pattern. The inner bar slides left to right in an infinite loop, with configurable width variants.

## Acceptance Criteria
- Inner bar slides left to right repeatedly
- Continuous infinite animation loop
- Pause / Resume toggle functionality
- Clean, minimal Material Design aesthetic

## Files
- `style.css` — Dark theme with indeterminate bar animation
- `demo.html` — Interactive loader demo with pause/resume
