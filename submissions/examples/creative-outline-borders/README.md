# Creative Outline Borders

## Overview

A collection of eight unique and creative CSS outline border effects demonstrating different animation techniques and visual styles. Each example uses pure CSS and showcases a distinct approach to border interactions, making this collection useful for buttons, cards, call-to-actions, and modern UI components.

## Included Effects

1. Border Draw
2. Neon Glow Border
3. Gradient Spin Border
4. Marching Ants Border
5. Electric Spark Border
6. Corner Expand Border
7. Scanner Border
8. Pixel Build Border

## Features

* 8 completely different border styles
* Pure CSS implementation
* No JavaScript dependencies
* Responsive showcase layout
* Hover-driven animations
* Lightweight and easy to integrate
* Suitable for modern UI and component libraries

## Usage

Include the desired class on an element:

```html
<div class="card border-draw">
  Border Draw
</div>

<div class="card neon-border">
  Neon Glow
</div>

<div class="card gradient-spin">
  Gradient Spin
</div>
```

## HTML Structure

```html
<div class="grid">
  <div class="card border-draw"></div>
```
