# 🧶 Ease Yarn Unwind

A CSS animation of a yarn ball spinning and unwinding with a decorative trail effect.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.yarn-ball` element.

## Custom CSS Variables

| Variable           | Default   | Description                  |
|--------------------|-----------|------------------------------|
| `--yarn-color`     | `#e8a87c` | Color of the yarn ball       |
| `--unwind-duration`| `3s`      | Duration of one full spin    |
| `--ball-size`      | `60px`    | Diameter of the yarn ball    |

## Interactive Controls

Open `demo.html` to use the color pickers and range sliders for live customization.
