❌ Problem: Broken Hover on Touch

Hover state gets stuck on touch

No touch feedback at all

Issue: On mobile, `:hover` states persist after tapping, making the UI feel "stuck" or unresponsive.

✅ Solution: Media Query Aware Hover

Works on desktop & mobile

Responds to your input type

Fixed: Uses @media (hover: hover) to only apply hover on devices that support it.

🧪 How to Test

Desktop:

  • Hover over buttons — smooth scale and color transition
  • Click buttons — they respond immediately

Mobile/Touch:

  • Tap buttons — active state shows, then clears
  • No "sticky" hover effects after tapping
  • Each tap gets fresh feedback

DevTools Simulation:

  • Open DevTools → Device Emulation → Select mobile device
  • Test hover behavior — should be disabled
  • Switch to desktop → Hover should work
