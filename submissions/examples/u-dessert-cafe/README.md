# Dessert Cafe

A dreamy dessert-focused cafe landing page built entirely using **EaseMotion CSS utility classes**.

This example showcases a modern dessert brand experience featuring handcrafted sweets, seasonal specialties, workshops, reviews, and online ordering.

---

## Features

### Hero Section

* Dessert showcase imagery
* Brand introduction
* Call-to-action buttons
* Entrance animations

### Dessert Menu

Organized by category:

* Cakes
* Waffles
* Crepes
* Drinks

Interactive hover cards create an engaging browsing experience.

### Seasonal Specials

Highlights limited-time menu offerings including:

* Strawberry Cloud Cake
* Lavender Honey Crepe
* Summer Mango Waffle

### Dessert Workshops

Promotes hands-on dessert experiences including:

* Cake decorating
* Plating techniques
* Pastry making
* Chef-guided sessions

### Online Ordering CTA

Dedicated section encouraging customers to:

* Place online orders
* Explore featured desserts
* Discover delivery options

### Customer Reviews

Testimonials designed to build trust and social proof.

### Location & Hours

Cafe details including:

* Opening hours
* Address
* Availability information

### Footer

Branding and ownership information.
