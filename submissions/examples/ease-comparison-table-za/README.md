# Comparison Table

## What does this do?
CSS feature comparison tables with check/cross indicators, hover row highlight, and dark striped variants.

## How is it used?
```html
<table class="cmt-table"><thead><tr><th>Feature</th><th>Plan A</th></tr></thead><tbody><tr><td>Item</td><td><span class="cmt-check">✓</span></td></tr></tbody></table>
```

## Why is it useful?
Clean data presentation without JavaScript. Three table variants. Responsive horizontal scroll. Accessible semantic HTML. Ideal for pricing and feature comparisons.
