# Inline Alert

## What does this do?
Inline Alert component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"ai-primary\">Inline Alert Primary</div>
```

## Why is it useful?
Provides a reusable inline alert UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
