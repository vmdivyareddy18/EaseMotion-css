# Level Up Glow

## What does this do?

Creates a level-up animation with an expanding glowing ring and a brief scale pulse on the element.

## How is it used?

```html
<div class="level-up-container">
  <div class="glow-ring"></div>
  <div class="level-up-item">⭐</div>
</div>
```

## Why is it useful?

This animation highlights achievements, rewards, or special moments with smooth, lightweight motion that aligns with EaseMotion CSS's animation-first philosophy.
