# Opacity Utilities

1. What does this do? Adds utility classes to control the opacity/transparency of elements from 0 to 100 in granular increments.
2. How is it used? Apply classes like `.opacity-50` or `.opacity-10` to elements to control transparency.
3. Why is it useful? It allows developers to create sophisticated hover overlays, subtle state changes, card backgrounds, and layered visual depth.
