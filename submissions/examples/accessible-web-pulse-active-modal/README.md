# Accessible Web Pulse-Active Modal

A modern **Pure CSS Pulse-Active Modal** designed for **Accessible Web** interfaces. The component emphasizes accessibility, responsiveness, keyboard navigation, and customization through CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Pulse-Active animation
- High-contrast accessible design
- Responsive layout
- Keyboard accessible
- Visible focus indicators
- CSS custom properties
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:1s;
    --easing:ease-in-out;
    --scale:1.03;

    --primary:#005fcc;
    --bg:#ffffff;
    --surface:#f8fafc;
}
```

You can customize:

- Animation duration
- Animation easing
- Pulse scale
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
