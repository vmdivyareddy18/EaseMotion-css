# Interactive Kanban Board Component

A modern Kanban board dashboard component built using pure HTML and CSS.

## Features

- Multiple workflow columns

- To Do, In Progress, Review, Completed states

- Animated task cards

- Visual drag-state feedback

- Status indicators

- Hover interactions

- Responsive layout

- Dashboard-style design

- Smooth transitions and animations

## Files

- demo.html

- style.css

- Project Management Dashboards

- Task Tracking Systems

- Productivity Applications

- Workflow Management Tools

- Team Collaboration Platforms

- SaaS Admin Panels
