# Animation: Interactive Analytics Dashboard

A modern analytics dashboard featuring animated KPI cards, traffic charts, and activity streams. Inspired by modern SaaS platforms such as Vercel, Stripe, and Plausible.

## Features

- Pure HTML & CSS
- Animated chart bars
- Interactive KPI cards
- Activity feed reveal
- Responsive dashboard layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="stat-card">
    <span class="label">Visitors</span>
    <h2>128.4K</h2>
</div>
```

Hovering KPI cards elevates them while charts and activity feeds animate into view.

## Why is it useful?

Analytics dashboards are among the most common interfaces in modern web applications. This example demonstrates how motion can improve readability, hierarchy, and user engagement in data-heavy layouts.
