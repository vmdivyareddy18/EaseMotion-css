# User Activity Card

A modern **User Activity Card** component built using **HTML5** and **CSS3** for the EaseMotion CSS examples collection.

---

## ✨ Overview

The **User Activity Card** is a reusable dashboard component that displays recent user actions in a clean and visually appealing layout. It combines avatars, activity descriptions, timestamps, and status badges to present activity feeds commonly found in SaaS platforms, admin dashboards, project management tools, and social applications.

This example demonstrates how modern activity cards can be built using only HTML and CSS with smooth animations, responsive layouts, and accessibility-friendly design.

---

## 🚀 Features

- 👤 User avatar section
- 📝 Activity description
- 🕒 Timestamp display
- 🏷️ Animated status badges
