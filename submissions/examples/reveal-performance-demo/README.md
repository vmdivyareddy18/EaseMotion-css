# Reveal Performance Demo

1. What does this do?  
   This demo shows a lightweight reveal animation that fades cards into view as they enter the viewport.

2. How is it used?  
   Open the demo in a browser and scroll down. Each card appears progressively when its position becomes visible.

3. Why is it useful?  
   It demonstrates a responsive, accessible reveal pattern that respects reduced-motion preferences and keeps the experience smooth.
