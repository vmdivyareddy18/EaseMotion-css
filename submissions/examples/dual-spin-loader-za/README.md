# Dual-Spin Counter-Rotating Ring Loader

A CSS loading spinner with two concentric rings rotating in opposite directions. The outer ring spins clockwise, the inner ring spins counter-clockwise, creating a dynamic counter-rotation effect.

## Files

- `demo.html` - Dual-ring loader with status text
- `style.css` - CW and CCW rotation keyframes, nested ring positioning

## Usage

Open `demo.html` to see the dual-spin loader animation.
