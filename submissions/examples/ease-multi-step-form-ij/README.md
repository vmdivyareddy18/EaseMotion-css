# Multi-Step Form

An animated multi-step wizard form with progress indicator, slide transitions between steps, field validation, and a celebration screen on completion.

## Features

- Step-by-step form with animated slide transitions
- Progress indicator showing current/completed steps
- Input validation before advancing
- Review step before submission
- Completion celebration with pop animation

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--msf-duration` | `0.35s` | Transition duration |
| `--msf-primary` | `#6366f1` | Primary accent color |
| `--msf-step-color` | `#c7d2fe` | Inactive step circle color |
| `--msf-active-color` | `#6366f1` | Active step circle color |
| `--msf-radius` | `12px` | Container border radius |
| `--msf-bg` | `#ffffff` | Container background |

## Usage

Open `demo.html` in a browser. Fill in each step and click Next to advance.
