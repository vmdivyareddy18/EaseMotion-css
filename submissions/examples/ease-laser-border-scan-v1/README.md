# Ease Laser Border Scan

## Overview

Ease Laser Border Scan is a futuristic border animation that simulates a glowing laser continuously scanning around an element. Built entirely with HTML and CSS, it is lightweight, reusable, and suitable for modern interfaces.

---

## Features

- 🔥 Animated laser border
- ✨ Continuous scan effect
- 🎨 Three color variants
- 🚀 Smooth hover elevation
- 📱 Responsive demo
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Default green scanner
- Blue cyber variation
- Purple futuristic variation

---

## Usage

```html
<div class="laser-card">
    Content
</div>
```

---

## Customization

You can customize:

- Laser color
- Scan speed
- Card size
- Border radius
- Glow intensity

---

## Browser Support

Compatible with all modern browsers supporting CSS animations and transforms.
