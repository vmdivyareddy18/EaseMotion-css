# Ghost Button Text Visibility Fix

## What does this do?

This submission improves the readability of the **Ghost** button by ensuring the text remains visible during hover. It increases the contrast between the text and the button background while preserving the clean Ghost button style.

---

## How is it used?

Include the stylesheet after `easemotion.min.css`.

```html
<link rel="stylesheet" href="../../easemotion.min.css">
<link rel="stylesheet" href="style.css">
```

Use the Ghost button normally.

```html
<button class="ease-btn ease-btn-ghost">
    Ghost
</button>
```

---

## Why is it useful?

- Improves button readability.
- Enhances accessibility with better color contrast.
- Maintains the minimal Ghost button appearance.
- Demonstrates a reusable styling improvement that aligns with EaseMotion CSS.

---

## Issue Addressed

The Ghost button text became difficult to read because of insufficient contrast between the text and background. This submission demonstrates an accessible styling approach that keeps the text clearly visible.
