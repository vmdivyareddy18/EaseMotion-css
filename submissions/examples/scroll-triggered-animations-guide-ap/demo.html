Observer Integration Guidelines

      

        

          
IntersectionObserver API

          

            Rather than listening to heavy window.onscroll events, IntersectionObserver runs asynchronously.
            It optimizes rendering performance by checking intersection thresholds on a separate browser thread.
          

        


        

          
Threshold and rootMargin

          

            The threshold determines how much of the target must intersect before triggering.
            The rootMargin expands or contracts the intersection bounding box (e.g. rootMargin: '0px 0px -50px 0px' offsets the trigger line).
          

        


        

          
CSS Scroll Timelines Alternative

          

            Modern browsers support CSS-only scroll triggers via animation-timeline: view().
            While excellent, this approach does not support class addition patterns yet, and requires javascript fallbacks for legacy systems.
