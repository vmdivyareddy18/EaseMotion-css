# Glow Subscribe Form

**Issue:** #42531

## Overview

Glow Subscribe Form is a responsive, Pure CSS subscription form created for the EaseMotion CSS library. It features a retro-inspired glowing button animation using EaseMotion CSS conventions.

## Features

- Pure CSS implementation
- No JavaScript required
- Responsive design
- Retro glow animation
- Uses EaseMotion `ease-*` naming convention
- CSS variables for easy customization
- Keyboard accessible
- Mobile friendly
