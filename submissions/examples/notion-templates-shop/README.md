# Notion Templates Shop

A premium Notion Templates Shop landing page built to demonstrate EaseMotion CSS animations.

## Description

This project is a copy-paste ready, fully responsive landing page for a productivity templates e-commerce store. It showcases the power of the EaseMotion CSS library by implementing beautiful entrance animations and hover interactions without requiring any custom animation keyframes or modifying the core library.

## Features

- **Hero Section**: Productivity-focused headline, dashboard mockup, and floating statistics with entrance animations.
- **Category Cards**: Interactive grid for browsing templates by category.
- **Featured Templates**: Responsive grid of template cards featuring hover interactions.
- **Template Preview**: Clean before/after comparison with desktop and mobile mockups.
- **Creator Story**: Profile section highlighting creator statistics.
- **Reviews**: Testimonial cards with a clean, Notion-inspired aesthetic.
- **Bundle Deals**: Pricing cards for Starter, Creator, and Business bundles.
- **Fully Responsive**: Adapts seamlessly to desktop, tablet, and mobile layouts.
