# Glow Pill Navigation

## What it does

This component provides a modern pill-style navigation menu for SaaS dashboards. It includes horizontal pill buttons with a glowing active state, smooth hover lift animation, and accessible rounded styling.

## How to use it

1. Open `demo.html` directly in the browser.
2. Include `style.css` in the same directory.
3. Add the `pill-nav` container and `pill` elements to your layout.

Example:

```html
<link rel="stylesheet" href="style.css">
<nav class="pill-nav" aria-label="Primary navigation">
  <a href="#" class="pill active">Overview</a>
  <a href="#" class="pill">Analytics</a>
  <a href="#" class="pill">Messages</a>
  <a href="#" class="pill">Projects</a>
  <a href="#" class="pill">Settings</a>
</nav>
```

## Why it fits EaseMotion CSS

This submission is pure HTML and CSS without any JavaScript or external libraries. It delivers a self-contained, responsive navigation pattern with polished motion and a clean SaaS dashboard aesthetic, aligning with EaseMotion CSS principles for reusable UI components.
