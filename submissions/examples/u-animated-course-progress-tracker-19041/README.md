# Animated Course Progress Tracker

A modern learning dashboard that visualizes course progress, milestones, achievements, and learning streaks.

## Features

- Course progress cards
- Animated progress bars
- Learning milestone timeline
- Achievement badges
- Statistics dashboard
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Course names
- Progress percentages
- Achievement badges
- Learning milestones
- Colors and animations

## Use Cases

- LMS platforms
- Student dashboards
- Online learning portals
- Training systems
- Certification platforms
