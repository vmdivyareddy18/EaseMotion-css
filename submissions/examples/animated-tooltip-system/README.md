# Animated Tooltip System

A clean, modern, and fully responsive **Tooltip System** with smooth directional animations. This example demonstrates how to combine **EaseMotion CSS** with pure CSS techniques to create polished, accessible tooltips that appear from all four directions with different color variations.

## 🚀 Features

- **Four Directional Positions**: Tooltips can appear on top, bottom, left, or right of the trigger element.
- **Smooth Directional Animations**: Each tooltip slides in from its respective direction with a smooth fade and translate effect.
- **Arrow Pointers**: CSS-generated arrows point from the tooltip to the trigger element for clear visual association.
- **Five Color Variations**: Dark, light, success, warning, and error themes for different use cases.
- **Pure CSS Implementation**: No JavaScript required - uses `::before` and `::after` pseudo-elements with `attr()` for content.
- **Interactive Demo Sections**: Showcases tooltips on buttons, icon buttons, and form help icons.
- **Fully Responsive**: On mobile, left/right tooltips automatically reposition to bottom for better usability.
