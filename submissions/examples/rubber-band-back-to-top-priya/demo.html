EaseMotion CSS Layout Playground

Scroll down, then hover or focus (Tab key) the button in the bottom-right corner to see the rubber-band bounce. Click it to jump back to top.

01. Elastic Interactions

The button uses a pure-CSS squash-and-stretch keyframe inspired by classic rubber-band easing curves — no JavaScript involved.

02. Keyboard Accessible

Try pressing Tab to focus the button directly — it triggers the same bounce animation and shows a visible focus ring.

03. Reduced Motion Respecting

If your OS has "Reduce Motion" enabled, the bounce and smooth scroll are automatically disabled for accessibility.

04. Fully Responsive

The button scales down on small screens so it never overwhelms mobile layouts.
