# Document Review Version Comparison

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive document comparison component that displays two document versions side by side, making it easier to review revisions before approval. Suitable for document management systems, knowledge bases, CMS platforms, legal review tools, and collaborative editing applications.

## Features

- Side-by-side version comparison
- Previous and latest version badges
- Highlighted additions
- Summary statistics
- Responsive layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="comparison">

    <div class="version old">
        ...
    </div>

    <div class="version new">
        ...
    </div>

</div>
```
