# Infinite Seamless CSS Marquee

A continuous horizontal scrolling badge component demonstrating gapless marquee translations.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting horizontal marquees, badge cards, and details descriptions.
2. **`style.css`**: Width calculations, translate animations, mask gradient boundaries, and hover pauses.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the horizontal badges scrolling from right to left.
3. Verify that the loop restarts seamlessly without gaps or jumps.
4. Hover your mouse over the marquee — confirm that the scrolling animation pauses.
5. Move the cursor away — confirm that the scroll restarts.
