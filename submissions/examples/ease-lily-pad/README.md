# Ease Lily Pad Animation

A gentle, fluid CSS animation showcasing a lily pad drifting lazily over soft pond water ripples, employing counter-balanced sinusoidal transformations and a custom ambient timing structure.

## How to View
1. Open `demo.html` inside a standard browser view.
2. Interact with the "Disturb Pond" button layout structure to re-align the active visual states.

## Technical Details
- **Architecture:** Dual-layered keyframe translation vectors paired with an expanding radial ripple component.
- **Easing Profile:** Uses a fluid `cubic-bezier(0.45, 0, 0.55, 1)` equation to model soft liquid buoyancy behaviors.
