💡 How it works

When color-scheme: light dark (or dark / light) is declared on the :root, the browser engine (Chromium, WebKit, Gecko) does the following:

  • Sets the default page background and text color before stylesheets load, preventing initial-load white flashes.
  • Adapts system-rendered UI elements (like form buttons, select menus, input widgets, dialog boxes, and scrollbars) to match.
