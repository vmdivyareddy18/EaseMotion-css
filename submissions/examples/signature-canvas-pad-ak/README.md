# Signature Canvas Pad

## What does this do?
A canvas-based signature drawing pad that supports mouse and touch input with clear and save functionality.

## How is it used?
Open demo.html in a browser. Draw on the canvas with mouse or touch. Click Clear to reset or Save as PNG to download the signature.

## Why is this useful?
Demonstrates the HTML5 Canvas API for freehand drawing with touch device support and PNG export.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
