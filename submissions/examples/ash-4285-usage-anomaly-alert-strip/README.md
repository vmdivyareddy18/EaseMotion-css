# Usage Anomaly Alert Strip

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive alert strip that highlights abnormal API or application usage with key metrics and a clear call-to-action. Suitable for cloud dashboards, analytics platforms, and monitoring tools.

## Features

- Usage anomaly alert banner
- Current vs average usage metrics
- Responsive layout
- Animated entrance
- Glassmorphism styling
- CTA button
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="alert-strip">

    <div class="content">
        <h2>Usage Anomaly Detected</h2>
    </div>

    <div class="stats">
        ...
    </div>

</div>
```

## Main Classes

```css
.alert-strip
.icon
.content
.stats
.metric
.label
```
