# Portfolio Command Center

A premium developer portfolio dashboard demonstrating enterprise-grade layouts, analytics panels, timelines, contribution visualizations, and interactive UI patterns using pure CSS.

## Features

* Professional command center layout
* Achievement dashboard metrics
* Animated skill matrix
* Contribution activity heatmap
* Project analytics cards
* Career timeline
* Technology stack overview
* Glassmorphism-inspired styling
* Responsive design
* Reduced-motion accessibility support

## Why It Fits EaseMotion CSS

This example demonstrates how EaseMotion CSS can be used to create modern, enterprise-quality portfolio experiences using animation-first design principles and reusable CSS patterns.

The dashboard combines:

* Layout systems
* Motion design
* Data visualization
* Professional UI presentation

into a single showcase example.

## Usage

Open:

```text
demo.html
```

directly in a browser.

No build tools or server required.

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

When enabled:

* Animations are disabled
* Transitions are disabled
* Layout remains fully functional
