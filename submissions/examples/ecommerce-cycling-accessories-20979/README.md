# App Landing: Cycling Accessories E-commerce (`ecommerce-cycling-accessories-20979`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a high-performance, tech-focused cycling and sports brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A performance cycling brand requires a UI that feels fast, aerodynamic, and highly technical. This template achieves an aggressive aesthetic using high-contrast colors (black, neon yellow/green, bright blue), sharp angled/skewed elements, and fast, slanted typography (`Teko` paired with `Inter`).

The layout includes:
- A high-visibility, aggressive Announcement Bar.
- A minimalist, sticky Navigation with slide-right hover effects.
- A Hero section featuring skewed speed lines and a CSS-animated spinning bicycle wheel.
- A "Categories" grid with stark, blocky cards and floating CSS product mockups.
- A "Fit Guide" technical section featuring a CSS radar animation simulating a 3D body scan.
- A "Latest Drops" product grid featuring a smooth slide-up hover reveal for quick actions.
- A "Ride Stories" blog section with overlay play buttons.
- A high-contrast Reviews layout.
- An angled, high-impact Newsletter signup.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use extremely fast, snappy `cubic-bezier(0.7, 0, 0.3, 1)` and `cubic-bezier(0.19, 1, 0.22, 1)` timings to convey speed and performance:
- `.ease-slide-left` / `.ease-fade-right` (Configured for sharp, fast entrances that complement the slanted typography).
- `.ease-reveal-spin` (A custom fast entrance animation for the hero wheel).
- `.ease-spin-fast` (An aggressive, fast continuous rotation for the CSS wheel).
- `.ease-spin-radar` / `.ease-pulse-neon` (Continuous animations driving the "Fit Guide" tech visualization).
- `.ease-hover-slide-right` (A fast, snappy hover interaction for links to simulate forward momentum).
- `.ease-hover-zoom-btn` (A custom interaction that scales buttons while maintaining their aggressive skew).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the slanted layout architecture, the high-contrast aesthetic, and the complex CSS wheel and radar simulations. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the rapid, snappy entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
