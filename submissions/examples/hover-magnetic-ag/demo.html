How it Works

This technique splits the button area into four invisible quadrant triggers absolute-positioned over and beyond the button bounds. When the cursor enters a quadrant, CSS :has() overrides custom properties to pull the button in that direction. A smooth cubic-bezier transition generates the physics-like bounce.

Snippet

<button class="magnetic-btn btn-primary">
  <!-- Hover Triggers -->
  <span class="magnetic-trigger tl"></span>
  <span class="magnetic-trigger tr"></span>
  <span class="magnetic-trigger bl"></span>
  <span class="magnetic-trigger br"></span>
  
  <!-- Visual Content -->
  <span class="btn-content">
    Explore Now
  </span>
</button>
