# ease-drag-reorder Component

This submission fixes Issue #39533 by adding a reusable `.ease-drag-reorder` list component.  
It allows users to drag and reorder list items with smooth CSS transitions.

## Usage

```html
<ul id="ease-list">
  <li draggable="true" class="ease-drag-item">Task A</li>
  <li draggable="true" class="ease-drag-item">Task B</li>
</ul>
