# Ease API Rate Limit Monitor

## Overview
A CSS-only component for monitoring API rate limit usage.

## Features
- Animated usage bars
- Responsive layout
- CSS variables
- Pure HTML & CSS
- Lightweight design

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for API dashboards, developer portals, and backend monitoring tools.
