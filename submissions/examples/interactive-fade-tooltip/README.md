# Interactive Fade-In Tooltip

A responsive **Pure CSS Fade-In Tooltip** designed for interactive user interfaces.

## Features

- Pure CSS
- Fade-In Animation
- Responsive Design
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript Required

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` directly in any modern browser.

## CSS Variables

```css
--duration
--ease
--tooltip-bg
--tooltip-color
--radius
--button-bg
```

These variables allow customization of animation timing, colors, and border radius.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## EaseMotion CSS

This component follows the EaseMotion CSS philosophy by providing a lightweight, reusable, animation-first tooltip implemented entirely with CSS.
