# Nth Last Child Count

## What does this do?
Demonstrates `:nth-last-child` for counting from the end of a list — styling the last N items, reverse pattern matching, and range selection from the tail.

## How is it used?
Apply to elements where you want to target items based on their position from the end:

    <li class="rev-item">Last items styled differently</li>

## Why is it useful?
`:nth-last-child` is ideal for "recent items" widgets, tail-end pagination, last-row visual adjustments, or any layout where the last few children need special treatment without knowing total count.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
