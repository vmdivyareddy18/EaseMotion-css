# Ease Infinity Loop Animation

## Overview

Ease Infinity Loop Animation creates a continuously drawing infinity symbol using SVG stroke animations and glowing effects.

---

## Features

- ♾️ Animated SVG infinity symbol
- ✨ Glowing stroke effect
- 🔄 Infinite drawing animation
- 🎯 Pure HTML & CSS
- 📱 Responsive demo

---

## Demo

Includes:

- Animated infinity loop
- Glow pulse
- Continuous stroke drawing

---

## Usage

```html
<svg class="infinity">
...
</svg>
```

---

## Customization

You can customize:

- Stroke color
- Stroke width
- Drawing speed
- Glow intensity
- SVG size

---

## Browser Support

Works in modern browsers supporting SVG and CSS animations.
