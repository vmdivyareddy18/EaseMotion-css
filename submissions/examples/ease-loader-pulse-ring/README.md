# ease-loader-pulse-ring

A radar/sonar-style pulsing ring loader that expands and fades infinitely.

## Usage

```html
<div class="ease-loader-pulse-ring"></div>
```

## CSS Variables

| Variable | Default | Description |
|---|---|---|
| `--ease-loader-size` | `48px` | Size of the loader |
| `--ease-loader-color` | `#6366f1` | Color of the rings |
| `--ease-loader-speed` | `1.5s` | Animation duration |
