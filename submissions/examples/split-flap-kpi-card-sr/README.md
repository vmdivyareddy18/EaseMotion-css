# Split-Flap KPI Dashboard Card

A responsive Split-Flap KPI Dashboard Card built with pure HTML and CSS that recreates a mechanical split-flap display for analytics dashboards, finance interfaces, admin panels, and operational monitoring screens.

---

## Features

- 🎯 Pure HTML & CSS (No JavaScript)
- 🎞️ Mechanical split-flap number transition
- 📊 Positive, negative, and neutral KPI states
- 🔄 Animated top and bottom flap sections
- 📈 Current metric with comparison text
- 📱 Fully responsive layout
- ♿ Keyboard-accessible focus states
- 🌙 Supports `prefers-reduced-motion`
- 🎨 Easy customization using CSS variables
- 🚫 No external libraries or frameworks
