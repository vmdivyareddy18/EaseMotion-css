# scroll-driven-parallax-tm

## What does this do?
A self-contained parallax scrolling example using CSS scroll-driven animations with scroll-timeline and view-timeline — no JavaScript needed.

## How is it used?
<div class="parallax-container">
  <div class="parallax-hero">Hero Section</div>
  <div class="parallax-section">Animated Section</div>
</div>

## Why is it useful?
Demonstrates modern CSS scroll-driven animations using scroll-timeline and view() for parallax effects. Fits EaseMotion's animation-first philosophy by replacing JavaScript scroll listeners with pure CSS declarations.
