# Glassmorphism UI 3D Perspective Tilt Accordion

A modern **Pure CSS 3D Perspective Tilt Accordion** designed for **Glassmorphism UI** layouts. The component features frosted-glass styling, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- 3D Perspective Tilt hover animation
- Glassmorphism (frosted glass) design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Soft gradient background
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.5s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --primary:#60a5fa;
    --secondary:#8b5cf6;
    --accent:#22d3ee;

    --glass-bg:rgba(255,255,255,.18);
    --glass-border:rgba(255,255,255,.35);
}
```

You can customize:

- Animation duration
- Animation easing
- Glass opacity
- Blur amount
- Accent colors
- Border transparency

Open `demo.html` directly in any modern browser.
