# ease-hover-skew

## What does this do?
The ease-hover-skew utility applies skew transforms on hover for dynamic interactive effects.

## How is it used?
    <div class="ease-hover-skew">Content</div>

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html in your browser.

## Contribution Notes
- Class naming follows the ease-* convention
- Uses CSS custom properties for theming
