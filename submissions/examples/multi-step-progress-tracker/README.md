# Multi-Step Progress Tracker

A modern responsive multi-step progress tracker built using pure HTML and CSS.

## Features

- Multiple progress steps
- Active step highlighting
- Completed step indicators
- Animated progress bar
- Smooth hover effects
- Responsive design
- Pure HTML and CSS
- Modern UI styling

## Folder Structure

```text
multi-step-progress-tracker/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in a browser.
2. Ensure `style.css` is in the same directory.
3. Customize colors, labels, and progress states.

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Responsive Design

## States

### Completed Step
Displays a success checkmark and completed styling.

### Active Step
Highlights the current step with scaling and pulse animation.

### Pending Step
Shows upcoming steps with default styling.
