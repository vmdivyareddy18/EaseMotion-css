# Sports Club Landing Page

## What does this do?

This is a complete, ready-to-copy Sports Organization / Club landing page built entirely using EaseMotion CSS components, utility classes, and transitions.

## How is it used?

Link the `easemotion.css` (or `easemotion.min.css`) inside your project's `<head>` and use the pre-configured layout and animation classes:

```html
<!-- Link to EaseMotion CSS bundle -->
<link rel="stylesheet" href="easemotion.css" />
<!-- Page-scoped custom design classes -->
<link rel="stylesheet" href="style.css" />
<!-- Reveal logic for scroll entrance animations -->
<script src="core/reveal.js" defer></script>
```

## Why is it useful?

This landing page provides an interactive, bold sports organization template (such as for football, basketball, or soccer teams) that developers can copy and customize. It showcases how to combine EaseMotion's variables, components (sticky navbar, cards, badges), utility classes (grids, flex centering, sr-only helpers), scroll-entrance animations, and interactive hover transitions to create a premium, fan-engagement focused user interface without writing complex custom animations or relying on bloated JavaScript frameworks.

## Showcase of EaseMotion CSS Classes

This template showcases the following EaseMotion features:

- **Components**:
  - Sticky glass navbar: `.ease-navbar-glass` with `.ease-navbar-glass-sticky` and `.ease-navbar-glass-blur`
  - Cards: `.ease-card`, `.ease-card-hover`, `.ease-card-glass`
  - Badges: `.ease-badge`, `.ease-badge-primary`
  - Buttons: `.ease-btn`, `.ease-btn-primary`, `.ease-btn-outline`, `.ease-btn-ghost`
- **Animations & Interactive states**:
  - Combined Pulse + Glow Hover: `.ease-hover-pulse-glow`
  - Grow on hover: `.ease-hover-grow`
  - Lift + Shadow on hover: `.ease-hover-lift`
  - Shimmer text: `.ease-shimmer-text`
  - Floating element keyframes: `.ease-float`
  - Text underline hover: `.ease-hover-underline`
  - Scroll entrance reveal classes: `.ease-reveal`, `.ease-reveal-up`, `.ease-reveal-slide-left`
  - Stagger delay helpers: `.ease-reveal-delay-1`, `.ease-reveal-delay-2`, `.ease-reveal-delay-3`, `.ease-reveal-delay-4`
- **Utilities**:
  - Flexbox and Alignment: `.ease-flex`, `.ease-items-center`, `.ease-justify-center`, `.ease-center`, `.ease-flex-wrap`
  - Grid layouts: `.ease-grid`, `.ease-grid-cols-2`, `.ease-grid-cols-3`, `.ease-gap-8`
  - Screen reader helper: `.ease-sr-only`
