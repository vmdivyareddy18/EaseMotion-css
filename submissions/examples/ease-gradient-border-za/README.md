# Gradient Border Component

## What does this do?
CSS gradient border techniques: mask-composite, pseudo-element overlay, and animated conic borders with color schemes.

## How is it used?
```html
<div class="gbd-card gbd-mask">Content</div>
<div class="gbd-card gbd-spin">Animated border</div>
```

## Why is it useful?
Three distinct gradient border techniques. Animated spinning and shimmer variants. Color scheme classes. Useful for cards and featured UI elements.
