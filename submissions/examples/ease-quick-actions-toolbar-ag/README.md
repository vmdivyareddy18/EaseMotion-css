# Quick Actions Toolbar

A floating quick actions toolbar designed to consolidate common actions into a sleek, space-saving interface. The toolbar expands dynamically upon hover or focus, triggering a staggered reveal of the sub-actions using cubic-bezier timing functions.

## Features
- **Hover/Focus Expand**: Hidden actions slide out in a smooth, springy transition when the primary button is hovered or focused.
- **staggered Animations**: Sub-action items enter with progressive CSS animation delays, creating a sophisticated motion sequence.
- **Micro-interactions**: Hovering over each individual actionbutton scales it slightly and applies a localized glow effect.
- **Pure CSS**: Fully responsive and animated using pure CSS transitions and keyframes, with no external JS dependencies.
- **EaseMotion Integration**: Fully styled using EaseMotion custom properties and typography variables.


## Files
- `demo.htm``: Live interactive demonstration of the floating quick-action toolbar.
- `style.css`: Styles for the wrapper container, expand/collapse transition, staggered entry, and glowing hover states.
