# Scroll Animations

## What does this do?
CSS scroll-driven animation components with fade/scale/slide/rotate reveals, scroll progress bar, clip-path section reveals, stagger children, and parallax effects.

## How is it used?
```html
<div class="sa-card sa-fade-up">Content</div>
<div class="sa-progress-bar"></div>
```

## Why is it useful?
Adds scroll-based entrance animations without JavaScript. 12 animation variants organized in families (fade, scale, slide, rotate). Scroll-driven progress bar. Staggered child animations. Customizable via CSS variables.
