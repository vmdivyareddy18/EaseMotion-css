# 👤 CSS Avatar Motion

A collection of beautiful, pure CSS animated profile avatars for modern web applications.

## 🚀 Features

### Avatar Styles (8 types)
1. **Pulse Ring** - Expanding ring animation
2. **Status Indicator** - Online/Offline/Busy dots
3. **Gradient Border** - Rotating conic gradient
4. **Hover Zoom** - Image zoom with overlay
5. **Stacked Group** - Overlapping avatar stack
6. **Badge Notification** - Bouncing notification badge
7. **Neon Glow** - Cyberpunk glow effect
8. **Flip Card** - 3D flip to show initials

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All avatar animations
- `readme.md` - Documentation

## 🛠️ How to Use

### Basic Avatar
```html
<div class="avatar pulse-ring">
    <img src="user.jpg" alt="User">
</div>
