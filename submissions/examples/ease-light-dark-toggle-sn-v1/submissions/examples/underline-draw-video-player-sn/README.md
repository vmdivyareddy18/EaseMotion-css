# Underline Draw Video Player

## What does this do?

This submission adds a responsive video player card featuring an animated underline draw effect on the video title. The component highlights content with smooth hover animations while maintaining a clean and modern layout.

## How is it used?

Wrap your video thumbnail and title inside the provided component structure and apply the corresponding CSS classes.

```html
<div class="video-card">
  <div class="video-thumbnail">
    <img src="thumbnail.jpg" alt="Video thumbnail">
    <button class="play-btn">▶</button>
  </div>

  <h3 class="video-title">
    Animated Video Player
    <span class="underline"></span>
  </h3>
</div>
```

## Why is it useful?

It provides an attractive, reusable video player component that improves visual feedback with a subtle underline animation. The component is responsive, accessible, and suitable for portfolios, media galleries, blogs, and documentation sites.
