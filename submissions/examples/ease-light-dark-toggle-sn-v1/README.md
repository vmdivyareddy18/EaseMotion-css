# ease-light-dark-toggle

## What does this do?

This submission adds a modern, animated light/dark theme toggle component with a sliding thumb, theme icons, and smooth transitions. It provides a clean, accessible switch that can be integrated into documentation sites, dashboards, portfolios, or web applications.

## How is it used?

Wrap a checkbox inside the `.ease-light-dark-toggle` component and style it using the accompanying CSS.

```html
<label class="ease-light-dark-toggle">
  <input type="checkbox">

  <span class="toggle-track">
    <span class="icon sun">☀️</span>
    <span class="toggle-thumb"></span>
    <span class="icon moon">🌙</span>
  </span>
</label>
```

## Why is it useful?

A light/dark theme toggle is a common UI component in modern websites. This submission demonstrates a clean, responsive, and framework-independent implementation that aligns with EaseMotion CSS's focus on reusable, animation-friendly, and developer-friendly components.
