# Serenity Yoga – Yoga Teacher Landing Page

A complete, ready‑to‑copy landing page for an independent yoga instructor built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Serene Hero** – calming yoga imagery with a class CTA
- **Class Types** – vinyasa, yin, meditation, pranayama
- **Teacher Training Program** – 200‑hour certified course
- **Schedule & Booking** – weekly class timetable
- **Retreat Announcements** – upcoming Himalayan and Goa retreats
- **Client Testimonials** – student experiences
- **Newsletter Signup** – email capture form
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-emerald-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`, `ease-input`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-emerald-700`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-full`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`, `ease-overflow-x-auto`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
