# City Skyline Loader

City skyline loader with sequential building reveals. Buildings rise from the bottom in sequence with a glowing sky gradient background, creating a loading animation effect.

## Features

- Sequential building reveal animations
- Gradient sky background with subtle pulse
- Customizable building color, sky color, and load duration
- CSS @keyframes for building rise and sky glow
- Repeatable loading loop

## Usage

Set CSS vars on `.sl-skyline`:
- `--building-color`: building fill (default `#1e293b`)
- `--sky-color`: sky gradient start (default `#0f172a`)
- `--load-duration`: full cycle duration (default `2.5s`)
