# ease-dashboard-donut-chart

Animated dashboard donut chart using pure CSS.

## Features

- CSS-only donut chart
- conic-gradient based fill
- Animated percentage fill
- Uses @property for smooth animation
- Center label support
- Responsive design

## Acceptance Criteria Covered

### Animated Donut Fill

Uses:

- conic-gradient
- @property
- animated percentage variable

### Center Label

Displays:

- Percentage value
- Status text

### No External Libraries

Built entirely using HTML and CSS.

## Use Cases

- Analytics dashboards
- KPI widgets
- Revenue tracking
- Progress visualization
