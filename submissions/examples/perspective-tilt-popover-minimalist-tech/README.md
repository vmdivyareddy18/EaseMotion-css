# 3D Perspective Tilt Popover – Minimalist Tech

## Description
A pure CSS popover featuring a smooth 3D perspective tilt animation designed for minimalist technology interfaces.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible using `:focus-within`
- Easy customization with CSS variables

## Usage

```html
<div class="popover">
  <button class="trigger">Hover Me</button>

  <div class="content">
    <h3>Minimal Tech</h3>
    <p>Popover Content</p>
  </div>
</div>
