# Battery Status Widget

## What does this do?
Displays real-time battery information using the Battery API: charge level, charging status, and estimated time remaining, with a visual battery icon that changes color based on level.

## How is it used?
Open demo.html in a supported browser. The widget updates automatically when battery status changes. Shows level, charging state, and remaining time.

## Why is this useful?
Demonstrates the Battery Status API with a polished visual indicator, color-coded states, and graceful fallback for unsupported browsers.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript (Battery API)

## Preview
Open demo.html directly in your browser.
