# Card Warning BG

1. What does this do? Adds a soft yellow/warning background tint to a card, complementing the existing success and danger card backgrounds.

2. How is it used? Apply `.card-warning-bg-zt` alongside your card class.

3. Why is it useful? It completes the semantic card color set (success, danger, warning) for alert-style cards, following the same pattern as `ease-card-success-bg` and `ease-card-danger-bg`.
