Appearance None

CSS appearance reset proposed for core/utilities.css.

The .appearance-none utility applies appearance: none; to UI elements like checkboxes, radios, and select menus. This strips away the browser's default styling, giving you a completely blank canvas to apply custom designs without fighting the user agent stylesheet.

Select Menu

Removing the native dropdown arrow to allow for custom SVG arrows or styling.

Checkbox

Removing the native checkmark box to create completely custom toggle switches.
