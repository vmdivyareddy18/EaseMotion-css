# Onboarding Step Indicator

A step progress indicator for onboarding flows.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--step-bg` | `#2d2d44` | Inactive step color |
| `--active-color` | `#4ecdc4` | Active step color |
| `--completed-color` | `#4ecdc4` | Completed step color |
| `--transition-speed` | `0.3s` | Animation speed |
