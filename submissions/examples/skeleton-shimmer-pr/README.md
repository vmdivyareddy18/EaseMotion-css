# Pure CSS Fluid Animated Skeleton Shimmer Loaders

### What does this do?
Introduces a modular collection of hardware-accelerated animated skeletal layout components (avatars, header titles, multi-line blocks, image cards) that pulse with an infinite translation shimmer matrix.

### How is it used?
Compose structural components natively by combining structural dimensions with the core utility modifier class `.skeleton-shimmer`.

### Why is it useful?
It creates high-performance async loading feedback indicators that reduce layout shift artifacts (CLS) without loading runtime scripts or rendering external tracking containers.
