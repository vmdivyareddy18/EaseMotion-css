# Fire Flicker Animation

## Overview
A pure CSS animation that mimics the organic, irregular flicker of a real flame. By combining dynamic opacity shifts, subtle scale transformations, and warm, glowing radial box-shadows, this effect brings a realistic fire element to the web without needing JavaScript or Canvas.

## Features
* **Organic Flickering:** Uses irregular keyframe percentages to simulate natural fire movements.
* **Dynamic Sizing:** Combines scaling with opacity to create depth and movement.
* **Warm Glow Effect:** Utilizes layered `box-shadow` properties to cast an ambient orange/red glow.
* **Infinite Loop:** Seamless `ease-in-out` animation cycle.
* **Pure CSS:** No JavaScript required, ensuring high performance.
* **Plug and Play:** Easy to implement with a single HTML class.
