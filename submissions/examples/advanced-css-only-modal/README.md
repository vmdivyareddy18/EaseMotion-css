# Advanced CSS-only Modal Component

A CSS-only modal component with animated overlay, centered dialog panel, close button, responsive layout, focus-visible styling, and reduced-motion support.

The modal can be opened and closed without JavaScript by using a hidden checkbox toggle. It includes a smooth fade/scale animation, backdrop overlay, keyboard-focusable controls, and a mobile-friendly layout.

This is an advanced component because it combines state handling, layered UI, accessibility-focused markup, transitions, responsive behavior, and interaction states using only HTML and CSS.

## Why is this useful for EaseMotion CSS?

Modals are a common UI pattern used for confirmations, alerts, onboarding flows, forms, previews, and focused content. A CSS-only modal fits EaseMotion CSS perfectly because it is animation-first, composable, human-readable, and does not require JavaScript or external dependencies.
