# Support Ticket Priority Stack

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive support ticket dashboard component that organizes tickets by priority using color-coded visual indicators for help desk and customer support interfaces.

## Features

- Priority-based ticket stack
- Critical, high, medium, and low priority levels
- Ticket status badges
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="ticket critical">

    <div class="priority">
        Critical
    </div>

    <div class="details">
        ...
    </div>

</div>
```

## Main Classes

```css
.ticket-stack
.ticket
.priority
.details
.status
.critical
.high
.medium
.low
```
