# Memory Card Game

## What does this do?
A memory card matching game with 8 pairs of emoji cards, 3D flip animation, moves counter, and win detection.

## How is it used?
Open demo.html in a browser. Click cards to flip them and find matching pairs. Match all 8 pairs to win. The moves counter tracks your attempts.

## Why is this useful?
This provides a complete memory game component demonstrating CSS 3D transforms for card flip animation, game state management, and match logic.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to play.
