# Countdown Timer (Issue #18263)

A countdown timer displaying MM:SS with animated number transitions. Each digit slides in from below when it changes. Configurable via CSS custom properties.

## Usage

Open `demo.html` in a browser. Click **Start** to begin the countdown, **Reset** to restart.

## Files

- `demo.html` — HTML5 page with countdown and JS logic
- `style.css` — Dark theme styles with `@keyframes slideInFromBottom`
- `README.md` — This file
