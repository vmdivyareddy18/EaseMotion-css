# ease-aspect-video

A 16:9 aspect ratio utility for video embeds and image containers.

## Usage

Include the `style.css` in your project and apply the class:

```html
<div class="ease-aspect-video">Video or image content</div>
```

## CSS Reference

```css
.ease-aspect-video {
  aspect-ratio: 16 / 9;
}
```

## Demo

Open `demo.html` in a browser to see the component in action.
