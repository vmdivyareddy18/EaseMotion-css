# Size Selector

## What does this do?
An animated size/variant selector for product pages, showing available options with a smooth active state transition.

## How is it used?
Apply the `.size-selector` class to a container with `.size-option` children. The active option animates with a border and background highlight.

## Why is this useful?
Provides clear visual feedback for size/variant selection in e-commerce interfaces, improving the shopping experience with minimal markup.
