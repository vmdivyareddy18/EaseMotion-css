# Particle Modal

A modern corporate-inspired modal with animated floating particle effects built entirely using HTML and CSS.

## Features

- ✨ Floating particle animation
- 💼 Corporate UI inspired
- 🚫 No JavaScript required
- 📱 Responsive layout
- ♿ Accessible structure using semantic HTML
- 🎯 Pure CSS modal using `:target`

## File Structure

```
particle-modal-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in a browser.

Click **Open Modal** to display the modal with animated particle effects.

## Use Cases

- Corporate Dashboards
- Landing Pages
- Product Announcements
- Confirmation Dialogs
- Modern Web Applications
