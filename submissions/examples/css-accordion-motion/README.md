# 🎹 CSS Accordion Motion

A collection of beautiful, pure CSS animated accordions using modern HTML5 `<details>` and `<summary>` elements.

## 🚀 Features
3 unique accordion styles:
1. **Classic Accordion** - Clean, minimal design with subtle animations
2. **Gradient Accordion** - Vibrant gradient backgrounds with smooth transitions
3. **Neon Accordion** - Glowing neon effects with cyberpunk vibes

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All animations (Pure CSS!)
- `readme.md` - Documentation

## 🛠️ How to Use
1. Open `demo.html` in any browser
2. Click on any accordion item to expand/collapse
3. Copy the CSS classes for your own projects

## 🧠 Key Techniques
- `<details>` and `<summary>` HTML5 elements
- `details[open]` CSS selector for state changes
- Smooth `max-height` transitions
- Icon rotation animations
- Hover and open state effects

## 💡 Why Pure CSS?
- No JavaScript dependencies
- Faster load times
- Better accessibility (native HTML elements)
- Works even if JS is disabled
