# Cyber Security Dashboard

A responsive **Cyber Security Dashboard** built using **HTML5** and **CSS3** that demonstrates threat monitoring, firewall protection, network security, and enterprise security analytics through a modern dashboard interface.

---

## Features

- Responsive dashboard layout
- Security overview cards
- Threat monitoring progress
- Network status monitoring
- Risk analysis panel
- Security activity feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Threats Detected
- Secure Devices
- Security Alerts
- Firewall Status

### Security Widgets

- Threat Timeline
- Network Status
- Risk Analysis

### Activity

- Security Logs
- Notifications Panel

### Quick Actions

- Scan Network
- Block Threat
- Export Logs

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
