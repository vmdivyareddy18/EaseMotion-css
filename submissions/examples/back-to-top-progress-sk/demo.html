Core Architecture

Engineered to deliver high visual fidelity and robust performance across all modern viewports.

Lightweight & Fast

Utilizes vanilla JavaScript with passive scroll listeners. Minimal layout shifting, optimized repaint cycle, and no bloated runtime libraries.

Themeable Architecture

Built with native CSS variables for seamless transitions between Light and Dark mode. Highly customizable palette to fit any existing branding.

Fully Accessible (A11y)

Includes precise ARIA attributes, semantic markup, focus state rings, keyboard triggers, and support for the prefers-reduced-motion media query.

Seamless Integration

Incorporate both elements into your document with minimal HTML wrapper tags. Our stylesheet automatically styles, animates, and positions the elements.

  • Fixed positions with Z-Index ordering
  • Dynamic height percentage tracking
  • Spring-back scale fade button animation

Accessibility-First Design

We believe motion should be inclusive. This component features advanced accommodations to ensure it functions perfectly for everyone:

  • Keyboard navigable via standard Tab rings
  • Automatic disabling of transitions under system Reduced Motion
  • Dynamic ARIA progress attribute values
