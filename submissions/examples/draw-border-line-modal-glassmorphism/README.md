# CSS Draw-Border Line Glassmorphism Modal

A modern **Glassmorphism Modal** featuring a sequential draw-border animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Glassmorphism UI
- Draw Border Line animation
- Frosted glass effect
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No JavaScript animation

## Files

```
demo.html
style.css
README.md
```

## Customization

Modify the CSS variables in `:root`.

```css
--primary
--glass
--border
--blur
--radius
--duration
```
