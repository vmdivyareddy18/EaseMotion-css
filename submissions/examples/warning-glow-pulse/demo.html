Warning Glow Pulse

A continuous amber / orange box-shadow pulse that signals a warning state on any element — cards, buttons, badges, inputs and icons. Controlled by --ease-warning-glow-color.

Each swatch overrides --ease-warning-glow-color locally. The keyframe reuses the variable, so no extra CSS is needed.

// How to use
<!-- 1. Link the stylesheet --> <link rel="stylesheet" href="style.css" /> <!-- 2. Add the class to any element --> <div class="warn-card warning-glow-pulse">…</div> /* 3. Override the color token when needed */ .my-element { --ease-warning-glow-color: rgba(249, 115, 22, 0.65); /* orange */ }

♿ Respects prefers-reduced-motion: animation is suppressed and replaced with a static amber ring so the warning remains visually distinct.
