# Pulse Voice Recorder

A lightweight animated voice recorder component built with pure HTML and CSS for EaseMotion CSS.

## Features

- Pure CSS animation
- No JavaScript required
- Smooth pulse recording effect
- Responsive layout
- Easy to customize

## Usage

Add this HTML:

```html
<div class="voice-recorder">
  <div class="pulse-ring"></div>
  <div class="mic">🎙️</div>
</div>
