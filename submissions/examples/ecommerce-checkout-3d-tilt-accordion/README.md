# E-Commerce Checkout 3D Perspective Tilt Accordion

A modern **Pure CSS 3D Perspective Tilt Accordion** designed for **E-Commerce Checkout** layouts. The component features smooth 3D tilt interactions, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- 3D Perspective Tilt hover animation
- E-Commerce Checkout inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.45s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --primary:#16a34a;
    --secondary:#22c55e;

    --radius:16px;
}
```

You can customize:

- Animation duration
- Animation easing
- Primary & secondary colors
- Border radius
- Shadow intensity
- Border color

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
