# 🧮 Glassmorphism Calculator UI

A modern **Glassmorphism-styled Calculator UI** built using **HTML, CSS, vanilla JavaScript, and EaseMotion CSS utilities**.

This project is a UI showcase that demonstrates how EaseMotion CSS can be used to build **interactive, animated, and visually appealing interfaces** with minimal effort.

---

## ✨ Features

- 🪟 Glassmorphism design with blur and transparency
- 🎞️ Smooth EaseMotion animations (`ease-fade-in`, `ease-slide-up`)
- 🎯 Hover interactions using `ease-hover-grow`
- 🧮 Fully functional calculator (basic arithmetic)
- ⌨️ Keyboard support (numbers, operators, Enter, Backspace, Esc)
- 📱 Fully responsive layout (mobile-friendly)
- 🎨 Modern UI with gradient background
