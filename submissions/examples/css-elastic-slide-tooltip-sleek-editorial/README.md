# CSS Elastic Slide Tooltip - Sleek Editorial Layouts

A pure CSS animated tooltip component inspired by sleek editorial and publishing interfaces.

## Features

- Pure CSS animation
- Elastic slide transition effect
- Editorial style design
- Responsive layout
- Keyboard accessible
- CSS custom property controls


## Usage

Open `demo.html` directly in a browser.

Hover or focus on buttons to display the animated tooltip.


## Customization

Update animation properties:

```css
:root {
    --tooltip-time: 0.45s;
    --tooltip-scale: 1;
    --tooltip-easing: cubic-bezier(.68,-0.55,.27,1.55);
}
