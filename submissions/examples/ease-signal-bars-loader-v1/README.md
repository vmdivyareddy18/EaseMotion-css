# Ease Signal Bars Loader

## Overview

Ease Signal Bars Loader is a lightweight loading animation inspired by network signal indicators. The bars animate sequentially to create a dynamic visual effect using only HTML and CSS.

---

## Features

- 📶 Animated signal bars
- ✨ Glowing effect
- 🔄 Sequential motion
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Five animated bars
- Sequential wave animation
- Infinite loading effect

---

## Usage

```html
<div class="loader">
    <div class="bar"></div>
</div>
```

---

## Customization

- Number of bars
- Animation speed
- Glow color
- Bar width
- Maximum height

---

## Browser Support

Compatible with modern browsers supporting CSS animations and transforms.

---

## Why This Fits EaseMotion CSS

Suitable for loading screens, dashboards, connectivity indicators, admin panels, and modern web applications. It provides a reusable animation utility with a clean and lightweight implementation.
