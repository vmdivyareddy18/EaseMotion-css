# Border Draw Hover Animation

**What does this do?**  
Draws a dynamic border around an element (buttons or cards) on hover using multiple style presets like clockwise, corner, or center-out drawing.

**How is it used?**  
```html
<!-- Clockwise sequential border drawing -->
<button class="draw-btn draw-clockwise">Clockwise Draw</button>

<!-- Opposite corners drawing simultaneously -->
<button class="draw-btn draw-corners">Corner Draw</button>

<!-- Edge expansion from the center outwards -->
<button class="draw-btn draw-center-out">Center-Out Draw</button>
```

**Why is it useful?**  
It provides premium, interactive visual feedback on hover without affecting the layout or causing reflow, as it leverages absolute-positioned pseudo-elements. It fits the EaseMotion CSS philosophy perfectly by being human-readable, self-contained, and highly visual.
