# ease-piggy-bank-coin-drop

Animated savings deposit effect where a coin drops into a piggy bank.

## Features

- Coin falls using translateY()
- Coin shrinks as it enters the piggy bank
- Piggy bank bounce/squash feedback
- Infinite looping animation
- Pure CSS implementation
- Suitable for finance and savings applications

## Usage

```html
<div class="coin">🪙</div>
<div class="piggy-bank">🐷</div>
```

## Animation Details

- Coin drops vertically using translateY()
- Coin scales down before disappearing
- Piggy bank performs a small squash-and-bounce reaction
