# Liquid Morphing Blob

## Description
A pure CSS liquid morphing blob animation. It uses `@keyframes` to continuously morph the `border-radius` of a gradient-filled div while simultaneously rotating and scaling it. This creates a smooth, organic, lava-lamp-like motion effect.

## Files
- `demo.html`: Contains the HTML structure for the blob container.
- `style.css`: Contains the gradient, box-shadow, and morphing/rotation animations.

## How to use
1. Open `demo.html` in your browser to see the effect.
2. Copy the HTML and CSS into your project.
3. Apply the `.liquid-blob-qn` class to a `div` element.
4. Customize the gradient colors (`#ff0076` and `#590fb7`) and the `box-shadow` in `style.css` to fit your design theme.
5. Adjust the animation duration in the `animation` property to make it faster or slower.
