# Weather Dashboard Widget

An animated weather widget showcasing multiple weather conditions.

## Features

- Sunny mode
- Rainy mode
- Cloudy mode
- Snowy mode
- Animated sun rotation
- Floating clouds
- Responsive layout
- Pure HTML, CSS and JavaScript

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in your browser and switch between weather conditions using the buttons.
