# Spring Toast

A Gaming-inspired Spring Toast component built with pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Spring entrance animation
- Gaming-themed notification design
- Responsive layout
- Accessible HTML
- No JavaScript required
