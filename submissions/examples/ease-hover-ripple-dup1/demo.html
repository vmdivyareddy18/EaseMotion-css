ease-hover-ripple

Move your cursor over any element below — a circular ripple expands outward from the exact point your pointer enters or moves to, fading as it grows.

CSS Variables

Large Surface

Move your cursor around this entire box — the ripple follows your exact position in real time.

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Add the class to any element */
<div class="ease-hover-ripple">Hover me</div>

/* 3. Include the tiny cursor-tracking script (see below) */
<script src="ripple.js"></script>

/* 4. Customise color or speed per-instance */
.my-card {
  --ease-ripple-color: #ff4d8d;
  --ease-ripple-duration: 0.8s;
}
