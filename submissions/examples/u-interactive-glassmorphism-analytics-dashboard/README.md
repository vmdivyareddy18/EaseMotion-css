# Interactive Glassmorphism Analytics Dashboard

Modern analytics dashboard built using glassmorphism design principles.

## Features

- Glassmorphism cards
- Sidebar navigation
- Analytics widgets
- Animated chart section
- Responsive table
- Hover interactions
- Responsive layout
- Pure HTML & CSS

## Components

### Sidebar
Navigation menu with active state.

### Analytics Cards
- Revenue
- Visitors
- Sales
- Growth

### Charts
Animated bar chart and activity section.

### Activity Table
Recent activity records with responsive layout.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
