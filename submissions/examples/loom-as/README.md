# Loom

A pure CSS and HTML floor loom weaving cloth: the shuttle flies across the warp, the reed beats the weft home and the finished fabric builds up on the cloth beam.

## What it shows

- Warp threads from a vertical repeating linear gradient, in two shades for the raised and lowered sheds
- A boat shuttle on a clip-path hull that flies across, laying a weft thread that grows in behind it
- The reed beating down to pack each pass, timed to the shuttle and weft cycle
- Woven cloth from two crossed repeating linear gradients blended into a plaid
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
