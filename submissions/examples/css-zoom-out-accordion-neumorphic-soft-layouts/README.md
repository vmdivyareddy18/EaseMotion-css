# CSS Zoom-Out Accordion - Neumorphic Soft Layouts

## Overview

A pure CSS animated accordion featuring a smooth Zoom-Out interaction inspired by modern neumorphic soft UI designs.

The component provides elegant expand/collapse motion without JavaScript.

## Features

- Pure HTML and CSS implementation
- Smooth zoom-out reveal animation
- Neumorphic soft shadows
- Fully responsive layout
- Keyboard accessible using native details/summary
- Customizable CSS variables

## Usage

```html
<div class="accordion">

  <details>
    <summary>Accordion Title</summary>

    <div class="content">
      <p>Your content goes here.</p>
    </div>

  </details>

</div>
