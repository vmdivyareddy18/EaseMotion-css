# Firefly Countdown Timer — Nature Design Pattern

A luminous, nature-inspired **Firefly Countdown Timer** component built with pure HTML and CSS for the EaseMotion CSS library.

## Features

- **Pure CSS Animation**: No JavaScript — the timer digits animate using CSS `@keyframes` and scroll-linked column animations.
- **Firefly Ambiance**: Floating firefly particles drift in the background using staggered `ease-kf-float` keyframes for an organic, living feel.
- **Pause / Resume Control**: A hidden checkbox state-control powers a fully accessible pause/play toggle button without any JavaScript.
- **Luxury Nature Aesthetic**: Deep forest dark theme with amber and emerald accent glow, premium Inter typography, and subtle card glow.
- **EaseMotion Tokens**: Uses `--ease-font-sans`, `--ease-radius-lg`, `--ease-space-*`, and `--ease-color-success` from the EaseMotion design system.
- **Responsive & Accessible**: Mobile-first layout, `prefers-reduced-motion` support, and semantic HTML structure.
