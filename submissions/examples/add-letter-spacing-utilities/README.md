# Letter Spacing Utilities

Utility classes for controlling the spacing between characters in text.

## Available Classes

| Class | CSS |
|---------|---------|
| `.tracking-tight` | `letter-spacing: -0.05em;` |
| `.tracking-normal` | `letter-spacing: 0;` |
| `.tracking-wide` | `letter-spacing: 0.05em;` |
| `.tracking-wider` | `letter-spacing: 0.1em;` |
| `.tracking-widest` | `letter-spacing: 0.2em;` |

## Usage

```html
<p class="tracking-tight">Text</p>
<p class="tracking-normal">Text</p>
<p class="tracking-wide">Text</p>
<p class="tracking-wider">Text</p>
<p class="tracking-widest">Text</p>
```

## Why Use It?

Letter spacing utilities help improve typography, readability, branding, headings, and UI consistency with simple, reusable classes.
