# CSS Micro Shake Tooltip

Pure CSS animated tooltip utilizing a Micro Shake interaction designed for Glassmorphism UI layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Glassmorphism styling
- Micro Shake animation
- CSS custom properties
- Soft blur effects
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--tooltip
--duration
--ease
```
