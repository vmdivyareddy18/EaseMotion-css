# 🧺 Catch the Falling Items

A classic arcade game where you catch falling items with a basket. Catch good items for points, avoid bombs, and survive as the speed increases!

## 🎮 How to Play

1. Click **"Start Game"** to begin
2. **Move** your basket left and right using your mouse or arrow keys
3. **Catch** good items (🍎🍋🍊🍉🍓⭐) for points
4. **Avoid** bombs (💣☠️💀) — they cost you a life!
5. **Level up** as your score increases
6. **Survive** as long as possible!

## 🖱️ Controls

| Action | Input |
|--------|-------|
| Move Basket | Mouse movement or `←` `→` arrow keys |
| Start Game | Click Start or press `Space` |
| Reset Game | Press `R` |

## ✨ Features

- 🎯 **6 Good Items** – Different point values (1-5 points)
- 💣 **3 Bad Items** – Bombs, skulls, and more
- 📈 **Progressive Difficulty** – Speed increases with level
- ❤️ **Lives System** – 3 lives to start
- 🏆 **Best Score** – Saved in browser local storage
- 🎨 **Visual Feedback** – Effects for catches and misses
- 📱 **Responsive Design** – Works on desktop and mobile
- 🖱️ **Touch Support** – Play on mobile devices
