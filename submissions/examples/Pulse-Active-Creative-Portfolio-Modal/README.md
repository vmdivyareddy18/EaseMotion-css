# Pulse Active Creative Portfolio Modal

A responsive creative portfolio modal featuring a smooth Pulse-Active interaction built entirely using HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Pulse-Active button animation
- Creative portfolio layout
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--scale
--primary
--secondary
--radius
```
