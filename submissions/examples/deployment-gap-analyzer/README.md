# Deployment Gap Analyzer

Deployment Gap Analyzer is a responsive release-audit panel for spotting readiness gaps before go-live. It separates passing checks, risky items, and blockers in a compact operational layout.

## Features

- Release gap summary
- Pass, risk, and block states
- Compact readiness cards
- Responsive mobile layout

## Files

- `demo.html` contains the deployment gap analyzer markup.
- `style.css` contains the card layout and status styles.
