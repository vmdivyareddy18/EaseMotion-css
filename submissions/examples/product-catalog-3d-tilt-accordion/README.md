# Product Catalog 3D Perspective Tilt Accordion

A responsive Pure CSS accordion designed for product catalog interfaces.

## Features

- Pure HTML & CSS
- 3D Perspective Tilt Animation
- Responsive Design
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any browser.

No build tools required.
