# Animated 3D Tilt Card Component

A modern CSS-only 3D tilt card component featuring perspective effects, depth shadows, smooth rotation animations, glowing highlights, and interactive hover transitions.

---

## Features

- Realistic 3D tilt effect
- CSS perspective and transform-style support
- Smooth hover rotation animation
- Floating card movement
- Glowing background highlights
- Depth shadows
- Responsive design
- Lightweight HTML & CSS implementation
- No JavaScript required
