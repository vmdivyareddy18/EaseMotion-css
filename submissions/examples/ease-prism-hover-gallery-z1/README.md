# Ease Prism Hover Gallery

## Overview

A premium hover gallery featuring colorful prism-inspired cards with subtle rotation and glassmorphism styling using only HTML and CSS.

## Features

- Prism gradient cards
- Hover lift animation
- Rotating glow effect
- Glassmorphism design
- Responsive grid
- Pure HTML & CSS

## Folder Structure

```text
ease-prism-hover-gallery-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Adapts smoothly to desktop, tablet, and mobile layouts.

## Why it fits EaseMotion CSS

Demonstrates reusable hover animations, modern gradients, and lightweight styling that can be integrated into portfolios, dashboards, and creative interfaces.
