# Education: Online Medical Education (`education-online-medical-21042`)

This proposal provides a complete, copy-paste ready landing page template designed for a professional Healthcare/Medical Education SaaS product, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

An Online Medical Education brand (like CME providers) requires a UI that feels highly authoritative, trustworthy, clinical, and professional. This template achieves this aesthetic using a clean light-mode architecture with a clinical Navy Blue primary color, Teal accents for action, traditional serif typography (`Merriweather`) for headings to denote authority, clean sans-serif (`Open Sans`) for legibility, and steady, reliable animations.

The layout includes:
- A clean Navigation with a CSS medical cross logo.
- A Hero section featuring a CSS-animated clinical dashboard mockup simulating course progress and module unlocking, flanked by floating achievement badges.
- A trusted accreditation logos banner.
- A Course Catalog grid with filtering tabs and medical-themed abstract CSS icons (brain, lungs, pill).
- A Faculty section highlighting expert instructors.
- A dark-mode Testimonials section with serif quote styling.
- A Pricing tier grid contrasting a-la-carte vs unlimited subscriptions.
- A clean, authoritative Footer.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use steady, reliable `cubic-bezier(0.4, 0.0, 0.2, 1)` timings tailored for a professional medical feel (avoiding overly bouncy or erratic motions):
- `.ease-slide-up-stagger` (Configured for a steady, controlled vertical entrance).
- `.ease-zoom-in-up` (Applied to the medical dashboard mockup for a smooth 3D entrance).
- `.ease-grow-width` (Used in the mockup to simulate the CME course progress bar filling up).
- `.ease-float` (Applied to the floating achievement badges in the hero).
- `.ease-pulse-slow` (A gentle pulse on the video play button).
- `.ease-hover-lift-card` (A subtle, professional hover elevation for course and faculty cards).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the clean layout architecture, the clinical color palette, and the complex abstract UI elements (dashboard mockup, medical icons). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the precise entrance animations. No JavaScript is required for this template.
