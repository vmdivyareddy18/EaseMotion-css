# CSS Draw-Border Line Modal

A responsive modal featuring a **Draw-Border Line animation** created entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Draw Border Line animation
- Interactive Interface styling
- Fully Responsive
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript animation

## Files

```

README.md
```

## CSS Variables

Customize the component by editing the variables inside `:root`.

```css
:root{

--primary:#2563eb;
--duration:.6s;
--radius:18px;

}
```

You can customize:

- Primary color
- Animation duration
- Border radius
- Shadow
- Background color
