# Personal Finance Tracker Dashboard

A responsive Personal Finance Tracker Dashboard built using HTML and CSS.

## Overview

This example demonstrates how EaseMotion CSS can be used to create modern financial dashboards with responsive layouts, animated cards, progress indicators, and transaction tracking sections.

## Features

- Responsive dashboard layout
- Financial summary cards
- Expense category tracking
- Savings goal progress bars
- Recent transactions section
- Financial insights panel
- Hover animations
- Fade-in effects
- Mobile-friendly design

## Dashboard Sections

### Financial Summary
- Total Income
- Total Expenses
- Total Savings
- Monthly Budget

### Expense Categories
- Food
- Shopping
- Transport
- Bills

### Savings Goals
- Emergency Fund
- Laptop Fund
- Travel Fund

### Recent Transactions
- Income entries
- Expense entries

### Insights
- Monthly overview
- Savings recommendations
- Budget tips
