Accordion

FAQ stack

How does this accordion open?

The browser manages the expanded state through the semantic details element, while CSS handles the motion.

Does it need JavaScript?

No. The interaction is native, keyboard accessible, and works when scripts are blocked.

Where can it be used?

It works well for FAQs, release notes, settings panels, and compact documentation sections.
