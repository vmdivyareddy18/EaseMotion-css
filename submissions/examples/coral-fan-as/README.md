# Coral Fan

Two pure CSS and HTML gorgonian sea fans swaying in the current above a rocky reef.

## What it shows

- The whole branching mesh drawn by two gradients: a repeating conic gradient for the radiating branches and a repeating radial gradient for the cross links
- mask-image cutting the gradient mesh down to a fan silhouette
- A combined rotate and skew sway so the fan bends with the current instead of tipping rigidly
- Light shafts, drifting fish and rising bubbles for depth
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
