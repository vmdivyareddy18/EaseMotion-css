# Circular Progress Ring (Issue #18256)

An SVG circle that animates from empty to filled using `stroke-dasharray`/`stroke-dashoffset`. Configurable color, size, and percentage via CSS custom properties.

## Usage

Open `demo.html` in a browser. The ring fills to 75% on load.

## Files

- `demo.html` — HTML5 page with SVG progress ring
- `style.css` — Dark theme styles with `@keyframes fillRing`
