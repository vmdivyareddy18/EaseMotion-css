# Release Risk Assessment Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive release risk assessment component that helps engineering teams evaluate deployment readiness through categorized risk cards. Suitable for DevOps dashboards, CI/CD pipelines, release management tools, QA portals, and software delivery platforms.

## Features

- Low, Medium, and High risk cards
- Release readiness summary
- Status badges
- Responsive layout
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="risk low">

    <div class="header">

        <h3>Code Quality</h3>

        <span class="badge success">
            Low
        </span>

    </div>

</div>
```

## Main Classes

```css
.cards
.risk
.header
.badge

```
