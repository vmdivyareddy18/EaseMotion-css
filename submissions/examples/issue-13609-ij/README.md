# ease-backdrop-blur-md

A medium backdrop blur utility (12px) for glassmorphism effects.

## Usage

Include the `style.css` in your project and apply the class:

```html
<div class="ease-backdrop-blur-md">Frosted glass content</div>
```

## CSS Reference

```css
.ease-backdrop-blur-md {
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
}
```

## Demo

Open `demo.html` in a browser to see the component in action.
