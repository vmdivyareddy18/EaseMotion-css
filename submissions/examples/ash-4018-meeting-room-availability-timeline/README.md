# Meeting Room Availability Timeline

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive meeting room scheduling component that displays room availability across different time slots using a visual timeline.

## Features

- Meeting room availability timeline
- Booked, available, and maintenance slots
- Status badge
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="slot booked">

    <span>09:00</span>

    <div class="bar"></div>

    <small>Booked</small>

</div>
```

## Main Classes

```css
.timeline-card
.room-header
.status
.timeline
.slot
.booked
.available-slot
.maintenance
.bar
```
