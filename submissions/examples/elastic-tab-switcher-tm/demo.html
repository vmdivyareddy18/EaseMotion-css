Elastic Tab Switcher

Spring-animated tab switching with accessible ARIA patterns

Default Style

Welcome to the Overview panel. This tab switcher uses aria-selected and aria-controls for full keyboard and screen reader support. The elastic spring animation on the indicator is driven by --ease-ease-bounce.

The Analytics panel shows usage metrics. Switch tabs using arrow keys (← →) when focused on the tab bar for full keyboard navigation following WAI-ARIA best practices.

Settings are fully configurable. All transitions respect prefers-reduced-motion and dark mode via CSS custom properties from the core/variables.css design token system.

Pill / Filled Variant

The filled pill variant works great for primary navigation. The selected tab floats up with a spring animation and gains a subtle shadow.

Explore shows trending content and recommendations based on your preferences. Updated in real time with WebSocket connections.
