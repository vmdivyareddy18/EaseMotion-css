# CSS Skew-Active Modal – Dashboard Analytics Layout

## Overview

A responsive Dashboard Analytics modal built with pure HTML and CSS. The modal uses a smooth skew-active opening animation and modern glassmorphism styling while remaining fully responsive and keyboard accessible.

## Features

- Pure HTML & CSS
- Skew-active opening animation
- Dashboard analytics layout
- Glassmorphism design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Reduced motion support

## CSS Custom Properties

The following variables can be customized:

- `--primary`
- `--secondary`
- `--duration`
- `--easing`
- `--modal-scale`
- `--modal-skew`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **View Dashboard** to open the modal and **×** to close it.

Customize the CSS variables in `style.css` to adjust colors, animation timing, skew angle, scaling, and overall appearance.
