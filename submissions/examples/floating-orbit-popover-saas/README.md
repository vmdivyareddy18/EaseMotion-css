# Floating Orbit Popover SaaS

A pure CSS animated Floating Orbit Popover component designed for SaaS showcase layouts.

## Features

- Pure CSS animation
- Floating orbit interaction
- No JavaScript required
- Responsive design
- Keyboard accessible
- CSS custom properties support

## Usage

```html
<div class="orbit-wrapper">

<button class="orbit-trigger">
View Product
</button>

<div class="orbit-popover">
Content
</div>

</div>
