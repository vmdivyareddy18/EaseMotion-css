# Comprehensive Exit Animations Suite

A modular collection of 6 hardware-accelerated layout exit utility variants matching corresponding entrance animations.

## Included Keyframe Trajectories
- `ease-fade-out`: Interpolates alpha metrics downwards linearly.
- `ease-slide-down`: Handles vertical translation exit patterns.
- `ease-slide-out-left` / `ease-slide-out-right`: Executes single-axis horizontal coordinate shifts.
- `ease-zoom-out`: Decrements bounding matrices to 0.92 cleanly.
- `ease-fade-out-scale`: Modern outward pop transition scaling tracking profiles.
