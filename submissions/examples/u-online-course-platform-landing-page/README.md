# Online Course Platform Landing Page

A complete Online Course Platform Landing Page built for EaseMotion CSS.

## Overview

This example showcases a modern EdTech SaaS landing page inspired by popular online learning platforms such as Udemy, Coursera, and Skillshare. It is designed to promote courses, instructors, student success stories, and subscription plans while encouraging newsletter signups.

## Sections Included

### Hero Section
- Platform introduction
- Course preview cards
- Primary call-to-action button

### Featured Courses
- Course cards
- Ratings display
- Pricing information

### Categories Navigation
- Programming
- Data Science
- Design
- Marketing
- Business

### Instructor Spotlight
- Featured instructor profile
- Professional bio section

### Student Testimonials
- Success stories
- Learning outcomes
- Student feedback

### Subscription Plans
- Basic plan
- Pro plan
- Enterprise plan

### Newsletter Signup
- Email subscription form
- Course updates CTA
