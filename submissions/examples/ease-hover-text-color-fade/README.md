# Text Color Fade

Text color fades to a new color on hover — pure CSS.

## Features
- Smooth color transition on hover
- Pure CSS, no JavaScript

## Usage
```css
.fade-text { color: #9ca3af; transition: color 0.4s ease; }
.fade-text:hover { color: #f472b6; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
