# ease-drawer-left

An animated left-side drawer panel that slides in from the left edge. Ideal for mobile navigation menus.

## Usage

Include the `style.css` in your project and use the HTML structure with `.drawer`, `.drawer-overlay`, and the toggle button. The demo uses JavaScript for toggling; a CSS-only version can use the checkbox technique.

## Demo

Open `demo.html` in a browser to see the component in action.
