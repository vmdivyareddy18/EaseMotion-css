# Pulse Theme Picker

A pure HTML and CSS animated Theme Picker component built for EaseMotion CSS.

## Features

- Smooth pulse animation
- Multiple theme options
- Pure CSS implementation
- No JavaScript dependency
- Responsive layout
- Modern dashboard-style design

## Usage

Add the component HTML:

```html
<div class="theme-picker">
  <button class="theme-card light">
    ☀️
    <span>Light</span>
  </button>
</div>
