# Animated Event Schedule Planner

A modern event schedule component featuring animated timeline cards, session details, speaker information, and responsive layouts.

## Features

- Timeline-based schedule
- Animated reveal effects
- Live and upcoming badges
- Speaker information
- Hover interactions
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Event sessions
- Speaker names
- Timeline colors
- Badge styles
- Animation timing

## Use Cases

- Conferences
- Hackathons
- Webinars
- Workshops
- Community events
- Tech meetups
