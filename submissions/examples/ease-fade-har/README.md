# Ease Fade Animation (har)

### 1. What does this do?
This adds a smooth fade-in and slide-up entrance animation for elements.

### 2. How is it used?
Simply add the class `ease-fade-har` to any HTML element you want to animate on load:
```html
<div class="ease-fade-har">Content</div>
```

### 3. Why is it useful?
It provides a clean, modern entrance effect that fits seamlessly with EaseMotion CSS's dynamic and aesthetic animation principles, perfect for cards or hero sections.
