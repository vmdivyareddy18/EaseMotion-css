# CSS Swing-Hover Modal for Neumorphic Soft Layouts

## Overview

This project demonstrates a responsive neumorphic modal featuring a smooth **Swing-Hover** animation built entirely with HTML and CSS. The modal showcases a soft user interface with interactive preference toggles, rounded elements, and subtle shadows that create a modern neumorphic design.

---

## Features

- Pure HTML5 and CSS3 implementation
- Smooth Swing-Hover entrance animation
- Soft neumorphic interface
- Responsive modal layout
- CSS-only toggle switches
- Keyboard-accessible controls
- Reusable and customizable component
- CSS custom properties for easy styling
- No JavaScript or external libraries required

## Preview

The modal includes:

- User Preferences heading

### Animation

- The background overlay fades in smoothly.
- The modal enters with a gentle **Swing-In** animation.
- Hovering over the modal triggers a subtle **Swing-Hover** effect, creating an engaging interactive experience.

## Customization

The component uses CSS custom properties for easy customization.

```css
:root{
    --primary:#4f7cff;
    --radius:22px;
    --duration:.8s;
    --bg:#eef2f7;
}
```

You can customize:

- Background colors
- Modal size
- Border radius
- Animation duration
- Animation easing
- Button colors
- Shadow intensity
- Typography
- Toggle switch appearance
