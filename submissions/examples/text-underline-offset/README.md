# text-underline-offset

## Overview

Demonstrates the CSS `text-underline-offset` property for precise control over underline positioning relative to text. Includes combinations with `text-decoration-color` and `text-decoration-thickness` to show how offsets interact with other text-decoration properties. Negative offset values and per-element comparisons are also covered.

## Usage

Open `demo.html` in any modern browser. The page displays multiple underline offset examples grouped by value range and styling combinations. Hover over any example to highlight its border. The design uses a dark theme with `#0a0f1e` background and `#e2e8f0` text.

## Features

- Six offset values from auto to 15px plus negative offset
- Combined styling with color, thickness, and line style
- Side-by-side comparison of three offset values on identical text
- Responsive layout with mobile breakpoint at 600px
- `prefers-reduced-motion` media query disables animations and transitions
- Dark theme throughout with carefully chosen accent colors
