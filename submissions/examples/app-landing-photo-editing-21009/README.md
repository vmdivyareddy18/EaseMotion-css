# App Landing: Photo Editing App (`app-landing-photo-editing-21009`)

This proposal provides a complete, copy-paste ready landing page template designed for a next-gen, AI-driven mobile Photo Editing App, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A modern, AI-powered photo editing app requires a highly creative, visual-forward, and premium aesthetic. This template achieves a sleek "Dark Mode/Neon" look using Deep Space Blacks (`#05050A`), Electric Purple (`#7000FF`), and Hot Pink/Magenta (`#FF0055`) gradients. It utilizes glassmorphism effects (`backdrop-filter: blur()`) to create depth and a modern geometric typography pairing (`Outfit` for display, `Inter` for body).

The layout includes:
- A glassmorphic Navigation bar.
- A Hero section featuring abstract floating neon orbs, a CSS-animated "phone mockup" simulating a before/after photo slider (using `clip-path` animation), and floating glass badges.
- A Features grid with glowing cards and pulsing CSS icons.
- A Masonry-style "Sample Edits" abstract image gallery with hover overlays.
- A single, high-impact Testimonial slider card.
- A Pricing tier grid highlighting a glowing "Pro" option.
- A visually striking CTA section for app downloads.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use extremely smooth, cinematic `cubic-bezier(0.25, 0.1, 0.25, 1)` and elastic `cubic-bezier(0.34, 1.56, 0.64, 1)` timings tailored for a premium, high-tech feel:
- `.ease-slide-up-stagger` (Configured for a smooth, premium vertical entrance).
- `.ease-pop-in` (For a snappy, elastic reveal on gallery items).
- `.ease-float-smooth` (Applied to the phone mockup for a premium floating effect).
- `.ease-pulse-slow` (Applied to the background gradient orbs for a breathing neon effect).
- `.ease-hover-scale-glow` (A high-impact hover interaction for primary buttons that adds a neon shadow).
- `.ease-hover-glow-card` (Applied to feature/pricing cards to add a glassmorphic glow on hover).
- `.ease-slide-horizontal` (A custom continuous CSS animation simulating the before/after slider handle moving back and forth).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the dark neon color palette to the glassmorphic panels and the abstract CSS illustrations (phone notch, slider UI, abstract landscape photos). The standard `ease-*` interaction classes are attached directly to HTML elements to provide precise, cinematic animations. No JavaScript is required for this template.
