# CSS Horizontal Roll Modal for Marketing Landing Pages

A responsive marketing modal built using pure HTML and CSS featuring a smooth horizontal roll animation.

## Features

- Pure HTML & CSS
- Responsive design
- Horizontal roll animation
- Landing page hero section
- Modern CTA button
- No JavaScript required

## Files

```
css-horizontal-roll-modal-marketing/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser and click **Get Started** to view the modal animation.
