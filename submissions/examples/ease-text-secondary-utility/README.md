# ease-text-secondary and ease-bg-secondary Utilities

Secondary color utility classes for EaseMotion CSS.

## Usage

```html
<span class="ease-text-secondary">Secondary colored text</span>
<div class="ease-bg-secondary">Secondary colored background</div>
```

## Classes

| Class                 | CSS Property                                  |
| --------------------- | --------------------------------------------- |
| `ease-text-secondary` | color: var(--ease-color-secondary)            |
| `ease-bg-secondary`   | background-color: var(--ease-color-secondary) |
