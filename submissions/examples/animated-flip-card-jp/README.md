# Animated Flip Card Component

## Description

A CSS-only 3D flip card component with smooth rotation animations. The front side displays basic information while the back side reveals additional content on hover.

---

## Features

- 3D flip animation
- Smooth hover transitions
- Floating animated icons
- Modern rounded card design
- Responsive layout
- CSS-only implementation
- Accessibility support using reduced motion

---

## Classes

- `.flip-container` – Card wrapper
- `.flip-card` – Individual card with perspective
- `.flip-card-inner` – Rotating card element
- `.flip-card-front` – Front side content
- `.flip-card-back` – Back side content
- `.icon` – Animated icon

---

## Accessibility

Supports `prefers-reduced-motion` to reduce animations for users who prefer less motion.

---

## Use Cases

- Service cards
- Product information
- Team member cards
- Portfolio sections
