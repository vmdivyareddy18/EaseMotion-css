# Two Factor Recovery Code Panel

## What does this do?

Shows recovery code rows with unused, copied, used, and warning states.

## How is it used?

```html
<div class="code-row is-used">
  <code>A9K2-LM8Q</code>
  <span>Used</span>
</div>
```

## Why is it useful?

Security settings need clear backup-code status feedback so users know which codes remain available.
