# Vertical Stepper Indicator

A vertical step progress indicator with animated transitions — pure CSS.

## Features
- Connecting line via `::before` pseudo-element
- Completed step with filled circle + checkmark
- Active step with pulse ring animation
- Pure CSS, no JavaScript

## Usage
```html
<div class="stepper">
  <div class="step completed"><div class="step-circle">✓</div><div class="step-content">...</div></div>
  <div class="step active"><div class="step-circle">2</div><div class="step-content">...</div></div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
