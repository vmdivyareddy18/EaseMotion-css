interpolate-size: allow-keywords

Smooth expand/collapse with auto-height transitions.

interpolate-size: allow-keywords is a CSS property that enables transitions on keyword-based sizes like auto, min-content, max-content, and fit-content.

Before this property, you could not animate to/from auto height. Developers had to use max-height hacks, JS-based height calculations, or grid tricks. Now you get native, performant animations.

Supported in Chrome 129+, Edge 129+, and Safari 18+. Check caniuse for updates.
