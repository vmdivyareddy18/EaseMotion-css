# Tech — Open Source Library Landing Page

## What does this do?
A complete, copy-paste ready full-page website for an Open Source State Management Library named `FastState`, featuring store creation code snippets, developer feature grids (bundle size, fine-grained reactivity, DevTools), quick start setup code examples, framework compatibility badges, GitHub stats count-ups, sponsor grids, and documentation links.

## How is it used?
Open `demo.html` directly in any web browser to view the page. No build step is required. The page transition, reveal animations, and hover components use the project's standard CSS classes.

## Why is it useful?
This showcases how EaseMotion CSS can be applied to design modern, dark-slate developer-centric open-source library pages with clean code highlight containers, compatibility badges, and download stats.

## EaseMotion CSS Classes Showcased
- **Entrance & Hover Animations**:
  - `.ease-slide-in-left` / `.ease-slide-in-right`: Used for the hero content entrances.
  - `.ease-hover-grow`: Applied to header logo elements and button states.
  - `.ease-hover-lift`: Used for framework compatibility badges and stats cards.
  - `.ease-hover-underline`: Applied to nav menu links.
  - `.ease-card-lift`: Used for the library feature cards.
- **Scroll Reveal Animations**:
  - `.ease-reveal`: Applied to sections to hide elements offscreen.
  - `.ease-reveal-up` / `.ease-reveal-scale`: Sliding directional reveals.
  - `.ease-reveal-delay-1` / `.ease-reveal-delay-2` / `.ease-reveal-delay-3`: Staggered timing delays for grid columns.
