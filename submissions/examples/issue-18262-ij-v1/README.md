# Circular Dot Spinner (Issue #18262)

8 dots arranged in a circle that fade and scale with staggered animation delays. Configurable dot color and size via CSS custom properties. Pure CSS.

## Usage

Open `demo.html` in a browser. The dots animate continuously.

## Files

- `demo.html` — HTML5 page with dot spinner
- `style.css` — Dark theme styles with `@keyframes dotFade`
- `README.md` — This file
