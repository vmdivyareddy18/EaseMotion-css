# Animated Glassmorphism Dashboard Card Component

A modern CSS-only dashboard card component built using the glassmorphism design style with transparent backgrounds, blur effects, glowing borders, and smooth hover animations.

---

## Features

- Glassmorphism transparent card design
- Backdrop blur effect
- Animated glowing border
- Smooth floating hover animation
- Soft shadows and depth effects
- Dashboard statistics layout
- Responsive grid design
- Pure HTML and CSS implementation

---

## Included Examples

- 👥 Total Users Statistics
- 💰 Revenue Analytics
- 📊 Performance Analytics
- 🚀 Project Tracking

---

## Files Included

- `demo.html` – Demonstrates the glass dashboard cards.
- `style.css` – Contains glass effects, animations, and responsive styles.
