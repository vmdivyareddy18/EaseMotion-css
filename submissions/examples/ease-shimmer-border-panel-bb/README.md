# Continuous Gradient Shimmer Border

A premium card panel featuring a continuous gradient shimmer tracing the inner border boundary.

## What does this do?
It builds cards where a conic-gradient overlay continuously rotates in the background, showing a glowing trace along the boundary outline.

## How is it used?
Configure standard elements with the shimmer panel wrappers:

```html
<div class="shimmer-panel" tabindex="0">
  <span class="shimmer-border"></span>
  <div class="panel-content">
    <!-- Panel Details -->
  </div>
</div>
```

## Why is it useful?
It provides premium visual feedback highlights for cards, utilizing standard CSS conic gradients.
