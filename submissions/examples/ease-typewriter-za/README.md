# CSS Typewriter Effect

## What does this do?
CSS-only typewriter typing animation using steps() timing function, overflow hidden, and border-right cursor blink.

## How is it used?
```html
<p class="tw-line tw-type-basic">Welcome to EaseMotion CSS.</p>
```

## Why is it useful?
Animated typing effect without JavaScript. Multiple lines with staggered delays. Blinking cursor. Variable speed. Looping and erase variants. Monospace font optimized.
