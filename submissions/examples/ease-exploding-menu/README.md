# Exploding Menu – Menu items explode outward

> A production-ready exploding menu component where menu items burst outward from a central trigger with smooth animations, multiple layouts, and customizable settings. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Exploding animation** – Items burst outward from center
- ✅ **3 layouts** – Circle, Grid, Spiral
- ✅ **Customizable** – Distance, speed, color
- ✅ **Auto-close** – Close after item selection
- ✅ **Keyboard accessible** – Enter/Space (toggle), Escape (close)
- ✅ **8 menu items** – Fully customizable
- ✅ **Keyboard shortcuts** – Enter/Space (toggle), Escape (close)
- ✅ **4 color variants** – Ocean, Sunset, Gold, Dream
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
