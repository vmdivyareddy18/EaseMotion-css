# Ease Hologram Card

## Overview

Ease Hologram Card is a futuristic UI animation built using pure HTML and CSS. It combines scan lines, glowing borders, glassmorphism, and hover elevation to create a premium holographic card effect.

---

## Features

- ✨ Animated scan line
- 💡 Glow border effect
- 🎨 Glassmorphism design
- 🚀 Hover lift animation
- 🎯 Multiple color variants
- ⚡ Pure HTML & CSS

---

## Included Variants

- Default
- Cyber Blue
- Neon Purple

---

## Usage

```html
<div class="ease-hologram-card">
    Content
</div>
```

---

## Browser Support

Works in all modern browsers supporting CSS animations, transforms, and backdrop-filter.

---

## Customization

You can easily customize:

- Glow color
- Scan animation speed
- Card size
- Border radius
- Background opacity
