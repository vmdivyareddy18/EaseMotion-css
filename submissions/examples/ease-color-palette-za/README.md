# Color Palette Grid

## What does this do?
CSS color palette swatch grids using aspect-ratio for square swatches with organized primary, neutral, warm, and dark color collections.

## How is it used?
```html
<div class="clp-grid"><div class="clp-swatch clp-bg-indigo"><span class="clp-name">Indigo</span><span class="clp-hex">#6366f1</span></div></div>
```

## Why is it useful?
Design system color reference without JavaScript. Aspect-ratio squares, hover scale effect, color name and hex labels. Ideal for design documentation and style guides.
