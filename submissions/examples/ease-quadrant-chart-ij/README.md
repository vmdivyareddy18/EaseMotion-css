# Ease Quadrant Chart

A CSS animation of four quadrants sequentially filling in with color on a chart grid.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.quadrant-chart` element.

## Custom CSS Variables

| Variable         | Default   | Description                |
|------------------|-----------|----------------------------|
| `--quad-color`   | `#4ecdc4` | Fill color of quadrants    |
| `--chart-bg`     | `#1a1a2e` | Background of chart area   |
| `--fill-duration`| `1s`      | Duration of each quadrant fill|

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
