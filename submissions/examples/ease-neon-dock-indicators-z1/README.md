# Ease Neon Dock Indicators

## Overview

A premium dock navigation featuring glowing activity indicators, glassmorphism styling, and floating hover interactions built entirely with HTML and CSS.

## Features

* Animated dock indicators
* Glassmorphism navigation
* Floating hover effect
* Responsive layout
* Lightweight CSS
* Pure HTML & CSS

## Folder Structure

```text
ease-neon-dock-indicators-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

* Chrome
* Firefox
* Edge
* Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Demonstrates reusable dock navigation patterns with subtle micro-interactions, modern styling, and maintainable CSS suitable for dashboards, operating-system-inspired interfaces, and developer tools.
