# Ease Liquid Glass Toggle Cluster

## Overview

A premium collection of glassmorphism toggle switches with smooth liquid animations built entirely using HTML and CSS.

## Features

- Animated toggle switches
- Glassmorphism cards
- Smooth knob transition
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS
