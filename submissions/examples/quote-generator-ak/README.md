# Random Quote Generator

## What does this do?
A random quote generator that displays curated quotes with a fade-in animation on each new quote. Includes attribution and a New Quote button.

## How is it used?
Open demo.html in a browser. A quote appears automatically. Click "New Quote" to cycle through the collection with an animated fade transition.

## Why is it useful?
This provides an inspirational quote component suitable for landing pages, dashboards, or morning routine apps. It demonstrates CSS keyframe animations and random selection logic.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see the quotes.
