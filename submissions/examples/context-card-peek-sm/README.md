# Context Card Peek

1. What does this do? Adds responsive cards that reveal a compact contextual detail panel on hover or keyboard focus with soft slide-and-fade motion.
2. How is it used? Apply `.peek-grid` to the wrapper and `.peek-card` to each focusable card with a `.peek-layer` for the extra details.
3. Why is it useful? It lets interfaces keep cards clean by default while still offering quick supporting context through purposeful CSS-only motion.
