# Ease API Key Manager

## Overview
A CSS-only API key management component for developer dashboards.

## Features
- Masked API keys
- Active/Disabled badges
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
Provides a reusable animation-first component for developer portals, cloud platforms, and admin dashboards.
