# Ease Constellation Navigation

## Overview

A lightweight navigation component featuring glowing constellation-inspired indicators on hover using only HTML and CSS.

## Features

- Pure HTML & CSS
- Hover glow animation
- Responsive layout
- Lightweight
- Easy to customize

## Folder Structure

```
ease-constellation-nav-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why This Example

This component demonstrates a subtle navigation micro-interaction with minimal code, making it suitable for modern websites and dashboards.
