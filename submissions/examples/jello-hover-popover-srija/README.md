# Cyberpunk Neon Jello Hover Popover

A pure CSS animated popover featuring a smooth jello interaction with a cyberpunk neon interface design.

## Features

- Pure CSS animation
- No JavaScript required
- Cyberpunk neon theme
- Jello hover/open animation
- Keyboard accessible using native HTML details element
- Responsive layout
- Custom CSS variables for animation control

## Customization

```css
:root {
 --jello-duration: .6s;
 --jello-easing: cubic-bezier(.36,1.6,.66,1);
 --popover-scale: 1;
}
