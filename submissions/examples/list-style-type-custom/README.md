# list-style-type-custom

## Overview

Demonstrates custom `list-style-type` values using string markers (emoji and text symbols) and the `@counter-style` at-rule for defining custom numbering systems. Includes cyclic, additive, and numeric counter styles. Each list type shows a practical use case with styled list items on a dark background.

## Usage

Open `demo.html` in a modern browser that supports `@counter-style` (Chrome, Firefox, Edge). Five sections show different custom list approaches: emoji bullets, arrow text markers, checkmark cycles, additive roman numerals, and bracketed decimal numbers.

## Features

- Emoji and Unicode symbol list markers via `list-style-type` strings
- `@counter-style` with cyclic, additive, and numeric systems
- Custom prefix, suffix, and symbol sets for counters
- Dark theme with teal accents and purple section headers
- `prefers-reduced-motion` media query
- Responsive layout with mobile breakpoint
- Self-contained with no external dependencies
