EaseMotion CSS is a lightweight collection of CSS animations and transitions designed to be easy to drop into any project. Each component uses custom properties so you can theme and tune everything without touching the core logic.

Copy the component's folder into your project. Link the style.css in your HTML head and the demo.js (if any) before the closing body tag. Customize the :root variables to match your design system.

Yes. The CSS is framework-agnostic. For JS-driven components like the FAB menu or fade-in-on-scroll, you can either use the provided vanilla JS or adapt the logic to your framework's lifecycle (e.g., useEffect in React, onMounted in Vue).
