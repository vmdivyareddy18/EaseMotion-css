# News Ticker Scroll

A scrolling news ticker with marquee-style text.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--ticker-bg` | `#1e1e36` | Ticker background |
| `--text-color` | `#eee` | Text color |
| `--scroll-speed` | `12s` | Scroll animation speed |
| `--accent-color` | `#f72585` | Label accent color |
