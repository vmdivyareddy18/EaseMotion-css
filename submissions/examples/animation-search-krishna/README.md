# Real-Time Animation Search Component

### What does this do?

Provides a client-side JavaScript input filter that instantly screens and updates visible animation cards based on user keystrokes.

### How is it used?

Attach the `id="search-bar"` to an input field and read the DOM classes of your container layout using `.querySelectorAll('.animation-card')` to toggle their CSS display properties dynamically.

### Why is it useful?

As the EaseMotion CSS catalog grows, this human-centric utility avoids heavy page reloads or infinite scrolling fatigue, instantly pointing developers to the exact transition they need.
