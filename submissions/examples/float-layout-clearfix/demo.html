Float Layout & Clearfix

Old Way — float with clearfix

This image is floated left. The .clearfix class uses ::after to clear the float so the container expands to wrap the floated child. Without clearfix, the background would collapse.

Modern Way — display: flow-root

display: flow-root creates a new block formatting context, containing floats without a pseudo-element hack. It is the modern, spec-native replacement for clearfix.
