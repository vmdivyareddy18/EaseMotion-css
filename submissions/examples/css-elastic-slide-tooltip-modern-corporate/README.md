# CSS Elastic Slide Tooltip - Modern Corporate Layouts

A pure CSS animated tooltip component designed for modern corporate dashboard interfaces.

## Features

- Pure CSS animation
- Elastic slide transition effect
- Professional corporate UI style
- Responsive layout
- Keyboard accessible
- Custom CSS animation controls


## Usage

Open `demo.html` directly in a browser.

Hover or focus on dashboard buttons to display the tooltip.


## Customization

Modify animation settings:

```css
:root {
    --tooltip-time: 0.45s;
    --tooltip-scale: 1;
    --tooltip-easing: cubic-bezier(.68,-0.55,.27,1.55);
}
