# ease-image-hover

Pure CSS image hover effects collection for EaseMotion CSS. 15+ gallery-ready effects including zoom, overlays, captions, filters, 3D transforms, and more. No JavaScript required.

## Features

- **Zoom effects** — Standard, slow, deep, and zoom-with-rotate
- **Filter effects** — Grayscale→color, sepia→color, blur→clear, brightness
- **Overlay slides** — Up, down, left, right, fade, scale-in, gradient
- **Caption reveals** — Slide up/down, fade, scale in
- **Shine/glare** — Light sweep across image on hover
- **Frame effect** — Animated border contraction
- **3D tilt** — Perspective rotation on hover
- **Flip** — Y-axis 3D flip
- **Ken Burns** — Continuous slow pan/zoom (pauses on hover)
- **Morph** — Border-radius morph to circle
- **Duotone** — CSS mix-blend-mode color overlay
- **Icon overlay** — Animated icon with spring physics
- **Shadow lift** — Physical elevation with depth shadow
- **Click ripple** — CSS-only radial ripple on active
- **Aspect ratios** — Square, landscape, portrait presets
- **Caption utilities** — Title, text, and tag styling
- **Accessible** — `focus-visible`, `focus-within`, `prefers-reduced-motion`
- **Zero JavaScript** — All effects are CSS-only
