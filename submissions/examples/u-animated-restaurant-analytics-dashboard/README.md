# Animated Restaurant Analytics Dashboard

A modern restaurant analytics dashboard built using pure HTML and CSS.

## Features

- Total Orders overview
- Reservations metrics
- Revenue indicators
- Customer Satisfaction tracking
- Activity feed
- Animated progress bars
- Responsive layout
- CSS Variables
- Dark mode support
- No JavaScript

## Folder Structure

u-animated-restaurant-analytics-dashboard/
├── demo.html
