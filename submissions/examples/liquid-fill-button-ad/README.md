# Liquid Fill Button

1. **What does this do?**  
   A button that appears to fill up with a liquid color from the bottom when hovered.

2. **How is it used?**  
   ```html
   <button class="liquid-btn">Hover Me</button>
   ```

3. **Why is it useful?**  
   It provides an elegant, engaging hover state for primary actions, aligning with EaseMotion CSS's smooth animation principles.
