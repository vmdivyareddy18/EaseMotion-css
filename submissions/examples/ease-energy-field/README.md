# ease-energy-field

Floating energy particles drift around a background container, creating an ambient field effect.

## Usage

```html
<div class="ease-energy-field" id="field">Content</div>
```

Populate with `.ep` child elements via script, setting random positions and animation timings.

## Why it fits EaseMotion CSS

CSS defines the float keyframe and particle style. Script is only a DOM generator loop.
