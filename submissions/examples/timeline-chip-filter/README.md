# Timeline Chip Filter

## What does this do?
A responsive filter chip row for timeline views with active, hover, and focus-visible states.

## How is it used?
```html
<div class="chip-filter">
  <button class="active">Today</button>
  <button>Week</button>
</div>
```

## Why is it useful?
It gives EaseMotion CSS a practical filter control for activity feeds, logs, and timeline dashboards.
