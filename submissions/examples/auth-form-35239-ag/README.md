# Animated Login/Signup Form

An interactive authentication form switcher demonstrating sliding tab gliders built using only HTML/CSS.

---

## What is Included

1. **`demo.html`**: A central login card containing sliding tab anchors, sliding viewports, credential text groups, and secure actions.
2. **`style.css`**: Card borders, tab gliders, transform translate offsets, and input warning glows.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Click the **Sign Up** tab — verify the background glider slides smoothly to the right and the signup form translates in.
3. Click the **Login** tab — verify the glider returns to the left and reveals the login form.
