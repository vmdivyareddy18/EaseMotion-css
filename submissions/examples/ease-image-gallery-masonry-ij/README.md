# Ease Image Gallery Masonry Component

An optimized, highly responsive column-based masonry gallery architecture featuring pure CSS loading entry staggers and elegant micro-hover scaling.

## Features
- **Pure CSS Stagger Calculations**: Utilizes inline variable index counts multiplied dynamically inside an `animation-delay: calc()` configuration.
- **Fluid Column Balancing**: Implements modern CSS columns layout mapping context blocks safely into auto-wrapping grid distributions.
- **Isolated Customization**: Control padding configurations, item sizing, and entry rates modularly via globally scopeable CSS variables.
