# Neobrutalism Toast Notification

A bold, high-contrast toast notification component that slides in with a harsh bounce.

## Files
- `demo.html`: The HTML structure of the toast.
- `style.css`: The CSS rules defining the toast's aesthetic and keyframe animations.
- `README.md`: Documentation.

## Features
- **Rigid Keyframes:** Animates into view with a fast, rigid slide-and-snap motion to match the brutalist aesthetic.
- **Harsh Contrast:** Uses a bright accent color, thick borders, and solid block shadows.
- **Auto-Hide:** Uses pure CSS keyframes to automatically slide out after a set delay.

## Usage
Simply inject the `.neo-brutalism-toast` element into the DOM, and the pure CSS keyframes will handle the entrance and exit automatically.
