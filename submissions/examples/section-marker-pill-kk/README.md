# Section Marker Pill

1. **What does this do?**  
   It creates a compact pill-style label for chapters, sections, and insights in editorial, documentation, and article-style layouts.

2. **How is it used?**  
   Place the marker directly above a heading or content block:

   ```html
   <div class="section-marker section-marker-primary">Chapter 01</div>
   <h2 class="section-title">Designing readable motion systems</h2>
   ```

3. **Why is it useful?**  
   It adds a small, readable, motion-forward hierarchy component that works naturally with headings, docs pages, editorial sections, and storytelling layouts.
