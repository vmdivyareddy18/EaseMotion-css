# Underline Tabs

## What does this do?

Creates a modern tab navigation with animated underline effects and gradient accents using pure HTML and CSS.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="tabs">

    <button class="tab active">
        Overview
    </button>

    <button class="tab">
        Features
    </button>

</div>
```

---

## Why is it useful?

Underline tabs are widely used for navigation in dashboards, landing pages, documentation sites, settings pages, and web applications. This standalone example demonstrates a clean and reusable tab interface with smooth underline animations while remaining lightweight and framework-independent.
