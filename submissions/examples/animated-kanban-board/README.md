# Animated Kanban Board UI

A clean, modern, and fully responsive **Kanban Board** interface. This example demonstrates how to use **EaseMotion CSS** to create a beautiful, staggered grid layout with smooth card entrance animations and interactive hover states, making the project management interface feel alive and responsive.

## 🚀 Features

- **Staggered Grid Entrance**: Uses `ease-fade-in-up` combined with `ease-delay-*` classes so the columns and task cards cascade in sequentially on page load.
- **Interactive Hover States**: Combines `ease-hover-grow` with custom CSS to add a subtle scale effect and an indigo border highlight when hovering over a card.
- **Modern SaaS Aesthetic**: Features soft shadows, rounded corners, color-coded status badges, and overlapping user avatars.
- **Fully Responsive**: The 3-column grid gracefully collapses into a single-column vertical layout on mobile devices.
- **Accessible & Semantic**: Uses proper heading hierarchy and `aria-labels` on icon buttons.
