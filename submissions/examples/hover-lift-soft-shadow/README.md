# Hover Lift & Soft Shadow

## Description
A smooth interaction effect that gently lifts an element upwards and adds a soft, diffused shadow underneath it when a user hovers over it.

## Usage
Simply apply the `hover-lift-shadow` class to interactive elements like buttons, cards, or feature highlights.

```html
<button class="hover-lift-shadow">
  Hover over me!
</button>

<!-- Or on a card -->
<div class="hover-lift-shadow" style="padding: 20px; border: 1px solid #ccc; border-radius: 8px;">
  Interactive Card
</div>
```

## Why is this useful for EaseMotion CSS?
This is a classic "human-readable" micro-interaction that provides immediate tactile feedback. It is highly composable and perfect for buttons, product cards, or feature highlights, making interfaces feel more responsive and premium without overwhelming the user.
