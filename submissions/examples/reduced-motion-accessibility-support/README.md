# Reduced Motion Accessibility Support

A responsive reduced-motion accessibility demo built using pure HTML and CSS.

## Features

- `prefers-reduced-motion` media query support
- Reduced animation intensity
- Motion-safe hover interaction helpers
- Accessibility-focused animation fallbacks
- Responsive demo layouts
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the reduced-motion accessibility support demo.
