# BMI Calculator

## What does this do?
A BMI calculator that computes Body Mass Index from height and weight, displaying the result on a color-coded gauge with health category labels.

## How is it used?
Open demo.html in a browser. Enter your height in centimeters and weight in kilograms. The BMI value updates in real-time with category color coding and a position indicator on the BMI scale.

## Why is this useful?
This provides a health tracking component suitable for fitness apps or wellness dashboards. It demonstrates conditional color coding and proportional gauge positioning.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
