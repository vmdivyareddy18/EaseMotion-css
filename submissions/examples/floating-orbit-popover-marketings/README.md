# Floating Orbit Popover

A pure CSS animated Floating Orbit Popover component designed for modern marketing landing page layouts.

## Features

- CSS-only animation
- No JavaScript required
- Smooth floating orbit transition
- Responsive design
- Keyboard accessible using focus-within
- Custom animation variables

## Usage

Add the component structure:

```html
<div class="orbit-popover">

  <button class="orbit-trigger">
    Explore Features
  </button>

  <div class="orbit-content">
    Content here
  </div>

</div>
