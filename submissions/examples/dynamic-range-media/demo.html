What is dynamic-range?

The @media (dynamic-range) media query detects whether a display supports high or standard dynamic range. HDR displays can show brighter highlights, deeper blacks, and a wider contrast ratio. Use this query to serve HDR-optimized colors and brightness levels without degrading the experience on SDR screens.

HDR-aware styling strategy

Serve HDR enhancements as progressive enhancements. Start with SDR-safe values, then layer HDR improvements inside @media (dynamic-range: high). Use color(display-p3 …) for wider colors and increase contrast or brightness for HDR displays. Always test on both SDR and HDR screens.

/* Base SDR styles — safe defaults */ .hero { background: linear-gradient(135deg, #ff6600, #ffcc00); box-shadow: 0 4px 20px rgba(255, 100, 0, 0.3); } /* HDR enhancement layer */ @media (dynamic-range: high) { .hero { background: linear-gradient(135deg, color(display-p3 1 0.3 0), color(display-p3 1 0.95 0.1) ); box-shadow: 0 4px 40px rgba(255, 100, 0, 0.5); filter: brightness(1.1) contrast(1.15); } } /* SDR fallback — no HDR enhancements applied */ @media (dynamic-range: standard) { .hero { filter: none; } }
