# Animation Playground

## Overview

Interactive playground for experimenting with EaseMotion animation utilities in real time.

## Features

* Animation selection
* Duration control
* Delay control
* Iteration count selection
* Direction selection
* Timing function selection
* Fill mode selection
* Animation restart button
* Copy configuration button
* Live preview panel
* Reduced-motion support
* Keyboard-accessible controls
* ARIA labels for accessibility

## Available Animations

* Bounce
* Fade
* Slide

## Configuration Options

### Duration

Controls how long the animation takes to complete.

### Delay

Specifies the waiting period before animation begins.

### Iteration Count

Determines how many times the animation runs.

### Direction

Controls playback direction.

### Timing Function

Defines the animation acceleration curve.

### Fill Mode

Determines how styles are applied before and after execution.

## Accessibility

The playground includes:

* Keyboard navigation support
* ARIA labels
* Focusable preview element
* Reduced-motion preference handling

## Usage

Open `demo.html` in your browser and experiment with different animation configurations.
