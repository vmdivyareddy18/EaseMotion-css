# Animated Loading States & Skeletons

A clean, modern, and fully responsive collection of **Loading Indicators and Skeleton Screens**. This example demonstrates how to use **EaseMotion CSS** continuous animations (`ease-pulse`, `ease-spin-slow`) combined with staggered delays to create a complete suite of accessible loading states without writing a single custom keyframe.

## 🚀 Features

- **Zero Custom Keyframes**: All continuous animations are handled purely by EaseMotion utility classes.
- **Profile Skeleton**: Uses `ease-pulse` with staggered delays to create a realistic, cascading shimmer effect for user profile placeholders.
- **List Skeleton**: Demonstrates how to apply staggered pulsing to multiple lines of text to mimic content loading.
- **Classic Spinner**: Uses `ease-spin-slow` to create a smooth, infinite rotating loading indicator.
- **Pulsing Dots**: Uses `ease-pulse` with staggered delays on three dots to create a smooth, wave-like loading effect.
- **Hover Interactions**: Cards lift slightly on hover for better tactile feedback.
