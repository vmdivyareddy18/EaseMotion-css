# CSS Variable Font Axis Animation

### What does this do?
Demonstrates fluid typography mutations using CSS keyframes and mouse interaction layers to morph variable font weight, width, and slant axes simultaneously.

### How is it used?
Apply the variation utility configurations directly onto structural headers supporting systemic or embedded variable typefaces.

### Why is it useful?
It replaces heavy, traditional, multi-file font loading routines with a single flexible asset file that performs hardware-accelerated fluid transitions on low-power displays.
