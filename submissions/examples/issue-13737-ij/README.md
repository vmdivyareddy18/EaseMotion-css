# Ease Combo Counter

Closes #13737

## Description
A combo counter component with scale pulse animation on increment. Color intensifies from green to yellow to red as the combo value increases, providing clear visual feedback.

## Acceptance Criteria
- Number display scales up with a pulse animation on each increment
- Color transitions from green (<5) to yellow (5-9) to red (10+)
- Reset button restores counter to zero
- Smooth CSS transitions on both transform and color

## Files
- `style.css` — Dark theme with combo counter animations
- `demo.html` — Interactive counter demo with Add and Reset buttons
