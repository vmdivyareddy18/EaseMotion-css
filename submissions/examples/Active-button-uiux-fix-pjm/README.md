# Active Navigation Button UI Enhancement

## Description

Enhances the active navigation button by improving text readability using a vibrant gradient text effect and a modern active-state background. The goal is to make the selected navigation item more visually distinct while maintaining EaseMotion CSS's clean and modern design language.

## Features

- Gradient text for active navigation item
- Smooth color transition
- Improved readability in light mode
- Modern active-state appearance
- Lightweight and responsive
- Easy to customize

## Usage

Apply the `active` class to the currently selected navigation item.

Example:

```html
<nav class="nav">
    <a href="#" class="nav-link active">Getting Started</a>
    <a href="#" class="nav-link">Utilities</a>
    <a href="#" class="nav-link">Animations</a>
</nav>
```
