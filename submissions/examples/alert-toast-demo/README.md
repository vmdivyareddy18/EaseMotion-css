# Alert and Toast Notification Component Demo

## What does this do?
Creates a CSS-only Alert and Toast Notification Component demo that showcases common notification patterns used in modern web applications.

## How is it used?
Use the `.alert` class combined with variants like `.success`, `.error`, `.warning`, or `.info` for inline alerts.
Use the `.toast` class for a floating notification.

```html
<div class="alert success">
  ✔ Operation completed successfully.
</div>

<div class="toast success-toast">
  ✔ File uploaded successfully.
</div>
```

## Why is it useful?
Notifications are essential UI elements in nearly every web application. Providing alert and toast demos would help developers quickly implement feedback systems while following EaseMotion CSS's human-readable, composable, and animation-first design philosophy. These demos also serve as practical examples for combining utility classes with entrance, exit, fade, slide, and attention-grabbing animations available in EaseMotion CSS.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
