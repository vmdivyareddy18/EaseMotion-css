# Recovery Treatment Center Website

This demo site showcases a compassionate, hope‑focused recovery center layout using **EaseMotion CSS classes** only.

## 🧩 Sections
- Hero with hopeful recovery messaging
- Programs (inpatient, outpatient, aftercare)
- Our approach section
- Staff credentials
- Patient success stories (anonymized)
- Insurance verification section
- 24/7 help line CTA
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — entrance fade animations
- `ease-slide` — sliding entrance for headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs and staff cards
- `ease-hover-opacity-reduce` — hover dimming for program cards and testimonials

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll through sections to see entrance animations.
3. Hover over buttons, cards, and testimonials to see interactive effects.
4. Replace placeholder images and text with real recovery center content.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion animation classes
