# Meeting Room Availability Timeline

## What does this do?

Shows available, held, booked, and current-time states for room scheduling.

## How is it used?

```html
<div class="room-timeline">
  <span class="slot is-booked">10:00</span>
</div>
```

## Why is it useful?

Scheduling UIs need dense but readable availability states that work on narrow screens.
