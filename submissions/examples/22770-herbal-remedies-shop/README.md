# Issue #22770 - Advanced: E-commerce - Herbal Remedies Shop

This submission adds a premium, responsive, one-page e-commerce landing page for **Verdant Vessel**, a wellness herbal remedies shop built with plain HTML, scoped CSS, and existing EaseMotion CSS classes.

## What This Submission Adds
- A complete editorial-style herbal wellness storefront that opens directly in the browser
- Inline SVG hero artwork featuring an apothecary bottle, botanical leaves, sunlight glow, and floating trust badges
- Four product cards with responsible wellness copy, herb details, aroma notes, prices, and CTAs
- Ingredient sourcing story, certification badges, testimonials, a wellness quiz CTA, and a branded footer
- Calm gradients, glassmorphism surfaces, and subtle botanical decoration using CSS only

## Required Sections Included
- Hero with botanical imagery-style visual
- Product grid with herb benefits
- Ingredient sourcing story
- Certification badges
- Reviews and testimonials
- Wellness quiz CTA
- Footer

## EaseMotion Classes Used
- Entrance motion: `ease-fade-in`, `ease-slide-up`, `ease-delay-100`, `ease-delay-200`, `ease-delay-300`
- Hover and interaction: `ease-btn-hover`, `ease-hover-glow`, `ease-hover-lift`, `ease-hover-underline`, `ease-card-hover`
- Components and utilities: `ease-container`, `ease-btn`, `ease-btn-primary`, `ease-btn-outline`, `ease-card`

## How To Open
1. Open `demo.html` directly in any modern browser.
2. No server, package install, build step, JavaScript, or external assets are required.

## Responsive Behavior
- Desktop: split hero layout, four-column product grid, multi-column story and review sections
- Tablet: stacked content blocks with two-column product and certification grids
- Mobile: single-column flow, full-width actions, simplified badge placement, and readable spacing

## Browser Compatibility
- Tested for standards-based modern browsers that support CSS Grid, gradients, `backdrop-filter`, and inline SVG
- Suitable targets include current Chrome, Edge, Firefox, and Safari
