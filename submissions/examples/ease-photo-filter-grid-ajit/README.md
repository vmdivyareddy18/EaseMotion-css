# Ease Photo Filter Grid

A CSS-only photo filter grid component with animated image cards and interactive filter buttons.

## Features

- CSS-only implementation
- Responsive grid layout
- Animated image hover effect
- Interactive filter buttons
- Modern card design
- Easy to customize

## Folder Structure

```
ease-photo-filter-grid-ajit/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<div class="ease-photo-filter-grid">
    <div class="card">
        <img src="image.jpg" alt="">
        <span>Category</span>
    </div>
</div>
```

Open `demo.html` directly in your browser.
