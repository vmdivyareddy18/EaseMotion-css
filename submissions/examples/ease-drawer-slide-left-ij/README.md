# Drawer Slide Left

A side panel (drawer) that slides in from the left edge of the viewport.

## Features

- Slides in from the left with a smooth transform animation
- Semi-transparent overlay backdrop
- Close button inside the drawer header
- Opens via button and closes via overlay click, close button, or Escape key
- Navigation links inside the drawer

## CSS Custom Properties

| Property                   | Default              | Description          |
| -------------------------- | -------------------- | -------------------- |
| `--drawer-duration`        | `0.3s`               | Slide animation time |
| `--drawer-width`           | `300px`              | Drawer width         |
| `--drawer-bg`              | `#ffffff`            | Drawer background    |
| `--drawer-text-color`      | `#1e293b`            | Text color           |
| `--drawer-overlay-color`   | `rgba(0,0,0,0.4)`   | Overlay color        |
| `--drawer-shadow`          | `2px 0 16px …`       | Drawer box-shadow    |
| `--font-family`            | `Inter`              | Font family          |

## Usage

Open `demo.html` in a browser and click **☰ Open Drawer** to see the drawer slide in.
