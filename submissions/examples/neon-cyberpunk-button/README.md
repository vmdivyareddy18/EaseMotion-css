# Neon Cyberpunk Button

## What does this do?
This adds a highly stylized, interactive "Neon Cyberpunk" button component. It features multi-layered neon box-shadow glows, an animated gradient sheen on hover, diagonal corner cuts via pseudo-elements, and an intense active/click glitch animation.

## How is it used?
Use this for call-to-actions, form submissions, or terminal-themed UI interfaces. Include the HTML snippet `<button class="cyberpunk-btn">...</button>` and copy the CSS styles from `style.css`. It's particularly effective on dark themes.

## Why is it useful?
It provides an out-of-the-box, highly engaging aesthetic for modern web applications aiming for a tech-heavy or retro-futuristic style. All animations and styling are handled through pure CSS, ensuring maximum performance without JavaScript overhead.
