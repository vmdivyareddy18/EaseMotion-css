# Staggered List Animation

A utility for animating list items sequentially with staggered fade-in and slide-in effects. Each item uses a `--i` custom property to control animation delay. The list items fade and slide from the left, one after another, creating a polished entrance sequence.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. The list items animate in sequence on page load.

## Accessibility notes

Reduced motion disables the animation entirely and shows all items immediately.
