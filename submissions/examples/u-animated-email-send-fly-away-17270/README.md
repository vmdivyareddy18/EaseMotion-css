# Animated Message Bubble Pop

## Overview

Animated Message Bubble Pop is a lightweight CSS animation that creates a subtle popping effect for chat messages and notification bubbles.

## Features

- CSS-only animation
- Smooth pop effect
- Chat UI demonstration
- Responsive layout
- Easy customization
- Accessibility support

## Usage

```html
<div class="message pop">
  New message received!
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
