# Supercomputing Fluid Dynamics Simulation Suite Showcase

A responsive, interactive UI dashboard concept for a fluid dynamics simulation suite. Built with **EaseMotion CSS** utility classes and minimal custom CSS. Zero JavaScript — all interactivity is handled natively by the browser (range sliders, hover effects).

## Sections
- **Header** — navigation with hover effect
- **KPI Row** — active simulations, CPU usage, time remaining, iterations
- **Visualization Area** — placeholder for real‑time flow rendering
- **Control Panel** — wind speed, temperature, particle count sliders + run button

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-min-h-screen`, `ease-py-*`, `ease-px-*`, `ease-mb-*`, `ease-mt-*`, `ease-h-64`, `ease-lg-col-span-2`
- **Background:** `ease-bg-gray-900`, `ease-bg-gray-800`, `ease-bg-blue-900`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-blue-*`, `ease-text-green-*`, `ease-text-yellow-*`, `ease-text-purple-*`, `ease-text-white`
- **Components:** `ease-card`, `ease-btn`, `ease-btn-primary`, `ease-badge`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-text-blue-300`
- **Animations:** `ease-fade-in`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-w-full`, `ease-block`, `ease-uppercase`, `ease-space-y-4`

## How to Use
1. Open `demo.html` in any modern browser.
2. Adjust the sliders and observe the UI.
3. Copy the HTML/CSS into your project for a simulation dashboard template.
