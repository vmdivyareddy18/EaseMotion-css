# 3D Tilt Select Menu

A 3D tilt select menu component for EaseMotion CSS inspired by modern admin panel designs.

## Features

- Pure CSS animation
- No JavaScript required
- 3D tilt hover effect
- Responsive design
- Accessible focus state

## Usage

Open demo.html to view the component.

## Animation

The select menu uses CSS 3D transforms to create a tilt interaction on hover.
