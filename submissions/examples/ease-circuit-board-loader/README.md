# Circuit Board Loader – Animated circuit path loading

> A production-ready circuit board loader component that creates a mesmerizing animated circuit path loading effect with glowing nodes, flowing traces, and futuristic aesthetics. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Animated Circuit Paths** – Glowing traces flow like electricity
- ✅ **Pulsing Nodes** – Bright nodes pulse along the circuit
- ✅ **Customizable Colors** – Primary, secondary, and background
- ✅ **Speed Control** – Adjust animation speed
- ✅ **Node Count** – Control number of circuit nodes
- ✅ **Pause/Resume** – Control the animation
- ✅ **Regenerate** – Generate new circuit patterns
- ✅ **Random Colors** – Randomize colors with one click
- ✅ **Keyboard Shortcuts** – Space (pause), R (regenerate), C (random color)
- ✅ **4 Color Variants** – Cyber, Neon, Matrix, Aurora
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark Mode** – Auto-adjusts to system preference
- ✅ **Zero Dependencies** – No libraries or frameworks
