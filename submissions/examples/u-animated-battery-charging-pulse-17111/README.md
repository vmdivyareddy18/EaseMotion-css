# Animated Battery Charging Pulse

## Overview

A reusable battery charging animation featuring a subtle pulse effect suitable for dashboards, device monitoring systems, charging indicators, and productivity applications.

## Features

- CSS-only animation
- Battery charging pulse effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="battery-button">
  <span class="battery-icon">🔋</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
