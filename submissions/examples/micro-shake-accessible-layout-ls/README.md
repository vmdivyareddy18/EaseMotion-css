# Micro Shake Accessible Accordion

## Description
A pure CSS animated accordion with a subtle micro shake interaction. Built using semantic HTML (`details` and `summary`) and CSS only.

## Features
- Pure HTML and CSS
- No JavaScript
- Responsive layout
- Keyboard accessible
- Smooth micro shake animation
- CSS custom properties for customization

## Custom Properties

- --duration
- --easing
- --shake-distance
- --radius
- --accent
