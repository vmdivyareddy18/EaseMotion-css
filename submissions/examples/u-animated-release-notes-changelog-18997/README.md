# Animated Release Notes Changelog

A modern changelog component featuring animated release cards, version badges, status indicators, and responsive layouts.

## Features

- Animated release cards
- Version badges
- Status tags
- Hover interactions
- Responsive design
- Staggered reveal animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

You can customize:

- Version numbers
- Release categories
- Colors
- Animation timing
- Layout spacing

## Use Cases

- Open-source project changelogs
- SaaS release notes
- Product update pages
- Documentation websites
- Software version history
