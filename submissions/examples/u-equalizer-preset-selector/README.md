# ease-equalizer-preset-selector

Animated EQ preset selector component for audio and music applications.

## Features

- Animated selected preset state
- Highlight pulse effect
- Smooth button transitions
- Equalizer visualization
- Audio settings UI
- Responsive layout

## Acceptance Criteria Covered

### Selected State Animation

The active preset receives:

- Animated highlight
- Pulse effect
- Glow feedback

### Smooth Transitions

Preset buttons transition smoothly between states.

### Use Cases

- Music players
- Audio settings panels
- Equalizer controls
- Podcast apps
- Streaming applications
