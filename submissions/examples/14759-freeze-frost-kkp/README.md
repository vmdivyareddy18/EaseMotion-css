# Freeze Frost Spread

1. What does this do? Creates a blue-white frost spread effect with growing crystal lines and a cool glow.
2. How is it used? Apply `.frost-card` to a card and include `.frost-crystal` spans for the spreading ice accents.
3. Why is it useful? It offers a CSS-only status or game-style freeze cue that is visual, compact, and reusable.
