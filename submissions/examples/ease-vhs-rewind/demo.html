Technical Note

This effect uses clip-path: inset() to slice the ::before and ::after pseudo-elements into horizontal bands. Each band is assigned a jittery translateX animation. The color separation is achieved using mix-blend-mode: screen, which beautifully blends the red and cyan channels over the original content.
