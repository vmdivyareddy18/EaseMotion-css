# Heart / Bookmark Toggle Button

Toggle buttons for heart (like) and bookmark actions. The heart uses a filled pink SVG on active state; the bookmark uses a filled indigo SVG. The button border also changes color to match. Both use a simple `.active` class toggle via JavaScript.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click each button to toggle its active (filled) state.

## Accessibility notes

Each button has an `aria-label` for screen readers. Reduced motion disables the fill and transform transitions.
