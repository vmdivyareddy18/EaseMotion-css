Problem: Overflow on Mobile

Wide animations (sliding panels, wide-scale transforms, offset-based effects) can cause horizontal scrollbars on narrow viewports. Use the toggle below to see the difference.

How It Works

The checkbox above toggles the overflow-x: hidden rule on the container. When unchecked, the animations extend beyond the viewport causing a horizontal scrollbar. When checked, the overflow is clipped.

/* Fix — apply to the wrapper element */
.animation-wrapper {
  overflow-x: hidden;
}

/* Alternative: per-element */
.wide-animation {
  max-width: 100%;
  overflow-x: hidden;
}

Best practice: Use overflow-x: hidden on the immediate parent container of animation elements rather than body, to avoid clipping tooltips or dropdowns.
