# Pulse-Active Tabs – Marketing Landing Page

**Issue:** #50169

## Overview

A pure CSS Pulse-Active Tabs component designed for Marketing Landing Pages with a smooth pulse animation, responsive layout, and keyboard accessibility.

## Features

- Pure CSS
- Pulse-active animation
- Responsive design
- Keyboard accessible
- CSS custom properties
- Lightweight
- No JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="tabs">
    <button class="tab active">Overview</button>
    <button class="tab">Features</button>
    <button class="tab">Testimonials</button>
    <button class="tab">Contact</button>
</div>
```

## Customization

```css
:root{
  --primary:#ff6b6b;
  --duration:.35s;
  --radius:12px;
}
```

## Why EaseMotion CSS?

- Pure CSS implementation
- Human-readable class names
- Animation-first philosophy
- Easy customization
- Responsive and reusable
