# Component Title Showcase 186

## Overview
A custom premium CSS-only component showcase featuring gradient cards, profile cards, and animated buttons with hover effects. Built entirely with EaseMotion CSS utility classes.

## Folder Structure

submissions/examples/new-comp-slug-186-sn250/
├── demo.html
├── style.css
└── README.md


## Components Included
- [x] Gradient Card with hover scale effect
- [x] Profile Card with user stats
- [x] Animated Buttons (3 variants: Hover Lift, Pulse, Glow)
- [x] Fully responsive (mobile, tablet, desktop)
- [x] Pure CSS (no JavaScript required)

## How to Use
1. Link the component stylesheet `style.css` in your HTML header.
2. Embed the structure code from `demo.html` into your body sections.
3. All EaseMotion CSS classes are loaded via CDN.

## EaseMotion Classes Used
- **Layout**: `ease-container`, `ease-grid`, `ease-flex`, `ease-flex-wrap`
- **Spacing**: `ease-py-*`, `ease-p-*`, `ease-mt-*`, `ease-mb-*`, `ease-gap-*`
- **Typography**: `ease-text-*`, `ease-font-bold`, `ease-text-center`
- **Colors**: `ease-bg-*`, `ease-text-*`, `ease-hover-bg-*`
- **Borders & Shadows**: `ease-rounded-*`, `ease-shadow-lg`, `ease-border-t`
- **Animations**: `ease-animate-fadeIn`, `ease-animate-slideUp`
- **Interactions**: `ease-hover-lift`, `ease-hover-scale-105`, `ease-transition`
- **Responsive**: `md:ease-grid-cols-2`, `lg:ease-grid-cols-3`
- **Utilities**: `ease-w-full`, `ease-w-24`, `ease-h-24`, `ease-mx-auto`, `ease-opacity-*`

## How to Run
1. Open `demo.html` in any modern browser
2. No build step, no server required
3. All CSS loaded via CDN
