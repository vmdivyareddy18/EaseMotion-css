Visibility vs Display
visibility-hidden

Hides the element from view, but keeps it in the document flow. The space it occupies remains blank, preventing surrounding elements from shifting positions.

display: none

Completely removes the element from the layout. Its space collapses, causing surrounding elements to fill in the empty gap.

Usage & Tables
visibility-collapse

Specifically designed for table rows, columns, column groups, and row groups. It hides the row/column and collapses the space it would have taken. For other elements, it behaves exactly like hidden.

Accessibility Note

Both visibility: hidden and display: none hide elements from screen readers. Use these utilities to hide content completely from all users.
