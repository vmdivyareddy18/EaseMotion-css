ease-btn-hover vs ease-hover-grow

Both classes react to mouse hover events, but target different elements and design surfaces. Explore this side-by-side interactive playground to master their differences, resolve layout collisions, and prevent CSS specificity issues.

Change the UI component below using the sandbox selector. Hover each container to visually compare how the classes execute their transitions.

ease-btn-hover Button Modifier
Effect: Translates element by -3px on the Y-axis and generates a deep indigo box-shadow. Designed for clickable components.
ease-hover-grow Generic Utility
Effect: Scales element size to 1.06 of its original footprint. Does not add box shadow or glow effects.

What happens when you stack both classes on the same element? Both utilities try to animate the transform property, causing collisions. Try it out below.

!
Transform Collision Because both classes write to the transform shorthand property, the browser cannot compose them. The class declared last in the stylesheet will completely override the other.
i
Shadow Composition Since ease-btn-hover applies a box-shadow and ease-hover-grow does not, the shadow effect is preserved even if the translation is overridden.

Decision Matrix — Which One When?

Use this quick reference to pick the right tool for the job. Avoid naming overlap confusion by focusing on the UI component type.

Component Type Primary Utility Visual Interaction Recommendation reason
Primary / Action Buttons ease-btn-hover Translational Lift (3px) + Glowing Glow Highlights primary call-to-actions with real-world elevation mimicking depth.
Bento Grid / Layout Cards ease-hover-grow Visual Scale (1.06) Scale makes large blocks feel active and responsive without rendering distracting shadows.
Interactive Chips / Badges ease-hover-grow Visual Scale (1.06) Maintains content layout alignment while giving micro-feedback on cursor entry.
Circular Avatars / Profile Images ease-hover-grow Visual Scale (1.06) Draws immediate user focus by popping the element forward in 2D space.
Small Control Icons ease-hover-grow Visual Scale (1.06) Scale is cleaner than translation and shadow for tight spacing environments.
