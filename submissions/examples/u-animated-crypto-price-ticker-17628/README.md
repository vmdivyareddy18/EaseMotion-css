# Animated Crypto Price Ticker

## Overview

Animated Crypto Price Ticker is a lightweight CSS animation that simulates cryptocurrency price updates using animated values and trend indicators.

## Features

- CSS-only animation
- Animated price ticker
- Trend pulse indicator
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="crypto-ticker">
  <span class="coin">₿</span>
  <span class="price">$105,000</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
