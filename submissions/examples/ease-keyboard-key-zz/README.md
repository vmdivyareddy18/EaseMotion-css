# Keyboard Key

## What does this do?
Keyboard Key component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"kk-primary\">Keyboard Key Primary</div>
```

## Why is it useful?
Provides a reusable keyboard key UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
