# Enhanced Pulse Button

A high-performance, accessible pulse animation for Call-To-Action (CTA) elements.

## Features
- **Customizable**: Control speed and color using CSS variables.
- **Variations**: Includes standard, fast, and soft-glow modes.
- **Accessible**: Automatically respects `prefers-reduced-motion` system settings.
- **Interactive**: Animation pauses on hover for a better user experience.

## Usage
```html
<button class="btn-pulse">Click Me</button>
<button class="btn-pulse fast">Fast Pulse</button>
```

## Why it fits Ease Motion-css?

This component provides a professional UI interaction that boosts engagement while maintaining the library's focus on lightweight, composable utilities.
