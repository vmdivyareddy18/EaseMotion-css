# Component Showcase

A comprehensive component showcase page showing live demos of every EaseMotion CSS component.

## Components Featured

- **Buttons**: Primary, Secondary, Outline, Ghost, Danger, Disabled, Sizes, Loading, Icon
- **Cards**: Standard, Hover, Bordered variants
- **Modals**: Modal dialog with header, body, and footer sections
- **Dropdowns**: Hover-activated dropdown menus with items, dividers, and danger variants
- **Forms**: Text input, Email, Select, Textarea, Checkbox, Radio, Toggle switch
- **Badges & Tags**: Color variants, pills, sizes, outline, dismissible tags
- **Alerts**: Info, Success, Warning, Danger with close buttons
- **Progress**: Determinate and indeterminate progress bars
- **Spinners & Loaders**: Spinner sizes, dots loader, pulse loader
- **Tooltips**: Top, Bottom, Left, Right positioning
- **Tabs**: Tab navigation with panel content switching
- **Accordion**: Expand/collapse content sections
- **Pagination**: Page navigation with active states
- **Breadcrumbs**: Hierarchical navigation links

## Directory Structure

- `demo.html` — Full component showcase page with live demos
- `style.css` — Dark theme styling for all components
