# CSS Tilt Hover Button – SaaS Showcase Layout

A modern pure CSS button featuring a smooth Tilt Hover animation designed for SaaS showcase interfaces.

## Features

- Pure HTML & CSS
- Responsive design
- Keyboard accessible
- Smooth tilt hover animation
- Uses CSS custom properties for customization
- No JavaScript required

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<button class="tilt-btn">
    Get Started
</button>
```

## Custom Properties

```css
:root{
    --duration:0.35s;
    --scale:1.06;
    --rotate:8deg;
    --primary:#5b5df0;
    --secondary:#7a5cff;
}
```
