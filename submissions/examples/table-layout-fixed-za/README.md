# Table Layout Fixed vs Auto

A CSS demo comparing table-layout: fixed and auto. Fixed layout distributes columns evenly based on table width, while auto adjusts to content.

## Files

- `demo.html` - Two tables with the same content, different layouts
- `style.css` - table-layout property, table styling

## Usage

Open `demo.html` to see how fixed vs auto distributes column widths differently.
