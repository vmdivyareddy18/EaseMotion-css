# Liquid Filling Circular Button

A pure CSS circular action button with an organic liquid fill effect on hover and focus.

## Features

- Circular button shape with a rising liquid wave.
- Uses CSS pseudo-elements for the soft wave surface.
- Supports multiple color variants through `--fill`.
- Includes hover and keyboard focus states.
- Respects reduced-motion preferences.

## Usage

Open `demo.html` to preview the button. Use `.ease-liquid-button` for the circular control and place `.liquid-wave` plus `.liquid-label` inside it.
