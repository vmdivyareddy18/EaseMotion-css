# Horizontal Card Layout

1. **What does this do?**  
   Adds a horizontal card layout idea that places media, content, and actions in a compact side-by-side card.

2. **How is it used?**  
   ```html
   <article class="horizontal-card">
     <div class="card-media"></div>
     <div class="card-content">Card content</div>
   </article>
   ```

3. **Why is it useful?**  
   It fits EaseMotion CSS by giving developers a readable pattern for scan-friendly product, article, and profile cards.
