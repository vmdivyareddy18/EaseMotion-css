# Seahorse Reef

A pure CSS and HTML seahorse hovering over a reef, its prehensile tail curling and its dorsal fin whirring to hold position.

## What it shows

- A prehensile tail from four shrinking discs that curl together on a shared keyframe
- The dorsal fin whirring fast with a striped repeating linear gradient, the way a real seahorse swims
- Bony body ridges from a diagonal repeating linear gradient, plus a spiky coronet and coral from masked conic gradients
- Swaying seagrass, drifting light shafts and rising bubbles for the reef
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
