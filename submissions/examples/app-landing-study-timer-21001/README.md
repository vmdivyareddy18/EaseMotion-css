# App Landing: Study Timer (Pomodoro) (`app-landing-study-timer-21001`)

This proposal provides a complete, copy-paste ready landing page template designed for a Productivity/Study Timer Mobile App, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A productivity and focus app requires a UI that feels clean, distraction-free, modern, and snappy. This template achieves this aesthetic using a crisp Light Mode palette with Indigo (`#4F46E5`), Pink (`#EC4899`), and Amber (`#F59E0B`) accents against off-white backgrounds. It utilizes a highly legible, modern geometric typography pairing (`Space Grotesk` for display, `DM Sans` for body).

The layout includes:
- A clean, semi-transparent Navigation bar.
- A Hero section featuring abstract floating shapes, a CSS-animated "phone mockup" simulating a Pomodoro timer interface (with a custom spinning progress ring), and floating UI cards representing app notifications.
- A Features grid with lifted hover states and spinning emoji icons.
- A dark-mode "Stats/Social Proof" section with bold typography.
- A Student Testimonials grid.
- A Pricing tier grid highlighting a "Most Popular" premium option.
- A clear, focused CTA section for app downloads.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use snappy, precise `cubic-bezier(0.175, 0.885, 0.32, 1.275)` timings tailored for a productivity tool:
- `.ease-slide-up-stagger` (Configured for a snappy, precise vertical entrance).
- `.ease-pop-in` (For a quick, energetic reveal on stat boxes).
- `.ease-hover-lift` / `.ease-hover-bounce` (For interactive, tactile button feedback).
- `.ease-wiggle` (Applied to floating notification cards to draw attention playfully).
- `.ease-hover-tilt` (Applied to the phone mockup to make it interactive on mouseover).
- `.ease-spin-slow` (A custom continuous CSS animation simulating the active timer progress ring).
- `.ease-pulse` (Applied to the active "Pause" button on the timer mockup to indicate state).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the clean color palette to the rounded UI elements and the abstract CSS illustrations (phone notch, timer circle, UI buttons). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the snappy, focused animations. No JavaScript is required for this template.
