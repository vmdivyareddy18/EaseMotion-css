Hanging Punctuation

A typographic refinement using the CSS hanging-punctuation property to push quotes outside the text margin, aligning the text body perfectly.

⚠️ Native support is currently limited to Safari. A seamless fallback using negative text-indent is provided for Chrome/Edge/Firefox!

Standard (Without Hanging)

"Design is not just what it looks like and feels like. Design is how it works."

Notice how the quote mark pushes the first letter 'D' inwards, breaking the vertical alignment.

With Hanging Punctuation

"Design is not just what it looks like and feels like. Design is how it works."

The quote mark hangs in the gutter, allowing the 'D' to perfectly align with the left margin.
