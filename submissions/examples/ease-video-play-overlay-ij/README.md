# Ease Video Play Overlay Component

An interactive, responsive CSS-animated video overlay component featuring a smooth pulsing play button on interaction (hover/focus).

## Features
- **Pure CSS Animation:** Uses `@keyframes` for a lightweight, high-performance pulse effect.
- **Accessible & Interactive:** Triggers flawlessly on mouse hover and keyboard focus (`:focus-within`).
- **Fully Customizable:** Easily change sizes, colors, and speeds using CSS root variables.
- **Responsive:** Fits perfectly inside any flexible video card or wrapper element.
