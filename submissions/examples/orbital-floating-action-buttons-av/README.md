# Orbital Floating Action Buttons

A modern **CSS-only Orbital Floating Action Button (FAB)** showcase where secondary action buttons smoothly expand around a primary floating button. This component is lightweight, responsive, and requires no JavaScript, making it suitable for dashboards, mobile applications, productivity tools, and admin panels.


## Features

- Pure HTML & CSS
- Smooth orbital expansion animation
- Circular floating action menu
- Four customizable action buttons
- Responsive layout
- No JavaScript required
- No external frameworks
- Easy to integrate into any project


## File Structure

orbital-floating-action-buttons-av/
├── demo.html
├── style.css
└── README.md


## Usage

1. Open `demo.html` in your browser.
2. Hover over the **main floating action button (+)**.
3. Watch the secondary action buttons smoothly expand in four directions.
4. Customize the icons, colors, spacing, and animation duration in `style.css` to match your project's design.
