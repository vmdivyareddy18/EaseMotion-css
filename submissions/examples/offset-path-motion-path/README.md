# Offset Path (Motion Path)

Demonstrates CSS `offset-path` for moving elements along defined geometric paths — circles, polygons, and custom SVG paths. Also includes a straight vs curved racing-track comparison.

## Features

- Circle path with an orbiting indigo dot and dashed guide circle.
- Polygon path following a hexagonal shape.
- Custom cubic bezier path using `path()` syntax.
- Racing track: straight slide vs curved motion-path side by side.
- Colour-coded movers for each path type.

## Topics Covered

- `offset-path` with circle, polygon, and path values
- `offset-distance` animation for progress along a path
- Comparison of straight linear movement vs curved paths
- Combining `offset-path` with existing animation properties
