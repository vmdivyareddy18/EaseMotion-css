# SCSS Utility: CSS Container Query Wrapper Classes

Part of the **EaseMotion CSS** design ecosystem. This utility module abstracts native CSS Container Queries into seamless, robust, and reusable SCSS mixins and structural utility classes. 

Unlike traditional Media Queries which respond to the global browser viewport, Container Queries style layout targets directly based on the structural size constraints of their immediate wrapper/parent component.
