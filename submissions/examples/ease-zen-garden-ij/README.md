# Ease Zen Garden

A CSS animation of a zen garden scene with sand texture, stones, and spreading ripple effects.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.garden` element.

## Custom CSS Variables

| Variable           | Default   | Description                   |
|--------------------|-----------|-------------------------------|
| `--sand-color`     | `#d4a574` | Base color of the sand garden |
| `--stone-color`    | `#888`    | Color of the garden stones    |
| `--ripple-duration`| `3s`      | Duration of one ripple cycle  |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
