# Notification Toast / Snackbar

A toast notification system with three variants: success (green), error (red), and info (indigo). Toasts slide in from the right and auto-dismiss after 4 seconds. Each toast has a close button for manual dismissal. A fixed container stacks toasts at the bottom-right corner.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click any trigger button to display a toast notification.

## Accessibility notes

Close buttons are `<button>` elements for keyboard activation. Reduced motion skips the slide animations and fades directly.
