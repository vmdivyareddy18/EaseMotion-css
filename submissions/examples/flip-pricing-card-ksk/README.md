# Animated 3D Flip Pricing Card (`flip-pricing-card-ksk`)

1. What does this do?  
Displays interactive pricing cards that flip 180 degrees in 3D space on hover using CSS animations to reveal features and Call-to-Action buttons.

2. How is it used?  
Use the card wrapper `.flip-card-container` containing a `.flip-card-inner` body with both `.flip-card-front` and `.flip-card-back` face layouts.

3. Why is it useful?  
It shows developers how to achieve rich, hardware-accelerated 3D micro-animations without JavaScript, enhancing dashboard conversion elements.
