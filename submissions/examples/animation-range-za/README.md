# CSS Animation Range Scroll Keyframes

A CSS demo of animation-range controlling which scroll timeline segment maps to keyframe progress in scroll-driven animations.

## Files

- `demo.html` - Cards with different animation-range values
- `style.css` - animation-range, animation-timeline, scroll-driven animation

## Usage

Open `demo.html` and scroll to see each card animate at different ranges.
