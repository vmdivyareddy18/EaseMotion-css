# CSS Top Layer Stacking with Dialog and Popover

## What does this do?

Demonstrates the CSS Top Layer — a dedicated stacking layer above all other content for `<dialog>` elements and `popover` attributes. Shows stacked dialogs, popover stacking order, and `::backdrop` pseudo-element styling.

## Key Features

- `<dialog>` elements render in the top layer
- `popover` attribute elements also use the top layer
- `::backdrop` pseudo-element styling for each layer
- Multiple stacked dialogs demonstrate top-layer ordering
- Combined dialog + popover interaction
- Custom backdrop animations (fade, blur)

## Preview

Open `demo.html` in a supporting browser (Chrome 90+, Edge 90+).
