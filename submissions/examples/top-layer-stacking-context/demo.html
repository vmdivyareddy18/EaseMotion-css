Top Layer Stacking

Dialogs and popovers render in the top layer, above all other content.

<dialog> Element

Modal dialogs render in the top layer, above all page content. Each dialog gets its own ::backdrop.

Popover API

Popovers also live in the top layer. When multiple top-layer elements are open, later elements stack on top.

::backdrop

Each top-layer element has a ::backdrop pseudo-element that can be styled with backgrounds, blur, and animations.
