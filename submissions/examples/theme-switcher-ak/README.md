# Live Theme Switcher

## What does this do?
A live theme switcher with 5 color themes (Dark, Light, Ocean, Forest, Sunset) using CSS custom properties. Theme selection persists in localStorage with instant live preview on a sample card.

## How is it used?
Open demo.html in a browser. Click any theme button to instantly see colors update across the page. The selected theme is saved and restored on reload.

## Why is this useful?
This demonstrates CSS custom properties for theming, dynamic stylesheet swapping via JS, and localStorage persistence. Useful for adding user-configurable themes to any web application.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser and switch between themes.
