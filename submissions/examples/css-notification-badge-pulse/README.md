# CSS Notification Badge with Pulse

A lightweight notification badge component built using pure HTML and CSS.

## Features

- Notification badge
- Pulse animation
- Hover effect
- Responsive design
- Pure HTML & CSS
- No JavaScript
- Beginner friendly

## Folder Structure

```text
css-notification-badge-pulse/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

## Browser Support

Works in all modern browsers.
