# Animated Subscription Renewal Countdown

## Overview

Animated Subscription Renewal Countdown is a lightweight CSS animation that simulates subscription renewal reminders using a pulsing renewal icon and countdown timer.

## Features

- CSS-only animation
- Renewal reminder effect
- Countdown timer animation
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="renewal-countdown">
  <span class="renewal-icon">🔄</span>
  <span class="timer">03:00</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
