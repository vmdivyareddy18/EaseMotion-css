# ease-product-quick-view

**Closes #13768**

Hover-to-reveal quick view button on product cards with a scale-in modal animation.

## Acceptance Criteria

- [ ] `.product-card` reveals `.quick-view-btn` on hover with fade + scale
- [ ] Modal opens on button click with a scale(0)→scale(1) transition
- [ ] Dark themed product grid with image placeholder, title, and price
- [ ] Clicking overlay or Close dismisses the modal

## Files

- `style.css` – Card hover, quick view button, modal scale animation
- `demo.html` – Grid of 3 product cards with interactive modal
- `README.md` – Issue documentation
