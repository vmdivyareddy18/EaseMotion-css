# Animated Expense Tracker Dashboard

A modern finance dashboard featuring animated budget cards, expense tracking, spending progress bars, and transaction history.

## Features

- Budget summary cards
- Animated amount counters
- Spending category progress bars
- Recent transaction list
- Hover interactions
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

- Budget values
- Spending categories
- Transactions
- Colors
- Animation timing

## Use Cases

- Personal finance apps
- Budget planners
- Expense trackers
- Fintech dashboards
- Savings management tools
