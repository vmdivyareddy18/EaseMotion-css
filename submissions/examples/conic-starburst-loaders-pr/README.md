# Conic-Gradient Starburst Animated Loaders

### What does this do?
Generates sharp, geometric loading elements like gears, sunbursts, and mechanical tickers using abrupt, non-uniform `conic-gradient()` color-stops.

### How is it used?
Apply structural modifier classes (`.starburst-classic`, `.starburst-gear`, etc.) directly to standard elements to instantiate vector spinners.

### Why is it useful?
It introduces a mathematical approach to web loaders, using clean browser masking matrices and native rotations to replace asset-heavy SVGs or complex Canvas logic.
