# Ease Card Team Member

This demo implements a **team member profile card** with photo, name, role, bio, and social links.

## 🧠 How It Works
- Circular avatar image for profile photo.
- Name, role, and bio text displayed clearly.
- Social icon links fade in on hover.
- Card lifts slightly with a shadow effect when hovered.

## 🧩 Files
- `demo.html` — main demo with card structure.
- `style.css` — styling, hover animations, and transitions.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Hover over the card to see the lift and social link fade-in animation.
3. Replace the image and links with your own team data.

## 🧰 Technologies
- HTML5
- CSS3 (transitions, hover effects)
