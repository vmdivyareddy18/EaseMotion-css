# 🏷️ CSS Chip Motion

A collection of beautiful, pure CSS animated chips and tags for modern web interfaces.

## 🚀 Features

### Chip Styles (10 types)
1. **Solid Chips** - Gradient background with lift effect
2. **Outline Chips** - Border-only with fill on hover
3. **Gradient Chips** - Gradient with shine sweep
4. **Icon Chips** - Chips with emoji icons
5. **Removable Chips** - With animated close button
6. **Selection Chips** - Checkbox-based selection
7. **Neon Chips** - Cyberpunk neon glow
8. **Avatar Chips** - User avatars with names
9. **Animated Chips** - Bounce, pulse, shake, rotate, glow
10. **Filter Chips** - Radio button-based filters

### Animation Features
- Lift and scale effects
- Shine sweep animations
- Icon rotations
- Close button spin
- Neon pulsing
- Avatar scaling
- Continuous animations
- Selection state changes
