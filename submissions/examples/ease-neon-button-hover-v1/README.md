# Ease Neon Button Hover

## Overview

Animated neon button hover utility with multiple color variations.

## Features

- Neon glow animation
- Hover lift effect
- Green variation
- Blue variation
- Pink variation
- Smooth transitions

## Files

- demo.html
- style.css

## Usage

```html
<button class="ease-neon-btn">
Hover Me
</button>
```

## Variations

```html
<button class="ease-neon-btn blue">
Blue
</button>

<button class="ease-neon-btn pink">
Pink
</button>
```

## Why this feature

Reusable hover animation utility suitable for landing pages, dashboards, portfolio websites and interactive UI.
