# Business Landing: Bounce House Rental (`business-landing-bounce-house-20955`)

This proposal provides a complete, copy-paste ready landing page template designed for a Party Equipment Rental Business (Bounce Houses/Inflatables), built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A family party rental website requires a UI that is extremely friendly, playful, and trustworthy. This template achieves a "Jelly Pop" aesthetic using a vibrant, primary color palette (Sky Blue, Fun Orange, Bright Yellow, Grass Green) against soft, rounded borders and shadows. It uses chunky, rounded typography (`Fredoka One` and `Quicksand`) to feel approachable and fun.

The layout includes:
- A high-visibility promo bar and sticky navigation with rounded CTA buttons.
- A Hero section featuring abstract CSS clouds drifting and balloons floating upwards, with a dynamic jelly-bounce entrance animation.
- An "Equipment Catalog" grid featuring pure CSS abstract illustrations of inflatables (Bounce Castle, Water Slide, Obstacle Course) with playful hover jiggle effects.
- A "Safety & Info" split section featuring a CSS-animated orbiting badge system around a central shield.
- A "Delivery Map" section featuring a custom CSS grid pattern with pulsing delivery zones and a bouncing map pin.
- A customer reviews section.
- A high-impact booking form/availability checker enclosed in a soft-shadowed card.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use highly elastic, jelly-like `cubic-bezier(0.5, -0.5, 0.5, 1.5)` and `cubic-bezier(0.25, 1.5, 0.5, 1)` timings tailored for a party brand:
- `.ease-pop-in` (Configured for an extreme, jelly-like bottom-up scaling entrance).
- `.ease-hover-lift-bounce` (For a highly elastic hover feedback that lifts product cards and expands their shadows).
- `.ease-hover-jiggle` (Applied to the abstract CSS inflatables to make them bounce slightly when the card is hovered).
- `.ease-drift-slow` / `.ease-float-up` (Custom continuous CSS animations for the background clouds and balloons).
- `.ease-orbit` (A custom animation making the safety badges rotate around the central shield).
- `.ease-pulse-slow` / `.ease-bounce-pin` (Used to animate the delivery zones and map pin).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the friendly rounded typography to the pure CSS illustrations (inflatables, clouds, balloons, map). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the bouncy, jelly-like animations. No JavaScript is required for this template.
