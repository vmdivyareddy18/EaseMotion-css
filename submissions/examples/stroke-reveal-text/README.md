# Typographic Stroke-to-Gradient Reveal

Closes #6855

Hover effect that transitions hollow outlined text to a rich gradient fill using `clip-path` sliding from right to left.

## Class

`.em-stroke-reveal-text` — add `data-text` attribute matching the text content.

## Usage

```html
<h1 class="em-stroke-reveal-text" data-text="Your Text Here">Your Text Here</h1>
```
