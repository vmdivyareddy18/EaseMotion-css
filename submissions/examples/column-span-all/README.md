## column-span-all

Demonstrates `column-span: all` for creating full-width headings and dividers within multi-column layouts.

### Key Features

- Article layout with section headings spanning all columns
- Newsletter layout with span-all headers and divider elements
- Print-style magazine layout patterns for editorial design

### Usage Notes

`column-span: all` pulls an element out of the multi-column flow to span the full width. Only block-level elements can span. The only valid values are `all` and `none`. This is ideal for section breaks, headings, pull quotes, and dividers.
