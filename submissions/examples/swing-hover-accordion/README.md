# Swing-Hover Accordion

A modern, marketing-oriented accordion component featuring a playful "swing" hover effect.

## Why Swing-Hover?
The swing interaction adds a tactile, "springy" feel that makes landing pages feel more interactive and engaging for users.

## Customization
You can tweak the feel of the swing effect using CSS custom properties:

```css
:root {
    --swing-duration: 0.5s;
    --swing-ease: cubic-bezier(0.175, 0.885, 0.32, 1.275);
}
