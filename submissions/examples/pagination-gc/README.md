# Animated Pagination UI Example

## 1. What does this do?

This submission provides pagination controls with animated active indicators and smooth transitions between page selections. Multiple pagination styles showcase different approaches to displaying page numbers with visual feedback on user interactions.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example demonstrates various pagination patterns including numbered pages, previous/next controls, and icon-based navigation. Copy the `.pagination` markup and CSS into your project to implement page navigation with animations.

---

## 3. Why is it useful?

Animated pagination enhances user experience by providing clear visual feedback on the current page and available navigation options. Smooth active indicator animations and hover effects make navigation feel responsive and polished while maintaining accessibility.
