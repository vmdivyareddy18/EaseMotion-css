# Animated Confirmation Dialog Component

Resolves #43661.

### What does this do?
Provides a modal confirmation dialog component featuring smooth entrance animations (backdrop fade-in and card slide-up/scale) and interactive buttons.

### How is it used?
Construct the markup using `.ease-modal-overlay`, `.ease-dialog-card`, and the associated internal classes for icons and buttons. The animations are triggered automatically on render.

### Why is it useful?
Confirmation dialogs are critical for destructive actions. Adding polished micro-interactions and smooth entrance animations reduces user friction and increases the perceived quality of the UI.
