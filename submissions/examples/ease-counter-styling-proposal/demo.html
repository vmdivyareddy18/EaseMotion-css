CSS Counter Utilities

Automatic numbering utilities proposed for core/utilities.css.

These utility classes allow you to automatically number elements without writing JavaScript or relying on basic <ol> tags. Perfect for highly styled lists, steps, or documentation headers.

Custom Step Counter

Using .counter-reset on the parent and .counter-increment with a custom styled pseudo-element on the children.
