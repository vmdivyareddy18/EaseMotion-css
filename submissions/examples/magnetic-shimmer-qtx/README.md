1. What does this do?
A micro-interaction effect combining an elastic hover scale with a synchronized, angled linear shimmer overlay.

2. How is it used?
```html
<button class="shimmer-card">
  <span class="shimmer-glow"></span>
  <span class="shimmer-content">Button Text</span>
</button>
```

3. Why is it useful?
It fulfills the framework's ethos by relying entirely on native CSS transitions and layout properties, removing any need for JavaScript listeners or heavy background assets to achieve premium micro-interactions.
