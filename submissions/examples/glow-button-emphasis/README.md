# Glow Button Emphasis

## What does this do?

Creates a glowing emphasis animation for buttons using pure CSS. The glow effect pulses smoothly around the button to attract attention to important actions such as sign-ups, purchases, submissions, or featured calls-to-action.

## How is it used?

```html
<button class="glow-button">
  Get Started
</button>
```

## Why is it useful?

Important actions often need visual emphasis to guide user attention. A subtle glow animation provides a modern and engaging way to highlight buttons without overwhelming the interface. The effect is lightweight, reusable, and requires no JavaScript.
