# Spiral Accordion

A CSS-only animated accordion component with a spiral opening effect.

## Features

- Pure CSS implementation
- No JavaScript required
- Smooth spiral reveal animation
- Responsive design
- Accessible using native HTML details/summary elements

## Usage

```html
<div class="ease-spiral-accordion">

  <details>
    <summary>Title</summary>
    <p>Content goes here.</p>
  </details>

</div>
