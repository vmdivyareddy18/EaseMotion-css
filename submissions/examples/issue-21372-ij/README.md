# Penguin Waddling

1. What does this do? A CSS-only penguin that waddles side to side with a rotating body.

2. How is it used? Place .penguin inside .scene. The body is a dark rounded shape; the white belly and beak are box-shadows.

3. Why is it useful? Shows how rotate combined with translate creates a convincing waddle effect.
