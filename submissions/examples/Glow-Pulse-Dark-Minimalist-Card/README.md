# Glow Pulse Dark Minimalist Card

A responsive Dark Minimalist themed card featuring a subtle Glow Pulse animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Glow Pulse animation
- Dark minimalist interface
- Clean typography
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--accent
--background
--surface
--text
--muted
--radius
```
