# Education: Debate Club & Competitive Speaking (`education-debate-21043`)

This proposal provides a complete, copy-paste ready extracurricular education landing page designed for a Debate/Speech academy, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

An academic coaching site needs to project authority, prestige, and success. This template achieves a collegiate aesthetic using a classic serif/sans-serif font pairing (Merriweather & Open Sans), a deep navy and crimson color palette, and structured, deliberate CSS entrance animations.

The layout includes:
- A high-impact hero section with a CSS-gradient background pattern, staggered text entrances, and an overlapping statistics card.
- A 3-column academic programs grid.
- A "Trophy Case" section featuring a right-sliding animation for tournament wins.
- A coach profile gallery.
- A dual-column layout containing pricing tiers and a dark-themed tournament schedule.
- A complete, stylized enrollment form with focus-ring interactions.
- A structured, informative footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-slide-up` / `.ease-fade-up` (For smooth, academic pacing of text and card entrances)
- `.ease-slide-right` (Used specifically on the trophy case items to draw the eye laterally)
- `.ease-zoom-in` (Applied to program imagery and the schedule box to emphasize importance without chaotic movement)
- `.ease-hover-lift` (Standard interaction for primary buttons, stat blocks, and pricing cards to create a tactile feel)
- `.ease-focus-ring` (Applied to form inputs to provide accessible, animated focus states)
- `.ease-hover-text` (For clean, subtle navigation and link hovers)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, typography, and academic styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.
