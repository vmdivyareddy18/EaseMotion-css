# Component: EaseMotion Recipes

This submission implements a recipes page for common EaseMotion animation patterns for issue **#13626**.

## Description

A card grid showing 10 ready-to-use EaseMotion animation recipes for common UI patterns including fade-in on load, slide-up on scroll, bounce CTA, pulse notification, shake on error, scale-in modal, rotate-in badge, loading spinner, staggered list, and card hover lift. Each card includes a code snippet.

## Acceptance Criteria

- Grid layout with 8-10 recipe cards
- Each card has title, description, and code snippet
- Covering entrance, emphasis, and hover patterns
- Responsive grid layout
