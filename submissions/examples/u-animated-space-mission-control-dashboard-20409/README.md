# Animated Space Mission Control Dashboard

A modern mission control dashboard component built using HTML and CSS.

## Features

- Animated mission progress bar
- Live telemetry cards
- Fuel and communication status
- Crew information panel
- Mission timeline
- Responsive dashboard layout
- Hover animations

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- Space mission dashboards
- Aerospace monitoring systems
- Sci-fi interfaces
- Data visualization demos
- Educational projects
