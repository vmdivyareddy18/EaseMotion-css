# Border Reveal Accordion

## What does this do?

This component creates a pure HTML and CSS accordion with a smooth border reveal animation. It is designed with a clean dashboard-style interface and expands or collapses content without using JavaScript.

## Features

- Pure HTML & CSS (No JavaScript)
- Smooth border reveal animation
- Responsive dashboard-style layout
- Expand/Collapse accordion effect
- Easy customization using CSS custom properties

## How is it used?

```html
<div class="accordion">
    <div class="accordion-item">
        ...
    </div>
</div>
```

## Why is it useful?

This component provides a clean and interactive way to organize dashboard analytics, FAQs, reports, or similar content. It is lightweight, responsive, and easy to customize for different projects.

## Technologies Used

- HTML5
- CSS3
