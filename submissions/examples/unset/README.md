# Unset Utilities

CSS utility classes for the `unset` property.

## Usage

```html
<div class="unset-all">...</div>
```

## Classes

- `.unset-all` — all: unset;
- `.unset-color` — color: unset;
- `.unset-display` — display: unset;
- `.unset-width` — width: unset;
- `.unset-height` — height: unset;
- `.unset-margin` — margin: unset;
- `.unset-padding` — padding: unset;
- `.unset-border` — border: unset;
- `.unset-bg` — background: unset;
- `.unset-font-size` — font-size: unset;
- `.unset-font-family` — font-family: unset;
- `.unset-text-align` — text-align: unset;
- `.unset-position` — position: unset;
- `.unset-overflow` — overflow: unset;
- `.unset-float` — float: unset;
- `.unset-clear` — clear: unset;
- `.unset-z-index` — z-index: unset;
- `.unset-cursor` — cursor: unset;
- `.unset-opacity` — opacity: unset;
- `.unset-visibility` — visibility: unset;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
