# CSS Image Filter Showcase Gallery

A gallery demonstrating eight CSS filter functions (blur, brightness, contrast, saturate, sepia, hue-rotate, invert, opacity) applied to gradient images. Hover to reveal the original unfiltered version.

## Files

- `demo.html` - Eight filter demo cards in a grid
- `style.css` - Filter classes, hover reset, aspect-ratio grid

## Usage

Open `demo.html` and hover over filter cards to see the original image without the filter.
