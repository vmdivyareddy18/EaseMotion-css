# Interactive Developer Portfolio Hero Component

A modern and responsive **Developer Portfolio Hero** built using pure HTML and CSS. This component features a clean glassmorphism-inspired profile card, animated hero section, technology badges, call-to-action buttons, skill progress bars, and responsive design.

---

## ✨ Features

- Modern developer portfolio hero section
- Glassmorphism profile card
- Animated introduction section
- Availability status indicator
- Technology stack badges
- Social media links
- Call-to-action buttons
- Developer statistics cards
- Skill progress bars
- Contact information section
- Smooth hover animations
- Responsive layout
- Pure HTML and CSS

---

## 📂 Folder Structure

```text
interactive-developer-portfolio-hero/

```

## 🚀 Usage

1. Open `demo.html` in your browser.
2. Ensure `style.css` is located in the same folder.
3. Replace the sample content with your own profile details.
4. Customize colors, typography, and animations as needed.
