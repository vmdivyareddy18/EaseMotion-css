# Animated Feature Showcase Card Component

## Description

A CSS-only feature card component with smooth hover animations, icon effects, glowing borders, and responsive layouts.

---

## Features

- Animated card lift effect
- Interactive icon rotation and scaling
- Glowing border hover effect
- Modern rounded card design
- Responsive grid layout
- Lightweight CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.feature-section` – Main feature area
- `.feature-container` – Responsive card wrapper
- `.feature-card` – Individual feature card
- `.feature-icon` – Animated feature icon

---

## Accessibility

Supports `prefers-reduced-motion` to disable animations for users who prefer reduced motion.

---

## Use Cases

- Product landing pages
- Service showcase sections
- Portfolio websites
- Business websites
- Skill presentation areas
