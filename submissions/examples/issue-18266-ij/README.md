# Step Progress Indicator (Issue #18266)

A 4-step horizontal progress indicator. Completed/current steps are filled, future steps are empty. Transitions between steps animate smoothly. Prev/Next buttons to navigate.

## Usage

Open `demo.html` in a browser. Use **Prev** and **Next** buttons to move through steps.

## Files

- `demo.html` — HTML5 page with step indicator and JS navigation
- `style.css` — Dark theme styles with CSS transitions
