Modal Hash Jump Prevention

Pure CSS modals use hash anchors (e.g. #modal) which scroll the page to top when cleared naively. Test the naive close vs the smooth close below.

Naive Close Modal

Clears hash using window.location.hash = ''. Watch the page jump violently back to the top of the viewport.

Smooth Close Modal (Remediated)

Clears hash using history.replaceState(). The modal closes while maintaining your scroll offset.
