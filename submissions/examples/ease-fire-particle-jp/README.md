# Ease Fire Particle

A premium CSS fire particle animation featuring rising embers, flame motion, glowing particles, and atmospheric fire effects.

## Features

- Pure HTML & CSS
- Rising Ember Particles
- Flame Motion Animation
- Glowing Fire Effects
- Multiple Particle Layers
- Responsive Design
- No JavaScript Required
