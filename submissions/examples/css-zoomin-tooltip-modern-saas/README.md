# CSS Zoom-In Tooltip – Modern SaaS Layout

## Overview

A responsive Modern SaaS tooltip component built entirely with HTML and CSS. The component features a smooth zoom-in animation, modern SaaS feature cards, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-In tooltip animation
- Modern SaaS interface
- Responsive feature cards
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--background`
- `--surface`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--zoom-scale`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any **Discover** button to display the animated Zoom-In Tooltip. Customize colors, spacing, animation timing, zoom scale, and shadows using the CSS variables defined in `style.css`.
