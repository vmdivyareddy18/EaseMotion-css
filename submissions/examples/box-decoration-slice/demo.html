Side by Side Comparison

Slice

Box decorations are sliced at each line break edge, like cutting a cake into pieces.

Clone

Box decorations are cloned at each line break edge, like each line is its own box.

Use box-decoration-slice: clone when you want each wrapped line to have full border-radius, padding, and background independently.
