# Ease Command Shortcuts Panel

## Overview
A CSS-only command shortcuts panel for developer tools and productivity applications.

## Features
- Responsive layout
- Animated shortcut badges
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` directly in your browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
Provides a lightweight, reusable, animation-first UI component suitable for IDE dashboards, admin panels, and SaaS applications.
