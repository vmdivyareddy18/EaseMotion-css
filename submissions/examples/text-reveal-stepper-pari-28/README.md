# Ease Text Reveal Stepper

A responsive portfolio-inspired stepper component featuring a smooth text reveal animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Accessible structure
- CSS custom properties
- Sequential text reveal animation
- Supports `prefers-reduced-motion`
- No JavaScript required

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` directly in a browser.

Customize the CSS variables in `style.css`:

```css
--accent
--ease-duration
--ease-easing
```

## Accessibility

- Semantic heading hierarchy
- Motion reduction support
- High color contrast
- Keyboard-friendly static content

## Fits EaseMotion CSS

This component demonstrates a reusable text reveal stepper suitable for portfolio, onboarding, roadmap, and timeline interfaces while remaining lightweight and dependency-free.
