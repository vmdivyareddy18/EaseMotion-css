# Vertical Timeline

Vertical timeline layout with alternating left/right content cards, central line, and scroll-triggered fade-in animation.

## Features

- Alternating left/right card layout with central timeline line
- Colored dot markers (blue / purple alternating)
- Date labels with uppercase styling
- Category tags per timeline item
- Staggered fade-in-up animation on load
- Hover effects with glow on cards
- Responsive: collapses to single-column on mobile
- `prefers-reduced-motion` support

## Files

- `demo.html` — Timeline with 6 sample events
- `style.css` — Self-contained CSS (162 lines)
