# CSS Form Builder Component

## What does this do?
A comprehensive CSS form component library with input fields, selects, checkboxes, radio buttons, toggle switches, textareas, file upload, and validation states.

## How is it used?
```html
<form class="frm-form">
  <div class="frm-group">
    <label class="frm-label">Name</label>
    <input type="text" class="frm-input" placeholder="Enter name">
  </div>
  <div class="frm-actions">
    <button class="frm-btn frm-btn-primary">Submit</button>
  </div>
</form>
```

## Why is it useful?
Provides a complete form styling system for any web application. Includes all common input types, interactive states, validation feedback, and responsive grid layout. Pure CSS with custom-styled form controls.
