# Esports Tournament Dashboard

## Overview
Create a competitive gaming management UI.

This project implements the Phase 111 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Tournament brackets**: Built with glassmorphism and CSS grid/flexbox.
- **Player cards**: Built with glassmorphism and CSS grid/flexbox.
- **Leaderboard section**: Built with glassmorphism and CSS grid/flexbox.
- **Match statistics**: Built with glassmorphism and CSS grid/flexbox.
- **Responsive design**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
