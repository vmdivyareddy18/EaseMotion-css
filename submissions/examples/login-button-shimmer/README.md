# Login Button Shimmer

A sign-in button with a passing shimmer on hover or keyboard focus.

## Files

- `demo.html` - login button markup.
- `style.css` - button lift, shimmer sweep, focus-visible state, and reduced-motion support.

## Highlights

- Keeps the shimmer decorative with a pseudo-element.
- Mirrors hover behavior on keyboard focus.
- Removes motion for reduced-motion users.
