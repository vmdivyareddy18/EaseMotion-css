# Noise Texture Background Utility

## What does this do?

Adds reusable CSS noise and grain texture overlays that create depth and visual richness in interfaces.

## How is it used?

```html
<section class="noise-bg">
    ...
</section>
```

## Why is it useful?

Subtle texture can reduce the flat appearance of solid colors while maintaining performance. It complements EaseMotion CSS by providing visually engaging backgrounds that pair naturally with animated UI components.
