# 🎵 Music Maker

A grid-based music sequencer that lets you create melodies by clicking cells. Compose patterns, play them back, and export your creations!

## 🎮 How to Use

1. **Click cells** on the grid to add notes
2. **Click Play** to hear your composition
3. **Switch instruments** for different sounds
4. **Adjust BPM** with arrow keys
5. **Export** your composition as a text file!

## 🎹 Instruments

| Instrument | Sound |
|------------|-------|
| Piano | Sine wave (smooth) |
| Synth | Square wave (retro) |
| Sawtooth | Sawtooth wave (brass-like) |
| Triangle | Triangle wave (soft) |

## ⌨️ Controls

| Action | Input |
|--------|-------|
| Play/Pause | Click Play or press `Space` |
| Stop | Click Stop or press `S` |
| Clear Grid | Click Clear or press `C` |
| Switch Instrument | Click Switch or press `I` |
| Export Composition | Click Export or press `E` |
| Increase BPM | `↑` or `+` |
| Decrease BPM | `↓` or `-` |
| Add Beats | `→` |
| Remove Beats | `←` |

## ✨ Features

- 🎵 **Grid-Based Sequencer** – 8 notes × 16 beats
- 🎹 **4 Instruments** – Piano, Synth, Sawtooth, Triangle
- 🎛️ **BPM Control** – Adjustable tempo (40-240 BPM)
- 🔊 **Real-time Playback** – Hear your composition instantly
- 💾 **Export** – Save your composition as a text file
- 🎨 **Visual Feedback** – Highlighted cells during playback
- 📱 **Responsive Design** – Works on desktop and mobile

## 🎵 Notes

The sequencer uses the C major scale:
- C4, D4, E4, F4, G4, A4, B4, C5

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid, responsive design)
- Vanilla JavaScript (Web Audio API)
