# Pulse Number Counter Component

A CSS-only animated number counter component created for EaseMotion CSS.

## Overview

The Pulse Number Counter adds a smooth pulse animation behind important
statistics and metrics. It is suitable for analytics dashboards,
landing pages, and business websites.

## Features

- Pure HTML and CSS
- No JavaScript required
- Smooth pulse animation effect
- Responsive layout
- Accessible semantic structure
- Lightweight and reusable
