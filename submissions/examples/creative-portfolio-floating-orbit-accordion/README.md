# Creative Portfolio Floating Orbit Accordion

A modern **Pure CSS Floating Orbit Accordion** designed for **Creative Portfolio** layouts. The component features subtle floating orbit animations, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Floating Orbit animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Creative portfolio inspired design
- Glassmorphism styling
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:4.5s;
    --easing:ease-in-out;
    --primary:#8b5cf6;
    --secondary:#06b6d4;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Glassmorphism appearance

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
