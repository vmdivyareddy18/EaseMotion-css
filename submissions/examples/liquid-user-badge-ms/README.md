# Liquid User Badge

Resolves #42435.

### What does this do?
Adds a Liquid User Badge component to EaseMotion CSS. It features a space-inspired glassmorphic card (`backdrop-filter`) with an organic, continuously morphing "liquid" border around the avatar using a custom `@keyframes liquidMorph` combined with `ease-pulse` and `ease-hover-scale`.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Hover over the badge to watch the liquid animation speed up dynamically.

### Why is it useful?
Provides a highly aesthetic, premium user profile badge that uses organic fluid shapes (border-radius morphing) instead of static circular borders, perfectly fitting space or sci-fi themes.
