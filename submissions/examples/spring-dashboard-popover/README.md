# CSS Spring Physics Popover

A responsive dashboard popover built using only HTML and CSS with a spring-inspired animation.

## Features

- Pure HTML & CSS
- Responsive dashboard design
- Spring physics animation
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Files

```
spring-dashboard-popover/
│── demo.html
│── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
