# Backdrop-Filter Glassmorphism

Explores `backdrop-filter` for frosted glass effects with animated background orbs, multiple filter combinations, and an interactive draggable panel.

## Features

- Frosted glass cards with configurable blur and opacity via custom properties
- Multiple backdrop-filter combinations (brightness, contrast, saturate, hue-rotate)
- Draggable glass panel that demonstrates real-time backdrop-filter updates
- Animated floating gradient orbs in the background

## Usage

Open `demo.html` in a modern browser. Drag the glass panel to see the backdrop-filter effect respond to different background areas. Fully self-contained with no external dependencies.
