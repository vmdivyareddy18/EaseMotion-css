# Ease Prism Border Card

A modern glass card featuring an animated prism-style border created using pure HTML and CSS.

## Features

- Animated gradient border
- Pure HTML & CSS
- Responsive
- Lightweight
- Reusable component

## Structure

```
ease-prism-border-card-n1/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any browser.

## Why EaseMotion CSS

Showcases a reusable animated border effect suitable for cards, pricing sections, dashboards, and feature highlights.
