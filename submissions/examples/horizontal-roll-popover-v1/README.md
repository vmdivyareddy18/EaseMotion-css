# CSS Horizontal Roll Popover

A modern SaaS-inspired horizontal roll popover built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth horizontal roll animation
- Responsive design
- Glassmorphism effect
- Keyboard accessible
- CSS custom properties
- Hover and focus interactions

## Technologies

- HTML5
- CSS3

## Folder Structure

```text
horizontal-roll-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser to see the component in action.
