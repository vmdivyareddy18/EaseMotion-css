# Grid Fill Animate

A grid that fills sequentially with animated cells.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--grid-bg` | `#1a1a2e` | Background color |
| `--fill-color` | `#4ecdc4` | Color of filled cells |
| `--cell-size` | `40px` | Size of each cell |
| `--fill-delay` | `0.05s` | Delay between each cell fill |
