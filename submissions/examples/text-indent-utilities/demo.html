Typography Utilities

Text Indentation

A lightweight collection of utility classes to control the horizontal indentation of the first line of text blocks, including positive, negative, and hanging alignments.

EaseMotion-css provides developers with atomic typography utility classes that optimize writing and structuring web interfaces. By using these text indentation utility classes, you can style paragraphs to mirror professional editorial layout standards, enhancing overall readability and readability across various screen resolutions. Select different classes above to see the indentation smoothly animate in real-time.

Hanging Indent (Small)

-1rem
.hanging-indent-example .ease-indent-neg-sm

A hanging indent is created by offsetting a positive left padding with a negative text-indent. Here, the first line hangs out to the left by 1rem, aligning the rest of the text.

Hanging Indent (Medium)

-2rem

This layout uses a medium hanging indent of 2rem. Hanging indents are extremely useful for bibliography listings, footnotes, glossaries, and bulleted lists.
