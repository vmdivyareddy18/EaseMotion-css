# Animated Testimonials Carousel

A clean, modern, and fully responsive **Testimonials Carousel** built entirely without JavaScript. This example demonstrates how to use the CSS radio-button hack to create smooth, horizontal sliding transitions between customer reviews, combined with **EaseMotion CSS** for beautiful staggered entrance animations.

## 🚀 Features

- **Zero JavaScript**: Fully interactive using only HTML and CSS (radio button hack).
- **Staggered Entrance Animations**: Uses `ease-fade-in-up` and `ease-delay-*` so the header and carousel slide in sequentially on page load.
- **Smooth Horizontal Sliding**: Content slides smoothly left/right using CSS `transform: translateX()` transitions.
- **Interactive Navigation Dots**: Clickable dots that highlight the active slide with a scale and color transition.
- **Clean, Modern Design**: Features a large quote icon, italicized testimonial text, and author avatars with a soft shadow card.
- **Fully Responsive**: Adjusts font sizes and padding for mobile devices.
