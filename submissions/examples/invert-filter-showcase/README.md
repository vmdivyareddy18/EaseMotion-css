# Invert Filter Showcase - EaseMotion CSS

A beautiful, interactive component showcase presenting the invert filter transition classes.

## Features Included

- `.ease-filter-invert`: Applies `filter: invert(100%) !important` to an element.
- `.ease-filter-invert-0`: Applies `filter: invert(0%) !important` to an element.
- Transition effects showing how invert styling integrates with interactive state designs.

## How to Run

1. Open `demo.html` directly in any modern web browser.
2. Hover over the cards to observe active color inversion states and transitions.

## Why is it useful?

Color inversion transitions are helpful for immediately flipping color palettes for widgets (e.g. creating rapid dark mode mockups, toggling state visual cues, or styling high-contrast dashboard states) natively through utilities.
