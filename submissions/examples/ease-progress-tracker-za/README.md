# Progress Tracker

## What does this do?
CSS progress tracking components with multi-step wizards, animated skill bars, milestone timelines, goal thermometers, and circular progress indicators.

## How is it used?
```html
<div class="pt-stepper"><div class="pt-step pt-complete"><div class="pt-circle">✓</div><span>Done</span></div><div class="pt-line pt-line-done"></div><div class="pt-step pt-active"><div class="pt-circle">2</div><span>Current</span></div></div>
```

## Why is it useful?
Provides comprehensive progress visualization without JavaScript. Five distinct tracking patterns. Animated bar fills on load. Ideal for onboarding flows, dashboards, project trackers, and portfolio skill sections.
