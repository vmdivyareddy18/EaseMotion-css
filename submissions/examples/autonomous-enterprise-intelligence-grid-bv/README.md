# Autonomous Enterprise Intelligence Grid

An autonomous enterprise intelligence grid dashboard built with EaseMotion CSS tokens. Tracks autonomy score, process automation rates, intelligence node status, active workflows, AI model performance, and system events in a sky-blue dark theme.

## Features

- **Stat cards** — autonomy score, active processes, intelligence nodes, decisions/day
- **Intelligence nodes** — 6 AI nodes (predictive analytics, NLP, computer vision, recommendation, anomaly detection, legacy bridge) with active/learning/degraded/offline status
- **Process automation** — 5 processes (data ingestion, report gen, customer triage, inventory, quality control) with color-coded bars
- **Active workflows** — 5 workflows with running/done/paused/failed status badges
- **AI model performance** — response time, throughput, model drift, retraining freq, GPU util, confidence
- **System events** — info/warn/err level events with timestamps
- **Enterprise grid overview** — active nodes, workflows, decision accuracy, models deployed, data processed, active alerts
- Responsive 12-column grid with sky-blue dark (default) and light mode support
- Reduced-motion compliant
