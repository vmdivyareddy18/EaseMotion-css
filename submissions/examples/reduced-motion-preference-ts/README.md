# Reduced Motion Preference Showcase

1. What does this do? Demonstrates a normal hover lift with a reduced-motion fallback for users who prefer less animation.
2. How is it used? Apply the showcased class names to the matching HTML pattern in the demo.
3. Why is it useful? It gives EaseMotion CSS a focused, reusable motion pattern that stays small and accessible.

To test the fallback, enable reduced motion in the operating system or browser accessibility settings and reload the demo.
