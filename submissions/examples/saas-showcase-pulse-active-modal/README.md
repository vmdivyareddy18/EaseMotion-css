# SaaS Showcase Pulse-Active Modal

A modern **Pure CSS Pulse-Active Modal** designed for **SaaS Showcase** interfaces. The component provides a smooth pulse animation, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Pulse-Active animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- SaaS-inspired design
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:1s;
    --easing:ease-in-out;
    --scale:1.04;

    --primary:#6366f1;
    --secondary:#8b5cf6;
}
```

These variables allow you to customize:

- Animation duration
- Animation easing
- Hover scale
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in your browser.

No JavaScript or build tools are required.
