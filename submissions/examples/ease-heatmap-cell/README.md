# Heatmap Cell – Color intensity animates on data change

> A production-ready heatmap cell component where color intensity animates smoothly when data changes. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Animated color transitions** – Smooth interpolation between values
- ✅ **4 color palettes** – Default, Inferno, Viridis, Plasma
- ✅ **Click selection** – Select cells to view values
- ✅ **Configurable grid size** – Small, Medium, Large
- ✅ **Toggle labels** – Show/hide cell values
- ✅ **Real-time stats** – Cell count, range, average
- ✅ **Keyboard shortcuts** – Space, R, L
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
