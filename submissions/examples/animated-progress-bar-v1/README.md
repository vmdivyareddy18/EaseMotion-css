# Animated Progress Bar

A responsive animated progress bar component built using pure HTML and CSS.

## Features

- Animated fill effects
- Gradient progress styling
- Percentage indicators
- Responsive layouts
- Smooth transitions and animations
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated progress bar component.

## Files

- `demo.html` → Progress bar structure
- `style.css` → Progress bar styles and animations
- `README.md` → Documentation
