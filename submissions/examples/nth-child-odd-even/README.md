# Nth Child Odd & Even

## What does this do?
Demonstrates `:nth-child(odd)` and `:nth-child(even)` for alternating row colors — striped tables, zebra lists, and pattern-based item coloring.

## How is it used?
Apply to any repeated element for alternating visual patterns:

    <tr class="zebra-row">
      <td>Item</td>
    </tr>

## Why is it useful?
Alternating row colors improve readability in data-heavy tables and lists. Using `:nth-child` keeps the pattern purely CSS-driven without server-side logic or JavaScript.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
