# Typing Loader – Text appears letter by letter with cursor

> A production-ready typing loader component that displays text letter by letter with a blinking cursor animation, customizable speed, color, phrases, and keyboard controls. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Letter-by-letter typing** – Text appears character by character
- ✅ **Blinking cursor** – Classic terminal-style cursor
- ✅ **Customizable speed** – Adjust typing speed
- ✅ **Multiple phrases** – Pre-defined and custom phrases
- ✅ **Loop mode** – Continuous typing cycle
- ✅ **Pause/Resume** – Control the animation
- ✅ **Skip** – Complete instantly
- ✅ **Random phrase** – Randomize loading messages
- ✅ **Keyboard shortcuts** – Space (toggle), R (reset), S (skip)
- ✅ **6 color variants** – Default, Green, Danger, Warning, Purple, Pink
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
