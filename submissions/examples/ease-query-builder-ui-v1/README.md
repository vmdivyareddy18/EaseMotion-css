# Ease Query Builder UI

## Overview

Ease Query Builder UI is a reusable interface component inspired by database and analytics tools. It provides a visual representation of building queries using connected filter blocks with smooth animations.

---

## Features

- 🔍 Visual query builder
- 🔗 Animated connectors
- 🎨 Interactive filter blocks
- 🚀 Action buttons
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Field selector
- Operator selector
- Value selector
- Animated connectors
- Action buttons

---

```html
<div class="builder">
...
</div>
```
