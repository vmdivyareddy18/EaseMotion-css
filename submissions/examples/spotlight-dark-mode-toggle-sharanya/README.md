# Spotlight Dark Mode Toggle

A reusable spotlight-style dark mode toggle component built with pure HTML and CSS for the EaseMotion CSS library.

## ✨ Features

- Spotlight-inspired dark mode toggle
- Smooth sliding animation
- Soft glow effects for both light and dark modes
- Pure HTML and CSS (No JavaScript)
- Responsive design
- Easy to customize and integrate
- Accessible and lightweight

## 🚀 Usage

Include the HTML structure in your project.

```html
<div class="spotlight-toggle">
    <!-- Toggle Component -->
</div>
```

Link the stylesheet.

```html
<link rel="stylesheet" href="style.css">
```
