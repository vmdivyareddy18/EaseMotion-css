# Ease Hypercube Loader

## Overview

A futuristic loading animation featuring synchronized glowing cubes that morph in a rhythmic sequence using only HTML and CSS.

## Features

* Animated cube morphing
* Staggered motion
* Gradient glow
* Glassmorphism container
* Responsive layout
* Pure HTML & CSS

## Folder Structure

```text
ease-hypercube-loader-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

* Chrome
* Firefox
* Edge
* Safari

## Responsive Design

Optimized for desktop, tablet, and mobile screens.

## Why it fits EaseMotion CSS

Demonstrates reusable loading animations with geometric motion, clean CSS architecture, and modern visual effects suitable for dashboards, splash screens, and AI-inspired interfaces.
