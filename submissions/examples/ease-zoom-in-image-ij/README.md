# Zoom In Image

Image gallery with smooth zoom-in hover effects, overlay captions, and click-to-expand modal viewer.

## Features

- Hover triggers scale transform on images with overlay and info slide-up
- Cursor changes to `zoom-in` on image cards
- Responsive grid layout with multiple images
- Click opens a full-size image modal with close on background click or Escape key
- CSS custom properties for full control

## CSS Custom Properties

| Property             | Default           | Description            |
|----------------------|-------------------|------------------------|
| `--zii-duration`     | 0.4s              | Transition duration    |
| `--zii-image-scale`  | 1.15              | Scale factor on hover  |
| `--zii-overlay-bg`   | rgba(0,0,0,0.65)  | Overlay background     |
| `--zii-info-color`   | #ffffff           | Info text color        |
| `--zii-radius`       | 12px              | Card border radius     |

## Usage

Open `demo.html` in any modern browser.
