# Mac-Style Dock (#36804)

A macOS-inspired dock with a magnification hover effect. Icons scale up smoothly on hover, with neighboring items parting to make room.

## CSS Custom Properties

| Property           | Default  | Description                         |
| ------------------ | -------- | ----------------------------------- |
| `--dock-bg`        | `#2c2c2e`| Background color of the dock bar    |
| `--icon-size`      | `48px`   | Base size of each dock icon         |
| `--magnify-scale`  | `1.8`    | Scale multiplier on hover           |
| `--hover-duration` | `0.3s`   | Duration of the magnification transition |

## Interactive Controls

Use the live variable panel below the dock to tweak colors, sizes, and timing in real time.
