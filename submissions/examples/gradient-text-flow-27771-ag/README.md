# Gradient Text Flow & Animation Utility Classes

A text layout showcase detailing sunset and ocean animated color shifting text utilities.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting multi-theme headers, alignment nodes, and descriptions.
2. **`style.css`**: Text masks, background positioning keyframes, color variables, and reduced motion limits.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Verify that **Sunset Gold Shift** shifts from orange to purple continuously.
3. Verify that **Ocean Wave Shift** shifts from cyan to blue continuously.
