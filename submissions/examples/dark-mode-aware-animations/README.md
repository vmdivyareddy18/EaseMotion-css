# Dark Mode Aware Animations

A responsive animation example that automatically adapts to the user's preferred color scheme using CSS.

## Features

- prefers-color-scheme support
- Automatic dark mode detection
- Reduced glow intensity in dark mode
- Manual `.dark-mode` override class
- Pure HTML & CSS
- No JavaScript

## Folder Structure

```text
dark-mode-aware-animations/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

Dark mode will automatically apply if your operating system uses a dark theme.
