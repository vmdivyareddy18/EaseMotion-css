Orbit Accordion

A neon-inspired accordion with orbiting accents, native keyboard interaction, responsive layout, and no JavaScript.

01 What is EaseMotion CSS? A lightweight motion-focused CSS library

EaseMotion CSS provides reusable animation patterns, hover effects, transitions, and interaction ideas that can be adapted to modern interfaces without adding unnecessary dependencies.

Reusable motion Pure CSS Accessible patterns
02 How does the orbit effect work? Layered rings rotate around the active item

Each accordion marker contains a rotating pseudo-orbit built from CSS borders and transforms. Opening or focusing an item increases the glow and activates the orbital motion.

Transforms Keyframes Focus states
03 Does it require JavaScript? Native HTML manages the open and closed state

No JavaScript is required. The component uses native details and summary elements, so it remains keyboard accessible and works when opened directly in a browser.

Native controls No dependencies Direct browser demo
04 Is reduced motion supported? Orbit animation can be disabled automatically

Users who prefer reduced motion receive the same expanded content and visible states without continuous rotation, entrance movement, or animated chevrons.

Reduced motion Responsive Readable states
