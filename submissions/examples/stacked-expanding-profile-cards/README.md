# Stacked Expanding Profile Cards

A modern **CSS-only stacked profile card** component where cards overlap by default and smoothly expand when hovered.

## Features

- Pure HTML & CSS
- Smooth hover animation
- Overlapping stacked layout
- Responsive design
- No JavaScript required
- Easy to customize

## Folder Structure

```
submissions/examples/stacked-expanding-profile-cards/
├── demo.html
├── style.css
└── README.md
```

## How It Works

- Cards are initially stacked using negative margins.
- Hovering over the container separates the cards.
- Hovering over an individual card raises and enlarges it using CSS transforms.
- Uses only CSS transitions and positioning.

## Customization

You can easily modify:

- Card width
- Border radius
- Colors
- Shadow
- Hover scale
- Animation duration
- Profile images
- Typography
