Usage

<div class="ease-hover-border-color-shift">
    <!-- Content -->
</div>

<!-- Custom color -->
<div class="ease-hover-border-color-shift" 
     style="--ease-hover-border-to: #10b981;">
    <!-- Content -->
</div>

<!-- Different speeds -->
<div class="ease-hover-border-color-fast">Fast</div>
<div class="ease-hover-border-color-slow">Slow</div>

🎨 Border Color Features

  • 🎨 border-color transition on hover
  • 📐 --ease-hover-border-to color variable
  • ⚡ Speed variations (fast, normal, slow)
  • 🌈 Unlimited color options via CSS variable
  • 🎯 Works on cards, inputs, any element with border
  • 💫 Smooth CSS transitions
