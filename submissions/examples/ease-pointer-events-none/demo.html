The Magic of pointer-events

Solve the "unclickable button" bug when icons block clicks, or create ghost overlays that users can interact through.

1. The Custom Icon Button

Ever tried to click a button, but you hit the SVG icon inside it instead, breaking your Javascript event delegation? pointer-events: none on the icon fixes this instantly.
