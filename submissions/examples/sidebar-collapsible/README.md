# Sidebar with Collapsible Sections

A vertical sidebar with collapsible accordion sections. Each section header toggles its child links open and closed with a smooth max-height transition and a rotating arrow indicator. The active link is highlighted in indigo.

## EaseMotion CSS classes used

- `ease-flex` — page-level flex layout

## How to run

Open `demo.html` in a browser. Click each section header to expand or collapse its items.

## Accessibility notes

Each section header is a `<button>` element for keyboard access. The sidebar uses semantic `<nav>` and `<a>` elements. Reduced motion disables the collapse transition.
