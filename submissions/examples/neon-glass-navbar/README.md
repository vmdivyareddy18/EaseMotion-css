# Neon Glass Navigation Bar

A modern CSS-only navigation bar featuring:

- ✨ Glassmorphism design
- 💡 Neon glow hover effects
- 🎯 Animated underline indicators
- 📱 Responsive layout
- 🚀 Smooth transitions

## Preview

This example demonstrates how to build a stylish navigation component using only HTML and CSS.

Features include:

- Frosted glass navbar
- Neon glowing navigation links
- Hover underline animation
- Glowing call-to-action button
- Responsive mobile layout

## Folder Structure
```

neon-glass-navbar/
├── demo.html
├── style.css
└── README.md

````

## Usage

1. Open `demo.html` in your browser.
2. Ensure `style.css` is in the same directory.
3. Customize colors, spacing, and glow intensity as desired.

## Customization

### Change Glow Color

Replace:

```css
#00ffff
````

with your preferred accent color.

### Adjust Blur Strength

Modify:

```css
backdrop-filter: blur(14px);
```

to increase or decrease the glass effect.

### Modify Border Radius

Update:

```css
border-radius: 18px;
```

to achieve a different visual style.
