# Image Compare Slider

CSS-first before/after image comparison slider demo for EaseMotion. The example uses layered panels and a range input to reveal the comparison state.

## Files

- `demo.html` - slider markup
- `style.css` - compare frame, clipped after layer, and handle styling

## Usage

Open `demo.html` and drag the slider to compare the before and after panels.
