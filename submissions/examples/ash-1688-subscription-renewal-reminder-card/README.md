# Subscription Renewal Reminder Card

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern subscription management card that highlights upcoming renewals, billing information, and renewal progress.

## Features

- Renewal reminder badge
- Countdown display
- Billing summary section
- Auto-renew status indicator
- Animated progress bar
- Call-to-action buttons
- Floating card animation
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="renewal-card">
  Subscription Content
</div>
```

## Main Classes

```css
.renewal-card
.countdown
.billing-section
.progress-bar
.progress-fill
.renew-btn
.manage-btn
```
