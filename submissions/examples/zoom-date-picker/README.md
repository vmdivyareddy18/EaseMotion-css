# Pure CSS Zoom Date Picker Component

A clean, responsive, and accessible date picker component built entirely with CSS for EaseMotion CSS. Featuring a neumorphism-inspired interface and a smooth zoom animation on interaction, it provides a modern input experience for scheduling forms, booking systems, and event planners.

## Features

- **Pure CSS:** No JavaScript required.
- **EaseMotion Ready:** Integrates seamlessly with EaseMotion CSS utility classes.
- **Zoom Animation:** Date input smoothly scales on hover and focus.
- **Neumorphism Design:** Soft shadows and subtle depth create a modern UI.
- **Responsive Layout:** Optimized for desktop and mobile devices.
- **Accessible:** Uses the native HTML date input for keyboard and screen reader compatibility.

## Configuration Parameters

The component can be customized using the following CSS custom properties:

| Variable | Description | Default Value |
| :--- | :--- | :--- |
| `--primary` | Primary accent color | `#6366f1` |
| `--bg` | Page background color | `#e7ecf3` |
| `--surface` | Card and input background | `#e7ecf3` |
| `--text` | Primary text color | `#1f2937` |
| `--muted` | Secondary text color | `#6b7280` |
| `--radius` | Border radius for the card | `20px` |
| `--shadow-light` | Light neumorphic shadow | `-8px -8px 18px rgba(255, 255, 255, 0.9)` |
| `--shadow-dark` | Dark neumorphic shadow | `8px 8px 18px rgba(163, 177, 198, 0.55)` |

## How to Run Locally

1. Clone your fork of the repository.
2. Navigate to `submissions/examples/zoom-date-picker-amna/`.
3. Open `demo.html` in any modern web browser to interact with the component.
