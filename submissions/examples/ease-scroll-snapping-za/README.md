# Scroll Snapping Component

## What does this do?
CSS scroll-snap based horizontal galleries and vertical section snapping with mandatory and proximity variants.

## How is it used?
```html
<div class="scs-hscroll">
  <div class="scs-hitem">Item 1</div>
  <div class="scs-hitem">Item 2</div>
</div>
```

## Why is it useful?
Provides smooth snap-based scrolling without JavaScript. Great for image galleries, step-by-step sections, and horizontal content browsing. Customizable snap alignment and type via CSS variables.
