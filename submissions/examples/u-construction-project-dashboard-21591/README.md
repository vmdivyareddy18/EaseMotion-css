# Construction Project Dashboard

A responsive **Construction Project Dashboard** built using **HTML5** and **CSS3** that demonstrates construction project tracking, workforce monitoring, material management, and site progress visualization through an interactive dashboard.

---

## Features

- Responsive dashboard layout
- Construction overview cards
- Project progress tracker
- Material usage panel
- Site timeline
- Recent construction activity
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Active Projects
- Workers On Site
- Pending Tasks
- Budget Utilized

### Construction Widgets

- Project Progress
- Material Usage
- Site Timeline

### Activity

- Recent Construction Activity
- Notifications Panel

### Quick Actions

- Add Project
- Assign Worker
- Export Report

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
