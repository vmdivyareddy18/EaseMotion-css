## Summary

Create a demonstration showing how CSS custom properties can be used to customize EaseMotion animations without modifying the framework source.

## Problem

Many developers want reusable animation settings such as duration, delay, and easing.

## Proposed Solution

Add a standalone example under the `submissions/` directory demonstrating CSS variables for animation customization.

## Deliverables

- demo.html
- style.css
- README.md

No framework source files will be modified.

## Benefits

- Educational example
- Beginner friendly
- Does not affect existing framework code
