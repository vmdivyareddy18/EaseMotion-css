# Ease Glass Signal Router

## Overview

A futuristic signal routing animation featuring a glowing central hub connected to multiple endpoints using pure HTML and CSS.

## Features

- Animated routing paths
- Pulsing central hub
- Glowing endpoint nodes
- Responsive layout
- Lightweight implementation
- Pure HTML & CSS

## Folder Structure

```text
ease-glass-signal-router-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

Provides a reusable routing visualization suitable for networking dashboards, AI systems, cloud infrastructure panels, and developer tools.
