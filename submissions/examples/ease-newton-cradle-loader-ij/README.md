# Newton Cradle Loader

A pendulum swing loader inspired by Newton's cradle.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--cradle-bg` | `#2d2d44` | Arm bar color |
| `--ball-color` | `#4ecdc4` | Ball fill color |
| `--string-color` | `#666` | String line color |
| `--swing-duration` | `1.2s` | Swing cycle duration |
