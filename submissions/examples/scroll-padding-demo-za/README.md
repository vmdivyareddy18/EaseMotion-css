# Scroll Padding Container Offset

A CSS demo of the scroll-padding property on scroll containers. Adds inward offset to prevent content from being clipped, useful with scroll-snap alignment.

## Files

- `demo.html` - Two scroll containers with and without scroll-padding
- `style.css` - scroll-padding, scroll-snap-type, horizontal scroll

## Usage

Open `demo.html` and scroll each panel to see how scroll-padding adds offset.
