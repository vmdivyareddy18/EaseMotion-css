# Text Decoration Style

text-decoration-style — Solid, dashed, dotted, double underline styles

## Features
- Uses `--ease-color-primary`
- Uses `--ease-color-danger`
- Uses `--ease-color-success`
- Uses `--ease-text-base`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Text Decoration Style</span>
</div>
```

## Why is it useful?

This submission demonstrates the text decoration style CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
