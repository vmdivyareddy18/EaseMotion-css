# Dice 3D — Rotating Die

CSS 3D die built with `preserve-3d` and six face elements. The `spin-die` keyframe rotates the cube continuously on X and Y axes.

## Files

- `demo.html` — 3D scene with die faces
- `style.css` — perspective, 3D transforms, face positioning

## Usage

Open `demo.html` to see the die tumble in 3D space.
