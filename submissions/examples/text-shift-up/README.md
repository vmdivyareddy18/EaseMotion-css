# Text Shift Up on Hover

Text shifts upward on hover — pure CSS.

## Features
- Smooth translateY(-4px) transition on hover
- Pure CSS, no JavaScript

## Usage
```css
.up-text { transition: transform 0.3s; }
.up-text:hover { transform: translateY(-4px); }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
