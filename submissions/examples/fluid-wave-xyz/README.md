1. What does this do?
An advanced micro-interaction button component that utilizes offset border-radius mechanics to generate a layered, organic liquid fill animation upon hovering.

2. How is it used?
```html
<button class="fluid-btn">
  <span class="btn-text">Click Me</span>
  <div class="wave-wrapper">
    <div class="wave-layer layer-one"></div>
    <div class="wave-layer layer-two"></div>
    <div class="wave-layer layer-three"></div>
  </div>
</button>
```
