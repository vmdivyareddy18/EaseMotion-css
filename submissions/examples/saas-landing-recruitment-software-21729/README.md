# Business Landing — Recruitment Software (ATS) SaaS (`saas-landing-recruitment-software-21729`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for HR Tech and Applicant Tracking Systems (ATS), using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A Recruitment SaaS landing page needs to feel fast, organized, and powerful (similar to Greenhouse, Lever, or Workable). This template uses a clean, modern aesthetic with highly legible typography and dynamic layouts to convey an organized hiring process.

The layout includes:
- A hero section featuring an animated CSS-built Kanban board / applicant pipeline mockup
- A logo banner for job board integrations
- A 3-column platform feature grid
- A split layout showcasing hiring workflows
- Customer testimonials highlighting time-saved
- A 2-tier pricing layout based on active job slots
- A high-impact dark CTA section

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth, authoritative text and card entrances)
- `.ease-slide-right` / `.ease-slide-left` / `.ease-slide-up` (For layout structural animations)
- `.ease-zoom-in` / `.ease-bounce-in` (For drawing attention to primary CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For buttons, feature cards, and mockups)
- `.ease-hover-glow` (For subtle link interactions)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
