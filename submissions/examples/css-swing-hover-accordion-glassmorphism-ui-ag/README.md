# CSS Swing-Hover Accordion (Glassmorphism)

A visually stunning accordion component designed for Glassmorphism UI layouts. Features frosted glass panels that swing realistically like physical glass panes when hovered, set against vibrant gradient backgrounds.

## Features
- **Heavy Glass Swing Animation:** A 3D swing animation calibrated to feel like a weighty pane of frosted glass swaying on a top hinge.
- **True Glassmorphism:** Uses `backdrop-filter: blur()`, translucent white backgrounds, and delicate white borders to create realistic frosted glass.
- **Vibrant Backdrop:** Designed to sit over a colorful, organic gradient background to showcase the frosted glass effect.
- **Pure CSS:** Completely JS-free implementation using semantic HTML (`<details>` and `<summary>`).

## Usage
Add the `glass-accordion-container` class to your wrapper. Each item should be a `<details>` element with the `glass-item` class, containing a `<summary>` with the `glass-header` class. 

## Files
- `demo.html`: An interactive preview featuring a vibrant background to showcase the glass effect.
- `style.css`: The source CSS containing the glassmorphism styles and the weighty glass swing animation.
