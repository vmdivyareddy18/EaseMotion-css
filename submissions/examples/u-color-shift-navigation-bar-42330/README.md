# Color Shift Navigation Bar

A Music Player-inspired Color Shift Navigation Bar built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Animated color-shifting gradient
- Responsive navigation layout
- Accessible semantic HTML
- No JavaScript required
