scrollbar layout shift in modals

Issue #18794 — body.style.overflow = 'hidden' removes the scrollbar, causing the page to jump ~15px right. Proposed fix: scrollbar-gutter: stable.

before (broken)

shift overflow: hidden removes scrollbar → layout jumps right.

after (fixed with scrollbar-gutter)

stable scrollbar-gutter: stable on body reserves space for the scrollbar at all times — no layout shift.

page width: measuring...  |  scrollbar visible: yes  |  body offset: 0px 
/* proposed change to core/modal.js */
body.style.overflow = 'hidden';
/* add: */ document.documentElement.style.scrollbarGutter = 'stable';

/* alternative CSS-only approach (no JS change needed) */
html { scrollbar-gutter: stable; }
