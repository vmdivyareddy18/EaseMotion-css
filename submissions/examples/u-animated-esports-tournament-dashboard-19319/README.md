# Animated Esports Tournament Dashboard

A modern esports tournament dashboard built with HTML and CSS. This example showcases animated statistic cards, live match tracking, team rankings, tournament progress indicators, prize pool distribution, and responsive layouts suitable for gaming and esports platforms.

## Features

* Animated tournament statistics cards
* Live match monitoring section
* Team rankings leaderboard
* Tournament progress tracking
* Prize pool distribution cards
* Upcoming match schedule
* Activity feed updates
* Hover animations and transitions
* Fully responsive layout
* Modern gaming-inspired design
