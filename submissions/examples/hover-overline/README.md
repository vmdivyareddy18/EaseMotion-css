# Hover Overline Effect

A simple CSS hover animation where an overline appears above text and expands from left to right.

## Features

- Overline appears on hover
- Uses `::before` pseudo-element
- Smooth left-to-right animation
- CSS variable support for color customization
- Responsive and lightweight
- No JavaScript required

## Files

- demo.html
- style.css

## Customization

Modify:
- `--ease-overline-color`
- Animation duration
- Font size
- Line thickness
