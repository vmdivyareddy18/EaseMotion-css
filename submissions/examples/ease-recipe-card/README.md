# 🥞 Fluffy Buttermilk Pancakes — Recipe Card

A beautifully designed, single-recipe card built with semantic HTML and custom CSS. Perfect for food blogs, personal projects, or just practicing front-end layout!

## ✨ Features

- 📱 **Fully responsive** — looks great on mobile, tablet, and desktop
- 🎨 **Attractive card design** with hover effects and soft shadows
- 📋 **Semantic HTML** — clean, accessible markup
- 🥞 **Complete recipe** with ingredients, instructions, and pro tips
- 🏷️ **Badge & meta info** for quick recipe overview

---

## 🛠️ Technologies

- **HTML5** — Semantic elements like `<article>`, `<section>`, `<ul>`, `<ol>`
- **CSS3** — Flexbox, Grid, custom properties, hover effects, responsive design

---

## 🚀 How to Use

1. Clone or download this repository.
2. Open `demo.html` in any modern browser.
3. No build tools or dependencies needed — it's pure HTML & CSS!
