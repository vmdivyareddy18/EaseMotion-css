# CSS Container Query Wrapper Classes

A component setup demonstrating responsive container layout modifications based on parent widths.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting narrow and wide parent containers, adaptive text structures, and custom icons.
2. **`style.css`**: Container contexts, inline-size declarations, container query blocks, and layout media rules.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the **Narrow Container** (300px) — confirm that the adaptive card layout renders vertically with centered text.
3. Observe the **Wide Container** (500px) — confirm that the adaptive card layout shifts horizontally with left-aligned text.
