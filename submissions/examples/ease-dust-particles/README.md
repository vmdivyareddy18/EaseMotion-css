# ease-dust-particles

Slow-drifting dust motes float upward across a container, creating a calm ambient texture.

## Usage

```html
<div class="ease-dust" id="scene">Content</div>
```

Populate `.dp` children via script with random positions and long animation durations.

## Why it fits EaseMotion CSS

Very slow timings (6–14s) make this feel organic rather than mechanical. CSS-first, script is only a generator.
