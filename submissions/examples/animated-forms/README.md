# Animated Form Components

A comprehensive suite of form components built purely with CSS. Includes text inputs with animated floating labels, custom checkboxes with bounce animations, and smooth sliding toggle switches.

## Files
- `demo.html` - Interactive demonstration of a login form using these components.
- `style.css` - The CSS implementation utilizing EaseMotion tokens.

## Details
This addresses the "Form components (inputs, checkboxes, toggles)" requirement currently listed as "Planned" in the project Roadmap. All components use native HTML form elements to ensure accessibility and submission compatibility while providing a beautiful, "animation-first" visual experience.
