# Black Ink Studio – Tattoo Artist Portfolio Landing Page

A complete, ready‑to‑copy landing page for a tattoo artist portfolio built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – bold tattoo artwork with consultation CTA
- **Portfolio Gallery** – realism, traditional, neo‑traditional, blackwork
- **About the Artist** – photo and bio
- **Shop Info** – location, hours, pricing
- **Booking Process** – four‑step flow
- **Reviews** – client testimonials
- **Appointment CTA** – dark call‑to‑action section
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-gray-900`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-full`, `ease-rounded-lg`, `ease-shadow-sm`, `ease-shadow-lg`, `ease-bg-*`, `ease-w-full`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
