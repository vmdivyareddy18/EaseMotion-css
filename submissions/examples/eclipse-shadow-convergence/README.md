# Eclipse Shadow Convergence Animation Utility

## Overview
This utility creates an eclipse-inspired animation where two shadow layers converge toward the center and overlap briefly before moving apart.

## Usage

```html
<div class="ease-eclipse-shadow">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first interaction
- Great for hero sections, card reveals, and futuristic UI effects
