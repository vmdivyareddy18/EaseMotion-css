# CSS Glow Pulse Button – Minimal Blog

A pure CSS Glow Pulse button designed for minimal blog interfaces. The component provides a subtle glowing animation, responsive layout, keyboard accessibility and customizable styling using CSS custom properties.

---

## Features

- Pure HTML & CSS
- Glow Pulse animation
- Minimal blog inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and reusable
- No JavaScript required

---

## Files

- `demo.html`
- `style.css`
- `README.md`

---

## Customization

You can customize:

- Background color
- Button color
- Glow color
- Animation duration
- Border radius
- Typography

using CSS variables defined inside `:root`.
