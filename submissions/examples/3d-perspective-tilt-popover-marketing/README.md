# 3D Perspective Tilt Popover - Marketing Layout

A pure CSS animated popover component featuring a smooth 3D perspective tilt effect for modern marketing landing pages.

## Features

- CSS-only animation
- No JavaScript required
- 3D perspective transition
- Responsive layout
- Keyboard accessible using focus states
- Customizable animation variables

## Usage

```html
<div class="tilt-popover">

  <button class="tilt-trigger">
    Explore Solution
  </button>

  <div class="tilt-card">
    Marketing content
  </div>

</div>
