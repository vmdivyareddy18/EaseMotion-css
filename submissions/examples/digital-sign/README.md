# E‑Signature SaaS Website

This demo site showcases a DocuSign‑style SaaS layout using **EaseMotion CSS classes** only.

## 🧩 Sections
- Hero with signature flow mockup
- Feature grid
- Legal compliance badges
- Workflow process timeline
- Pricing tiers
- Customer logos
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — entrance fade animations
- `ease-slide` — sliding entrance for headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs, logos, and mockups
- `ease-hover-opacity-reduce` — hover dimming for feature and pricing cards

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll through sections to see entrance animations.
3. Hover over buttons, cards, and logos to see interactive effects.
4. Replace placeholder images and text with real SaaS content.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion animation classes
