# Ease Hydra Hiss Animation

1. What does this do? Creates a rapid, vibrating animation mimicking a venomous snake rearing back, expanding its hood (scaling up), and fiercely hissing/shaking side-to-side before striking back to its original position.
2. How is it used? Apply `.ease-hydra-hiss-ps` for an immediate visual warning, or `.ease-hydra-hiss-hover-ps` to trigger the hiss upon hovering.
3. Why is it useful? It serves as a fantastic, aggressive error state or a thematic warning interaction. Unlike a standard CSS horizontal shake, it scales the element up slightly and applies an alternating `skewX` to create a more organic, "slithering" vibration rather than a mechanical digital shake.
