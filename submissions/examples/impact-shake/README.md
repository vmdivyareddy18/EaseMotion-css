# Impact Screen Shake

A rapid, brief screen/element shake animation simulating an impact force (like game damage, explosive hits, or dramatic visual feedback).

## EaseMotion CSS Classes Showcased

This feature submits a self-contained prototype demonstrating keyframe offsets simulating a rapid, high-frequency, decaying 2D shake.

### Classes:
- `.ease-impact-shake`: Triggers a rapid impact vibration over `0.35s` using linear translation offsets.
- Can be triggered programmatically by adding/removing the class on click, hit, or collision events.

### Usage in HTML:
```html
<div class="ease-impact-shake">
  Content to shake on impact
</div>
```
