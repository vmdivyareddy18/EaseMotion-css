# Component: WCAG AA Color Pairs

This submission documents WCAG AA contrast ratios for EaseMotion color pairs for issue **#13624**.

## Description

A table showing 16 foreground/background color combinations from the EaseMotion design system, their contrast ratios, and whether they pass WCAG AA (4.5:1 for normal text). Includes visual swatches and pass/fail status for each pair.

## Acceptance Criteria

- Table with 15+ color pair rows
- Foreground and background color swatches
- Contrast ratio displayed per pair
- WCAG AA pass/fail status
- Mix of passing and failing examples
