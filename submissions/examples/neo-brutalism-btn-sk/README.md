# Neobrutalism Push Button

A bold, trendy Neobrutalist push button component.

## Files
- `demo.html`: The HTML structure of the button.
- `style.css`: The CSS rules defining the shadow and active click state.
- `README.md`: Documentation.

## Features
- **Neobrutalist Aesthetic:** Features a thick solid border and a harsh, non-blurred solid drop shadow (`box-shadow: 4px 4px 0px #000;`).
- **Mechanical Click:** On `:active`, the button physically translates downwards and to the right by the exact dimension of the shadow, while the shadow drops to `0px`. This mimics a satisfying mechanical keypress.
- **Zero JS:** Fully powered by native CSS pseudo-classes.

## Usage
Simply drop the `.neo-brutalism-btn` class onto any `<button>` or `<a>` tag.
