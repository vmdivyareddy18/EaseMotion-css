# Workspace Invite Role Picker

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern workspace invitation interface featuring role selection cards, invite summaries, team management UI patterns, and responsive layouts built entirely with HTML and CSS.

## Features

- Workspace invitation panel
- Email input field
- Role selection cards
- Admin, Editor, Viewer roles
- Active role highlighting
- Invite summary section
- Hover animations
- Glassmorphism styling
- Responsive design
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="role-card active">
  <h3>Admin</h3>
</div>
```

## Main Classes

```css
.invite-card
.role-card
.active
.summary-panel
.invite-btn
```
