# Animated Toggle Switch UI Example

## 1. What does this do?

This submission provides a CSS-first animated toggle switch component with smooth thumb movement, track transitions, and color variants. It uses a hidden checkbox input and custom track/thumb styling to show on/off state clearly.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example uses `style.css` for the toggle appearance and motion. Copy the `.toggle-switch` markup along with the CSS to reuse the component in your project.

---

## 3. Why is it useful?

Toggle switches are widely used for settings, preferences, and binary controls. This example demonstrates how to build a polished switch with animated state changes, accessible checkbox semantics, and reusable color variants.
