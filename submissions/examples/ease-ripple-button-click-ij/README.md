# Ripple Button Click

Button with ripple effect on click. Ripple expands from click point with opacity fade.

## Features

- Button with ripple effect on click
- Ripple expands from click point
- Opacity fade as ripple grows
- CSS keyframe animation, JS creates ripple element

## Usage

Click the button. JS creates a `.rbc-ripple` span at click position. CSS handles the animation.
