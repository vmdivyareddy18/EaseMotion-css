# 15 — grid-auto-columns-rows

## Overview
Demonstrates `grid-auto-rows` and `grid-auto-columns` for sizing implicit grid tracks when items overflow defined explicit tracks. The auto-rows example places 6 items in a 3×2 explicit grid, with the 5th and 6th items falling into implicit rows sized at 120px. The auto-columns example uses `grid-auto-flow: column` with implicit columns at 160px.

## Files
- `demo.html` — page with two grid demos (auto-rows and auto-columns)
- `style.css` — dark-themed styles showing implicit track sizing

## Usage
Open `demo.html` to see implicit grid items highlighted with accent borders, demonstrating how `grid-auto-rows` and `grid-auto-columns` control their size.
