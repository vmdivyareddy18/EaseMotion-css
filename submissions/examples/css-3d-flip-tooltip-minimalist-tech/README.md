# CSS 3D Flip Tooltip – Minimalist Tech Layout

## Overview

A responsive Minimalist Tech tooltip component built entirely with HTML and CSS. The component features a smooth 3D flip animation, clean monochrome styling, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip tooltip animation
- Minimalist Tech interface
- Responsive layout
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--surface`
- `--background`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on the action buttons to display the animated 3D Flip Tooltip. Colors, animation timing, flip angle, spacing, and shadows can all be customized using the CSS variables defined in `style.css`.
