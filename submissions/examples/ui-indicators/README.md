# UI Indicators Component

A collection of essential micro-components including badges, tags, avatars, and progress bars. These indicators provide clear visual feedback to users and act as the "glue" that binds larger interface components together.

## Files
- `demo.html` - Interactive demonstration of all indicator types and sizes.
- `style.css` - The CSS implementation utilizing EaseMotion tokens.

## Details
This addresses the "Badge, tag, avatar, progress bar" requirement currently listed as "Planned" in the project Roadmap. 
- **Badges**: Pill/rounded indicators with pulse animations.
- **Avatars**: Circular images/initials with hover-lifts and status dots.
- **Progress Bars**: Animated width transitions and infinite indeterminate loading states.
