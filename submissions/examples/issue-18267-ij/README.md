# Animated Timeline (Issue #18267)

A vertical timeline with items on alternating sides of a central line. Each item fades and slides in when scrolled into view using `IntersectionObserver`. Uses `@keyframes fadeInLeft`/`fadeInRight`-equivalent CSS transitions.

## Usage

Open `demo.html` in a browser. Scroll down to see timeline items animate into view.

## Files

- `demo.html` — HTML5 page with timeline and IntersectionObserver
- `style.css` — Dark theme styles with slide/fade transitions
