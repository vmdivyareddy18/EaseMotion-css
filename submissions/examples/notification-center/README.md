# Animation: Notification Center

A modern notification center component featuring staggered card animations, status indicators, and interactive hover effects.

## Features

- Pure HTML & CSS
- Staggered notification reveal
- Status-specific styling
- Interactive hover effects
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="notification success">

    <div class="icon">✅</div>

    <div>
        <h3>Deployment Successful</h3>
        <p>Your project was deployed successfully.</p>
    </div>

</div>
```

## Why is it useful?

Notification centers are commonly used in dashboards, productivity tools, developer platforms, and SaaS applications. This example demonstrates how animated feedback can improve visibility and user engagement while maintaining a clean interface.
