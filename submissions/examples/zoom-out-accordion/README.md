# Zoom-Out Accordion (Cyberpunk Neon)

Ek pure CSS animated accordion jo 'Zoom-Out' interaction transition use karta hai. Iska design Cyberpunk interfaces aur neon aesthetics se inspire hai.

## Features
- **Pure CSS:** JavaScript ka zero usage.
- **Animation:** `transform: scale()` ka use karke smooth zoom-out/zoom-in effect.
- **Theme:** Cyberpunk inspired neon green glow.

## How to customize
- `--neon-green`: Color change karne ke liye.
- `--roll-speed`: Transition speed adjust karne ke liye CSS mein time update karein.

## Testing
- Browser mein `demo.html` file ko open karein.
- Accordion labels par click karke zoom-out animation dekhein.
