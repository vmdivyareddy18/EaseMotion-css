# Animated Countdown Timer Component

## Overview

A modern countdown timer component featuring glowing timer cards, smooth pulse animations, hover effects, and a clean event-style layout.

## Features

- Glowing countdown cards
- Smooth pulse animation
- Hover lift and scaling effects
- Responsive card layout
- Modern UI design
- Pure HTML and CSS implementation

## Files

- demo.html – Demonstrates the countdown timer layout
- style.css – Contains timer styles and animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in a modern browser to view the animated countdown timer component.

Hover over the timer cards to experience the interactive animation effects.
