# Component: Background Grid

This submission implements a grid line pattern background for issue **#13616**.

## Description

An `.ease-bg-grid` class that creates a grid pattern using two `linear-gradient` layers for horizontal and vertical lines. Customizable via `--ease-grid-size` and `--ease-grid-color` CSS custom properties. Includes interactive controls for grid size, opacity, and visibility toggle.

## Acceptance Criteria

- `.ease-bg-grid` class with dual `linear-gradient` grid lines
- `--ease-grid-size` controls cell size
- `--ease-grid-color` controls line color
- Interactive size slider, opacity slider, and toggle button
