# Event Management Dashboard

A responsive **Event Management Dashboard** built using **HTML5** and **CSS3** that demonstrates event planning, attendee registration, ticket management, and schedule monitoring through a modern interactive dashboard.

---

## Features

- Responsive dashboard layout
- Event overview cards
- Registration progress tracker
- Event schedule panel
- Ticket availability monitor
- Recent registrations feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

### Event Widgets

- Registration Progress

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
