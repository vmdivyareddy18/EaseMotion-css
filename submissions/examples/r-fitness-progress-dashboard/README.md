# Fitness Progress Dashboard

A responsive Fitness Progress Dashboard example built with EaseMotion CSS.

## Features

- Daily fitness statistics
- Weekly progress tracking
- Achievement badges
- Activity timeline
- Responsive dashboard layout
- Smooth animations and hover effects

## Folder Structure

```text
r-fitness-progress-dashboard/
├── README.md
├── demo.html
└── style.css
```

## Usage

Open `demo.html` in a modern web browser.

## EaseMotion Features Used

- Fade-in animations
- Hover transitions
- Responsive grid layouts
- Interactive dashboard cards
