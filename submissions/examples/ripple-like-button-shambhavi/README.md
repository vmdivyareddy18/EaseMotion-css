# Ripple Like Button

A Nature-inspired **Ripple Like Button** component for the EaseMotion CSS library.

## ✨ Features

- 🌿 Nature-inspired green gradient design
- 💧 Pure CSS ripple animation
- ❤️ Interactive heart icon
- 📱 Responsive layout
- ♿ Accessible with `:focus-visible`
- 🚫 No JavaScript required

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in your browser to preview the component.

## Accessibility

- Keyboard focus support
- Semantic HTML button
- Responsive on different screen sizes

## Animation

The ripple expands smoothly from the center of the button on hover, creating a subtle water-inspired interaction.
