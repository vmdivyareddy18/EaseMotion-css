# Next-Gen Neural Interface Dashboard

A next-gen neural interface (BCI) dashboard built with EaseMotion CSS tokens. Monitors brain signal clarity, neural connections, bandwidth by brain region, active pathways, mental state, and neural activity in a futuristic purple-neon dark theme.

## Features

- **Stat cards** — signal clarity, active neurons, avg latency, bandwidth
- **Neural connections** — 6 connections with synced/drift/loss/idle status
- **Bandwidth by region** — frontal, temporal, parietal, occipital with color-coded bars
- **Active pathways** — 5 brainwave frequencies (theta, alpha, beta, gamma, delta)
- **Mental state** — focus level gauge, cognitive load, fatigue, stress, engagement
- **Neural activity log** — timestamped BCI events with severity icons
- **Interface overview** — active regions, channels, uptime, alerts, packets/s, session duration
- Responsive 12-column grid with purple-neon dark (default) and light mode support
- Reduced-motion compliant
