# Ease XP Bar Fill

Closes #13735

## Description
An experience bar component with a gradient fill and a glowing animated leading edge. The bar smoothly transitions width as XP is added.

## Acceptance Criteria
- Gradient fill from blue to purple
- Glowing dot/edge at the fill boundary with pulse animation
- Transition on width change is smooth
- Supports overflow beyond max for visual feedback

## Files
- `style.css` — Dark theme with XP bar and glow styles
- `demo.html` — Interactive XP bar demo with Add XP and Reset buttons
