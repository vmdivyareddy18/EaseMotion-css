# Workout Exercise Timer

## What does this do?
An interval workout timer with a exercise list, round tracking, rest periods between exercises, and start/skip/reset controls.

## How is it used?
Open demo.html in a browser. Click play to start the workout. Each exercise runs for its set duration with rest periods in between. Skip or reset to control the workout.

## Why is this useful?
Demonstrates an interval timer pattern suitable for HIIT workouts, circuit training, or any timed exercise routine with round tracking.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
