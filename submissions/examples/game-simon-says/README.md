# 🎮 Simon Says

A classic Simon Says memory game built with HTML, CSS, and JavaScript. Watch the sequence of colored buttons, then repeat it from memory. Each round adds one more step to the sequence — how long can you keep up?

## 🎮 How to Play

1. **Click "Start Game"** or press **Space** to begin
2. **Watch** the sequence of colors light up with sounds
3. **Repeat** the sequence by clicking the colored buttons in the same order
4. Each correct sequence adds one more step
5. Make a mistake and the game ends!
6. **Challenge yourself** to beat your high score!

## ⌨️ Controls

| Action | Key / Input |
|--------|-------------|
| Start Game | Click center circle, press `Space`, or click Start button |
| Color 1 (Green) | Click green button or press `1` |
| Color 2 (Red) | Click red button or press `2` |
| Color 3 (Yellow) | Click yellow button or press `3` |
| Color 4 (Blue) | Click blue button or press `4` |
| Reset Game | Press `R` |

## ✨ Features

- **Progressive Difficulty** – Sequence grows longer each round
- **Sound Feedback** – Each color plays a unique tone
- **Visual Feedback** – Colors light up with glow effects
- **Score Tracking** – Tracks your current score
- **Best Score** – Saved in browser local storage
- **Level Display** – Shows current sequence length
- **Keyboard Shortcuts** – Quick input with number keys
- **Responsive Design** – Plays perfectly on all devices
- **Touch Support** – Works on mobile and tablet
- **Animated Feedback** – Shake animation on wrong guesses

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations, gradients, responsive design)
- Vanilla JavaScript (Web Audio API for sounds)
