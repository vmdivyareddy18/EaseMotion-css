# Event Ticketing Management Dashboard Showcase

## What does this do?

Provides an animated event-ticketing dashboard with event tracking, ticket sales monitoring, registration analytics, and attendee activity updates.

## How is it used?

Open `demo.html` directly in a browser and explore the event management interface.

## Why is it useful?

It demonstrates practical event-management UI patterns and combines multiple animation techniques into a realistic ticketing and registration monitoring experience.
