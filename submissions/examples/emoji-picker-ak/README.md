# Emoji Picker

## What does this do?
An emoji picker with category tabs (Smileys, Gestures, Food, Travel, Symbols), search functionality, and one-click copy to clipboard.

## How is it used?
Open demo.html in a browser. Browse emoji by category tab or search by keyword. Click any emoji to copy it to clipboard — the preview bar shows the selected emoji.

## Why is it useful?
This provides a complete emoji picker component suitable for chat applications, comment forms, or social media tools. It demonstrates category filtering, search, and clipboard integration.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the picker.
