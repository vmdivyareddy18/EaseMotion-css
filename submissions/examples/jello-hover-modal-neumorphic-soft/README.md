# CSS Jello-Hover Modal for Neumorphic Soft Layouts

A responsive, **pure CSS modal** featuring a smooth **Jello-Hover** animation with a modern **Neumorphic Soft** design. The component requires **no JavaScript** and is fully customizable using CSS custom properties.

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Smooth Jello-Hover animation
- Neumorphic Soft UI design
- Responsive layout
- Keyboard accessible
- Customizable using CSS variables
- Lightweight and reusable

## 🚀 Usage

1. Open `demo.html` directly in your browser.
2. Click the **Open Modal** button.
3. Hover over the modal to experience the **Jello-Hover** animation.
4. Click **Close** to dismiss the modal.

## 🎨 Customization

Adjust the following CSS variables in `style.css`:

```css
:root {
    --duration: 0.7s;
    --ease: cubic-bezier(.22,1,.36,1);
    --scale: 0.92;

    --primary: #4f46e5;
    --primary-hover: #4338ca;

    --radius: 22px;
}
```

These variables allow you to customize:

- Animation duration
- Animation easing
- Initial modal scale
- Primary colors
- Border radius
