# CSS Jello-Hover Accordion (Accessible Web Layouts)

A pure CSS responsive accordion designed for accessible web structures. Features a smooth, responsive "jello" wobble effect when hovering over the card headers, adding playfulness while adhering to accessible guidelines.

## Features
- **Jello-Hover Wobble**: High-fidelity keyframe transition that skews header cards back and forth dynamically on hover or keyboard focus.
- **Pure CSS checked State**: Built utilizing native checkbox inputs for zero-JS responsive toggling.
- **High Contrast Accessibility**: Large text, explicit contrast colors (WCAG 2.1 AA compliant), and prominent focus outlines for keyboard users.
- **EaseMotion Style Tokens**: Custom timing and skew variable support.

## Files
- `demo.html`: Live interactive demonstration of the accessible jello-hover accordion.
- `style.css`: Stylesheet containing the flex/width horizontal transitions and active state rules.
