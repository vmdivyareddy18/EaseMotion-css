Jello-Hover Accordion

An accessible web accordion component with a playful wobble effect. Hover or focus on cards to trigger the jello entry sequence.

It is a pure CSS physics-mimicking keyframe animation that skews and stretches cards along their 3D boundaries, creating a delightful interactive response when hovering or keyboard navigating.
