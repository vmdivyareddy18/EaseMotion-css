# ease-stock-ticker-flash

Animated stock ticker component that flashes green or red when prices change.

## Features

- Green flash for price increase
- Red flash for price decrease
- Fade back to normal state
- Stock market dashboard UI
- Finance themed component
- Responsive design

## Acceptance Criteria Covered

### Direction Aware Feedback

- Green flash = stock moving up
- Red flash = stock moving down

### Fade Animation

Background briefly highlights then smoothly returns to normal.

### Use Cases

- Trading platforms
- Stock dashboards
- Crypto exchanges
- Finance applications
- Market monitoring tools
