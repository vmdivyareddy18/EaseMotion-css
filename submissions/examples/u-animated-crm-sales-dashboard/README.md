# Animated CRM Sales Dashboard

A modern **CRM Sales Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example showcases revenue analytics, customer pipeline management, lead tracking, and sales performance with smooth CSS-only animations.

## Features

* 💰 Revenue overview card
* 🎯 New Leads tracker
* 👥 Active Customers statistics
* 📈 Circular conversion rate indicator
* 📋 Multi-stage sales pipeline
* 📊 CSS-only monthly revenue chart
* 📉 Regional sales progress bars
* 🔔 Customer activity feed
* ✨ Hover interactions and smooth animations
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

## Customization

All colors, spacing, and theme settings can be customized using CSS variables.

Example:

```css
:root{
    --primary:#2563eb;
    --success:#10b981;
    --warning:#f59e0b;
    --surface:#ffffff;
    --bg:#f5f7fb;
}
```
