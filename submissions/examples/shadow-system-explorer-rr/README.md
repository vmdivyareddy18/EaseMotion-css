# Elevation & Shadow System Explorer Component

## What does this do?

Provides a visual explorer for comparing different shadow and elevation levels commonly used in modern design systems.

## How is it used?

```html
<div class="shadow-card md">
    <h3>Shadow MD</h3>
</div>
```

## Why is it useful?

Elevation and shadow systems help communicate hierarchy, focus, and depth in user interfaces. This component demonstrates multiple shadow levels and interactive hover states using pure HTML and CSS.
