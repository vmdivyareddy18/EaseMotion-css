# Animated Background Patterns

## 1. What does this do?

This submission provides a collection of reusable CSS-only animated background patterns, including Animated Gradient, Floating Blobs, Aurora, Animated Grid, Mesh Gradient, and Radiant Rays. Each pattern is displayed in a responsive showcase with a live preview, title, and short description.

---

## 2. How is it used?

Open `demo.html` directly in a browser to preview the collection. Copy the desired background class and its corresponding CSS from `style.css`, then apply it to any container in your project. Colors and animation settings can be customized using the provided CSS custom properties.

---

## 3. Why is it useful?

Animated backgrounds are widely used in hero sections, landing pages, dashboards, authentication screens, and marketing websites. This submission demonstrates lightweight, responsive, reusable, and accessible CSS-only background patterns that require no external libraries or frameworks.
