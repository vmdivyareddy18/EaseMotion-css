ease-baseline-ruler-grid

Visual baseline grid ruler overlay for alignment measurement

Default (--grid-color: rgba(79,70,229,0.15); --grid-size: 16px)

This content sits above the baseline grid. The grid lines help you verify vertical rhythm, line-height alignment, and spacing consistency in your layouts.

Custom (--grid-color: rgba(220,38,38,0.12); --grid-size: 12px)

A finer 12px red-tinted grid. Useful for checking tighter vertical spacing and smaller font baselines.

Double Height (--grid-size: 24px; --ruler-height: 200px)

A larger 24px grid with extended 200px ruler height for measuring taller elements.
