# Animated Renewable Energy Dashboard

A modern dashboard for monitoring renewable energy production, battery storage, sustainability goals, and carbon reduction metrics.

## Features

- Solar and wind energy metrics
- Battery storage monitoring
- Carbon savings tracking
- System status indicators
- Sustainability goal progress bars
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Renewable energy platforms
- Sustainability dashboards
- Smart grid monitoring
- Solar farm analytics
- Environmental reporting systems
