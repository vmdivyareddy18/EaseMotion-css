# Wobble Card Hover

Cards with a wobble animation on hover, including a bouncing icon effect.

## Features

- Wobble keyframe animation on hover (rotate and scale)
- Icon bounce and rotation effect
- Glassmorphism card design
- Responsive flex layout with wrapping
- Pure CSS animation via `:hover`

## Usage

Include `style.css` and structure cards with `.wobble-card` and `.wobble-card-inner`. The wobble animation triggers on hover using CSS `@keyframes`.
