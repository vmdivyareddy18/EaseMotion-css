# Executive Recruiter Template
A discreet, sophisticated, and high-impact template for executive search firms, headhunting agencies, and leadership placement consultancies.

- **Design**: Uses a "Leadership-Sophistication" palette with charcoal-suit professional tones, success-mint highlights, and clean, modular layouts to convey exclusivity, precision, and trust.
- **Use Case**: Retained search firms, executive headhunters, and C-suite placement consultancies.
