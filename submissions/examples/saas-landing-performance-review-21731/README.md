# Business Landing — Performance Review Software SaaS (`saas-landing-performance-review-21731`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for HR Tech and Performance Management software, using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A Performance Review SaaS landing page needs to feel modern, reliable, and user-friendly (similar to Lattice, 15Five, or CultureAmp). This template utilizes a clean slate and indigo color palette to achieve a trustworthy HR-tech aesthetic.

The layout includes:
- A hero section featuring an animated CSS-built review cycle dashboard mockup
- A logo banner for enterprise trust
- A 3-column platform feature grid
- A cycle timeline example paired with a progress UI card
- An integration ecosystem pill grid
- Customer testimonials
- A 2-tier per-employee pricing layout
- A high-impact free trial CTA

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth, authoritative text and card entrances)
- `.ease-slide-right` / `.ease-slide-left` / `.ease-slide-up` (For layout structural animations)
- `.ease-zoom-in` / `.ease-bounce-in` (For drawing attention to primary CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For buttons, feature cards, and mockups)
- `.ease-hover-glow` (For subtle link interactions)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
