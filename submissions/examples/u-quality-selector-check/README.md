# ease-quality-selector-check

Animated video quality selector component with checkmark feedback.

## Features

- Animated checkmark on selected quality
- Smooth dropdown open animation
- Video player settings UI
- Responsive design
- Pure HTML and CSS

## Acceptance Criteria Covered

### Animated Checkmark

Uses a checkbox-style animated checkmark on the selected quality option.

### Dropdown Animation

Dropdown menu opens with a smooth transition effect.

### Video Settings UI

Supports common quality options:

- 1080p
- 720p
- 480p
- 360p
- 240p

## Use Cases

- Video players
- Streaming platforms
- Media applications
- Learning portals
