# Auto-Complete Search

## What does this do?
A search input with a dropdown of programming language suggestions that filter as the user types.

## How is it used?
Open demo.html in a browser. Type in the input field to see matching suggestions. Click a suggestion or use the keyboard (↑↓ arrows, Enter, Esc) to select.

## Why is this useful?
Demonstrates a common UI pattern for type-ahead search with keyboard navigation and accessible interaction.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
