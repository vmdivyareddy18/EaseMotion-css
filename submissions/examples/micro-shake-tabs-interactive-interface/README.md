# CSS Micro Shake Tabs – Interactive Interface

## Issue
Closes #50165

## Description
A pure CSS animated tab component featuring a subtle Micro Shake effect for interactive interfaces. The component is responsive, keyboard accessible, and customizable using CSS variables.

## Features
- Pure HTML & CSS
- Micro Shake hover animation
- Responsive layout
- Keyboard accessible
- CSS custom properties for customization
- No JavaScript required

## Files
- demo.html
- style.css
- README.md

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="tabs">
  <button class="tab active">Home</button>
  <button class="tab">Services</button>
  <button class="tab">Projects</button>
</div>
