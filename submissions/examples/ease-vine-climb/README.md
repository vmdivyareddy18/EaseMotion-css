# Ease Vine Climb Animation

An intricate, timed CSS vector animation depicting a winding vine growing vertically against a wall structure, leveraging SVG stroke offsets combined with staggered keyframed scale expansions for structural leaves.

## How to View
1. Double-click or serve `demo.html`.
2. Interact with the "Grow Vine" control button to review the sequence lifecycle.

## Technical Details
- **Features:** SVG `stroke-dashoffset` path manipulation paired with layered tracking delays.
- **Timing:** Primary interpolation curve uses a balanced standard `cubic-bezier(0.4, 0, 0.2, 1)` transition profile.
