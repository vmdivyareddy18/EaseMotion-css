# Stat Label

## What does this do?
Stat Label component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"sl-primary\">Stat Label Primary</div>
```

## Why is it useful?
Provides a reusable stat label UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
