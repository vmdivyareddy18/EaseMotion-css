# Enterprise Knowledge Graph Explorer Component

## What does this do?

Provides a centralized interface for exploring relationships between applications, APIs, services, teams, data assets, and business capabilities.

## How is it used?

```html
<div class="entity-card connected">
    <h3>Customer API</h3>
</div>
```

## Why is it useful?

Enterprise knowledge graphs help organizations understand complex relationships across systems and domains. This component provides a reusable interface for visualizing governance, ownership, connectivity, and metadata insights.
