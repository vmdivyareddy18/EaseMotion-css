# Ease Underline

Animated underline effect that expands smoothly from left to right on hover.

## Features

- Pure HTML & CSS

- Uses ::after pseudo-element

- Smooth transform animation

- Left-to-right underline reveal

- Responsive design

- Lightweight implementation

## Files

- demo.html

- style.css

- README.md

## Usage

Add the class:

```html
<a class="ease-underline">Hover Me</a>
```

The underline animates from left to right on hover using CSS transforms.
