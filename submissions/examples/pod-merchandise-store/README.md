# ThreadCraft — Print on Demand Merchandise Store

A complete, copy-paste ready full-page e-commerce template for creator economy merchandise. Styled with a bold modern dark theme, using responsive layout systems and verified EaseMotion CSS animation classes.

## Tech Stack
- HTML
- CSS (Vanilla layouts)
- Standard EaseMotion CSS Framework classes (No custom `@keyframes` or keyframe animations)
- Vanilla `reveal.js` Scroll Trigger

## EaseMotion Classes Showcase

This advanced template showcases the following core classes from EaseMotion CSS:

1. **Entrance Animations**
   - `.ease-fade-in`: Fades in the navigation bar and hero badges upon page load.
   - `.ease-slide-up`: Slides up the main heading, subtext, and CTAs in the hero section.
   - `.ease-zoom-in`: Smoothly scales up the custom product mockup in the hero section.

2. **Looping Animations**
   - `.ease-float`: A slow, hardware-accelerated floating vertical motion applied to the hero graphic tee mockup.

3. **Text Emphasis Animations**
   - `.ease-shimmer-text`: A moving linear-gradient shimmer applied to the main tagline highlights.

4. **Hover & Micro-interactions**
   - `.ease-hover-underline`: Animates an underline growing from left-to-right under navbar links and footer links.
   - `.ease-hover-grow`: A subtle scale-up effect applied to primary action buttons.
   - `.ease-squish-button`: A playful organic scale transform applied to click targets on hover/click.
   - `.ease-hover-lift-shadow`: Lifts card elements upwards and increases shadow blur (used on creator stories spotlight).
   - `.ease-card-lift`: Lifts and scales cards on hover (used on design categories).
   - `.ease-hover-shimmer`: Sweeps a light shine across card backgrounds on hover (used on design categories).
   - `.ease-hover-lift`: Lifts cards on hover (used on step cards and customer reviews).

5. **Scroll Reveal Animations**
   - `.ease-reveal` + `.ease-reveal-up`: Automatically animates elements (design categories, step cards grid, creator stories, reviews, newsletter signup) sliding up into place as the user scrolls them into the viewport.
   - `.ease-reveal` + `.ease-reveal-scale`: Scales up sections (like the central Build CTA banner) upon viewport entry.

6. **Animation & Delay Controls**
   - `.ease-delay-100`, `.ease-delay-200`, `.ease-delay-300`: Staggers the loading order of elements inside the hero section for a premium cinematic layout build.
