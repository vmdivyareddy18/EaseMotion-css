# Ease Notification Toast

## What does this do?

Provides animated toast notification components for success, error, warning, and information messages.

## How is it used?

```html
<div class="toast toast-success">
  <div class="toast-icon">✓</div>
  <div class="toast-content">
    <h4>Success</h4>
    <p>Your changes have been saved.</p>
  </div>
</div>
