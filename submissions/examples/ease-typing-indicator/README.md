# Typing Indicator – 3 bouncing dots (chat apps)

> A production-ready typing indicator component featuring 3 bouncing dots commonly used in chat applications, with customizable colors, speed, and name display. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **3 bouncing dots** – Classic chat typing animation
- ✅ **Customizable color** – Match your brand
- ✅ **Adjustable speed** – Control animation tempo
- ✅ **Name display** – Shows who is typing
- ✅ **Avatar support** – User avatar with status
- ✅ **Multiple sizes** – Small, Medium, Large, Dark
- ✅ **Chat simulation** – Demo chat with typing
- ✅ **Keyboard shortcuts** – Ctrl+Enter (toggle), Escape (stop)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
