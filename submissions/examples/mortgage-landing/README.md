# TrustMark Financial – Mortgage & Loan Landing Page

A fully responsive, animated landing page for a mortgage and loan company, built with **EaseMotion CSS** utility classes and custom refinements.

## Sections
- Hero with rate calculator CTA
- Loan types grid (mortgage, refinance, personal)
- Process timeline (apply → approve → close)
- Rate comparison table
- Customer testimonials
- Pre‑qualify CTA form
- FAQ accordion

## EaseMotion Classes Showcased
- **Layout:** ease-container, ease-grid, ease-flex, ease-overflow-x-auto
- **Typography:** ease-text-*, ease-font-*
- **Colours:** ease-bg-*, ease-text-*
- **Components:** ease-card, ease-btn, ease-input
- **Hover Effects:** ease-hover-scale-105, ease-hover-shadow-*
- **Animations:** ease-fade-in, ease-slide-up, ease-delay-*
- **Spacing:** ease-p-*, ease-m-*, ease-gap-*

## Custom CSS Additions
- Process timeline with numbered badges
- Styled rate comparison table
- FAQ accordion with open/close icons and active state
- Reduced‑motion accessibility
- Mobile timeline stacking

## How to Use
1. Copy demo.html into your project.
2. Ensure the path to easemotion.css is correct (usually from core/).
3. Open demo.html in any modern browser — no build step required.
