# Animated Statistics Dashboard Cards

A responsive dashboard card component displaying key metrics with hover animations and trend indicators.

## Features

- Responsive dashboard layout
- Metric cards with icons
- Positive and negative trend indicators
- Hover lift animations
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add additional statistics
- Replace icons with SVGs
- Change colors and typography
- Modify animation timing
