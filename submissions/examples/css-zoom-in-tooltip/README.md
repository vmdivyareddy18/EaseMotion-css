# CSS Zoom-In Tooltip

A lightweight tooltip component built entirely with CSS using a smooth zoom-in animation.

## Features

- Pure CSS
- Responsive
- Keyboard Accessible
- Zoom-In Animation
- CSS Custom Properties
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Customization

The following CSS variables are available:

```css
--duration
--ease
--scale
--tooltip-bg
--tooltip-color
--radius
```

These variables allow you to customize the animation timing, colors, border radius, and zoom effect.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## EaseMotion CSS

This component follows the EaseMotion CSS philosophy by providing a reusable, animation-first, JavaScript-free UI component suitable for checkout pages and other modern interfaces.
