# Neon Glow Modal

## What does this do?
A pure HTML and CSS modal featuring a vibrant neon glow effect, smooth opening animation, and a modern SaaS-inspired design without using JavaScript.

---

## How does a developer use it?
```html
<a href="#modal" class="btn">Launch Modal</a>

<div id="modal" class="modal">
  <div class="modal-box">
    <!-- Modal content goes here -->
  </div>
</div>
```

---

## Why is it useful?
This component aligns with EaseMotion CSS by providing a visually engaging, animation-first modal using only HTML and CSS. It is lightweight, responsive, easy to customize, and demonstrates modern neon interface aesthetics without JavaScript.

---

## Features
- Pure HTML & CSS
- Neon glow border and shadow effects
- Smooth open and close animation
- Responsive SaaS-inspired layout
- No JavaScript required
- Easy to customize
