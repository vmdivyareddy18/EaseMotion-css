# Holy Light Rays Animation

## What does this do?

Creates shimmering beams of holy light shining down from above with varied timing and opacity, all in pure CSS.

## How is it used?

Add `.ray` elements inside a `.rays-container`. Each ray gets a staggered animation delay.

```html
<div class="rays-container">
  <div class="ray r1"></div>
  <div class="ray r2"></div>
  <!-- more rays... -->
</div>
```

## Why is it useful?

Holy light effects enhance fantasy RPG UIs, divine spell indicators, church/temple scenes, and religious or fantasy game atmospheres without JavaScript.

## Tech Stack

- HTML
- CSS (no frameworks, no JavaScript)

## Preview

Open `demo.html` directly in your browser to see the effects.
