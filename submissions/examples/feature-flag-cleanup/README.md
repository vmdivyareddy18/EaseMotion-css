# Feature Flag Cleanup

Feature Flag Cleanup is a responsive governance panel for reviewing stale feature flags. It separates flags that should be removed, reviewed, or kept so release owners can reduce long-term rollout debt.

## Features

- Stale flag summary
- Remove, review, and keep states
- Compact flag review cards
- Responsive layout for mobile screens

## Files

- `demo.html` contains the feature flag cleanup markup.
- `style.css` contains the card layout and status labels.
