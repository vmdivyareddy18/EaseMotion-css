# Animated Statistics Counter Card Component

## Description

A modern statistics card component with animated icons, hover effects, and responsive layouts for showcasing achievements, analytics, and business metrics.

---

## Features

- Animated statistic cards
- Floating and bouncing icons
- Smooth card hover effects
- Modern shadows and rounded design
- Responsive grid layout
- Lightweight CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.stats-container` – Statistics card wrapper
- `.stat-card` – Individual statistics card
- `.icon` – Animated icon
- `.stats-section` – Main section container

---

## Accessibility

Supports `prefers-reduced-motion` to reduce animations for users who prefer less motion.

---

## Use Cases

- Business dashboards
- Portfolio websites
- Startup landing pages
- Analytics sections
