# Filter Drop-Shadow vs Box-Shadow

Compares CSS `filter: drop-shadow()` with `box-shadow` across transparent PNGs, clipped elements, and text. Demonstrates how drop-shadow respects the alpha channel while box-shadow does not.

## Features

- Side-by-side comparison on SVG transparent images and clip-path shapes
- Interactive live editor for offset and blur values
- Reference table of key property differences
- Fully self-contained dark-themed demo

## Usage

Open `demo.html` in a modern browser. Use the sliders to compare how each shadow type renders in real time. No external dependencies required.
