ease-ripple

A CSS-only Material Design-style click ripple effect. No JavaScript required.

Default Ripple

White ripple on dark surfaces

Dark Ripple

Dark ripple for light surfaces

Primary Ripple

Tinted to match brand color

Success Ripple

Green-tinted feedback

Danger Ripple

Red-tinted destructive action

Top-Left Origin

Ripple expands from corner

/* Basic usage */
<button class="ease-ripple">Click me</button>

/* Dark ripple on light surface */
<div class="ease-ripple ease-ripple-dark">...</div>

/* Custom color & speed */
<button class="ease-ripple"
  style="--ease-ripple-color: rgba(0,0,0,0.2);
  --ease-ripple-duration: 0.8s">
  Custom Ripple
</button>
