# Dashboard Analytics Floating Orbit Accordion

A modern **Pure CSS Floating Orbit Accordion** designed for **Dashboard Analytics** interfaces. The component features a subtle floating orbit animation, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Floating Orbit animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Dashboard analytics inspired design
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:5s;
    --easing:ease-in-out;
    --primary:#22d3ee;
    --secondary:#3b82f6;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Border styling

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
