# Animated File Diff Highlight Viewer

A CSS-only file diff viewer that highlights added, removed, and modified lines with animations.

## Features

- Added lines slide in from the left
- Removed lines highlighted with strike-through
- Modified lines pulse gently
- Line number gutter
- Responsive layout
- Pure HTML & CSS

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in a browser.

## Classes

- `.ease-diff-added`
- `.ease-diff-removed`
- `.ease-diff-modified`
- `.ease-diff-line-number`

## Purpose

Useful for:
- Git diff previews
- Changelogs
- Version comparison UIs
- Code review demonstrations

Created for EaseMotion CSS.
