# Music Streaming Dashboard

A responsive **Music Streaming Dashboard** built using **HTML5** and **CSS3** that showcases streaming analytics, listener engagement, playlist performance, artist statistics, and subscriber insights through a modern dashboard interface.

---

## Features

- Responsive dashboard layout
- Music streaming overview cards
- Listening activity tracker
- Top artists panel
- Playlist performance monitor
- Recent streaming activity
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Active Listeners

### Music Widgets

- Listening Activity

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
