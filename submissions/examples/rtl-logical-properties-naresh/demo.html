🛠️ Logical Direction Implementation

By declaring a CSS variable --ease-direction: 1 at the root and updating it to -1 inside the [dir="rtl"] scope, you can reverse horizontal values dynamically using calc() in your keyframe definitions:

:root {
  --ease-direction: 1; /* 1 for LTR, -1 for RTL */
}

/* Reverse the direction multiplier in RTL scope */
[dir="rtl"] {
  --ease-direction: -1;
}

/* Define keyframes using calc with direction variable */
@keyframes logicalSlideLeft {
  from {
    transform: translateX(calc(-80px * var(--ease-direction)));
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

.ease-slide-left {
  animation: logicalSlideLeft 1s cubic-bezier(0.16, 1, 0.3, 1) forwards;
}
