Text Animation Utilities

Five dedicated CSS utility classes for animated text effects: glow, shimmer, typewriter, glitch, and gradient-flow.

.ease-text-glow

Smoothly pulsing text-shadow glow. Animates between a subtle halo and a more intense bloom using the primary brand colour. Lightweight — only animates text-shadow.

Glowing Text 
<h1 class="ease-text-glow">
  Welcome to EaseMotion CSS
</h1>
.ease-text-shimmer

A shimmering gradient that sweeps across the text using background-clip: text. Customisable via --text-shimmer-from, --text-shimmer-via, and --text-shimmer-to.

Stunning UI Effects 
<h1 class="ease-text-shimmer">
  Stunning UI Effects
</h1>
.ease-text-typewriter

Simulates text being typed character by character, then holds with a blinking cursor. Set --text-typewriter-length and --text-typewriter-steps to match your text.

EaseMotion CSS 
<!-- Set length = character count of text -->
<p class="ease-text-typewriter"
   style="--text-typewriter-length: 18ch;
          --text-typewriter-steps:  18;">
  EaseMotion CSS
</p>
Note: Use the replay button below to restart.
.ease-text-glitch

A modern digital glitch effect using clip-path slice animations on ::before / ::after pseudo-elements for cyan and magenta ghost layers. Requires a data-text attribute matching the element's text content for the pseudo-element clones.

GLITCH MODE 
<!-- data-text must match inner text -->
<h1 class="ease-text-glitch"
    data-text="GLITCH MODE">
  GLITCH MODE
</h1>
.ease-text-gradient-flow

An animated flowing multi-stop gradient text. Uses background-position animation for smooth, continuous movement. Customisable via --text-gradient-flow-* tokens.

Gradient Flow 
<h1 class="ease-text-gradient-flow">
  Gradient Flow
</h1>

<!-- Custom colours -->
<h1 class="ease-text-gradient-flow"
  style="
    --text-gradient-flow-from: #6c63ff;
    --text-gradient-flow-via1: #a78bfa;
    --text-gradient-flow-via2: #ec4899;
    --text-gradient-flow-to:   #6c63ff;
    --text-gradient-flow-duration: 4s;
  ">
  Custom Gradient
</h1>
Composable Example

Combine multiple text animation utilities. Each class is independent and can be stacked on the same element where CSS properties don't conflict.

Glow Effect Shimmer Effect Gradient Flow 
<h1 class="ease-text-glow">Glow Effect</h1>
<h2 class="ease-text-shimmer">Shimmer Effect</h2>
<h2 class="ease-text-gradient-flow">Gradient Flow</h2>
