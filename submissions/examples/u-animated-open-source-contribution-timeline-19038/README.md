# Animated Open Source Contribution Timeline

A modern timeline component for visualizing open-source milestones, pull requests, issues, contribution streaks, and achievements.

## Features

- Contribution statistics cards
- Animated timeline entries
- Milestone indicators
- Achievement badges
- Hover interactions
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Contribution milestones
- Timeline dates
- Achievement badges
- Statistics values
- Colors and animations

## Use Cases

- GSSoC portfolios
- GitHub profile showcases
- Developer portfolios
- Open-source dashboards
- Contribution tracking pages
