# Elastic Font Switcher

A responsive Portfolio-inspired **Elastic Font Switcher** built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Elastic button animation
- Keyboard accessible radio controls
- CSS custom properties
- Supports `prefers-reduced-motion`
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Customization

Edit the CSS variables in `:root` to change colors, spacing, border radius, and animation timing.
