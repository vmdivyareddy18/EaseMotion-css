Column Width

Specifies the optimal width of columns in a multi-column layout.

CSS Property: column-width

Variants Demo

column-width: 180px Column Width variant
column-width: 250px Column Width variant
column-width: 350px Column Width variant
column-width: 450px Column Width variant
180px columns

Multi-column text with 180px column width. Browser fits as many columns as possible. Great for creating magazine-style layouts with balanced line lengths.

250px columns

Wider columns create a more traditional newspaper-style layout with longer, more readable lines. Good for long-form content and documentation.

350px columns

Even wider columns provide generous line lengths suitable for articles and editorial content where readability is the primary concern.

Usage

<div class="column-width">Content here</div>
