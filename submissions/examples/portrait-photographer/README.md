# 📷 Portrait Photographer Portfolio

A responsive portfolio landing page for freelance portrait photographers built using EaseMotion CSS.

## Features

- Hero Section
- Portfolio Gallery
- About Section
- Pricing Packages
- Client Reviews
- Booking CTA
- Responsive Footer
