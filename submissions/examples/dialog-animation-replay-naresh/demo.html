Dialog Replay Workarounds

Replaying animations inside native HTML dialog elements

1. JS Reflow Toggling (Keyframe animations)

Removes and re-adds the animation class (e.g. .ease-fade-in) before calling .showModal(). This forces the browser to evaluate the keyframe animation from the beginning on every open event.

2. CSS @starting-style (Modern browsers, transition-based)

CSS-only solution using transition properties. Defining initial state within @starting-style triggers the transition naturally every time the modal goes from closed (display: none) to open.
