# Animated Gradient Text Component

A modern CSS-only animated typography component featuring moving gradients, glowing colors, smooth transitions, and responsive design.

---

## Features

- Animated moving gradient text
- Vibrant multi-color effects
- Smooth infinite animation
- Glowing typography style
- Responsive layout
- Lightweight HTML & CSS only
- No JavaScript required

---

## Files Included

- `demo.html` — Demonstrates animated gradient text.
- `style.css` — Contains gradients, animations, and responsive styles.

---

## Use Cases

Perfect for:

- Hero section headings
- Landing pages
- Advertisements
- Portfolio websites
- Technology products
- Modern UI designs

---

## Customization

You can modify:

- Gradient colors
- Animation speed
- Font size
- Glow intensity
- Background styles

---

Compatible with all modern browsers:

- Safari
