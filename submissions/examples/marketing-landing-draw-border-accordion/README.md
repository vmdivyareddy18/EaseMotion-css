# Marketing Landing Page Draw-Border Line Accordion

A modern **Pure CSS Draw-Border Line Accordion** designed for **Marketing Landing Page** layouts. The component features a smooth animated border effect, responsive design, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Draw-Border Line animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Marketing landing page inspired design
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.45s;
    --easing:ease;
    --primary:#f97316;
    --secondary:#ec4899;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Border styling

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
