# Status Page Incident Banner

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A status page incident banner designed to communicate active outages, degraded performance, and ongoing investigations.

## Features

- Incident status banner
- Severity badge
- System status dashboard
- Service health indicators
- Responsive layout
- Slide-down animation
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="incident-banner">

    <span class="severity-badge">
        Major Incident
    </span>

</div>
```

## Main Classes

```css
.incident-banner
.incident-content
.severity-badge
.status-card
.service-row
.healthy
.degraded
```
