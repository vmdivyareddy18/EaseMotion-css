# Animation Duration Examples

## Purpose

This demo compares different animation durations to help users choose appropriate timing.

---

## Durations Included

- 0.5 seconds
- 1 second
- 2 seconds
- 3 seconds

---

## Folder Structure

```
animation-duration-examples/
│
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

Open `demo.html` in your browser.

Observe how changing the duration affects the speed of the animation.

---

## Learning Outcomes

- Understand animation speed.
- Compare duration values visually.
- Learn when fast or slow animations are appropriate.

---

## Benefits

- Beginner friendly
- Easy to extend
- Helpful reference for developers
