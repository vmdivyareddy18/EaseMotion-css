# Ease File Upload Dropzone

## Overview

Ease File Upload Dropzone is a reusable upload component featuring a modern dashed border, floating upload icon, hover glow, and smooth animations built entirely with HTML and CSS.

---

## Features

- 📁 Drag-and-drop inspired UI
- ⬆️ Floating upload icon
- ✨ Animated dashed border
- 🚀 Hover glow effect
- 📱 Responsive layout
- ⚡ Pure HTML & CSS
