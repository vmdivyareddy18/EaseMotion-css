# Search with Debounce – Loading spinner while typing

> A production-ready search component with debounced input that shows a loading spinner while typing and displays results with smooth animations. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Debounced search** – Prevents excessive API calls
- ✅ **Loading spinner** – Visual feedback while typing
- ✅ **Clear button** – Quick clear with one click
- ✅ **Result highlighting** – Matches are highlighted
- ✅ **Result count** – Shows number of results
- ✅ **Search time** – Displays response time
- ✅ **API call counter** – Tracks search requests
- ✅ **Keyboard shortcuts** – Ctrl+/ (focus), Escape (clear)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
