# Expandable Card – Click to expand with smooth height transition

> A production-ready expandable card component that reveals content with a smooth height transition when clicked. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Smooth height transition** – Elastic easing animation
- ✅ **Click to expand/collapse** – Toggle with one click
- ✅ **Keyboard accessible** – Enter/Space to toggle
- ✅ **Accordion mode** – Only one card open at a time
- ✅ **Bulk operations** – Expand all, collapse all, toggle all
- ✅ **Dynamic cards** – Add new cards programmatically
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Highly customizable** – Colors, animation speed, behavior
