# Ease Count-Up on Scroll

This demo implements a **scroll-triggered count-up animation** using `IntersectionObserver`.

## 🧠 How It Works
- Each `.stat` element starts at `0`.
- When scrolled into view, the number counts up to its `data-target` value.
- The animation runs **once per session** (one-shot trigger).

## 🧩 Files
- `demo.html` — main demo with JavaScript logic.
- `style.css` — styling for layout and animation.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Scroll down until the stats section enters view.
3. Watch the numbers smoothly count up!

## 🧰 Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- IntersectionObserver API
