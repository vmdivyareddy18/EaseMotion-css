# Animated CSS-Only Weather Icons

A set of pure CSS weather icons featuring the Sun, a Cloud, and Rain.

## Features

- **No Images or SVGs**: Constructed entirely out of `div` elements and pseudo-classes.
- **Engaging Animations**: Combines keyframe animations for spinning, floating, and pulsating.
- **Zero JavaScript**: Runs entirely on CSS.
- **Easily customizable**: Colors and sizes can be easily modified in the CSS.

## Classes Introduced

This component uses standard HTML/CSS. If integrated into EaseMotion CSS, these could be transformed into utility classes like:
- `.ease-weather-sun`
- `.ease-weather-cloud`
- `.ease-weather-rain`
