# Issue #22759 - Advanced: E-commerce - Art Prints Posters

## What this submission adds
- A complete premium one-page e-commerce landing page for an art prints and posters brand.
- A direct-open `demo.html` experience built with only HTML, CSS, and existing EaseMotion CSS classes.
- A fully custom hero visual using inline SVG and CSS-only framed poster mockups with gallery-wall styling.

## Required sections included
- Hero with lifestyle art mockup
- Collections by style
- Room visualizer CTA
- Artist spotlight section
- Framing options
- Reviews/testimonials
- Custom print CTA
- Footer

## E-commerce details included
- Four art print product cards
- Print name, style/category, artist name, available sizes, price, and CTA on each card
- Style collections: Abstract, Minimal, Botanical, Typography
- Framing options and room visualizer callout
- Custom print consultation CTA

## How to open demo.html
1. Open `submissions/examples/22759-art-prints-posters/demo.html` directly in a browser.
2. No server, package install, or build step is required.

## Responsive behavior
- Desktop: editorial two-column hero, multi-column product and content grids.
- Tablet: stacked feature sections with two-column product/review layouts where space allows.
- Mobile: single-column flow, touch-friendly buttons, readable typography, and simplified hero badge placement.

## Browser compatibility
- Designed for current Chrome, Edge, Firefox, and Safari releases.
- Uses inline SVG, CSS gradients, modern layout primitives, and graceful reduced-motion handling.
