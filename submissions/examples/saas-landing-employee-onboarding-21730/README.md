# Business Landing — Employee Onboarding Software SaaS (`saas-landing-employee-onboarding-21730`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for HR Tech and Employee Onboarding software, using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

An Employee Onboarding SaaS landing page needs to convey efficiency, automation, and a welcoming experience (similar to Workday, BambooHR, or Rippling). This template uses a clean, modern aesthetic with highly legible typography and dynamic layouts.

The layout includes:
- A hero section featuring an animated CSS-built onboarding checklist mockup
- A logo banner for enterprise trust
- A 3-column platform feature grid
- A split layout showcasing time-to-productivity stats with animated cards
- An integration ecosystem pill grid for HRIS connectivity
- Customer testimonials highlighting saved time
- A 2-tier per-employee pricing layout
- A high-impact CTA section

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth, authoritative text and card entrances)
- `.ease-slide-right` / `.ease-slide-left` / `.ease-slide-up` (For layout structural animations)
- `.ease-zoom-in` / `.ease-bounce-in` (For drawing attention to primary CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For buttons, feature cards, and mockups)
- `.ease-hover-glow` (For subtle link interactions)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
