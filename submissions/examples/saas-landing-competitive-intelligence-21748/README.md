# Business Landing — Competitive Intelligence SaaS (`saas-landing-competitive-intelligence-21748`)

This proposal demonstrates how to construct a professional, high-converting B2B SaaS landing page (specifically for a Competitive Intelligence tool like Crayon or Klue) using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A B2B SaaS landing page needs to look trustworthy, clean, and modern. This template achieves a premium aesthetic through careful use of typography (Inter), a refined color palette (Slate & Blue), and smooth, professional animations.

The layout includes:
- A hero section with a CSS-built dashboard mockup
- A tracked sources logo banner
- A 3-column feature grid
- A split-layout explanation section
- A 3-tier pricing table with a "Most Popular" callout
- A high-impact CTA block

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth, professional text and card entrances)
- `.ease-slide-right` / `.ease-slide-left` / `.ease-slide-up` (For layout structural animations)
- `.ease-zoom-in` / `.ease-bounce-in` (For drawing attention to primary CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For buttons, feature cards, and mockups)
- `.ease-focus-ring` (For accessible form inputs)
- `.ease-hover-glow` (For subtle link interactions)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
