# ease-marquee-3d

A CSS-only 3D perspective marquee that scrolls text horizontally with a `rotateX` tilt for depth.

**Usage:** Include `style.css` only. The `.marquee-3d-inner` element uses `perspective: 400px` and `rotateX(14deg)` to create depth. Content must be duplicated for seamless looping. Adjust `animation-duration` for speed.
