# Ease Backflip Out

### 1. What does this do?

This animation causes an element to flip backward over its horizontal axis (X-axis) while fading out, creating a dynamic exit effect.

### 2. How is it used?

Add the `backflip-out` class to any element to trigger the exit animation.

```html
<div class="backflip-out">...</div>
```

### 3. Why is it useful?

It provides a high-energy transition for elements being removed from the DOM or hidden from view. It complements entry animations like "backflip-in" and adds variety to the framework's rotation-based effects.
