# ::first-letter Dropcap

## Description
Explores the `::first-letter` pseudo-element to create decorative drop caps in three distinct typographic styles: serif (gold, shadowed), script (purple, glowing), and brush (red, bold). Each article demonstrates a different mood and font pairing while maintaining a clean dark-themed layout.

## Usage
Open `demo.html` in any modern browser. Each article displays its first letter styled with a unique font, size, color, and shadow effect. The drop caps use `float: left` for proper text wrapping. Resize the browser to see responsive scaling.

## Browser Support
`::first-letter` is supported in all modern browsers (Chrome 1+, Firefox 1+, Safari 1+, Edge 12+, Opera 7+). The `float` technique for drop caps works universally. Decorative borders and shadows degrade gracefully in older browsers. Respects `prefers-reduced-motion` by disabling all animations.
