# Double Focus Border Glow

## What does this do?

Creates an interactive double-focus outline ring designed to provide high visibility and accessibility for keyboard navigation focus states.

## How is it used?

Apply the `double-focus-glow-uj` class to focusable elements like inputs, buttons, or links:

```html
<input type="text" class="double-focus-glow-uj" placeholder="Tab to focus me...">
```

## Why is this useful?

It aligns with EaseMotion's focus on lightweight, highly visible, and accessible focus outlines. It avoids generic browser-default outlines while guaranteeing excellent color contrast on both light and dark backgrounds.
