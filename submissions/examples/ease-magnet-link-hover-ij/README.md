# Magnet Link Hover (#36805)

Links that spring toward the cursor on hover, simulating a magnetic pull. An underline trails behind in the accent color.

## CSS Custom Properties

| Property           | Default   | Description                        |
| ------------------ | --------- | ---------------------------------- |
| `--link-color`     | `#0a84ff` | Default link text color            |
| `--magnet-color`   | `#ff375f` | Hover / accent color               |
| `--pull-distance`  | `12px`    | Distance the link translates       |
| `--hover-duration` | `0.35s`   | Duration of the magnetic animation |

## Interactive Controls

Use the variable panel to adjust colors, pull distance, and animation speed. The effect uses an elastic cubic-bezier for a lively feel.
