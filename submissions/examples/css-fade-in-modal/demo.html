Accessibility Features Integration:

  • Native Tab-indexing: Open/Close triggers use standard anchor elements (`<a>`), which natively support keyboard interaction (`Enter` key) and auto-focusing on activation.
  • Focus Shifting: Setting `tabindex="-1"` on the overlay enables browsers to programmatically direct active focus to the modal card on transition.
  • Screen-Reader Landmarks: Built using appropriate WAI-ARIA tags, specifically `role="dialog"`, `aria-modal="true"`, and header labels referencing `aria-labelledby`.
  • Motion Comfort: Built-in `@media (prefers-reduced-motion)` resets the transitions instantly for users with sensitivity to animations.
