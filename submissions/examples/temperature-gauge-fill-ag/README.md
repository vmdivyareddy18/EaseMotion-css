# Temperature Gauge Fill

## What does this do?
An animated temperature gauge/thermometer fill effect where a colored column rises and pulses to indicate temperature level.

## How is it used?
Apply the .temp-gauge class to a container. The .gauge-fill child animates its height to represent temperature, with .hot and .cold modifiers for color states.

## Why is this useful?
Provides a visual temperature reading pattern for weather apps, dashboards, IoT interfaces, and environmental monitoring without JavaScript.
