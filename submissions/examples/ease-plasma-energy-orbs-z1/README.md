# Ease Plasma Energy Orbs

## Overview

A futuristic animation featuring glowing plasma orbs that pulse with soft energy using only HTML and CSS.

## Features

* Animated glowing orbs
* Plasma-inspired colors
* Soft pulse animation
* Responsive layout
* Lightweight CSS
* Pure HTML & CSS

Optimized for desktop, tablet, and mobile devices.
