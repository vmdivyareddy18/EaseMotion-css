# Form Autosave Conflict Resolver

## What does this do?

Shows local draft, remote version, conflict, and resolved autosave states.

## How is it used?

```html
<section class="conflict-panel has-conflict">
  <article class="version-card is-local">Local draft</article>
</section>
```

## Why is it useful?

Autosave workflows need a calm way to explain conflicting edits and resolution state.
