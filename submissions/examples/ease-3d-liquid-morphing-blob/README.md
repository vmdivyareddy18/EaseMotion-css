# 3D Liquid Morphing Blob

A pure CSS organic, morphing liquid blob animation that changes shape seamlessly and features an orbiting border and 3D inner shadow effects.

## Features
- **Organic Morphing**: Uses complex keyframe border-radius animations to simulate a liquid or biological shape.
- **3D Depth**: Utilizes inset box-shadows to give the blob a 3D bubble-like appearance.
- **Hover Transitions**: Smoothly scales up and transitions the gradient colors when hovered.

## How to use

```html
<div class="ease-liquid-blob-container">
    <div class="ease-liquid-blob">
        <span class="ease-blob-text">Hover Me</span>
    </div>
</div>
```
