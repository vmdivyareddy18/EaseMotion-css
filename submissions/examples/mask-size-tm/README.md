# Mask Size — EaseMotion CSS Utilities

CSS `mask-size` utility classes for the EaseMotion CSS framework.

## Quick Start

```html
<link rel="stylesheet" href="style.css">
```

## Utility Classes

| Class | Declaration |
|-------|-------------|
| `.mask-auto` | `mask-size: auto;` |
| `.mask-cover` | `mask-size: cover;` |
| `.mask-contain` | `mask-size: contain;` |
| `.mask-100` | `mask-size: 100%;` |
| `.mask-50` | `mask-size: 50%;` |
| `.mask-25` | `mask-size: 25%;` |
| `.mask-75` | `mask-size: 75%;` |
| `.mask-150` | `mask-size: 150%;` |
| `.mask-200` | `mask-size: 200%;` |

## Responsive Variants

Prefix with `sm-`, `md-`, `lg-` for responsive behavior:

```html
<div class="util-class sm-util-class md-util-class lg-util-class">
  Responsive Mask Size
</div>
```

## Dark Mode

Use the `-dark` variant:

```html
<div class="util-class-dark">
  Dark mode Mask Size
</div>
```

## Reduced Motion

Use the `-nomotion` variant:

```html
<div class="util-class-nomotion">
  No motion Mask Size
</div>
```

## Framework Integration

All utilities use EasMotion design tokens from `core/variables.css`: `--ease-primary`, `--ease-secondary`, `--ease-accent`, `--ease-success`, `--ease-danger`, `--ease-warning`, `--ease-info`.
