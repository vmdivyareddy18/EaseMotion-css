# Flash Chip

A responsive animated chip component built for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Flash highlight animation on hover
- Responsive layout
- Accessible semantic HTML
- No JavaScript required
- Uses EaseMotion naming conventions

## Files

```
demo.html
style.css
README.md
```

## Animation

The component uses a moving glossy highlight that sweeps across the chip on hover to create a flash effect while maintaining smooth transitions.
