# 🚀 Asteroids

A classic Asteroids arcade game built with HTML, CSS, and JavaScript. Control a triangular ship, shoot asteroids, and survive in the depths of space!

## 🎮 How to Play

1. Click **"Start Game"** or press **Space** to begin
2. Use **Arrow Keys** to control your ship
   - **↑** – Thrust forward
   - **←** / **→** – Rotate ship
   - **Space** – Shoot bullets
   - **H** – Hyperspace jump (escape danger!)
3. **Destroy all asteroids** to advance to the next level
4. **Avoid colliding** with asteroids
5. **Score points** by destroying asteroids (larger asteroids give fewer points)

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `↑` | Thrust Forward |
| `←` / `→` | Rotate Ship |
| `Space` | Shoot Bullet |
| `H` | Hyperspace Jump |
| `Space` (Menu) | Start Game |
| `R` | Reset Game |

## ✨ Features

- 🚀 **Classic Asteroids Gameplay** – Shoot and destroy asteroids
- 🔺 **Triangular Ship** – Smooth rotation and thrust mechanics
- 💥 **Asteroid Splitting** – Large asteroids break into smaller ones
- ⭐ **Scoring System** – Points based on asteroid size (20, 50, 100)
- 📈 **Progressive Difficulty** – More asteroids with each level
- 🏆 **High Score** – Saved in browser local storage
- ❤️ **Lives System** – 3 lives to survive
- 🌀 **Hyperspace Jump** – Teleport to escape danger
- ✨ **Particle Effects** – Explosions and thrust effects
- 🎨 **Beautiful Visuals** – Glowing ship, detailed asteroids, starfield
- 📱 **Responsive Design** – Works on desktop and mobile
