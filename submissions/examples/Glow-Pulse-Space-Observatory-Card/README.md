# Glow Pulse Space Observatory Card

A responsive Space Observatory themed card featuring a smooth Glow Pulse animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Glow Pulse animation
- Space Observatory interface
- Observatory statistics
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--glow
--bg
--surface
--primary
--secondary
--text
--muted
--radius
```
