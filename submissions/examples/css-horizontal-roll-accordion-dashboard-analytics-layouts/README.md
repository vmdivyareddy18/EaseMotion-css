# CSS Horizontal Roll Accordion - Dashboard Analytics Layouts

A pure CSS animated accordion designed for dashboard analytics interfaces with a smooth horizontal roll transition.

## Features

- Pure HTML and CSS
- No JavaScript required
- Dashboard-focused UI design
- Smooth horizontal roll animation
- Responsive layout
- Keyboard accessible using native details/summary
- Custom animation controls using CSS variables


## Usage

```html
<div class="accordion">

    <details>

        <summary>
            Analytics Section
        </summary>

        <div class="content">
            Dashboard information
        </div>

    </details>

</div>
