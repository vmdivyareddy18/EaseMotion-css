# CSS Speed Dial Component

## What does this do?
A floating action button that expands to reveal multiple action items in vertical, horizontal, or circular layouts using CSS checkbox hack or hover.

## How is it used?
```html
<div class="spd-fab spd-up">
  <input type="checkbox" id="tog" class="spd-toggle">
  <label for="tog" class="spd-main">+</label>
  <div class="spd-actions">
    <a href="#" class="spd-action">A</a>
  </div>
</div>
```

## Why is it useful?
Provides a compact multi-action UI pattern for mobile-friendly interfaces. Pure CSS toggle activation, multiple layout directions, and customizable colors. No JavaScript needed.
