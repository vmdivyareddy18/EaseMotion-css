# scrollbar-gutter-stable

## Purpose
Demonstrates `scrollbar-gutter: stable` for preventing layout shift when scrollbars appear, comparing fixed vs overlay scrollbars.

## Features
- Side-by-side comparison of `stable` vs `auto` scrollbar-gutter
- Fixed vs overlay scrollbar behavior
- Content width remains stable when scrollbar visibility changes
- Visual measurement indicators

## Usage
Open `demo.html`. Scroll the content areas and observe how the stable gutter prevents horizontal layout shifting.
