# Animated Pricing Table Comparison Component

A modern animated pricing table showcasing multiple plans with smooth reveal effects, highlighted recommendations, and responsive layouts.

## Features

- Animated card reveal effects
- Featured plan highlighting
- Hover interactions
- Responsive layout
- SaaS-style pricing UI
- Clean comparison structure

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in your browser.

## Customization

You can customize:

- Plan names
- Pricing values
- Colors
- Feature lists
- Animation timings
- Featured plan styling

## Use Cases

- SaaS websites
- Startup landing pages
- Product pricing sections
- Subscription platforms
- Business websites
