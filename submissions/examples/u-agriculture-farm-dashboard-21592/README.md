# Pet Care Dashboard

A responsive **Pet Care Dashboard** built using **HTML5** and **CSS3** that demonstrates veterinary appointment scheduling, pet health monitoring, vaccination tracking, and clinic management through an interactive dashboard interface.

---

## Features

- Responsive dashboard layout
- Pet care overview cards
- Appointment progress tracker
- Vaccination schedule panel
- Pet health status
- Recent checkup activity
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Registered Pets
- Appointments Today
- Vaccinations Due
- Active Veterinarians

### Pet Care Widgets

- Appointment Progress
- Vaccination Schedule
- Pet Health Status

### Activity

- Recent Checkups
- Notifications Panel

### Quick Actions

- Add Pet

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
