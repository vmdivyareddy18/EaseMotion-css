# Animated Digital Resume Portfolio

A modern resume and portfolio component featuring animated profile cards, skill progress bars, project showcases, achievement badges, and responsive layouts.

## Features

- Profile hero card
- Animated skill bars
- Project showcase section
- Achievement badges
- Hover interactions
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Personal information
- Skills and progress values
- Projects
- Achievements
- Colors and animations

## Use Cases

- Student portfolios
- Developer portfolios
- Personal websites
- Resume showcases
- Open source profiles
