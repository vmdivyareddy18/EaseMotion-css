# Ease Stepper Vertical

This demo implements a **vertical step progress indicator** with animation and checkmarks for completed steps.

## 🧠 How It Works
- Steps are stacked vertically.
- A connecting line links each step.
- The active step is highlighted with a smooth animation.
- Completed steps show a checkmark.

## 🧩 Files
- `demo.html` — main demo with JavaScript logic.
- `style.css` — styling and animation.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Watch the steps activate sequentially every 2 seconds.
3. Customize step content and timing as needed.

## 🧰 Technologies
- HTML5
- CSS3 (transitions, transforms)
- JavaScript (timed animation)
