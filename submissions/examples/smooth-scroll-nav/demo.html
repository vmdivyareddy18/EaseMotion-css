🚀 Smooth Scrolling

❓ FAQ

Does smooth scrolling work on all browsers?

Yes, scroll-behavior: smooth is supported in all modern browsers. The prefers-reduced-motion media query disables it for users who prefer reduced motion.

What is :target?

The :target pseudo-class matches the element whose id matches the URL fragment, letting you style the currently-active section.

Why use scroll-margin?

Without scroll-margin, a fixed or sticky header would overlap the top of the targeted section. scroll-margin-block-start adds breathing room equal to the nav height.
