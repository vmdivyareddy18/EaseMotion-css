# ClearBudget App Landing Page

This submission resolves issue `#20994` by providing a complete, high-fidelity landing page for a personal finance mobile app. The design emphasizes a trustworthy, financial wellness aesthetic using deep navies and emerald greens.

## 🚀 Deployment Instructions
This page is 100% self-contained and utilizes zero external dependencies (no CDNs, no external images, no JavaScript frameworks). 
1. Navigate to this directory.
2. Open `demo.html` in any modern web browser.
3. Everything, including inline SVG icons and native CSS animations, runs locally via the `file:///` protocol.

## 🎨 EaseMotion Classes Showcased
This layout extensively utilizes the EaseMotion framework utility conventions. The following classes have been polyfilled in `style.css` for isolation purposes:

* **Animations**: `ease-animate-slide-up`, `ease-animate-fade-in`, `ease-animate-slide-up-delay-*`
* **Interactions**: `ease-hover-scale`, `ease-hover-float`, `ease-hover-shadow-xl`
* **Layouts**: `ease-flex`, `ease-grid`, `ease-hidden`, `md:ease-flex-row`
* **Components**: `ease-btn-primary`, `ease-mockup-card`, `ease-card`
