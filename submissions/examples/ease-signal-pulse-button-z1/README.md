# Ease Signal Pulse Button

## Overview

A CSS-only button with a continuous pulse animation designed for call-to-action elements.

## Features

- Pure HTML & CSS
- Pulse animation
- Responsive
- Lightweight
- Reusable

## Folder Structure

```
ease-signal-pulse-button-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox and Safari.

## Why This Example

This example provides a reusable animated button suitable for landing pages, dashboards, and modern interfaces while keeping the implementation minimal.
