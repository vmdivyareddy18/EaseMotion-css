# CSS 3D Perspective Tilt Modal (Glassmorphism UI)

A **pure CSS 3D animated modal component** designed for **Glassmorphism UI layouts** in EaseMotion CSS.

---

## ✨ Features

- Pure CSS 3D perspective animation
- Glassmorphism (blur + transparency)
- Hover-based 3D tilt interaction
- Smooth modal transitions
- Fully responsive design
- Keyboard accessible (ESC support)

---

## 🎯 Purpose

This component enhances modern UI systems with:
- frosted glass effect
- depth-based 3D motion
- interactive hover feedback

Built entirely using CSS, no JavaScript dependencies for animation.

---

## 🧪 Usage

```html
<button class="open-btn" onclick="openModal()">Open Modal</button>

<div class="modal active">
  <div class="modal-content">
    Glass modal content
  </div>
</div>
