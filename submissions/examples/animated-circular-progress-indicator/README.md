# Animated Circular Progress Indicator Component

A modern CSS-only circular progress indicator featuring gradient progress rings, smooth animations, percentage labels, and responsive cards.

## Features

- Circular progress visualization
- Gradient progress rings using CSS conic-gradient
- Smooth entry animation
- Percentage labels inside the circle
- Glassmorphism-inspired cards
- Hover elevation and glow effect
- Fully responsive design
- Lightweight HTML & CSS implementation

## Files Included

- `demo.html` — Demonstration of circular progress indicators
- `style.css` — Contains all styles, gradients, animations, and responsive layouts
