# Student Learning Analytics Dashboard

A responsive dashboard for tracking learning progress, assignments, achievements, and study performance.

## Features

- Learning statistics cards
- Course progress tracking
- Upcoming assignments section
- Achievement badges
- Performance overview
- Responsive dashboard design
- Smooth animations and hover effects

## Dashboard Sections

### Statistics
- Courses Completed
- Active Courses
- Assignments Due
- Study Hours

### Progress Tracking
- Course completion cards
- Progress bars

### Tasks
- Upcoming assignments
- Priority indicators

### Achievements
- Study streaks
- Certificates earned
