# Fluid Toggle Switch (`ease-toggle-has`)

### 1. What does this do?
A custom checkbox toggle switch with organic elastic sliding motion and glowing active state.

### 2. How is it used?
Wrap inside `.toggle-switch` structure:
```html
<label class="toggle-switch">
  <input type="checkbox">
  <span class="slider"></span>
</label>
```

### 3. Why is it useful?
Replaces raw HTML check boxes with high-performance CSS animated toggles that elevate interface touchpoints.
