# Advanced: Business Landing — Law Firm

A professional, responsive landing page for a modern law firm built using **EaseMotion CSS**. The page demonstrates how existing EaseMotion animation and interaction classes can be used to create an elegant business website without custom animation keyframes or JavaScript.

## Features

* Responsive navigation bar
* Professional hero section with consultation CTA
* Practice areas showcase
* Attorney profile section
* Why Choose Us section
* Case results highlights
* Client testimonials
* Free consultation form
* Office information
* Responsive footer

## EaseMotion CSS Classes Used

### Entrance Animations

* `ease-fade-up`
* `ease-fade-down`
* `ease-fade-left`
* `ease-fade-right`
* `ease-scale-in`
* `ease-slide-up`

### Hover Effects

* `ease-hover-lift`
* `ease-hover-grow`
