# Business Landing — White-label Reporting SaaS (`saas-landing-whitelabel-reporting-21715`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for white-label reporting tools and agency software (similar to AgencyAnalytics, DashThis, or Databox), using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

An agency-focused B2B SaaS needs to project professionalism, customization capabilities, and time-saving automation. This template uses a clean, modern aesthetic with a vibrant primary brand color, soft shadows, and clean typography.

The layout includes:
- A hero section featuring an animated CSS-built branded dashboard mockup (demonstrating the white-labeling value prop)
- An integration ecosystem logo banner
- A dark-mode showcase section explicitly highlighting white-labeling features (custom domain, themes)
- A 6-card platform feature grid
- An agency use-case section demonstrating time saved
- Testimonial cards
- An agency-friendly, per-client tiered pricing layout
- A comprehensive footer

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth text entrances, list items, and chart bars)
- `.ease-slide-up` (For structural layout animations in pricing, features, and the cascading stat cards in the dashboard mockup)
- `.ease-zoom-in` (For drawing attention to the primary mockups)
- `.ease-bounce-in` (For primary trial CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For interactive buttons, feature cards, and the 3D dashboard effect)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
