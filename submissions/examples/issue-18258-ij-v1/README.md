# Animated Checkbox (Issue #18258)

A custom-styled checkbox with an SVG checkmark that draws itself on check using `stroke-dasharray`/`stroke-dashoffset` animation. Configurable color and size.

## Usage

Open `demo.html` in a browser. Click the checkbox to see the checkmark draw in.

## Files

- `demo.html` — HTML5 page with checkbox and SVG checkmark
- `style.css` — Dark theme styles with CSS transition
- `README.md` — This file
