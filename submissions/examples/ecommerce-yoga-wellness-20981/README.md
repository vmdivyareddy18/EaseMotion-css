# App Landing: Yoga & Wellness E-commerce (`ecommerce-yoga-wellness-20981`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a calm, mindful yoga and wellness brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A wellness brand requires a UI that feels like taking a deep breath. This template achieves a serene aesthetic using muted, earthy colors (sage green, clay, sand, stone), ample white space, elegant typography (`Cormorant Garamond` paired with `Outfit`), and extremely smooth, slow transitions.

The layout includes:
- A subtle Announcement Bar.
- A minimalist, sticky Navigation.
- A Hero section featuring abstract CSS shapes that simulate a slow breathing animation.
- A "Shop Categories" grid featuring CSS representations of physical yoga products (a mat unrolling, a block floating).
- A "Style Guide" educational split-section with an elegant accordion layout.
- A curated "Featured Products" grid with a soft blur hover effect for quick add.
- A full-width Community/Classes CTA banner.
- An elegant, typography-focused Reviews section.
- A Subscription Box CTA.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use very slow, smooth `cubic-bezier(0.25, 0.46, 0.45, 0.94)` timings to create a calming, deliberate pace:
- `.ease-fade-up` / `.ease-slide-down` (Configured for slow, soft, low-distance entrances).
- `.ease-breathe` (An ambient animation applied to background shapes to simulate a slow inhale/exhale via scale and opacity).
- `.ease-float-slow` (An ambient, subtle floating animation).
- `.ease-unroll` (A custom CSS animation applied to a block element to simulate a yoga mat unrolling).
- `.ease-hover-fade` (A simple opacity transition for links).
- `.ease-hover-blur` (Applied to product images to soften them upon hover, revealing the Quick Add button).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the minimalist layout, the elegant typography, and the CSS product simulations. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the slow, deliberate entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/ecommerce-yoga-wellness-20981/` directory to avoid modifying core files directly and to ensure zero deletions.*
