# Searchable Filter Tags

## What does this do?
A tag filter component with a search input to filter tags, clickable tag chips with selection state, and an active filters display bar showing selected tags with remove functionality.

## How is it used?
Open demo.html in a browser. Type in the search field to filter the tag list. Click any tag chip to toggle selection. Selected tags appear in the "Active Filters" bar and can be removed individually.

## Why is it useful?
This provides a reusable tag filtering UI pattern useful for e-commerce category filters, blog topic filtering, or any multi-select content filtering interface.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see the filter tags in action.
