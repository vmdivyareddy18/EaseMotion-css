New Component

ease-timeline-vertical

An elegant vertical timeline component that synchronizes line filling and dot glows with viewport scrolling via pure CSS view timelines. Natively collapses to a clean left-aligned list on mobile devices.

Installation & Usage

Include the base timeline class .ease-timeline-vertical and wrap child entries containing dates, titles, and text cards as shown below:

<div class="ease-timeline-vertical">
  <!-- Track progress line -->
  <div class="ease-timeline-line" aria-hidden="true">
    <div class="ease-timeline-line-fill"></div>
  </div>

  <!-- Timeline Entry -->
  <div class="ease-timeline-entry">
    <div class="ease-timeline-dot"></div>
    <div class="ease-timeline-card">
      <span class="ease-timeline-date">Q1 2026</span>
      <h4 class="ease-timeline-title">Ideation</h4>
      <p class="ease-timeline-description">Description here...</p>
    </div>
  </div>
</div>

Modifier Classes

Class Name Target Element Description
.ease-timeline-vertical Container The core timeline wrapper. Centers container and establishes absolute layout context.
.ease-timeline-line Inner Element Static track background track running behind cards.
.ease-timeline-line-fill Inner Element Colored fill bar that scales dynamically with scroll position.
.ease-timeline-entry Inner Element Represents a single step container. Automatically alternates sides (odd/even).
.ease-timeline-vertical--left Container Mod Appended to base class to force a left-aligned layout on desktop viewports.

Design Customization Tokens

Variable Token Default Value Description
--ease-timeline-accent #6366f1 Primary theme color applied to dot borders, active text, and outline triggers.
--ease-timeline-accent-gradient linear-gradient(...) Gradient color block used to color the scroll progress fill line.
--ease-timeline-card-bg #111827 Backdrop color of the descriptive cards.
--ease-timeline-dot-bg #0b0f17 Inner center fill color of the timeline dot markers.
--ease-timeline-radius 16px Border-radius rounding applied to card boxes.
