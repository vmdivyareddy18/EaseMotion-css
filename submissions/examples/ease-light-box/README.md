# Lightbox – Fullscreen gallery with swipe support

> A production-ready lightbox component that displays images in a fullscreen gallery with smooth transitions, swipe support, keyboard navigation, and thumbnail previews. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Fullscreen gallery** – Immersive image viewing
- ✅ **Swipe support** – Touch gestures for mobile
- ✅ **Keyboard navigation** – Arrow keys, Escape, Space
- ✅ **Thumbnail previews** – Quick navigation
- ✅ **Auto-play** – Slideshow mode
- ✅ **Image counter** – Current position indicator
- ✅ **Dynamic content** – Add/remove images
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
