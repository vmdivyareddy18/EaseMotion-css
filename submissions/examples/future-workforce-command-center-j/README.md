# Future Workforce Command Center

A modern workforce analytics dashboard built with pure HTML and CSS.

## Features

- Workforce Overview Statistics
- AI Hiring Analytics
- Skill Gap Tracking
- Project Allocation Metrics
- Team Activity Feed
- Regional Workforce Distribution
- Glassmorphism Design
- Amber-Cyan Dark Theme
- Gradient Highlights
- Hover Animations
- Fully Responsive Layout

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Glassmorphism UI
- Responsive Design
