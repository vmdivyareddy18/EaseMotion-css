Popover guidance

Popover attributes

  • popover or popover="auto" — light dismiss + Escape.
  • popovertarget on trigger links to popover id.
  • popovertargetaction="hide" for explicit close buttons.

Entrance animations

  • Use :popover-open to run EaseMotion keyframes on show.
  • @starting-style enables smooth opacity/transform transitions.
  • Prefer ease-fade-in for hints; ease-zoom-in for menus.

Accessibility

  • Set role="tooltip", menu, or dialog.
  • Return focus to trigger on close (browser handles for auto popovers).
  • Respect prefers-reduced-motion in custom entrance CSS.
