# Scrollbar Gutter Stable Layout

A CSS demo of the scrollbar-gutter property showing how scrollbar-gutter: stable reserves space for the scrollbar, preventing content reflow when scrollbars appear or disappear.

## Files

- `demo.html` - Two side-by-side scrollable panels
- `style.css` - scrollbar-gutter property, comparison layout

## Usage

Open `demo.html` and scroll the panels to see the stable gutter effect.
