# CSS Fade-Out Accordion for Glassmorphism UI Layouts

## Overview

A modern glassmorphism accordion built using only HTML and CSS. The component combines frosted glass effects, smooth fade transitions, and semantic HTML elements to create an elegant FAQ or information panel.

---

## Features

- Pure HTML & CSS
- Glassmorphism interface
- Smooth fade animation
- CSS-only accordion
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript
