Focus Ring Patterns

Accessible focus indicators with :focus-visible, :focus-within, and outline-offset

Default outline

:focus-visible { outline: 2px solid Highlight; }

Offset ring

:focus-visible {
  outline: 3px solid #6366f1;
  outline-offset: 3px;
}

Double ring

:focus-visible {
  outline: 2px solid #f59e0b;
  outline-offset: 2px;
  box-shadow: 0 0 0 4px #1a1a2e;
}

Glow ring

:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px #38bdf8,
              0 0 20px rgba(56,189,248,0.4);
}

Focus-within card

Focus this card — all children inherit the ring

Inner link 
:focus-within { outline: 2px solid #a78bfa; }

Custom checkbox

input:focus-visible + .check-visual {
  outline: 2px solid #34d399;
  outline-offset: 2px;
}

Guidelines

  • Never remove outlines unconditionally — outline: none without a replacement traps keyboard users
  • Use :focus-visible instead of :focus to avoid showing rings on mouse clicks
  • Minimum 2px thickness, ensure 3:1 contrast against the background
  • outline-offset prevents rings from blending into rounded corners
