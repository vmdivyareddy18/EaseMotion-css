# Focus Ring Accessibility Patterns

1. **What does this do?** Compares six accessible focus ring patterns using `:focus-visible`, `:focus-within`, outline-offset, glow shadows, and custom checkbox styling.

2. **How is it used?** Tab through the demo buttons to see each ring style. The patterns use native CSS — no JavaScript required.

3. **Why is it useful?** Removing focus outlines breaks keyboard navigation for users with motor or visual impairments. This demo provides accessible alternatives that are both usable and visually refined.
