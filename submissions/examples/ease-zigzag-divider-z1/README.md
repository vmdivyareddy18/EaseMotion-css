# Ease Zigzag Divider

## Overview

A lightweight animated zigzag divider created using HTML and CSS.

## Features

- Pure HTML & CSS
- Animated zigzag pattern
- Responsive
- Lightweight
- Easy to customize

## Folder Structure

```
ease-zigzag-divider-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why This Example

This example provides a reusable section divider with a subtle animated zigzag pattern suitable for landing pages, documentation, and dashboard layouts.
