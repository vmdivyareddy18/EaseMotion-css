# ease-gradient-shift

A pure CSS animation that smoothly shifts a multi-color gradient across a container using `background-position` and `background-size: 400% 400%`.

## Usage

Open `demo.html` in any modern browser. No build steps or JavaScript required.

## Files

| File       | Description                        |
|------------|------------------------------------|
| demo.html  | HTML5 preview page (dark theme)    |
| style.css  | Animation keyframes and styles     |
| README.md  | This file                          |

## Customisation

Adjust the colour stops in the `background` property and tweak the `background-position` keyframes to change direction or timing.
