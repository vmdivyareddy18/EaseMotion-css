# Letter Spacing Tight on Hover

Letter spacing tightens on hover — pure CSS.

## Features
- Smooth letter-spacing transition from loose to tight
- Pure CSS, no JavaScript

## Usage
```css
.tight-text { letter-spacing: 0.2em; transition: letter-spacing 0.3s; }
.tight-text:hover { letter-spacing: -0.05em; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
