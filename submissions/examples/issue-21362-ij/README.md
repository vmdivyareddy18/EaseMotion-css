# Fish Swimming

1. What does this do? A CSS-only fish that swims back and forth with a tail fin that swishes.

2. How is it used? Place .fish inside .scene. The body is an ellipse with a glow, the tail is a triangle animated with rotation.

3. Why is it useful? Combines translation (swim path) with rotation (tail fin) for layered animal motion using pure CSS.
