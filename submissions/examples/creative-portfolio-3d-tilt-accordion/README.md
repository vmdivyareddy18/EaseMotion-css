# Creative Portfolio 3D Perspective Tilt Accordion

A modern **Pure CSS 3D Perspective Tilt Accordion** designed for **Creative Portfolio** layouts. The component features subtle 3D tilt interactions, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- 3D Perspective Tilt hover animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Creative portfolio inspired design
- Glassmorphism styling
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.45s;
    --easing:cubic-bezier(.2,.8,.2,1);
    --primary:#7c3aed;
    --secondary:#6366f1;
    --accent:#06b6d4;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Border styling
- Glassmorphism appearance

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
