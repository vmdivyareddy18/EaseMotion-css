# Animated Payment Success Checkmark

## Overview

Animated Payment Success Checkmark is a lightweight CSS animation that simulates a successful payment using a checkmark and expanding success waves.

## Features

- CSS-only animation
- Success confirmation effect
- Expanding ripple waves
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="payment-success">
  <span class="checkmark">✓</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
