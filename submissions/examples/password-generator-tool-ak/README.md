# Password Generator Tool

## What does this do?
A configurable password generator with adjustable length (4-32), character type toggles (uppercase, lowercase, numbers, symbols), and a real-time strength indicator.

## How is it used?
Open demo.html in a browser. Adjust the length slider and toggle character types, then click "Generate Password" or change any setting to auto-generate. Click the copy button to copy the password.

## Why is this useful?
Demonstrates a practical utility tool with real-time configuration feedback, strength analysis, and clipboard integration.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
