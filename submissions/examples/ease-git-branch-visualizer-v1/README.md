# Ease Git Branch Visualizer

## Overview

Ease Git Branch Visualizer is a reusable developer-focused UI component that illustrates a Git workflow with animated commit nodes, branch paths, and merge visualization using pure HTML and CSS.

---

## Features

- 🌿 Git branch visualization
- 🔵 Animated commit nodes
- 🌈 Multiple branch colors
- 📍 Branch labels
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Branch labels

## Usage

```html
<div class="graph">
...
</div>
```

## Customization

- Branch colors
- Commit size

- Labels

## Browser Support

Compatible with all modern browsers supporting CSS transforms and animations.
