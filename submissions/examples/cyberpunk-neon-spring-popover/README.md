# CSS Cyberpunk Neon Spring Popover

A pure HTML & CSS popover featuring a spring physics animation and cyberpunk neon styling.

## Features

- Pure HTML & CSS
- Spring-inspired animation
- Cyberpunk neon theme
- Keyboard accessible
- Responsive design
- CSS Custom Properties
- No JavaScript

## Folder Structure

```
cyberpunk-neon-spring-popover/
│── demo.html
│── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser.

## Customization

```css
:root{
  --primary:#ff00ff;
  --secondary:#00ffff;
  --duration:.45s;
  --scale:1.08;
}
```

- Chrome
- Firefox
