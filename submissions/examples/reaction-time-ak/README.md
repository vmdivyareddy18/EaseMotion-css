# Reaction Time Test

## What does this do?
A reaction time test that measures how quickly you respond when the screen turns green. Tracks last, average, and best reaction times.

## How is it used?
Open demo.html in a browser. Click the area to start. Wait for the color to change from purple to green, then click as fast as possible. Your reaction time in milliseconds is displayed.

## Why is this useful?
This provides a cognitive assessment tool for measuring response times. It demonstrates state machine logic, random delay generation, and performance timing.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to test your reactions.
