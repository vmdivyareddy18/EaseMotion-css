# Animation Timing Comparison Demo

## Purpose

This submission demonstrates how different CSS animation timing functions affect motion.

---

## Timing Functions Included

- linear
- ease
- ease-in
- ease-out
- ease-in-out

---

## Folder Structure

```
animation-timing-comparison/
│
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in a browser.
2. Observe how each timing function changes the animation speed.
3. Compare the movement of each box.

---

## Learning Outcomes

- Understand the difference between common CSS timing functions.
- See visual comparisons side by side.
- Learn when each timing function is appropriate.
