# Breathing Status Badge

A CSS-only animated status badge example for EaseMotion CSS.

## What it does

This example shows small status badges with a soft breathing pulse animation. It can be used for user profiles, dashboards, chat apps, admin panels, and team member cards.

## Features

- CSS-only animation
- Online, Away, Busy, and Offline variants
- Soft breathing pulse effect
- Hover lift effect
- Responsive layout
- No JavaScript required
