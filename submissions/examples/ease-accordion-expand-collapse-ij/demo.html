What is EaseMotion?

EaseMotion is a lightweight, modern CSS animation library built for performance and ease of use. It provides pre-styled components, utility classes, and keyframe animations to help developers build engaging UI interactions with minimal effort.

How do I install it?

You can install EaseMotion via npm, or download the CSS file directly and link it in your project. The library works with any framework or vanilla HTML/CSS setup.

Is it free to use?

Yes! EaseMotion is completely free and open source under the MIT license. You can use it in personal and commercial projects without any restrictions.
