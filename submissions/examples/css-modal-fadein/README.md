# CSS Fade-In Modal for SaaS Showcase Layouts

A modern SaaS landing page featuring a smooth CSS fade-in pricing modal. This component is ideal for product showcases, startup websites, software landing pages, and marketing pages.

---

## Features

- ✨ Smooth fade-in modal animation
- 📈 SaaS-style hero section
- 💳 Pricing plans modal
- 🎨 Modern UI with soft shadows
- 📱 Fully responsive layout
- ❌ Close button support
- 🖱️ Click outside to dismiss
- ⌨️ ESC key support
- 🚀 Lightweight (HTML, CSS & Vanilla JavaScript)
- 🌐 No external libraries required

## How to Run

1. Download all three files.
2. Keep them in the same folder.
3. Open `demo.html` in any modern browser.
4. Click **View Pricing** or **Start Free Trial** to open the modal.

## Animation Details

### Overlay

- Fades in using `opacity`
- Dark translucent background
- Smooth transition

### Modal

Initial State:

- `opacity: 0`
- `translateY(35px)`
- `scale(0.95)`

Visible State:

- `opacity: 1`
- `translateY(0)`
- `scale(1)`

This creates a clean fade-and-slide entrance suitable for SaaS products.
