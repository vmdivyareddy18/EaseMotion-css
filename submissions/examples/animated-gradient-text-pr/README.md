# CSS-Only Animated Gradient Text

### What does this do?
Applies a continuous, flowing multi-stop linear gradient fill that shifts across text shapes seamlessly using pure CSS background properties.

### How is it used?
Attach the utility modifier `.ease-gradient-text` onto large headings or emphasis spans.

### Why is it useful?
It provides an elegant, eye-catching focal point for premium branding elements without introducing heavy video assets, asset images, or layout recalculations.
