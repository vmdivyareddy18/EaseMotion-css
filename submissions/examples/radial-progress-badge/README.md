# Radial Progress Badge

## What does this do?
A compact metric badge with a radial progress ring, load animation, and readable center value.

## How is it used?
```html
<article class="progress-badge badge-one">
  <span class="ring"></span>
  <strong>82</strong>
  <small>Quality</small>
</article>
```

## Why is it useful?
It gives EaseMotion CSS a dashboard-friendly progress component for scores, health checks, and completion metrics.
