# App Landing: Astrology & Horoscope App (`app-landing-astrology-21008`)

This proposal provides a complete, copy-paste ready B2C landing page designed for an astrology, horoscope, or mystical entertainment mobile app, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

An astrology application requires a highly atmospheric, mystical aesthetic. This template achieves a "celestial" look using deep space backgrounds (`#070914`), rich metallic gold accents, faint ethereal borders, and classic typography combining Cinzel (a mystical serif) and Montserrat (a modern sans-serif).

The layout includes:
- A purely CSS-rendered background starfield.
- A Hero section featuring a rotating CSS Zodiac Wheel mockup.
- A 3-column features grid highlighting daily horoscopes and chart readings.
- A split layout showcasing a CSS-rendered geometric "Birth Chart" preview.
- A "Trusted by the cosmos" review grid.
- A 2-tier subscription/pricing architecture (Free vs Premium "Oracle" tier).
- Bold, glassmorphic App Store download CTAs.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities, heavily stylized via CSS to have an ethereal, slow-paced transition feel:
- `.ease-fade-up` / `.ease-slide-up` (Configured with a long, slow `1.2s` duration and an ethereal `cubic-bezier(0.22, 1, 0.36, 1)` curve for a floating, mystical reveal)
- `.ease-zoom-in` (Applied to the celestial graphics, incorporating a blur-to-focus filter transition)
- `.ease-slide-left` / `.ease-slide-right` (Used for floating cosmic elements)
- `.ease-hover-lift` (Applied to metallic gold buttons and feature cards)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, the complex rotating zodiac wheel geometry, the birth chart visual, and the atmospheric styling. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the entrance animations. The design is fully responsive and requires zero JavaScript to execute the entrance, hover, or infinite rotating effects.
