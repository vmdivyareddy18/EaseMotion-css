# Global Education Analytics Dashboard

A modern education-focused analytics dashboard built with pure HTML and CSS.

## Features

- Education Hero Section
- Student Overview Statistics
- Course Analytics Cards
- Completion Rate Widgets
- Instructor Metrics Panel
- Learning Outcomes Timeline
- Violet-Green Dark Theme
- Glassmorphism Design
- Responsive Layout
- Accessibility-Friendly Structure
- Smooth Hover Animations

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Glassmorphism Effects
- Responsive Design
