# Animated Bell Notification Ring

## Overview

A reusable bell icon animation featuring a notification ringing effect suitable for dashboards, messaging applications, productivity tools, admin panels, and notification systems.

## Features

- CSS-only animation
- Bell ringing effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="bell-button">
  <span class="bell-icon">🔔</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
