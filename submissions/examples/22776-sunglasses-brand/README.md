# Issue #22776 - Advanced: E-commerce - Sunglasses Brand

## What this submission adds

This submission adds a premium one-page e-commerce landing page for `Solstice Atelier`, a luxury lifestyle sunglasses brand. The page is fully static, built with only HTML and CSS, opens directly in the browser, and combines custom editorial styling with existing EaseMotion CSS utility, card, button, and animation classes.

## Required sections included

- Hero with inline SVG lifestyle-style sunglasses visual
- Collection grid with 4 product cards
- Face shape guide
- Lens technology explainer
- Reviews
- Virtual try-on CTA
- Newsletter signup
- Footer

## How to open demo.html

Open `demo.html` directly in a browser from this folder. No server, build step, JavaScript, external images, or external assets are required.

## Responsive behavior

- Desktop: two-column editorial hero, four-column product grid, multi-column content sections
- Tablet: stacked hero with two-column products and simplified spacing
- Mobile: single-column layout, full-width buttons, static floating badges for readability, and touch-friendly spacing

## Browser compatibility

Tested as standards-based static HTML and CSS intended for modern browsers with support for:

- CSS Grid
- Flexbox
- `backdrop-filter`
- Inline SVG
- CSS gradients

Works best in current versions of Chrome, Edge, Firefox, and Safari.
