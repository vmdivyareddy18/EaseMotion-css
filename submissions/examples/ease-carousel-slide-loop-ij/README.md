# Carousel Slide Loop

A smooth, auto-sliding image carousel with loop functionality built using HTML, CSS, and JavaScript.

## Features

- Auto-slides with configurable duration
- Pause on hover
- Previous / Next navigation buttons
- Dot indicators for direct slide selection
- Smooth `translateX` transitions
- Responsive layout

## Usage

Include the `style.css` in your HTML and use the provided markup structure. Customize via CSS custom properties:

```css
--carousel-duration    /* Transition speed (default: 0.5s) */
--carousel-slide-width /* Maximum carousel width (default: 600px) */
--carousel-bg          /* Page background (default: #f8f9fa) */
--carousel-btn-bg      /* Navigation button background (default: rgba(0,0,0,0.5)) */
```
