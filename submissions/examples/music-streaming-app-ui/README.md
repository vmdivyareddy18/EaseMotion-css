# Animation: Music Streaming App UI

A modern music streaming interface inspired by Spotify, Apple Music, YouTube Music, and Tidal. It includes album cards, navigation, and a persistent music player.

## Features

- Pure HTML & CSS
- Sidebar navigation
- Album showcase cards
- Persistent music player
- Hover interactions
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="album-card">

    <div class="album-cover">🎵</div>

    <h3>Album Title</h3>

</div>
```

## Why is it useful?

Music streaming interfaces combine content discovery, navigation, and media controls into a single experience. This example demonstrates a modern application layout suitable for entertainment, media, and dashboard-style products.
