# Interactive Range Slider Morphing

## Description
An interactive range capacity slider with a floating tooltip indicator that dynamically updates layout and values based on user input.

## Installation
Drop the CSS styles in your layouts and import the script rules.

## Usage
Standard HTML structures with custom input elements.

## Accessibility
- Implements aria-valuemin, aria-valuemax, and aria-valuenow attributes.
- Fully focusable and keyboard navigable thumb.
