🚫 Layout Thrashing Detection

The benchmark flags any animation that triggers layout/reflow by animating physical properties like width, height, top, left, or margin. Only transform and opacity are GPU-composited and safe.

PropertyGPU Composited?ExampleBenchmark Verdict
opacity ✓ Yes animate-fade Pass
transform ✓ Yes animate-slide-in Pass
width / height ✗ No (reflow) animate-grow-width Fail
top / left ✗ No (reflow) animate-slide-top Fail
margin ✗ No (reflow) animate-slide-margin Fail

⚠️ Layout Thrash Detection Logic

During the 3-second measurement window, the script injects requestAnimationFrame callbacks and checks element.offsetHeight before and after each frame. If the layout dimensions change on every frame (forced synchronous layout), the animation is flagged as causing layout thrashing. The benchmark script exits with code 1.

Safe pattern: transform: translateX(100px) — No reflow, GPU composited ✅

Unsafe pattern: left: 100px — Triggers reflow every frame ❌

📜 scripts/benchmark-animations.mjs

Playwright script that launches a headless browser, creates HTML with each animation class, measures FPS via requestAnimationFrame counter over 3 seconds, and reports results.

// scripts/benchmark-animations.mjs import { chromium } from 'playwright'; const ANIMATIONS = [ 'animate-fade', 'animate-slide-in', 'animate-spin', 'animate-pulse', 'animate-shake', ]; const THRESHOLD = 30; // minimum average FPS const browser = await chromium.launch(); const page = await browser.newPage(); let exitCode = 0; for (const name of ANIMATIONS) { const html = `<div class="${name}" style="width:100px;height:100px"></div> <script> let frames = 0; function count() { frames++; requestAnimationFrame(count); } requestAnimationFrame(count); setTimeout(() => window.__frames = frames, 3000); <\/script>`; await page.setContent(html); await page.waitForTimeout(3100); const fps = (await page.evaluate(() => window.__frames)) / 3; const status = fps >= THRESHOLD ? '✅' : '❌'; console.log(`${status} ${name}: ${fps.toFixed(1)} FPS`); if (fps < THRESHOLD) exitCode = 1; } await browser.close(); process.exit(exitCode);
