## Features

- Premium dark sci-fi telemetry dashboard styled like a fusion reactor control room
- Glassmorphism panels with cyan, blue, violet, and plasma orange accents
- Central reactor core visualization built with CSS and inline SVG only
- Live telemetry cards for temperature, pressure, plasma stability, magnetic field, coolant flow, and containment status
- Reactor stability gauges, energy output chart, cooling and pressure system cards, status timeline, and control panel actions
- Smooth CSS-only motion including pulse glow, floating telemetry cards, rotating reactor rings, scanline sweep, and hover lift
- Responsive layout for desktop, tablet, and mobile browsers
