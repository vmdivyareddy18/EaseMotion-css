Stripping Default Styles

Using appearance: none to remove the ugly default Operating System styles from form elements, allowing us to build beautiful, custom-branded controls.

1. The Native vs Custom Select

Select dropdowns look radically different on Mac, Windows, iOS, and Android. By stripping the appearance, we can build a unified, custom look.

2. The Fully Custom Checkbox

Standard checkboxes are impossible to style. By using appearance: none, we turn the checkbox into a blank canvas and style it completely from scratch.
