# Auto-resize Textarea – Grows as you type

> A production-ready auto-resizing textarea component that grows dynamically as you type, with manual resize support and real-time stats. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Auto-resize** – Grows as you type
- ✅ **Manual resize** – Drag handle to resize
- ✅ **Real-time stats** – Characters, words, lines
- ✅ **Min/Max height** – Configurable limits
- ✅ **Keyboard shortcuts** – Ctrl+Enter, Tab indent
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Highly customizable** – Colors, sizes, limits
- ✅ **Programmatic API** – Full control via JavaScript
