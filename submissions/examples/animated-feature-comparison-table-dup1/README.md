# Animated Feature Comparison Table

## Overview

An animated pricing comparison table that displays multiple plans with a highlighted Pro column. Rows animate into view with a smooth staggered effect, while check and cross icons use a pop animation to improve visual engagement.

## Features

- Smooth row entrance animation
- Highlighted Pro plan
- Animated check and cross icons
- Responsive layout
- Pure HTML, CSS and JavaScript
- No external libraries

## Files

- `demo.html` – Demo page
- `style.css` – Styling and animations

## How to Use

1. Open `demo.html` in any modern browser.
2. Scroll to see the row animations.
3. Customize the table content as needed.

## Browser Support

- Chrome
- Edge
- Firefox
- Safari
