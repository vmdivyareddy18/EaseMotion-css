# Ease Holographic Badge

A CSS-only holographic badge with a sweeping shine animation.

## Features

- Pure HTML & CSS
- Shine animation
- Glass-style design
- Responsive
- Lightweight

## Structure

```
ease-holographic-badge-n1/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Why EaseMotion CSS

A reusable animated badge component demonstrating subtle motion with minimal code.
