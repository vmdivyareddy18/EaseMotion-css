# Scale on Click – Button depresses and rebounds

> A production-ready scale-on-click component that creates a satisfying depress-and-rebound animation on buttons, with customizable intensity, variations, and keyboard shortcuts. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Depress and rebound** – Satisfying scale animation
- ✅ **3 intensity levels** – Low, Medium, High
- ✅ **Multiple variations** – Bounce, elastic, slow, fast
- ✅ **Color variants** – Primary, Success, Danger, Warning, Dark, Outline
- ✅ **Size variants** – Small, Medium, Large
- ✅ **Toggle scale** – Enable/disable effect
- ✅ **Reset all** – Reset all buttons
- ✅ **Keyboard shortcuts** – Ctrl+S (toggle), Escape (reset), Ctrl+1-3 (intensity)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
