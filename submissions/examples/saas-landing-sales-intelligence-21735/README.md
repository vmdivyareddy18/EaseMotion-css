# Business Landing — Sales Intelligence SaaS (`saas-landing-sales-intelligence-21735`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for a Sales Intelligence Platform (similar to ZoomInfo or Apollo) using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A B2B Sales tech landing page must look highly authoritative, data-driven, and corporate. This template achieves that aesthetic through a clean layout, solid colors (deep navy and corporate blue), sharp corners, and precise animations.

The layout includes:
- A hero section featuring a CSS-built prospect profile dashboard mockup
- A logo banner for customer trust
- A 3-column platform feature grid (Enrichment, Intent, Outreach)
- A split-layout data coverage section with count-up statistics styling
- An integration ecosystem grid
- A 2-tier per-seat pricing layout
- A high-impact free trial CTA

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth, authoritative text and card entrances)
- `.ease-slide-right` / `.ease-slide-left` / `.ease-slide-up` (For layout structural animations)
- `.ease-zoom-in` / `.ease-bounce-in` (For drawing attention to primary CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For buttons, feature cards, and mockups)
- `.ease-focus-ring` (For accessible form inputs)
- `.ease-hover-glow` (For subtle link interactions)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
