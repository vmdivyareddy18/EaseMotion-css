# Vertical Tabs / Sidebar Tabs

A vertical tab interface with four tabs on the left and corresponding content panels on the right. The active tab is highlighted in indigo with a light indigo background. Clicking a tab switches the visible panel.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click each tab to switch the content panel.

## Accessibility notes

Each tab is a `<button>` element. The active state uses both color and background changes for non-color-based identification.
