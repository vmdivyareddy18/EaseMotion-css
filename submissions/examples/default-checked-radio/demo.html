:default & :checked Radio

Default vs Checked

The first radio is the :default (pre-selected in HTML). The second has been marked :checked via user interaction. Notice the distinct visual states.

Default Button

The :default pseudo-class also applies to the default button in a form. The "Save" button is the default.

State Legend

  • :default — Set via HTML attribute
  • :checked — Currently selected by user
  • Changed — User picked a non-default option
