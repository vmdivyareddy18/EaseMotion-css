# Letter Spacing Expand on Hover

Letter spacing grows on hover — pure CSS.

## Features
- Smooth letter-spacing transition on hover
- Pure CSS, no JavaScript

## Usage
```css
.expand-text { letter-spacing: normal; transition: letter-spacing 0.3s; }
.expand-text:hover { letter-spacing: 0.3em; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
