Animated Border Beam

Glowing light beam that travels along the border of any element

Default Beam

A glowing beam travels continuously along the border

Reverse Direction

Beam travels counter-clockwise around the border

Fast Speed

Beam moves at 2x speed for more energetic feel

Slow Speed

Beam moves at 0.5x speed for elegant effect

Dual Beam

Two beams traveling in opposite directions

Custom Color

Beam color can be customized with CSS variables

💡 How to use: Add class="ease-border-beam" to any element with a border-radius. Works great on cards, buttons, modals, and images.
