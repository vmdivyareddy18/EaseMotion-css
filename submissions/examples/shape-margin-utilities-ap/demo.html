CSS Shape Wrap Layouts

Observe how text wraps organically along circular, elliptical, and polygonal boundaries with defined margins.

1. Circular Wrapping (Left Float) .shape-circle .shape-margin-4
CIRCLE

This article utilizes the .shape-circle utility, which sets the shape-outside property to a circular boundary. Together with the .float-left property, this circular shape pushes surrounding inline text elements along its curve. To ensure the text doesn't touch the edges of the circular element, we've added the .shape-margin-4 utility class. This adds a clean, 1rem (16px) margin around the circle.

Without the shape margin utility, inline text would press right against the outer edge of the circular graphic, making the paragraph hard to read. By using a standard spacing system, layout designers can maintain vertical rhythm while making typography wrap around organic paths.

2. Elliptical Wrapping (Right Float) .shape-ellipse .shape-margin-6
ELLIPSE

Here we have an elliptical float positioned on the right-hand side using the .shape-ellipse and .float-right utility classes. An elliptical path has different horizontal and vertical radius values, which creates a stretched wrap contour. We have applied the .shape-margin-6 utility class, giving it a 1.5rem (24px) spacing boundary.

The text flows smoothly down and around the curve of the ellipse, maintaining an even, legible distance of 24px at all points. This layout is extremely useful for magazine-style web editorials, landing page hero showcases, and featured headers where images have non-rectangular aspects.

3. Polygonal Triangle Wrapping (Left Float) .shape-triangle .shape-margin-8
TRIANGLE

Polygonal wrapping allows for complex geometric shapes. In this example, the element is clipped into a triangle using CSS clip-path, and the text wrapping is defined using a matching shape-outside: polygon(50% 0%, 0% 100%, 100% 100%) via the .shape-triangle class. We have also applied .shape-margin-8, creating a generous 2rem (32px) buffer zone.

Notice how the text forms a diagonal block, mirroring the slope of the triangle's right side. Staging spacing this way around a polygon creates a futuristic, editorial feeling in modern web design.

Technical Deep-Dive

Key requirements and details to keep in mind when implementing CSS shapes in production.

1. Float Requirement

The CSS shape-outside property (which forms the basis for shape margins) only applies to floated elements. If an element is not set to float: left or float: right, the shape-outside boundaries and their shape-margins will be ignored completely.

2. Sizing Constraint

Float shapes must have explicit width and height properties defined. The coordinates for polygons, circles, and ellipses are relative to the bounding box of the element. Without explicit height and width, the shape boundary cannot be calculated.

3. clip-path vs shape-outside

It is important to remember that shape-outside does not clip the background or visual content of the element itself; it only affects the flow of surrounding inline text. To visually clip the element to match the flow boundary, you must pair it with a matching clip-path (as seen in the triangle polygon class).
