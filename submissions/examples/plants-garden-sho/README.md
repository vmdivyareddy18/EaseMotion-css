# 🌿 Plants & Garden Shop Demo

A responsive horticulture e-commerce landing page built using **EaseMotion CSS** animation and hover utility classes.

## 📁 Folder Structure

```
submissions/
└── examples/
    └── plants-garden-shop/
        ├── demo.html
        ├── style.css
        └── README.md
```

## ✨ Features

- 🌱 Hero section with call-to-action
- 🪴 Plant categories
- 🌼 Featured product showcase
- 💧 Plant care guide section
- 📦 Monthly subscription CTA
- ⭐ Customer reviews
- 🌿 Plant quiz CTA
- 📱 Fully responsive layout
- 🎨 Natural green theme

## 🎬 EaseMotion Classes Used

- ease-fade-up
- ease-fade-in
- ease-slide-up
- ease-zoom-in
- ease-hover-grow
- ease-hover-glow
- ease-hover-lift
- ease-hover-shadow
- ease-hover-border

## 🚀 How to Run

1. Open the project folder.
2. Open `demo.html` in any modern browser.
3. No build tools or dependencies are required.
