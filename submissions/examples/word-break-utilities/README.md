# Word Break Utilities

Control how long words and text wrap within containers using simple utility classes.

## Included Utilities

- `ease-break-word`
- `ease-break-all`
- `ease-break-keep`

## Usage

```html
<div class="ease-break-word">LongText...</div>
```
