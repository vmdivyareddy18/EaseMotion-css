Content Visibility Utilities

CSS content-visibility utilities to control element rendering lifecycle. Skip rendering of off-screen items to boost initial page speed and loading performance.

What is content-visibility?
It is a powerful performance feature that allows the browser to skip rendering an element's contents (including layout and painting) until it is needed.
  • cv-visible: Renders normally (default browser behavior).
  • cv-auto: Skips rendering of elements that are off-screen, boosting performance.
  • cv-hidden: Skips rendering completely. Good for hiding views while keeping their cached layout state.

Visibility States Comparison

Visible Content .cv-visible

This content is rendered normally on page load. Its layout, style, and painting are calculated immediately by the browser.

Standard behavior. No optimization applied.

Hidden Content .cv-hidden

You will not see this text. The browser completely skips layout and painting calculations for this container's children.

Similar to display: none, but keeps rendering state cached for faster toggling.
