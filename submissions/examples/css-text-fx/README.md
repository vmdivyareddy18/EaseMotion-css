# ✍️ CSS Text Effects

A minimal collection of beautiful, pure CSS text animations for modern web design.

## 🚀 Features
- 🌈 **Gradient Flow:** Smooth moving gradient text.
- ⌨️ **Typing Effect:** Classic typewriter animation with blinking cursor.
- 🌊 **Wave Text:** Bouncing letters with staggered delays.

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All CSS animations
- `readme.md` - Documentation

## 🛠️ How to Use
1. Open `demo.html` in your browser.
2. Copy the CSS classes to use in your own projects!

## 📱 Browser Support
Works perfectly on all modern browsers (Chrome, Firefox, Safari, Edge).
