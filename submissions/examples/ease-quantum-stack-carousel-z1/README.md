# Ease Quantum Stack Carousel

## Overview

A futuristic stacked carousel concept featuring layered glass cards with floating motion and depth effects using only HTML and CSS.

## Features

- Layered carousel layout
- Floating card animation
- Glassmorphism styling
- Responsive design
- Lightweight CSS
- Pure HTML & CSS
