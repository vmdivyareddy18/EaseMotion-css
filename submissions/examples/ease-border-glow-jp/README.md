# Ease Border Glow

A premium animated glowing border effect built using pure HTML and CSS.

## Features

- Pulsing Border Glow
- Animated Box Shadow
- CSS Keyframes
- Glassmorphism Cards
- Responsive Design
- Pure HTML & CSS
- Multiple Showcase Examples

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="ease-border-glow">
    Content
</div>
```
