# Animated Ocean Research Dashboard

A modern dashboard for monitoring ocean conditions, marine ecosystems, research missions, and conservation projects.

## Features

- Ocean condition metric cards
- Marine monitoring panel
- Research progress trackers
- Activity feed
- Responsive layout
- Smooth animations

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Oceanographic research
- Marine conservation portals
- Environmental monitoring systems
- Research institutions
- Educational science dashboards
