# Dot Progress Steps

A horizontal step progress indicator with animated dot transitions.

## Features

- Five-step horizontal progress indicator
- Active dot scales and pulses
- Completed dots fill with a distinct color
- Connecting line animates as steps progress
- Previous / Next navigation buttons

## CSS Custom Properties

| Property                   | Default   | Description             |
| -------------------------- | --------- | ----------------------- |
| `--dot-duration`           | `0.35s`   | Transition duration     |
| `--dot-inactive-color`     | `#d1d5db` | Inactive dot color      |
| `--dot-active-color`       | `#6366f1` | Active dot color        |
| `--dot-completed-color`    | `#22c55e` | Completed dot color     |
| `--dot-size`               | `40px`    | Dot diameter            |
| `--dot-line-color`         | `#e5e7eb` | Connecting line color   |
| `--font-family`            | `Inter`   | Font family             |

## Usage

Open `demo.html` in a browser and click **Next** / **Previous** to navigate steps.
