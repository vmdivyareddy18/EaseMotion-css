# Skeleton Loading Rectangle (Issue #18251)

A skeleton loading placeholder with a shimmer animation. A dark gray rectangle with a moving gradient highlight that creates a scanning shimmer effect.

## Usage

Open `demo.html` in a browser. The skeleton rectangle will animate automatically.

## Files

- `demo.html` — HTML5 page with the skeleton element
- `style.css` — Dark theme styles with `@keyframes shimmer`
- `README.md` — This file
