# CSS Tada-Click Modal for Dashboard Analytics Layouts

## Overview

This project demonstrates a responsive dashboard analytics modal featuring a smooth **Tada-Click** entrance animation built entirely with HTML and CSS. The component presents key business metrics in a clean, modern interface while maintaining responsiveness, accessibility, and lightweight performance without JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- Smooth Tada-Click entrance animation
- Responsive dashboard analytics modal
- Modern analytics card layout
- Statistics summary section
- CSS custom properties for easy customization
- Hover and active button effects
- Keyboard-accessible focus styles
- No JavaScript or external libraries required

## Customization

The component uses CSS custom properties for easy customization.

```css
:root{
    --primary:#2563eb;
    --primary-dark:#1d4ed8;
    --accent:#10b981;
    --bg:#eef3fb;
    --radius:22px;
    --duration:.8s;
}
```

You can customize:

- Primary and accent colors
- Background color
- Animation duration
- Border radius
- Typography
- Shadow intensity
- Analytics card styling
- Button appearance
- Modal width
