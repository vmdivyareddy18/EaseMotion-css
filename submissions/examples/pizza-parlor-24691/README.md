# Pizza Parlor Template
A rustic, high-appetite template designed for pizzerias, pizza parlors, and casual Italian eateries.

- **Design**: Uses a "Wood-Fired" palette with deep sauce-reds, warm dough-beiges, and charcoal-black accents to evoke the traditional pizza-making process.
- **Use Case**: Neighborhood pizzerias, gourmet pizza shops, and casual Italian restaurants.
