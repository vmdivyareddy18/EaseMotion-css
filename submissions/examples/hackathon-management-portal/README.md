# Hackathon Management Portal

## Overview
Build a hackathon organizer dashboard interface.

This project implements the Phase 97 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Team management cards**: Built with glassmorphism and CSS grid/flexbox.
- **Submission tracking**: Built with glassmorphism and CSS grid/flexbox.
- **Leaderboard section**: Built with glassmorphism and CSS grid/flexbox.
- **Event timeline**: Built with glassmorphism and CSS grid/flexbox.
- **Organizer Console**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
