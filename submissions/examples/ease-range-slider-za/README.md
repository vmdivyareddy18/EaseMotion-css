# Custom Range Slider

## What does this do?
CSS-styled range input slider with color themes, sizes, thumb shapes, and filled track using -webkit-appearance override.

## How is it used?
```html
<input type="range" min="0" max="100" value="50" class="rsl-slider rsl-primary">
```

## Why is it useful?
Native range input with custom styling. Five color themes, three sizes, three thumb shapes. Cross-browser compatible. Disabled state included.
