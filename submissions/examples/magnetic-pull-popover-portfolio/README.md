# CSS Magnetic Pull Popover for Creative Portfolio Layouts

A pure CSS animated popover with a smooth magnetic pull interaction designed for creative portfolio interfaces.

## Features

- CSS-only animation
- Magnetic pull movement effect
- Responsive design
- Keyboard accessible
- Custom CSS variables
- No JavaScript required

## Usage

```html
<div class="magnetic-popover">

<button class="magnetic-trigger">
View Portfolio
</button>

<div class="magnetic-content">
Content here
</div>

</div>
