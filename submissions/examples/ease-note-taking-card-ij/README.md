# Note Taking Card

Sticky-note style cards with staggered appear animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--card-bg` | `#2d2d44` | Note card background |
| `--card-color` | `#ccc` | Note text color |
| `--pin-color` | `#f72585` | Pin dot color |
| `--appear-duration` | `0.35s` | Appear animation speed |
