# Ease Polaris Beacon

## Overview

A futuristic beacon animation featuring expanding signal rings and a glowing center built entirely with semantic HTML and pure CSS.

## Features

- Expanding beacon waves
- Glowing center core
- Glassmorphism card
- Responsive layout
- Lightweight CSS animations
- No JavaScript

## Folder Structure

```text
ease-polaris-beacon-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Provides a reusable beacon animation that demonstrates clean CSS motion, modern gradients, and lightweight effects for dashboards, maps, IoT interfaces, and monitoring systems.
