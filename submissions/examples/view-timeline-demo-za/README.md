# CSS View Timeline Scroll Reveal

A CSS demo of view-timeline creating scroll-driven animations tied to element visibility within the scrollport, enabling pure CSS scroll reveals without JavaScript.

## Files

- `demo.html` - Cards that reveal on scroll via view-timeline
- `style.css` - view-timeline-name, view-timeline-axis, animation-timeline

## Usage

Open `demo.html` and scroll down to see each card animate as it enters view.
