Resize Utilities

Control element resizability natively with simple CSS classes.

Resize Both

Allows both horizontal and vertical resizing.

Drag the handle in the bottom-right corner to resize this box in both directions.

.resize

Resize Horizontal

Restricts resizing to the horizontal axis.

You can drag the handle to resize this box only horizontally.

.resize-x

Resize Vertical

Restricts resizing to the vertical axis.

You can drag the handle to resize this box only vertically.

.resize-y

Resize None

Disables resizability (often used to lock textareas).

Why use these utilities?

In CSS, the resize property only works if the element's overflow is set to something other than visible (like auto or scroll). EaseMotion's utilities handle this requirement automatically, making native resizing seamless.
