# Search Field

## What does this do?
Search Field component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"sf-primary\">Search Field Primary</div>
```

## Why is it useful?
Provides a reusable search field UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
