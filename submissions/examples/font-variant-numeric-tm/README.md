# Font Variant Numeric — EaseMotion CSS Utilities

CSS `font-variant-numeric` utility classes for the EaseMotion CSS framework.

## Quick Start

```html
<link rel="stylesheet" href="style.css">
```

## Utility Classes

| Class | Declaration |
|-------|-------------|
| `.numeric-normal` | `font-variant-numeric: normal;` |
| `.numeric-ordinal` | `font-variant-numeric: ordinal;` |
| `.numeric-slashed-zero` | `font-variant-numeric: slashed-zero;` |
| `.numeric-lining` | `font-variant-numeric: lining-nums;` |
| `.numeric-oldstyle` | `font-variant-numeric: oldstyle-nums;` |
| `.numeric-proportional` | `font-variant-numeric: proportional-nums;` |
| `.numeric-tabular` | `font-variant-numeric: tabular-nums;` |
| `.numeric-diagonal` | `font-variant-numeric: diagonal-fractions;` |
| `.numeric-stacked` | `font-variant-numeric: stacked-fractions;` |

## Responsive Variants

Prefix with `sm-`, `md-`, `lg-` for responsive behavior:

```html
<div class="util-class sm-util-class md-util-class lg-util-class">
  Responsive Font Variant Numeric
</div>
```

## Dark Mode

Use the `-dark` variant:

```html
<div class="util-class-dark">
  Dark mode Font Variant Numeric
</div>
```

## Reduced Motion

Use the `-nomotion` variant:

```html
<div class="util-class-nomotion">
  No motion Font Variant Numeric
</div>
```

## Framework Integration

All utilities use EasMotion design tokens from `core/variables.css`: `--ease-primary`, `--ease-secondary`, `--ease-accent`, `--ease-success`, `--ease-danger`, `--ease-warning`, `--ease-info`.
