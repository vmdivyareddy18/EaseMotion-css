# Fade In Cookie Notification

A responsive **Fade In Cookie Notification** component built with pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Corporate-inspired design
- Fade-in animation
- Responsive layout
- Accessible structure
- Uses EaseMotion variables and `ease-*` animation classes

## Folder Structure

```text
fade-in-cookie-notification-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Technologies

- HTML5
- CSS3

## Animation

- `ease-fade-in`
- CSS keyframe fade animation
- Responsive and lightweight
