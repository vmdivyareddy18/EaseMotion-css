# Morph Timeline

A reusable timeline component with a soft neumorphic design and subtle morph-style hover interactions.It is built using HTML and CSS and is designed to be lightweight,responsive, and easy to reuse in different web projects.

## Features

- Neumorphic card design
- Smooth hover morph effect
- Vertical timeline layout
- Responsive across different screen sizes
- Pure HTML and CSS
- Easy to customize

## Folder Structure

```
morph-timeline-S-BOT-DESIGN/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern web browser.
2. Ensure `style.css` is in the same directory.
3. Hover over the timeline cards to view the morph interaction.

## Customization

You can easily modify:

- Timeline colors
- Card spacing
- Shadow intensity
- Border radius
- Timeline labels and descriptions

by editing the CSS variables and styles inside `style.css`.
