# Morphing Hamburger Menu Icon

## Description

An animated hamburger menu icon that smoothly transforms into a close icon.

## Features

- Smooth icon morphing
- Responsive design
- Lightweight implementation
- Common navigation pattern

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and click the icon.
