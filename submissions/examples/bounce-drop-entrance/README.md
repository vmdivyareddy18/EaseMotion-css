 # Bounce Drop Entrance Animation

A lightweight entrance animation utility that drops elements from above with a springy bounce settle effect.

## Features

- Pure CSS animation
- Spring-like cubic-bezier easing
- Smooth overshoot and settle effect
- Lightweight and reusable
- Cross-browser compatible

## Usage

```html
<div class="ease-bounce-drop">
  Content
</div>
```

## Animation Behavior

- Starts from `translateY(-120%)`
- Overshoots slightly downward
- Settles smoothly at original position

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- No JavaScript required
