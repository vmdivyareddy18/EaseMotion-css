# Neon Glow Card Component

## What does this do?
Animated neon glow cards with rotating conic gradient borders, pulsing glow effects, 3D flip animations, and six color themes.

## How is it used?
```html
<div class="gl-card gl-cyan">
  <div class="gl-border"></div>
  <div class="gl-content"><h3>Title</h3><p>Description</p></div>
</div>
```

## Why is it useful?
Creates visually striking cards for dark-themed modern UIs. Six color themes, 3D flip variants, corner accent styles. Responsive with reduced-motion support.
