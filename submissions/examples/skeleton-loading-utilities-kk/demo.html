Skeleton Loading Utilities

Five composable CSS utility classes for skeleton loading states: a base shimmer, text lines, avatar circles, card placeholders, and a lightweight pulse animation modifier.

Individual Utility Classes
.ease-skeleton

Base skeleton placeholder. Displays a sweeping shimmer gradient. Rounds corners via --skeleton-radius. Can be customised with --skeleton-bg, --skeleton-highlight, and --skeleton-shine. 

<div class="ease-skeleton"
     style="height: 80px;"></div>
.ease-skeleton-text

Mimics a line of body text — 0.85em tall, full-width. Stack multiple lines to simulate a paragraph. The last sibling narrows to 65% automatically for a natural ragged edge. 

<div class="ease-skeleton ease-skeleton-text"></div>
<div class="ease-skeleton ease-skeleton-text"></div>
<div class="ease-skeleton ease-skeleton-text"></div>
<!-- last-child auto-narrows to 65% -->
<div class="ease-skeleton ease-skeleton-text"></div>
.ease-skeleton-avatar

Circular placeholder for profile photos and user avatars. Defaults to 48×48 px. Override with --skeleton-avatar-size or use the -sm / -lg / -xl variants. 

<!-- sm · default · lg · xl -->
<div class="ease-skeleton ease-skeleton-avatar
             ease-skeleton-avatar-sm"></div>
<div class="ease-skeleton ease-skeleton-avatar"></div>
<div class="ease-skeleton ease-skeleton-avatar
             ease-skeleton-avatar-lg"></div>
<div class="ease-skeleton ease-skeleton-avatar
             ease-skeleton-avatar-xl"></div>
.ease-skeleton-card

Card-shaped placeholder for dashboard widgets, product cards, and content previews. Defaults to 200px tall. Override with --skeleton-card-height or use the -sm / -lg variants. 

<div class="ease-skeleton ease-skeleton-card
             ease-skeleton-card-sm"></div>
<div class="ease-skeleton ease-skeleton-card"></div>

<!-- custom height -->
<div class="ease-skeleton ease-skeleton-card"
     style="--skeleton-card-height: 260px;"></div>
.ease-skeleton-pulse

Animation modifier — replaces the shimmer sweep with a simpler, lower-overhead opacity pulse. Combine with any shape class. Ideal for reduced-complexity contexts or lists with many placeholders. 

<!-- Replace shimmer with pulse -->
<div class="ease-skeleton ease-skeleton-card
             ease-skeleton-pulse"></div>
<div class="ease-skeleton ease-skeleton-text
             ease-skeleton-pulse"></div>
<div class="ease-skeleton ease-skeleton-avatar
             ease-skeleton-pulse"></div>

CSS Variable API — Theming
Custom Colours

Override the three colour tokens to match any brand palette. All skeleton elements inside the container inherit the custom tokens. 

<div style="
  --skeleton-bg:        #1a1a3e;
  --skeleton-highlight: #2d2d6e;
  --skeleton-shine:     #4040a0;
">
  <div class="ease-skeleton ease-skeleton-card"></div>
  <div class="ease-skeleton ease-skeleton-text"></div>
</div>
Duration Tuning

Control animation speed with --skeleton-shimmer-duration for the sweep and --skeleton-pulse-duration for the pulse modifier. 

<!-- Faster shimmer -->
<div class="ease-skeleton ease-skeleton-card"
     style="--skeleton-shimmer-duration: 0.9s;"></div>

<!-- Faster pulse -->
<div class="ease-skeleton ease-skeleton-text
             ease-skeleton-pulse"
     style="--skeleton-pulse-duration: 0.8s;"></div>
