# Horizontal Auto-Scroll Marquee

A CSS-only marquee that seamlessly loops content using duplicate tracks and translateX animation. Perfect for news tickers, announcements, and brand carousels.

## Files

- `demo.html` - Marquee with duplicate tracks
- `style.css` - Infinite scroll animation with mask edge fade

## Usage

Open `demo.html` to see the content auto-scroll horizontally.
