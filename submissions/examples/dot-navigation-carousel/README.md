# Dot Navigation Indicator for Carousels

A CSS dot navigation indicator for carousels and slideshows. Active dots scale up with a smooth transition and use an accent color. Includes left, center, and right alignment variants. A small script handles active state switching on click.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click any dot to set it as active. Three alignment examples are shown.

## Accessibility notes

Each dot is a `<button>` with an `aria-label` describing which slide it represents. Focus-visible outlines indicate keyboard focus. Reduced motion disables the scale transition.
