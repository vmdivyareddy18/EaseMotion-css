# Animated Empty State Block — EaseMotion CSS

An elegant, lightweight feedback component tailored for blank-canvas scenarios (empty search results, unpopulated tables, empty shopping carts, or fresh account profiles).

## 🛠 Features & Class Syntax

| Selector Flag | target Level | Action / Micro-Interaction Animation |
|---|---|---|
| `.ease-empty-state` | Main Wrapper | Controls alignment properties, inner padding scales, and basic formatting bounds. |
| `.ease-empty-icon-wrap` | Graphic Container | Hosts an infinite, gentle `translateY` floating animation to bring dead space to life. |
| `.ease-empty-title` | Header Text | Slides upwards and fades in instantly upon component mount sequence (`100ms` delay). |
| `.ease-empty-desc` | Paragraph Block | Secondary staggered typography entrance block (`250ms` delay). |
| `.ease-empty-cta` | Interactive Link | Final staggered component focus target revealing a prompt button (`400ms` delay). |
| `.ease-empty-state-compact` | Container Modifier | Scales layout boundaries and font sizes down to fit perfectly within dashboards or cards. |
