Appearance Textfield

CSS appearance control proposed for core/utilities.css.

The .appearance-textfield utility applies appearance: textfield; to custom elements or inputs. It is most commonly used to remove the default increment/decrement spinners from type="number" inputs, making them behave and look like standard text inputs.

Default Browser Behavior

Standard number input with default browser spinners (arrows on hover/focus).

.appearance-textfield Applied

The utility is applied, removing the spinners and making it render like a standard text field while retaining number-type validation.
