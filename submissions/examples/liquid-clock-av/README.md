# Liquid Clock

## What does this do?

Creates a futuristic liquid-inspired digital clock interface featuring animated waves, floating gradients, and a glassmorphism container.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="clock">
    <div class="time">
        <span>09</span>
        <span>:</span>
        <span>41</span>
        <span>:</span>
        <span>26</span>
    </div>
</div>
```

---

## Why is it useful?

Digital clock interfaces are commonly used in dashboards, landing pages, productivity apps, smart home interfaces, and futuristic UI concepts. This standalone example demonstrates how layered gradients, liquid wave animations, and glassmorphism can create a premium visual experience using only HTML and CSS.
