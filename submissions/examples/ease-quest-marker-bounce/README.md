# ease-quest-marker-bounce – Quest Marker Bounce

A continuous gentle bounce animation for a location pin or objective marker. Ideal for map markers, tutorial hints, or quest indicators.

## EaseMotion classes used
- **Layout:** ease-container, ease-flex, ease-items-center, ease-justify-center, ease-min-h-screen, ease-mx-auto
- **Background:** ease-bg-gray-100
- **Typography:** ease-text-3xl, ease-font-bold, ease-text-gray-500, ease-text-sm, ease-text-gray-400
- **Spacing:** ease-mb-2, ease-mb-8, ease-mt-8
- **Animation:** ease-fade-in, ease-delay-200, ease-delay-500

## How it works
- The pin emoji is animated with 	ranslateY moving up and down in a continuous loop.
- A shadow element underneath scales down and fades as the pin rises, adding a sense of depth.
- The animation respects prefers-reduced-motion by disabling the movement.

## How to use
1. Copy the HTML, CSS, and JS from demo.html into your project.
2. Ensure the path to easemotion.css is correct.
3. Open demo.html in any modern browser.
