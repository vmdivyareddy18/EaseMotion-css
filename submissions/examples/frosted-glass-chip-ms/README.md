# Frosted Glass Chip

Resolves #42400.

### What does this do?
Adds a Frosted Glass Chip component to EaseMotion CSS tailored for organic, nature-inspired interfaces or photography portfolios. It uses CSS `backdrop-filter: blur()` combined with semi-transparent rgba backgrounds to create a glassmorphism effect.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. This chip is actually a hidden checkbox, so it maintains a toggled state perfectly without JavaScript. 

### Why is it useful?
Glassmorphism is a popular design trend that allows background imagery to subtly shine through UI elements, maintaining immersion. It's particularly effective for filter chips overlaid on top of rich hero images or maps.
