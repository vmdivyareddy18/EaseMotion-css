📏 Flexbox-Driven Base

By controlling a single panel's flex-basis and allowing the other to flex-grow, layouts stay incredibly stable and avoid subpixel rounding errors common in percentage grid systems.

🚀 Nested Architecture

Primitives are fully recursive. You can easily nest horizontal split containers inside vertical ones and vice-versa, allowing for unlimited dashboard grids and modular side panels.

✨ Clean Drag Performance

Dragging triggers on top of transparent pointer masks, preventing pointer loss over iframes or code blocks, and ensuring smooth 60fps tracking without lagging or stuttering.
