# Premium Fitness Gym Landing Page

A complete, high-energy, and fully responsive landing page for a premium fitness club ("IronForge Gym") built under issue #14806 using the **EaseMotion CSS** design system.

## EaseMotion Features Showcased

- **Entrance Animations:** Leverages `.ease-reveal` and direction helpers (`.ease-reveal-up`, `.ease-reveal-right`) with scroll-reveal triggers.
- **Hover Micro-interactions:** Showcases `.ease-card-lift` and `.ease-hover-shimmer` on active buttons, classes, and pricing cards.
- **Component Systems:** Utilizes standardized `.ease-btn`, `.ease-btn-primary`, `.ease-btn-outline`, and layout classes (`.ease-flex`, `.ease-grid`, `.ease-card`).

## Required Sections Included

1. **High-Energy Hero:** Bold dark-orange theme with strong contrast and a free trial booking CTA.
2. **Animated Class Schedule Table:** An interactive schedule table showcasing daily class offerings with hover animation highlights.
3. **Trainer Profiles Grid:** Interactive cards showcasing certified personal trainers and head coaches.
4. **Before/After Transformation Gallery:** Interactive slider showing client fitness achievements.
5. **Membership Pricing Tiers:** Tiered pricing cards (Base, Elite, VIP) to drive memberships.
6. **Member Testimonials:** Motivating reviews from real gym members.
7. **Free Trial CTA Banner:** High-energy conversion banner at the bottom.
