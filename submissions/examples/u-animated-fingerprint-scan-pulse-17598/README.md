# Animated Fingerprint Scan Pulse

## Overview

Animated Fingerprint Scan Pulse is a lightweight CSS animation that simulates biometric authentication using a pulsing fingerprint and scanning line.

## Features

- CSS-only animation
- Fingerprint pulse effect
- Moving scan line
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="fingerprint-scan">
  <span class="fingerprint">👆</span>
  <span class="scan-line"></span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
