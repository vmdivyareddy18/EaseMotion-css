# Ease Log Stream Viewer

## Overview

Ease Log Stream Viewer is a reusable monitoring component inspired by developer consoles and DevOps dashboards. It displays color-coded log entries with smooth animations using pure HTML and CSS.

---

## Features

- 📜 Live log viewer layout
- 🟢 Streaming status indicator
- 🎨 Color-coded log levels
- ✨ Animated log entries
- 📱 Responsive design
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- INFO logs
- SUCCESS logs
- WARNING logs
- ERROR logs
- Live streaming indicator

---

## Usage

```html
<div class="viewer">
...
</div>
```

## Customization

- Log colors
- Typography
- Animation speed

## Browser Support

Compatible with all modern browsers supporting CSS animations and Flexbox.

Suitable for monitoring dashboards, admin panels, developer tools, DevOps interfaces, and educational demos. It provides a reusable log viewer component with clean animations and no JavaScript.
