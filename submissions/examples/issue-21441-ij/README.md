## Sunset Animation

A pure CSS sunset scene. The sun descents through a sky that shifts from bright warm oranges to deep purples. Glitter particles shimmer across the water as daylight fades.

**Features:**
- Rich sunset gradient with multiple color stops
- Descending sun with shrinking glow
- Darkening sky filter over time
- Respects `prefers-reduced-motion`
