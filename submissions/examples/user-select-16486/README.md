# User-Select Utilities
This submission introduces utilities to control whether text content can be highlighted and selected by the user.

- `.select-none`: Disables text selection.
- `.select-all`: Selects the entire content on a single click.
- `.select-text`: Allows standard text selection.
- `.select-auto`: Resets selection to browser default.

These utilities are useful for improving UI behavior in buttons, labels, or interactive components.
