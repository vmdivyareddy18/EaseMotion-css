# Timeline Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive vertical timeline.

---

## Included Components

- Timeline heading
- Four timeline events
- Dates
- Titles
- Descriptions
- Responsive layout

---

## Folder Structure

```
submissions/examples/timeline-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to verify responsiveness.
3. Observe the animation effects applied to each timeline event.
