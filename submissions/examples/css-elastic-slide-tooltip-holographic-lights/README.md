# CSS Elastic Slide Tooltip - Holographic Light Layout

A pure CSS animated tooltip featuring a smooth elastic slide transition with a futuristic holographic light appearance.

## Features

- Pure CSS animation (No JavaScript)
- Elastic slide entrance effect
- Hover and keyboard focus support
- Fully responsive
- Custom CSS animation variables
- Modern glassmorphism holographic style

## Usage

Add the tooltip structure:

```html
<div class="tooltip-wrapper">

  <button class="tooltip-trigger">
    Hover Me
  </button>

  <div class="tooltip">
    Tooltip content
  </div>

</div>
