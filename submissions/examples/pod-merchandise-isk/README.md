# E-commerce — Print on Demand Merchandise

## What does this do?
A complete, copy-paste ready full-page website for a Creator Print-on-Demand merchandise storefront styled with a creative and vibrant visual aesthetic, featuring product mockup layouts, operational flow steps, creator stories, and interactive pricing buttons.

## How is it used?
Open `demo.html` directly in any web browser to view the page. No build steps or server configurations are required. The layout animations utilize EaseMotion CSS's built-in scroll and hover classes.

## Why is it useful?
This showcases how EaseMotion CSS variables, animations, and transitions can be combined to build a modern, high-converting creator economy product page with fluid and playful interactions that elevate user engagement.

## EaseMotion CSS Classes Showcased
- **Entrance & Hover Animations**:
  - `.ease-slide-in-left` / `.ease-slide-in-right`: Used for slide-in entrance animations on load.
  - `.ease-hover-grow`: Used for navigational branding logos and button states.
  - `.ease-hover-lift-shadow`: Used for creator review panels and primary link actions.
  - `.ease-hover-underline`: Used for header menu links.
  - `.ease-card-lift`: Used for interactive feature list containers.
  - `.ease-hover-glow`: Used for call-to-action designer registration button highlights.
- **Scroll Reveal Animations**:
  - `.ease-reveal`: Applied to sections to hide elements offscreen until visible.
  - `.ease-reveal-up` / `.ease-reveal-left` / `.ease-reveal-right` / `.ease-reveal-scale`: Sliding directional reveals.
  - `.ease-reveal-delay-1` / `.ease-reveal-delay-2` / `.ease-reveal-delay-3`: Staggered timing delays for grid columns.
- **Looping & Floating Animations**:
  - `.ease-bounce` / `.ease-float`: Applied to product info badges to create dynamic vertical floating motions.
