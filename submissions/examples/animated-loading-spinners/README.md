# Animated Loading Spinners Collection

A diverse collection of **Animated Loading Spinners** perfect for indicating progress and keeping users engaged during wait times. This example demonstrates how to combine **EaseMotion CSS** for continuous animations, staggered dot movements, and smooth scaling effects to create modern, accessible loading indicators.

## 🚀 Features

- **9 Different Spinner Styles**: Classic circle, bouncing dots, dual ring, pulsing square, ripple effect, animated bars, gradient spinner, dots grid, and rotating squares.
- **Continuous Animations**: Uses `ease-spin-slow` for rotating elements and `ease-pulse` for scaling effects.
- **Staggered Delays**: Multiple elements use `ease-delay-*` classes to create wave-like, sequential animations.
- **Reverse Spin Support**: Some spinners rotate in opposite directions for visual interest.
- **Pure CSS Implementation**: No JavaScript required for the animations.
- **Responsive Grid Layout**: Adapts gracefully to different screen sizes.
- **Clean Card Design**: Each spinner is displayed in a card with hover effects and code snippets showing which EaseMotion classes are used.
