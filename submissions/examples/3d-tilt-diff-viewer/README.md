# 3D Tilt Diff Viewer

A responsive **3D Tilt Diff Viewer** component built using pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Finance dashboard inspired UI
- 3D tilt hover animation
- Responsive layout
- Accessible structure
- Uses EaseMotion CSS variables and `ease-*` animation classes

## Folder Structure

```text
3d-tilt-diff-viewer-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Technologies

- HTML5
- CSS3

## Animation

- `ease-fade-in`
- 3D tilt hover effect
- Responsive and lightweight
