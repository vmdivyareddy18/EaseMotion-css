# Alarm Clock

Digital alarm clock with set time, ring animation, and snooze functionality. The clock face glows when the alarm is enabled and pulses when ringing.

## Features

- Live time display updating every second
- Set alarm via time input with AM/PM indicator
- Toggle alarm on/off with glow effect
- Pulse animation on alarm trigger
- Snooze button adds 5 minutes
- Bell/shake visual feedback during alarm

## Usage

Enable the alarm and set a time. When the clock matches the alarm time, the face pulses and rings. Press Snooze to delay 5 minutes or Disable to stop.
