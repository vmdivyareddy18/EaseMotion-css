Floating Animation Utilities

Five reusable CSS utility classes that add smooth, performant floating animations to any element — no JavaScript required.

Floating Card .ease-float

A smooth, infinite vertical float at medium speed (3 s). Ideal for feature cards, product images, and dashboard widgets.

🚀

Feature Card

Floating gently, endlessly.

Floating Image .ease-float-slow

Slower, more serene floating (5 s). Perfect for hero illustrations, background decorations, and ambient UI accents.

🌐
Floating Icon .ease-float-fast

A quicker cadence (1.5 s) to draw attention. Best for notification badges, action icons, or small UI accents that need urgency.

🔔
Bounce Float .ease-float-bounce

Combines floating with a gentle spring bounce for a playful, energetic feel. Great for mascots, emojis, and fun CTAs.

✨
Hover Float Button .ease-float-hover

Float activates only on hover with a smooth transition — no layout shift. Hover the button below to see it lift and float.
