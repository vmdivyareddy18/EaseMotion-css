# Bounce Icon Animation

A lightweight CSS utility that adds a bounce effect to icons and small UI elements.

## Usage

```html
<div class="ease-bounce-icon">⭐</div>
```

## Features

 Pure CSS
 Lightweight and reusable
 Suitable for icons, badges, and notifications
 Easy to integrate into existing projects

## Files

 `style.css` – animation implementation
 `demo.html` – working example
