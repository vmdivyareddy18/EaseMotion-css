# Modern Button Collection

## Description
A collection of modern, interactive button styles designed to enhance user interfaces while maintaining EaseMotion CSS's composable and animation-first philosophy.

## Features
- Neon Glow Button
- Glassmorphism Button
- Gradient Button
- 3D Press Button
- Slide Fill Button
- Shine Hover Effect
- Smooth hover animations
- Responsive and lightweight
- Easy to customize

## Usage
Include the HTML structure and CSS styles, then apply the desired button class to any `<button>` element.

Examples:

```html
<button class="btn btn-neon">Neon</button>
<button class="btn btn-glass">Glass</button>
<button class="btn btn-gradient">Gradient</button>
<button class="btn btn-press">3D Press</button>
<button class="btn btn-slide">Slide Fill</button>
<button class="btn btn-shine">Shine</button>
```
