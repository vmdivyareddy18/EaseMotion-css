# Premium Pricing Card (pricing-card-ag)

### What does this do?
This is a modern, premium, glassmorphic pricing card with a popular badge, dark theme styling, and an engaging hover-lift animation with a glowing shadow effect.

### How is it used?
Include the provided `style.css` and use the following HTML structure:

```html
<div class="pricing-card pricing-card-pro">
  <div class="pricing-badge">Most Popular</div>
  
  <div class="pricing-header">
    <h3 class="pricing-title">Pro Plan</h3>
    <p class="pricing-description">For professionals and growing teams</p>
  </div>
  
  <div class="pricing-price-container">
    <span class="pricing-currency">$</span>
    <span class="pricing-amount">29</span>
    <span class="pricing-period">/mo</span>
  </div>
  
  <ul class="pricing-features">
    <li class="pricing-feature">...</li>
  </ul>
  
  <button class="pricing-btn">Get Started Now</button>
</div>
```

### Why is it useful?
Pricing pages are critical for SaaS applications. This component provides a highly professional and conversion-optimized design that fits perfectly within EaseMotion's animation-first philosophy, utilizing elegant scaling and shadow transitions without needing JavaScript.
