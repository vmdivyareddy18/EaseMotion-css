# Catapult Stopwatch

A responsive **Catapult Stopwatch** component built using pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Responsive layout
- Accessible structure
- Catapult launch animation
- Uses EaseMotion-style variables and animation class (`ease-fade-in`)
