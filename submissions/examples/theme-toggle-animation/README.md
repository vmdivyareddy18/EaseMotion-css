# Cosmic Theme Toggle

## What does this do?

Creates a premium animated theme toggle switch inspired by day and night transitions. The component smoothly transforms a glowing sun into a crescent moon while animating stars, background gradients, and a floating glassmorphism-style toggle knob.

## How is it used?

```html
<input type="checkbox" id="theme-toggle">

<label for="theme-toggle" class="toggle">

    <div class="stars">
        <span></span>
        <span></span>
        <span></span>
        <span></span>
        <span></span>
    </div>

    <div class="knob">
        <div class="sun-moon"></div>
    </div>

</label>
```

Include the accompanying `style.css` file and customize colors, animation timings, and glow effects as needed.

## Why is it useful?

This component demonstrates multiple CSS animation techniques in a single, reusable example:

* Smooth state transitions
* Morphing visual effects
* Glassmorphism styling
* Gradient background animation
* Glow and lighting effects
* Star particle animations

It aligns with the animation-first philosophy of EaseMotion CSS by providing a visually engaging, interactive UI component that can be used in portfolios, dashboards, landing pages, and modern web applications.
