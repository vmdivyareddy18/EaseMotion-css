# Animated User Profile Card

A clean, modern, and fully responsive **User Profile Card** perfect for social media, team pages, and user directories. This example demonstrates how to combine **EaseMotion CSS** for staggered section entrances, continuous status pulsing, and smooth hover transitions to create an engaging user summary component.

## 🚀 Features

- **Staggered Section Entrances**: Each section (avatar, name, bio, stats, buttons) fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a premium, cascading load effect.
- **Continuous Status Pulsing**: The online status indicator uses `ease-pulse` to draw attention to the user's availability.
- **Interactive Hover States**: Uses `ease-hover-grow` on buttons and social icons for satisfying tactile feedback.
- **Modern Card Design**: Features a gradient cover image, circular avatar with border, and clean typography.
- **Stats Grid**: Clean, responsive grid showing followers, following, and posts with hover effects.
- **Action Buttons**: Primary and secondary buttons with distinct styles and hover transitions.
- **Social Links**: Circular icon buttons with lift effects on hover.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted padding and font sizes.
