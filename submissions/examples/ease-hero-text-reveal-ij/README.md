# Hero Text Reveal

Large heading text that reveals with a `clip-path` animation from right to left on page load. Creates an impactful entrance for hero sections.

## Features

- Clip-path inset reveal animation
- Customizable reveal color, duration, and font size via CSS custom properties
- Triggered by adding `.htr-revealed` class on window load
- Smooth cubic-bezier easing

## Usage

Add `.htr-heading` to the heading element. On `window.load`, add `.htr-revealed` class to trigger the clip-path animation.
