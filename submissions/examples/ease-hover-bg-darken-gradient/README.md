# Darken Gradient on Hover

A darker gradient overlay appears on hover — pure CSS.

## Features
- Dark gradient overlay fades in on hover
- Smooth opacity transition
- Pure CSS, no JavaScript

## Usage
```css
.darken-overlay { opacity: 0; transition: opacity 0.3s; }
.darken-card:hover .darken-overlay { opacity: 1; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
