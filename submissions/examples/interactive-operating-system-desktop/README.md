# Animation: Interactive Operating System Desktop

A complete desktop environment inspired by modern operating systems. It combines desktop icons, floating windows, a glass dock, notifications, and system controls into a cohesive interactive showcase.

## Features

- Pure HTML & CSS
- Desktop icons
- Floating application windows
- Glassmorphism dock
- Notification center
- Top system bar
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="window explorer">
    ...
</div>

<div class="dock">
    ...
</div>
```

## Why is it useful?

This example demonstrates how multiple interactive UI patterns can work together to create a cohesive desktop-like experience. It showcases layouts, animations, glassmorphism, navigation, and window management concepts in a single premium demo.
