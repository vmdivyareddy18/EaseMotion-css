# City Marathon 2026 – Running Event Landing Page

A complete, ready‑to‑copy landing page for a marathon/running event built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – high‑energy race imagery with a register CTA
- **Race Categories** – 5K, 10K, half marathon, full marathon
- **Course Map** – image of the route
- **Registration Pricing** – four tiered prices
- **Training Resources** – free plans and guides
- **Sponsor Logos** – partner brands
- **Register CTA** – blue call‑to‑action section
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-blue-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-blue-800`, `ease-hover-opacity-100`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`, `ease-opacity-80`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
