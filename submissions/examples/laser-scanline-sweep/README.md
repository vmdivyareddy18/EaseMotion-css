# Laser Scanline Sweep Animation Utility

## Overview
This utility creates a futuristic laser scanline effect where a glowing beam sweeps across an element and fades out.

## Usage

```html
<div class="ease-laser-scan">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first interaction
- Great for cyberpunk UI, dashboards, cards, and futuristic interfaces
