# Holographic Navigation Bar

A responsive **Holographic Navigation Bar** inspired by modern food delivery applications. It features a glassmorphism design with animated holographic glow effects using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Glassmorphism UI
- Holographic glowing animation
- Accessible navigation
- No JavaScript required

## Folder Structure

```
holographic-navigation-bar-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
