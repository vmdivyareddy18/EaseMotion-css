# Interactive Testimonial Carousel Cards

A responsive testimonial showcase demonstrating animated customer review cards.

## Features

- Multiple testimonial cards
- Hover animations
- Featured testimonial highlight
- Responsive layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more testimonials
- Change colors and typography
- Modify hover effects
- Adjust card spacing and layout
