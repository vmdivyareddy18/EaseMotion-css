The all Property

Using initial, unset, and revert to isolate components from inherited styles.

The heavily styled parent

This parent applies extreme styles that cascade down to everything inside it.

Normal component (no reset)

This card inherits the parent's chaotic font, colors, alignment, and borders.

all: initial

Everything resets to initial values — including display (back to inline for div!).

all: unset

Inheritable props get parent values; non-inheritable get initial. Behaves like a fresh start.

all: revert

Rolls back to browser/user-agent stylesheet defaults — not quite a full reset.
