# Animated AI Model Training Dashboard

A modern dashboard for visualizing AI model training metrics, experiment progress, and deployment readiness.

## Features

- Training metrics cards
- Accuracy and loss tracking
- GPU utilization indicators
- Progress bars
- Experiment timeline
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any browser.

## Use Cases

- AI dashboards
- ML experiment tracking
- Educational AI projects
- Data science portfolios
