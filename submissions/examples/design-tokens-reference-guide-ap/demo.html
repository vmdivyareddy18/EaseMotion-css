How to Override Tokens

      

        

          
1. Global Overrides

          

            Redefining tokens inside the :root selector applies custom values globally across the application.
          

        


        

          
2. Local/Component Overrides

          

            Limit customizations to specific components by re-declaring variables inside the target class selector.
