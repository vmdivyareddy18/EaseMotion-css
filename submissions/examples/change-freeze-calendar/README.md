# Change Freeze Calendar

Change Freeze Calendar is a responsive release-planning view that shows which days are open for normal deployment, which days require extra review, and which days are under a freeze window.

## Features

- Seven-day calendar for release planning
- Open, review, and freeze status labels
- Freeze window emphasis for restricted days
- Responsive stacked layout on small screens

## Files

- `demo.html` contains the weekly freeze calendar markup.
- `style.css` contains the calendar grid, status labels, and responsive styles.
