# Animated Select Dropdown – Smooth expand with icons

> A production-ready animated select dropdown component that expands with smooth animations and supports icons for each option. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Smooth expand animation** – Elastic opening with staggered options
- ✅ **Icon support** – Each option can have an icon
- ✅ **Keyboard accessible** – Arrow keys, Enter, Escape
- ✅ **Programmatic control** – Open, close, select, reset via API
- ✅ **4 color variants** – Primary, Success, Danger, Dark
- ✅ **Selected state** – Visual feedback for selected option
- ✅ **Click outside to close** – Auto-close when clicking outside
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
