# Ease Liquid Glass Navbar

A modern glassmorphism navigation bar featuring floating liquid background animations, soft blur effects, glowing buttons, and smooth hover interactions. Built entirely with **HTML** and **CSS** without any JavaScript.

---

## ✨ Features

- Liquid glass navigation bar
- Glassmorphism UI design
- Animated floating background blobs
- Smooth hover underline effect
- Gradient action buttons
- Responsive navigation layout
- Pure HTML & CSS
- Lightweight and reusable

---

## 📁 Folder Structure

```

submissions/

```

---

## 🚀 Usage

1. Open `demo.html` in your browser.
2. Ensure `style.css` is in the same folder.
3. Customize colors, text, buttons, or animations as needed.

---

## 🎨 Customization

You can easily change:

- Gradient colors
- Blur intensity
- Border radius
- Navbar spacing
- Animation speed
- Floating blob size
- Button colors
- Typography

Works in all modern browsers supporting:

Optimized for:
