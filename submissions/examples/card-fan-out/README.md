# Stacked Cards Fan-Out Animation

A CSS-only card stack that smoothly fans outward on hover using transform-based animations.

## Features

- Pure CSS
- Transform-based animation
- Responsive layout
- No JavaScript required

## Usage

```html
<div class="card-stack">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
  <div class="card">Card 3</div>
</div>
```

Hover over the card stack to see the cards spread outward in a fan-like arrangement.
