# Stagger Error Page

A responsive corporate-style **404 Error Page** featuring staggered entrance animations created entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Staggered text reveal animation
- Keyboard accessible button
- CSS custom properties
- Supports `prefers-reduced-motion`
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Customization

Modify the CSS variables inside `:root` to change colors, spacing, shadows, border radius, and animation timing.
