# Firefly Particle Drift

Small glowing dots that randomly blink and drift slowly, simulating fireflies hovering at dusk. Ideal for adding ambient nighttime backgrounds or dramatic decoration overlays.

## EaseMotion CSS Classes Showcased

This feature submits a self-contained prototype demonstrating keyframe offsets simulating slow 2D drifting motion combined with asynchronous opacity pulsing.

### Classes:
- `.ease-firefly-container`: Context parent wrapper for positioning the hovering fireflies.
- `.ease-firefly`: Base firefly class establishing shape, color, absolute positioning, glow, and drifting animation rules.
- Pre-configured variations (`.ease-firefly-1` through `.ease-firefly-10`) with staggered offsets, animation delays, durations, and translation patterns.

### Usage in HTML:
```html
<div class="ease-firefly-container">
  <div class="ease-firefly ease-firefly-1"></div>
  <div class="ease-firefly ease-firefly-2"></div>
</div>
```
