# Meditation Timer App

## What does this do?
A meditation timer with a circular progress indicator, selectable durations (3/5/10/15 min), play/pause and reset controls, and a session counter.

## How is it used?
Open demo.html in a browser. Select a duration, click play to start the timer, pause/reset as needed. The circle fills as time elapses.

## Why is this useful?
Demonstrates a circular countdown timer pattern with SVG progress ring, session tracking, and start/pause/reset controls.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
