Animation Accessibility Guidelines

Animations can improve user experience, but excessive motion may cause discomfort for some users. This example demonstrates accessible animation practices.

Accessibility Recommendations

  • Keep motion subtle and purposeful.
  • Avoid rapid flashing animations.
  • Respect user motion preferences.
  • Provide reduced-motion alternatives.
  • Use animations to support usability, not distract users.
If a user enables reduced motion in their operating system, animations should be minimized whenever possible.
