# Animated Stock Market Bull Run

## Overview

Animated Stock Market Bull Run is a lightweight CSS animation that simulates bullish market momentum using a charging bull and an upward-trending growth indicator.

## Features

- CSS-only animation
- Bull market motion effect
- Growth indicator animation
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="bull-run">
  <span class="bull">🐂</span>
  <span class="trend-arrow">📈</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
