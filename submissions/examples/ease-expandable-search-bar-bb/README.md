# Ease Expandable Search Bar

A modern, accessible expandable search bar built with plain HTML, CSS, and vanilla JavaScript.

## Features

- Circular search button that expands into a full input field
- Smooth width transition using CSS custom properties
- Animated icon motion and placeholder reveal
- Enter-key submission that logs the query to the console
- Collapses when clicked outside and the input is empty
- Responsive layout tuned for mobile and desktop
