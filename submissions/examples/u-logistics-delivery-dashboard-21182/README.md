# Logistics Delivery Dashboard

A responsive **Logistics Delivery Dashboard** built using **HTML5** and **CSS3** to demonstrate shipment tracking, fleet monitoring, delivery progress, and logistics workflow animations with EaseMotion CSS.

---

## Features

- Responsive dashboard layout
- Delivery overview cards
- Shipment progress tracker
- Fleet status panel
- Route performance widget
- Recent shipment activity feed
- Notifications panel
- Quick action buttons
- Smooth hover effects
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Active Deliveries
- Completed Deliveries
- Drivers Available
- Daily Revenue

### Logistics Widgets

- Delivery Progress
- Fleet Status
- Route Performance

### Activity

- Shipment Activity Feed
- Notifications Panel

### Quick Actions

- Create Shipment
- Assign Driver
- Export Report

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations.
