# Orthodontist Template
A clean, reassuring, and clinical-forward template for orthodontist offices, dental specialist practices, and oral health clinics.

- **Design**: Uses a "Clinical-Confidence" palette with bright clinical-blue primaries, sterile smile-white surfaces, and accessible grids to convey hygiene, precision, and welcoming care.
- **Use Case**: Orthodontic practices, dental clinics, and specialist oral care centers.
