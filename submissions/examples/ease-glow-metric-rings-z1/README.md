# Ease Glow Metric Rings

## Overview

A premium dashboard widget featuring glowing metric rings with animated pulse effects and glassmorphism styling using only HTML and CSS.

## Features

- Glowing metric rings
- Pulse animation
- Glassmorphism cards
- Responsive Flexbox layout
- Lightweight implementation
- Pure HTML & CSS
