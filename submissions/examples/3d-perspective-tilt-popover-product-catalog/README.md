# 3D Perspective Tilt Popover

A pure CSS animated popover component with a smooth 3D perspective tilt interaction.

## Features

- CSS-only animation
- No JavaScript required
- 3D perspective tilt effect
- Responsive design
- Keyboard accessible using focus states
- Customizable CSS variables

## Usage

```html
<div class="tilt-popover">

  <button class="tilt-trigger">
    Product Details
  </button>

  <div class="tilt-content">
    Product information
  </div>

</div>
