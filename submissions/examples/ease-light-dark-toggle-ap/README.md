# ease-light-dark-toggle

## What is this?

A sleek, animated light/dark mode toggle built with pure HTML, CSS, and JavaScript. It features a smooth sliding switch, dynamic mode labels, responsive design, and remembers the user's preferred theme using `localStorage`.

## How do I use it?

Include the HTML structure, add the provided CSS, and initialize the JavaScript. The toggle automatically switches themes and saves the selected mode for future visits.

```html
<label class="toggle">
  <input type="checkbox" id="toggle">

  <span class="thumb"></span>

  <span class="text light">LIGHT MODE</span>
  <span class="text dark">DARK MODE</span>
</label>
```

## Why use this component?

This toggle offers a lightweight, dependency-free solution for adding theme switching to any website. With smooth animations, accessibility support, responsive styling, and persistent user preferences, it can be easily integrated into portfolios, dashboards, landing pages, or any modern web application.
