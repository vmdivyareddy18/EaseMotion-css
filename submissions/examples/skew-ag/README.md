# Skew Animation Example

A demonstration of a smooth Skew animation for EaseMotion CSS.

## What does this do?
This example demonstrates a continuous skew animation applied to a UI element to create an engaging visual effect.

## How is it used?
Apply the `.skew-box` class to any HTML element you wish to animate:

```html
<div class="skew-box">
    Skew Me!
</div>
```

## Why is it useful?
Skew animations add dynamic motion to static elements, drawing user attention and making the interface feel more alive, aligning perfectly with EaseMotion's goal of bringing effortless motion to the web.
