# Animated AI Chat Conversation Component

A modern AI chat conversation interface inspired by ChatGPT, Claude, and Gemini, built using **pure HTML & CSS**. This component features elegant glassmorphism, smooth animations, responsive layouts, and interactive UI elements, making it ideal for AI products, SaaS dashboards, portfolios, and modern web applications.

## ✨ Features

* 🤖 Modern AI chat interface
* 💬 Animated message appearance
* ⌨️ Typing indicator animation
* ✨ Glassmorphism design
* 🎨 Gradient background with blur effects
* 🌟 Hover animations
* 📱 Fully responsive layout
* 🔄 Smooth scrolling chat area
* 🟢 Online status indicator
* 🚀 Attractive send button interaction
* 🎯 Custom scrollbar styling
* 💻 Built using only HTML & CSS
* 📦 Easy to integrate into existing projects

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/<username>/<repository>.git
```

2. Navigate to the project folder.

```bash
cd <repository>
```

3. Open `demo.html` in your preferred web browser.

No dependencies or build tools are required.
