# Stack Layout

## What does this do?
Adds `.ease-stack` — a vertical flex layout with consistent spacing.

Set gap via `--stack-gap`.

```html
<div class="ease-stack" style="--stack-gap: 1rem">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```

Open `demo.html` to see it in action.

## Preview
