# Animation: ease-text-pop-up

### What does this do?
Reveals text phrases word-by-word sequentially by sliding each element up through an isolated layout overflow masking viewport window.

### How is it used?
Wrap text words inside an `.ease-word-mask` element containing a nested `span` with an inline `--ease-word-index` integer tracking assignment parameter.

### Why is it useful?
It replaces complex JavaScript text-slicing script overhead with ultra-smooth native hardware-accelerated translation layers to yield perfect layout performance metrics.
