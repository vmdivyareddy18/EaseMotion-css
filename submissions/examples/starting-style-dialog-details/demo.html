Part 2: Pure CSS Accordion

What is @starting-style?

It defines the initial visual state of an element the moment it becomes visible (transitions from display: none to block).

Why use allow-discrete?

Because display is a discrete property, it can't be interpolated normally. allow-discrete tells the browser to flip the display value at the end of the exit transition, keeping the element painted while it fades out.

Any JavaScript needed here?

Zero JavaScript. This uses the native HTML <details> element's built-in toggle behavior, enhanced purely by CSS transitions. No max-height hacks required!
