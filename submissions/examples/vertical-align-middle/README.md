# Vertical-Align

## Overview

Explores the `vertical-align` property for inline and table-cell elements. Demonstrates every keyword value (`baseline`, `top`, `middle`, `bottom`, `text-top`, `text-bottom`, `super`, `sub`) and shows practical use cases like icon-text alignment and table cell positioning.

## Key Concepts

- **Baseline alignment**: The default — aligns the baseline of inline elements with the parent's baseline.
- **Top / bottom**: Aligns the top or bottom of the element with the highest/lowest element in the line.
- **Middle**: Aligns the middle of the element with the baseline plus half the x-height of the parent.
- **Text-top / text-bottom**: Aligns with the top/bottom of the parent's font (descender/ascender lines).
- **Super / sub**: Aligns to the superscript or subscript position, useful for math or footnotes.
- **Table cells**: `vertical-align` is fully supported in table cells, where it positions content within the cell's height.

## Usage

Open `demo.html` to see all vertical-align keyword values applied to inline elements in a single row. The icon section shows a practical icon+text pattern. The table section demonstrates top/middle/bottom within table cells. The mixed-heights section shows inline-block elements with different vertical-align values.
