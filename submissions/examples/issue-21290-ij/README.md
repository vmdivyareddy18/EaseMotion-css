# Balloon Float — Hot Air Balloon

CSS hot air balloon floating with drifting background clouds. The `float-balloon` keyframe creates a gentle bobbing motion while clouds drift horizontally.

## Files

- `demo.html` — balloon envelope, basket, ropes, clouds
- `style.css` — float cycle, cloud drift keyframes

## Usage

Open `demo.html` to see the balloon bob among drifting clouds.
