# Neumorphic Rubber Band Toast Component

A gorgeous, highly elastic **Rubber Band Toast** notification component written exclusively for the **EaseMotion CSS** library framework. This component leverages custom structural keyframe weights to perfectly mirror soft geometric Neumorphism alongside bouncy, elastic mechanical physics.

## 🚀 Features
- **Neumorphic Architecture**: Uses dual-source directional outer and inner shadows to simulate real-world surface extrusion.
- **Pure CSS Keyframes**: Hand-crafted multi-step physics interpolation simulating authentic kinetic squashing and stretching (`scale3d`).
- **Zero-Friction Runtime**: Fully functional layout engine execution utilizing hardware-accelerated transforms—no JavaScript required.
