# Skew Hashtag Input Component

An admin panel-inspired dashboard tag entry field layout emphasizing responsive, geometric spatial motion built for the **EaseMotion CSS** UI animation bundle.

## ✨ Features
- ⚡ **Pure CSS Action Layers:** Implements dynamic `:focus-within` selectors to skew layout boundaries fluidly.
- 📐 **Counter-Balanced Text:** Automatically inverts transform matrices inside the active panel field container to guarantee user content legibility.
- ❌ **Interactive Deletion:** Uses structural checkbox hacks to make template tags hideable on click without a single line of JS.
- ♿ **Motion Adaptive:** Seamless dropbacks inside strict environments using `prefers-reduced-motion`.

## 🛠️ Usage

1. Mount the tag structural markup found in `demo.html` into your presentation block forms.
2. Link the core design assets (`style.css`).
3. Tweak structural angles and thematic variables at your container layer:

```css
:root {
  --skew-angle: -6deg;
  --color-primary: #ec4899;
}
