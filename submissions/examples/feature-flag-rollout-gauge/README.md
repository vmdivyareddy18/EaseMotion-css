# Feature Flag Rollout Gauge

An intermediate EaseMotion CSS example for launch-control dashboards. A circular rollout gauge reveals current adoption progress while health signals animate underneath.

## Files

- `demo.html` - Semantic rollout summary and health signal markup.
- `style.css` - Responsive panel layout, conic gauge animation, rotating ring, and staggered signal entrance.

## Usage

Open `demo.html` in a browser to preview the animated rollout gauge.
