# Animated Link Underline Component

## What does this do?
Decorative CSS hover underline effects for links with multiple direction, style, and creative variants.

## How is it used?
```html
<a href="#" class="ani-link ani-ltr">Hover Me</a>
<nav class="ani-nav"><a href="#" class="ani-link ani-center">Link</a></nav>
```

## Why is it useful?
Enhances link interactivity with smooth animated underlines. Multiple styles suit different design contexts from minimal to playful. Pure CSS, no JavaScript required.
