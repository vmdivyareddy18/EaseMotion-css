@property Typed Color Token Animation

@property with syntax: '<color>' makes CSS custom properties type-aware — enabling native color interpolation in transitions and @keyframes.

Token cascade — one keyframe updates all consumers

--ease-token-primary

Swatch background, border, and text all read the same token.

One animation, many consumers

Change the token once — every element using it updates in sync.

Without @property, animating a custom property is impossible — the browser treats it as an opaque string and snaps between values. syntax: '<color>' enables true interpolation.

Theme switch — transition tokens between theme states

Design Token Theme

Background, heading color, and accent bar all transition smoothly when a theme class is applied.

Toggling a class changes --ease-token-bg, --ease-token-primary, and --ease-token-accent. Because they are typed as <color>, the transition interpolates through LAB color space instead of snapping.

Per-element typed token — hover to transition

Each button uses its own --ease-btn-bg typed property. transition: --ease-btn-bg 0.4s ease interpolates smoothly — try hovering.
