# Elastic Maintenance Page

A responsive **Elastic Maintenance Page** inspired by space-themed interfaces. The component uses a CSS-only elastic animation to create a lively maintenance screen without JavaScript.

## Features

- Pure HTML & CSS
- Elastic animation
- Responsive layout
- Accessible structure
- No JavaScript required

## Folder Structure

```
elastic-maintenance-page-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
