# Universal Card Hover Effect

**What does this do?**
Adds a consistent, smooth hover effect (lift and shadow) to all card components across the framework.

**How is it used?**
Any element using the `.ease-card` class will automatically inherit this hover effect. No additional utility classes are required.

```html
<div class="ease-card">
  <!-- Content here -->
</div>
```

**Why is it useful?**
Currently, many card variants (Base Card, Shadow Card, etc.) lack interactive feedback. This update ensures all cards respond consistently when hovered, improving visual hierarchy and user engagement in alignment with the Animation First philosophy.
