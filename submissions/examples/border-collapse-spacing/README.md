# border-collapse-spacing

## Overview

Demonstrates `border-collapse: collapse` versus `border-collapse: separate` with various `border-spacing` values for HTML tables. Shows how adjacent borders merge in collapse mode, and how spacing between cells appears in separate mode with different spacing configurations.

## Usage

Open `demo.html` in any browser. Four table examples show collapse, default separate, wide 8px spacing, and asymmetric 12x4 spacing. A visual comparison grid at the bottom shows all four modes on identical mini tables for easy comparison.

## Features

- Collapse vs separate border modes on identical data
- Symmetric and asymmetric border-spacing values
- Visual comparison grid with labeled mini tables
- Consistent data across all examples for accurate comparison
- Responsive grid layout for the comparison section
- `prefers-reduced-motion` support
- Dark theme with green accent and blue headers
