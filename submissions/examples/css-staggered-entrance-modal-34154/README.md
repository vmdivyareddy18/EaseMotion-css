# CSS Staggered Entrance Modal

## What does this do?

Creates a responsive e-commerce checkout modal with a smooth staggered entrance animation using only HTML and CSS.

## How is it used?

Open `demo.html` in a browser.

Customize animation timing and scale through the CSS custom properties defined in `style.css`.

## Why is it useful?

Provides a lightweight, accessible checkout modal with modern staggered animations that require no JavaScript.
