# Acupuncture & TCM Template
A mindful, serene, and tradition-respecting template for acupuncture clinics, TCM (Traditional Chinese Medicine) practitioners, and integrative wellness centers.

- **Design**: Uses a "Harmony-Balance" palette with sage-toned neutrals, qi-gold accent energy, and soft, rounded elements to communicate peace, wisdom, and holistic healing.
- **Use Case**: Acupuncture practices, TCM clinics, and holistic integrative health centers.
