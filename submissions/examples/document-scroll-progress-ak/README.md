# Document Scroll Progress

## What does this do?
A scroll progress indicator for a document container with a gradient progress bar, percentage readout, and automatic section detection.

## How is it used?
Open demo.html in a browser and scroll the document area. The sticky progress bar fills as you scroll, and the current section name updates below the document.

## Why is this useful?
Demonstrates a container-scoped scroll progress pattern with section detection, suitable for documentation, long-form articles, or embedded content readers.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
