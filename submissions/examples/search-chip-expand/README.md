# Search Chip Expand

A responsive search chip that starts compact and expands when users hover or focus the control.

## Files

- `demo.html` - accessible search label and input markup.
- `style.css` - width transition, focus-within state, icon scaling, and reduced-motion handling.

## Highlights

- Uses `focus-within` so keyboard users get the same expanded state.
- Keeps the input inside a stable pill-shaped control.
- Includes a reduced-motion fallback.
