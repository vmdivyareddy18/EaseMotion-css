# Ease Select Arrow Rotate Component

An elegant, highly customizable HTML/CSS-only solution for custom drop-down menus featuring a smooth, animated rotating indicator arrow.

## Features
- **Pure CSS**: Zero JavaScript required; uses modern CSS pseudo-classes (`:focus-within`) to detect interactions.
- **Smooth Animations**: Clean transition effects utilizing `cubic-bezier` timing variables.
- **Fully Customizable**: Handled completely via globally scopeable CSS Variables (`--arrow-rotation`, `--transition-speed`, etc.).
- **Accessible & Responsive**: Accessible via keyboard navigation (`Tab` and native option handling) and responds perfectly across device widths.
