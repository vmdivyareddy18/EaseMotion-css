# Bounce Map Embed

A modern **Bounce Map Embed** component built entirely with **HTML** and **CSS**, inspired by SaaS dashboard design patterns. The component features animated location markers, a dashboard-style map interface, responsive layouts, and accessibility support—all without requiring JavaScript.

---

## Features

- Pure HTML & CSS
- Zero JavaScript
- SaaS dashboard-inspired design
- Animated bouncing location marker
- Responsive card layout
- Keyboard accessible
- Supports `prefers-reduced-motion`
- Easily customizable using CSS

## Customization

You can easily customize the component by modifying the colors, spacing, animation timing, and marker positions in `style.css`.

Common customizations include:

- Map background colors
- Location marker colors
- Card border radius
- Shadow intensity
- Animation duration
- Typography
- Button styles

---

## Usage

```html
<link rel="stylesheet" href="style.css">

<section class="map-card">

    <div class="map">

        <div class="pin pin-primary">
            <span></span>
        </div>

        <div class="pin pin-secondary">
            <span></span>
        </div>

        <div class="pin pin-tertiary">
            <span></span>
        </div>

    </div>

</section>
```

Open **demo.html** directly in any modern browser to preview the component.
