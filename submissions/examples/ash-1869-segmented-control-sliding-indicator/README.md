# Segmented Control Sliding Indicator

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern segmented control component with an animated sliding indicator inspired by iOS navigation and dashboard filters.

## Features

- Sliding active indicator
- Segmented navigation
- Dashboard filter UI
- Theme switcher example
- Smooth transitions
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="segmented-control">

  <div class="slider"></div>

  <button class="segment active">
    Overview
  </button>

</div>
```

## Main Classes

```css
.segmented-control
.segment
.slider
.active
.compact
```
