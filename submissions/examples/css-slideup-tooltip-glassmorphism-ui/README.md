# CSS Slide-Up Tooltip – Glassmorphism UI Layout

## Overview

A responsive Glassmorphism UI tooltip component built entirely with HTML and CSS. The component features a smooth slide-up animation, frosted glass cards, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Slide-Up tooltip animation
- Glassmorphism UI design
- Responsive frosted-glass cards
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--glass-bg`
- `--glass-border`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--slide-distance`
- `--shadow`
- `--blur`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any **Preview** button to display the animated Slide-Up Tooltip. Colors, blur intensity, spacing, animation timing, and shadows can all be customized using the CSS variables defined in `style.css`.
