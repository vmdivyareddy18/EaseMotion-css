🔘 Split Button & Segmented Control

Split button with primary action + dropdown menu, button groups with connected segments, and radio-based segmented control with active state.

Button Group

Connected buttons without gaps — buttons are joined via border-radius: 0 on middle items with margin-inline-start: -1px.

Split Button

Primary action with a toggle arrow that opens a dropdown menu of related options. Click the arrow to toggle the menu. Includes divider and danger item support.

Click the arrow ▾ to open split button menu

Segmented Control

Radio-based mutually exclusive segments using hidden inputs. Uses role="radiogroup" for accessibility. Active segment gets highlighted background.
