# CSS Spring Physics Accordion

A pure CSS accordion component engineered for accessible web layouts, mimicking physical spring mechanics (stiffness, mass, and damping) entirely without JavaScript.

## 🚀 Features
- **Zero JavaScript:** Built entirely on top of high-performance CSS transforms and custom property configuration states.
- **Spring Physics Simulation:** Uses an optimized multi-stage spring cubic easing sequence to create a micro-rebound/snappy bounce interaction.
- **A11y Core Blueprint:** Structured explicitly around keyboard navigability, high contrast visibility states, and clear focus indicators.
- **Exposed Design Tokens:** Completely themeable via exposed parameters like `--spring-bounce`, `--accordion-speed`, and state-scaling factors.

## 📂 File Structure
- `demo.html` - Semantic accessible document structure.
- `style.css` - Custom spring keyframe states and theme tokens.
