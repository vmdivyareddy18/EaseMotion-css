# Animated Before & After Image Comparison Component

A CSS-only image comparison slider with smooth reveal animations, sliding divider effects, and responsive design.

---

## Features

- Smooth before and after reveal animation
- Animated sliding divider
- Hover-based interaction
- Responsive layout
- Modern rounded card design
- No JavaScript required
- Accessibility support using `prefers-reduced-motion`

---

## Classes

| Class | Description |
|--------|-------------|
| `.comparison-container` | Main comparison wrapper |
| `.before-image` | Original image |
| `.after-wrapper` | Animated overlay container |
| `.after-image` | Transformed image |
| `.slider-line` | Vertical divider line |
| `.slider-circle` | Drag indicator icon |

---

## Use Cases

- Photography comparisons
- Image editing showcases
- Product transformations
- Fitness progress images
- Design before/after examples

---

## Accessibility

Supports reduced motion preferences by disabling transition animations.
