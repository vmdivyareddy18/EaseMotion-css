# Ease Orbit Menu

## Overview

Ease Orbit Menu is a premium CSS-only navigation component that displays menu items in a circular orbit around a central action button. It combines glassmorphism, gradient accents, floating animations, and rotating orbit rings to create a futuristic user interface suitable for modern web applications.

---

## Features

- 🌌 Circular orbit navigation layout
- ✨ Animated rotating orbit rings
- 💎 Glassmorphism card design
- 🎯 Central floating action button
- 🚀 Smooth hover lift animations
- 🎨 Modern gradient color palette
- 📱 Fully responsive layout
- ⚡ Pure HTML & CSS (No JavaScript)

## Usage

Simply open `demo.html` in any modern web browser.

No installation, JavaScript, or external libraries are required.
