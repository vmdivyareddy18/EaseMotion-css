ease-typewriter-once

A typewriter effect that types out text character by character on hover — then stays fully visible.

CSS typewriter effect |

Types once per hover — cursor fades out at the end.

How It Works

The text width animates from 0 to 100% using steps(22) (one step per character) with animation-fill-mode: forwards, so the text stays revealed. The cursor blinks during typing via a separate @keyframes and fades out at the end. On hover, both animations trigger simultaneously using :hover on the parent wrapper.
