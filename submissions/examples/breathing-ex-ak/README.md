# Guided Breathing Exercise

## What does this do?
A guided breathing exercise following the 4-7-8 pattern (inhale 4s, hold 7s, exhale 8s) with an animated expanding/contracting circle and phase guidance.

## How is it used?
Open demo.html in a browser. Click "Start" to begin the breathing cycle. Follow the circle animation and text prompts. The timer shows remaining seconds in each phase.

## Why is this useful?
This provides a mindfulness and relaxation tool for stress reduction. It demonstrates CSS keyframe animations for breathing visualization and timed phase transitions.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to practice breathing.
