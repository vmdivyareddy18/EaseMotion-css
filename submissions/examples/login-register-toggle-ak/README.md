# Login / Register Toggle

## What does this do?
A toggle interface that switches between a login form and a registration form with a slide-in animation. Includes basic client-side validation for required fields.

## How is it used?
Open demo.html in a browser. Click "Login" or "Register" to switch forms. Fill in the fields and submit to see validation feedback.

## Why is it useful?
This provides a compact authentication UI pattern suitable for landing pages, dashboards, or any app that needs sign-in and sign-up flows without navigating away.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see the toggle in action.
