# Gradient Patterns

A gallery of repeating gradient patterns using `repeating-linear-gradient`, `repeating-radial-gradient`, and `repeating-conic-gradient`. Includes stripes, checkerboard, zigzag, polka dots, waves, and tartan patterns.

## Features

- Eight distinct pattern swatches with live CSS code labels
- Interactive pattern builder with color pickers and size controls
- Pattern overlay demonstrations on gradient backgrounds
- Fully self-contained with no external dependencies

## Usage

Open `demo.html` in any modern browser. Use the custom builder to experiment with different gradient types, colors, and sizes in real time.
