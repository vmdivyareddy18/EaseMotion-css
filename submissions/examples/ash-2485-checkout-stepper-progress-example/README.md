# Checkout Stepper Progress Example

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern checkout progress stepper that visually guides users through a multi-step purchasing workflow.

## Features

- Multi-step checkout indicator
- Active and completed step states
- Progress line visualization
- Responsive layout
- Glassmorphism checkout card
- Modern e-commerce pattern
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="stepper">

    <div class="step active">

        <div class="step-circle">
            3
        </div>

    </div>

</div>
```

## Main Classes

```css
.stepper
.step
.step-circle
.completed
.active
.progress-line
.checkout-card
```
