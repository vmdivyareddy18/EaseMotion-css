# Global Security Operations Center

An ultra-premium Security Operations Center dashboard and command platform featuring live DDoS threat indices, real-time security log tables, automated mitigation controls, and monospaced console shells. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **SOC Command Header**: Platform logo, classified threat flags, and uptime badges.
- **Threat Matrix Cards**: Four metrics (Active Intrusions, Blocked Exploits, Analysts, Compliance Rating).
- **Incident Logger Table**: Detailed alert history tracking timestamp, IP source, incident types, severities, and mitigations.
- **Incident Playbook Controls**: Interactive mitigation triggers simulating threat isolation commands.
- **SOC Telemetry Shell**: Interactive terminal streaming security logs.

## 🎨 Theme Details

- **Palette**: Dark Cyber (`#030712`), Slate Surface (`#0f172a`), Cyan Accent (`#06b6d4`), Threat Red (`#ef4444`), Status Green (`#10b981`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Blinking cyber highlights, warning pulses, and slide-ups.
