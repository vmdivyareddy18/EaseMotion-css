# 🃏 3D Flip Cards

A collection of beautiful, pure CSS 3D flip cards with different animation styles.

## 🚀 Features
4 unique flip animations:
1. **Horizontal Flip** - Classic left-to-right rotation
2. **Vertical Flip** - Top-to-bottom rotation
3. **Diagonal Flip** - Combined X & Y axis rotation
4. **3D Cube Flip** - Realistic cube rotation with depth

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All 3D animations
- `readme.md` - Documentation

## 🛠️ How to Use
1. Open `demo.html` in any browser
2. Hover over cards to see the flip effect
3. Copy the CSS classes for your own projects

## 🧠 CSS Concepts Used
- `perspective` for 3D depth
- `transform-style: preserve-3d` for child elements
- `backface-visibility: hidden` to hide back side
- `rotateX`, `rotateY` for rotation
- `translateZ` for depth positioning
