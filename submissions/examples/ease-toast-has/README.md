# Stacking Notification Toasts (`ease-toast-has`)

### 1. What does this do?
An animated notification stacker displaying toasts that enter with springy translate offsets and fade away.

### 2. How is it used?
Target the `.toast-card` class for the toasts:
```html
<div class="toast-card">
  <span class="toast-icon">💡</span>
  <div class="toast-content">
    <h4>Title</h4>
    <p>Body</p>
  </div>
</div>
```

### 3. Why is it useful?
Provides non-blocking application alerts with robust responsive layouts and micro-engineered sliding animations.
