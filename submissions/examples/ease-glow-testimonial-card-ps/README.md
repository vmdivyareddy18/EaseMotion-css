# Glow Testimonial Card (#41887)

### 1. What does this do?
This is an intermediate interactive customer feedback showcase card optimized with neon ambient radial glows, stylized quote marks, and layout transformations on user hover states.

### 2. How is it used?
Integrate the raw component anywhere on your pages by referencing the `.glow-testimonial-card` base class:

```html
<div class="glow-testimonial-card">
  <div class="ambient-glow"></div>
  <p class="testimonial-text">"Customer Quote Here..."</p>
  <div class="user-profile">...</div>
</div>
