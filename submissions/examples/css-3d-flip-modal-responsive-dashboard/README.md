# CSS 3D Flip Modal – Responsive Dashboard Layout

## Overview

A responsive Dashboard modal built entirely with HTML and CSS. The component features a smooth 3D flip opening animation, responsive dashboard cards, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip animation
- Responsive Dashboard layout
- Modern card-based interface
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--success`
- `--bg`
- `--surface`
- `--text`
- `--muted`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Open Dashboard** to display the animated Responsive Dashboard 3D Flip Modal. Colors, animation timing, flip angle, scaling, spacing, and shadows can be customized using the CSS custom properties defined in `style.css`.
