ease-scroll-fade

Elements fade into view as they enter the viewport — driven by IntersectionObserver. CSS handles every frame; JS only adds a class when the threshold is crossed.

Scroll to see it

Default — Fade Up

Content reveals as you scroll

Each element starts transparent and slightly below its resting position. As it crosses the IntersectionObserver threshold it transitions to full opacity and translates up into place.

🎯

One Class

Add ease-scroll-fade to any element — the observer handles the rest.

⚡

GPU Animated

Only opacity and transform animate — zero layout or paint work.

♿

Accessible

prefers-reduced-motion makes elements immediately visible with no transition.

🎛️

Configurable

Duration, easing, distance, and delay are all CSS custom properties.

Direction Variants

Fade from Left

.ease-scroll-fade-left slides in from the left. Great for feature descriptions paired with a visual on the right.

🖼️

Fade from Right

.ease-scroll-fade-right mirrors the motion for alternating layout rows — the standard content + visual pattern.

📊

Fade + Scale

.ease-scroll-fade-scale grows from 90% to 100% as it fades in — a subtler entrance for cards and images.

🔍

Usage

Drop-in, zero config

Link the stylesheet, add the class, include the script. One IntersectionObserver instance watches every .ease-scroll-fade* element on the page.

HTML

<!-- 1. Link stylesheet -->
<link rel="stylesheet" href="style.css" />

<!-- 2. Add class to any element -->
<h2 class="ease-scroll-fade">Fades in on scroll</h2>

<!-- Direction variants -->
<div class="ease-scroll-fade-left">Slides from left</div>
<div class="ease-scroll-fade-right">Slides from right</div>
<div class="ease-scroll-fade-scale">Scales in</div>

<!-- Stagger delays for grouped elements -->
<div class="ease-scroll-fade stagger-1">First</div>
<div class="ease-scroll-fade stagger-2">Second</div>
<div class="ease-scroll-fade stagger-3">Third</div>

<!-- Per-instance overrides via CSS variables -->
<div class="ease-scroll-fade"
     style="--ease-fade-duration:1s; --ease-fade-distance:60px">
  Slower, longer travel
</div>

<!-- 3. Include the observer script -->
<script src="scroll-fade.js"></script>
