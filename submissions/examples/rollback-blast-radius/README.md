# Rollback Blast Radius

Rollback Blast Radius is a responsive release recovery panel that separates rollback risk into low, medium, and high impact zones. It gives operators a quick view of which services can safely roll back and which need extra recovery attention.

## Features

- Rollback window summary
- Low, medium, and high impact zones
- Compact card layout for operational scanning
- Responsive single-column mobile view

## Files

- `demo.html` contains the rollback impact markup.
- `style.css` contains the layout, labels, and responsive rules.
