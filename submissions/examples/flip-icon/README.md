# Flip Icon Animation

A lightweight 3D flip animation utility for icons using pure CSS and `@keyframes`.

## Features

- Pure CSS animation
- Smooth 3D flip effect
- Lightweight and reusable
- Uses `rotateY()` transform
- Responsive and browser-friendly

## Preview

Hover over the icons to see the flip animation.

## Usage

```html
<span class="flip-icon">⭐</span>
