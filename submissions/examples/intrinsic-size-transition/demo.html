Accordion — Click to Expand

Each panel smoothly expands to its natural height: auto on click.

interpolate-size: allow-keywords is a new CSS property that allows browsers to smoothly animate between a numeric value (like 0) and a keyword value (like auto or max-content) — something that was previously impossible without JavaScript.

Chrome 129+, Edge 129+, and Safari 18+ support interpolate-size. For older browsers, this demo falls back gracefully — content is still accessible, just without the smooth animation.

We use @supports (interpolate-size: allow-keywords) to only apply the transition in browsers that understand it. On unsupported browsers, the content simply snaps open/closed instantly — still fully accessible.
