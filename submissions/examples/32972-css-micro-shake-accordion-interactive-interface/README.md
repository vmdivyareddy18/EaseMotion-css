# CSS Micro-Shake Accordion (Interactive Interface Layouts)

P pure CSS responsive accordion designed for interactive configuration panel interfaces. Features a tactile "micro-shake" horizontal wobble animation when hovering or focusing on the panel headers.

## Features
- **Tactile Micro-Shake Transition**: High-frequency keyframe shake animation that gives immediate hover/focus feedback.
- **Bespoke Interactive Coral Theme**: Styled with a profissional settings layout in deep slate, cool gray, and bright coral accents.
- **Pure CSS checked Logic**: Built using native checkbox inputs for zero-JS toggle support.
- **Full Keyboard Focus outlines**: Standard WCAG compliant focus outlines on header label items.

## Files
- `demo.html`: Live interactive demonstration showing settings/configuration options.
- `style.css`: Stylesheet containing the keyframe shaking animation and accordion panel layouts.
