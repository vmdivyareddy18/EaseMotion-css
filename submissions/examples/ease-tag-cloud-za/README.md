# CSS Tag Cloud

## What does this do?
A CSS tag cloud with weighted font-size, color, and opacity based on data-weight attributes with category color coding and interactive hover.

## How is it used?
```html
<div class="tgc-cloud"><span class="tgc-tag" data-weight="50">Tag</span></div>
```

## Why is it useful?
Visual tag importance without JavaScript. Weight-based sizing via data attributes. Category color themes. Interactive hover variant. Ideal for blog sidebars and content metadata.
