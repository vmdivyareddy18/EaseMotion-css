# Floating Orbit Popover Neumorphic

A pure CSS animated Floating Orbit Popover designed with a soft neumorphic UI style.

## Features

- Pure CSS animation
- Neumorphic soft shadows
- Floating orbit interaction
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="soft-orbit">

<button class="soft-trigger">
Explore
</button>

<div class="soft-popover">
Content
</div>

</div>
