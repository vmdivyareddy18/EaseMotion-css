# Draw Border Line Popover

A responsive **Draw-Border Line Popover** built using pure CSS for EaseMotion CSS.

## Features

- Pure CSS animation
- Draw-border line effect
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Usage

```html
<div class="popover-container">
    <button class="popover-btn">Hover Me</button>

    <div class="popover">
        Popover content
    </div>
</div>
```
