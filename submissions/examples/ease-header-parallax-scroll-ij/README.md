# Header Parallax Scroll (#36723)

A header with layered background elements that move at different speeds during scroll to create a parallax depth effect.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--header-bg` | `#1e1b4b` | Base background color of the header |
| `--parallax-speed` | `0.4` | Speed multiplier for parallax layers (0–1) |
| `--text-color` | `#fef3c7` | Color of the header text |
| `--overlay-color` | `rgba(15,23,42,0.5)` | Overlay tint applied over the layers |

## Interactive Demo

Open `demo.html` and scroll the card content. Adjust the speed slider to control layer motion intensity.
