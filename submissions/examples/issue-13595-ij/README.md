# ease-drawer-right

An animated right-side drawer panel that slides in from the right edge. Ideal for shopping carts, filter panels, or notification trays.

## Usage

Include the `style.css` in your project and use the HTML structure with `.drawer`, `.drawer-overlay`, and the toggle button. The demo uses JavaScript for toggling.

## Demo

Open `demo.html` in a browser to see the component in action.
