# Animated Floating Action Button (`ease-fab-cv`)

A lightweight Floating Action Button (FAB) component built using **HTML5** and **Pure CSS**.

## Features

- Expanding FAB
- Rotating icon animation
- Pulse animation
- Ripple hover effect
- Primary, Info, Success and Danger variants
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-fab-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
