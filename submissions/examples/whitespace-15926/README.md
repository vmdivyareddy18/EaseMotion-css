# White-space Utilities
This submission introduces utilities to control how white space inside an element is handled.

- `.ws-normal`: Default behavior (sequences of whitespace collapse).
- `.ws-nowrap`: Sequences of whitespace collapse, but text never wraps.
- `.ws-pre`: Respects original whitespace and line breaks.
- `.ws-pre-line`: Collapses whitespace but preserves line breaks.
- `.ws-pre-wrap`: Preserves whitespace and line breaks, while allowing text to wrap.

These utilities provide fine-grained control over text layout and formatting.
