# Cyberpunk Neon Elastic Bounce Tooltip

A futuristic, high-contrast, zero-dependency pure CSS interactive tooltip component engineered to complement cyberpunk, arcade, and tech-noir dark interface configurations. Implements an overshooting physics-elastic micro-interaction bounce during focus transitions.

Resolves Issue: #34384

## ✨ Features
* **Zero JavaScript Pipeline Lifecycle:** Leverages hardware-accelerated CSS pseudo-triggers (`:hover` and `:focus-visible`) to compute overlay tracking properties instantly on the style thread.
* **Elastic Overshoot Spring Physics:** Maps a dedicated `cubic-bezier` vector profile that extends properties briefly beyond bounds before pulling down into a locked target state, mimicking a high-impact mechanical component response.
* **Exposed Parameter Matrices:** Tweak total transition duration spans, baseline scale reduction baselines, and layout timing functions right from clean properties within the global `:root` node.
* **Immersive Gaming Visual Palette:** Uses stylized geometry, neon box-shadow radials, and `clip-path` cuts while respecting full accessibility guidelines.
