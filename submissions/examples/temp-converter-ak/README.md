# Temperature Converter

## What does this do?
A temperature converter with bidirectional conversion between Celsius, Fahrenheit, and Kelvin, plus a visual color-coded thermometer.

## How is it used?
Open demo.html in a browser. Type a value in any of the three fields (C, F, or K) to see instant conversion in the others. The thermometer bar animates and changes color from blue (cold) to red (hot).

## Why is this useful?
This provides an intuitive temperature conversion tool with visual feedback. It demonstrates bidirectional data binding and color-coded state representation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the converter.
