# Stagger Reveal Animations

## What does this do?
Pure CSS staggered reveal animations for sequential entrance effects on lists, grids, and grouped elements.

## How is it used?
```html
<ul class="st-list st-fade-up">
  <li class="st-item">Item 1</li>
  <li class="st-item">Item 2</li>
</ul>
```

## Why is it useful?
Creates professional sequential entrance animations without JavaScript. Multiple animation types (fade-up, scale-in, slide-left, rotate-in). Customizable delay and duration via CSS variables.
