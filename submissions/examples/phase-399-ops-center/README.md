# Operations Center Dashboard

A small example dashboard built with pure CSS — server status, traffic chart, and an alert feed.

## Features

- CSS Grid layout, responsive at 768px
- Dark mode (default) and light mode, with auto system preference detection
- CSS-only pulse animation for healthy server nodes
- Respects `prefers-reduced-motion`
- Sharp corners, no `border-radius`
- Poppins font

## Files

- `demo.html` — markup + small inline script for the theme toggle
- `style.css` — all styles
- `README.md` — this file

## Usage

Open `demo.html` in a browser. Click the theme toggle in the top right to switch between dark and light mode.
