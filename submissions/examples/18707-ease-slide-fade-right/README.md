# Animation: Ease Slide Fade Right

## Overview

Ease Slide Fade Right is a smooth entrance animation that combines a fade-in effect with horizontal motion from left to right. To create a more natural appearance, the animation also includes a subtle scale transition and a modern easing curve.

The effect is designed to draw attention to content while maintaining a clean and professional user experience.

## Features

* Smooth fade-in transition
* Left-to-right slide motion
* Subtle scale enhancement
* Lightweight CSS-only implementation
* Modern cubic-bezier easing
* Accessibility support using `prefers-reduced-motion`
* Reusable across different UI elements

## Use Cases

* Hero sections
* Navigation menus
* Feature cards
* Buttons
* Landing page content
* Portfolio websites
* Product showcases

## Example Usage

```html
<div class="ease-slide-fade-right">
  Welcome to EaseMotion CSS
</div>
```

## Animation Behavior

1. Starts with zero opacity.
2. Appears slightly left of its final position.
3. Includes a subtle scale transition.
4. Gradually becomes fully visible.
5. Smoothly settles into its final position.

## Accessibility

This animation respects the user's motion preferences through the `prefers-reduced-motion` media query.
