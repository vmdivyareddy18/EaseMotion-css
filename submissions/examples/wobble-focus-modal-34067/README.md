# CSS Wobble-Focus Modal

A responsive, high-impact marketing modal layout designed for landing page conversion points.

## Core Mechanics
- **Elastic Keyframe Vectoring**: Uses a pure CSS multi-stage elastic scale and rotation keyframe pattern to snap user focus instantly.
- **Fluid Structural Design**: Fully self-contained inside its sandbox directory with zero global styling leak risks.
- **Premium Dark Aesthetics**: Uses luxury dark panels and purple focal indicators optimized for professional SaaS and tech contexts.
