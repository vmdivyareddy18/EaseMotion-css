# CSS Dropdown Menu

## What does this do?
Pure CSS dropdown menus with hover submenus, nested menus, mega menu, and focus-based dropdown.

## How is it used?
```html
<nav class="dd-nav"><ul><li class="dd-has"><a href="#">Menu</a><ul class="dd-sub"><li>Item</li></ul></li></ul></nav>
```

## Why is it useful?
Multi-level navigation without JS. Hover dropdowns, nested submenus, focus accessibility, mega menu columns.
