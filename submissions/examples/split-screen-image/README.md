# Split-Screen Image Comparison

A side-by-side split-screen layout comparing two gradient panels with a white divider in the middle. The left panel (Before) uses slate gradients, and the right panel (After) uses indigo-to-violet gradients. Labels are centered in each half.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the static split-screen comparison.

## Accessibility notes

The panels are decorative gradients. Labels provide textual context. No JavaScript is required.
