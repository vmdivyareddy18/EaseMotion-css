# Morph Glow Feature Grid

A lightweight CSS-only feature grid component with smooth morphing glow hover effects.

## Features

- Pure HTML and CSS
- No JavaScript required
- Responsive grid layout
- Smooth glow animation
- Accessible reduced motion support

## Usage

Add the feature card structure:

```html
<div class="feature-card">
    <span>⚡</span>
    <h2>Feature</h2>
    <p>Description</p>
</div>
