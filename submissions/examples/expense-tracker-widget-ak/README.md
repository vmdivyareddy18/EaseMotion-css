# Expense Tracker Widget

## What does this do?
A personal expense tracker with add/delete entries, total/weekly/monthly summaries, category selection, and localStorage persistence.

## How is it used?
Open demo.html in a browser. Add expenses with description, category, and amount. View totals and delete individual entries. Data persists across sessions.

## Why is this useful?
Demonstrates a complete CRUD expense tracking widget with period-based summaries and persistent local storage.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript (localStorage)

## Preview
Open demo.html directly in your browser.
