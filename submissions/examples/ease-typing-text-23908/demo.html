ease-typing-text

CSS-only typewriter animation with no JavaScript required.

Single Line

Basic typewriter that types one line and stops. Uses steps() for character-by-character effect.

Hello, I'm a typewriter! 🖋️ 
<span class="ease-typing-text">Hello, I'm a typewriter!</span>

Multi-Line

Each line types sequentially with staggered delays.

Line one appears first. Line two follows after a delay. And line three finishes the set. 
<span class="ease-typing-multiline">
  <span class="ease-typing-line">Line one</span>
  <span class="ease-typing-line" style="--ease-typing-line-index: 1">Line two</span>
  <span class="ease-typing-line" style="--ease-typing-line-index: 2">Line three</span>
</span>

Speed Variants

Use ease-typing-fast (1s) and ease-typing-slow (4s) modifiers.

Fast typing speed!
Default speed
Slow typing speed...
class="ease-typing-text ease-typing-fast"  /* 1s */
class="ease-typing-text"                /* 2s (default) */
class="ease-typing-text ease-typing-slow"  /* 4s */

Cursor Colors

Green cursor variant
Red cursor variant
No cursor at all

All variants respect prefers-reduced-motion and show full text immediately.
