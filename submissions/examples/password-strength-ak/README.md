# Password Strength Checker

## What does this do?
A live password strength checker that evaluates password criteria (length, uppercase, lowercase, number, special character) with a progress bar and visual feedback.

## How is it used?
Open demo.html in a browser. Type a password to see real-time strength evaluation. Each criterion lights up green when met. The bar color shifts from red (weak) to purple (very strong).

## Why is it useful?
This provides a reusable password strength indicator for registration forms or settings pages, helping users create more secure passwords with immediate visual feedback.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to test the checker.
