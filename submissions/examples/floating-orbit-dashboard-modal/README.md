# Floating Orbit Modal for Responsive Dashboard Layouts

A pure HTML & CSS modal featuring floating and orbiting decorative elements, designed for modern dashboard interfaces.

## Features

- Pure HTML & CSS
- Floating modal entrance
- Orbiting decorative elements
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No animation JavaScript

## Files

```
demo.html
style.css
README.md
```

## Customization

Update the CSS variables inside `:root`:

- Animation duration
- Primary color
- Border radius
- Shadow
- Orbit sizes

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
