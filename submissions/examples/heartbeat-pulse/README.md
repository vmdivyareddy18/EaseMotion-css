# Heartbeat Pulse Animation

A realistic double-beat heartbeat pulse animation for like buttons, health indicators, favorite icons, and live status dots.

## Demo

Open `demo.html` in your browser to see the heartbeat animation on:
- Heart icons (default and love variants)
- Health status pulse dot
- Interactive like button

## Usage

```html
&lt;!-- Heart icon --&gt;
&lt;div class="heart-icon heartbeat-anim"&gt;&lt;/div&gt;

&lt;!-- Status indicator --&gt;
&lt;div class="status-dot heartbeat-anim"&gt;&lt;/div&gt;

&lt;!-- Any element --&gt;
&lt;span class="heartbeat-anim"&gt;❤️&lt;/span&gt;
