# CSS Floating Orbit Accordion for Neumorphic Soft Layouts

## Overview

This project demonstrates a responsive accordion component featuring a smooth **Floating Orbit** animation built entirely with HTML and CSS. Designed with a neumorphic soft UI aesthetic, the accordion provides an elegant way to organize and display expandable content without requiring JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- CSS-only Floating Orbit animation
- Neumorphic soft UI design
- Responsive accordion layout
- Native `<details>` and `<summary>` elements
- Keyboard-accessible interaction
- CSS custom properties for easy customization
- Smooth expand/collapse transitions
- No JavaScript or external libraries required

- Accordion panels smoothly expand and collapse.
- The active panel gently floats using CSS transforms.
- A glowing decorative element continuously orbits around the active accordion item.

## Customization

The component uses CSS custom properties for quick customization.

```css
:root{
    --bg:#e9edf3;
    --surface:#eef2f7;
    --primary:#4f7cff;
    --ring:#9cb7ff;
    --text:#2b3445;
    --radius:22px;
}
```

You can customize:

- Background color
- Primary accent color
- Orbit glow color
- Border radius
- Shadow intensity
- Typography
- Animation duration
- Accordion spacing
- Hover effects

This project demonstrates how a CSS-only Floating Orbit animation can enhance a neumorphic accordion interface while maintaining accessibility and performance. By using semantic HTML elements and modern CSS techniques, the component remains lightweight, reusable, responsive, and easy to customize for FAQs, documentation, help centers, and information panels.
