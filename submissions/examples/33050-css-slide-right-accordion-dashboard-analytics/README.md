# CSS Slide-Right Accordion (Dashboard Analytics Layouts)

A horizontal accordion component designed for dashboard layouts. The panels expand horizontally, sliding their right boundary to reveal detailed real-time warning logs and system analytics.

## Features
- **Horizontal Slide-Right Expansion**: Active panels expand horizontally from `60px` to `380px` width, showcasing an elegant way to layout side-by-side metric cards.
- **Bespoke Obsidian & Amber Theme**: Styled with a dark mode warm alert aesthetic, highlighting real-time incident levels.
- **Pure CSS checked State**: Built utilizing native hidden radio inputs for clean toggle state tracking.
- **Full Keyboard Access**: Labels bind to inputs, allowing keyboard focus and navigation via arrow/space keys.

## Files
- `demo.html`: Live interactive demonstration of the horizontal slide-right accordion.
- `style.css`: Stylesheet containing the flex/width horizontal transitions and active state rules.
