# Browser Compatibility Checklist

This documentation guide triages and fixes consistency issues within the browser specifications section by organizing feature support parameters into a consistent list structure.

---

## Modern Feature Specifications

Below is the structured list of browser specifications supported by EaseMotion CSS:

- **CSS Subgrid**
  - Chrome support version: `104+`
  - Safari support version: `16+`
  - Firefox support version: `71+`
- **Scroll Viewport Timelines**
  - Chrome support version: `115+`
  - Safari support version: `17+`
  - Firefox support version: `120+`
- **Backdrop Filter Saturation**
  - Chrome support version: `76+`
  - Safari support version: `9+`
  - Firefox support version: `70+`
- **preserve-3d Perspective**
  - Chrome support version: `36+`
  - Safari support version: `9+`
  - Firefox support version: `16+`

---

## Verification Steps

1. Open `demo.html` in a browser.
2. Confirm the alignment of checklist items is consistent and legible.
