# Swing Hover Minimalist Tech Modal

A clean and responsive Swing-Hover modal built entirely with HTML and CSS for minimalist technology-inspired interfaces.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Minimal monochrome UI
- Swing-Hover animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--rotate
--radius
--bg
--surface
--text
--muted
--border
```
