# CSS Dropdown Select

## What does this do?
Custom styled dropdown selects with appearance removal, icon prefixes, color themes, and size variants.

## How is it used?
```html
<div class="dd-wrap"><select class="dd-select"><option>Option</option></select></div>
```

## Why is it useful?
Replaces browser default select styling with consistent custom design. No JavaScript required. Multiple color themes, sizes, icon prefix support. Focus states with ring. Disabled state included.
