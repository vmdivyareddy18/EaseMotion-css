# ease-hover-border-expand

A smooth hover animation where the outline expands outward, creating a subtle breathing effect.

## Preview

When hovering over the element:

- Outline becomes visible
- Outline expands outward using `outline-offset`
- Smooth easing transition

---

## Features

- Pure HTML & CSS
- No JavaScript required
- Lightweight
- Easy to customize
- Uses CSS variables

---

## CSS Variable

```css
--ease-border-expand-offset: 10px;
```

Change this variable to control how far the outline expands.

Example:

```css
:root{
    --ease-border-expand-offset: 14px;
}
```
