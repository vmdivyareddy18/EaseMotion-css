# CSS Elastic Slide Tooltip - Medical Dashboard Layouts

A pure CSS animated tooltip component designed for medical dashboard interfaces.

## Features

- Pure CSS animation
- Elastic slide transition effect
- Healthcare dashboard inspired design
- Responsive layout
- Keyboard accessible
- CSS variable customization


## Usage

Open `demo.html` directly in a browser.

Hover or focus on dashboard buttons to view the animated tooltip.


## Customization

Modify animation values using CSS variables:

```css
:root {
    --tooltip-time: 0.45s;
    --tooltip-scale: 1;
    --tooltip-easing: cubic-bezier(.68,-0.55,.27,1.55);
}
