# ease-hover-bg-expand-circle

## What does this do?
A beautiful CSS hover animation where a background color grows outwards as a circle from the center of the element on hover, smoothly filling the background of buttons, cards, or other containers.

## How is it used?
Add the `ease-hover-bg-expand-circle` class to any container or button. Define the custom `--ease-circle-bg-color` property on the element to control the color of the expanding circle:

```html
<button class="btn ease-hover-bg-expand-circle" style="--ease-circle-bg-color: #6366f1;">
  Hover me
</button>
```

## Why is it useful?
It provides a high-quality, modern ripple or radial expanding transition without any JavaScript dependencies. It leverages CSS transforms and hardware-accelerated scaling, ensuring smooth 60fps animations.

## Tech Stack
- HTML
- CSS (Vanilla, No JavaScript)
