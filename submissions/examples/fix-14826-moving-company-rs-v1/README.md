# Issue #14826: Advanced: Full Moving Company Landing Page

This folder contains a standalone demonstration of a Moving Company Landing Page as requested in Issue #14826.

## Files
- `demo.html`: The HTML structure for the landing page.
- `style.css`: The CSS styling emphasizing a clean, trustworthy, and actionable design.

## How it works
The demo provides a complete layout for a moving company featuring:
- A trustworthy color palette combining professional blue with energetic yellow accents.
- A prominent "Get a Quote" hero section.
- A clear, icon-based services overview (Residential, Commercial, Packing).
- A step-by-step "How It Works" timeline.
- Fully responsive design using modern CSS Flexbox and Grid.
