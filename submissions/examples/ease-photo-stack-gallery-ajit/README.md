# Ease Photo Stack Gallery

A CSS-only stacked photo gallery component with a smooth spread animation.

## Features

- 📷 CSS-only component
- 🎴 Stacked photo layout
- ✨ Hover spread animation
- 📱 Responsive design
- 🚀 No JavaScript required
- 🎨 Easy to customize

## Folder Structure

```
ease-photo-stack-gallery-ajit/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Create the gallery:

```html
<div class="ease-photo-stack-gallery">
    <img src="image1.jpg" alt="">
    <img src="image2.jpg" alt="">
    <img src="image3.jpg" alt="">
    <img src="image4.jpg" alt="">
    <img src="image5.jpg" alt="">
</div>
```

Open `demo.html` directly in any modern browser to view the component.
