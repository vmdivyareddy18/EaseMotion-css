# Ease Spotlight Audio Player

A cyberpunk-inspired animated audio player component built with pure CSS.

## Features

- Spotlight glowing animation
- Animated audio waveform
- Cyberpunk visual style
- Pure CSS implementation
- Responsive layout
- No JavaScript required

## Usage

Add the HTML structure:

```html
<div class="audio-player">
  <div class="spotlight"></div>
</div>
