## Neon Glow Pulsation** 

**1. What does this do?
Adds a rhythmically pulsing neon glow border animation to any element.

**2. How is it used?** 
Add the class `neon-glow-pulse` to my HTML element:

```html
<div class="neon-glow-pulse">Hello World</div>

**3. Why is it useful?**
It creates a visually striking effect for cards, buttons, or callouts in dark themed UIs - fitting EaseMotion CSS's philosophy of lightweight, single-class utilities.
