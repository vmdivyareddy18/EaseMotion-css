# ease-loader-bars

Equalizer-style bouncing bars loader with staggered animation.

## Usage

```html
<div class="ease-loader-bars">
  <span></span><span></span><span></span>
  <span></span><span></span>
</div>
```

## CSS Variables

| Variable | Default | Description |
|---|---|---|
| `--ease-loader-color` | `#6366f1` | Bar color |
| `--ease-loader-speed` | `1s` | Animation duration |
| `--ease-loader-bar-width` | `6px` | Width of each bar |
| `--ease-loader-bar-height` | `40px` | Height of each bar |
| `--ease-loader-gap` | `4px` | Gap between bars |
