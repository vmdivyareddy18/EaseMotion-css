# Compact Card Variant

1. **What does this do?**  
   Adds a compact card variant idea for dense lists, dashboards, and small content previews.

2. **How is it used?**  
   ```html
   <article class="compact-card">
     <h3>Card title</h3>
     <p>Short supporting text.</p>
   </article>
   ```

3. **Why is it useful?**  
   It fits EaseMotion CSS by making tight card layouts readable and reusable without forcing every card to feel oversized.
