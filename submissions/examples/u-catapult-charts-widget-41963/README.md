# Catapult Charts Widget

A Blog-inspired Catapult Charts Widget built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Catapult animation
- Responsive chart layout
- Accessible HTML
- No JavaScript required
