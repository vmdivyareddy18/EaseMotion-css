# Swing Hover Cyberpunk Neon Modal

A futuristic cyberpunk-inspired modal built using pure HTML and CSS with a glowing Swing-Hover animation.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Cyberpunk neon styling
- Swing-Hover animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--rotate
--bg
--card
--cyan
--pink
--radius
```
