Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive data table component featuring hover interactions, keyboard focus states, and mobile-friendly layouts.

## Features

- Responsive table design
- Row hover highlighting
- Keyboard focus support
- Accessibility-friendly navigation
- Status badges
- Mobile card layout transformation
- Smooth transitions
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<tr tabindex="0">
    <td>User Data</td>
</tr>
```

## Main Classes

```css
.table-card
.table-wrapper
.badge
.active
.pending
.inactive
```
