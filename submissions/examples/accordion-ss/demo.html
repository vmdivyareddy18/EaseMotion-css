What browsers are supported?
Chrome 49+, Firefox 31+, Safari 9.1+, Edge 15+. Internet Explorer is not supported. The framework uses CSS custom properties, CSS Grid, Flexbox, and modern animations.
Can I customize colors?
Absolutely! Override CSS custom properties like --ease-color-primary in your own stylesheet. EaseMotion wraps everything in @layer so your styles always win — even if the framework loads after your CSS.
How big is the bundle?
Only ~15 kB minified! You can also load granular files — just import the animation categories you need. The core variables.css is always required first.
