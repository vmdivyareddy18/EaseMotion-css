# Ease Quantity Input Stepper

A modern **Quantity Input Stepper** built using **pure HTML and CSS**. It provides a clean quantity selector interface with smooth hover animations, responsive design, and customizable styling using CSS custom properties.

## Features

- Pure HTML & CSS
- Quantity input with increment/decrement controls
- CSS-only animation using `@keyframes`
- Hover and active button effects
- CSS Custom Properties for easy customization
- Responsive design
- Keyboard accessible
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
ease-quantity-input-stepper/
├── demo.html
├── style.css
└── README.md
```

## Customization

Modify the CSS variables in the `:root` section to customize:

- Primary color
- Background color
- Border radius
- Shadow
- Transition speed

## Browser Support

Compatible with all modern browsers.
