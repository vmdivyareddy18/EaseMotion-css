# Ease Yin Yang Spin

A yin-yang symbol rotating continuously in a smooth loop.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.yin-yang` element.

## Custom CSS Variables

| Variable          | Default  | Description                    |
|-------------------|----------|--------------------------------|
| `--yin-color`     | `#333`   | Dark (yin) color of the symbol |
| `--yang-color`    | `#fff`   | Light (yang) color of the symbol|
| `--spin-duration` | `8s`     | Duration of one full rotation  |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
