Animated Accordion

Pure CSS accordion using native <details> / <summary> with a smooth grid-template-rows height transition and rotating chevron indicator. Zero JavaScript.

What is EaseMotion CSS? ▸

EaseMotion CSS is a curated, animation-first CSS framework. It provides utility classes and components for smooth transitions, hover effects, scroll reveals, and premium UI patterns — all without JavaScript.

How do I add animations to my project? ▸

Simply link easemotion.css in your HTML and apply utility classes like ease-fade-in, ease-slide-up, or ease-zoom-in to any element. No build step required.

Does EaseMotion CSS support dark mode? ▸

Yes. The framework uses CSS custom properties throughout, making it straightforward to override color tokens for dark mode using prefers-color-scheme or a class-based theme toggle.

Is it accessible? ▸

This accordion uses native <details> and <summary> elements which are keyboard-navigable and screen-reader friendly out of the box. aria-hidden is applied to the decorative chevron icon.

Can I contribute to EaseMotion CSS? ▸

Absolutely! Submit a self-contained demo under submissions/examples/your-feature-name/ containing demo.html, style.css, and README.md. The maintainer reviews and integrates accepted submissions into the core framework.
