# Animated Toast Notification Component

A reusable, framework-independent, animated toast notification component built using semantic HTML, vanilla CSS, and JavaScript. It demonstrates multiple notification variants, smooth CSS transitions, accessibility best practices, and complete responsive design for both light and dark modes.

## Features
* **4 Distinct Variants**: Success, Error, Warning, and Info.
* **Stacking Support**: Allows multiple toast notifications to appear simultaneously.
* **Smooth Animations**: Clean slide-in and fade-out animations using pure CSS.
* **Theme Support**: Adaptive styling for both Light and Dark modes.
* **Accessibility**: Implements appropriate ARIA roles (`role="alert"` or `role="status"`) and keyboard-accessible close actions.
* **Zero Dependencies**: Pure HTML, CSS, and JS.
