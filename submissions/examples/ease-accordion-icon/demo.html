EaseMotion Accordion

Pure CSS accordion using the modern grid-template-rows animation trick.

EaseMotion CSS is a lightweight library focused on beautiful micro-interactions and transitions. It prioritizes pure CSS solutions over JavaScript whenever possible.

Historically, animating height: auto in CSS was impossible. This component bypasses that limitation by animating a CSS Grid container's grid-template-rows property from 0fr to 1fr.
