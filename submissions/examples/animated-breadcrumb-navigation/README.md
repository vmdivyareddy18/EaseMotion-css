# Animated Breadcrumb Navigation

A responsive animated breadcrumb navigation component built using pure HTML and CSS.

## Features

- Animated active states
- Hover underline effects
- Icon separators
- Responsive layouts
- Smooth transitions and hover animations
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated breadcrumb navigation component.

## Files

- `demo.html` → Breadcrumb navigation structure
- `style.css` → Breadcrumb styles and animations
- `README.md` → Documentation
