# Animated Breadcrumb Navigation

A clean, modern, and fully responsive **Breadcrumb Navigation System** with smooth hover transitions and animated indicators. This example demonstrates how to combine **EaseMotion CSS** with custom CSS to create engaging, accessible navigation breadcrumbs in multiple styles.

## 🚀 Features

- **Smooth Hover Effects**: Each breadcrumb link smoothly transitions color, background, and position on hover using CSS transitions.
- **Animated Separators**: Chevron and arrow separators with subtle color transitions.
- **Sliding Current Page Indicator**: An underline indicator that smoothly scales in from left to right when the page loads.
- **Icon Animations**: Icons scale up slightly on hover for tactile feedback.
- **Four Style Variations**:
  - **Basic**: Clean, minimal design with slash separators
  - **Arrow**: Connected arrow-style with background fills
  - **Pill**: Rounded pill-shaped crumbs with hover color inversions
  - **Compact**: Mobile-optimized with collapsible middle items
- **Accessible**: Uses proper `<nav>`, `<ol>`, and `aria-current="page"` attributes for screen readers.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted spacing and font sizes.
