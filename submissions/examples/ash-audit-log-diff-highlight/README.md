# Audit Log Diff Highlight

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern audit log interface that visualizes configuration changes using before-and-after comparisons, highlighted diffs, activity timelines, and status indicators.

## Features

- Audit log dashboard
- Before vs after comparison
- Added value highlighting
- Removed value highlighting
- Change indicators
- Activity timeline
- User metadata section
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="diff-card added">
  <strong>Admin</strong>
</div>
```

## Main Classes

```css
.audit-panel
.diff-grid
.diff-card
.added
.removed
.timeline-item
```
