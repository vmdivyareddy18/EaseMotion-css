# Springy Tooltip (`ease-tooltip-has`)

### 1. What does this do?
An animated tooltip that reveals itself on hover using elastic scaling transitions.

### 2. How is it used?
Wrap elements inside `.tooltip-container` and target the tooltip content:
```html
<div class="tooltip-container">
  <button>Hover Me</button>
  <span class="tooltip-text">Tooltip Text</span>
</div>
```

### 3. Why is it useful?
Provides high-fidelity context help that matches the animated and expressive philosophy of EaseMotion CSS.
