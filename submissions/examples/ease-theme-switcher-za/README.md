# Theme Switcher

## What does this do?
A CSS-only theme switcher using custom properties and radio buttons to toggle between dark, light, and colorful themes.

## How is it used?
```html
<div data-theme="dark">
<label class="ts-btn ts-dark"><input type="radio" name="ts" value="dark"><span>Dark</span></label>
</div>
```

## Why is it useful?
Enables multi-theme support without JavaScript. Smooth transitions via CSS variables. Easy to extend with new themes.
