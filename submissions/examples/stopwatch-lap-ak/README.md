# Stopwatch with Lap Recording

## What does this do?
A stopwatch with start, stop, resume, and lap recording functionality. Displays elapsed time in HH:MM:SS.mm format with an accumulative lap list.

## How is it used?
Open demo.html in a browser. Click Start to begin timing. Press Lap to record split times without stopping. Stop pauses the timer; Reset clears all data.

## Why is it useful?
This provides a complete stopwatch component for timing applications, sports tracking, or productivity tools. It demonstrates precise interval timing and lap management with vanilla JavaScript.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the stopwatch.
