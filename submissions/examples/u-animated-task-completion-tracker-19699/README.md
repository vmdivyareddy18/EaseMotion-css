# Animated Task Completion Tracker

A modern task and productivity tracking component built using HTML and CSS.

## Features

- Animated progress bar
- Completed and pending task tracking
- Productivity score card
- Achievement badge
- Responsive layout
- Hover animations

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- Project dashboards
- Productivity apps
- Habit trackers
