🔑 Modern OTP/PIN Custom Carets

When design calls for unique inputs like validation blocks, standard cursors look misplaced. By applying .ease-caret-transparent, we can create custom pulsing bottom bars using pure CSS triggers when fields are focused.

HTML / CSS Integration Code 
<!-- CSS Core Utility -->
.ease-caret-transparent {
  caret-color: transparent !important;
}

<!-- HTML Markup -->
<input type="text" class="ease-caret-transparent" placeholder="No caret...">
