# Profile Avatar Hover

This example adds a profile card button where the avatar lifts and gains a soft ring on hover or focus.

## Files

- `demo.html` contains the profile card button markup.
- `style.css` defines the avatar lift, ring state, focus style, and reduced-motion fallback.

## Highlights

- Uses a real button for the interactive profile card.
- Keeps the card and avatar dimensions stable while animating.
- Applies the same treatment to hover and keyboard focus.
- Removes avatar movement when reduced motion is requested.
