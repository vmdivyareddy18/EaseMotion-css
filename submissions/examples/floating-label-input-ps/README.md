# Floating Label Input Component

1. What does this do? Creates a sleek input field where the placeholder smoothly floats up to become the input label when focused or filled.
2. How is it used? Copy the `.floating-input-group` markup. It is critical that the `<input>` element has `placeholder=" "` (a single space) for the CSS `:placeholder-shown` selector to correctly detect when the input has content.
3. Why is it useful? It provides a modern, space-saving form aesthetic commonly found in Material Design, implemented here cleanly with zero JavaScript, ensuring high performance and ease of use.
