# CSS 3D Flip Card

## What does this do?
CSS 3D card flip animations with horizontal/vertical directions, perspective tilt, and profile card variant.

## How is it used?
```html
<div class="fl-card"><div class="fl-inner"><div class="fl-front">Front</div><div class="fl-back">Back</div></div></div>
```

## Why is it useful?
Creates interactive flip card experiences without JavaScript. Two flip directions. Perspective depth. Profile card template. Hover-triggered with smooth cubic-bezier transition.
