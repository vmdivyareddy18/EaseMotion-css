# Animated Skill Progress Bars

A set of horizontal progress bars that animate from zero to their target width on load. Each skill item shows a label and percentage alongside a gradient-filled bar. The animation is triggered via `requestAnimationFrame`.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to see the skill bars animate on load.

## Accessibility notes

The percentage is displayed as text next to each bar. The bar fill animation is decorative. Reduced motion disables the animation.
