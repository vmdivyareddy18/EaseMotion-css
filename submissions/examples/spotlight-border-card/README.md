````md
# Spotlight Border Card

A modern CSS-only card hover effect featuring a glowing spotlight border.

## Features

- CSS-only implementation
- Smooth hover animation
- Glowing spotlight border
- Reusable card component
- Responsive design

## Usage

Include the HTML structure:

```html
<div class="spotlight-card">
  <h2>Spotlight Border Card</h2>
  <p>Your content here.</p>
  <button>Explore</button>
</div>
```
````

Link the stylesheet:

```html
<link rel="stylesheet" href="style.css" />
```

## Files

- `demo.html` – Live demonstration
- `style.css` – Component styling
- `README.md` – Documentation

## Preview

Hover over the card to see:

- Spotlight border glow
- Elevation effect
- Smooth animation
