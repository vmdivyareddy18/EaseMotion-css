# ease-dots-loader

Three dots bounce in sequence as a loading indicator, fully CSS-only.

## Files

- `demo.html` – loader demo with dark theme
- `style.css` – dot bounce keyframes, staggered delays, configurable custom properties
- `README.md` – this file

## Usage

Open `demo.html` in a browser. Customize via `--dot-color`, `--dot-size`, and `--dot-speed` on the `.loader` element.
