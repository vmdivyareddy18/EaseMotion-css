# Magnetic Pull Popover

## What it does

Magnetic Pull Popover is a pure CSS animated popover component that creates a smooth attraction-style interaction effect.

It uses CSS transitions and transforms without requiring JavaScript.

## Features

- CSS-only animation
- Smooth magnetic pull interaction
- Keyboard accessible with focus states
- Responsive design
- Customizable animation variables

## Usage

```html
<div class="magnetic-popover">
  <button class="magnetic-trigger">
    Hover Me
  </button>

  <div class="popover-content">
    Content goes here
  </div>
</div>
