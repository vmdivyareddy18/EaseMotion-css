# Animated Open Source Contribution Dashboard

A responsive dashboard component that visualizes open-source activity, contribution statistics, achievements, and activity heatmaps.

## Features

- Contribution statistics cards
- Activity heatmap
- Achievement badges
- Hover interactions
- Animated card reveals
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Contribution counts
- Achievement badges
- Heatmap colors
- Animation timing
- Dashboard sections

## Use Cases

- Open source portfolios
- GSSoC contributor dashboards
- GitHub profile showcases
- Developer portfolios
- Community contribution pages
