# Rotate Click Tooltip

Dashboard analytics tooltip component featuring a smooth rotate-click interaction.

---

## Features

- Pure CSS implementation
- Rotate-Click animation
- Dashboard inspired styling
- Responsive layout
- Keyboard accessibility
- Focus support
- CSS variables
- Zero JavaScript dependency
