# Kanban WIP Limit Column Indicator

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A Kanban board column indicator showcasing Work-In-Progress (WIP) limits with visual status indicators for healthy, warning, and completed states.

## Features

- Kanban board layout
- WIP limit indicators
- Warning state animation
- Task card styling
- Responsive design
- Glassmorphism UI
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<span class="wip warning">
    5 / 5
</span>
```

## Main Classes

```css
.kanban-board
.column
.column-header
.wip
.normal
.warning
.success
.task-card
```
