# Hover Tilt

**EaseMotion Submission** · Contributor: kavin553

---

## 1. What does this do?

Gives an element a subtle 3D tilt on both the X and Y axes when hovered, creating an interactive, Apple-style card effect.

## 2. How is it used?

Apply the `hover-tilt-card` class to any element:

```html
<div class="hover-tilt-card">
  Hover me
</div>
