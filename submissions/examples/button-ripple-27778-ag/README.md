# Micro-Interaction Button Ripple Effect

An interactive button component demonstrating CSS scale-expanding ripple click effects.

---

## What is Included

1. **`demo.html`**: A showcase dashboard containing interactive triggers, position overlays, and click script coordinates.
2. **`style.css`**: Button styles, relative layouts, overflow clipping, expanding animation scales, and motion rules.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Click anywhere on either the Primary or Accent button.
3. Confirm that a white circular ripple animation expands outward from the exact cursor coordinate clicked.
