# inverted-colors Media Query

Demonstrates the `inverted-colors` media query for handling inverted display modes. Shows image correction with `filter: invert(1)`, text and UI element adjustments, and icon handling for emoji in inverted mode.

## Usage

Open `demo.html` and enable color inversion in your OS settings (Windows: Magnifier > Invert colors, macOS: Accessibility > Display > Invert Colors). The demo's corrected images re-invert to display naturally, while the uncorrected examples show the default inverted appearance.

## Browser Support

`inverted-colors` is supported in Chrome, Edge, Safari, and Firefox. The demo applies `filter: invert(1)` to re-invert images and icons back to their original appearance. Dark theme uses `#0a0f1e` background with `#e2e8f0` text for the default (non-inverted) state.
