# ease-permission-request-icon

Animated permission request icons for mobile and web applications.

## Features

- Camera permission icon
- Microphone permission icon
- Location permission icon
- Gentle pulse animation
- Bounce attention animation
- Responsive design
- Pure HTML and CSS

## Acceptance Criteria Covered

- Subtle scale pulse effect
- Gentle bounce animation
- Attention-seeking motion
- Common permission dialog UI pattern
