# Data Pipeline Lineage Health Map

A modern Data Pipeline Lineage Health Map built entirely with HTML and CSS.

## Overview

This example showcases an enterprise dashboard for monitoring data pipeline lineage, tracking upstream and downstream dependencies, visualizing pipeline health, identifying incidents, and ensuring reliable data delivery. The interface is fully responsive and built using only HTML and CSS.

## Features

- Responsive pipeline monitoring dashboard
- Data lineage overview
- Pipeline health monitoring
- Dependency status tracking
- Incident reporting
- Throughput analytics
- Dataset monitoring
- Modern enterprise UI
- Pure HTML & CSS (No JavaScript)
