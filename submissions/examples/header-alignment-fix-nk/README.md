# Header Alignment Fix Example

## What does this do?

Demonstrates common header alignment issues such as missing logo padding, element clipping, and inconsistent spacing, alongside a corrected responsive implementation.

## How is it used?

```html
<header class="fixed-header">
  <div class="logo">EaseMotion CSS</div>
  <nav class="nav-links">
    <a href="#">Docs</a>
    <a href="#">Components</a>
    <a href="#">Animations</a>
  </nav>
  <button class="demo-btn">Live Demo</button>
</header>
```

## Why is it useful?

Proper header spacing and alignment improve usability, readability, and responsiveness. This example demonstrates best practices for modern navigation layouts while showcasing CSS transitions and accessibility support.
