Transform Origin Utilities

CSS transform-origin utilities to specify the pivot point around which element transitions and transformations (like rotation and scaling) are calculated.

What is transform-origin?
The transform-origin property defines the location of the pivot point for an element's transformations. For example, a card rotates around its center by default, but you can set the origin to top left to pivot from the corner instead.

Note: The cyan dots indicate the active transform origin (pivot point) for each example box.

Origin Point Comparisons

Center Pivot (Default) .to-center
Rotation
SPIN

Rotates around the exact geometric center. This is the initial browser behavior.

Top Left Pivot .to-top-left
Rotation
SPIN

Rotates around the top-left corner, causing the entire card to swing wide below and to the right.

Top Right Pivot .to-top-right
Rotation
SPIN

Pivots around the top-right corner, swinging wide below and to the left.

Bottom Left Scale .to-bottom-left
Scale
SHRINK

Scales down toward the bottom-left corner. The bottom-left edge remains anchored in place.

Bottom Right Scale .to-bottom-right
Scale
SHRINK

Scales down toward the bottom-right corner. The bottom-right corner point remains stationary.

Top Pivot Scale .to-top
Scale
SHRINK

Shrinks vertically and horizontally toward the top-center edge. The top edge remains aligned.

Bottom Pivot Rotation .to-bottom
Rotation
SPIN

Rotates using the bottom-center edge as the anchor axis, creating a pendulum style swing.

Left Pivot Scale .to-left
Scale
SHRINK

Collapses toward the left-center boundary. The left edge stays stationary.

Right Pivot Rotation .to-right
Rotation
SPIN

Rotates using the right-center edge as the anchor point, spinning wide to the left.
