Object View Box Crop & Pan

CSS-native image cropping with object-view-box

View Box Coordinate System

object-view-box uses an embedded SVG coordinate system where inset(20% 10%) defines the visible portion of the replaced element. The values are inset(top right bottom left).

Crop Variations

No Crop
object-view-box: none

Full image visible with object-fit: cover

Center Crop
object-view-box: inset(15%)

15% cropped from all sides

Crop Bottom
object-view-box: inset(0 0 40% 0)

Crops 40% from the bottom

Crop Right
object-view-box: inset(0 40% 0 0)

Crops 40% from the right

Pan Effects

Pan Left
object-view-box: inset(0 0 0 20%)

Shifts visible area to reveal more of the right side

Pan Up
object-view-box: inset(20% 0 0 0)

Shifts visible area down to reveal more of the top

Pan Top-Right
object-view-box: inset(0 0 30% 30%)

Reveals top-right region of the image

Zoom In
object-view-box: inset(25%)

Equivalent to zooming in — tighter view box

Comparison with object-fit

object-fit: cover

Fills the container; crops automatically but you cannot control what is cropped.

object-fit: contain

Fits the whole image; may leave letterboxing.

object-view-box pan

Manual control over the visible region — you choose what to show.

Uses a placeholder gradient as the replaced element. Replace with an <img> for real images.
