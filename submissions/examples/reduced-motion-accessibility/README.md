# Reduced Motion Accessibility Utilities

A responsive reduced-motion accessibility demo built using pure HTML and CSS.

## Features

- `prefers-reduced-motion` media query support
- Reduced animation utility classes
- Motion-safe hover interaction helpers
- Accessibility-focused animation handling
- Smooth fallback behavior for motion-sensitive users
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the reduced-motion accessibility utilities demo.
