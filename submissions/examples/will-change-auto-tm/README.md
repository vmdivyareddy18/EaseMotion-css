# Will Change Auto

will-change — Hints browser to optimize element rendering

## Features
- Uses `--ease-color-surface`
- Uses `--ease-radius-lg`
- Uses `--ease-shadow-md`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Will Change Auto</span>
</div>
```

## Why is it useful?

This submission demonstrates the will change auto CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
