# FAQ Accordion Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be combined with minimal JavaScript to create a responsive FAQ accordion.

---

## Components

- FAQ Heading
- Five Accordion Items
- Expand/Collapse
- Responsive Layout

---

## Folder Structure

```
submissions/examples/faq-accordion-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Click on any question.
3. Observe the accordion expanding and collapsing.

---

## Features

- Responsive layout
- Minimal JavaScript
- Accordion interaction
- Uses existing EaseMotion CSS animation classes
