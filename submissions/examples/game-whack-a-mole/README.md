# 🔨 Whack-a-Mole

A classic Whack-a-Mole game built with HTML, CSS, and JavaScript. Moles pop up randomly from holes, and you need to click them to score points. The game features a timer and increasing difficulty as your score rises.

## 🎮 How to Play

- **Click** a mole when it appears to whack it and earn 1 point.
- The game lasts **30 seconds**.
- As your score increases, the **moles appear faster** — the game gets progressively harder!
- Try to achieve the **highest score** possible before time runs out.

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `Space` or `Enter` | Start the game |
| `R` | Reset the game |
| Click / Tap | Whack a mole |

## ✨ Features

- **Timer** – 30-second countdown with visual display
- **Increasing Speed** – Moles appear faster as your score goes up
- **High Score** – Saved locally in your browser
- **Visual Feedback** – Moles have a "hit" animation, holes glow when active
- **Responsive Design** – Plays well on desktop and mobile
- **Keyboard Shortcuts** – Quick access to start and reset

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, animations, responsive design)
- Vanilla JavaScript (no dependencies)
