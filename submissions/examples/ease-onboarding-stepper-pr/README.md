# Ease Onboarding Stepper

## What does this do?

Provides an animated onboarding stepper component for multi-step workflows and user journeys.

## How is it used?

```html
<div class="stepper">

  <div class="step completed">
    <div class="circle">✓</div>
    <span>Create Account</span>
  </div>

  <div class="step active">
    <div class="circle">2</div>
    <span>Profile Setup</span>
  </div>

</div>
```

## Why is it useful?

Onboarding steppers are widely used in signup flows, checkout processes, surveys, profile setup screens, and SaaS onboarding experiences. This component provides a reusable, animation-first solution that aligns with EaseMotion CSS's philosophy of delivering polished UI components with minimal setup.
