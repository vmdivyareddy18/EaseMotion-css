# Animated Fitness Goal Tracker Card

A modern fitness tracking component built using HTML and CSS.

## Features

- Animated progress bar
- Daily goal completion tracking
- Steps and calories statistics
- Workout streak tracker
- Achievement section
- Responsive design

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- Fitness apps
- Wellness dashboards
- Workout trackers
- Health monitoring platforms
- Activity management systems
