# Animated Toast Notifications

A clean, modern, and fully responsive **Toast Notification System** with smooth slide-in/slide-out animations and auto-dismiss functionality. This example demonstrates how to combine **EaseMotion CSS** for smooth transitions, continuous progress bar countdowns, and interactive hover states to create non-intrusive user feedback.

## 🚀 Features

- **Smooth Slide-In Animation**: Uses `ease-slide-in-right` to elegantly introduce toasts from the right side of the screen.
- **Smooth Slide-Out Animation**: Custom `ease-slide-out-right` animation for graceful dismissal.
- **Auto-Dismiss with Progress Bar**: Toasts automatically disappear after 5 seconds with a visual countdown progress bar.
- **Four Toast Types**: Success (green), Error (red), Warning (amber), and Info (blue) with distinct colors and icons.
- **Stackable Notifications**: Multiple toasts stack vertically with proper spacing.
- **Interactive Close Button**: Manual dismiss button with `ease-hover-grow` for tactile feedback.
- **Color-Coded Progress Bars**: Each toast type has a matching progress bar color.
- **Fully Responsive**: Adapts to mobile screens with full-width toasts.
