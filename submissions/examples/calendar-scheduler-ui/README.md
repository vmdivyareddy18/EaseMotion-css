# Animation: Calendar Scheduler UI

A modern calendar scheduler inspired by Google Calendar, Notion Calendar, Outlook, and Cron. It features event indicators, current-day highlighting, and interactive date cards.

## Features

- Pure HTML & CSS
- Monthly calendar layout
- Event badges
- Current day highlight
- Responsive design
- Hover interactions
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="day today">

    <span class="date">11</span>

    <div class="event-badge">
        Launch Event
    </div>

</div>
```

## Why is it useful?

Calendars are one of the most common productivity interfaces used in scheduling, project management, booking systems, and team collaboration tools. This example demonstrates a clean and modern implementation suitable for dashboards and web applications.
