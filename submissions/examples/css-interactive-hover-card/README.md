# Interactive Hover Card

A modern interactive card component built using pure HTML and CSS.

## Features

- Hover lift animation
- Image zoom effect
- Animated button
- Responsive layout
- Pure HTML & CSS
- Beginner Friendly

## Folder Structure

```text
css-interactive-hover-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Browser Support

Works in all modern browsers.
