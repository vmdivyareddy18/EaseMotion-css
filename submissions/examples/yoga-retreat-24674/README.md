# Yoga Retreat Template
A calm, minimalist, and holistic template designed for yoga retreats, wellness spas, and mindful travel experiences.

- **Design**: Uses a "Harmony & Balance" palette with zen-sage greens, soft-sand neutrals, and lotus-lavender accents to convey peace, wellness, and alignment.
- **Use Case**: Yoga studios, holistic wellness resorts, and spiritual retreat organizers.
