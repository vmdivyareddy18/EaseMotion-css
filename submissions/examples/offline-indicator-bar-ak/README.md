# Offline Indicator Bar

## What does this do?
A fixed notification bar that appears at the top of the page when the user goes offline, with automatic dismissal on reconnection, dismiss button, and an event log.

## How is it used?
Open demo.html in a browser. Toggle airplane mode or use the "Simulate Offline" button to see the bar appear. Click "Dismiss" to hide it manually.

## Why is this useful?
Demonstrates an offline detection UI pattern with a non-intrusive notification bar and event logging, suitable for PWAs and connectivity-aware apps.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
