# Workload Balance Meter

A responsive capacity dashboard demo with animated workload meters, hover lift cards, and keyboard focus states.

## Files

- `demo.html` contains the workload meter markup.
- `style.css` contains the responsive layout, animated meter bars, and card interaction states.

## Usage

Open `demo.html` in a browser. Hover or focus each workload card to see the lift state while the meter animates.
