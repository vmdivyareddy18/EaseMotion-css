Search Bar with Close Icon

A modal-style search overlay — hidden by default, opens on button click or Ctrl+K / ⌘K, closes via close icon, Escape, or backdrop click.

Trigger

🔍
↑↓ Navigate Enter Select Esc Close

How it works

  • Hidden by default — .ease-search-overlay has opacity:0; visibility:hidden
  • Opens by adding .is-open — smooth fade + slide-down with bounce
  • Close icon (✕) removes .is-open
  • Backdrop click closes — click outside .ease-search-box
  • Escape key closes
  • Ctrl+K / ⌘K keyboard shortcut
  • Live filtering as you type
  • Keyboard navigation with ↑↓ arrows
