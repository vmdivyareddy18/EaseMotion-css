# Animated Pagination Component

A responsive animated pagination component built using pure HTML and CSS.

## Features

- Animated active page states
- Hover transition effects
- Previous and next navigation buttons
- Responsive layouts
- Smooth interaction animations
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated pagination component.

## Files

- `demo.html` → Pagination structure
- `style.css` → Pagination styles and animations
- `README.md` → Documentation
