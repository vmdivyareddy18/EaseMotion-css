# CSS-Only DVD Bouncing Logo

## Description
A classic nostalgic DVD screensaver logo implemented entirely with CSS keyframes. No JavaScript required!

## How it works
- Uses **two separate animation sequences**: `moveX` and `moveY`.
- The `alternate` keyword is key here—it makes the logo "bounce" when it hits the edge of the viewport.
- `linear` timing ensures a constant velocity for that authentic retro feel.

## Usage
Add the `.dvd-logo` element inside a relative container to see it bounce. You can adjust the duration (e.g., 5s, 3s) to change the "bouncing" speed.
