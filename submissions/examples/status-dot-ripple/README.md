# Status Dot Ripple

A live status badge with a pulsing dot that signals an active connection.

## Files

- `demo.html` - status badge markup.
- `style.css` - dot ripple animation and reduced-motion fallback.

## Highlights

- Uses `role="status"` for the live label.
- Keeps the ripple decorative through a pseudo-element.
- Stops the pulse for reduced-motion users.
