# Orbit Menu – Circular menu with orbiting items

> A production-ready orbit menu component that creates a circular menu with orbiting items, smooth animations, auto-rotation, and full customization. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Circular orbiting menu** – Items orbit around a central trigger
- ✅ **Auto-rotation** – Menu items rotate continuously
- ✅ **Direction control** – Clockwise or counter-clockwise
- ✅ **Customizable** – Radius, speed, color
- ✅ **Keyboard accessible** – Enter/Space (toggle), Escape (close)
- ✅ **8 menu items** – Fully customizable
- ✅ **4 color variants** – Ocean, Sunset, Gold, Dream
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
