# Ease Database Connection Pool Monitor

## Overview

A responsive HTML and CSS component for monitoring database connection pool health, utilization, waiting requests, and capacity limits.

## Features

- Active connection monitoring
- Idle pool visibility
- Waiting request tracking
- Pool utilization alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable monitoring component for database connection pools, helping teams identify bottlenecks, prevent connection exhaustion, and improve application reliability.
