# AI Ocean Coral Reef Regeneration Monitoring Showcase

## Overview
This submission is a premium HTML and CSS showcase for an AI-powered coral reef regeneration monitoring platform. It presents a futuristic but scientific marine operations dashboard focused on reef recovery, live telemetry, autonomous field systems, predictive analytics, and conservation impact.

## Features
- Large immersive hero with custom inline SVG coral reef artwork
- CSS-only ambient motion including bubbles, wave drift, shimmer, glow, and pulse effects
- Monitoring dashboard for ocean temperature, acidity, water quality, biodiversity, fish population, and coral cover
- AI prediction cards, live operations timeline, restoration progress, and environmental impact summaries
- Reef vitality heatmap, regeneration chart, coral species panels, and connected global observatory map
- Fully responsive layout designed for desktop, tablet, and mobile browsing
- Self-contained implementation with no JavaScript and no external assets

## Design Decisions
- Uses a dark oceanic palette with high-contrast text for readability in a dashboard context.
- Keeps all cards on dark surfaces to preserve contrast and match the scientific mission-control tone.
- Relies on original inline SVG artwork instead of external imagery so the page remains portable and direct-open.
- Uses CSS-only animation with a reduced-motion fallback to keep the experience polished and accessible.
- Favors dense information layout over empty decorative space so the page feels like real monitoring software.
