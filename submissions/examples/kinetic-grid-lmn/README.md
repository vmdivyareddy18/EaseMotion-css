1. What does this do?
A highly customizable, mathematical grid-based loading state that creates fluid kinetic wave motions utilizing dynamic CSS variables instead of JavaScript.

2. How is it used?
```html
<div class="kinetic-matrix">
  <div class="matrix-dot" style="--col: 1; --row: 1;"></div>
  <div class="matrix-dot" style="--col: 2; --row: 1;"></div>
  </div>
```
