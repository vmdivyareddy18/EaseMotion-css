# Counter Number Ticker

An animated number counter using CSS `counter-increment` and `@keyframes` for a ticker-style count-up effect.

## Features

- Numbers animate from 0 to target using CSS counters
- Gradient text styling on numbers
- 2x2 grid layout for multiple stats
- Fade-in animation on page load
- Dark theme card design

## Usage

Include `style.css` and structure counter items with `.counter-number` and `.counter-label`. The counter animation uses CSS `counter-increment` driven by `@keyframes` percentages.
