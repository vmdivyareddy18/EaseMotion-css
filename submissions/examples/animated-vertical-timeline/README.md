# Animated Vertical Timeline

A clean, modern, and fully responsive **Vertical Timeline** component. This example demonstrates how to use **EaseMotion CSS** to create a beautiful cascading entrance effect using alternating directional slides and bouncy dot animations.

## 🚀 Features

- **Alternating Layout**: Cards alternate between the left and right sides of a central vertical line.
- **Directional Animations**: Uses `ease-slide-in-left` and `ease-slide-in-right` for a dynamic, alternating entrance effect.
- **Bouncy Nodes**: The timeline dots use `ease-bounce-in` to add a playful, engaging feel as they appear.
- **Staggered Delays**: Combines `ease-delay-*` classes so the timeline builds itself sequentially down the page.
- **Fully Responsive**: On mobile devices, the layout gracefully collapses into a single-column, left-aligned timeline.
- **Hover Interactions**: Cards lift slightly on hover with an enhanced drop shadow.
