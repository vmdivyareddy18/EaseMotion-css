# Segmented Micro-Interaction Circular Loader

## Description
A premium dual-stage circular loading spinner featuring counter-rotating segmented tracks built on physics-based transition curves.

## Usage
Add the loader container to your markup:
```html
<div class="loader-container">
  <div class="circular-segment-loader"></div>
  <div class="circular-segment-loader-inner"></div>
</div>
```

## Why is it useful?
Delivers a highly fluid status verification indicator. Relies exclusively on CSS transition variables to keep runtime thread utilization low. Automatically overrides high-frequency transitions to safe modes in reduced motion environments.
