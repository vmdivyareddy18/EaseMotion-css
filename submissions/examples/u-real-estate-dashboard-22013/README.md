# Real Estate Dashboard

A responsive **Real Estate Dashboard** built using **HTML5** and **CSS3** that demonstrates property listing management, sales tracking, client inquiries, and revenue analytics through a modern interactive dashboard.

---

## Features

- Responsive dashboard layout
- Property overview cards
- Sales progress tracker
- Property availability panel
- Client inquiry management
- Recent sales activity feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Properties Listed
- Active Clients

- Revenue

### Real Estate Widgets

- Sales Progress
- Property Availability
- Client Inquiries

### Activity

- Recent Property Sales
- Notifications Panel

- Add Property

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
