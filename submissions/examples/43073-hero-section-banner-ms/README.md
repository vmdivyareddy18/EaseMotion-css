# Hero Section Banner

A modern, full-screen, high-impact hero section banner designed for landing pages and SaaS sites.

## Features
- **Immersive Background**: Features layered animated background elements, including a drifting radial gradient, soft floating ambient shapes, and an infinitely panning geometric grid.
- **Gradient Typography**: A beautiful linear-gradient text highlight using `background-clip: text`.
- **Entrance Animation**: The main content gracefully slides up and fades in on load.
- **Glassmorphic Buttons**: Premium secondary buttons utilizing translucent borders and backdrop filters, alongside high-contrast primary calls-to-action.

## Files
- `demo.html`: The semantic HTML5 markup for the header/hero section.
- `style.css`: The styling and CSS animations for the grid, floating shapes, and typography.

## Usage
Add the `<header class="ease-hero-banner">` section to the top of your landing page. The banner automatically occupies the full viewport height (`min-height: 100vh`) and centers the content. Adjust the variables or specific colors in the `ease-hero-bg-gradient` and `ease-hero-shape` blocks to match your brand's color palette.
