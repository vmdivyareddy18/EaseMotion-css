# Animation Play State Demo

## Description

This example demonstrates how the CSS `animation-play-state` property can be used to pause an animation without JavaScript.

## Features

- Infinite bouncing animation
- Pause animation on hover
- Simple and reusable CSS

## Files

- demo.html
- style.css

## Usage

Hover over the green box to pause the animation.

## CSS Used

```css
.demo-box:hover{
    animation-play-state: paused;
}
```

## Benefits

- Improves user interaction
- Useful for animated cards, buttons, and banners
- No JavaScript required
