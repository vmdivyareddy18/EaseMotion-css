# CSS Zoom-Out Accordion - Minimalist Tech Layouts

## Overview

A pure CSS animated accordion featuring a smooth Zoom-Out interaction designed for minimalist technology interfaces.

The component creates elegant content reveal animations without JavaScript.

## Features

- Pure HTML and CSS
- Zoom-out expand animation
- Minimal tech inspired design
- Responsive layout
- Keyboard accessible using native details/summary
- Custom CSS animation variables
- No JavaScript dependency

## Usage

```html
<div class="accordion">

  <details>
    <summary>Accordion Title</summary>

    <div class="content">
      <p>Your content goes here.</p>
    </div>

  </details>

</div>
