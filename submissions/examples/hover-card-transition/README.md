# Hover Card Transition

## Overview

Demonstrates smooth hover transitions on card examples using EaseMotion CSS `ease-card-hover` class.

## Features

- **Smooth Transitions**: Cards have fluid hover animations
- **Multiple Effects**: Lift, shadow, and border-color transitions
- **Accessibility**: Respects `prefers-reduced-motion`

## Usage

Add `ease-card-hover` class to any card:

```html
<article class="ease-card ease-card-shadow ease-card-hover">
  <div class="card-content">
    <h3>Card Title</h3>
    <p>Card content here.</p>
  </div>
</article>
```

## CSS Properties

The `ease-card-hover` class provides transitions for:

- `transform` - Smooth lift effect on hover
- `box-shadow` - Enhanced shadow on hover  
- `border-color` - Primary color border highlight
