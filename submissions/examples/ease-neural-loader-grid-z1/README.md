# Ease Neural Loader Grid

## Overview

A compact loading animation inspired by neural network activity using a 3×3 animated grid.

## Features

- Pure HTML & CSS
- Animated grid
- Responsive
- Lightweight
- Easy to customize

## Folder Structure

```
ease-neural-loader-grid-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox and Safari.

## Why This Example

This example demonstrates a simple grid-based loading animation that can be reused in dashboards, AI interfaces, and admin panels.
