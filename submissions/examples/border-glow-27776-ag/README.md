# Animated Border Draw & Glow Keyframe Mixins

A card component demonstrating rotating conic-gradient glow border effects on hover.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting relative card wrappers, gradient border layers, and textual details.
2. **`style.css`**: Conic gradients, absolute center offsets, rotation keyframes, overflow wraps, and media settings.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Hover over the **Interactive Panel** card.
3. Verify that a beautiful neon purple-to-cyan gradient border rotates smoothly around the card edges.
