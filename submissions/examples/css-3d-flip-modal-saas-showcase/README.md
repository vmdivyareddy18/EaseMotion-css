# CSS 3D Flip Modal – SaaS Showcase Layout

## Overview

A responsive SaaS Showcase modal built entirely with HTML and CSS. The component features a smooth 3D flip opening animation, a modern SaaS landing experience, responsive feature cards, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip animation
- SaaS showcase interface
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--bg`
- `--surface`
- `--text`
- `--muted`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Explore Features** to display the animated 3D Flip Modal showcasing SaaS platform highlights. Colors, animation timing, spacing, and flip effects can be customized using the CSS variables defined in `style.css`.
