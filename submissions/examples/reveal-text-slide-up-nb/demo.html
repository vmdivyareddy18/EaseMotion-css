How It Works

Animation Breakdown

Four simple CSS concepts work together to create a polished, progressive content reveal.

01

Overflow Hidden

The parent wrapper clips its contents using overflow: hidden, ensuring the translated text is invisible until it moves into the visible area.

.reveal-wrapper { overflow: hidden; }
02

translateY(100%)

The child element starts positioned one full element-height below its normal position, placing it entirely outside the parent's visible area.

.ease-reveal-text { transform: translateY(100%); }
03

Parent Hover Trigger

Hovering the parent wrapper triggers the child's state change via the descendant selector, moving it to its natural position.

.reveal-wrapper:hover .ease-reveal-text { transform: translateY(0); }
04

Transition Timing

A 0.4s ease-out transition smooths the motion, starting quickly and decelerating naturally for a polished feel.

transition: transform 0.4s ease-out;
Property Value Description
overflow hidden Clips child content outside the parent bounds
transform (initial) translateY(100%) Positions text one full height below
transform (hover) translateY(0) Moves text to its natural position
transition 0.4s ease-out Smooth deceleration curve
Trigger :hover Parent hover activates child animation
