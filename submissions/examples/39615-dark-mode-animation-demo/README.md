# Dark Mode Animation Demo

1. What does this do? Demonstrates EaseMotion CSS animations (such as fade-in, slide-up, zoom-in, bounce, and hover effects) in a responsive dark-mode dashboard interface.
2. How is it used? Open `demo.html` in a browser and check how EaseMotion delay and hover classes are combined on `.demo-card` and other layout elements.
3. Why is it useful? It showcases how to build fluid, high-performance, dark-first interfaces with staggered animations without writing custom JavaScript.

## Features

- Dark-first UI with a CSS-only light theme preview
- Demonstrates `ease-fade-in`, `ease-slide-up`, `ease-zoom-in`, `ease-bounce`, and hover animation utilities
- Staggered entrance timing with EaseMotion delay classes
- Responsive card layout for desktop and mobile
- Reduced-motion fallback for accessibility

## Files

- `demo.html` - Demo markup and EaseMotion utility classes
- `style.css` - Dark mode theme styles, toggle styling, and responsive layout
