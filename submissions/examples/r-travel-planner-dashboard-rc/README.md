# Travel Planner Dashboard Showcase

## What does this do?

Provides an animated travel-planning dashboard with trip management, itinerary tracking, booking monitoring, and travel-budget insights.

## How is it used?

Open `demo.html` directly in a browser and explore the travel planner interface.

## Why is it useful?

It demonstrates practical travel-platform UI patterns and combines multiple animation techniques into a realistic trip-planning and booking-management experience.
