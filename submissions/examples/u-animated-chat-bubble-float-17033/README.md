# Animated Chat Bubble Float

## Overview

A reusable chat bubble animation featuring a gentle floating effect suitable for messaging platforms, support widgets, communication tools, and collaboration interfaces.

## Features

- CSS-only animation
- Floating chat bubble effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="chat-button">
  <span class="chat-icon">💬</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
