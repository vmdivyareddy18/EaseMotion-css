# CSS Gradient Text Component

## What does this do?
A collection of gradient text effects using background-clip for typography with direction variants, color themes, animations, and decorative styles.

## How is it used?
```html
<span class="grt-sample grt-left">Gradient Text</span>
<span class="grt-sample grt-animated-rainbow">Rainbow</span>
```

## Why is it useful?
Creates eye-catching gradient typography without images. Multiple directions, themes, and animations including rainbow cycling. Pure CSS, accessible, and responsive.
