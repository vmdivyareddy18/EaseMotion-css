# Info Banner Component

## What does this do?
CSS announcement and notification banners with severity color coding, layout variants, dismiss buttons, and action buttons.

## How is it used?
```html
<div class="ib-banner ib-info"><span class="ib-ico">i</span><div class="ib-txt"><strong>Title</strong><p>Message</p></div><button class="ib-cls">x</button></div>
```

## Why is it useful?
Provides clear user feedback without JavaScript. Four severity levels, three layout modes, action button support. Accessible and customizable.
