# Grid Card Layout

A grid of cards with staggered pop-in entrance animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--card-bg` | `#2d2d44` | Card background color |
| `--card-border` | `#4a4a6a` | Card border color |
| `--pop-scale` | `1.08` | Hover scale factor |
| `--stagger-delay` | `0.08s` | Delay between each card entrance |
