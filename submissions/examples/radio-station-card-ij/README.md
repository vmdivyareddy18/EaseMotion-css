# Radio Station Card

1. What does this do? A radio station selector card where the active station gets an animated border glow and a pulsing now-playing indicator dot.

2. How is it used? Add `.station-card` elements with a `.now-playing` dot inside. Toggle `.active` class to trigger the border glow animation and the pulsing green dot.

3. Why is it useful? It creates a polished radio/streaming app UI — stations are clearly distinguished by their active state with an engaging glow animation and a subtle live-indicator pulse.
