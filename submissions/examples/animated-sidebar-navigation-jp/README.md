# Animated Sidebar Navigation Component

## Description

A CSS-only animated sidebar navigation component with smooth slide-in effects, interactive menu items, and responsive layouts.

---

## Features

- Sidebar slide-in animation
- Active menu item highlighting
- Smooth hover transitions
- Responsive navigation layout
- Modern dashboard-style design
- CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.sidebar` – Main sidebar container
- `.logo` – Sidebar heading/logo
- `.menu` – Navigation wrapper
- `.menu-item` – Individual navigation link
- `.active` – Current active menu item
- `.content` – Main page content area

---

## Accessibility

Supports `prefers-reduced-motion` to disable animations for users who prefer reduced motion.

---

## Use Cases

- Admin dashboards
- Analytics panels
- Portfolio websites
- Web applications
- User account panels
