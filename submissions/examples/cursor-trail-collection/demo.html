Cursor Trail Effect Collection

Five lightweight cursor-reactive effects. Three are 100% CSS (cursor-trail, cursor-pulse, cursor-magnetic). Two (cursor-glow, cursor-spotlight) use a 4-line mousemove listener only to feed live coordinates into CSS custom properties — all rendering is still done by CSS gradients and transitions.

ease-cursor-magnetic

Button lifts and scales toward the cursor on hover/focus, pure CSS.

True per-pixel magnetic pull toward the exact pointer position requires JavaScript to read pointer offset; this pure-CSS version approximates the feel with a directional lift + scale on hover/focus.
