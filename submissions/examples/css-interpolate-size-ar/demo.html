Interpolate‑size Demo

Smoothly expand/collapse sections with height: auto using interpolate-size.

interpolate-size is a CSS property that allows you to transition to and from intrinsic sizing keywords like auto, min-content, and max-content — previously impossible with pure CSS.

This demo expands and collapses using height: auto with a smooth transition. Check the browser support note below.

Requires a browser with interpolate-size support (Chrome 117+). Falls back to instant snap otherwise.
