Vue.js Integration Guide

Learn how to use EaseMotion CSS in a Vue 3 or Nuxt project — from static classes to dynamic animations and scroll-triggered effects.

1. Installation

CDN (quickest)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/SAPTARSHI-coder/EaseMotion-css@main/easemotion.min.css">

npm (Vue CLI / Vite)

npm install easemotion-css
/* main.js */
import 'easemotion-css/easemotion.min.css'

2. Static Class Usage

<template>
  <h1 class="ease-fade-in">Welcome to Vue!</h1>
</template>

Just add the class directly in your template — no extra setup needed.

3. Dynamic Binding

<script setup>
import { ref } from 'vue';
const isVisible = ref(false);
</script>

<template>
  <div :class="{ 'ease-fade-in': isVisible }">Dynamic animation</div>
</template>

Toggle EaseMotion classes reactively with Vue's data binding.

4. With Vue <Transition>

<template>
  <Transition name="fade">
    <div v-if="show" class="ease-card">Content</div>
  </Transition>
</template>

<style scoped>
.fade-enter-active { transition: opacity 0.5s ease; }
.fade-leave-active { transition: opacity 0.3s ease; }
.fade-enter-from,
.fade-leave-to { opacity: 0; }
</style>

Combine Vue's transition system with your own custom CSS (or extend with EaseMotion keyframes).

5. Nuxt.js Setup

// nuxt.config.ts
export default defineNuxtConfig({
  css: ['easemotion-css/easemotion.min.css']
})

Import the stylesheet in the CSS array — available globally.

6. Scroll‑Triggered Animations

// composables/useScrollAnimation.js
import { ref, onMounted } from 'vue';

export function useScrollAnimation() {
  const isVisible = ref(false);
  onMounted(() => {
    const observer = new IntersectionObserver(([entry]) => {
      if (entry.isIntersecting) isVisible.value = true;
    }, { threshold: 0.1 });
    observer.observe(document.querySelector('.scroll-target'));
  });
  return { isVisible };
}
<template>
  <div :class="{ 'ease-slide-up': isVisible }" class="scroll-target">Scroll‑triggered</div>
</template>

Use an IntersectionObserver inside a Vue composable to add classes when elements enter the viewport.

All examples can be copied directly into your Vue project. Make sure EaseMotion CSS is loaded before your custom styles.
