# ease-menu-accordion-sidebar (#36812)

Menu accordion sidebar with nested expand/collapse submenus.

## CSS Custom Properties

| Variable            | Default    | Description                    |
|---------------------|------------|--------------------------------|
| `--sidebar-bg`      | `#1a1a2e`  | Sidebar card background        |
| `--menu-item-bg`    | `#262640`  | Menu item hover background     |
| `--expand-duration` | `0.3s`     | Expand / collapse animation    |
| `--active-color`    | `#7c5cfc`  | Active menu item background    |

## Features

- Two-level accordion with nested submenus
- Chevron rotation on expand
- Smooth `grid-template-rows` expand animation
- Active state highlighting
- Dark theme card layout
