# Typing Speed Test

## What does this do?
A typing speed test that measures words per minute (WPM) and accuracy over a 30-second timed session with real-time character highlighting.

## How is it used?
Open demo.html in a browser. Click "Start Test" and begin typing the displayed text. Each character is highlighted green (correct) or red (incorrect). When time runs out or you finish the text, your WPM and accuracy are displayed.

## Why is this useful?
This provides a typing practice and assessment tool. It demonstrates timed test logic, real-time character comparison, and performance metrics calculation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to test your typing speed.
