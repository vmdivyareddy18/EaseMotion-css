# Bulk Selection Action Bar

**What does this do?** Reveals a contextual action bar while showing selected, unselected, active, and away list states.

**How is it used?** Apply `action-bar` to the contextual toolbar and `selected` to selected rows inside a `directory` container.

**Why is it useful?** It gives multi-item workflows clear visual feedback with restrained motion, strong focus states, responsive controls, and a reduced-motion fallback.
