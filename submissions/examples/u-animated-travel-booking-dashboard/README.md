# Animated Travel Booking Dashboard

A modern travel booking dashboard built entirely with HTML and CSS.

## Features

- Trip analytics
- Booking statistics
- Destination tracking
- Revenue overview
- Booking status section
- Progress indicators
- Responsive layout
- CSS Variables
- Pure CSS animations

## Folder Structure

u-animated-travel-booking-dashboard/
├── demo.html
├── style.css
└── README.md

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
- CSS Variables
- Media Queries

## Included Animations

- ease-fadeUp
- ease-progress
- ease-slideIn
- ease-pulse
