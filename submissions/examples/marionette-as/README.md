# Marionette

A pure CSS and HTML marionette dancing on strings, its jointed limbs swinging from a rocking control bar.

## What it shows

- A fully jointed puppet: arms and legs split into hinged segments, each swinging on its own keyframe so elbows and knees bend as it dances
- Five taut strings running from a control bar down to the head, hands and feet, the bar and strings rocking together
- A ruffled clown collar from a masked repeating conic gradient
- A face with rosy cheeks, a smile drawn with border-bottom and a peaked hat
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
