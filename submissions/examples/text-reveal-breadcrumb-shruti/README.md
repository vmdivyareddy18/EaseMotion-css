# Text Reveal Breadcrumb

A responsive breadcrumb component with a text reveal animation inspired by modern music player interfaces.

## Features

- Pure HTML & CSS
- Responsive layout
- Accessible navigation
- Smooth text reveal effect
- No JavaScript required

## Files

```text
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in a browser and hover over the breadcrumb links to see the reveal animation.
