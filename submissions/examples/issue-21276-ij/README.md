# Domino Fall — Domino Chain Reaction

CSS domino chain reaction using staggered `animation-delay` on each tile. Each domino rotates from its bottom edge to simulate toppling.

## Files

- `demo.html` — row of ten domino tiles
- `style.css` — staggered fall keyframes, gradient tile styling

## Usage

Open `demo.html` to watch the dominoes fall in sequence.
