# 🗡️ Knife Throw

A classic Knife Throw arcade game built with HTML, CSS, and JavaScript. Throw knives at a spinning wheel, hit target areas for points, and avoid hitting previously thrown knives!

## 🎮 How to Play

1. Click **"Start Game"** to begin
2. **Click** the canvas or press **Space** to throw
3. **Hit target areas** on the spinning wheel for points
4. **Avoid hitting** previously thrown knives
5. **Score points** based on target difficulty (10-30 points)
6. **Complete rounds** with increasing difficulty!

## 🖱️ Controls

| Action | Input |
|--------|-------|
| Throw Knife | Click canvas or press `Space` |
| Start Game | Click Start or press `Space` |
| Reset Game | Press `R` |
| Change Difficulty | Press `D` |

## ✨ Features

- 🎯 **Target Areas** – 8 segments with varying point values (10-30)
- ⚔️ **Collision Detection** – Avoid hitting previously thrown knives
- 📈 **Progressive Difficulty** – Speed increases with difficulty level
- 🎨 **Visual Feedback** – Particle effects on hits
- 🏆 **Best Score** – Saved in browser local storage
- 📱 **Responsive Design** – Works on desktop and mobile
- 🖱️ **Touch Support** – Play on mobile devices

## 🎯 Scoring

| Target | Points |
|--------|--------|
| 10-point areas | 10 |
| 20-point areas | 20 |
| 30-point areas | 30 |

## 📊 Difficulty Levels

| Level | Speed | Starting Knives |
|-------|-------|-----------------|
