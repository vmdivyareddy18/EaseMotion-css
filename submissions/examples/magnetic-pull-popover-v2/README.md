# CSS Magnetic Pull Popover

A lightweight **pure HTML & CSS** popover with a smooth magnetic pull hover effect designed for minimalist tech interfaces.

## Features

- Pure HTML & CSS
- No JavaScript
- Smooth magnetic pull animation
- Responsive design
- Keyboard accessible
- Easy customization using CSS variables

## Folder Structure

```
minimalist-tech-magnetic-popover/
│── demo.html
│── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser.

## Customization

Edit the CSS variables inside `style.css`.

```css
:root{
    --accent:#38bdf8;
    --duration:.35s;
    --scale:1.08;
}
```
