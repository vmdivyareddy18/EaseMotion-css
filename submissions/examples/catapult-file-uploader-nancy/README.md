# Catapult File Uploader Component

A pure CSS file uploader design implementation optimized for medical dashboard user environments. Built explicitly to showcase fluid catapult transition properties within EaseMotion CSS modules.

## Architectural Notes
- **Medical UX Integration:** Clean lines, medical-themed layouts, and clinical visual states.
- **No Script Engines:** Managed purely via sibling state configurations inside standard markup tags.
- **Catapult Feedback Matrix:** Transitions the successful processing state using an upward launching spring action (`ease-catapult-upload`).
