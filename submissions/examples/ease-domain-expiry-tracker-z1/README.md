# Ease Domain Expiry Tracker

## Overview
A CSS-only component for monitoring domain expiration dates.

## Features
- Expiry status badges
- Responsive layout
- Hover animation
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for hosting dashboards, admin panels, and domain management systems.
