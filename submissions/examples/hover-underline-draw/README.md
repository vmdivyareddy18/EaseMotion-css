# Hover Underline Draw

## What it does

Creates an underline that animates into view when hovering over links or text.

## How to use it

<a href="#" class="ease-underline-draw">Link</a>

## Variants

- ease-underline-center - Draws from center outward
- ease-underline-reverse - Draws from right to left
- ease-underline-thick - 3px underline (default 2px)
- ease-underline-purple/pink/cyan - Colored underline
- ease-underline-fast/slow - Speed variants

## Why it fits EaseMotion CSS

- Pure CSS, zero JavaScript
- Smooth width transitions
- Multiple directions and colors
- Accessible, respects reduced motion
