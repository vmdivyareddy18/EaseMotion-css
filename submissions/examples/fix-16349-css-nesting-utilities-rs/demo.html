Native CSS Nesting Patterns

Write component-scoped CSS safely and cleanly using the native `&` nesting syntax. No SASS compilation required.

Developer Experience

Native CSS nesting drastically improves code readability. You can encapsulate all logic related to a component within a single block, including hover states and media queries.

Performant Rendering

Because it is parsed natively by the browser engine, there's no build step required. The styling cascade correctly targets children when the parent is hovered or focused.
