# Shimmer Pulse Audio Player Animation

A responsive Audio Player animation featuring a smooth Shimmer Pulse progress bar built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive layout
- Shimmer Pulse animation
- Audio Player inspired interface
- CSS custom properties
- `prefers-reduced-motion` support
- Keyboard accessible controls
- No JavaScript required

--duration
--accent
--shimmer
--background
--surface
