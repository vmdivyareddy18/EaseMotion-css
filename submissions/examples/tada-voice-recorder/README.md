# Tada Voice Recorder

A responsive **Tada Voice Recorder** component inspired by minimalist design. The microphone icon performs a CSS-only **tada animation**, creating an engaging recording interface without JavaScript.

## Features

- Pure HTML & CSS
- CSS Tada animation
- Responsive layout
- Accessible structure
- No JavaScript required

## Folder Structure

```
tada-voice-recorder-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
