# Neumorphic Badge

A responsive **Neumorphic Badge** component built using pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Cyberpunk-inspired neumorphic design
- Smooth hover interaction
- Animated status indicator
- Responsive layout
- Uses EaseMotion CSS variables and `ease-*` animation classes

## Folder Structure

```text
neumorphic-badge-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Technologies

- HTML5
- CSS3

## Animation

- `ease-fade-in`
- `easePulse` status animation
- Smooth hover transition
