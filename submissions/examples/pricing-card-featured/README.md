# Pricing Card with Featured Tier Highlight

A pricing card component with three tiers: Starter, Professional (featured), and Enterprise. The featured tier has an accent border, shadow glow, and a "Popular" badge. Each card lists features with checkmark bullets and a call-to-action button.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the three pricing tiers side by side.

## Accessibility notes

Buttons are `<button>` elements for proper keyboard interaction. The "Popular" badge uses a styled span for visual emphasis only.
