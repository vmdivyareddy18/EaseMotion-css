# 🍅 Pomodoro Timer - AK

A beautiful, functional Pomodoro timer to boost your productivity with the proven Pomodoro Technique.

## 🚀 Features
- ⏱️ Three modes: Focus (25 min), Short Break (5 min), Long Break (15 min)
- 🎯 Circular animated progress ring
- ▶️ Start / Pause / Reset controls
- 🔔 Audio notification when session ends
- 📊 Tracks sessions & total focus minutes
- 📱 Fully responsive design
- 🎨 Modern gradient UI

## 📂 Files
- `demo.html` — Main HTML + JavaScript
- `style.css` — Styling & animations
- `readme.md` — Documentation

## 🛠️ How to Use
1. Open `demo.html` in any browser
2. Select a mode (Focus / Short Break / Long Break)
3. Click **Start** and stay focused!
4. Take breaks when the timer ends

## 🧠 About Pomodoro Technique
Work for 25 minutes → 5 min break → repeat. After 4 sessions, take a 15 min long break.
