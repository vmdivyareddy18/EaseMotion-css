# Animated SaaS Metrics Dashboard

A modern SaaS analytics dashboard designed with pure HTML and CSS.

## Features

- Subscriber metrics
- Monthly recurring revenue
- Growth analytics
- Engagement tracking
- Activity feed
- Animated progress bars
- Hover interactions
- Responsive layout
- CSS Variables
