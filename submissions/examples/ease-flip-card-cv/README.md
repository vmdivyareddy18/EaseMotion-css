# Animated Flip Card (`ease-flip-card-cv`)

A responsive flip card component built entirely with **HTML5** and **Pure CSS**.

## Features

- Horizontal flip animation
- Vertical flip animation
- Hover-triggered interaction
- Multiple color themes
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-flip-card-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Compatible with all modern browsers supporting CSS 3D transforms.
