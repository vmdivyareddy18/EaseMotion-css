# Getting Started Quickstart Guide

This submission provides a step-by-step quickstart guide for EaseMotion CSS (Issue **#14192**).

## What It Covers

- CDN setup (jsDelivr, unpkg, GitHub Raw)
- npm installation
- Your first animation
- Staggered sequences with delay utilities
- Dark mode setup and theme toggle
- Duration helpers (fast, medium, slow)
- Hover effects (grow, shimmer, card lift)
- Scroll-triggered reveal
- CSS variable customization
- Troubleshooting FAQ

## Files

- `demo.html` — Interactive guided tutorial with copyable code blocks for each step
- `style.css` — Modern dark-themed styling for the tutorial page
- `README.md` — This documentation
