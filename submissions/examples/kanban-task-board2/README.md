# Animation: Kanban Task Board

A modern Kanban board inspired by Trello, Jira, GitHub Projects, and Linear. Tasks are organized into workflow stages using responsive columns and interactive task cards.

## Features

- Pure HTML & CSS
- Multi-column Kanban layout
- Priority badges
- Hover lift effects
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="task-card">

    <span class="badge high">
        High
    </span>

    <h3>Create Landing Page</h3>

</div>
```

## Why is it useful?

Kanban boards are widely used in project management, issue tracking, product development, and team collaboration tools. This component demonstrates how structured task workflows can be presented in a clean and interactive interface.
