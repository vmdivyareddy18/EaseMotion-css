# Approval Request Decision Bar

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern approval workflow component that displays pending requests, approval actions, request metadata, and priority indicators.

## Features

- Approval notification bar
- Approve and reject actions
- Priority badge
- Deadline indicator
- Request summary panel
- Animated attention pulse
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="decision-bar">
  Approval Request
</div>
```

## Main Classes

```css
.decision-bar
.request-info
.priority-badge
.actions
.approve-btn
.reject-btn
```
