# Morph Floating Action Button

A Banking-inspired Morph Floating Action Button built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Morph expansion animation
- Rotating action icon
- Responsive layout
- Accessible HTML
- No JavaScript required
