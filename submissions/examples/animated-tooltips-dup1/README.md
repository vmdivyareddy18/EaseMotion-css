# Animated Tooltip System

A clean, modern, and fully responsive **Tooltip System** with smooth directional animations and multiple style variations. This example demonstrates how to combine **EaseMotion CSS** for smooth fade-in/scale animations with custom CSS for positioning, arrow creation, and color variations to create an engaging contextual help system.

## 🚀 Features

- **Directional Animations**: Tooltips animate in from their respective positions using `ease-fade-in-up` (top), `ease-fade-in-down` (bottom), `ease-fade-in-left` (right), and `ease-fade-in-right` (left).
- **Four Position Options**: Tooltips can appear on top, bottom, left, or right of the trigger element.
- **CSS Arrows**: Each tooltip has a directional arrow created with pure CSS borders.
- **Color Variations**: Support for default (dark), success (green), warning (amber), and error (red) tooltip styles.
- **Dynamic Positioning**: JavaScript calculates and positions tooltips dynamically based on trigger element location.
- **Icon Support**: Works with both text buttons and icon-only triggers.
- **Real-World Example**: Includes a form validation example showing tooltips in context.
- **Fully Responsive**: Adapts gracefully to mobile screens.
