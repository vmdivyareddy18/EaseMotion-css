CSS Rainbow Animated Borders

Observe how conic-gradients and continuous keyframe rotations create smoothly cycling rainbow borders on interactive cards and buttons.

Premium Feature

Glow Dashboard

This card uses a rotating conic-gradient pseudo-element masked by a dark background container to form a sharp, continuous rainbow border.

Filter Shift

Aurora Wave

This variant uses a static gradient combined with a keyframe animation that cycles the hue-rotate() filter for a pulsing spectrum transition.

Implementation Strategy

The border is built by placing a rotating pseudo-element containing a conic gradient behind a content box that is offset by 1.5px to cover the center:

.rainbow-card::before {
  content: '';
  position: absolute;
  inset: -2px; /* Expand slightly outside */
  background: conic-gradient(red, yellow, green, blue, violet, red);
  border-radius: inherit;
  animation: rotateBorder 4s linear infinite;
  z-index: -1;
}

@keyframes rotateBorder {
  to { transform: rotate(360deg); }
}
