# Elastic List

## What it does

List items that stretch horizontally on hover with an elastic bounce effect.

## How to use it

<ul class="ease-elastic-list">
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

## Variants

- ease-elastic-fast - 0.15s transition
- ease-elastic-slow - 0.5s transition
- ease-elastic-colored - Different colors per item
- ease-elastic-glow - Neon glow on hover
- ease-elastic-rounded - Pill-shaped corners
- ease-elastic-border - Bordered items

## Why it fits EaseMotion CSS

- Pure CSS, zero JavaScript
- Elastic cubic-bezier easing
- Multiple style variants
- Smooth hover transitions
