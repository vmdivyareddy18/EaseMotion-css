# Navigation Checklist Progress

A step-by-step checklist with progress transitions.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--checklist-bg` | `#2d2d44` | Checklist background |
| `--item-color` | `#888` | Inactive item color |
| `--active-color` | `#4ecdc4` | Active/completed color |
| `--progress-duration` | `0.25s` | Transition speed |
