1. What does this do?
A sleek segmented navigation bar utilizing concealed radio inputs to power a fluid, hardware-accelerated background highlight glider that follows active tab states.

2. How is it used?
```html
<nav class="pill-nav-zkf">
    <input type="radio" id="tab1" name="group" class="nav-radio-zkf" checked>
    <label for="tab1" class="nav-item-zkf">Tab 1</label>
    <div class="nav-glider-zkf"></div>
</nav>
```
3. Why is it useful?
It creates sophisticated sliding micro-interactions entirely with native structural CSS layout selectors, bypassing the performance overhead and execution lag introduced by client-side JavaScript calculations.
