# CSS Micro-Shake Accordion (E-Commerce Checkout Layouts)

A pure CSS responsive accordion designed for premium e-commerce checkout check-flows. Features a tactile "micro-shake" horizontal wobble animation on step headers to highlight interactivity.

## Features
- **Tactile Micro-Shake Interaction**: Subtle horizontal vibration triggered on hover/focus to suggest clickability.
- **Premium Boutique Aesthetics**: Designed with a Warm Ivory and Dark Espresso / Charcoal color scheme matching premium coffee, apparel, or boutique shops.
- **Pure CSS checked Logic**: Built using native radio inputs for zero-JS toggle support.
- **Accessible Step Indicators**: Standard WCAG compliant focus outlines and step statuses.

## Files
- `demo.html`: Live interactive demonstration of the e-commerce checkout flow.
- `style.css`: Stylesheet containing the custom checkout cards, step layout, and keyframe shaking animation.
