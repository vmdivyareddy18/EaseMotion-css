# CSS Tooltip System

## What does this do?
Pure CSS tooltip system with multiple positions, color themes, animation styles, and various trigger element types.

## How is it used?
```html
<span class="tp-trigger tp-top" data-tip="Tooltip text">Hover me</span>
```

## Why is it useful?
Provides lightweight tooltip functionality without JavaScript. Four positions, five color themes, four animation styles. Customizable via data-tip attribute and CSS variables.
