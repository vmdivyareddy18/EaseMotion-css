transition-behavior: allow-discrete

Smoothly transition discrete properties like display and overlay. Chrome 117+

⚡ Requires Chrome 117+. The default normal behavior instantly toggles discrete properties — no smooth transition.

normal (default)

display toggles instantly — no transition.

I appear instantly
transition-behavior: normal

allow-discrete

display fades in/out smoothly.

I fade in smoothly
transition-behavior: allow-discrete

How It Works

Without allow-discrete, the browser cannot interpolate values for discrete properties (like display: none → block). It snaps instantly.

With allow-discrete, combined with @starting-style, the browser can interpolate opacity and transform while toggling display — giving a smooth entry/exit.

Implementation

Proposed .ease-allow-discrete utility
.ease-allow-discrete {
  transition-behavior: allow-discrete;
}
Component usage (modal panel)
.ease-modal-panel {
  transition:
    opacity 0.3s,
    transform 0.3s,
    display 0.3s allow-discrete;
}

/* Entry state */
@starting-style {
  .ease-modal-panel.active {
    opacity: 0;
    transform: scale(0.95);
  }
}
