# Animated University Management Dashboard

An interactive layout environment built specifically for handling student numbers, curriculum changes, scheduling logs, and live administration monitoring panels.

## Features
- **Dynamic Counters:** Tracks metrics for Students, Faculty, and Curriculum Courses.
- **Micro-Animations:** Includes GPU-accelerated transition transforms (`@keyframes slideUp`, custom tracking progress metrics).
- **Administration Actions:** Dedicated event buttons for adding students, scheduling system examinations, and generating analytics report parameters.
