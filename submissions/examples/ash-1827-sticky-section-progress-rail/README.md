# Sticky Section Progress Rail

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A sticky vertical progress navigation component that visually tracks progress through multiple sections of content.

## Features

- Sticky progress rail
- Active section indicators
- Current step highlighting
- Gradient progress fill
- Milestone markers
- Reading progress visualization
- Responsive layout
- Glassmorphism content cards
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="progress-rail">

  <div class="rail-line">

    <div class="rail-fill"></div>

  </div>

</div>
```

## Main Classes

```css
.progress-rail
.rail-line
.rail-fill
.marker
.active
.current
.content-card
```
