# Glow Drag Drop Zone for Cyberpunk

### What does this do?
Adds a pure-CSS, neon-infused drag-and-drop file upload container themed around classic Cyberpunk UI design elements that pulses beautifully when active.

### How is it used?
```html
<div class="drop-zone-container-sap">
  <input type="file" class="drop-zone-input-sap" aria-label="Upload files here" />
  <div class="drop-zone-visual-sap">
    <div class="cyber-icon-sap">⤓</div>
    <p class="cyber-text-primary-sap">READY FOR PAYLOAD</p>
    <p class="cyber-text-secondary-sap">DRAG & DROP OR CLICK</p>
  </div>
</div>
```

### Why is it useful?
It updates a standard boring HTML form layout asset into a striking, high-performance visual interaction experience without needing a single line of JavaScript overhead.
