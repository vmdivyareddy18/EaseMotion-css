# Data Retention Policy Selector

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive data retention policy selector for choosing organizational data retention periods. Suitable for admin dashboards, privacy settings, compliance portals, and enterprise management systems.

## Features

- CSS-only policy selection
- Interactive radio cards
- Policy badges
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<label class="policy">

    <input type="radio" name="retention">

    <span class="indicator"></span>

    <div class="content">
        ...
    </div>

</label>
```
