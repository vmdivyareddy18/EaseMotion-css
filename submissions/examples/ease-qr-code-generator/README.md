# QR Code Generator – Pulsing scan effect

> A production-ready QR code generator component with a pulsing scan animation, customizable colors, size, and logo, plus download and share functionality. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **QR Code Generation** – Generate QR codes from text/URL
- ✅ **Pulsing Scan Effect** – Animated scan line with pulse
- ✅ **Customizable** – Size, color, background, logo toggle
- ✅ **Download** – Save QR code as PNG
- ✅ **Share** – Share QR code or copy text
- ✅ **Presets** – Website, Text, Email, WiFi
- ✅ **Keyboard shortcuts** – Ctrl+Enter (generate), Ctrl+S (download), Space (toggle scan)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
