# ease-number-typing-animate

## What this is
A CSS-only number typing animation with a cursor-style blink effect.

## Why
It makes numeric values feel more dynamic and engaging while keeping the implementation lightweight and easy to reuse.

## How to test
1. Open `demo.html` in a browser.
2. Confirm the number animates in with a typing-style reveal.
3. Confirm the cursor blinks during the animation.
4. Resize the browser to check responsive behavior.
5. Update the CSS variables and verify the styling still works.

## Note
This is a visual demo component and uses only HTML and CSS, so no JavaScript is needed.
