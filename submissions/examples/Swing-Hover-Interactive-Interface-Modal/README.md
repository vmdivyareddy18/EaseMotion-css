# Swing Hover Interactive Interface Modal

A modern interactive modal built entirely with HTML and CSS featuring a smooth Swing-Hover animation and responsive interface.

## Features

- Pure CSS
- Responsive layout
- Keyboard accessible
- Interactive cards
- Swing-Hover animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--rotate
--radius
--primary
--secondary
```
