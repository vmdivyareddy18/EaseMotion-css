# Footer Light Mode Fix Showcase

A footer component layout demonstrating text contrast fixes and flexbox alignment to prevent overlapping.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting central content cards, theme switches, and bottom relative footer bands.
2. **`style.css`**: Light and dark mode overrides, relative margin offsets, link colors, and contrast improvements.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Confirm the page displays in **Light Mode** by default.
3. Verify that the footer text and links have excellent contrast (dark gray/slate) and do not overlap with the main content.
4. Click **Toggle Dark/Light Mode** to transition to Dark Mode.
5. Verify the footer style adapts correctly and maintains legibility.
