# Micro Shake Neumorphic Soft Modal

A responsive neumorphic modal featuring a subtle Micro Shake interaction built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Soft neumorphic interface
- Micro Shake animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--shake
--bg
--surface
--shadow-dark
--shadow-light
--radius
--primary
```
