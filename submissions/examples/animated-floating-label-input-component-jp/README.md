# Animated Floating Label Input Component

## Overview

A modern CSS-only form input component with animated floating labels, glowing focus borders, and smooth transition effects.

## Features

- Floating label animation
- Smooth input focus transitions
- Glowing border effects
- Responsive and modern form design
- Lightweight CSS-only implementation

## Files

- demo.html – Demonstrates the animated input fields
- style.css – Contains floating label styling and animations
- README.md – Provides component documentation

## Usage

Open demo.html in a browser and click on any input field to see the floating label animation and glowing focus effect.
