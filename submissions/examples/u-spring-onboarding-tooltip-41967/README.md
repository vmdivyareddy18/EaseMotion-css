# Spring Onboarding Tooltip

A Minimalist-inspired Spring Onboarding Tooltip built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Spring tooltip animation
- Responsive layout
- Accessible HTML
- No JavaScript required
