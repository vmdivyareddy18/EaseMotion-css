# Animated Lock Security Bounce

## Overview

A reusable lock icon animation featuring a subtle bounce effect suitable for authentication systems, login pages, password managers, banking dashboards, and cybersecurity interfaces.

## Features

- CSS-only animation
- Lock bounce effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="lock-button">
  <span class="lock-icon">🔒</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
