# CSS Micro Shake Tooltip

Pure CSS animated tooltip utilizing a Micro Shake interaction designed for Dashboard Analytics layouts.

## Features

- Pure CSS implementation
- Micro Shake animation
- Dashboard analytics aesthetics
- Responsive design
- Keyboard accessible
- CSS custom properties
- Glassmorphism styling
- Smooth 60fps animations
- Zero JavaScript dependency

## Variables

```css
--primary
--secondary
--tooltip-bg
--duration
--ease
```
