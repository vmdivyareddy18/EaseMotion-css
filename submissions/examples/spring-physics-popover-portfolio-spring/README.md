# CSS Spring Physics Popover Component

An accessible, fully responsive overlay popover designed for high-end portfolio cards leveraging customized transition matrices to emulate realistic mechanical spring responses.

## Customizable Parameters
The following custom properties are exposed via `:root` to control the layout physics:
- `--spring-duration`: Controls the total acceleration time path.
- `--spring-easing`: The `cubic-bezier` timing configuration driving structural recoil thresholds.
- `--spring-pullback`: The initial offset matrix used prior to snap deployment.
