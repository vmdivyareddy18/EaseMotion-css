# Liquid Ripple Medical Dashboard Accordion

A responsive Medical Dashboard accordion featuring a smooth Liquid Ripple animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Native `<details>` and `<summary>`
- Keyboard accessible
- Responsive design
- Liquid Ripple animation
- Medical Dashboard inspired layout
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--primary
--secondary
--surface
--background
--text
--radius
```
