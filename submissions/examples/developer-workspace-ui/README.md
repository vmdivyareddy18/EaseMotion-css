# Animation: Developer Workspace UI

A complete IDE-inspired workspace combining an activity bar, file explorer, code editor, terminal, and status bar. Inspired by modern developer tools such as VS Code, Cursor, and Windsurf.

## Features

- Pure HTML & CSS
- Activity sidebar
- Interactive file explorer
- Code editor layout
- Integrated terminal
- Status bar
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="workspace">
    <aside class="activity-bar"></aside>
    <aside class="explorer"></aside>
    <main class="editor"></main>
</div>
```

## Why is it useful?

Developer-focused interfaces are increasingly common for documentation sites, code playgrounds, AI coding tools, and SaaS products. This example demonstrates a complete workspace layout with modern UI patterns and subtle interactions.
