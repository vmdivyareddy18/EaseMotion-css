# Shimmer Pulse Tooltip

A pure CSS animated tooltip featuring a smooth shimmer pulse interaction.

## Features

- Pure CSS implementation
- No JavaScript required
- Smooth shimmer animation
- Pulse hover interaction
- Keyboard accessible
- Responsive design
- CSS custom properties support

## Customization

You can modify:

```css
:root {
  --tooltip-duration: 2s;
  --tooltip-scale: 1.05;
  --tooltip-radius: 12px;
}
