Space Design System

Catapult Stopwatch

A cockpit launch-control stopwatch animated in pure CSS. Click the safety cover to open, pull the lever down to launch the stopwatch, and hit Reset to clear.

Launch Instructions: 1. Click the red SAFETY COVER to open it.  |  2. Click the LAUNCH LEVER to start or pause the stopwatch.  |  3. Click the red RESET button to clear.

Features & Accessibility Details

  • Pure CSS Timer Logic: Uses CSS Custom Properties (@property) and CSS Counters (counter-reset) to compute and display minutes, seconds, and centiseconds without loading Javascript.
  • Catapult Mechanics: Pulling the launch lever down triggers a multi-step spring animation (catapult-launch-ag) where the dial hand pulls back elastically (tension) before launching forward.
  • Tactile Safety cover: Implements a 3D perspective glass flip cover that shields the launch lever until explicitly toggled.
  • Keyboard Accessibility: Interactive labels are fully focusable with keyboard outline indicators. The toggle elements can be activated using standard Space/Enter triggers.
  • Zero-JS Reset: Leverages standard form resets to instantly clear all animated states and reset variables back to 0.
