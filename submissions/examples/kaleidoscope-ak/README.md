# Kaleidoscope Animation

## What does this do?
An interactive kaleidoscope animation with adjustable symmetry segments, rotation speed, and color schemes rendered on a canvas.

## How is it used?
Open demo.html in a browser. Adjust the segments slider (3-12) for symmetry, speed slider for rotation rate, and click "Next Scheme" to cycle through color palettes.

## Why is this useful?
This provides a mesmerizing visual effect suitable for creative tools, screensavers, or ambient displays. It demonstrates canvas 2D rendering, symmetry math, and real-time animation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to explore the kaleidoscope.
