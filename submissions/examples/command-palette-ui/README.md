# Animation: Command Palette UI

A modern command palette inspired by Raycast, VS Code, Linear, and Spotlight Search. It provides a centralized interface for navigation and actions with keyboard shortcut indicators.

## Features

- Pure HTML & CSS
- Command palette layout
- Search input
- Keyboard shortcut badges
- Hover interactions
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="command-item">

    <span>Open Settings</span>

    <kbd>⌘S</kbd>

</div>
```

## Why is it useful?

Command palettes have become a standard navigation pattern in modern productivity applications, developer tools, and SaaS platforms. This component demonstrates a clean and accessible implementation suitable for dashboards and web applications.
