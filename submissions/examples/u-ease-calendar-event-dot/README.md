# ease-calendar-event-dot

Animated calendar event indicator dots.

## Features

- Event dots scale in from 0 to 1
- Fade-in entrance animation
- Staggered appearance
- Different colors for event categories
- Pure HTML and CSS

## Usage

```html
<div class="event-dot work"></div>
<div class="event-dot meeting"></div>
<div class="event-dot personal"></div>
```

## Animation Details

- Uses `scale(0)` → `scale(1)` transition.
- Includes opacity fade-in.
- Supports staggered timing for multiple events.
- Different colors represent different event types.
