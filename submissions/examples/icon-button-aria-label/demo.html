Icon Button Aria Labels

Icon-only buttons need accessible labels for screen readers. Add aria-label to provide descriptive text for assistive technology users.

Icon Buttons with Aria Labels

Common Icon Button Patterns

Close Button

aria-label="Close dialog"

Add Button

aria-label="Add new item"

Share Button

aria-label="Share this content"

Like Button

aria-label="Like this post"

Implementation

HTML

<!-- Icon button with aria-label -->
<button class="ease-btn ease-btn-icon" aria-label="Close menu">
  <svg>...</svg>
</button>

<!-- Icon button with aria-label for screen readers -->
<button class="ease-btn ease-btn-icon" aria-label="Delete item">
  <svg>...</svg>
</button>

CSS for Icon Buttons

.ease-btn-icon {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  width: 40px;
  height: 40px;
  padding: 0;
  border-radius: var(--ease-radius-md);
  background: var(--ease-color-surface);
  border: 1px solid var(--ease-color-neutral-200);
  cursor: pointer;
  transition: all var(--ease-speed-fast) var(--ease-ease);
}

.ease-btn-icon:hover {
  background: var(--ease-color-primary);
  color: #fff;
  border-color: var(--ease-color-primary);
}

.ease-btn-icon:focus {
  outline: 2px solid var(--ease-color-primary);
  outline-offset: 2px;
}

Accessibility Checklist

  • ✅ Always add aria-label to icon-only buttons
  • ✅ Use descriptive labels that explain the action
  • ✅ Test with screen readers (NVDA, VoiceOver, JAWS)
  • ✅ Ensure focus states are visible
  • ✅ Don't use title attribute alone (not reliable)
  • ✅ Consider aria-pressed for toggle buttons
