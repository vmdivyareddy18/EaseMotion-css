# Deep-Space Satellite Constellation Telemetry Showcase
A futuristic, responsive HTML & CSS telemetry dashboard inspired by deep-space satellite constellation monitoring. The interface showcases live satellite health, communication status, telemetry logs, and mission analytics with smooth animations—built entirely without JavaScript.

---

## Features
- 🚀 Deep-space telemetry inspired dashboard
- 🛰️ Satellite constellation overview cards
- 📡 Live communication status indicators
- 📊 Animated signal strength progress bars
- 🌌 Futuristic dark-space interface
- 💻 Fully responsive layout
- ⚡ Pure HTML & CSS (No JavaScript)

## How to Use
1. Open `demo.html` in any modern browser.
2. Explore the responsive telemetry dashboard.
3. Customize colors, cards, and animations inside `style.css`.

---

## HTML Usage
```html
<div class="dashboard">
  <div class="card">

    <span class="badge online">Online</span>

      <div class="bar"></div>
    </div>
  </div>

```

The smooth transitions, responsive dashboard layout, glowing cards, and animated telemetry indicators align perfectly with the framework's animation-first philosophy while remaining lightweight and easy to customize.
