# Vortex Spin Animation (`anim-vortex-spin`)

**What does this do?**  
It creates a complex, "hard level" 3D entrance animation that rotates the element across all three axes (X, Y, Z) while simultaneously scaling up and shifting from a blurred state to sharp focus.

**How is it used?**  
Apply the `.anim-vortex-spin` class to any element you want to animate upon entrance or on hover:
```html
<div class="anim-vortex-spin">
  I spin into view!
</div>
```

**Why is it useful?**  
It fits EaseMotion CSS's philosophy of providing advanced, complex motion effects out-of-the-box without requiring users to manually write convoluted 3D transform keyframes. It's perfect for high-impact hero elements or showcasing premium components.
