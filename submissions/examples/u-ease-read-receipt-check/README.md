# ease-read-receipt-check

A messaging-style read receipt animation built using HTML and CSS.

## Features

- Single checkmark for sent state
- Double checkmark for delivered state
- Blue checkmarks for read state
- Fade and scale animation
- Messaging UI inspired design
- Responsive layout

## States

### Sending
Single checkmark shown.

### Sent
Message successfully sent.

### Delivered
Second checkmark fades in.

### Read
Both checkmarks become blue.

## Animation

The second checkmark uses:

- opacity transition
- scale animation
- smooth appearance effect
