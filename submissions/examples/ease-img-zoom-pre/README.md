## ease-img-zoom

**What does this do?**
Adds a smooth zoom effect to images on hover for product and
portfolio cards, with overflow clipping and reduced-motion support.

**How is it used?**
<div class="ease-img-zoom">
  <img src="product.jpg" alt="Product" />
</div>

<!-- Slow zoom -->
<div class="ease-img-zoom-slow">
  <img src="product.jpg" alt="Product" />
</div>

<!-- Zoom with color overlay -->
<div class="ease-img-zoom-overlay">
  <img src="product.jpg" alt="Product" />
</div>

**Why is it useful?**
Provides a polished image hover interaction for product and portfolio
grids with reduced-motion accessibility support — consistent with
EaseMotion CSS's animation-first philosophy.
