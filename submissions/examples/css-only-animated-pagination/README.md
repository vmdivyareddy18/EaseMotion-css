# CSS-Only Animated Morphic Pagination

An interactive, fluid pagination tracking component powered entirely via semantic HTML labels, native radio inputs, and composite sibling CSS selectors.

## Features
- **Zero Scripting Overheads**: Utilizes functional native browser state loops (`:checked ~`) to trigger component states smoothly.
- **Elastic Micro-Interactions**: Leverages an energetic, customized bouncy cubic-bezier path config to produce liquid movement.
- **Fully Modular Controls**: Tailor container gaps, element scales, and backdrop colors effortlessly via variable assignments.
