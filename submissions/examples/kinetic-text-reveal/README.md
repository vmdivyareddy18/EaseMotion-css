# Kinetic Typography Split-Text Reveal

A high-end motion graphics effect where a headline's characters are masked and rotate/translate into view with staggered delays.

## Features
- Staggered animation using `nth-child`
- Smooth translation and rotation
- Clean and performant pure CSS implementation
- Cubic bezier easing for a natural feel

## Usage
```html
<div class="split-text">
  <span>T</span><span>E</span><span>X</span><span>T</span>
</div>
```

## Browser Support
- Chrome, Firefox, Safari, Edge (Modern versions)

## Tech Stack
- HTML + CSS only, no JavaScript required
