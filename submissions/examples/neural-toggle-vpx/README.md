1. What does this do?
A micro-interaction checkbox toggle containing responsive, organic sub-particles that visually displace when the input state updates.

2. How is it used?
```
<label class="switch-label">
  <input type="checkbox" class="switch-input">
  <span class="switch-track">
    <span class="switch-thumb"></span>
    <span class="switch-particle"></span>
    <span class="switch-particle"></span>
  </span>
</label>
```

3. Why is it useful?
It replaces heavy JS physics solvers or spring libraries with localized, pure CSS delayed keyframes, keeping user dashboards tactile and engaging with minimal runtime overhead.
