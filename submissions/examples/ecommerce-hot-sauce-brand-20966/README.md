# App Landing: Hot Sauce Brand (`ecommerce-hot-sauce-brand-20966`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a bold, personality-driven hot sauce brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A spicy food brand requires a UI that feels explosive, loud, and playfully aggressive. This template achieves a bold aesthetic using high-contrast colors (red, yellow, orange, stark black/white), heavy borders and hard shadows (brutalism/neo-brutalism influence), chunky `Anton` typography, and bouncy, popping animations.

The layout includes:
- A sliding warning Promo Banner.
- A Navigation bar with shaking/wiggling interactions.
- A Hero section featuring abstract CSS flames, bursting splashes, and an oversized floating hot sauce bottle.
- A Product Grid categorized by heat level, featuring CSS-rendered bottles and shaking alert badges.
- A Story section with a rotating CSS pepper graphic and jagged CSS dividers.
- A Recipe cards grid with CSS-rendered abstract food icons (taco, wing, pizza).
- An infinite scrolling Review Marquee.
- A massive, high-contrast Shop CTA.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use explosive `cubic-bezier(0.175, 0.885, 0.32, 1.275)` and hyper-bouncy `cubic-bezier(0.68, -0.55, 0.265, 1.55)` timings that convey heat and energy:
- `.ease-pop-in` (Configured for an aggressive, skewed scaling entrance).
- `.ease-slide-up-bounce` (A high-tension vertical entrance).
- `.ease-burst` / `.ease-flicker` (Continuous ambient animations for the abstract splashes and flames).
- `.ease-drip-down` (A continuous background-position animation simulating dripping sauce).
- `.ease-wiggle` / `.ease-pulse-alert` (Attention-grabbing infinite animations for badges).
- `.ease-marquee` (An infinite linear translation for the reviews banner).
- `.ease-hover-shake` (A brutalist hover effect that translates the element and drops a hard shadow).
- `.ease-hover-tilt` (A bouncy rotational hover interaction).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the brutalist layout architecture, the vibrant palette, and the complex abstract CSS illustrations (bottles, peppers, food items). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the bouncy, popping animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
