# Pure CSS Holographic Follow Button Component

A visually engaging, responsive, and accessible follow button built entirely with CSS for EaseMotion CSS. Inspired by nature-themed interfaces, the component combines holographic gradients, shimmering highlights, and soft glow effects to create a polished call-to-action for profile cards, social platforms, and community pages.

## Features

- **Pure CSS:** No JavaScript required.
- **EaseMotion Ready:** Integrates seamlessly with EaseMotion CSS utility classes.
- **Holographic Effect:** Animated shimmer and layered gradient highlights.
- **Nature-Inspired Design:** Soft green and teal palette with subtle glow effects.
- **Responsive Layout:** Adapts smoothly across desktop and mobile devices.
- **Accessible:** Uses a semantic HTML button with keyboard focus support.

## Configuration Parameters

The component can be customized using the following CSS custom properties:

| Variable | Description | Default Value |
| :--- | :--- | :--- |
| `--primary` | Primary button color | `#22c55e` |
| `--secondary` | Secondary holographic accent | `#06b6d4` |
| `--accent` | Additional highlight color | `#84cc16` |
| `--bg` | Page background color | `#eefbf2` |
| `--surface` | Card background color | `rgba(255, 255, 255, 0.75)` |
| `--text` | Primary text color | `#1f2937` |
| `--muted` | Secondary text color | `#64748b` |
| `--radius` | Card border radius | `22px` |

## How to Run Locally

1. Clone your fork of the repository.
2. Navigate to `submissions/examples/holographic-follow-button-amna/`.
3. Open `demo.html` in any modern web browser to interact with the component.
