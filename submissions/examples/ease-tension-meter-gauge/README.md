# Ease Tension Meter Gauge

A pure CSS animated tension meter gauge component for dashboards, analytics pages, and monitoring interfaces.

## Features

- Pure CSS implementation
- Smooth gauge needle animation
- Responsive design
- Customizable CSS variables
- No JavaScript required

## Usage

Add the component markup:

```html
<div class="tension-card">
    <h2>Tension Meter</h2>

    <div class="gauge">
        <div class="needle"></div>
    </div>

    <div class="level">
        Medium Tension
    </div>
</div>
