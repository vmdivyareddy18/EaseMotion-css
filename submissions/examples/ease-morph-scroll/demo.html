How It Works

IntersectionObserver + CSS transition

No keyframes. No animation libraries. Just three browser APIs working together.

1

Define the Observer

Create an IntersectionObserver with a fine-grained threshold array — 101 values from 0 to 1 — so the callback fires at every 1% of intersection. 

const thresholds =
  Array.from({length: 101},
    (_, i) => i / 100);

const observer =
  new IntersectionObserver(
    callback,
    { threshold: thresholds }
  );
2

Map Ratio to Radius

Inside the callback, read entry.intersectionRatio (0 → 1) and map it linearly to border-radius (50% → 0%). No math library needed. 

const callback = entries => {
  entries.forEach(entry => {
    const r = entry
      .intersectionRatio;
    // 0 visible → 50%
    // 1 visible → 0%
    const radius = 50 - r * 50;
    entry.target.style
      .borderRadius =
      `${radius}%`;
  });
};
3

CSS Transition Smooths It

Even though JavaScript updates the value at every threshold step, the transition property on the element fills the gaps between callbacks for a silky-smooth result. 

/* style.css */
.morph-card {
  border-radius: 50%;
  transition:
    border-radius 120ms
    ease-out;
}

/* JS sets inline style —
   transition handles
   the interpolation */

Why Scroll-Linked Morphing Works

🎯
Directs attention

The transformation draws the eye exactly where the designer needs focus — no additional animation required.

⚡
Zero paint cost

border-radius is a compositor-only property. No layout recalculation, no repaint — pure GPU compositing.

📐
Reveals hierarchy

The circle-to-square arc signals "I'm arriving" — a visual metaphor for focus sharpening as content becomes readable.

♿
Respects preferences

Pair with prefers-reduced-motion to set border-radius: 0 immediately — the structure requires zero extra code.

Practical Applications

Where to Use This Effect

The morph technique is versatile — anywhere a rectangular element would benefit from a more dynamic entrance.

🖼️

Image Reveals

Photography portfolios and editorial sites where images unfurl from abstract blobs into sharp frames as they scroll into view. Creates a memorable "developing film" feeling.

👤

Team Profiles

"About Us" and leadership pages where each avatar starts as a minimal dot and expands into a full portrait. Guides the user's eye through each team member in sequence.

🛍️

Product Galleries

E-commerce grids where each product card enters as a coloured orb, teasing the palette and category before snapping into a recognisable card layout.

💼

Portfolio Cards

Creative portfolios and agencies where work items bloom from circles into their natural aspect ratios, giving each case study a distinct, memorable entrance moment.

🚀

Landing Pages

SaaS and product landing pages where feature icons or screenshot mockups use the morph to introduce each section with a tactile, motion-rich moment of arrival.

🌟

Hero Sections

Page heroes where a central visual element transitions from an abstract branded circle on load to a sharp product image or illustration as scroll begins — a powerful first impression.
