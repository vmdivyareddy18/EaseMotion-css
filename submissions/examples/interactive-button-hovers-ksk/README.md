# Interactive Button Hover Animations (`interactive-button-hovers-ksk`)

1. What does this do?  
Adds a collection of four distinct, lightweight button hover animations: Icon Glide, Border Draw, Gradient Shimmer, and Neon Glow.

2. How is it used?  
Apply the `.btn-hover-demo` base class alongside one of the animation modifiers: `.icon-glide`, `.border-draw`, `.animated-gradient`, or `.neon-glow`.

3. Why is it useful?  
It provides developers with easy-to-use, isolated hover animations that enhance micro-interactions and visual feedback for button elements while keeping styles cleanly decoupled.
