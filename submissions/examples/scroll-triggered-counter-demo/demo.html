Scroll-Triggered Counter Demo (Pure CSS)

Numbers count up from 0 to their target value only as they scroll into view, using animation-timeline: view() — no JavaScript, no IntersectionObserver. Scroll down to see the stats animate.

💡 Browser support: Chrome/Edge 115+ trigger the count-up on scroll. Firefox/Safari fall back to the existing .ease-count-up on-load behavior from core/animations.css — numbers still animate, just on page load instead of on scroll.
