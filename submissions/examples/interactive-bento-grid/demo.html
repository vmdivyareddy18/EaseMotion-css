How to Use

To use the Bento Grid, copy the structure below. Link the stylesheet `style.css`. Staggered entrance animations are managed entirely in CSS via `:nth-child` delays. Use the click listener script to toggle the active `.ease-active` class, expanding details drawers inline without shifting adjacent cards.

HTML Markup Structure

<div class="ease-bento-grid">
  <!-- Card 1 (Span 2x2 grid columns & rows) -->
  <div class="ease-bento-card ease-span-2x2">
    <!-- Card Face -->
    <div class="card-title">Cluster Core</div>
    
    <!-- Slide up details drawer overlay -->
    <div class="ease-bento-details">
      <button class="ease-bento-close-btn"><!-- Close Icon --></button>
      <h4>Core Metrics</h4>
      <!-- Expanded content -->
    </div>
  </div>
  
  <!-- Card 2 (Span 2x1 grid columns) -->
  <div class="ease-bento-card ease-span-2x1">
    <!-- Content -->
  </div>
</div>
