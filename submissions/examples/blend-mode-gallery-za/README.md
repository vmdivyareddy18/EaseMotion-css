# CSS Mix Blend Mode Gallery

An interactive gallery demonstrating six CSS mix-blend-mode values (multiply, screen, overlay, difference, exclusion, hard-light) using layered colored shapes over gradient backgrounds.

## Files

- `demo.html` - Six blend mode demo cards
- `style.css` - Mix-blend-mode classes, gradient backgrounds, card layout

## Usage

Open `demo.html` to see the different blend mode effects.
