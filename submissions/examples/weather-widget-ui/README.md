# Animated Weather Widget

A clean, modern weather widget card with an animated sun icon (pulsing core + rotating rays), built using pure HTML and CSS.

## What it does

- Displays temperature, condition, and location.
- Sun icon has a continuously pulsing core and slowly rotating rays.
- Shows humidity and wind stats in a footer row.
- Fully responsive — card width adapts to smaller screens.

## How to use

1. Copy `demo.html` and `style.css` into your project.
2. Open `demo.html` directly in a browser — no build step or server required.
3. Reuse the `.weather-card` markup, swapping in your own temperature, condition, location, and detail values. Swap `.weather-icon.sun` for other conditions by adding matching icon variants.

## Why it fits EaseMotion CSS

- Pure HTML/CSS, no JavaScript or external dependencies.
- Beginner-friendly: clear class names, simple keyframe animations.
- Meets all requirements from issue #47329: temperature display, animated weather icon, clean modern design, mobile responsive.
