# Configurable Easing Functions for Transition Animations

This feature adds standard mathematical easing functions to replace rigid linear transitions.

## Features Included
- **Linear**: Default constant speed.
- **Ease-In**: Slow start, fast finish.
- **Ease-Out**: Fast start, slow finish.
- **Ease-In-Out**: Smooth acceleration and deceleration.
- **Elastic**: Playful overshoot and bounce effect.

## How to Run
Open `demo.html` in any browser, select your preferred easing algorithm, and click "Animate Box".
