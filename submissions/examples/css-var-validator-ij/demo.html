CSS Variable Validator

Validate EaseMotion CSS custom properties against expected formats

A Node.js script that scans a stylesheet, parses all --ease-* custom properties, and validates them against expected patterns (colors, lengths, durations, keywords). Below is a sample validation report.

Variable Expected Status
--ease-progress-height <length> ✓ Valid
--ease-loader-color <color> ✓ Valid
--ease-loader-size <length> ✓ Valid
--ease-fade-duration <time> ✓ Valid
--ease-mask-fade-direction to right | to left | to top | to bottom ✓ Valid
--ease-dot-size <length> ✓ Valid
--ease-dot-spacing <length> ✓ Valid
--ease-dot-color <color> ✓ Valid
--ease-grid-size <length> ✓ Valid
--ease-grid-color <color> ✓ Valid
--ease-noise-opacity <number> 0–1 ⚡ Warning (out of range)
