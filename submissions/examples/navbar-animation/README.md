# ✨ Animated Navbar

A modern, responsive, and animated navigation bar built using pure **HTML** and **CSS**. This project features a sleek dark UI inspired by contemporary landing pages, complete with smooth hover effects, an animated active navigation item, and a stylish theme toggle button.

---

## 📸 Preview

A clean navigation bar featuring:

- Modern dark theme
- Animated active navigation button
- Smooth hover transitions
- Responsive layout
- Circular theme toggle icon
- Glassmorphism-inspired design

---

- 🎨 Modern dark user interface

## 🎯 Components

A circular button positioned on the right with:

- Hover rotation
- Scale animation
- Glow effect

---

## ✨ Animations Included

### Hover Animation

Navigation items gently move upward while changing background color.

### Active Button Animation

The active button displays:

- Soft glowing shadow
- Animated underline
- Smooth transition

### Theme Button Animation

Hovering over the theme button triggers:

- Rotation
- Scale effect
- Background transition

## 🎨 Customization

### Change Background

```css
body{
    background:#09090f;
}
```

---

### Change Active Color

```css
background:linear-gradient(180deg,#24234d,#18172f);
```

---

### Change Glow Color

```css
box-shadow:
0 0 20px rgba(94,91,255,.35);
```

---

### Change Animation Speed

```css
transition:.35s;
```

or

```css
animation:underline 2s infinite ease-in-out;
```
