## neon-border-trace

**What does this do?**
Adds a glowing neon line that continuously travels around
the border of an element using conic-gradient and CSS @property.

**How is it used?**
<div class="neon-border-trace">
  <button>Click me</button>
</div>

<div class="neon-border-trace green">
  <div>Card content</div>
</div>

Color variants: purple (default), green, orange, pink

**Why is it useful?**
Creates an eye-catching animated border effect using pure CSS —
perfect for CTAs, featured cards, and premium UI elements.
Consistent with EaseMotion CSS's animation-first philosophy.
