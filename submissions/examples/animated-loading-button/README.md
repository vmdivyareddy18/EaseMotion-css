# Animated Loading Button

A responsive animated loading button component built using pure HTML and CSS.

## Features

- Loading spinner animations
- Success state styling
- Hover interactions
- Disabled/loading state behavior
- Responsive layout
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated loading button component.

## Files

- `demo.html` → Loading button structure
- `style.css` → Button styles and animations
- `README.md` → Documentation
