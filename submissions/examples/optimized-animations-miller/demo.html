Optimized Animations: Transform & Opacity

Side-by-side comparison of a layout-triggering animation (using left / width) versus a GPU-accelerated version using transform and opacity.

❌ Unoptimized

Animates left and width — triggers layout/paint every frame.

.box-unoptimized {
  position: relative;
  left: 0;
  width: 60px;
  animation: move-left 2.5s ease-in-out infinite alternate;
}

@keyframes move-left {
  from { left: 0;    width: 60px;  }
  to   { left: 240px; width: 100px; }
}

✅ Optimized

Animates transform and opacity — runs on the GPU compositor.

.box-optimized {
  transform: translateX(0) scale(1);
  opacity: 1;
  animation: move-transform 2.5s ease-in-out infinite alternate;
}

@keyframes move-transform {
  from { transform: translateX(0) scale(1);     opacity: 1;   }
  to   { transform: translateX(240px) scale(1.4); opacity: 0.7; }
}
