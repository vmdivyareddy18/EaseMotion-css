# Education: Homeschool Curriculum Provider (`education-homeschool-curriculum-21040`)

This proposal provides a complete, copy-paste ready landing page template designed for a Homeschool Curriculum Provider, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A Homeschool Curriculum brand requires a UI that feels warm, welcoming, family-oriented, and joyful. This template achieves this aesthetic using a "natural" color palette with Deep Blue (`#3D6B82`), Mustard Yellow/Sun (`#E5A93D`), Sage Green (`#698362`), and Warm Sand/Paper backgrounds. It pairs a friendly, rounded display font (`Nunito`) with a highly readable body font (`Quicksand`). 

The layout includes:
- A Navigation with a CSS sun icon.
- A Hero section featuring abstract floating shapes, a CSS-animated "learning scene" mockup (with a CSS book that turns pages), and organic sticker badges.
- A Teaching Philosophy section utilizing an abstract CSS blob mask and family reading illustration.
- A Curriculum by Grade grid featuring book spine illustrations and soft, lifted hover states.
- A Community section displaying tilted, polaroid-style photos with handwriting fonts.
- A transparent Pricing bundle grid.
- A warm CTA section.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use bouncy, joyful `cubic-bezier(0.34, 1.56, 0.64, 1)` timings tailored for a family-friendly feel:
- `.ease-slide-up-stagger` (Configured for a bouncy vertical entrance).
- `.ease-pop-in` / `.ease-pop-stagger` (For playful, springy component reveals).
- `.ease-float` (Applied to background shapes for a gentle, organic floating effect).
- `.ease-wiggle` (Applied to the open-and-go sticker badges to draw attention playfully).
- `.ease-hover-bounce` (A joyful, springy hover interaction for primary buttons).
- `.ease-hover-tilt` (Interactive hover state for the community polaroids and learning scene).
- `.ease-page-turn` (A custom continuous CSS animation simulating a book's pages turning in the hero section).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the warm color palette to the rounded UI elements and the abstract CSS illustrations (sun, trees, compass, books). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the precise, joyful animations. No JavaScript is required for this template.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/education-homeschool-curriculum-21040/` directory to avoid modifying core files directly and to ensure zero deletions.*
