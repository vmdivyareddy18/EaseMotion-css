# ease-media-player-controls (#36809)

Media player controls with animated play button and progress bar.

## CSS Custom Properties

| Variable            | Default    | Description                |
|---------------------|------------|----------------------------|
| `--control-bg`      | `#2a2a3e`  | Card / control background  |
| `--btn-color`       | `#00d4aa`  | Play / skip button color   |
| `--progress-color`  | `#00d4aa`  | Progress bar fill color    |
| `--transition-speed`| `0.3s`     | Duration of all transitions |

## Features

- Animated play / pause toggle with pulse ring effect
- Simulated progress bar with live time display
- Skip prev / next buttons
- Hover scaling on all controls
- Dark theme card layout
