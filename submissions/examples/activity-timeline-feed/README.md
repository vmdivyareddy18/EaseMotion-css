# Animation: Activity Timeline Feed

A vertical timeline component designed for displaying activity history, changelogs, milestones, and project updates. Timeline entries animate into view and respond to hover interactions.

## Features

- Pure HTML & CSS
- Vertical timeline layout
- Staggered entry animations
- Interactive timeline cards
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="timeline-item">

    <div class="timeline-dot"></div>

    <div class="timeline-card">
        <h3>Milestone Title</h3>
    </div>

</div>
```

Timeline entries appear sequentially and highlight on hover.

## Why is it useful?

Activity feeds and timelines are widely used in changelogs, project tracking systems, developer portfolios, release notes, and productivity applications. This component provides a clean and engaging way to present chronological information.
