# scroll-margin-top

Demonstrates `scroll-margin-top` for anchor links with fixed headers. Each section has different scroll-margin values to show how offset affects scroll position.

## Usage

Add `scroll-margin-top` to section elements. Navigate via the fixed nav bar — each link scrolls to its section with the specified top margin. Adjust the fixed header height with different margin values per section.

## Browser Support

`scroll-margin-top` is supported in all modern browsers (Chrome, Firefox, Safari, Edge) since 2020. Also works with `scroll-behavior: smooth` for animated scrolling.
