# Flip Newsletter Popup

A modern newsletter popup component featuring a smooth flip animation built with pure HTML and CSS.

## Features

- Pure CSS (No JavaScript)
- Flip animation
- Responsive layout
- Accessible form
- Modern dark UI
- Reusable component
- Beginner-friendly implementation

## Folder

```
demo.html
style.css
README.md
```

## Usage

```html
<input type="checkbox" id="popup-toggle" />

<label for="popup-toggle" class="open-btn"> Subscribe Newsletter </label>

<div class="overlay">
  <div class="popup">...</div>
</div>
```

The popup is toggled using a hidden checkbox, making the component completely JavaScript-free.

## Accessibility

- Keyboard accessible
- Proper email input
- Responsive design

## Animation

The popup flips into view using a CSS `rotateY()` transform with smooth transitions.
