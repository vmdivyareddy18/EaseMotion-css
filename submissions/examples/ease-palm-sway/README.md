# Ease Palm Sway Animation

An asymmetric, organic CSS keyframe animation designed to replicate the natural, fluid physics of environmental objects (like palm trees, foliage, or banners) catching a rhythmic breeze.

## Features
- **Bottom-Anchored System**: Configured with `transform-origin: bottom center` to prevent float-slipping during standard axis rotations.
- **Asymmetric Timing**: Non-linear rotation steps make the sway feel authentic and random, rather than mechanical.
- **Multiple Modifier Weights**: Comes equipped with three pre-configured speeds (`default`, `fast`, `slow`).

## Core Utility Usage
```html
<div class="your-element ease-palm-sway">🌴</div>
<div class="your-element ease-palm-sway-fast">🌴</div>
<div class="your-element ease-palm-sway-slow">🌴</div>
