# CSS Pulse-Active Popover

A responsive **Pure CSS Pulse-Active Popover** designed for **Product Catalog** layouts. The component uses a smooth pulse animation and reveals additional product information on hover or keyboard focus.

## Features

- Pure HTML & CSS
- Responsive Design
- Keyboard Accessible (`:focus-visible`)
- Pulse Animation
- Hover & Focus Popover
- CSS Custom Properties
- No JavaScript

## Technologies

- HTML5
- CSS3

## CSS Variables

```css
--pulse-duration
--pulse-scale
--primary
--popover-bg
--transition
--radius
--shadow
```
