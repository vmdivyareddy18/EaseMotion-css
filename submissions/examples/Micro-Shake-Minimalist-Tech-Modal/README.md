# Micro Shake Minimalist Tech Modal

A clean, responsive minimalist tech modal featuring a subtle Micro Shake interaction built entirely using HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Minimal monochrome interface
- Micro Shake animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--shake
--bg
--surface
--text
--muted
--border
--radius
```
