Animation Breakdown

Each property plays a specific role in creating the lower-left exit rotation effect.

↺

Rotation Angle

rotate(45deg) rotates the element 45 degrees clockwise. The positive angle drives the element downward and leftward, making it exit toward the lower-left corner of the viewport.

45°
◆

Transform Origin

transform-origin: left bottom sets the pivot to the element’s lower-left corner. Instead of rotating around its centre, the element swings from its bottom-left anchor, creating a natural drop-away motion.

Pivot (left bottom)
◇

Opacity Transition

Opacity moves from 1 to 0 in parallel with the rotation. The fade-out prevents a harsh disappearance, making the exit feel soft and polished — ideal for dismissible UI elements.

Use Cases

Origin-based exit rotations lend themselves to a variety of real-world UI scenarios.

Notification Dismissal

Toast notifications can drop away with a lower-left rotation, making dismissals feel intentional and spatially grounded.

Card Removal

Removing cards from a grid with a lower-left rotation exit distinguishes dismissed content and keeps the UI feeling fluid and dynamic.

Modal Closing

Dismiss modal overlays with a grounded lower-left exit, providing visual closure that signals the action is complete.

Dashboard Interactions

Dismissable dashboard widgets and panels benefit from a directional exit that reinforces spatial awareness in data-dense interfaces.

@keyframes rotate-exit-lower-left {
  0% {
    opacity: 1;
    transform: rotate(0deg);
  }
  100% {
    opacity: 0;
    transform: rotate(45deg);
  }
}
