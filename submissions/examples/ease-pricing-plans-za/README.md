# CSS Pricing Plans Component

## What does this do?
A complete pricing section with monthly/yearly toggle, three tier cards with feature lists, comparison table, FAQ accordion, testimonial, and call-to-action section.

## How is it used?
```html
<section class="pr-plans">
  <div class="pr-plan pr-starter">
    <h3>Starter</h3>
    <div class="pr-price-wrap">
      <span class="pr-amount">$9</span>
    </div>
    <ul class="pr-features">
      <li>Feature</li>
    </ul>
    <button class="pr-btn">Get Started</button>
  </div>
</section>
```

## Why is it useful?
Provides a complete pricing page for SaaS products and subscription services. Includes toggle between monthly/yearly billing, highlighted popular plan, detailed feature comparison table, FAQ accordion section with expandable questions, social proof testimonial block, and a conversion-focused call-to-action. Fully responsive and customizable via CSS variables with dark theme default.
