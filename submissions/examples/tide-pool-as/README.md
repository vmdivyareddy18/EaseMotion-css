# Tide Pool

A pure CSS and HTML rock pool seen from above: a sea star, anemones, an urchin and a shell resting under rippling water.

## What it shows

- A five armed sea star built with the radial rotate pattern, each arm parking its own angle and dotted with tube feet
- Anemones and an urchin from repeating conic gradients, one masked into a ring of tentacles
- Rippling water from a diagonal repeating linear gradient that shifts back and forth
- A round pool clipped inside a rocky rim drawn with a conic gradient
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
