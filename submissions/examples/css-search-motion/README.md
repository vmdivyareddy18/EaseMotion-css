# 🔍 CSS Search Motion

A collection of beautiful, pure CSS animated search bars for modern web interfaces.

## 🚀 Features

### Search Bar Styles (10 types)
1. **Expand on Focus** - Collapses to icon, expands on focus
2. **Slide from Icon** - Bottom underline slides in
3. **Neon Glow** - Cyberpunk neon pulsing glow
4. **Gradient Border** - Animated gradient border
5. **Underline Style** - Minimal underline with gradient fill
6. **Pill with Button** - Rounded pill with search button
7. **Glassmorphism** - Frosted glass effect
8. **3D Press** - 3D depth with press effect
9. **Bounce Icon** - Icon bounces on focus
10. **Rotating Border** - Conic gradient rotating border

### Animation Features
- Smooth focus transitions
- Icon transformations
- Glow and pulse effects
- Gradient animations
- 3D depth effects
- Bounce animations
