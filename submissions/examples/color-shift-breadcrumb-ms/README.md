# Color Shift Breadcrumb

Resolves #42401.

### What does this do?
Adds a Color Shift Breadcrumb component to EaseMotion CSS tailored for Portfolios and personal sites. It uses a CSS linear gradient with a 400% background size. When hovered, the `color` becomes transparent, and the `background-clip: text` becomes visible while animating smoothly through the gradient colors.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Apply the `shift-link` class to the `<a>` tags in your breadcrumb navigation.

### Why is it useful?
Provides an elegant, subtle pop of creativity for text links. It feels premium and playful without disrupting the core accessibility or semantics of standard breadcrumb navigation.
