# Animated Testimonial Wall

A modern testimonial wall component featuring animated review cards, customer profiles, ratings, and responsive layouts.

## Features

- Animated testimonial cards
- Customer profile section
- Star ratings
- Featured testimonial highlight
- Hover interactions
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

- Customer names
- Ratings
- Testimonials
- Colors
- Animations
- Layout spacing

## Use Cases

- SaaS landing pages
- Product websites
- Agency portfolios
- Business websites
- Customer review sections
