# CSS Resume / CV Card

## What does this do?
Professional CSS resume/CV card with sidebar, avatar, contact info, skill tags, experience timeline, and education sections.

## How is it used?
```html
<div class="rc-resume"><div class="rc-sidebar"><div class="rc-avatar">JD</div><h3>Name</h3><div class="rc-skills"><div class="rc-skill-tags"><span>Skill</span></div></div></div><div class="rc-main"><div class="rc-block"><strong>Experience</strong><div class="rc-item"><span class="rc-role">Role</span><span class="rc-company">Company | Date</span><p>Description</p></div></div></div></div>
```

## Why is it useful?
Provides a complete resume/CV card layout without JavaScript. Sidebar with avatar, contact info, and skill tags. Main area with experience and education sections. Professional dark theme. Responsive sidebar flex layout. Ideal for portfolios, team pages, and profile sections.
