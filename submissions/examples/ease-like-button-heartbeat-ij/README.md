# Like Button Heartbeat

A heart / like button component with a heartbeat animation on click. The heart fills with color when active and the count increments with a bounce animation. Supports multiple independent like buttons for different items.

## Usage

Include `style.css` and copy the HTML structure. Each `.like-btn` toggles its `active` class on click.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--lhb-duration` | 0.3s | Transition duration |
| `--lhb-heart-color` | #e2e8f0 | Heart color (inactive) |
| `--lhb-active-color` | #ef4444 | Heart color (active / liked) |
| `--lhb-count-color` | #64748b | Count text color |
| `--lhb-size` | 24px | Heart icon size |

## Browser Support

Chrome, Firefox, Safari, Edge
