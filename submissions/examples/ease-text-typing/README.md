# Typing Text Animation

## What does it do?
A pure CSS typewriter effect that reveals text character by character with a blinking cursor.

## Features
- Character-by-character reveal using `width` + `steps()` animation
- Blinking cursor via animated `border-right`
- Pure CSS, no JavaScript

## Usage
```html
<div class="typewriter">
  <p class="typewriter-text">Your text here.</p>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
