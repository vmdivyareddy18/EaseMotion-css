# Animation Timing Function

A side-by-side comparison of CSS timing functions: ease, linear, ease-in, ease-out, ease-in-out, and a custom cubic-bezier. Each card shows both the animated ball and its timing graph.

## Features

- Six timing functions displayed in a responsive grid.
- Each card includes an SVG timing-diagram preview of the curve.
- Custom cubic-bezier (overshoot) highlighted in amber vs standard indigo.
- Pause/Resume button to freeze all animations for close inspection.

## Topics Covered

- Named easing functions (ease, linear, ease-in, ease-out, ease-in-out)
- `cubic-bezier()` with overshoot values
- `animation-timing-function` property
- Relationship between the timing graph and perceived motion
- `animation-play-state` toggle
