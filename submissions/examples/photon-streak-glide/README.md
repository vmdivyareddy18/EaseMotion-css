# Photon Streak Glide Animation Utility

## Overview
This utility creates a high-speed glowing photon streak that glides across an element, producing a futuristic light trail effect.

## Usage

```html
<div class="ease-photon-streak">
  Animated Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable utility naming
- Animation-first interaction
- Great for futuristic UI, dashboards, hero sections, and cyberpunk-inspired interfaces
