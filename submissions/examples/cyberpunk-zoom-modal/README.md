# Cyberpunk Neon Zoom-In Modal

A responsive **pure CSS** modal component featuring a cyberpunk-inspired neon design and a smooth zoom-in animation. The component is intended as a documentation/example submission for the EaseMotion CSS examples library.

## Features

* Pure HTML & CSS (no JavaScript required)
* Smooth zoom-in modal animation
* Cyberpunk neon-inspired visual theme
* Responsive layout for desktop, tablet, and mobile
* CSS Custom Properties for easy customization
* Keyboard-friendly HTML structure
* Lightweight and easy to integrate

## Files

```text
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in any modern web browser.

The component uses a hidden checkbox and CSS selectors to control the modal state without JavaScript.

## Customization

The component exposes CSS custom properties that can be modified to match your design.

Example variables:

```css
:root {
  --accent-color: #00f5ff;
  --secondary-color: #ff2bd6;
  --modal-duration: 0.4s;
  --modal-scale: 0.9;
  --modal-radius: 14px;
}
```

These variables allow developers to customize colors, animation timing, scale, and appearance without changing the component structure.
