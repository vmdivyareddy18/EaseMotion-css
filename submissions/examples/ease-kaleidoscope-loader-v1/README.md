# Ease Kaleidoscope Loader

## Overview

Ease Kaleidoscope Loader is a geometric loading animation built with rotating triangular segments that create a colorful kaleidoscope effect using pure HTML and CSS.

---

## Features

- 🔷 Kaleidoscope-inspired animation
- 🎨 Eight colorful segments
- 🔄 Continuous rotation
- ✨ Morphing geometry
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Rotating kaleidoscope
- Morphing triangular slices
- Infinite loading animation

---

## Usage

```html
<div class="loader">
...
</div>
```

## Customization

- Segment colors
- Rotation speed
- Number of slices
- Loader size
- Morph duration

Compatible with modern browsers supporting CSS transforms, clip-path, and animations.
