# Ease High Five Animation

### What does this do?
Adds a pure-CSS micro-interaction animation featuring two structural vector hands that seamlessly slide into coordinates and collide to perform a high-five animation loop paired with expanding impact shockwaves when triggered.

### How is it used?
```html
<div class="high-five-wrapper-sap">
  <div class="high-five-stage-sap">
    <div class="hand-sap hand-left-sap"></div>
    <div class="hand-sap hand-right-sap"></div>
    <div class="burst-sap ripple-1-sap"></div>
    <div class="burst-sap ripple-2-sap"></div>
  </div>
  <span class="action-text-sap">HOVER TO HIGH FIVE</span>
</div>
```

### Why is it useful?
It introduces a modern, highly engaging gamified celebration asset to alternative form submissions or completion tracking states using fast hardware-accelerated transitions and zero JavaScript processing overhead.
