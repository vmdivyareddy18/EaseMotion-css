# Animated Reading Challenge Card

A modern reading challenge tracking component built using HTML and CSS.

## Features

- Animated progress bar
- Reading goal tracker
- Books completed statistics
- Reading streak display
- Achievement section
- Responsive design

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- Reading challenge apps
- Book communities
- Educational platforms
- Personal development dashboards
- Library management systems
