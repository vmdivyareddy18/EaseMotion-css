# Business Landing — User Research SaaS (`saas-landing-heatmap-replay-21714`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for User Research, Heatmap, and Session Replay tools (similar to Hotjar, FullStory, or LogRocket), using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A user research SaaS needs to look approachable, empathetic, and vibrant. This template uses a warm aesthetic (Hotjar-style) with rounded typography and high-contrast primary colors.

The layout includes:
- A hero section featuring an animated CSS-built browser mockup with a CSS-rendered heatmap and moving mouse cursor overlay.
- An integration ecosystem logo banner
- A 6-card platform feature grid
- A dark-mode privacy compliance section with badge cards
- Testimonial cards
- A session-volume based tiered pricing layout
- A large, focused CTA section
- A comprehensive footer

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth text entrances, and fading in the CSS heatmap blur layers sequentially)
- `.ease-slide-up` (For structural layout animations in pricing, features, and testimonials)
- `.ease-zoom-in` (For drawing attention to the primary mockups and badges)
- `.ease-bounce-in` (For primary trial CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For interactive buttons, feature cards, and the mockups)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes. The heatmap effect is built entirely using CSS radial gradients and blur filters.
