# Kubernetes Multi-Cluster Fleet Manager Component

## What does this do?

Provides a centralized interface for monitoring Kubernetes clusters, fleet health, upgrades, compliance status, and resource utilization across multiple environments.

## How is it used?

```html
<div class="cluster-card healthy">
    <h3>Production-EKS-US</h3>
</div>
```

## Why is it useful?

Organizations running Kubernetes at scale require centralized visibility into cluster operations. This component provides a reusable fleet management dashboard for monitoring health, upgrades, compliance, and capacity planning.
