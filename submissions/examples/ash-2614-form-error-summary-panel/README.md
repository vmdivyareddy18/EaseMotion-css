# Form Error Summary Panel

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern form validation pattern featuring a centralized error summary panel that displays multiple validation issues in one place.

## Features

- Error summary panel
- Validation issue list
- Error-highlighted fields
- Animated panel reveal
- Accessible visual hierarchy
- Responsive layout
- Glassmorphism design
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="error-summary">

  <ul class="error-list">
    <li>Email address is required.</li>
  </ul>

</div>
```

## Main Classes

```css
.error-summary
.error-header
.error-icon
.error-list
.error-input
.field
```
