# 🧱 Brick Breaker

A classic Brick Breaker game built with HTML, CSS, and JavaScript. Control a paddle to bounce a ball and break all the bricks at the top of the screen.

## 🎮 How to Play

- **Break all the bricks** by bouncing the ball off the paddle.
- **Control the paddle** using your mouse, touch, or arrow keys (← →).
- **Score points** for each brick you break.
- **Don't let the ball fall** – you have 3 lives.
- **Win** by clearing all bricks from the screen!

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `←` / `→` | Move paddle left/right |
| `Space` | Start the game |
| `R` | Reset the game |
| Mouse / Touch | Move paddle by dragging |

## ✨ Features

- **6 Rows of Colorful Bricks** – Each row has a different color
- **3 Lives** – You have 3 chances before game over
- **Score Tracking** – Points increase with each brick broken
- **High Score** – Saved locally in your browser
- **Progressive Difficulty** – Ball speed remains consistent for fair play
- **Responsive Design** – Works on desktop and mobile
- **Rich Visual Effects** – Gradients, shadows, glow effects, and animations
- **Keyboard Shortcuts** – Quick access to start and reset

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3 (Gradients, shadows, responsive design)
- Vanilla JavaScript (no dependencies)
