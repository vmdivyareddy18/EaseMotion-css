# LGBTQ+ Support Center Template
An inclusive, vibrant, and safe-feeling template for LGBTQ+ nonprofits, advocacy groups, and community support centers.

- **Design**: Uses a "Community-Pride" palette with high-energy violet/teal gradients, inclusive-pink accents, and welcoming, rounded structures to signify safety, joy, and collective strength.
- **Use Case**: Community centers, mental health nonprofits, and LGBTQ+ advocacy organizations.
