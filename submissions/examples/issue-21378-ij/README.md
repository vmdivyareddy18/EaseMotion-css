# Eagle Soaring

1. What does this do? A CSS-only eagle that soars in a wide arc with wings that glide and adjust.

2. How is it used? Place .eagle inside .scene. The body is a small ellipse; wings are a large curved shape that rotates and scales.

3. Why is it useful? Combines wide horizontal translation with wing adjustments for majestic soaring motion.
