# Scroll-Driven Animations

## What does this do?
This demo shows three native CSS scroll-driven animation variants: a fade-in card, a slide-up card and a fade-in + fade-out card.

## How is it used?
Open `demo.html` in a browser. The demo uses these classes:
- `.fade-demo`
- `.slide-demo`
- `.fade-in-out-demo`

## Why is it useful?
It fits EaseMotion CSS because it adds readable, motion-first behavior with no JavaScript, and it degrades safely in browsers that do not support `animation-timeline: scroll()`.
