# ease-kanban-card

Animated Kanban board card with drag-handle hint.

## Features

- Hover lift animation
- Drag handle hint animation
- Elevated shadow effect
- Responsive layout
- Pure HTML and CSS

## Acceptance Criteria Covered

### Card Lift

The card lifts upward on hover using:

- translateY()
- enhanced shadow

### Drag State Hint

The drag handle subtly animates to suggest draggable behavior.

### Common Use Cases

- Project management boards
- Task tracking apps
- Scrum boards
- Workflow dashboards
