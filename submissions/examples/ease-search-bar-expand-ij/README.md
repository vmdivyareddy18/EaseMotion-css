# Search Bar Expand

An expandable search bar that morphs from an icon into a full-width input field.

## Features

- Icon-to-input expansion with elastic easing
- Close button to collapse back
- Results dropdown with staggered entrance animation
- Focus-state border highlight

## CSS Custom Properties

| Variable | Default | Description |
|---|---|---|
| `--sbe-duration` | `0.35s` | Expand/collapse duration |
| `--sbe-bar-bg` | `#ffffff` | Input background |
| `--sbe-focus-color` | `#6366f1` | Focus border/accent color |
| `--sbe-icon-color` | `#6c6c80` | Search icon color |
| `--sbe-radius` | `12px` | Border radius |

## Usage

Click the search icon to expand the bar. Type to reveal results. Click the close button or blur to collapse.
