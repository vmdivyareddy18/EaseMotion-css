# Issue #22746 - Advanced: E-commerce - Perfume Fragrance

## What this submission adds

This submission adds a complete one-page luxury perfume e-commerce landing page for **Maison Velours** using only HTML, CSS, and existing EaseMotion CSS classes. The page is designed to open directly in the browser with no build step, JavaScript, external assets, or third-party dependencies.

## Required sections included

- Hero with inline SVG perfume bottle visual, scent mist, petals, reflections, glow, and CTA area
- Fragrance product grid with 4 perfume cards
- Fragrance notes storytelling section
- Cross-sell / layering rituals section
- Reviews / testimonials section
- Gift packaging section
- Footer

## EaseMotion classes used

- `ease-fade-in`
- `ease-slide-up`
- `ease-delay-100`
- `ease-delay-200`
- `ease-delay-300`
- `ease-delay-400`
- `ease-delay-500`
- `ease-hover-lift`

## How to open demo.html

Open `demo.html` directly in any modern browser. No server, package install, or build command is required.

## Responsive behavior

- Mobile-first layout with stacked content on smaller screens
- Product, review, and cross-sell cards expand into multi-column layouts on tablet and desktop
- Hero content and SVG visual shift into a two-column editorial layout on wider screens
- Navigation and CTAs wrap cleanly on narrow viewports

## Browser compatibility

Tested for direct-open compatibility in modern Chromium, Firefox, Safari, and Edge browsers. The page uses standard HTML/CSS features plus locally linked EaseMotion CSS classes.
