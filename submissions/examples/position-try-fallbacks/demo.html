position-try-fallbacks

CSS Anchor Positioning with automatic fallback positions

hover buttons to show tooltips
top fallback
Tooltip content
Uses position-try-fallbacks to flip when near edges.
near right edge
Edge-aware positioning
Automatically flips to left/right fallback.
near bottom edge
Viewport-safe
Falls back to top positioning when bottom overflows.
← resize or scroll to test fallbacks → 
/* CSS snippet */
.anchor-button {
  anchor-name: --anchor-el;
}

.tooltip {
  position: absolute;
  position-anchor: --anchor-el;
  position-try-fallbacks:
    --try-top, --try-bottom,
    --try-left, --try-right;
  bottom: anchor(top);
  left: anchor(center);
  translate: -50% -8px;
}
