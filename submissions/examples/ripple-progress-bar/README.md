# Ripple Progress Bar

A CSS-only animated progress bar with a smooth ripple effect.

## Features

- Pure CSS animation
- No JavaScript required
- Smooth ripple movement
- Responsive layout
- Accessible progress indicator

## Usage

```html
<div class="ease-ripple-progress">
  <div class="progress-track">
    <div class="progress-fill">
      <span class="ripple"></span>
    </div>
  </div>
</div>
