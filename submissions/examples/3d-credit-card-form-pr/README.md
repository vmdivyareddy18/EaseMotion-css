# CSS-Only 3D Credit Card Form

### What does this do?
Orchestrates a realistic three-dimensional card-flipping animation that automatically shows the front or back of a credit card mockup layout based on active form inputs without using JavaScript listeners.

### How is it used?
Wrap form inputs and the perspective container inside a shared context box utilizing `:focus-within` selectors to pass 3D rotation metrics down into adjacent `.credit-card-mesh` assets.

### Why is it useful?
It creates an incredibly engaging, interactive micro-experience for billing and payment configurations entirely in the standard rendering pipeline, boosting conversion with zero script overhead.
