# Business Landing — Data Pipeline SaaS (`saas-landing-data-pipeline-21727`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for Data Engineering / ELT tools (like Fivetran, Airbyte, or dbt), using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A Data Engineering SaaS landing page needs to project reliability, scale, and connectivity. This template uses a clean, light-mode aesthetic with monospace accents to appeal to data engineers and analysts.

The layout includes:
- A hero section featuring an animated CSS-built data pipeline diagram showing packets moving from source to destination
- A logo banner for data-forward customer companies
- A 3-column platform feature grid detailing ELT concepts (CDC, Scheduling, Transformations)
- An integration source/destination tag cloud visualization
- A volume-based (Monthly Active Rows) pricing layout
- A dedicated documentation CTA

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For text and grid entrances)
- `.ease-slide-up` (For structural layout animations)
- `.ease-slide-right` (Used innovatively with `animation-iteration-count: infinite` to simulate data packets flowing across a pipeline track)
- `.ease-zoom-in` / `.ease-bounce-in` (For primary mockups and CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For buttons, feature cards, diagram nodes, and tags)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes.
