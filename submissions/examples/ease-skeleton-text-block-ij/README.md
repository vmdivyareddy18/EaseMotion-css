# Skeleton Text Block

A loading skeleton placeholder with a circular avatar and multiple text lines. CSS handles the pulsing shimmer animation via `@keyframes` with staggered delays using `--sk-delay`.

## Features

- Circular avatar skeleton with pulse animation
- Multiple text line skeletons with variable widths
- Staggered animation delays via `--sk-delay` custom property
- Opacity and background color pulse
- Card layout with flex alignment

## Usage

Include `style.css` and structure with `.skeleton-avatar` and `.skeleton-line` elements. Use `--sk-width` for line widths and `--sk-delay` for staggered animation timing.
