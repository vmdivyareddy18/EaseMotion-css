# Crystallize Weather Widget

Resolves #41903.

### What does this do?
Adds a Crystallize Weather Widget component to EaseMotion CSS, inspired by Travel App design patterns. It presents current temperature, condition, humidity, and wind speed in a frosted glass-style card with a glowing crystalline icon accent. The widget entrance uses EaseMotion's `ease-scale-up-vertical` animation, and the weather icon uses `ease-focus-in` for a subtle reveal effect.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the styling. Swap in real weather data (location, temperature, condition, humidity, wind) to make it dynamic.

### Why is it useful?
Provides a polished, no-JS weather display component suited for travel and outdoor-focused apps, giving users an immediately readable glanceable weather summary with a distinctive frosted, crystalline aesthetic.
