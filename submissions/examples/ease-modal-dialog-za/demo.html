CSS Modal Dialog

Pure CSS modal dialog using checkbox hack with overlay, close button, and focus trap

Modal Demo

Click the button below to open a modal dialog. The checkbox hack toggles visibility without JavaScript.

Modal Title

This modal dialog is built entirely with CSS. It uses the checkbox hack to toggle visibility, with a smooth fade-in and scale animation.

Features include:

  • Overlay backdrop with blur
  • Smooth open/close animation
  • Close button and overlay click to close
  • Responsive sizing

Usage

<input type="checkbox" id="my-modal" class="md-toggle">
<div class="md-overlay">
  <label for="my-modal" class="md-overlay-close"></label>
  <div class="md-modal">
    <div class="md-head">
      <h3>Title</h3>
      <label for="my-modal" class="md-close">&times;</label>
    </div>
    <div class="md-body">Content here</div>
  </div>
</div>
