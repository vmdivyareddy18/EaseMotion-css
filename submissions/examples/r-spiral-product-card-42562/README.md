# Spiral Product Card

A modern **Spiral Product Card** component built using **pure HTML and CSS** for the EaseMotion CSS library. The component features a rotating spiral background animation, interactive hover effects, responsive layout, and accessibility-friendly structure without requiring JavaScript.

## ✨ Features

- 🌀 Spiral rotating background animation
- 🎨 Modern product card UI
- 🖱️ Smooth hover interactions
- 📱 Fully responsive design
- ♿ Accessible semantic HTML
- ⚡ Pure HTML & CSS (No JavaScript)
- 🎯 Lightweight and easy to customize

## 📂 Project Structure

```
spiral-product-card-richa/
├── demo.html
├── style.css
└── README.md
```

## 🚀 Usage

1. Open `demo.html` in any modern web browser.
2. Ensure `style.css` is in the same directory.
3. Hover over the card to view the spiral animation and interactive effects.

## 🎨 Customization

You can easily customize the component by editing the CSS variables in `style.css`.

```css
:root {
    --ease-primary: #6c63ff;
    --ease-secondary: #8f94fb;
    --ease-accent: #ff6b81;
    --ease-radius: 24px;
}
```

You can modify:

- Primary and secondary colors
- Accent color
- Border radius
- Shadow intensity
- Animation duration
- Card dimensions
- Typography
