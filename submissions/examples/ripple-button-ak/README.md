# Ripple Button

A modern gradient ripple button built using pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML and CSS (No JavaScript)
- CSS-only ripple effect triggered on hover
- Modern gradient button design
- Responsive layout
- Smooth hover animation
- Lightweight and reusable component

## Files

```
ripple-button-ak/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<button class="ripple-button">
    Click Me
</button>
```

The button can be customized by modifying the CSS properties in `style.css`.

## Preview

The component showcases a gradient button with a CSS-only ripple animation and smooth hover interactions.

## Why is it useful?

This lightweight, dependency-free component follows EaseMotion CSS's philosophy of creating simple, reusable, and visually engaging UI interactions.
