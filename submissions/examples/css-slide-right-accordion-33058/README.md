# CSS Slide-Right Accordion for Neumorphic Soft Layouts

## Overview

A pure HTML and CSS accordion featuring a neumorphic interface with a slide-right content animation. Built using semantic HTML elements for accessibility and responsive behavior.

---

## Features

- Pure HTML5 & CSS3
- Neumorphic soft UI
- Slide-right animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
