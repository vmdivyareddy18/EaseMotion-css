# Neon Border Trace Animation

A CSS-only neon border effect where a glowing line appears to travel around the edge of an element.

## Features

- Pure CSS
- Smooth looping animation
- Works on buttons and cards
- No JavaScript required

## Usage

```html
<button class="neon-trace">
  Neon Button
</button>

<div class="neon-trace card">
  Neon Card
</div>
```

The effect uses a rotating conic gradient to simulate a glowing border trace around the element.
