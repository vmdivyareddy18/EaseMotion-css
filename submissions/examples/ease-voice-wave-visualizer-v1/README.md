# Ease Voice Wave Visualizer

## Overview

Ease Voice Wave Visualizer is a reusable audio-inspired animation that simulates a live voice waveform using animated vertical bars. Built entirely with HTML and CSS.

---

## Features

- 🎤 Audio waveform animation
- 📊 Animated equalizer bars
- ✨ Soft glow effect
- 🔄 Infinite looping animation
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- 10 animated waveform bars
- Sequential motion
- Smooth looping animation
- Glowing visual style

---

## Usage

```html
<div class="visualizer">
...
</div>
```

---

## Customization

- Bar count
- Animation speed
- Glow color
- Bar width
- Maximum height

---

## Browser Support

Compatible with modern browsers supporting CSS animations and transforms.
