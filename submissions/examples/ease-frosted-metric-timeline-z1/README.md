# Ease Frosted Metric Timeline

## Overview

A modern animated timeline featuring frosted milestone cards, glowing indicators, and a gradient progress track built entirely with HTML and CSS.

## Features

- Frosted milestone cards
- Animated status indicators
- Gradient timeline
- Responsive Flexbox layout
- Lightweight implementation
- Pure HTML & CSS

## Folder Structure

```text

```

Open `demo.html` in any modern browser.
