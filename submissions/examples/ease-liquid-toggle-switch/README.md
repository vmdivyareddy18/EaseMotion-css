# Ease Liquid Toggle Switch

A minimal CSS-only liquid toggle switch with smooth transitions and a glowing thumb.

## Features

- Pure HTML & CSS
- Smooth toggle animation
- Responsive
- Lightweight
- Easy to customize

## Structure

```
ease-liquid-toggle-switch/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any browser.

## Why EaseMotion CSS

A compact, reusable animation component demonstrating smooth state transitions with zero dependencies.
