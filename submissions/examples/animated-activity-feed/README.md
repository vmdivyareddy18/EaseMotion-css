# Animated Activity Feed/Timeline

A clean, modern, and fully responsive **Activity Feed Component** perfect for social platforms, dashboards, and collaboration tools. This example demonstrates how to combine **EaseMotion CSS** for staggered item entrances, smooth hover effects, and interactive status indicators to create an engaging, real-time feed experience.

## 🚀 Features

- **Staggered Item Entrances**: Each activity item slides in from the left using `ease-fade-in-left` and `ease-delay-*` for a cascading load effect.
- **Live Indicator**: A pulsing "Live" badge using `ease-pulse` to indicate real-time updates.
- **Interactive Hover States**: Activity items shift right on hover with enhanced shadows and accent borders.
- **Type Indicators**: Color-coded badges showing the activity type (code, design, project, comment, task).
- **Action Buttons**: Like and comment buttons with `ease-hover-grow` for tactile feedback.
- **User Avatars**: Circular avatar placeholders with emoji icons.
- **Timestamp Display**: Relative time stamps (e.g., "2 minutes ago").
- **Load More Button**: Interactive button to load additional activities.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted layout and sizing.
