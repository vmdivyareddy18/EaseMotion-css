Isolated Sheet Content

When scrolling reaches the top or bottom limits of this independent scroll container, the browser natively attempts to pass the remaining velocity to the main background document wrapper.

Applying the .ease-overscroll-none property isolates this boundary layout completely. The velocity chain breaks right at the edge of this element.

This behavior is critical for building custom drawer elements, slide-out contextual sidebars, and blocking mobile browser scroll-to-refresh pull actions cleanly.
