# Caption Side Top

caption-side — Positions table captions above or below

## Features
- Uses `--ease-color-surface`
- Uses `--ease-color-neutral-200`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Caption Side Top</span>
</div>
```

## Why is it useful?

This submission demonstrates the caption side top CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
