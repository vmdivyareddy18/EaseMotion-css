# Timeline / Roadmap Component Demo

## What does this do?

Creates a CSS-only Timeline / Roadmap Component demo that visually displays events, milestones, project phases, product updates, or development progress in a chronological format.

## How is it used?

Add the timeline structure inside your HTML container:

```html
<div class="timeline">
  <div class="timeline-item">
    <div class="timeline-dot"></div>
    <div class="timeline-content">
      <h3>Phase 1</h3>
      <p>Project planning and research completed.</p>
    </div>
  </div>
</div>
```

## Why is it useful?

Timeline and roadmap layouts are widely used in modern websites and dashboards. Adding a dedicated demo showcases how EaseMotion CSS can build structured, visually appealing content sections without JavaScript. The component aligns with EaseMotion CSS's animation-first and composable philosophy, allowing developers to easily combine it with existing animation utilities for milestone reveals, scroll-triggered effects, and interactive storytelling experiences.

## Tech Stack

- HTML
- CSS (no frameworks, no JavaScript)

## Preview

Open `demo.html` directly in your browser to see the timeline effect.
