# Tada Click Neumorphic Modal

A responsive, pure CSS modal featuring a smooth Tada-Click entrance animation with a soft neumorphic design.

## Features

- Pure CSS (No JavaScript)
- Responsive layout
- Keyboard accessible
- Customizable CSS variables
- Neumorphic soft UI
- Smooth Tada animation

## CSS Variables

```css
--duration
--easing
--scale
```
