Simulated Subfolder Page (/playground/index.html)

The Bug: Using a relative path like href="index.html" makes the browser search inside the current /playground/ folder, causing a broken or dead-end redirect.

The Fix: Updating the attribute to use the absolute base site path (href="/EaseMotion-css/") ensures that clicking the logo always targets the main homepage layout, regardless of directory depth.
