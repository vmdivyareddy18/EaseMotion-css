# Sprouting Seed

Seed growing into plant with unfolding leaves, showing the complete life cycle from seed to mature plant.

## Features

- **Complete Growth Cycle** – Seed → Sprout → Seedling → Young Plant → Mature
- **Unfolding Leaves** – Leaves unfurl as the plant grows
- **Root System** – Visible roots spreading underground
- **Growth Stages** – 5 distinct growth stages with visual changes
- **4 Speed Settings** – Slow, Normal, Fast, Very Fast
- **Interactive Growth** – Click to grow the plant
- **Particle Effects** – Growth particles during development
- **Detailed Plant Structure** – Stem, leaves, roots, and seed
- **Soil Visualization** – Underground soil with root system
- **Keyboard Shortcuts** – G (grow), S (speed), R (reset), SPACE (grow)
- **FPS Counter** – Real-time frame rate display
- **Responsive** – Adapts to all screen sizes

## Technologies Used

- HTML5
- CSS3 (Animations, Gradients, Flexbox)
- Canvas API (2D context with gradients and shadows)
- Vanilla JavaScript (Growth simulation, Animation loop)

## Installation

## Usage

### Controls

| Action | Result |
|--------|--------|
| "Grow" | Grow the plant to next stage |
| "Speed: Normal" | Cycle through 4 speeds |
| "Reset" | Reset to seed |
| G | Grow |
| S | Cycle speeds |
| R | Reset |
| SPACE | Grow |
