# Identity Federation Dashboard Component

## What does this do?

Provides a centralized interface for monitoring identity providers, federation trusts, authentication activity, and access governance metrics.

## How is it used?

```html
<div class="provider-card healthy">
    <h3>Okta</h3>
</div>
```

## Why is it useful?

Modern enterprises rely on multiple identity providers and federation relationships. This component provides a reusable IAM dashboard for monitoring trust relationships, SSO health, authentication activity, and governance controls.
