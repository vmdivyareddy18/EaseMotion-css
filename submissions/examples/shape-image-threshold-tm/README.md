# Shape Image Threshold

shape-image-threshold — Sets alpha threshold for float shaping

## Features
- Uses `--ease-radius-full`
- Uses `--ease-color-bg`
- Uses `--ease-color-surface`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Shape Image Threshold</span>
</div>
```

## Why is it useful?

This submission demonstrates the shape image threshold CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
