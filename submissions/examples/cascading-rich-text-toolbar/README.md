# Cascading Rich Text Toolbar

A pure CSS animated rich text toolbar component for EaseMotion CSS.

## Features

- Pure CSS interaction
- Cascading dropdown animations
- Responsive design
- Accessible editable content area
- No JavaScript required


## Animation

The dropdown menus use smooth:

- opacity transition
- translate animation
- hover interaction


## Usage

Copy the HTML and CSS files into your project.
