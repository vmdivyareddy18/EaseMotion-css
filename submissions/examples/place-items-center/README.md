## Overview
Demonstrates the `place-items`, `place-content`, and `place-self` shorthand properties for single-property centering. Shows grid centering with `place-items: center`, flexbox centering with `place-content: center`, and individual item positioning with `place-self: center` inside a three-column grid.

- `demo.html` ‚Äî page with three centering demos (grid, flex, and place-self)
- `style.css` ‚Äî dark-themed styles comparing centering approaches

## Usage
Open `demo.html` to see how each shorthand property centers content. The note section explains the shorthand-to-longhand mappings for all three properties.
