# Neumorphism Slide-Right Modal

A pure CSS modal component with smooth slide-right animation using Neumorphism design.

---

## ✨ Features
- Pure HTML + CSS (No JavaScript)
- Slide-right animation
- Neumorphism UI style
- Fully responsive
- Lightweight and reusable

---

## ⚙️ How it works
Uses the CSS checkbox hack to toggle modal visibility and trigger animations.
