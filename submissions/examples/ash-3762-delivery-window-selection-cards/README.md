# Delivery Window Selection Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive delivery window selection component featuring interactive delivery slot cards for e-commerce, grocery, and logistics applications.

## Features

- Delivery window cards
- CSS-only radio selection
- Availability badges
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<label class="card">

    <input type="radio" name="delivery">

    <span class="indicator"></span>

    <div class="details">
        ...
    </div>

</label>
```

## Main Classes

```css

```
