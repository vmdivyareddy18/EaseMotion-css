# CSS Micro Shake Tooltip

Pure CSS animated tooltip utilizing a Micro Shake interaction designed for Product Catalog layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Micro Shake animation
- Product catalog aesthetics
- CSS custom properties
- Modern card layouts
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--tooltip
--duration
--ease
```
