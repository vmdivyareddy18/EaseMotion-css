# Nested Tab Accordion

A tab-based accordion with nested panel support.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--tab-bg` | `#2d2d44` | Tab bar background |
| `--content-bg` | `#252540` | Content panel background |
| `--active-color` | `#4ecdc4` | Active tab color |
| `--slide-duration` | `0.25s` | Transition speed |
