# CSS Neon Glow Modal

Pure CSS animated modal featuring a Neon Glow interaction designed for Cyberpunk Neon interfaces.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS custom properties
- Cyberpunk aesthetic
- Neon glow effects
- Glassmorphism styling
- Smooth 60fps animations

## Variables

```css
--duration
--ease
--primary
--secondary
--glow
```
