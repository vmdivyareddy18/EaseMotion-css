# ease-level-up-glow

A gamification-inspired animation that creates a level-up effect with a glowing expanding ring and a pulse animation on the element itself.

## Features

- Expanding glow ring
- Bright glowing effect
- Scale + opacity fade outward
- Brief scale pulse on the element
- Pure CSS animation

## Usage

```html
<div class="ease-level-up-glow">
  ⭐
</div>
```

## Fits EaseMotion CSS

This animation provides a playful achievement effect suitable for badges, rewards, and gamification interfaces.
