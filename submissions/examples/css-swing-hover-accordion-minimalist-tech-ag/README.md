# CSS Swing-Hover Accordion (Minimalist Tech)

A sophisticated, ultra-clean accordion component designed for Minimalist Tech layouts. It uses stark monochrome tones with a single primary accent color, thin precision borders, and a physically constrained "swing" animation on hover.

## Features
- **Constrained Swing Animation:** A crisp, professional 2D pendulum swing that adds interactivity without feeling loose or playful.
- **Minimalist Tech Aesthetic:** High contrast monochrome, ultra-thin 1px borders, and generous whitespace.
- **Pure CSS:** No JavaScript required for the accordion logic or animations.
- **Semantic HTML:** Built directly on top of `<details>` and `<summary>`.

## Usage
Add the `min-tech-accordion` class to your container. Each item should be a `<details>` element with the `min-tech-item` class, containing a `<summary>` with the `min-tech-header` class.

## Files
- `demo.html`: The interactive demonstration.
- `style.css`: The source CSS defining the tight swing animation and minimalist styles.
