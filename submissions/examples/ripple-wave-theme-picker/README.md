# Ripple Wave Theme Picker

A CSS-only animated theme picker component inspired by banking design patterns.

## Features

- Ripple wave animation effect
- Smooth hover interaction
- Modern theme selection UI
- Responsive design
- Pure CSS implementation
- No JavaScript required

## Usage

```html
<div class="theme-picker">

  <button class="theme theme-blue">
    Blue
  </button>

</div>
