# Word Break Normal

word-break — Controls how words break at line boundaries

## Features
- Uses `--ease-color-text`
- Uses `--ease-text-base`
- Uses `--ease-color-muted`
- Uses `--ease-color-bg`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Word Break Normal</span>
</div>
```

## Why is it useful?

This submission demonstrates the word break normal CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
