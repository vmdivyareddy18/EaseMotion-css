# Planetary Risk Intelligence Grid

A responsive **Planetary Risk Intelligence Grid** dashboard built entirely with **HTML** and **CSS**.

## Features

- Executive risk intelligence dashboard
- Responsive CSS Grid layout
- Planetary threat monitoring table
- Circular global risk indicator
- Analytics metric cards
- Forecast progress bars
- Timeline-based response actions
- Modern glassmorphism-inspired UI
- CSS-only implementation
- No JavaScript or external libraries

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in any modern browser.
