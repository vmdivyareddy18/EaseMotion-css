# 🌌 CSS Background Motion

A collection of beautiful, pure CSS animated backgrounds for modern web interfaces.

## 🚀 Features

### Background Styles (10 types)
1. **Gradient Flow** - Smooth flowing gradient animation
2. **Floating Particles** - Rising particle effects
3. **Wave Animation** - Ocean wave SVG animation
4. **Geometric Pattern** - Moving geometric shapes
5. **Aurora Effect** - Northern lights simulation
6. **Bubble Effect** - Rising bubble animation
7. **Starfield** - Twinkling star effect
8. **Mesh Gradient** - Animated blob gradients
9. **Diagonal Lines** - Moving diagonal pattern
10. **Pulse Rings** - Expanding ring animation

### Animation Features
- Smooth gradient transitions
- Particle floating effects
- SVG wave animations
- Geometric pattern movement
- Aurora color shifts
- Bubble rising effects
- Star twinkling
- Blob morphing
- Pattern scrolling
- Ring expansion

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All background animations
- `readme.md` - Documentation
