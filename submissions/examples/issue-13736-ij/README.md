# Ease Health Bar Damage

Closes #13736

## Description
A game-style health bar component with smooth width transition on damage/heal. A red damage flash overlay appears briefly on the depleted section when hit.

## Acceptance Criteria
- Health bar smoothly transitions width on damage and heal
- Red damage flash overlay appears briefly on each hit
- Health text reflects current value
- "Take Damage" and "Heal" buttons for interaction

## Files
- `style.css` — Dark theme with health bar and damage flash styles
- `demo.html` — Interactive health bar demo
