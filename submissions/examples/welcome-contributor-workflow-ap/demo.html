                Check Contributor History
                Leverages GitHub GraphQL or REST API to fetch previous merged PRs.
                Returns user previous counts.
              

              

                Post Personalised Welcome Comment
                Appends welcoming comments with personalized author handles.
                Triggers issues.createComment().
              

              

                Include Guide Links
                Injects links to Contribution guides, CODEOWNERS, and roadmap rules.
                Ensures links point to files inside main branch.
              

              

                Tag Contributor Label
                Appends label tag to active pull request details page.
                Triggers issues.addLabels().
              

            
          

        

      


      
      
Detailed Devops Guidelines

      

        

          
1. Check Contributors Merge Histories

          

            To prevent spamming returning builders, queries must filter out closed PRs that were not merged.
            Only count contributions that successfully closed into main branch.
          

        


        

          
2. GitHub Actions Secrets security

          

            The action requires access to write permissions to append comments and labels.
            Use GitHub default tokens (secrets.GITHUB_TOKEN) with restricted repository scopes.
