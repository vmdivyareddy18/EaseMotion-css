# Animated Liquid Submit Button

A creative, interactive submit button that features a rising "liquid" or "wave" animation upon hovering. This component is built purely with HTML and CSS, utilizing pseudo-elements and keyframe rotations to achieve the wave effect.

## Features
- Pure CSS implementation
- Custom CSS keyframe animations for endless liquid waves
- Dynamic hover state that fills the button
- Semantic markup

## Usage
Add the HTML structure to your document and include the accompanying `style.css`. The animation relies on the background color of the body to create the wave cutout effect. Update the background variables in the CSS if you place this component on a differently colored background.
