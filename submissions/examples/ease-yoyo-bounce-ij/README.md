# 🪀 Ease Yoyo Bounce

A CSS animation of a yoyo bouncing up and down on a string, with a spinning motion on the body.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.yoyo` element.

## Custom CSS Variables

| Variable            | Default  | Description                |
|---------------------|----------|----------------------------|
| `--yoyo-color`      | `#ff6b6b`| Color of the yoyo body     |
| `--string-color`    | `#ddd`   | Color of the yoyo string   |
| `--bounce-duration` | `0.6s`   | Duration of one bounce cycle|

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
