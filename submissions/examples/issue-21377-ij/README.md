# Dolphin Leaping

1. What does this do? A CSS-only dolphin that leaps from the water in an arc with tail flicks.

2. How is it used? Place .dolphin inside .scene. The body is a cyan ellipse; the tail is a triangle with rotation.

3. Why is it useful? Shows arc-based jumping motion with combined translate and rotate keyframes.
