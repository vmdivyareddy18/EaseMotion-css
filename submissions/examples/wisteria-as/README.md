# Wisteria

A pure CSS and HTML wisteria in full bloom, long violet racemes hanging from a pergola and swaying while a bee works the flowers.

## What it shows

- Each raceme built from a single element: a banded repeating linear gradient gives the tiers of florets, a repeating radial gradient adds the pea flower highlights
- Two masks composited with mask-composite: intersect, tapering the raceme to a point and fading its tail so it hangs naturally
- Racemes of different lengths swaying out of phase, each parking its rest angle in a custom property
- Falling petals and a bee crossing the bloom on a wandering path
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
