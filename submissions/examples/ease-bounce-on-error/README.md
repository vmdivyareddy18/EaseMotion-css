# Bounce on Error – Element shakes on invalid action

> A production-ready validation component that shakes or bounces elements when invalid actions occur, providing immediate visual feedback for errors. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Shake animation** – Elements shake on invalid input
- ✅ **3 intensity levels** – Low, Medium, High
- ✅ **Real-time validation** – Check as you type
- ✅ **Visual feedback** – Error/success states with icons
- ✅ **Form validation** – Email, password, confirm password
- ✅ **Keyboard shortcuts** – Ctrl+Enter (submit), Ctrl+R (reset), Ctrl+I (intensity)
- ✅ **Customizable** – Colors, duration, easing
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
