# Animated Date Picker / Calendar Widget

A clean, modern, and fully responsive **Date Picker / Calendar Widget** perfect for booking systems, scheduling apps, and forms. This example demonstrates how to combine **EaseMotion CSS** for smooth card entrances, bouncy date selections, and interactive hover states to create a polished calendar interface.

## 🚀 Features

- **Smooth Card Entrance**: The calendar card fades in smoothly using `ease-fade-in-down`.
- **Interactive Navigation**: Previous/Next month arrows use `ease-hover-grow` for satisfying tactile feedback.
- **Bouncy Date Selection**: When a date is clicked, it pops into view using `ease-bounce-in` for a playful confirmation.
- **Hover Effects**: Days smoothly change background color on hover for clear visual feedback.
- **Event Indicators**: Days with scheduled events display a small colored dot below the date number.
- **Today Highlight**: The current date is visually distinguished with accent-colored text.
- **Legend Footer**: Clear visual mapping for "Today" and "Has Event" indicators.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing and spacing.
