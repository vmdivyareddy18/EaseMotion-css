# Hotel Reservation Dashboard

A responsive **Hotel Reservation Dashboard** built using **HTML5** and **CSS3** to demonstrate hotel booking management, room occupancy monitoring, guest activities, and reservation workflow animations with EaseMotion CSS.

---

## Features

- Responsive dashboard layout
- Hotel overview cards
- Room occupancy tracker
- Room status panel
- Booking timeline
- Recent guest activity feed
- Notification center
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Rooms Booked
- Available Rooms
- Guests Checked In
- Revenue

### Hotel Widgets

- Room Occupancy Progress
- Room Status Panel
- Booking Timeline

### Activity

- Recent Guest Activity Feed
- Notifications Panel

### Quick Actions

- Book Room
- Check In Guest
- Generate Invoice

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations.
