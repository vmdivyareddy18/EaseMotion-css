# Gradient Slider Track (#36692)

Gradient slider track with color-preview in a dark-themed card layout.

## CSS Custom Properties

| Property           | Default     | Description                  |
|--------------------|-------------|------------------------------|
| `--track-height`   | `10px`      | Height of the slider track   |
| `--gradient-start` | `#ff6b6b`   | Start color of the gradient  |
| `--gradient-end`   | `#4ecdc4`   | End color of the gradient    |
| `--thumb-color`    | `#ffffff`   | Color of the slider thumb    |

## Interactive Controls

- Range slider for track height
- Color pickers for gradient start/end
- Color picker for thumb color
- Draggable slider with live hex color preview
