# Animated AI Prompt Card Component

A modern **Animated AI Prompt Card** built using pure HTML and CSS. This reusable UI component is designed for AI prompt libraries, prompt marketplaces, developer tools, SaaS platforms, and AI-powered applications.

The component features a glassmorphism-inspired layout, smooth hover animations, AI model badges, category tags, prompt statistics, preview section, and responsive design.

---

## ✨ Features

- Modern glassmorphism card design
- AI model badge
- Prompt title and description
- Prompt preview section
- Category tags
- Prompt usage statistics
- Copy Prompt button (UI only)
- Favorite button (UI only)
- Smooth hover animations
- Pulse animation for AI model badge
- Responsive layout
- Pure HTML and CSS

## 🚀 Usage

1. Open `demo.html` in your browser.
2. Ensure `style.css` is in the same directory.
3. Replace the sample prompt with your own content.
4. Customize colors, badges, and categories as needed.
