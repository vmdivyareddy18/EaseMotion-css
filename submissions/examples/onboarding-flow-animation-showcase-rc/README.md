# Onboarding Flow Animation Showcase

1. What does this do?

Provides a complete onboarding flow showcasing animated welcome screens, feature highlights, setup forms, progress tracking, and completion states.

2. How is it used?

Open `demo.html` and explore the onboarding sections and animations.

3. Why is it useful?

It demonstrates real-world onboarding patterns and combines multiple motion techniques into a practical user experience example.
