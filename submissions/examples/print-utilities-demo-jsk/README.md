# Print Utility Demo

## Description

This example demonstrates how CSS print utility classes can improve printed documents.

## Files

- demo.html
- style.css

## Features

- Hide unnecessary UI when printing.
- Display print-only content.
- Simple implementation using `@media print`.

## CSS Utilities

```css
.print-hidden
.print-visible
```

## Usage

Open `demo.html` in a browser.

Press **Ctrl + P** (or **Cmd + P** on macOS).

Observe that:

- Buttons disappear.
- Print-only message appears.
- Layout becomes printer-friendly.

## Benefits

- Better printed pages
- Cleaner layouts
- Beginner-friendly example
