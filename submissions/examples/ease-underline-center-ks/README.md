# Growing Underline from Center

## What does this do?
Animates a text underline expanding outwards from the center on hover.

## How is it used?
Add the class to your inline link or text wrapper:

```html
<a href="#" class="underline-center-link">Hover Center Underline</a>
```

## Why is it useful?
It offers a modern, elegant alternative to standard static underlines that draws attention smoothly to navigation and header links.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
