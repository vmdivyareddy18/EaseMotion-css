# Animated Countdown Timer

A sleek, modern, and fully responsive **Countdown Timer** perfect for product launches, flash sales, event registrations, and limited-time offers. This example demonstrates how to combine **EaseMotion CSS** for smooth number transitions, pulsing separators, and staggered unit entrances to create an engaging, urgency-driving UI component.

## 🚀 Features

- **Smooth Number Transitions**: When numbers change, they animate with a flip effect using `ease-flip` for a polished, dynamic feel.
- **Pulsing Separators**: The colon separators between time units continuously pulse using `ease-pulse` to draw attention and create visual rhythm.
- **Staggered Unit Entrances**: Each time unit (days, hours, minutes, seconds) fades in sequentially using `ease-fade-in-up` and `ease-delay-*`.
- **Gradient Accents**: Modern gradient styling on the badge, title, and CTA button for visual appeal.
- **Real-time Updates**: JavaScript updates the countdown every second with smooth animations.
- **Call-to-Action Button**: "Notify Me" button with hover lift effect using `ease-hover-grow`.
- **Glassmorphism Design**: Frosted glass card with backdrop blur and subtle borders.
- **Fully Responsive**: Adapts gracefully to mobile screens, stacking units vertically on small devices.
