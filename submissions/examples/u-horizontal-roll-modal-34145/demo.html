Tactile Keyboard Access

This component operates without JavaScript using standard URL target anchors. When a close target (backdrop or close key) is clicked, the URL target is redirected to the open button (e.g. href="#trigger-soft-light"). This collapses the overlay and returns tab focus straight back to the trigger button to preserve keyboard navigation flow.
