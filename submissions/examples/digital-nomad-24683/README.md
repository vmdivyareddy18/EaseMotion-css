# Digital Nomad Community Template
A social and utility-focused template designed for remote work hubs, co-living platforms, and digital nomad networks.

- **Design**: Uses a "Global-Office" palette with professional dark greys, connectivity-focused teals, and warm neutral tones to emphasize both work-focus and community-belonging.
- **Use Case**: Co-living booking sites, nomad networking platforms, and location-independent worker resources.
