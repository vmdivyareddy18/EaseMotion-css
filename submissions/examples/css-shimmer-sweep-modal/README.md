# CSS Shimmer Sweep Modal

A responsive modal component featuring a smooth shimmer sweep animation created using only CSS.

## Features

- Pure CSS
- No JavaScript required
- Responsive layout
- Keyboard accessible (`tabindex="0"`)
- Customizable using CSS variables
- Lightweight and reusable

## Folder Structure

```
css-shimmer-sweep-modal/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Customization

You can modify these CSS custom properties inside `:root`:

```css
--duration
--ease
--radius
--scale
```

These variables control the animation speed, easing, border radius, and modal scaling.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari (modern versions)

## Why EaseMotion CSS?

This example demonstrates a modern shimmer sweep animation built entirely with CSS, making it easy to integrate into interfaces without any JavaScript dependencies.
