# Shadow Pop Navigation Bar

A responsive Space-themed navigation bar featuring a smooth Shadow Pop hover animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Keyboard accessible
- Shadow Pop hover animation
- CSS custom properties
- Supports `prefers-reduced-motion`
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Customization

Modify the CSS variables in `:root` to adjust colors, shadows, border radius, and animation timing.
