# Side Sheet Inspector Panel

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A side-sheet inspector panel commonly used in dashboards, design tools, content management systems, and admin interfaces for editing selected item properties.

## Features

- Side sheet layout
- Inspector panel UI
- Editable property fields
- Responsive design
- Slide-in animation
- Glassmorphism styling
- Dashboard-friendly pattern
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<aside class="inspector-panel">

    <div class="panel-header">
        <h2>Inspector</h2>
    </div>

</aside>
```
