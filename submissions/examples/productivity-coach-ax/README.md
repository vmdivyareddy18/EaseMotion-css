# Peak Productivity – Coaching Landing Page

A complete, ready‑to‑copy landing page for a productivity coach built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – organized lifestyle messaging with a free clarity call CTA
- **Programs** – 1‑on‑1, group, and self‑paced courses
- **Methodology** – the PEAK System (Plan, Execute, Analyse, Keep Improving)
- **Testimonials** – real‑world results from clients
- **Pricing Packages** – three tiered plans
- **Free Clarity Call CTA** – bold blue call‑to‑action section
- **Newsletter Signup** – email capture form
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-blue-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`, `ease-input`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-blue-700`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
