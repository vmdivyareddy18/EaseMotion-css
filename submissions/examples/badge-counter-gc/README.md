# Animated Badge Counter UI Example

## 1. What does this do?

This submission adds a badge counter component that animates count changes with scale and shake effects. It includes multiple badge variants for notifications and status updates.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes buttons to increase and reset counts, and animated badge transitions. Copy the `.badge-counter` markup and CSS into your project to add attention-grabbing badge updates.

---

## 3. Why is it useful?

Animated badge counters help draw attention to updates without overwhelming the UI. The motion provides clear visual feedback for count changes and improves the perceived responsiveness of notifications.
