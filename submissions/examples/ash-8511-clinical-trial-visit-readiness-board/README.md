# Clinical Trial Visit Readiness Board

A modern Clinical Trial Visit Readiness Board built entirely with HTML and CSS.

## Overview

This example showcases a research-site operations dashboard for tracking participant visit preparation across multiple readiness stages. It helps clinical teams monitor scheduled visits, identify action items, manage delays, and confirm completed visit preparations using a Kanban-style workflow.

## Features

- Research-site operations dashboard
- Kanban-style readiness board
- Scheduled visit tracking
- Ready-for-visit monitoring
- Action-required workflow
- Delayed visit management
- Completed visit tracking
- Responsive layout
- Pure HTML & CSS (No JavaScript)
