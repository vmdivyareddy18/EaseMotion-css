# CSS Zoom-Out Accordion for E-Commerce Checkout Layouts

## Overview

This project demonstrates a responsive **Zoom-Out Accordion** designed for e-commerce checkout interfaces. Built entirely with HTML and CSS, the component uses native `<details>` and `<summary>` elements to provide an accessible accordion experience enhanced with subtle zoom-out animations and modern styling.

---

## Features

- Pure HTML5 & CSS3
- CSS-only zoom-out animation
- Responsive checkout FAQ layout
- Native accessible accordion
- Smooth expand/collapse interaction
- Hover zoom-out effect
- Rotating toggle icon
- CSS custom properties for easy customization
- No JavaScript required

## Animation

- Cards gently zoom out on hover.
- Opening an accordion triggers a smooth scale animation.
- Toggle icon rotates to indicate the expanded state.
- Soft shadows enhance depth without overwhelming the layout.

## Customization

Modify the variables in `:root` to quickly personalize the design.

```css
:root{
    --primary:#f97316;
    --secondary:#fff7ed;
    --radius:18px;
}
```
