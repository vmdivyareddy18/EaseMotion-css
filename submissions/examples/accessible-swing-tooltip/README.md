# Accessible Swing-Hover Tooltip (Pure CSS)

An educational, responsive tooltip component built with 100% hardware-accelerated motion profiles, tuned specifically to pass stringent accessibility (A11y) checks.

## Features
- **WCAG Double-A Compliant Contrast:** Follows color contrast metrics perfectly to keep all text elements readable.
- **Native Focus Routing:** Leverages standard semantic button interaction mapping alongside `:focus` selectors to support standard keyboard-only environments.
- **Prefers-Reduced-Motion Guard:** Intercepts operating system settings to kill high-vibrancy animations automatically for individuals dealing with vestibular visual disorders.
- **Custom properties customization:**
  - `--swing-duration`: Fine-tune overall physics runtime.
  - `--swing-angle-intensity`: Control the swing sway variance.
