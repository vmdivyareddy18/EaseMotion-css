# 🐸 Frogger

A classic Frogger arcade game built with HTML, CSS, and JavaScript. Guide your frog across busy roads and treacherous rivers to reach the finish line!

## 🎮 How to Play

1. Click **"Start Game"** or press **Space** to begin
2. Use **Arrow Keys** (↑ ↓ ← →) to move your frog
3. **Avoid cars and trucks** on the road
4. **Jump on logs and turtles** to cross the river
5. **Reach the finish line** at the top to score points
6. The game gets **harder** as you level up!

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `↑` | Move Up |
| `↓` | Move Down |
| `←` | Move Left |
| `→` | Move Right |
| `Space` | Start Game |
| `R` | Reset Game |

## ✨ Features

- 🐸 **Classic Frogger Gameplay** – Dodge traffic and cross rivers
- 🚗 **Dynamic Traffic** – Cars, trucks, and buses with varying speeds
- 🚣 **River Navigation** – Ride logs and turtles across the water
- 🐢 **Submerging Turtles** – Turtles occasionally dive underwater!
- 📈 **Progressive Difficulty** – Speed increases with each level
- 🎯 **Scoring System** – Points for reaching the finish line
- 🏆 **High Score** – Saved in browser local storage
- ❤️ **Lives System** – 3 lives to reach the top
- 🎨 **Beautiful Visuals** – Gradient backgrounds, glowing frog, detailed vehicles
- 📱 **Responsive Design** – Works on desktop and mobile

## 🎯 Scoring

- **+10 points** for reaching the finish line
- **+5 bonus points** per level
- **Level increases** after 5 successful crossings

## 📊 Level Progression

- More vehicles appear on the road
- Vehicles and logs move faster
- More river obstacles to navigate

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3 (Gradients, responsive design)
- Vanilla JavaScript
