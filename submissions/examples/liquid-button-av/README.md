# Liquid Button

## What does this do?

Creates a premium liquid-inspired call-to-action button with animated flowing waves and smooth hover interactions.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<button class="liquid-button">

    <span>Explore Now</span>

    <div class="liquid"></div>

</button>
```

---

## Why is it useful?

Animated call-to-action buttons attract user attention while maintaining a clean interface. This standalone component demonstrates creative CSS animations using layered pseudo-elements and gradient backgrounds without relying on JavaScript or external libraries.
