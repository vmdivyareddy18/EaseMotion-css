Line Height Utilities

CSS line-height utility classes governing vertical text leading. Ensures high-legibility layout structures for headings, body copies, and dense cards.

1. Typography Showcase

None .lh-none line-height: 1

This paragraph has a line-height value of 1. It is best suited for single-line headers or multi-line headings where tight vertical compactness is desired. However, it should be used with caution on long paragraphs, as the text lines can feel crowded and hard to track.

Tight .lh-tight line-height: 1.25

This paragraph has a line-height value of 1.25. It works beautifully for larger text components like subtitles, blockquotes, or heading elements that stretch across multiple lines, striking a clean balance between spacing and compact aesthetics.

Normal .lh-normal line-height: 1.5

This paragraph has a line-height value of 1.5. This is the standard leading size recommended for modern web reading layouts. It ensures excellent baseline tracking and reading speed for primary paragraph copy blocks across most viewports.

Relaxed .lh-relaxed line-height: 1.75

This paragraph has a line-height value of 1.75. Ideal for long-form reading platforms, blogs, and documentation panels. The extra vertical space gives the eyes room to breathe, reducing cognitive strain during deep reading.

Loose .lh-loose line-height: 2

This paragraph has a line-height value of 2. It represents a double-spaced baseline alignment. Highly effective for decorative captions, educational modules for visually impaired users, or large callout sections.

The Importance of Baseline Rhythm in Web Design

Published June 2026 • Typography Guide

When designing user interfaces, designers often prioritize grids, fonts, and colors, overlooking the subtle influence of line heights. Line spacing, or leading, directly influences baseline rhythm. If lines are too close together, readers struggle to distinguish letters, causing fatigue. Conversely, if lines are too far apart, the eye has trouble tracking from the end of one line to the beginning of the next.

Using structured, logical utility classes like .lh-normal and .lh-relaxed allows developer teams to enforce consistent reading proportions globally. It ensures layouts scale correctly relative to font-size updates without manually setting pixel leadings in custom stylesheets.
