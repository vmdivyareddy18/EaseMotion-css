# Scale Audio Player

A responsive **Scale Audio Player** inspired by food delivery applications. The component uses a subtle CSS scaling animation to highlight the player while maintaining a clean and modern interface.

## Features

- Pure HTML & CSS
- CSS scale animation
- Responsive layout
- Accessible audio controls
- No JavaScript required

## Folder Structure

```
scale-audio-player-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
