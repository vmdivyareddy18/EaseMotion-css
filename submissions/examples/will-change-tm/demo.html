Will Change

Hints to the browser that an element will be transformed, allowing optimization.

CSS Property: will-change

Variants Demo

will-change: auto Will Change variant
will-change: scroll-position Will Change variant
will-change: transform Will Change variant
will-change: contents Will Change variant
will-change: opacity Will Change variant
transform (hover me)
Hover for scale and rotate
opacity (hover me)
Hover for opacity fade
contents
Contents hint applied

Usage

<div class="will-change">Content here</div>
