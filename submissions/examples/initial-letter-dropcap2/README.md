# Initial Letter Drop Cap Demo

This example demonstrates the CSS `initial-letter` property for creating a drop cap effect.

## What is a Drop Cap?

A drop cap is a large decorative first letter that spans multiple lines of text, commonly used in books, magazines, and editorial layouts.

## Usage

```css
.dropcap::first-letter {
  -webkit-initial-letter: 3;
  initial-letter: 3;
}
```
