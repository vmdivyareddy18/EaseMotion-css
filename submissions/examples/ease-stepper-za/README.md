# CSS Step Stepper

## What does this do?
Multi-step progress stepper with horizontal, vertical, icon, and badge variants for wizards and onboarding flows.

## How is it used?
```html
<div class="st2-stepper"><div class="st2-step st2-active"><div class="st2-circle"><span>1</span></div><div class="st2-text"><strong>Step</strong><span>Description</span></div></div></div>
```

## Why is it useful?
Guides users through multi-step processes without JavaScript. Four layout variants. Highlighted active/completed states. Ideal for checkout flows, onboarding, and multi-page forms.
