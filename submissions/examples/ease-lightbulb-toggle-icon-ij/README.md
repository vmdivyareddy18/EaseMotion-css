# Lightbulb Toggle Icon

A lightbulb icon component that toggles between off and on states. The on state features a yellow glow with a pulsing animation and a visible filament inside the bulb.

## Usage

Include `style.css` and copy the HTML structure. The `.on` / `.off` class on `body` controls the bulb state.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--lbt-duration` | 0.4s | Transition duration |
| `--lbt-off-color` | #64748b | Bulb color when off |
| `--lbt-on-color` | #fbbf24 | Bulb color when on |
| `--lbt-glow-color` | rgba(251,191,36,0.4) | Glow effect color |
| `--lbt-bg` | #0f172a | Page background |

## Browser Support

Chrome, Firefox, Safari, Edge
