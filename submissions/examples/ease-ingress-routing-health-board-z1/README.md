# Ease Ingress Routing Health Board

## Overview

A responsive HTML and CSS component for monitoring Kubernetes Ingress routing health and configuration status.

## Features

- Ingress health overview
- Route update monitoring
- Backend warning detection
- Failed routing alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable networking dashboard for monitoring ingress routing, detecting configuration issues, and improving application availability across Kubernetes environments.
