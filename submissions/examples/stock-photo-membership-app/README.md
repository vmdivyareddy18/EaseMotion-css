# Stock Photo Membership Landing Page

A premium, copy-paste ready landing page for a stock photography subscription service built with EaseMotion CSS.

## Description

This project showcases a complete e-commerce landing page tailored for digital media memberships. It features a clean, high-contrast aesthetic designed to let photography stand out, enhanced by subtle, smooth animations from the EaseMotion library. The design is fully responsive and requires zero custom keyframes or external dependencies.

## Features Included

- **Hero Section**: Impactful headline, search bar, and an asymmetrical gallery grid.
- **Collection Categories**: Interactive grid cards with hover scaling and gradients.
- **Quality Assurance**: Minimalist three-column feature highlights.
- **Contributor Spotlight**: Creator cards with avatars and mini-photo galleries.
- **Membership Tiers**: Clear pricing cards comparing credit packs vs. pro subscription.
- **Reviews & Testimonials**: Simple, elegant quote section.
- **Free Trial CTA**: High-conversion bottom call-to-action block.
- **Responsive Design**: Adapts beautifully across mobile, tablet, and desktop views.
