# Parallax Image – Image shifts based on mouse/cursor

> A production-ready parallax image component that creates an immersive 3D effect by shifting the image based on mouse/cursor position. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Mouse-driven parallax** – Image shifts with cursor movement
- ✅ **Smooth interpolation** – Fluid, natural movement
- ✅ **3 intensity levels** – Low, Medium, High
- ✅ **Toggle effect** – Enable/disable parallax
- ✅ **Real-time feedback** – Position and intensity display
- ✅ **Keyboard shortcuts** – Space (toggle), R (reset), 1-3 (intensity)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
