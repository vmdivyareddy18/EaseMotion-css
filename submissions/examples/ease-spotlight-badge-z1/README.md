# Ease Spotlight Badge

## Overview

This example demonstrates a reusable CSS-only spotlight badge with a moving light sweep animation. It works well for highlighting important labels in user interfaces.

## Features

- Pure HTML and CSS
- Animated spotlight effect
- Responsive layout
- Multiple color variants
- Reusable utility classes
- No JavaScript required

## Folder Structure

```text
ease-spotlight-badge-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

1. Open the project folder.
2. Launch `demo.html` in a browser.
3. Customize colors and labels as needed.

## Browser Support

Compatible with all modern browsers supporting CSS animations.

## Why This Example

This component provides a lightweight visual emphasis utility suitable for dashboards, product listings, announcements, and marketing interfaces.
