# Video Production Company Landing Page

Closes #14900 — Full cinematic landing page for a video production agency
built entirely with EaseMotion CSS utility classes.

## Sections

| Section | Description |
|---------|-------------|
| Hero | Cinematic headline + showreel mockup with play button |
| Services | 6-card grid (commercial, corporate, social, motion, post, events) |
| Portfolio | 6-item video gallery with hover overlays and badges |
| Process | 5-step vertical timeline from brief to delivery |
| Clients | 8 trusted brand logo pills |
| Testimonials | 3 client review cards with star ratings |
| CTA | Get a quote form with email capture |

## EaseMotion Classes Used

| Class | Purpose |
|-------|---------|
| ease-fade-in | Hero badge and buttons entrance |
| ease-slide-up | Hero headline and paragraph |
| ease-zoom-in | Showreel mockup entrance |
| ease-on-scroll | All below-fold section elements |
| ease-delay-1 to 6 | Stagger timing for grids |
| ease-hover-lift | Cards and client logos |
| ease-hover-scale | Buttons and portfolio items |
| ease-hover-glow | Showreel gold glow on hover |

## Files
- demo.html - self-contained landing page
- style.css  - all EaseMotion utilities + page styles
- README.md  - this file

## Features
- Fully responsive (mobile, tablet, desktop)
- IntersectionObserver scroll-triggered animations
- No build step — opens directly in browser
- Realistic copy (no lorem ipsum)
- prefers-reduced-motion respected
