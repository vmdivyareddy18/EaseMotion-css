# Bounce Spinner

A modern **Bounce Spinner** component built using pure HTML and CSS.

The component features a glassmorphism-inspired loading card with animated bouncing dots that provide subtle visual feedback during loading states.

## Features

- Pure HTML & CSS
- No JavaScript
- Responsive
- Glassmorphism design
- Smooth bounce animation
- Lightweight
- Accessible
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<div class="bounce-spinner">
  <span class="dot ease-bounce"></span>

  <span class="dot ease-bounce"></span>

  <span class="dot ease-bounce"></span>
</div>
```

Include the provided `style.css` in your project.

## Accessibility

- Decorative loading indicator
- Responsive layout
- High-contrast colors
- Pure CSS implementation

## Why EaseMotion CSS?

The Bounce Spinner demonstrates how lightweight motion can enhance loading states while remaining reusable, responsive, and dependency-free. The subtle bounce animation provides visual feedback without distracting users.
