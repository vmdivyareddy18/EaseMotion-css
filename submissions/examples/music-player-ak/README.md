# Minimal Music Player UI

## What does this do?
A minimal music player UI with play/pause, next/prev track controls, animated progress bar, volume slider, and track information display.

## How is it used?
Open demo.html in a browser. Click play to start simulated playback with progress animation. Use next/prev to cycle through tracks.

## Why is this useful?
This provides a polished music player component for web applications. It demonstrates playback simulation, progress tracking, and audio control UI patterns.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the player.
