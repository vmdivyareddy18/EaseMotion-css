A digital glitch effect with horizontally displaced color channels and a moving scan line. Pseudo-elements create red and cyan channel offsets while a glowing horizontal bar sweeps down the screen. Customize via --glitch-color-1, --glitch-color-2, --glitch-duration, and --scan-duration.
