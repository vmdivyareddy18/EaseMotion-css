# CSS Stats Grid Component

## What does this do?
A complete statistics dashboard with primary metric cards showing numbers with trend indicators, secondary cards with progress bars, performance metrics with response time breakdown, timeline milestones, and geographic distribution circles.

## How is it used?
```html
<div class="sg-grid sg-grid-4">
  <div class="sg-card sg-card-primary">
    <div class="sg-card-num">12,480</div>
    <div class="sg-card-label">Active Users</div>
    <div class="sg-card-trend sg-trend-up">+12.5%</div>
  </div>
</div>
```

## Why is it useful?
Creates a professional statistics and metrics section for admin dashboards and analytics pages. Features four-card primary stats grid with colored left borders and trend indicators, three-card secondary grid with animated progress bars, two-card performance layout with metric breakdown rows, vertical timeline with colored dots and milestone markers, and distribution percentage circles. Fully responsive with grid breakpoints and CSS custom property theming.
