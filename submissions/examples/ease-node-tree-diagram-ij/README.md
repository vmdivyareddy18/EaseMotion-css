# Node Tree Diagram

A directory tree diagram with node connection animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--node-color` | `#4ecdc4` | Node fill color |
| `--line-color` | `#4a4a6a` | Connector line color |
| `--branch-color` | `#f72585` | Branch accent color |
| `--connect-duration` | `0.4s` | Node connection speed |
