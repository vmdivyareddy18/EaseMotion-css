# Global Digital Civilization Observatory

A global digital civilization observatory dashboard built with EaseMotion CSS tokens. Monitors internet users, mobile subscriptions, digital adoption, platform user bases, cybersecurity metrics, internet penetration by region, and digital milestones in a green-themed dark interface.

## Features

- **Stat cards** — internet users, mobile subscriptions, digital adoption rate, digital economy value
- **Digital adoption by sector** — e-commerce, digital banking, online education, telemedicine, remote work with color-coded bars
- **Platform user base** — 6 platforms (YouTube, WhatsApp, Instagram, TikTok, X, LinkedIn) with user counts and growth changes
- **Internet penetration** — 5 regions (North America, Europe, Asia-Pacific, Middle East, Africa) with coverage bars
- **Cybersecurity metrics** — attacks blocked, data breaches, breach cost, ransomware, phishing, zero-day exploits
- **Digital milestones** — key events with timestamps
- **Civilization overview** — countries monitored, connected devices, mobile coverage, avg. hours online, new users, active websites
- Responsive 12-column grid with green dark (default) and light mode support
- Reduced-motion compliant
