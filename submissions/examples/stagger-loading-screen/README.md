# Stagger Loading Screen

A clean glassmorphism-inspired loading screen featuring staggered animated dots built with **EaseMotion CSS** and pure CSS.

## Features

- ✨ Glassmorphism card
- 🔄 Staggered loading animation
- 🚀 Uses EaseMotion CSS animation utilities
- 📱 Responsive layout
- ♿ Accessible semantic HTML
- 🚫 No JavaScript required

## Folder Structure

```text
stagger-loading-screen-amna/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser after including EaseMotion CSS.

## EaseMotion Classes Used

- `ease-fade-in`
- `ease-slide-up`

## Customization

You can modify:

- Dot colors
- Animation speed
- Glass background
- Card size
- Loading text

using the CSS variables in `style.css`.

## Accessibility

- Semantic HTML
- Responsive design
- Keyboard-friendly
- Pure CSS implementation
