# Wishlist Heart

## What does this do?
A heart icon that fills with animation when clicked/liked, simulating the wishlist/favorite toggle interaction.

## How is it used?
Use the `.heart-like` class on a span containing the heart symbol. The active state (`.liked`) triggers a scale and color animation.

## Why is this useful?
Provides satisfying micro-interaction for wishlist/favorite toggles in e-commerce and content platforms, improving engagement.
