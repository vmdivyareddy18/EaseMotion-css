# Color Picker Swatch Row

## What does this do?
Adds `.ease-swatch-row` — a row of color swatches with the hex label visible on hover.

```html
<div class="ease-swatch-row">
  <button class="ease-swatch" style="--swatch: #6c63ff"></button>
  <button class="ease-swatch" style="--swatch: #ec4899"></button>
</div>
```

Open `demo.html` to see it in action.
