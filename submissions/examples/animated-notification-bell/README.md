# Animated Notification Bell with Dropdown Panel

A clean, modern, and fully responsive **Notification Bell** with an animated dropdown panel. This example demonstrates how to combine **EaseMotion CSS** for continuous icon pulsing, staggered list entrances, and smooth panel transitions to create an engaging notification system.

## 🚀 Features

- **Continuous Bell Animation**: Uses `ease-pulse` on the bell icon to draw attention to new notifications.
- **Animated Badge**: The notification count badge uses `ease-bounce-in` for a playful entrance effect.
- **Smooth Panel Dropdown**: The notification panel slides down using `ease-fade-in-down` for a polished reveal.
- **Staggered List Items**: Each notification fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a premium, cascading effect.
- **Visual Distinction**: Unread notifications have a distinct background color and a blue dot indicator.
- **Categorized Icons**: Different notification types (messages, mentions, tasks, updates, system) have unique colored icons.
- **Interactive Hover States**: Uses `ease-hover-grow` on buttons and items for satisfying tactile feedback.
- **Fully Responsive**: The panel adapts to full-width on mobile devices.
