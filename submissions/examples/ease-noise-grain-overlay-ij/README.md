# Noise Grain Overlay

A film grain / noise texture overlay with animated shift.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--overlay-opacity` | `0.15` | Grain opacity |
| `--grain-color` | `#000` | Grain tint color |
| `--blend-mode` | `overlay` | CSS mix-blend-mode |
| `--anim-speed` | `0.3s` | Animation rate factor |
