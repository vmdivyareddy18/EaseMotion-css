# Slot Reel — Slot Machine Spinning Reel

CSS-animated slot machine reel with bounce easing. Click "Spin" to retrigger the reel animation via a minimal JS handler.

## Files

- `demo.html` — reel window, symbols, spin button
- `style.css` — reel window mask, spin keyframe, button styles

## Usage

Open `demo.html` and click the Spin button to animate the reel.
