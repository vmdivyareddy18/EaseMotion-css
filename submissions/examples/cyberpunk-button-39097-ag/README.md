# Cyberpunk Glitch & Slicing Button

A cyberpunk action button demonstrating digital glitch slice transitions using clip-path overlays.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting cybernetic layouts, trigger button nodes, and headers.
2. **`style.css`**: Clip-path coordinate masks, glitch steps keyframes, shadow offsets, and neon variables.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Hover your cursor over the **Initiate Hack** button.
3. Verify that the button triggers a fast digital slice "glitch" transition effect with cyan/magenta offsets.
