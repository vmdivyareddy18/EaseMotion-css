# Spotlight QR Code Display — Dark Mode Design Patterns

A pure CSS QR code display component implementing a moving spotlight effect with dark mode aesthetics through layered gradients, declarative animations, and glassmorphism-inspired composition.

## Specifications

- **Spotlight Animation**: Utilizes CSS `@keyframes` with animated radial gradients to simulate a moving spotlight sweeping across the QR code display.
- **Declarative Interaction**: Implements continuous visual effects entirely through native CSS animations without JavaScript dependencies.
- **Motion Architecture**: Integrates EaseMotion CSS variables and utility classes for framework-consistent animation timing and presentation.
- **Dark Mode Styling**: Combines translucent surfaces, subtle shadows, backdrop blur, and high-contrast elements to create a modern dark interface.
- **Accessible Structure**: Uses semantic HTML with ARIA labels to ensure the QR display remains understandable for assistive technologies.
- **Responsive Composition**: Preserves sizing, spacing, and animation behavior across desktop and mobile viewports.
