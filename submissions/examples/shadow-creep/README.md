# Shadow Creep Animation Effect

A dark shadow creeping/spreading effect where a shadow gradually expands from below an element, creating a sense of looming presence under issue #14753.

## Design Concept

- **Atmospheric Visuals:** A dark mystery/horror themed card element (e.g. "The Obelisk") floats above a surface.
- **Creeping Shadow:** A soft shadow beneath the card gradually expands, darkens, and stretches outward, simulating a looming shadow creeping across the floor.
- **Pure CSS Motion:** Implemented using CSS keyframes animating scale, opacity, blur-radius, and box-shadow dimensions dynamically.

## Setup & Showcase

Open `demo.html` in your favorite browser to see the effect in action.
