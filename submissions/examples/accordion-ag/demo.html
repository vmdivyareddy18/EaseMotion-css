Performant Accordion

This accordion uses modern CSS grid-template-rows to animate smoothly between 0fr and 1fr without heavy layout repaints.

This is the content for Section 1. Animating the height of elements has historically been difficult and poorly performing. Using CSS Grid changes that.

By animating grid-template-rows on the wrapper element from 0fr to 1fr, the browser can perform the animation efficiently. We also set opacity for a smoother visual effect.

The accordion uses aria-expanded on the buttons to ensure that screen readers can accurately interpret the open or closed state of each section.
