# Enterprise Architecture Repository Explorer Component

## What does this do?

Provides a centralized interface for exploring business capabilities, application portfolios, technology domains, ownership structures, and system dependencies.

## How is it used?

```html
<div class="domain-card healthy">
    <h3>Customer Management</h3>
</div>
```

## Why is it useful?

Enterprise architecture teams need visibility into applications, business capabilities, and technology ecosystems. This component provides a reusable repository explorer interface that supports governance, dependency analysis, and architecture management.
