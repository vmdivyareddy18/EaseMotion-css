# Toast Stack SK

Animated stacked toast notifications for success, info, and error states.

## Usage

```html
<div class="toast-stack">
  <article class="toast toast--success">
    <span class="toast__icon">✓</span>
    <div class="toast__body">
      <p class="toast__title">Saved</p>
      <p class="toast__message">Your preferences were updated.</p>
    </div>
  </article>
</div>
```

Variants: `toast--success`, `toast--info`, `toast--error`.

## Why EaseMotion CSS

Pure CSS entrance animation with staggered delays — no JavaScript required.
