# Fill Stroke

fill & stroke — SVG element fill and stroke color utilities

## Features
- Uses `--ease-color-primary`
- Uses `--ease-color-success`
- Uses `--ease-color-danger`
- Uses `--ease-radius-full`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Fill Stroke</span>
</div>
```

## Why is it useful?

This submission demonstrates the fill stroke CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
