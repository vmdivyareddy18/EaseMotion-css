Introduction

EaseMotion is a modern CSS animation library built for performance and simplicity. It offers a wide range of pre-built components, transitions, and keyframe animations that make it easy to create engaging user interfaces.

Whether you are building a landing page, a dashboard, or a full-featured web application, EaseMotion provides the building blocks to bring your UI to life with smooth, performant animations.

Features

Explore the core features of EaseMotion including custom properties for easy theming, hardware-accelerated animations, accessibility support, and a modular component system designed for scalability.

Each component is self-contained with its own CSS custom properties, making customization straightforward. The library also respects prefers-reduced-motion for accessibility.

Usage

Getting started with EaseMotion is simple. Include the CSS file in your project, add the appropriate class names to your HTML elements, and customize via the :root custom properties.

The library works with any framework or vanilla HTML/CSS setup. No JavaScript dependencies are required for most components.

Examples

Browse through hundreds of example components ranging from accordions and buttons to complex UI patterns like modals, carousels, and animated charts.

Each example includes a live demo, source code, and documentation for easy integration into your own projects.
