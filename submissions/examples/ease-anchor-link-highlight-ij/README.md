# Anchor Link Highlight

Page with anchor links that highlight on scroll. Features smooth scrolling to sections, an active link indicator with underline animation, and a scroll progress bar.

## Features

- Active link highlights on scroll with color + border-bottom transition
- Smooth scroll to section on link click
- Scroll progress indicator bar
- Sections fade in on page load with staggered animation
- Customizable via CSS custom properties

## Usage

Add `alh-link` to anchor elements and `alh-section` to content sections. The JS updates the active link based on scroll position and animates the progress bar.
