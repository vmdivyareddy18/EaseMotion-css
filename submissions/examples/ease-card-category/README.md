# Ease Card Category

This demo implements a **category/tag card** with a colored label, hover lift, and smooth entrance animation.

## 🧠 How It Works
- Each card uses a CSS variable `--ease-category-color` for the label color.
- The card lifts slightly on hover for interactivity.
- A fade-in animation runs when the page loads.

## 🧩 Files
- `demo.html` — main demo with two sample cards.
- `style.css` — styling and animations.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Observe the colored category labels and hover effect.
3. Adjust `--ease-category-color` for different themes.

## 🧰 Technologies
- HTML5
- CSS3 (variables, transitions, keyframes)
- JavaScript (entrance animation)
