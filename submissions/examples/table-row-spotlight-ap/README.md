# Comparison Row Spotlight

A responsive feature comparison table with staggered row entrance animation and a subtle hover/focus spotlight for scanning pricing or product plan differences!

## What does this do?

Adds a beautiful, animated feature comparison table that highlights rows on hover and focus, with staggered entrance animations!

## How is it used?

Copy the HTML structure from demo.html and include style.css in your project!

## Why is this useful for EaseMotion CSS?

It fits EaseMotion CSS's human-readable, animation-first philosophy by improving dense comparison tables with purposeful motion and accessible focus feedback!

## Features

- 📱 Responsive design
- ⏱️ Staggered row entrance animation
- ✨ Hover and focus spotlight effect
- 🎨 Beautiful gradient header
- 🫙 Glass-effect styling
- ♿ Accessible and clear UI
