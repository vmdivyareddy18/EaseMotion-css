# Table Sort Arrows

Sortable table with animated arrow indicators and smooth row transitions.

## Features

- Click column headers to sort ascending/descending
- Animated arrow direction change with rotate transition
- Rows fade in/out on sort with staggered opacity
- Active column highlighted with accent color
- Alternating row stripes
- Custom properties for easy theming

## Custom Properties

| Property | Default | Description |
|---|---|---|
| `--tsa-duration` | 0.3s | Transition duration |
| `--tsa-header-bg` | #f8fafc | Table header background |
| `--tsa-sort-color` | #3b82f6 | Active sort column color |
| `--tsa-arrow-color` | #94a3b8 | Inactive arrow color |
| `--tsa-stripe-color` | #f1f5f9 | Alternating row stripe |

## Usage

Include `style.css` and copy the HTML structure. Table data is fully customizable.
