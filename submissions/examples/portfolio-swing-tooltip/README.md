# Creative Portfolio Swing Tooltip (Pure CSS)

An artistic, high-end editorial tooltip interaction module featuring highly expressive elastic pendulum swing physics, optimized explicitly for creative gallery indexes and modern artist portfolios.

## Features
- **Elastic Kinetic Physics:** Implements custom `cubic-bezier` anchors that generate organic overshoots and wind-sway settling movements.
- **Top-Right Hanging Pivot Frame:** Anchors shifts natively to the upper right quadrant via deliberate `transform-origin` adjustments.
- **Avant-Garde Architectural Styling:** Drops traditional rounded pill layouts in favor of clean corner framing and sleek, modern color gradients.
- **Parametric Configuration Controls:** Tuning constants are managed inside centralized custom root parameters:
  - `--swing-duration`: Structural runtime control.
  - `--swing-max-angle`: Sets the maximum displacement of the rotation arch.
