# Ease Storage Usage Meter

## Overview
A CSS-only storage usage component for cloud dashboards and admin panels.

## Features
- Animated progress bar
- Responsive card
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any modern browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A lightweight, reusable dashboard component with smooth animations and an easy-to-customize design.
