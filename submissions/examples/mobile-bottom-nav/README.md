# Mobile Bottom Navigation Bar

A fixed bottom navigation bar with four icon-labeled tabs: Home, Search, Favorites, and Profile. The active tab is highlighted in indigo with a small top indicator bar. Clicking a tab switches the active state via JavaScript.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click each tab to see the active state switch.

## Accessibility notes

Each tab is a `<button>` element for keyboard accessibility. The active state is conveyed by color and an indicator bar. Reduced motion disables the color transition.
