Frequently Asked Questions

Is EaseMotion CSS free to use?

Yes, the starter components are completely free under the MIT license. Pro and Enterprise plans unlock premium components, priority support, and custom themes.

Can I use it with my framework?

Absolutely. EaseMotion CSS is framework-agnostic. It works with React, Vue, Angular, Svelte, or plain HTML. Just import the CSS file and use the component classes.

How do I customize the theme?

All components use CSS custom properties. Override variables like --lpg-primary, --lpg-bg, and --lpg-radius in your stylesheet to create your own theme.

Is there TypeScript support?

While the library is CSS-only, we provide TypeScript definitions for component class names and CSS variable types for better IDE integration.

What browsers are supported?

All modern browsers including Chrome, Firefox, Safari, and Edge. IE11 is not supported. We target the last 2 versions of each major browser.
