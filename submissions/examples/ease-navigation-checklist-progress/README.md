# Ease Navigation Checklist Progress

A modern **CSS-only navigation checklist with a step progress indicator** built using HTML and CSS.

## Features

- Pure HTML & CSS
- CSS-only animation using `@keyframes`
- CSS Custom Properties for easy customization
- Animated active progress step
- Completed, active, and pending states
- Responsive design
- Hover effects
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
ease-navigation-checklist-progress/
├── demo.html
├── style.css
└── README.md
```

## Customization

Edit the CSS variables in `:root` to customize:

- Primary color
- Success color
- Background color
- Border radius
- Shadow
- Transition speed

## Browser Support

Compatible with all modern browsers.
