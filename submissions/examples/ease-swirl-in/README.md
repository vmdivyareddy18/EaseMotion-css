# Ease Swirl In

An entrance animation that simultaneously spins and scales up an element from 0 to 100% size, creating a swirling entrance effect.

## Usage

Simply add the `.ease-swirl-in` class to any element.

```html
<div class="ease-swirl-in">
    I am swirling in!
</div>
```

## Files
- `demo.html`: Interactive preview of the animation.
- `style.css`: The CSS implementation with keyframes for `swirlIn`.
