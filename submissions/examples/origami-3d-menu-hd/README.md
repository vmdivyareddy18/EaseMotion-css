# 3D Origami Folding Menu (origami-3d-menu-hd)

## What does this do?
An accordion-style dropdown menu where the items unfold with a beautiful 3D paper-folding effect and staggered delays.

## How is it used?
Wrap a button trigger and list elements in a container:
```html
<div class="origami-menu-container">
  <button class="origami-trigger">Explore Features</button>
  <ul class="origami-list">
    <li class="origami-item"><a class="origami-link" href="#">Link 1</a></li>
    <li class="origami-item"><a class="origami-link" href="#">Link 2</a></li>
  </ul>
</div>
```

## Why is it useful?
It provides a premium, highly tactile spatial animation for dropdown selectors and navigation menus that enhances spatial feedback using pure CSS transitions and 3D transforms.
