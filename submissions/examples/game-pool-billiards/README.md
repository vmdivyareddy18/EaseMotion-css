# 🎱 Pool / Billiards

A classic Pool / Billiards game built with HTML, CSS, and JavaScript. Aim, set power, and pocket balls in order with realistic physics-based collisions!

## 🎮 How to Play

1. Click **"Start Game"** to begin
2. **Aim** by clicking and dragging on the cue ball
3. **Set power** by dragging back (further = more power)
4. **Release** to shoot the cue ball
5. **Pocket balls** in any order (8-ball rules simplified)
6. **Score points** for each ball pocketed
7. **First to 7 points** wins!

## 🖱️ Controls

| Action | Input |
|--------|-------|
| Aim/Shoot | Click and drag on cue ball |
| Shoot | Release mouse/touch |
| Start Game | Click Start or press `Space` |
| Reset Game | Press `R` |
| Toggle Mode | Press `M` |

## ✨ Features

- 🎱 **Realistic Physics** – Ball-ball and ball-cushion collisions
- 👥 **2-Player Mode** – Take turns with a friend

- 🏆 **Best Score** – Saved in browser local storage

- 📱 **Responsive Design** – Works on desktop and mobile
- 🖱️ **Touch Support** – Play on mobile devices

## 🎯 Scoring

| Action | Points |
|--------|--------|
| Pocket a Ball | +1 |
| Win Game | +7 total |

## 📊 Game Rules

- **2-Player Mode**: Players alternate turns
- **Foul**: Cue ball pocketed = turn switches
- **Win**: First player to pocket 7 balls wins
- **Best Score**: Highest score achieved in a single game
