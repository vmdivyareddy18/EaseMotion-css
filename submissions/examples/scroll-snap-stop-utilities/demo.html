Scroll Snap Stop Utility Classes

Control whether a fast swipe/flick can skip past intermediate snap points via scroll-snap-stop. Try flicking quickly through both rows below on a touch device or trackpad.

💡 Note: No scroll-snap-stop utility exists today — only scroll-snap-type (.ease-snap-x/.ease-snap-y) and scroll-snap-align (.ease-snap-center/-start/-end) in core/utilities.css.

Common use cases

  • Image carousels/galleries — leave default (.ease-snap-stop-normal)
  • Onboarding flows, story viewers, step wizards — .ease-snap-stop-always
  • Any UI where skipping a step would break context or comprehension
