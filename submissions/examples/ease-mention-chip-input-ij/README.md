# ease-mention-chip-input (#36811)

Mention chip input with pop-in tag animation and suggestion buttons.

## CSS Custom Properties

| Variable         | Default    | Description                   |
|------------------|------------|-------------------------------|
| `--input-bg`     | `#1a1a2e`  | Card / input area background  |
| `--chip-bg`      | `#3b3b6b`  | Chip tag background           |
| `--chip-color`   | `#c6b8ff`  | Chip text color               |
| `--pop-duration` | `0.3s`     | Pop-in / removal animation    |

## Features

- Type a name + Enter to create a chip
- Click suggestion buttons to add chips
- Pop-in scale animation on chip add
- Smooth scale-out on chip remove (× button)
- Dark theme card layout
