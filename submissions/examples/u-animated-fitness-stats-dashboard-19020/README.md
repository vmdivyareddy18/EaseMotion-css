# Animated Fitness Stats Dashboard

A modern fitness dashboard featuring animated stat cards, goal tracking, progress bars, achievement badges, and responsive layouts.

## Features

- Fitness statistics cards
- Animated counters
- Goal progress bars
- Achievement badges
- Hover interactions
- Responsive design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Fitness metrics
- Goal percentages
- Badge labels
- Colors
- Animations

## Use Cases

- Fitness apps
- Wellness platforms
- Health dashboards
- Activity trackers
- Workout planners
