Backdrop Filter Saturate

CSS filter: saturate() and backdrop-filter: saturate() for vibrant, muted, and desaturated UI patterns

How it works: filter: saturate(N%) adjusts color intensity of images. backdrop-filter: saturate(N%) adjusts saturation of content behind an element. Values below 100% desaturate; above 100% intensify vibrancy. Combine with blur() for glass effects.
