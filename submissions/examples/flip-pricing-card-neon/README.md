# CSS-only 3D Flip Pricing Card with Neon Hover Glow

A modern pricing card that flips in 3D on hover using only HTML and CSS. The front side presents the pricing plan, while the back side reveals additional features and a call-to-action button.

## Features

- Pure HTML & CSS
- 3D flip animation
- CSS perspective and transforms
- Neon glow hover effect
- Responsive layout
- No JavaScript required

## Folder Structure

```
flip-pricing-card-neon/
├── demo.html
├── style.css
└── README.md
```

## How to Use

1. Open `demo.html` in any modern browser.
2. Hover over the pricing card.
3. The card flips smoothly to reveal additional information.

## Customization

You can easily modify:

- Card colors
- Glow color
- Flip duration
- Pricing information
- Features list
- Button text

## Technologies Used

- HTML5
- CSS3
- CSS 3D Transforms
- CSS Transitions
- Flexbox

## Browser Support

Works in all modern browsers that support:

- CSS3 Transforms
- CSS Transitions
- Perspective
- Backface Visibility
