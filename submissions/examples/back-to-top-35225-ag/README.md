# Animated Back to Top Button

A floating action button demonstrating scale transition reveals on page scroll.

---

## What is Included

1. **`demo.html`**: A scrolling viewport layout containing multiple dummy text blocks and the floating action button.
2. **`style.css`**: Page scroll rules, floating dimensions, visibility classes, scale transitions, and hover scale increases.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Scroll down the page past 200px.
3. Verify the floating arrow button scales up smoothly in the bottom right corner.
4. Click the button — confirm the window scrolls back to the top and the button disappears.
