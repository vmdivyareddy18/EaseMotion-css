ease-page-transition-slide

Current page slides left while new page slides in from the right on navigation.

Bug: No Slide Transition

myapp.com/about
Page A — appears instantly, no animation.

Fix: Slide Page Transition

Click "Navigate →" to trigger the slide transition between pages.

myapp.com/home

🏠 Home Page

Welcome! You are on the home page.

📋 About Page

This is the about page, slid in from the right.

CSS Implementation Reference

/* ease-page-transition-slide */
@view-transition {
    navigation: auto;
}

::view-transition-old(root) {
    animation: slideOutLeft 0.4s ease-in-out;
}

::view-transition-new(root) {
    animation: slideInRight 0.4s ease-in-out;
}

@keyframes slideOutLeft {
    from { transform: translateX(0); }
    to   { transform: translateX(-100%); }
}

@keyframes slideInRight {
    from { transform: translateX(100%); }
    to   { transform: translateX(0); }
}
