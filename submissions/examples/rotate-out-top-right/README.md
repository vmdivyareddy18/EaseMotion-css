# Rotate Out Top Right

A lightweight rotate-out animation utility built using pure CSS.

## Features

- Rotates toward upper-right corner
- Uses transform-origin positioning
- Smooth fade-out effect
- Lightweight CSS animation
- Pure CSS implementation

## Usage

```html
<div class="rotate-box">
  ROTATE OUT
</div>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable exit animation
- Pure CSS motion effect
