# Global Artificial General Intelligence Command Center

Phase 271 dashboard — AGI command center for monitoring capability scores, intelligence evolution, model performance, governance, and operational metrics.

## Features

- **Stat cards** — 4 key metrics: AGI capability index, active models, governance score, training compute
- **Capability scores** — 6 AGI capabilities with gradient bars (reasoning, language, vision, planning, creativity, meta-learning)
- **Intelligence evolution** — 6 evolution tracks with active/training/scheduled statuses
- **Model performance** — 6 models with accuracy scores
- **Governance & ethics** — 6 compliance metrics (safety, bias, transparency, privacy, audit, red team)
- **Operational metrics** — 6 operational KPIs (cost, token rate, efficiency, uptime, PUE, carbon)
- **AGI overview grid** — 8-cell summary of command-wide stats

## Theme

Purple/neon AGI theme with embedded dark and light mode support via `prefers-color-scheme`. Reduced motion respected.
