# design-universe-v1

## 1. What does this do?

Provides a comprehensive, modern design system showcase ("EaseMotion Design Universe") containing design tokens, color palettes, typography scales, buttons, cards, forms, and micro-animations, all built with pure CSS.

## 2. How is it used?

This is a comprehensive system. You can apply the various utility and component classes to your HTML structure:

```html
<!-- Container for the universe -->
<div class="design-universe-container">
  
  <!-- Buttons -->
  <button class="du-btn du-btn-primary">Primary Button</button>
  <button class="du-btn du-btn-outline">Outline Button</button>

  <!-- Cards -->
  <div class="du-card du-card-glass">
    <h3 class="du-text-gradient">Glass Card</h3>
    <p>A beautiful glassmorphism card.</p>
  </div>

  <!-- Form Controls -->
  <div class="du-form-group">
    <label class="du-label">Username</label>
    <input type="text" class="du-input" placeholder="Enter username" />
  </div>

</div>
```

## 3. Why is it useful?

EaseMotion CSS empowers developers to create stunning interfaces quickly. This Design Universe acts as a master showcase of what is possible when combining structured design tokens with EaseMotion's philosophy of rich, declarative motion and premium aesthetics (such as glassmorphism, dynamic gradients, and smooth hover effects), without relying on external JavaScript frameworks. It serves as both a beautiful template and a rigorous stress-test of modern CSS capabilities.
