# Micro Shake Glassmorphism UI Modal

A responsive glassmorphism modal featuring a subtle Micro Shake interaction built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Glassmorphism interface
- Micro Shake animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--shake
--glass
--border
--primary
--secondary
--radius
```
