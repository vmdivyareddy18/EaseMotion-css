# Dynamic Analytics Dashboard

An intermediate responsive frontend dashboard built as a submission for GSSoC '26.

## 🚀 Features
- **Responsive Layout:** Built with CSS Flexbox and Grid to ensure it scales perfectly on desktop and tablet screens.
- **Modern UI Dark Theme:** Clean look inspired by modern developer ecosystems.
- **Interactive State Simulation:** Implements JavaScript logic to simulate real-time metrics updates dynamically.
