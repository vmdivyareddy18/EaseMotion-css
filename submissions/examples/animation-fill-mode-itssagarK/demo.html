Animation Fill Mode Utilities

See how elements behave before and after keyframe animations execute.

anim-fill-none (Default)

Snaps back to start style after completion.

anim-fill-forwards

Retains the style of the last keyframe.

anim-fill-backwards

Applies the style of the first keyframe during delay.

anim-fill-both

Applies first keyframe in delay, and retains last keyframe.
