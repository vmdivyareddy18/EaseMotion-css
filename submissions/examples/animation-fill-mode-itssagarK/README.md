# Animation Fill Mode Utilities

1. What does this do? Adds animation-fill-mode utility classes (`.anim-fill-none`, `.anim-fill-forwards`, `.anim-fill-backwards`, `.anim-fill-both`) to control CSS animation state before and after execution.
2. How is it used? Apply `.anim-fill-forwards` to retain the styles set by the last keyframe of an animation.
3. Why is it useful? It allows developers to create seamless transitions that do not snap back to their original state once the animation completes.
