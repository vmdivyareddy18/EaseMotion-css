# Animated Electric Vehicle Charging Dashboard

A responsive Electric Vehicle (EV) charging dashboard built using HTML and CSS.

## Features

- Animated battery charging indicator
- Charging progress bar
- Vehicle range and charging statistics
- Charging station status
- Responsive dashboard layout
- Smooth hover animations

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

## Use Cases

- EV mobile applications
- Charging station dashboards
- Automotive interfaces
- Smart mobility platforms
- IoT monitoring dashboards
