# View Switcher

## What does this do?
View Switcher component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"vs-primary\">View Switcher Primary</div>
```

## Why is it useful?
Provides a reusable view switcher UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
