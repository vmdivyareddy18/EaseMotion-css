# Ease macOS Dock

## Overview

Ease macOS Dock recreates the iconic dock interaction with magnifying icons, hover elevation, and smooth scaling effects using only HTML and CSS.

---

## Features

- 🍎 macOS-inspired dock
- 🔍 Icon magnification
- 🚀 Hover lift animation
- ✨ Glassmorphism container
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Seven dock icons
- Hover magnification
- Smooth scaling
- Glass dock container

---

## Usage

```html
<div class="dock">
    <div class="icon">🏠</div>
</div>
```

---

## Customization

- Icon size
- Hover scale
- Dock spacing
- Background color
- Border radius

---

## Browser Support

Compatible with all modern browsers supporting CSS transitions and backdrop filters.
