# CSS Fade-Out Accordion

A pure, modern, zero-JS animated Accordion component featuring smooth opacity transitions and subtle scale manipulations.

## Customization Parameters

You can fully customize the presentation parameters using standard CSS custom properties defined inside `:root`:

```css
:root {
  --accordion-timing: 0.35s; /* Animation duration */
  --accordion-easing: cubic-bezier(...); /* Transition timing curve */
  --accordion-scale-start: 0.98; /* Entrance scaling factor */
}
```
