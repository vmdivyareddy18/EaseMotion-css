# Tip Pool Calculator

A service-industry tip pool calculator with employee management, automatic distribution calculations, local persistence, and individual payout summaries.

## Features

* Add employees
* Edit employee details
* Delete employees
* Automatic tip pool distribution
* Hours-based payout calculation
* Pool participation percentage support
* INR (₹) currency formatting
* Local storage persistence
* Validation and error handling
* Summary dashboard cards
* Responsive layout

## Usage

1. Enter the total tip pool amount.
2. Set the pool participation percentage.
3. Add employees and their hours worked.
4. Click **Calculate Distribution**.
5. View payout summaries and contribution percentages.

Changes are automatically saved and restored using browser local storage.

## Validation

The calculator prevents:

* Empty employee names
* Missing hours worked
* Zero or negative hours
* Calculations without employees
