    How stagger works

    Each item shares the same entrance animation class but gets a progressively larger ease-delay-* value. Item 1 starts immediately; item 2 waits one base-delay step; item 3 waits two steps, and so on — creating a choreographed sequence without JavaScript timers. 

    <div class="ease-slide-up ease-delay-100">First</div>
<div class="ease-slide-up ease-delay-200">Second</div>
<div class="ease-slide-up ease-delay-300">Third</div>
