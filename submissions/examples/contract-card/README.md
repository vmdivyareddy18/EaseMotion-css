# Contract Card

**What does this do?**
Applies a subtle shrink effect to cards on hover, creating a pressed or contracted interaction state.

**How is it used?**

```html
<div class="contract-card">
  Content
</div>
```

**Why is it useful?**

The contract effect provides visual feedback without changing layout. It can be used to indicate selection, focus, or interaction while maintaining a clean and lightweight UI experience.
