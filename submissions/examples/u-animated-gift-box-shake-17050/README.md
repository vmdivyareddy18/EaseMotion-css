# Animated Gift Box Shake

## Overview

A reusable gift box animation featuring a playful shake effect suitable for rewards, promotions, loyalty programs, gamification systems, and celebration interfaces.

## Features

- CSS-only animation
- Gift box shake effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="gift-button">
  <span class="gift-icon">🎁</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
