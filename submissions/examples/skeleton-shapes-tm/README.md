# Component: ease-skeleton variants

Demonstrates the `.ease-skeleton-text`, `.ease-skeleton-circle`, and `.ease-skeleton-heading` variants for the skeleton loader component.

## Features

- Pure HTML & CSS
- Text skeleton lines with varying widths
- Circular skeleton for avatar placeholders
- Heading skeleton with extra width
- Shimmer animation using ease-kf-shimmer
- Dark mode compatible
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="ease-skeleton ease-skeleton-circle"></div>
<div class="ease-skeleton ease-skeleton-heading"></div>
<div class="ease-skeleton ease-skeleton-text"></div>
```

## Why is it useful?

Skeleton loaders are essential for perceived performance. The three most common shapes — text lines, circular avatars, and headings — should be built-in variants so developers don't need custom CSS for every loading state.
