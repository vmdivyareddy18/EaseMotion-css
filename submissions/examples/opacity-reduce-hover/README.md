# Opacity Reduce on Hover

Element dims with reduced opacity on hover — pure CSS.

## Features
- Smooth opacity transition from 1 to 0.6
- Pure CSS, no JavaScript

## Usage
```css
.dim-card { opacity: 1; transition: opacity 0.3s; }
.dim-card:hover { opacity: 0.6; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
