# Volume Control Slider

A volume slider with animated speaker icon that changes based on audio level.

## Features

- Slider with animated track fill and custom thumb
- Speaker icon morphs through mute/low/medium/high states
- Mute toggle with X icon animation
- Percentage display updates in real time
- Fully customizable via CSS custom properties

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--vcs-duration` | `0.3s` | Animation duration |
| `--vcs-track-bg` | `#334155` | Slider track background |
| `--vcs-fill-color` | `#6366f1` | Slider fill color |
| `--vcs-thumb-color` | `#fff` | Slider thumb color |
| `--vcs-icon-color` | `#e2e8f0` | Speaker icon color |

## Usage

Open `demo.html` in any modern browser. Drag the slider to adjust volume. Click Mute to toggle mute state.

## Browser Support

Chrome, Firefox, Safari, Edge — any modern browser supporting CSS custom properties and animations.
