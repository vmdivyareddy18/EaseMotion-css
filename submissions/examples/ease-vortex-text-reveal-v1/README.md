# Ease Vortex Text Reveal

## Overview

Ease Vortex Text Reveal is a typography animation where individual letters spiral inward from different directions before settling into their final positions. The effect combines rotation, translation, blur, and glow using only HTML and CSS.

---

## Features

- 🌪️ Vortex-inspired text reveal
- 🔤 Sequential letter animation
- ✨ Glow finish
- 💨 Blur-to-sharp transition
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Usage

```html
<h1 class="title">
    <span>E</span>
    <span>A</span>
</h1>
```
