# Ripple Radio Button

A clean, blog-inspired radio button component with a subtle ripple selection animation. It is built using pure HTML and CSS, making it lightweight, responsive, and easy to integrate into any project.

---

## Features

- 🎯 Pure HTML & CSS (No JavaScript required)
- 💧 Smooth ripple animation on selection
- 📰 Blog-inspired card layout
- 📱 Fully responsive design
- ♿ Accessible using semantic HTML elements
- 🎨 Easy to customize colors and spacing
- 🚀 Compatible with EaseMotion CSS utility classes

## Usage

```html
<label class="radio-item ease-hover-lift">

    <input
        type="radio"
        name="category"
        checked>

    <span class="custom-radio"></span>

    <span class="label-text">
        Technology
    </span>

</label>
```

## Why it fits EaseMotion CSS

This component follows the EaseMotion CSS philosophy by providing a lightweight, reusable, and animation-focused UI element. The ripple animation gives users clear visual feedback while keeping the implementation simple, accessible, and completely free of JavaScript. It works well for blog category selectors, settings pages, forms, surveys, and preference selection interfaces.
