# Object Fit Cover

object-fit: cover — Scales images to fill container without distortion

## Features
- Uses `--ease-radius-lg`
- Uses `--ease-shadow-md`
- Uses `--ease-color-surface`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Object Fit Cover</span>
</div>
```

## Why is it useful?

This submission demonstrates the object fit cover CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
