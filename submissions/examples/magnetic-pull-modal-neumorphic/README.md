# CSS Magnetic Pull Modal for Neumorphic Soft Layouts

A responsive pure CSS modal featuring a smooth **Magnetic Pull** opening animation with a neumorphic design.

## Features

- Pure HTML + CSS
- No JavaScript
- Responsive layout
- Keyboard accessible
- Uses CSS custom properties
- Neumorphic soft UI
- Smooth magnetic pull animation

## Files

```
demo.html
style.css
README.md
```

## CSS Variables

```css
:root{
    --duration:.6s;
    --ease:cubic-bezier(.22,1,.36,1);
    --scale:.75;
}
```

These variables allow customization of:

- Animation duration
- Animation easing
- Initial modal scale
