# Utility: glow-warning-token

Demonstrates the `--ease-glow-warning` design token for consistent warning-color glow effects.

## Features

- Pure HTML & CSS
- Uses the `--ease-glow-warning` CSS custom property token
- Multiple glow intensity variants
- Dark mode compatible
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="warning-card glow-warning">
  Warning: Please review your settings
</div>
```

## Why is it useful?

Warning actions need visual prominence. The `--ease-glow-warning` token standardizes warning glow effects across the framework, matching the pattern already established for success, danger, primary, secondary, and info glows.
