# Navigation Menu Bar

A menu bar with animated active indicator.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--nav-bg` | `#2d2d44` | Navigation background |
| `--link-color` | `#888` | Inactive link color |
| `--active-color` | `#4ecdc4` | Active indicator color |
| `--slide-duration` | `0.3s` | Indicator slide speed |
