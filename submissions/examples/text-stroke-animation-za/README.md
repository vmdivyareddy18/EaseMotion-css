# Animated Text Stroke Effect

A CSS text effect using -webkit-text-stroke with animations that transition stroke color and width. The outlined text pulses between different colors and stroke thicknesses.

## Files

- `demo.html` - Stroked heading with hover pause
- `style.css` - Text-stroke keyframes, color and width transitions

## Usage

Open `demo.html` to see the animated text stroke. Hover over the text to pause the animation.
