# Ease Orbit Loader

## Overview

A lightweight orbit-style loading animation built with pure HTML and CSS.

## Features

- Pure HTML & CSS
- Orbit animation
- Responsive
- Lightweight
- Easy to customize

## Folder Structure

```
ease-orbit-loader-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why This Example

This example demonstrates a reusable loading animation suitable for dashboards, portfolio sites, and application interfaces.
