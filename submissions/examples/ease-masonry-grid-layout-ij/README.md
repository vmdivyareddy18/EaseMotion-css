# Masonry Grid Layout (#36808)

A CSS columns-based masonry grid with staggered card entrance animations. Each card fades and slides up sequentially for a polished reveal.

## CSS Custom Properties

| Property           | Default   | Description                                |
| ------------------ | --------- | ------------------------------------------ |
| `--card-bg`        | `#3a3a3c` | Background color of each masonry card      |
| `--card-radius`    | `12px`    | Border radius of each card                 |
| `--stagger-delay`  | `0.05s`   | Per-card delay increment for entrance      |
| `--card-min-width` | `160px`   | Minimum column width (controls column count)|

## Interactive Controls

Use the variable panel to adjust card appearance, column sizing, and stagger timing. Resize the browser to see responsive column wrapping in action.
