# Animated Select Dropdown

A modern animated select dropdown component built using **HTML, CSS, and Vanilla JavaScript**. It features smooth expand/collapse animations, icons for each option, keyboard accessibility, and a responsive design.

## Features

- Smooth expand/collapse animation
- Icons for each dropdown option
- Responsive layout
- Keyboard accessible
- Click outside to close
- Selected option updates automatically
- Vanilla JavaScript (no frameworks)
- Lightweight and reusable

## Project Structure

```
animated-select-dropdown/
├── demo.html
├── style.css
├── script.js
└── README.md
```

## Usage

1. Open `demo.html` in your browser.
2. Click the dropdown button to expand the menu.
3. Select any option to update the selected value.
4. Use the keyboard:
   - **Enter** or **Space** to open/select.
   - **Escape** to close the dropdown.
