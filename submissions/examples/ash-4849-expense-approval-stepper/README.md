# Expense Approval Stepper

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive finance workflow component that visualizes each stage of an expense approval process using a CSS-only stepper.

## Features

- Multi-step approval workflow
- Completed, active, and pending states
- Summary information cards
- Responsive vertical layout on mobile
- Glassmorphism styling
- Hover animations
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="step current">

    <div class="circle">3</div>

    <div class="content">

        <h3>Finance Review</h3>

    </div>

</div>
```

## Main Classes

```css
.stepper
.step
.completed
.current
.pending
.circle
.line
.summary
.item
```
