# 18 — align-content-distribute

## Overview
Compares `space-between`, `space-around`, and `space-evenly` distribution keywords in both flexbox and grid contexts. Each keyword is shown with three numbered items so the spacing difference between items and edges is visually apparent.

## Files
- `demo.html` — page with flexbox and grid distribution demos
- `style.css` — dark-themed styles with all six distribution combinations

## Usage
Open `demo.html`. Compare how `space-between` places first/last items at the edges, `space-around` adds half-spacing on edges, and `space-evenly` gives equal spacing everywhere.
