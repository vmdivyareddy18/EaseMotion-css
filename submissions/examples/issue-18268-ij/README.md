# Dual Range Slider (Issue #18268)

A range slider with two handles. The track between the handles is filled with a highlight color. Uses two `<input type="range">` elements with JavaScript to sync the handles and fill the middle track.

## Usage

Open `demo.html` in a browser. Drag either handle to set the range.

## Files

- `demo.html` — HTML5 page with dual range slider and JS
- `style.css` — Dark theme styles with range slider styling
