# FAQ Accordion

A FAQ-specific accordion with smooth answer reveal using the CSS grid `grid-template-rows` trick, chevron icon rotation, and single-open mode.

## Usage

Include `style.css` in your project and structure your FAQ with `.faq-item` containing `.faq-question` and `.faq-answer` elements.

## Demo

Open `demo.html` in a browser to see the component in action.
