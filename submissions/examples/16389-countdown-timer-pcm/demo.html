📖 API Reference

AttributeUsed OnDescription
data-target.countdown / .countupISO date string (countdown) or number (count-up)
data-duration.countupAnimation duration in ms (default 2000)
data-prefix.countupText before the number (e.g. "$")
data-suffix.countupText after the number (e.g. "+" or "%")
data-decimals.countupNumber of decimal places (default 0)
data-observe.countup"true" to start on scroll into view
aria-label.countdownAccessible label for the timer
aria-live.countdown"polite" for live region announcements
