# Enabled & Disabled Form

## What does this do?
Demonstrates the `:enabled` and `:disabled` pseudo-classes for form controls, providing clear visual distinction between interactive and disabled states.

## How is it used?
Apply to any form input, button, or control that can be disabled:

    <input type="text" class="form-control" disabled>
    <button class="form-btn" disabled>Submit</button>

## Why is it useful?
Disabled controls need clear visual cues so users understand why they cannot interact. Proper styling prevents frustration and guides users toward completing required fields or conditions before proceeding.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
