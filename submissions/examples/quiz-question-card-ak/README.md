# Quiz Question Card

## What does this do?
An interactive quiz component with multiple-choice questions, instant correct/wrong feedback, a progress bar, and a final score summary.

## How is it used?
Open demo.html in a browser. Select an answer to receive instant feedback, then click "Next" to advance. The final screen shows the total score with a restart option.

## Why is this useful?
Demonstrates a reusable quiz card pattern suitable for educational apps, trivia games, or knowledge-check widgets.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
