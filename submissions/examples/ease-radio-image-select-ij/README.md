# Ease Radio Image Select (`#38924`)

A modern, highly polished CSS animation component that transforms basic HTML radio inputs into rich, accessible visual cards using image triggers.

## Key Features & Animations
* **Smooth Hover Expansion:** Cards shift upward and subtly expand inner images on hover utilizing clean `cubic-bezier` timing curves.
* **Selection State Highlights:** Clicking an item smoothly snaps an accent ring around the target border container and alters its background palette.
* **Pop-in Checkmarks:** A circular confirmation badge scales from `0` to `1` instantly on selection via an elegant elastic overshoot spring animation.
* **Fully Accessible:** Raw controls are masked securely to preserve default keyboard arrow navigation, structural form submissions, and screen-reader tree parsing.

## Directory Layout
* `demo.html` - Structural sandbox layout representing alternative UI options.
* `style.css` - Animation pipelines, layout variables, and conditional pseudo-state targets.

## How to Test
1. Launch `demo.html` inside your favorite modern browser window.
2. Hover over card blocks to trigger spatial elevations and zooming imagery frames.
3. Click any alternative card to observe smooth checkmark expansions and contextual ring switches.
