# Logistics Delivery Dashboard

A responsive **Logistics Delivery Dashboard** built using **HTML5** and **CSS3** that demonstrates shipment tracking, fleet monitoring, delivery management, and logistics analytics through a modern interactive dashboard.

---

## Features

- Responsive dashboard layout
- Logistics overview cards
- Route progress tracker
- Vehicle tracking panel
- Delivery timeline
- Recent delivery activity
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Active Deliveries
- Drivers Online
- Packages In Transit
- Delivery Success Rate

### Logistics Widgets

- Route Progress
- Vehicle Tracking
- Delivery Timeline

### Activity

- Recent Deliveries
- Notifications Panel

### Quick Actions

- Assign Driver
- Dispatch Shipment
- Export Report

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
