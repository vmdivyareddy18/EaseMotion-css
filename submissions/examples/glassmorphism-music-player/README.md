# 🎵 Glassmorphism Music Player

A premium **Glassmorphism Music Player UI** built with **HTML, CSS, and Vanilla JavaScript**. This project features a modern frosted-glass design, animated backgrounds, an interactive playlist, and smooth music player controls, making it a great UI showcase for modern web applications.

---

## ✨ Features

- 🎵 Modern Glassmorphism UI
- 💿 Rotating album artwork with vinyl effect
- ▶️ Play / Pause controls
- ⏮ Previous / ⏭ Next track navigation
- 🔀 Shuffle mode
- 🔁 Repeat mode
- ❤️ Favorite button
- 📋 Interactive playlist
- 🔍 Playlist search
- 📈 Animated progress bar
- 🔊 Volume slider
- 🎚 Animated equalizer
- 🌈 Animated gradient background
- 🎵 Floating music note animations
- 🌙 Light / Dark theme toggle
- 📢 Toast notifications
- ⌨ Keyboard shortcuts
- 📱 Fully responsive design
- ✨ Smooth hover and transition effects

---

## 🎮 Controls

| Action | Description |
|---------|-------------|
| ▶️ Play / Pause | Toggle music playback |
| ⏮ Previous | Play previous track |
| ⏭ Next | Play next track |
| 🔀 Shuffle | Toggle shuffle mode |
| 🔁 Repeat | Toggle repeat mode |
| ❤️ Favorite | Mark or unmark a song as favorite |
| 🔊 Volume | Adjust playback volume |
| 🔍 Search | Filter songs in the playlist |

---

## ⌨ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **Space** | Play / Pause |
| **←** | Previous Track |
| **→** | Next Track |
| **M** | Mute / Unmute |
