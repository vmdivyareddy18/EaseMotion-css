# Hornbill

A pure CSS and HTML great hornbill perched on a branch, tossing a fig up and catching it in its huge casqued bill.

## What it shows

- The oversized bill and casque built from clip-path wedges and a rounded box in warm yellows
- The lower mandible gaping open on the same cycle that flips the fig up, so the bird catches its food
- A pied tail from a two stop linear gradient and a scaly foot from a clip-path sawtooth
- An eye with a real lash arc via border-top, blinking on its own timer
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
