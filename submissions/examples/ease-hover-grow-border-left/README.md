# ease-hover-grow-border-left

A lightweight hover animation that reveals a colored left border by growing it from **0% to 100% height** using a `::before` pseudo-element.

---

## ✨ Features

- Pure CSS
- No JavaScript
- Smooth 0.25s transition
- Customizable accent color
- Great for navigation menus, sidebars, and list items

---

## 🚀 Usage

```html
<a href="#" class="ease-hover-grow-border-left">
    Home
</a>
```

---

## 🎨 CSS Variables

```css
:root {
    --ease-accent-border-color: #4f46e5;
}
```
