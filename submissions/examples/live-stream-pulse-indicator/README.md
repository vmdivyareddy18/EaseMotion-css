# Live Stream Pulse Indicator

## Overview
This feature simulates a live streaming UI using animated pulse and glow effects.

## ✨ Features
- Pulsing live indicator dot
- Glow ripple animation
- Animated UI card
- Hover interactions

## 🎯 Utility Classes
- ease-fade-in
- ease-slide-up
- ease-hover-grow

## 💡 Usage

```html
<span class="live-badge">
  <span class="dot"></span> LIVE
</span>
