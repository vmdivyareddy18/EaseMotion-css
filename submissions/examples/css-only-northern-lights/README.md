# CSS-Only Northern Lights Effect

## Overview

A pure CSS recreation of the northern lights featuring animated aurora ribbons, floating particles, atmospheric glow, and color-shifting scenes.

## Features

- CSS-only implementation
- Flowing aurora animations
- Floating star particles
- Atmospheric glow effects
- Multiple color variations
- Responsive layout
- No dependencies

## Tech Stack

- HTML5
- CSS3
  - Gradients
  - Blur filters
  - Keyframe animations
  - Blend modes

## Usage

1. Open `demo.html` in a browser.
2. Explore different northern light themes.
3. Customize gradient colors and animation timings as needed.

## Browser Support

- Chrome
- Edge
- Firefox
