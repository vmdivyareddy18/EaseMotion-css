# Ease Ripple Hover Effect

## Overview

Ease Ripple Hover Effect is a lightweight CSS-only animation that creates a smooth ripple expanding from the center of an element when the user hovers over it. The animation provides subtle visual feedback and improves user interaction without requiring JavaScript.

---

## Features

- Pure CSS implementation
- Modern ripple animation
- Lightweight and reusable
- Smooth hover interaction
- Responsive design
- Accessibility support using `prefers-reduced-motion`

---

## Use Cases

- Primary buttons
- Call-to-action buttons
- Navigation items
- Cards
- Product actions
- Interactive UI elements

---

## Example Usage

```html
<button class="ease-ripple-hover">
    Hover Me
</button>
```

---

## Animation Behavior

1. User hovers over the element.
2. A circular ripple appears at the center.
3. The ripple expands smoothly.
4. The ripple gradually fades away.
5. The button returns to its normal state.

---

## Accessibility

Users who prefer reduced motion automatically receive a static version using the `prefers-reduced-motion` media query.

---

## Browser Compatibility

Supports all modern browsers including:

- Chrome
- Edge
- Firefox
- Safari

---

## Why it fits EaseMotion CSS

This animation follows the EaseMotion CSS philosophy by providing a reusable, human-readable, animation-first utility that enhances user interaction while remaining lightweight and easy to integrate into any project.
