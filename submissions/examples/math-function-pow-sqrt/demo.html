CSS pow(), sqrt(), hypot()

Mathematical sizing, positioning, and scaling — entirely in CSS

pow() — Exponential Scale

Each square's size is calc(pow(2, n) * 1rem), creating an exponential sequence.

2¹=2
2²=4
2³=8
2⁴=16
