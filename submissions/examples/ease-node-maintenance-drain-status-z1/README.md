# Ease Node Maintenance Drain Status

## Overview

A responsive HTML and CSS dashboard component for tracking Kubernetes node maintenance operations and drain progress.

## Features

- Node maintenance overview
- Drain progress monitoring
- Cordoned node visibility
- Maintenance failure alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in a modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable infrastructure component for visualizing node maintenance workflows, helping DevOps teams coordinate upgrades, reduce downtime, and improve cluster reliability.
