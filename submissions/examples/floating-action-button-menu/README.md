# Floating Action Button (FAB) with Menu

## What it does

A floating action button that expands to show a menu of options when clicked. Pure CSS using checkbox hack.

## How to use it

<div class="ease-fab-container">
    <input type="checkbox" id="fabToggle" class="ease-fab-checkbox">
    <label for="fabToggle" class="ease-fab-button">+</label>
    <div class="ease-fab-menu">
        <a href="#" class="ease-fab-item">📁</a>
        <a href="#" class="ease-fab-item">❤️</a>
    </div>
</div>

## Variants

- ease-fab-left - Positions FAB on left side
- ease-fab-top - Positions FAB at top
- ease-fab-rainbow - Rainbow animated gradient

## Why it fits EaseMotion CSS

- Pure CSS, zero JavaScript
- Staggered menu item animations
- Multiple positioning options
- Smooth transform transitions
