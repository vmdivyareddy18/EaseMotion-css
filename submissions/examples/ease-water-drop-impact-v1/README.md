# Ease Water Drop Impact

## Overview

Ease Water Drop Impact is a realistic CSS animation where a water droplet falls, strikes the surface, generates expanding ripples, and creates a small splash effect.

---

## Features

- 💧 Falling water drop
- 🌊 Expanding ripple animation
- ✨ Splash particle effect
- 🔄 Infinite animation loop
- 📱 Responsive demo
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Falling droplet
- Ripple waves
- Splash particles
- Continuous animation

---

## Usage

```html
<div class="water">
...
</div>
```

---

## Customization

You can customize:

- Water color
- Ripple size
- Splash distance

- Drop size
