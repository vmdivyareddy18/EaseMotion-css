# Caret Color Utilities

Utility classes for customizing the text cursor (caret) color in form fields.

## Available Classes

| Class | CSS |
|---------|---------|
| `.caret-red` | `caret-color: #ef4444;` |
| `.caret-blue` | `caret-color: #3b82f6;` |
| `.caret-green` | `caret-color: #22c55e;` |
| `.caret-purple` | `caret-color: #8b5cf6;` |
| `.caret-black` | `caret-color: #000;` |

## Usage

```html
<input class="caret-red" />
<input class="caret-blue" />
<input class="caret-green" />
<input class="caret-purple" />
```

## What It Does

The `caret-color` property controls the color of the text insertion cursor inside editable elements such as inputs and textareas.

## Use Cases

- Branded form controls
- Themed input fields
- Custom UI kits
- Enhanced accessibility and visual feedback
