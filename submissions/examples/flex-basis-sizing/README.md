# 20 — flex-basis-sizing

## Overview
Explores `flex-basis` for initial sizing in flexbox, comparing it with `width` and demonstrating `flex` shorthand variations. Three sections cover: `flex-basis` vs `width`, growing/shrinking behavior with different `flex-grow` and `flex-shrink` values, and the four shorthand keywords (`initial`, `auto`, `none`, plus a custom `1 2 300px`).

## Files
- `demo.html` — page with three flex demos comparing sizing strategies
- `style.css` — dark-themed styles with inline `flex-basis` and `flex` shorthand usage

## Usage
Open `demo.html` and resize to see how items grow, shrink, or stay fixed based on their `flex` values. Items with `flex: none` never grow or shrink, while `flex: auto` fills available space.
