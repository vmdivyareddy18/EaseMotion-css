# Cyberpunk Swing-Hover Tooltip

A pure CSS tooltip featuring a smooth swing-hover animation inspired by cyberpunk neon interfaces.

## Features

- Pure CSS
- Responsive
- Keyboard Accessible
- Swing Hover Animation
- CSS Custom Properties
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` directly in any modern browser.

## CSS Variables

```css
--duration
--ease
--rotate
--tooltip-bg
--tooltip-color
--border
```

Customize these variables to change the appearance and animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## EaseMotion CSS

This component demonstrates an animation-first, reusable tooltip suitable for futuristic and cyberpunk UI designs while remaining lightweight and dependency-free.
