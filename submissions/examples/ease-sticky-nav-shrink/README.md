# ease-sticky-nav-shrink

## Description

A sticky navigation animation that smoothly shrinks the navigation bar and logo after scrolling beyond a threshold.

## Features

- Sticky navbar
- Smooth height transition
- Logo scaling
- Body class toggled using JavaScript
- Pure HTML/CSS/JavaScript

## Usage

Open demo.html in any browser.
