# Ease Email Delivery Status

## Overview
A CSS-only component for displaying email delivery status.

## Features
- Status badges
- Responsive layout
- Hover animation
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component suitable for email dashboards, CRM platforms, and marketing tools.
