`hyphens-none-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/hyphens-none-tm/style.css">
<div class="hyphens-none-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .hyphens-none: disables hyphenation

## Impact

Prevents unexpected hyphenation in headings
