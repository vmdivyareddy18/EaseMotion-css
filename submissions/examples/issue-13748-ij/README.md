# ease-freeze-frost

**Closes #13748**

A frost effect that spreads over an element using a `::after` pseudo-element with a crystalline gradient pattern and `clip-path` animation.

## Acceptance Criteria

- [ ] `::after` pseudo-element spreads with `clip-path` animation
- [ ] Crystalline blue/white gradient pattern
- [ ] Trigger button starts the freeze; button changes to melt
- [ ] Dark background with icy colour scheme

## Files

- `style.css` – Frost pseudo-element, clip-path spread transition
- `demo.html` – Frost container with freeze/melt trigger
- `README.md` – Issue documentation
