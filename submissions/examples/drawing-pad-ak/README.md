# Drawing Pad

## What does this do?
A canvas-based drawing pad with color picker, brush size control, eraser, clear, and save-as-image functionality.

## How is it used?
Open demo.html in a browser. Draw on the canvas using mouse or touch. Change colors with the picker, adjust brush size with the slider, toggle eraser mode, clear the canvas, or save your drawing as a PNG.

## Why is this useful?
This provides a lightweight drawing tool for note-taking, sketching, or annotation. It demonstrates canvas API usage, touch support, and tool state management.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to start drawing.
