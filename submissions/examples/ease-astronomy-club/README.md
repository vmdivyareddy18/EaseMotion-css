# Astronomy Club / Observatory Landing Page

A responsive, copy-paste ready landing page built with existing EaseMotion CSS utility classes.

## Overview

This demo page uses only vanilla HTML and EaseMotion CSS classes to create a modern astronomy club landing page with a deep navy space theme, responsive layout, and interactive visual structure.

## Folder structure

```
submissions/examples/ease-astronomy-club/
├── demo.html
└── README.md
```

## Responsive behavior

- Mobile-first layout stacking content vertically.
- Auto-fit responsive grids adjust cards for tablet and desktop widths.
- Buttons and sections scale naturally without extra CSS.

## Sections included

- Hero with headline, CTA buttons, and feature highlights
- Upcoming Events cards
- Educational Programs cards
- Observatory Visit details and map placeholder
- Membership pricing cards with a recommended plan
- Telescope Guide resource cards
- Newsletter subscription section
- Footer with about, quick links, programs, contact, and social links

## EaseMotion classes showcased

- Layout: `ease-container`, `ease-grid`, `ease-flex`, `ease-stack`, `ease-grid-auto`
- Cards: `ease-card`, `ease-card-shadow`, `ease-card-hover`, `ease-card-glow`, `ease-card-accent`, `ease-card-glass`, `ease-card-outlined`
- Buttons: `ease-btn`, `ease-btn-primary`, `ease-btn-outline`, `ease-btn-link`, `ease-btn-hover`, `ease-btn-lg`, `ease-btn-sm`, `ease-btn-icon`
- Typography and spacing: `ease-text-xl`, `ease-text-3xl`, `ease-text-4xl`, `ease-text-muted`, `ease-text-info`, `ease-font-semibold`, `ease-font-bold`, `ease-padding-8`, `ease-my-14`
- Animations: `ease-fade-in`, `ease-slide-in-from-bottom`

## Browser compatibility

The landing page works in modern browsers that support CSS custom properties, CSS grid, and Flexbox. Open `demo.html` directly in any up-to-date desktop or mobile browser.

## Usage

Open `demo.html` in a browser. No build or JavaScript is required.
