Z-Index Scale Utilities

Control the stacking order of overlapping positioned elements. Use standard positive scales for layers, negative values to place items behind backgrounds, and hover modifiers to bring hovered items to the front dynamically.

Success Criterion 2.4.3

Focus Order & Keyboard Focus

Reordering elements visually using z-index does not change the tab key order. Keyboard navigation follows the DOM tree structure. Ensure focusable elements in modal dialogs or high Z-index menus are reachable in a logical keyboard sequence, rather than forcing the focus indicator to disappear behind overlays.

Prerequisites

Positioning Rule

The z-index property only works on elements that have a position value other than static (such as relative, absolute, fixed, or sticky). Ensure positioned classes (e.g. relative) are applied alongside z-index scale classes.
