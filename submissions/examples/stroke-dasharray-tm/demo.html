SVG Stroke Dash Array

Creates dashed or dotted lines in SVG by specifying dash and gap patterns.

CSS Property: stroke-dasharray

Variants Demo

stroke-dasharray: none SVG Stroke Dash Array variant
stroke-dasharray: 5 5 SVG Stroke Dash Array variant
stroke-dasharray: 10 5 SVG Stroke Dash Array variant
stroke-dasharray: 20 5 5 5 SVG Stroke Dash Array variant
stroke-dasharray: 30 10 SVG Stroke Dash Array variant

Usage

<div class="stroke-dasharray">Content here</div>
