# Scroll-Triggered Fade-In Sections

Sections that fade in with a smooth upward motion as the user scrolls down. Uses the Intersection Observer API to detect when each section enters the viewport. Alternating light and dark backgrounds. Once visible, a section stays revealed.

## EaseMotion CSS classes used

None required — built with plain CSS transitions.

## How to run

Open `demo.html` in a browser and scroll down to see each section fade in.

## Accessibility notes

Content is fully visible in the DOM. The fade-in is purely decorative. Reduced motion disables the transition — sections are fully visible by default.
