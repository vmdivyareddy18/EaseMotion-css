Why is this structure keyboard accessible?

In pure CSS, tabbing out of a modal typically leaves the user stranded. This component uses a Focus-Restoring URL-Hash Redirect. By matching the backdrop and close button target anchors directly to the triggering button's ID (e.g. href="#trigger-dark"), clicking or pressing Enter on them will navigate back to the button, instantly closing the modal and restoring tab focus to the trigger.
