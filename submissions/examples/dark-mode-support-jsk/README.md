# Dark Mode UI Demo

## Description

This submission demonstrates a clean and modern dark mode user interface.

## Features

- Dark theme layout
- Responsive container
- Styled cards
- Dark input fields
- Hover effects for buttons

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in any web browser.

No installation or external dependencies are required.

## Why is this useful?

Dark mode is widely used in modern web applications. This demo provides a reusable design reference that can later be adapted into EaseMotion CSS components.
