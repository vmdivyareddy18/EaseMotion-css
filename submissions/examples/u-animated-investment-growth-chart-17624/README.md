# Animated Investment Growth Chart

## Overview

Animated Investment Growth Chart is a lightweight CSS animation that simulates portfolio and investment growth using an animated rising chart and appearing data points.

## Features

- CSS-only animation
- Rising chart effect
- Animated data points
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="growth-chart">
  <div class="chart-line"></div>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
