CSS Text Rendering Optimization

Control browser text rendering strategy with the `text-rendering` property. Optimize for speed, legibility, or geometric precision depending on content type.

text-rendering: auto

Default Browser Behavior

The browser makes the rendering decision automatically based on the content and context. This is the fallback — no explicit instruction given to the rendering engine. Best for general-purpose body text.

text-rendering: optimizeSpeed

Speed-Optimized Rendering

The browser prioritizes rendering speed over glyph quality. Disable ligatures and kerning to reduce rendering overhead. Best for large amounts of text where performance matters more than typography.

text-rendering: optimizeLegibility

Legibility-Optimized (Recommended for UI)

The browser enables kerning and ligatures for clearer, more readable text. Combined with -webkit-font-smoothing: antialiased, this produces crisp text on high-DPI displays. Recommended for all UI body text.

text-rendering: geometricPrecision

Geometric Precision

Prioritizes accurate glyph sizing over speed. Particularly useful for SVG text, rotated typography, and logo text where rounding errors would cause visual artifacts.
