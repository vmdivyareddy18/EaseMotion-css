# Mood Tracker Calendar

## What does this do?
A mood tracker calendar with monthly view, 5 emoji-based mood options, color-coded days, and localStorage persistence.

## How is it used?
Open demo.html in a browser. Click any date in the calendar, then select a mood emoji to log it. Navigate between months to see past moods.

## Why is this useful?
This provides a personal mood journal component for wellness tracking apps. It demonstrates calendar generation, emoji-based input, and date-keyed localStorage.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to track moods.
