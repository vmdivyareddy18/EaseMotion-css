# Animated Face ID Scan Frame

## Overview

Animated Face ID Scan Frame is a lightweight CSS animation that simulates facial recognition authentication using a scanning frame and moving scan line.

## Features

- CSS-only animation
- Face ID scan frame effect
- Moving scan line
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="faceid-scan">
  <span class="avatar">👤</span>
  <span class="scan-line"></span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
