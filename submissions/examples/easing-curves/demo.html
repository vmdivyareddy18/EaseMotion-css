Easing Curve Reference

Visual previews of all major easing functions – ball moves right on hover or automatically.

ease

Default – gentle start & end. Good for UI transitions.

ease-in

Starts slow, ends fast. Use for exit animations.

ease-out

Starts fast, ends slow. Use for entrance animations.

ease-in-out

Symmetrical slow‑fast‑slow. Good for looping or emphasis.

linear

Constant speed. Feels mechanical – use sparingly.

spring (cubic-bezier)

Bouncy settle. cubic-bezier(0.68, -0.55, 0.265, 1.55)

Hover over each track to see the ball move. All animations use pure CSS.
