# Cyberpunk Neon Glitch Text Effect

## Description
This feature provides a highly styled futuristic glitch text animation, perfect for titles, loading prompts, header highlights, and cyberpunk themed user interfaces.

## Usage
Apply the class `.cyber-glitch-text` to any text element, and supply the same text in the `data-text` attribute.
```html
<div class="cyber-glitch-container">
  <h1 class="cyber-glitch-text" data-text="NEURAL LINK">NEURAL LINK</h1>
</div>
```

## Why is it useful?
It offers a premium, lightweight CSS-only interactive glitch effect without requiring heavy JavaScript rendering engines. In addition, it respects the user's system preferences by disabling motion transitions automatically when `prefers-reduced-motion: reduce` is configured.
