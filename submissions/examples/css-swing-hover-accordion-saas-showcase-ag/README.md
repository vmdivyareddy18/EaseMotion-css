# CSS Swing-Hover Accordion (SaaS Showcase)

A conversion-focused accordion component designed for SaaS product showcase pages. Uses a branded card-based layout with a corporate swing animation and feature-benefit structure ideal for pricing pages, feature tours, and onboarding flows.

## Features
- **Branded Card Swing:** A subtle, professional top-hinge swing that feels corporate yet engaging — not playful, not stiff.
- **SaaS-Optimized Layout:** Each accordion section has a structured header with an icon slot, title, and badge area — perfect for feature tiers and pricing breakdowns.
- **Conversion-Ready:** Designed with CTA-friendly spacing and visual hierarchy that guides the user's eye.
- **Pure CSS:** No JavaScript needed. Uses `<details>` and `<summary>` for semantic markup.

## Usage
Add the `saas-accordion` class to your container. Each item should be a `<details>` element with `saas-item` class. Use `saas-badge` spans inside headers for tier labels.

## Files
- `demo.html`: Interactive preview showing a SaaS pricing feature breakdown.
- `style.css`: The source CSS with branded card styling and corporate swing animation.
