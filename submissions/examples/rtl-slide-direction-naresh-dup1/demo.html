RTL Slide Animation Fix

Ensuring horizontal animations adapt dynamically to Right-To-Left layout systems

Active Mode: LTR (Left-To-Right)

1. Legacy Hardcoded Absolute Slides

These elements use absolute coordinate keyframes (translateX(-100%) / translateX(100%)). Under RTL, they slide in from the wrong/opposite direction.

.ease-absolute-left (Slide Left)
Absolute Left
.ease-absolute-right (Slide Right)
Absolute Right

2. Refactored Logical RTL-Aware Slides

These elements use keyframes backed by CSS variables. When dir="rtl" is set on the container, the start/end offsets automatically swap.

.ease-logical-left (Slide Left -> Logical Start)
Logical Left
.ease-logical-right (Slide Right -> Logical End)
Logical Right
