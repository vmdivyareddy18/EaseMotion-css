# SaaS Showcase Draw-Border Line Accordion

A modern **Pure CSS Draw-Border Line Accordion** designed for **SaaS Showcase** interfaces. The component features a smooth border-drawing animation, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Draw-Border Line animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- SaaS-inspired FAQ accordion
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.4s;
    --easing:ease;
    --primary:#2563eb;
    --secondary:#60a5fa;
}
```

You can customize:

- Animation duration
- Animation easing
- Border colors
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
