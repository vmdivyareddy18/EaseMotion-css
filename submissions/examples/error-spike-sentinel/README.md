# Error Spike Sentinel

Error Spike Sentinel is a responsive reliability panel for monitoring sudden error-rate changes. It summarizes spike confidence, affected region, and recovery trend so incident owners can decide whether to escalate.

## Features

- Spike confidence summary
- Error, region, and recovery signal cards
- Distinct danger, warning, and stable states
- Responsive single-column layout

## Files

- `demo.html` contains the error spike sentinel markup.
- `style.css` contains the signal card layout and responsive styling.
