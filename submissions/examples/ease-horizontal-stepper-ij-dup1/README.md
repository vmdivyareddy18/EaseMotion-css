# Horizontal Stepper

A horizontal step progress indicator for multi-step forms and wizards.

## Features

- Connected step indicators with animated transitions
- Active step with pulsing glow animation
- Completed step with checkmark styling
- Gradient connecting lines between steps
- Animated content area with fade-in effect
- Previous/Next navigation buttons

## Usage

Include `style.css` and structure the stepper with `.step` elements containing `.step-indicator` and `.step-label`. Use `.active` and `.completed` classes to indicate progress.
