# Cursor Glow Countdown Timer

A retro-inspired countdown timer component featuring a glowing cursor, neon effects, and subtle CRT-style animations. Built entirely with HTML and CSS, it is lightweight, responsive, and requires no JavaScript.

## Features

* Pure HTML & CSS
* Blinking cursor glow animation
* Neon countdown display
* CRT scan-line and glow effects
* Responsive layout
* Easy to customize and integrate

## Usage

Include `style.css` in your project and use the HTML structure from `demo.html` to add the countdown timer to your page.

## Why EaseMotion CSS?

This component follows EaseMotion CSS's animation-first philosophy by providing a reusable, visually engaging UI element with smooth CSS-only animations, making it suitable for dashboards, landing pages, countdown sections, and futuristic interfaces.
