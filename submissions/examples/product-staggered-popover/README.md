# Product Catalog Staggered Entrance Popover

A pure CSS animated popover designed for product catalog interfaces with a smooth staggered entrance effect.

## Features

- Pure HTML and CSS
- Staggered entrance animation
- Product catalog UI style
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript


## Usage

Open `demo.html` directly in the browser.


## Customization

Animation timing can be changed using CSS variables.

```css
:root{
 --duration:.4s;
}
