# Profile Avatar Reveal

A profile preview card where the avatar ring expands and supporting details settle into view.

## Files

- `demo.html` - focusable profile card markup.
- `style.css` - avatar ring motion, detail reveal, focus-visible state, and reduced-motion support.

## Highlights

- Keeps the card compact while revealing detail on interaction.
- Uses matching hover and keyboard focus behavior.
- Includes reduced-motion handling.
