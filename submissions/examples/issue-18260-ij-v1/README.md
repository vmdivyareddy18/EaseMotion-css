# Interactive Star Rating (Issue #18260)

Five stars that fill with color on hover. Click to set a rating. Uses CSS transitions on color and transform. Configurable active and inactive colors.

## Usage

Open `demo.html` in a browser. Hover over stars to preview, click to rate.

## Files

- `demo.html` — HTML5 page with interactive star rating
- `style.css` — Dark theme styles with CSS transitions
- `README.md` — This file
