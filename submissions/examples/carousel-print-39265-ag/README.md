# Carousel Print Media Query Optimization

A carousel component layout demonstrating clean page-break print overrides.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting horizontal carousel slides, descriptions, and header titles.
2. **`style.css`**: Media print overrides, grid layout flatteners, flex alignments, and font color overrides.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Verify that you can scroll horizontally through the slides.
3. Open the **Print Preview** (Ctrl + P) in your browser.
4. Verify that in the print layout, all three slides are displayed flat vertically, ensuring complete readability.
