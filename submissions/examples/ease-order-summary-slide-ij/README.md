# Order Summary Slide

An expandable order summary with slide-down details.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--summary-bg` | `#2d2d44` | Card background |
| `--item-bg` | `#3a3a5a` | Line item background |
| `--expand-color` | `#4ecdc4` | Expand button accent |
| `--slide-duration` | `0.35s` | Slide animation speed |
