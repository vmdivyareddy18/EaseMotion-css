# Incident Escalation Path Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern incident response workflow component that visualizes escalation paths between support, engineering, and specialist teams.

## Features

- Escalation path visualization
- L1, L2, and L3 workflow cards
- Active escalation highlighting
- SLA indicators
- Team ownership display
- Escalation timeline
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="path-card active">
  Engineering Team
</div>
```

## Main Classes

```css
.path-card
.active
.completed
.level-badge
.timeline-card
.timeline-item
```
