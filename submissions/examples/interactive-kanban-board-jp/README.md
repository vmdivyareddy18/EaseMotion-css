# Interactive Kanban Board Component

A modern Kanban Board UI built using pure HTML and CSS.

## Features

- Responsive multi-column board layout

- Animated task cards

- Drag-state visual feedback

- Column highlighting effects

- Hover interactions

- Status indicators

- Dashboard-style UI

- Smooth transitions and glow effects

- Mobile friendly design

- demo.html

- style.css

- Productivity Apps

- Team Collaboration Platforms

- Task Tracking Systems

- Admin Panels
