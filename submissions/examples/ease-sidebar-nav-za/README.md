# CSS Sidebar Navigation

## What does this do?
Responsive sidebar nav with grouped links, active states, and mobile hamburger toggle via checkbox hack.

## How is it used?
```html
<aside class="sb-sbar"><div class="sb-brand"><h2>Brand</h2></div><nav class="sb-nav"><a class="sb-a sb-act">Link</a></nav></aside>
```

## Why is it useful?
Fixed sidebar without JS. Mobile slide-in toggle. Icon support, section grouping, active indicators.
