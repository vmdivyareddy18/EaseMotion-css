# Full Pet Products Store Landing Page

1. What does this do? A playful pet products e-commerce landing page with pet categories, bestsellers, features, and community CTA built entirely with EaseMotion CSS framework classes.

2. How is it used? Open `demo.html` — the page uses `ease-card-hover` for product cards, `ease-bounce-in` for playful entrances, `ease-float` for decorative elements, `ease-reveal-*` for scroll animations, and `ease-grid-cols-*` for responsive layouts.

3. Why is it useful? Demonstrates EaseMotion's fun side with bouncing entrance animations, floating decorative elements, and warm brand colors — all framework-native without custom keyframes, proving EaseMotion suits playful brands too.
