# Mention Highlight Pulse

## What does this do?
Adds a subtle pulsing background highlight to @mentions when a new message arrives, drawing attention to the referenced user.

## How is it used?
Apply the `.mention-pulse` class to a span wrapping an @mention. The background pulses briefly then settles to draw attention.

## Why is this useful?
Provides conversational context cues in chat UIs and comment sections, improving awareness of relevant mentions without intrusive notifications.
