# Shake Sticky Header

A landing page inspired sticky header with a subtle shake animation built using pure HTML and CSS.

## Features

- 📌 Sticky navigation header
- 🎬 Shake hover animation
- 💻 Pure HTML & CSS
- 📱 Responsive layout
- ♿ Accessible navigation
- 🚫 No JavaScript required

## File Structure

```
shake-sticky-header-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

Hover over the sticky header to trigger the shake animation while the header remains fixed at the top during scrolling.

## Use Cases

- Landing Pages
- Business Websites
- Product Pages
- Portfolio Sites
- Marketing Websites
