# Heartbeat Time Picker

Resolves #42410.

### What does this do?
Adds a Real-Estate styled Heartbeat Time Picker component to EaseMotion CSS. It uses CSS radio buttons to maintain state without JavaScript. The active time slot utilizes `ease-zoom-in` for feedback, and the main confirm button features a continuous `ease-heartbeat` animation on the clock icon to draw attention.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. The pure-CSS radio logic handles the active states.

### Why is it useful?
Provides an elegant, responsive time selection interface with subtle heartbeat motion to encourage user conversion, completely decoupled from JavaScript.
