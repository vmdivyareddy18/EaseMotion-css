# Glowing Glassmorphic Card (`ease-glowing-card-has`)

### 1. What does this do?
This is a glassmorphic card component featuring an interactive neon radial glow background that dynamically follows the user's mouse movements.

### 2. How is it used?
Wrap content inside `.glow-card` and add the glow element:
```html
<div class="glow-card">
  <div class="card-glow"></div>
  <div class="card-content">
    <h3>Glowing Card</h3>
    <p>Interactive glass container...</p>
  </div>
</div>
```

The position of the radial gradient is updated dynamically via mouse coordinates.

### 3. Why is it useful?
It adds high-fidelity, interactive aesthetics to modern web pages using micro-interactions that feel premium and tactile, which aligns with EaseMotion's aesthetic philosophy.
