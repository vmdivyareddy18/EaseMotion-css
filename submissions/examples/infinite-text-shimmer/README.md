# Infinite Text Shimmer

A continuously looping shimmer effect for text using animated gradient backgrounds and `background-clip: text`.

## Features

- Infinite shimmer animation
- Uses `background-clip: text`
- Customizable speed with `--ease-shimmer-speed`
- Lightweight and pure CSS

## Usage

```html
<h1 class="ease-text-shimmer">
  EaseMotion CSS
</h1>
