# Swing Hover Creative Portfolio Modal

A responsive Creative Portfolio modal featuring a smooth Swing-Hover animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Creative portfolio layout
- Swing-Hover animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--rotate
--radius
--primary
--secondary
```
