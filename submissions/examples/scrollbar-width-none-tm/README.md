`scrollbar-width-none-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/scrollbar-width-none-tm/style.css">
<div class="scrollbar-width-none-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .scrollbar-width-none: hides scrollbar

## Impact

Hides scrollbar while maintaining scroll
