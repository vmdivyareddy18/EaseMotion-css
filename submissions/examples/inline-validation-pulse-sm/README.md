# Inline Validation Pulse

1. What does this do? Adds responsive form rows with success, warning, and error validation states using pulsing dots, animated helper chips, and clear focus styles.
2. How is it used? Apply `.field-row` with state classes like `.success-row`, `.warning-row`, or `.error-row` around each labeled input.
3. Why is it useful? It makes inline form feedback easier to scan while keeping the interaction CSS-only, accessible, and motion-aware.
