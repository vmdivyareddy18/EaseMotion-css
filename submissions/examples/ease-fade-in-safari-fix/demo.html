Fade In Safari Fix

Fixed ease-fade-in animation for Safari by adding missing -webkit- vendor prefixes. Now works across all major browsers.

🔍 Before vs After

❌ Before Fix

This element uses animation without -webkit- prefix.

Broken on Safari — no fade animation plays.

animation: fade-in 0.5s ease forwards;
✅ After Fix

This element uses -webkit- prefixed animation.

Works correctly on Safari, Chrome, Firefox, Edge.

-webkit-animation: fade-in 0.5s ease forwards;

📄 The Fix

Added -webkit- vendor prefix to all fade animations:

/* ✅ Fixed: Added -webkit- prefix for Safari */
.ease-fade-in {
  -webkit-animation: fade-in 0.6s ease forwards;
  animation: fade-in 0.6s ease forwards;
}

@-webkit-keyframes fade-in {
  from { opacity: 0; }
  to   { opacity: 1; }
}

@keyframes fade-in {
  from { opacity: 0; }
  to   { opacity: 1; }
}
