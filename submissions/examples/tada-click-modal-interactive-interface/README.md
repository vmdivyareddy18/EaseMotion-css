# CSS Tada-Click Modal for Interactive Interface Layouts

A lightweight, responsive **CSS-only Tada-Click Modal** designed for interactive interface layouts. The modal uses a smooth "Tada" entrance animation to create an engaging user experience without requiring JavaScript.

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Smooth Tada-Click entrance animation
- Fully responsive layout
- Keyboard accessible
- Easy to customize with CSS custom properties
- Modern interactive interface design

## 🚀 Usage

1. Open `demo.html` directly in any modern web browser.
2. Click the **Launch Modal** button.
3. The modal appears with a smooth Tada-Click animation.
4. Click **Close** to dismiss the modal.

## 🎨 Customization

Modify the CSS variables in `style.css` to match your project.

```css
:root {
  --duration: 0.65s;
  --ease: cubic-bezier(0.175, 0.885, 0.32, 1.275);
  --scale-start: 0.8;

  --primary: #6d5dfc;
  --primary-hover: #5948f2;
}
```

You can customize:

- Animation duration
- Animation easing
- Initial scale
- Primary colors
- Border radius
- Shadows
