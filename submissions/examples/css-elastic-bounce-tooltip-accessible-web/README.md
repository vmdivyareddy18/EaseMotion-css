# CSS Elastic Bounce Tooltip – Accessible Web Layouts

## Overview

A responsive tooltip component built using pure HTML and CSS. The tooltip features a smooth elastic bounce animation, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Elastic bounce animation
- Responsive design
- Keyboard accessible (`:hover` and `:focus`)
- CSS custom properties for easy customization
- No JavaScript required
- Reduced motion support

## CSS Custom Properties

The following variables can be customized:

- `--tooltip-bg`
- `--tooltip-color`
- `--primary`
- `--duration`
- `--easing`
- `--radius`
- `--padding`
- `--shadow`
- `--scale`
- `--translate`

## Files

- `demo.html` – Demo page
- `style.css` – Tooltip styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser. Hover over or focus on the button to display the elastic bounce tooltip. Customize the CSS variables in `style.css` to adjust colors, animation, spacing, and appearance.
