# SVG Filter Playground

Interactive playground for CSS filter effects applied to SVG graphics and text.

## Features

- Multiple filter presets (Blur, Grayscale, Sepia, Vibrant, Invert, Shadow)
- Real-time range controls for blur and brightness
- Smooth filter transitions between presets
- SVG illustration and styled text both affected by the active filter

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--sfp-duration` | `0.4s` | Filter transition duration |
| `--sfp-control-bg` | `#1e293b` | Control card background |
| `--sfp-active-preset` | `#6366f1` | Active preset button color |
| `--sfp-radius` | `14px` | Border radius |
| `--sfp-bg` | `#0f172a` | Page background |
