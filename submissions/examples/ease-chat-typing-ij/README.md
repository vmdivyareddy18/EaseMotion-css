# Chat Typing Indicator

An animated typing indicator showing three bouncing dots to indicate someone is typing in a chat interface.

## Features

- Three bouncing dots with staggered animation delays
- Smooth message slide-in animation for chat bubbles
- Dark theme chat interface design
- Pure CSS keyframe animations
- Responsive layout

## Usage

Include `style.css` in your project and use the `.typing-indicator` HTML structure with three `.typing-dot` elements with staggered `animation-delay` values (0s, 0.15s, 0.3s).
