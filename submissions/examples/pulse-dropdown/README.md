# Pulse Dropdown

A lightweight animated dropdown component created with pure HTML and CSS for EaseMotion CSS.

## Features

- Pure CSS implementation
- No JavaScript required
- Smooth pulse animation
- Hover-based dropdown interaction
- Responsive design
- Easy customization

## Usage

Add the dropdown structure:

```html
<div class="dropdown">

  <button class="dropdown-btn">
    Select Option
  </button>

  <div class="dropdown-menu">
    <a href="#">Dashboard</a>
    <a href="#">Profile</a>
  </div>

</div>
