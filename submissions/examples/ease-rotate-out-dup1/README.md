# Ease Rotate Out

## Overview

A CSS exit animation where an element rotates away while fading out.

This animation is intended to pair with `ease-rotate-in`, providing a matching exit transition.

## Features

- Rotates to 200 degrees
- Fades to opacity 0
- Uses ease-in timing
- Pure CSS implementation
- Infinite demo preview

## Usage

```html
<div class="ease-rotate-out">
  Content
</div>
```

```css
.ease-rotate-out {
  animation: easeRotateOut 600ms ease-in forwards;
}
```

## HTML Structure

```html
<div class="ease-rotate-out">
  Content
</div>
```

## CSS Classes

### `.ease-rotate-out`

Applies the exit rotation animation.

## Why It Fits EaseMotion CSS

- Lightweight
