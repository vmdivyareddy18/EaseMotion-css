# Animated Job Application Tracker Card

A modern job application tracking component built using HTML and CSS.

## Features

- Animated application progress bar
- Interview stage tracking
- Recruiter update section
- Application timeline
- Responsive design
- Interactive action buttons

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

- Job portals
- Career dashboards
- Recruitment platforms
- Internship tracking systems
- Applicant management tools
