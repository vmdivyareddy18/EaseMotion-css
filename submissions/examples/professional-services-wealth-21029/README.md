# Professional Services: Private Wealth Management (`professional-services-wealth-21029`)

This proposal provides a complete, copy-paste ready B2B/B2C professional services landing page designed for a high-net-worth private wealth management firm or multi-family office, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A private wealth advisory site must convey heritage, prestige, extreme trust, and exclusivity. This template achieves an "institutional" aesthetic using a classic serif/sans-serif pairing (Lora & Lato), a sophisticated color palette (Deep Forest Green, Champagne/Gold, and Warm Off-White), and intentionally slow, luxurious CSS entrance animations.

The layout includes:
- A stately hero section with an italicized serif headline and a subtle, textured CSS background.
- A full-width "Minimum Investment" banner.
- A side-by-side Investment Philosophy section detailing core pillars.
- An elegant, card-based services grid (Portfolio, Estate, Tax).
- A Partner/Team profile section with a slow, grayscale-to-color hover transition.
- A dramatic, dark-themed client reputation block.
- A private consultation request form featuring refined typography and minimalist input lines.
- A comprehensive footer complete with mandatory financial disclosures.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (Crucial for this design: animations are purposefully paced slower (via CSS `animation-duration: 1s` overrides) to feel luxurious and unhurried)
- `.ease-slide-right` (Used for primary section headers)
- `.ease-expand` (Applied to champagne-colored divider lines for a sophisticated accent)
- `.ease-hover-lift` (Applied to service cards and primary buttons to provide a soft, tactile, high-end feel)
- `.ease-focus-line` (Applied to form inputs to create a minimalist, single-line focus state that shifts to the accent gold color)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, elegant typography, and high-net-worth styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript.
