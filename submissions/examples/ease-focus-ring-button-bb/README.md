# Accessible Dynamic Focus Ring Button

## Description
A highly visible button styling with an animated pulse-ring outline that triggers on focus (`:focus-visible`). It improves website usability and complies with WCAG accessibility standards.

## Usage
Simply apply the `.focus-ring-btn` class to any `<button>` element:
```html
<button class="focus-ring-btn">Interact With Me</button>
```

## Why is it useful?
Provides distinct keyboard interaction states without adding distracting borders or outline elements for mouse users. Supports `prefers-reduced-motion` safety fallbacks.
