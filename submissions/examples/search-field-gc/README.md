# Inline Search Field UI Example

## 1. What does this do?

This submission provides an inline search field component with icon integration, animated focus border, and floating placeholder styling. It demonstrates a polished search input pattern using CSS `:focus-within`, `@keyframes`, and custom icon placement.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The demo uses `style.css` for search field styling and focus animation. Copy the `.search-field` wrapper and CSS into your project for a reusable search input component.

---

## 3. Why is it useful?

Search input fields are a key navigation component for many apps and sites. This example improves clarity with a strong focus state, clear icon affordance, and accessible layout while remaining CSS-only.
