# Scroll-Driven SVG Path Animation

### 1. What does this do?
This component demonstrates a scroll-driven animation where an SVG path is progressively drawn as the user scrolls down the page. It maps the window scroll percentage to the `stroke-dashoffset` CSS property using a minimal JavaScript listener.

### 2. How is it used?
Include the HTML structure with your inline SVG. Load `style.css` in your head block and `script.js` before your closing `body` tag. The configuration details are noted in `config.json` for extending the paths. 

### 3. Why is it useful?
Scroll-driven animations are vital for modern storytelling and interactive landing pages. This example provides a highly performant, lightweight pattern without relying on heavy third-party animation libraries like GSAP.
