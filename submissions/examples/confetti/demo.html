🎉 ease-confetti

CSS-only confetti burst · EaseMotion CSS submission · Issue #2720

.ease-confetti

Fires a confetti burst on click / tap. No JS needed.

.ease-confetti-auto
✨ Auto-plays on load

Plays once automatically when the page loads. Great for success screens.

.ease-confetti-loop
🔁 Looping burst

Repeats the confetti burst every 3 seconds indefinitely.

.ease-confetti-small

Tiny, delicate particles. Stack with .ease-confetti for click trigger.

.ease-confetti-large

Big chunky particles for maximum celebration impact.

combo example

Combine .ease-confetti + .ease-confetti-large on any element.

<!-- Fires on click -->
<button class="ease-btn ease-btn-primary ease-confetti">
  🎉 Celebrate!
</button>

<!-- Auto-plays on load -->
<div class="ease-confetti-auto">You did it!</div>

<!-- Loops every 3s -->
<div class="ease-confetti-loop">Keep going!</div>

<!-- Size modifiers (stack with ease-confetti) -->
<button class="ease-confetti ease-confetti-small">Tiny</button>
<button class="ease-confetti ease-confetti-large">Chunky</button>
