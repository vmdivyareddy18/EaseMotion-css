# Label Badge Pulse

A notification badge with attention-grabbing pulse ring animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--badge-color` | `#ff6b6b` | Badge background color |
| `--pulse-color` | `rgba(255,107,107,0.4)` | Pulse ring color |
| `--badge-size` | `24px` | Badge diameter |
| `--pulse-duration` | `1.5s` | Ring pulse cycle duration |
