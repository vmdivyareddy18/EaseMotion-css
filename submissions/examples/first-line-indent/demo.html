::first-line Indent

Styled with ::first-line

This is the very first paragraph of the demo. The first line of this paragraph should appear with a distinct font, increased letter-spacing, and a subtle indentation thanks to the pseudo-element. All subsequent lines wrap normally with the body styling.

Here is a second paragraph to demonstrate that each paragraph's first line is styled independently. Notice how the first line gets its own color and spacing while the rest of the text remains unchanged. This creates a pleasant reading rhythm.

A third paragraph reinforces the effect. The initial line stands out with a slighly larger font weight and a light gold hue, guiding the reader's eye naturally down the left edge of the text block.

Styled with <span> (comparison)

This is the equivalent paragraph where the first line is manually wrapped in a <span>. The span approach requires breaking the text in the HTML, which is fragile and does not reflow when the viewport changes. Resize the browser to see how the span fails to track the actual first line.

Another paragraph with a manually defined "first line" span. Unlike the pseudo-element approach above, the span's text is static and will not adapt to wider or narrower screens, making it impractical for responsive layouts.
