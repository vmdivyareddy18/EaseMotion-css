# App Landing: Protein & Supplement Brand (`ecommerce-protein-supplement-brand-20969`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for a high-performance sports nutrition and supplement brand, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

An athletic performance brand requires a UI that feels bold, aggressive, and highly energetic. This template achieves a high-octane aesthetic using a dark mode foundation (`#0C0D10`), neon/electric yellow accents, slanted/skewed UI elements, condensed bold typography (`Teko`), and snappy, punchy animations with high tension.

The layout includes:
- An urgent Promo Bar and sticky Navigation.
- A Hero section featuring a skewed background layout and a CSS-rendered abstract athlete graphic with pulsing energy rings and a lifting barbell.
- A "Shop By Goal" grid (Bulk, Cut, Endurance) featuring color-coded linear gradients and hover zoom interactions.
- A Product Arsenal grid featuring abstract CSS supplement containers (protein tub, pre-workout tub, pill bottle, BCAA pouch).
- A Bundle/Subscribe CTA overlay box with a diagonal slash pattern.
- A Science & Ingredients section showcasing a transparent label philosophy alongside an animated CSS data/chart graphic.
- A bold Testimonial grid for athletes.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use snappy `cubic-bezier(0.175, 0.885, 0.32, 1.275)` and aggressive bouncing `cubic-bezier(0.68, -0.55, 0.265, 1.55)` timings that convey power and speed:
- `.ease-slide-up-fast` (Configured for quick, skewed vertical entrances).
- `.ease-zoom-in-fast` (An aggressive, punchy scale-in animation).
- `.ease-reveal-polygon` (A custom clip-path entrance that reveals content using a skewed polygon shape).
- `.ease-lift` / `.ease-pulse-fast` (Continuous energetic movements applied to the hero graphic).
- `.ease-grow-up` / `.ease-draw-line` (Animations used to build the CSS chart in the Science section).
- `.ease-hover-punch` (A high-tension hover scale and skew effect used on primary buttons).
- `.ease-hover-glitch` (A quick text-shadow and color shift hover effect for links).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the dark, angular layout architecture, the high-contrast palette, and the complex abstract CSS illustrations (supplement tubs, athlete silhouette, charts). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the aggressive entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
