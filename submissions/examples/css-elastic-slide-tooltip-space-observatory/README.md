# CSS Elastic Slide Tooltip - Space Observatory

A pure CSS animated tooltip component inspired by space observatory interfaces.

## Features

- Pure CSS animation
- Elastic slide transition effect
- Space-themed UI design
- Responsive layout
- Keyboard accessible
- Customizable animation variables


## Usage

Open `demo.html` directly in a browser.

Hover or focus on celestial object buttons to display the animated tooltip.


## Customization

Modify CSS variables:

```css
:root {
    --tooltip-time: 0.45s;
    --tooltip-scale: 1;
    --tooltip-easing: cubic-bezier(.68,-0.55,.27,1.55);
}
