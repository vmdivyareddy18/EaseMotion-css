# CSS Media Player Controls

## What does this do?
CSS-styled media player UI with audio and video player skins, spinning album art, playlist tracks, and interactive controls.

## How is it used?
```html
<div class="mp-player mp-audio"><div class="mp-controls"><button class="mp-btn mp-play">▶</button></div><div class="mp-progress"><div class="mp-track"><div class="mp-track-fill"></div></div></div></div>
```

## Why is it useful?
Provides media player UI patterns without JavaScript. Audio and video player variants. Spinning album art via CSS animation. Playlist with hover states. Progress bars and volume controls. Customizable via CSS variables.
