# Frog Jumping

1. What does this do? A CSS-only frog that jumps with squash-and-stretch (scaleY) and blinking eyes.

2. How is it used? Place .frog inside .scene. The body is a green ellipse; eyes are two circles via box-shadow.

3. Why is it useful? Shows bounce easing combined with squash/stretch for organic jumping motion.
