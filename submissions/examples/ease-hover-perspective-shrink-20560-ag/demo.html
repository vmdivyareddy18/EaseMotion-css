CSS Code Guide

To enable 3D perspective scaling, set a perspective on the parent wrapper element, and translate the card's Z-axis on hover:

/* Parent Wrapper */
.card-perspective-container {
  perspective: 1000px;
}

/* Card Element */
.showcase-card {
  transition: transform 0.4s cubic-bezier(0.25, 1, 0.5, 1);
}

/* Hover Action */
.showcase-card:hover {
  transform: scale(0.92) translateZ(-20px);
}
