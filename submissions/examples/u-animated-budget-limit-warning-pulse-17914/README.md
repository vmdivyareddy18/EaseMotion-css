# Animated Budget Limit Warning Pulse

## Overview

Animated Budget Limit Warning Pulse is a lightweight CSS animation that simulates budget alerts using a pulsing warning icon and animated progress bar.

## Features

- CSS-only animation
- Warning pulse effect
- Budget limit visualization
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="budget-warning">
  <span class="warning-icon">⚠️</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
