# Estate Planning Attorney Template
A calm, distinguished, and reassurance-focused template for estate planning attorneys, trust consultants, and family succession practices.

- **Design**: Uses a "Legacy-Preservation" palette with heritage-green deep tones, legacy-bronze accents, and parchment-inspired backgrounds to communicate stability, timelessness, and familial care.
- **Use Case**: Estate planning law firms, trust management services, and family succession consultancies.
