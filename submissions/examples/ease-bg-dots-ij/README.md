# Component: Background Dots

This submission implements a repeating dot pattern background for issue **#13617**.

## Description

An `.ease-bg-dots` class that creates a repeating dot pattern using `radial-gradient`. Customizable via `--ease-dot-size`, `--ease-dot-spacing`, and `--ease-dot-color` CSS custom properties. Includes interactive controls for size, spacing, and visibility toggle.

## Acceptance Criteria

- `.ease-bg-dots` class with `radial-gradient` dot pattern
- `--ease-dot-size` controls dot radius
- `--ease-dot-spacing` controls grid spacing
- `--ease-dot-color` controls dot color
- Interactive size slider, spacing slider, and toggle button
