# Stirling Engine

A pure CSS and HTML model Stirling engine running on a spirit flame, the flywheel spinning while the piston strokes and the con rod rocks in time.

## What it shows

- The linkage kept honest by sharing one 1.5s period: the flywheel turns, the piston strokes and the con rod rocks on the same cycle, so the mechanism reads as connected
- A spoked brass flywheel using the radial rotate pattern, with a crank pin offset from the hub
- Cooling fins on the hot cylinder from a repeating linear gradient over a brass body
- A spirit flame from layered radial gradient stops, guttering on a fast cycle with heat shimmer rising off it
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
