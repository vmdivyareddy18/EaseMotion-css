# Animated Text Decoration Styles

A typography demo showcasing five CSS text-decoration-style values (solid, wavy, dotted, dashed, double) with animated color transitions on each item.

## Files

- `demo.html` - Five decorated text lines with different styles
- `style.css` - Text-decoration properties, per-item keyframes, hover offset shift

## Usage

Open `demo.html` to see animated text decorations. Hover over each line to see the underline offset increase.
