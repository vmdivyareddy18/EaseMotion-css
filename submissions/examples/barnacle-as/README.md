# Barnacle

Three pure CSS and HTML acorn barnacles on a tidal rock, opening their plates to sweep the water with their cirri.

## What it shows

- Shell cones cut from a clip-path polygon, with the same clip reused on a repeating conic gradient layer for the plate seams
- Aperture plates that swing apart on mirrored keyframes like a beak
- Feathery cirri drawn as a masked repeating conic gradient that unfurls and sweeps in time with the plates
- A translucent wave that surges over the rock
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
