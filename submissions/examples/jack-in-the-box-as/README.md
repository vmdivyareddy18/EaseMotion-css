# Jack in the Box

A pure CSS and HTML jack in the box: turn the crank, the lid flies open and the clown springs out with a wobble, then drops back down to start again.

## What it shows

- The whole pop cycle choreographed on one timeline: crank spins, lid swings open, jack launches up on his spring, wobbles to a stop, then folds back in
- A coiled spring from stacked half circle borders that compress and stretch with the launch
- A harlequin hat from a diagonal repeating linear gradient clipped to a peak, and a ruffled collar from a masked conic gradient
- A clown face with rosy cheeks, a red nose and a border-bottom grin, and musical notes on the pop
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`, leaving the jack popped and the lid open.
