# EaseMotion CSS — Draw Border Modal (CSS-only)

This example demonstrates a CSS-only modal animation using the `:target` selector.

## Files
- `demo.html` — Landing page + modal markup
- `style.css` — Styling, overlay + scale animation, and draw-border animation via pseudo-elements

## Customization
Edit CSS custom properties in `style.css`:
- `--primary-color`
- `--background-color`
- `--surface-color`
- `--text-color`
- `--radius`
- `--duration`
- `--easing`
- `--scale`
- `--shadow`
- `--border-width`
