# Morphing Blob Button Hover Effect

A modern CSS-only button featuring a liquid-inspired morphing blob animation. The effect uses animated `border-radius`, pseudo-elements, smooth transitions, and hover interactions to create an eye-catching call-to-action button.

## Preview

- CSS-only
- No JavaScript
- Responsive layout
- Smooth hover animation
- Animated blob background

## Folder Structure

```
morphing-blob-button/
├── demo.html
├── style.css
└── README.md
```

## How to Run

1. Open `demo.html` in any modern web browser.
2. Hover over the button to view the morphing blob animation.

## Features

- Liquid-style blob animation
- Animated pseudo-elements
- Smooth scaling on hover
- Soft glowing shadow
- Easy to customize
- Beginner-friendly code

## Customization

You can easily modify:

- Button color (`background`)
- Animation speed
- Border-radius values
- Shadow intensity
- Button size
- Font styles

## Browser Support

Works in all modern browsers that support:

- CSS Animations
- CSS Transitions
- Pseudo-elements
- Border-radius animations
