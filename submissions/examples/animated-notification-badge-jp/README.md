# Animated Notification Badge Component

## Description

A modern notification badge component featuring pulse and bounce animations, multiple color variants, and smooth hover interactions.

---

## Features

- Animated pulse effect
- Bounce notification animation
- Multiple color variants
- Works with icons and buttons
- Hover lift interaction
- Responsive layout
- Reduced motion accessibility support

---

## Variants

### Pulse Badge
Draws user attention using a glowing pulse animation.

### Bounce Badge
Creates an active notification effect with vertical movement.

### Color Styles
- Red
- Blue
- Green

---

## Classes

- `.badge` – Base notification badge
- `.pulse` – Pulse animation
- `.bounce` – Bounce animation
- `.badge-red` – Red style
- `.badge-blue` – Blue style
- `.badge-green` – Green style

---

## Accessibility

Supports `prefers-reduced-motion` to disable unnecessary animations.
