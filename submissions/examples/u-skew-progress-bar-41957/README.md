# Skew Progress Bar

A Neon-inspired Skew Progress Bar built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Skew animation
- Animated progress fill
- Responsive layout
- Accessible HTML
- No JavaScript required
