# Software License Renewal Portfolio

A high-fidelity **Software License Renewal Portfolio** built entirely with HTML and CSS.

## Overview

This example demonstrates an enterprise Software Asset Management (SAM) dashboard for managing software license renewals, subscription portfolios, vendor agreements, utilization metrics, procurement planning, and compliance monitoring across business units.

The interface combines executive licensing KPIs, a comprehensive renewal portfolio, vendor tracking, utilization analytics, procurement readiness, and renewal timelines into a modern enterprise dashboard suitable for IT asset management and procurement teams.

---

## Features

- Enterprise software asset management dashboard
- Advanced software license renewal portfolio
- Sticky renewal portfolio table
- Executive licensing KPI cards
- Vendor contract tracking
- License utilization monitoring
- Renewal deadline management
- Procurement readiness workflow
- Budget forecasting overview
- Interactive renewal checklist
- License renewal timeline
- Animated portfolio health visualization
- Renewal readiness analytics
- Responsive enterprise layout
- Pure HTML & CSS (No JavaScript)
