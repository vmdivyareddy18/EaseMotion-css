# Animated Isometric Developer Workspace

A premium CSS-only isometric developer workstation hero component with floating browser windows, terminal panels, code blocks, and tech badges.

## Usage

Add the HTML markup below and include `style.css` alongside `demo.html`.

```html
<section class="workspace-hero">
  <div class="workspace">
    <div class="workspace__glow"></div>
    <div class="workspace__desk"></div>

    <div class="workspace__panel workspace__panel--browser">...</div>
    <div class="workspace__panel workspace__panel--terminal">...</div>
    <div class="workspace__panel workspace__panel--code">...</div>
    <div class="workspace__panel workspace__panel--laptop">...</div>
    <div class="workspace__tech-bar">...</div>
  </div>
</section>
```

## Why it fits EaseMotion CSS

This feature is built entirely with HTML and CSS, delivering motion-first visual storytelling through reusable hero elements. It demonstrates composable animation patterns, glassmorphism styling, and responsive layout behavior without requiring JavaScript, making it a strong match for EaseMotion CSS's animation-first philosophy.
