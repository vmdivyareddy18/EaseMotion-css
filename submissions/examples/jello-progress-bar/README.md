# Jello Progress Bar

A CSS-only animated jello-style progress bar component for EaseMotion CSS.

## Features

- Pure CSS animation
- Jello bounce effect
- Responsive design
- No JavaScript required
- Accessible progress indicator

## Usage

Include the HTML structure and CSS file in your project.

Example:

```html
<div class="progress-bar">
  <span class="progress-fill"></span>
</div>
