# Train Chug — Train Moving with Smoke

CSS train that chugs along a track with spinning wheels and puffing smoke. The locomotive moves horizontally while wheel rotation and smoke bursts play in parallel.

## Files

- `demo.html` — train engine, cabin, wheels, smoke puffs
- `style.css` — train locomotion, wheel spin, smoke puff keyframes

## Usage

Open `demo.html` to see the train move with steam effects.
