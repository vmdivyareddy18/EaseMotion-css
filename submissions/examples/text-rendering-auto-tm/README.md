`text-rendering-auto-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/text-rendering-auto-tm/style.css">
<div class="text-rendering-auto-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .text-rendering-auto: sets text-rendering to auto

## Impact

Resets text rendering to browser default
