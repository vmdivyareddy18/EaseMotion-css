# Progress Button – Button fills with color showing upload progress

> A production-ready progress button component that visually fills with color to show upload or task progress. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

![Progress Button Demo](https://via.placeholder.com/280x56/4facfe/ffffff?text=Uploading...)

**Live Preview:** Open `demo.html` in your browser.

---

## ✨ Features

- ✅ **Visual progress fill** – Button fills from left to right
- ✅ **Real-time percentage** – Shows progress percentage
- ✅ **Multiple states** – Idle, Progress, Complete, Error
- ✅ **Pause/Resume** – Pause and resume uploads
- ✅ **Manual control** – Slider for manual progress adjustment
- ✅ **Auto-complete** – Automatically completes on reaching 100%
- ✅ **Keyboard accessible** – Enter/Space to interact
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Highly customizable** – Colors, text, durations, and callbacks
