# Ease Newsletter Subscribe Card

A modern **newsletter subscribe card** built using **pure HTML and CSS**. It features a smooth slide-in animation, responsive layout, and customizable styling using CSS custom properties.

## Features

- Pure HTML & CSS
- CSS-only animation using `@keyframes`
- Modern newsletter card design
- Email input with focus styles
- Animated subscribe button
- CSS Custom Properties for easy customization
- Responsive design
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
ease-newsletter-subscribe-card/
├── demo.html
├── style.css
└── README.md
```

## Customization

You can customize the component by editing the CSS variables in the `:root` section, including:

- Primary color
- Background color
- Border radius
- Shadow
- Transition speed

## Browser Support

Compatible with all modern browsers.
