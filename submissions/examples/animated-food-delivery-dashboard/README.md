# Animated Food Delivery Dashboard

An administrative delivery fulfillment dashboard environment built specifically for tracking active culinary orders, logistical fleet dispatch counters, and merchant status loops.

## Features
- **Summary Metrics Counters:** Quick access overview tracking metrics for Total Orders, Active Riders, Partner Restaurants, and Payout Revenue.
- **Hardware-Accelerated Fluid Animations:** Smooth loading transformations utilizing `@keyframes loadIn` tracking vectors and interactive status dropdown micro-animations.
- **Dispatch Quick Controls:** Fast admin hooks tracking instant rider assignments, tracking modifications, and CSV analytical file extractions.
