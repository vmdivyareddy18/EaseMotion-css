# Glitch Hashtag Input

A reusable cyberpunk-inspired hashtag input component built using only HTML and CSS.

## Features

- Pure HTML + CSS
- Responsive
- No JavaScript
- Accessible
- Animated glitch border
- Animated hashtag icon
- RGB glow
- Scan line animation
- CSS Variables
- Supports prefers-reduced-motion

## Preview

Click inside the input field to activate:

- RGB glitch border
- Animated hashtag icon
- Moving scan line
- Placeholder flicker

## Accessibility

Supports:

- Keyboard navigation
- Screen readers
- prefers-reduced-motion

## Use Case

Ideal for:

- Blog editors
- CMS
- Portfolio websites
- Admin dashboards
- Social media publishing tools
- AI prompt tagging interfaces
