# Recipe Meal Planner App Template
A wholesome, organized, and appetizing landing page template for meal planners, recipe discovery apps, and healthy lifestyle tools.

- **Design**: Uses a "Kitchen-Fresh" palette with sage-green primary branding, spice-orange action accents, and clean, modular layouts designed to promote health, organization, and culinary creativity.
- **Use Case**: Meal planning apps, recipe aggregators, and nutritional management platforms.
