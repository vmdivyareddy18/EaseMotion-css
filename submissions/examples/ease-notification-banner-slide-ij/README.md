# Notification Banner Slide

A slide-down notification banner with dismiss action.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--banner-bg` | `#2d2d44` | Banner background |
| `--text-color` | `#eee` | Message text color |
| `--icon-color` | `#f72585` | Alert icon color |
| `--slide-duration` | `0.35s` | Slide animation speed |
