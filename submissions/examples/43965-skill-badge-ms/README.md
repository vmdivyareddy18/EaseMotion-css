# Animated Skill Badge Component

Resolves #43965.

### What does this do?
Provides a reusable skill badge component with a satisfying bounce and glow hover animation, including a micro-interaction for the badge icon.

### How is it used?
Wrap text and an optional icon inside an element with the `.ease-skill-badge` class. Use `.ease-skill-badge-icon` for the icon to enable the rotation micro-interaction.

### Why is it useful?
Skill badges are commonly used on portfolio websites and contributor profiles. This component makes them visually engaging and aligned with EaseMotion's animation-first philosophy.
