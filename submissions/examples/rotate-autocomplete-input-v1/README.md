# Rotate Autocomplete Input

A responsive autocomplete input with a smooth rotate entrance animation inspired by modern News Portal search interfaces.

## Features

- Pure HTML & CSS
- Rotate entrance animation
- Responsive layout
- Accessible using native HTML `<datalist>`
- No JavaScript required

## Files

```
rotate-autocomplete-input-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
