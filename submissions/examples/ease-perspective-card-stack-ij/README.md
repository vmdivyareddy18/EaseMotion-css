# Perspective Card Stack

A stacked card layout with 3D perspective hover effects using CSS transforms.

## Features

- Stacked cards with alternating rotation offsets
- Hover lifts card with translateZ for 3D depth
- Adjacent cards slide down when one is hovered
- Color swatch with rotate animation on hover
- Glassmorphism card design with backdrop blur
- Pure CSS 3D transforms using perspective and preserve-3d

## Usage

Include `style.css` and structure cards with `.stack-card` elements inside `.card-stack`. Use CSS custom properties `--rotate`, `--offset`, and `--index` for positioning each card in the stack.
