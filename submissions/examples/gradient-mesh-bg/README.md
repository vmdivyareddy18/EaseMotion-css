# Animated Gradient Mesh Background

1. **What does this do?** Creates a living, breathing background from three layered radial gradients that orbit and morph at different speeds, with a frosted-glass content card on top.

2. **How is it used?** Copy the `.mesh-bg` structure into any page. The pseudo-elements create depth. The card is optional — the mesh works behind any content.

3. **Why is it useful?** Gradient meshes typically require canvas, WebGL, or heavy JavaScript. This uses two pseudo-elements and three keyframe animations — zero dependencies, smooth 60fps performance.
