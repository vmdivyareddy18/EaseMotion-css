# Ease Floating Orbit Modal

A pure HTML and CSS glassmorphism modal featuring floating orbit animations and a modern UI.

## Features

- Pure HTML & CSS
- No JavaScript required
- Glassmorphism design
- Floating orbit animation
- Smooth modal open/close transition
- CSS custom properties for easy customization
- Responsive layout
- Keyboard-friendly checkbox toggle

## Files

```
demo.html
style.css
README.md
```

## Technologies

- HTML5
- CSS3
- CSS Variables
- CSS Animations
- Glassmorphism UI

## Usage

1. Open `demo.html` in a browser.
2. Click **Open Glass Modal**.
3. Click **Close** or **✕** to dismiss the modal.

## Customization

Edit the CSS variables in `:root` to change:

- Colors
- Blur amount
- Border radius
- Animation speed
- Orbit sizes
