# Minimalist Tech Fade-Out Accordion

A clean, stark, monospace implementation of an interactive accordion component designed specifically for terminal and minimalist tech system themes.

## Customization Variables

Configure custom timing and scaling constraints using global variables inside your stylesheet root context:

```css
:root {
    --tech-duration: 0.25s;
    --tech-scale-factor: 0.99;
    --tech-accent: #00ff66;
}
