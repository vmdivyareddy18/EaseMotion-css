# Ease Polaris Glass Compass

## Overview

A futuristic compass widget featuring a rotating orbital ring, animated navigation needle, and glassmorphism center hub built entirely with HTML and CSS.

## Features

- Animated compass needle
- Rotating orbital ring
- Glassmorphism center hub
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-polaris-glass-compass-z1/

```

Open `demo.html` in any modern browser.

Optimized for desktop, tablet, and mobile devices.
