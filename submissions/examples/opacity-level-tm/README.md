# Opacity Level

opacity — Controls transparency level of elements

## Features
- Uses `--ease-color-primary`
- Uses `--ease-radius-lg`
- Uses `--ease-shadow-md`
- Uses `--ease-color-surface`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Opacity Level</span>
</div>
```

## Why is it useful?

This submission demonstrates the opacity level CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
