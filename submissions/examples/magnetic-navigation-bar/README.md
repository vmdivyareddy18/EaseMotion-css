# Magnetic Navigation Bar

A responsive navigation bar with a magnetic hover animation inspired by modern News Portal layouts.

## Features

- Pure HTML & CSS
- Responsive layout
- Accessible keyboard focus
- Magnetic hover animation
- Beginner-friendly implementation
- No JavaScript required

## Folder Structure

```
magnetic-navigation-bar-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in a browser.
2. Hover over the navigation links.
3. Resize the browser to see the responsive layout.

## Technologies

- HTML5
- CSS3
- EaseMotion-style animation classes
