# Text Orientation Upright Demo

A vertical text demo using CSS writing-mode and text-orientation properties. Shows mixed, upright, and sideways text orientations in vertical-rl writing mode.

## Files

- `demo.html` - Three vertical text orientation cards
- `style.css` - writing-mode, text-orientation properties, dark theme

## Usage

Open `demo.html` to see the different text orientation styles in vertical mode.
