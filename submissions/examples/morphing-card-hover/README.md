# Animation: Morphing Card Hover

## Description
A smooth, pure CSS morphing animation for cards. On hover, the standard rectangular card seamlessly transitions into an organic, blob-like shape using the 8-value `border-radius` property.

## Features
- Pure CSS (No JavaScript required)
- Fully responsive and highly performant (`transform` and `border-radius` transitions)
- Cross-browser compatible (Chrome, Firefox, Safari, Edge)
- Follows EaseMotion naming conventions (`em-morph-card`)

## Usage
Apply the `.em-morph-card` class to any `div` or container element.
