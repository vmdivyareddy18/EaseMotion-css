# Ease Holographic Metric Tile

## Overview

A premium dashboard metric tile featuring holographic gradients, animated data bars, and a glassmorphism surface built with pure HTML and CSS.

## Features

- Holographic glow effect
- Animated metric bars
- Glassmorphism container
- Responsive layout
- Smooth hover animation
- No JavaScript required

## Folder Structure

```text
ease-holographic-metric-tile-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

- Firefox

Scales smoothly across desktop, tablet, and mobile devices.
