# Animated CSS Icons

## What does this do?
CSS-drawn icons with keyframe and hover animations: heart beat/pulse, star spin/bounce, spinner, bell ring, mail shake, weather icons.

## How is it used?
```html
<div class="aic-icon aic-heart aic-beat"><div class="aic-heart-shape"></div><span>Heart</span></div>
```

## Why is it useful?
Pure CSS icon animations without image or SVG dependencies. Hover and auto-play variants. Customizable colors and timing.
