Email-Safe CSS Subset

Which EaseMotion utilities work inside email clients

⚠️ Email client CSS support varies widely. Gmail strips <style> tags and most animations. Outlook uses Word's rendering engine. Always test emails across clients before sending.
CSS Feature Email Safe? Notes
animation ✗ Unsafe Removed by Gmail, Outlook
@keyframes ✗ Unsafe Not supported in most clients
transition ✗ Unsafe No hover/interaction in email
transform ✗ Unsafe Stripped by Outlook, Gmail
inline styles ✓ Safe Use inline for all email CSS
color ✓ Safe Universally supported
background-color ✓ Safe Universally supported
font-family ✓ Safe Use web-safe fallbacks
text-align ✓ Safe Universally supported
display: none ✓ Safe Works in most clients
@media queries ~ Partial Works in Apple Mail, iOS; not Gmail
border-radius ~ Partial Not supported in Outlook
