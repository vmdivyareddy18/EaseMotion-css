# CSS Floating Orbit Modal

A high-performance, pure CSS animated modal component designed specifically for modern marketing landing page configurations.

## Features
- **Zero JavaScript Overhead:** Relies entirely on the CSS checkbox state hack for performance critical environments.
- **Floating Orbit Motion Architecture:** Uses standard 3D scaling, translation, and custom rotational easing timelines to display an entry effect reminiscent of orbit dynamics.
- **Customizable Properties:** Exposes intuitive variables (`--orbit-duration`, `--orbit-easing`, `--scale-factor-start`) for lightning-fast configuration mapping.
- **Fully Responsive & Accessible:** Fluid scaling transitions that degrade nicely on reduced-motion viewports. Includes basic native keyboard mapping triggers.
