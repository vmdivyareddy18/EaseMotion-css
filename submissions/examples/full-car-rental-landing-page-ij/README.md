# DriveEasy – Full Car Rental Landing Page

## 1. What does this do?
This is a complete car rental landing page for DriveEasy, featuring a hero section with a vehicle search/booking widget, vehicle categories grid (economy, SUV, luxury), why-choose-us benefits, a location coverage map section, customer reviews, a loyalty/rewards call-to-action, and a full booking form. All motion effects are powered by EaseMotion CSS framework classes.

## 2. How is it used?
Open `demo.html` in any modern browser. The page is fully responsive across mobile, tablet, and desktop. The booking widget and form are static UI demonstrations. Users can browse vehicle categories, read customer reviews, and interact with CTAs as a showcase template.

## 3. Why is it useful?
This provides a ready-to-customize template for car rental or mobility service websites. It demonstrates how to build a feature-rich landing page using only EaseMotion animation classes, keeping the CSS free of custom animation logic and making the design easy to maintain and extend.
