# Glow Pulse Art Deco Card

A responsive Art Deco themed card featuring a smooth Glow Pulse animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Glow Pulse animation
- Elegant Art Deco styling
- Gold geometric accents
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--gold
--gold-light
--background
--surface
--text
--muted
--radius
```
