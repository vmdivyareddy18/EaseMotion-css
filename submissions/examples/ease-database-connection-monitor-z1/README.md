# Ease Database Connection Monitor

## Overview
A CSS-only database monitoring component with animated connection indicators.

## Features
- Connection status indicators
- Responsive layout
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for monitoring dashboards, cloud platforms, and backend administration tools.
