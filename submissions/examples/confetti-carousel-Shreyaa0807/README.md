# Confetti Carousel (EaseMotion Variant)

A fluid, pure-CSS responsive carousel component featuring a celebratory confetti explosion animation on slide transition. Built as a standard addition to the `EaseMotion-css` ecosystem.

## Features
- **Pure CSS**: Driven completely by radio state selectors (`:checked`), removing the need for JavaScript overhead.
- **EaseMotion Sync**: Leverages core variables (`--ease-motion-timing`, `--ease-motion-duration`) for transition alignments.
- **Accessible Design**: Formatted with standard ARIA roles (`carousel`, `slide`), fully navigable by assistive devices, and responds visibly to focus indicators.

## How to Run the Demo
1. Save `demo.html` and `style.css` together in the same folder directory (`submissions/examples/confetti-carousel-<suffix>/`).
2. Open `demo.html` directly in any standard modern web browser to view the implementation.
