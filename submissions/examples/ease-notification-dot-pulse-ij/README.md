# Notification Dot Pulse

A pulsing notification indicator dot with ring animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--dot-color` | `#f72585` | Notification dot color |
| `--ring-color` | `#f72585` | Pulse ring color |
| `--dot-size` | `12px` | Dot diameter |
| `--pulse-duration` | `1.5s` | Pulse cycle duration |
