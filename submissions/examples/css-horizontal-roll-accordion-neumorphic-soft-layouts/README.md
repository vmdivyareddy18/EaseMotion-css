# CSS Horizontal Roll Accordion - Neumorphic Soft Layouts

A pure CSS animated accordion featuring a smooth horizontal roll interaction with a soft neumorphic design style.

## Features

- Pure HTML and CSS
- No JavaScript required
- Horizontal roll opening animation
- Neumorphic soft UI appearance
- Responsive layout
- Keyboard accessible using native details/summary
- Custom CSS variables for animation control


## Usage

```html
<section class="accordion">

    <details>

        <summary>
            Accordion Title
        </summary>

        <div class="content">
            Content goes here
        </div>

    </details>

</section>
