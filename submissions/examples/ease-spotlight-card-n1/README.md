# Ease Spotlight Card

A modern glassmorphism card featuring a continuous spotlight sweep animation created with pure HTML and CSS.

## Features

- Glassmorphism design
- Spotlight animation
- Responsive layout
- Pure HTML & CSS
- Zero dependencies

## Folder Structure

```
ease-spotlight-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Why EaseMotion CSS

Demonstrates a reusable hover-friendly UI pattern with a subtle animated spotlight effect that fits EaseMotion CSS's animation-first philosophy.
