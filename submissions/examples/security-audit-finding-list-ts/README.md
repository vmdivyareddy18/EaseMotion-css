# Security Audit Finding List

## What does this do?

Lists security audit findings with severity, affected item, and remediation timing.

## How is it used?

```html
<article class="finding high">
  <span>High</span><strong>Exposed token scope</strong><em>Due today</em>
</article>
```

## Why is it useful?

It gives EaseMotion CSS a focused compliance review pattern for security dashboards.
