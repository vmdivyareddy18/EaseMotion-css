# Spell Cast Burst Animation

## What does this do?
Creates a magical burst effect with expanding rings and particles radiating outward when a spell is cast.

## How is it used?
Add `.ring` and `.part` elements inside a `.burst` container. The burst animation loops continuously.

## Why is it useful?
Spell burst effects provide visual feedback for magic casting in fantasy games and interactive storytelling.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
