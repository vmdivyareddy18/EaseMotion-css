# Tab Pill Swap

A segmented tab control where the active pill settles with a short scale animation.

## Files

- `demo.html` - tab-style navigation markup with the active page.
- `style.css` - active pill animation, hover/focus states, and reduced-motion support.

## Highlights

- Uses `aria-current` for the active tab.
- Keeps the active item visually stable after the animation.
- Preserves keyboard focus visibility.
