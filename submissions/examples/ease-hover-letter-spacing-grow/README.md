# ease-hover-letter-spacing-grow

A lightweight hover animation that smoothly increases text letter spacing, creating a subtle "breathing" effect for headings, labels, and interactive text.

---

## ✨ Features

- Pure CSS
- No JavaScript
- Smooth 0.3s transition
- Customizable using CSS variables
- Ideal for headings, labels, buttons, and navigation

---

## 🚀 Usage

```html
<h1 class="ease-hover-letter-spacing-grow">
    EaseMotion CSS
</h1>
```

---

## 🎨 CSS Variables

```css
:root {
    --ease-letter-spacing-hover: 0.12em;
    --ease-letter-spacing-duration: 0.3s;
}
```
