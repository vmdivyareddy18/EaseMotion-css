CSS Zoom-In Accordion

A pure CSS animated accordion component utilizing a smooth zoom-in scale transition and grid-rows height interpolation. Fully accessible, keyboard-navigable, and responsive.

Standard Transition (300ms Bounce)
How does the Zoom-In mechanism operate?
This component combines CSS Grid grid-template-rows interpolation with a transform scale transition on the interior child. When the parent details element gains the [open] state, the track expands from 0fr to 1fr, and the scale grows seamlessly from 0.95 to 1.0.
Is it fully compliant with web accessibility guidelines?
Absolutely. By utilizing native HTML5 <details> and <summary> tags, the component exposes standard roles to assistive technologies out-of-the-box. Screen readers correctly interpret expanded state and keyboard focus can be cycled natively using the Tab key, with Enter or Spacebar triggering toggle events.
How can I customize the animation speed and scale?
You can override the behavior easily by specifying the following CSS Custom Properties:
  • --ease-accordion-zoom-duration: Adjusts transition speed (e.g. 500ms).
  • --ease-accordion-zoom-easing: Adjusts animation curve (e.g. cubic-bezier(...)).
  • --ease-accordion-zoom-scale-start: Adjusts initial scale compression (e.g. 0.85).
