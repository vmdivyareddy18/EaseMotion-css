# Ease Apple Ripen Animation

A dual-layered timeline utility built to replicate biological development cycles by pairing physical expansion boundaries together with advanced chromatic `filter` shifting matrices.

## Features
- **Hanging Axis Orientation**: Utilizes a customized `transform-origin: top center` calculation profile to simulate stem-bound branch loading.
- **Dynamic Chromatic Shifts**: Leverages precision `hue-rotate` filters to transform baseline green assets cleanly into a rich red state.
- **Three Speed Presets**: Ships out of the box containing `default`, `fast`, and `slow` growth variations.

## Core Utility Usage
```html
<div class="element ease-apple-ripen">🍏</div>
<div class="element ease-apple-ripen-fast">🍏</div>
<div class="element ease-apple-ripen-slow">🍏</div>
