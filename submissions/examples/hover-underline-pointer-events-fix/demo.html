Bug Fix #6829 — Underline ::after Blocks Clicks

The invisible 2px ::after strip intercepts clicks on elements just below nav links. pointer-events: none fixes it completely.

❌ Before Fix — ::after blocks the button below
Hover me (buggy link)
Try clicking the very top edge of the button above…
::after has no pointer-events: none
✅ After Fix — button always clickable
Hover me (fixed link)
Try clicking the very top edge of the button above…
pointer-events: none on ::after

From Center .ease-hover-underline-center
