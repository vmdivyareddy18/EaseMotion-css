# Animated Customer Testimonial Carousel

A responsive testimonial showcase component featuring animated cards, customer avatars, ratings, and smooth reveal effects.

## Features

- Animated testimonial cards
- Customer avatars
- Star ratings
- Hover interactions
- Responsive layout
- Modern SaaS-style design

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in your browser.

## Customization

You can customize:

- Customer names
- Avatar styles
- Ratings
- Colors
- Animation timing
- Layout spacing

## Use Cases

- SaaS landing pages
- Portfolio websites
- Startup websites
- Product showcases
- Customer review sections
