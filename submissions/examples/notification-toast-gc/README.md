# Notification Toast UI Example

## 1. What does this do?

This submission provides a CSS-first notification toast component with animated entrance and dismiss transitions. Each toast type (success, info, warning, error) includes icon support and motion-driven visual feedback.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example uses `style.css` for toast styling and animations. Copy the `.notification-toast` structure and CSS into your project to use the toast pattern.

---

## 3. Why is it useful?

Toasts are ideal for transient feedback and status messages. This example demonstrates how to create lightweight notification UI with strong visual hierarchy and appealing motion patterns without any JavaScript.
