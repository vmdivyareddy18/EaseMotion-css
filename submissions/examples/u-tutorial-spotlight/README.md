# ease-tutorial-spotlight

Tutorial spotlight animation that highlights UI elements while dimming the rest of the screen.

## Features

- Spotlight circle effect
- Background dimming overlay
- Huge box-shadow hole technique
- Smooth spotlight movement
- Tutorial coachmark panel
- Responsive layout
- Pure HTML and CSS
