CSS Relative Color Syntax

Deconstruct and manipulate base CSS variables directly in the browser. You can inject opacity into hex codes, alter lightness, and calculate complementary hues dynamically.

1. Alpha/Opacity Injection

Traditionally, to add opacity to a CSS variable, you had to define the variable as `255, 255, 255` and wrap it in `rgba()`. With RCS, you can pass a standard HEX code `#3b82f6` and automatically inject an alpha channel: `rgb(from var(--brand) r g b / 20%)`.

2. Lightness Math

By destructing a color into `hsl`, we can use `calc()` on the `l` (lightness) channel. Here, we calculate a much lighter background and a darker text color directly from the base `rgb(239, 68, 68)` Danger variable.

Danger Light Alert (l + 40%)

3. Hue Rotation Math

By manipulating the `h` (hue) channel, we can automatically generate analogous palettes or perfect complementary colors.
