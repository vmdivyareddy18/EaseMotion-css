# 💪 IRON PULSE · gym landing

A clean, three‑file gym project — **HTML**, **CSS** and this **README**.  
No external libraries, no frameworks, just a modern one‑pager for a fictional gym.

---

## 📁 File structure
/
├── index.html (main structure)
├── style.css (all styles, responsive)
└── README.md (you are here)


---

## 🏋️ What’s inside

- **Hero section** – bold headline, trial CTA, and a pure‑CSS dumbbell icon.
- **3 feature cards** – HIIT, strength tracking, recovery.
- **Weekly class schedule** – Monday to Saturday, with class names & times.
- **Footer** – location, opening hours, and a small tagline.

All colours, spacing, and fonts are custom‑designed for a dark, sporty vibe.

---

## 🎨 Design notes

- Font: **Inter** (Google Fonts) – clean and athletic.
- Primary accent: `#e7513a` (energetic orange‑red).
- Dark background (`#0b0d10` → `#14171c`) with subtle card layers.
- Fully responsive: works on mobile, tablet, and desktop.

---

## 🚀 How to use

1. Save all three files in the same folder.
2. Open `index.html` in any browser.
3. (Optional) Tweak the schedule or copy to build your own gym site.

No build tools, no dependencies — just open and go.

---

## 📌 Possible extensions
