## Sunrise Animation

A pure CSS recreation of a sunrise scene. The sun rises from below the horizon while the sky gradient brightens from deep night to warm dawn colors. Soft glow rays emanate from the sun as it climbs.

**Features:**
- Smooth sky gradient transition from night to dawn
- Rising sun with radial glow and halo effect
- Pseudo-horizon with curved silhouette
- Respects `prefers-reduced-motion`
