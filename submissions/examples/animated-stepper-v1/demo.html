CSS Classes & Variables

Integrate the component using clean class declarations:

<!-- Stepper Layout -->
<div class="ease-stepper">
  <!-- Active step -->
  <div class="ease-step ease-step-active" aria-current="step">
    <div class="step-marker">2</div>
    <div class="step-label">Security</div>
  </div>
</div>

Customize variables to fit your theme aesthetic:

:root {
  --ease-stepper-primary: #6366f1;       /* Active accent color */
  --ease-stepper-completed: #10b981;     /* Completed success color */
  --ease-stepper-marker-size: 40px;      /* Step circle dimensions */
  --ease-stepper-line-height: 4px;       /* Line connector height */
}
