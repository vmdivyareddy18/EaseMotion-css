# Slider Range with Dual Thumbs

## What does this do?
Adds `.ease-slider-dual` — a range slider with two thumbs for selecting a range.

```html
<div class="ease-slider-dual">
  <input type="range" min="0" max="100" value="20">
  <input type="range" min="0" max="100" value="80">
</div>
```

Open `demo.html` to see it in action.
