# Ease Black Hole Transition

A futuristic CSS animation inspired by a black hole with glowing energy rings and a cosmic hover transition.

---

## Features

- Animated black hole
- Rotating energy rings
- Neon glow effects
- Hover transition
- Responsive layout
- Pure HTML & CSS

---

```
submissions/examples/ease-black-hole-transition-v1/

```

---

## Usage

Open `demo.html` with `style.css` in the same folder.

## Browser Support

Works in all modern browsers supporting CSS animations and Flexbox.
