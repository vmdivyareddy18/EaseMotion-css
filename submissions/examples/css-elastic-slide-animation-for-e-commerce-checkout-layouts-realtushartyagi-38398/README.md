# CSS Elastic Slide Animation (E-Commerce Checkout)

## Overview
A pure CSS implementation of an elastic slide animation effect designed specifically for E-Commerce Checkout layouts.
It uses a `cubic-bezier` transition to create a bouncy, elastic motion when the user hovers over the element.

## Usage
Simply load `style.css` and view `demo.html` for the markup structure.
