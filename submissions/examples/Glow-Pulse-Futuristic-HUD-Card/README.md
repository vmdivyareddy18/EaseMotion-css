# Glow Pulse Futuristic HUD Card

A responsive Futuristic HUD card featuring a smooth Glow Pulse animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Glow Pulse animation
- Futuristic HUD interface
- Glassmorphism styling
- Animated scan line
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--glow
--primary
--secondary
--surface
--background
--text
--muted
--radius
```
