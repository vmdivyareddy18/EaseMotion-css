# Vibrant Backdrop Filter Glass (`ease-backdrop-filter-saturate`)

A showcase of how combining `saturate()` with `blur()` inside a CSS `backdrop-filter` creates highly premium, vibrant frosted glass effects.

## 🚀 Features & EaseMotion Showcase

- **Vibrant Glassmorphism**: Standard `backdrop-filter: blur()` often leaves background colors looking muddy and gray. Adding `saturate(180%)` forces the refracted colors to pop, creating an iOS-style premium glass effect.
- **Floating Environment**: The demo includes CSS-animated floating blurred orbs in the background to perfectly illustrate how the glass interacts with moving colors in real-time.
- **Hardware Accelerated**: Native browser backdrop filters leverage GPU rendering for buttery smooth performance even over complex backgrounds.

## 🛠️ Usage

This demo is self-contained. Open `demo.html` in your browser. All required CSS is inside `style.css`.
