# Text Scramble Reveal Animation

A CSS-only text reveal effect that mimics a scramble transition before revealing readable text.

## Features

- Pure CSS
- No JavaScript
- Works on headings and paragraphs
- Lightweight and reusable

## Usage

```html
<h1 class="scramble" data-text="EaseMotion CSS">
  EaseMotion CSS
</h1>
```

The effect uses blur, opacity, and letter-spacing transitions to simulate a scrambled text reveal.
