# Animated Cashback Coin Burst

## Overview

Animated Cashback Coin Burst is a lightweight CSS animation that simulates cashback rewards using a money icon and bursting coins.

## Features

- CSS-only animation
- Coin burst effect
- Reward notification animation
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="cashback-burst">
  <span class="reward">💰</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
