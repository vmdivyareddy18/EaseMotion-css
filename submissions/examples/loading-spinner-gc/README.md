# Loading Spinner UI Example

## 1. What does this do?

This submission showcases CSS-only loading spinners with multiple animation styles. No images or JavaScript required—just pure CSS animations using `@keyframes` and pseudo-elements to create smooth, performant loading indicators.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example demonstrates various spinner styles including rotating rings, pulsing dots, and gradient animations. Copy the `.loader-*` classes and CSS into your project to display loading states anywhere.

---

## 3. Why is it useful?

CSS-only spinners reduce dependencies and bundle size while providing fast, smooth animations. Multiple design options allow spinners to match any UI aesthetic, and the lightweight approach ensures minimal performance impact during loading states.
