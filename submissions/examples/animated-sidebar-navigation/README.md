# Animated Sidebar Navigation

A responsive animated sidebar navigation component built using pure HTML and CSS.

## Features

- Collapsible responsive sidebar
- Animated navigation hover effects
- Active navigation states
- Responsive mobile layout
- Smooth transitions and interactions
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated sidebar navigation component.

## Files

- `demo.html` → Sidebar navigation structure
- `style.css` → Sidebar styles and animations
- `README.md` → Documentation
