# Ease Stellar Path Navigator

## Overview

A futuristic navigation path component featuring glowing milestones, animated nodes, and a gradient connection line built entirely with HTML and CSS.

## Features

- Animated navigation nodes
- Gradient path connector
- Glassmorphism styling
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-stellar-path-navigator-z1/
├── demo.html
├── style.css

```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Introduces a reusable navigation path suitable for onboarding flows, checkout progress, deployment stages, AI workflows, and dashboard interfaces.
