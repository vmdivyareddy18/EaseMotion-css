# Animated Pricing Cards Comparison

A responsive pricing card example demonstrating subscription plans with hover animations and a featured plan highlight.

## Features

- Three pricing tiers
- Featured plan badge
- Hover lift effects
- Animated CTA buttons
- Responsive layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add or remove pricing plans
- Change colors and spacing
- Modify animation timing
- Update pricing and features
