A CSS loading animation featuring three glowing dots that bounce in sequence with a staggered delay. Each dot pulses with a neon blue glow against a dark background. Use the .loader container with .dot children. Customize via CSS custom properties --dot-size, --dot-color, --dot-gap, and --bounce-duration.
