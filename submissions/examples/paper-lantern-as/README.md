# Paper Lantern

A pure CSS and HTML string of red paper lanterns glowing and swinging on a cord at a night festival.

## What it shows

- Each lantern lit from within by a radial gradient with a warm outer glow, flickering its brightness like a real candle
- Fine paper ribs from a vertical repeating linear gradient over the round body
- A character glyph built from stacked hard stop linear gradient strokes, and a fringed tassel from a fine repeating linear gradient
- Three lanterns swinging out of phase at different scales, with embers drifting up
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
