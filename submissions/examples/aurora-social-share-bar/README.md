# Aurora Social Share Bar

## What it does

Aurora Social Share Bar provides a modern sharing component inspired by banking dashboard designs.

Features:

- Animated aurora gradient background
- Smooth hover motion
- Responsive layout
- Accessible labels
- No JavaScript required
