How to Use

Basic HTML Structure

<div class="bounce-popover" data-popover="Your text here">
  <span class="bounce-label">Hover me</span>
</div>

CSS Custom Properties (Optional)

/* Customize animation timing */
--bounce-duration: 0.45s;

/* Customize popover appearance */
--bounce-bg: #dbeafe;
--bounce-text: #0c4a6e;
--bounce-accent: #0284c7;

/* Direction variations */
.bounce-from-top
.bounce-from-bottom
.bounce-from-left
.bounce-from-right

Practical Examples

/* Form input helper */
<div class="bounce-popover bounce-from-bottom" 
     data-popover="Required field">
  <span class="info-icon">ℹ️</span>
</div>

/* Action button */
<button class="bounce-popover bounce-from-top" 
        data-popover="Save changes">
  Save
</button>

/* Custom colors and timing */
<div class="bounce-popover" 
     data-popover="Custom bounce"
     style="--bounce-duration: 0.4s;
            --bounce-bg: #fee2e2;
            --bounce-text: #991b1b;">
  Alert
</div>
