# Chameleon Color Shift

An object that smoothly shifts colors like a chameleon's skin with multiple color modes, palettes, and interactive controls.

## Features

- **5 Color Modes** – Smooth, Wave, Pulse, Reactive (mouse-responsive), Morph
- **8 Color Palettes** – Rainbow, Ocean, Sunset, Forest, Galaxy, Pastel, Neon, Monochrome
- **4 Speed Settings** – Slow, Normal, Fast, Very Fast
- **Chameleon Shape** – Stylized chameleon silhouette with color-shifting skin
- **Hexagonal Cells** – Scales that shift colors individually
- **Mouse Interaction** – Reactive mode responds to mouse position
- **Glow Effects** – Subtle glow around color-shifting cells
- **Keyboard Shortcuts** – M (mode), S (speed), P (palette), R (reset), SPACE (cycle modes)
- **FPS Counter** – Real-time frame rate display
- **Responsive** – Adapts to all screen sizes

## Technologies Used

- HTML5
- CSS3 (Animations, Gradients, Flexbox)
- Canvas API (2D context with gradients and clipping)
- Vanilla JavaScript (Color interpolation, Animation loop)

## Usage

### Controls

| Action | Result |
|--------|--------|
| "Mode: Smooth" | Cycle through 5 color modes |
| "Speed: Normal" | Cycle through 4 speeds |
| "Palette: Rainbow" | Cycle through 8 color palettes |
| "Reset" | Reset to default |
| M | Cycle modes |
| S | Cycle speeds |
| P | Cycle palettes |
| R | Reset |
| SPACE | Cycle modes |

### Color Modes

1. **Smooth** – Continuous smooth color shifting
2. **Wave** – Wave-like color patterns
3. **Pulse** – Pulsing color with brightness changes
4. **Reactive** – Responds to mouse position
5. **Morph** – Organic morphing color patterns

### Color Palettes

1. **Rainbow** – Full spectrum
2. **Ocean** – Calm blue/teal tones
3. **Sunset** – Warm sunset tones
4. **Forest** – Earthy green tones
5. **Galaxy** – Deep purple/pink tones
6. **Pastel** – Soft gentle colors
