# Gradient Stat Cards

## Overview

A responsive dashboard stat card collection with gradient accent bars, soft glow effects, and hover lift animation. Each card highlights a key metric with polished dark SaaS styling.

## Features

- Revenue, Active Users, Growth Rate, Conversion Rate, and New Signups stats
- Gradient top accent bars for variant emphasis
- Hover lift and soft glow transitions
- Responsive layout for desktop and mobile
- Pure HTML and CSS implementation

## HTML usage example

```html
<link rel="stylesheet" href="style.css">
<section class="stats-grid">
  <article class="stat-card stat-revenue">
    <div class="stat-accent"></div>
    <div class="stat-body">
      <p class="stat-label">Revenue</p>
      <h2>$92.4k</h2>
      <p class="stat-detail">Last 30 days revenue performance.</p>
    </div>
  </article>
</section>
```

## Why it fits EaseMotion CSS

This submission is self-contained and built with responsive CSS patterns only. It delivers a production-quality dashboard UI with motion-focused hover states, gradient accents, and modern spacing—all without JavaScript or external dependencies.
