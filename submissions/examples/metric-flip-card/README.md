# Metric Flip Card

A responsive dashboard card demo that uses a 3D flip interaction to reveal supporting metric context on hover or keyboard focus.

## Files

- `demo.html` contains the metric card markup.
- `style.css` contains the responsive grid, 3D flip interaction, and focus-visible styles.

## Usage

Open `demo.html` in a browser. Hover a card or tab into it to reveal the back face with extra metric details.
