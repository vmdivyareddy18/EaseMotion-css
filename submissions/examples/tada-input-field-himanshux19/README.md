# Tada Input Field

A responsive animated input field for EaseMotion CSS inspired by modern startup UI patterns.

## Features

- Pure CSS
- Responsive
- Accessible labels
- Floating label animation
- Tada focus animation
- Uses EaseMotion animation classes
- No JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<input
    class="tada-input ease-duration-400 ease-out"
    placeholder=" "
>
```

## Accessibility

- Uses proper `<label>`
- Keyboard accessible
- Visible focus state
- Responsive layout

## Animation

The component performs a subtle "Tada" animation on focus by scaling and rotating slightly before returning to its normal state.
