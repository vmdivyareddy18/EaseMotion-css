# Zoom-Out Analytics Popover

A production-quality **pure HTML5 + CSS3** Zoom-Out Popover component designed for modern dashboard interfaces.

---

## Features

- Pure HTML & CSS
- No JavaScript
- Modern Analytics Dashboard UI
- Glassmorphism design
- KPI Cards
- Revenue Metrics
- Growth Badge
- Chart Placeholder
- Hover + Keyboard Focus support
- Smooth Zoom-Out animation
- Responsive layout
- CSS Custom Properties
- Accessible semantic HTML
- ARIA attributes
- Visible focus indicators
- prefers-reduced-motion support
- Modular, commented CSS
