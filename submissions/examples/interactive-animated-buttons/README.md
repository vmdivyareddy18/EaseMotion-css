# Interactive Animated Button Variants

An advanced compilation of 5 distinct, highly interactive, modern motion buttons engineered completely using pure CSS. These components leverage performant layout techniques, pseudo-element mapping, and localized variables to create rich click/hover feedback mechanics with zero JavaScript runtime overhead.

## Included Variants
1. **Ripple Button:** Simulates a radial expanding click-wave accent directly on active frame thresholds.
2. **Shimmer Hover Button:** Sweeps an angled specular highlight gradient mask smoothly across the surface interface.
3. **Magnetic Hover Button:** Uses subtle transform displacement ranges to mimic microscopic physical floating pulling attraction.
4. **Neon Glow Button:** Casts vibrant double-layered radial box-shadow illumination outwards upon selection.
5. **Liquid Fill Button:** Triggers a fluid, rising wave element inside the button silhouette via clip-path masks.

## Configuration Tokens
Tweak or augment the behavior natively inside `style.css` using these exposed parameters:
- `--btn-transition-speed`: Global response timing offset for interactive transitions (default: `0.35s`).
- `--neon-accent-color`: Base illumination color for the glow variance (default: `#00f2fe`).
- `--liquid-wave-color`: Surface color applied to the rising liquid element structure (default: #6366f1).
