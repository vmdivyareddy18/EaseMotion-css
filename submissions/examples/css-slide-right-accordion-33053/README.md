# CSS Slide-Right Accordion for E-Commerce Checkout Layouts

## Overview

A responsive accordion component designed for e-commerce checkout pages using HTML and CSS only.

---

## Features

- Pure HTML & CSS
- CSS-only slide-right animation
- Responsive design
- Keyboard accessible
- Semantic `<details>` & `<summary>`
- CSS custom properties
- No JavaScript
