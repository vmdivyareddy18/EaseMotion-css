# Hand-Drawn Underline (#36722)

An animated sketch-style underline that draws on hover with a rough, hand-drawn aesthetic.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--underline-color` | `#f97316` | Color of the underline stroke |
| `--line-thickness` | `4px` | Thickness of the underline |
| `--draw-duration` | `0.8s` | Duration of the draw animation |
| `--sketch-roughness` | `3px` | Offset distance for the rough sketch layer |

## Interactive Demo

Open `demo.html` to see the effect in action. Use the sliders to tweak each variable and the color picker to change the underline color.
