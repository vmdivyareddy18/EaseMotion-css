# Animated Pagination Navigation

A CSS-only pagination navigation component with smooth transitions, disabled states, and hover effects.

## Features
- Previous / Next navigation buttons
- Active page highlighting with elevation shadow
- Disabled state for boundary pages
- Ellipsis truncation for large page sets
- Hover lift transitions on all interactive elements
- Fully responsive (collapses button labels on mobile)
- Accessible with aria-labels and aria-current

## Usage
Include `style.css` in your project and use the BEM-structured HTML as shown in `demo.html`.

## Browser Compatibility
Modern browsers supporting CSS Grid, Flexbox, and CSS Custom Properties.
