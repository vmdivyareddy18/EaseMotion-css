# Glassmorphism Cards

## What does this do?
CSS glassmorphism cards using backdrop-filter blur with frosted glass effect, gradient borders, interactive hover states, and a glass login form.

## How is it used?
```html
<div class="glc-card"><div class="glc-icon">🚀</div><h3>Title</h3><p>Content</p></div>
```

## Why is it useful?
Modern glass UI without JavaScript. Multiple variants (tint, dark, border, hover, glass-only). Hardware accelerated. Customizable blur/opacity/border via CSS variables.
