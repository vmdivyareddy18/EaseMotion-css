# Spinning Top

A pure CSS and HTML spinning top whirling on its point, its painted swirl blurring into rings of colour as it precesses.

## What it shows

- The painted disc from a repeating conic gradient that spins fast in place, so the colours blur into bands
- A gyroscopic wobble: an outer element precesses slowly while the swirl spins quickly, two independent timings
- The teardrop body, brass crown and metal tip built from rounded boxes and a clip-path point
- A drifting contact shadow and sparks flicking off the spinning tip
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
