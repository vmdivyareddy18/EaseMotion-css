# Shake Alert Banner

A responsive alert banner with a subtle shake animation built using pure HTML and CSS.

## Features

- Pure HTML and CSS (No JavaScript)
- Smooth shake hover animation
- Responsive layout
- Clean alert banner design
- Lightweight and reusable

## Files

```text
shake-alert-banner-ak/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<div class="alert-banner">
    <span class="icon">⚠️</span>
    <span class="message">Warning! Please check your dashboard alerts.</span>
</div>
```

The appearance, colors, and animation can be customized by modifying the CSS properties in `style.css`.

## Preview

Displays a clean warning alert banner with a subtle shake animation on hover to draw the user's attention.

## Why is it useful?

This lightweight, dependency-free component provides an engaging alert notification while following EaseMotion CSS's simple, reusable, animation-first philosophy.
