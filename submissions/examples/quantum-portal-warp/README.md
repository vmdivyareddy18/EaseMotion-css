# Quantum Portal Warp Animation Utility

## Overview
Adds a futuristic portal entrance animation using glowing expanding rings and warp distortion.

## Usage

```html
<div class="ease-quantum-portal">
  Portal Content
</div>
```

## Why it fits EaseMotion CSS
- Human-readable naming
- Animation-first design
- Reusable for cards, hero sections, modals, and loaders
