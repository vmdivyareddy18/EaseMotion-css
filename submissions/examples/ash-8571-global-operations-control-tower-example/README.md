# Global Operations Control Tower

A responsive **Global Operations Control Tower** dashboard built entirely with **HTML** and **CSS**.

## Features

- Executive operations command dashboard
- Global operations monitoring
- Operations intelligence matrix
- Fleet and logistics overview
- Operations workflow timeline
- Executive analytics cards
- Circular operational health indicator
- Responsive CSS Grid layout
- Modern enterprise control center
- No JavaScript or external libraries

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in any modern browser.

- HTML5
