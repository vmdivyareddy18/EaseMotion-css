CSS Ruby Annotation

Ruby annotations for CJK typography with ruby-align and ruby-position.

What is Ruby?

Ruby annotation is a small text placed alongside base CJK characters to indicate pronunciation or meaning. The HTML <ruby> element pairs base text (<rb>) with annotation (<rt>).
