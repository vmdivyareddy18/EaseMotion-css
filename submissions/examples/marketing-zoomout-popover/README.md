# Marketing Landing Page Zoom-Out Popover

A pure CSS animated zoom-out popover component designed for marketing landing pages.

## Features

- Pure CSS animation
- Smooth zoom-out transition
- Responsive design
- Keyboard accessible
- No JavaScript required
- Custom CSS variables


## Usage

Add the popover structure:

```html
<button class="launch-btn">

 Explore More

 <div class="popover">
   Content
 </div>

</button>
