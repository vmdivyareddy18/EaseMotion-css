# Interactive Bento Grid

A modern, dark-mode **Bento Grid** layout showcasing product features. This example demonstrates how to use CSS Grid alongside **EaseMotion CSS** to create a visually striking, responsive dashboard-style layout with smooth entrance animations and interactive hover states.

## 🚀 Features

- **Asymmetrical Layout**: Uses CSS Grid to create a dynamic "Bento box" layout (large, small, and wide cards).
- **Staggered Animations**: Uses `ease-fade-in-up` combined with `ease-delay-*` classes for a cascading entrance effect.
- **Interactive Hover States**: Uses `ease-hover-grow` combined with custom CSS to create a glowing border and lift effect on hover.
- **Dark Mode Aesthetic**: Clean, modern dark theme with gradient text and subtle shadows.
- **Fully Responsive**: Collapses into a single-column layout on mobile devices.
