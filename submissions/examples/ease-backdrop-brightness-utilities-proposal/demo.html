Backdrop Brightness Utilities

Background filter controls proposed for core/utilities.css.

These utility classes apply the backdrop-filter: brightness() property, allowing you to dim or brighten the area behind an element. This is especially useful for focus overlays, modals, and glassmorphism UI designs.

.backdrop-brightness-50

Dims the background significantly.

.backdrop-brightness-100

Default brightness (no change).

.backdrop-brightness-150

Brightens the background.

.backdrop-brightness-200

Overexposes the background.
