EaseMotion · CSS Component

CSS-Only Ripple Button

A smooth, hover-triggered ripple effect built with pure CSS — zero JavaScript, full colour customisation via CSS variables.

Colour Variants — hover each button to see the ripple

Standard variants

--blue
--green
--purple
💡 Hover any button above. The ripple expands from the centre using a CSS @keyframes animation on ::before — no JavaScript involved.

Usage

HTML snippet

<!-- link the stylesheet -->
<link rel="stylesheet" href="style.css">

<button class="ease-ripple-btn ease-ripple-btn--blue" type="button">
  <span class="ease-ripple-effect">Click Me</span>
</button>

Custom colour via CSS variables

.my-brand-btn {
  --ripple-bg: #e11d48;
  --ripple-bg-hover: #be123c;
  --ripple-text: #ffffff;
  --ripple-wave: rgba(255,255,255,0.4);
  --ripple-transition-duration: 0.5s;
}
