# Ease Elf Bow Animation

1. What does this do? Creates an animation resembling a drawn bowstring: the element pulls back slowly, holds its tension, and then violently snaps forward incredibly fast while fading out.
2. How is it used? Apply `.ease-elf-bow-ps` for an immediate firing effect, or `.ease-elf-bow-hover-ps` to trigger the shot when hovered. 
3. Why is it useful? Excellent for send buttons, "fly away" exit animations, or gamified elements where you want an exaggerated, high-velocity departure that conveys precision and speed.
