# CSS Elastic Slide Tooltip - Fitness Tracker Layouts

A pure CSS animated tooltip component designed for fitness tracker interfaces.

## Features

- Pure CSS animation
- Elastic slide transition effect
- Fitness dashboard inspired design
- Responsive layout
- Keyboard accessible
- Custom CSS animation variables


## Usage

Open `demo.html` directly in a browser.

Hover or focus on tracker buttons to display the tooltip.


## Customization

Update animation values:

```css
:root {
    --tooltip-time: 0.45s;
    --tooltip-scale: 1;
    --tooltip-easing: cubic-bezier(.68,-0.55,.27,1.55);
}
