Sticky Header

Scroll down to see the header shrink and transform

How it works

The header uses position: sticky combined with JavaScript scroll detection to toggle a compact state. When scrolled past a threshold, the header reduces its height, padding, and font size, while the logo resizes and nav items slide into place with smooth transitions.

Customization

All timing, colors, and sizing are controlled via CSS custom properties on :root. Change --sh-duration for speed, --sh-height for the default header height, and background colors for light/dark variants.

Scroll behavior

The header transitions height, padding, box-shadow, and background-color on scroll. The sticky state applies a larger shadow and a more compact layout. Content scrolls underneath while the header stays pinned to the top.

Accessibility

Reduced motion is respected. When prefers-reduced-motion is set, all header transitions are disabled to prevent vestibular disturbance.
