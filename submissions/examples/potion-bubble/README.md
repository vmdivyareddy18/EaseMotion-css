# Potion Bubble Animation

## What does this do?
A potion bottle with bubbling liquid and rising bubble particles, all in pure CSS.

## How is it used?
Add the `.potion-body` with `.liquid` and `.bubble` elements inside.

## Why is it useful?
Potion animations enhance alchemy UIs, fantasy game shops, and crafting systems.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
