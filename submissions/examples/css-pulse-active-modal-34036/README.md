# CSS Pulse-Active Modal for Interactive Interface Layouts

## Overview

This project demonstrates a responsive interactive settings modal featuring a smooth **Pulse-Active** entrance animation built entirely with HTML and CSS. The modal presents common user preferences in a clean interface with CSS-only toggle switches and a subtle pulse animation that draws attention without relying on JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- Smooth Pulse-Active entrance animation
- Responsive interactive settings modal
- CSS-only toggle switches
- Keyboard-accessible design
- Modern UI with clean layout
- CSS custom properties for easy customization
- No JavaScript or external libraries required

## Customization

Modify the CSS variables in `style.css` to customize the component.

```css
:root {
    --modal-width: 430px;
    --radius: 18px;
    --duration: 0.8s;
    --primary: #2563eb;
}
```

You can customize:

- Modal width
- Border radius
- Animation duration
- Animation easing
- Colors
- Typography
- Button appearance
- Toggle switch colors
- Shadow intensity

## Why This Component?

This component demonstrates how a CSS-only Pulse-Active animation can enhance interactive interfaces by drawing user attention to important settings while maintaining a lightweight, responsive, and accessible design. It aligns with the EaseMotion CSS philosophy of creating reusable, animation-first UI components without JavaScript.
