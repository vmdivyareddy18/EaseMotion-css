# Floating Ripple Ring Button

A CSS-only floating action button with a continuously expanding ripple ring effect.

## Features

- Pure CSS
- Infinite ripple animation
- Reduced-motion support
- Lightweight and reusable

## Usage

```html
<button class="fab">
  +
</button>
```

The ripple effect is created using a pseudo-element that expands and fades continuously around the button.
