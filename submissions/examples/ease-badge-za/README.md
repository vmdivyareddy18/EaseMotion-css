# CSS Badge / Status Component

## What does this do?
CSS badges, status dots, notification counters, and tag chips with color variants, sizes, and pill shapes.

## How is it used?
```html
<span class="bd-badge bd-primary">Label</span>
<span class="bd-status bd-online"></span>
```

## Why is it useful?
Lightweight badge system without JavaScript. Six color themes, five sizes, pill mode. Positioned badges for buttons. Status indicators. Removable tag chips.
