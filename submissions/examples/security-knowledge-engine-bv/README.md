# Autonomous Security Knowledge Engine

An interactive, premium threat correlation and vulnerability indexer dashboard showcasing advanced CSS absolute positioning, node relationship lines, and keyframe loading animations.

This dashboard component simulates a cybersecurity knowledge engine displaying correlated threat vectors (CVEs, MITRE ATT&CK tactics, and malware targets).

---

## Features

- **Correlated Node Map:** Dynamic nodes positioned via absolute coordinates, linked by pure CSS border/transform lines representing vulnerability exploit chains.
- **Vulnerability Search Filter:** Interactive logs listing simulated threat query matches and CVSS severity details.
- **Staggered Loading Progress:** Simulated scanning bars styled with horizontal slide keyframes.
- **Ambient Glow Backgrounds:** Fluid radial glows emphasizing cybersecurity enterprise theme accents.

---

## File Structure

- `demo.html` - Dashboard interface showcasing the security knowledge graph.
- `style.css` - Custom colors, card grids, rotate transforms, and connecting lines.
- `README.md` - Technical outline and documentation.
