# What does this do?

An interactive playground that allows developers to visually configure, test, and generate EaseMotion CSS classes for animations in real-time.

# How is it used?

Apply the generated animation and utility classes to your HTML elements:

```html
<div class="fade duration-500 delay-200 timing-ease-in-out iteration-infinite direction-alternate"></div>
```

# Why is it useful?

It significantly improves developer experience by eliminating manual trial-and-error, allowing developers to visually preview animation speeds, delays, directions, and timing curves before generating copy-pasteable HTML snippets for direct integration.
