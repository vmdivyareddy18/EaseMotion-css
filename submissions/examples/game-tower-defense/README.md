# 🏰 Tower Defense

A classic Tower Defense game built with HTML, CSS, and JavaScript. Build towers along a path, defend against waves of enemies, and upgrade your defenses to survive!

## 🎮 How to Play

1. **Select a tower** from the tower buttons
2. **Click** on the map to place your tower (avoid the path!)
3. Click **"Start Wave"** or press **Space** to begin
4. **Watch** enemies follow the path
5. **Earn gold** by defeating enemies
6. **Upgrade** towers by clicking on them
7. **Survive** all waves!

## ⌨️ Controls

| Key | Action |
|-----|--------|
| `1-4` | Select tower type |
| `Space` | Start wave |
| `S` | Toggle speed (1x, 2x, 3x) |
| `R` | Reset game |
| Click Tower | Upgrade tower |

## ✨ Features

- 🗼 **4 Tower Types** – Basic, Sniper, Cannon, Ice
- 📈 **Upgrade System** – Each tower can be upgraded 5 levels
- 🌊 **Wave System** – Progressive difficulty with each wave
- 🎯 **Tower Targeting** – Auto-targets nearest enemy
- ❄️ **Slow Effects** – Ice towers slow enemies
- 💰 **Gold Management** – Earn gold from kills, spend on towers
- 🏆 **Score Tracking** – Track your performance
- ⏩ **Speed Control** – Speed up the game (1x, 2x, 3x)

## 🗼 Tower Types

| Tower | Cost | Damage | Range | Special |
|-------|------|--------|-------|---------|
| Basic | 20g | 15 | 100 | – |
| Sniper | 35g | 40 | 180 | High damage, long range |
| Cannon | 50g | 30 | 120 | Splash damage |
| Ice | 40g | 8 | 110 | Slows enemies |

## 📊 Upgrade System

- Each tower can be upgraded up to **Level 5**
- Upgrades increase **damage** and **range**
- Upgrade cost increases with level

## 🎯 Scoring

- **+10 points** per enemy killed
- **+10-30 gold** per enemy killed (based on wave)
- **+50-150 gold** bonus per wave completed

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3 (Gradients, responsive design)
- Vanilla JavaScript
