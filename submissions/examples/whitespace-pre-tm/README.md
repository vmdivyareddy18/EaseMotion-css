# whitespace-pre-tm

## What does this do?
Adds utility classes to control the `white-space` CSS property for preserving whitespace in code blocks and preformatted text.

## How is it used?
```html
<div class="whitespace-pre">  code with preserved spaces</div>
<div class="whitespace-pre-wrap">wraps but preserves spaces</div>
<div class="whitespace-nowrap">never wraps</div>
```

## Why is it useful?
Essential for displaying code blocks, terminal output, and preformatted text correctly. Fills a gap in EaseMotion's text utility suite with simple, human-readable class names.
