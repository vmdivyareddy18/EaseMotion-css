# Text Reveal Calendar

A modern **Text Reveal Calendar** component built with pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Responsive layout
- Text reveal animation
- Health & Fitness inspired design
- Accessible structure
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<div class="day">
  <span>1</span>

  <small>Cardio</small>
</div>
```

Include the provided `style.css` in your project.

## Accessibility

- Responsive design
- Keyboard focus support
- High-contrast text
- Semantic content structure

## Why EaseMotion CSS?

The Text Reveal Calendar demonstrates how subtle motion can improve the presentation of calendar events using lightweight CSS animations. The reveal effect provides additional context while keeping the interface clean and engaging.
