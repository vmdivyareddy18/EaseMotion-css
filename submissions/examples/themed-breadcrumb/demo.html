Note: Toggle your OS color scheme to see dark/light mode changes. The breadcrumb uses @media (prefers-color-scheme: dark) and @media (prefers-color-scheme: light) to set text, link, and separator colors — light text on dark backgrounds, dark text on light backgrounds. Links have :focus-visible for keyboard accessibility and smooth hover transitions.
