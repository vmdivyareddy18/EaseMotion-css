# ease-gradient-progress-bar

A horizontal progress bar with gradient fill animation. The bar smoothly fills from 0 to a target percentage with a color gradient, and the percentage label updates in real time.

## Features

- Gradient-filled progress bar with smooth easing
- Real-time percentage label updates during animation
- Preset value buttons (25%, 50%, 75%, 100%) and a reset button
- Customizable via CSS custom properties

## CSS Custom Property Control

- `--gp-duration`: Transition duration for the bar width
- `--gp-height`: Height of the progress track
- `--gp-gradient-start`: Start color of the gradient fill
- `--gp-gradient-end`: End color of the gradient fill
- `--gp-track-bg`: Background color of the empty track
- `--gp-radius`: Border radius for the track and bar
- `--gp-label-color`: Color of the percentage label and heading
