# Animated Weather Icons

Animated weather icons with unique animations for each weather type.

## Features

- Six weather types: Sun, Clouds, Rain, Snow, Thunder, Wind
- Sun rotates/pulses with glowing effect
- Clouds float horizontally
- Rain drops fall with staggered timing
- Snowflakes fall with swaying motion
- Lightning flashes intermittently
- Wind lines blow across the scene
- Temperature display animates on weather change
- Fully customizable via CSS custom properties

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--wia-duration` | `0.6s` | Scene transition duration |
| `--wia-sun-color` | `#fbbf24` | Sun color |
| `--wia-cloud-color` | `#94a3b8` | Cloud color |
| `--wia-rain-color` | `#60a5fa` | Rain drop color |
| `--wia-snow-color` | `#e2e8f0` | Snowflake color |
| `--wia-bg` | `#1a1a2e` | Container background |

## Usage

Open `demo.html` in any modern browser. Click the weather type buttons to switch between animated weather scenes.

## Browser Support

Chrome, Firefox, Safari, Edge — any modern browser supporting CSS custom properties and animations.
