# CSS 3D Flip Modal – Glassmorphism UI Layout

## Overview

A responsive Glassmorphism UI modal built entirely with HTML and CSS. The component features a smooth 3D flip opening animation, frosted glass panels, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip animation
- Glassmorphism interface
- Frosted glass effect using `backdrop-filter`
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--bg`
- `--glass`
- `--border`
- `--text`
- `--muted`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Open Preview** to display the animated Glassmorphism 3D Flip Modal. Colors, animation timing, blur intensity, spacing, shadows, and flip effects can all be customized using the CSS variables defined in `style.css`.
