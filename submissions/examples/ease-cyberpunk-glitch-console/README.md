# Cyberpunk Glitch Console Layout

## Description
A retro-futuristic Cyberpunk/Matrix-styled developer terminal interface showcasing CRT scanlines, neon glowing text structures, typing cursors, and custom responsive frame borders.

## Installation
Clone and run `demo.html` in any browser.

## Usage
Simply embed inside dashboard modules to show activity logs.

## Accessibility
- Uses high-contrast glowing monospace fonts.
- Supports `prefers-reduced-motion` settings.
