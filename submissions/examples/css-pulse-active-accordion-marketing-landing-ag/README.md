# CSS Pulse-Active Accordion (Marketing Landing Page)

A bold, conversion-optimized accordion for Marketing Landing Page layouts. Uses oversized typography, full-width hero-style sections, testimonial pull-quotes, and a pulsing CTA beacon on the primary section to maximize engagement and click-through.

## Features
- **CTA Beacon Pulse:** The primary marketing section features a glowing beacon animation on its call-to-action area — a radial pulse emanating from the section border to draw the eye toward conversion.
- **Hero-Scale Typography:** Accordion headers use bold, large-scale headline text with gradient overlays — styled like marketing hero sections.
- **Testimonial Pull-Quote:** Content sections include a styled blockquote with attribution, formatted like a customer testimonial.
- **Metric Counters:** Each section includes large numerical metric displays (e.g., "10K+ Users", "99.9% Uptime") with animated reveal.
- **Full-Width Dividers:** Sections are separated by gradient dividers rather than borders, creating a landing page flow.
- **Pure CSS:** No JavaScript. Built on `<details>`/`<summary>`.

## Usage
Wrap items in `.landing-accordion`. Each `<details>` gets `.landing-section`. Add `.landing-section--primary` for the hero/CTA section.

## Files
- `demo.html`: A marketing landing page FAQ/features section.
- `style.css`: Hero typography, CTA beacon pulse, testimonial cards, and metric counters.
