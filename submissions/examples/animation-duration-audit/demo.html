⏱ Audit: Animation Duration

Scans animation-related CSS files for animation-duration and transition-duration values

Purpose

Identify all hardcoded duration values across framework CSS files and recommend migration to --ease-duration-* CSS custom properties for consistency.

What It Checks

  1. Greps css/ and src/ directories for animation-duration and transition-duration declarations
  2. Extracts distinct duration values (e.g. 0.3s, 1s, 2.5s)
  3. Reports a frequency-sorted summary
  4. Flags values that should be replaced with tokens

Run the Audit

bash submissions/examples/animation-duration-audit/audit.sh

Execute from the repository root.

Sample Output

=== Animation Duration Audit ===

css/animations.css:42:  animation-duration: 0.3s;
src/effects/fade.css:12: transition-duration: 0.2s;

=== Summary ===
      5 0.3s
      3 0.5s
      2 1s
      1 0.2s
