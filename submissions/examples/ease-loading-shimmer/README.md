# Ease Loading Shimmer

A reusable CSS shimmer loading utility for EaseMotion CSS. It provides a smooth skeleton loading animation that can be used as a placeholder while content is loading.

## Features

- Smooth shimmer animation
- Lightweight and reusable
- Pure HTML & CSS
- Responsive design
- Easy to customize
- Suitable for cards, avatars, text blocks, and buttons

## Folder Structure

```
ease-loading-shimmer/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Use the shimmer class:

```html
<div class="ease-loading-shimmer"></div>
```

## Customization

You can adjust:

- Animation duration
- Background color
- Border radius
- Element width and height
