# Multi-Column Text Layout

## What does this do?
CSS multi-column text layouts with column-count, column-width, column-rule, column-gap, and drop caps for magazine-style typography.

## How is it used?
```html
<div class="mct-cols mct-two"><p>Text content flows into columns automatically.</p></div>
```

## Why is it useful?
Magazine-style text layout without JavaScript. Two/three/auto column variants. Column rules and drop caps. Responsive column-width for adaptive layouts.
