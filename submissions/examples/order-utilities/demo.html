Order Utilities

CSS order utilities to rearrange child elements within Flexbox and Grid containers, independent of their physical DOM source order.

1. Original DOM Order Default

Elements are displayed in the exact sequence they are written in the HTML markup. All cards have the default order (equivalent to .order-none).

1 Card One
2 Card Two
3 Card Three
4 Card Four
5 Card Five
2. Rearranged Order (.order-1 to .order-5) Reordered

Using the numeric order utilities to completely reverse the visual sequence of cards, without modifying the underlying HTML order.

1 .order-5
2 .order-4
3 .order-3
4 .order-2
5 .order-1
3. Order First (.order-first) .order-first

Applying .order-first to Card Four shifts it to the absolute beginning of the layout container because its order is set to -9999.

Applying .order-last to Card Two pushes it to the absolute end of the layout container because its order is set to 9999.
