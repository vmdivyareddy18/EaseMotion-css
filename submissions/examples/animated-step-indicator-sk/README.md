# Animated Step Indicator

A CSS-only horizontal step progress indicator with completed, active, upcoming, and error states.

## Features
- Horizontal steps connected by animated lines
- Four states: completed (filled purple), active (pulsing ring), upcoming (outlined), error (red pulse)
- Checkmark icon on completed steps
- Smooth spring transitions on active step scale
- Error step with pulse animation
- Fully responsive (scrolls on mobile)
- Accessible with aria-current support

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`. Add modifier classes: `--completed`, `--active`, or `--error` to `.steps__step`.

## Browser Compatibility
Modern browsers supporting CSS Flexbox, Transitions, Animations, and Custom Properties.
