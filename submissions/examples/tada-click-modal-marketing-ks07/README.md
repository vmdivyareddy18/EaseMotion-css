# Tada Click Modal for Marketing Landing Page

## 1. What does this do?
Creates a responsive modal with a smooth Tada click animation using only HTML and CSS.

## 2. How is it used?
Open `demo.html` in a browser and click the "Open Modal" button to view the animation.

## 3. Why is it useful?
It provides an attractive, reusable modal effect for marketing landing pages without requiring JavaScript.
