# Animated FAQ Accordion

A responsive animated FAQ accordion component built using pure HTML and CSS.

## Features

- Expandable and collapsible sections
- Smooth height animations
- Icon rotation effects
- Responsive layout
- Hover interactions
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated FAQ accordion component.

## Files

- `demo.html` → FAQ accordion structure
- `style.css` → Accordion animations and styles
- `README.md` → Documentation
