# Admin Dashboard Sidebar Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive admin dashboard sidebar.

---

## Components

- Sidebar Navigation
- Dashboard Header
- Statistics Cards
- Sidebar Toggle
- Responsive Layout

---

## Folder Structure

```
submissions/examples/admin-sidebar-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Click the menu button to show or hide the sidebar.
3. Resize the browser window to test responsiveness.

---

## Features

- Responsive admin dashboard
- Sidebar navigation
- Dashboard statistics cards
- Minimal JavaScript
- Uses existing EaseMotion CSS animation classes
