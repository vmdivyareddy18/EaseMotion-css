# ✂️ CSS Divider Motion

A collection of beautiful, pure CSS animated dividers and separators for modern web interfaces.

## 🚀 Features

### Divider Styles (10 types)
1. **Gradient Line** - Flowing gradient animation
2. **Dashed Animated** - Moving dashed line
3. **Dots Pattern** - Scrolling dot pattern
4. **Wave SVG** - Animated SVG wave
5. **Zigzag Pattern** - Moving zigzag design
6. **Icon Center** - Lines with rotating icon
7. **Neon Glow** - Pulsing neon glow effect
8. **Pulse Line** - Expanding pulse animation
9. **Gradient Fade** - Fading gradient ends
10. **Animated Border** - Shine effect with text

### Animation Features
- Gradient flow animations
- Pattern scrolling
- SVG wave motion
- Icon rotation
- Neon pulsing
- Line expansion
- Shine sweep effects
- Smooth transitions

## 📂 Files
- `demo.html` - Main HTML structure
