# 💬 CSS Tooltip Motion

A collection of beautiful, pure CSS animated tooltips for modern web interfaces.

## 🚀 Features

### Basic Tooltips (4 positions)
- Top tooltip (default)
- Bottom tooltip
- Left tooltip
- Right tooltip

### Animated Tooltips (4 effects)
- Fade In - Smooth opacity transition
- Scale Up - Grows from small to full size
- Slide In - Slides into position
- Bounce - Playful bounce animation

### Styled Tooltips (4 styles)
- Neon Glow - Cyberpunk neon effect
- Gradient - Vibrant gradient background
- Rounded - Pill-shaped tooltip
- Custom Arrow - Larger arrow indicator

### Multi-line Tooltips
- Wide tooltip for longer text
- Multi-line with line breaks

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All tooltip animations
- `readme.md` - Documentation

## 🛠️ How to Use

### Basic Tooltip
```html
<button class="btn tooltip" data-tooltip="Your tooltip text">
    Hover Me
</button>
