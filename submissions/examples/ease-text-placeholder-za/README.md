# Text Placeholder Loading

## What does this do?
CSS placeholder loading effects including shimmer lines, pulse animation, gradient shimmer text, and avatar+card skeletons.

## How is it used?
```html
<div class="tph-line tph-shimmer" style="width:100%"></div>
<p class="tph-grad">Loading text...</p>
```

## Why is it useful?
Lightweight content loading placeholders without JavaScript. Shimmer and pulse variants. Avatar, text, and card patterns. Accessible reduced-motion support.
