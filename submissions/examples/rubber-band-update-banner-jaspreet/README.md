# Retro Rubber Band Update Banner

A pure HTML/CSS responsive update banner component built for the EaseMotion library using a classic 8-bit retro gaming design pattern.

## Features
- Pure CSS animation without JavaScript dependencies
- Navigation-ready structural anchor links styled identically to tactile native push buttons
- Clean retro style with thick borders, pixel text fallback fonts, and a strict color palette
- Highly responsive fluid flexbox structure shifting from row layout to stacked column view on smaller screens
- Keyboard accessible layout using explicit semantic outline indicators
- Smooth macro-interactions on banner hover and micro-interactions on action items
