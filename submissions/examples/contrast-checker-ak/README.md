# WCAG Color Contrast Checker

## What does this do?
A color contrast checker that evaluates foreground/background color pairs against WCAG AA and AAA standards for normal and large text.

## How is it used?
Open demo.html in a browser. Pick foreground and background colors using the color pickers or enter hex values. The contrast ratio and WCAG pass/fail status update in real-time.

## Why is it useful?
This helps designers and developers ensure their color combinations meet accessibility standards. It demonstrates color math, luminance calculation, and WCAG compliance checking.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to test color combinations.
