EaseMotion is a lightweight CSS utility library that provides drop-in animations and transitions for modern web interfaces. It emphasizes performance, accessibility, and ease of use.
You can install EaseMotion via npm with npm install easemotion or include it directly from a CDN. Refer to the getting started guide for detailed instructions.
