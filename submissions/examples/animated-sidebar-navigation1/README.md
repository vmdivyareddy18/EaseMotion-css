# Animated Sidebar Navigation

A clean, modern, and fully responsive **Sidebar Navigation Menu** perfect for dashboards, admin panels, and content management systems. This example demonstrates how to combine **EaseMotion CSS** for smooth slide-in entrances, staggered menu item animations, and interactive hover effects to create a polished, professional navigation component.

## 🚀 Features

- **Smooth Sidebar Entrance**: Uses `ease-slide-in-left` to elegantly introduce the sidebar from the left side of the screen.
- **Staggered Menu Items**: Each navigation item fades in sequentially using `ease-fade-in-right` and `ease-delay-*` for a premium, cascading load effect.
- **Interactive Hover States**: Menu items smoothly translate right and change background color on hover, with icons scaling up using `ease-hover-grow`.
- **Animated Active Indicator**: The active menu item features a left border indicator that smoothly animates in on page load.
- **Notification Badges**: Includes both regular and "new" badges with a continuous pulse animation for urgent notifications.
- **User Profile Section**: A clean footer section showing the logged-in user with hover effects.
- **Fully Responsive**: On mobile devices, the sidebar transforms into a bottom navigation bar with icon-only menu items.
- **Dark Theme**: Professional dark sidebar with clean typography and subtle shadows.
