# Tooltip Icon

## What does this do?
Tooltip Icon component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"ti-primary\">Tooltip Icon Primary</div>
```

## Why is it useful?
Provides a reusable tooltip icon UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
