# CSS Loading Button

## What does this do?
Buttons with CSS loading states including spinners, dot animations, progress bars, and success/error indicators.

## How is it used?
```html
<button class="lb-btn lb-loading lb-spinner">Save</button>
<button class="lb-btn lb-success">Done</button>
```

## Why is it useful?
Provides visual feedback for async operations without JavaScript. Five loading animations. Success/error states. Four sizes. Progress bar variants. Accessible aria-busy ready.
