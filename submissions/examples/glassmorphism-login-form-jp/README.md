# Glassmorphism Login Form Component

## Description

A modern glass-style authentication form featuring transparent blurred backgrounds, smooth entrance animations, interactive input focus effects, and animated buttons.

---

## Features

- Glassmorphism UI design
- Frosted glass blur effect
- Smooth fade-up entrance animation
- Animated input focus effects
- Hover effects for buttons
- Responsive layout for mobile devices
- Reduced motion accessibility support

---

## Classes

- `.glass-card` – Main glass container
- `.login-form` – Authentication form
- `.input-group` – Input field wrapper
- `.login-btn` – Animated login button
- `.extra-links` – Additional account links

---

## Accessibility

Supports `prefers-reduced-motion` to reduce animations for users who prefer less motion.

---

## Browser Support

Works in modern browsers supporting `backdrop-filter`.
