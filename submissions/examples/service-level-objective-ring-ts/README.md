# Service Level Objective Ring

## What does this do?

Shows service level objective attainment with healthy, warning, and breached ring states.

## How is it used?

```html
<section class="slo healthy">
  <div class="ring" style="--score: 94%"><span>99.94%</span></div>
</section>
```

## Why is it useful?

It gives EaseMotion CSS a reliability dashboard pattern for SLO targets and error budget status.
