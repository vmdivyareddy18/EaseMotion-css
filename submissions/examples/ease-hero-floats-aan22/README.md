# Ease Animated Hero Section with Floating Elements

## What does this do?
A complete hero section component featuring a headline with animated gradient text, animated subtitle, CTA buttons with hover effects, and decorative floating UI elements that bob and rotate in the background. The floating elements (cards, badges, icons) orbit and float at different speeds creating a dynamic 3D-like background.

## How is it used?
```html
<section class="ease-hero">
  <div class="ease-hero__floats">
    <div class="ease-hero__float ease-hero__float--1">⚡ Fast</div>
  </div>
  <div class="ease-hero__content">
    <div class="ease-hero__badge">⚡ EaseMotion CSS v2.0</div>
    <h1 class="ease-hero__title">
      Animate Anything<br/>
      <span class="ease-hero__title--gradient">Effortlessly</span>
    </h1>
    <p class="ease-hero__subtitle">Description text here.</p>
    <div class="ease-hero__cta">
      <button class="ease-hero__btn ease-hero__btn--primary">Get Started ↗</button>
      <button class="ease-hero__btn ease-hero__btn--outline">View Examples</button>
    </div>
  </div>
</section>
```

## Why is it useful?
Hero sections are the most important section of any landing page. This fits EaseMotion's animation-first philosophy — floating decorative elements create visual depth and make hero sections feel alive and premium without any external libraries.
