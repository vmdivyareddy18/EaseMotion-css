# Slide-out Drawer / Off-canvas Navigation Panel

A left-sliding off-canvas navigation drawer with a semi-transparent overlay backdrop. Opening and closing is controlled by JavaScript toggling `.open` classes. The drawer slides in from the left edge with a smooth `transform` transition.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click "Open Drawer" to slide the panel in from the left. Click the backdrop or the close button to dismiss.

## Accessibility notes

The backdrop and close button provide multiple ways to dismiss the drawer. Reduced motion disables the slide transition entirely.
