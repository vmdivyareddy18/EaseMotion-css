# Floating Action Menu UI Example

## 1. What does this do?

This submission provides an expanding floating action menu with animated child actions that appear on hover or focus. It creates a compact floating button that reveals additional action buttons using motion and scale transitions.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes a floating menu with primary and child actions. Copy the `.floating-action-menu` markup and CSS into your project to use a responsive floating action button with animated expansion.

---

## 3. Why is it useful?

Floating action menus save screen space while exposing important actions on demand. Animated expansion makes the interactions feel intuitive and keeps the UI clean when the menu is inactive.
