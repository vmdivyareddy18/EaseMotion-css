# Neumorphic UI Components

## What does this do?
A complete set of neumorphic (soft UI) components with extruded and inset shadow effects for a tactile interface aesthetic.

## How is it used?
```html
<button class="nm-btn nm-btn-primary">Button</button>
<div class="nm-card nm-card-extruded">Card content</div>
```

## Why is it useful?
Provides a modern soft UI design system with buttons, cards, toggles, sliders, and inputs all using CSS layered shadows. Customizable via CSS variables, no JavaScript required.
