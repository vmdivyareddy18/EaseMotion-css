3D Interactive Hourglass Timer

An animation-first, interactive, pure CSS sand timer utilizing 3D rotation transitions and synchronized draining keyframes.
How it works: The hourglass uses two separate local coordinate animations. Draining sand shrinks towards the neck, while accumulating sand grows from the base. Clicking directly on the Hourglass triggers a 3D flip (checkbox hack), which physically swaps the bulbs and swaps the animations, restarting the flow. You can use the controls in the dashboard to change the Timer Limit (5s, 10s, or 20s). The ambient alert border and status dot will flash red as soon as the sand finishes—all natively in pure CSS.
