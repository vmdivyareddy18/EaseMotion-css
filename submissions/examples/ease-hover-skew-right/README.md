# ease-hover-skew-right

A hover animation utility that skews an element slightly to the right.

## Features

- Smooth hover transition
- Uses `skewX(5deg)`
- Lightweight CSS utility
- Pairs with `ease-hover-skew-left`

## Usage

```html
<div class="ease-hover-skew-right">
  Hover Me
</div>
```

## CSS

```css
.ease-hover-skew-right:hover {
  transform: skewX(5deg);
}
```

## Use Cases

- Buttons
- Cards
- Navigation items
- Interactive UI elements
