# Course Module Progress Map

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive learning progress component that visually maps course modules and their completion status for e-learning platforms.

## Features

- Course progress timeline
- Completed, active, locked, and upcoming modules
- Progress status badges
- Timeline connectors
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="module active">

    <div class="indicator">
        2
    </div>

    <div class="details">
        <h3>CSS Fundamentals</h3>
    </div>

</div>
```

## Main Classes

```css
.progress-map
.module
.completed
.active
.indicator
.details
.status
.connector
```
