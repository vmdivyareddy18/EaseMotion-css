# Movie Card Showcase

## What does this do?
A responsive grid of movie cards with poster placeholders, title, year, star ratings, and hover overlay effects displaying a synopsis. Movies are rendered from a JavaScript data array.

## How is it used?
Open demo.html in a browser to see a grid of movie cards. Hover over any card to reveal a synopsis overlay. The grid is fully responsive.

## Why is it useful?
This demonstrates a movie/showcase card UI pattern suitable for streaming apps, review sites, or portfolio galleries. Includes responsive grid layout and hover interactions.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see the movie card showcase.
