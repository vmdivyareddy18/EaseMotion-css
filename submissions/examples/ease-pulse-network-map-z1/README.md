# Ease Pulse Network Map

## Overview

A futuristic network visualization featuring glowing connection lines, pulsing nodes, and a central hub using pure HTML and CSS.

## Features

- Animated network links
- Pulsing nodes
- Central hub indicator
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text

```

Open `demo.html` in any modern browser.

Optimized for desktop, tablet, and mobile devices.
