Automatic CHANGELOG Generator

Generate release-ready CHANGELOG.md files automatically from merged Pull Request titles and labels using GitHub Actions.

View Workflow

Overview

This example demonstrates a documentation page for an automated CHANGELOG generator. The workflow collects merged Pull Requests, categorizes them by labels such as Animation, Components, Bug Fixes, and Documentation, and produces a clean release-ready CHANGELOG following the Keep a Changelog format.

GitHub Actions Workflow 

name: Generate CHANGELOG

on:
  workflow_dispatch:

jobs:
  changelog:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Read Merged Pull Requests
        run: python scripts/generate_changelog.py

      - name: Commit CHANGELOG
        run: git add CHANGELOG.md
