# Gradient Fill Bar (#36691)

Gradient fill bar with animated gradient sweep in a dark-themed card layout.

## CSS Custom Properties

| Property           | Default     | Description                   |
|--------------------|-------------|-------------------------------|
| `--bar-bg`         | `#2a2a3e`   | Background color of the bar   |
| `--gradient-start` | `#ff6b6b`   | Start color of the fill       |
| `--gradient-end`   | `#4ecdc4`   | End color of the fill         |
| `--fill-duration`  | `2s`        | Duration of sweep animation   |

## Interactive Controls

- Color picker for bar background
- Color pickers for gradient start/end
- Range slider for sweep speed
- Range slider to adjust fill progress
