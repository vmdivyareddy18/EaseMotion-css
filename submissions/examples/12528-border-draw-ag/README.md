# Border Draw Effect

1. What does this do? Animates the border lines drawing sequentially around an element on hover (Top → Right → Bottom → Left).
2. How is it used? Apply the `.border-draw` class to a relatively positioned element:

   ```html
   <button class="border-draw">Hover Me</button>
   ```

3. Why is it useful? It provides a highly polished, interactive micro-interaction for cards, buttons, or navigation links without adding extra HTML nodes or relying on SVG stroke animations.
