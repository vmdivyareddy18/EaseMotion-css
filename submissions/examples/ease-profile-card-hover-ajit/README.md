# Ease Profile Card Hover

A CSS-only profile card component with a smooth hover reveal animation.

## Features

- CSS-only implementation
- Profile information reveal on hover
- Smooth image zoom effect
- Animated content panel
- Social media icons
- Responsive layout
- No JavaScript required

## Folder Structure

```
ease-profile-card-hover-ajit/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<div class="ease-profile-card-hover">

    <img src="profile.jpg" alt="Profile">

    <div class="profile-content">
        <h2>John Doe</h2>
        <span>Frontend Developer</span>
        <p>Profile description</p>
    </div>

</div>
```

Simply open `demo.html` in any modern browser.
