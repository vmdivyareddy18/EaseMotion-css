# Ease QR Code Scanner Frame

A modern **QR Code Scanner Frame** built using **pure HTML and CSS**. It features an animated scanning line, stylized scanner corners, responsive design, and customizable styling using CSS custom properties.

## Features

- Pure HTML & CSS
- Animated QR scanner frame
- CSS-only scanning line animation using `@keyframes`
- Scanner corner highlights
- CSS Custom Properties for easy customization
- Responsive design
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
ease-qr-code-scanner-frame/
├── demo.html
├── style.css
└── README.md
```

## Customization

Modify the CSS variables in the `:root` section to customize:

- Scanner color
- Background color
- Border radius
- Shadow
- Animation speed

## Browser Support

Compatible with all modern browsers.
