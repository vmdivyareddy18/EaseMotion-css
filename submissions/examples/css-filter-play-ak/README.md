# CSS Filter Effects Playground

## What does this do?
An interactive CSS filter playground with 8 adjustable filters (blur, brightness, contrast, grayscale, hue-rotate, saturate, sepia, invert) applied to a gradient image with side-by-side comparison.

## How is it used?
Open demo.html in a browser. Drag any slider to apply the corresponding CSS filter to the preview. The original unfiltered image is shown alongside for comparison. Click "Reset All" to restore defaults.

## Why is this useful?
This provides a visual tool for experimenting with CSS filter combinations. It demonstrates real-time filter application and CSS property string construction.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to experiment with filters.
