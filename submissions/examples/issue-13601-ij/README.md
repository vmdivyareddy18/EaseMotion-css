# ease-aspect-portrait

A 3:4 portrait aspect ratio utility for profile cards and image containers.

## Usage

Include the `style.css` in your project and apply the class:

```html
<div class="ease-aspect-portrait">Portrait content</div>
```

## CSS Reference

```css
.ease-aspect-portrait {
  aspect-ratio: 3 / 4;
}
```

## Demo

Open `demo.html` in a browser to see the component in action.
