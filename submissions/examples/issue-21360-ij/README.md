# Dog Wagging Tail

1. What does this do? A CSS-only dog silhouette whose tail wags rapidly from side to side using a rotate animation.

2. How is it used? Place .dog inside .scene. The body is a rounded rectangle; the tail is a pseudo-element that rotates around its bottom center.

3. Why is it useful? Shows how a single pseudo-element and transform-origin can create lively tail-wagging motion with minimal CSS.
