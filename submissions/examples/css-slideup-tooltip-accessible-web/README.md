# CSS Slide-Up Tooltip – Accessible Web Layout

## Overview

A responsive Accessible Web tooltip component built entirely with HTML and CSS. The component features a smooth slide-up animation, accessibility-focused feature cards, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Slide-Up tooltip animation
- Accessible Web interface
- Responsive feature cards
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--background`
- `--surface`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--slide-distance`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any **Learn More** button to display the animated Slide-Up Tooltip. Customize colors, spacing, animation timing, slide distance, and shadows using the CSS variables defined in `style.css`.
