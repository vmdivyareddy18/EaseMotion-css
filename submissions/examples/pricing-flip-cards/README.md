# CSS Only Flip Pricing Cards

A modern **CSS-only pricing card component** featuring a Monthly/Yearly pricing toggle without using JavaScript.

## Features

- 🎨 Glassmorphism design
- 💳 Three pricing cards
- 🔄 Monthly / Yearly pricing switch
- ✨ Hover lift animation
- 📱 Fully responsive
- 🚫 No JavaScript required
- 🎯 Pure HTML & CSS

## Folder Structure

```
pricing-flip-cards/
├── demo.html
├── style.css
└── README.md
```

## How to Run

1. Open `demo.html` in any modern browser.
2. Click the Monthly/Yearly toggle.
3. Hover over the cards to view the animation.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Transforms
- CSS Transitions
- Glassmorphism UI
