# Ease Holographic Button

A futuristic animated button component built with pure **HTML** and **CSS**. It combines holographic lighting, RGB gradients, animated scan lines, glassmorphism, and smooth hover interactions to create a premium user interface element.

---

## ✨ Features

* Holographic button effect
* RGB animated gradient
* Scan line animation
* Shine sweep hover effect
* Glassmorphism cards
* Floating background orbs
* Responsive design
* Pure HTML & CSS
* No JavaScript required

## 🚀 Installation

Open `demo.html` in your browser with `style.css` in the same folder.

## 🌐 Browser Support

Works in all modern browsers supporting CSS animations, Flexbox, CSS Variables, and Backdrop Filter.
