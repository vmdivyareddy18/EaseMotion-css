# Ease Floating Theme Switcher

## Overview

A modern theme selection panel featuring floating color indicators, smooth hover animations, and glassmorphism styling using only HTML and CSS.

## Features

- Floating color indicators
- Glassmorphism panel
- Animated pulse effect
- Smooth hover motion
- Responsive layout
- Pure HTML & CSS

## Folder Structure

```text
ease-floating-theme-switcher-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Designed for desktop, tablet, and mobile screens.

Provides a reusable animated settings component demonstrating clean motion design, lightweight CSS, and modern UI patterns suitable for dashboards and developer tools.
