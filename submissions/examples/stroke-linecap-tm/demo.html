SVG Stroke Line Cap

Defines the shape of the ends of SVG strokes: butt, round, or square.

CSS Property: stroke-linecap

Variants Demo

stroke-linecap: butt SVG Stroke Line Cap variant
stroke-linecap: round SVG Stroke Line Cap variant
stroke-linecap: square SVG Stroke Line Cap variant

Usage

<div class="stroke-linecap">Content here</div>
