# React Interactive Animation Playground

This is a premium, interactive playground designed to showcase the power of EaseMotion CSS within a React environment.

## Features
- **Standalone Execution**: Uses Babel standalone and React CDNs. You can just open `demo.html` in any browser! No node server required.
- **Live Preview**: See animations applied to components instantly as you tweak settings.
- **Dynamic Controls**: Adjust animation types, speeds (100ms - 3000ms), timing curves, and hover triggers.
- **Live Code Generation**: The playground automatically builds the correct React JSX based on your configuration so you can easily copy and paste it into your own projects.
- **Premium UI**: Designed with modern glassmorphism panels, dark mode, and responsive CSS grid.

## How to run
Simply open `demo.html` in your favorite modern browser (Chrome, Firefox, Safari, Edge).

## Why it fits EaseMotion CSS
It demonstrates how EaseMotion's zero-dependency utility classes map beautifully to declarative component abstractions (like `<Animate>` and `<Hover>`). This playground lowers the barrier to entry for React developers, encouraging them to adopt EaseMotion CSS in modern JSX stacks.
