# Floating Orbit Modal – SaaS Showcase

A modern SaaS-themed Floating Orbit Modal built entirely with HTML and CSS for the EaseMotion CSS collection.

## Features

- Pure HTML & CSS
- Floating orbit animation
- SaaS showcase interface
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- Zero JavaScript
- Smooth scale entrance
- Lightweight component

## Folder Structure

```
floating-orbit-modal-saas-showcase/
├── demo.html
├── style.css
└── README.md
```

## CSS Variables

```css
:root{
--primary:#2563eb;
--secondary:#38bdf8;
--duration:.65s;
--orbit-speed:8s;
--radius:22px;
}
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in any browser.
