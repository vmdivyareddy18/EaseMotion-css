# DTC Tea E‑Commerce Website

This demo site showcases a calm, elegant tea culture aesthetic using **EaseMotion CSS classes** only.

## 🧩 Sections
- Hero with calming tea imagery
- Collections by type (green, herbal, oolong)
- Tea sourcing origins story
- Brewing guide
- Subscription CTA
- Reviews
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — entrance fade animations
- `ease-slide` — sliding entrance for headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs and images
- `ease-hover-opacity-reduce` — hover dimming for collection cards and reviews

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll through sections to see entrance animations.
3. Hover over buttons, cards, and testimonials to see interactive effects.
4. Replace placeholder images and text with real tea e‑commerce content.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion animation classes
