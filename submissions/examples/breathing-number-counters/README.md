# Breathing Number Counter

A responsive **Breathing Number Counter** component inspired by Education Platform dashboards for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript required
- Smooth breathing animation
- Animated number highlight
- Responsive layout
- Accessible structure
- Uses EaseMotion-style `ease-*` animation class

## Folder Structure

```text
breathing-number-counter-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<div class="counter-card ease-breathing">
  <h2>Total Students</h2>
  <div class="counter">25,000+</div>
  <p>Learning Worldwide</p>
</div>
```

## Animations

- `ease-breathing`
- Number glow animation

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component is lightweight, reusable, responsive, and follows EaseMotion CSS's animation-first philosophy using only CSS.
