# Music Player Playlist

## What does this do?
A music player component with a track playlist, play/pause/next/prev controls, a seekable progress bar, volume slider, and track selection from the list.

## How is it used?
Open demo.html in a browser. Click play to start playback, use next/prev to navigate tracks, click any playlist item to select a track, and adjust volume with the slider.

## Why is this useful?
Demonstrates a complete audio player UI pattern with playlist management and simulated playback controls.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
