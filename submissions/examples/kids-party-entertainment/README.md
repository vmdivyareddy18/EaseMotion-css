# 🎉 Kids Party Entertainment Landing Page

A colorful and responsive business landing page for a Kids Party Entertainment service built using EaseMotion CSS.

## ✨ Features

- Hero section with booking CTA
- Services section
- Party package pricing
- Entertainment gallery
- Parent testimonials
- Booking form
- Responsive footer

## 📁 Project Structure

```
kids-party-entertainment/
├── demo.html
├── style.css
└── README.md
```
