# Number Typing Animate

A terminal-style number display with blinking cursor.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--text-color` | `#4ecdc4` | Number text color |
| `--cursor-color` | `#4ecdc4` | Cursor color |
| `--type-speed` | `1.5s` | Typing rhythm speed |
| `--bg-color` | `#2d2d44` | Display background |
