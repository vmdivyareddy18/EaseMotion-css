# Morphing Blob Background

## What does this do?

Creates an animated organic blob background using pure CSS morphing border-radius animations, gradients, and blur effects.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="blob blob-one"></div>
<div class="blob blob-two"></div>
<div class="blob blob-three"></div>
```

---

## Why is it useful?

Morphing blob backgrounds are widely used in hero sections, SaaS landing pages, portfolio websites, login screens, and modern dashboards. This standalone component demonstrates advanced CSS animation techniques while remaining lightweight, reusable, and framework-independent.
