# Food Menu Restaurant

## What does this do?
A restaurant menu display with categorized items (starters, mains, desserts, drinks), filterable by category tabs, showing item names, descriptions, and prices.

## How is it used?
Open demo.html in a browser. Click category tabs to filter menu items. Each item shows a name, description, and price.

## Why is this useful?
Demonstrates a filterable restaurant menu layout suitable for food service websites or digital menu boards.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
