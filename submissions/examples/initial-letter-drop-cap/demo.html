CSS initial-letter

Native drop caps without pseudo-elements or span wrappers

Raise: 3, Sink: 1

The initial-letter property lets you create typographic drop caps with just one CSS declaration. This paragraph uses a drop cap raised three lines and sunk one line deep. The effect is purely decorative and respects the user's prefers-reduced-motion setting.

Raise: 4, Sink: 2

A larger drop cap spans four lines of text. This style suits more dramatic editorial layouts. The ::first-letter pseudo-element is the traditional approach, but initial-letter gives finer control over sizing and alignment without extra markup.

Raise: 2, Sink: 1

Drop caps work best with serif or display typefaces where the letterform has strong visual character. A subtle two-line drop cap adds a refined editorial feel without overwhelming the text block. Use them sparingly on opening paragraphs.

Browser support

initial-letter is supported in Safari 16.4+ and Chrome 123+ (behind flag). For cross-browser fallback, this demo also includes a ::first-letter approach for browsers that do not yet support the property.
