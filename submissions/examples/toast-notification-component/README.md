# Toast Notification

Animated toast notification component with slide-in animation — pure CSS.

## Features
- Slide-in from right via translateX keyframes
- Error variant with red accent
- Hover to pause animation
- Close button with hover effect

## Usage
```html
<div class="toast">
  <div class="toast-icon">✓</div>
  <div class="toast-content"><strong>Success!</strong><p>Message</p></div>
  <span class="toast-close">×</span>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
