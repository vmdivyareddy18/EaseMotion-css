# Orbit Lightbox

Issue: **#42527**

## Overview

Orbit Lightbox is a responsive Pure CSS lightbox component created for the EaseMotion CSS library. It features an orbit-style entrance animation inspired by modern corporate UI patterns.

## Features

- Pure CSS implementation
- No JavaScript required
- Orbit entrance animation
- Uses EaseMotion `ease-*` naming convention
- Responsive design
- Keyboard accessible
- CSS variables for customization
- Smooth transitions
