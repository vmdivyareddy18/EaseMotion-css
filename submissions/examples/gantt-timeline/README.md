# Animated Gantt Timeline Bar Collection

An advanced project roadmap visualization layout tracking task durations across custom grid systems via hardware-accelerated timeline extensions.

## Architecture Selectors
- `.ease-gantt-container` : The master context container organizing nested coordinate rows.
- `.ease-gantt-track` : Individual structural tracks isolating overlapping progress layers.
- `.ease-gantt-bar` : The timeline graphic vector node computing progress thresholds and hover scales.

- `.ease-span-45`
- `.ease-span-70`
