# Alert Dialog Modal

Modal dialog with backdrop overlay and multiple alert types (info, warning, error, success). Scales in with a fade overlay for smooth appearance.

## Features

- Four dialog types with colored top borders
- Overlay fade-in with backdrop click to dismiss
- Scale + opacity entrance animation
- Confirm and Cancel buttons
- Stack multiple dialogs on top of each other
- Customizable via CSS custom properties

## Usage

Click a type button to open that dialog. Click Confirm, Cancel, or the overlay to dismiss. Use Stack Dialog to layer multiple dialogs.
