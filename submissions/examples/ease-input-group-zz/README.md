# Input Group

## What does this do?
Input Group component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"ig-primary\">Input Group Primary</div>
```

## Why is it useful?
Provides a reusable input group UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
