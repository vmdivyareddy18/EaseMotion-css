# Quotes Auto

quotes — Sets custom quotation mark characters for different languages

## Features
- Uses `--ease-color-text`
- Uses `--ease-color-muted`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Quotes Auto</span>
</div>
```

## Why is it useful?

This submission demonstrates the quotes auto CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
