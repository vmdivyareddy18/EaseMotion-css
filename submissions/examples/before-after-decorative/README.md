# Before & After Decorative

## What does this do?
Demonstrates the `::before` and `::after` pseudo-elements for purely decorative purposes — quote marks, ribbons, icons, and separators, all generated via CSS without extra HTML markup.

## How is it used?
Add to any element that needs decorative accents:

    <blockquote class="deco-quote">Text</blockquote>
    <h2 class="deco-ribbon">Heading</h2>

## Why is it useful?
Pseudo-elements keep the DOM clean while enabling rich visual decoration. This pattern reduces HTML clutter, improves maintainability, and allows theme changes without touching markup.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
