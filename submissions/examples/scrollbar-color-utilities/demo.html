Scrollbar Color Utility Classes

Apply EaseMotion's brand colors to scrollbar-color and the WebKit scrollbar pseudo-elements. Existing .ease-scrollbar-thin controls width but hardcodes a neutral gray palette — these classes add themed color variants.

💡 Note: Scroll inside each box below to see the colored thumb. Firefox uses native scrollbar-color; Chrome/Edge/Safari use the WebKit pseudo-elements.

Common use cases

  • Branded content panels — .ease-scrollbar-primary
  • Secondary sidebars/drawers — .ease-scrollbar-secondary
  • Success/confirmation modals with scrollable content — .ease-scrollbar-success
  • Destructive/warning dialogs with scrollable content — .ease-scrollbar-danger
