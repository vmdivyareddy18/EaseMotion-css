# Command Palette Component

## What does this do?

Provides a modern command palette interface with grouped commands, keyboard shortcut display, and smooth animations.

## How is it used?

```html
<div class="command-palette">

  <div class="search-box">
    <input type="text" placeholder="Search commands...">
  </div>

  <div class="command-item">
    <span>Open Settings</span>
    <kbd>⌘ S</kbd>
  </div>

</div>
```

## Why is it useful?

Command palettes are widely used in modern SaaS platforms, developer tools, dashboards, and productivity applications. This component provides a reusable, animation-first command interface that aligns with EaseMotion CSS's design philosophy.
