# Liquid Ripple Cryptocurrency Exchange Accordion

A responsive Cryptocurrency Exchange accordion featuring a smooth Liquid Ripple animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Native `<details>` and `<summary>`
- Keyboard accessible
- Responsive layout
- Liquid Ripple opening animation
- Cryptocurrency exchange inspired design
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--accent
--accent2
--background
--surface
--text
--muted
--radius
```
