EaseMotion CSS

Northern Lights,
in your browser.

Soft radial gradients drifting in a slow 10s cycle — atmospheric backgrounds for hero sections and portfolios.

Configurable Palettes

Default
Arctic
Solar Flare
Rosé
Moonlit

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Add class + inner layer div */
<div class="ease-aurora">
  <div class="aurora-layer"></div>
  <!-- your content here -->
</div>

/* 3. Customise colors via CSS variables */
.my-hero {
  --aurora-1: rgba(255, 80, 0, 0.60);   /* band 1 */
  --aurora-2: rgba(255, 200, 0, 0.50); /* band 2 */
  --aurora-3: rgba(255, 40, 80, 0.45);  /* band 3 */
  --aurora-4: rgba(200, 0, 255, 0.35);  /* band 4 */
  --aurora-5: rgba(255, 150, 0, 0.30);  /* band 5 */
  --aurora-duration: 8s;               /* 8–12s */
}
