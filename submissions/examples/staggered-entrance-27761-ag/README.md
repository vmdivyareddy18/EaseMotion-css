# Staggered Entrance Delay Utility Classes

A grid component demonstrating cascading item slides and fades using index delay multipliers.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting reload buttons, staggered grids, item cards, and stagger indexes.
2. **`style.css`**: Animation delay formulas, entrance keyframe slides, grid layouts, and media rules.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the grid items sliding and fading in sequence.
3. Click the **Restart** button to re-trigger the staggered animation sequence.
