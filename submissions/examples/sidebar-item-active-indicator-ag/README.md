# Sidebar Active Indicator

## What does this do?
Adds a sliding active indicator to sidebar navigation items, showing which section is currently selected with a smooth animated accent bar.

## How is it used?
Apply the `.sidebar-nav` class to a nav element and `.nav-item.active` to the current item. The active item shows a colored accent bar with smooth transition.

## Why is this useful?
Provides clear visual context for the user's current location in the application, improving navigation UX without JavaScript.
