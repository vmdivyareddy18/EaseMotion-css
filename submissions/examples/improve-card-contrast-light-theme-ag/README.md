# Improve Card Contrast and Hover Feedback in Light Theme

A light theme enhancement that improves visual hierarchy by slightly darkening the card background, adding a subtle border, and introducing smooth elevation and accent borders on hover.

## What does this do?
Enhances `.ease-card` with better contrast against white backgrounds and adds a `.ease-card-hover` class for smooth, interactive lift effects.

## How is it used?
```html
<div class="ease-card ease-card-hover">
  <h3>Your Title</h3>
  <p>Your content goes here.</p>
</div>
```

## Why is it useful?
It prevents cards from blending into light page backgrounds, improving readability and making interactive elements easily identifiable through fluid motion.
