# Approval Request Decision Bar

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive approval request component that presents pending approvals with request details and clear decision actions. Suitable for admin dashboards, workflow systems, document approval portals, and enterprise applications.

## Features

- Approval request notification bar
- Request metadata cards
- Approve and Reject actions
- Responsive layout
- Slide-in animation
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="decision-bar">

    <div class="content">
        <h2>Approval Request Pending</h2>
    </div>

    <div class="actions">

        <button class="approve">Approve</button>

        <button class="reject">Reject</button>

    </div>

</div>
```

## Main Classes

```css
.decision-bar
.icon
.content
.details
.item
.label
.actions
.approve

```
