# Enterprise Project Portfolio Center

A premium enterprise portfolio management dashboard built using pure HTML and CSS.

## Features

- Portfolio Overview Statistics
- Multi-Project Analytics
- Delivery Performance Metrics
- Budget Tracking Dashboard
- Resource Allocation Monitoring
- Team Timeline Feed
- Glassmorphism UI
- Purple-Blue Theme
- Responsive Layout

## Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox
