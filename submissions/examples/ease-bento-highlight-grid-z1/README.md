# Ease Bento Highlight Grid

## Overview

This example demonstrates a responsive CSS-only bento grid layout using asymmetric card sizing for feature sections and dashboards.

## Features

- Pure HTML and CSS
- Asymmetric bento layout
- Hover lift animation
- Responsive grid
- Highlighted feature card
- No JavaScript required

## Folder Structure

```text
ease-bento-highlight-grid-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

1. Open the example folder.
2. Launch `demo.html`.
3. Replace the sample cards with your own content.

## Browser Support

Compatible with modern browsers supporting CSS Grid.

## Why This Example

Bento layouts are widely used in modern landing pages, AI products, dashboards, and portfolios because they organize content while creating strong visual hierarchy.
