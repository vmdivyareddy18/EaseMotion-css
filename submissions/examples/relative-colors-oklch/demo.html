Relative Colors with oklch()

Manipulate Lightness, Chroma, and Hue from a base color

Base Color

oklch(55% 0.2 180)

Manipulated Variants

Darker (L - 20%) l: calc(L - 0.2)
Lighter (L + 25%) l: calc(L + 0.25)
More Saturated (C + 0.1) c: calc(C + 0.1)
Less Saturated (C - 0.1) c: calc(C - 0.1)
Hue Shift (H + 60) h: calc(H + 60)
Complement (H + 180) h: calc(H + 180)

How it works

The oklch(from ...) relative color syntax lets you read the L, C, and H channels of any color and derive new colors by adjusting individual channels independently.
