# Tada Click Tabs – Creative Portfolio

A responsive **pure CSS Tada-Click Tabs** component designed for creative portfolio interfaces.

## Features

- Pure HTML & CSS
- Responsive design
- Keyboard accessible
- Tada click animation
- Customizable using CSS variables
- No JavaScript required

## Usage

```html
<div class="tabs">
    <button class="tab active">Projects</button>
    <button class="tab">Gallery</button>
    <button class="tab">Skills</button>
    <button class="tab">Contact</button>
</div>
```

## Customize

```css
:root{
    --primary:#ff4d6d;
    --duration:.5s;
    --radius:12px;
}
```
