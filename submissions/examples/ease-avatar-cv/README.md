# Animated Avatar (`ease-avatar-cv`)

A lightweight animated avatar component built using **HTML5** and **Pure CSS**.

## Features

- 🟢 Online/Offline status indicator
- 💫 Pulse border animation
- 🔍 Hover zoom effect
- 👥 Group avatar stack
- 📱 Responsive layout
- No JavaScript
- No dependencies

## Folder Structure

```text
ease-avatar-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
