# Account Security Card

A modern **Account Security Card** component built using **HTML5** and **CSS3** for the EaseMotion CSS examples collection.

---

## ✨ Overview

The **Account Security Card** is a reusable dashboard component that helps users understand and improve the security of their account. It visually presents security status using progress bars, checklists, status indicators, and actionable buttons.

This component is suitable for account settings pages, admin dashboards, SaaS applications, banking portals, developer platforms, and enterprise applications.

---

## 🚀 Features

- 🛡️ Security score indicator
- 📊 Animated security progress bar
- ✅ Security checklist
- ⚠️ Warning & critical security states
- 🟢 Excellent security variant
- 🪟 Glassmorphism card
- 🏢 Enterprise dashboard variant
- 🎨 Modern responsive design
- ✨ Smooth hover animations
- 📱 Mobile-friendly layout
- 🌙 Dark mode support
- ♿ Reduced motion support
- 🎯 Pure HTML & CSS implementation
