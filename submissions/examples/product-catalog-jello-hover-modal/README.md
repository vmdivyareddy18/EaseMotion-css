# Product Catalog Jello Hover Modal

A responsive product catalog modal built using only HTML and CSS.

## Features

- Pure CSS
- Jello Hover animation
- Responsive design
- Keyboard accessible
- CSS custom properties
- No JavaScript

## Files

- demo.html
- style.css

## Custom Properties

```css
--duration
--easing
--scale
--primary
```

These variables make it easy to customize animation speed, easing, scaling, and colors.

## Use Case

Designed for modern product catalog and e-commerce layouts where product details can be displayed in an animated modal without JavaScript.
