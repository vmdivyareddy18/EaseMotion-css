# Radar Sweep Animation Effect

A rotating radar sweep line effect with concentric rings, simulating scanning/detection activity under issue #14745.

## Design Concept

- **Concentric Radar Grid:** Interactive glowing circles that expand outward representing radar signals.
- **Rotating Sweep Line:** A linear gradient sweep line rotating 360 degrees to detect virtual targets.
- **Blinking Targets:** Green dot targets fade in and out as the sweep line passes over them.
- **Pure CSS Layout:** Uses keyframes for constant rotation and opacity pulsing for target fades.
