PWA Display Modes

Progressive Web Apps can run in different display modes: browser, minimal-ui, standalone, and fullscreen. Each mode affects the chrome around your app.

Display Mode Previews

Each card shows how the app would look in a different display mode.

browser

Browser Mode

Full browser chrome — address bar, tabs, bookmarks.

minimal-ui

Minimal UI

Basic browser chrome — back/forward, minimal URL bar.

standalone

Standalone Mode

No browser chrome — feels like a native app.

fullscreen

Fullscreen Mode

Immersive — no browser UI, fills entire screen.
