# AI Chat Conversation UI Component

A modern AI chat interface component built with HTML and CSS.

## Features

- AI and user message bubbles
- Animated typing indicator
- Chat header with online status
- Message timestamps
- Responsive layout
- Dark mode support
- Reduced motion support
- Modern conversation design

## Files

- demo.html
- style.css
- README.md

## Use Cases

- AI assistants
- Customer support bots
- Learning platforms
- Productivity tools
- SaaS dashboards
- Coding assistants

Supports:

- Responsive design
- Dark mode
