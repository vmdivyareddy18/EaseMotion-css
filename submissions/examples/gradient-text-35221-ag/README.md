# Animated Gradient Text Component

A shifting linear color loop text component utilizing background-clip properties.

---

## What is Included

1. **`demo.html`**: A showcase dashboard holding shifting headers, speed select boxes, and background clips.
2. **`style.css`**: Text masks, fill variables, loop timing, and reduced motion configs.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the header text shifting colors in a continuous loop.
3. Change the speed selection dropdown to verify speed adjustment works correctly.
