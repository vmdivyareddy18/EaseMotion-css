# Stacked Cards

## What it does

Creates a stack of cards that fan out when hovered, like spreading a deck of cards.

## How to use it

<div class="ease-stack-cards">
    <div class="ease-stack-card">Card 1</div>
    <div class="ease-stack-card">Card 2</div>
    <div class="ease-stack-card">Card 3</div>
    <div class="ease-stack-card">Card 4</div>
</div>

## Variants

- ease-stack-left - Fans out to the left
- ease-stack-right - Fans out to the right
- ease-stack-full - Fans out to both sides
- ease-stack-fast - 0.15s transition
- ease-stack-slow - 0.6s transition
- ease-stack-glow - Adds neon glow effect

## Why it fits EaseMotion CSS

- Pure CSS, zero JavaScript
- 3D transforms and rotations
- Multiple fan-out directions
- Smooth hover transitions
