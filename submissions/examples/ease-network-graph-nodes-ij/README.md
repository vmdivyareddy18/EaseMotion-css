# Network Graph Nodes

A network graph with pulsing node indicators.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--node-color` | `#4ecdc4` | Node fill color |
| `--edge-color` | `#4a4a6a` | Edge line color |
| `--pulse-color` | `rgba(78,205,196,0.3)` | Pulse ring color |
| `--pulse-duration` | `2s` | Pulse cycle duration |
