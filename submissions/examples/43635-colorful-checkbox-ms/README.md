# Colorful Checkbox Variants Showcase

Resolves #43635.

### What does this do?
Provides a set of custom, animatable checkbox components with various color theme variants (primary, success, danger, warning, purple, pink). It uses a pop animation on check and a satisfying icon scale-in effect.

### How is it used?
Hide the native input with `.ease-checkbox-input`. Use the adjacent sibling selector to style `.ease-checkbox-box`. Apply a color variant class like `.ease-cb-success` to customize the check color.

### Why is it useful?
Native checkboxes are notoriously hard to style across browsers. This CSS-only solution ensures cross-browser consistency while providing premium micro-interactions.
