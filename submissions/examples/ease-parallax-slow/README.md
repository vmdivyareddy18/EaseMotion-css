# Ease Parallax Slow

This demo shows a **slow parallax animation** where an element moves vertically at half the scroll speed.

## 🧠 How It Works
- JavaScript updates `translateY` based on scroll position.
- CSS variable `--ease-parallax-speed` controls the speed (default `0.5`).
- GPU-accelerated transform ensures smooth performance.
- Ideal for background elements.

## 🧩 Files
- `demo.html` — main demo with JS logic.
- `style.css` — styling and parallax setup.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Scroll down to see the background move slower than the foreground.
3. Adjust `--ease-parallax-speed` in `style.css` for different effects.

## 🧰 Technologies
- HTML5
- CSS3 (custom properties)
- JavaScript (scroll-driven transform)
