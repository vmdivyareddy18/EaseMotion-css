# Perspective Newsletter Popup

A responsive **Perspective Newsletter Popup** inspired by modern neon interfaces. The popup uses a CSS-only perspective animation to create a subtle 3D appearance while keeping the design lightweight and accessible.

## Features

- Pure HTML & CSS
- CSS perspective animation
- Responsive layout
- Accessible form controls
- No JavaScript required

## Folder Structure

```
perspective-newsletter-popup-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
