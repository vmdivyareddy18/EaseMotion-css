# Business Landing: Restaurant Marketing Agency (`business-landing-restaurant-agency-20935`)

This proposal provides a complete, copy-paste ready landing page template designed for a niche Digital Marketing Agency specializing in Restaurants, Cafes, and Hospitality Groups, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A restaurant-focused agency website must feel warm, hospitable, and appetizing, while still projecting business competence. This template achieves an "Appetizing Hospitality" aesthetic using a cream background paired with food-adjacent colors (Tomato Red, Mustard Yellow, and Olive Green). It pairs a welcoming, elegant serif font (`Playfair Display`) for headings with a clean, readable sans-serif (`Work Sans`) for body copy.

The layout includes:
- A hospitable, warm navigation bar.
- A Hero section featuring abstract CSS shapes resembling plates, and a custom CSS-built "Restaurant Receipt" data visualization graphic demonstrating ROI.
- A "Recipe for Growth" services grid highlighting Local SEO, Paid Ads, and Reputation Management.
- A "Success Stories" case study section showcasing reservation growth metrics.
- A transparent "Menu" pricing grid with a highlighted "Chef's Special" tier.
- A testimonial section featuring direct quotes from restaurant owners.
- A "Free Audit" form section designed to capture leads.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use warm, slightly bouncy `cubic-bezier(0.34, 1.56, 0.64, 1)` timings tailored for a friendly, hospitable brand:
- `.ease-float` & `.ease-float-delayed` (Used on the receipt graphic and review badge to create a lightweight, organic hovering effect).
- `.ease-spin-slow` (Used on the background abstract shapes to provide gentle ambient movement).
- `.ease-slide-up-stagger` (Configured for a friendly, slightly bouncy vertical entrance).
- `.ease-hover-lift` (Provides an inviting, tactile elevation effect on service and pricing cards).
- *Custom UI*: The hero section features a pure CSS "Receipt" graphic complete with a jagged bottom edge (created via CSS gradients) and a CSS barcode.

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the appetizing color palette and typography to the pure CSS illustrations (the receipt graphic). The standard `ease-*` interaction classes are attached directly to HTML elements to provide warm, responsive animations. No JavaScript is required for this template.
