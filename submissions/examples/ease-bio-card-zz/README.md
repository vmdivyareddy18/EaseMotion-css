# Team Bio Card

## What does this do?
Team Bio Card component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"bc-primary\">Team Bio Card Primary</div>
```

## Why is it useful?
Provides a reusable team bio card UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
