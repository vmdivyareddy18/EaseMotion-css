# Disability Support Services Template
A compassionate, highly accessible, and inclusive template designed for disability support non-profits, advocacy groups, and community care organizations.

- **Design**: Uses an "Inclusion-First" palette with professional access-blue primaries, warm-peach supportive accents, and soft-slate neutral backgrounds to foster trust and approachability.
- **Use Case**: Disability support services, community advocacy non-profits, and inclusive technology foundations.
