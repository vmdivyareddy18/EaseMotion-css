# Ease Centaur Run Animation

1. What does this do? Creates a rhythmic, infinite galloping animation that simulates a four-legged running cycle (bouncing up and down with alternating rotational pitches).
2. How is it used? Apply `.ease-centaur-run-ps` to any element to make it continuously gallop in place.
3. Why is it useful? It is perfect for loading indicators, thematic game sprites, or playful UI elements where a standard linear translation isn't dynamic enough. It brings characters or icons to life with an organic "striding" rhythm.
