# CSS Text Truncation Showcase

Resolves #42071

A lightweight, CSS-only showcase demonstrating common text overflow strategies including single-line ellipsis, multi-line clamping, word breaking, and scrollable overflow. The component helps developers understand and compare different CSS text truncation techniques without JavaScript.

## Features

- Pure CSS, zero JavaScript
- Responsive card layout
- Demonstrates multiple text overflow techniques
- Includes CSS snippets for each example
- Beginner-friendly educational showcase

## Files

- demo.html
- style.css
- README.md

## Techniques Included

- Single-line ellipsis
- Two-line clamp
- Three-line clamp
- Word breaking
- Scrollable overflow
