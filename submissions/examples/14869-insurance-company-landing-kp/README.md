# Insurance Company Landing Page

## What does this do?
It provides a full responsive landing page for an insurance company with plan cards, a premium quote panel, and a claim workflow section.

## How is it used?
```html
<section class="hero">
  <h1>Protect home, health, travel, and business from one clear dashboard.</h1>
  <a class="button primary" href="#plans">Explore plans</a>
</section>
```

## Why is it useful?
It demonstrates how EaseMotion-style motion can support a trust-focused financial services page with clear calls to action and self-contained CSS.
