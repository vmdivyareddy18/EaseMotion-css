# Mini Profile Card

## What does this do?
A compact profile card with avatar styling, hover lift, and animated online presence badge.

## How is it used?
```html
<section class="profile-card">
  <span class="avatar">K</span>
  <h2>Mini Profile Card</h2>
</section>
```

## Why is it useful?
It gives EaseMotion CSS a team-member card for dashboards, directories, and collaboration screens.
