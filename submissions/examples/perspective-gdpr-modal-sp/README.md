# Perspective GDPR Modal

A portfolio-inspired GDPR consent modal featuring a 3D perspective entrance animation built entirely with HTML and CSS.

## Features

- 🍪 GDPR consent modal
- 🎬 Perspective 3D entrance animation
- 💻 Pure HTML & CSS
- 📱 Responsive design
- ♿ Accessible structure
- 🚫 No JavaScript required

## File Structure

```
perspective-gdpr-modal-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` and click **Open GDPR Modal**.

The modal appears using a smooth perspective animation and presents Accept and Decline actions.

## Use Cases

- Portfolio websites
- Business websites
- SaaS dashboards
- Landing pages
- Cookie consent interfaces
