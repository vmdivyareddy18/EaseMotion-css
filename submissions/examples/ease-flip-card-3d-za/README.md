# CSS 3D Flip Card

## What does this do?
Interactive 3D flip card component using CSS perspective, preserve-3d, and backface-visibility. Flips on hover to reveal back content with smooth cubic-bezier transition.

## How is it used?
```html
<div class="fc-card"><div class="fc-inner"><div class="fc-front"><h3>Front</h3></div><div class="fc-back"><h3>Back</h3></div></div></div>
```

## Why is it useful?
Creates engaging interactive cards without JavaScript. Uses CSS 3D transforms for realistic flip animation. Front shows preview, back reveals details. Perfect for portfolios, flashcards, product showcases, and team member cards.
