# Empty State / Placeholder Illustration

An empty state card with a custom SVG illustration (a document with a plus icon overlay), a heading, description text, and a Call to Action button. Useful for pages where no data exists yet.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the empty state card.

## Accessibility notes

The SVG illustration is decorative. The CTA is a `<button>` element with clear text.
