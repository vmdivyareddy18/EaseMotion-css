# Animated Lock Security Pulse

## Overview

Animated Lock Security Pulse is a lightweight CSS animation that creates a pulsing lock effect to indicate security, protection, and authentication.

## Features

- CSS-only animation
- Smooth pulse effect
- Security-focused UI example
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="lock-security-pulse">
  🔒
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
