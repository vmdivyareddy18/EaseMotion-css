# Full Dental Clinic Landing Page (`ease-page-dental`)

A complete, production-ready healthcare landing page template demonstrating a clean, trustworthy aesthetic built entirely with EaseMotion CSS utilities.

## 🚀 Features & EaseMotion Showcase

This landing page integrates multiple core `.ease-*` classes alongside an `IntersectionObserver` script to create a calming, professional scroll-triggered experience:

- **Friendly Entrances**: Utilizes `.ease-slide-up` and `.ease-fade-in` with CSS `animation-delay` staggered properties to gently introduce the Services grid and Doctor profiles.
- **Card Hover States**: The service cards, pricing cards, and doctor profiles utilize `.ease-hover-lift` to provide tactile feedback and depth on mouseover, reinforcing a modern, premium feel.
- **Pill-Shaped Buttons**: All Call-to-Action buttons use `.ease-hover-scale` combined with a trustworthy medical blue palette and pill-border-radius for an inviting UI.

## 🛠️ Usage

This demo is entirely self-contained. You can open `demo.html` directly in your browser. All required EaseMotion CSS classes have been defined at the top of the `style.css` file for immediate testing. 

If using this as a template for a real project, simply migrate the HTML structure into your application and ensure your global CSS bundle includes the `ease-*` utility classes.
