# Animated Feature Spotlight/Highlight Reel

A clean, modern, and fully responsive **Feature Spotlight Component** that showcases key product features with smooth transitions and auto-play functionality. This example demonstrates how to combine **EaseMotion CSS** for staggered entrances, continuous animations, and interactive navigation to create an engaging feature showcase.

## 🚀 Features

- **Staggered Card Entrances**: Each feature card zooms in sequentially using `ease-zoom-in` and `ease-delay-*` for a cascading load effect.
- **Continuous Icon Animations**: Feature icons use `ease-pulse` with staggered delays for a lively, attention-grabbing effect.
- **Smooth Transitions**: Active feature cards fade in/out with cubic-bezier easing for natural movement.
- **Interactive Navigation**: Clickable dots with `ease-hover-grow` allow manual navigation between features.
- **Keyboard Support**: Arrow keys (← →) navigate between features for accessibility.
- **Auto-play with Progress Bar**: Automatically cycles through features every 5 seconds with a visual progress indicator.
- **Pause on Hover**: Auto-play pauses when hovering over the spotlight area.
- **Responsive Design**: Adapts gracefully to mobile screens with adjusted sizing and layout.
