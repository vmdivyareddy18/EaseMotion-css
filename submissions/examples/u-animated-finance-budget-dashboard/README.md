# Animated Finance Budget Dashboard

A modern **Finance Budget Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example provides a responsive interface for managing budgets, tracking expenses, monitoring savings, and visualizing financial analytics with smooth CSS-only animations.

## Features

* 💰 Total balance overview
* 📈 Monthly income card
* 💳 Monthly expenses tracking
* 🏦 Savings overview
* 🍔 Expense category cards
* 📊 CSS-only weekly spending chart
* 📉 Budget progress indicators
* 💸 Recent transaction activity
* ✨ Smooth hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

* `ease-progressGrow`
* `ease-slideIn`
* `ease-pulse`
* `ease-hoverLift`
* `ease-cardReveal`
* `ease-statusPulse`
