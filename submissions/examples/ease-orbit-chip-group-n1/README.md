# Ease Orbit Chip Group

A lightweight chip/tag component with a rotating highlight animation using pure HTML and CSS.

## Features

- Animated chip highlight
- Responsive layout
- Pure HTML & CSS
- Zero dependencies
- Easy to customize

## Structure

```
ease-orbit-chip-group-n1/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Why EaseMotion CSS

A reusable micro-interaction component that demonstrates smooth hover animations with minimal code.
