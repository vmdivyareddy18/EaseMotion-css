# Image Compare Slider

## What does this do?
Before/after image comparison component using CSS and minimal JavaScript for the range slider input.

## How is it used?
```html
<div class="ic-container"><div class="ic-after">After</div><div class="ic-before">Before</div><input type="range" class="ic-slider"></div>
```

## Why is it useful?
Useful for showcasing design iterations, photo edits, or product comparisons. Horizontal and vertical layouts supported. Accessible range input control.
