# ✕ ◯ Tic-Tac-Toe (XO Game)

A classic two-player Tic-Tac-Toe game built with **HTML**, **CSS**, and **vanilla JavaScript** (embedded in HTML). Clean, modern design with smooth interactions.

---

## 🎮 How to Play

1. **Players**: Two players take turns — **X** goes first, then **O**.
2. **Objective**: Get **three of your marks** in a row (horizontally, vertically, or diagonally).
3. **Click** any empty cell to place your mark.
4. The game automatically detects **wins** and **draws**.
5. Winning cells are **highlighted** in blue.
6. Press the **"New Game"** button to restart anytime.

## 🚀 Getting Started

1. Download all three files into the same folder.
2. Open `index.html` in any modern web browser.
3. Start playing immediately — no server or dependencies required.
