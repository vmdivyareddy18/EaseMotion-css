# Minimalist Tech Pulse-Active Modal

A modern **Pure CSS Pulse-Active Modal** designed for **Minimalist Tech** interfaces. The component provides a smooth pulse animation, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Pulse-Active hover animation
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- Minimalist Tech inspired UI
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:1s;
    --easing:ease-in-out;
    --scale:1.03;

    --bg:#0f172a;
    --card:#1e293b;
    --primary:#38bdf8;
}
```

You can easily customize:

- Animation speed
- Animation easing
- Hover scale
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in your browser.

No build tools or JavaScript are required.
