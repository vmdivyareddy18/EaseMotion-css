# Floating Action Button (FAB)

A fixed-position action button in the bottom-right corner with a plus icon. The FAB hides when scrolling down and reappears when scrolling up. Uses a scale hover effect and a subtle box-shadow glow.

## EaseMotion CSS classes used

None. This example uses plain CSS for all styling.

## How to run

Open `demo.html` in a browser. Scroll down to see the FAB hide; scroll up to reveal it again.

## Accessibility notes

The button has `aria-label="Add new item"` for screen readers. Reduced motion disables the scale transitions.
