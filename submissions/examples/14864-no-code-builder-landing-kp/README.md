# No-Code Website Builder Landing Page

## What does this do?
It provides a full responsive landing page for a no-code website builder with a builder preview, feature cards, and launch workflow.

## How is it used?
```html
<section class="hero">
  <h1>Design, launch, and iterate websites without touching code.</h1>
  <a class="button" href="#blocks">Browse blocks</a>
</section>
```

## Why is it useful?
It demonstrates a product-focused SaaS page pattern with lightweight CSS motion and a self-contained visual builder mockup.
