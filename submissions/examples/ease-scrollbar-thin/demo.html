Themed Scrollbar

Scroll inside the panel below. The scrollbar's color is controlled entirely by the --ease-scrollbar-color custom property — change it on .ease-scroll to retheme it instantly.

This panel has a custom thin scrollbar styled to match the EaseMotion color system.

On WebKit/Blink browsers (Chrome, Edge, Safari) it uses ::-webkit-scrollbar and friends.

On Firefox it uses the standard scrollbar-color / scrollbar-width properties.

Both read from the same single source of truth: --ease-scrollbar-color.

Try the color swatches above — they update the variable live with one line of JS.

Keep scrolling to see the full thumb and track styling in both directions.

The track stays subtle and neutral so the colored thumb is what draws the eye.

The thumb has a soft glow hover state so it feels responsive without being loud.

This pattern works for any scrollable container — sidebars, code blocks, modals, etc.
