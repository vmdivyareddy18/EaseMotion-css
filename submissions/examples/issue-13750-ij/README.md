# ease-poison-bubble

**Closes #13750**

A green bubbling overlay effect using multiple pseudo-elements with erratic bubble keyframes and a toxic colour palette.

## Acceptance Criteria

- [ ] Green/toxic gradient bubbles float upward erratically
- [ ] Multiple pseudo-elements (`::before`, `::after`) with different timing
- [ ] Toggle button activates/deactivates the effect
- [ ] Dark background with green accent

## Files

- `style.css` – Bubble keyframes, pseudo-element overlays
- `demo.html` – Poison container with toggle
- `README.md` – Issue documentation
