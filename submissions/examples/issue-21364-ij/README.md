# Snake Slithering

1. What does this do? A CSS-only snake that slithers in an S-curve motion using skew transforms and shifting border-radius.

2. How is it used? Place .snake inside .scene. The body animates translation and skew to create a serpentine wiggle.

3. Why is it useful? Demonstrates how combining skew and border-radius changes can simulate slithering motion.
