# output-element-badge

## Purpose
Demonstrates the `output` element for live form calculations with badge and result indicator styling.

## Features
- Live calculation display using the `output` element
- Styled as badges, pills, and result indicators
- Form input events update output in real-time
- Multiple visual variants (badge, tag, card, inline)

## Usage
Open `demo.html` and interact with the form controls. The output elements update dynamically as you adjust values.
