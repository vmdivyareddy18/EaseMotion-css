# Swing GDPR Modal

A Neumorphism-inspired Swing GDPR Modal built using pure HTML and CSS for EaseMotion CSS.

## Features

- Pure HTML & CSS
- No JavaScript
- Swing opening animation
- Responsive layout
- Accessible structure
- Neumorphism design
- Uses EaseMotion-style variables and animation naming

## Folder Structure

```
swing-gdpr-modal-sanju-rajak/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Click **Open GDPR Modal** to preview the swing animation.
