# Animated Horizontal Timeline

A clean, modern, and fully responsive **Horizontal Timeline/Roadmap** perfect for showcasing project milestones, company history, or step-by-step processes. This example demonstrates how to combine **EaseMotion CSS** for staggered card entrances and smooth connecting line animations with custom CSS for horizontal scrolling and interactive hover effects.

## 🚀 Features

- **Horizontal Scrollable Layout**: Cards are arranged horizontally with smooth scrolling behavior and a custom scrollbar.
- **Staggered Card Entrances**: Each milestone card fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a cascading load effect.
- **Bouncy Milestone Dots**: The timeline dots use `ease-bounce-in` with staggered delays to playfully pop into view.
- **Animated Connecting Line**: A gradient line connects all milestones and fades in smoothly on page load.
- **Interactive Hover Effects**: Cards lift on hover with enhanced shadows and accent-colored borders.
- **Status Indicators**: Clear visual distinction between completed, in-progress, and upcoming milestones with color-coded badges.
- **Custom Scrollbar**: Styled scrollbar that matches the design aesthetic.
- **Fully Responsive**: Adapts to mobile screens with adjusted sizing and spacing.
