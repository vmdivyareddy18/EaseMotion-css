# CSS-only Modal Demo

A CSS-only modal demo that opens and closes without JavaScript.

The component uses a hidden checkbox (`:checked`) to control the open/close state. It includes a full-screen overlay, centered modal dialog, close button, responsive layout, smooth open/close animation, and focus-visible styling for keyboard users. It also honors `prefers-reduced-motion` settings.

This modal can be used for confirmations, alerts, onboarding messages, forms, previews, or short focused content blocks.

## Why is this useful for EaseMotion CSS?

A modal is a common UI pattern in modern interfaces and is aligned with the project roadmap. A CSS-only modal fits EaseMotion CSS because it is lightweight, animation-first, human-readable, and does not require JavaScript for state handling. 

It demonstrates how to combine layout, layered UI, accessibility-minded markup, and motion in one self-contained, advanced example.
