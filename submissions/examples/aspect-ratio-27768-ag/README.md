# Responsive Aspect Ratio Utility Classes

A responsive layout component demonstrating container aspect-ratio consistency.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting 16:9 widescreen grids, 1:1 square media panels, and size details.
2. **`style.css`**: Aspect ratio rules, width clamps, alignment parameters, and container styles.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Resize the browser window.
3. Verify that the **16:9 Widescreen** box scales proportionally, retaining its precise widescreen format.
4. Verify that the **1:1 Square** box remains perfectly square at all viewport dimensions.
