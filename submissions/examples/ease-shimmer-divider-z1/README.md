# Ease Shimmer Divider

## Overview

A reusable divider with a subtle shimmer animation created using HTML and CSS.

## Features

- Pure HTML & CSS
- Shimmer animation
- Responsive
- Lightweight
- Easy to customize

## Folder Structure

```
ease-shimmer-divider-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.
