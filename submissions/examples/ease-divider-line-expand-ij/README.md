# Ease Divider Line Expand Component

A sleek, modern content divider component that transitions smoothly, expanding outward from the center when its parent container receives hover or focus.

## Features
- **Pure CSS Transition:** Leverages optimized hardware-accelerated `transform: scaleX()` for buttery-smooth performance.
- **Center Expansion:** Driven by `transform-origin: center` to grow dynamically outward.
- **Accessible Design:** Features `:focus-within` triggers alongside hover states for standard keyboard navigability.
- **CSS Variables:** The color gradient, line thickness, total width, and animation speeds are instantly modifiable.
