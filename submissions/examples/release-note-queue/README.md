# Release Note Queue

Release Note Queue is a responsive launch-readiness panel for tracking release notes before publication. It groups documentation items by ready, review, and hold states so launch owners can see what still needs action.

## Features

- Release note review summary
- Ready, review, and hold states
- Compact queue cards
- Responsive single-column layout

## Files

- `demo.html` contains the release note queue markup.
- `style.css` contains the layout, cards, and state labels.
