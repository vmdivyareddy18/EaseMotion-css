# Vertical Wobble Animation

A lightweight vertical wobble animation utility built using pure CSS.

## Features

- Pure vertical wobble movement
- Diminishing oscillation effect
- Smooth ease-out motion
- No rotation involved
- Pure CSS implementation

## Usage

```html
<div class="wobble-box">
  WOBBLE
</div>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable motion effect
- Pure CSS animation
