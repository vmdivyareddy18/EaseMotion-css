# Spotlight Voice Recorder

Resolves #42608.

### What does this do?
Adds a Spotlight Voice Recorder component to EaseMotion CSS. It features a glowing background spotlight (`ease-pulse`), a flashing red recording indicator (`ease-flash`), and simulated audio waves using staggered `ease-bounce` animations.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. It seamlessly integrates with `easemotion.min.css` to provide the animations.

### Why is it useful?
Provides a polished, ready-to-use audio recording UI that leverages EaseMotion's built-in animations for instant visual feedback.
