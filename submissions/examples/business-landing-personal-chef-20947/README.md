# Business Landing: Personal Chef (`business-landing-personal-chef-20947`)

This proposal provides a complete, copy-paste ready landing page template designed for a Private Chef or Luxury Culinary Service, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A luxury culinary website requires a UI that feels refined, elegant, and bespoke. This template achieves a "Culinary Luxury" aesthetic using a sophisticated palette of Deep Charcoal (`#1A1A1A`), Warm Paper/Off-White (`#F9F7F2`), and Champagne Gold (`#C6A87C`). It uses an elegant serif (`Playfair Display`) for primary display text and a clean, legible sans-serif (`Lato`) for body copy and UI elements.

The layout includes:
- An elegant, sticky navigation bar with subtle hover indicators.
- A Hero section featuring a dark, moody aesthetic and a complex custom CSS-constructed abstract background simulating an overhead view of a plated dish.
- A "Services" grid highlighting dinner parties, meal prep, and events with subtle float animations.
- A "Sample Menus" section utilizing a dark-mode card layout to present a tasting menu.
- A "Meet the Chef" bio section featuring an abstract CSS portrait in an image frame.
- A clean, centered testimonial block.
- A highly polished booking inquiry form.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use luxurious, smooth `cubic-bezier(0.33, 1, 0.68, 1)` timings tailored for a high-end brand:
- `.ease-slide-up-stagger` (Configured for a slow, elegant vertical entrance that feels effortless).
- `.ease-hover-lift-shadow` (Provides a gentle elevation effect on service cards).
- `.ease-float-slow` (Applied to icons and UI elements to give a weightless, premium feel).
- *Custom Animations*: The template includes unique keyframes (`ease-spin-very-slow`) applied to a CSS-constructed background element, simulating the slow rotation of a plate.
- `.ease-hover-scale-img` (Creates an elegant press-down effect on the portrait frame).
- `.ease-scale-x` (Used to draw focus lines and dividers organically).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the elegant typography and color palette to the pure CSS illustrations (the abstract plate background and the chef portrait). The standard `ease-*` interaction classes are attached directly to HTML elements to provide smooth, luxurious animations. No JavaScript is required for this template.
