# Glassmorphism Login Form

A modern glassmorphism-inspired login form component built using pure HTML and CSS.

## Features

- Glassmorphism card design
- Animated floating labels
- Responsive layout
- Hover animations
- Smooth transitions
- Password field UI
- Social login buttons
- Modern gradient background
- Pure HTML and CSS
- Mobile friendly

## Folder Structure

```text
glassmorphism-login-form/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in a browser.
2. Ensure `style.css` is in the same directory.
3. Customize colors, fonts, and animations as required.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Glassmorphism Design
