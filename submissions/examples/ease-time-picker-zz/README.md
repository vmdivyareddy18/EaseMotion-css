# Time Picker

## What does this do?
Time Picker component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"tp-primary\">Time Picker Primary</div>
```

## Why is it useful?
Provides a reusable time picker UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
