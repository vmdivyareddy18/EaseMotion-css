# Accessible Range Slider with Tooltip and Tick Marks

## 1. What does this do?

This submission provides a modern, accessible range slider featuring a floating value tooltip, labeled tick marks, a gradient progress track, and smooth thumb interactions. It is built with semantic HTML, modern CSS, and lightweight vanilla JavaScript while preserving the native behavior and accessibility of the HTML range input.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The component uses the accompanying `style.css` for styling and includes a small amount of vanilla JavaScript to keep the tooltip position and displayed value synchronized with the slider. Multiple slider instances can be added by following the same markup structure shown in the demo.

---

## 3. Why is it useful?

Range sliders are commonly used in settings panels, dashboards, media controls, and filtering interfaces. This submission demonstrates a reusable, responsive, and accessible implementation with keyboard support, visible focus indicators, reduced-motion support, and modern styling—all without relying on external libraries or frameworks.
