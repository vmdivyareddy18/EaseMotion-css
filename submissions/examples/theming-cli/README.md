# Theming CLI

1. What does this do? A small command-line utility (Node.js) that generates a custom EaseMotion CSS token file.
2. How is it used? Run `node cli.js` to get a step-by-step wizard, then link the generated `easemotion-theme.css` after `easemotion.min.css`.
3. Why is it useful? It lets users consistently customize colors, spacing, and durations without manually editing variables, fitting the project's zero-dependency philosophy.
