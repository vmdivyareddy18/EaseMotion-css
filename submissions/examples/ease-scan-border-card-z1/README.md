# Ease Scan Border Card

## Overview

A card component with a continuous scanning light animation created using HTML and CSS.

## Features

- Pure HTML & CSS
- Animated scan effect
- Responsive
- Lightweight
- Easy to customize

## Folder Structure

```
ease-scan-border-card-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in a modern browser.

## Browser Support

Chrome, Edge, Firefox, Safari.

## Why This Example

This example demonstrates a reusable scanning animation suitable for security dashboards, analytics cards, and futuristic interfaces.
