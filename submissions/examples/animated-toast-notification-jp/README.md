# Animated Toast Notification Component

## Description

A modern toast notification component with smooth slide-in animations, hover interactions, and multiple status variants.

---

## Features

- Slide-in entrance animation
- Success, error, warning, and information styles
- Smooth hover movement
- Rounded cards with shadows
- Responsive layout
- Reduced motion accessibility support
- Easy customization

---

## Variants

### Success
Displays positive feedback with green styling.

### Error
Shows failures or invalid actions with red styling.

### Warning
Highlights important notices with yellow styling.

### Information
Provides general updates with blue styling.

---

## Usage

Include `demo.html` and `style.css` to use the toast component.

---

## Accessibility

Supports `prefers-reduced-motion` for users who prefer fewer animations.
