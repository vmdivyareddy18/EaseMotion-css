# Animated Bar Chart

1. What does this do? Creates an animated bar chart where bars grow from the bottom with staggered animation delays.

2. How is it used? Apply `.bar` class to bar elements with custom CSS variables `--bar-height` for height and `--bar-delay` for staggered timing.

3. Why is it useful? It provides a visually engaging way to present data with smooth, performant CSS animations that enhance user experience without JavaScript dependencies.
