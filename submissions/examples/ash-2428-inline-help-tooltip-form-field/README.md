# Inline Help Tooltip Form Field

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern form field component with inline help tooltips that provide contextual guidance without leaving the form.

## Features

- Inline help tooltips
- Hover and focus interactions
- Accessible guidance pattern
- Smooth tooltip animations
- Modern form styling
- Responsive layout
- Glassmorphism card design
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="tooltip-wrapper">

    <span class="help-icon">?</span>

    <div class="tooltip">
        Helpful information here.
    </div>

</div>
```

## Main Classes

```css
.field
.label-row
.tooltip-wrapper
.help-icon
.tooltip
.submit-btn
```
