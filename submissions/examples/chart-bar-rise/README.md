# Chart Bar Rise

A compact analytics chart where bars rise from the baseline with staggered timing.

## Files

- `demo.html` - chart figure with custom bar heights.
- `style.css` - bar rise animation, staggered delays, and reduced-motion fallback.

## Highlights

- Uses CSS custom properties for bar heights.
- Animates bars from the bottom with `scaleY`.
- Keeps bars static for reduced-motion users.
