# Info and Warning Badge Semantic Variants

An isolated component expansion delivering the `.ease-badge-info` and `.ease-badge-warning` layout presets. These semantic blocks categorize system context states cleanly across high-density lists, profile chips, and telemetry grids.

## Semantic Token API

- `.ease-badge-info`: Maps a soft, high-contrast blue/sky profile. Engineered to identify non-critical system tips, platform tracking milestones, or instructional metadata updates.
- `.ease-badge-warning`: Maps a vivid amber contrast profile. Formatted to indicate unverified user permissions, threshold alarms, lagging processes, or elements requiring closer evaluation.

## Usage Layout Structure
```html

<span class="ease-badge ease-badge-info">System Tip</span>

<span class="ease-badge ease-badge-warning">Attention Required</span>
```
