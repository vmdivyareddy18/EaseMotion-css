# Pulsing Status Dot (Issue #18253)

A small pulsing circle that scales up and fades in/out to indicate active status. Configurable color via CSS custom property.

## Usage

Open `demo.html` in a browser. The dot pulses continuously.

## Files

- `demo.html` — HTML5 page with the pulsing dot
- `style.css` — Dark theme styles with `@keyframes pulseDot`
- `README.md` — This file
