SCSS Mixin Recipe Book

Semantic HTML? Use our SCSS mixins instead of utility classes.

Hover Lift Mixin

Apply the hover-lift micro-interaction to any semantic class without muddying your HTML markup.

@use 'easemotion' as *; .my-semantic-button { background: blue; color: white; @include ease-hover-lift(); }

Fade In Up Mixin

Trigger entering animations purely through your stylesheets.

@use 'easemotion' as *; .article-card { /* Animate in over 0.5s */ @include ease-fade-in-up(0.5s); }

Pulse Alert Mixin

Draw attention to a specific notification dot or alert banner.

@use 'easemotion' as *; .notification-dot { /* Infinite pulse */ @include ease-pulse(); }
