Animation Customization via CSS Variables

This demo proves how global --em- variables can control animation states without touching library source code.



Default

Uses global :root defaults.
(0.5s, 0s delay, 1 iteration)

Slow & Linear

Locally overrides duration and easing.
(--em-duration: 2s; --em-easing: linear;)

Delayed

Locally overrides the delay.
(--em-delay: 1.5s;)

Infinite

Locally overrides iteration count.
(--em-iteration: infinite;)
