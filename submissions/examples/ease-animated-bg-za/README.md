# CSS Animated Backgrounds Component

## What does this do?
A collection of pure CSS animated background effects including gradient shifts, moving stripes, pulse glows, floating orbs, and grid scans for hero sections.

## How is it used?
```html
<section class="abg-panel abg-grad-shift">
  <div class="abg-overlay">Content</div>
</section>
```

## Why is it useful?
Creates engaging dynamic backgrounds for heroes, dividers, and call-to-action sections without JavaScript. GPU-accelerated animations ensure smooth performance. Easily customizable via CSS variables.
