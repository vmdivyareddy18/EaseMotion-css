# Team Capacity Allocation Bars

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive team capacity dashboard that visualizes workload allocation using animated progress bars. Suitable for project management tools, agile dashboards, resource planning systems, and workforce management applications.

## Features

- Animated capacity bars
- Team workload visualization
- Capacity summary cards
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="team">

    <div class="header">

        <span>Engineering</span>

        <strong>82%</strong>

    </div>

    <div class="bar">

        <div class="fill engineering"></div>

    </div>

</div>
```

## Main Classes

```css
.team
.header
.bar
.fill

.design

```
