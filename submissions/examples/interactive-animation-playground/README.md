# Interactive Animation Playground

## Purpose

This submission provides an interactive playground for previewing EaseMotion CSS animation classes.

---

## Features

- Select an animation
- Click **Play Animation**
- Instantly preview the selected animation
- Beginner-friendly interface

---

## Folder Structure

```
interactive-animation-playground/
│
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Select an animation from the dropdown.
3. Click **Play Animation**.
4. Watch the preview box animate.
