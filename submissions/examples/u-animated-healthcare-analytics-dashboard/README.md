# Animated Healthcare Analytics Dashboard

A modern **Healthcare Analytics Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example demonstrates patient management, appointment tracking, healthcare insights, doctor availability, and medical analytics with smooth CSS-only animations.

## Features

* 🩺 Total Patients overview
* 📅 Appointment tracking
* 👨‍⚕️ Doctor availability indicators
* 💊 Treatment completion metrics
* 🏥 Department analytics
* 👤 Doctor profile cards
* 🟢 Online/Offline status indicators
* 📊 CSS-only weekly patient analytics
* 📈 Progress indicators
* 🔔 Activity feed
* ✨ Hover interactions
* 🌙 Dark mode support
* 📱 Fully responsive
* 🎨 CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries
