# Shopping Cart Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive shopping cart interface.

---

## Components

- Shopping Cart
- Product Cards
- Quantity Input
- Remove Button
- Order Summary
- Checkout Button

---

## Folder Structure

```
submissions/examples/shopping-cart-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to verify responsiveness.
3. Observe the applied EaseMotion CSS animations.

---

## Features

- Responsive shopping cart
- Product list
- Quantity selector
- Order summary
- Checkout section
- Uses existing EaseMotion CSS animation classes
