# Error State

## What does this do?
Error State component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"es-primary\">Error State Primary</div>
```

## Why is it useful?
Provides a reusable error state UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
