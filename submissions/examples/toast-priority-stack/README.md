# Toast Priority Stack

## What does this do?
A stacked notification pattern with priority colors, staged slide-in motion, and compact timestamps.

## How is it used?
```html
<section class="toast-stack">
  <article class="toast toast-critical">
    <span class="pulse"></span>
    <strong>Deploy blocked</strong>
  </article>
</section>
```

## Why is it useful?
It gives EaseMotion CSS a dashboard-ready notification component where motion and color clarify message priority.
