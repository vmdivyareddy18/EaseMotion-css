# Custom Counter Style Bullet List

A CSS demo using the list-style-type property with custom string symbols as list markers. Shows arrow, checkmark, and star bullet styles using Unicode characters.

## Files

- `demo.html` - Three list cards with custom bullet styles
- `style.css` - list-style-type with custom strings, card layout

## Usage

Open `demo.html` to see the custom bullet styles.
