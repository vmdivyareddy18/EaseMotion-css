# Ease DNS Query Analytics

## Overview

A responsive HTML and CSS component for visualizing DNS resolver performance, query volume, cache efficiency, and network health.

## Features

- DNS resolver overview
- Query analytics
- Cache hit information
- Health status badges
- Responsive design
- CSS variables
- Pure HTML & CSS
