Column Rule

Adds a dividing line between columns in a multi-column layout.

CSS Property: column-rule

Variants Demo

column-rule: 1px solid #e2e8f0 Column Rule variant
column-rule: 2px dashed #cbd5e1 Column Rule variant
column-rule: 3px double #94a3b8 Column Rule variant
column-rule: 1px solid #6c63ff Column Rule variant

Usage

<div class="column-rule">Content here</div>
