# CSS-only Animated Loading Skeleton Component

## Description

A lightweight CSS-only loading skeleton component featuring smooth shimmer animations for profile cards, image cards, and text placeholders.

---

## Features

- Pure CSS implementation
- Smooth shimmer loading effect
- Profile skeleton layout
- Image and text placeholders
- Responsive design
- Hover interactions
- No JavaScript required
- Reduced motion accessibility support

---

## Components

- `.skeleton` – Base loading placeholder
- `.avatar` – Circular profile placeholder
- `.title` – Heading placeholder
- `.text` – Text line placeholder
- `.image` – Image placeholder

---

## Accessibility

Supports `prefers-reduced-motion` to disable shimmer animations for users who prefer less motion.
