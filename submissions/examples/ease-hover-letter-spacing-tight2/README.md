# ease-hover-letter-spacing-tight

A lightweight CSS hover animation that smoothly reduces letter spacing from **0.08em** to **0**, creating a clean text-condensing effect. It is ideal for pre-spaced headings, labels, and display typography.

---

## ✨ Features

- Pure CSS
- No JavaScript
- Smooth transition
- Lightweight and reusable
- Customizable with CSS variables
- Complements `ease-hover-letter-spacing-grow`

---

## 🚀 Usage

```html
<h1 class="ease-hover-letter-spacing-tight">
    EASEMOTION CSS
</h1>
```

---

## 🎨 CSS Variables

```css
:root {
    --ease-letter-spacing-default: 0.08em;
    --ease-letter-spacing-duration: 0.3s;
}
```
