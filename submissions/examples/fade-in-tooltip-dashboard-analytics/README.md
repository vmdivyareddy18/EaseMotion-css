# CSS Fade-In Tooltip for Dashboard Analytics Layouts

A modern **Dashboard Analytics Tooltip** built entirely with **HTML and CSS**. The tooltip smoothly fades into view on hover and keyboard focus, making it suitable for analytics dashboards, admin panels, and reporting interfaces.

---

## Features

- Pure HTML & CSS
- Smooth Fade-In animation
- Dashboard Analytics styling
- Fully Responsive
- Keyboard Accessible (`:focus-within`)
- CSS Custom Properties
- No JavaScript Required
- Lightweight & Reusable

## CSS Variables

Customize the component using CSS variables.

```css
:root{

--primary:#2563eb;
--duration:.35s;
--radius:14px;
--shadow:0 12px 30px rgba(0,0,0,.15);
--translate:10px;

}
```

You can easily change:

- Primary color
- Animation duration
- Tooltip offset
- Border radius
- Shadow
- Background colors
