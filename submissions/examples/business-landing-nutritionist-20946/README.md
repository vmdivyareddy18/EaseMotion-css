# Business Landing: Nutritionist / Dietitian (`business-landing-nutritionist-20946`)

This proposal provides a complete, copy-paste ready landing page template designed for a Clinical Nutritionist, Dietitian, or Health Professional, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A professional health services website must feel scientific, trustworthy, and approachable. This template achieves a "Clinical & Approachable" aesthetic using a clean palette of pure whites, soft grays, and two primary accent colors: Medical Green (`#2B7A3E`) and Science Blue (`#246B8C`). It pairs a friendly, modern geometric sans-serif (`Outfit`) for display text with a highly legible body font (`Plus Jakarta Sans`).

The layout includes:
- A clean, sticky navigation bar.
- A Hero section featuring abstract CSS representations of cells and a DNA helix, alongside a dynamic CSS-constructed "data visualization" graphic with rotating rings and growing chart bars.
- An "Approach & Credentials" section featuring a CSS-abstracted portrait of a doctor and stacked credential lists.
- A "Targeted Interventions" service grid with icon hover effects.
- A data-driven "Clinical Results" section showcasing animated progress bars for key patient metrics, paired with a dark-mode testimonial card.
- A comprehensive appointment booking form designed for patient intake.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use soft, organic `cubic-bezier(0.25, 1, 0.5, 1)` and quick data-reveal `cubic-bezier(0.16, 1, 0.3, 1)` timings:
- `.ease-slide-up-stagger` (Configured for a soft, friendly vertical entrance).
- `.ease-grow-up` and `.ease-grow-right` (Used heavily in the data visualization and clinical results sections to animate charts and progress bars).
- `.ease-draw-line` (Used in the hero section to animate an SVG underline beneath a key word).
- `.ease-float-slow` (Applied to abstract background cells and data cards to create a dynamic, floating environment).
- `.ease-spin-slow` (Applied to data rings in the hero visual).
- `.ease-hover-lift-shadow` (Provides a gentle elevation effect on service cards).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the typography and color palette to the pure CSS illustrations (the abstract hero graphic and the data charts). The standard `ease-*` interaction classes are attached directly to HTML elements to provide smooth, professional animations. No JavaScript is required for this template.
