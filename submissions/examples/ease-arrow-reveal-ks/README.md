# Reveal Arrow Icon

## What does this do?
Slides in an arrow icon from the right and shifts the button text slightly to the left on hover.

## How is it used?
Add the class to your button:

```html
<button class="arrow-reveal-btn">
  <span>Get Started</span>
</button>
```

## Why is it useful?
It provides clean, dynamic feedback for user navigation flow, helping to indicate interactive progress or direction.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
