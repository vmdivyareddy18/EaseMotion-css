1. What does this do?
An advanced typography navigation link component that splits text geometry along a precise geometric fault line on hover, simultaneously animating a secondary informational string into the fissure.

2. How is it used?
```html
<a href="#" class="shatter-link">
  <span class="shatter-track shatter-top">LINK TEXT</span>
  <span class="shatter-track shatter-bottom">LINK TEXT</span>
  <span class="shatter-ghost">SUB TEXT</span>
</a>
```
