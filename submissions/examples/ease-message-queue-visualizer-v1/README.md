# Ease Message Queue Visualizer

## Overview

Ease Message Queue Visualizer is a reusable backend-inspired UI component that demonstrates the flow of messages between a producer, queue, and consumer using animated packets and queue indicators.

---

## Features

- 📤 Producer node
- 📬 Queue visualization
- 📥 Consumer node
- 🟢 Animated message packets
- ✨ Queue pulse animation
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---
