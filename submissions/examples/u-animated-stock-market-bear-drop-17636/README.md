# Animated Stock Market Bear Drop

## Overview

Animated Stock Market Bear Drop is a lightweight CSS animation that simulates bearish market momentum using a moving bear icon and a declining trend indicator.

## Features

- CSS-only animation
- Bear market motion effect
- Downtrend indicator animation
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="bear-drop">
  <span class="bear">🐻</span>
  <span class="trend-arrow">📉</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
