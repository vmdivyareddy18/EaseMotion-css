# Seesaw — Teeter-Totter

CSS seesaw with pivoting board and two seats that move in opposite vertical arcs. The board rotates around its center while seats interpolate height.

## Files

- `demo.html` — pivot, board, left/right seats
- `style.css` — board tilt keyframe, seat vertical motion

## Usage

Open `demo.html` to see the seesaw rock.
