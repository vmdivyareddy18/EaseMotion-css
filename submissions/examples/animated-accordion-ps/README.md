# Animated Accordion Component

1. What does this do? Creates a fully pure CSS animated accordion menu with smooth expanding and collapsing panels.
2. How is it used? Copy the `.animated-accordion-ps` HTML structure which relies on hidden radio buttons (`.accordion-toggle`) and labels (`.accordion-header`).
3. Why is it useful? It provides a robust, Javascript-free way to present dense information neatly, utilizing CSS Grid interpolation for buttery smooth opening/closing transitions.
