# CSS-only Gooey Floating Action Menu

A modern Floating Action Button (FAB) menu built entirely using HTML and CSS.

## Features

- Pure HTML & CSS
- No JavaScript
- Floating Action Button
- Expanding action menu
- Smooth staggered animations
- Responsive design
- Easy to customize

## Folder Structure

```
gooey-fab-menu/
├── demo.html
├── style.css
└── README.md
```

## How to Run

Simply open `demo.html` in any modern web browser.

## Customization

You can easily modify:

- Button colors
- Icons
- Animation duration
- Size
- Position
- Number of menu items

## Technologies Used

- HTML5
- CSS3
