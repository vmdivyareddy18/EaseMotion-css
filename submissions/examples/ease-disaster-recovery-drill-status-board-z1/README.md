# Ease Disaster Recovery Drill Status Board

## Overview

A responsive HTML and CSS component for monitoring disaster recovery drills, recovery objectives, and failback readiness.

## Features

- Disaster recovery drill summary
- RTO monitoring
- RPO validation status
- Failback readiness
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable UI component for disaster recovery planning, business continuity dashboards, cloud infrastructure monitoring, and enterprise resilience reporting.
