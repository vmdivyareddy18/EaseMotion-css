# Swing Color Picker

A Neon-inspired Swing Color Picker built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Swing animation
- Neon-inspired color palette
- Responsive layout
- Accessible HTML
- No JavaScript required
