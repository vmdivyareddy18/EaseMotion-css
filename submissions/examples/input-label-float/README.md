# Input Label Float

A floating-label form field where the label moves upward when the input is focused or contains text.

## Files

- `demo.html` - email input with floating label markup.
- `style.css` - focus state, label transform, valid content state, and reduced-motion support.

## Highlights

- Uses `:placeholder-shown` to detect empty input state.
- Keeps the label visible while the user types.
- Includes keyboard focus styling and reduced-motion handling.
