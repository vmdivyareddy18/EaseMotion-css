# Ease Cerberus Bark Animation

1. What does this do? Creates a three-stage aggressive "barking" animation. The element snaps to the left, snaps to the right, and then lunges violently forward and up, mimicking the three heads of Cerberus barking in succession.
2. How is it used? Apply `.ease-cerberus-bark-ps` for an immediate sequence, or `.ease-cerberus-bark-hover-ps` to trigger the three barks when the user hovers over the element.
3. Why is it useful? It is an extremely engaging error state or warning animation. When standard "shakes" aren't aggressive enough to grab a user's attention, the Cerberus bark ensures they notice the interaction.
