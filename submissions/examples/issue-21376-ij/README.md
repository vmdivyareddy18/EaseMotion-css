# Bee Buzzing

1. What does this do? A CSS-only bee that hovers erratically with rapidly vibrating wings.

2. How is it used? Place .bee inside .scene. The body is a striped ellipse; wings are a translucent shape with a fast scaleX animation.

3. Why is it useful? The extremely fast wing animation (0.08s) creates the illusion of buzzing flight.
