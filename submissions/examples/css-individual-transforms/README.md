# CSS Individual Transform Properties Example

## Description
A comprehensive interaction showcase demonstrating modern CSS individual transform properties. Illustrates how declaring `translate`, `rotate`, and `scale` independently simplifies transition logic and prevents cascade bugs.

## Key Features Demonstrated
- **Decoupled Properties**: Manipulating distinct spatial variables without re-declaring complete shorthand `transform` expressions.
- **Micro-Interaction States**: Isolated hover actions showing clean, performant hardware-accelerated animations.
