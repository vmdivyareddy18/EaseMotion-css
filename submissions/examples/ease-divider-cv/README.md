# Animated Divider (cv variant)

## What does this do?
A collection of pure CSS animated dividers with 4 variants (Gradient Flow, Glow Line, Dotted with Icon, Animated Arrow) and 3 color options. Perfect for separating content sections with visual flair.

## How is it used?

### Basic Usage:

```html
<!-- Gradient flow divider -->
<div class="ease-divider-cv ease-divider-cv-flow"></div>

<!-- Glow line divider -->
<div class="ease-divider-cv ease-divider-cv-glow"></div>

<!-- Dotted with icon -->
<div class="ease-divider-cv ease-divider-cv-dotted">
    <span class="ease-divider-cv-icon">✨</span>
</div>

<!-- Animated arrow -->
<div class="ease-divider-cv ease-divider-cv-arrow"></div>

<div class="ease-divider-cv ease-divider-cv-flow ease-divider-cv-primary"></div>
<div class="ease-divider-cv ease-divider-cv-flow ease-divider-cv-info"></div>
<div class="ease-divider-cv ease-divider-cv-flow ease-divider-cv-success"></div>
