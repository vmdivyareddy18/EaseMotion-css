# AI Chat Assistant Workspace

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive AI chat interface for interacting with conversational assistants. Includes a conversation sidebar, chat window, message bubbles, and message composer. Suitable for AI chatbots, customer support tools, productivity assistants, and enterprise AI platforms.

## Features

- Chat conversation sidebar
- Responsive messaging interface
- AI and user message bubbles
- Message composer
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="dashboard">

    <aside class="sidebar">
        ...
    </aside>

    <main class="chat">
        ...
    </main>

</div>
```
