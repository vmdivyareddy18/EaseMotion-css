# Cubic Bezier Playground

An interactive playground for exploring CSS `cubic-bezier()` curves. Predefined curves are shown alongside their visual graph, and a custom input lets you enter any four-value bezier and see the result in real time.

## Features

- Six predefined curves with SVG timing-diagram previews.
- Custom cubic-bezier input field with Apply & Animate button.
- Each card shows the curve graph and an animated ball on a track.
- Colour-coded custom curve (amber) vs standard curves (indigo).

## Topics Covered

- `cubic-bezier()` timing function with four control points
- Visual relationship between the bezier graph and motion
- Named easings as shorthand cubic-bezier values
- Overshoot / bounce effects with control points outside 0–1
