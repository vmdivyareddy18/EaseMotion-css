# Blur Language Switcher

A CSS-only animated language switcher component inspired by glassmorphism design patterns.

## Features

- Blur glass effect
- Smooth entrance animation
- Hover interactions
- Responsive layout
- Pure CSS implementation
- No JavaScript required

## Usage

```html
<div class="language-switcher">
  <button class="active">English</button>
  <button>Spanish</button>
</div>
