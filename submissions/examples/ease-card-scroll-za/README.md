# Card Scroll Carousel

## What does this do?
A CSS horizontal scroll-snap carousel with custom scrollbar styling, snap alignment options, and gradient cards.

## How is it used?
```html
<div class="crs-wrapper"><div class="crs-track"><div class="crs-card">Card</div><div class="crs-card">Card</div></div></div>
```

## Why is it useful?
Touch-friendly horizontal card browsing without JavaScript. Scroll-snap for precise alignment. Custom styled scrollbar. Responsive and accessible.
