# Wobble Sticky Header

A CSS-only animated sticky header component inspired by modern landing page designs.

## Features

- Sticky navigation header
- Smooth wobble entrance animation
- Pure CSS implementation
- Responsive layout
- Accessible navigation

## Usage

Add the header structure:

```html
<header class="wobble-header">
  <nav>
    <a href="#">Home</a>
  </nav>
</header>
