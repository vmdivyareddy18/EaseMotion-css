# Incident Command Center

An intermediate EaseMotion CSS example for incident response dashboards. Status cards sweep into place with subtle scanning motion for severity, owner, and next action.

## Files

- `demo.html` - Semantic incident command summary markup.
- `style.css` - Responsive panel layout, staggered entrance motion, and scanning status cards.

## Usage

Open `demo.html` in a browser to preview the animated incident command center.
