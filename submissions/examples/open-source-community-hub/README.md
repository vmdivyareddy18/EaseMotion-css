# Open Source Community Hub

## Overview
Design a contributor engagement dashboard.

This project implements the Phase 103 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Contribution cards**: Built with glassmorphism and CSS grid/flexbox.
- **Community metrics**: Built with glassmorphism and CSS grid/flexbox.
- **Activity feed**: Built with glassmorphism and CSS grid/flexbox.
- **Achievement widgets**: Built with glassmorphism and CSS grid/flexbox.
- **Community Stats**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
