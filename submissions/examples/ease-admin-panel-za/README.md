# CSS Admin Panel Component

## What does this do?
A complete CSS admin panel with sidebar navigation, user management table, search/filter toolbar, pagination, stats cards, activity timeline, and modal dialog.

## How is it used?
```html
<div class="adm-layout">
  <aside class="adm-sidebar">Navigation</aside>
  <main class="adm-main">
    <div class="adm-card">
      <table class="adm-table">...</table>
    </div>
  </main>
</div>
```

## Why is it useful?
Provides a full-featured admin interface for user management dashboards. Includes data tables with search, filter, pagination, role badges, status indicators, and modals. Responsive with sidebar collapse on mobile.
