# Loader Bounce Animation Variant

## What does this do?

Creates a lightweight loading indicator using three bouncing dots animated with pure CSS.

## How is it used?

```html
<div class="loader-bounce">
  <span></span>
  <span></span>
  <span></span>
</div>
```

## Why is it useful?

Loading indicators provide visual feedback while content is being fetched or processed. The bouncing animation is simple, recognizable, and works well across dashboards, forms, and applications.
