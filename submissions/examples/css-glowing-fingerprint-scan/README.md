# CSS Glowing Fingerprint Scan Animation Component

A futuristic, accessible biometric scanner component designed in lightweight, pure CSS featuring standard hardware-accelerated looping sweep transitions.

## Features
- **Dynamic Mask Lighting Effect:** Fingerprint details smoothly transition colors and expand drop shadows using performant filters on user interaction.
- **Bi-Directional Keyframes Sweep:** An animated sweeping horizontal gradient tracks perfectly down the height bounds of the wrapper.
- **Accessible & Focusable:** Supports standard `:focus-within` triggers along with mouse hover gestures for standard access paths.
- **Modular Config Options:** Easily scale the timing curves, tint colors, background limits, and glow intensities using native CSS variables.
