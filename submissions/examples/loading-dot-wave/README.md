# Loading Dot Wave

A compact loading indicator with four staggered dots and clear status text for async UI states.

## Files

- `demo.html` - status region markup for the loader.
- `style.css` - staggered dot animation, pill container styling, and reduced-motion fallback.

## Highlights

- Uses `role="status"` and visible status copy.
- Staggers each dot with small animation delays.
- Disables movement for reduced-motion users.
