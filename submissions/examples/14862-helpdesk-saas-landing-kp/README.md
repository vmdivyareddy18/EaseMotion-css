# Helpdesk SaaS Landing Page

## What does this do?
It creates a full responsive landing page for a helpdesk SaaS with ticket cards, support features, and metric highlights.

## How is it used?
```html
<section class="hero">
  <h1>Resolve customer tickets faster with one shared support inbox.</h1>
  <a class="button" href="#inbox">View inbox</a>
</section>
```

## Why is it useful?
It gives EaseMotion a practical SaaS support page pattern with clear motion for queue activity and responsive marketing sections.
