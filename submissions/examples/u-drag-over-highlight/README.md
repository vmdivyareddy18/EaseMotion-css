# ease-drag-over-highlight

Animated drag-and-drop file upload zone with highlight effect.

## Features

- Border color transition
- Background highlight animation
- Scale effect on drag-over
- Responsive layout
- Pure HTML and CSS

## Acceptance Criteria Covered

### Drag Over Highlight

When `.ease-drag-over` class is applied:

- Border color changes
- Background color changes
- Glow effect appears
- Scale animation occurs

### JavaScript Integration

Can be triggered using:

- dragenter
- dragleave
- dragover
- drop

events.

### Use Cases

- File upload forms
- Cloud storage apps
- Dashboard upload panels
- Media upload tools
