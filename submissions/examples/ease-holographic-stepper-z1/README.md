# Ease Holographic Stepper

## Overview

A futuristic multi-step progress component featuring holographic gradients, floating cards, and smooth animations using only HTML and CSS.

## Features

* Holographic step cards
* Animated conic gradients
* Floating hover interaction
* Responsive Flexbox layout
* Lightweight CSS
* Pure HTML & CSS

## Folder Structure

```text
ease-holographic-stepper-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

* Chrome

Optimized for desktop, tablet, and mobile screens.
