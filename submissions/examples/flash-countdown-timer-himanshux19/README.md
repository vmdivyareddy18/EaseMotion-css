# Flash Countdown Timer

A responsive Flash Countdown Timer component built using pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- Flash animation
- Responsive design
- Glassmorphism timer cards
- Modern landing-page style
- Accessible markup
- No JavaScript required

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<div class="countdown">
  <div class="card ease-flash">
    <h2>07</h2>
    <span>Days</span>
  </div>

  <div class="card ease-flash">
    <h2>12</h2>
    <span>Hours</span>
  </div>

  <div class="card ease-flash">
    <h2>45</h2>
    <span>Minutes</span>
  </div>

  <div class="card ease-flash">
    <h2>18</h2>
    <span>Seconds</span>
  </div>
</div>
```

Include the provided `style.css` to enable the component styling and animation.

## Accessibility

- Semantic HTML
- Responsive layout
- Keyboard-friendly button
- High-contrast text

## Why EaseMotion CSS?

The component showcases subtle motion through a lightweight flash animation, making countdown sections more engaging for landing pages and promotional campaigns without requiring JavaScript.
