# Wobble Rating Stars

Resolves #42449.

### What does this do?
Adds a Wobble Rating Stars component to EaseMotion CSS tailored for News Portal/Blog designs. It utilizes a CSS-only rating system via hidden radio buttons and `flex-direction: row-reverse` sibling logic. The selected star triggers an `ease-wobble` animation for satisfying feedback upon rating.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Click on any star to rate the article; watch the star playfully wobble as it locks in the rating.

### Why is it useful?
Provides an elegant, no-JS rating component perfectly suited for typography-heavy news sites, giving readers immediate, delightful micro-interaction feedback.
