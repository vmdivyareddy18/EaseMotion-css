# Ease Liquid Signal Bars

## Overview

A premium animated signal indicator featuring liquid-style vertical bars with glowing gradients and smooth wave motion using pure HTML and CSS.

## Features

- Liquid signal animation
- Gradient glowing bars
- Glassmorphism card
- Responsive layout
- Lightweight implementation
- Pure HTML & CSS

Works smoothly across desktop, tablet, and mobile devices.
