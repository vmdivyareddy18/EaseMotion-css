# Animated Logistics Dashboard

A logistics analytics dashboard built using pure HTML and CSS.

## Features

- Deliveries overview
- Shipment analytics
- Warehouses
- Revenue metrics
- Activity feed
- Responsive layout
- CSS Variables
- Pure CSS animations
