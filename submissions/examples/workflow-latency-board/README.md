# Workflow Latency Board

An intermediate EaseMotion CSS example for workflow telemetry dashboards. Staggered rows and animated bars compare latency across intake, review, and deploy stages.

## Files

- `demo.html` - Semantic workflow latency board markup.
- `style.css` - Responsive board layout, staggered row motion, and animated latency bars.

## Usage

Open `demo.html` in a browser to preview the animated workflow latency board.
