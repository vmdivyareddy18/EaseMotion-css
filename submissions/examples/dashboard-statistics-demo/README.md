# Dashboard Statistics Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be applied to dashboard statistic cards.

---

## Included Cards

- Users
- Revenue
- Orders
- Products

---

## Folder Structure

```
submissions/examples/dashboard-statistics-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to verify responsiveness.
3. Observe the animation effects applied to each statistics card.

## Benefits

- Responsive dashboard layout
