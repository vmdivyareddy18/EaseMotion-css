# CSS Elastic Slide Tooltip - Minimalist Tech Layouts

A pure CSS animated tooltip with a smooth elastic slide transition designed
for minimalist technology interfaces.

## Features

- Pure CSS animation
- No JavaScript required
- Minimal dark tech UI style
- Responsive design
- Keyboard accessible
- Custom CSS animation properties

## Usage

```html
<div class="tooltip-wrapper">

  <button class="tech-button">
    Explore Feature
  </button>

  <div class="tooltip">
    Tooltip content
  </div>

</div>
