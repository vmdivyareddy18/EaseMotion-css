# Sci-Fi Console Shimmer Pulse Modal

## What does this do?
This submission provides a pure CSS shimmer pulse modal for a futuristic console interface. The modal uses a dark sci-fi palette with neon highlights and a sweeping shimmer effect that activates on hover and keyboard focus.

## How is it used?
Open the launch button in the demo to view the modal overlay. The component is fully semantic and uses no JavaScript.

## Why is it useful?
It offers a lightweight, accessible modal pattern for dashboard and console-style interfaces while remaining easy to customize.

## Customization variables
- `--em-duration`
- `--em-delay`
- `--em-easing`
- `--em-scale`
- `--em-shimmer-color`
- `--em-glow-color`
