# Circuit Trace

## What does this do?
Animated circuit board trace lines that glow and pulse, simulating data flowing through circuit paths.

## How is it used?
Apply the `.circuit-trace` class to a container. Horizontal and vertical lines with dots create a circuit board aesthetic.

## Why is this useful?
Adds a tech-themed visual element to dashboards, loading screens, and developer tools, reinforcing a technical or futuristic brand.
