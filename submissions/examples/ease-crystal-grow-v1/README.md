# Ease Crystal Grow

## Overview

Ease Crystal Grow is a futuristic CSS animation where crystal-like polygons emerge from the center using scale, blur, rotation, and glow effects.

---

## Features

- 💎 Crystal growth animation
- ✨ Blur to sharp transition
- 🌈 Three color variants
- ⚡ Pure HTML & CSS
- 🎯 Polygon-based crystal shape
- 🚀 Smooth easing animation

---

## Included Variants

- Cyan Crystal
- Blue Crystal
- Purple Crystal

---

## Usage

```html
<div class="crystal"></div>
```

---

## Customization

You can customize:

- Crystal colors
- Animation duration
- Glow intensity
- Crystal size
- Polygon shape

---

## Browser Support

Works in modern browsers supporting CSS clip-path, transforms and animations.
