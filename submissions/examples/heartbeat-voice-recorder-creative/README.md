# Heartbeat Voice Recorder — Creative Agency Patterns

An expressive pure CSS audio capturing node implementing continuous kinetic concentric pulse arrays optimized for production media elements.

## Specifications
- **Mechanical Physics**: Tracks propagation loops via specialized cubic timelines (`cubic-bezier(0.215, 0.610, 0.355, 1)`) to mimic organic heart contractions.
- **Declarative Operations**: Implements layered delay intervals using inline variable matrices (`--wave-delay`) directly within clean presentation files.
- **Hardware Layering**: Offloads volumetric expanding updates wholly to GPU composite draw paths to prevent active frame drops under heavy layouts.
