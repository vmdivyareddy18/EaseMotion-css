# CSS 3D Flip Modal – Product Catalog Layout

## Overview

A responsive Product Catalog modal built with pure HTML and CSS featuring a smooth 3D flip animation. The component uses CSS custom properties for easy customization and requires no JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D flip animation
- Responsive design
- Product catalog interface
- Glassmorphism modal
- CSS custom properties
- Keyboard accessible
- No JavaScript required

## CSS Custom Properties

You can customize the animation and appearance using:

- `--primary`
- `--secondary`
- `--duration`
- `--easing`
- `--flip-angle`
- `--modal-scale`
- `--modal-perspective`
- `--blur`
- `--radius`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in a browser. Click **View Details** to see the animated 3D flip modal. Modify the CSS custom properties in `style.css` to customize the appearance and animation.
