# Full Creative Agency Portfolio (`ease-page-agency`)

A bold, dark-themed, and highly interactive landing page template designed for creative agencies, design studios, and high-end portfolios.

## 🚀 Features & EaseMotion Showcase

This landing page integrates multiple core `.ease-*` classes alongside custom interactive elements to create an immersive, premium studio experience:

- **Custom Cursor Integration**: Features a JS-driven custom cursor that interacts with the UI, expanding when hovering over interactive elements. This pairs beautifully with `.ease-hover-scale` and `.ease-hover-lift` classes.
- **Asymmetric Masonry Gallery**: Uses a mix of standard and large spanning grid items to create a staggered, editorial layout for the portfolio, revealed using `.ease-slide-up` with progressive animation delays.
- **Continuous Marquee**: Includes an infinite, CSS-only text marquee for client logos.
- **Alternating Timeline**: A vertical process timeline that uses opposing `.ease-slide-left` and `.ease-slide-right` animations to draw the user's eye back and forth down the page.

## 🛠️ Usage

This demo is entirely self-contained. You can open `demo.html` directly in your browser. All required EaseMotion CSS classes have been defined at the top of the `style.css` file for immediate testing. 

If using this as a template for a real project, simply migrate the HTML structure into your application and ensure your global CSS bundle includes the `ease-*` utility classes.
