# Ease Floating Label Badge

## Overview

A lightweight floating badge component with a subtle animation created using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Floating badge animation
- Responsive layout
- Easy to customize
- Zero dependencies

## Folder Structure

```
ease-floating-label-badge-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox and Safari.

## Why This Example

This example provides a reusable floating badge suitable for cards, product highlights, and dashboard widgets.
