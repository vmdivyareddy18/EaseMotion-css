# Voice Input Button – Pulsing mic button

> A production-ready voice input button component with a pulsing microphone, speech-to-text support, real-time transcription, and full customization. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Pulsing mic button** – Animated pulsing effect when listening
- ✅ **Speech-to-text** – Web Speech API integration
- ✅ **Real-time transcription** – Live transcript display
- ✅ **Confidence tracking** – Shows recognition confidence
- ✅ **Auto-stop timer** – Stops on silence
- ✅ **Sensitivity control** – Adjust recognition sensitivity
- ✅ **Demo mode** – Simulated voice input
- ✅ **Keyboard shortcuts** – Space (toggle), Escape (stop), C (clear)
- ✅ **4 color variants** – Ocean, Sunset, Gold, Dream
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
