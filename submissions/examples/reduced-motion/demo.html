Accessibility Demo: Reduced Motion

This page demonstrates how CSS animations can respect the user's prefers-reduced-motion accessibility preference.

Default Animation

Fade In

When reduced motion is NOT enabled, this animation plays normally.

Reduced Motion

Static Box

When the operating system requests reduced motion, animations are automatically disabled.

CSS Used

@media (prefers-reduced-motion: reduce) {

    * {
        animation: none !important;
        transition: none !important;
        scroll-behavior: auto !important;
    }

}

Why This Matters

  • Reduces motion sickness.
  • Improves accessibility.
  • Supports users with vestibular disorders.
  • Respects operating system accessibility settings.
  • Follows modern accessibility best practices.

How to Test

  1. Open Chrome DevTools.
  2. Press Ctrl + Shift + P.
  3. Search for Rendering.
  4. Open Rendering panel.
  5. Find Emulate CSS prefers-reduced-motion.
  6. Select Reduce.
  7. Refresh the page.
