# Ease Notification Stack

## Overview

Ease Notification Stack is a reusable toast notification animation featuring staggered slide-in effects, stacked layout, hover interactions, and multiple notification styles using only HTML and CSS.

---

## Features

- 🔔 Stacked notifications
- ✨ Slide-in animation
- 🎨 Success, Info & Warning variants
- 🚀 Hover interaction
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Success notification
- Info notification
- Warning notification
- Smooth staggered entrance

---

## Usage

```html
<div class="notifications">
...
</div>
```

---

## Customization

- Toast colors
- Animation speed
- Stack spacing
- Width
- Border styles

---

## Browser Support

Compatible with all modern browsers supporting CSS animations and transitions.
