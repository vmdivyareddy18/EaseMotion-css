# Accessible Floating Orbit Popover

A pure CSS animated Floating Orbit Popover component built for accessible web layouts.

## Features

- CSS-only animation
- No JavaScript dependency
- Smooth floating orbit transition
- Keyboard accessible using focus states
- Responsive layout
- Custom animation variables

## Usage

```html
<div class="orbit-popover">

  <button class="orbit-trigger">
    Open Details
  </button>

  <div class="orbit-content">
    Popover content
  </div>

</div>
