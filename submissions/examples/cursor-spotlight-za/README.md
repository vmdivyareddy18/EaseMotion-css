# Cursor Spotlight Effect

A spotlight effect that follows the cursor using CSS radial gradients and a small JS event listener to update custom properties.

## Files

- `demo.html` - Spotlight wrapper with JS mouse tracking
- `style.css` - Radial gradient spotlight, gradient text, dot animations

## Usage

Open `demo.html` and move your cursor over the page to see the spotlight reveal content.
