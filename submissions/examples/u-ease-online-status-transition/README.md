# ease-online-status-transition

Smooth online/offline/away status transition animation component built with HTML and CSS.

## Features

- Online status with ease-ping animation
- Away status with soft pulse
- Busy status indicator
- Offline static state
- Smooth color transitions
- Status badge examples
- Responsive design

## States

### Online
Green status indicator with animated ping effect.

### Away
Orange indicator with gentle pulse animation.

### Busy
Red indicator for unavailable users.

### Offline
Gray static indicator without animation.

## Acceptance Criteria Covered

- Smooth color transitions between states
- Online state uses ease-ping effect
- Offline state remains static
- Based on status badge pattern
