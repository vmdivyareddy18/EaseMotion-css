# Ease Backdrop Blur Large

Closes #13610

## Description
A utility class for heavy backdrop blur (`blur(24px)`) providing strong glassmorphism. Perfect for overlay panels, modals, and navigation elements that need maximum frosted glass effect.

## Acceptance Criteria
- `backdrop-filter: blur(24px)` for heavy glassmorphism
- Works over colorful gradient backgrounds
- Toggleable blur levels for comparison
- Smooth transition between blur values

## Files
- `style.css` — Dark theme with heavy backdrop blur utility
- `demo.html` — Demo with blur level comparison controls
