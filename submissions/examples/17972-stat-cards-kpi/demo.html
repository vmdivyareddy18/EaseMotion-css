CSS Techniques Used

The hover lift effect, trend badges, and sparkline bars are all pure CSS.

/* Hover lift + glow */
.stat-card:hover {
  transform: translateY(-6px);
  box-shadow: var(--ease-card-hover-shadow), 0 0 0 1px var(--card-accent);
}

/* Sparkline bars via CSS custom property */
.sparkline-bar {
  height: var(--h, 50%);
}

/* Number counter animation (JS IntersectionObserver) */
.stat-card__value {
  font-variant-numeric: tabular-nums;
}
