# Sci-Fi Glow Pulse Animation

Resolves Issue #38415

## Description
This submission adds a futuristic `ease-glow-pulse-scifi` utility class. It applies a neon cyan/blue glowing box-shadow animation that pulses along with a scale effect, creating a "Sci-Fi Console" look. It also handles the `prefers-reduced-motion` media query for accessibility.

## Usage
Simply apply the `.ease-glow-pulse-scifi` class to any UI element (like a button or a badge) to give it a futuristic glow effect.

```html
<button class="ease-glow-pulse-scifi">Activate Core</button>
```

## Why it fits EaseMotion CSS
This animation provides a complex, accessible neon glow effect that aligns with EaseMotion's goal of offering ready-to-use motion utilities. It takes the heavy lifting out of creating detailed box-shadow keyframes for sci-fi styled components.
