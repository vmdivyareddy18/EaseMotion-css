# 404 Error Card

## What does this do?

Creates a modern 404 error card with a glassmorphism layout, responsive design, and call-to-action buttons.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="card">
    <h1>404</h1>
    <h2>Page Not Found</h2>
</div>
```

---

## Why is it useful?

404 pages are a common part of modern websites. This component provides a polished and reusable error interface with a clean visual style that can be easily adapted for portfolios, SaaS platforms, documentation sites, and dashboards.
