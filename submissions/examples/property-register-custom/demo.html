@property — Registered Custom Properties

Typed CSS custom properties with @property for animation

Why Register Custom Properties?

Using @property defines a property's syntax, initial value, and inheritance, enabling native transitions and animations.

Animated Gradient

A live gradient whose angle, colors, and stops are registered custom properties.

Registered Property Types

<length>

Animated --box-size transitions smoothly.

<color>

Animated --box-color interpolates in color space.

<angle>

Animated --box-angle rotates continuously.

<number>

Animated --box-opacity fades smoothly.

<percentage>

Animated --box-scale scales in and out.

<integer>

Animated --box-count renders steps.
