# @property — Registered Custom Properties

Demonstrates CSS `@property` for registering typed custom properties with defined syntax, inherits, and initial-value. Features animated gradients and boxes that transition <length>, <color>, <angle>, <number>, <percentage>, and <integer> types.

## Usage

Open `demo.html` to see live animations of registered properties. Use Play/Pause to control the gradient animation. Six property-type boxes demonstrate how typed properties enable smooth transitions that unregistered custom properties cannot achieve.

## Browser Support

`@property` is supported in Chrome 85+, Edge 85+, and Firefox 128+. Safari support is behind a flag. The demo includes `prefers-reduced-motion` fallbacks that disable animations while preserving the layout. Dark theme base is `#0a0f1e` with `#e2e8f0` text.
