# ease-number-flip-counter

## What this is
A CSS-only number flip counter with a digit-roll clock effect.

## Why
It makes numeric values feel more dynamic and polished while keeping the component lightweight and easy to theme.

## How to test
1. Open `demo.html` in a browser.
2. Confirm the digits animate with a flip-style rolling motion.
3. Check that the motion loops smoothly.
4. Resize the browser to verify responsive behavior.
5. Update the CSS custom properties and confirm the styling still works.

## Note
This is a visual demo component and uses only HTML and CSS, so no JavaScript is needed.
