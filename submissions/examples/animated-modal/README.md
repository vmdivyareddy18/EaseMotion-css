# Animated Modal Component

### 1. What does this do?
Provides a smooth, self-contained modal overlay box that utilizes physics-based cubic-bezier transitions for an elegant entrance animation.

### 2. How is it used?
```html
<div class="modal-overlay">
  <div class="modal-box">
    <h3 class="modal-title">Title</h3>
    <p class="modal-text">Content</p>
  </div>
</div>
