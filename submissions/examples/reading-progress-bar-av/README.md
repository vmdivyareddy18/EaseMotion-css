# Reading Progress Bar

## What does this do?

Creates a modern reading progress indicator with a sticky gradient progress bar and a premium article layout.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="progress-track">
    <div class="progress-fill"></div>
</div>
```

---

## Why is it useful?

Reading progress indicators are commonly used in blogs, documentation websites, knowledge bases, and long-form articles. This showcase demonstrates a polished visual implementation that can later be connected to JavaScript for dynamic scroll tracking while remaining a clean, standalone HTML/CSS example.
