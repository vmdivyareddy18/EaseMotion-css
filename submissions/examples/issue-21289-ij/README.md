# Parachute Drop — Parachute Descending

CSS parachute descending through the sky. The `descend-parachute` keyframe moves the assembly downward while the `sway-chute` animation rocks the canopy side-to-side.

## Files

- `demo.html` — chute dome, suspension lines, person figure
- `style.css` — descent trajectory, canopy sway person styling

## Usage

Open `demo.html` to see the parachute float down.
