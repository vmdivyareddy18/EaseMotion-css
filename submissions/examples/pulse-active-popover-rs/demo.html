Pulse-Active Popover

Pure CSS animated popover with pulse-active interaction for accessible web layouts

Basic Usage

Default (Bottom)

Click the button to open the popover with a smooth pulse animation.

Information

This is a basic popover with pulse animation. The subtle pulse effect draws attention without being distracting.

Top Position

Popover appears above the trigger with custom pulse timing.

Top Popover

Appears above the trigger element. Custom pulse duration of 1.5s with pink accent color.

Right Position

Popover positioned to the right with green accent.

Right Side

Positioned to the right with a slower 2.5s pulse and green accent.

Left Position

Popover positioned to the left with custom scale.

Left Side

Left positioned with increased base scale (1.05) and pulse scale (1.04).

Keyboard Accessibility

Focus-Triggered

Tab to the button and the popover opens automatically. Press Escape or Tab away to close.

Keyboard Accessible

This popover opens on focus (Tab key) and closes when focus leaves. No pulse animation on focus trigger for better UX.

With Form Elements

Popover containing interactive form controls with proper focus trapping.

Preferences

Custom Duration

Faster pulse animation for urgent notifications.

Warning

Fast pulse (1s) with red accent for urgent messages. Larger pulse scale for higher visibility.

No Pulse (Static)

Disable pulse animation for subtle interactions.

Static Mode

Pulse animation disabled by setting --popover-pulse-duration: 0s. Clean fade/scale transition only.

HTML Structure

Minimal Setup

<!-- Include EaseMotion CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css">
<link rel="stylesheet" href="style.css">

<!-- Popover Component -->
<div class="pulse-popover-rs">
  <input type="checkbox" id="my-popover" class="pulse-popover-toggle-rs" />
  <label for="my-popover" class="pulse-popover-trigger-rs">
    Open Popover
  </label>
  <div class="pulse-popover-content-rs" role="dialog" aria-label="Popover content">
    <div class="pulse-popover-header-rs">
      <h4 class="pulse-popover-title-rs">Title</h4>
      <button type="button" class="pulse-popover-close-rs" aria-label="Close">&times;</button>
    </div>
    <div class="pulse-popover-body-rs">
      <p>Your content here...</p>
    </div>
    <div class="pulse-popover-footer-rs">
      <button type="button" class="pulse-popover-btn-primary-rs">Action</button>
    </div>
  </div>
</div>

Custom Properties API

CSS Variables

Override these variables on the .pulse-popover-rs element or globally via :root:

Variable Default Description
--popover-duration 0.25s Open/close transition duration
--popover-ease cubic-bezier(0.4, 0, 0.2, 1) Transition easing function
--popover-pulse-duration 2s Pulse animation cycle duration
--popover-pulse-ease ease-in-out Pulse easing function
--popover-scale 1 Base scale factor when open
--popover-pulse-scale 1.02 Scale factor at pulse peak
--popover-gap 8px Gap between trigger and popover
--popover-max-width 280px Maximum popover width
--popover-bg --ease-color-bg Background color
--popover-border --ease-color-neutral-200 Border color
--popover-shadow --ease-shadow-lg Box shadow
--popover-text --ease-color-text Text color
--popover-radius --ease-radius-lg Border radius
--popover-z-index --ease-z-dropdown Z-index for layering
