# Ease Cherry Blossom Animation

A multi-axis ambient trajectory animation designed to mirror lightweight, organic descent profiles like falling petals or festive confetti streams.

## Features
- **3D Air Flipping**: Combines standard 2D rotation parameters alongside `rotateY` properties to simulate fluttering face-turns.
- **Asymmetric Sway Grid**: Uses alternating horizontal `translateX` shifts across the descent profile to create a true wind-drifted appearance.
- **Continuous Ambient Looping**: Seamless entry and exit opacity masks make it easy to drop directly onto backgrounds.

## Core Utility Usage
```html
<div class="element ease-cherry-blossom">🌸</div>
<div class="element ease-cherry-blossom-fast">🌸</div>
<div class="element ease-cherry-blossom-slow">🌸</div>
