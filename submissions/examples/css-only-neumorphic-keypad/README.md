# CSS-only Animated Neumorphic Keypad Component

A fully responsive, pure CSS Animated Neumorphic Keypad component designed for PIN entry, calculators, or dialer interfaces. This component showcases advanced usage of CSS box-shadows, gradients, and transition delays to create a realistic 3D soft-UI (neumorphism) effect when buttons are pressed, without relying on JavaScript.

## Files
- `demo.html` - Interactive demonstration of the neumorphic keypad with realistic press effects.
- `style.css` - The CSS implementation utilizing EaseMotion tokens and complex shadow stacking.

## Details
This addresses the need for premium, modern, soft-UI components within the EaseMotion CSS framework. It uses layered `box-shadow` techniques (both outset and inset) coupled with CSS custom properties to simulate a physical button being pressed down into a soft surface. Everything is hardware-accelerated via `transform` and strictly CSS-only.
