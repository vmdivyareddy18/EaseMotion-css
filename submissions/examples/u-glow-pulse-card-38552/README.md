# Glow Pulse Card

A premium **Glow Pulse Card** built with pure CSS for **SaaS Pricing Layouts**. The component highlights featured pricing plans with a smooth glowing pulse animation while remaining lightweight, responsive, and accessible.

---

## Features

- Pure CSS implementation
- Smooth Glow Pulse animation
- Responsive pricing card
- Modern SaaS-inspired design
- Keyboard accessible
- CSS Custom Properties
- Zero JavaScript dependency
