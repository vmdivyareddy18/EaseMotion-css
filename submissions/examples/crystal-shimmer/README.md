# Crystal Shimmer Animation

## What does this do?
A faceted crystal shape with shifting spectral colors and moving light reflections.

## How is it used?
Add the `.crystal` container with `.crystal-shape` and `.shine` elements.

## Why is it useful?
Crystal effects enhance fantasy inventories, gem displays, and magical object interactions.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
