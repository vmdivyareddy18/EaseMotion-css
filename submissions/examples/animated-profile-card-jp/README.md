# Animated Profile Card Component

## Description

A modern profile card component with smooth hover animations, animated avatars, social media buttons, and responsive layouts.

---

## Features

- Smooth card lift animation
- Avatar zoom and rotation effect
- Animated social media buttons
- Responsive card layout
- Rounded modern design
- Lightweight CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.profile-container` – Profile card wrapper
- `.profile-card` – Individual profile card
- `.avatar` – Profile image
- `.role` – User designation
- `.social-icons` – Social links container

---

## Accessibility

Supports `prefers-reduced-motion` to reduce animation effects.

---

## Use Cases

- Team member sections
- Portfolio websites
- Developer profiles
- Company websites
