📦 Block Flex vs. Inline Flex

Standard display: flex generates a block-level container that fills its parent's width and breaks the text flow. In contrast, display: inline-flex generates an inline-level container that sits on the same line as surrounding text and occupies only as much width as its children require.

🎛️ Inline-Flex Alignment Playground

Aligning inline items alongside raw text is tricky. You can combine CSS vertical-align (to align the container box relative to the text baseline) and flexbox align-items (to align elements inside the container).
