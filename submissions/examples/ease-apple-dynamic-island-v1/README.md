# Ease Apple Dynamic Island

## Overview

Ease Apple Dynamic Island recreates Apple's expanding notification area using only HTML and CSS. The island smoothly grows on hover to reveal compact content.

---

## Features

- 📱 Dynamic Island animation
- ✨ Smooth expansion
- 🎵 Notification preview
- 🚀 Hover interaction
- 📱 Responsive demo
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Phone mockup
- Expandable island

- Music notification

---

## Usage

```html
<div class="island">
...
</div>
```

## Customization

- Colors
- Notification content

## Browser Support

Compatible with all modern browsers supporting CSS transitions.

## Why This Fits EaseMotion CSS

Ideal for modern UI showcases, mobile mockups, portfolios, landing pages, and Apple-inspired interfaces. It provides a reusable animated component with no JavaScript dependency.
