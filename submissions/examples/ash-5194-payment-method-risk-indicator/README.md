# Payment Method Risk Indicator

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive payment risk assessment component that displays payment methods with visual fraud risk indicators. Suitable for payment gateways, banking dashboards, fintech platforms, checkout systems, and fraud detection interfaces.

## Features

- Payment method cards
- Low, Medium, and High risk indicators
- Fraud summary metrics
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="method high">

    <div class="info">

        <h3>Virtual Card</h3>

    </div>

    <span class="badge">
        High Risk
    </span>

</div>
```
