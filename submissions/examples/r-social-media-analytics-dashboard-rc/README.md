# Social Media Analytics Dashboard Showcase

## What does this do?

Provides an animated social media analytics dashboard with follower tracking, engagement monitoring, audience growth metrics, and content-performance insights.

## How is it used?

Open `demo.html` directly in a browser and explore the social-media analytics interface.

## Why is it useful?

It demonstrates practical social-media analytics UI patterns and combines multiple animation techniques into a realistic audience-growth and engagement-monitoring experience.
