# CSS Search Bar

## What does this do?
Animated expandable search bar component with icon prefix, text input, clear button, and optional filter button. Supports multiple layout modes including expand-on-click and full-width.

## How is it used?
```html
<div class="sb-wrap sb-expand"><div class="sb-bar"><span class="sb-icon">🔍</span><input class="sb-input" placeholder="Search..."><button class="sb-clear">✕</button></div></div>
```

## Why is it useful?
Provides search UI without JavaScript dependency. Smooth expand transition on focus using CSS :focus-within. Icon prefix for visual clarity. Clear button for reset. Dark theme with accent color support. Ideal for navigation headers and content filtering.
