# Rubber Band Context Menu — Glassmorphism Design Patterns

A pure CSS context menu component implementing elastic rubber band motion with frosted glass aesthetics using declarative animations and layered visual composition.

## Specifications

- **Rubber Band Animation**: Utilizes CSS `@keyframes` with `scale3d()` transformations to create an elastic stretch-and-settle interaction inspired by rubber band motion.
- **Declarative Interaction**: Implements hover-driven animation and menu item transitions entirely through native CSS selectors without JavaScript.
- **Motion Architecture**: Integrates EaseMotion CSS variables and utility classes for framework-consistent animation timing and interaction behavior.
- **Glassmorphism Styling**: Combines translucent surfaces, backdrop blur, subtle borders, and soft shadows to achieve a modern frosted glass interface.
- **Accessible Controls**: Uses semantic navigation and button elements with keyboard focus indicators for improved accessibility.
- **Responsive Composition**: Preserves spacing, visual hierarchy, and animation behavior across desktop and mobile layouts.
