# CSS Slide-Right Accordion for Accessible Web Layouts

## Overview

A responsive accordion built using semantic HTML and CSS, designed with accessibility in mind. The component features a smooth slide-right animation while maintaining keyboard usability and responsive behavior.

---

## Features

- Pure HTML5 & CSS3
- CSS-only slide-right animation
- Semantic `<details>` and `<summary>`
- Keyboard accessible
- Visible focus styles
- Responsive layout
- CSS custom properties
- No JavaScript required
