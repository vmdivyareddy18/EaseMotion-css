# Animated Spotlight Hover Card Component

A modern CSS-only spotlight card component featuring glowing radial highlights, smooth scaling animations, shadow transitions, and interactive hover effects.

## Features

- Animated spotlight glow effect
- Smooth card lift and scaling
- Dynamic shadow transitions
- Animated icons
- Gradient call-to-action button
- Responsive design
- Pure HTML and CSS implementation
- Easy customization

## Files Included

- `demo.html` — Demonstration of spotlight cards
- `style.css` — Contains styling, animations, and responsive behavior

## Hover Effects

- Spotlight moves toward the card center
- Card scales and lifts upward
- Glow shadow appears around the card
- Icons rotate and enlarge
- Buttons receive interactive animations

## Use Cases

- Product showcase cards
- Feature sections
- Service cards
- Team member profiles
- Portfolio highlights

## Customization

You can modify:

- Spotlight colors
- Card dimensions
- Animation duration
- Border radius
- Typography
- Button styles

## Browser Support

Compatible with modern browsers:

- Chrome
- Firefox
- Edge
- Safari
