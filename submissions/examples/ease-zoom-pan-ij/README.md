# Ease Zoom &amp; Pan

A CSS animation simulating a viewport panning and zooming across content with a smooth camera-like motion.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.viewport` element.

## Custom CSS Variables

| Variable         | Default   | Description                      |
|------------------|-----------|----------------------------------|
| `--zoom-scale`   | `1.5`     | Maximum zoom multiplier          |
| `--pan-duration` | `4s`      | Duration of one zoom-pan cycle   |
| `--img-color`    | `#4ecdc4` | Base color of the content grid   |

## Interactive Controls

Open `demo.html` to use the color pickers and range sliders for live customization.
