# Liquid Fill Button Hover Effect

## Description

A button hover animation where a liquid-like fill rises from the bottom and covers the button background.

## Features

- Smooth liquid fill animation
- Pure CSS implementation
- Lightweight and reusable
- Modern call-to-action effect

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` in a browser and hover over the button.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
