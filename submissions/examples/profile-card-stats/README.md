# Profile Card with Stats Grid

A profile card featuring a gradient avatar, name, job title, description, and a 4-column stats grid showing projects, followers, articles, and rating. Includes a Follow button with an indigo hover state.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the profile card.

## Accessibility notes

Content is semantic HTML. The avatar is a decorative gradient. The Follow button is a `<button>` element.
