# Fade In File Uploader Component

A sleek, banking-inspired custom file uploader styled for the **EaseMotion CSS** library ecosystem. It offers a premium, high-security aesthetic equipped with a smooth fade-in entry animation.

## Features
- **Smooth Transition**: Employs an explicit `cubic-bezier` timing curve tailored for clean financial dashboards.
- **Pure CSS**: Implemented without external JavaScript frameworks.
- **Accessible**: Uses an off-screen accessible pattern (`visually-hidden`) keeping the input readable for screen-readers while maintaining custom styles.
- **Responsive**: Adapts gracefully across desktop and mobile screens.

## File Map
- `demo.html` - Live proof-of-concept viewport page.
- `style.css` - Component-specific styles and CSS keyframes.

## Usage
Simply append the dynamic structural class helper alongside your container block:
```html
<div class="uploader-container ease-fade-in-uploader">
  <!-- Content -->
</div>
```
