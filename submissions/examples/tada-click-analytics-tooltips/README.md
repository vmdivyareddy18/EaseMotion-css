# Pure CSS Tada-Click Dashboard Analytics Tooltip

A declarative, high-performance CSS tooltip engineered precisely to fit rich data-dense environments like control consoles and enterprise analytics layouts.

## Features
- **Zero JS Dependency Engine:** Leverages CSS custom properties combined natively with the `:active` structural token indicator.
- **Data Console Aesthetic:** Styled with dark mode panels, high contrasting status metrics, and structural double borders.
- **Impact-Click Feedback Loop:** Shakes on multiple rotation planes when pressed to emulate confirmation signaling.
- **Highly Responsive Layouts:** Safely scales down dynamically within standard CSS flex and grid matrices.

## Exposed Variables Properties

| Variable Parameter | Utility Profile | Baseline Value |
| :--- | :--- | :--- |
| `--analytics-accent` | Core informational theme branding color | `#3b82f6` |
| `--tooltip-bg` | Interior surface coloring block metric | `#1f2937` |
| `--tooltip-fade-speed` | Visibility fade speed entry duration | `0.2s` |
| `--tooltip-tada-speed` | Total loop time duration of the click wiggle | `0.55s` |
