# Magnetic Pull Popover - Responsive Dashboard Layouts

## What it does

Magnetic Pull Popover is a pure CSS animated component designed for responsive dashboard interfaces.

It creates a smooth magnetic attraction effect when users interact with the trigger element.

## Features

- CSS-only animation
- Responsive dashboard-friendly design
- Keyboard accessible focus interaction
- Smooth transform transitions
- Custom CSS variables for animation control
- No JavaScript required

## Usage

```html
<div class="magnetic-popover">

  <button class="magnetic-trigger">
    View Stats
  </button>

  <div class="popover-content">
    Dashboard information
  </div>
