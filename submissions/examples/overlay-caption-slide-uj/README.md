# Overlay Caption Slide-Up

## What does this do?

Hovering an image reveals a background-blurred overlay panel with slide-in details.

## How is it used?

Apply the `overlay-slide-uj` class to the wrapper card containing the image and the caption-panel:

```html
<div class="overlay-slide-uj">
  <img src="https://via.placeholder.com/400" alt="Nature">
  <div class="caption-panel-uj">
    <h5>Deep Woods</h5>
    <p>Scenic forestry capture.</p>
  </div>
</div>
```

The caption panel will automatically slide up on hover.

## Why is this useful?

It provides a clean, premium visual design for gallery grids, portfolio cards, and team listings. By utilizing hardware-accelerated transitions and backdrop blurs, it delivers high performance and smooth animation while keeping classes semantic and human-readable.
