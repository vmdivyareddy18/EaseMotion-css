# Subscription Seat Utilization Table

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive subscription seat utilization dashboard that displays seat allocation, usage, and utilization percentages in a modern table. Suitable for SaaS admin panels, subscription management systems, and enterprise licensing dashboards.

## Features

- Responsive utilization table
- Animated utilization bars
- Color-coded status badges
- Seat summary cards
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<tr>

    <td>Engineering</td>

    <td>80</td>

    <td>74</td>

    <td>

        <div class="bar">

            <div class="fill engineering"></div>

        </div>

    </td>

</tr>
```
