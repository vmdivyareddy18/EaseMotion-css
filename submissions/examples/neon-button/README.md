# Neon Button

A responsive **Neon Button** component built with pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- E-Commerce inspired design
- Neon glow hover animation
- Responsive layout
- Accessible button element
- Uses EaseMotion CSS variables and `ease-*` animation classes

## Folder Structure

```text
neon-button-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Technologies

- HTML5
- CSS3

## Animation

- `ease-fade-in`
- Neon glow hover effect
- Responsive and lightweight
