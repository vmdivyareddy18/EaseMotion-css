# Ball Bounce — Bouncing Ball Squash/Stretch

CSS bouncing ball with squash-and-stretch deformation. The `bounce-ball` keyframe scales Y at impact points to simulate elasticity. Shadow pulses in sync.

## Files

- `demo.html` — ball and shadow elements
- `style.css` — bounce trajectory, squash/stretch, shadow pulse

## Usage

Open `demo.html` to see the ball bounce with realistic deformation.
