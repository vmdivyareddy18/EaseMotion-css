Platform Features

Built for Precision

Every tool you need for clinical-grade neurofeedback training

◉
Adaptive Thresholds

Reward criteria auto-adjust each session based on your personal baseline trajectory — no manual recalibration needed between appointments.

⬡
Multi-Site Coherence

Simultaneous training of inter-electrode coherence, targeting default mode network and task-positive network balance.

∿
Artifact Intelligence

Real-time ICA separates eye blinks, muscle artifact, and ECG from genuine EEG so rewards never fire on noise.

◈
Longitudinal Reporting

Session-over-session trend charts with protocol comparison, effect-size statistics, and exportable clinician summaries.

▤
Protocol Library

Over 60 evidence-based neurofeedback protocols mapped to clinical presentations, customisable per client.

◎
Reward Engine

Audio, visual, and gamified feedback modes with latency under 40 ms from signal to reward — within operant conditioning windows.
