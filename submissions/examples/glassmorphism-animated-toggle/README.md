# Glassmorphism Animated Toggle

A modern, fluid toggle switch built completely with CSS. It utilizes backdrop-filters to achieve a glassmorphic aesthetic that blends seamlessly into dark or colorful backgrounds.

## Features
- Pure CSS implementation
- Custom cubic-bezier transitions for a natural, spring-like feel
- Accessible label-wrapping structure
- Glassmorphism design trend with inset shadowing and gradient lighting

## Usage
Simply drop the HTML and CSS into your project. The toggle uses standard `input[type="checkbox"]` mechanics, meaning it can be easily hooked up to forms or JavaScript frameworks.
