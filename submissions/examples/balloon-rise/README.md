# Balloon Rise & Sway

A gentle, slow balloon rise animation combined with subtle horizontal swaying and rotation oscillations. Ideal for party, celebration, or achievement unlock UI feedback.

## EaseMotion CSS Classes Showcased

This feature submits a self-contained prototype demonstrating how standard translation and rotation offsets can be combined to simulate lighter-than-air float dynamics.

### Classes:
- `.ease-balloon-container`: Context parent wrapper for bounds control.
- `.ease-balloon-rise`: Applies continuous upward rising (`translateY`) and swaying (`rotate`) oscillations over a slow looping duration.

### Usage in HTML:
```html
<div class="ease-balloon-container">
  <div class="ease-balloon-rise">
    🎈
  </div>
</div>
```
