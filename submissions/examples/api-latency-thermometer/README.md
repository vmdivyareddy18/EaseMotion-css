# API Latency Thermometer

API Latency Thermometer is a responsive health panel for comparing service latency levels. It highlights cool, warm, and hot API routes so teams can quickly spot which endpoint needs attention.

## Features

- P95 latency summary
- Cool, warm, and hot service states
- Compact reading cards
- Responsive single-column layout

## Files

- `demo.html` contains the latency thermometer markup.
- `style.css` contains the layout, state colors, and responsive rules.
