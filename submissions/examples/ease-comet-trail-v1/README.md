# Ease Comet Trail

## Overview

Ease Comet Trail is a reusable animation that simulates a glowing comet traveling across the screen while leaving behind a fading light trail. Built entirely with HTML and CSS.

---

## Features

- ☄️ Animated comet flight
- ✨ Multi-layer glowing trail
- 🌠 Smooth diagonal motion
- 🎨 Lightweight implementation
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

- Animated comet
- Layered fading trail
- Infinite motion

---

## Usage

```html
<div class="comet">
    ...
</div>
```

---

## Customization

- Comet color
- Trail length
- Animation speed
- Glow intensity
- Flight direction

---

## Browser Support

Compatible with all modern browsers supporting CSS transforms and animations.

---

## Why This Fits EaseMotion CSS

Useful for hero sections, landing pages, loading screens, space-themed websites, and creative UI effects. It provides a reusable visual animation with no JavaScript dependency.
