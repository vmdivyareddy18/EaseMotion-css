# Slider — Perspective Variant Component Block (`ease-perspective-slider`)

An advanced interactive layout container introducing high-fidelity spatial 
depth visual fields to metric showcases using CSS perspective pipelines.

---

## 📋 Overview

The `perspective-slider` component brings a smooth 3D display to SaaS metrics cards. 

It highlights data layers by tilting side elements while locking the center display grid flat.

This creates a high-impact presentation layer built for modern SaaS metric centers, 
optimization layouts, or digital health score displays.

---

## ✨ Features

* **Advanced 3D Perspective Grid Mapping**
  Utilizes native CSS `perspective` combined with localized `rotateY` matrix 
  offsets to generate realistic viewing shifts.

* **Fluid Transform Physics**
  Card movements transition into custom scale routines upon interactive user hovering cycles.

* **JavaScript-Free Component Architecture**
  Orchestrates all transitions on hardware GPU channels without rendering runtime script tags.

* **Accessible Focus Pipelines**
  Fully supports screen readers and sequential keyboard routing paths across all interactive metric cards.
