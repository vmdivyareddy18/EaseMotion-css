# Border Image Gradient Animation

Animated gradient borders using the CSS border-image property with conic, linear, and radial gradients. Each card demonstrates a different border-image animation technique.

## Files

- `demo.html` - Three cards with different border-image gradient styles
- `style.css` - @property animation, repeating-gradient, border-image slice

## Usage

Open `demo.html` to see animated gradient borders in action.
