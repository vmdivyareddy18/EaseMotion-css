# 🚀 Portfolio Home Speed Dial Pro UI

An advanced Speed Dial (Expandable FAB Menu) component with animation and labeled actions.

---

## ✨ Features

- Expandable Floating Action Button (FAB)
- Smooth open/close animation
- Icon + label support
- Hover scaling effects
- Clean and modern UI
- Fully reusable component

---

## 🧩 Component Structure

- Main FAB button
- Expandable action buttons
- Label support for each action

---

## 📦 Usage

```html
<div class="fab-container">
  <button class="fab main">+</button>
  <div class="fab-options">
    <button>🏠</button>
    <button>📧</button>
  </div>
</div>
