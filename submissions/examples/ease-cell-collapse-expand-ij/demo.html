Cell Collapse / Expand

Click a cell header to collapse or expand its content

Introduction ▾

This is the introductory content. Click the header above to collapse or expand this section. The animation uses max-height transitions for a smooth reveal.

Features ▾

Collapse and expand individual cells with smooth CSS transitions. Each cell operates independently, allowing multiple cells to be open at once.

Usage ▾

Add the collapsed class to any .cell-item to hide its body content. Remove the class to reveal it with a smooth expand animation.

Customization ▾

Use CSS variables to customize the background color, padding, collapse duration, and expand delay to match your design system.
