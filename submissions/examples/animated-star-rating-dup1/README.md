# Animated Star Rating Component

A clean, modern, and fully responsive **Star Rating Component** with smooth hover effects, bouncy click animations, and multiple rating variations. This example demonstrates how to combine **EaseMotion CSS** for interactive hover scaling, click feedback, and smooth state transitions to create an engaging rating system.

## 🚀 Features

- **Smooth Hover Scaling**: Stars scale up smoothly when hovered using `ease-hover-grow` for tactile feedback.
- **Bouncy Click Animation**: When a star is clicked, it bounces in using `ease-bounce-in` for satisfying confirmation.
- **Hover Preview**: Hovering over a star previews the rating by highlighting all stars up to that position.
- **Half-Star Display**: Support for displaying half-star ratings (e.g., 4.5/5) using CSS clipping.
- **Emoji Rating Variation**: An alternative emoji-based rating system with grayscale-to-color transitions.
- **Dynamic Rating Text**: The rating text updates dynamically after selection with a smooth fade-in animation.
- **Display-Only Mode**: Pre-filled ratings that are non-interactive for showing existing reviews.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing.
