# Release Risk Matrix

An intermediate EaseMotion CSS example for release review dashboards. Risk cells pop into a two-by-two matrix to compare impact and confidence before launch.

## Files

- `demo.html` - Semantic release risk matrix markup.
- `style.css` - Responsive matrix layout, axis labels, and staggered cell animation.

## Usage

Open `demo.html` in a browser to preview the animated release risk matrix.
