Brightness & Contrast Utilities

CSS filter controls proposed for core/utilities.css.

These utility classes apply the filter: brightness() and filter: contrast() properties. They are extremely useful for adjusting image assets without needing an image editor, creating dynamic hover states, or building dark mode adjustments.

Brightness

.filter-brightness-50

.filter-brightness-100 (Default)

.filter-brightness-150

Contrast

.filter-contrast-50

.filter-contrast-100 (Default)

.filter-contrast-150
