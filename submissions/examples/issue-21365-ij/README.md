# Butterfly Caught in Net

1. What does this do? A CSS-only butterfly fluttering inside a net enclosure. The wings scaleX while the net shakes.

2. How is it used? Place .butterfly inside .scene. The butterfly body uses box-shadow for wing shapes; the net is a dashed border circle.

3. Why is it useful? Shows confined-animal motion with container shake plus internal rapid flutter using pure CSS.
