# Wheels of Flavor

A responsive landing page for Wheels of Flavor, built using EaseMotion CSS.

## Usage

Open demo.html in any modern browser. No build step required.

## Features

- Uses EaseMotion CSS framework classes (animations, utilities, buttons)
- Fully responsive layout
- Zero JavaScript dependencies
- Dark theme optimized

## Files

- demo.html — HTML structure with EaseMotion classes
- style.css — Custom color scheme and layout
- README.md — Documentation
