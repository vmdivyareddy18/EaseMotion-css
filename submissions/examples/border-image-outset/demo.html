Border Image Outset

Extending the border image beyond the element's border box

Outset Value Demonstrations

No Outset (0)
border-image-outset: 0

Border image stays within the border box

Small (10px)
border-image-outset: 10px

Image extends 10px beyond each edge

Medium (25px)
border-image-outset: 25px

Image extends 25px beyond the border box

Large (50px)
border-image-outset: 50px

Image extends 50px — overlaps surrounding content

Asymmetric Outsets

Horizontal Only
border-image-outset: 0 30px

Extends left and right edges only

Vertical Only
border-image-outset: 30px 0

Extends top and bottom edges only

Four Values
border-image-outset: 10px 20px 30px 40px

Different outset per edge (top, right, bottom, left)
