Class Count Badge Generator

A script that counts CSS classes by category and generates shields.io badge URLs for the README.

How it works

  • Parses the core CSS files in the core/ directory
  • Counts classes by category: animation, utility, component
  • Generates shields.io badge URLs with the current counts
  • Outputs ready‑to‑paste Markdown for the README

How to run

# Install dependencies
npm install fs path

# Run the script
node count-classes.js

The script reads core/ and prints three shields.io badge URLs (animation, utility, component).

Built for EaseMotion CSS — keep the README badges accurate automatically.
