Animation Fill Mode Utilities

None Mode

fill-mode-none

The default setting. Style changes from keyframes do not persist. Elements instantly snap back to their default idle styles upon completion.

N

Forwards Mode

fill-mode-forwards

Retains style values defined in the last keyframe (100% or to). Useful for entrance animations where items must remain visible.

F

Backwards Mode

fill-mode-backwards

Applies style values from the first keyframe (0% or from) during the delay phase before the animation actively starts playing.

B

Both Mode

fill-mode-both

Combines forwards and backwards parameters. Applies the 0% style during delay, and holds the 100% style upon completion.
