# Animated Feature Comparison Table

A responsive comparison table component with animated rows, plan comparison indicators, and hover interactions.

## Features

- Animated row reveal effects
- Feature comparison layout
- Hover highlighting
- Responsive mobile design
- SaaS-style comparison interface

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

- Plan names
- Feature list
- Colors
- Animation timing
- Responsive breakpoints

## Use Cases

- SaaS pricing comparisons
- Product feature matrices
- Subscription plan comparisons
- Service comparison pages
- Marketing websites
