# Avatar Status

## What does this do?
Avatar Status component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"as-primary\">Avatar Status Primary</div>
```

## Why is it useful?
Provides a reusable avatar status UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
