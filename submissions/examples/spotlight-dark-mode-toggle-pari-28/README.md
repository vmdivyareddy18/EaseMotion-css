# Spotlight Dark Mode Toggle

A responsive **Cyberpunk-inspired Spotlight Dark Mode Toggle** built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Spotlight thumb animation
- Responsive layout
- Keyboard accessible checkbox
- CSS custom properties
- Supports `prefers-reduced-motion`
- No JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` directly in any modern browser.

## Customization

Adjust the CSS variables in `:root` to change colors, glow intensity, sizing, and animation speed.
