# Animated Onboarding Tour / Product Guide

A sleek, modern, and fully responsive **Onboarding Tour Component** perfect for guiding new users through key features of your application. This example demonstrates how to combine **EaseMotion CSS** for smooth spotlight transitions, staggered tooltip entrances, and interactive navigation to create an engaging, educational user experience.

## 🚀 Features

- **Smooth Spotlight Transition**: The overlay uses CSS `clip-path` to create a spotlight effect that smoothly transitions between different UI elements.
- **Slide-In Tooltip**: The tooltip card uses `ease-slide-in-up` to elegantly appear with each step.
- **Bouncy Progress Dots**: Progress indicators use `ease-bounce-in` with staggered delays for a playful entrance.
- **Interactive Navigation**: Previous/Next buttons with hover effects using `ease-hover-grow`.
- **Step Counter**: Clear indication of current progress (e.g., "Step 2 of 4").
- **Skip Tour Option**: Users can skip the tour at any time.
- **Dynamic Positioning**: Tooltip automatically positions itself based on the highlighted element.
- **Pulsing Start Button**: The "Start Tour" button uses `ease-pulse` to draw attention.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted tooltip sizing.
