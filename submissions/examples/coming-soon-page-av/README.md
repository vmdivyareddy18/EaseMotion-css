# Coming Soon Page

## What does this do?

Creates a modern "Coming Soon" landing page featuring a glassmorphism countdown section, email subscription form, and responsive layout.

---

## How is it used?

Open `demo.html` directly in any browser.

```html
<div class="launch-info">
    <div class="card">
        <h2>48</h2>
        <span>Days</span>
    </div>
</div>
```

---

## Why is it useful?

Coming Soon pages are widely used before product launches, startups, SaaS platforms, portfolios, and events. This standalone example demonstrates a polished landing page using only HTML and CSS while aligning with EaseMotion CSS's clean, animation-first design philosophy.
