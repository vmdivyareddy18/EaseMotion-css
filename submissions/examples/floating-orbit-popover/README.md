# Floating Orbit Popover

A responsive **Cyberpunk Neon Floating Orbit Popover** built using pure CSS for the EaseMotion CSS examples library.

## Features

- Pure CSS animation
- Floating orbit motion
- Cyberpunk neon theme
- Responsive layout
- Keyboard accessible (`:focus-within`)
- No JavaScript required
- Customizable with CSS variables

## Usage

```html
<div class="popover-wrapper">
    <button class="orbit-btn">Hover Me</button>

    <div class="popover">
        Popover content
    </div>
</div>
```

Import `style.css` into your project.

## Custom Properties

- `--ease-duration`
- `--ease-easing`
- `--ease-scale`
- `--ease-primary`
- `--ease-secondary`

## Why EaseMotion CSS?

This component demonstrates reusable, animation-first UI with smooth floating motion, customizable CSS variables, accessibility support, and zero JavaScript.
