# Elastic Bounce Creative Portfolio Modal

A responsive Creative Portfolio modal featuring a smooth Elastic Bounce entrance animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Elastic Bounce animation
- Creative portfolio layout
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--scale
--primary
--secondary
--bg
--surface
--text
--muted
--radius
```
