# Background Fill Swipe

## What does this do?
Sweeps a solid background color from left to right inside an outline button when hovered.

## How is it used?
Add the class to your button:

```html
<button class="fill-swipe-btn">Hover Swipe</button>
```

## Why is it useful?
It provides a high-contrast transition that changes text and background colors concurrently, producing a polished, solid fill state.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
