# Firefly Coming Soon Page

## 1. What does this do?
Creates a "Coming Soon" page with floating firefly particles that glow and animate around the page in dark mode. The fireflies appear and disappear with smooth animations, creating an elegant and magical atmosphere.

## 2. How is it used?
Apply `.firefly` class to particle elements and `.coming-soon-content` to your main content:

```html
<!-- Firefly particles -->
<div class="firefly firefly-1"></div>
<div class="firefly firefly-2"></div>

<!-- Content -->
<div class="coming-soon-content">
  <h1 class="coming-soon-title">Coming Soon</h1>
  <p class="coming-soon-subtitle">Something magical is on the way...</p>
</div>
