# Animated FAQ Accordion Cards

A responsive FAQ accordion component built with semantic HTML and CSS.

## Features

- Accordion-style FAQ cards
- Smooth hover interactions
- Responsive design
- Accessible `<details>` and `<summary>` elements
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more FAQ items
- Change colors and spacing
- Modify animation timing
- Customize typography
