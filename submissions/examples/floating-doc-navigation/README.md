# Floating Dock Navigation

## Description

A macOS-inspired floating navigation dock with animated icon magnification.

## Features

- Smooth hover magnification
- Floating glassmorphism dock
- Lightweight implementation
- Responsive design

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and hover over the navigation icons.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
