How It Works

CSS animation-timeline: scroll(root) binds the animation progress to the document scroll position. The animation-range property maps the scroll range to the animation's keyframes.

This eliminates JavaScript scroll listeners entirely — the browser handles all frame interpolation natively on the compositor thread.
