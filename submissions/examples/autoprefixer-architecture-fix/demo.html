CSS Architecture Fix: Autoprefixer Pipeline

This submission provides the architectural requirement to ban manual vendor prefixing, removing massive codebase bloat across 1,002 files.

Incorrect Architecture

-webkit-transform: scale(1.5);
-moz-transform: scale(1.5);
-ms-transform: scale(1.5);
-o-transform: scale(1.5);
transform: scale(1.5);

Contributors are manually typing dozens of prefixes, causing severe maintenance nightmares and bloating the CSS files.

Correct Architecture

transform: scale(1.5);

Contributors write standard CSS. The automated PostCSS Autoprefixer pipeline dynamically injects exact prefixes at build time based on browser support.
