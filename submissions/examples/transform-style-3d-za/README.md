# Transform Style Preserve 3D Card

A CSS demo of transform-style: preserve-3d creating genuine 3D card flips with perspective and backface-visibility. Hover to flip the card.

## Files

- `demo.html` - A flip card with front and back faces
- `style.css` - transform-style: preserve-3d, backface-visibility, perspective

## Usage

Open `demo.html` and hover over the card to see the 3D flip effect.
