# Stacked Cards

Interactive card stack that fans out on click or hover. Built with CSS transforms and JavaScript interactivity.

## Features

- Click to fan the stack, click individual cards to lift them
- Hover effect with elevated shadow
- Deal animation that re-stacks cards with staggered entrance
- Live card count display

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--sc-duration` | `0.35s` | Transition speed |
| `--sc-card-bg` | `#ffffff` | Card background |
| `--sc-card-color` | `#1e293b` | Card text color |
| `--sc-offset` | `8px` | Vertical offset between stacked cards |
| `--sc-radius` | `16px` | Card border radius |
