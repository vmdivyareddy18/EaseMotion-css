# Ferris Wheel — Rotating Ferris Wheel

CSS ferris wheel with rotating frame and counter-rotating cabins to keep them upright. Spokes and center hub included.

## Files

- `demo.html` — wheel frame, spokes, cabins
- `style.css` — wheel spin, cabin counter-spin keyframes

## Usage

Open `demo.html` to see the ferris wheel rotate.
