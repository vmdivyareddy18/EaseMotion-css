# Pulse Status Dot

A system status indicator with pulsing dot animations for live, warning, and offline states.

## Features

- Animated pulse ring effect on live and warning dots
- Color-coded states: green (live), yellow (warning), red (down)
- Status label with matching color backgrounds
- Row hover highlight effect
- Pure CSS keyframe animations

## Usage

Include `style.css` and add `.status-dot` with `.live`, `.warning`, or `.down` class. The dot pulses with a `box-shadow` ring animation on live and warning states.
