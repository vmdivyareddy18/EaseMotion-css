# Animated Search Bar Collection Component

## What does this do?

Provides a collection of modern animated search bar designs showcasing different interaction patterns and CSS animation techniques.

## How is it used?

```html
<div class="search-box expand">
    <input type="text" placeholder="Search...">
</div>
```

## Why is it useful?

Search bars are one of the most common UI elements in modern applications. This component demonstrates multiple search styles including expandable inputs, floating labels, glassmorphism effects, and animated focus states using pure HTML and CSS.
