# Animation: Settings Panel UI

A modern settings interface inspired by popular SaaS platforms. It combines navigation menus, interactive setting cards, and animated toggle switches into a clean dashboard experience.

## Features

- Pure HTML & CSS
- Interactive toggle switches
- Sidebar navigation
- Responsive layout
- Hover elevation effects
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="setting-card">

    <h3>Email Notifications</h3>

    <label class="switch">
        <input type="checkbox">
        <span class="slider"></span>
    </label>

</div>
```

## Why is it useful?

Settings pages are one of the most common interfaces in web applications. This example demonstrates how motion and interaction can improve usability while maintaining a clean and professional design.
