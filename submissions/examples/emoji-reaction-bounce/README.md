# Emoji Reaction Bounce

A social-media style emoji reaction component inspired by Slack and Discord.

## Features

- Bounce animation using scale(0) → scale(1.3) → scale(1)
- Reaction counter pulse animation
- Pure CSS implementation
- Common chat and social UI pattern

## Usage

```html
<div class="ease-reaction">
    <span class="ease-reaction-emoji">👍</span>
    <span class="ease-reaction-count">12</span>
</div>
