# Legal Tech SaaS Landing Page

## Overview

A complete ContractPodAi-inspired legal technology SaaS landing page built entirely with EaseMotion CSS utilities and interactions.

## Features

- Hero section with contract workflow mockup
- Feature grid for contract lifecycle management
- Contract cycle reduction statistics
- Integration logos section
- Pricing tiers
- Customer legal team logos
- Responsive footer
- Entrance animations and hover interactions

## Tech Stack

- HTML5
- CSS3
- EaseMotion CSS

## Usage

Open `demo.html` directly in a browser. No build process or dependencies are required.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

- `ease-fade-in`
