# Snack Subscription Box

A responsive e-commerce landing page built for EaseMotion CSS.

## Features

- Hero Section
- Countries Featured
- Past Subscription Boxes
- Pricing Plans
- Customer Reviews
- Subscribe CTA
- FAQ Accordion
- Footer

## Files

- demo.html
- style.css
- README.md

## Animations

Uses EaseMotion CSS animation classes such as:

- ease-fade-up
- ease-hover-grow
- ease-float
