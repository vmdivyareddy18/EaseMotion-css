Class Reference

This demo uses the examples-* prefix so it can run standalone. In the framework itself, the equivalent classes follow the ease-* convention shown below.

Demo class (this file) Framework class What it does
examples-fade ease-fade Fade + lift in on load
examples-pulse-hover ease-pulse-hover Outward pulse on hover
examples-bounce ease-bounce Vertical bounce motion
examples-slide-left ease-slide-left Slide in from the left
examples-duration-* duration-* Sets animation length (300/400/500/1000ms shown here)
examples-delay-* delay-* Sets animation start delay (200ms shown here)
examples-infinite ease-infinite Repeats the animation indefinitely
