Optimization Strategies

  1. Extract SVGs — Move 7 inline SVG data URIs (~5.5KB) to a separate icons file
  2. Consolidate light mode — Replace @media block with [data-theme="light"] selectors
  3. Remove duplicate transitions — Set transition on parent, inherit on children
  4. Merge orb code — Use CSS vars for ::before/::after differences
  5. Combine breakpoints — Fold 480px rules into the 768px block
  6. Use shorthand — margin-inline, remove !important, shorter selectors
