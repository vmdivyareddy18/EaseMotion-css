# Carousel Gallery Thumbs

An image gallery carousel with thumbnail navigation. Click a thumbnail to transition the main image with a crossfade effect. Previous/next arrow buttons provide sequential navigation.

## Features

- Main image crossfade transition
- Thumbnail navigation with active border indicator
- Previous / next arrow controls
- Accessible ARIA roles and labels
- Customisable via CSS custom properties

## Custom Properties

| Property                   | Default  | Description             |
| -------------------------- | -------- | ----------------------- |
| `--gallery-duration`       | `0.3s`   | Transition duration     |
| `--gallery-main-height`    | `450px`  | Main image area height  |
| `--gallery-thumb-size`     | `100px`  | Thumbnail width         |
| `--gallery-bg`             | `#f4f4f9`| Page background         |
| `--gallery-active-border`  | `#4a90d9`| Active thumbnail border |
