# Clock Dial Loader

A CSS-only clock dial loader with a rotating hand and premium dashboard styling.

## What it is

This component is a circular loader inspired by a clock dial. It features 12 hour markers, a smoothly rotating hand, and a glowing center hub for a polished, modern loading experience.

## Features

- Pure HTML + CSS implementation
- Circular clock-like dial with 12 markers
- Infinite smooth hand rotation
- Center hub with soft glow styling
- Premium dark theme appearance
- Responsive sizing for small and large screens
- No JavaScript or external libraries

## Usage

1. Copy `demo.html` and `style.css` into your submission folder.
2. Open `demo.html` in a browser.
3. Use the loader inside dashboards, loading states, or premium UI previews.

```html
<link rel="stylesheet" href="style.css" />
<div class="loader-page">
  <section class="loader-card ease-fade-in">
    <!-- loader markup here -->
  </section>
</div>
```

## Why it fits EaseMotion CSS

This submission is a great fit for EaseMotion CSS because it emphasizes motion-first styling, self-contained markup, and a clean dark aesthetic. The loader uses smooth CSS animation and polished detail, matching the framework's focus on premium, animation-friendly UI patterns.
