# CSS Micro Shake Modal

Pure CSS animated modal utilizing a subtle Micro Shake interaction for Product Catalog layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS Custom Properties
- Product showcase styling
- Micro shake animation
- Smooth 60fps interactions

## Variables

```css
--duration
--primary
--secondary
--ease
```
