# CSS Zoom-Out Accordion (Neumorphism)

A simple accordion component created using pure HTML and CSS with a soft neumorphic design and zoom-out opening animation.

## Features

- Pure HTML & CSS
- No JavaScript
- Neumorphic UI
- Zoom-Out Animation
- Responsive Design
- Beginner Friendly

## Folder Structure

```text
css-zoom-out-accordion-neumorphism/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Browser Support

Works in all modern browsers.
