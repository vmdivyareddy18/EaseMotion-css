# Character Counter – Counts down with color change near limit

> A production-ready character counter component that displays real-time character count with color changes and visual warnings as the limit approaches. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Real-time counting** – Updates as you type
- ✅ **Color-coded warnings** – Changes color near limit
- ✅ **Progress bar** – Visual representation of usage
- ✅ **Character limit** – Configurable limit
- ✅ **Word count** – Shows word count alongside
- ✅ **Visual states** – Good, Warning, Danger, Exceeded
- ✅ **Responsive design** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Highly customizable** – Colors, thresholds, messages, callbacks
