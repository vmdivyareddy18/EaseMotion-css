# Content Moderation Decision Queue

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive moderation queue component designed for reviewing reported content with clear priority labels and action buttons.

## Features

- Moderation queue cards
- High, medium and low risk labels
- Approve and reject actions
- Responsive layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="card">

    <div class="severity high">
        High Risk
    </div>

    <div class="content">
        ...
    </div>

</div>
```

## Main Classes

```css
.queue
.card
.severity
.high
.medium
.low
.content
.actions
.approve
.reject
```
