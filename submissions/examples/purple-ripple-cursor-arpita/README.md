# Purple Ripple Cursor Showcase
An interactive landing page showcasing a glowing purple ripple cursor effect that follows mouse movement with smooth animations and automatic cleanup.

---

## Features
- Interactive purple ripple cursor trail
- Smooth ripple expansion and fade animation
- Glowing water droplet effect
- Mouse tracking across the viewport
- Glassmorphism-inspired interface
- Responsive landing page layout
- Lightweight and easy to customize

## How to Use

1. Open `demo.html` in any modern browser.
2. Move the mouse around the page.
3. Purple glowing ripple droplets will appear and fade smoothly while preserving normal page interactions.

---

## HTML Usage

```html
<div class="cursor-container"></div>

<section class="hero">
    <h1>Purple Ripple Cursor Showcase</h1>
</section>
```

---

## Customization

Modify the ripple appearance in `style.css`.

```css
.ripple{
    width:16px;
    height:16px;
    background:radial-gradient(circle,#d76cff,#8b5cf6,transparent 70%);
}
```

You can customize:

- Ripple size
- Glow intensity
- Colors
- Animation duration
- Shadow effects
