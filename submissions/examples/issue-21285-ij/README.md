# Roller Coaster — Car on Track

CSS roller coaster car riding along a curved track. The `ride-track` keyframe moves the car horizontally while varying vertical position to follow the track arc.

## Files

- `demo.html` — track, rail, car elements
- `style.css` — track shape, car motion, wheel dots

## Usage

Open `demo.html` to see the car traverse the coaster track.
