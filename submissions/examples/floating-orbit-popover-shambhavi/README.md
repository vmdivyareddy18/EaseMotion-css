# Floating Orbit Popover

A modern Floating Orbit Popover built using pure HTML and CSS with a Glassmorphism-inspired interface.

## Features

- Pure HTML & CSS
- Floating Orbit animation
- Glassmorphism design
- Responsive layout
- Keyboard accessible (`:focus-within`)
- No JavaScript required
- Easy to customize using CSS variables

## Files

- demo.html
- style.css
- README.md

## Customization

```css
--orbit-duration
--orbit-scale
--orbit-easing
--glass-bg
--glass-border
```

## Browser Support

Works in all modern browsers supporting:

- CSS Variables
- CSS Animations
- Backdrop Filter

## Preview

Hover or focus on the circular button to reveal the animated floating popover.
