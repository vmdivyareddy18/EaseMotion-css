# Albatross

A pure CSS and HTML wandering albatross dynamic soaring over the southern ocean, banking on stiff outstretched wings without a single flap.

## What it shows

- Dynamic soaring built from stacked animations: an outer element banks and climbs while the bird itself drifts, so the arc reads as riding the wind gradient
- Wings that never flap: they only flex with a rotate and skewY, the way a real albatross locks its wings and soars
- Enormous wingspan cut from clip-path vanes with dark trailing edges from a two stop gradient
- Rolling swell, drifting cloud banks and a low sun for the open ocean
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
