# Weather Lightning Icon

1. What does this do? Renders a lightning bolt icon inside a weather app icon container that flashes with irregular opacity and brightness timing.

2. How is it used? Apply `.weather-icon` and `.lightning-icon` to a container, with `.lightning-bolt` and `.bolt-shard` elements inside.

3. Why is it useful? It delivers a realistic storm-icon flash effect using pure CSS, suitable for weather forecast widgets and app launchers without JavaScript.
