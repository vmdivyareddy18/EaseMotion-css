# Interactive Layout Playground

## 1. What does this do?
An interactive, live preview utility playground built to showcase Flexbox and Grid class helpers inside EaseMotion CSS.

## 2. How is it used?
Open `demo.html` in your web browser to explore classes interactively, or apply layout utility classes directly to your container elements:

```html
<div class="ease-flex ease-flex-row ease-justify-center ease-items-center ease-gap-4">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

## 3. Why is it useful?
It provides an immediate visual playground for developer exploration, aligning with EaseMotion CSS's animation-first, human-readable, and zero-dependency philosophy.
