# Animated Skeleton Card

A collection of card skeleton loading shimmer components with multiple layout variants. Built with pure CSS animations.

## Created By

[@saurabhhhcodes](https://github.com/saurabhhhcodes)

## Variants

- **Text Block** — Paragraph lines with shimmer
- **Avatar + Text** — Circular avatar placeholder alongside text lines
- **Card Image** — Image placeholder with title and description
- **Full Page** — Complex layout skeleton

## Usage

1. Include the Inter font in your `<head>`:
   ```html
   <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
   ```
2. Link `style.css` in your HTML.
3. Use any skeleton variant:
   ```html
   <div class="skeleton-card skeleton-card--text-block">...</div>
   ```
