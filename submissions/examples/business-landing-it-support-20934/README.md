# Business Landing: IT Support Company (`business-landing-it-support-20934`)

This proposal provides a complete, copy-paste ready landing page template designed for a Managed IT Services Provider (MSP) or IT Support Company, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

An IT services website must project reliability, technical competence, and approachability. This template achieves a "Trustworthy Tech" aesthetic using a clean Corporate Blue and Slate color palette. It pairs a geometric, tech-forward display font (`Space Grotesk`) with a highly legible, standard corporate sans-serif (`Roboto`) for body copy.

The layout includes:
- A clean, authoritative navigation bar with prominent contact information.
- A Hero section featuring a CSS grid background, glowing status indicators, and a custom CSS-built "Server Rack" data visualization graphic demonstrating uptime and security.
- A high-impact "Metrics" banner focused on response times and guarantees.
- A comprehensive "Core Competencies" services grid with subtle, professional hover states.
- A transparent, multi-tiered "Predictable IT Costs" pricing table highlighting a recommended professional plan.
- A customer reviews section.
- A "Free Assessment" lead capture form section.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, modified locally via CSS to use precise, smooth `cubic-bezier(0.25, 1, 0.5, 1)` timings tailored for a reliable, professional brand:
- `.ease-float` & `.ease-float-delayed` (Used on the server rack and status cards to create a smooth, continuous levitation effect).
- `.ease-slide-right-stagger` (Configured for a smooth, confident horizontal entrance in the hero).
- `.ease-hover-lift` (Provides a professional, constrained elevation effect on service and pricing cards).
- *Custom Animations*: Includes a live `ease-blink` utility on the server graphic to simulate network activity, and an `animate-pulse` dot in the hero badge to simulate 24/7 active monitoring.

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the complete aesthetic overhaul, from the trustworthy color palette and typography to the pure CSS illustrations (the server rack graphic). The standard `ease-*` interaction classes are attached directly to HTML elements to provide smooth, reliable animations. No JavaScript is required for this template.
