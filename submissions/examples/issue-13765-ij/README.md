# ease-upload-bounce

**Closes #13765**

A continuously bouncing upload arrow that emphasises upward direction.

## Acceptance Criteria

- [ ] `.upload-arrow` bounces upward using `translateY` keyframes
- [ ] Animation loops continuously by default
- [ ] Toggle button pauses/resumes the animation
- [ ] Dark theme with minimal, centered layout

## Files

- `style.css` – bounceUp keyframes, upload arrow styling
- `demo.html` – Upload arrow with start/stop toggle
- `README.md` – Issue documentation
