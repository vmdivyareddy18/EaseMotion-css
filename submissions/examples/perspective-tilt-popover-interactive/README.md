# CSS 3D Perspective Tilt Popover

## Description

A pure CSS popover with a smooth 3D perspective tilt animation designed for Interactive Interface layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS Custom Properties
- Easy to customize

## Usage

```html
<div class="popover">
  <button class="trigger">Open Popover</button>

  <div class="content">
    <h3>Interactive Interface</h3>
    <p>Popover Content</p>
  </div>
</div>
```

## Custom Properties

- `--duration`
- `--primary`
- `--radius`
- `--bg`
- `--card`

## Browser Support

- Chrome
