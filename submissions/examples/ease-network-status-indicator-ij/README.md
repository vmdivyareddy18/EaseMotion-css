# Network Status Indicator

An online/offline status indicator with pulse animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--indicator-bg` | `#2d2d44` | Card background |
| `--online-color` | `#4ecdc4` | Online status color |
| `--offline-color` | `#f72585` | Offline status color |
| `--pulse-duration` | `2s` | Pulse animation speed |
