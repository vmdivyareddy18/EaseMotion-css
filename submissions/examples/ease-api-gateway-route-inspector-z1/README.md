# Ease API Gateway Route Inspector

## Overview
A responsive HTML and CSS component for monitoring API Gateway routes, request traffic, authentication methods, and route health.

## Features
- Route overview cards
- HTTP method information
- Authentication details
- Request volume metrics
- Animated status badges
- Responsive design
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

Provides a reusable animation-first UI component for API gateways, backend services, cloud-native dashboards, and infrastructure monitoring platforms.
