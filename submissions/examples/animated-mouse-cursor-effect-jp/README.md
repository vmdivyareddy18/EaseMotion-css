# Animated Mouse Cursor Effect

## Overview
This component creates a modern custom mouse cursor with smooth movement, glowing effects, and interactive hover animations.

## Features

- Glowing circular cursor effect
- Smooth cursor tracking
- Expanding ring animation on hover
- Interactive button hover state
- Minimal and lightweight implementation
- Pure HTML, CSS, and JavaScript

## Files

- `demo.html` – Main demonstration page with cursor interaction
- `style.css` – Styling and animation effects
- `README.md` – Component documentation

## Usage

1. Open `demo.html` in any modern browser.
2. Move the mouse to see the glowing cursor effect.
3. Hover over the button to activate the expanding ring animation.

## Technologies Used

- HTML5
- CSS3 animations and transitions
- Vanilla JavaScript
