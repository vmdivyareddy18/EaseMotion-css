# Echo Spectrum Ease

A modern animated voice spectrum visualizer created with pure HTML and CSS.

---

## Features

- AI voice visualizer
- Animated spectrum bars
- Neon glow effect
- Smooth wave animation
- Responsive layout
- Pure HTML & CSS
