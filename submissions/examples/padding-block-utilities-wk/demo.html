Logical Block Content

Logical block properties apply padding relative to the text flow axis. In horizontal layouts, this represents the top and bottom. In vertical layouts, it automatically maps to the left and right sides.

Guidelines & Browser Behavior

  • Logical Orientation: Flow-relative styles keep design alignment consistent across multiple localized languages and vertical text directions.
  • Clean Hierarchy: Uses framework spacing tokens from var(--ease-space-1) through var(--ease-space-16).
