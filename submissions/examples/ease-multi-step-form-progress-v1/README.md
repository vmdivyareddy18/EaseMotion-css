# Ease Multi Step Form Progress

## Overview

Ease Multi Step Form Progress is a reusable progress indicator for multi-step forms. It highlights completed, active, and upcoming steps with smooth animations using only HTML and CSS.

---

## Features

- ✅ Completed step indicators
- 🔵 Active step highlight
- 📈 Animated progress line
- 🎨 Hover interaction
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo showcases:

- Four-step progress flow
- Completed states
- Active state
- Animated connecting line

---

## Usage

```html
<div class="progress">
...
</div>
```

---

## Customization

- Step colors
- Progress width
- Circle size
- Labels
- Animation speed

---

## Browser Support

Compatible with all modern browsers supporting CSS animations and Flexbox.

---

## Why This Fits EaseMotion CSS

Ideal for checkout flows, onboarding screens, registration forms, surveys, dashboards, and wizard-style interfaces. It provides a reusable multi-step progress component with no JavaScript dependency.
