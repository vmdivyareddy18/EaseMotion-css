# Skew Tilt Card Hover

## What does this do?
Applies a slight 3D perspective rotation, skew, and translation lift to card elements on hover.

## How is it used?
Add the class to your card container:

```html
<div class="tilt-skew-card">
  <h3>Card Title</h3>
  <p>Content goes here.</p>
</div>
```

## Why is it useful?
It provides dynamic, physical depth to layout containers (like dashboards or feature grids) to draw attention and increase clicks.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
