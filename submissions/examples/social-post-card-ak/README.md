# Social Post Card

## What does this do?
A social media post card component with interactive like (heart toggle animation), comment simulation, and share button. Displays the post author, content, image placeholder, and interaction stats.

## How is it used?
Open demo.html in a browser. Click the heart to like/unlike with a pop animation. Type a comment and click Comment to increment the count. Click Share to see a simulated share notification.

## Why is it useful?
This demonstrates a social feed post UI pattern suitable for social media apps, community platforms, or content feeds. Includes micro-interactions and animated feedback.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to see the social post card in action.
