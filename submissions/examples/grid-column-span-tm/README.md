# Grid Column Span

grid-column: span — Makes grid items span multiple columns

## Features
- Uses `--ease-color-surface`
- Uses `--ease-radius-lg`
- Uses `--ease-shadow-md`
- Uses `--ease-color-primary`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Grid Column Span</span>
</div>
```

## Why is it useful?

This submission demonstrates the grid column span CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
