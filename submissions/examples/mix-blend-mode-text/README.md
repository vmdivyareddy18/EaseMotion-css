# mix-blend-mode-text

Applies `mix-blend-mode` to text elements over images and gradients. Features an interactive blend mode selector to switch between difference, exclusion, screen, multiply, and more.

## Usage

Text overlays an image or gradient background. Use the dropdown to change blend modes in real time. Each mode produces unique color interactions between the text and underlying content.

## Browser Support

`mix-blend-mode` is supported in all modern browsers. The `difference`, `exclusion`, `screen`, and `multiply` values work universally in Chrome, Firefox 90+, Safari 15+, and Edge.
