Outline to gradient reveal mechanics in pure CSS

  • -webkit-text-stroke: 1px [color]; defines the hollow text outline, giving it a lightweight, editorial structure.
  • data-text & attr(data-text): maps content dynamically onto a pseudo-element overlay (`::after`), ensuring standard screen-readers and SEO bots read the heading cleanly.
  • clip-path: inset(0 100% 0 0); crops the gradient overlay to 0% width by clipping it from the right-hand edge.
  • clip-path: inset(0 0 0 0); on hover reveals the full gradient fill, sliding the color sweep across from left to right.
