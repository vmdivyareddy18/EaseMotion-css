# 3D Perspective Tilt Popover - Dashboard Analytics

A pure CSS animated popover component designed for dashboard analytics interfaces with a smooth 3D perspective tilt interaction.

## Features

- CSS-only animation
- No JavaScript dependency
- 3D perspective tilt transition
- Responsive layout
- Keyboard accessible using focus states
- Custom CSS variables for animation control

## Usage

```html
<div class="tilt-popover">

<button class="tilt-trigger">
View Analytics
</button>

<div class="tilt-panel">
Dashboard content
</div>

</div>
