# Squish Text Utility

**What does this do?**  
Adds a quick squish-and-rebound animation for text-sized interactive elements such as buttons, links, and chips.

**How is it used?**  
```html
<button class="squish-text">Save Changes</button>
<a class="squish-text" href="/docs">Read the docs</a>
<span class="squish-text" tabindex="0">Synced</span>
```

**Why is it useful?**  
Text controls often need feedback that feels tactile without taking up extra space or shifting nearby content. This utility fits EaseMotion CSS because it is readable, composable, dependency-free, and focused on one clear motion behavior.
