# CSS Voting / Poll Widget

## What does this do?
CSS voting/poll widget with bar chart results, radio selection, rating stars with distribution, and quick emoji reactions.

## How is it used?
```html
<div class="vp-poll"><h3>Question?</h3><label class="vp-option"><input type="radio" name="poll" class="vp-radio"><div class="vp-bar" style="width:45%"><span>Option</span></div></label><button class="vp-btn">Vote</button></div>
```

## Why is it useful?
Creates interactive poll and voting UIs without JavaScript. Bar chart results with animated fill. Radio selection with visual feedback. Star rating with distribution breakdown. Quick emoji reaction buttons. Total vote count display. Ideal for surveys, feedback widgets, and community polls.
