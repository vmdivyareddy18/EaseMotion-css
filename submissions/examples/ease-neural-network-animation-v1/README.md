# Ease Neural Network Animation

## Overview

Ease Neural Network Animation creates a futuristic AI-inspired visualization using animated nodes and connecting lines.

---

## Features

- Animated network nodes
- Flowing signal effect
- Pulsing glow animation
- SVG connection lines
- Pure HTML & CSS
- Responsive layout

---

## Demo

Includes:

- Connected AI nodes

- Pulsing network

```html
<div class="network">
...
</div>
```
