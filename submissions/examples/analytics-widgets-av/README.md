# Analytics Widgets

## What does this do?

Creates a colorful analytics dashboard featuring responsive KPI cards with gradients, icons, and growth indicators.

---

## How is it used?

Open `demo.html` directly in a browser.

```html
<div class="grid">
    <div class="card revenue">
        ...
    </div>
</div>
```

---

## Why is it useful?

Analytics widgets are commonly used in admin dashboards, SaaS platforms, CRM systems, finance applications, and monitoring tools. This standalone example demonstrates a modern dashboard layout with colorful gradients and responsive cards using only HTML and CSS.
