
# Animated Gradient Border Pulse

A reusable CSS animation that creates a vibrant animated gradient border with a subtle pulsing glow. The effect is built entirely with CSS and can be applied to cards, buttons, pricing sections, feature boxes, and other UI components to draw attention without relying on JavaScript.

## Features

* 🎨 Animated multi-color gradient border
* ✨ Smooth pulsing glow effect
* ⚡ Pure CSS implementation
* 📱 Responsive design
* 🔧 Easy to customize using CSS variables
* 🚀 Lightweight and reusable

## Usage

1. Include the `style.css` file in your HTML.

2. Apply the class to any element.

```html
<div class="gradient-border-pulse">
    <h2>Premium Plan</h2>
    <p>Beautiful animated border using pure CSS.</p>
</div>
```

## Customization

The effect can be customized using the following CSS variables:

| Variable        | Description           |
| --------------- | --------------------- |
| `--border-size` | Border thickness      |
| `--radius`      | Border radius         |
| `--color-1`     | First gradient color  |
| `--color-2`     | Second gradient color |
| `--color-3`     | Third gradient color  |

Different color themes can be created by overriding these variables.

## Browser Support

Works in all modern browsers that support:

* CSS Custom Properties
* CSS Animations
* Pseudo-elements
* CSS Gradients
* CSS Masking (for border effect)
