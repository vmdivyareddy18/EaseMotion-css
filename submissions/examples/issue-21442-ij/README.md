## Rainbow Arch Animation

A pure CSS rainbow that arcs across the sky with all seven colors. The rainbow fades in and pulses gently while soft clouds frame the arch on both sides.

**Features:**
- Conic gradient with seven spectral colors
- Radial mask for arched reveal
- Flanking clouds with gentle drift
- Respects `prefers-reduced-motion`
