# Animated WiFi Signal Wave

## Overview

A reusable WiFi signal animation featuring a subtle wave effect suitable for connectivity indicators, smart devices, IoT dashboards, router interfaces, and communication platforms.

## Features

- CSS-only animation
- WiFi signal pulse effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<div class="wifi-container">
  <span class="wifi-icon">📶</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
