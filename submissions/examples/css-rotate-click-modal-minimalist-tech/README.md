# CSS Rotate-Click Modal – Minimalist Tech Layout

## Overview

A responsive Minimalist Tech modal built entirely with HTML and CSS. The component features a smooth rotate-click opening animation, a clean modern interface, responsive design, and keyboard accessibility without requiring JavaScript.

## Features

- Pure HTML & CSS
- Rotate-click opening animation
- Minimalist Tech design
- Glassmorphism effect
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Reduced motion support

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--duration`
- `--easing`
- `--rotate`
- `--scale`
- `--radius`
- `--blur`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Open Dashboard** to display the animated rotate-click modal.

Customize colors, animation timing, rotation angle, scaling, blur, and spacing using the CSS custom properties in `style.css`.
