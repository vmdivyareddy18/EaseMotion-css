# CSS Text Truncation Showcase

A clean, CSS-only side-by-side component showcasing the three most common ways to handle text overflow and layout wrapping issues.

## Features Included
1. **Single-line Ellipsis:** Using `text-overflow: ellipsis` for strict single line constraints.
2. **Multi-line Clamping:** Utilizing `-webkit-line-clamp` to truncate paragraphs gracefully at a specific line threshold.
3. **Word Breaking:** Preventing layout breakage from unbroken text strings or long URLs using `word-break`.
