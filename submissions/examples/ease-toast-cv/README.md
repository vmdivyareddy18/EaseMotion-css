# Animated Toast Notification (`ease-toast-cv`)

A lightweight toast notification component built using **HTML5** and **Pure CSS**.

## Features

- ✅ Slide-in animation
- ✨ Fade-in animation
- 🏀 Bounce entry animation
- 📚 Stacked toast layout
- 🟢 Success variant
- 🔵 Info variant
- 🟡 Warning variant
- 🔴 Error variant
- 📱 Responsive design
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-toast-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
