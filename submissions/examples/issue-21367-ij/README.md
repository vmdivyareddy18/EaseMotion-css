# Rabbit Hopping

1. What does this do? A CSS-only rabbit that hops with body squash/stretch and ear flop animation.

2. How is it used? Place .rabbit inside .scene. Body is a white ellipse; ears are pink pill shapes animated with rotation.

3. Why is it useful? Combines vertical hop with secondary ear motion for more lifelike animation.
