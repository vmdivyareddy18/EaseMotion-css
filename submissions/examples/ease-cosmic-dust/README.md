# ✨ Ease Cosmic Dust Animation

A smooth CSS-based cosmic dust particle animation built for the **EaseMotion CSS** library.

This effect simulates floating cosmic dust particles drifting through space using pure CSS animations with lightweight JavaScript for particle generation.

---

## 🌌 Preview

> A dark cosmic background with glowing dust particles gently floating and fading in random motion.

---

## 🚀 Features

- Pure CSS animation (no external libraries)
- Lightweight JavaScript for dynamic particle generation
- Smooth easing motion (`ease-in-out`)
- Realistic floating + fading dust effect
- Fully responsive full-screen animation
- Easy to integrate into any project
