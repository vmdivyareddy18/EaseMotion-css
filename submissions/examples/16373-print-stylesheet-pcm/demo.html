Print Stylesheet · @media print module

Optimized for printing: hides UI chrome, shows link URLs, black-on-white, no page breaks inside content

Article Content

This is the main article content. When printing, the sidebar, navbar, footer, and other UI chrome will be hidden automatically. Link URLs will appear after each anchor tag. For example: EaseMotion Documentation and GitHub Repository.

Cards and tables will not break across pages. The print stylesheet ensures black text on white background, removes all shadows and background images to save ink, and sets appropriate margins.
