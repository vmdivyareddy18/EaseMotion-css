# Network Status Indicator

## What does this do?
Displays the current online/offline status using the Navigator API, with connection details (type, effective type, downlink speed, RTT) from the Network Information API.

## How is it used?
Open demo.html in a browser. The status updates automatically when connectivity changes. Click "Check Connection" to refresh. Toggle airplane mode to see the indicator switch.

## Why is this useful?
Demonstrates real-time network status monitoring using browser APIs, useful for progressive web apps and offline-aware interfaces.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript (Navigator API)

## Preview
Open demo.html directly in your browser.
