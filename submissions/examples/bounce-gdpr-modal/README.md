# Bounce GDPR Modal

A lightweight GDPR consent modal component with a smooth bounce entrance animation built using pure HTML and CSS.

## Features

- Pure CSS animation
- No JavaScript dependency
- Responsive design
- Accessible modal layout
- Smooth bounce entrance effect
- Easy customization

## Usage

Add the modal structure:

```html
<section class="gdpr-overlay">
  <div class="gdpr-modal">
    <h2>Privacy & Cookies</h2>
    <p>Your privacy matters to us.</p>
  </div>
</section>
