# Interactive Command Palette Component

A modern command palette inspired by VS Code, Linear, Notion, and Raycast.

## Features

- Search input field
- Categorized command groups
- Quick action shortcuts
- Keyboard shortcut badges
- Recent searches section
- Hover animations
- Smooth modal animation
- Responsive layout
- Pure HTML and CSS

## Folder Structure

```text
interactive-command-palette/
├── demo.html
├── style.css
└── README.md
```

## Included Sections

- Quick Actions
- Navigation Commands
- Recent Searches
- Keyboard Help Footer

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Responsive Design

## Usage

1. Open `demo.html`
2. Ensure `style.css` is in the same folder.
3. Customize commands and categories as needed.
