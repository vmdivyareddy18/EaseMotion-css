EaseMotion CSS — Form Controls

CSS accent-color Customization

Styling native interactive form elements has historically required replacing them with custom divs. The accent-color property solves this by recoloring native controls elegantly.

Native Form Controls

Recolored using the CSS accent-color property.

Color Theme Selector

Choose a theme color to dynamically override the CSS variable controlling accent-color.

.accented-control {
  accent-color: #7c3aed;
}
