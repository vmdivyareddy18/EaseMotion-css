# Resume Preview Builder

## What does this do?
A live resume preview builder with editable form fields on the left and a real-time updating preview card on the right. Users can type their name, bio, skills, and experience to see a styled resume render instantly.

## How is it used?
Open demo.html in a browser. Fill in the form fields in the editor panel. The resume card preview updates in real time as you type.

## Why is it useful?
This provides a two-way data binding demo using input events, useful for profile editors, dashboard settings, or any live-preview use case.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to try the resume builder.
