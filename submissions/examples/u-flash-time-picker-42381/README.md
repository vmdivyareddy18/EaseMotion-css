# Flash Time Picker

A Landing Page-inspired Flash Time Picker built using pure CSS for the EaseMotion CSS library.

## Features

- Pure CSS implementation
- Flash animation on hover
- Responsive layout
- Accessible HTML
- No JavaScript required
