# Animated Data Table

CSS-only data table with sortable headers, striped rows, and hover animations.

## Features

- Sticky table header with sort indicator icons
- Alternating row stripes for readability
- Hover lift effect with elevated shadow
- Responsive scroll wrapper for mobile
- Pagination bar with active page state
- Status badges with distinct color coding
- Custom properties for easy theming

## Usage

Include the `style.css` in your HTML and use the BEM class structure. The table is fully responsive with horizontal scroll on narrow viewports. Sort indicators are purely visual — add your own JavaScript for interactive sorting.

## Browser Support

All modern browsers (Chrome, Firefox, Safari, Edge). Uses `sticky`, `flex`, `grid`, and CSS custom properties.
