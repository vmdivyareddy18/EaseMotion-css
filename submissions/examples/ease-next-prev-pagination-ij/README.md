# Next/Prev Pagination

A page navigation control with animated button transitions.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--btn-bg` | `#2d2d44` | Button background |
| `--arrow-color` | `#888` | Arrow/text color |
| `--hover-bg` | `#3a3a5a` | Hover background |
| `--slide-duration` | `0.25s` | Transition speed |
