# CSS light-dark() Color Function

A CSS demo of the light-dark() CSS color function that outputs different colors based on the current color-scheme value, enabling theme switching without media queries.

## Files

- `demo.html` - Cards with light-dark() colors and toggle buttons
- `style.css` - light-dark() function, color-scheme toggling

## Usage

Open `demo.html` and click Light/Dark to see light-dark() adapt.
