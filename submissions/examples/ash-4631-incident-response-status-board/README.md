# Incident Response Status Board

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive incident response status board that visualizes incidents across different response stages, making operational progress easy to monitor.

## Features

- Multi-column status board
- Detected, Investigating, Mitigating, and Resolved stages
- Color-coded incident cards
- Responsive grid layout
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="card investigating">

    <strong>API Gateway Errors</strong>

    <p>Engineering reviewing logs</p>

</div>
```

## Main Classes

```css
.board
.columns
.column
.card
.detected
.investigating
.mitigating
.resolved
```
