# Number Roll Animate

A vertical scroll number animation for counters.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--roll-bg` | `#2d2d44` | Counter background |
| `--digit-color` | `#4ecdc4` | Digit color |
| `--roll-height` | `48px` | Digit font size |
| `--roll-duration` | `0.5s` | Roll animation speed |
