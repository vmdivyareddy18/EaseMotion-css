# Notification Panel Slide

A notification panel where message rows slide in with small staggered delays.

## Files

- `demo.html` - notification panel markup.
- `style.css` - slide-in motion, row hover state, and reduced-motion fallback.

## Highlights

- Uses sequential delays for a lightweight panel entrance.
- Keeps rows readable while they animate into place.
- Provides a static fallback for reduced-motion users.
