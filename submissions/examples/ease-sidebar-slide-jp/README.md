# Ease Sidebar Slide

A modern sidebar animation component that smoothly slides in and out from screen edges using pure HTML and CSS.

## Features

- Left Sidebar Slide Animation
- Right Sidebar Slide Animation
- Smooth Transform Transitions
- Backdrop Overlay
- Responsive Design
- Glassmorphism Styling
- Pure HTML & CSS
- No JavaScript Required
