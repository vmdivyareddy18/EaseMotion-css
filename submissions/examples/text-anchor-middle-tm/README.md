# Text Anchor Middle

text-anchor — Aligns SVG text horizontally

## Features
- Uses `--ease-color-primary`
- Uses `--ease-color-success`
- Uses `--ease-color-danger`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Text Anchor Middle</span>
</div>
```

## Why is it useful?

This submission demonstrates the text anchor middle CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
