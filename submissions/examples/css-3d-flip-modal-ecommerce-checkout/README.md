# CSS 3D Flip Modal – E-Commerce Checkout Layout

## Overview

A responsive E-Commerce Checkout modal built entirely with HTML and CSS. The component features a smooth 3D flip opening animation, a modern checkout summary, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip animation
- E-Commerce Checkout interface
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--success`
- `--bg`
- `--surface`
- `--text`
- `--muted`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--radius`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Click **Proceed to Checkout** to display the animated 3D Flip Checkout Modal. Colors, animation timing, flip angle, spacing, and shadows can all be customized using the CSS custom properties defined in `style.css`.
