# writing-mode-horizontal

## Overview

Demonstrates the `writing-mode` CSS property with `horizontal-tb`, `vertical-rl`, and `vertical-lr`. Shows how text direction and line stacking change across each mode, including a mixed-mode comparison section where identical content is rendered in all three modes side by side.

## Usage

Open `demo.html` in any browser. Sections show each writing mode individually with descriptive paragraphs, followed by a three-column mixed comparison using identical line-based content. Vertical boxes have fixed heights to make the direction visible.

## Features

- Three writing modes displayed individually with explanatory text
- Side-by-side mixed comparison with identical content in all modes
- Vertical height constraints for readable vertical text
- Responsive flex layout that stacks on mobile
- `prefers-reduced-motion` support
- Dark theme with purple accent and orange section headers
