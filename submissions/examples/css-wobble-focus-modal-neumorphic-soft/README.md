# CSS Wobble-Focus Modal for Neumorphic Soft Layouts

1. **What does this do?**
   It renders a pure CSS animated modal component utilizing a smooth **Wobble-Focus** interaction transition, styled for modern **Neumorphic Soft** interface aesthetics.

2. **How is it used?**
   Open `demo.html` in any modern web browser. Click the **Open Modal** button to display the modal with the Wobble-Focus entrance animation. Close it using the **Close** button, the **×** icon, or by clicking the overlay. No JavaScript is required.

3. **Why is it useful?**
   It provides a lightweight, zero-JavaScript modal solution featuring a responsive layout, keyboard accessibility, and customizable CSS custom properties for animation timing, easing, scale, colors, shadows, border radius, and other visual styles. The soft neumorphic design makes it easy to integrate into modern dashboards, landing pages, and web applications while maintaining a clean, minimal user experience.
