# Seven-Segment Display (Issue #18265)

A 2-digit seven-segment display (like a calculator). Each segment lights up with a color transition. JavaScript sets the displayed number. Configurable color and size.

## Usage

Open `demo.html` in a browser. Displays "42" by default. Click **Set** to enter a different 2-digit number.

## Files

- `demo.html` — HTML5 page with 2-digit segment display and JS
- `style.css` — Dark theme styles with segment transitions
- `README.md` — This file
