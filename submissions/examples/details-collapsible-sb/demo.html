No-JavaScript pattern

CSS-only collapsible sections

Use native details and summary markup for accessible expandable content with a rotating arrow indicator.

What is EaseMotion?
EaseMotion is a lightweight CSS framework for motion utilities, reusable components, and expressive UI details.
Does this need JavaScript?
No. The browser handles expanded state natively, while CSS styles the summary, body, and open-state arrow rotation.
Can it be used in FAQs?
Yes. It works well for FAQ lists, settings panels, release notes, and compact documentation sections.
