# Release Countdown Banner

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern release countdown banner designed to promote upcoming launches, product releases, events, or feature rollouts.

## Features

- Countdown timer layout
- Floating time cards
- Release announcement banner
- Call-to-action button
- Responsive design
- Glassmorphism styling
- Smooth animations
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="countdown">

    <div class="time-card">

        <span class="number">12</span>

        <span class="label">Days</span>

    </div>

</div>
```

## Main Classes

```css
.banner
.badge
.countdown
.time-card
.number
.notify-btn
```
