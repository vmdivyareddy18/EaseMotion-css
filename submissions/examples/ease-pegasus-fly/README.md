# ease-pegasus-fly

A pegasus fly animation for EaseMotion CSS.

## Usage

```html
<!-- Fly across screen -->
<div class="ease-pegasus-fly">
  <span>🦄</span>
</div>

<!-- Hover float -->
<div class="ease-pegasus-hover">
  <span>🦄</span>
</div>
```

## Speed Variants

```html
<div class="ease-pegasus-fly ease-pegasus-fly-slow">...</div>
<div class="ease-pegasus-fly ease-pegasus-fly-fast">...</div>
```

## Classes

| Class | Description |
|---|---|
| `ease-pegasus-fly` | Flies across screen with arc path |
| `ease-pegasus-hover` | Gentle floating hover animation |
| `ease-pegasus-wings` | Wing flap on child element |
| `ease-pegasus-fly-slow` | Slow speed (5s) |
| `ease-pegasus-fly-fast` | Fast speed (1.5s) |

## Features
- Fly across screen with natural arc path
- Gentle hover float variant
- Wing flap animation
- Speed variants
- Works on any element
- Respects prefers-reduced-motion
- Uses EaseMotion CSS design tokens
