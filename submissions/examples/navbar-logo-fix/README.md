# Navbar Logo Visibility Fix

## Description

This example improves the visibility of the EaseMotion CSS logo within the navbar by increasing its rendered size.

The original logo appeared relatively small compared to the navbar height and surrounding content. Increasing the logo size helps strengthen brand visibility while maintaining responsiveness and layout stability.

---

## Usage

HTML:

```html
<a href="index.html" class="docs-logo">
  <img src="assets/logo.svg" alt="EaseMotion CSS" class="docs-logo-img" />
</a>
