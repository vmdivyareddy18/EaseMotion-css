# Gradient Animated Border (#36690)

Gradient animated border with color-shift effect in a dark-themed card layout.

## CSS Custom Properties

| Property          | Default     | Description                    |
|-------------------|-------------|--------------------------------|
| `--border-width`  | `4px`       | Thickness of the border        |
| `--gradient-start`| `#ff6b6b`   | Start color of the gradient    |
| `--gradient-end`  | `#4ecdc4`   | End color of the gradient      |
| `--shift-duration`| `3s`        | Duration of one rotation cycle |

## Interactive Controls

- Range slider to adjust border width
- Color pickers for gradient start/end colors
- Range slider to adjust rotation speed
