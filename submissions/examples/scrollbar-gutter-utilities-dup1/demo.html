scrollbar-gutter Stability Utilities

Prevent jarring layout shifts when scrollbars appear or disappear. The CSS scrollbar-gutter property reserves space for the scrollbar so the content width never changes.

📏 scrollbar-gutter: stable ⚡ CLS Prevention 🖥️ Dashboard Layouts

1. The Problem Without scrollbar-gutter

By default, when a scrollbar appears, it takes up space from the content area, causing Cumulative Layout Shift (CLS). Buttons jump, text reflows, and the layout feels unstable. Click the button below to toggle content and observe the shift.

2. Dashboard Layout Preset

The .ease-scroll-layout utility creates a fixed sidebar + scrollable main content layout. The scrollbar-gutter: stable both-edges on the main area ensures the sidebar and content never shift when the scrollbar appears.

🔧 CSS Reference

/* 1. Basic: Reserve space for scrollbar on one side */
.ease-gutter-stable {
  overflow: auto;
  scrollbar-gutter: stable;
}

/* 2. Symmetric: Reserve space on BOTH sides (keeps layout centered) */
.ease-gutter-both {
  overflow: auto;
  scrollbar-gutter: stable both-edges;
}

/* 3. Dashboard Layout Preset */
.ease-scroll-layout {
  display: grid;
  grid-template-columns: 280px 1fr;
  height: 100vh;
  overflow: auto;
  
  /* Reserve gutter on both edges so sidebar never shifts */
  scrollbar-gutter: stable both-edges;
}

/* Main scrollable area within the layout */
.ease-scroll-layout main {
  padding: 2rem;
  overflow-y: auto;
  scrollbar-gutter: stable;
}
