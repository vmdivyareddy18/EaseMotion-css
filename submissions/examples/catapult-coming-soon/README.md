# Catapult Coming Soon Component

A CSS-only animated Coming Soon component built for EaseMotion CSS.

## Preview

The Catapult Coming Soon component creates a playful launch animation where the
catapult arm moves and launches a ball to represent upcoming features,
products, or announcements.

## Features

- Pure HTML and CSS
- No JavaScript required
- Smooth catapult launch animation
- Responsive design
- Accessible structure
- Lightweight and reusable
