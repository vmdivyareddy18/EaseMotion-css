# 3D Perspective Tilt Popover – SaaS Showcase

## Description

A responsive CSS-only popover featuring a smooth 3D perspective tilt animation designed for modern SaaS showcase interfaces.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS custom properties
- Smooth 3D perspective animation

## Usage

```html
<div class="popover">
  <button class="trigger">View Pricing</button>

  <div class="popover-content">
    <h3>Premium Plan</h3>
    <p>Your content here.</p>
  </div>
</div>
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
