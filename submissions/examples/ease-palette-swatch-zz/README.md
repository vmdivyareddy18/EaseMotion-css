# Palette Swatch

## What does this do?
Palette Swatch component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"ps-primary\">Palette Swatch Primary</div>
```

## Why is it useful?
Provides a reusable palette swatch UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
