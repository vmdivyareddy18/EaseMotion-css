# Ease Crystal Focus Rings

## Overview

A futuristic targeting interface featuring rotating crystal rings, a glowing center hub, and elegant glassmorphism styling built entirely with HTML and CSS.

## Features

- Rotating focus rings
- Glassmorphism center hub
- Dual-direction animations
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-crystal-focus-rings-z1/
├── demo.html

└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox

- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

Provides a reusable futuristic targeting animation suitable for AI dashboards, monitoring systems, gaming interfaces, cybersecurity tools, and developer showcases.
