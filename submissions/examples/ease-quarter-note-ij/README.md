# Ease Quarter Note

A musical quarter note pulsing with the beat, creating a rhythmic audio-visual cue.

## Usage

Include the stylesheet in your HTML:

```html
<link rel="stylesheet" href="style.css" />
```

The animation runs automatically on the `.note` element.

## Custom CSS Variables

| Variable          | Default  | Description                  |
|-------------------|----------|------------------------------|
| `--note-color`    | `#333`   | Color of the note            |
| `--pulse-color`   | `#4ecdc4`| Glow color of the pulse      |
| `--pulse-duration`| `0.8s`   | Duration of one pulse cycle  |

## Interactive Controls

Open `demo.html` to use the color pickers and range slider for live customization.
