# CSS Elastic Slide Tooltip - Creative Portfolio Layout

A pure CSS animated tooltip designed for creative portfolio websites with a smooth elastic slide interaction.

## Features

- Pure CSS animation
- No JavaScript required
- Creative portfolio inspired design
- Elastic slide entrance effect
- Hover and keyboard focus support
- Responsive layout
- Custom CSS animation variables

## Usage

```html
<div class="tooltip-wrapper">

  <button class="project-btn">
    View Project
  </button>

  <div class="tooltip">
    Project details
  </div>

</div>
