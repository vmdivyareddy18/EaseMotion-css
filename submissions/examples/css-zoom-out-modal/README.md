# Pure CSS Spatial Zoom-Out Modal Component

An interaction element modeled around premium e-commerce showcases. It replicates deep visual perspective layering by shifting an expanded entrance node back into normal focus proportions seamlessly without running custom evaluation scripts.

## Core Properties
Customize the properties directly in `style.css` (`:root` selector context):

| Custom CSS Token | Default Setting | Technical Role |
|---|---|---|
| `--zoom-duration` | `0.5s` | The timeframe allocated for structural entry velocity. |
| `--zoom-easing` | `cubic-bezier(0.16, 1, 0.3, 1)` | Power-4 deceleration layout providing natural visual dampening. |
| `--zoom-scale-start` | `1.25` | Spatial scaling index configuration before alignment settles. |
