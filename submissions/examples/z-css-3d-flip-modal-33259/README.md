# CSS 3D Flip Modal

EaseMotion CSS component implementing a pure CSS animated modal using a smooth 3D flip transition.

## Features

- Pure CSS
- No JavaScript
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- Marketing landing page design
- Glassmorphism styling
- Smooth 3D Flip animation
- 60fps performance

## CSS Variables

```css
--duration
--ease
--scale
--primary
--secondary
```
