# Interactive Demo Page

## Overview

This submission adds a simple interactive demo page to showcase the available animations in the EaseMotion-css library.

## Features

- Live animation preview
- Responsive layout
- Easy-to-read interface
- Hover interaction
- Animation cards

## Folder Structure

```
interactive-demo-page/
│── demo.html
│── style.css
└── README.md
```

## Usage

1. Open `demo.html` in a browser.
2. Hover over or interact with the demo cards.
3. Replace the placeholder animation classes with the project's actual animation classes if needed.

## Expected Benefits

- Better user experience
- Quick animation preview
- Easier documentation
- Improved onboarding for new users
