# Animated Portfolio Pie Chart Rotate

## Overview

Animated Portfolio Pie Chart Rotate is a lightweight CSS animation that simulates portfolio allocation visualization using a rotating pie chart.

## Features

- CSS-only animation
- Rotating pie chart
- Portfolio analytics visualization
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="portfolio-pie-chart">
  <div class="pie-chart"></div>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
