# Animated Kanban Task Board

A responsive Kanban-style task management board with animated task cards and workflow columns.

## Features

- To Do, In Progress, and Completed columns
- Interactive task cards
- Hover animations
- Responsive layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Add more workflow columns
- Add additional tasks
- Change colors and spacing
- Modify animation effects
