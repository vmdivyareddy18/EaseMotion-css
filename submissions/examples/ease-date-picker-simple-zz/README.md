# Date Picker

## What does this do?
Date Picker component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"dp-primary\">Date Picker Primary</div>
```

## Why is it useful?
Provides a reusable date picker UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
