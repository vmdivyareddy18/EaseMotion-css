# Animated Order Tracking Dashboard

A modern **Order Tracking Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example provides a responsive dashboard for monitoring order progress, delivery timelines, courier status, and shipment analytics with smooth CSS-only animations.

## Features

* 📦 Order overview cards
* ✅ Delivered orders tracking
* 🚚 In Transit status
* ⏳ Pending orders indicator
* 🛣️ Delivery timeline
* 👨‍✈️ Courier status cards
* 📊 CSS-only weekly delivery chart
* 📈 Delivery progress indicators
* 🔔 Recent order activity feed
* ✨ Smooth hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

## Customization

All colors, spacing, and theme settings are controlled using CSS variables.

Example:

```css
:root{
    --primary:#2563eb;
    --success:#10b981;
    --warning:#f59e0b;
    --danger:#ef4444;
    --surface:#ffffff;
    --bg:#f5f7fb;
}
```
