# Ease Holographic Dial Menu

## Overview

A futuristic radial command menu with holographic styling, animated rotation, and glowing center control built entirely with HTML and CSS.

## Features

- Radial menu layout
- Rotating holographic dial
- Animated center button
- Glassmorphism interface
- Responsive design
- Pure HTML & CSS

## Folder Structure

```text
ease-holographic-dial-menu-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Demonstrates reusable circular layouts, holographic motion, and elegant micro-interactions suitable for futuristic dashboards, gaming interfaces, productivity tools, and developer UIs.
