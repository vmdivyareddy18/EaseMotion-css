# Animated Command Palette (`ease-command-palette-cv`)

A lightweight command palette UI built using **HTML5** and **Pure CSS**.

## Features

- Zoom entrance animation
- Backdrop blur
- Styled search input
- Animated command list
- Responsive layout
- Pure CSS
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-command-palette-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

> Note: This example demonstrates the visual design and animations only. Interactive keyboard shortcuts and open/close behavior require JavaScript and are intentionally omitted to comply with the repository's Pure CSS requirement.

## Browser Support

Works in all modern browsers.
