# Music Producer Personal Brand Landing Page

Closes #24820 — Full personal brand site for an independent music producer
built with EaseMotion CSS classes. Underground dark aesthetic.

## Sections

| Section | Content |
|---------|---------|
| Hero | Studio waveform visual + animated bars + CTA |
| Beat Catalog | 6 beat cards with mini waveforms, play buttons, pricing |
| Genres | 6 genre specialty cards with beat counts |
| Client Artists | 6 artist profile cards |
| Pricing | 4 tier cards (Basic, Premium, Exclusive, Custom) |
| Testimonials | 3 artist review cards with star ratings |
| Inquiry Form | Full project inquiry with service selector |
| Footer | Links, social icons, copyright |

## EaseMotion Classes Used

| Class | Where |
|-------|-------|
| ease-fade-in | Hero eyebrow, buttons, studio visual |
| ease-slide-up | Hero headline |
| ease-zoom-in | Studio waveform visual |
| ease-on-scroll | All below-fold sections |
| ease-delay-1 to 6 | Staggered grid entrances |
| ease-hover-lift | Beat cards, pricing cards, testimonials |
| ease-hover-scale | Nav CTA, submit button, buy buttons |
| ease-hover-glow | Client cards, studio visual |

## Files
- demo.html — self-contained landing page, no build step
- style.css  — all EaseMotion utilities + page styles
- README.md  — this file

## Features
- Animated studio waveform in hero
- Mini waveform visualisers on each beat card
- IntersectionObserver scroll-triggered entrances
- Fully responsive (mobile/tablet/desktop)
- Realistic copy and pricing — no lorem ipsum
- prefers-reduced-motion respected
