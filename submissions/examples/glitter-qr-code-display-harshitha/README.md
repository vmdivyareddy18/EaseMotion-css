# ✨ Glitter QR Code Display

A reusable QR Code display component with a glitter shimmer effect, built using **EaseMotion CSS**.  
This component is responsive, accessible, and works without JavaScript.

---

## 📂 Project Structure

The component is organized as follows:

submissions/examples/glitter-qr-code-display-harshitha/
├── demo.html      # Live demo page
├── style.css      # Glitter animation styles
└── README.md      # Documentation


---

## 🚀 Features
- Pure CSS glitter animation (no JavaScript required)
- Uses EaseMotion CSS variables (`--ease-*`)
- Responsive and accessible design
- Easy to drop into any project

---
