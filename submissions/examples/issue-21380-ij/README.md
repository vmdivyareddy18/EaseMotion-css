# Lion Roaring

1. What does this do? A CSS-only lion that roars with a pulsing mane that expands like a sonic ripple.

2. How is it used? Place .lion inside .scene. The face is a gold circle; the mane is a larger circle that scales and emits expanding ring shadows.

3. Why is it useful? Shows how box-shadow with growing spread values can create a ripple/shockwave effect.
