# Checkout Summary Accordion

### 1. What does this do?

Creates a responsive checkout summary card with an animated accordion for displaying order details.

### 2. How is it used?

Include `style.css` and use the provided HTML structure with the checkout card and accordion button in your page.

### 3. Why is it useful?

It provides a clean, interactive way to organize checkout information while keeping the interface lightweight, responsive, and aligned with EaseMotion CSS's animation-first philosophy.
