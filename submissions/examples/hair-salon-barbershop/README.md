# 💇 LuxeCuts Salon Landing Page

A stylish and responsive **Hair Salon & Barbershop Landing Page** built using **EaseMotion CSS**. This demo showcases elegant animations, hover effects, and a modern UI without requiring any build tools.

## ✨ Features

- Modern responsive hero section
- Navigation bar with CTA button
- Services showcase with pricing
- Smooth entrance animations
- Interactive hover effects
- Mobile-friendly layout
- Clean and minimal design
