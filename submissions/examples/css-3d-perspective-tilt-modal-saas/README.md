# CSS 3D Perspective Tilt Modal (SaaS Showcase UI)

A **pure CSS 3D animated modal component** designed for **SaaS showcase interfaces** in EaseMotion CSS.

---

## ✨ Features

- Clean SaaS-style UI design
- Pure CSS 3D perspective animation
- Hover-based tilt interaction
- Smooth modal transitions
- Fully responsive layout
- Keyboard accessible (ESC support)

---

## 🎯 Purpose

This component enhances SaaS dashboards and landing pages with:

- modern UI structure
- interactive 3D motion
- lightweight CSS-only implementation

---

## 🧪 Usage

```html
<button class="open-btn" onclick="openModal()">Open Modal</button>

<div class="modal active">
  <div class="modal-content">
    SaaS modal content
  </div>
</div>
