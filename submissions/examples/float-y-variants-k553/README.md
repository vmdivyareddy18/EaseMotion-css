# Float Y Variants

**EaseMotion Submission** · Contributor: kavin553

---

## 1. What does this do?

Adds a continuous, smooth vertical floating animation with configurable speed and direction modifiers, and automatically pauses on hover/focus.

## 2. How is it used?

```html
<div class="float-y">Default speed</div>
<div class="float-y float-y-slow">Slower</div>
<div class="float-y float-y-fast">Faster</div>
<div class="float-y float-y-reverse">Reversed direction</div>
