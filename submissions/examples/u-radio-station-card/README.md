# ease-radio-station-card

Animated radio station selector card component for music and streaming applications.

## Features

- Animated selected station state
- Glowing active border
- Pulsing now-playing indicator
- Equalizer animation
- Responsive design
- Pure HTML and CSS

## Acceptance Criteria Covered

### Selected State

The active station card receives:

- Animated border highlight
- Glow effect
- Visual selection feedback

### Now Playing Indicator

Includes:

- Pulsing status indicator
- Animated equalizer bars
- Live playback visualization

### Use Cases

- Radio apps
- Streaming platforms
- Podcast applications
- Music dashboards
