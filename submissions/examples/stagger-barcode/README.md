# Stagger Barcode

A reusable animated barcode component with staggered wave motion built using pure CSS.

## Features

- Staggered animation effect
- Smooth vertical wave movement
- Neon glow styling
- Pure CSS implementation
- Responsive design
- No JavaScript required

## Usage

Add the HTML:

```html
<div class="ease-stagger-barcode">

  <span></span>
  <span></span>
  <span></span>
  <span></span>
  <span></span>

</div>
