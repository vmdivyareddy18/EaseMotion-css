# Recipe Card Component

## What does this do?
An interactive recipe card showing ingredients with adjustable servings, expandable cooking instructions, cook time, and a visual placeholder image.

## How is it used?
Open demo.html in a browser. Use "Adjust Servings" to scale ingredients up, "Reset" to restore defaults, and "Show Instructions" to expand the cooking steps.

## Why is this useful?
Demonstrates a practical UI pattern for recipe display with dynamic content scaling, collapsible sections, and clean responsive layout.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
