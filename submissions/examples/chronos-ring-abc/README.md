1. What does this do?
An advanced state-selection checkbox replacement that uses nested, timed border transitions to generate structural depth changes and dual radial ripple pulses upon selection.

2. How is it used?
```html
<label class="chronos-checkbox-label">
  <input type="checkbox" class="chronos-input">
  <div class="chronos-frame">
    <div class="chronos-core"></div>
    <div class="pulse-ring ring-alpha"></div>
    <div class="pulse-ring ring-beta"></div>
  </div>
  <span class="label-text">Option Label</span>
</label>
```
