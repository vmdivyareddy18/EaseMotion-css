# Retro Parallax Hashtag Input Component

An interactive, multi-layer CSS parallax input element built for the **EaseMotion CSS** library. Inspired by classic bold retro geometries, this module utilizes pseudo-states to shift layout depths perfectly across Z-index levels without touching a single line of JavaScript.

## 🚀 Features
- **Pure CSS Interaction**: Implements lightweight hardware-accelerated movements via pseudo transitions.
- **Retro Aesthetic**: Uses highly contrasted styling borders matching classic 90s visual motifs.
- **A11y Compliant**: Retains proper keyboard focus wrappers (`:focus-within`) ensuring seamless screen reader interactions.
