# Animated Profile Completion Card

A modern profile completion tracking component built using HTML and CSS.

## Features

- Animated progress bar
- Completion tracking
- Completed and pending tasks
- Reward notification
- Action buttons
- Responsive layout

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in a browser.

## Technologies

- HTML5
- CSS3
- CSS Animations

- User onboarding
- Job portals
- Learning platforms
- Social networking apps
- SaaS dashboards
