# 🌐 CSS Social Motion

A collection of beautiful, pure CSS animated social media icons for modern websites.

## 🚀 Features

### Icon Styles (8 types)
1. **Classic Brand Colors** - Official brand colors with lift effect
2. **Rounded Hover** - Circular icons with 360° rotation
3. **Slide Up Reveal** - Icon slides up to reveal label
4. **Neon Glow** - Cyberpunk neon glow effect
5. **3D Rotate** - 3D flip animation
6. **Gradient Pulse** - Gradient backgrounds with pulse
7. **Outline Fill** - Circle fills from center on hover
8. **Bounce Effect** - Playful bounce animation

### Supported Platforms
- Facebook
- Twitter / X
- Instagram
- LinkedIn
- YouTube
- GitHub

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All icon animations
- `readme.md` - Documentation
