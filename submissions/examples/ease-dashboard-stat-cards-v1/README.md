# Ease Dashboard Stat Cards

## Overview

Ease Dashboard Stat Cards is a reusable dashboard component displaying key business metrics with icons, trend indicators, and hover animations using pure HTML and CSS.

---

## Features

- 📊 Dashboard metric cards
- 📈 Positive & negative trends
- ✨ Hover lift animation
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Sales card
- Users card
- Orders card
- Revenue card
- Trend indicators

---

## Usage

```html
<div class="dashboard">
...
</div>
```

---

## Customization

- Colors
- Icons
- Card spacing
- Typography
- Animation speed

## Browser Support

Compatible with all modern browsers supporting CSS Grid, Flexbox, transitions, and animations.

Useful for admin dashboards, SaaS applications, analytics platforms, business reports, and portfolio projects. It provides a lightweight, reusable dashboard statistics component.
