# Draw Border Line Popover Interactive

A pure CSS animated popover featuring a smooth draw-border line transition for interactive interface layouts.

## Features

- Pure CSS animation
- No JavaScript required
- Smooth border drawing effect
- Responsive design
- Keyboard accessible
- Custom CSS variables
- Lightweight component

## Usage

```html
<div class="interface-popover">

<button class="trigger">
Open Info
</button>

<div class="popover">
Content here
</div>

</div>
