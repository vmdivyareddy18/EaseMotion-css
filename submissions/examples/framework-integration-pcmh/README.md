# Framework Integration Guides — Issue #7422

## Overview

Interactive documentation for using EaseMotion CSS with React/Next.js, Vue/Nuxt, and Astro. Covers installation, global import, component examples, SSR handling, and scoped styles.

## Contents

| Tab | Framework | Topics |
|-----|-----------|--------|
| React / Next.js | React 18+ | App Router, Pages Router, SSR, CSS Modules |
| Vue / Nuxt | Vue 3+ | Global import, scoped styles, Nuxt config |
| Astro | Astro 4+ | Layout imports, SSR, scoped styles |

## Usage

Open `demo.html` in any browser. Tab interface switches between framework-specific guides.

## Why This Matters

Framework integration guides reduce onboarding friction for developers moving from utility-first frameworks. These patterns ensure SSR compatibility, style isolation, and predictable class usage.
