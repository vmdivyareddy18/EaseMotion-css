EaseMotion CSS

ease-hover-tilt-right

Hover any card below — it pivots in 3D space around its Y-axis, tilting to the right as if leaning away from you.

Hover the Cards

📐

3D Pivot

rotateY(-8deg) rotates the card around its vertical axis.

🪟

Perspective Parent

The perspective lives on the wrapper, not the card itself.

➡️

Lean Right

A negative rotateY angle tilts the right edge away from view.

Tilt Direction Comparison

tilt-left (8deg)
Hover ←
tilt-right (-8deg)
Hover →

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Wrap the element in a perspective parent */
<div class="tilt-wrap">
  <div class="ease-hover-tilt-right">Hover me</div>
</div>

/* 3. Customise the tilt per-instance */
.my-card {
  --tilt-angle: -12deg;          /* steeper tilt */
  --tilt-duration: 0.5s;         /* slower */
  --tilt-perspective: 1000px;    /* flatter look */
}

/* Opposite direction (ease-hover-tilt-left) */
.tilt-left { --tilt-angle: 8deg; }
