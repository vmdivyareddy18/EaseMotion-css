# Fraud Detection Command Center Example Showcase

This directory contains the files for the **Fraud Detection Command Center** example showcase, illustrating the advanced integration of EaseMotion CSS utilities, layout features, card components, animations, and responsive utilities in building modern dashboard systems.

## Features Included
- **Fraud heatmaps**: Implemented using pure responsive grids and CSS widgets.
- **Threat monitoring**: Implemented using pure responsive grids and CSS widgets.
- **Risk indicators**: Implemented using pure responsive grids and CSS widgets.
- **Case tracking**: Implemented using pure responsive grids and CSS widgets.
- **Alert center**: Implemented using pure responsive grids and CSS widgets.
- **Executive reports**: Implemented using pure responsive grids and CSS widgets.

## Naming Standards & Best Practices
- **Isolation**: All custom animations and element variables are isolated using `--primary` and specific dashboard selectors to prevent scope bleeding.
- **Accessibility**: Includes ARIA compliance labels on functional buttons and visual gauges.
- **Responsiveness**: Utilizes CSS grid auto-fit systems to stack columns naturally on mobile device viewports without custom frameworks.
