# Stagger Rating Stars

## 1. What does this do?
Creates an interactive 5-star rating component where each star animates in with a staggered pop-in effect on page load. When you hover over the stars, they bounce up in sequence with smooth scale and elevation animations.

## 2. How is it used?
Apply `.rating-stars` to a flex container with `.star` elements:

```html
<div class="rating-stars">
  <span class="star star-1">★</span>
  <span class="star star-2">★</span>
  <span class="star star-3">★</span>
  <span class="star star-4">★</span>
  <span class="star star-5">★</span>
</div>
