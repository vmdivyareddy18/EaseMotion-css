# CSS Micro Shake Tooltip

Pure CSS animated tooltip utilizing a Micro Shake interaction designed for SaaS Showcase layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Micro Shake animation
- SaaS showcase styling
- CSS custom properties
- Glassmorphism cards
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--tooltip
--duration
--ease
```
