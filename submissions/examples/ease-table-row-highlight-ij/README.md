# Ease Table Row Highlight Component

An elegant, micro-interactive data table rows enhancement using fluid CSS state transitions.

## Features
- **Fluid Visual Transitions**: Features a smooth color shifts blended with scaling logic (`cubic-bezier`).
- **Complete Theme Flexibility**: Configured strictly via cleanly isolated `:root` variables.
- **Responsive Architecture**: Packaged neatly inside a horizontal overflow context block safely accommodating mobile views.
