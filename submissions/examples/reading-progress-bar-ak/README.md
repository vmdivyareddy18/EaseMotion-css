# Reading Progress Bar

## What does this do?
A fixed progress bar at the top of the page that fills as the user scrolls through an article, showing the percentage of content read.

## How is it used?
Open demo.html in a browser and scroll down. The thin purple bar at the top fills progressively, and a percentage indicator appears at the bottom-right.

## Why is this useful?
Demonstrates a scroll-position tracking pattern commonly used on blogs and documentation sites to give readers a visual cue of remaining content.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
