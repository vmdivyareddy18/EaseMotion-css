📤 Scroll Reveal Features

  • 📈 translateY(20px) + opacity(0) → 0 on scroll
  • 👁️ IntersectionObserver triggers class
  • ⏱️ Stagger-friendly with nth-child delays
  • ⚡ Speed variations (fast, normal, slow)
  • 🎨 Works on any element
  • 💫 Smooth CSS transitions
