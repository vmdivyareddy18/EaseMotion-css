# ease-card-news

A modern news article card component with breaking/category badge, image hover zoom, and headline underline interaction.

---

## ✨ Features

- Large header image with hover zoom
- Breaking/category badge
- Pulsing animation for breaking news
- Headline with hover underline (`ease-hover-underline`)
- Source and timestamp display
- Clean card layout
- Pure HTML + CSS

---

## 🚀 Usage

```html
<article class="ease-card-news breaking">
    ...
</article>
```
