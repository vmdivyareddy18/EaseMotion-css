# Glow Timeline

Resolves #42444.

### What does this do?
Adds a Glow Timeline component to EaseMotion CSS tailored for Corporate/Dark mode designs. It uses a gradient line and glowing markers that utilize the `ease-pulse` animation to draw attention. The content cards use a glassmorphic effect and `ease-hover-pop` for interactivity.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. It works perfectly on dark backgrounds. Modify the `animation-delay` on `.timeline-item`s to sequence the initial fade-in.

### Why is it useful?
Provides an elegant way to display corporate roadmaps, company history, or project milestones. The glowing pulse animations keep a static list feeling active and modern.
