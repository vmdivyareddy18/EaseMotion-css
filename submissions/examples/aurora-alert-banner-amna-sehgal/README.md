# Aurora Alert Banner

A responsive Aurora-inspired alert banner featuring an animated gradient glow, glassmorphism styling, and a prominent call-to-action for SaaS products, blogs, and modern landing pages.

## What is included

* Responsive HTML structure for the Aurora Alert Banner.
* CSS-only animated aurora gradient and glow pulse effects.
* Glassmorphism-inspired banner with accessible semantic markup.
* Call-to-action button with hover and keyboard focus states.
* Pure CSS implementation with no JavaScript required.
