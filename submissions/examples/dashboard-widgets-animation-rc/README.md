# Dashboard Widgets Animation Collection

1. What does this do?

Provides a collection of animated dashboard widgets including statistics cards, progress indicators, activity panels, notifications, and loading states.

2. How is it used?

Open `demo.html` and interact with the dashboard widgets to preview their animations and motion effects.

3. Why is it useful?

It demonstrates practical dashboard UI motion patterns commonly used in analytics platforms, admin panels, and SaaS applications.
