# Fade In Maintenance Page

A pure HTML and CSS animated maintenance page component built for EaseMotion CSS.

## Features

- Smooth fade-in entrance animation
- Travel app inspired design
- Responsive layout
- Clean maintenance message UI
- No JavaScript dependency

## Usage

Add the component:

```html
<section class="maintenance-page">

  <div class="maintenance-card">
    <h1>Under Maintenance</h1>
    <p>Please check back soon.</p>
  </div>

</section>
