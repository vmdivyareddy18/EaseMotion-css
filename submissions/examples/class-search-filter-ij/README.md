# Component: EaseMotion Class Search

This submission implements a searchable filter for EaseMotion animation classes for issue **#13625**.

## Description

A searchable card grid displaying all available EaseMotion CSS classes grouped by type (Entrance, Exit, Emphasis, Loading, Modifier, Hover). Users can filter classes by name or type in real-time. Useful as a quick reference for developers.

## Acceptance Criteria

- Search input that filters in real-time
- Grid of 12+ animation class cards
- Each card shows class name and type
- Filters by class name and type
- Responsive grid layout
