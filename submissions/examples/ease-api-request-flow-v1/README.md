# Ease API Request Flow

## Overview

Ease API Request Flow is a reusable visualization component that illustrates how a request travels from the client through an API, server, and database using animated data packets built entirely with HTML and CSS.

---

## Features

- 🌐 API request visualization
- 🟢 Animated data packets
- 📦 Client, API, Server, Database nodes
- ✨ Hover interactions
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Client node
- API Gateway node
- Server node
- Database node
- Animated request packets
