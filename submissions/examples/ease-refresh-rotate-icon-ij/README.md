# Refresh Rotate Icon

Refresh icon that spins on click with a smooth 360° rotation. Demonstrates a simple one-time spin, a loading state with continuous spin and success checkmark, and an auto-refreshing indicator.

## Features

- Smooth 360° rotation on click using CSS transform transition
- Continuous spin keyframe for loading state
- Success checkmark fades in on completion
- Three demo variations: simple, loading, auto
- Customizable via root CSS variables

## Usage

Toggle `.rri-spin-once` for a single rotation, `.rri-spin-continuous` for infinite spin, and `.rri-check-show` to reveal the checkmark. CSS handles all transitions and keyframes.
