# Subscription Renewal Reminder Card

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern subscription renewal reminder card featuring countdown timers, billing details, renewal progress visualization, and action buttons.

## Features

- Subscription reminder UI
- Renewal countdown boxes
- Billing summary section
- Auto-renew status indicator
- Animated progress bar
- Renewal urgency badge
- CTA buttons
- Glassmorphism styling
- Floating animation
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="renewal-card">
  Subscription Content
</div>
```

## Main Classes

```css
.renewal-card
.countdown
.billing-info
.progress-bar
.progress-fill
.primary-btn
.secondary-btn
```
