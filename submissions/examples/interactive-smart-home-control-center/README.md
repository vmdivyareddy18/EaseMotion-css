# Animation: Interactive Smart Home Control Center

A modern smart-home dashboard inspired by Apple Home, Google Home, and SmartThings. It combines room management, energy monitoring, security tracking, and device control into a unified interface.

## Features

- Pure HTML & CSS
- Smart device cards
- Energy overview
- Security center
- Responsive dashboard layout
- Glassmorphism styling
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="room-card">
    <div class="room-icon">💡</div>
    <h3>Living Room</h3>
</div>
```

## Why is it useful?

Smart-home interfaces are becoming increasingly common across consumer technology products. This example demonstrates how complex dashboard layouts, status indicators, and interactive cards can work together to create a modern control center experience.
