# Animated Floating Label Inputs

A clean, modern, and fully responsive **Form Input Component with Floating Labels** that provides elegant user feedback and clear context. This example demonstrates how to combine **EaseMotion CSS** for smooth form entrances and validation state animations with custom CSS for the floating label effect and border transitions.

## 🚀 Features

- **Floating Label Animation**: Labels smoothly float up and shrink when the input is focused or contains a value, using CSS transforms and transitions.
- **Staggered Form Entrances**: Each form field fades in sequentially using `ease-fade-in-up` and `ease-delay-*` for a cascading load effect.
- **Animated Input Borders**: A colored border expands from the center when an input is focused, providing clear visual feedback.
- **Icon Animations**: Input icons scale up slightly when the field is focused for tactile feedback.
- **Smooth Focus States**: Inputs highlight with accent colors and subtle shadows when focused.
- **Success State Animation**: After form submission, a success message appears with a bouncy checkmark icon using `ease-bounce-in`.
- **Multiple Input Types**: Supports text, email, tel, select, and textarea inputs with consistent styling.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing and spacing.
