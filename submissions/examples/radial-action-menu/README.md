# CSS-only Radial Action Menu

A modern floating radial action menu built using only HTML and CSS.

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Responsive layout
- Smooth circular expansion animation
- Hover scaling effects
- Floating Action Button (FAB)
- Easy to customize

## 📁 Files

```
demo.html
style.css
README.md
```

## 🚀 How to Use

1. Open `demo.html` in any modern browser.
2. Click the center floating button.
3. The action buttons expand in a circular layout.
4. Click again to collapse the menu.

## 🎨 Customization

You can easily modify:

- Button colors
- Menu radius
- Animation duration
- Icons
- Number of menu items
- Background gradient

## 📚 Technologies Used

- HTML5
- CSS3
- CSS Transforms
- CSS Transitions
- Flexbox

## 📷 Preview

A floating action button expands into six animated circular action buttons positioned around the center, creating a clean and modern radial navigation menu.
