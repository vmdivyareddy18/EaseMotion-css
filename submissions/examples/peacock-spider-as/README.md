# Peacock Spider

A pure CSS and HTML peacock spider performing its courtship dance: the rainbow abdominal fan snaps up, the side flaps unfurl and the third legs wave overhead.

## What it shows

- The famous fan built from stacked hard stop radial gradients in concentric rings of colour, with a fine repeating conic gradient for the hair rays
- Fan, side flaps and legs all firing on one timeline so the display raises and folds as a single performance
- Eight jointed legs and two raised arms, each parking its own angle in a custom property so shared keyframes drive them
- The four forward facing eyes of a jumping spider from layered radial gradients
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`, leaving the fan raised.
