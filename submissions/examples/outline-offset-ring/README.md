# outline-offset-ring

Uses `outline-offset` to create focus rings and decorative effects. Demonstrates standard positive offsets, negative offsets for inner outlines, and multi-ring combinations.

## Usage

Add `class="focus-ring"` to any focusable element, or apply decorative ring classes. Negative offsets create inset outlines that appear inside the element. Stack multiple rings using `box-shadow` combined with outlines.

## Browser Support

`outline-offset` is supported in all modern browsers. Negative values work universally. Double/triple ring effects use `box-shadow` as fallback for older browsers.
