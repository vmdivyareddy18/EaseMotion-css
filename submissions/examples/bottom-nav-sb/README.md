# Bottom Navigation Bar

A mobile-first fixed bottom navigation bar inspired by iOS tab bars.

## Features

- Supports 3-5 icon and label tabs.
- `.ease-active` highlights the current tab.
- Uses thumb-friendly tap targets.
- Includes hover and focus-visible states.
- Keeps the navigation pinned to the bottom of the app shell.

## Usage

Open `demo.html` to preview the bottom navigation. Use `.ease-bottom-nav` around `.ease-bottom-nav-item` links, then add `.ease-active` to the current route.
