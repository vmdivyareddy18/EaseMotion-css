# Ease Aurora Ripple Buttons

## Overview

A premium button collection featuring animated ripple effects, aurora gradients, and glassmorphism styling using only HTML and CSS.

## Features

- Ripple hover animation
- Aurora gradient buttons
- Glassmorphism effect
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS
