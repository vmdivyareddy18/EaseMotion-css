# Turtle Crawling

1. What does this do? A CSS-only turtle that slowly crawls back and forth with a domed shell.

2. How is it used? Place .turtle inside .scene. The body is a green ellipse; the shell is a darker semi-circle on top.

3. Why is it useful? Uses a very long duration (4s) to convey the slow pace of a turtle.
