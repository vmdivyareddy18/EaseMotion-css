# Animated Radio Button (Issue #18257)

A custom-styled radio button with a smooth expanding inner circle animation on check. Uses CSS `::after` with a scale transition.

## Usage

Open `demo.html` in a browser. Click between radio options to see the animation.

## Files

- `demo.html` — HTML5 page with radio inputs
- `style.css` — Dark theme styles with CSS transition
