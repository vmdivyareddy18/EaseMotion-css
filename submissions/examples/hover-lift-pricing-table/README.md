# Hover Lift Pricing Table

A responsive pricing table demo with three plan cards, hover lift motion, focus-visible button feedback, and a highlighted popular tier.

## Files

- `demo.html` contains the pricing table markup and accessible plan labels.
- `style.css` contains the responsive layout, card lift animation, and button focus styles.

## Usage

Open `demo.html` in a browser and hover or focus each pricing card to see the lift effect.
