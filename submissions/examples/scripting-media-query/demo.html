Adaptive Component

Uses @media (scripting) for progressive enhancement.

JavaScript is detected

This component shows enhanced styling when JavaScript is available and a clean fallback when scripting is disabled.

How it works

  • scripting: enabled — Adds interactive hover effects, smooth transitions, and enhanced visual states.
  • scripting: none — Provides a functional, static experience. All core content remains accessible.
  • scripting: initial-only — Applies styles for pages that only run script during page load.
