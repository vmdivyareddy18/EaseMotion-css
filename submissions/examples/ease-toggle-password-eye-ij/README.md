# Toggle Password Eye

A password visibility toggle with animated eye icon morph, input type switch, and strength indicator.

**Directory:** `ease-toggle-password-eye-ij`

## Features
- Eye open/close animation via SVG path opacity morph
- Input fades between dot/password text on type switch
- Visual strength indicator per field
- Multiple example fields (strong, weak, empty)
- Focus border highlight

## CSS Custom Properties
| Property | Default | Description |
|---|---|---|
| `--tpe-duration` | `0.3s` | Animation duration |
| `--tpe-eye-color` | `#64748b` | Eye icon default color |
| `--tpe-active-color` | `#3b82f6` | Eye icon active color |
| `--tpe-input-bg` | `#1e293b` | Input background |
| `--tpe-radius` | `10px` | Border radius |

## Usage
Open `demo.html` in a browser. Click the eye icon on any field to toggle password visibility.
