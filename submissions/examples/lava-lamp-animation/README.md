# Lava Lamp Animation

A lightweight CSS-only lava lamp animation utility with floating organic blobs.

## Features

- Morphing organic blob shapes
- Smooth floating movement
- Multiple blobs with different speeds
- Ambient decorative animation
- Pure CSS implementation

## Usage

```html
<div class="lava-container">
  <div class="blob blob1"></div>
  <div class="blob blob2"></div>
  <div class="blob blob3"></div>
</div>
```

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable ambient motion effect
- Pure CSS visual animation
