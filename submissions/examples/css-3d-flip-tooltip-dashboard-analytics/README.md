# CSS 3D Flip Tooltip – Dashboard Analytics Layout

## Overview

A responsive Dashboard Analytics tooltip component built entirely with HTML and CSS. The component features a smooth 3D flip animation, interactive KPI cards, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth 3D Flip tooltip animation
- Dashboard Analytics interface
- Interactive KPI cards
- Responsive layout
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--surface`
- `--background`
- `--text`
- `--muted`
- `--tooltip-bg`
- `--tooltip-text`
- `--radius`
- `--duration`
- `--easing`
- `--flip-angle`
- `--scale`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on any KPI card to display the animated 3D Flip Tooltip. Colors, animation timing, spacing, shadows, and flip effects can all be customized using the CSS variables defined in `style.css`.
