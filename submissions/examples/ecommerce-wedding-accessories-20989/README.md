# App Landing: Bridal & Wedding Accessories E-commerce (`ecommerce-wedding-accessories-20989`)

This proposal provides a complete, copy-paste ready B2C landing page template designed for an elegant, romantic bridal accessories brand (veils, jewelry, hairpieces), built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A bridal brand requires a soft, dreamy, and highly sophisticated aesthetic. This template achieves a romantic feel using a delicate color palette of whites, creams, blush, and rose gold accents. The typography pairs a classic serif (`Playfair Display`) with a clean, light sans-serif (`Lato`).

The layout includes:
- A subtle top Announcement Bar.
- A sticky Navigation with a frosted glass (blur) effect.
- A Hero section featuring a CSS-animated abstract lace/tulle overlay and a frosted content box.
- A 3-column "Curated for the Bride" category grid with simulated accessory icons.
- A Split-layout highlighting a featured collection (pearls) with scattered CSS pearl accents.
- A "Real Brides" masonry-style gallery with elegant hover overlays.
- A dark-mode Customization Service CTA.
- A Customer Reviews section with pagination dots.
- A Newsletter/Checklist download form.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use extremely slow, graceful `cubic-bezier(0.25, 1, 0.5, 1)` timings. This creates a floating, dream-like pacing:
- `.ease-fade-up` / `.ease-slide-up` (Configured for very slow, sweeping text entrances (1.5s - 1.8s) to feel romantic and unhurried).
- `.ease-reveal-right` / `.ease-reveal-left` (Uses a `clip-path` inset animation to elegantly unveil sections).
- `.ease-zoom-in` (Applied to the hero background and custom CTA for a slow, cinematic zoom).
- `.ease-hover-glow` (Custom interaction for the logo, adding a soft rose gold text-shadow on hover).
- `.ease-hover-float` (Applied to buttons and category cards for a gentle, weightless lift).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the elegant layout, the frosted glass effects, the abstract CSS lace/pearl simulations, and the typography styling. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the graceful entrance animations. The design is fully responsive and requires zero JavaScript to execute the animations or hover states.
