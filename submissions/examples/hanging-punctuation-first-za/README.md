# CSS Hanging Punctuation Quotes

A CSS demo of hanging-punctuation property allowing opening punctuation marks to hang outside the text edge for cleaner alignment in quotes and lists.

## Files

- `demo.html` - Blockquotes and lists comparing default vs hanging punctuation
- `style.css` - hanging-punctuation: first, serif typography

## Usage

Open `demo.html` to see how hanging punctuation improves text alignment.
