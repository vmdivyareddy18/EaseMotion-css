# Profile Dashboard Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to create a responsive user profile dashboard.

---

## Components

- User Profile Card
- Avatar
- Statistics Cards
- Recent Activity
- Edit Profile Button

---

## Folder Structure

```
submissions/examples/profile-dashboard-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to test responsiveness.
3. Observe the applied EaseMotion CSS animations.

---

## Features

- Responsive dashboard
- Statistics cards
- Activity section
- Uses existing EaseMotion CSS animation classes
