# Professional Services: Management Consulting (`professional-services-consulting-21033`)

This proposal provides a complete, copy-paste ready B2B professional services landing page designed for a high-tier strategy or management consulting firm, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A top-tier consulting site (e.g., McKinsey, BCG, Bain) demands an aesthetic of authority, trust, and clarity. This template achieves a corporate, results-driven look using a strong serif/sans-serif font pairing (Playfair Display & Inter), a strict navy/white/slate color palette, and extremely deliberate, unhurried CSS entrance animations.

The layout includes:
- A commanding hero section with a simulated architectural background and staggered text entrances.
- A client logo trust bar.
- A 4-column services/expertise grid.
- A high-contrast case studies section highlighting numerical results.
- A side-by-side methodology/approach layout.
- A team profile gallery featuring grayscale-to-color hover effects.
- A clean, dual-pane corporate inquiry form with accessible bottom-border focus states.
- A structured, multi-column corporate footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (Provides the smooth, deliberate pacing essential for corporate sites)
- `.ease-slide-right` (Used consistently on section headers to guide the user's eye)
- `.ease-expand` (Applied to divider lines under headings for a sophisticated, animated accent)
- `.ease-hover-lift` (Standard interaction for primary buttons, service cards, and case studies to create a tactile, premium feel)
- `.ease-focus-border` (Applied to form inputs for a minimal, modern focus state that shifts the bottom border color)
- `.ease-hover-text` (For clean, subtle navigation and link hovers)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, typography, and corporate styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.
