# Issue 18306 - Strikethrough

A line that draws across text from left to center using ::after pseudo-element with scaleX transition.

## Usage

Open `demo.html` and click the text to toggle the strikethrough effect.

## Files

- `demo.html` - Demo page with clickable strikethrough text
- `style.css` - Styles with scaleX transition on ::after
