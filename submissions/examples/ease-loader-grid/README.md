# ease-loader-grid

A 3x3 grid of dots that pulse in diagonal wave sequence.

## Usage

```html
<div class="ease-loader-grid">
  <span></span><span></span><span></span>
  <span></span><span></span><span></span>
  <span></span><span></span><span></span>
</div>
```

## CSS Variables

| Variable | Default | Description |
|---|---|---|
| `--ease-loader-size` | `12px` | Size of each dot |
| `--ease-loader-color` | `#6366f1` | Color of the dots |
| `--ease-loader-speed` | `1.2s` | Animation duration |
| `--ease-loader-gap` | `4px` | Gap between dots |
