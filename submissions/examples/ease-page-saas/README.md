# Full SaaS Landing Page (`ease-page-saas`)

A polished, high-converting landing page template designed for SaaS products, featuring analytics dashboard mockups, animated stat counters, and enterprise-grade styling.

## 🚀 Features & EaseMotion Showcase

This template uses a clean, professional aesthetic while leaning heavily on subtle `.ease-*` animations to make the page feel modern and alive:

- **Animated Stat Counters**: Implements `.ease-count-up` paired with a small `IntersectionObserver` script to create dynamic numbers that count up from 0 when they enter the viewport.
- **Alternating Feature Rows**: Uses `.ease-slide-left` and `.ease-slide-right` on feature text and visuals to create a cohesive reading rhythm as the user scrolls.
- **Floating Dashboard Elements**: Uses the `.ease-float` infinite animation on abstract UI widgets around the hero mockup to add depth and continuous motion.
- **Interactive Pricing**: Leverages `.ease-hover-lift` and `.ease-hover-scale` on pricing cards and CTA buttons to encourage user interaction.
- **Staggered Grids**: Employs staggered `animation-delay` utilities on the integration logo grid and security badges for satisfying waterfall reveal effects.

## 🛠️ Usage

This demo is entirely self-contained. You can open `demo.html` directly in your browser. All required EaseMotion CSS classes have been defined at the top of the `style.css` file for immediate testing. 

If using this as a template for a real project, simply migrate the HTML structure into your application and ensure your global CSS bundle includes the `ease-*` utility classes.
