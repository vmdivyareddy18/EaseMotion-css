Class Usage Statistics

Use the included scripts to scan all demo.html files and generate a popularity report of EaseMotion classes.

How to run

bash scan_classes.sh        # Unix / Git Bash
or
powershell -File scan_classes.ps1   # Windows PowerShell

The script scans submissions/examples/*/demo.html, extracts all class names, and prints a sorted frequency report.

Data‑driven insights into community usage of EaseMotion classes.
