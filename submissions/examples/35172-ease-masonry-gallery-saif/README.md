# Ease Masonry Gallery

A modern **responsive Masonry Gallery** built using **pure HTML & CSS** for the EaseMotion CSS component library.

The component automatically arranges gallery items of different heights into a clean masonry layout with smooth entrance animations, elegant hover effects, responsive breakpoints, and dark mode support.

---

## ✨ Features

- Responsive Masonry Layout
- Pure HTML & CSS
- Zero JavaScript
- Smooth staggered fade-in animation
- Hover lift effect
- Image zoom animation
- Gradient caption overlay
- Rounded modern card design
- CSS Variables
- Dark Mode Support
- Mobile-first responsive design
- Lightweight and reusable

---

## 🚀 Usage

1. Copy the component folder into your project.
2. Link the stylesheet.

```html
<link rel="stylesheet" href="style.css">
```

3. Copy the gallery HTML.

```html
<div class="ease-masonry-gallery">

    <article class="gallery-card">
        <img src="image.jpg" alt="Gallery Image">

        <div class="overlay">
            <h3>Gallery Title</h3>
        </div>

    </article>

</div>
```

## 📱 Responsive Breakpoints

| Device | Columns |
|---------|---------|
| Desktop | 4 |
| Tablet | 3 |
| Mobile | 2 |
| Small Mobile | 1 |

---

## 🎨 Customization

Modify the CSS variables inside `:root`.

```css
:root{

    --bg:#f8fafc;

    --surface:#ffffff;

    --text:#0f172a;

    --radius:18px;

    --transition:.4s;

}
```

You can easily customize:

- Background colors
- Border radius
- Shadows
- Animation speed
- Overlay colors
- Typography
