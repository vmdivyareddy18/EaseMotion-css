# ease-image-zoom-hover

A lightweight, reusable CSS utility that smoothly zooms an image on hover while keeping overflow hidden, creating a polished gallery-style interaction.

## Use Cases
- Galleries
- Product images
- Portfolio websites
- Blog thumbnails

## Usage

Wrap your image in a container with the `ease-image-zoom-hover` class:

```html
<div class="ease-image-zoom-hover">
  <img src="demo.jpg" alt="Description">
</div>
