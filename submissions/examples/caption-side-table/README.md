# caption-side-table

## Overview

Demonstrates the `caption-side` CSS property for positioning table captions above (`top`) or below (`bottom`) the table. Includes styled captions with colored backgrounds and a comparison grid showing both positions on identical table data.

## Usage

Open `demo.html` in any browser. Three sections show: a top-positioned caption on a quarterly revenue table, a bottom-positioned caption on a headcount table, and a side-by-side styled comparison of both positions with identical two-column data.

## Features

- Caption positioned at top (default behavior) and bottom
- Styled captions with background, font weight, and color
- Side-by-side split comparison of both positions
- Responsive grid layout
- `prefers-reduced-motion` support
- Dark theme with blue accent and green section headers
