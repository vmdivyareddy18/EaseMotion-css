# Ease Strikethrough

Animated strikethrough effect that draws a line from left to right across text on hover.

## Features

- Pure HTML & CSS
- Uses pseudo-element animation
- Smooth left-to-right line drawing
- Works on text, links, pricing, tasks, and headings
- Responsive demo examples

## Usage

```html
<span class="ease-strikethrough">
    Hover Me
</span>
```

The strike line is generated using `::after` and animated with CSS transforms.
