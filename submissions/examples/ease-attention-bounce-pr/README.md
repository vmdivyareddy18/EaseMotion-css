# Animation: ease-attention-bounce

### What does this do?
Triggers an elegant, physics-tapered vertical bounce oscillation to subtly guide user focus toward persistent UI controls like chat vectors or primary inputs.

### How is it used?
Apply the class `.ease-attention-bounce` to any layout component, optionally adjusting bounds inline via `--ease-bounce-height` and `--ease-bounce-speed` runtime tokens.

### Why is it useful?
It provides an organic, friction-free alternative to sudden modal interruptions, executing smoothly on the compositor layer without triggering browser layout updates.
