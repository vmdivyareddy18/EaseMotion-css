# Smart Home Devices Demo

A responsive **Smart Home Devices** landing page built using EaseMotion CSS.

## Features

- Hero section
- Products by Room (Bedroom, Kitchen, Living Room)
- Compatibility Badges (Alexa, Google Home, Apple HomeKit)
- 3-Step Setup Guide
- Customer Reviews
- Bundle Deals
- Newsletter Signup
- Responsive Layout
- Hover Effects
- EaseMotion CSS Animations
