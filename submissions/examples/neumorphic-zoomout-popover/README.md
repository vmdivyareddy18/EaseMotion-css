# Neumorphic Zoom-Out Popover

A pure CSS animated popover with a smooth zoom-out interaction inspired by Neumorphic Soft UI design.

## Features

- Pure CSS implementation
- Smooth zoom-out animation
- Neumorphic shadows
- Responsive layout
- Keyboard accessible
- No JavaScript required
- CSS variable customization


## Usage

```html
<button class="soft-button">

 Open Details

 <div class="popover">
   Content
 </div>

</button>
