# Splash / Loading Screen

## What does this do?
CSS-animated splash screens with logo reveals, ring/dot/pulse spinners, progress indicators, and transition-out effects.

## How is it used?
```html
<div class="sh2-splash sh2-logo-splash"><div class="sh2-logo"><h2>Brand</h2><div class="sh2-loader-bar"><div class="sh2-loader-fill"></div></div></div></div>
```

## Why is it useful?
Provides engaging loading experiences without heavy animation libraries. Multiple splash styles (logo, spinner, progress, minimal). SVG progress ring. CSS-only transitions.
