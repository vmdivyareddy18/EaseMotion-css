# Daily Habit Tracker

## What does this do?
A habit tracking app with checkable daily habits, completion statistics, streak counting, add/delete habits, and localStorage persistence.

## How is it used?
Open demo.html in a browser. Click the circle to mark habits done, add new habits with the input field, delete unwanted ones, and track your daily streak.

## Why is this useful?
Demonstrates a complete CRUD habit tracker with local storage persistence, streak logic, and responsive card-based UI.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript (localStorage)

## Preview
Open demo.html directly in your browser.
