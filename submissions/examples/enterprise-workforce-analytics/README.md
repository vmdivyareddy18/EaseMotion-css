# Enterprise Workforce Analytics

## Overview
Build an HR and workforce management dashboard.

This project implements the Phase 115 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Employee statistics**: Built with glassmorphism and CSS grid/flexbox.
- **Department analytics**: Built with glassmorphism and CSS grid/flexbox.
- **Performance widgets**: Built with glassmorphism and CSS grid/flexbox.
- **Workforce insights**: Built with glassmorphism and CSS grid/flexbox.
- **Responsive dashboard**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
