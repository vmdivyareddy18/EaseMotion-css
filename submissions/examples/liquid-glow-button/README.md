# Liquid Border Glow & Pop Button

An animation-first, highly expressive premium button effect that introduces a rotating fluid gradient border, matching drop-shadow glow, and an elastic scale-up pop upon hover.

## Classes Added
- `.ease-btn-liquid-glow` : Applies the base setup, active states, and hover micro-animations.

## How to use
Stack it straight onto standard EaseMotion buttons:
```html
<button class="ease-btn ease-btn-liquid-glow">Hover Me</button>
<button class="ease-btn ease-btn-pill ease-btn-liquid-glow">Pill Variant</button>
