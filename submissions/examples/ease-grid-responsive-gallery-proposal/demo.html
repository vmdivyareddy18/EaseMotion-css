CSS Grid Auto Gallery

A fully responsive, zero-media-query grid layout proposal for core/layout.css.

This utility uses CSS Grid's auto-fit and minmax() functions to create a perfectly responsive gallery that automatically calculates column counts based on available screen width. No breakpoints or media queries required!

.grid-auto-gallery

Resize your browser window. Watch as the cards reflow automatically to fill the space without leaving gaps or squishing too tight.
