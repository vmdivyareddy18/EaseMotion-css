# Floating Underline Reveal Text

## Description

A text hover effect featuring an animated underline reveal and floating motion.

## Features

- Pure CSS animation
- Smooth underline reveal
- Lightweight implementation
- Great for navigation links

## Files

- demo.html
- style.css
- script.js

## Usage

Open demo.html and hover over the text.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
