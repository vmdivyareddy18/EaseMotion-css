`overscroll-behavior-none-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/overscroll-behavior-none-tm/style.css">
<div class="overscroll-behavior-none-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .overscroll-behavior-none: prevents scroll chaining

## Impact

Stops scroll propagation to parent containers
