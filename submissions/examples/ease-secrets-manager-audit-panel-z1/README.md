# Ease Secrets Manager Audit Panel

## Overview

A responsive HTML and CSS component for monitoring secret rotation, expiration, and access activity.

## Features

- Secret rotation status
- Expiration tracking
- Recent access information
- Security status badges
- Responsive layout
- Pure HTML & CSS

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable animation-first component for security operations, secrets management, DevSecOps dashboards, and cloud infrastructure monitoring.
