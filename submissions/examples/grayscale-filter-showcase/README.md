# Grayscale Filter Showcase - EaseMotion CSS

A beautiful, interactive component showcase presenting the grayscale filter transition classes.

## Features Included

- `.ease-filter-grayscale`: Applies `filter: grayscale(100%) !important` to an element.
- `.ease-filter-grayscale-0`: Applies `filter: grayscale(0%) !important` to an element.
- Smooth transitions built-in to demonstrate transition performance and visual utility.

## How to Run

1. Open `demo.html` directly in any modern web browser.
2. Hover over the cards to observe the interactive color-to-grayscale and grayscale-to-color transition states.

## Why is it useful?

Grayscale filter states are widely used in UI design to create visually clean inactive or default card states that colorize on hover, as well as providing support for accessibility and styling options for avatars, photography, or logo grids.
