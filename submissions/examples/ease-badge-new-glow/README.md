# ease-badge-new-glow – "NEW" Badge Glow Pulse

A promotional badge (e.g., “NEW”, “Sale”) that draws attention with a continuous, subtle box‑shadow glow animation.

## EaseMotion classes used
- **Layout:** ease-container, ease-flex, ease-items-center, ease-justify-center, ease-min-h-screen
- **Background:** ease-bg-gray-100
- **Typography:** ease-text-3xl, ease-font-bold, ease-text-gray-500, ease-font-semibold, ease-text-blue-700, ease-text-lg, ease-text-sm, ease-text-gray-400, ease-font-bold, ease-text-blue-600
- **Spacing:** ease-mb-6, ease-mb-10, ease-p-4, ease-mb-1, ease-mb-2, ease-mt-8
- **Components:** ease-card, ease-overflow-hidden
- **Hover Effects:** ease-hover-shadow-xl
- **Animation:** ease-fade-in, ease-delay-200, ease-delay-500, ease-transition

## How it works
- The .badge-new-glow element is absolutely positioned on a product card.
- It uses a @keyframes badge-glow that oscillates the ox-shadow from transparent to a bright blue glow and back.
- The animation loops continuously and pauses for reduced‑motion users.

## How to use
1. Add the class adge-new-glow to any badge element.
2. Copy demo.html and style.css into your project.
3. Ensure the path to easemotion.css is correct (usually from core/).
