# ease-meeting-scheduler-card (#36810)

Meeting scheduler card with interactive time-slot selection.

## CSS Custom Properties

| Variable             | Default    | Description                   |
|----------------------|------------|-------------------------------|
| `--card-bg`          | `#1e1e32`  | Card background               |
| `--slot-hover-bg`    | `#2d2d4a`  | Time-slot hover background    |
| `--selected-color`   | `#7c5cfc`  | Selected slot & confirm color |
| `--select-duration`  | `0.25s`    | Selection animation duration  |

## Features

- 3-column grid of 30-min time slots
- Click-to-select with visual feedback
- Confirm button (disabled until a slot is picked)
- Selection confirmation message
- Dark theme card layout
