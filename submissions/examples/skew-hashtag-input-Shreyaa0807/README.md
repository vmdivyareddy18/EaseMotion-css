# Skew Hashtag Input

A pure-CSS reusable hashtag input field featuring a sharp, admin-panel-inspired skew transition on focus.

## Features
- **Pure CSS:** Completely operational without requiring JavaScript.
- **Accessible:** Uses semantic elements and interactive `:focus-within` states.
- **Responsive:** Fluid layout that can easily scale to fit grid configurations.

## File Structure
- `demo.html` - Live implementation demo.
- `style.css` - Component keyframes and layout rules.
