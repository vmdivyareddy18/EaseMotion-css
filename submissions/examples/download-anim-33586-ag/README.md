# Download State Animation

An interactive download button demonstrating multi-state transitions built using only HTML/CSS.

---

## What is Included

1. **`demo.html`**: A clean user card containing the interactive Download Button, progress tracks, loading states, and completion messages.
2. **`style.css`**: Button dimensions, state offsets, active loaders, spin indicators, and checkmark highlights.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Click the **Download Package** button.
3. Verify that the button switches to the loading screen showing progress fill lines.
4. After 3 seconds, confirm the button successfully transitions to the green complete status.
