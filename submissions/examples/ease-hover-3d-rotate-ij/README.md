# Hover 3D Rotate

A card that rotates in 3D on hover with configurable perspective.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--card-bg` | `#2d2d44` | Card background color |
| `--rotate-x` | `15deg` | X-axis rotation on hover |
| `--rotate-y` | `15deg` | Y-axis rotation on hover |
| `--transition-speed` | `0.3s` | Animation speed |
