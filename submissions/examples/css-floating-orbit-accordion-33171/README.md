# CSS Floating Orbit Accordion for Product Catalog Layouts

## Overview

This project showcases a modern **Product Catalog Accordion** featuring a smooth **Floating Orbit** animation built entirely with HTML and CSS. The component presents expandable product information cards with an animated orbit indicator, making it ideal for product showcases, e-commerce catalogs, and feature comparison sections without requiring JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- CSS-only Floating Orbit animation
- Product catalog inspired accordion layout
- Responsive design for desktop and mobile
- Smooth expand and collapse transitions
- Animated orbit indicator for active items
- Keyboard-accessible using native `<details>` and `<summary>`
- CSS custom properties for easy customization
- No JavaScript or external libraries required

### Animation

- The active accordion item displays a glowing orbit indicator.
- The orbit follows a smooth circular motion using CSS keyframes.
- Accordion content expands with a soft fade-in effect.
- Cards slightly elevate on hover to enhance depth.
- The expand icon transitions from a plus to a minus for improved user feedback.

## Customization

The component uses CSS custom properties for quick styling.

```css
:root{
    --primary:#4f46e5;
    --accent:#06b6d4;
    --surface:#ffffff;
    --text:#1f2937;
    --muted:#6b7280;
    --radius:18px;
}
```

You can customize:

- Primary and accent colors
- Background gradients
- Typography
- Card spacing
- Border radius
- Orbit size and animation speed
- Shadows
- Hover effects
- Accordion transitions

## Why This Component?

This component demonstrates how lightweight CSS animations can enhance traditional product catalog accordions by introducing a subtle floating orbit effect that draws attention to active content. It remains responsive, accessible, reusable, and dependency-free, making it suitable for product catalogs, e-commerce websites, and marketing interfaces.
