# Ease Crystal Command Dock

## Overview

A premium floating command dock inspired by desktop operating systems, featuring crystal glass styling and animated gradient highlights using only HTML and CSS.

## Features

- Crystal glass effect
- Animated conic gradients
- Floating hover motion
- Responsive dock layout
- Lightweight component
- Pure HTML & CSS

## Folder Structure

```text
ease-crystal-command-dock-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Scales smoothly across desktop, tablet, and mobile devices.

Provides a reusable animated command dock with elegant motion, clean structure, and modern styling suitable for dashboards, developer tools, and productivity interfaces.
