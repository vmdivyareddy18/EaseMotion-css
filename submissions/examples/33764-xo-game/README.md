# XO Game

A simple Tic-Tac-Toe (XO) game built using HTML, CSS, and JavaScript.

## Features

- 🎮 Two-player gameplay
- ✅ Win detection
- 🤝 Draw detection
- 🔄 New Game / Reset button
- 📱 Responsive design
- 🌙 Clean and modern UI

## Project Structure

```
xo-game/
├── demo.html
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Navigate to `submissions/33764-xo-game`.
3. Open `demo.html` in any modern web browser.

No installation or dependencies are required.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
