# Organization Tree

An org chart with hierarchy reveal animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--tree-bg` | `#1a1a2e` | Page background |
| `--node-color` | `#2d2d44` | Card background |
| `--line-color` | `#4a4a6a` | Connector line color |
| `--reveal-duration` | `0.4s` | Reveal animation speed |
