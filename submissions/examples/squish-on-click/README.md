# Squish On Click

**What does this do?**
Creates a fun, physical reaction where an element compresses downwards (gets shorter and wider) when clicked, simulating rubber or jelly.

**How is it used?**
Apply the class to an interactive element like a button:
```html
<button class="squish-button">Click Me!</button>
```

**Why is it useful?**
It provides immediate, playful visual feedback to user interactions, reinforcing the idea that UI elements are tangible objects. This aligns perfectly with EaseMotion's highly interactive, animation-first principles.
