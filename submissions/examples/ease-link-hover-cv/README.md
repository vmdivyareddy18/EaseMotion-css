# Hover Underline Link (cv variant)

## What does this do?
A collection of 5 different animated link underline effects triggered on hover. Includes color variants and pure CSS animations for that premium website feel.

## How is it used?

### Basic Usage:

```html
<!-- Slide Right (default direction) -->
<a href="#" class="ease-link-hover-cv ease-link-hover-cv-slide-right">Learn More</a>

<!-- Slide Left -->
<a href="#" class="ease-link-hover-cv ease-link-hover-cv-slide-left">Back to Home</a>

<!-- Expand from Center -->
<a href="#" class="ease-link-hover-cv ease-link-hover-cv-expand-center">Read Documentation</a>

<!-- Gradient Wave -->
<a href="#" class="ease-link-hover-cv ease-link-hover-cv-gradient">Explore Features</a>

<!-- Double Line -->
<a href="#" class="ease-link-hover-cv ease-link-hover-cv-double">View Portfolio</a>
