# SLA Renewal Countdown

An intermediate EaseMotion CSS example for customer success dashboards. Countdown blocks rise into place and softly tick to highlight time left before an SLA renewal window.

## Files

- `demo.html` - Semantic renewal countdown markup.
- `style.css` - Responsive card layout, staggered block entrance, and subtle tick animation.

## Usage

Open `demo.html` in a browser to preview the animated SLA renewal countdown.
