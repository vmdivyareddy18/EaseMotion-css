# CSS-only Morphing Floating Action Button (FAB) Menu

A modern Floating Action Button (FAB) that expands into multiple action buttons using only HTML and CSS.

## Features

- Pure HTML & CSS
- No JavaScript
- Smooth open/close animation
- Morphing "+" to "×" icon
- Staggered menu animation
- Responsive design

## Folder Structure

```
morphing-fab-menu/
├── demo.html
├── style.css
└── README.md
```

## Preview

- Floating circular action button
- Click to reveal multiple action icons
- Smooth scaling and fade animation
- Hover effects on each action button

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Transforms
- CSS Transitions
- Hidden Checkbox Toggle

## Usage

1. Open `demo.html` in any modern browser.
2. Click the floating "+" button.
3. The action menu expands with smooth animations.

## Customization

You can easily customize:

- Background colors
- FAB size
- Number of menu items
- Animation duration
- Position (bottom-left, bottom-right, etc.)
- Icons or text inside action buttons

## Browser Support

Works in all modern browsers that support CSS Transforms and Transitions.
