ease-video-play-button-pulse

A circular play button centered over a video thumbnail pulses gently and continuously — the familiar invitation-to-click pattern from video embed UIs everywhere.

Usage

/* 1. Link the stylesheet */
<link rel="stylesheet" href="style.css" />

/* 2. Standard video thumbnail + centered play button pattern */
<div class="video-thumb">
  <img src="thumbnail.jpg" alt="..." />
  <div class="video-thumb-overlay"></div>
  <button class="play-btn ease-video-play-button-pulse" aria-label="Play video">
    <svg viewBox="0 0 24 24"><polygon points="5,3 21,12 5,21"/></svg>
  </button>
</div>

/* 3. Customise the pulse color or speed */
.my-play-btn {
  --ease-play-pulse-color: #6c63ff;     /* ring color */
  --ease-play-pulse-scale: 1.18;        /* deeper pulse */
  --ease-play-pulse-duration: 2.2s;     /* slower */
}
