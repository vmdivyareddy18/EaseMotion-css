# Progress Bar Fill

A setup progress example with an animated fill and a subtle shine pass across the active bar.

## Files

- `demo.html` - progress panel markup with ARIA progressbar values.
- `style.css` - fill animation, shine effect, and reduced-motion fallback.

## Highlights

- Uses `role="progressbar"` with explicit min, max, and current values.
- Animates the fill with `scaleX` from the left edge.
- Disables movement for reduced-motion users.
