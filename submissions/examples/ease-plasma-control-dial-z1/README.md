# Ease Plasma Control Dial

## Overview

A futuristic control dial featuring rotating plasma rings, an animated pointer, and a glassmorphism center panel built with pure HTML and CSS.

## Features

- Rotating plasma rings
- Animated control pointer
- Glassmorphism center panel
- Responsive layout
- Lightweight implementation
- Pure HTML & CSS
