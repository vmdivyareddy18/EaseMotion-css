Contributor Onboarding Automation

Welcome first-time contributors automatically using GitHub Actions and Discord webhooks. Celebrate every contributor's first merged pull request with an instant community message.

Explore Workflow

Why Contributor Onboarding?

Recognizing first-time contributors helps build a welcoming open-source community. By combining GitHub Actions with a Discord webhook, repositories can automatically celebrate contributors whenever their first pull request is merged.

Automation Workflow

Contributor Opens Pull Request
↓
Pull Request Merged
↓
GitHub Action Runs
↓
Checks First-Time Contributor
↓
Sends Discord Webhook 🎉

GitHub Action Workflow 

name: First Contributor Welcome

on:
  pull_request:
    types:
      - closed

jobs:
  welcome:
    if: github.event.pull_request.merged == true
    runs-on: ubuntu-latest

    steps:
      - name: Check First Merge
        run: echo "First merged PR detected"

      - name: Send Discord Webhook
        run: echo "Posting welcome message..."

Discord Webhook Payload 

{
  "username": "EaseMotion Bot",
  "avatar_url": "https://example.com/bot.png",
  "content": "🎉 Welcome @newContributor! Congratulations on your first merged Pull Request. Thank you for contributing to EaseMotion CSS!"
}

Key Features

Automatic Detection

Detects when a contributor's first Pull Request has been merged.

GitHub Actions

Uses GitHub Actions to automate contributor onboarding.

Discord Integration

Sends a welcome message through a Discord webhook automatically.

Community Recognition

Celebrates first-time contributors and encourages future contributions.

No Manual Work

The complete workflow runs automatically after every merged Pull Request.

Developer Friendly

Easy to understand, customize, and integrate into open-source repositories.

Benefits of Automated Contributor Onboarding

  • 🎉 Welcomes first-time contributors automatically.
  • 🚀 Improves contributor engagement and retention.
  • 🤖 Eliminates manual welcome messages.
  • 💬 Strengthens the open-source community through recognition.
  • 🔔 Integrates seamlessly with GitHub Actions and Discord.
  • 🌍 Encourages more developers to continue contributing.
