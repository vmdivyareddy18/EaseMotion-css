# Kinetic Typography Split-Text Reveal

A typography showcase page demonstrating staggered letters rising and rotating into position.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting split-text letters, delay triggers, and description details.
2. **`style.css`**: Height clamps, letter clip masks, transform translate and rotate keyframes, stagger values, and media checks.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the letters **K I N E T I C** sliding up and rotating into position sequentially.
3. Click **Replay Animation** to confirm the staggered letter draw sequence repeats correctly.
