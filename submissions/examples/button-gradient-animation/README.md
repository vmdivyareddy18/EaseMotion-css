# Button Gradient Animation on Hover 

#### 1. **What does this do?**
Provides four interactive gradient animation button styles that activate smooth, vibrant color transitions when hovered, creating eye-catching visual feedback with rotating, sliding, and pulsing gradient effects.

#### 2. **How is it used?**
```html
<!-- Gradient Slide Animation -->
<button class="gradient-btn btn-slide">Slide Gradient</button>

<!-- Gradient Glow Animation -->
<button class="gradient-btn btn-glow">Glow Gradient</button>
```

#### 3. **Why is it useful?**
These button animations embody EaseMotion CSS's philosophy by delivering smooth, professional animations that enhance user engagement without sacrificing performance. Each animation style leverages CSS gradients and keyframe animations to create a polished, modern interaction pattern that gives users satisfying visual feedback on hover, making interfaces feel more responsive and interactive while maintaining accessibility and minimalist elegance.
