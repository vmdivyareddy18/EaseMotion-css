# Profile Cards Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used with responsive profile cards.

---

## Included Components

- Avatar
- Name
- Role
- Biography
- Social Icons
- Follow Button

---

## Folder Structure

```
submissions/examples/profile-cards-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html` in your browser.
2. Resize the browser window to test responsiveness.
3. Observe the animation effects applied to the profile cards.
