# Catapult Date Picker Component

A pure CSS, high-performance glassmorphic date picker layout built for EaseMotion CSS. It implements a unique catapult expansion layer bounce interaction natively without relying on JavaScript runtimes.

## Features
- **Zero JavaScript:** Utilizes native checkbox state management hacks.
- **Glassmorphism Design Theme:** Multi-layered blur layers leveraging `backdrop-filter` rules.
- **Catapult Animation Stack:** Custom keyframe layers mapped closely to EaseMotion physics engine patterns (`--ease-bounce`).
- **Responsive Geometry:** Scales down elegantly to smaller screen formats.

## Structure Map
- `demo.html` - Live demonstration sandbox container.
- `style.css` - Component structural layout, aesthetics, and animation code.
