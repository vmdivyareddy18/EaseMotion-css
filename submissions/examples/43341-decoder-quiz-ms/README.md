# Animation Class Decoder Quiz

Resolves #43341.

### What does this do?
Provides an interactive multiple-choice quiz component that tests users on their knowledge of EaseMotion CSS class names and their corresponding effects.

### How is it used?
The component uses Vanilla JS to cycle through a set of questions, injecting options dynamically and providing immediate visual feedback (green for correct, red for incorrect) alongside a "Next Question" workflow.

### Why is it useful?
Gamification improves documentation engagement. By offering a quick quiz, contributors can learn the core EaseMotion class naming conventions in a fun and interactive way.
