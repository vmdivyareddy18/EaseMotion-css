# Icon Morph

## What does this do?
An icon morphing effect where two icon states transition smoothly between shapes (e.g. play/pause, menu/close, plus/minus).

## How is it used?
Apply the .morph-icon class to a container. Toggle the .morphed class to trigger the morph transition between two icon states.

## Why is this useful?
Provides smooth state transition for interactive controls like play/pause, expand/collapse, and menu/close without icon font swaps or JavaScript-heavy solutions.
