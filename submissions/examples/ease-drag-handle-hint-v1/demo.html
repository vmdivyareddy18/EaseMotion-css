How It Works

On :hover of the .drag-card, the .drag-handle (≡ icon) gets a subtle opacity pulse and horizontal micro-bounce using a @keyframes animation. The effect signals to the user that the element can be dragged. The animation plays once on each hover entry using animation-fill-mode: forwards.
