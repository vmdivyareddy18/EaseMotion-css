# Animated Trash Bin Toss

## Overview

Animated Trash Bin Toss is a lightweight CSS animation that simulates a file being deleted into a trash bin.

## Features

- CSS-only animation
- File drop effect
- Trash bin bounce effect
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="trash-bin-toss">
  <span class="file">📄</span>
  <span class="bin">🗑️</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
