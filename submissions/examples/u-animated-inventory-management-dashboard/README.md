# Animated Inventory Management Dashboard

A modern **Inventory Management Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example provides an interactive interface for monitoring inventory, warehouse operations, stock availability, and product categories with smooth CSS-only animations.

## Features

* 📦 Inventory overview cards
* ✅ In Stock indicator
* ⚠️ Low Stock monitoring
* ❌ Out of Stock tracking
* 🏷️ Product category cards
* 📊 CSS-only weekly stock movement chart
* 📈 Warehouse inventory status
* 🚚 Warehouse activity feed
* ✨ Hover interactions and smooth animations
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

## Customization

All colors, spacing, and theme settings are controlled using CSS variables.

Example:

```css
:root{
    --primary:#2563eb;
    --success:#10b981;
    --warning:#f59e0b;
    --danger:#ef4444;
    --surface:#ffffff;
    --bg:#f5f7fb;
}
```
