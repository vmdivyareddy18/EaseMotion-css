Frequently Asked Questions

Find answers to common questions about EaseMotion CSS.

EaseMotion CSS is a lightweight, utility-first animation library that provides pre-built classes for common animations like fades, slides, zooms, and pulses. It allows you to add smooth, professional animations to your website without writing custom keyframes or JavaScript.

You can easily install EaseMotion CSS via CDN by adding the following link to your HTML <head>:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/SAPTARSHI-coder/EaseMotion-css@main/easemotion.min.css" />

Alternatively, you can download the source files from GitHub and include them in your project.
