 
# Stripe Scroll Animation

Alternating color stripe background animates its position on scroll using repeating-linear-gradient linked to scrollY.

## Files

- demo.html - Interactive demo
- style.css - Stripe animation styles
- README.md - Documentation

## Classes

| Class | Colors |
|-------|--------|
| `stripe-primary` | Blue → Purple |
| `stripe-secondary` | Green → Teal |
| `stripe-warm` | Orange → Red |
| `stripe-cool` | Cyan → Blue |
| `stripe-neon` | Pink → Purple |

## Features

- Diagonal stripes via repeating-linear-gradient
- background-position linked to scrollY
- Dynamic stripe motion effect
- Multiple color variants

## Demo

👉 Open [demo.html](./demo.html) and scroll up/down to see stripes move!
