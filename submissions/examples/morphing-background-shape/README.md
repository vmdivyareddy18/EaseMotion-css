# Morphing Background Shape

## What it does

Creates a fluid, organic blob shape that continuously morphs behind content. Perfect for hero sections and decorative backgrounds.

## How to use it

<div class="ease-morph-bg"></div>

Make sure parent has position: relative.

## Variants

- ease-morph-fast - 3s morph cycle
- ease-morph-slow - 12s morph cycle
- ease-morph-rainbow - Cycling gradient colors
- ease-morph-purple - Purple-only theme
- ease-morph-small - 200x200px
- ease-morph-large - 600x600px

## Why it fits EaseMotion CSS

- Pure CSS, zero JavaScript
- Fluid organic animation
- Multiple speed and color variants
- GPU-accelerated border-radius
