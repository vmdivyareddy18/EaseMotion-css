# Animated Employee Attendance Dashboard

A modern **Employee Attendance Dashboard** built using **pure HTML and CSS** for the EaseMotion CSS project. This example showcases attendance monitoring, employee status, department summaries, and activity tracking with smooth CSS-only animations.

## Features

* 👨‍💼 Employee attendance overview cards
* ✅ Present employee tracking
* ❌ Absent employee monitoring
* ⏰ Late check-in indicator
* 🏢 Department attendance summary
* 📊 CSS-only weekly attendance chart
* 👤 Employee profile cards
* 🟢 Online/Offline status indicators
* 📋 Daily attendance activity feed
* ✨ Smooth hover interactions
* 🌙 Automatic Dark/Light mode support
* 📱 Fully responsive layout
* 🎨 Easy customization using CSS variables
* 🚫 No JavaScript
* 🚫 No external libraries

* `ease-pulse`
* `ease-slideIn`

* `ease-statusPulse`
