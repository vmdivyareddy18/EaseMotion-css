# Neon Glow Card – Card with animated neon border glow

> A production-ready neon glow card component with an animated border that pulses and rotates with a vibrant glow effect, fully customizable colors, speed, and intensity. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Animated neon border** – Rotating gradient glow effect
- ✅ **Customizable colors** – Pick any color
- ✅ **Adjustable speed** – Control animation speed
- ✅ **Adjustable intensity** – Control glow brightness
- ✅ **Toggle glow** – Enable/disable glow effect
- ✅ **Toggle animation** – Pause/resume animation
- ✅ **6 color variants** – Blue, Green, Pink, Purple, Gold, Rainbow
- ✅ **Keyboard shortcuts** – Ctrl+G (toggle glow), Ctrl+A (toggle animation), Ctrl+R (reset), Ctrl+C (random color)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
