# CSS Micro-Shake Accordion (Dashboard Analytics Layouts)

A pure CSS responsive accordion designed for operations monitoring and server health dashboards. Features a tactile "micro-shake" horizontal wobble animation on active cluster card headers.

## Features
- **Tactile Micro-Shake Interaction**: Subtle horizontal vibration triggered on hover/focus to suggest activity and alert status.
- **Cyber Command Center Aesthetics**: Designed with a Dark Slate and Electric Neon Cyan color scheme matching operations or K8s dashboard monitors.
- **Pure CSS checked Logic**: Built using native checkbox inputs for zero-JS toggle support.
- **Exposed custom properties**: Intensity, timing, and borders can be configured using CSS variables.

## Files
- `demo.html`: Live interactive demonstration of the cluster node dashboard.
- `style.css`: Stylesheet containing the custom node cards, status badges, and keyframe shaking animation.
