Modern Text Underlines

Using text-underline-offset and text-decoration-thickness to create highly legible, animated typography links that don't cut through descenders (like g, p, y).

The "Descender-Safe" Hover

Notice how this typography link pushes the line down below the letters "p" and "y". On hover, the thickness animates up.

The "Marker Highlight" Effect

By increasing the thickness and shifting the offset upward, we can create a highlight marker effect using pure text-decoration.

The Wavy Error State

Offsets work beautifully with other styles like wavy decorations to give them room to breathe.
