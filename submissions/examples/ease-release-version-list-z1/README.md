# Ease Release Version List

## Overview
A CSS-only component for displaying software release versions and status badges.

## Features
- Animated status badges
- Responsive card layout
- CSS variables
- Hover effects
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for release dashboards, changelog pages, and developer tools.
