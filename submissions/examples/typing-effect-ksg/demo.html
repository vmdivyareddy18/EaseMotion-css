Typing Effect with Cursor

Pure CSS character-by-character reveal using steps() animation and a blinking cursor — no JavaScript required for the animation.

Demo 1

Hero Headline

The platform to

The platform to Build. Ship. Scale.

Uses .typing-effect with --typing-steps: 19 (matching the character count of "Build. Ship. Scale."). The steps() timing function snaps the width in discrete jumps — one per character — creating the typewriter illusion.

Three .code-line elements each use a separate @keyframes with a matching steps(N) value and staggered animation-delay to create a sequential code-writing effect. The cursor appears only on the currently active line.
