# 📅 CSS Timeline Motion

A collection of beautiful, pure CSS animated timelines for showcasing processes, histories, and roadmaps.

## 🚀 Features

### Timeline Types (4 variations)
1. **Vertical Timeline** - Classic top-to-bottom layout
2. **Horizontal Timeline** - Left-to-right layout
3. **Alternating Timeline** - Zigzag pattern with alternating sides
4. **Gradient Timeline** - Colorful gradient dots and line

### Animation Features
- Staggered fade-in animations
- Pulsing dot effects
- Hover lift effects
- Smooth transitions
- Responsive design

## 📂 Files
- `demo.html` - Main HTML structure
- `style.css` - All timeline animations
- `readme.md` - Documentation

## 🛠️ How to Use

### Vertical Timeline
```html
<div class="timeline vertical">
    <div class="timeline-item">
        <div class="timeline-dot"></div>
        <div class="timeline-content">
            <span class="date">2024</span>
            <h3>Event Title</h3>
            <p>Event description here.</p>
        </div>
    </div>
</div>
