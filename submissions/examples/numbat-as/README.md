# Numbat

A pure CSS and HTML numbat licking termites out of a fallen log in eucalyptus woodland.

## What it shows

- White back stripes from a repeating linear gradient, masked so they fade out over the pale belly
- A long tongue driven by scaleX from a snout side origin, disappearing into the termite hole
- The hole drawn above the animal so the tongue tip is genuinely swallowed by it
- A bushy tail textured with a repeating conic gradient
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
