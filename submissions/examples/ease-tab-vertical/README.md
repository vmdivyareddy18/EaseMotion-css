# Ease Tab Vertical

This demo implements a **vertical tab navigation** component with a sliding active indicator and smooth content transitions.

## 🧠 How It Works
- Tabs are stacked vertically on the left.
- The active indicator slides vertically to the selected tab.
- Content panels transition smoothly on change.
- Uses the radio input technique for accessibility and simplicity.

## 🧩 Files
- `demo.html` — main demo with tab logic.
- `style.css` — styling and animations.
- `README.md` — documentation and usage notes.

## 🚀 Usage
1. Open `demo.html` in a browser.
2. Click each tab to see the indicator slide and content transition.
3. Customize colors and layout as needed.

## 🧰 Technologies
- HTML5
- CSS3 (transitions, transforms)
- JavaScript (indicator animation)
