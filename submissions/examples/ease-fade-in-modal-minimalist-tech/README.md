# CSS Fade-In Modal – Minimalist Tech

A clean, responsive CSS-only fade-in modal inspired by modern Minimalist Tech interfaces. The component features a smooth fade and scale animation, soft backdrop blur, and elegant action buttons without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth fade-in and scale animation
- Minimalist Tech inspired design
- Responsive layout
- Backdrop blur overlay
- Accessible checkbox-based interaction
- No JavaScript or external libraries

## Preview

A centered modal with a subtle fade and scale entrance, designed for confirmations, notifications, and lightweight dialog interfaces.

## Files

- `demo.html`
- `style.css`

## How It Works

- Uses a hidden checkbox to control the modal state.
- Labels act as the trigger to open and close the modal.
- CSS transitions animate the overlay opacity and modal appearance.
- Fully responsive using flexible sizing and media queries.

## Use Cases

- Success confirmation dialogs
- Project completion notifications
- Login or signup prompts
- Settings confirmation
- Portfolio and SaaS interfaces
- Minimalist dashboard applications

## Browser Support

Works in all modern browsers supporting:

- CSS Transitions
- Flexbox
- Backdrop Filter (with graceful fallback)
- CSS Transforms
