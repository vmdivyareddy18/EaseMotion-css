# Animated Tabs Navigation Component

## Overview

A modern CSS-only tabs navigation component with smooth transitions, hover effects, active indicators, and responsive layouts.

## Features

- Smooth tab hover animations
- Active tab highlighting
- Glowing active indicator
- Responsive tab layout
- Smooth content fade-in effect
- Lightweight CSS-only implementation

## Files

- demo.html – Demonstrates the animated tabs navigation
- style.css – Contains tab styling and animation effects
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in any modern browser to view the animated tabs component.

Hover over the tabs to experience smooth interaction effects.
