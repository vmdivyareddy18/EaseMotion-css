# CSS 3D Perspective Tilt Popover

A pure CSS animated popover featuring a smooth 3D perspective tilt effect for modern SaaS layouts.

## Features

- CSS-only animation
- 3D perspective tilt transition
- Responsive design
- Keyboard accessible with focus-within
- Customizable CSS variables

## Usage

```html
<div class="tilt-popover">

<button class="tilt-trigger">
SaaS Features
</button>

<div class="tilt-content">
Content here
</div>

</div>
