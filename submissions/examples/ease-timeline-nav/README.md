# Timeline Nav – Vertical navigation with timeline

> A production-ready timeline navigation component that combines a vertical timeline with content sections, providing an elegant way to navigate through chronological content. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Vertical timeline navigation** – Chronological navigation
- ✅ **Interactive timeline items** – Click to navigate
- ✅ **Progress indicator** – Shows current position
- ✅ **Keyboard shortcuts** – Arrow keys, Home, End
- ✅ **Smooth transitions** – Elegant content switching
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
