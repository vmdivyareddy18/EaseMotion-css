# Food Delivery Operations Dashboard Showcase

## What does this do?

Provides an animated food-delivery dashboard with order tracking, delivery monitoring, restaurant operations insights, and customer satisfaction analytics.

## How is it used?

Open `demo.html` directly in a browser and explore the food-delivery operations interface.

## Why is it useful?

It demonstrates practical food-delivery platform UI patterns and combines multiple animation techniques into a realistic order-management and delivery-monitoring experience.
