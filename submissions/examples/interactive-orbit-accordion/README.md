# Interactive Orbit Accordion

A modern **Pure CSS Floating Orbit Accordion** designed for interactive interface layouts.

## ✨ Features

- Pure HTML & CSS
- No JavaScript
- Responsive Design
- Keyboard Accessible
- Floating Orbit Animation
- CSS Custom Properties
- Futuristic Purple/Blue Theme

## 📂 Files

- demo.html
- style.css

## ⚙️ Customization

Modify these CSS variables:

```css
:root{
    --primary:#7c3aed;
    --secondary:#38bdf8;
    --duration:8s;
}
```

## 🚀 Usage

Simply open `demo.html` in any modern browser.

No dependencies required.
