Integrate in minutes

Install the SDK, initialize the client with your environment key, and start gating features behind flags. Lightweight, thread-safe, and built for production traffic.

JavaScript Python Go Java

<1ms Avg latency

app.js JavaScript 
import { createFlagClient } from '@flagflow/js-sdk';

// Initialise with your environment key
const client = createFlagClient({
  apiKey: process.env.FLAGFLOW_KEY,
  user: {
    id:    user.id,
    email: user.email,
    plan:  'pro',
  },
});

// Gate a feature behind a flag
if (client.isEnabled('new-dashboard')) {
  renderNewDashboard();
} else {
  renderLegacyDashboard();
}

// Get a variant for A/B testing
const variant = client.getVariant('checkout-flow');
// → 'control' | 'treatment-a' | 'treatment-b'
