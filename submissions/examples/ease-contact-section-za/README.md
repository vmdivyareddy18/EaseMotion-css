# CSS Contact Section Component

## What does this do?
A complete contact page layout with a multi-field contact form, contact information cards, social media links, and an interactive map placeholder section.

## How is it used?
```html
<form class="ct-form">
  <div class="ct-row">
    <div class="ct-field">
      <label>Name</label>
      <input type="text" placeholder="Your name">
    </div>
  </div>
  <div class="ct-field">
    <label>Message</label>
    <textarea rows="4"></textarea>
  </div>
  <button class="ct-submit">Send</button>
</form>
```

## Why is it useful?
Creates a professional contact page for business websites. Features a responsive two-column layout with form fields for name email phone subject and message including select dropdown and checkbox for privacy consent. Contact info cards display location email phone and office hours with icons. Social media link row with hover effects and a dashed map placeholder section. Fully responsive with grid stacking on mobile viewports. CSS custom properties enable easy dark theme branding.
