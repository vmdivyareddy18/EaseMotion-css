# Feature Flag Rollout Slider States

## What does this do?

Shows paused, canary, gradual, complete, and rollback states for feature flag rollout.

## How is it used?

```html
<div class="rollout-row is-half">
  <span>Gradual</span>
  <b class="track"><i></i></b>
</div>
```

## Why is it useful?

Feature flag tools need clear rollout status visuals for staged releases and rollback decisions.
