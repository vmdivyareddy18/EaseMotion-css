Animation Definition

The transition starts by scaling slightly upward (to 1.1) to grab attention before collapsing down to 0 and vanishing:

.ease-pop-out {
  animation: easePopOut 0.35s cubic-bezier(0.34, 1.56, 0.64, 1) forwards;
}

@keyframes easePopOut {
  0% { transform: scale(1); opacity: 1; }
  25% { transform: scale(1.1); opacity: 1; }
  100% { transform: scale(0); opacity: 0; }
}
