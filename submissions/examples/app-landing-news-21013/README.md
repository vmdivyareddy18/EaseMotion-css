# App Landing: News Aggregator (`app-landing-news-21013`)

This proposal provides a complete, copy-paste ready B2C landing page designed for a mobile news aggregator app, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A news or information product needs to project authority, neutrality, and clarity. This template implements a classic "journalism" aesthetic adapted for modern SaaS/App landing pages. It features a high-contrast black-and-white color scheme, sharp borders, and classic typography combining Playfair Display (serif) for headings and Inter (sans-serif) for functional UI elements.

The layout includes:
- A prominent Hero section featuring a detailed, pure CSS mobile app mockup (complete with sliding news feed items and a floating "Breaking News" alert).
- A trusted publisher logo strip.
- A 3-column features grid outlining personalization and bias indicators.
- A "Bento Box" style CSS grid for topics categories.
- A sample article preview layout with a CSS-rendered "Bias/Perspective Analysis" meter.
- A clean, 2-tier pricing architecture.
- Bold App Store download Call-To-Action (CTA) sections.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (Used with a snappy, standard bezier curve `0.16, 1, 0.3, 1` to give the site a crisp, responsive, and authoritative feel)
- `.ease-slide-left` (Applied sequentially to the news feed items inside the CSS phone mockup to simulate app loading states)
- `.ease-zoom-in` (Used on the topics bento grid cards and the floating breaking news alert)
- `.ease-hover-lift` (Applied to all buttons and app store CTAs to encourage click-through)
- `.ease-hover-scale` (Applied to social icons and the main logo)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, bento grid, phone mockup geometry, typography, and sharp styling. The standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/app-landing-news-21013/` directory to avoid modifying core files directly and to ensure zero deletions.*
