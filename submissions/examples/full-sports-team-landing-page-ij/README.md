# Sports Team/Club Landing Page

1. What does this do? A complete landing page for a sports team or club featuring a bold hero with team branding and next-match CTA, player roster grid with cards, match schedule/results table, ticket package options, team news feed, merchandise store preview, and a fan membership CTA. Motion is driven entirely by EaseMotion CSS classes.

2. How is it used? Open demo.html in a browser. The page uses EaseMotion CSS entrance classes (ease-fade-in, ease-slide-up, ease-bounce-in) and scroll-reveal classes (ease-reveal, ease-reveal-slide-up) to animate sections into view. Player cards use ease-hover-lift-shadow and ease-hover-glow for interactive feedback. The reveal.js script enables scroll-triggered animations.

3. Why is it useful? Sports team sites need to convey energy, team spirit, and urgency (next match, ticket sales). The staggered card animations and bold hero motion create an immersive fan experience that mirrors the excitement of live sports, while the reveal-based scrolling keeps content discovery engaging.
