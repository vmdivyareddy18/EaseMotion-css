# Jello Drawer

A responsive CSS-only navigation drawer featuring a smooth jello-style entrance animation inspired by organic nature design patterns.

## What is included

* Responsive HTML structure for the Jello Drawer component.
* CSS-only slide-in and jello animation using keyframes.
* Pure CSS toggle implementation with no JavaScript required.
* Accessible navigation drawer with semantic HTML and keyboard-friendly controls.
* Nature-inspired styling suitable for modern dashboards, landing pages, and websites.
