React Integration Guide

Official developer documentation demonstrating how to install, import, and dynamically trigger EaseMotion CSS animations in React projects.

1. Installation

To start using the framework in your React application, install the package from npm:

terminal 
npm install easemotion-css

2. Importing Files

Import the stylesheet package inside your project's root entry file (e.g., src/main.jsx or src/index.js):

src/main.jsx 
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App.jsx';

// Import EaseMotion CSS core tokens and base styles
import 'easemotion-css/core/variables.css';
import 'easemotion-css/core/base.css';
import 'easemotion-css/core/animations.css';
import 'easemotion-css/core/utilities.css';

// Import optional pre-built components
import 'easemotion-css/components/buttons.css';
import 'easemotion-css/components/cards.css';

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>,
);
💡
Note on import order: Always load variables.css first. The other modules rely on custom properties defined inside the variables file to trigger colors and easing speeds correctly.

3. Static Class Usage

Apply animations using the JSX-compliant className attribute instead of the standard HTML class attribute:

src/App.jsx 
function App() {
  return (
    <div className="ease-container ease-center">
      <div className="ease-card ease-slide-up ease-delay-100">
        <h2 className="ease-card-title">React + EaseMotion CSS</h2>
        <p>This card slides in gracefully when the component mounts.</p>
      </div>
    </div>
  );
}

4. Dynamic Class Toggling (State-based)

You can dynamically add or trigger animations on interactive elements by bindings React state variables to conditional className evaluations:

InteractiveCard.jsx 
import React, { useState } from 'react';

export default function Card() {
  const [isShaking, setIsShaking] = useState(false);

  const triggerShake = () => {
    setIsShaking(true);
    // Reset state after animation finishes
    setTimeout(() => setIsShaking(false), 500);
  };

  return (
    <div className="ease-flex ease-flex-col">
      <div className={`ease-card ` + 
        (isShaking ? 'ease-shake' : '')
      }>
        Interactive Card
      </div>
      <button 
        className="ease-btn ease-btn-primary"
        onClick={triggerShake}
      >
        Trigger Shake
      </button>
    </div>
  );
}
