# Flip Chip

A responsive **Flip Chip** component built using pure HTML and CSS for the EaseMotion CSS library.

## Features

- Pure HTML & CSS
- No JavaScript
- Portfolio-inspired design
- 3D flip animation
- Responsive layout
- Accessible structure
- Uses EaseMotion CSS variables and `ease-*` animation classes

## Folder Structure

```text
flip-chip-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Technologies

- HTML5
- CSS3

## Animation

- `ease-fade-in`
- 3D flip hover effect
- Responsive and lightweight
