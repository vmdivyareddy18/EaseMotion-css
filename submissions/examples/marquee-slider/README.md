# Hardware-Accelerated Infinite Marquee Slider

An advanced, independent horizontal layout utility designed to display seamless, looping content tracks using synchronized twin translation sequences.

## Global Structural Classes
- `.ease-marquee-container` : Configures the core clipping viewport mask window.
- `.ease-marquee-track` : Instantiates the infinite layout loop stream.
- `.ease-marquee-slide` : Wraps standalone nested asset modules inside the vector field.

## Performance Modifiers
- `.ease-marquee-reverse` : Flips the linear translation direction seamlessly.
- `.ease-marquee-fast` : Adjusts keyframe loop duration to execute a faster crawl cadence.
