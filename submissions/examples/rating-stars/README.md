# Rating Stars Component

## Description

A CSS-only Rating Stars Component that allows developers to display product ratings, service reviews, testimonials, course ratings, and feedback scores in a clean and visually appealing format.

The component supports:

- Full star ratings (e.g., 5/5)
- Half-star ratings (e.g., 4.5/5)
- Review count display
- Read-only/static rating display
- Optional hover-preview version for interactive demonstrations

## Why is this useful for EaseMotion CSS?

Star ratings are one of the most common UI patterns in e-commerce platforms, SaaS products, review websites, portfolios, and testimonials.

Adding a Rating Stars Component demo expands EaseMotion CSS's collection of practical UI examples while maintaining its human-readable and composable design philosophy. The component can also be combined with EaseMotion animation utilities for smooth hover and rating interactions.
