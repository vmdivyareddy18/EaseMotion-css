# Animated Testimonial Card Slider

## Description

A modern testimonial slider component with smooth card animations, profile images, user ratings, and interactive hover effects.

---

## Features

- Smooth slide-up entrance animation
- Horizontal slider layout
- Profile images and customer details
- Star rating display
- Hover lift animation
- Responsive design
- Reduced motion accessibility support

---

## Classes

- `.slider` — Testimonial container
- `.testimonial-card` — Individual review card
- `.rating` — Star rating section

---

## Accessibility

Supports `prefers-reduced-motion` for users who prefer less animation.
