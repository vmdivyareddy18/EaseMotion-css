# Liquid Gooey Loader Animation

A loader animation showcase demonstrating gooey liquid physics using CSS contrast and blur filter values.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting high-contrast loading containers, bubble nodes, and descriptors.
2. **`style.css`**: Contrast filter parameters, blur coefficients, keyframe translate animations, and motion rules.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the two bubble loaders on the dark card.
3. Verify that as they slide past each other, they merge with a realistic liquid-like "gooey" connector bridge.
