# Cascading Cookie Banner

**Issue:** #42533

## Overview

The **Cascading Cookie Banner** is a responsive, Pure CSS cookie consent component for the EaseMotion CSS library. It features a cascading entrance animation inspired by modern music player interfaces while maintaining accessibility and responsiveness.

## Features

- Pure CSS implementation
- No JavaScript required
- Cascading entrance animation
- Uses EaseMotion `ease-*` naming convention
- Responsive layout
- Keyboard accessible
- CSS variables for easy customization
- Beginner-friendly example
