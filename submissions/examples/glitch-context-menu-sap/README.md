# Glitch Context Menu

### What does this do?
Adds a pure-CSS native context dropdown interface themed around agency brutalist layout frameworks that instantly fires a high-speed chromatic text slicing glitch animation loop across option slots during hover actions.

### How is it used?
```html
<details class="context-details-sap">
  <summary class="menu-trigger-sap">AGENCY ACTIONS</summary>
  <ul class="menu-list-sap">
    <li class="menu-item-sap">
      <a href="#" class="glitch-link-sap" data-text="INITIALIZE">INITIALIZE</a>
    </li>
  </ul>
</details>
```
### Why is it useful?
It provides creative developers with a stylized contextual interaction platform that achieves hyper-responsive, digital distortion animation visual accents using optimized string mirroring vectors without importing bloated JavaScript intervals.
