# Issue #14820: Advanced: Full Wedding Planner Landing Page

This folder contains a standalone demonstration of a full Wedding Planner Landing Page as requested in Issue #14820.

## Files
- `demo.html`: The HTML structure for the landing page.
- `style.css`: The CSS styling using advanced and premium aesthetic designs, including soft gradients, elegant typography, and subtle micro-animations.

## How it works
The demo provides a complete layout for a wedding planning service, featuring:
- A hero section with a romantic gradient and call to action.
- A services section highlighting offerings like Venue Selection, Catering, and Decor.
- A gallery/portfolio section using CSS grid.
- A contact/booking form section.
- Premium styling with hover effects, glassmorphism, and responsive design.
