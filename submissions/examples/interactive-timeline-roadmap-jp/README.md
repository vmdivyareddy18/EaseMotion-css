# Interactive Timeline Roadmap Component

A modern roadmap and timeline component built using pure HTML and CSS.

## Features

- Animated vertical roadmap
- Milestone indicators
- Completed, Active, and Upcoming states
- Status badges
- Progress visualization
- Hover interactions
- Pulse animation for active stage
- Responsive layout
- Reusable timeline structure

## Files

- demo.html
- style.css

## Use Cases

- Product Roadmaps
- Startup Growth Timelines
- Educational Journeys
- Open Source Project Plans
- Company Milestones
- Development Schedules

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Responsive Design

## Responsive

Optimized for desktop, tablet, and mobile devices.
