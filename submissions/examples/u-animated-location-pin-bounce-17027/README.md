# Animated Location Pin Bounce

## Overview

A reusable location pin animation featuring a gentle bouncing effect suitable for maps, travel applications, delivery tracking systems, and location-based interfaces.

## Features

- CSS-only animation
- Location marker bounce effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="location-button">
  <span class="location-pin">📍</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
