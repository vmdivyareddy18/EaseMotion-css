# Animated Bug Tracking Board

A modern bug tracking board inspired by GitHub Issues, Jira, and Linear.

## Features

- Multiple workflow columns
- Priority badges
- Status indicators
- Animated bug cards
- Responsive layout
- Hover interactions

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Customization

- Bug titles
- Priority levels
- Workflow stages
- Status colors
- Animation timing

## Use Cases

- Issue tracking systems
- Development dashboards
- Open source projects
- Team collaboration tools
- Project management platforms
