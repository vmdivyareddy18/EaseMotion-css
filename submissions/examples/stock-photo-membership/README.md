# Stock Photo Membership - EaseMotion CSS Example

A premium stock photography membership platform landing page, demonstrating modern UI components using EaseMotion CSS.

## Features

- Premium stock photo aesthetic
- Interactive image collage and category grid
- Responsive pricing cards
- Contributor spotlight section
- Feature showcase with icons
- Smooth scroll animations using core `reveal.js`
- Clean, semantic HTML structure

## Folder Structure

```text
submissions/examples/stock-photo-membership/
├── demo.html     # Main HTML template
├── style.css     # Custom UI styling and color overrides
└── README.md     # Project documentation
```

## EaseMotion Classes Used

- `ease-slide-up`, `ease-slide-in-left`, `ease-slide-in-right`, `ease-zoom-in`
- `ease-reveal`, `ease-reveal-up`, `ease-reveal-left`, `ease-reveal-right`, `ease-reveal-scale`
- `ease-hover-lift`, `ease-hover-lift-shadow`, `ease-hover-scale`
- `ease-card`, `ease-card-glass`, `ease-card-neumorphic`, `ease-card-image`
- `ease-grid`, `ease-grid-cols-1`, `ease-md-grid-cols-2`, `ease-md-grid-cols-3`, `ease-md-grid-cols-4`
- `ease-bg-primary`, `ease-bg-neutral`, `ease-bg-white`

## Browser Compatibility

Compatible with all modern browsers:
- Chrome, Firefox, Safari, Edge

## Responsive Support

Fully responsive across Desktop, Tablet, and Mobile views, switching gracefully from multi-column grids to stacked layouts.
