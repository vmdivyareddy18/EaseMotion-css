# 💣 Minesweeper

A classic Minesweeper puzzle game built with HTML, CSS, and JavaScript. Reveal cells, flag suspected mines, and use number clues to clear all safe cells without detonating any mines!

## 🎮 How to Play

1. Select **difficulty** (Easy, Medium, Hard)
2. Click **"Start Game"** or press **Space** to begin
3. **Left-click** a cell to reveal it
4. **Right-click** (or long-press on mobile) to flag a suspected mine
5. **Numbers** indicate how many mines are adjacent
6. **Clear all safe cells** to win!
7. **Avoid clicking** on mines!

## ⌨️ Controls

| Action | Input |
|--------|-------|
| Reveal Cell | Left-click |
| Flag/Unflag Cell | Right-click |
| Start Game | Click Start or press `Space` |
| Reset Game | Press `R` |

## ✨ Features

- 💣 **Classic Minesweeper Gameplay** – Reveal, flag, and avoid mines
- 📊 **Three Difficulty Levels** – Easy (9×9, 10 mines), Medium (16×16, 40 mines), Hard (30×16, 99 mines)
- ⏱️ **Timer** – Tracks your completion time
- 🏆 **Best Times** – Saved per difficulty in browser local storage
- 🎨 **Visual Feedback** – 3D cell effects, number colors, mine and flag icons
- 📱 **Responsive Design** – Works on desktop and mobile
- 💾 **Auto-save** – Best times persist across sessions
- 🔄 **Reset Functionality** – Start over anytime
