# Weathervane

A pure CSS and HTML rooster weathervane on a barn roof, veering back and forth as gusts blow past.

## What it shows

- The rooster silhouette assembled from layered rounded boxes, a clip-path tail and a border triangle beak, all spinning as one group about the hub
- The four fixed direction pointers with real N, S, E, W letters set through ::before content
- A barn roof and gable cut from a single clip-path polygon
- Wind gusts and clouds streaming across behind the vane
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

The rooster and wind stop under `prefers-reduced-motion: reduce`.
