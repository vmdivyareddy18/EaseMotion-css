# profile-badge-tm

## What does this do?
A pure CSS Profile Badge Avatar component with hover scaling micro-interaction and an online status dot indicator.

## How is it used?
<div class="badge-card">
  <div class="avatar-wrapper">
    <img class="avatar-img" src="avatar.jpg" alt="User" />
    <span class="status-dot"></span>
  </div>
  <div class="badge-info">
    <h4>User Name</h4>
    <p>Role or description</p>
  </div>
</div>

## Why is it useful?
Uses clean flexbox layout with basic CSS transitions (transform: scale, border-radius, box-shadow) for a smooth hover micro-interaction. No JavaScript or state logic needed. Ideal for profile cards, user lists, and comment sections in EaseMotion-powered UIs.
