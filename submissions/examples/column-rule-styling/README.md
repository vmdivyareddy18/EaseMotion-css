## column-rule-styling

Showcases the `column-rule` property for decorative dividers between columns in multi-column layouts.

### Key Features

- Solid, dashed, dotted, and double column rule styles
- Themed rule colors coordinated with drop cap styling
- Width control from subtle (1px) to prominent (4px)

### Usage

`column-rule` follows the same syntax as `border`: `<width> <style> <color>`. The rule sits centered in the `column-gap` and spans the full height of the multi-column container. Rule styles apply uniformly between all columns.
