column-rule — Decorative Column Dividers

Width, style, and color options for column separators in multi-column layouts.

Solid Rule — 2px solid #3b82f6

A solid column rule creates a clean, professional divider between columns. Use a subtle color for readability without distraction.

The column-rule sits in the center of the column-gap. It spans the full height of the multi-column element, matching the tallest column.

You can set width, style, and color independently or use the shorthand column-rule property.

Dashed Rule — 2px dashed #10b981

Dashed rules add a lighter visual touch. The dashes break up the line, making the separator feel less rigid and more organic.

Dashed and dotted rules work well for informal or creative content where a solid line might feel too formal.

Dotted Rule — 3px dotted #f59e0b

Dotted column rules provide the lightest visual weight. They separate columns without drawing too much attention.

The dot size is determined by the rule width. A 3px dotted rule creates more visible dots than 1px.

Double Rule — 4px double #ec4899

Double rules create a distinctive, elegant separator. Two parallel lines add visual interest and work especially well in formal or editorial layouts.

A double rule requires at least 3px width to render properly — the minimum is 3px for two visible lines with a gap between them.

Gradient Effect — 2px solid with Themed Colors

While column-rule does not support gradients directly, you can create visual variety by using different colors per section or by combining rule style with a background gradient on the container.

Each section in this page uses a different rule color to demonstrate how theming works with multi-column layouts.

Choose rule colors that complement your design system for a cohesive look.

column-rule shorthand: <width> <style> <color>. Accepts same values as border.
