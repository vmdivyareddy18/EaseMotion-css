ease-scramble

CSS matrix-style text scramble animation · EaseMotion CSS · Issue #2727

.ease-scramble

HOVER ME

↑ hover to trigger

Matrix decode fires on :hover. Glyphs cycle, then text snaps into place.

.ease-scramble-auto SYSTEM READY

Plays once on page load. Ideal for hero text or loading screens.

.ease-scramble-loop DECODING...

Repeats the scramble every 4 seconds. Great for ambient UI elements.

.ease-scramble + .ease-scramble-fast FAST DECODE

↑ hover to trigger

Speed modifier: 0.5s snappy decode. Stack with any trigger class.

.ease-scramble + .ease-scramble-slow SLOW REVEAL

↑ hover to trigger

Speed modifier: 2s dramatic decode. Perfect for cinematic intros.

.ease-scramble-green MATRIX MODE

↑ hover to trigger

Classic green matrix colour with phosphor glow. Stack with any trigger.

combo · auto + green + slow

WELCOME TO EASEMOTION CSS

Combine classes freely: ease-scramble-auto ease-scramble-green ease-scramble-slow

<!-- Hover trigger -->
<h1 class="ease-scramble" data-text="EaseMotion CSS">
  EaseMotion CSS
</h1>

<!-- Auto on load -->
<span class="ease-scramble-auto" data-text="LOADING SYSTEM">
  LOADING SYSTEM
</span>

<!-- Loop every 4s -->
<div class="ease-scramble-loop" data-text="DECODING...">
  DECODING...
</div>

<!-- Speed modifiers — stack with trigger class -->
<span class="ease-scramble ease-scramble-fast" data-text="QUICK">QUICK</span>
<span class="ease-scramble ease-scramble-slow" data-text="SLOW">SLOW</span>

<!-- Green matrix mode -->
<span class="ease-scramble ease-scramble-green" data-text="MATRIX">MATRIX</span>

<!-- Combine freely -->
<h1 class="ease-scramble-auto ease-scramble-green ease-scramble-slow"
    data-text="WELCOME">WELCOME</h1>
