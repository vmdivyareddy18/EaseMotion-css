# CSS Wobble-Focus Modal for Cyberpunk Neon Layouts

1. **What does this do?**  
   It renders a pure CSS animated modal component featuring a smooth Wobble-Focus entrance transition, designed with a futuristic Cyberpunk Neon aesthetic using glowing accents, vibrant colors, and modern UI effects.

2. **How is it used?**  
   Open `demo.html` in any modern web browser. Click the **Launch Modal** button to display the modal with the Wobble-Focus animation. Close it using the **Close** button, the **×** icon, or by clicking the overlay. The component is built entirely with HTML and CSS, requiring no JavaScript for the animation.

3. **Why is it useful?**  
   It provides a lightweight, responsive, and accessible modal component with zero JavaScript animation overhead. The design is fully customizable through CSS custom properties, allowing developers to easily adjust animation duration, easing, scale, colors, and glow effects while integrating it into cyberpunk-themed websites, developer portfolios, gaming interfaces, dashboards, or futuristic web applications.
