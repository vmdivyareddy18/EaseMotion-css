# Ease Label Edit Inline

A modern **inline label edit** component built using **pure HTML and CSS**. It demonstrates a clean click-to-edit style interface with smooth CSS animations and customizable styling using CSS custom properties.

## Features

- Pure HTML & CSS
- CSS-only animation using `@keyframes`
- Inline edit style interface
- Smooth focus transition
- CSS Custom Properties for easy customization
- Responsive design
- Dark mode support
- Reduced motion support
- No JavaScript required

## Folder Structure

```text
ease-label-edit-inline/
├── demo.html
├── style.css
└── README.md
```

## Customization

Modify the CSS variables in the `:root` section to customize:

- Primary color
- Background color
- Border radius
- Shadow
- Transition speed

## Browser Support

Compatible with all modern browsers.
