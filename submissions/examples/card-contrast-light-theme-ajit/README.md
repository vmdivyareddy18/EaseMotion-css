# Improve Card Contrast and Hover Feedback in Light Theme

## Overview

This submission improves the appearance of cards in the Light Theme by increasing contrast and enhancing hover interactions while preserving EaseMotion CSS's clean and minimal design philosophy.

## Features

- Slightly darker card background
- Subtle border for better separation
- Improved shadow depth
- Smooth lift animation
- Accent-colored border on hover
- Soft hover glow
- Smooth transitions

## Files

```
card-contrast-light-theme-ajit/
├── demo.html
├── style.css
└── README.md
```

## How to Run

Simply open `demo.html` in any modern web browser.

## Hover Effects

- Card lifts upward
- Shadow becomes deeper
- Border changes to accent color
- Smooth transition animation

## Notes

- Only the Light Theme is enhanced.
- Dark Theme remains unchanged.
- No core library files are modified.
