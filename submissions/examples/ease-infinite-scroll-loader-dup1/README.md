# Infinite Scroll Loader – Shows spinner while loading next page

> A production-ready infinite scroll loader component that automatically loads more content as you scroll, with a beautiful animated spinner and comprehensive controls. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Infinite scroll** – Loads content as you scroll
- ✅ **Animated spinner** – Beautiful loading indicator
- ✅ **End of content** – Clear visual indication
- ✅ **Error handling** – Retry on error with click
- ✅ **Toggle loader** – Enable/disable infinite scroll
- ✅ **4 loader variants** – Spinner, Dots, Pulse, Bar
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
