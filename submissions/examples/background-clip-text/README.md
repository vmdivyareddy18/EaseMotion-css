# background-clip-text

Uses `background-clip: text` to render gradient and image backgrounds through text. Multiple gradient styles, image-masked text, and hover transitions.

## Usage

Apply `class="gradient-text"` for linear-gradient text, `class="rainbow-text"` for animated gradients, or `class="image-text"` to clip an image through text. Hover effects trigger gradient shifts.

## Browser Support

`background-clip: text` requires the `-webkit-background-clip` prefix in Safari. Works in Chrome, Firefox 115+, Edge, and Safari 16+.
