background-clip: text

Gradient and image backgrounds clipped to text shapes.

Image as Text

This text masks an image through the letters. The background image shows through each character.

Every character reveals part of the background image, creating a striking typographic effect.

Hover Effects

Hover to shift Hover to reveal Hover to pulse Hover to spin

Usage

.gradient-text {
  background: linear-gradient(135deg, #6366f1, #ec4899);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
}

Combine background-clip: text with color: transparent for the effect to work.
