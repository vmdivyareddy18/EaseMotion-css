# Ease Orbital Hex Grid

## Overview

A futuristic hexagonal dashboard grid featuring floating glassmorphism tiles and a highlighted central node built entirely with HTML and CSS.

## Features

* Hexagonal UI layout
* Floating tile animation
* Highlighted center node
* Responsive grid
* Lightweight CSS
* Pure HTML & CSS

## Folder Structure

```text
ease-orbital-hex-grid-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

* Chrome
* Firefox
* Edge
* Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

## Why it fits EaseMotion CSS

Introduces a reusable hexagonal dashboard pattern with subtle motion and modern CSS techniques suitable for analytics panels, AI dashboards, developer tools, and futuristic interfaces.
