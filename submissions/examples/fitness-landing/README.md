# Vortex Kinetics // High-Performance Training Matrix

A professional, high-intensity landing page template designed for elite athletic training facilities and performance conditioning centers.

## Structure
- `index.html`: The core content and layout structure.
- `style.css`: All design systems, animation engines, and responsive media queries.

## Design Philosophy
- **Performance-First**: Uses CSS Grid and hardware-accelerated animations for smooth transitions.
- **Metric-Driven**: Focuses on data presentation with clean table layouts and pricing matrix blocks.
- **Brand Identity**: High-contrast dark theme with signature red accents to convey intensity and precision.

## How to use
1. Ensure both `index.html` and `style.css` are in the same folder.
2. Open `index.html` in any modern web browser.
