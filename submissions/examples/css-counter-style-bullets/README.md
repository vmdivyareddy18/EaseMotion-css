# CSS @counter-style Custom Bullets Example

## Description
A clean, self-contained educational implementation demonstrating modern CSS `@counter-style` configurations. Showcases how to generate cyclical bullet symbols, hexadecimal numeric prefixes, and custom alphabetic list counter sequences entirely in pure CSS.

## Key Elements Demonstrated
- `@counter-style`: The native API engine creating rule structures.
- `system`: Dictates list iterations (`cyclic`, `numeric`, `alphabetic`).
- `prefix` & `suffix`: Injects custom padding, labels, or decorative character trails around numbers.
