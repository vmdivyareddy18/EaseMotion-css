# Compass Needle Spin

## What does this do?
It animates a compass needle as it spins and settles on a target heading.

## How is it used?
```html
<div class="compass-face" aria-hidden="true">
  <span class="needle"></span>
  <span class="hub"></span>
</div>
```

## Why is it useful?
It fits maps, travel apps, outdoor dashboards, and onboarding screens where direction changes should feel responsive and understandable.
