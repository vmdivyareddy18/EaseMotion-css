# Language Switcher – Dropdown with flag animations

> A production-ready language switcher component with a dropdown menu featuring flag animations, smooth transitions, and keyboard accessibility. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Flag animations** – Animated flags in dropdown
- ✅ **Smooth transitions** – Content changes with animation
- ✅ **8 languages** – English, Spanish, French, German, Italian, Portuguese, Japanese, Chinese
- ✅ **Keyboard accessible** – Arrow keys, Enter, Escape
- ✅ **Keyboard shortcuts** – Ctrl+Shift+L (toggle), Alt+1-8 (switch languages)
- ✅ **Toggle dropdown** – Enable/disable dropdown
- ✅ **Toggle animation** – Enable/disable content transitions
- ✅ **4 variants** – Small, Default, Rounded, Dark
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
