# Ease Task Priority Stack

## Overview
A responsive HTML and CSS component for displaying task priorities.

## Features
- Priority labels
- Hover animations
- Responsive design
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A lightweight, reusable component with smooth animations, ideal for productivity dashboards and task management systems.
