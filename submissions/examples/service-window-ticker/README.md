# Service Window Ticker

An intermediate EaseMotion CSS example for maintenance dashboards. A scrolling ticker keeps service window timing, owner, impact, and rollback status visible.

## Files

- `demo.html` - Semantic service window ticker markup.
- `style.css` - Responsive panel layout, gradient edge fades, and continuous ticker motion.

## Usage

Open `demo.html` in a browser to preview the animated service window ticker.
