# Weaver Nest

A pure CSS and HTML weaver bird flitting back to the hanging nests it has knotted onto a branch, a strand of grass in its beak.

## What it shows

- The woven nest wall from two crossed repeating linear gradients, one shadow and one straw coloured, so the basketry reads as knotted fibres
- Nests hanging from stalks and swinging out of phase, each parking its scale in a custom property so the swing keyframe cannot wipe it
- A bird whose wings beat on a fast keyframe while it flies a curved return path, releasing its straw as it arrives
- A downward facing entrance tunnel with an inset shadow, the weaver's trademark
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
