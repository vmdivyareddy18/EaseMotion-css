Features & Accessibility Details

  • Faceted Pendulum Swing: Hovering swatches triggers an elastic pendulum keyframe animation (swing-hover-ag) that simulates gravitational damping and pivot tension.
  • Responsive Hover-and-Focus: Triggers open popovers on hover for desktop users, and on `:focus-within` for keyboard and mobile touch users, requiring zero JavaScript logic.
  • Modern Discrete Transitions: Uses new CSS transition attributes (transition-behavior: allow-discrete and @starting-style) to animate popover displays smoothly without JS toggles.
  • Keyboard Accessibility: Interactive buttons are fully focusable with keyboard outline indicators. The toggle elements can be activated using standard Space/Enter keys.
  • Custom Parameters: Variables such as swing angle, wobble duration, and timing eases are fully exposed at the component level using CSS Custom Properties (e.g., --swing-angle).
