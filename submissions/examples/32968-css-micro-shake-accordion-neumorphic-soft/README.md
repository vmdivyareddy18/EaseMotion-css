# CSS Micro-Shake Accordion (Neumorphic Soft Layouts)

A pure CSS responsive accordion designed for Neumorphic Soft UI dashboards. Features a tactile "micro-shake" horizontal wobble animation on header cards that are styled to look like extruded elements from the parent background.

## Features
- **Horizontal Micro-Shake Effect**: A subtle horizontal vibration animation triggered on hover or focus to suggest interactivity.
- **Premium Neumorphic Soft UI**: Implements double drop-shadow extrusions (`#e0e5ec` base) that transition into recessed inset containers when active.
- **Pure CSS checked Logic**: Built using native checkbox inputs for zero-JS toggle support.
- **Accessible Outlines**: High visibility outlines for keyboard-focused states.

## Files
- `demo.html`: Live interactive demonstration of the Neumorphic Soft micro-shake accordion.
- `style.css`: Stylesheet containing the Neumorphic shadows, panel layouts, and keyframe shaking animation.
