Toast Notification Component

Pure CSS auto-dismissing toasts. No JavaScript timers. The animation lifecycle handles entrance, dwell, and exit automatically using CSS keyframes. Staggered delays simulate a realistic notification stream.

Key Features

  • Auto-dismiss — Entire lifecycle (in → stay → out) controlled by one CSS animation
  • Progress bar — Visual countdown via ::after pseudo-element
  • Zero JavaScript — No timers, no state management
  • Direction variants — Slide from right or slide from bottom
  • Color variants — Success, error, info with accessible contrast
  • Dark mode — .toast-dark class swaps the palette instantly
