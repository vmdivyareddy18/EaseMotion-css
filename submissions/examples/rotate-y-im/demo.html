ease-rotate-y

A utility class that smoothly rotates elements around the Y-axis in 3D space — on hover, on click, or continuously. Pure CSS, GPU-accelerated.

How to Use

html
<!-- Rotates 60deg on hover by default -->
<div class="ease-rotate-y">
  Hover me
</div>

<!-- Custom angle via data attribute -->
<div class="ease-rotate-y" data-angle="180">
  Flips 180°
</div>

<!-- Negative angle reverses direction -->
<div class="ease-rotate-y" data-angle="-90">
  Rotates the other way
</div>

<!-- Continuous spin (no hover needed) -->
<div class="ease-rotate-y-spin">⬡</div>

<!-- Speed variants -->
<div class="ease-rotate-y-spin ease-rotate-y-spin--slow">◈</div>
<div class="ease-rotate-y-spin ease-rotate-y-spin--fast">✦</div>
css vars
/* Customize duration and perspective */
.ease-rotate-y {
  --rotate-y-duration: 0.6s;   /* default: 0.6s */
  --rotate-y-perspective: 800px; /* default: 800px */
}
Note: data-angle values are mapped via CSS custom properties set inline by a tiny optional JS snippet, OR you can use the preset angle classes .ease-rotate-y-45, .ease-rotate-y-90, .ease-rotate-y-180, .ease-rotate-y-neg-180 directly with zero JS.
