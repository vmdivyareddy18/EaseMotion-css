# Cyberpunk Neon Fade-Out Accordion

A modern **Pure CSS Fade-Out Accordion** designed for **Cyberpunk Neon** layouts. The component features glowing neon styling, smooth fade-out interactions, responsive design, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Smooth Fade-Out interaction
- Cyberpunk neon inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and reusable

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.45s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --primary:#00f7ff;
    --secondary:#ff00ff;

    --radius:18px;
}
```

You can customize:

- Animation duration
- Animation easing
- Neon accent colors
- Border radius
- Glow intensity
- Background colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
