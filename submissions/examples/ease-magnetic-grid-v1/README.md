# Ease Magnetic Grid

## Overview

Ease Magnetic Grid creates a futuristic grid of glowing nodes that react to hover interactions, giving the appearance of a magnetic field. Built entirely with HTML and CSS, it is lightweight, reusable, and easy to customize.

---

## Features

- 🧲 Magnetic-inspired hover effect
- ✨ Glowing grid nodes
- 🚀 Smooth scaling animation
- 🎨 Color transition on interaction
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- 5×5 glowing node grid
- Hover expansion effect
- Active node highlight
- Soft glow transitions

---

## Usage

```html
<div class="grid">
    <span></span>
</div>
```

---

## Customization

You can customize:

- Grid size
- Node colors
- Glow intensity
- Animation duration
- Node spacing

---

## Browser Support

Works in all modern browsers supporting CSS Grid, transforms, and transitions.
