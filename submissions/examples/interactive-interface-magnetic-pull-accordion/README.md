# Interactive Interface Magnetic Pull Accordion

A modern **Pure CSS Magnetic Pull Accordion** designed for **Interactive Interface** layouts. The component features smooth magnetic pull hover interactions, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Magnetic Pull hover animation
- Interactive interface inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Smooth scaling and glow effects
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.35s;
    --easing:cubic-bezier(.22,1,.36,1);

    --primary:#38bdf8;
    --secondary:#22d3ee;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Border colors
- Glow intensity

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
