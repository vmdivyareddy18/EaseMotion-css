# Advanced: Business Landing — Dental Clinic

A responsive dental clinic landing page built with **HTML**, **CSS**, and **EaseMotion CSS**. This example demonstrates how to create a clean, modern healthcare website using only existing EaseMotion utility and animation classes.

## Features

* Responsive navigation bar
* Hero section with appointment call-to-action
* Dental services grid
* Before & after smile gallery
* Modern responsive layout
* Professional healthcare design
* Smooth entrance animations
* Interactive hover effects

## EaseMotion CSS Classes Showcased

### Entrance Animations

* `ease-fade-down`
* `ease-fade-up`
* `ease-fade-left`
* `ease-fade-right`
* `ease-scale-in`

### Hover Animations

* `ease-hover-grow`
* `ease-hover-lift`
