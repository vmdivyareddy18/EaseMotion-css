# Advanced Text Reveal Animations

A collection of modern, CSS-only text reveal animations for **EaseMotion CSS**. These effects are lightweight, responsive, and require no JavaScript.

---

## ✨ Features

- 🎯 Pure CSS animations
- ⚡ No JavaScript required
- 📱 Responsive design
- 🎨 Easy to customize with CSS variables
- ♿ Semantic HTML
- 🌙 Compatible with light and dark themes

---

## Available Animations

- Slide Up Reveal
- Blur Reveal
- Clip Path Reveal
- Letter-by-Letter Reveal

---

## Installation

Include the stylesheet in your project.

```html
<link rel="stylesheet" href="text-reveal.css" />
```
