# Product Catalog Shimmer Sweep Modal

A responsive product quick-view modal built using **pure HTML & CSS** featuring a smooth shimmer sweep animation.

## Features

- Pure HTML & CSS
- Product Catalog design
- Shimmer Sweep animation
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No animation JavaScript

## Files

```
demo.html
style.css
README.md
```

## Customization

Modify the variables inside `:root`:

- Animation duration
- Primary color
- Border radius
- Shadow

## Use Cases

- Product Quick View
- E-commerce Catalog
- Shopping Websites
- Store Landing Pages
