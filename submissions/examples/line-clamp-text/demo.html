Fallback Demo

This box simulates how browsers that do not support -webkit-line-clamp would render the truncated text — the fallback shows the full text with a different visual indicator (a fade-out gradient) so content is never hidden from users. The fade gracefully hints that there is more content beyond the visible area.

In non-supporting browsers, the .clamp-* fallback removes the overflow: hidden and shows full text — ensuring accessibility.
