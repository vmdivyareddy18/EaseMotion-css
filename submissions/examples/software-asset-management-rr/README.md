# Software Asset Management Center Component

## What does this do?

Provides a centralized interface for monitoring software licenses, vendor relationships, renewal schedules, utilization metrics, and compliance status.

## How is it used?

```html
<div class="asset-card healthy">
    <h3>Microsoft 365</h3>
</div>
```

## Why is it useful?

Organizations often struggle with software sprawl, unused licenses, and renewal management. This component provides a reusable Software Asset Management dashboard for tracking software utilization, compliance, and optimization opportunities.
