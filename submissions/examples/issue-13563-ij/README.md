# ease-progress-circular

Circular/ring progress indicator. SVG-based with stroke-dasharray/stroke-dashoffset technique, smooth fill animation, and percentage text in center.

## Usage

Include `style.css` and use `.ease-progress-circular` with an SVG circle. Set `--ease-progress-value` or update `stroke-dashoffset` via JS.

## Demo

Open `demo.html` in a browser.
