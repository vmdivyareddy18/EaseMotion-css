# ease-backdrop-blur

## What does this do?
The ease-backdrop-blur utility class provides a reusable CSS pattern following the EaseMotion CSS design system.

## How is it used?
    <div class=".ease-backdrop-blur-md">Content</div>

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html in your browser.

## Contribution Notes
- Class naming follows the ease-* convention
- Uses CSS custom properties for theming
