# CSS Progress Tracker

A simple and responsive CSS-only Progress Tracker component built using HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive design
- Completed, Active, and Upcoming steps
- Smooth hover animation
- Beginner-friendly implementation

## Files

- demo.html
- style.css

## Use Cases

- Multi-step forms
- Checkout process
- User onboarding
- Registration workflow
