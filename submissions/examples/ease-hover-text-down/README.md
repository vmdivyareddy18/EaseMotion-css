# ease-hover-text-down

A simple hover animation where text shifts **downward by 3px**, creating a subtle interactive effect. Works great for navigation links, menus, and buttons.

---

## ✨ Features

- Pure CSS (no JavaScript)
- Smooth 0.25s transition
- Lightweight and reusable
- Customizable via CSS variables
- Pairs well with `ease-hover-text-up`

---

## 🚀 Usage

```html
<a href="#" class="ease-hover-text-down">
    Home
</a>
```

---

## 🎛 CSS Variables

```css
:root {
    --ease-hover-text-down-distance: 3px;
    --ease-hover-text-down-duration: 0.25s;
}
```
