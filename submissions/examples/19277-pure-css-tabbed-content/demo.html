Getting Started

Welcome to EaseMotion! This guide walks you through the fundamentals and helps you ship your first animation in under 5 minutes.

/* Include EaseMotion in your HTML */
<link rel="stylesheet" href="easemotion.css" />

Installation

Install EaseMotion via npm or link directly from our CDN.

# Via npm
npm install easemotion-css

# Via CDN
<link rel="stylesheet" href="https://cdn.easemotion.dev/latest/easemotion.min.css" />

Configuration

Customize EaseMotion by overriding CSS custom properties in your root or a scoped selector.

:root {
  --ease-duration: 400ms;
  --ease-timing: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-primary: hsl(221, 83%, 53%);
}

How It Works

The entire mechanism is a single CSS trick — no JavaScript required.

/* 1. Hidden radio inputs act as state holders */
.tab-radio {
  position: absolute;
  opacity: 0;
  width: 0;
  height: 0;
}

/* 2. All panels are hidden by default */
.tab-panel {
  display: none;
}

/* 3. :checked + sibling combinator reveals the active panel */
#tab-1:checked ~ .tab-panels .tab-panel:nth-child(1) {
  display: block;
}

/* 4. :checked label styling highlights the active tab */
#tab-1:checked ~ .tab-list label[for="tab-1"] {
  background: var(--ease-primary);
  color: #fff;
}
