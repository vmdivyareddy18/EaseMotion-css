Outline Offset Utilities

Configure the spacing offset between an element's outline and its border or visual bounds. Use positive offsets to project focus rings outward, and negative offsets to draw interior frame decorations.

Design Use-Cases

1. Accessible Focus Visible

A WCAG 2.1 compliant focus ring with outwards offset prevents focus boundaries from clashing with the element's border radius or overlapping adjacent labels.

2. Inside Inset Framing

Using a negative offset draws an elegant border inside the bounds of the element. Perfect for editorial profile frames, banners, and digital certificates.

Inner Frame

3. Multi-Layered Border

Offsetting a dashed outline outward from a solid border produces a sophisticated, double-ringed layout resembling vintage packaging or technological brackets.

Layered Frame

4. Expanding Outline Trigger

An interactive micro-animation where hovering or focusing changes the outline offset, causing the ring to expand outward smoothly for improved feedback.

Accessibility Guidance (WCAG 2.1)

Criterion 2.4.7

Focus Visible

Any keyboard-operable user interface must have a mode of operation where the keyboard focus indicator is visible. Using outline-offset ensures focus states remain visually detached from background styling, making them clear and unmistakable to users with low vision.

Contrast Rule

Non-Text Contrast (1.4.11)

The visual focus indicator must have a contrast ratio of at least 3:1 against the adjacent background. Native outlines are preserved in OS-level High Contrast Modes (e.g. Windows Contrast Themes), whereas alternative focus indicators built using box-shadow or custom border colors are often stripped, creating severe accessibility barriers.
