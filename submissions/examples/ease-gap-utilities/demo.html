Flexbox & Grid Gap Utilities

Using the CSS gap property to consistently space elements without relying on messy margin overrides or first/last child selectors.

1. Flexbox with Gap (.ease-gap-md)

Instead of adding margins to every button and removing it from the last one, simply apply a gap to the flex parent container. It automatically spaces items cleanly.

2. CSS Grid with Gap (.ease-gap-lg)

Gap works beautifully in 2D grid layouts as well, spacing out both columns and rows uniformly.
