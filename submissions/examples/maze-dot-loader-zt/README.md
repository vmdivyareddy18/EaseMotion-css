# Maze Dot Loader

A CSS-only maze dot loader with a glowing dot moving through a square path for premium dashboard-style loading states.

## Description

This component animates a small glowing dot along a maze-like square path using only HTML and CSS. It features a clean geometric layout, soft glow effects, and smooth looping motion.

## Features

- Pure HTML + CSS implementation
- No JavaScript or external libraries
- Dot moves through a square maze pattern
- Smooth infinite animation
- Modern dark theme with glow accents
- Responsive sizing for mobile and desktop
- Easy color and speed customization through CSS variables

## Usage

1. Copy `demo.html` and `style.css` into your submission folder.
2. Open `demo.html` in a browser.
3. Update the loader styles or animation speed using `--accent` and `--speed` variables.

```html
<link rel="stylesheet" href="style.css" />
<div class="loader-page">
  <section class="loader-card ease-fade-in">
    <!-- loader markup here -->
  </section>
</div>
```

This loader aligns with EaseMotion CSS by prioritizing motion-first design, self-contained markup, and polished visual styling. It feels premium and modern while remaining lightweight and easy to customize.
