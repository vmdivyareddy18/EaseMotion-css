# Gap Utilities: Grid And Flex Gap Utilities

Demonstrates CSS gap utilities for grid and flexbox layouts using EaseMotion spacing tokens, covering row-gap, column-gap, and the shorthand gap property with responsive variants.

## Features

- Demonstrates **gap** CSS property with multiple practical variants
- Uses EaseMotion design tokens (`--ease-color-*`, `--ease-space-*`, `--ease-radius-*`)
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`
- Multiple size/style variants for real-world usage

## Usage

Apply the utility class to any element:

```html
<div class="gap-xs">Content</div>
```

## Why is it useful?

The **gap** CSS property is part of the modern CSS specification and enables
fine-grained control over gap-utilities. It integrates seamlessly with EaseMotion's design token
system, making it easy to maintain consistent spacing, colors, and typography across your project.
