Clip Path

Creates clipping regions to show only parts of an element using geometric shapes.

CSS Property: clip-path

Variants Demo

clip-path: ellipse(50% 40% at 50% 50%) Clip Path variant
clip-path: polygon(50% 0%, 100% 100%, 0% 100%) Clip Path variant
clip-path: inset(20% 10% 30% 15%) Clip Path variant
