# Shimmer Skeleton Loading

## What does this do?
A comprehensive CSS shimmer/skeleton loading system with multiple animation directions and common layout patterns for content placeholders.

## How is it used?
```html
<div class="sh-line sh-line-60 sh-shimmer-ltr"></div>
<div class="sh-avatar-lg sh-shimmer-wave"></div>
```

## Why is it useful?
Provides performant GPU-accelerated loading placeholders. Multiple animation variants (LTR, TTB, diagonal, wave, pulse) suit different design needs. Includes reusable layout patterns for articles, profiles, dashboards, and card grids.
