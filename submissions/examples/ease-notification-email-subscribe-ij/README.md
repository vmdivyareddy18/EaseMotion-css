# Notification Email Subscribe

An email subscription prompt with bell-ring animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--prompt-bg` | `#2d2d44` | Card background |
| `--bell-color` | `#4ecdc4` | Bell icon accent |
| `--btn-color` | `#4ecdc4` | Subscribe button color |
| `--ring-duration` | `2s` | Bell ring cycle |
