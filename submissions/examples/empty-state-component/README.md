# Empty State Component for Dashboards and Apps

## Description

Create a CSS-only Empty State Component that helps developers display meaningful placeholders when content, data, search results, notifications, tasks, or records are unavailable.

The component is illustration-free and focuses on clean typography, an optional icon slot, descriptive messaging, and a call-to-action button. It supports both compact and full-page variants to accommodate dashboards, admin panels, SaaS products, mobile apps, and web applications.

Supported features:

- Title and description
- Optional icon slot
- Primary action button
- Compact variant
- Full-page variant
- Responsive layout
- Easy customization with CSS

## Why is this useful for EaseMotion CSS?

Empty states are a critical part of modern user experiences. Instead of showing blank screens, they guide users toward meaningful actions such as creating content, adding data, or exploring features.

This component aligns with EaseMotion CSS's human-readable and composable philosophy while providing a reusable UI pattern commonly found in dashboards, productivity tools, SaaS platforms, and admin interfaces.

It can also be enhanced with existing EaseMotion animations for subtle entrance and attention effects.
