# Pulse Context Menu

A modern **Pulse Context Menu** component built with pure HTML and CSS.

The component provides a clean, reusable context menu featuring a subtle pulse animation inspired by modern blogging platforms.

## Features

- Pure HTML & CSS
- Responsive
- Accessible
- Pulse animation
- Blog-inspired design
- No JavaScript
- Easy to customize

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<div class="context-menu">
    <button class="menu-button">
        ⋮
    </button>
    <div class="menu">
        <a href="#">Edit</a>
        <a href="#">Duplicate</a>
        <a href="#">Delete</a>
    </div>
</div>
```

## Accessibility

- Keyboard-focusable button
- Responsive layout
- High-contrast text
- Semantic links for menu actions

## Why EaseMotion CSS?

The Pulse Context Menu demonstrates how subtle motion can improve user interaction by smoothly revealing contextual actions. The pulse animation adds visual emphasis while keeping the component lightweight, reusable, and entirely CSS-based.
