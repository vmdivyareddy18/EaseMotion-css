# Image Color Reveal

**EaseMotion Submission** · Contributor: kavin553

---

## 1. What does this do?

Displays an image in grayscale by default, smoothly transitioning to full color on hover or keyboard focus.

## 2. How is it used?

```html
<img
  class="image-color-reveal"
  src="team-member.jpg"
  alt="Team member portrait"
/>
