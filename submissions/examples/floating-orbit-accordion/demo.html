Floating Orbit

A minimalist, zero-JavaScript accordion interface driven entirely by semantic HTML and customizable CSS motion parameters.

01 Adaptive Motion Transitions designed around natural movement.

Adjust --orbit-duration and --orbit-easing to control the timing and motion profile of the interface.

02 Orbital Feedback Visual states move beyond standard expansion.

Floating indicators, circular accents, and subtle scaling create an orbital interaction language without requiring JavaScript.

03 Custom Parameters Fine-tune scale and movement using CSS properties.

Change --orbit-scale and --orbit-distance from the root configuration to quickly adapt the interaction to different layouts.

04 Accessible by Default Semantic controls with keyboard focus support.

Native details and summary elements provide keyboard interaction while visible focus styles and reduced-motion support improve accessibility.
