# SaaS Landing: B2B Data Enrichment (`saas-landing-b2b-data-21754`)

This proposal provides a complete, copy-paste ready B2B landing page template designed for a modern Data Enrichment/RevOps SaaS product, built entirely with HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A Data Enrichment SaaS brand requires a UI that feels highly accurate, technical, and reliable. This template achieves a Clearbit-style aesthetic using a crisp data-focused palette (white/gray, deep blues, cyan accents), modern geometric typography (`Work Sans` & `Roboto Mono`), and fast, precise animations.

The layout includes:
- A clean, structured Navigation.
- A Hero section featuring a CSS-animated API JSON mockup simulating lines of code typing out and highlighting matched data.
- A trusted customer RevOps logos banner with staggered fade-ups.
- A Stats row featuring JavaScript-assisted count-up numbers.
- A Features grid highlighting firmographics, contacts, and intent with abstract CSS icons.
- An Integrations section showcasing an abstract CSS data pipeline animation simulating data flowing between CRMs and the SaaS platform.
- A Pricing tier grid highlighting API vs Platform plans.
- A clean, professional Footer.

### Showcased Classes
The template leverages standard EaseMotion animation and interaction utilities, heavily modified locally via CSS to use precise, snappy `cubic-bezier(0.16, 1, 0.3, 1)` and slightly bouncy `cubic-bezier(0.34, 1.56, 0.64, 1)` timings:
- `.ease-slide-up-stagger` (Configured for a fast, geometric vertical entrance).
- `.ease-type-code` (A fast, sequential reveal animation simulating JSON data loading).
- `.ease-flow` (A continuous linear gradient animation simulating data moving through a pipeline).
- `.ease-pulse` (A rigid, structural pulse animation on the central node).
- `.ease-hover-lift-card` (A sharp, pronounced hover elevation for pricing/feature cards).
- `.ease-hover-glow-box` (A technical glow transition for the hero API mockup).
- `.ease-count-up` (A JS-assisted counting animation for the stats bar).

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the sharp layout architecture, the tech-focused palette, and the complex abstract UI elements (API JSON mockup, data pipeline flow). The standard `ease-*` interaction classes are attached directly to HTML elements to provide the precise entrance animations. A tiny block of inline vanilla JavaScript is included at the bottom of `demo.html` solely to trigger the `.ease-count-up` number effect.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/saas-landing-b2b-data-21754/` directory to avoid modifying core files directly and to ensure zero deletions.*
