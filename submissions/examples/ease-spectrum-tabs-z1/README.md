# Ease Prism Wave Tabs

## Overview

A premium animated tab navigation featuring glowing prism-inspired underline effects and glassmorphism styling built entirely with HTML and CSS.

## Features

- Animated underline wave
- Glassmorphism tabs
- Hover lift animation
- Responsive Flexbox layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure

```text
ease-prism-wave-tabs-z1/
├── demo.html
├── style.css
└── README.md
```

Open `demo.html` in any modern browser.
