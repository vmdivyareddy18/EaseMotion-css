# Ime Mode Auto

ime-mode — Controls Input Method Editor state for text inputs

## Features
- Uses `--ease-color-surface`
- Uses `--ease-color-text`
- Uses `--ease-radius-md`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Ime Mode Auto</span>
</div>
```

## Why is it useful?

This submission demonstrates the ime mode auto CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
