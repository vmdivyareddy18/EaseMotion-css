# Breathing Glow Effect

## What it does

Creates a pulsing glow that expands and contracts like breathing. Perfect for attention-grabbing icons, status indicators, or call-to-action elements.

## How to use it

<div class="ease-breathing-glow">✨</div>

## Variants

- ease-breathing-fast - 1s cycle
- ease-breathing-slow - 4s cycle
- ease-breathing-blue - Blue glow
- ease-breathing-green - Green glow
- ease-breathing-pink - Pink glow

## Why it fits EaseMotion CSS

- Pure CSS animation
- Multiple color variants
- Scale + shadow animation
- Respects prefers-reduced-motion
