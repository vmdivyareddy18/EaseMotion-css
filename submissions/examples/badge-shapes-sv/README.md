# Badge Shape Variants

## What does this do?

Creates pill-shaped and square-shaped badge variants that can be used for labels, notifications, statuses, and UI indicators.

## How is it used?

```html
<span class="badge badge-pill success">
  Success
</span>

<span class="badge badge-square info">
  i
</span>
```

## Why is it useful?

Badges are commonly used throughout dashboards, admin panels, and applications. Providing multiple shape variants increases flexibility and allows developers to choose a style that best matches their interface.
