# Custom Focus Rings

## What does this do?
CSS :focus-visible custom focus ring patterns including solid, dashed, dotted, glow, neon, underline, offset, and bottom bar indicators.

## How is it used?
```html
<button class="fcr-btn fcr-glow">Focus me</button>
<input type="text" class="fcr-inp fcr-glow">
```

## Why is it useful?
Accessible custom focus indicators that only show on keyboard focus. Multiple ring styles for different design contexts. Works on buttons, inputs, links, and checkboxes.
