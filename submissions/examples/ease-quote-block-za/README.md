# CSS Quote Block Component

## What does this do?
A collection of styled blockquote and testimonial components with decorative quote marks, border accents, gradient backgrounds, and avatar card styles.

## How is it used?
```html
<blockquote class="qtb-classic">
  <p>Quote text here</p>
  <cite>Author Name</cite>
</blockquote>
```

## Why is it useful?
Provides visually appealing quote presentations for testimonials, article pull quotes, and review sections. Multiple styles from minimal to decorative. Pure CSS, responsive, and customizable.
