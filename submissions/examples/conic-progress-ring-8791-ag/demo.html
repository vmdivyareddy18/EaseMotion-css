Technique & Code

By nesting an absolute-positioned inner circle inside a block containing a conic-gradient, we mask out the center to form a ring. The gradient boundary is bound to a CSS property:

.conic-ring {
  background: conic-gradient(
    #7c3aed var(--progress), 
    rgba(255, 255, 255, 0.05) 0deg
  );
  transition: --progress 1s cubic-bezier(0.4, 0, 0.2, 1);
}
