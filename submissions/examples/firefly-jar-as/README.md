# Firefly Jar

A pure CSS and HTML jar of fireflies on a summer night, the insects drifting and blinking inside the glass.

## What it shows

- Fireflies as glowing dots with layered box-shadow halos, each running two animations at once: a wandering drift and an independent blink
- A glass jar from translucent gradients and borders, with a vertical shine streak and a ribbed metal lid
- A soft pool of light gathering at the base of the jar, breathing in and out
- A starry sky and swaying grass around the jar
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
