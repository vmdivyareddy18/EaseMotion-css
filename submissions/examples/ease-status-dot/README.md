# Status Dot – Pulsing green/red for online/offline

> A production-ready status indicator component that displays pulsing green/red dots for online/offline states with status history and uptime tracking. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Pulsing status dot** – Animated pulse for live status
- ✅ **4 status states** – Online, Offline, Warning, Idle
- ✅ **Status history** – Tracks all status changes
- ✅ **Uptime tracking** – Real-time uptime counter
- ✅ **Multiple sizes** – Small, Medium, Large
- ✅ **Keyboard shortcuts** – 1-4 (status), Space (toggle), C (clear)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
