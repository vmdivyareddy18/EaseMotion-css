# Ease Griffin Soar Animation

A complex, majestic multi-axis flight simulation animation utility classes for the EaseMotion CSS framework. It pairs vertical loft adjustments with responsive vertical scales to perfectly mimic a heavy creature cycling high altitude airways.

## Features
- **Composite Transforms**: Combines translations, pitch shifts, and scale modulators in a single non-breaking timeline thread.
- **Micro-Physics Gliding**: Leverages smooth custom ease curves to create a floating sensation without mechanical snapping.
- **Three Utility Modifiers**: Ships complete with standard `default`, high frequency `climb`, and majestic `heavy` air speeds.

## Core Utility Usage
```html
<div class="element ease-griffin-soar">🦅🦁</div>
<div class="element ease-griffin-soar-climb">🦅🦁</div>
<div class="element ease-griffin-soar-heavy">🦅🦁</div>
