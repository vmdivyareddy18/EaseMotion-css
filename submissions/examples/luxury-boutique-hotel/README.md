# EaseMotion CSS Showcase — Boutique Luxury Hospitality

An editorial, luxury-tier single-page application built for intimate boutique resorts. This implementation documents how structural grid patterns and custom typographic scales interact fluidly when driven by the performance-optimized **EaseMotion CSS motion protocol**.

---

## 🚀 Key Features

* **Zero Frame Pre-requisites:** Executable raw vanilla code layer requiring zero compiler infrastructure or Tailwind modules.
* **Premium Editorial Palette:** Centered completely on a luxury composition utilizing soft linen creams (`#f9f6f0`), deep slate carbons (`#11141a`), and muted architectural gold tones (`#c5a059`).
* **Asymmetrical Flex Architecture:** Leverages alternating flex layouts and structural aspect ratios to mirror classic, high-end design magazines.
* **Declarative Motion Registry:** Applies physics-based, smooth curve adjustments via custom transition modules to provide tactile feedback across the user experience.

---

## 🎯 EaseMotion Class Registry

The structural views are animated via the following natively packaged utility classes:

### 1. Spatial Entrance Vectors
These definitions trigger upon document completion to sequentially elevate components into place without blocking interactive processes:
* `ease-fade-in` — Safely brings macro canvas backgrounds up from zero opacity using a prolonged `1.2s` timeline.
* `ease-slide-up` — Applies an upward motion from a `translateY(40px)` threshold, giving editorial headers room to breathe.
* `ease-slide-down` — Seamlessly transitions navigation modules from above the header view down into static visibility thresholds.
* `ease-slide-right` — Drags complex left-hand photographic arrays horizontally outward into position smoothly.
* `ease-slide-left` — Drags right-hand text layers into position synchronously with matching image elements.
* `ease-scale-in` — Elevates focal interactive containers using an expansive structural zoom trajectory (`scale(0.97)` to origin). (Ideal for reservation panels).

### 2. Delay Sequence Modifiers
Enables precise coordinate staggering across repeated element maps:
* `ease-delay-1` — Delays transition ignition by exactly `0.2s`.
* `ease-delay-2` — Delays transition ignition by exactly `0.4s`.
* `ease-delay-3` — Delays transition ignition by exactly `0.6s`.
* `ease-delay-4` — Delays transition ignition by exactly `0.8s`.

### 3. Idle Ambient Interactions
* `ease-pulse-soft` — Oscillates elements gently over a long `4s` frequency to simulate an atmospheric pulse. (Used to draw attention to booking updates).

### 4. Interactive Hover Hooks
* `ease-hover-grow` — Smoothly translates element depth up the vertical scale (`translateY(-8px)`) while applying a luxurious blurred box shadow.
* `ease-hover-glow` — Expands a gold hue across thin element borders while projecting a soft ambient perimeter mist.
* `ease-hover-lift` — Adds micro-elevation shifts (`translateY(-3px)`) tailored specifically for text menus and interactive anchors.
