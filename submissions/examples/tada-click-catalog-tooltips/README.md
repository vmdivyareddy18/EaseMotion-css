# Pure CSS Tada-Click Product Catalog Tooltip

An efficient, lightweight, and hardware-accelerated tooltip element crafted cleanly to match modern retail grids and digital e-commerce item displays.

## Features
- **Zero JS Dependencies:** Operates using declarative CSS animation states attached directly to user interaction pseudotokens (`:active`).
- **Product UI Harmony:** Leverages a modern layout format to frame clean information badges and pricing metrics.
- **Micro-Interaction Polish:** Scales and shifts across dual axises on user clicks to create confirmation responses.
- **A11y Validated Framework:** Keyboard-navigable layout structure compatible with assistive readers.

## Configuration Parameters

| Custom Property Name | Assignment Target Scope | Default Setup Vector |
| :--- | :--- | :--- |
| `--catalog-accent` | Core branding element highlights | `#4f46e5` |
| `--tooltip-bg` | Interior surface coloring matrix | `#171717` |
| `--tooltip-fade-duration` | Base visibility reveal duration speed | `0.2s` |
| `--tooltip-tada-duration` | Execution cycle duration of the click shake | `0.5s` |
