# Masonry Grid Layout

A CSS-only masonry-style grid using `grid-template-columns` with `auto-fill` and `grid-auto-rows` for variable-height items. Each item spans a different number of rows using a `--rows` custom property, creating the staggered masonry effect. Items are shown in pastel colors.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the masonry grid of colored blocks.

## Accessibility notes

The colored blocks are decorative and purely visual. Reduced motion has no impact since there is no animation.
