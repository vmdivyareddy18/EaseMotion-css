# Scale Share Button

A responsive **Scale Share Button** component inspired by Glassmorphism design patterns for the EaseMotion CSS library.

## Features

- Pure HTML and CSS
- No JavaScript
- Glassmorphism inspired UI
- Smooth scale hover animation
- Responsive design
- Accessible button element
- Uses EaseMotion CSS variables and `ease-*` animation classes
