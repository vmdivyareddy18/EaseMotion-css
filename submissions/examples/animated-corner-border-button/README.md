# Animated Corner Border Button

## Description

A hover effect where border corners expand into a complete frame around the button.

## Features

- Pure CSS implementation
- Smooth border animation
- Lightweight and reusable
- Modern UI styling

## Files

- demo.html
- style.css
- script.js

## Usage

Open demo.html and hover over the button.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
