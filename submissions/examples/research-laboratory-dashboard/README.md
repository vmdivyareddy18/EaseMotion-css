# Research Laboratory Dashboard

## Overview
Design a scientific research tracking workspace.

This project implements the Phase 124 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Experiment cards**: Built with glassmorphism and CSS grid/flexbox.
- **Lab analytics**: Built with glassmorphism and CSS grid/flexbox.
- **Research timeline**: Built with glassmorphism and CSS grid/flexbox.
- **Resource management**: Built with glassmorphism and CSS grid/flexbox.
- **Mobile optimization**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
