# Text Overflow Utilities (#16469)

## Overview
A set of CSS utility classes designed to handle text overflow gracefully. This feature provides standard solutions for both single-line truncation and multi-line clamping.

## Features
- **.truncate-text:** Standard ellipsis for single-line overflow.
- **.clamp-2-lines:** Intelligent multi-line clamping using standard CSS flex-box logic.

## Usage
Apply the class directly to your text container. Ensure the parent container has a defined width for truncation to take effect.
