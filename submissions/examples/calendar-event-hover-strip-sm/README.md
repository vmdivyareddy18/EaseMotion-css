# Calendar Event Hover Strip

1. What does this do? Adds compact calendar event strips that expand extra details on hover or keyboard focus with animated priority rails and status chips.
2. How is it used? Apply `.event-list` to the schedule wrapper and `.event-strip` with tone classes such as `.design-event`, `.build-event`, `.review-event`, or `.release-event` to each event.
3. Why is it useful? It keeps busy schedules scannable while revealing useful context through purposeful CSS-only motion.
