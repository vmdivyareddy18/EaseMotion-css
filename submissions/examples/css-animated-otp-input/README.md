# Animated OTP Verification Input

A modern OTP verification component built using pure HTML and CSS.

## Features

- Animated OTP input boxes
- Focus scale effect
- Responsive layout
- Clean modern UI
- Pure HTML & CSS
- Beginner Friendly

## Folder Structure

```text
css-animated-otp-input/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser.

## Browser Support

Works in all modern browsers.
