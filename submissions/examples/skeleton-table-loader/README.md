# Skeleton Table Loader

## What does this do?
A responsive skeleton loading table with shimmer rows, avatar placeholders, and column placeholders.

## How is it used?
```html
<div class="table-shell">
  <div class="table-row">
    <span class="avatar"></span>
    <span class="line wide"></span>
  </div>
</div>
```

## Why is it useful?
It gives EaseMotion CSS a practical loading-state pattern for dashboards and data-heavy interfaces.
