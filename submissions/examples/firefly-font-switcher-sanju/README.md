# Firefly Font Switcher

A blog-inspired Firefly Font Switcher component for EaseMotion CSS.

## Features

- Pure CSS
- No JavaScript
- Responsive layout
- Accessible radio-button controls
- Firefly glow animation
- Blog-style design
- Uses EaseMotion animation classes

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in a browser.

Select any font card to preview the typography.

## Accessibility

- Keyboard accessible
- Uses semantic radio inputs
- Responsive on mobile and desktop
