# Teleport Fade

## What does this do?
A teleportation fade and reform effect where an element dissolves, scatters, and then reassembles at its new position.

## How is it used?
Apply the `.teleport-fade` class to trigger the teleport effect. The element fades out with a scatter particle effect, then reassembles.

## Why is this useful?
Adds a sci-fi/fantasy transition effect for element repositioning, page transitions, or character movement in gamified UIs.
