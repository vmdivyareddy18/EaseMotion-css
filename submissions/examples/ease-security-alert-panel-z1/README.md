# Ease Security Alert Panel

## Overview
A CSS-only security alert component for monitoring dashboards.

## Features
- Severity badges
- Hover animations
- Responsive layout
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for security dashboards, SOC platforms, monitoring tools, and admin interfaces.
