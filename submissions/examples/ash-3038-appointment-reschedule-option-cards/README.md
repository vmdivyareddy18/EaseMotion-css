# Appointment Reschedule Option Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive appointment rescheduling component featuring selectable time-slot cards for booking and scheduling interfaces.

## Features

- Appointment option cards
- Selected state styling
- Doctor and time information
- Availability badges
- Responsive layout
- Hover animations
- Glassmorphism UI
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="option-card selected">

    <div class="date">
        <h2>Mon</h2>
    </div>

    <div class="details">
        <h3>09:30 AM</h3>
    </div>

</div>
```

## Main Classes

```css
.cards
.option-card
.selected
.date
.details
.confirm-btn
```
