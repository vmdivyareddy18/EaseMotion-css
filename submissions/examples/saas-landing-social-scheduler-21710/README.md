# Business Landing — Social Media Scheduler SaaS (`saas-landing-social-scheduler-21710`)

This proposal provides a complete, copy-paste ready B2B SaaS landing page template specifically designed for Social Media Management, Content Scheduling, and Marketing platforms (similar to Buffer, Hootsuite, or Later), using pure HTML/CSS and EaseMotion utility classes.

## 📌 Feature Overview

A marketing SaaS needs to feel creative, approachable, and organized. This template uses a clean UI with soft shadows, a distinct primary color (Rose), and highly visual components.

The layout includes:
- A hero section featuring an animated CSS-built visual content calendar mockup, complete with a floating "New Post" composer window.
- A social platform integration logo banner.
- A 6-card platform feature grid with emoji icons.
- An interactive analytics section featuring a CSS-animated bar chart.
- A customer testimonial grid.
- A tiered pricing layout.
- A comprehensive footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-in` / `.ease-fade-up` (For smooth text entrances and simulating posts appearing in the calendar mockup)
- `.ease-slide-up` (For structural layout animations in pricing, features, and the floating composer modal)
- `.ease-zoom-in` (For drawing attention to the primary mockups and the analytics chart)
- `.ease-bounce-in` (For primary trial CTAs)
- `.ease-hover-lift` / `.ease-hover-scale` / `.ease-hover-zoom` (For interactive buttons, feature cards, and the mockups)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the structural layout and styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations without needing custom JavaScript or specific CSS keyframes. The calendar, floating composer, and bar charts are built entirely with HTML/CSS.
