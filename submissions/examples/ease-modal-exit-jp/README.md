# Ease Modal Exit

A premium modal exit animation built using pure HTML and CSS featuring backdrop fade-out, content scale-down, and smooth closing transitions.

## Features

- Backdrop Fade-Out Animation
- Modal Scale-Down Exit
- Smooth Opacity Transition
- Glassmorphism UI
- Responsive Design
- Pure HTML & CSS
- No JavaScript Required
