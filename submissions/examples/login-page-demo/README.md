# Login Page Demo

## Purpose

This example demonstrates how existing EaseMotion CSS animation classes can be used to build a responsive login page.

---

## Components

- Login Card
- Email Input
- Password Input
- Login Button
- Forgot Password Link
- Sign Up Link

---

## Folder Structure

```
submissions/examples/login-page-demo/
├── demo.html
├── style.css
└── README.md
```

---

## How to Run

1. Open `demo.html`.
2. Resize the browser window.
3. Observe the applied EaseMotion animations.

---

## Benefits

- Responsive login page
- Practical authentication UI example
- Uses existing EaseMotion CSS animation classes
- Easy to customize
