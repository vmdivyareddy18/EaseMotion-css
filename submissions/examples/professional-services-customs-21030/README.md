# Professional Services: Customs Broker & Trade Logistics (`professional-services-customs-21030`)

This proposal provides a complete, copy-paste ready B2B professional services landing page designed for a Customs Broker or Global Trade/Logistics firm, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A logistics and customs brokerage site needs to convey reliability, structural integrity, and regulatory competence. This template achieves an industrial, heavy-duty aesthetic using a strong, condensed font (Roboto Condensed), a highly structured layout with solid block borders, and an authoritative palette of dark slate, trust blue, and freight/safety yellow.

The layout includes:
- A hero section featuring an abstract CSS-animated shipping lane simulation.
- A bold, high-contrast client trust bar.
- A highly structured 3-column service capabilities grid.
- A vertical alternating process timeline (document receipt to freight delivery).
- A grid of specific industries served.
- A robust, dual-pane CTA section encompassing contact info and a detailed quote request form.
- An extensive, multi-column footer suitable for a multi-office agency.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-slide-right` / `.ease-slide-left` (Utilized to alternate the entrance of the timeline steps for a dynamic, process-driven flow)
- `.ease-fade-up` (For standard, clean entrances of text blocks and client logos)
- `.ease-expand` (Applied to divider lines under headings for a structural, industrial accent)
- `.ease-zoom-in` (Applied to the dense industry grid to bring items forward efficiently)
- `.ease-hover-lift` (Standard interaction for primary buttons, capability cards, and timeline steps to indicate interactivity)
- `.ease-focus-ring` (Applied to form inputs to provide an accessible, distinct focus state)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, typography, and heavy-duty industrial styling, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive (the timeline automatically collapses to a single-column layout on smaller screens) and requires zero JavaScript.
