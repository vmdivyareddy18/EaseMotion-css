# Full Mortgage/Loan Company Landing Page

1. What does this do? A complete, responsive mortgage company landing page showcasing loan services (home, auto, personal, business), rates, application process, and client testimonials using EaseMotion CSS framework classes.

2. How is it used? Include `demo.html` in the browser — the page uses EaseMotion classes (`ease-navbar-glass`, `ease-card`, `ease-btn-primary`, `ease-reveal-up`, `ease-hover-lift`, etc.) for all layout, components, and animations with zero custom keyframes.

3. Why is it useful? Demonstrates how EaseMotion can build production-ready financial services pages entirely with framework utility classes — no custom animations needed, just composable `ease-*` classes for professional UI.
