# Timeline Status Rail

## What does this do?
A vertical status timeline with completed, active, and pending states plus animated entry and pulse feedback.

## How is it used?
```html
<ol class="status-rail">
  <li class="done"><span class="dot"></span><strong>Request received</strong></li>
  <li class="active"><span class="dot"></span><strong>Maintainer review</strong></li>
</ol>
```

## Why is it useful?
It adds a clear progress pattern for onboarding, orders, reviews, and task flows while keeping the motion semantic and lightweight.
