EaseMotion CSS is a lightweight, utility-first animation library that provides pre-built CSS animations for modern web development. It includes fade-ins, slides, bounces, pulses, and more—all without requiring JavaScript.

Simply add classes like ease-fade-in-up or ease-bounce-in to your HTML elements to bring them to life.

You can install EaseMotion CSS via CDN or npm:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/SAPTARSHI-coder/EaseMotion-css@main/easemotion.min.css" />

Or install via npm:

npm install easemotion-css
