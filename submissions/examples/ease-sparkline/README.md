# Sparkline – Line draws itself with data points

> A production-ready sparkline component that draws a line chart with animated data points. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

![Sparkline Demo](https://via.placeholder.com/400x100/4facfe/ffffff?text=📈+Sparkline)

**Live Preview:** Open `demo.html` in your browser.

---

## ✨ Features

- ✅ **Self-drawing line** – Animates from left to right
- ✅ **Data points** – Shows values on hover/draw
- ✅ **Fill area** – Gradient fill under the line
- ✅ **Real-time stats** – Current value and percentage change
- ✅ **Play/Pause** – Control the animation
- ✅ **Random data** – Generate new data on demand
- ✅ **Speed control** – Slow, Normal, Fast
- ✅ **Toggle points** – Show/hide data points
- ✅ **Multiple variants** – Different color themes
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
