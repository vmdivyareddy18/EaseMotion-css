# Floating Scroll Top FAB

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A floating action button designed for scroll-to-top interactions. The component features a floating animation, pulse ring effect, gradient styling, and responsive positioning using pure HTML and CSS.

## Features

- Scroll-to-top FAB
- Floating animation
- Pulse ring effect
- Gradient styling
- Hover interaction
- Responsive positioning
- Dashboard-style demo
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<a href="#" class="scroll-top-fab">
  ↑
</a>
```
