# Glowing Neon Button Effect

A set of buttons with a neon glow effect on hover. Three colors are available: cyan, pink, and green. On hover, the buttons emit a layered box-shadow glow. The buttons are displayed on a dark background for maximum glow visibility.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Hover over each button to see the neon glow effect.

## Accessibility notes

Buttons are `<button>` elements for proper keyboard interaction. Reduced motion disables the glow transition.
