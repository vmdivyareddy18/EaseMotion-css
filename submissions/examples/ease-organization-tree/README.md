# ease-organization-tree

## What this is
A CSS-only organization tree with hierarchy reveal animation.

## Why
It helps display team structure in a clean visual format while keeping the component lightweight and easy to theme.

## How to test
1. Open `demo.html` in a browser.
2. Confirm the root node appears first.
3. Confirm child branches reveal in sequence.
4. Check that the connector lines and spacing clearly show the hierarchy.
5. Resize the browser to verify the tree remains usable on smaller screens.

## Note
This is a visual demo component and uses only HTML and CSS.
