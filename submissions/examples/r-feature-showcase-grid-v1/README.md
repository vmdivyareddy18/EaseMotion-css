# Animated Feature Showcase Grid

A responsive feature showcase grid demonstrating animated cards using pure HTML and CSS.

## Features

- Responsive grid layout
- Six feature cards
- Hover lift and scale effects
- Icon placeholders
- Mobile-friendly design
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Replace icons with SVGs
- Add additional feature cards
- Adjust animation timing
- Change colors and spacing
