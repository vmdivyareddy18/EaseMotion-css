# Animated Team Member Profile Cards

A responsive team showcase component featuring profile cards with avatar placeholders, social links, and hover animations.

## Features

- Four profile cards
- Avatar placeholders
- Team member names and roles
- Social media placeholders
- Hover lift and scale effects
- Responsive grid layout
- Pure HTML and CSS

## Files

- demo.html
- style.css

## Usage

Open `demo.html` directly in a browser.

## Customization

- Replace avatar placeholders with images
- Add additional team members
- Customize colors and typography
- Modify animation timing and effects
