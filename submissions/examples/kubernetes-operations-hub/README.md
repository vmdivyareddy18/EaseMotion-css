# Enterprise Kubernetes Operations Hub

A responsive enterprise dashboard demo built using **EaseMotion CSS**.

## Features

- Cluster Status
- Pod Monitoring
- Namespace Analytics
- Node Utilization
- Deployment History
- Container Health
- Responsive Layout

## EaseMotion Classes Used

- ease-fade-up
- ease-slide-up
- ease-fade-in
- ease-hover-lift
- ease-hover-shadow
- ease-hover-grow
