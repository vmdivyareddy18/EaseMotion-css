# FAQ Accordion Component

## Overview

A lightweight, responsive FAQ accordion component built with semantic HTML and pure CSS.

## Features

* Pure HTML and CSS
* No JavaScript required
* Responsive design
* Smooth expand/collapse transition
* Accessible semantic structure using `details` and `summary`

## Usage

```html
<details class="faq-item">
  <summary>Question</summary>
  <div class="faq-content">
    Answer goes here.
  </div>
</details>
```
