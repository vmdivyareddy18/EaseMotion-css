# Ease Aurora Notification Stack

## Overview

A premium notification stack featuring floating glass alerts, glowing status indicators, and subtle motion effects built entirely with HTML and CSS.

## Features

- Animated notification cards
- Aurora glow accents
- Glassmorphism styling
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS

## Folder Structure


Open `demo.html` in any modern browser.
