# Morphing Blob Animation

A pure CSS organic blob animation for hero sections, backgrounds, and decorative panels.

## Features

- Uses border-radius animation to morph the blob shape.
- Includes slow and fast variants.
- Adds subtle floating and rotation changes for life.
- Works as a decorative hero/background accent.
- Respects reduced-motion preferences.

## Usage

Open `demo.html` to preview the blob variants. Use `.ease-blob` and apply `.slow` or `.fast` to adjust pacing.
