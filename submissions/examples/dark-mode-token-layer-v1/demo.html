Dark Mode Token Layer

Click the button to toggle between light and dark themes using data-theme attribute.

Surface Card New

This card uses --ease-color-surface and --ease-color-border tokens. It automatically adapts to the active theme.

How it works

Set data-theme="dark" on the <html> element to activate dark mode. Remove it or set data-theme="light" to go back to light mode. Falls back to system preference automatically.
