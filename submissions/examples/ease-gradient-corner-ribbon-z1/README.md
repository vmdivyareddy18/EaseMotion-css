# Ease Gradient Corner Ribbon

## Overview

A corner ribbon component featuring a smooth animated gradient built with HTML and CSS.

## Features

- Pure HTML & CSS
- Animated gradient ribbon
- Responsive layout
- Lightweight
- Easy to customize

## Folder Structure

```
ease-gradient-corner-ribbon-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

Chrome, Edge, Firefox, and Safari.

## Why This Example

This example provides a reusable corner ribbon that can be added to cards, pricing sections, products, or feature banners with minimal code.
