# Animated Savings Goal Progress Ring

## Overview

Animated Savings Goal Progress Ring is a lightweight CSS animation that simulates savings progress using a circular progress indicator and animated target icon.

## Features

- CSS-only animation
- Circular progress visualization
- Savings goal tracking UI
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="savings-progress-ring">
  <span class="target">🎯</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
