CSS Parallax

Pure CSS parallax with perspective transforms

How It Works

The parallax effect uses CSS perspective and transform: translateZ() with scale(). Elements with negative translateZ appear to move slower, creating depth. No JavaScript needed - hardware-accelerated compositing ensures smooth performance.

Multi-Layer Depth

Background

translateZ(-3px) scale(4) - slowest

Midground

translateZ(-1.5px) scale(2.5) - medium

Foreground

translateZ(0) - normal speed
