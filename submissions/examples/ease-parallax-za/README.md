# CSS Parallax Scrolling

## What does this do?
Pure CSS parallax scrolling using perspective and transform: translateZ() with scale() to create multi-layer depth effects.

## How is it used?
```html
<div class="px-container">
  <div class="px-layer px-back">Background</div>
  <div class="px-layer px-front">Foreground</div>
</div>
```

## Why is it useful?
Creates immersive 3D depth with hardware-accelerated CSS transforms. No JavaScript needed. Multi-layer system for rich parallax scenes. Responsive mobile fallback.
