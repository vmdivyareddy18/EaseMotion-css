# Animated Roadmap Progress Timeline

A modern roadmap timeline component featuring animated milestones, progress tracking, status badges, and responsive layouts.

## Features

- Animated roadmap timeline
- Milestone pulse effects
- Completed, In Progress, and Planned states
- Hover interactions
- Responsive design
- Product roadmap visualization

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

You can customize:

- Roadmap stages
- Milestone colors
- Status labels
- Timeline spacing
- Animation duration

## Use Cases

- Product roadmaps
- Open source project planning
- Startup launch timelines
- Feature release schedules
- Development progress tracking
