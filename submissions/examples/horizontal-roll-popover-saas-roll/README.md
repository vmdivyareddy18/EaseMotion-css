# CSS Horizontal Roll Popover Component

An accessible, highly structured horizontal roll popover overlay designed for modern SaaS dashboards that exposes transition properties via custom styles metrics.

## Exposed Parameters
- `--roll-duration`: Controls total unrolling velocity path speeds.
- `--roll-easing`: The `cubic-bezier` timing configuration driving smooth lateral unfold arcs.
- `--roll-initial-angle`: Base rotation parameters applied along the Y-axis coordinates prior to launch updates.
