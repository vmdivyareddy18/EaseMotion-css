# Animated Shield Protection Wave

## Overview

Animated Shield Protection Wave is a lightweight CSS animation that simulates active protection using a security shield and expanding wave rings.

## Features

- CSS-only animation
- Expanding protection waves
- Security dashboard UI example
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="shield-wave">
  <span class="shield">🛡️</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
