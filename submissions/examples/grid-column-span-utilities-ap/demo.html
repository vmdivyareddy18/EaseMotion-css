Accessibility Guidance (WCAG 2.1)

      

        

          
Success Criterion 1.3.2

          
Sensory Order

          

            When utilizing placement classes like `col-start-*` or `col-end-*`
            to re-align elements visually, ensure that the DOM hierarchy is
            maintained in logical reading order. Screen readers read DOM order,
            so visual reordering can confuse blind or low-vision users.
          

        


        

          
Reflow & Wrapping

          
Reflow & Zoom (1.4.10)

          

            Ensure grid containers wrap columns cleanly on small viewport
            viewports. Always use responsive modifiers (e.g. `.sm-col-span-full`
            or `.md-col-span-6`) on elements spanning large spans on desktop
            screens to support up to 400% zoom.
