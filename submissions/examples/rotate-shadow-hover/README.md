# Rotate Shadow Hover

**What does this do?**  
Creates a hover effect where a soft colored shadow appears to orbit around a card or button while the element lifts gently.

**How is it used?**  
```html
<article class="shadow-rotate-card">
  <span class="shadow-rotate-kicker">Analytics</span>
  <h2>Revenue pulse</h2>
  <p>Hover the card to rotate the glow.</p>
</article>
```

**Why is it useful?**  
It gives static cards, pricing panels, and call-to-action surfaces a clear hover affordance without adding JavaScript or extra markup. The shadow is decorative, configurable through CSS variables, and falls back to a regular elevated shadow when motion is reduced.
