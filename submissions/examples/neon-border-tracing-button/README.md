# Neon Border Tracing Button

## Description

A futuristic button featuring a continuously animated neon border.

## Features

- Animated border tracing
- Neon glow effect
- Pure CSS implementation
- Lightweight and reusable

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and hover over the button.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
