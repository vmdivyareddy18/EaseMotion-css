# Alert & Toast Notifications

## What does this do?
CSS alert banners and toast notifications with multiple severity levels and dismissible variants.

## How is it used?
```html
<div class="at-alert at-info"><span>i</span><div><strong>Title</strong><p>Message</p></div></div>
```

## Why is it useful?
User feedback without JS. Color-coded left border, dismissible alerts, toast stacking.
