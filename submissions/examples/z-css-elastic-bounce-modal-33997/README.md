# CSS Elastic Bounce Modal

Pure CSS animated modal utilizing an Elastic Bounce transition for Marketing Landing Page layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS custom properties
- Elastic bounce interaction
- Marketing aesthetics
- Glassmorphism styling
- Smooth 60fps animations

## Variables

```css
--duration
--ease
--primary
--secondary
```
