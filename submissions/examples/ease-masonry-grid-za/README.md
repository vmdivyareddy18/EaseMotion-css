# Masonry Grid Layout

## What does this do?
A CSS masonry grid layout with multi-column layout, animated staggered item reveals, and CSS-only category filtering using radio buttons.

## How is it used?
```html
<div class="ms-masonry">
  <figure class="ms-item ms-photo">...</figure>
</div>
```

## Why is it useful?
Creates Pinterest-style irregular grid layouts using CSS columns. No JavaScript for layout or filtering. Responsive, with shimmer loading and staggered animations.
