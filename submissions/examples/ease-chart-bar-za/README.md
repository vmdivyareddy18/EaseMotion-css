# CSS Bar Chart

## What does this do?
Pure CSS bar chart component with vertical and horizontal orientations, animated bar growth, grouped bars, and color-coded categories.

## How is it used?
```html
<div class="bc-chart bc-vertical"><div class="bc-bar-wrap"><div class="bc-bar bc-bar-1" style="height:80%"></div><span class="bc-label">Label</span></div></div>
```

## Why is it useful?
Creates data visualization charts without JavaScript or SVG. Vertical and horizontal bar charts with animated transitions. Grouped bars for comparison. Color gradients per category. Responsive layout. Customizable via CSS variables.
