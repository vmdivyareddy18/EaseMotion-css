# Staggered Fade-In Grid Animation

A CSS grid where each cell fades in with staggered delays using CSS custom properties for timing. Each item sequentially reveals with a smooth upward fade.

## Files

- `demo.html` - 3x3 grid with staggered items
- `style.css` - fade-up keyframes with calc(var(--i) * delay)

## Usage

Open `demo.html` to see grid items fade in from top-left to bottom-right.
