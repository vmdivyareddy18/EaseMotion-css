# CSS-Only Spotlight Hover Card

### What does this do?
Implements an interactive modern card component that fades in an ambient radial spotlight illumination mask completely over pure CSS layout tracking filters upon cursor focus.

### How is it used?
Incorporate the structural modifier class `.spotlight-card` directly onto element containers. Ambient masks execute cleanly via nested pseudo-elements (`::before`).

### Why is it useful?
It updates default dashboard lists with highly immersive micro-interactions without spinning up heavy JavaScript cursor coordinate listeners, minimizing layout calculations.
