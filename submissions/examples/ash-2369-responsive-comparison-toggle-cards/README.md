# Responsive Comparison Toggle Cards

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive pricing comparison component with toggle controls and feature comparison cards.

## Features

- Pricing comparison cards
- Billing toggle UI
- Featured plan highlighting
- Responsive grid layout
- Glassmorphism styling
- Hover animations
- Mobile-friendly design
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="billing-toggle">
    <button class="toggle-btn active">
        Monthly
    </button>
</div>
```

## Main Classes

```css
.billing-toggle
.toggle-btn
.comparison-grid
.plan-card
.featured
.badge
```
