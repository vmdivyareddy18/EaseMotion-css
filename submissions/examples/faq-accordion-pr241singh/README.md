# FAQ Accordion Component

A modern **FAQ Accordion** component built using **HTML5** and **CSS3** for the EaseMotion CSS examples collection.

---

## ✨ Overview

The **FAQ Accordion Component** provides a clean and responsive way to organize frequently asked questions using semantic HTML and pure CSS. It includes multiple visual styles, smooth expand/collapse interactions, and accessibility-friendly markup without requiring JavaScript.

This component is suitable for documentation websites, landing pages, SaaS products, help centers, knowledge bases, portfolios, and e-commerce websites.

---

## 🚀 Features

- 📖 CSS-only accordion
- 📝 Semantic HTML using `<details>` and `<summary>`
- ➕ Animated expand/collapse indicator
- 🎨 Multiple visual variants
- 🪟 Glassmorphism style
- 🌈 Gradient style
- 🎯 Filled accordion
- 📄 Minimal accordion
- ✨ Smooth hover animations
- 📱 Fully responsive layout
- 🌙 Dark mode support
- ♿ Reduced motion support
- ♿ Keyboard accessible
- 🎯 No JavaScript required
