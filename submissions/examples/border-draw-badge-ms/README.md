# Border Draw Badge

Resolves #42433.

### What does this do?
Adds a Border Draw Badge component to EaseMotion CSS. It uses SVG `stroke-dasharray` and `stroke-dashoffset` to create a smooth, continuous line drawing effect around the badge on hover, paired with EaseMotion's `ease-hover-pop` for subtle scaling.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Hover over the badge to trigger the line draw animation.

### Why is it useful?
Provides a highly interactive, modern portfolio style badge that draws attention to skills or tags without overwhelming the UI.
