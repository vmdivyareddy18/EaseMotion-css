# ease-explosion-shockwave

**Closes #13746**

Expanding shockwave rings that rapidly scale from `scale(1)` to `scale(3)` with opacity fade, staggered for a multi-ring effect.

## Acceptance Criteria

- [ ] Rings expand from `scale(1)` to `scale(3)` with opacity fading to 0
- [ ] Animation completes in ~0.5s (fast)
- [ ] Multiple rings with staggered delays (0.1s, 0.25s, 0.4s)
- [ ] Center point with "Explode" trigger button

## Files

- `style.css` – Shockwave keyframes, ring styles, staggered delays
- `demo.html` – Center point with explode button
- `README.md` – Issue documentation
