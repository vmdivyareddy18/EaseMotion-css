# Animated Chat Interface / Messenger UI

A clean, modern, and fully responsive **Chat Interface Component** with smooth message animations, typing indicators, and auto-scroll behavior. This example demonstrates how to combine **EaseMotion CSS** for message entrance animations, continuous typing effects, and smooth transitions to create an engaging messaging experience.

## 🚀 Features

- **Smooth Message Entrances**: Received messages slide in from the left using `ease-slide-in-left`, while sent messages slide in from the right using `ease-slide-in-right`.
- **Typing Indicator**: Three pulsing dots using `ease-pulse` with staggered delays simulate someone typing.
- **Auto-Scroll**: Chat automatically scrolls to the newest message for seamless conversation flow.
- **Interactive Send Button**: Send button scales up on hover using `ease-hover-grow` for tactile feedback.
- **Code Snippets in Messages**: Support for displaying code snippets within message bubbles.
- **Date Separators**: Visual date dividers with smooth fade-in animations.
- **Online Status Indicator**: Green dot indicator for user availability.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing and layout.
