# Animated Notification Chip UI Example

## 1. What does this do?

This submission provides notification chips with bounce animations and color transitions. The chips animate when updates arrive and support multiple status variants.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example includes chips that bounce and change color on interaction. Copy the `.notification-chip` markup and CSS into your project for animated notification badges.

---

## 3. Why is it useful?

Animated notification chips grab attention and provide clear feedback on system updates. The color variants and bounce effects make status changes feel responsive and rewarding.
