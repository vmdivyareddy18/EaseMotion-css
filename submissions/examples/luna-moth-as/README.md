# Luna Moth

A pure CSS and HTML luna moth resting on a twig in the moonlight, its pale green wings slowly beating.

## What it shows

- Four wings from radial gradients, the hindwings cut with clip-path into the moth's long trailing tails
- Eyespots layered from stacked radial gradients on each forewing
- The wings beating in mirrored pairs, hinged at the body, while the whole moth breathes
- A cratered full moon glowing behind, with wing scales drifting off as dust
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
