# CSS Wobble-Focus Modal for E-Commerce Checkout Layouts

## Overview

This project showcases a responsive checkout modal featuring a smooth **Wobble-Focus** entrance animation built entirely with HTML and CSS. The modal presents essential checkout information in a clean and modern interface while emphasizing accessibility, responsiveness, and ease of customization without relying on JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- CSS-only Wobble-Focus entrance animation
- Responsive checkout modal layout
- Animated overlay with centered modal
- Interactive call-to-action button
- Keyboard-accessible focus styles
- CSS custom properties for easy customization
- Modern card-based design
- No JavaScript or external libraries required

## Customization

The component uses CSS custom properties for quick styling.

```css
:root{
    --primary:#3b82f6;
    --primary-hover:#2563eb;
    --surface:#ffffff;
    --text:#1f2937;
    --muted:#6b7280;
    --radius:22px;
}
```

You can customize:

- Primary button color
- Background overlay
- Modal border radius
- Typography
- Shadows
- Animation duration
- Button styles
- Spacing and layout
