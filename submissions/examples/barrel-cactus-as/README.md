# Barrel Cactus

A pure CSS and HTML barrel cactus in the desert, its crown of yellow flowers opening in the heat.

## What it shows

- Vertical ribs from a four stop repeating linear gradient that gives each rib a shaded and a sunlit face
- A halo of spines from a repeating conic gradient masked to a ring, so they bristle from the body edge only
- Ray flowers from repeating conic gradients masked to rings, opening in sequence on staggered delays
- A layered desert of grit, pebbles and a faded mesa behind
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`, leaving the flowers open.
