# Elastic Bounce Accordion

A pure HTML and CSS accordion for minimalist tech layouts, designed for system dashboards and AI control surfaces.

## What does this do?

It presents compact configuration sections with a controlled elastic bounce interaction that feels technical, calm, and responsive.

## How is it used?

Use the provided HTML structure in `demo.html` and link `style.css` in your page.

```html
<link rel="stylesheet" href="style.css" />
```

## Why is it useful?

It fits EaseMotion CSS because it combines expressive motion with restraint, making a UI interaction feel more deliberate and polished without becoming noisy. The layout also gives each panel a clearer information hierarchy for technical interfaces, and the implementation is keyboard-friendly and reduced-motion aware.
