# Device Session Trust List

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A modern security dashboard component for managing trusted devices, active sessions, and account access history.

## Features

- Device trust management
- Active session list
- Current device highlighting
- Trusted status badges
- Risk review indicators
- Session revoke actions
- Security summary cards
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="session-card current">
  Current Device
</div>
```

## Main Classes

```css
.sessions-list
.session-card
.current
.trusted
.review
.revoke-btn
```
