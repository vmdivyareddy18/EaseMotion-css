# Doodle Jump Clone

A classic Doodle Jump game clone where you control a character that bounces on platforms to reach new heights!

## 🎮 Gameplay

Guide your character upward by bouncing on platforms. The higher you go, the more points you score! But be careful - if you fall off the screen, it's game over!

## ✨ Features

- **Smooth Controls** – Arrow keys or A/D to move left/right
- **Jump Mechanics** – Space bar or click to jump
- **Dynamic Platforms** – Some platforms move left and right
- **Score Tracking** – Score increases as you climb higher
- **Visual Effects** – Particle effects, glow, trails, and parallax stars
- **Responsive Design** – Works on desktop and mobile devices
- **Touch Support** – Touch to jump, drag to move
- **Game Over & Restart** – Restart with button or click

## 🚀 How to Play

1. **Move** – Use ← → or A D keys
2. **Jump** – Press SPACE or click/tap the screen
3. **Climb** – Land on platforms to bounce higher
4. **Score** – Points increase as you climb
5. **Game Over** – Fall off the screen to end

## 🎨 Visual Features

- Colorful gradient platforms
- Character with animated eyes and smile
- Glowing effects and particle bursts
- Parallax star background
- Smooth camera following

## 🛠️ Technologies Used

- HTML5
- CSS3 (Animations, Gradients, Flexbox)
