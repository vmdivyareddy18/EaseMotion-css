# CSS 3D Flip Tooltip for SaaS Showcase Layouts

A modern, pure CSS 3D Flip Tooltip designed for SaaS pricing cards and showcase interfaces. The component uses smooth flip animations, CSS custom properties for easy customization, and supports keyboard accessibility.

## Features

- 🎯 Pure CSS implementation (No JavaScript)
- 🔄 Smooth 3D flip animation
- 📱 Fully responsive
- ♿ Keyboard accessible using `:focus`
- 🎨 Easily customizable with CSS variables
- 💼 Designed for modern SaaS interfaces

## Customization

You can modify the following CSS variables:

```css
:root {
  --tooltip-bg: #1e293b;
  --tooltip-color: #ffffff;
  --tooltip-radius: 12px;
  --tooltip-width: 280px;
  --flip-duration: 0.6s;
  --flip-easing: ease;
}
```
