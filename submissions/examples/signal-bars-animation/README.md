# Signal Bars Animation

## Overview

A CSS animation inspired by mobile network indicators where signal bars fill sequentially and loop continuously.

## Features

- Sequential bar filling
- Pure CSS animation
- Infinite loop
- Connectivity status indicator
- Lightweight and reusable

## Usage

```html
<div class="signal">
  <span class="bar"></span>
  <span class="bar"></span>
  <span class="bar"></span>
  <span class="bar"></span>
</div>

```

## CSS Classes

### `.signal`

Container for the bars.

### `.bar`

Individual animated signal bars.
