# Responsive Role Access Matrix Card

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive role-based access matrix component for displaying permissions across different user roles in admin dashboards and enterprise applications.

## Features

- Responsive permission matrix
- Role comparison layout
- Allow/Deny indicators
- Mobile-friendly design
- Hover interactions
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="matrix-row">

    <span>Manage Users</span>

    <span class="allow">✓</span>

    <span class="deny">✕</span>

</div>
```

## Main Classes

```css
.matrix-card
.matrix-header
.matrix-row
.allow
.deny
```
