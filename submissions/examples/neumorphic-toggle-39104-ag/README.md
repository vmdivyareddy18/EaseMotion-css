# Neumorphic Fluid Toggle Switch

An interactive toggle component demonstrating soft shadow indentation transitions.

---

## What is Included

1. **`demo.html`**: A showcase dashboard hosting switch checkboxes, slider elements, and mode labels.
2. **`style.css`**: Extruded shadows, inset shadow formulas, color-mix values, translate transitions, and motion overrides.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Observe the soft neumorphic card containing the toggle switch.
3. Click the toggle switch.
4. Verify that the knob slides smoothly to the right, and the track transitions into a glowing indigo theme with inset depth.
