# Seoul Street Craft // High-Energy Street Food Landing Page

A bold, high-contrast brutalist design showcase tailored for modern food trucks and street food collectives using EaseMotion CSS layout logic.

## Design Architecture
- **Brutalist Overlays:** Implements industrial heavy borders (`2px solid`) matched with radial neon highlights to emulate dark night-market aesthetics.
- **Asymmetric Hover Physics:** Menu cards use custom hardware-accelerated translations to dynamic state vectors when triggered.
- **Clean Document Isolation:** Follows rigid zero-pollution structure limits.

## Framework Tokens Showcased
- Structural Scoping: `.ease-layout-section`, `.ease-container`
- Presentation Matrix: `.ease-grid-3x`, `.ease-grid-2x`
- Micro Animation Engine: `.ease-fade-up`, `.ease-delay-*`
