# Animated Glassmorphism Pricing Card

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A premium pricing card featuring glassmorphism styling, floating animations, gradient glow effects, feature lists, and modern SaaS-inspired design patterns.

## Features

- Glassmorphism card design
- Animated floating effect
- Gradient glow background
- Popular badge
- Pricing display
- Feature checklist
- Hover lift interaction
- CTA button animation
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="pricing-card">
  Pricing Content
</div>
```

## Main Classes

```css
.pricing-card
.popular-badge
.price
.features
.cta-btn
```
