# Ease API Endpoint Health Matrix

## Overview
A CSS-only dashboard component for displaying API endpoint health in a responsive matrix.

## Features
- Responsive grid layout
- Animated endpoint status
- Hover effects
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` in any modern browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Designed for desktop, tablet, and mobile screens.

## Why EaseMotion CSS?
Provides a reusable animation-first component for API monitoring dashboards, backend systems, cloud platforms, and DevOps interfaces.
