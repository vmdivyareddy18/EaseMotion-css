# CSS Scroll Marker Navigation Utilities

Relates to feature request **#41195**.

## 1. What does this do?

Provides utility classes (`.ease-scroll-container` and `.ease-scroll-section`) that use the emerging CSS **Scroll Marker** specification. It automatically generates interactive navigation dots (or markers) along the edge of a scrolling container — all powered natively by the browser without any JavaScript event listeners or Intersection Observers.

## 2. How is it used?

The implementation uses the emerging CSS Scroll Markers specification (`scroll-marker-group`, `scroll-marker`, `::scroll-marker`, `:target-current`).

**HTML Setup**
```html
<div class="ease-scroll-container">
  <section class="ease-scroll-section">Section 1</section>
  <section class="ease-scroll-section">Section 2</section>
  <section class="ease-scroll-section">Section 3</section>
</div>
```

**CSS Mechanism**
```css
.ease-scroll-container {
  /* Tells the container to group the markers generated by its descendants */
  scroll-marker-group: after; 
}

.ease-scroll-section {
  /* Tells the browser to generate a marker for this section */
  scroll-marker: auto;
}

/* Style the generated marker (the dots!) */
.ease-scroll-section::scroll-marker {
  content: "";
  width: 12px; height: 12px;
  border-radius: 50%;
  background-color: gray;
}

/* Style the active dot when its section is scrolled into view */
.ease-scroll-section::scroll-marker:target-current {
  background-color: purple;
}
```

## 3. Why is this useful for EaseMotion CSS?

Long documents, presentations, and carousels often rely on JavaScript to create section indicators and sync them with scroll position. CSS Scroll Markers provide a modern, native approach to building these interfaces, vastly reducing complexity and improving performance. 

By adding these utilities, EaseMotion CSS provides developers with bleeding-edge CSS capabilities in an easy-to-use format.

*(Note: CSS Scroll Markers are currently an emerging specification. You may need to enable Experimental Web Platform Features in Chromium-based browsers to see the demo working natively.)*
