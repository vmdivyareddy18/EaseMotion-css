CSS Scroll Marker Navigation Utilities

Native, CSS-only scroll navigation dots generated automatically using the emerging scroll-marker specification. Zero JavaScript required.

📜 scroll-marker-group 🎯 ::scroll-marker ✨ target-current

🎯 How it works: The dots on the right side of the scroll area are generated entirely by CSS. They automatically track the active section in view and can be clicked to scroll to that section.

01 / Introduction

This is a native CSS scroll container. Notice the navigation dots on the right? They are not HTML elements.

02 / Features

The dots are generated via the ::scroll-marker pseudo-element and grouped using scroll-marker-group.

03 / Examples

When this section enters the view, the corresponding marker automatically highlights via the :target-current pseudo-class.

04 / Conclusion

Clicking a marker natively scrolls to its associated section because it behaves like an anchor link automatically.

🔧 CSS Reference

/* 1. Define the container and marker group placement */
.ease-scroll-container {
  overflow-y: auto;
  scroll-behavior: smooth;
  scroll-marker-group: after; /* Places markers at the end edge */
}

/* 2. Tell each section to generate a marker */
.ease-scroll-section {
  scroll-marker: auto;
}

/* 3. Style the auto-generated markers */
.ease-scroll-section::scroll-marker {
  content: "";
  width: 12px;
  height: 12px;
  border-radius: 50%;
  background-color: #475569;
}

/* 4. Style the active state when the section is in view */
.ease-scroll-section::scroll-marker:target-current {
  background-color: #a78bfa;
  transform: scale(1.3);
}
