# Advanced: Business Landing — Spa Wellness

A responsive spa and wellness business landing page built using **HTML**, **CSS**, and **EaseMotion CSS**. This example showcases a luxurious, calming website experience using only existing EaseMotion CSS animation and utility classes.

## Features

* Responsive navigation bar
* Elegant hero section with booking CTA
* Signature treatments section
* Wellness packages and pricing
* Modern responsive layout
* Clean, spa-inspired aesthetic
* Smooth entrance animations
* Interactive hover effects

## EaseMotion CSS Classes Showcased

### Entrance Animations

* `ease-fade-down`
* `ease-fade-up`
* `ease-fade-left`
* `ease-fade-right`
* `ease-scale-in`

### Hover Animations

* `ease-hover-grow`
* `ease-hover-lift`
