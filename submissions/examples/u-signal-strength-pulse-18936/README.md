# Signal Strength Pulse

An animated network signal strength indicator.

## Features

- Five signal bars
- Sequential pulse animation
- Weak signal mode
- Medium signal mode
- Strong signal mode
- Responsive layout
- Pure HTML, CSS and JavaScript

## Files

- demo.html
- style.css

## Usage

Open demo.html in your browser.

Use the buttons to switch between signal strengths and observe the pulsing animation.
