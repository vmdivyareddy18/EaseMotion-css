# Global Future Economy Intelligence Platform

A global future economy intelligence dashboard built with EaseMotion CSS tokens. Tracks GDP, trade volume, inflation, unemployment, sector performance, currency exchange rates, economic indicators, commodity prices, trade partners, and economy snapshot in a gold-amber dark theme.

## Features

- **Stat cards** — global GDP, daily trade volume, inflation rate, unemployment
- **Sector performance** — 6 sectors (financial, technology, healthcare, energy, agriculture, transport) with color-coded QoQ bars
- **Currency exchange** — 6 currency pairs (EUR, GBP, JPY, BRL, CNY, INR vs USD) with change indicators
- **Key indicators** — consumer confidence, PMI manufacturing/services, industrial production, retail sales, housing index
- **Commodity prices** — crude oil, gold, silver, copper, wheat, natural gas
- **Top trade partners** — 5 countries ranked by trade volume with bars
- **Economy snapshot** — economies tracked, market cap, new businesses, digital economy share, workforce, projected growth
- Responsive 12-column grid with gold-amber dark (default) and light mode support
- Reduced-motion compliant
