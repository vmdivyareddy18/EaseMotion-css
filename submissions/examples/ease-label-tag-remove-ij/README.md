# Label Tag Remove

Tags/labels with an animated remove button. Hover reveals the X button; clicking it animates the tag out with scale + opacity. Add new tags via the input.

## Features

- Tags with colored left border accent
- Remove button fades in on tag hover
- Scale + opacity animation on removal
- Add new tags via input field (Enter or click Add)
- Random accent color on each new tag

## Usage

Set `--ltr-removing` to `1` on `.ltr-tag` to trigger the removal animation. The tag fades out, shrinks, and collapses width before being removed from the DOM after 400ms.
