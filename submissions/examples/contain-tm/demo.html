Contain Layout

Declares that an element is isolated from layout changes in its subtree.

CSS Property: contain

Variants Demo

contain: layout Contain Layout variant
contain: style Contain Layout variant

contain: strict Contain Layout variant
contain: layout
Layout isolated from parent
contain: paint
Paint containment hides overflow
contain: strict
Strict: all containment types
contain: content
Content containment applied

Usage

<div class="contain">Content here</div>
