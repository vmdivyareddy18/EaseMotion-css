# Animation: Documentation Sidebar Navigation

A documentation-style sidebar navigation component inspired by GitBook, Docusaurus, Stripe Docs, and Vercel Docs. Sections can be expanded and collapsed using native HTML details elements while maintaining a clean developer-focused layout.

## Features

- Pure HTML & CSS
- Expandable navigation groups
- Documentation-style layout
- Responsive design
- Hover interactions
- Reduced motion support
- No JavaScript required

## Usage

```html
<details>

    <summary>Components</summary>

    <a href="#">Buttons</a>
    <a href="#">Cards</a>

</details>
```

Sections can be expanded or collapsed to organize documentation content.

## Why is it useful?

Documentation websites often contain large amounts of structured content. This component provides a clean navigation pattern that helps users explore guides, APIs, components, and utilities while maintaining readability and organization.
