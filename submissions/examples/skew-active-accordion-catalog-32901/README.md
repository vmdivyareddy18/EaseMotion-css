# Skew-Active Accordion — Product Catalog Layouts

A production-optimized pure CSS accordion variant introducing horizontal skew displacement physics to active storefront rows.

## Specifications
- **Mechanical Physics**: Executes instant dynamic shear transformations along the horizontal axis to highlight user focus shifts.
- **Declarative Operations**: Commands layout dimension modifications and text visibility exclusively via structural selector trees to operate cleanly without scripts.
- **Hardware Layering**: Offloads style calculations directly onto background composition pipelines to shield browser paint cycles.
