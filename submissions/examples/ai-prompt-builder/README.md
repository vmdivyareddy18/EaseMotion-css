# AI Prompt Builder Studio

A modern **AI Prompt Builder Studio** built using **HTML** and **CSS**. This example demonstrates a clean, responsive interface for creating, organizing, previewing, and managing AI prompts for various real-world applications.

---

## ✨ Features

- 🤖 Modern AI Prompt Builder interface
- 📂 Prompt template categories sidebar
- 📝 Large prompt editor
- 🏷️ Variable placeholder chips
- 👀 Live prompt preview section
- 📊 Prompt statistics panel
- 📚 Recent prompt history cards
- 🎨 Smooth hover effects and transitions
- 📱 Fully responsive layout
- 🌙 Automatic dark mode support
- ♿ Reduced motion accessibility support

## 🚀 Included Components

### Prompt Categories

- Content Writing
- Coding Assistant
- Email Generator
- Marketing
- Education
- Product Description
- Blog Generator
- Brainstorm Ideas

### Prompt Editor

- Large editable prompt area
- Responsive textarea
- Clean writing interface

### Variable Panel

Supports reusable variables such as:

- `{{client_name}}`
- `{{company}}`
- `{{product}}`
- `{{tone}}`
- `{{language}}`
- `{{industry}}`

### Live Preview

Displays a formatted preview of the generated prompt for a better editing experience.

### Prompt Statistics

Shows useful information including:

- Character count
- Word count
- Variable count
- Estimated reading time

### Recent Prompts

Example prompt cards including:

- SQL Generator
- Resume Optimizer
- Blog Writer
- Code Reviewer
- Cold Email Generator
- Meeting Summary
