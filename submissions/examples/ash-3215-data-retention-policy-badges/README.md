# Data Retention Policy Badges

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive component that displays data retention policies using visually distinct badges for privacy, governance, and compliance dashboards.

## Features

- Data retention policy cards
- Color-coded retention badges
- Responsive layout
- Compliance-focused design
- Hover animations
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="policy-card">

    <div class="policy-info">
        <h3>Financial Records</h3>
    </div>

    <span class="badge long">
        7 Years
    </span>

</div>
```

## Main Classes

```css
.policy-list
.policy-card
.policy-info
.badge
.short
.medium
.long
.permanent
```
