# Animated Theme Toggle Switch

A modern and responsive **Dark/Light Theme Toggle Switch** built using **HTML, CSS, and JavaScript**. The component features smooth animations, icon transitions, and uses CSS custom properties for easy customization.

## ✨ Features

* 🌞 Light/Dark theme toggle
* 🌙 Smooth Moon ↔ Sun icon transition
* 🎯 Sliding thumb animation
* 🎨 Smooth background color transition
* ⚡ CSS Custom Properties (Variables)
* 📱 Responsive design
* ♿ Accessible button with `aria-label`
* 🧩 Easy to integrate into any project
