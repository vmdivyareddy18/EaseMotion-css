# Holographic Glassmorphic Profile Card

A premium dashboard profile card featuring shifting light reflection sweeps.

## What does this do?
It builds a glassmorphic content module containing dynamic color-dodge overlays that shift back-and-forth along gradients depending on pointer positions, offering an iridescent style.

## How is it used?
Configure standard glass layouts with the holographic containers and connect the active movement shaders:

```html
<div class="holographic-card" tabindex="0">
  <div class="holographic-foil"></div>
  <div class="card-inner">
    <!-- Profile Details -->
  </div>
</div>
```

## Why is it useful?
It provides premium visual feedback options for dashboard layouts, leveraging standard CSS background adjustments.
