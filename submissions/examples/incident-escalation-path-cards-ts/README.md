# Incident Escalation Path Cards

## What does this do?

Shows detected, triaged, escalated, mitigated, and resolved incident response states.

## How is it used?

```html
<div class="escalation-path">
  <article class="path-card is-active">Escalated</article>
</div>
```

## Why is it useful?

Incident teams need a compact visual path that shows current response state at a glance.
