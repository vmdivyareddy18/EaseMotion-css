# Escalation Path Map

Escalation Path Map is a responsive incident-response panel that shows the current support escalation path. It separates completed, active, and upcoming escalation stages so teams can see what happens next.

## Features

- Escalation timing summary
- Three-stage incident response path
- Completed and active status markers
- Responsive layout for smaller screens

## Files

- `demo.html` contains the escalation path markup.
- `style.css` contains the timeline styling and responsive rules.
