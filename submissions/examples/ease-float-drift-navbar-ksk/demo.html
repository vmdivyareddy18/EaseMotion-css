Float-Drift Navigation

Interactive Parameters

Hover over the links to see a sliding pill drift animation. Scroll down the page—the navbar remains fixed at the top, gently floating vertically using fluid CSS keyframe loops. Click the "Theme" nav link to switch between Dark and Light mode.

Creative portfolios rely on subtle, premium interaction design. Instead of standard block layouts, pill-shaped floating navbars create a distinct, modern spatial layout. Adding a very low-frequency vertical oscillation animation simulates a natural "drift" that makes the interface feel organic.

This component is implemented 100% in pure HTML and CSS. Mobile responsive viewports utilize a checkbox hack to morph the hamburger menu button and slide-reveal the dropdown nav drawer with matching spring transitions.
