# Unicode Bidi Utilities

1. What does this do? Adds unicode-bidi utility classes to control text directionality override and isolation.
2. How is it used? Apply `.bidi-isolate` or `.bidi-bidi-override` along with direction properties to elements.
3. Why is it useful? It ensures proper rendering and layout of mixed direction (LTR and RTL) text for internationalized web pages.
