# Isometric Skew Calendar (`ease-skew-calendar-ksk`)

1. What does this do?  
Displays a monthly calendar widget tilted in 3D isometric space where individual day cells translate outwards on the Z-axis with smooth hover scale and glowing shadows.

2. How is it used?  
Nest the `.calendar-card` dashboard widget inside a `.calendar-perspective` frame, containing a week grid layout of `.calendar-day` items.

3. Why is it useful?  
It showcases advanced CSS-only 3D transforms (`transform-style: preserve-3d`) and Z-axis elevation changes, providing a premium SaaS component layout.
