Accessibility Guidance (WCAG 2.1)

Success Criterion 1.4.10

Reflow & Writing-Mode Flexibility

When text wraps or reading coordinates rotate, padding must scale flow-relative. Using physical padding (like top/bottom) on rotated text boxes often compresses lines, clipping letters. Logical properties preserve character clearances, preventing readability degradation.

Bi-directional

Directional Reading Order (1.3.2)

For users reading right-to-left (RTL) or vertical layouts, preserving correct logical text blocks is crucial. Applying logical padding block ensures spacing aligns correctly with the flow of read blocks, creating a comfortable visual pacing that matches the domestic syntax direction.
