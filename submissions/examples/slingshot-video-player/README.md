# Slingshot Video Player

A responsive travel-inspired video player component with a slingshot-style play button animation using pure HTML and CSS.

## Features

- Pure HTML & CSS
- No JavaScript required
- Slingshot play button animation
- Responsive design
- Accessible keyboard focus
- Travel app inspired UI

## Folder Structure

```
slingshot-video-player-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in a browser.
2. Hover or focus on the play button to experience the animation.
3. Customize colors using the CSS variables.

## Technologies

- HTML5
- CSS3
- EaseMotion-style animation classes

## Accessibility

- Keyboard-focusable play button
- Responsive layout
- Semantic HTML structure
