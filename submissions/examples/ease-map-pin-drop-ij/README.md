# Map Pin Drop (#36806)

Pins animate in from above with a bouncy landing, simulating a drop placement on a map surface. Staggered delays create a sequential reveal.

## CSS Custom Properties

| Property         | Default   | Description                        |
| ---------------- | --------- | ---------------------------------- |
| `--pin-color`    | `#ff375f` | Accent color for pin dropshadow    |
| `--pin-size`     | `40px`    | Font size of the pin emoji         |
| `--bounce-height`| `60px`    | Drop height before bounce          |
| `--drop-duration`| `0.8s`    | Duration of the drop animation     |

## Interactive Controls

Click "Drop Pins" to replay the animation. Tweak drop height, pin size, and timing with the live variable panel.
