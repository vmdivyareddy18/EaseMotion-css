# Morph Context Menu

A responsive **Morph Context Menu** inspired by dark mode interfaces. The menu smoothly scales into view using CSS transitions without JavaScript.

## Features

- Pure HTML & CSS
- Morph (scale) animation
- Responsive design
- Accessible structure
- No JavaScript required

## Folder Structure

```
morph-context-menu-srujana/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
