# Vector Effect

vector-effect: non-scaling-stroke — Constant SVG stroke width regardless of scale

## Features
- Uses `--ease-color-primary`
- Uses `--ease-color-success`
- Uses `--ease-radius-full`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Vector Effect</span>
</div>
```

## Why is it useful?

This submission demonstrates the vector effect CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
