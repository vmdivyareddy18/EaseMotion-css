# Gravity Hub – Indoor Climbing Gym Landing Page

A complete, ready‑to‑copy landing page for an indoor climbing gym built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – climbing action imagery with a day pass CTA
- **Wall Types** – bouldering, top rope, lead climbing
- **Membership Tiers** – monthly, quarterly, annual
- **Classes & Instruction** – beginner courses, private coaching
- **Kids Programs** – mini climbers, junior squad, teen competitive
- **Reviews** – climber testimonials
- **Day Pass CTA** – orange call‑to‑action section
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-orange-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-orange-700`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
