# Handoff Readiness Grid

Handoff Readiness Grid is a shift-transition dashboard for operations teams. It presents each handoff area as a clear readiness card so the outgoing and incoming teams can quickly see what is ready, what needs review, and what is blocked.

## Features

- Responsive grid of handoff readiness cards
- Distinct ready, review, and blocked states
- Header callout for time-sensitive shift context
- Compact mobile layout for smaller screens

## Files

- `demo.html` contains the handoff board content.
- `style.css` contains the grid layout, card states, and responsive rules.
