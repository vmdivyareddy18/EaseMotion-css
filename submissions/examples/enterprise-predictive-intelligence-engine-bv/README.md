# Enterprise Predictive Intelligence Engine

Phase 278 dashboard — enterprise predictive intelligence engine for monitoring active predictions, model performance, forecast accuracy, pipeline status, engine metrics, and data drift.

## Features

- **Stat cards** — 4 key metrics: active predictions, models deployed, avg accuracy, forecast horizon
- **Model performance** — 6 models with R² scores and gradient bars
- **Segment forecasts** — 6 forecast vs actual comparisons across regions and metrics
- **Pipeline status** — 5 pipeline stages with health indicators and timestamps
- **Engine metrics** — 6 core KPIs (latency, TPS, refresh cycle, feature store size, uptime, MAE)
- **Data drift monitor** — 5 feature drift scores with severity-coded bars
- **Predictive overview grid** — 8-cell summary of engine-wide stats

## Theme

Teal/cyan predictive theme with embedded dark and light mode support via `prefers-color-scheme`. Reduced motion respected.
