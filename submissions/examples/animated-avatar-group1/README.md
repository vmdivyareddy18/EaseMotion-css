# Animated Avatar Group with Status Indicators

A clean, modern, and fully responsive **Avatar Group Component** perfect for collaboration tools, chat apps, and team dashboards. This example demonstrates how to combine **EaseMotion CSS** for staggered avatar entrances, pulsing online status indicators, and smooth hover spread effects to create an engaging, informative component.

## 🚀 Features

- **Staggered Avatar Entrances**: Each avatar bounces in sequentially using `ease-bounce-in` and `ease-delay-*` for a playful cascading load effect.
- **Pulsing Status Indicators**: Online status dots use `ease-pulse` to continuously draw attention to active users.
- **Hover Spread Effect**: Avatars smoothly spread apart when hovering over the group for better visibility.
- **Interactive Hover States**: Individual avatars scale up on hover using `ease-hover-grow` for tactile feedback.
- **Overflow Counter**: Displays "+X more" when there are too many avatars to show.
- **Tooltips on Hover**: Shows user names when hovering over avatars.
- **Multiple Variations**: Includes compact, standard, and with-names layouts.
- **Status Types**: Supports online (green), away (amber), offline (gray), and busy (red) statuses.
- **Real-World Example**: Includes a project card showing the avatar group in context.
- **Fully Responsive**: Adapts gracefully to mobile screens with adjusted sizing.
