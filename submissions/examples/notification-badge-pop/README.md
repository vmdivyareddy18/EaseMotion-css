# Notification Badge Pop

A CSS-only notification button with a small badge pulse that draws attention without moving the whole layout.

## Files

- `demo.html` - button markup with accessible unread count text.
- `style.css` - button hover, focus-visible, badge pulse, and reduced-motion support.

## Highlights

- Keeps the badge positioned independently of the button label.
- Uses a short scale animation for the unread count.
- Includes a reduced-motion fallback.
