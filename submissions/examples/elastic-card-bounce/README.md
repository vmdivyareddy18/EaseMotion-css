# Elastic Card Bounce

**What does this do?**

Creates a spring-like elastic bounce effect when a user hovers over a card.

**How is it used?**

```html
<div class="elastic-card">
  Content
</div>
```

**Why is it useful?**

The elastic bounce effect provides playful and responsive visual feedback. It helps highlight interactive content while keeping the implementation lightweight and purely CSS-based.
