# CSS Invoice Card

## What does this do?
CSS invoice/receipt card with company details, bill-to section, line item table, totals calculation, and payment status badges.

## How is it used?
```html
<div class="iv-invoice"><div class="iv-top"><div class="iv-brand"><strong>Company</strong></div></div><table class="iv-table"><tr><td>Item</td><td>$Amount</td></tr></table><div class="iv-totals"><div class="iv-total-row iv-grand-total"><span>Total</span><span>$0</span></div></div></div>
```

## Why is it useful?
Provides a complete invoice UI pattern without JavaScript. Professional layout with brand info, billing details, line items table, subtotal/tax/total, and color-coded payment status badges. Ideal for dashboards, order confirmation pages, and billing sections.
