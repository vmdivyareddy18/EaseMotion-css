# Freeze Frost

## What does this do?
A frost/ice crystallization spreading effect that simulates freezing with expanding icy crystals and a blue-white glow.

## How is it used?
Apply the `.freeze-frost` class to a container. Ice crystals spread outward from the center with a cold shimmer.

## Why is this useful?
Provides cold/freeze themed effects for weather apps, game status effects, winter-themed interfaces, and environmental storytelling.
