# CSS Horizontal Roll Accordion for Creative Portfolio Layouts

## Overview

A pure HTML and CSS accordion featuring a smooth horizontal roll animation, designed for creative portfolio websites.

---

## Features

- Pure HTML5 & CSS3
- Horizontal roll animation
- Responsive layout
- Keyboard accessible
- Semantic HTML (`details` / `summary`)
- CSS custom properties
- No JavaScript
