# Ease Ripple Avatar Ring

A CSS-only avatar with expanding ripple rings.

## Features

- Pure HTML & CSS
- Ripple animation
- Responsive
- Lightweight
- Reusable

## Structure

```
ease-ripple-avatar-ring-z1/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any browser.

## Why EaseMotion CSS

A reusable profile animation showcasing clean, lightweight motion effects.
