# Animated Volume Bars Equalizer

A lightweight CSS animation that mimics the moving bars seen in audio players and music visualizers.

## Features

- Pure CSS animation
- Smooth wave-like motion
- Staggered timing for each bar
- Responsive layout
- No JavaScript required

## Files

- `demo.html` — demonstration page
- `style.css` — animation and styling

## Preview

Five animated bars move continuously to create a music equalizer effect.
