# Global Logistics Dashboard

## Overview
Design a shipment and logistics monitoring system.

This project implements the Phase 120 requirements for the EaseMotion CSS catalog, demonstrating advanced UI development using modern CSS techniques without external UI frameworks.

## Features Included
- **Shipment tracking cards**: Built with glassmorphism and CSS grid/flexbox.
- **Delivery metrics**: Built with glassmorphism and CSS grid/flexbox.
- **Fleet overview**: Built with glassmorphism and CSS grid/flexbox.
- **Analytics widgets**: Built with glassmorphism and CSS grid/flexbox.
- **Responsive layout**: Built with glassmorphism and CSS grid/flexbox.

## Architecture
- **Pure CSS**: No preprocessors or Tailwind used.
- **Interactivity**: Utilizes CSS pseudo-classes (`:hover`, `:checked`) for interactivity, avoiding JavaScript where possible.
- **Animations**: Custom keyframes for charts, loaders, and state transitions.
- **Variables**: Fully themeable using CSS custom properties.

## Setup
Simply open `demo.html` in any modern browser to view the interactive dashboard.
