# Fade In Pricing Card (Space Theme)

A pure CSS, responsive pricing card grid featuring a staggered fade-in transition system inspired by cosmic UI design patterns.

## Features
- **Zero JavaScript**: Runs entirely on pure CSS animations.
- **Staggered Delays**: Leverages inline CSS custom variables (`--delay`) to trigger sequencing.
- **Accessible**: Built-in `prefers-reduced-motion` media queries.
- **Responsive**: Adapts perfectly across mobile, tablet, and desktop layouts.

## Usage
Simply structure your pricing container with the utility animation classes:

```html
<div class="pricing-card animate-fade-in" style="--delay: 1;">
  <!-- Card content -->
</div>
```
