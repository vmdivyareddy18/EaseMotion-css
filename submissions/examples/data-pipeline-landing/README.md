## EaseMotion Classes Demonstrated

### Entrance Animations
- `ease-fade-in` вҖ” nav, badge, hero trust line, proof bar
- `ease-slide-up` вҖ” hero heading, subtitle, CTAs, feature cards, pricing cards
- `ease-slide-in-left` вҖ” integrations source column
- `ease-slide-in-right` вҖ” integrations destination column

### Hover Effects
- `ease-hover-lift` вҖ” buttons, logo cards, feature cards, pricing cards, customer cards
- `ease-hover-grow` вҖ” pipeline source/destination nodes, proof logos
- `ease-hover-glow` вҖ” primary CTA button, featured pricing card
- `ease-hover-underline` вҖ” nav links, footer links, feature card links
- `ease-hover-pulse-glow` вҖ” pipeline engine center node

### Looping / Continuous Animations
- `ease-pulse` вҖ” badge dot, decorative orbs
- `ease-rotate` вҖ” pipeline engine icon
- `ease-float` вҖ” integration arrow, docs CTA icon
- `ease-gradient-text-animated` вҖ” hero headline accent, stat numbers

### Glassmorphism
- `ease-glass-card` вҖ” pipeline illustration, feature cards, logo cards, pricing cards, docs CTA

### Delay Staggering
- `ease-delay-100` through `ease-delay-600` вҖ” entrance stagger on all grid items

### Layout Utilities
- `ease-container` вҖ” max-width centered wrapper
- `ease-flex`, `ease-justify-between`, `ease-justify-center`, `ease-items-center`
- `ease-gap-4`, `ease-gap-8`, `ease-flex-wrap`
- `ease-text-center`

## Usage Instructions

1. Copy the `data-pipeline-landing/` folder anywhere on your machine.
2. Open `demo.html` in any modern browser.
3. The page links to `../../easemotion.css` (relative path within this repo). If you move the folder outside the repo, update the `<link>` tag to point to your EaseMotion CSS build:

```html
<!-- Local repo path (default) -->
<link rel="stylesheet" href="../../easemotion.css" />

<!-- CDN or custom path -->
<link rel="stylesheet" href="/path/to/easemotion.css" />
```

4. All content is placeholder-ready вҖ” swap in real copy, real logos, and real links.

## Design Tokens

Colors and spacing are driven by CSS custom properties in `style.css`. Override any token in your own stylesheet:

```css
:root {
  --brand-purple: #7c3aed;
  --brand-blue:   #3b82f6;
  --bg-base:      #080b14;
  --text-primary: #f1f5f9;
}
```

## Technical Notes

- **No build tools** вҖ” pure HTML + CSS, opens directly in a browser.
- **No JavaScript frameworks** вҖ” one small inline `<script>` uses `IntersectionObserver` to trigger scroll-reveal animations by unpausing them when elements enter the viewport.
- **No custom `@keyframes`** вҖ” all animation timing is delegated to EaseMotion classes. The only exception is `flowPulse` on the decorative pipeline connector lines inside the hero illustration, which is a layout-level visual effect (not an entrance or hover animation).
- **Semantic HTML5** вҖ” uses `<nav>`, `<header>`, `<section>`, `<footer>`, `<blockquote>`, `<code>` appropriately.
- **Accessible** вҖ” social icon links have `aria-label` attributes; color contrast meets WCAG AA for body text.
