EaseMotion is a modern CSS animation library that focuses on performance and developer experience. Every component is built with accessibility in mind.

The highlight marker effect works by animating the background-size property from zero to full width, creating a smooth sweep-in effect that feels natural.

Choose from multiple highlight colors using the palette above. Each paragraph tracks its own highlighted state independently.

Built with pure CSS transitions and custom properties, making it easy to integrate into any project without JavaScript frameworks.
