# Bounce Tooltip

Resolves #42411.

### What does this do?
Adds a Bounce Tooltip component to EaseMotion CSS suited for Food Delivery cards. It utilizes pure CSS hover states (`:hover` and `:focus-within`) on the image wrapper to trigger an `ease-bounce` animation on the hidden tooltip bubble, bringing it into view playfully.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. Hover over the burger image to see the tooltip pop and bounce.

### Why is it useful?
Provides an engaging, fun way to show extra information (like ingredients or spice level) without cluttering the card design, using CSS-only physics animations.
