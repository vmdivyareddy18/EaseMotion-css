# Masonry Layout – Items load with stagger animation

> A production-ready masonry layout component that displays items in a responsive grid with a beautiful staggered loading animation. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Responsive masonry grid** – Auto-adjusts columns
- ✅ **Stagger animation** – Items load with graceful delay
- ✅ **Load more** – Add items dynamically
- ✅ **Shuffle** – Randomize item order
- ✅ **Toggle stagger** – Enable/disable animation
- ✅ **Keyboard shortcuts** – L (load), R (reset), S (shuffle), T (toggle)
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
