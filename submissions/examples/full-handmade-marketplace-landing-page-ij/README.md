# Full Handmade Goods Marketplace Landing Page

1. What does this do? A warm, artisanal marketplace landing page showcasing handmade product categories, featured items, artisan stories, and a newsletter CTA using only EaseMotion CSS framework classes.

2. How is it used? Open `demo.html` — every layout, card, button, animation, and responsive behavior is powered by EaseMotion classes (`ease-grid-cols-3`, `ease-card`, `ease-reveal-up`, `ease-hover-grow`, `ease-marquee`, etc.) with zero custom keyframes.

3. Why is it useful? Proves EaseMotion can create visually rich, warm-tone e-commerce pages entirely with framework classes — from product grids to artisan spotlights to scroll-triggered reveals.
