Getting Started

Welcome to the EaseMotion CSS documentation demo. Use the sidebar on the left to explore sections. Click “Animations” to expand its nested categories.

Entrance Animations

Entrance animations bring elements onto the screen — fade-in, slide-in, zoom-in, bounce-in and more.

Looping Animations

Looping animations run continuously — pulse, spin, float, breathing, shimmer, etc.

Hover Animations

Hover animations react to user interaction — lift, glow, tilt, scale and gradient shifts.

Delay Helpers

Utility classes for staggered or delayed animation timing — useful when sequencing multiple elements.

Components

Reusable UI building blocks such as buttons, cards, modals and navbars.

Utilities

Small atomic helpers — spacing, color, typography, layout shortcuts.

Themes

Pre-built theme tokens and dark / light mode support.
