# Glass Surface Panel

## What does this do?

Adds a self-contained glassmorphism surface panel demo with frosted blur, translucent borders, light and dark variants, and accessible focus states.

## How is it used?

```html
<article class="glass-panel">
  <h2>Live metrics</h2>
  <p>Place dashboard, modal, or floating panel content here.</p>
</article>

<article class="glass-panel glass-panel-dark">
  <h2>Dark panel</h2>
  <p>Use the dark variant on darker or high-contrast backgrounds.</p>
</article>
```

## Why is it useful?

Glass panels let EaseMotion users place readable content on rich gradient or image backgrounds while keeping the component lightweight, responsive, and JavaScript-free.
