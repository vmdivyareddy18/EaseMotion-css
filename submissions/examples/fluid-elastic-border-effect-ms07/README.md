# Fluid Elastic Border Effect

A pure CSS hover animation that gives buttons an elastic, liquid-like border interaction.

## Features

- Pure CSS
- No JavaScript required
- Smooth cubic-bezier easing
- Minimal and reusable
- Animation-first design

## Usage

```html
<button class="em-btn-elastic">
  Hover Me
</button>
```

## Effect

- Idle state:
  - Crisp outlined button.
- Hover state:
  - Border stretches elastically.
  - Background smoothly fills.
  - Text color transitions for contrast.

## Why it fits EaseMotion CSS

This effect embraces the EaseMotion philosophy by making interactions feel lively and premium instead of rigid and mechanical.

- Human-readable.
- Animation-first.
- Easily composable with colors, shadows and scaling utilities.
