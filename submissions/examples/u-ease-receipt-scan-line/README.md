# ease-receipt-scan-line

Animated scan-line effect that simulates receipt or document scanning.

## Features

- Horizontal scan line
- Top-to-bottom sweep animation
- Bright glowing scan effect
- Infinite looping animation
- Pure CSS implementation
- Suitable for fintech and document-scanning interfaces

## Usage

Add the scan-line element inside a container:

```html
<div class="receipt">
    <div class="scan-line"></div>
</div>
