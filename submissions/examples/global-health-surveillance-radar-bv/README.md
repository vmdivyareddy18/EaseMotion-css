# Global Health Surveillance Radar

A global health surveillance radar dashboard built with EaseMotion CSS tokens. Tracks active outbreaks, vaccination coverage by region, health metrics, and real-time alerts across the world.

## Features

- **Stat cards** — active outbreaks, global vaccination, recovery rate, active alerts
- **Region surveillance** — 6 countries with critical/elevated/monitoring/clear status
- **Outbreak radar** — visual concentric radar plot with color-coded outbreak dots
- **Vaccination coverage** — progress bars for 5 continents with color severity
- **Key metrics** — infection rate, mortality, hospital capacity, test positivity, R0
- **Live alerts** — timestamped event feed with severity icons
- **Global snapshot** — aggregate data grid (195 countries, 847K active cases, etc.)
- Responsive 12-column grid with dark mode and reduced motion support
