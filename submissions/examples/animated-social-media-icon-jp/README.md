    # Animated Social Media Icon Component

## Description

A CSS-only collection of animated social media icons with smooth hover rotations, scaling effects, glowing shadows, and color transitions.

---

## Features

- 360-degree rotation animation
- Icon scaling on hover
- Glowing shadow effects
- Social platform color variants
- Responsive icon layout
- Lightweight CSS-only implementation
- Reduced motion accessibility support

---

## Classes

- `.social-container` – Wrapper for social icons
- `.social-icon` – Base icon style
- `.facebook` – Facebook color variant
- `.twitter` – Twitter color variant
- `.instagram` – Instagram color variant
- `.linkedin` – LinkedIn color variant
- `.youtube` – YouTube color variant

---

## Accessibility

Supports `prefers-reduced-motion` to disable animations for users who prefer less motion.

---

## Use Cases

- Portfolio websites
- Landing pages
- Website footers
- Contact sections
- Personal profiles
