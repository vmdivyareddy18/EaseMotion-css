# Usage Anomaly Alert Strip

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern alert strip component designed to highlight abnormal usage spikes, unusual traffic patterns, and operational anomalies.

## Features

- Usage anomaly alert banner
- Traffic spike metrics
- Severity indication
- Animated warning pulse
- Action buttons
- Alert details panel
- Glassmorphism styling
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="alert-strip">
  Usage anomaly detected
</div>
```

## Main Classes

```css
.alert-strip
.alert-left
.alert-metrics
.metric
.investigate-btn
.dismiss-btn
```
