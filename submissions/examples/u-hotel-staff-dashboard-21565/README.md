# Hotel Staff Management Dashboard

A responsive **Hotel Staff Management Dashboard** built using **HTML5** and **CSS3** to demonstrate hotel staff scheduling, housekeeping management, attendance monitoring, and daily hotel operations through an interactive dashboard.

---

## Features

- Responsive dashboard layout
- Hotel overview cards
- Staff attendance tracker
- Housekeeping progress panel
- Shift schedule widget
- Recent staff activity feed
- Notifications panel
- Quick action buttons
- Smooth hover animations
- CSS keyframe animations
- Accessibility support

---

## Dashboard Components

### Overview Cards

- Staff On Duty
- Rooms Serviced
- Pending Tasks
- Shift Coverage

### Hotel Widgets

- Staff Attendance
- Housekeeping Progress
- Shift Schedule

### Activity

- Recent Staff Activity
- Notifications Panel

### Quick Actions

- Assign Shift

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable non-essential animations for users who prefer reduced motion.
