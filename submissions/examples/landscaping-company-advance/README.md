# Outdoor Services Business Website

This demo site showcases a natural, green outdoor-services layout using **EaseMotion CSS classes** only.

## 🧩 Sections
- Hero with lush garden imagery
- Services grid
- Project portfolio gallery
- Seasonal packages
- Free estimate form
- Customer testimonials
- Service area map
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — entrance fade animations
- `ease-slide` — sliding entrance for headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs, images, and map
- `ease-hover-opacity-reduce` — hover dimming for service and package cards, testimonials

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll through sections to see entrance animations.
3. Hover over buttons, cards, and images to see interactive effects.
4. Replace placeholder images and text with real outdoor services content.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion animation classes
