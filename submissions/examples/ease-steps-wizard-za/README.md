# CSS Steps Wizard

## What does this do?
Multi-step wizard progress indicator with numbered circles, connector lines, completion states, and form content area. Tracks progress through 5 steps.

## How is it used?
```html
<div class="sw-steps"><div class="sw-step sw-complete"><div class="sw-circle">✓</div><div class="sw-label">Step 1</div></div><div class="sw-connector sw-complete"></div><div class="sw-step sw-active"><div class="sw-circle">2</div><div class="sw-label">Step 2</div></div></div>
```

## Why is it useful?
Visual progress indicator for multi-step forms and wizards without JavaScript. CSS-only states for completed, active, and pending steps. Smooth transitions. Checkmark on completion. Ideal for checkout flows, onboarding, and multi-page forms.
