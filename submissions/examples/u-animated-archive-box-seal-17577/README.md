# Animated Archive Box Seal

## Overview

Animated Archive Box Seal is a lightweight CSS animation that simulates a document being archived into a storage box.

## Features

- CSS-only animation
- Archive box effect
- Document storage animation
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="archive-box">
  <span class="document">📄</span>
  <span class="box">📦</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
