# MusiRent – Musical Instrument Rentals E‑commerce Landing Page

A complete, ready‑to‑copy landing page for an online musical instrument rental business built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – instrument imagery with a browse CTA
- **Rental Categories by Instrument** – guitar, piano, drums, violin
- **How It Works** – choose, ship, play, return
- **Pricing by Instrument/Duration** – monthly rental rates
- **Student Discount** – 25% off with verification
- **Reviews** – customer testimonials
- **Start Renting CTA** – dark call‑to‑action section
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-indigo-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-indigo-700`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
