# Breadcrumb Chevron

## What does this do?
Breadcrumb Chevron component with primary, secondary, and outline variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"bc2-primary\">Breadcrumb Chevron Primary</div>
```

## Why is it useful?
Reusable breadcrumb chevron UI element. Dark theme with CSS custom properties.
