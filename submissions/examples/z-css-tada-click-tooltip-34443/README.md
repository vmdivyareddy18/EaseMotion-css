# CSS Tada Click Tooltip

Pure CSS animated tooltip utilizing a Tada-Click interaction designed for Responsive Dashboard layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Tada animation
- Dashboard aesthetics
- CSS custom properties
- Glassmorphism cards
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--tooltip
--duration
--ease
```
