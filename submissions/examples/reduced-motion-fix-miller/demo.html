Reduced Motion Fix (#11476)

To test this fix:

  1. Open Browser DevTools (F12).
  2. Open Command Menu (Ctrl+Shift+P or Cmd+Shift+P).
  3. Type Reduced Motion and select Emulate CSS prefers-reduced-motion: reduce.
  4. Observe that the animations below stop immediately.

Bouncing Box

Bounce

Spinning Box

Spin

This fix applies a universal selector * override within the prefers-reduced-motion media query to catch all animations, including those defined outside the core variable system.
