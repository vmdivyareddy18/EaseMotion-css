# Animation: Animated Testimonial Wall

A modern testimonial showcase featuring a responsive bento-style grid, profile avatars, and interactive quote cards.

## Features

- Pure HTML & CSS
- Bento-style testimonial layout
- Hover elevation effects
- Responsive design
- Profile avatars
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="testimonial-card">

    <div class="avatar">A</div>

    <h3>Aryan Kumar</h3>

    <p>
        Amazing experience using this product.
    </p>

</div>
```

## Why is it useful?

Testimonials help build trust and credibility for products, services, and portfolios. This component provides an engaging way to showcase user feedback while maintaining a clean and modern design.
