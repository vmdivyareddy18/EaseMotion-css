# Project Roadmap Tracker

A premium CSS-only roadmap tracker for startup launch planning and product journey visualization.

## Description

This component showcases a horizontal project roadmap with connected stages, progress states, and rich stage cards. It is designed to feel like a startup product launch dashboard rather than a generic progress bar.

## Features

- Pure HTML + CSS implementation
- Connected horizontal roadmap with stage cards
- Completed stages visually distinguished by gradient markers
- Current stage highlighted with pulse animation
- Future stages muted for visual hierarchy
- Stage labels, descriptions, and timeline details
- Smooth hover interactions for each stage card
- Fully responsive layout for mobile and desktop

## Usage

1. Copy `demo.html` and `style.css` into your submission folder.
2. Open `demo.html` in the browser.
3. Update stage text and progress structure as needed.

```html
<link rel="stylesheet" href="style.css" />
<main class="roadmap-page">
  <section class="roadmap-panel">
    <!-- roadmap content here -->
  </section>
</main>
```

## Customization

- Change the primary theme by updating `--accent` and `--accent-alt`.
- Adjust the roadmap fill width inside `.roadmap-line--fill`.
- Replace stage text and timelines to fit your own product workflow.
- Modify card spacing and radius for tighter or softer roadmap visuals.
