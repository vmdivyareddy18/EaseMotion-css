# Ease Goblin Sneak Animation

1. What does this do? Creates a suspicious, jerky "sneaking" animation where the element crouches down (squashes), shifts cautiously side to side, freezes momentarily, and then stands back up in a loop.
2. How is it used? Apply `.ease-goblin-sneak-ps` to an element to make it infinitely sneak in place.
3. Why is it useful? Perfect for loading states, playful character animations in web games, or drawing attention to an element in a quirky, non-aggressive way.
