# CSS Scale-Hover Tooltip – Glassmorphism UI Layout

## Overview

A responsive Glassmorphism tooltip component built entirely with HTML and CSS. The component features a smooth scale-on-hover animation, frosted glass styling, keyboard accessibility, responsive layout, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth scale-hover tooltip animation
- Glassmorphism design
- Frosted glass effect using `backdrop-filter`
- Responsive layout
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

Customize the component using:

- `--primary`
- `--secondary`
- `--accent`
- `--glass`
- `--border`
- `--text`
- `--muted`
- `--radius`
- `--duration`
- `--easing`
- `--tooltip-scale`
- `--shadow`

## Files

- `demo.html` – Demo page
- `style.css` – Component styles
- `README.md` – Documentation

## Usage

Open `demo.html` directly in any modern browser.

Hover over or focus on the action buttons to display the animated Glassmorphism tooltip. Colors, animation timing, blur intensity, spacing, and tooltip scaling can all be customized using the CSS variables defined in `style.css`.
