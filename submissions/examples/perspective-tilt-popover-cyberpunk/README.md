# CSS 3D Perspective Tilt Popover

## Description

A pure CSS animated popover featuring a smooth 3D perspective tilt effect designed for Cyberpunk Neon layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- CSS Custom Properties
- Lightweight and reusable

## Usage

```html
<div class="popover">
    <button class="trigger">Launch Panel</button>

    <div class="content">
        <h3>Cyberpunk Neon</h3>
        <p>Pure CSS 3D Perspective Tilt Popover.</p>
    </div>
</div>
```

## Custom Properties

- `--duration`
- `--primary`
- `--bg`
- `--card`
- `--text`

- Chrome
