# Opacity Full on Hover

Element becomes fully opaque on hover — pure CSS.

## Features
- Smooth opacity transition from 0.4 to 1
- Pure CSS, no JavaScript

## Usage
```css
.fade-card { opacity: 0.4; transition: opacity 0.3s; }
.fade-card:hover { opacity: 1; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
