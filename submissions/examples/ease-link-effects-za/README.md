# Animated Link Effects

## What does this do?
CSS link hover animation library with underline slides, highlight bars, icon reveals, text swaps, border effects, background fills, and combined animations.

## How is it used?
```html
<a href="#" class="lk-link lk-underline-slide">Hover me</a>
```

## Why is it useful?
Enhances navigation and inline links with subtle hover animations without JavaScript. 20+ effect variations. Zero JavaScript. Customizable colors and timing via CSS variables. Accessible hover states.
