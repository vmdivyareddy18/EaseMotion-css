# Command Shortcut Cheat Sheet Component

## What does this do?

Provides a modern keyboard shortcut reference interface using keycap-style elements and categorized command groups.

## How is it used?

```html
<div class="shortcut">
    <span>Open Search</span>
    <kbd>⌘</kbd><kbd>K</kbd>
</div>
```

## Why is it useful?

Keyboard shortcut guides improve user productivity and discoverability. This component demonstrates modern documentation and productivity UI patterns using pure HTML and CSS.
