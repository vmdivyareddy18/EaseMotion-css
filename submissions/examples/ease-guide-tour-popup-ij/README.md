# Guide Tour Popup

An interactive tour popup with step indicators and navigation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--popup-bg` | `#2d2d44` | Popup background color |
| `--accent-color` | `#4ecdc4` | Accent color for active step |
| `--step-color` | `#ffd93d` | Color for completed steps |
| `--transition-speed` | `0.3s` | Animation speed |
