# Animated Space Colony Management Dashboard

A futuristic dashboard for managing colony resources, habitat systems, research progress, and expansion initiatives.

## Features

- Colony resource monitoring
- Population statistics
- Habitat status indicators
- Expansion progress tracking
- Activity feed
- Responsive layout

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern browser.

## Use Cases

- Space colony simulations
- Sci-fi dashboards
- Resource management systems
- Educational space projects
- Strategy game interfaces
