# Ease Project Progress Steps

## Overview
A responsive project progress component built using only HTML and CSS.

## Features
- Animated active step
- Responsive layout
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any modern browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
This reusable component follows the animation-first approach and is ideal for dashboards, SaaS products, admin panels, and project management interfaces.
