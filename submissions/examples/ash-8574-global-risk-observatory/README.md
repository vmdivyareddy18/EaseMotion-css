# Global Risk Observatory

A responsive **Global Risk Observatory** dashboard built entirely with **HTML** and **CSS**.

## Features

- Executive global risk dashboard
- AI-powered threat intelligence
- Global risk assessment matrix
- Predictive risk forecasting
- Risk assessment workflow timeline
- Executive analytics cards
- Circular risk confidence indicator
- Responsive CSS Grid layout
- Modern enterprise observatory
- No JavaScript or external libraries

Open `demo.html` in any modern browser.
