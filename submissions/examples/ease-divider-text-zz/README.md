# Text Divider

## What does this do?
Text Divider component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"dt-primary\">Text Divider Primary</div>
```

## Why is it useful?
Provides a reusable text divider UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
