# Neon Pulse Card

## Description
A pure CSS neon-glowing card component with a pulsing border animation. It features a smooth hover lift effect for the card and an interactive fill-and-glow effect for the button. This requires zero JavaScript.

## Files
- `demo.html`: Contains the HTML structure for the card, title, text, and button.
- `style.css`: Contains the neon styling, hover transitions, and the `pulse-glow-qn` keyframe animation.

## How to use
1. Open `demo.html` in your browser to see the effect.
2. Copy the HTML and CSS into your project.
3. Apply the `.card-qn`, `.card-title-qn`, `.card-text-qn`, and `.card-btn-qn` classes to your respective elements.
4. Customize the neon color (`#00ffc8`) in `style.css` to match your brand or theme.
5. Adjust the `pulse-glow-qn` animation duration to make the pulsing faster or slower.
