# Scrollbar Color

scrollbar-color — Customizes scrollbar track and thumb colors

## Features
- Uses `--ease-color-neutral-300`
- Uses `--ease-color-primary`
- Uses `--ease-color-surface`

- Responsive grid layout adapts across breakpoints
- Dark mode support via `prefers-color-scheme: dark`
- Reduced motion support via `prefers-reduced-motion: reduce`

## Usage

```html
<div class="demo-card">
  <span class="card-label">Scrollbar Color</span>
</div>
```

## Why is it useful?

This submission demonstrates the scrollbar color CSS feature within the EaseMotion CSS design system, leveraging existing `--ease-*` tokens for cohesive styling.
