# ✨ Slide Text Entrance

## What does this do?

Creates a smooth slide-up entrance animation for text elements using pure CSS. The text appears with a gentle upward motion and fade-in effect, making content introductions feel more modern, fluid, and visually appealing.

---

## How is it used?

```html
<h1 class="em-slide-text-entrance">
  Welcome to EaseMotion
</h1>

## Why it is useful?

Text content often appears abruptly on webpages, which feels static and less engaging. This animation improves user experience by adding a smooth entrance transition that guides attention naturally. It is lightweight, reusable, and works without JavaScript, making it ideal for modern UI design systems.
