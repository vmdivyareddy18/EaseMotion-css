# Responsive Dashboard Slide-Right Accordion

A modern **Pure CSS Slide-Right Accordion** designed for **Responsive Dashboard** layouts. The component features a smooth slide-right interaction, responsive design, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Smooth Slide-Right interaction
- Responsive dashboard inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Lightweight and reusable

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:.45s;
    --easing:cubic-bezier(.25,.8,.25,1);

    --primary:#2563eb;
    --secondary:#38bdf8;

    --radius:18px;
}
```

You can customize:

- Animation duration
- Animation easing
- Accent colors
- Border radius
- Shadows
- Background colors

---

## 🚀 Usage

Open `demo.html` directly in any modern browser.

No JavaScript or build tools are required.
