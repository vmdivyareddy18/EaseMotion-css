# Ease Aurora Command Grid

## Overview

A premium dashboard command grid featuring floating glassmorphism action tiles with subtle aurora-inspired motion using only HTML and CSS.

## Features

- Floating action tiles
- Glassmorphism styling
- Responsive CSS Grid
- Gradient hover effects
- Lightweight implementation
- Pure HTML & CSS
