# Object Fit Utilities

1. What does this do? Adds object-fit utility classes (`.object-fill`, `.object-contain`, `.object-cover`, `.object-none`, `.object-scale-down`) to control how replaced content (like images or videos) fits into its container box.
2. How is it used? Apply `.object-cover` to an image to scale it to cover its container, cropping if necessary, while preserving aspect ratio.
3. Why is it useful? It solves common layout issues related to responsive images stretching or squashing by providing native CSS sizing behaviors.
