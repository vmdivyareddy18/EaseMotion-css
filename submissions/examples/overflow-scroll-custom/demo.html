overflow property variations

overflow: auto

Content that exceeds the container height will scroll automatically.

This allows the user to see all content without leaving the container.

overflow: hidden

Content is clipped and no scrollbar appears.

Anything overflowing is invisible.

Useful for image containers and cards.

Similar to hidden but scroll is programmatically blocked.
