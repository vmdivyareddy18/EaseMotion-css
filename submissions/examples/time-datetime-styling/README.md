# time-datetime-styling

## Purpose
Demonstrates `time` element and `datetime` attribute styling for formatted date/time display with relative time indicators.

## Features
- Semantic `time` element with `datetime` attribute
- Relative time indicators (ago, from now)
- Formatted date/time display with custom styling
- Timeline-style layout

## Usage
Open `demo.html` to see various time elements displayed with semantic markup and custom visual styles for dates and times.
