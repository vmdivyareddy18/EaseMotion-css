# Sunbird

A pure CSS and HTML sunbird hovering at a tubular flower, its decurved bill dipped into the corolla.

## What it shows

- Iridescent plumage from a conic gradient, shimmered by an animated hue-rotate filter rather than a colour swap
- A decurved bill drawn as a border-top arc on a rounded box, no image or SVG path
- Wings blurred with filter and flapped on a 0.16s alternate keyframe so they read as motion, not shapes
- Flower corollas flared with a masked repeating conic gradient
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
