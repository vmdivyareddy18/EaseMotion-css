# Glassmorphism Pricing Cards

A beautiful, responsive pricing cards component using the **EaseMotion CSS** framework. This example demonstrates how to combine EaseMotion's utility classes (`ease-card-glass`, `ease-fade-in`, `ease-hover-grow`) with a little bit of custom CSS to create a modern glassmorphism effect.

## 🚀 Features

- **Glassmorphism Design**: Uses `backdrop-filter` and semi-transparent backgrounds.
- **Smooth Animations**: Entrance animations using `ease-fade-in` and `ease-slide-up` with staggered delays.
- **Hover Effects**: Cards lift up on hover using custom CSS transitions and button scaling using `ease-hover-grow`.
- **Fully Responsive**: Uses EaseMotion's grid system (`ease-grid-auto`) to adapt to any screen size.
- **Zero Dependencies**: Only requires the EaseMotion CSS CDN.
