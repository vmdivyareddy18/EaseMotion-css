# Cards Gallery Page

A dedicated gallery that displays a collection of animated card variants in a responsive grid. Each card uses a different hover effect — Lift, Glow, Flip, Pulse, and Bounce — demonstrating the animation‑first philosophy of EaseMotion CSS.

## Sections
- **Grid Layout** – responsive grid with multiple card styles
- **Lift Card** – translates upward on hover
- **Glow Card** – adds a glowing outline
- **Flip Card** – rotates on the Y axis
- **Pulse Card** – scales gently in and out
- **Bounce Card** – bounces vertically

## EaseMotion Classes Showcased
- **Layout:** `ease-flex`, `ease-items-center`, `ease-justify-center`, `ease-min-h-screen`, `ease-py-12`, `ease-px-4`
- **Background:** `ease-bg-gray-50`, `ease-bg-white`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`
- **Spacing:** `ease-mb-*`, `ease-mt-*`, `ease-p-6`
- **Components:** `ease-card`
- **Animation:** `ease-fade-in`, `ease-delay-200`, `ease-delay-500`
- **Utilities:** `ease-shadow-md`, `ease-w-full`

## How to Use
1. Open `demo.html` in any modern browser.
2. Hover over the cards to see the effects.
3. Copy the `.card-*` classes and CSS into your project to reuse the animations.
