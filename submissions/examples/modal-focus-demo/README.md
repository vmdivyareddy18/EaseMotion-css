# Modal Focus Demo

1. What does this do?  
   This demo shows a simple modal that moves keyboard focus to the first input field when it opens, improving accessibility.

2. How is it used?  
   Open the demo in a browser and click the button to open the modal. The focus should land inside the modal content automatically.

3. Why is it useful?  
   It highlights a practical accessibility pattern that makes modal dialogs easier to use with keyboards and assistive technologies.
