# Grayscale to Color Image Hover

An image gallery where all images start in grayscale and transition to full color on hover using CSS filter property transitions with smooth cubic-bezier easing.

## Files

- `demo.html` - Three image cards in a gallery layout
- `style.css` - Filter grayscale transition, hover-based color restoration

## Usage

Open `demo.html` and hover over the images to see them transition from grayscale to color.
