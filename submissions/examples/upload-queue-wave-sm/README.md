# Upload Queue Wave

1. What does this do? Adds responsive upload queue rows with animated wave progress fills, status chips, staggered entry motion, and hover or focus feedback.
2. How is it used? Apply `.upload-list` to the queue wrapper and `.upload-row` with state classes like `.complete`, `.active`, `.waiting`, or `.paused` for each file.
3. Why is it useful? It makes file upload progress easier to scan with purposeful CSS-only motion and readable queue states.
