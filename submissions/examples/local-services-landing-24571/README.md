# Local Services Marketplace Template
A community-oriented, high-trust, and transaction-ready landing page template for local service marketplaces, gig platforms, and home-service directories.

- **Design**: Uses a "Market-Trust" palette with market-blue authority headers, trust-gold service highlights, and clean, modular cards to emphasize provider vetting and local proximity.
- **Use Case**: Gig economy platforms, home-service booking sites, and hyper-local service directories.
