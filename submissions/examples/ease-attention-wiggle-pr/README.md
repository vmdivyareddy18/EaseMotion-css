# Animation: ease-attention-wiggle

### What does this do?
Triggers a smooth, low-frequency horizontal wiggle oscillation across an object layer to pull user focus towards high-value points elegantly.

### How is it used?
Apply the utility class `.ease-attention-wiggle` directly to any structural item, optionally tweaking behavior inline with `--ease-wiggle-amount` and `--ease-wiggle-speed` variables.

### Why is it useful?
It provides a gentle, non-obtrusive alternative to stressful validation alerts, running efficiently on composited transformation layers.
