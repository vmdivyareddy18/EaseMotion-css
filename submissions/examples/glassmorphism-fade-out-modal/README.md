# Glassmorphism Fade-Out Modal

A responsive Glassmorphism modal built using only HTML and CSS.

## Features

- Glassmorphism design
- Pure HTML & CSS
- Fade-in / Fade-out effect
- Responsive layout
- CSS custom properties
- No JavaScript

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in any modern web browser.
