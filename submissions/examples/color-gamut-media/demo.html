What is color-gamut?

The @media (color-gamut) media query lets you detect the color space your display supports. srgb is the baseline (most monitors), p3 covers wider Display P3 / DCI-P3 colors (Apple displays, many new monitors), and rec2020 targets ultra-wide gamut (professional HDR reference monitors).

How to see the difference

On a standard sRGB display, the P3 swatches will appear clipped or indistinguishable from sRGB equivalents. On a wide-gamut display (MacBook Pro, iPad Pro, Pro Display XDR, many OLED monitors), the P3 colors will appear noticeably more vivid and saturated. The @media (color-gamut: p3) query can serve P3-specific styles only to capable displays.

/* Serve richer colors to wide-gamut displays */ .swatch { background: #ff3333; /* sRGB fallback */ } @media (color-gamut: p3) { .swatch { background: color(display-p3 1 0.2 0.05); } } @media (color-gamut: rec2020) { .swatch { background: color(rec2020 0.9 0.15 0.05); } }
