EaseMotion is a CSS utility library focused on smooth, performant animations using modern CSS features like custom properties, transitions, and keyframes. It provides drop-in components for common UI motion patterns.
Each component ships with a demo.html, style.css, and README.md. Copy the CSS custom properties into your project, import the styles, and adapt the markup to fit your app. All animations are controlled via :root variables.
Yes. All components respect prefers-reduced-motion and use safe animation durations. Interactive elements have visible focus styles and appropriate ARIA attributes.
