# Poison Bubble Animation Effect

A toxic bubbling green animation effect simulating poisonous liquid with rising and popping bubbles under issue #14755.

## Design Concept

- **Toxic Cauldron/Vat Aesthetic:** A dark-themed layout featuring a bubbling pool of vibrant neon green poisonous liquid.
- **Bubble Mechanics:** Bubbles of various sizes rise continuously from the pool. Each bubble has a distinct horizontal sway (wobble), speed, and elevation path before "popping" at a randomized height.
- **Pure CSS Realization:** Implemented using keyframe animations for the floating rise, horizontal wave sway, and the popping contraction/expansion burst.

## Setup & Showcase

Open `demo.html` in your favorite browser to see the effect in action.
