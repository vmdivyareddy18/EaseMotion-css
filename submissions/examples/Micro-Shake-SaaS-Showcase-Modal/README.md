# Micro Shake SaaS Showcase Modal

A responsive SaaS-inspired modal built entirely with HTML and CSS featuring a smooth Micro Shake interaction.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- SaaS dashboard interface
- Micro Shake animation
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--shake
--primary
--secondary
--bg
--surface
--text
--muted
--radius
```
