Pipeline

How It Works

Four automated steps transform a raw CSS file into a fully interactive animation gallery.

01
📄

Read animations.css

The script opens core/animations.css with Node's fs.readFileSync and loads the full stylesheet text into memory.

const css = fs.readFileSync(
  'core/animations.css',
  'utf-8'
);
02
🔍

Extract .ease-* Classes

A single regular expression scans the CSS and collects every unique class selector that begins with .ease-.

const RE = /\.(ease-[\w-]+)/g;
const classes = [
  ...new Set(
    [...css.matchAll(RE)]
      .map(m => m[1])
  )
];
03
🃏

Generate HTML Cards

For each class name, the script produces a card element — a preview box with the class applied, the name in monospace, and a copy-ready code snippet.

const cards = classes.map(cls => `
  <div class="card">
    <div class="${cls}">
      Preview
    </div>
    <code>${cls}</code>
  </div>
`).join('');
04
📦

Output demo-gallery.html

The script wraps the cards in a full HTML document — including the EaseMotion stylesheet link — and writes it to disk as a ready-to-open file.

fs.writeFileSync(
  'demo-gallery.html',
  htmlTemplate(cards)
);
console.log('✓ Done');

Why Generate?

Benefits

Manual gallery pages go stale. A generator ensures your docs always match your codebase.

📚

Automatic Documentation

Every animation class is documented the moment it's added to the CSS — no manual entry required.

🧪

Easier Testing

Open one generated file in a browser to visually inspect every animation in isolation before release.

🔄

Always Up to Date

Re-run the script after any change to animations.css and the gallery rebuilds itself in under a second.

🚀

Better Onboarding

New contributors open demo-gallery.html and immediately see what every class does — zero guesswork.

👁️

Visual Reference

A living style guide — designers and developers share a single source of truth for all available animations.

⚡

Developer Productivity

Browse, search, and copy class names from a visual gallery instead of hunting through raw CSS source.
