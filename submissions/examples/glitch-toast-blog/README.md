# Glitch Toast (#41538)

## Description
A high-impact toast notification featuring a jittery "glitch" animation. Designed for cyberpunk or developer-focused blog interfaces.

## Key Features
- **Glitch Effect:** Uses CSS `keyframes` with randomized `translate` values for a digital distortion effect.
- **Pure CSS:** Zero JavaScript; utilizes the checkbox hack for state toggling.
- **Responsive:** Fixed positioning ensures it stays visible on all screen sizes.

## Usage
Simply drop the `toast-wrapper` HTML into your page. The `glitch-trigger` checkbox handles the visibility and the animation loop.
