ease-page-transition-fade

Full page fades out, then new page fades in using the View Transitions API pattern.

Bug: Instant Page Switch

myapp.com/products

Page appears instantly — no fade transition.

Fix: Fade Page Transition

Click "Navigate" to trigger the full-page fade-out → fade-in transition.

myapp.com/home

🏠 Home Page

This page fades out when you navigate away.

🛍️ Products Page

This page fades in from invisible after the transition.

CSS Implementation Reference

/* ease-page-transition-fade */
@view-transition {
    navigation: auto;
}

::view-transition-old(root) {
    animation: fadeOut 0.35s ease-in-out;
}

::view-transition-new(root) {
    animation: fadeIn 0.35s ease-in-out;
}

@keyframes fadeOut {
    from { opacity: 1; }
    to   { opacity: 0; }
}

@keyframes fadeIn {
    from { opacity: 0; }
    to   { opacity: 1; }
}
