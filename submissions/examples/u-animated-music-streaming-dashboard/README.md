# Animated Music Streaming Dashboard

A modern streaming analytics dashboard built with HTML and CSS.

## Features

- Streams overview
- Listener analytics
- Playlist insights
- Followers statistics
- Activity feed
- Responsive layout
- CSS Variables
- Hover animations
- Pure CSS animations
- No JavaScript
