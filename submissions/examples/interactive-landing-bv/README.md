# Interactive Landing Page System

A premium interactive landing page built with pure CSS animations.

## Features
- Sticky Navigation
- Animated Hero Banner
- Problem Solution Cards with hover effects
- Feature Grid
- Pricing Tiers (Basic/Pro/Enterprise)
- Pure CSS FAQ Accordion
- Newsletter Signup
- Cookie Consent Banner
- Scroll to Top Button

## Preview
Open `demo.html` in any browser.

## Tech Used
- Pure HTML5
- Pure CSS3 (animations, transitions, grid)
- No JavaScript (except cookie dismiss)
