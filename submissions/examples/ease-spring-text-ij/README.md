# Spring Text

A text animation where each letter springs upward with overshoot on hover, using staggered cubic-bezier timing.

## Features

- Each letter animates individually with staggered delays
- Spring-like bounce with overshoot using cubic-bezier timing
- Color transitions through pink and purple during animation
- Subtitle fades out on hover
- Pure CSS animation

## Usage

Wrap each letter in a `<span>` inside `.spring-text`. The animation triggers on hover with staggered `animation-delay` per letter.
