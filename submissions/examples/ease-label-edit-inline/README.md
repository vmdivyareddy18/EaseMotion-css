## Description
Added the `ease-label-edit-inline` component, a CSS-only inline label edit interaction with click-to-edit motion.

## What changed
- Added a new demo for the component in `demo.html`.
- Added component styles and keyframe animations in `style.css`.
- Documented usage, customization, and behavior in `README.md`.

## Why
This component matches the EaseMotion CSS library’s animation-first approach and provides a reusable inline editing interaction with a polished motion effect.

## How to test
1. Open `demo.html`.
2. Find the `ease-label-edit-inline` demo.
3. Click the label to verify the inline edit transition animation.
4. Check that the component responds correctly with the provided CSS custom properties.

## Note
This component is CSS-only and focuses on the visual transition behavior. it does not include form submission logic.
