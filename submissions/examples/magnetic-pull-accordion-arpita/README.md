# Magnetic Pull Accordion

## What does this do?
A pure HTML and CSS accordion featuring a smooth magnetic pull hover interaction designed for modern marketing landing page layouts.

---

## How is it used?

```html
<div class="accordion">

  <div class="item">
    <input type="checkbox" id="item1">

    <label for="item1">
      <span>Accordion Title</span>
      <span class="icon">+</span>
    </label>

    <div class="content">
      Accordion content goes here.
    </div>

  </div>

</div>
```

---

## Why is it useful?
The Magnetic Pull Accordion provides an engaging way to organize content for marketing websites, product showcases, FAQs, and landing pages. Its subtle magnetic hover effect, smooth accordion expansion, responsive design, and pure CSS implementation align with EaseMotion CSS's animation-first and developer-friendly philosophy.
