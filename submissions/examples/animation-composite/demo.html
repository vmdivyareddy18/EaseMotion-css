animation-composite

Controls how multiple animations combine when they target the same property — replace, add, or accumulate.

replace (default)

The last animation's values override the first. The box scales then fades — the fade wins.

R

add

Values are summed. The box both scales AND fades at the same time.

A

accumulate

Values accumulate intelligently — transforms chain, opacity layers, colors blend.

C

Each box runs two conflicting animations. Only animation-composition changes how they merge.
