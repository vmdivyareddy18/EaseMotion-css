# LittleSprout — Organic Baby Essentials E-Commerce

A full-page, fully responsive e-commerce landing page designed for a safe, nurturing, and premium baby essentials brand. Built exclusively utilizing the `EaseMotion` CSS utility class architecture to deliver a soft, highly-rounded, and approachable aesthetic.

## 🍼 Theme & Aesthetic
**Safe & Nurturing Parenthoood:**
* **Colors:** Soft Blush Pink (`#FF8FA3`), Warm Cream (`#FFFDFB`), Pure White (`#FFFFFF`), and Soft Warm Brown (`#4A3B39`).
* **Typography:** `Nunito` (a highly rounded, bubbly sans-serif providing a friendly, safe feel) paired with `Work Sans` (a highly legible, clean sans-serif for body copy).
* **Vibe:** Gentle, safe, organic, premium, and trustworthy.

## 🚀 How to Run
This template requires **zero build steps**. 
1. Clone or download the repository.
2. Ensure `demo.html` and `style.css` are in the same directory.
3. Open `demo.html` directly in your favorite web browser.

## 🛠️ EaseMotion Classes Showcased
This project relies entirely on the core EaseMotion CSS utility framework to build complex grids and smooth micro-interactions.

### Layout & Structure
* `ease-container`
* `ease-flex`, `ease-grid`
* `ease-grid-cols-2`, `ease-grid-cols-3`, `ease-grid-cols-4`
* `ease-gap-4`, `ease-gap-6`, `ease-gap-8`, `ease-gap-12`
* `ease-items-center`, `ease-justify-between`, `ease-justify-center`
* `ease-relative`

### Theming Utilities
* `ease-bg-main`, `ease-bg-surface`, `ease-bg-primary`, `ease-bg-light`
* `ease-text-main`, `ease-text-primary`, `ease-text-white`, `ease-text-muted`
* `ease-border`, `ease-border-primary`, `ease-border-y`
* `ease-font-bold`, `ease-font-display`

### Animations & Scroll Reveals
* `ease-reveal`, `ease-reveal-up`, `ease-reveal-scale` (Intersection Observer triggered for smooth scroll-in)
* `ease-fade-in`, `ease-slide-up`, `ease-bounce-in` (Initial page load transitions)
* `ease-delay-100` through `ease-delay-500` (Staggered element loading)

### Micro-Interactions (Hover)
* `ease-hover-lift` (Y-axis translation + soft shadow expansion for category and review cards)
* `ease-hover-scale` (Image zoom effect applied to blog/community articles)
* `ease-hover-fade` (Subtle opacity shift on logos)
* `ease-hover-underline` (Animated rounded bottom border on navigation links)

## 📱 Responsiveness
The layout adapts gently for different viewports. The 4-column age category grids and 3-column review grids smoothly degrade to a 2-column view on tablets and a stacked, single-column design on mobile devices to ensure a seamless shopping experience for parents on the go.
