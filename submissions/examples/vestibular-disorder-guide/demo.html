Vestibular Disorder-Safe Animation Guide

Building inclusive motion experiences with prefers-reduced-motion

Understanding Vestibular Disorders

Vestibular disorders affect the inner ear and brain, causing dizziness, vertigo, nausea, and balance problems. The Vestibular Disorders Association (VEDA) reports that up to 35% of adults over 40 experience some form of vestibular dysfunction. For these users, certain CSS animations can trigger debilitating symptoms.

1 in 4 adults will experience a vestibular disorder in their lifetime

High-Risk Animation Patterns

The following animation types are most likely to trigger adverse reactions. Each should be paired with a safe alternative via prefers-reduced-motion.

Large Parallax Scrolling

Parallax (unsafe)
Reduced motion (safe) 
/* Risky — large background offset */
.parallax-bg {
  transform: translateY(-30%);
}

/* Safe alternative */
@media (prefers-reduced-motion: reduce) {
  .parallax-bg {
    transform: none;
  }
}

Spinning / Rotation Animations

Spin (unsafe)
Reduced motion (safe) 
/* Risky — continuous rotation */
.spinner {
  animation: spin 2s linear infinite;
}

/* Safe alternative */
@media (prefers-reduced-motion: reduce) {
  .spinner {
    animation: none;
  }
}

Zooming / Scale Pulses

Zoom pulse (unsafe)
Reduced motion (safe) 
/* Risky — rapid scale oscillation */
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50%      { transform: scale(1.4); }
}

@media (prefers-reduced-motion: reduce) {
  .pulse {
    animation: none;
    /* or use a subtle opacity alternative */
  }
}

How It Works

The CSS media feature prefers-reduced-motion: reduce allows developers to detect when a user has requested minimal animation at the OS level (Windows: Settings → Ease of Access → Display; macOS: System Settings → Accessibility → Display → Reduce motion).

WCAG Success Criterion 2.3.3 — Animation from Interactions (Level AAA) requires that "motion animation triggered by interaction can be disabled, unless the animation is essential to the functionality or the information being conveyed."
