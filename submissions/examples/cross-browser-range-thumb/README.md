# Cross-Browser Range Thumb

**What does this do?**  
It gives the HTML range input a consistent thumb and track style across Chromium, Firefox, and Safari.

**How is it used?**  
```html
<input class="range-control" type="range" min="0" max="100" value="64" />
```

**Why is it useful?**  
Native form controls often look different from browser to browser, and this keeps the slider feeling polished while staying lightweight and CSS-only.
