# Font Foundry - EaseMotion CSS Example

A premium typography and font foundry e-commerce landing page demonstrating the capabilities of EaseMotion CSS.

## Features

- Premium typography-focused design
- Interactive font collection showcase
- Comprehensive character map preview
- Responsive pricing cards
- Smooth scroll animations using core `reveal.js`
- Clean, aesthetic layouts

## Folder Structure

```text
submissions/examples/font-foundry/
├── demo.html     # Main HTML template
├── style.css     # Custom typography styling overrides
└── README.md     # Project documentation
```

## EaseMotion Classes Used

- `ease-fade-up`, `ease-slide-up`, `ease-slide-in-left`, `ease-slide-in-right`
- `ease-reveal`, `ease-reveal-up`, `ease-reveal-left`, `ease-reveal-right`, `ease-reveal-scale`
- `ease-hover-lift`, `ease-hover-lift-shadow`, `ease-hover-scale`
- `ease-card`, `ease-card-glass`, `ease-card-neumorphic`, `ease-card-flat`
- `ease-grid`, `ease-grid-cols-1`, `ease-md-grid-cols-2`, `ease-lg-grid-cols-3`
- `ease-bg-primary`, `ease-bg-neutral`, `ease-bg-white`
