# Wave Error Page

Resolves #42412.

### What does this do?
Adds a Wave Error Page component to EaseMotion CSS. It creates a calming ocean wave effect at the bottom of the screen using inline SVG backgrounds. The waves utilize the `ease-swing` animation with overridden durations and directions to create a continuous, soothing parallax motion.

### How is it used?
Copy the HTML from `demo.html` into your project and use `style.css` for the design layout. The wave SVGs are inline so no external assets are required.

### Why is it useful?
Provides an aesthetic and calming 404 error page. Error pages often cause frustration; soft, continuous wave animations help reduce user anxiety while maintaining a high-quality corporate design.
