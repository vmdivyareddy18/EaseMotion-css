# Floating Label Input Component

A modern **Floating Label Input Component** built with **HTML5** and **CSS3** for EaseMotion CSS.

---

## ✨ Features

- Animated floating labels
- Smooth focus transitions
- Hover interactions
- Outline variant
- Filled variant
- Rounded variant
- Glassmorphism variant
- Success state
- Error state
- Disabled state
- Read-only state
- Responsive layout
- Dark mode support
- Reduced motion support
