# Jello Barcode

A CSS-only animated barcode component with a smooth jello motion effect.

## Features

- Pure CSS animation
- No JavaScript required
- Nature-inspired visual style
- Responsive design
- Accessible animation handling

## Usage

```html
<div class="ease-jello-barcode">

  <span></span>
  <span></span>
  <span></span>

</div>
