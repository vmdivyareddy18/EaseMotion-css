# Cart Icon Bounce

## What does this do?
A shopping cart icon that bounces with a badge count animation when an item is added, providing add-to-cart feedback.

## How is it used?
Use `.cart-bounce` on the cart container. The `.bumped` class triggers a bounce animation on the icon and a pop on the badge.

## Why is this useful?
Provides satisfying add-to-cart feedback that confirms the action without redirecting or showing a toast, keeping the user in context.
