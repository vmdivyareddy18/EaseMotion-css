# Tag Input / Chip Input Component

A tag input component where users can add and remove tags as styled chips. Pressing Enter or clicking "Add" creates a new chip. Each chip has a remove button. The inputs are bounded and validated to prevent empty tags.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Type a tag name and click "Add" or press Enter. Click the &times; on any chip to remove it.

## Accessibility notes

Chip remove buttons are `<button>` elements for keyboard accessibility. The input has a placeholder for guidance.
