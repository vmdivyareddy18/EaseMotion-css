# Animated Rocket Launch Lift-Off

## Overview

A reusable rocket icon animation featuring a smooth lift-off effect suitable for startup websites, deployment dashboards, achievement systems, growth metrics, and launch interfaces.

## Features

- CSS-only animation
- Rocket lift-off effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="rocket-button">
  <span class="rocket-icon">🚀</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
