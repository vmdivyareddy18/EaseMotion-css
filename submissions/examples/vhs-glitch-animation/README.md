# VHS Glitch Animation

A retro VHS-style glitch animation utility built using pure CSS.

## Features

- RGB color channel separation
- Horizontal distortion bands
- Smooth jitter animation
- Retro VHS aesthetic
- Pure CSS implementation

## Usage

```html
<div class="vhs-box">
  VHS GLITCH

  <div class="band band1"></div>
  <div class="band band2"></div>
</div>
```

- Human-readable naming

- Lightweight implementation
