# Full Subscription Box Landing Page

1. What does this do? A vibrant subscription box landing page with how-it-works steps, three-tier pricing comparison, themed box showcases, FAQ accordion, and CTA built entirely with EaseMotion CSS framework classes.

2. How is it used? Open `demo.html` — every layout, pricing card, scroll reveal, hover effect, and responsive behavior uses EaseMotion classes (`ease-card`, `ease-reveal-up`, `ease-hover-lift`, `ease-grid-cols-3`, `ease-delay-*`, `ease-btn-pill`, etc.) with zero custom keyframes.

3. Why is it useful? Proves EaseMotion handles conversion-focused SaaS-style landing pages with pricing tiers, FAQ accordions, and trust signals — all framework-native, showing EaseMotion's utility for subscription and service businesses.
