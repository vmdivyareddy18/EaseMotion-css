# Marquee — Horizontal Scrolling Text

CSS-only scrolling marquee with infinite loop. Duplicated content creates a seamless scroll effect using `translateX` keyframe animation on a flex row.

## Files

- `demo.html` — marquee markup with repeated items
- `style.css` — flex layout, scroll animation, glow accents

## Usage

Open `demo.html` in any browser. The marquee scrolls continuously. Respects `prefers-reduced-motion`.
