Flexible CSS Grid Auto-Fit Generator Mixin

A reusable SCSS mixin that produces responsive auto-fit grids using minmax() — no media queries needed.

Section 01

Base .ease-grid

Columns auto-fit to a 200 px minimum, growing to fill available space. Resize the window to see columns collapse.

.ease-grid
Item 1 Auto-Fit Column Collapses below 200 px min-width.
Item 2 Auto-Fit Column Grows to fill available space equally.
Item 3 Auto-Fit Column No media queries required.
Item 4 Auto-Fit Column Add or remove items freely.

Section 02

Capped column grid — .ease-grid--3

Uses the min() trick inside minmax() to enforce a maximum of 3 columns on wide viewports while still collapsing gracefully on narrow screens.

.ease-grid--3
Feature A Max 3 Columns Wide viewport: exactly 3 per row.
Feature B Responsive Collapse Narrows to 2 or 1 column on small screens.
Feature C No JS Pure CSS Grid — zero JavaScript.
Feature D Wraps Cleanly 4th item starts a fresh row.
Feature E Design-Token Ready Override gap via --ease-grid-gap.
Feature F Dark & Light Adapts to prefers-color-scheme.

Section 03

Span & dense helpers

.ease-grid__item--span-2 makes an item span two columns. .ease-grid--dense back-fills gaps automatically.

.ease-grid--4 .ease-grid--dense
Featured Wide Card — span 2 Uses .ease-grid__item--span-2 to occupy two columns.
Item B Normal Card 1 column wide.
Item C Normal Card 1 column wide.
Item D Dense Back-fill Grid fills gaps with dense.
Item E Normal Card 1 column wide.

Section 04

Gap modifier utilities

Chain a gap modifier class to adjust spacing independently of the column layout.

.ease-grid--2 .ease-grid--gap-xl
Left Extra Large Gap --gap-xl sets gap: 2rem.
Right Extra Large Gap Available: sm · md · lg · xl · 2xl.
