# Scale Hover Tooltip

Responsive tooltip component designed for modern E-Commerce Checkout interfaces.

---

## Features

- Pure CSS implementation
- Scale-Hover animation
- Checkout inspired styling
- Responsive design
- Keyboard accessibility
- Focus-visible support
- CSS variables
- Zero JavaScript dependency
