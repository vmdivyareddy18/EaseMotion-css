# Owl Blinking

1. What does this do? A CSS-only owl face with large eyes that blink periodically.

2. How is it used? Place .owl inside .scene. The body is a brown oval; eyes are yellow circles with dark pupils created via box-shadow.

3. Why is it useful? Shows how scaleY on the Y axis creates a natural eyelid blink effect.
