# CSS Avatar / Initials

## What does this do?
CSS avatar component with size variants, initials fallback, status indicators, stacked groups, and ring styles.

## How is it used?
```html
<div class="av-avatar av-md" style="background:#667eea"><span>JD</span></div>
<div class="av-stack"><div class="av-avatar av-md">A</div><div class="av-more av-md">+3</div></div>
```

## Why is it useful?
User avatars without JavaScript. Five sizes. Initials-based. Online/away/busy status dots. Stacked overlapping groups with hover effects. Colored rings. Square variant.
