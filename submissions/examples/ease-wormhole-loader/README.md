# Wormhole Loader – Spinning tunnel effect

> A production-ready wormhole loader component that creates a mesmerizing spinning tunnel effect with customizable colors, speed, rings, and direction controls. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Spinning tunnel effect** – Mesmerizing wormhole animation
- ✅ **Customizable colors** – Primary and background colors
- ✅ **Adjustable speed** – Control animation speed
- ✅ **Adjustable rings** – Number of ring segments
- ✅ **Direction control** – Clockwise or counter-clockwise
- ✅ **Pause/Resume** – Control the animation
- ✅ **Reset** – Reset the animation
- ✅ **Random colors** – Randomize colors with one click
- ✅ **Keyboard shortcuts** – Space (pause), R (reset), D (direction)
- ✅ **4 color variants** – Aurora, Sunset, Gold, Dream
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
