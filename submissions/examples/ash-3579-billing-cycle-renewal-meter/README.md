# Billing Cycle Renewal Meter

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive billing cycle progress component that visually displays subscription renewal progress using a CSS-only circular meter.

## Features

- Circular renewal progress meter
- Billing information summary
- Subscription status badge
- Responsive layout
- CSS-only progress visualization
- Hover interactions
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="progress-ring">

    <div class="progress-value">
        73%
    </div>

</div>
```

## Main Classes

```css
.card
.meter
.progress-ring
.progress-value
.info
.item
.badge
.renew-btn
```
