# Animated PIN/OTP Input Collection Component

A secure, multi-segmented credential access interface engineered for Two-Factor Authentication (2FA) models, secure gateway clearances, and promotional processing inputs under **EaseMotion CSS**.

## ✨ Motion Ecosystem
* **Elastic Scale Injections (`cellPop`)**: Digit additions trigger localized transformations using custom easing equations to scale characters without affecting line heights.
* **Declarative Shadow Carets (`inlineCaretBlink`)**: Leverages infinite inset box shadows inside focus vectors to cleanly replace default platform inputs with unified carets.
* **Chassis Validation Waves**: Wraps full rows using distinct shake (`rowErrorShake`) and completion expansion profiles (`rowSuccessPulse`) to provide instant validation feedback.
