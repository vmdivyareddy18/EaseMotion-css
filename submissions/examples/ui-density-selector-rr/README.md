# UI Density Selector Component

## What does this do?

Provides a modern UI density selector interface that allows users to compare Compact, Comfortable, and Spacious layout modes.

## How is it used?

```html
<div class="density-card comfortable active">
    <h3>Comfortable</h3>
</div>
```

## Why is it useful?

Many SaaS applications and productivity tools allow users to customize interface density. This component demonstrates density-selection patterns, animated previews, and modern settings UI using pure HTML and CSS.
