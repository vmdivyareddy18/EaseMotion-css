# Random Quote Generator

## What does this do?
Displays a random quote from a curated collection with a new quote button and a share-to-Twitter option.

## How is it used?
Open demo.html in a browser. Click "New Quote" to cycle through quotes or the share button to tweet the current quote.

## Why is this useful?
Demonstrates a simple randomized content display pattern with social sharing integration.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
