# Navbar Shrink on Scroll

> A production-ready navbar component that elegantly shrinks as you scroll down the page, with smooth transitions, responsive mobile menu, and full customization. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

---

## ✨ Features

- ✅ **Shrink on scroll** – Navbar smoothly shrinks as you scroll
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Mobile menu** – Hamburger menu with slide-in animation
- ✅ **Smooth transitions** – Elastic easing for polished feel
- ✅ **Accessibility** – Keyboard navigable with ARIA labels
- ✅ **Customizable** – Easy to style and configure
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
