# Ease Focus Session Timeline

## Overview

Ease Focus Session Timeline is a modern CSS-only productivity timeline component for SaaS dashboards and developer tools. It presents focus sessions in a clean vertical timeline with animated milestones.

---

## Features

- Pure HTML & CSS
- Vertical timeline layout
- Animated active session
- Timeline indicators
- Hover interactions
- Responsive design
- Dashboard-friendly appearance
- CSS variables
- Keyframe animations
- No JavaScript required

---

## Folder Structure

```text
ease-focus-session-timeline-z1/

```

---

## Usage

1. Open `demo.html`
2. Ensure `style.css` is in the same folder
3. View in any modern browser

---

## Customization

- Change theme colors using CSS variables
- Add more timeline events
- Modify animation speed
- Adjust spacing and typography
- Replace sample productivity data
