# Rotate Share Button

A modern, animated rotate share button built using pure HTML and CSS for EaseMotion CSS.

## Features

- 🔄 Smooth rotating share button
- 🎨 Pure CSS (No JavaScript)
- 📱 Responsive design
- ♿ Accessible with keyboard navigation (`:focus-within`)
- ✨ Lightweight and easy to customize

## File Structure

```
rotate-share-button-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Hover over the **Share** button (or focus it using the keyboard) to reveal the social sharing buttons with a smooth rotating animation.

## Why is it useful?

The Rotate Share Button provides a reusable, elegant sharing UI that can be integrated into portfolios, blogs, dashboards, landing pages, and web applications while maintaining a clean, modern user experience.
