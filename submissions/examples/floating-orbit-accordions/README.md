# CSS Floating Orbit Accordion

A pure CSS, highly animated accordion component designed for modern marketing landing pages. It features a unique "Floating Orbit" transition effect using CSS keyframes and custom properties, achieving complex motion states with **zero JavaScript**.

## 🚀 Features
- **100% Pure CSS:** No JS overhead, leveraging the native `<input type="radio">` hack for state management.
- **Floating Orbit Motion:** Subtle background orbit/glow animations that respond dynamically when a tab is expanded.
- **Customization Ready:** Exposed CSS Custom Properties (`--orbit-speed`, `--accordion-easing`, etc.) for seamless branding integration.
- **Fully Responsive & Accessible:** Fluid scaling across all device sizes and supports keyboard navigation via labels.

## 📂 File Structure
- `demo.html` - The markup and structure of the marketing accordion.
- `style.css` - The layout, orbit animations, and custom theme properties.
