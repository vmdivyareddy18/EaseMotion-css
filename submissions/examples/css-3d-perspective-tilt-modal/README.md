# CSS 3D Perspective Tilt Modal

A modern **pure CSS 3D animated modal** with perspective tilt interaction designed for minimalist tech UI layouts.

---

## ✨ Features

- Pure CSS 3D perspective animation
- Smooth open/close transitions
- Hover-based tilt interaction (rotateX / rotateY)
- Responsive and centered modal
- Keyboard accessible (ESC to close)
- Customizable via CSS variables

---

## 🎯 CSS Variables

You can customize behavior easily:

```css
:root {
  --transition: 0.35s;
  --tilt: 12deg;
  --scale: 1.05;
}
