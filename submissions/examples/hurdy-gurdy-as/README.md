# Hurdy Gurdy

A pure CSS and HTML hurdy gurdy, the crank driven medieval string instrument, with keys tapping and musical notes floating up as the wheel turns.

## What it shows

- The rosetta sound hole and the carved scroll drawn from repeating conic gradients
- Drone strings from a fine repeating linear gradient stretched over the belly
- Tangent keys pressing in sequence on staggered delays while the crank turns at a steady rate
- Musical notes set as unicode glyphs through ::before content, drifting up and fading
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
