# CSS Spring Physics Accordion for Marketing Landing Page Layouts

A **pure CSS Spring Physics Accordion** designed for **Marketing Landing Pages**. The component delivers smooth spring-inspired expand/collapse animations while remaining lightweight, responsive, accessible, and completely JavaScript-free.

## Features

- 🚀 Pure HTML & CSS
- 🌱 Smooth spring-inspired accordion animation
- 📢 Ideal for FAQs and marketing landing pages
- 📱 Fully responsive
- ♿ Keyboard accessible using native `<details>` and `<summary>`
- 🎨 Easily customizable with CSS Custom Properties
- ⚡ No JavaScript required

## Folder Structure

## Usage

1. Open `demo.html` in any modern browser.
2. Click an accordion section to expand or collapse it.
3. Customize colors, spacing, and animation using the CSS variables in `style.css`.

## Customization

Modify the variables in the `:root` selector:

```css
:root {
  --primary: #4f46e5;
  --secondary: #7c3aed;
  --radius: 18px;
  --duration: .45s;
  --easing: cubic-bezier(.22,1,.36,1);
}
```

You can customize:

- Primary and secondary colors
- Border radius
- Animation duration
- Easing function
- Shadows and spacing
