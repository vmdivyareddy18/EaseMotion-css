# Rune Glow Pulse Animation

## What does this do?
Ancient rune symbols that pulse with glowing magical energy, each with staggered timing.

## How is it used?
Create a grid of `.rune` elements each with a `--h` custom property for delay.

## Why is it useful?
Rune animations enhance fantasy UIs, enchantment displays, and magical item interactions.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
