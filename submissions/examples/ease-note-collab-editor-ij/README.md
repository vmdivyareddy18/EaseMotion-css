# Note Collab Editor

A collaborative text editor with cursor-position indicators.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--editor-bg` | `#1e1e36` | Editor background |
| `--cursor-color` | `#4ecdc4` | Blinking cursor color |
| `--accent-color` | `#4a4a6a` | Border accent |
| `--blink-duration` | `1s` | Cursor blink speed |
