# Chip Filter Group

## What does this do?
Adds `.ease-chip-filter` — a group of toggleable chips that act as filters.

```html
<div class="ease-chip-filter">
  <button class="ease-chip-filter-item">React</button>
  <button class="ease-chip-filter-item ease-chip-active">Vue</button>
</div>
```

Open `demo.html` to see it in action.
