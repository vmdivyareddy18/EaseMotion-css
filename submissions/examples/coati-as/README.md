# Coati

A pure CSS and HTML coati walking a jungle branch, ringed tail held high and long snout tipping to sniff.

## What it shows

- A banded tail from a single repeating linear gradient at a slight angle
- The snout rotating from a hinge at the skull, with the nose nested inside so it travels with the tip
- Far legs stepping in reverse on the same keyframe, dimmed with filter: brightness for depth
- Dappled canopy light pulsing over the scene
- No images, no JavaScript, no external dependencies

## Usage

Open `demo.html` in any modern browser.

## Accessibility

All motion stops under `prefers-reduced-motion: reduce`.
