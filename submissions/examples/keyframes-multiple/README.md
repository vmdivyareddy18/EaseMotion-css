# Multiple @keyframes on One Element

Shows how to combine multiple `@keyframes` animations on a single element for complex motion — position, scale, rotation, and colour all running simultaneously.

## Features

- Combined demo: one element animating position, scale, rotation, and colour at once.
- Breakdown grid: each property shown in isolation (position, scale, rotation, colour).
- Layer stack: four separate animations (move, pulse, spin, fade) applied via comma-separated `animation`.
- Responsive layout adapting to mobile.

## Topics Covered

- Comma-separated `animation` shorthand for multiple keyframes
- Combining transform sub-properties (translate, scale, rotate)
- Animating `background` gradients across colours
- Separating concerns: isolating individual animation properties
