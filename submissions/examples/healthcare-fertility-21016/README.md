# Healthcare: Fertility Clinic (`healthcare-fertility-21016`)

This proposal provides a complete, copy-paste ready B2C healthcare landing page designed for a fertility clinic or reproductive medicine practice, utilizing pure HTML/CSS and standard EaseMotion utility classes.

## 📌 Feature Overview

A fertility clinic website must communicate hope, warmth, and deep medical expertise. This template achieves an "expert yet compassionate" aesthetic using a soft, calming color palette (peaches, warm corals, slate blue), an elegant serif font for headings (Playfair Display), and gentle, slow-paced CSS entrance animations.

The layout includes:
- A hero section featuring soft CSS background glows and a stacked abstract image frame to evoke a sense of care and depth.
- A 3-column "Comprehensive Care" services grid.
- A split-layout section emphasizing clinical success rates with simulated count-up stats blocks.
- Professional team profile cards for the REI specialists.
- Patient story testimonial cards set against a calming blue background.
- A dedicated patient care and financial planning banner.
- A clean, minimalist appointment consultation form.
- A robust clinic footer.

### Showcased Classes
The template leverages the following EaseMotion animation and interaction utilities:
- `.ease-fade-up` / `.ease-slide-up` (Configured with slightly longer durations (0.8s) for soft, reassuring entrances of text and cards)
- `.ease-zoom-in` (Applied to the hero image stack and success rate boxes to draw focus gently)
- `.ease-hover-scale` (Applied to team portraits and social icons for subtle interactivity)
- `.ease-hover-lift` (Standard interaction for booking buttons)
- `.ease-slide-left` / `.ease-slide-right` (Used in split-layout sections, such as the booking form, to bring content together seamlessly)
- `.ease-focus-line` (Applied to form inputs to provide a clean, minimalist bottom-border highlight upon focus, avoiding bulky input boxes)

## ⚙️ How to Use

To view this full-page template locally, simply open the `demo.html` file in your web browser. 

The CSS handles the layout, soft glowing shapes, and typography mapping, while standard `ease-*` interaction classes are attached directly to HTML elements to provide the animations. The design is fully responsive and requires zero JavaScript to execute the entrance and hover effects.

*Note: As per the contributing guidelines, this proposal is implemented inside a unique `submissions/examples/healthcare-fertility-21016/` directory to avoid modifying core files directly and to ensure zero deletions.*
