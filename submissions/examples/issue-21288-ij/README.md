# Rocket Launch — Rocket with Flame

CSS rocket that launches upward with a flickering flame exhaust. The `launch-rocket` keyframe sends the rocket up, while `flicker-flame` animates the thrust.

## Files

- `demo.html` — rocket body, nose, fins, window, flames
- `style.css` — launch trajectory, flame flicker keyframes

## Usage

Open `demo.html` to watch the rocket launch and reset.
