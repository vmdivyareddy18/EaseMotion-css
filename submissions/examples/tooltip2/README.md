# Tooltip Component

Pure CSS tooltips using `data-tooltip` attributes. No JavaScript required.

## Demo

Open `demo.html` in any modern browser.

## Features

- ✅ **Pure CSS** — zero JavaScript
- ✅ **4 positions** — top, bottom, left, right
- ✅ **Arrow indicator** — automatic direction matching
- ✅ **Delay option** — delayed appearance on hover
- ✅ **2 themes** — dark (default), error
- ✅ **Multi-line** — wraps long text automatically
- ✅ **Accessible** — keyboard focus support
- ✅ **Responsive** — adapts to mobile
