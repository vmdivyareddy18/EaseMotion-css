# Enterprise Hyperautomation Command Hub

Phase 274 dashboard — enterprise automation command center for monitoring bot fleets, automation workflows, process efficiency, queue status, and system alerts.

## Features

- **Stat cards** — 4 key metrics: active bots, processes automated, hours saved, success rate
- **Bot status overview** — real-time status of 6 automation bots with health indicators
- **Process efficiency bars** — 5 process lines with gradient fill bars
- **Queue status** — 5 queues with pending counts and occupancy bars
- **System metrics** — 6 core KPIs (response time, throughput, error rate, uptime, API calls, savings)
- **System alerts** — 5 live alert entries with severity badges and timestamps
- **Automation overview grid** — 8-cell summary of cluster-wide stats

## Theme

Orange/amber hyperautomation theme with embedded dark and light mode support via `prefers-color-scheme`. Reduced motion respected.
