How to Contribute a New Translation

      

        

          
1. Create Locale Folder

          

            Inside the root directory, create a new folder under /locales/<language-code>/.
            Copy the original English README.md file into this new folder.
          

        


        

          
2. Translate Content

          

            Translate the headings, descriptions, class lists, and setup guidelines.
            Keep code blocks and class names in English to prevent integration bugs.
