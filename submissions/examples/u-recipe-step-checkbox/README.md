# ease-recipe-step-checkbox

Animated recipe step checkbox component with strikethrough completion effects.

## Features

- Animated strikethrough line
- Text dimming on completion
- Recipe step checklist
- Completion indicator
- Responsive design
- Pure HTML and CSS

## Acceptance Criteria Covered

### Strikethrough Animation

The completion line is animated using:

- CSS pseudo-elements
- scaleX animation
- left-to-right drawing effect

### Completion State

Completed steps:

- Show animated strikethrough
- Reduce text opacity
- Display checkmark indicator

### Use Cases

- Recipe applications
- Cooking assistants
- Task checklists
- Step-by-step tutorials
