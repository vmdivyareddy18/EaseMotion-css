Resize Direction

Indicates which directions an element can be resized by the user.

CSS Property: resize

Variants Demo

resize: none Resize Direction variant
resize: both Resize Direction variant
resize: horizontal Resize Direction variant
resize: vertical Resize Direction variant
resize: both
Drag corner to resize both directions
resize: horizontal
Drag corner to resize horizontally
resize: vertical
Drag corner to resize vertically

Usage

<div class="resize">Content here</div>
