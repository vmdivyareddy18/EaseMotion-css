# Monochrome Neo Glow Pulse Card

A pure-CSS interactive component utilizing keyframe glowing pulses and responsive transformation layouts, custom-styled for minimal, high-contrast dark mode interfaces.

## Features
* **Zero JavaScript Footprint:** Runs smooth ambient glowing shadow pulses on user focus or hover actions natively via hardware-accelerated CSS engines.
* **Fully Configurable Parameters:** Easily calibrate pulse frequencies, border transitions, and geometric scaling thresholds using standard CSS variables inside the root node.
* **A11y Compliant:** Leverages semantic block structures paired with high-contrast `:focus-visible` states to preserve structural integrity across access devices.
