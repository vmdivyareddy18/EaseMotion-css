# Image Hover Effects

## What does this do?
CSS image hover effect library with zoom, overlay fade/slide, filter transitions, caption reveals, and border/shine effects.

## How is it used?
```html
<div class="ih-frame"><div class="ih-img ih-zoom-in" style="background-image:url(image.jpg)"><div class="ih-overlay"><h3>Title</h3></div></div></div>
```

## Why is it useful?
Provides 12 distinct hover effects for images without JavaScript. GPU-accelerated transforms and filters. Ideal for galleries, portfolio grids, and product cards. Customizable timing and easing via CSS variables.
