# Glow Pulse Holographic Light Card

A responsive Holographic Light themed card featuring a smooth Glow Pulse animation built entirely with HTML and CSS.

## Features

- Pure CSS
- Responsive design
- Keyboard accessible
- Glow Pulse animation
- Holographic glassmorphism interface
- Soft iridescent gradients
- Customizable CSS variables
- No JavaScript required

## CSS Variables

```css
--duration
--primary
--secondary
--accent
--glass
--text
--muted
--radius
```
