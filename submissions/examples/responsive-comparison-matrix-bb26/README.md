# Responsive Comparison Matrix

## What does this do?

This submission adds a responsive comparison matrix with sticky row labels, animated row entry, and readable status styling.

## How is it used?

Use `.matrix-card` as the scrollable container and `.matrix-row` for each comparable feature row.

```html
<section class="matrix-card">
  <div class="matrix-row">
    <strong>Dashboard cards</strong>
    <span class="good">Staggered lift</span>
  </div>
</section>
```

## Why is it useful?

It helps EaseMotion showcase motion decisions in dense documentation or product UI while preserving context on small screens.
