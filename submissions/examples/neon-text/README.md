# Neon Glow Text Effect

A cyberpunk-style flickering neon sign effect using text-shadow and alternating keyframes. Perfect for dark mode headers, retro themes, or nightlife branding.

## Demo

Open `demo.html` in your browser to see:
- Default pink neon glow
- Cyan, green, and purple color variants
- Flickering "faulty neon sign" animation
- Subtle glow for body text

## Usage

```html
&lt;!-- Basic neon text --&gt;
&lt;h1 class="neon-text" data-text="NEON"&gt;NEON&lt;/h1&gt;

&lt;!-- Color variants --&gt;
&lt;h1 class="neon-text neon-cyan" data-text="CYBER"&gt;CYBER&lt;/h1&gt;
&lt;h1 class="neon-text neon-green" data-text="MATRIX"&gt;MATRIX&lt;/h1&gt;
&lt;h1 class="neon-text neon-purple" data-text="FUTURE"&gt;FUTURE&lt;/h1&gt;
&lt;h1 class="neon-text neon-red" data-text="DANGER"&gt;DANGER&lt;/h1&gt;
&lt;h1 class="neon-text neon-orange" data-text="WARNING"&gt;WARNING&lt;/h1&gt;
&lt;h1 class="neon-text neon-yellow" data-text="CAUTION"&gt;CAUTION&lt;/h1&gt;
&lt;h1 class="neon-text neon-white" data-text="PURE"&gt;PURE&lt;/h1&gt;

&lt;!-- Flickering effect --&gt;
&lt;h1 class="neon-text neon-flicker" data-text="OPEN"&gt;OPEN&lt;/h1&gt;

&lt;!-- Subtle glow --&gt;
&lt;p class="neon-text neon-subtle" data-text="nightlife"&gt;nightlife&lt;/p&gt;

&lt;!-- With border box --&gt;
&lt;h1 class="neon-text neon-box" data-text="SIGN"&gt;SIGN&lt;/h1&gt;

&lt;!-- Size variants --&gt;
&lt;h1 class="neon-text neon-large" data-text="BIG"&gt;BIG&lt;/h1&gt;
&lt;h1 class="neon-text neon-small" data-text="mini"&gt;mini&lt;/h1&gt;
