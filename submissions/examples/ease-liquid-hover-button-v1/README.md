# Ease Liquid Hover Button

## Overview

This example introduces a reusable liquid hover button animation built with pure HTML and CSS. The button uses a liquid fill transition combined with a glowing border for a modern interactive effect.

---

## Features

- 🌊 Liquid fill animation
- ✨ Smooth hover transition
- 💡 Glow border effect
- 🎨 Multiple color variants
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Included Variants

- Default
- Blue
- Purple
- Green

---

## Usage

```html
<button class="ease-liquid-btn">
    Default
</button>

<button class="ease-liquid-btn blue">
    Blue
</button>

<button class="ease-liquid-btn purple">
    Purple
</button>

<button class="ease-liquid-btn green">
    Green
</button>
```

---

## Customization

You can easily customize:

- Border color
- Liquid fill color
- Transition speed
- Border radius
- Button size

---

## Browser Support

Works in all modern browsers supporting CSS transitions.
