# Word Spacing Grow on Hover

Word spacing expands when hovering — pure CSS.

## Features
- Smooth word-spacing transition on hover
- Pure CSS, no JavaScript

## Usage
```css
.ws-text { word-spacing: normal; transition: word-spacing 0.3s; }
.ws-text:hover { word-spacing: 0.5em; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
