# Stock Indicator

## What does this do?
Animated low-stock urgency indicator that pulses to draw attention to limited inventory items.

## How is it used?
Apply the `.stock-indicator` class with a `.low` modifier to show urgent low-stock status. The indicator pulses red when stock is critically low.

## Why is this useful?
Creates urgency cues in e-commerce interfaces without JavaScript, helping drive purchase decisions through visual inventory feedback.
