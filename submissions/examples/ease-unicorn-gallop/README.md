# Ease Unicorn Gallop Animation

A vibrant CSS animation simulating an active gallop rhythm using sinusoidal transform keyframes paired with a highly custom tracking magical trail layout.

## How to View
1. Load `demo.html` in your favorite internet browser tool.
2. The sequence runs automatically on an endless loop, or click "Gallop Again" to force reset.

## Technical Details
- **Architecture:** CSS3 Keyframe translations & scale matrix parameters
- **Easing Curve:** `cubic-bezier(0.45, 0, 0.55, 1)` applied continuously to smooth out bounding transitions.
