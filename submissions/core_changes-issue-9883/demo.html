Issue #9883 — CSS Containment Utility Classes

Proposed additions to core/utilities.css: content-visibility and contain utilities for rendering performance optimization.

Content Visibility

.ease-content-visibility-{auto, hidden, visible}

content-visibility: auto skips rendering of off-screen elements, improving initial load performance on long pages. Scroll down to see the effect — on a real page these sections would only render when scrolled into view.

content-visibility: auto — only renders when near the viewport

Hidden Section

content-visibility: hidden — like display:none but with retained rendering state

Visible Section

content-visibility: visible — default, always renders

ClassDescription
ease-content-visibility-autoSkip off-screen rendering automatically (major perf win)
ease-content-visibility-hiddenHide element but preserve rendering state (faster show/hide)
ease-content-visibility-visibleAlways render (default behavior)

Contain
