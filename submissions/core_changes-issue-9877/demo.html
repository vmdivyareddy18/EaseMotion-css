Text Wrap

.ease-text-balance / .ease-text-pretty / .ease-text-nowrap / .ease-text-wrap

The text-wrap CSS property controls how text wraps within its container. Modern browsers support balance (optimal line distribution) and pretty (orphan prevention).

Use Cases

Headlines & Titles: ease-text-balance prevents uneven line lengths in headings, making them look professionally typeset.

Article Body: ease-text-pretty avoids orphaned single words on the last line of paragraphs.

Navigation & Tags: ease-text-nowrap keeps button labels, nav items, and tags from breaking across lines.

Reset: ease-text-wrap restores default wrapping behavior.
