⌨️ ease-kbd

Keyboard key component — Issue #10103

Single Keys

Press Esc to close the dialog.

Press Enter to submit the form.

Press Tab to move to the next field.

Press Space to toggle the checkbox.

Press Delete to remove the selected item.

Key Combinations

Search: Ctrl + K

Save: Ctrl + S

DevTools: Ctrl + Shift + I

Mac screenshot: ⌘ + ⇧ + 3

Group Container

Using ease-kbd-group for natural spacing:

⌘ ⇧ P — Command Palette
Ctrl Alt Del

Size Variants

Small: F1 ease-kbd-sm

Default: F1

Large: F1 ease-kbd-lg

HTML Structure

<!-- Single key -->
<kbd class="ease-kbd">Esc</kbd>

<!-- Key combination -->
<kbd class="ease-kbd">Ctrl</kbd> + <kbd class="ease-kbd">K</kbd>

<!-- Group -->

  <kbd class="ease-kbd">⌘</kbd>

  <kbd class="ease-kbd">P</kbd>
</span>
