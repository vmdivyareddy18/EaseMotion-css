Issue #9881 — Text Utility Classes

Proposed additions to core/utilities.css: text-indent, text-justify, and vertical-align utilities.

Text Indent

.ease-text-indent-{0,1,2,4,6,8}

This paragraph has ease-text-indent-4. The first line is indented using var(--ease-space-4). Useful for article layouts, print-style typography, and blog content where paragraphs traditionally start with an indented first line.
This paragraph has ease-text-indent-0 — no indentation, useful for resetting.

Text Justify

.ease-text-justify-{auto,none,inter-word,inter-character,distribute}

ClassPreview
ease-text-justify-autoLorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
ease-text-justify-noneNo justification algorithm applied. Browser default spacing is used.
ease-text-justify-inter-wordAdjusts spacing between words for clean edges. Best for English and Latin text.
ease-text-justify-inter-characterAdjusts spacing between characters. Best for CJK (Chinese/Japanese/Korean) text.
ease-text-justify-distributeDistributes spacing between both characters and words for precise alignment.
