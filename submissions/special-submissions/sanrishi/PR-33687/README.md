# Sable Meadow

A unique landing page for Sable Meadow built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: contact, services, gallery, stats

## Setup
Open `demo.html` in any modern browser.

## Structure
```

  demo.html
  style.css

```
