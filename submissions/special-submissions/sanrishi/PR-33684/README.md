# Ivory Glen

A unique landing page for Ivory Glen built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: pricing, services, gallery, contact

## Setup
Open `demo.html` in any modern browser.

## Structure
```
ease-shadow-breeze-sablehub/
  demo.html
  style.css
```
