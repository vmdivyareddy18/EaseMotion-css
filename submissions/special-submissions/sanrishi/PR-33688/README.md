# Velvet Hub

A unique landing page for Velvet Hub built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: gallery, testimonials, pricing, team

## Setup
Open `demo.html` in any modern browser.

## Structure
```
ease-slate-studio-azurevale/
  demo.html
  style.css

```
