# Brass Ridge

A unique landing page for Brass Ridge built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: contact, testimonials, features, faq

## Setup
Open `demo.html` in any modern browser.

## Structure
```

  demo.html
  style.css
```
