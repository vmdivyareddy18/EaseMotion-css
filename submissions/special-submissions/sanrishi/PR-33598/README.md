# Shadow Forge

A distinctive landing page for Shadow Forge built with EaseMotion CSS framework.

## Live Demo

Open \demo.html\ in any modern web browser to view the landing page.

## Features

- **Pure EaseMotion CSS** - Uses the EaseMotion CDN
- **Responsive Layout** - Adapts to all viewports
- **Unique Composition** - Each page has a different section layout
- **Custom Theme** - Unique color scheme
- **Zero JavaScript** - No JS required
- **Multiple Sections** - 4 varied content sections

## Sections

- Hero with custom layout
  - team
  - services-4col
  - timeline
  - pricing

## EaseMotion Classes Used

- \ase-fade-in\ - Fade in animation
- \ase-slide-up\ - Slide up animation
- \ase-delay-{100,200,300}\ - Animation delays
- \ase-bg-surface\ - Background section
