# Azure Loom

A unique landing page for Azure Loom built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: gallery, team, services, stats

## Setup
Open `demo.html` in any modern browser.

## Structure
```
ease-mauve-studio-emberbreeze/
  demo.html
  style.css
  README.md
```
