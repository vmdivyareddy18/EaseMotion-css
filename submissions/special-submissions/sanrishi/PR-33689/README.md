# Frost Craft

A unique landing page for Frost Craft built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: cta, stats, faq, services

## Setup
Open `demo.html` in any modern browser.

## Structure
```

  demo.html
  style.css
```
