# Jade Breeze

A unique landing page for Jade Breeze built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: contact, stats, team, faq

## Setup
Open `demo.html` in any modern browser.

## Structure
```
ease-velvet-breeze-vaporforge/
  demo.html
  style.css

```
