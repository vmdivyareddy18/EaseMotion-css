# Shadow Forge

A unique landing page for Shadow Forge built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: features, testimonials, gallery, stats

## Setup
Open `demo.html` in any modern browser.
