# Crystal Studio

A unique landing page for Crystal Studio built with EaseMotion CSS framework.

## Features
- Pure EaseMotion CSS from CDN
- Responsive across all devices
- Zero JavaScript
- 4 unique sections: gallery, testimonials, services, cta

## Setup
Open `demo.html` in any modern browser.

## Structure
```
ease-azure-hub-ivorycoast/
  demo.html
  style.css
  README.md
```
