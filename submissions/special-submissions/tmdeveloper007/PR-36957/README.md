# ease-page-pagination-nav

**Issue:** #36863 | Pure HTML + CSS

A CSS pagination nav with animated page indicators, prev/next arrows,
ellipsis, dark mode, and reduced-motion safety.

## Tokens

`--pager-duration-base: 280ms` | `--pager-ease: cubic-bezier(0.22, 1, 0.36, 1)` | `--pager-primary`

## Variants

`.pager--sm` / `.pager--lg` sizes | `.pager__link--active` pop animation | `.pager__link--disabled`

## Accessibility

`<nav>` + `<ul>` + `<a>` | `aria-current="page"` | `prefers-reduced-motion` supported
