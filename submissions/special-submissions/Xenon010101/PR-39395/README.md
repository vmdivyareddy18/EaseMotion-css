# Animated Filterable Portfolio Gallery

## What does this do?
A CSS-only filterable portfolio gallery that lets users filter items by category using hidden radio buttons. Items animate in with a smooth fade-and-scale transition when the active filter changes.

## How is it used?
```html
<input type="radio" name="filter" id="filter-web">
<label for="filter-web">Web Design</label>

<div class="gallery-item web">...</div>
```

## Why is it useful?
Portfolio galleries are a core pattern for creative professionals. This component shows how EaseMotion CSS principles create a polished, interactive filtering experience without a single line of JavaScript — just clean HTML and CSS transitions.
