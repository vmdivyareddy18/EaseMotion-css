Issue #9875 — Multi-Column Layout Utility Classes

Proposed additions to core/utilities.css: column-rule, column-span, and column-fill utilities.

Column Rule

.ease-column-rule-{style, width, color}

Controls the vertical rule drawn between columns. The examples below use column-count: 3.

Column Span

.ease-column-span-all / .ease-column-span-none

Makes an element span across all columns — ideal for headings in multi-column layouts.

First paragraph in column layout. Lorem ipsum dolor sit amet, consectetur adipiscing elit.

This Heading Spans All Columns

Second paragraph after the spanning heading. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

Third paragraph. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris.

Column Fill

.ease-column-fill-auto / .ease-column-fill-balance

ClassDescription
ease-column-fill-autoColumns are filled sequentially — first column fills completely, then the next
ease-column-fill-balanceContent is balanced across columns (browser default)
